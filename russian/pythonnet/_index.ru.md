**Добро пожаловать на Aspose.Cells for Python via .NET**

Aspose.Cells for Python via .NET позволяет вашим приложениям выполнять широкий спектр задач по обработке электронных таблиц. Aspose.Cells for Python via .NET поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые ваша компания использует каждый день. Он также позволяет экспортировать файлы Excel® в PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, обычный текст и популярные форматы изображений, включая TIFF, JPG, PNG, BMP и SVG.
