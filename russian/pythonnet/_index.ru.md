**Добро пожаловать на Aspose.Cells для Python через .NET**

Aspose.Cells для Python через .NET позволяет вашим приложениям выполнять широкий спектр задач обработки электронных таблиц. Aspose.Cells для Python через .NET поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые ваша компания использует каждый день. Он также позволяет экспортировать файлы Excel® в форматы PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, обычный текст и популярные форматы изображений, включая TIFF, JPG, PNG, BMP и SVG.
