**Добро пожаловать в Aspose.Cells для Python via .NET**

Aspose.Cells для Python via .NET позволяет вашим приложениям выполнять широкий спектр задач по обработке электронных таблиц. Aspose.Cells для Python via .NET поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые использует ваш бизнес каждый день. Он также позволяет экспортировать файлы Excel® в форматы PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, обычный текст и популярные форматы изображений, включая TIFF, JPG, PNG, BMP и SVG.
