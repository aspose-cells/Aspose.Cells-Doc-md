## **Ограничения ознакомительной версии**

Оценочная версия Aspose.Cells Python через продукт .Net (без указания лицензии) обеспечивает полную функциональность продукта, но она ограничена открытием 100 файлов в одной программе и дополнительным рабочим листом с оценочным водяным знаком.

Ограничения показаны ниже:

- **Количество открытых файлов** (Aspose.Cells Python через .Net)
 При запуске вашей программы вы можете открыть только 100 файлов Excel, используя Aspose.Cells Python через библиотеку .Net. Если ваше приложение превысит это число, будет выдано исключение.


Более того, рабочий лист с оценочным водяным знаком всегда будет отображаться как активный рабочий лист в сгенерированном файле Excel с использованием Aspose.Cells Python через библиотеку. Только в лицензионной версии вы можете установить активный рабочий лист на другие рабочие листы. В выходной PDF-файл или файл изображения по номеру Aspose.Cells Python через водяной знак оценки будет вставлен в верхней части документа/изображения.

## **Применение лицензии в Aspose.Cells Python через компонент**

Лицензия представляет собой простой текстовый XML-файл, который содержит такие сведения, как название продукта, количество разработчиков, для которых он лицензируется, дату истечения срока действия подписки и т. д. Файл имеет цифровую подпись, поэтому не изменяйте файл. Даже непреднамеренное добавление дополнительного разрыва строки в файл сделает его недействительным. Вам необходимо установить лицензию перед использованием Aspose.Cells Python через, если вы хотите избежать ограничения его оценки. Для каждого приложения (или процесса) требуется установить лицензию только один раз. Лицензию можно загрузить из файла.

Aspose.Cells Python via пытается найти лицензию в явном пути.

Существует два распространенных метода применения лицензии из файла.

### **Применение лицензии с диска**

Самый простой способ установить лицензию — указать путь к файлу лицензии.

{{< highlight "csharp" >}}

 //Instantiate an instance of license and set the license file through its path

license = License();
 //For Windows
license.set_license("D:\Aspose.Cells.lic");
 //For Linux or MacOS
license.set_license("/home/yourusername/Aspose.Cells.lic"); 
{{< /highlight >}}

{{% alert color="primary" %}}

Когда вы вызываете множество_лицензионный метод, имя лицензии должно совпадать с именем вашего файла лицензии. Например, вы можете изменить имя файла лицензии на **Aspose.Cells.lic.xml**. Затем в своем коде вы должны использовать измененное имя лицензии (**Aspose.Cells.lic.xml**) для набора_лицензионный метод.

{{% /alert %}}
