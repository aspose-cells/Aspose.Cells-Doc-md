## **Ограничения оценочной версии**

Оценочная версия продукта Aspose.Cells для Python через .Net (без указанной лицензии) обеспечивает полную функциональность продукта, но она ограничена открытием 100 файлов в одной программе и использованием дополнительного листа с оценочным водяным знаком.

Ограничения показаны ниже:

- **Количество открытых файлов** (Aspose.Cells Python через .Net)
  При запуске вашей программы вы можете открыть только 100 файлов Excel с помощью Aspose.Cells Python через .Net библиотеку. Если ваше приложение превысит это количество, будет сгенерировано исключение.


Более того, в рабочем листе с оценочным водяным знаком всегда будет показываться активный рабочий лист в созданном файле Excel с использованием Aspose.Cells Python через библиотеку. Только в лицензированной версии можно установить активный рабочий лист на другие рабочие листы. В выходном PDF- или изображении Aspose.Cells Python через, оценочный водяной знак будет вставлен в верхнюю часть документа/изображения.

## **Применение лицензии в Aspose.Cells Python через компонент**

Лицензия представляет собой обычный текстовый файл XML, содержащий такие детали, как название продукта, количество лицензированных разработчиков, дата истечения подписки и т.д. Файл цифровой подписи, поэтому не изменяйте файл. Даже непреднамеренное добавление дополнительного переноса строки в файл приведет к его аннулированию. Необходимо установить лицензию перед использованием Aspose.Cells Python через, если вы хотите избежать ограничений оценки. Для каждого приложения (или процесса) требуется установить лицензию всего один раз. Лицензию можно загрузить из файла.

Aspose.Cells Python через пытается найти лицензию в явных местах пути.

Существует два общих метода для применения лицензии из файла.

### **Применение лицензии с диска**

Самый простой способ установить лицензию - поместить файл лицензии в явное место пути.

{{< highlight csharp >}}

# Instantiate an instance of license and set the license file through its path

license = License();
# For Windows
license.set_license("D:\Aspose.Cells.lic");
# For Linux or MacOS
license.set_license("/home/yourusername/Aspose.Cells.lic"); 
{{< /highlight >}}

{{% alert color="primary" %}}

Когда вы вызываете метод set_license, имя лицензии должно совпадать с именем вашего файла лицензии. Например, вы можете изменить имя файла лицензии на **Aspose.Cells.lic.xml**. Затем в вашем коде вы должны использовать измененное имя лицензии (**Aspose.Cells.lic.xml**) для метода set_license.

{{% /alert %}}
