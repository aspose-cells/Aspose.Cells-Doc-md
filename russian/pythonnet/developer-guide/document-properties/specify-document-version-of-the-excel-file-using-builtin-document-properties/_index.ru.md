## **Возможные сценарии использования**

Вы можете изменить **номер версии** файла Excel, щелкнув правой кнопкой мыши по файлу, выбрав Свойства > Детали и отредактировав поле **Номер версии**. Используйте свойство [**BuiltInDocumentPropertyCollection.document_version**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/document_version) для изменения его программно с помощью API Aspose.Cells для Python via .NET.

## **Указание версии документа Excel с использованием встроенных свойств документа**

В следующем образце кода создается книга и изменяются ее встроенные свойства документа, которые включают в себя название, авторов и номер версии. Пожалуйстa, посмотрите [выходной файл Excel](64716811.xlsx), сгенерированный кодом, и снимок экрана, который показывает измененный номер версии с использованием свойства [**BuiltInDocumentPropertyCollection.document_version**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/document_version).
