## **Как нарисовать линейку времени при преобразовании Excel в PDF с использованием библиотеки Aspose.Cells для Python Excel**
Если у вас есть файл Excel, к которому применена линейка времени, и вы хотите экспортировать Excel в PDF с учетом настроек линейки времени, Aspose.Cells для Python via .NET теперь поддерживает это по умолчанию. Просто экспортируйте файл Excel с линейкой времени в PDF, и сгенерированный PDF покажет примененную линейку времени.

Приведенный ниже образец кода загружает [образец Excel-файла](input.xlsx), содержащий существующую временную шкалу. Затем он сохраняет книгу как [файл PDF на выходе](out.pdf). На следующем снимке экрана сравниваются исходный файл Excel и сгенерированный PDF-файл.
