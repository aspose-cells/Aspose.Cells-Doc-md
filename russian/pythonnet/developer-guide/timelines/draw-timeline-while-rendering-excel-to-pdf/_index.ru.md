##  **Нарисуйте временную шкалу при рендеринге Excel до PDF**
Если у вас есть файл Excel, к которому применена временная шкала, и вы хотите экспортировать Excel в PDF с настройками временной шкалы, Aspose.Cells for Python via .NET теперь поддерживает это по умолчанию. Вы просто экспортируете файл Excel с временной шкалой в PDF, сгенерированный PDF покажет примененную временную шкалу.

 Следующий пример кода загружает[образец файла Excel](input.xlsx) который содержит существующую временную шкалу. Затем он сохраняет книгу как[выходной файл PDF](out.pdf). На следующем снимке экрана сравнивается исходный файл Excel и сгенерированный файл PDF.
