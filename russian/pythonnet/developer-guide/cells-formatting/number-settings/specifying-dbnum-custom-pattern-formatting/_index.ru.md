## **Возможные сценарии использования**

Aspose.Cells для Python via .NET поддерживает пользовательский формат *DBNum*. Например, если значение в ячейке 123, а вы задаете формат как [DBNum2][$-804]General, то оно будет отображаться как 壹佰贰拾叁. Вы можете задавать пользовательский формат ячейки с помощью метода [**Cell.get_style()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_style/#) и свойства [**Style.custom**](https://reference.aspose.com/cells/python-net/aspose.cells/style/custom).
