{{% alert color="primary" %}} 

Microsoft Excel поддерживает две системы дат: систему дат 1900 (стандартную систему дат, реализованную в Excel для Windows) и систему дат 1904. Система дат 1904 обычно используется для обеспечения совместимости с файлами Excel для Mac и является системой по умолчанию при использовании Excel для Mac. Вы можете установить систему дат 1904 для файлов Excel с помощью Aspose.Cells для Python via .NET.

{{% /alert %}} 

Для реализации 1904-ой годовой системы в Microsoft Excel (например, Microsoft Excel 2003):

1. В меню **Инструменты** выберите **Параметры**, а затем выберите вкладку **Расчет**.
1. Выберите опцию **1904 годовая система**.
1. Нажмите **ОК**.
