Aspose.Cells для Python via .NET предоставляет возможность применять эффект верхнего индекса (текст над линией базы) и нижнего индекса (текст под линией базы).

{{% /alert %}}

## **Работа с надстрочным и подстрочным индексами**

Примените эффект надстрочного, установив свойство [**is_superscript**](https://reference.aspose.com/cells/python-net/aspose.cells/font/is_superscript) объекта [**Style.font**](https://reference.aspose.com/cells/python-net/aspose.cells/font) в значении **true**. Для применения подстрочного установите свойство [**is_subscript**](https://reference.aspose.com/cells/python-net/aspose.cells/font/is_subscript) объекта [**Style.font**](https://reference.aspose.com/cells/python-net/aspose.cells/font) в значении **true**.
