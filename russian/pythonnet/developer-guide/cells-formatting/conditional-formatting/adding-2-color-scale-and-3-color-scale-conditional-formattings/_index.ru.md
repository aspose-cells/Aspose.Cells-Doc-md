{{% alert color="primary" %}}

Условное форматирование двухцветной шкалы и трехцветной шкалы добавляются одинаковым образом, за исключением свойства [**FormatCondition.color_scale.is_3_color_scale**](https://reference.aspose.com/cells/python-net/aspose.cells/colorscale/is_3_color_scale). Это свойство **false** для двухцветной шкалы и **true** для условного форматирования трехцветной шкалы.

{{% /alert %}}
