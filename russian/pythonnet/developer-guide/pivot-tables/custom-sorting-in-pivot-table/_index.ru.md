##  **Пользовательская сортировка в сводной таблице**
Используя Aspose.Cells for Python via .NET API, вы можете сортировать сводные таблицы по значениям полей. Следующий фрагмент кода загружает образец файла Excel и добавляет три сводные таблицы. Первая сводная таблица не имеет пользовательской сортировки, вторая сводная таблица отсортирована по значениям полей строк «Морепродукты», а третья сводная таблица отсортирована по значениям полей столбцов «28/07/2000».
