## ** Как установить пользовательскую сортировку в сводной таблице с использованием библиотеки Aspose.Cells для Excel Python**
С помощью API Aspose.Cells для Python via .NET вы можете сортировать сводные таблицы по значениям полей. В следующем фрагменте кода загружается образец excel-файла и добавляются три сводные таблицы. Первая сводная таблица без пользовательской сортировки, вторая сводная таблица отсортирована по значениям поля строк "Морепродукты", а третья сводная таблица отсортирована по значениям поля столбцов "28/07/2000".
