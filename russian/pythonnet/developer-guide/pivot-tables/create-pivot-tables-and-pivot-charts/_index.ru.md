Сводная таблица — это интерактивная сводка записей. Например, у вас могут быть сотни записей счетов в списке на листе. Сводная таблица позволяет суммировать счета по клиентам, продуктам или датам. С помощью Excel Microsoft можно быстро переупорядочить информацию в сводной таблице, перетаскивая кнопки в новое положение.

Сводная диаграмма — это интерактивное графическое представление данных в сводной таблице. Сводные диаграммы появились в Excel 2000. Использование сводной диаграммы еще больше упрощает понимание данных, поскольку сводная таблица автоматически создает промежуточные и итоговые суммы.

 Aspose.Cells for Python via .NET опоры[сводные таблицы](/cells/ru/python-net/create-pivot-tables-and-pivot-charts/) и[сводные диаграммы](/cells/ru/python-net/create-pivot-tables-and-pivot-charts/).

##  **Добавление сводных таблиц и диаграмм**

Aspose.Cells for Python via .NET предоставляет специальный набор классов, используемых для создания сводных таблиц. Эти классы используются для создания и установки объектов сводной таблицы, которые действуют как основные строительные блоки объекта сводной таблицы:

- PivotField — поле в отчете сводной таблицы.
- PivotFields — коллекция всех объектов PivotField в сводной таблице.
- Сводная таблица — отчет сводной таблицы на листе.
- Сводные таблицы — коллекция всех объектов сводных таблиц на листе.

###  **Подготовка к использованию Aspose.Cells for Python via .NET**
1.  Установить Aspose.Cells for Python via .NET с[пипи](https://pypi.org/project/aspose-cells-python/)используйте команду: $ pip install aspose-cells-python.
1. Вы также можете следовать пошаговым инструкциям по установке «Aspose.Cells for Python via .NET» в вашу среду разработчика.

1. Добавьте данные в ячейки листа, используя метод put_value объекта Cell. Вы также используете файл шаблона, уже заполненный данными. Данные будут использоваться в качестве источника данных сводной таблицы.
