##  **Опция «Показать страницы фильтра отчета»**
Excel поддерживает создание сводных таблиц, добавление фильтров отчетов и включение опции «Показать страницы фильтров отчетов». Aspose.Cells for Python via .NET также поддерживает эту функцию, позволяя включить опцию «Показать страницы фильтра отчета» в созданной сводной таблице.
