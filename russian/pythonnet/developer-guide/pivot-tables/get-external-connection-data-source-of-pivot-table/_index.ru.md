##  **Получить источник данных внешнего соединения сводной таблицы**

Aspose.Cells for Python via .NET обеспечивает возможность получения внешнего источника данных подключения сводной таблицы. Для этого API предоставляет[**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) собственность[**Сводная таблица**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/) сорт.[**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/)свойство возвращает[**Внешнее соединение**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/) объект. Следующий фрагмент кода демонстрирует использование[**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) свойство, чтобы получить источник данных внешнего соединения сводной таблицы.
