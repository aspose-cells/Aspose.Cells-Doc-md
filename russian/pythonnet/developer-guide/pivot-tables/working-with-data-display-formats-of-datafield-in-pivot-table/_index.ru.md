Aspose.Cells for Python via .NET поддерживает все форматы отображения данных DataField.

{{% /alert %}}

##  **Параметр формата отображения «Ранг от наименьшего к наибольшему» и «Ранг от наибольшего к наименьшему».**

Aspose.Cells for Python via .NET предоставляет возможность установить параметр формата отображения для сводных полей. Для этого API предоставляет[**PivotField.data_display_format**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfield/data_display_format/) свойство. Чтобы ранжировать от большего к меньшему, вы можете установить[**PivotField.data_display_format**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfield/data_display_format/)собственность[**PivotFieldDataDisplayFormat.RANK_LARGEST_TO_SMALLEST**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfielddatadisplayformat/). В следующем фрагменте кода демонстрируется настройка параметров формата отображения.
