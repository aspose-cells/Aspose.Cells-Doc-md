{{% alert color="primary" %}}

Вы можете установить различные параметры сводной таблицы, используя Aspose.Cells for Python via .NET. Одна из таких опций — «Для отображения пустых ячеек». При установке этой опции все пустые ячейки сводной таблицы отображаются как указанная строка.

{{% /alert %}}

##  **Настройка параметра сводной таблицы в Microsoft Excel**

Чтобы найти и установить эту опцию в Excel Microsoft:

1. Выберите сводную таблицу и щелкните правой кнопкой мыши.
1. Выберите *Параметры сводной таблицы**.
1.  Выберите**Макет и формат** вкладка.
1.  Выберите**Для пустых ячеек покажите** параметр и укажите строку.

##  **Настройка параметра сводной таблицы с помощью Aspose.Cells for Python via .NET**

Aspose.Cells for Python via .NET обеспечивает[**PivotTable.display_null_string**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/display_null_string/) и[**PivotTable.null_string**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/null_string/) свойства для настройки опции сводной таблицы «Показывать пустые ячейки».
