{{% alert color="primary" %}}

Отчеты на основе сводных таблиц полезны, но подвержены ошибкам, если целевые пользователи не обладают подробными знаниями Excel для настройки этих отчетов. В этих обстоятельствах организации захотят ограничить пользователям возможность изменять отчет на основе сводной таблицы. Общие функции сводных таблиц, такие как добавление дополнительных фильтров, срезов, полей или изменение порядка определенных элементов в отчете, в большинстве случаев не рекомендуются каждому пользователю. С другой стороны, эти пользователи также смогут обновлять отчет и использовать существующие фильтры или срезы. Aspose.Cells for Python via .NET предоставил разработчикам эту возможность, чтобы запретить пользователям изменять эти отчеты во время их создания. Для этой цели в Excel предусмотрена функция отключения ленты сводной таблицы, и то же самое предусмотрено в Aspose.Cells for Python via .NET, т.е. разработчик может отключить ленту, содержащую элементы управления для изменения этих отчетов.

{{% /alert %}}
