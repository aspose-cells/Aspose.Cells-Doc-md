Aspose.Cells для Python via .NET предоставляет функцию замены текста в рабочей книге с помощью регулярного выражения. Для этого API предоставляет свойство [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) класса [**ReplaceOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions). Установка [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) в значение **true** указывает, что искомый ключ будет регулярным выражением.

Приведенный ниже фрагмент кода демонстрирует использование свойства [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key), используя [образец файла Excel](101089318.xlsx). [Выходной файл](101089319.xlsx), сгенерированный данным фрагментом кода, прикреплен для справки.
