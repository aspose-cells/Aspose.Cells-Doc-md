## **Создание настраиваемых диаграмм**

До сих пор, когда мы обсуждали диаграммы, мы рассматривали стандартные диаграммы с их стандартными настройками форматирования. Мы только задаем источник данных, устанавливаем несколько свойств, и диаграмма создается с настройками по умолчанию. Но API Aspose.Cells для Python via .NET также поддерживают создание пользовательских диаграмм, позволяющих разработчикам создавать диаграммы с собственными настройками форматирования.

Разработчики могут создавать пользовательские диаграммы в режиме выполнения с помощью Aspose.Cells для Python via .NET.

Диаграмма состоит из серии данных. Каждая серия данных в Aspose.Cells для Python via .NET представлена объектом [**Series**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/series), в то время как объект [**SeriesCollection**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/seriescollection) служит коллекцией [**Series**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/series) объектов. При создании пользовательской диаграммы разработчики имеют свободу использовать разные типы диаграмм для различных серий данных (собранных в объект [**SeriesCollection**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/seriescollection)).

Приведенный ниже пример кода демонстрирует, как создать пользовательские графики. В этом примере мы собираемся использовать столбчатую диаграмму для первой серии данных и линейную диаграмму для второй серии. Результатом будет добавление столбчатой диаграммы, объединенной с линейной диаграммой, на лист.
