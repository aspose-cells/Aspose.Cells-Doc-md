## **Введение**
Диаграмма — это визуальное отображение информации. Aspose.Cells для Python via .NET позволяет разработчикам визуализировать информацию в виде диаграмм так же, как это делает Microsoft Excel. Представление данных в виде диаграмм всегда полезно для оперативных решений, позволяя быстро видеть сравнения, закономерности и тенденции. Создание диаграмм во время выполнения на основе данных в электронной таблице — одна из самых мощных функций Aspose.Cells.

## **Типы диаграмм**
Aspose.Cells для Python via .NET поддерживает практически все типы диаграмм, поддерживаемые Microsoft Excel. Диаграммы делятся на две основные категории:

- Стандартные типы
- Пользовательские типы
### **Стандартные типы**

- Column

- Line

- Scatter
- Area
- Doughnut
- Radar
- Funnel
- Surface 3D
- Bubble

Каждый из стандартных типов диаграмм подробно описан ниже.
#### **Гистограммы**
Гистограммы используют вертикальные столбцы (называемые столбцами), чтобы отображать разные значения одного или нескольких элементов данных. Гистограммы в основном используются для сравнения значений по различным категориям. Они идеально подходят для отображения изменений значения элемента со временем, например, при анализе бюджета.

Следующие столбчатые диаграммы поддерживаются в Aspose.Cells для Python via .NET:

- Clustered column chart
- Stacked column chart
- 100% stacked column chart
- 3D clustered column chart
- 3D stacked column chart
- 3D 100% stacked column chart
- 3D column chart
#### **Горизонтальные гистограммы**
Горизонтальные гистограммы практически тот же тип, что и вертикальные гистограммы. Разница заключается в том, что они используют горизонтальные столбики вместо вертикальных.

Следующие гистограммы поддерживаются в Aspose.Cells для Python via .NET:

- Clustered bar chart
- Stacked bar chart
- 100% stacked bar chart
- 3D clustered bar chart
- 3D stacked bar chart
- 3D 100% stacked bar chart
#### **Линейные графики**
Линейные графики используют линии для отображения информации. Эти графики очень полезны для показа временной тенденции с течением времени и часто используются для отображения изменений значения более чем одного элемента с течением времени.

Следующие линейные диаграммы поддерживаются в Aspose.Cells для Python via .NET:

- Simple line chart
- Stacked line chart
- 100% stacked line chart
- Line chart with data markers
- Stacked line chart with data markers
- 100% stacked line chart with data markers
- 3D line chart

Круговые диаграммы в основном используются для отображения информации, когда вы хотите представить различные части целого или проценты от общего.

Следующие круговые диаграммы поддерживаются в Aspose.Cells для Python via .NET:

Следующие диаграммы разброса поддерживаются в Aspose.Cells для Python via .NET:

Воронкообразная диаграмма поддерживается в Aspose.Cells для Python via .NET.

Диаграмма пузырьков может быть использована вместо точечной диаграммы, если ваши данные содержат три серии данных, каждая из которых содержит набор значений. Диаграммы пузырьков в основном используются для отображения финансовых данных.

Следующие пузырьковые диаграммы поддерживаются в Aspose.Cells для Python via .NET:

#### **Диаграмма дерева**
Диаграмма древовидной карты обеспечивает иерархический обзор ваших данных и упрощает обнаружение шаблонов, таких как наиболее продаваемые товары в магазине. Ветки дерева представлены прямоугольниками, и каждая подветка отображается в виде меньшего прямоугольника. Диаграмма древовидной карты отображает категории цветом и близостью и легко позволяет отображать большое количество данных, что было бы сложно с другими типами диаграмм.

Деревооблачная карта поддерживается в Aspose.Cells для Python via .NET.

#### **Диаграмма солнечной радуги**
Диаграмма солнечной радуги идеально подходит для отображения иерархических данных. Каждый уровень иерархии представлен одним кольцом или кругом, причем внутренний круг является верхним уровнем иерархии. Диаграмма солнечной радуги без иерархических данных (один уровень категорий) выглядит похоже на диаграмму круга. Однако диаграмма солнечной радуги с несколькими уровнями категорий показывает, как внешние кольца относятся к внутренним кольцам. Диаграмма солнечной радуги наиболее эффективна в том, чтобы показать, как одно кольцо можно разбить на составляющие его части, в то время как другой тип иерархической диаграммы - диаграмма дерева - идеально подходит для сравнения относительных размеров.

Карта солнечного луча поддерживается в Aspose.Cells для Python via .NET.

#### **Гистограммная диаграмма**
Гистограмма - это столбчатая диаграмма, показывающая частотные данные.

Следующие радарные диаграммы поддерживаются в Aspose.Cells для Python via .NET:

- Histogram chart
- Pareto chart

#### **Диаграмма ящика с усами**
Диаграмма ящика с усами показывает распределение данных по квартилям, выделяя среднее значение и выбросы. У ящиков могут быть вертикальные линии, называемые «усами». Эти линии указывают на изменчивость за пределами верхних и нижних квартилей, и любая точка за пределами этих линий или усов считается выбросом.

Диаграмма коробка с усами поддерживается в Aspose.Cells для Python via .NET.

#### **Диаграмма водопада**
Диаграмма водопада показывает накопленный итог при добавлении или вычитании значений. Она полезна для понимания того, как начальное значение (например, чистая прибыль) влияет на серию положительных и отрицательных значений.

Диаграмма потоковой воды поддерживается в Aspose.Cells для Python via .NET.
