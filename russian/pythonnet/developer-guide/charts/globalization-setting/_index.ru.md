В этой теме мы покажем вам, как преобразовать график в локализованное изображение и как установить регион локализации для графика.

{{% /alert %}}

## **Сценарий**

 Когда может понадобиться установить регион локализации для графика?

Если открыть файл XLSX с графиком в Excel с настройками региона на испанском языке, такие элементы, как название графика и легенда, отображаются на испанском. Однако сохранение этого графика как изображения с помощью Aspose.Cells может оставлять эти элементы по умолчанию на английском языке:

**![Глобальная проблема](GlobalIssue.png)**

Это происходит потому, что легенда графика в результатирующем изображении не совпадает с локализацией Excel. Вы можете решить эту проблему, настроив региональные параметры графика.

## **Поддерживаемые элементы**

Следующие элементы графика будут отображаться в соответствии с вашими настройками локализации:

| **Поддерживаемые элементы** | **Значение по умолчанию (на английском)** |
|-----------------------------|-----------------------------------|
| Название оси               | Название оси                     |
| Название единицы оси      | сотни, тысячи...                |
| Название заголовка графика | Заголовок графика               |
| Название увеличения легенды | Увеличение                     |
| Название уменьшения легенды | Уменьшение                     |
| Название общего итога легенды | Итоговая сумма                |
| Другое название             | Другие                         |
| Название серии             | Серия                          |
