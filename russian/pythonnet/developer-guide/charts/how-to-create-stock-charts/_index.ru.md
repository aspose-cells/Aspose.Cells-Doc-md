В этом абзаце вы узнаете, как создать диаграмму акций, включая четыре типа:
- **HLC** – Высокая-Низкая-Закрытая.
- **OHLC** – Открытие-Высокая-Низкая-Закрытая.
- **VHLC** – Объем-Высокая-Низкая-Закрытая.
- **VOHLC** – Объем-Открытие-Высокая-Низкая-Закрытая.

## **Что такое диаграмма акций?**

Гистограммы акций — это специфический тип диаграмм, используемых для отслеживания изменений цены торгуемых активов, таких как товары, акции и криптовалюты. Они позволяют видеть максимальные и минимальные значения за период, а также цены открытия и закрытия в одной диаграмме. Aspose.Cells для Python via .NET предлагает 4 типа гистограмм акций, и для их использования необходимо иметь соответствующие наборы данных и правильно выбрать столбцы в нужном порядке.

Следующий набор данных показывает ежедневную торговую информацию по акциям. Мы будем использовать эти данные для создания четырех типов диаграмм акций: График акций высокой-низкой-закрытой цены (HLC), График открытия-высокой-низкой-закрытой цены (OHLC), График акций объем-высокой-низкой-закрытой цены (VHLC) и График акций объем-открытие-высокая-низкая-закрытая цена (VOHLC).
