{{% alert color="primary" %}}

Вы можете добавлять пользовательские метки к точкам данных серии диаграммы. Aspose.Cells для Python via .NET предоставляет свойство [**ChartPoint.data_labels.text**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/datalabels/text) для добавления этих меток. В этой статье объясняется, как использовать это свойство для добавления пользовательских меток к точкам данных серии диаграммы.

{{% /alert %}}

Каждая пользовательская метка показывает **Название серии** и **Имя точки**. Вы можете использовать любой другой текст вместо этого.
