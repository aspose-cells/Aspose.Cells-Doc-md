## **Возможные сценарии использования**
Aspose.Cells для Python via .NET теперь поддерживает чтение и изменение диаграмм Microsoft Excel 2016, которых нет в Microsoft Excel 2013 или более ранних версиях.

## **Чтение и манипулирование диаграммами Excel 2016**
В следующем образцовом коде загружается [исходный файл Excel](22774101.xlsx), который содержит диаграммы Excel 2016 на первом листе. Он читает все диаграммы по очереди и изменяет их заголовок в соответствии с их типом диаграммы. Ниже показан скриншот исходного файла Excel до выполнения кода.
