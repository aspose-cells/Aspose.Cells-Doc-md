{{% alert color="primary" %}}

Если вы хотите установить текст заливки легенды графика на никуда, чтобы он не отображался внутри легенды графика, то установите [**LegendEntry.is_text_no_fill**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/legendentry/is_text_no_fill) в **true**.

{{% /alert %}}
