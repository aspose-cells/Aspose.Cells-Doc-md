Aspose.Cells для Python via .NET позволяет устанавливать градиенты, текстуры, узоры или изображения в качестве эффектов заливки для различных объектов, таких как область построения, область диаграммы или легенда диаграммы. Этот документ показывает, как добавить изображение в фон диаграммы.

{{% /alert %}}
