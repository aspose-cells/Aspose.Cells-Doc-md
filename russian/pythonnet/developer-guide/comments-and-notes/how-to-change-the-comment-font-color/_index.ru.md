{{% alert color="primary" %}}

Microsoft Excel позволяет добавлять комментарии к ячейкам для добавления дополнительной информации и выделения данных. Разработчики могут потребовать настройки комментария для определения настроек выравнивания, направления текста, цвета шрифта и др. Aspose.Cells для Python via .NET предоставляет API для выполнения этой задачи.

{{% /alert %}}

Aspose.Cells для Python via .NET предоставляет свойство [**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body) для цвета шрифта комментария. Следующий пример кода демонстрирует использование свойства [**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body) для установки направления текста в комментарии.
