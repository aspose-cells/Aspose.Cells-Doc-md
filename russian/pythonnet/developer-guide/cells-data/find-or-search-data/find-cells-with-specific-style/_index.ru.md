{{% alert color="primary" %}}

Иногда вам может понадобиться найти ячейки с определенным примененным стилем. Вы можете использовать Aspose.Cells для Python via .NET, чтобы найти все ячейки с общим стилем. Aspose.Cells для Python via .NET предоставляет свойство [**FindOptions.style**](https://reference.aspose.com/cells/python-net/aspose.cells/findoptions/style/), которое вы можете использовать, чтобы указать стиль для поиска ячеек.

{{% /alert %}}

Код в этом примере находит все ячейки, у которых такой же стиль, как у ячейки A1. После выполнения кода все ячейки, у которых такой же стиль, как у A1, содержат текст "Найдено".
