{{% alert color="primary" %}}

Вы можете использовать Aspose.Cells для Python via .NET, чтобы получить примененную валидацию для ячейки. Aspose.Cells для Python via .NET предоставляет метод [**Cell.get_validation()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) для этой цели. Если на ячейке не применена валидация, возвращается значение null.

Точно так же можно использовать метод [**Worksheet.validations.get_validation_in_cell**](https://reference.aspose.com/cells/python-net/aspose.cells/validationcollection/get_validation_in_cell/#int-int), чтобы получить примененную валидацию для ячейки, указав её индексы строки и столбца.

{{% /alert %}}
