## **Получить проверку ячейки в файлах ODS**

С помощью Aspose.Cells вы можете получить проверку, примененную к ячейке в файлах ODS. Для этого API предоставляет метод [**get_validation**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) класса [**Cell**](https://reference.aspose.com/cells/python-net/aspose.cells/cell).
