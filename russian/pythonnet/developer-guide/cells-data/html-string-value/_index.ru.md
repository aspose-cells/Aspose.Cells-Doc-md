{{% alert color="primary" %}}

Aspose.Cells для Python via .NET поддерживает преобразование ориентированного на Microsoft Excel HTML в формат XLS/XLSX. Это означает, что HTML, созданный в Microsoft Excel, может быть преобразован обратно в формат XLS/XLSX с использованием Aspose.Cells для Python via .NET.

Точно так же, если есть простой HTML, Aspose.Cells для Python via .NET может преобразовать его в HTML Rich Text. Aspose.Cells для Python via .NET предоставляет свойство [**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/), которое может принимать такой простой HTML и преобразовывать его в отформатированный текст ячейки.
