## **Возможные сценарии использования**

Вы можете форматировать срез в Microsoft Excel, установив его количество столбцов или установив стиль и т. д. Aspose.Cells для Python via .NET также позволяет вам сделать это, используя свойства [**Slicer.number_of_columns**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/number_of_columns/) и [**Slicer.style_type**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/style_type/).
