##  **Возможные сценарии использования**

 Если вы хотите удалить срез в Excel Microsoft, просто выберите его и нажмите кнопку*Удалить* кнопка. Аналогично, если вы хотите удалить его с помощью Aspose.Cells for Python via .NET API программно, используйте[**Рабочий лист.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/)метод. Это удалит срез с листа.

##  **Удаление слайсера**
