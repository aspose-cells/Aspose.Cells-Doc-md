## **Возможные сценарии использования**

Если вы хотите удалить срез в Microsoft Excel, просто выберите его и нажмите кнопку *Delete*. Точно так же, если вы хотите удалить его с использованием API Aspose.Cells для Python via .NET программно, используйте метод [**Worksheet.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/). Он удалит срез из листа книги Excel.
