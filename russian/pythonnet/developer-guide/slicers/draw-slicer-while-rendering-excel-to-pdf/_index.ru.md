##  **Нарисуйте срез при рендеринге Excel до PDF.**
Если у вас есть файл Excel, к которому применен срез, и вы хотите экспортировать Excel в PDF с настройками среза, Aspose.Cells for Python via .NET теперь поддерживает это по умолчанию. Вы просто экспортируете файл Excel со срезом в номер PDF, сгенерированный номер PDF покажет примененный срез.

 Следующий пример кода загружает[образец файла Excel](94044165.xlsx) который содержит существующий срез. Затем он сохраняет книгу как[выходной файл PDF](94044166.pdf). На следующем снимке экрана сравнивается исходный файл Excel и сгенерированный файл PDF.
