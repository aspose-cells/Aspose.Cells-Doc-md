## **Возможные сценарии использования**

Если вы хотите разорвать связь среза и сводной таблицы в Excel, вам нужно щелкнуть правой кнопкой мыши по срезу и выбрать элемент "Связи отчетов...". В списке опций вы можете выполнять действия с помощью флажка. Точно так же, если вы хотите разорвать связь среза и сводной таблицы используя Aspose.Cells для Python via .NET API программно, пожалуйста, используйте метод [**Slicer.remove_pivot_connection(pivot)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/remove_pivot_connection/#aspose.cells.pivot.PivotTable). Он разорвет связь среза и сводной таблицы.
