## **Возможные сценарии использования**

Обычно именованные диапазоны являются прямоугольными, с ячейками, непрерывными и смежными друг с другом. Но иногда может потребоваться использовать неупорядоченный диапазон ячеек, в котором ячейки не являются смежными. Aspose.Cells для Python via .NET поддерживает создание именованного диапазона с несмежными ячейками.
