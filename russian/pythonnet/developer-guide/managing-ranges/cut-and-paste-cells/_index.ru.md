## **Вырезать и вставить ячейки**

Aspose.Cells предоставляет возможность вырезать и вставить ячейки в пределах рабочего листа с помощью метода [**insert_cut_cells **](https://reference.aspose.com/cells/python-net/aspose.cells/cells/insert_cut_cells/) коллекции [**Cells**](https://reference.aspose.com/cells/python-net/aspose.cells/cells). Параметры [**insert_cut_cells **](https://reference.aspose.com/cells/python-net/aspose.cells/cells/insert_cut_cells/) принимают следующие параметры.

- [**Range**](https://reference.aspose.com/cells/python-net/aspose.cells/range): Диапазон ячеек для вырезания.
- Индекс строки: Индекс строки для вставки ячеек.
- Индекс столбца: Индекс столбца для вставки ячеек.
- [**ShiftType**](https://reference.aspose.com/cells/python-net/aspose.cells/shifttype): Направление сдвига столбцов.
