## **Возможные сценарии использования**
Aspose.Cells для Python via .NET предоставляет объект Range, который имеет различные утилиты, облегчающие пользователю работу с диапазонами Excel. В этой статье иллюстрируется использование следующих методов или свойств объекта Range.

- **адрес**

Получает адрес диапазона.

- **cell_count**

Получает общее количество ячеек в диапазоне.

- **get_offset**

Получает диапазон смещения.

- **вся_колонка**

Получает объект Range, представляющий весь столбец (или столбцы), содержащий указанный диапазон.

- **вся_строка**

Получает объект Range, представляющий всю строку (или строки), содержащий указанный диапазон.
