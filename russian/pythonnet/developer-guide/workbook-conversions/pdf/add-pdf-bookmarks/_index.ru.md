В этой статье представлена информация о том, как вставлять закладки PDF при преобразовании электронной таблицы в PDF.

Aspose.Cells for Python via .NET позволяет добавлять закладки на лету. PDF Закладки могут значительно улучшить навигацию по длинным документам. Добавляя ссылки на закладки в документ PDF, вы можете точно контролировать желаемый вид, вы не ограничены ссылками на страницу. Вы можете настроить точный вид, расположив целевую страницу, а затем создав закладку.

{{% /alert %}}
