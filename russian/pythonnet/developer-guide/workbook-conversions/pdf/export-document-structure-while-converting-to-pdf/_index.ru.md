Средства логической структуры PDF обеспечивают механизм включения информации, касающейся структуры содержимого документа, в файл PDF. Aspose.Cells for Python via .NET сохраняет информацию о структуре документа Excel Microsoft, например ячейку, строку, таблицу, лист, изображение, фигуру, верхний/нижний колонтитул и т. д.

С помощью опции [PdfSaveOptions.export_document_structure]https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/export_document_structure/) вы можете сохранить в файл с тегом PDF с экспортированной структурой документа.
