PDF logical structure facilities предоставляют механизм для включения информации о структуре содержимого документа в файл PDF. Aspose.Cells для Python via .NET сохраняет информацию о структуре из документа Microsoft Excel, такую как ячейка, строка, таблица, лист, изображение, форма, заголовок/нижний колонтитул и т. д.

С помощью параметра [PdfSaveOptions.export_document_structure](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/export_document_structure/) можно сохранить тегированный PDF с экспортированной структурой документа.
