{{% alert color="primary" %}} 

Aspose.Cells для Python via .NET позволяет конвертировать файлы XLS, содержащие изображения и диаграммы, в документы PDF. Aspose.Cells для Python via .NET API может работать независимо для конвертации электронных таблиц в PDF: для конвертации не требуется Aspose.PDF для .NET. Процесс может быть выполнен в памяти, так как он не зависит от временных или промежуточных XML-файлов. Это означает, что большие файлы Excel, например, содержащие изображения, диаграммы и другие объекты для рисования, могут быть конвертированы быстро и эффективно.
