Aspose.Cells for Python via .NET поддерживает преобразование XLS файлов, содержащих изображения и диаграммы, в документы PDF. Aspose.Cells for Python via .NET API может работать независимо для преобразования электронной таблицы в PDF: Aspose.PDF for .NET не требуется для преобразования. Этот процесс можно выполнить в памяти, поскольку он не зависит от временных или промежуточных XML-файлов. Это означает, что большие файлы Excel, например содержащие изображения, диаграммы и другие графические объекты, можно конвертировать быстро и эффективно.
