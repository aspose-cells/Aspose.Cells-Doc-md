##  **Преобразовать книгу Excel в JSON**

Aspose.Cells for Python via .NET API обеспечивает поддержку преобразования электронных таблиц в формат JSON. Чтобы экспортировать книгу на номер JSON, введите[**СохранитьФормат.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) в качестве второго параметра[**Рабочая книга.Сохранить**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) метод. Вы также можете использовать[**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) class, чтобы указать дополнительные настройки для экспорта листа в JSON.

 В следующем примере кода демонстрируется экспорт активного листа в Json с помощью[**СохранитьФормат.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) член перечисления. Пожалуйста, посмотрите код для конвертации[исходный файл](book1.xlsx) к[выходной файл Json](book1.Json)генерируется кодом для справки.
