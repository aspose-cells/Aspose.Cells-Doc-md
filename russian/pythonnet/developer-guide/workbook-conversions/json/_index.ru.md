## **Конвертировать книгу Excel в JSON**

API Aspose.Cells для Python via .NET предоставляет поддержку для конвертации электронных таблиц в формат JSON. Для экспорта книги в JSON передайте [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) вторым параметром метода [**Workbook.save**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions). Также вы можете использовать [**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) класс для указания дополнительных настроек экспорта рабочего листа в JSON.

Приведенный ниже пример кода демонстрирует экспорт активного рабочего листа в Json с использованием члена перечисления [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat). Пожалуйста, ознакомьтесь с кодом для конвертации [исходного файла](book1.xlsx) в [выходной файл Json](book1.Json) для справки.
