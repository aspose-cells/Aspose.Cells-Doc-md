## **Конвертировать Excel в HTML с заголовками**

Aspose.Cells для Python via NET предоставляет возможность экспорта заголовков при преобразовании Excel в HTML. Для этого вы можете установить свойство [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) в **true**. Значение по умолчанию свойства [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) это **false**.
