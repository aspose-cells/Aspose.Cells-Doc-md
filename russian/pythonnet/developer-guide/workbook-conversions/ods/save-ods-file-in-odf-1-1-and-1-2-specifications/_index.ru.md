{{% alert color="primary" %}}

 Aspose.Cells for Python via .NET поддерживает сохранение файла ODS (**Электронная таблица OpenDocument**) в формате ODF (**Формат OpenDocument**) версий 1.1 и 1.2. Aspose.Cells for Python via .NET есть[**OdsSaveOptions.is_strict_schema11**](https://reference.aspose.com/cells/python-net/aspose.cells/odssaveoptions/is_strict_schema11/)свойство, которое указывает использование спецификации ODF 1.1 для сохранения файлов ODS. Значение этого свойства по умолчанию — *false**, поэтому файл ODS, сохраненный без этого параметра, использует спецификации 1.2.

{{% /alert %}}
