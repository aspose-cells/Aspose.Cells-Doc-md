##  **Фон в файлах ODS**

Фон можно добавить на листы в файлах ODS. Фон может быть цветным или графическим. Фон не виден, когда файл открыт, но если файл напечатан как PDF, фон виден в сгенерированном PDF. Фон также виден в диалоговом окне предварительного просмотра печати.

Aspose.Cells for Python via .NET предоставляет возможность читать фоновую информацию и добавлять фон в файлы ODS.

##  **Прочтите справочную информацию из файла ODS.**

Aspose.Cells for Python via .NET обеспечивает[**ОдСтраницаФон**](https://reference.aspose.com/cells/python-net/aspose.cells.ods/odspagebackground) класс для управления фоном в ODS Files.
