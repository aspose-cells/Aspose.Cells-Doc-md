Aspose.Cells for Python via .NET также поддерживает эту функцию с помощью методов [**Workbook.protect_shared_workbook()**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/protect_shared_workbook) и [**Workbook.unprotect_shared_workbook()**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/unprotect_shared_workbook).
