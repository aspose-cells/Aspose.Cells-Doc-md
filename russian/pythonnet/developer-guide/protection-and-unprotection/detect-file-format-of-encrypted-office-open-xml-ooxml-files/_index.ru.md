**Office Open XML** (также известный как **OOXML** или **Microsoft Open XML** (MOX)) - это основанный на XML формат файлов, разработанный Microsoft для представления документов офиса, таких как таблицы, диаграммы, презентации и текстовые документы.

{{% /alert %}} 



API Aspose.Cells for Python via .NET предоставляет способ определения формата файла зашифрованных файлов **Microsoft Open XML**. Для идентификации типа файла используйте метод [FileFormatUtil.detect_file_format](https://reference.aspose.com/cells/python-net/aspose.cells/fileformatutil/detect_file_format/#str), показанный ниже в примере кода.
