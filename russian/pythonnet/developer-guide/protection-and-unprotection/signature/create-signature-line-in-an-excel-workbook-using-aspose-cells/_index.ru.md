## **Введение**

Microsoft Excel предоставляет возможность добавлять **Строку подписи** в рабочие книги Excel. Вы можете добавить строку подписи, нажав на вкладку **Вставка** и затем выбрав **Строка подписи** из группы **Текст**.

## **Как создать строку подписи для Excel**

Aspose.Cells для Python via .NET также предоставляет эту функцию и реализует свойство [**Picture.signature_line**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/picture/signature_line) для этой цели. В этой статье объясняется, как использовать это свойство для добавления линии подписи с помощью Aspose.Cells для Python via .NET.
