С увеличением популярности экрана с высоким разрешением изображения, сгенерированные по умолчанию при 96 DPI, часто выглядят размытыми и нечеткими. Чтобы обеспечить ясность на дисплеях с высоким разрешением, важно создавать изображения с более высокой плотностью точек (DPI). API Aspose.Cells for Python via .NET позволяет установить [**ImageOrPrintOptions.horizontal_resolution**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/horizontal_resolution/) и [**ImageOrPrintOptions.vertical_resolution**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/vertical_resolution/), что позволяет создавать высококачественные изображения из файлов Excel, которые выглядят острыми на дисплеях с высоким разрешением.
