## **Возможные сценарии использования**

В основном, вы указываете директорию шрифтов или список шрифтов для всех рабочих книг, но иногда необходимо указывать отдельный или приватный набор шрифтов для ваших рабочих книг. API Aspose.Cells for Python via .NET предоставляет класс [**IndividualFontConfigs**](https://reference.aspose.com/cells/python-net/aspose.cells/individualfontconfigs), который можно использовать для указания индивидуального или приватного набора шрифтов для вашей рабочей книги.
