Вы можете указать название задания или документа при печати вашей рабочей книги или листа, используя объекты WorkbookRender или SheetRender. API Aspose.Cells for Python via .NET предоставляет методы WorkbookRender.ToPrinter(printerName, jobName) и SheetRender.ToPrinter(printerName, jobName), которые позволяют указать название задания при печати вашей рабочей книги или листа.

{{% /alert %}}
