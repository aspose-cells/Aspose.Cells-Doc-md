{{% alert color="primary" %}}

Вы можете использовать Aspose.Cells for Python via .NET для экспорта вашего листа или диаграммы в изображение с желаемой шириной и высотой. Предоставляется метод [**ImageOrPrintOptions.set_desired_size()**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/set_desired_size) для установки желаемой ширины и высоты экспортируемого изображения. Ширина и высота задаются в пикселях.
