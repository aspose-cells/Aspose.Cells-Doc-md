## **Введение**

В этой статье мы узнаем, как замораживать верхние строки. Когда у вас большое количество данных под общим заголовком, вы не можете видеть заголовок при прокрутке рабочего листа вниз. Вы можете заморозить верхние строки, чтобы видеть эту замороженную часть даже при прокрутке остальных данных. Вы легко можете видеть заголовки в верхних строках.

## **Как закрепить строки в Excel**

**![Заморозить верхнюю строку(и) в Excel](Freeze-Rows.png)**


1. Если вы хотите заморозить верхние строки, то сначала выберите строку под строкой, которую нужно заморозить
2. Нажмите Вид > Заморозка областей.
3. В выпадающем меню нажмите "Заморозить верхнюю строку".
4. Если вы прокрутите вниз, первая строка всегда будет в верхнем виде.

**![Замороженная строка](Frozen-Row.png)**

Как видно на изображении, 1-я строка заморожена, она всегда остается сверху при прокрутке вниз.

Замораживание строк позволяет просматривать ваши большие данные без отслеживания меток строк.




## **Как закрепить строки с помощью библиотеки Aspose.Cells для Python Excel**
Простое закрепление строк с помощью Aspose.Cells для Python via .NET. Используйте метод [**Worksheet.freeze_panes**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/freeze_panes/#str-int-int) для закрепления выбранных строк.
1. Создать рабочую книгу для открытия файла или создать пустой файл.
2. Заморозить первую строку с методом Worksheet.FreezePanes().
