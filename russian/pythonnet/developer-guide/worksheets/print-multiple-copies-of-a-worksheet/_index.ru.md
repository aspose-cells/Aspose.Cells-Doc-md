## **Печать нескольких копий листа книги Excel**

Aspose.Cells для Python via .NET позволяет распечатывать несколько копий листа, используя метод [**SheetRender.to_printer(printer_settings)**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/sheetrender/to_printer/#aspose.pydrawing.printing.PrinterSettings). Следующий пример демонстрирует использование метода [**SheetRender.to_printer(printer_settings)**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/sheetrender/to_printer/#aspose.pydrawing.printing.PrinterSettings) для печати нескольких копий листа.
