## **Возможные сценарии использования**

*Свойство Sheet.SheetId* находится внутри пространства имён *DocumentFormat.OpenXml.Spreadsheet* и является частью OpenXml. Это свойство и его значение можно просмотреть внутри файла *workbook.xml*, как показано на следующем скриншоте. Aspose.Cells для Python via .NET предоставляет эквивалентное свойство как [**Worksheet.tab_id**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/tab_id).
