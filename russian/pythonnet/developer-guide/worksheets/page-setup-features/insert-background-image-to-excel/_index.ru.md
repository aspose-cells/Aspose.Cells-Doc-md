Вы можете сделать лист более привлекательным, добавив изображение в качестве фонового рисунка. Эта функция может быть довольно эффективной, если у вас есть специальная корпоративная графика, дополняющая фон, не заслоняя данные на листе. Вы можете установить фоновое изображение для листа с помощью API Aspose.Cells для Python via .NET.

{{% /alert %}} 

## **Как установить фон листа в Microsoft Excel**

Чтобы установить фоновое изображение листа в Microsoft Excel (например, Microsoft Excel 2019):

1. Из меню **Макет страницы** найдите опцию **Настройка страницы**, затем щелкните опцию **Фон**.
1. Выберите изображение для установки фонового изображения листа.
