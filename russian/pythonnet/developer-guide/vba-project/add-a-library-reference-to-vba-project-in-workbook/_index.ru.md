Это можно сделать с помощью метода [**VbaProject.references.add_registered_reference()**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaprojectreferencecollection/add_registered_reference/#str-str) Aspose.Cells for Python via .NET.

## **Добавить ссылку на библиотеку в проект VBA в Microsoft Excel**

В Microsoft Excel вы можете добавить ссылку на библиотеку в проект VBA, нажав **Инструменты > Ссылки...** вручную.
