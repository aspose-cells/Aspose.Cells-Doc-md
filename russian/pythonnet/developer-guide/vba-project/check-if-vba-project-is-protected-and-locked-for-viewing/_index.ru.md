## Проверьте, защищен ли VBA-проект и заблокирован ли для просмотра в Python

Aspose.Cells для Python via .NET позволяет проверить, защищен ли VBA (Visual Basic for Applications) проект Excel-файла и заблокирован ли он для просмотра. Для этого API предоставляет свойство [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/). Если он заблокирован для просмотра, то свойство [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/) возвращает **true**.
