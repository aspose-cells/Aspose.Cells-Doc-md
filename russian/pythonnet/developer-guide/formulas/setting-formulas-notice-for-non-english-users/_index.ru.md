{{% alert color="primary" %}} 

Aspose.Cells для Python via .NET поддерживает большинство формул/функций, входящих в состав Microsoft Excel. Разработчики могут использовать эти формулы через API или [дизайнерские таблицы](/cells/ru/net/what-is-a-designer-spreadsheet/). Aspose.Cells для Python via .NET поддерживает огромный набор математических, строковых, логических, дат/времени, статистических, баз данных, поиска и ссылочных формул. Формулы должны задаваться в английском (US) стиле.

{{% /alert %}} 

## **Примечание для пользователей, не говорящих на английском языке**
Есть два совета, которых должны придерживаться пользователи, не использующие английский язык, при создании формул с Aspose.Cells для Python via .NET:

1. Формулы должны вводиться на английском языке. Например, используйте английское "=SUM()", а не немецкое "=SUMME()".
1. Всегда используйте запятую (,) для разделения параметров функций. В некоторых языковых настройках разделитель параметров функции — точка с запятой, но Aspose.Cells для Python via .NET использует английский стиль запятой. Например, используйте "=IF (C5=0,0,C3/C4)" вместо "=IF(C5=0;0;C3/C4)".
