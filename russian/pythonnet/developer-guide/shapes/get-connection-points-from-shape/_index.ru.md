Иногда возникает необходимость в получении точек соединения формы для выравнивания или размещения форм в нужном месте. Для этой цели требуется получить все точки соединения. Нижеприведенный код может использоваться для получения списка точек соединения формы с использованием свойства [Shape.ConnectionPoints](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/connection_points).
