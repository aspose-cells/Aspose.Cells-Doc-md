## **Возможные сценарии использования**
Aspose.Cells для Python via .NET предоставляет свойство [Shape.reflection](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/reflection) вместе с классом [ReflectionEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect) для работы с эффектом отражения фигуры или диаграммы. Класс [ReflectionEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect) содержит следующие свойства, которые можно установить для достижения различных результатов в соответствии с требованиями приложения.

- [размытость](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/blur)
- [направление](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/direction)
- [расстояние](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/distance)
- [направление исчезания](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/fade_direction)
- [tot_with_shape](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/rot_with_shape)
- [размер](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/size)
- [прозрачность](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/transparency)
- [тип](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/type)
