## **Возможные сценарии использования**
Aspose.Cells для Python via .NET предоставляет свойство [Shape.shadow_effect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/shadow_effect) вместе с классом [ShadowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect) для работы с эффектом тени фигуры или диаграммы. Класс [ShadowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect) содержит следующие свойства, которые можно установить для достижения различных результатов в соответствии с требованиями приложения.

- [угол](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/angle)
- [размытость](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/blur)
- [цвет](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/color)
- [расстояние](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/distance)
- [тип_предустановки](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/preset_type)
- [размер](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/size)
- [прозрачность](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/transparency)
