## **Возможные сценарии использования**

Иногда необходимо отображать текст шрифтом восточноазиатских языков, например японским, китайским, тайским и т.д. Aspose.Cells for Python via .NET предоставляет свойство [**TextOptions.far_east_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/far_east_name), которое позволяет указать название шрифта для восточноазиатских языков. Кроме того, можно указать латинский шрифт, используя свойство [**TextOptions.latin_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/latin_name).
