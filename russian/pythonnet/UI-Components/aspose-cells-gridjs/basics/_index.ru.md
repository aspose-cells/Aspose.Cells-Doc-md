## Основы GridJs

Aspose.Cells.GridJs - это библиотека, которая позволяет пользователям разрабатывать переносимое веб-приложение для просмотра или редактирования электронных таблиц быстро и легко. 

## Почему использовать Aspose.Cells.GridJs


- Позволяет пользователям создавать, редактировать, форматировать, совместно использовать и безопасно обмениваться электронными таблицами с обновлениями в реальном времени, поддержкой формул и инструментами визуализации данных, аналогичными традиционным настольным приложениям.
- Поддерживает ввод и редактирование данных, форматирование, навигацию по электронной таблице, вычисление формул, манипулирование данными, диаграммы и визуализацию, импорт и экспорт, безопасность, дополнения и настройку для разработчиков, чтобы настраивать редактор под конкретные потребности бизнеса.

## Особенности


- Импорт, просмотр и редактирование популярных форматов электронных таблиц.
- Импорт, просмотр и редактирование популярных форматов электронных таблиц.
- Экспорт электронных таблиц в различные поддерживаемые форматы файлов.
- Отображение и управление изображениями, формами или файлами диаграмм.
- Выполнение настройки дизайна и макета Grid.
- Управление несколькими листами электронной таблицы.

- Создание и вычисление формул Excel®.

## Общее использование

Ниже приведены основные этапы разработки веб-приложения GridJs.

- Установите каталог кэширования файлов с помощью Config.set_file_cache_directory(`ваш_каталог_хранения`)
- Установите лицензию с помощью Config.set_license(`путь_к_лицензии`)
- Установите маршрут к изображению с помощью GridJsWorkbook.set_image_url_base(`маршрут_для_просмотра_изображения`)
- Настройте действие маршрута для получения `json` из файла электронной таблицы. Для этого вы можете использовать API `GridJsWorkbook.ImportExcelFile` и `GridJsWorkbook.ExportToJson`, `GridJs` автоматически сохранит файл таблицы в кэше.
- Настройте действие маршрута для получения `json` для операции обновления. Вы можете использовать API `GridJsWorkbook.UpdateCell`, `GridJs` выполнит операцию обновления в кэше и вернет обновленный `json`.
- Настройте действие маршрута для получения файла из кэша, чтобы получить zip-файл изображений/форм или файл таблицы в кэше.
- Настройте действие маршрута для загрузки электронной таблицы. Вы можете использовать API `GridJsWorkbook.SaveToCacheWithFileName`.
