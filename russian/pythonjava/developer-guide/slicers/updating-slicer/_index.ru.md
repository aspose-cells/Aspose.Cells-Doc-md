## **Обновление слайсера**
Aspose.Cells for Python via Java поддерживает обновление слайсеров. Для этого API предоставляет свойство Slicer.SlicerCache.SlicerCacheItems, которое используется для выбора или отмены выбора элементов среза. Следующий фрагмент кода загружает[образец файла Excel](106365050.xlsx)который содержит слайсер. Он отменяет выбор 2-го и 3-го элементов слайсера и обновляет слайсер с помощью метода Slicer.refresh(). Затем он сохраняет книгу как[выходной файл Excel](106365051.xlsx). На следующем снимке экрана показано влияние примера кода на пример файла Excel. Как вы можете видеть на снимке экрана, обновление слайсера с выбранными элементами также соответствующим образом обновило сводную таблицу.
