## **Возможные сценарии использования**
Слайсеры используются для быстрой фильтрации данных. Их можно использовать для фильтрации данных как в таблице, так и в сводной таблице. Microsoft Excel позволяет создать срез, выбрав таблицу или сводную таблицу, а затем щелкнув значок*Вставка > Слайсер*Aspose.Cells для Python через Java предоставляет метод Worksheet.getSlicers().add() для создания слайсера.
## **Создать слайсер для сводной таблицы**
