## **Draw Slicer при рендеринге Excel на PDF**
Aspose.Cells for Python via Java поддерживает экспорт Excel в PDF с настройками слайсера. API поддерживает это по умолчанию, вы просто экспортируете файл Excel со слайсером в PDF, сгенерированный PDF покажет примененный слайсер. Следующий фрагмент кода загружает[образец файла Excel](106365058.xlsx)который содержит срез и сохраняет книгу как[выходной файл PDF](106365052.xlsx). На следующем снимке экрана сравнивается исходный файл Excel и сгенерированный файл PDF.
