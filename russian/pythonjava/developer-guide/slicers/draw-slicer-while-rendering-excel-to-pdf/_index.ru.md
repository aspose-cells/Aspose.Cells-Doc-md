## **Нарисуйте фильтр при рендеринге Excel в PDF**
Aspose.Cells для Python via Java поддерживает экспорт Excel в PDF с настройками среза. API поддерживает это по умолчанию, вы просто экспортируете Excel-файл со срезом в PDF, и сгенерированный PDF покажет примененный срез. В следующем фрагменте кода загружается [образец Excel-файла](106365058.xlsx), который содержит срез, и книга сохраняется в формате [PDF](106365052.xlsx). На следующем скриншоте сравнивается исходный файл Excel и сгенерированный файл PDF.
