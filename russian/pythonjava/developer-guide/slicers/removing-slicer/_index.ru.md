## **Удаление слайсера**
 Чтобы удалить слайсер в Microsoft Excel, вы просто выбираете слайсер и нажимаете кнопку*Удалить* кнопка. Чтобы добиться сохранения с помощью Aspose.Cells for Python via Java, используйте метод Worksheet.getSlicers().remove(). Это удалит слайсер с рабочего листа.

Следующий фрагмент кода загружает[образец файла Excel](106364970.xlsx)который содержит существующий слайсер. Он обращается к слайсеру, удаляет его и сохраняет[выходной файл Excel](106364971.xlsx). На следующем снимке экрана показан слайсер, который будет удален.
