## **Удаление срезки**
Чтобы удалить срез в Microsoft Excel, просто выберите срез и нажмите кнопку *Удалить*. Чтобы осуществить это с помощью Aspose.Cells для Python via Java, используйте метод Worksheet.getSlicers().remove(). Он удалит срез с листа. 

В следующем фрагменте кода загружается [образец Excel-файла](106364970.xlsx), который содержит существующий срез. Он получает доступ к срезу, удаляет его и сохраняет [выходной Excel-файл](106364971.xlsx). На следующем скриншоте показан срез, который будет удален.
