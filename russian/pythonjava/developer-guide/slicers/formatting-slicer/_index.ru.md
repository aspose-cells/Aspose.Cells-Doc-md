## **Форматирование фильтра**
С помощью Microsoft Excel можно форматировать срез, установив количество его столбцов, стили и т.д. Aspose.Cells для Python via Java предоставляет свойства Slicer.NumberOfColumns и Slicer.StyleType для достижения этого.

В следующем фрагменте кода загружается [образец Excel-файла](106364964.xlsx), который содержит срез. Он получает доступ к срезу, устанавливает количество его столбцов и тип стиля, а затем сохраняет его в [Excel-файл](106364965.xlsx). Снимок экрана показывает, как выглядит срез после выполнения образца кода.
