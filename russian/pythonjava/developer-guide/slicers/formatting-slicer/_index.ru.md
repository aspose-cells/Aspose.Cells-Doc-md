## **Форматирование слайсера**
Используя Microsoft Excel, вы можете отформатировать срез, задав количество столбцов, стилей и т. д. Aspose.Cells for Python via Java предоставляет свойства Slicer.NumberOfColumns и Slicer.StyleType для достижения этой цели.

Следующий фрагмент кода загружает[образец файла Excel](106364964.xlsx)который содержит слайсер. Он обращается к слайсеру, устанавливает количество столбцов и тип стиля и сохраняет его как[выходной файл Excel](106364965.xlsx). На скриншоте показано, как выглядит слайсер после выполнения примера кода.
