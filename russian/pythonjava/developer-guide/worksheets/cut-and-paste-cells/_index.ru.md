## **Вырезать и вставить Cells**
Aspose.Cells для Python через Java предоставляет возможность вырезать и вставлять ячейки. Для этого API предоставляет[вставкаCutCells](https://reference.aspose.com/cells/python/asposecells.api/cells#insertCutCells\(com.aspose.cells.Range,%20int,%20int,%20int\)) метод[Cells](https://reference.aspose.com/cells/python/asposecells.api/Cells)коллекция.[вставкаCutCells](https://reference.aspose.com/cells/python/asposecells.api/cells#insertCutCells\(com.aspose.cells.Range,%20int,%20int,%20int\)) принимает следующие параметры.

- [Диапазон](https://reference.aspose.com/cells/python/asposecells.api/Range)Диапазон ячеек, которые нужно вырезать.
- Индекс строки: индекс строки для вставки ячеек.
- Индекс столбца: индекс столбца для вставки ячеек.
- [ShiftType](https://reference.aspose.com/cells/python/asposecells.api/ShiftType): Направление сдвига столбцов.
