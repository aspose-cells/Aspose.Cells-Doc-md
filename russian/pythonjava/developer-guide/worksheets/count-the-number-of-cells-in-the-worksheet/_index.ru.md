## **Подсчитайте количество ячеек на рабочем листе**
Aspose.Cells for Python via Java позволяет подсчитать количество ячеек на листе. Для этого API предоставляет[Cells.getCount](https://reference.aspose.com/cells/python/asposecells.api/cells#Count)() или же[Cells.getCountLarge](https://reference.aspose.com/cells/python/asposecells.api/cells#CountLarge)() характеристики.

В следующем фрагменте кода показано, как использовать[Cells.getCount](https://reference.aspose.com/cells/python/asposecells.api/cells#Count)() или же[Cells.getCountLarge](https://reference.aspose.com/cells/python/asposecells.api/cells#CountLarge)(), чтобы получить количество ячеек на листе.
