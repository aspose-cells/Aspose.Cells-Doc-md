## **Преобразовать Excel в Markdown**
Aspose.Cells для Python via Java поддерживает преобразование файлов Excel в формат Markdown. Для экспорта активного рабочего листа в Markdown передайте [SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN) в качестве второго параметра метода [Workbook.Save](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)). Также вы можете использовать класс [MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions) для указания дополнительных параметров для экспорта рабочих листов в Markdown.
