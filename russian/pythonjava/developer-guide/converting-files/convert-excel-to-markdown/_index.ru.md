## **Преобразование Excel в Markdown**
Aspose.Cells for Python via Java поддерживает преобразование файлов Excel в формат Markdown. Чтобы экспортировать активный рабочий лист в Markdown, передайте[СохранитьФормат.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN)в качестве второго параметра[Книга. Сохранить](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)) метод. Вы также можете использовать[MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions)class, чтобы указать дополнительные параметры для экспорта листов в Markdown.

В следующем примере кода демонстрируется экспорт активного рабочего листа в Markdown с помощью[СохранитьФормат.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN)член перечисления. Пожалуйста, смотрите[выходной файл уценки](Book1.txt)сгенерированный кодом для справки.
