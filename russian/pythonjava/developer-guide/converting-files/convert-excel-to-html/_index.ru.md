## **Конвертировать Excel в HTML**
API Aspose.Cells для Python via Java предоставляет возможность преобразования файлов Excel в документы HTML. Для этого API предоставляет класс [HtmlSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/HtmlSaveOptions), который позволяет разработчикам контролировать несколько аспектов вывода HTML.
