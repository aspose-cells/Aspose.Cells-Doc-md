## **Преобразование Excel в HTML**
 Aspose.Cells for Python via Java API предоставляет возможность конвертировать файлы Excel в документы HTML. Для этого API предоставляет[Хтмлсавеоптионс](https://reference.aspose.com/cells/python/asposecells.api/HtmlSaveOptions) class, который позволяет разработчикам контролировать несколько аспектов вывода HTML.

Следующий фрагмент кода демонстрирует использование[Хтмлсавеоптионс](https://reference.aspose.com/cells/python/asposecells.api/HtmlSaveOptions)класс для экспорта файлов Excel в формат HTML.
