##  **Преобразовать Excel в HTML**
 Aspose.Cells for Python via Java API предоставляет возможность конвертировать файлы Excel в документы HTML. Для этого номер API предоставляет[HtmlSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/HtmlSaveOptions) класс, который позволяет разработчикам контролировать некоторые аспекты вывода HTML.
