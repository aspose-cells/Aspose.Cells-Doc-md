## **Преобразовать Excel в изображение**
Aspose.Cells для Python via Java поддерживает преобразование файлов Excel в различные формы изображений. Для этого API предоставляет классы [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender) и [ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions). Этот класс представляет рабочий лист, который будет преобразован в изображение. Класс [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender) предоставляет метод [toImage()](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage\(int,%20java.lang.String\)) для преобразования рабочего листа в файл изображения. Поддерживаются форматы BMP, PNG, JPEG, TIFF и EMF.
