## **Преобразование текста в столбцы**
Вы можете преобразовать текст в столбцы, используя Microsoft Excel. Эта функция доступна из*Инструменты данных*под*Данные*вкладка Чтобы разделить содержимое столбца на несколько столбцов, данные должны содержать определенный разделитель, например запятую (или любой другой символ), на основе которого Microsoft Excel разделяет содержимое ячейки на несколько ячеек. Aspose.Cells также предоставляет эту функцию через[тексттоколоннс](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) метод. Следующий фрагмент кода демонстрирует использование[тексттоколоннс](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) путем преобразования текста в столбцы с пробелом в качестве разделителя.
