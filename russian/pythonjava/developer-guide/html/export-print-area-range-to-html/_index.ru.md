## **Экспортировать диапазон области печати в HTML**
Aspose.Cells для Python via Java поддерживает экспорт только области печати, т.е. выбранного диапазона ячеек, вместо всего листа в HTML. Для этого API предоставляет свойство ExportPrintAreaOnly в HtmlSaveOptions. Установка этого свойства в True экспортирует только область печати.
