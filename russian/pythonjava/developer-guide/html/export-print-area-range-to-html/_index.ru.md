## **Экспорт диапазона области печати в HTML**
Aspose.Cells for Python via Java поддерживает экспорт только области печати, т.е. выделенного диапазона ячеек, а не всего ост HTML. Для этого API предоставляет[Хтмлсавеоптионс. Экспортпринтареаонли](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) имущество. Установка этого свойства в**Истинный** будет экспортировать только область печати.

Следующий пример кода демонстрирует использование[Хтмлсавеоптионс. Экспортпринтареаонли](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly)свойство для экспорта только области печати на HTML.
