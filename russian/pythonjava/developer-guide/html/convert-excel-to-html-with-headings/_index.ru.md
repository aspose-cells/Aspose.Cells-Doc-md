## **Конвертировать Excel в HTML с заголовками**
Aspose.Cells предоставляет опцию экспорта заголовков строк и столбцов при преобразовании Excel в HTML. Для этого используется свойство [HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings), предоставляемое API. Значение по умолчанию [HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) - **False**. Передайте **True** в качестве параметра, чтобы отображать заголовки в выходном файле HTML. На следующем изображении показан выходной файл, сгенерированный следующим кодом.
