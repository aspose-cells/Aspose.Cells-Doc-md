## **Форматирование объекта списка**
Таблица - это серия строк и столбцов, содержащих связанные данные, управляемые независимо от данных в других строках и столбцах. По умолчанию в каждом столбце таблицы в заголовке строки включено фильтрование, чтобы можно было быстро фильтровать или сортировать данные объекта списка. Вы можете добавить итоговую строку (специальную строку в списке, предоставляющую выбор агрегатных функций, полезных для работы с числовыми данными) в объект списка, предоставляющую раскрывающийся список агрегатных функций для каждой ячейки строки итогов.

Aspose.Cells поддерживает форматирование объектов списка. Для этого API предоставляет классы ListObject и TableStyleType. Класс TableStyleType содержит константы, которые представляют встроенные стили таблиц.
