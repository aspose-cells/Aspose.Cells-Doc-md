## **Системные Требования**
 Aspose.Cells for Python via Java не зависит от платформы API и может использоваться на любой платформе (Windows, Linux и MacOS), где[Python](https://www.python.org/downloads/) установлено. Перед настройкой установки на компьютере должна быть установлена версия Java 8 или выше.

## **Python Версия**
- Python 3,5 или выше
## **Java Версия**
- Java 1,8 или выше

## **Установка**
### **Windows:**
- **Установите Java и установите переменные среды JAVA_HOME и PATH.**
Например:
{{< highlight "java" >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}
  
- **Установите Aspose.Cells for Python via Java из pypi**
 Вы можете легко использовать Aspose.Cells for Python via Java от[пипи](https://pypi.org/project/aspose-cells/) с помощью следующей команды.
{{< highlight "java" >}}

 $ pip install aspose-cells

{{< /highlight >}}
