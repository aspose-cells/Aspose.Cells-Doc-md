## **Системные требования**
Aspose.Cells для Python via Java является кроссплатформенным API и может использоваться на любой платформе (Windows, Linux и MacOS), где установлен [Python](https://www.python.org/downloads/). На компьютере должна быть установлена Java 8 или более новые версии перед установкой.

## **Версия Python**
- Python 3.5 или выше
## **Версия Java**
- Java 1.8 или выше

## **Установка**
### **Windows:**
- **Установите Java и установите переменные среды JAVA_HOME и PATH**
Например:
{{< highlight java >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}

- **Установите Aspose.Cells для Python via Java из pypi**
Вы можете легко использовать Aspose.Cells для Python via Java из [pypi](https://pypi.org/project/aspose-cells/) следующей командой.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}
