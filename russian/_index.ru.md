**![Aspose.Cells for .NET Логотип продукта](cells-for-net.png)**

**Добро пожаловать в Aspose.Cells for .NET**

Aspose.Cells for .NET - это библиотека программирования электронных таблиц, которая позволяет разработчикам программного обеспечения создавать, считывать, конвертировать, обновлять и печатать файлы электронных таблиц из своих собственных приложений. Комбинация API и графических элементов управления Aspose.Cells for .NET ускоряет программирование и конвертацию Microsoft Excel®. Aspose.Cells for .NET поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые использует ваш бизнес каждый день. Он также позволяет экспортировать файлы Excel в форматы PDF, XPS, HTML, MHTML, обычный текст и популярные форматы изображений, включая TIFF, JPG, PNG, BMP и SVG.

**Добро пожаловать в Aspose.Cells для Python via .NET**

Aspose.Cells для Python via .NET позволяет вашим приложениям выполнять большой спектр задач по обработке электронных таблиц. Aspose.Cells для Python via .NET поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые использует ваш бизнес каждый день. Он также позволяет экспортировать файлы Excel® в PDF, XPS, HTML, MHTML, обычный текст и популярные графические форматы, включая TIFF, JPG, PNG, BMP и SVG.
