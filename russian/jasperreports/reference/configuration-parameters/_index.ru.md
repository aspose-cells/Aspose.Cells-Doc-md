{{% alert color="primary" %}} 

В следующей таблице перечислены параметры конфигурации. 

{{% /alert %}} 

|**Имя параметра** |**Описание** |
| :- | :- |
|FORMAT_TYPE |Может быть установлено на "EXCEL97TO2003" или "EXCEL2007" для создания файлов формата Microsoft Excel 97 0 2003 или Excel 2007. |
|IS_ONE_PAGE_PER_SHEET |Значение логического типа, указывающее, должна ли каждая страница отчета быть записана в отдельный лист XLS. |
|IS_REMOVE_EMPTY_SPACE_BETWEEN_ROWS |Значение логического типа, указывающее, должны ли удаляться пустые места, которые могут возникать между строками, или нет. |
|IS_REMOVE_EMPTY_SPACE_BETWEEN_COLUMNS |Значение логического типа, указывающее, должны ли удаляться пустые места, которые могут возникать между столбцами, или нет. |
|IS_WHITE_PAGE_BACKGROUND |Значение логического типа, указывающее, должен ли фон страницы быть белым или цветом фона XLS по умолчанию. Цвет фона XLS может изменяться в зависимости от свойств просмотрщика XLS или цветовой схемы операционной системы. |
|IS_DETECT_CELL_TYPE |Флаг для указания, следует ли экспортировщику учитывать тип выражений исходного текстового поля и соответственно устанавливать типы ячеек и значения. |
|SHEET_NAMES |Массив строк, представляющих пользовательские имена листов. Это полезно при использовании параметра IS_ONE_PAGE_PER_SHEET. |
|IS_FONT_SIZE_FIX_ENABLED |Флаг для уменьшения размера шрифта, чтобы текст умещался в указанную высоту ячейки. |
|MAXIMUM_ROWS_PER_SHEET |Целочисленное значение, указывающее максимальное количество строк, разрешенных для отображения на листе. При установке создается новый лист для отображения оставшихся строк. Отрицательные значения или ноль означают, что ограничение не установлено. |
|IS_IGNORE_GRAPHICS |Флаг для игнорирования графических элементов и экспорта только текстовых элементов. |
|IS_COLLAPSE_ROW_SPAN |Флаг для схлопывания объединения ячеек по строкам и избегания объединения ячеек между строками. |
|IS_IGNORE_CELL_BORDER |Флаг для игнорирования границ ячейки. |
|IS_USE_EXCEL_CHART |Флаг для использования редактируемой диаграммы в формате Microsoft Excel вместо статических изображений. Значение по умолчанию - true. |
