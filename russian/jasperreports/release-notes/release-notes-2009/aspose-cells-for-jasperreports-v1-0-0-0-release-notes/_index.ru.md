 Эта страница содержит примечания к выпуску для[Aspose.Cells for JasperReports v1.0.0.0](https://downloads.aspose.com/cells/jasperreports/new-releases/aspose.cells-for-jasperreports-v1.0.0.0/)

 Мы рады объявить Aspose.Cells for JasperReports v1.0.0.0!

 Какие новости:

 l Предоставление редактируемых диаграмм в формате Microsoft Excel, а не статических изображений.

 l Поддержка интеграции Aspose.Cells for JasperReports с JasperServer.

 Устранить зависимость библиотеки POI.

 l Поддержка различных видов диаграмм, включая гистограмму, диаграмму с областями, круговую диаграмму, линейную диаграмму и т. д.
