**Добро пожаловать в Aspose.Cells for JasperReports**

Aspose.Cells for JasperReports - это библиотека, которая работает в качестве замены оригинального экспортера Microsoft Excel [JasperReports](https://community.jaspersoft.com/project/jasperreports-library), обладающая рядом великолепных улучшений. Она устраняет зависимость от библиотеки POI и использует Aspose.Cells for Java на серверной стороне для экспорта отчетов в форматах Microsoft Excel (XLS, XLSX).
