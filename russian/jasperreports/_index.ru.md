**Добро пожаловать в Aspose.Cells for JasperReports**

 Aspose.Cells for JasperReports — это библиотека, которая работает как замена[ДжасперРепортс](https://community.jaspersoft.com/project/jasperreports-library)s оригинальный Microsoft экспортер Excel, в комплекте с некоторыми большими улучшениями. Он устраняет зависимость от библиотеки POI и использует Aspose.Cells for Java на серверной части для экспорта отчетов в форматы Microsoft Excel (XLS, XLSX).
