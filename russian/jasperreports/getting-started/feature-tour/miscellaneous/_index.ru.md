{{% alert color="primary" %}} 

Некоторые эффективные функции Aspose.Cells for JasperReports перечислены ниже:

1. Aspose.Cells for JasperReports поддерживает диаграммы Microsoft Excel, таким образом, диаграммы экспортируются в виде редактируемых диаграмм, а не статических изображений. Таким образом, пользователи могут редактировать диаграммы вручную в Microsoft Excel, если им это потребуется.
2. Aspose.Cells for JasperReports [легко интегрируется с JasperServer](/cells/ru/jasperreports/integration-with-jasperserver/).

{{% /alert %}}
