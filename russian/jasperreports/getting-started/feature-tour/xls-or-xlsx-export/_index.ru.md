{{% alert color="primary" %}}

 Aspose.Cells для JasperReports позволяет предоставлять отчеты в формате XLS и XLSX без потери макета или форматирования. Для создания электронных таблиц в форматах Excel Microsoft Aspose.Cells для JasperReports используется встроенная версия[Aspose.Cells for Java](https://products.aspose.com/cells/java/), ведущая библиотека для обработки электронных таблиц от Aspose.

Microsoft Excel не требуется для создания документов с номером Aspose.Cells.
