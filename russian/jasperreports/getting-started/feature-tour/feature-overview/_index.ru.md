{{% alert color="primary" %}}

Aspose.Cells for JasperReports - это продукт для экспорта отчетов из JasperReports и JasperServer в формат Microsoft Excel. Aspose.Cells for JasperReports наследует все возможности встроенного экспортера, а также компенсирует некоторые недостатки оригинального рендерера XLS в JasperReports. Этот раздел содержит подробное описание возможностей Aspose.Cells for JasperReports. Нажмите на соответствующие ссылки в таблице, чтобы узнать больше о возможностях.
