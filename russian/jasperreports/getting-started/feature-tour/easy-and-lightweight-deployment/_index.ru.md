{{% alert color="primary" %}}

 Aspose.Cells для JasperReports является внешним экспортером для[ДжасперРепортс](https://community.jaspersoft.com/project/jasperreports-library).

 Aspose.Cells для JasperReports предоставляется в виде одного файла JAR и может быть легко установлен на компьютеры, на которых работает JasperReports.[Загрузите его, чтобы начать пробную версию](https://downloads.aspose.com/cells/jasperreports).

[Чтобы установить экспортер](/cells/ru/jasperreports/installation/), **aspose.cells.jasperreports.jar** следует скопировать в**JasperReports\lib** каталог.

{{% /alert %}}
