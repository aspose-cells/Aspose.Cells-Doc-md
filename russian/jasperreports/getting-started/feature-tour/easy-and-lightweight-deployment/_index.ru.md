{{% alert color="primary" %}}

Aspose.Cells for JasperReports - внешний экспортер для [JasperReports](https://community.jaspersoft.com/project/jasperreports-library).

Aspose.Cells for JasperReports предоставляется в виде одного JAR-файла и легко устанавливается на компьютерах, работающих с JasperReports. [Загрузите его, чтобы начать пробный период](https://downloads.aspose.com/cells/jasperreports).

[Для установки экспортера](/cells/ru/jasperreports/installation/), **aspose.cells.jasperreports.jar** должен быть скопирован в каталог **JasperReports\lib**.

{{% /alert %}}
