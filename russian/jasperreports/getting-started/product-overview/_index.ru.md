### **Добро пожаловать в документацию по Aspose.Cells for JasperReports!**

Aspose.Cells for JasperReports - это библиотека, которая работает в качестве замены оригинального экспортера Microsoft Excel [JasperReports](https://community.jaspersoft.com/project/jasperreports-library), обладающая рядом великолепных улучшений. Она устраняет зависимость от библиотеки POI и использует Aspose.Cells for Java на серверной стороне для экспорта отчетов в форматах Microsoft Excel (XLS, XLSX).

{{% /alert %}}

### **Описание продукта**

Aspose.Cells for JasperReports - гибкое решение, которое облегчает экспорт отчетов из JasperReports и JasperServer в файлы Microsoft Excel. Все особенности отчета конвертируются в электронные таблицы Microsoft Excel с высочайшей степенью точности.

Помимо функциональности, предоставляемой оригинальным экспортером Microsoft Excel на основе POI, этот компонент предоставляет несколько ценных функций. Основные инновации, связанные с экспортом в XLS, включают:

- Предоставление редактируемых диаграмм в формате Microsoft Excel вместо статических изображений.
