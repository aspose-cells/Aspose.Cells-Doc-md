### **Добро пожаловать в документацию Aspose.Cells for JasperReports!**

 Aspose.Cells for JasperReports — это библиотека, которая работает как замена[ДжасперРепортс](https://community.jaspersoft.com/project/jasperreports-library)s оригинальный Microsoft экспортер Excel, в комплекте с некоторыми большими улучшениями. Он устраняет зависимость от библиотеки POI и использует Aspose.Cells for Java на серверной части для экспорта отчетов в форматы Microsoft Excel (XLS, XLSX).

{{% /alert %}}

### **Описание товара**

Aspose.Cells for JasperReports — это гибкое решение, упрощающее экспорт отчетов из JasperReports и JasperServer в форматы файлов Excel Microsoft. Все функции отчета преобразуются в электронные таблицы Excel Microsoft с высочайшей степенью точности.

Помимо функций, предоставляемых исходным экспортером Micrsofot Excel на основе POI, этот компонент предоставляет некоторые ценные функции. Основные новшества, связанные с экспортом в XLS, включают:

- Предоставляйте редактируемые диаграммы в формате Microsoft Excel, а не статические изображения.
