{{% alert color="primary" %}} 

Чтобы установить Aspose.Cells для JasperReports в вашу систему, ваша система должна соответствовать следующим требованиям:

- Java 2 Платформа Standard Edition 1.6 или выше.
- JasperReports 6.3.0 или выше.

{{% /alert %}}
