Мы рады объявить о первом общедоступном выпуске Aspose.Cells для PHP через Java. API-интерфейсы включают функции обработки электронных таблиц MS Excel в PHP (широко используемый язык сценариев с открытым исходным кодом). Aspose.Cells для PHP через Java — это компонент подмножества, который включает в себя все важные и полезные функции, присутствующие в его собственном[Aspose.Cells for Java](https://products.aspose.com/cells/java/). Новый API является таким же надежным и многофункциональным компонентом. Он поддерживает преобразование форматов файлов с высокой точностью в XLS, XLSX, XLSM, SpreadsheetML, CSV, Tab Delimited, HTML, MHTML и OpenDocument Spreadsheet и обратно в PHP. Пожалуйста, посетите[Страница сравнения характеристик продукта](/cells/ru/java/aspose-cells-for-php-via-java-features/)для вашей справки. Кроме того, см. документ о том, как[Среда установки и другие рекомендации по установке](/cells/ru/java/setup-and-installation-guidelines/)для вашей справки.
