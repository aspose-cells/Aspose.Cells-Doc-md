Убедитесь, что у вас установлен пакет ADT и что вы загрузили последнюю версию Aspose.Cells для Android via Java.

- Извлеките **aspose-cells-X.X.X.jar** из последнего архива версии и добавьте его в новую папку в примере, который вы хотите запустить. Добавьте файл в папку с именем **libs**.
- Откройте проект в своем рабочем пространстве, используя Eclipse с ADT.
- Нажмите **Run**, чтобы запустить, и выберите соответствующий эмулятор для запуска пакета.
