{{% alert color="primary" %}}

Ознакомительная версия Aspose.Cells для SharePoint не имеет ограничений по времени, и для ознакомительных пользователей также доступна бесплатная техническая поддержка.

{{% /alert %}} {{% alert color="primary" %}}

 Загрузка пробной и полной версий Aspose.Cells для SharePoint одинакова. Просто загрузите Aspose.Cells для SharePoint из[страница загрузки](https://downloads.aspose.com/cells/sharepoint) а также[установить его](/cells/ru/sharepoint/install-aspose-cells-for-sharepoint/). Aspose.Cells по умолчанию работает в ознакомительном режиме.

{{% /alert %}}

### **Ограничения ознакомительной версии**

 Режим оценки добавляет предупреждение об оценке во все экспортированные документы. Когда вы приобрели лицензию, просто[установить лицензию](/cells/ru/sharepoint/installing-aspose-cells-for-sharepoint-license/) и он будет работать в лицензионном режиме.
