Aspose.Cells for SharePoint предоставляет панель инструментов, включающую несколько ценных опций и команд:

- Преобразовать с помощью Aspose.Cells
- Предварительный просмотр документа с Aspose.Cells
- Поиск в документе с помощью Aspose.Cells
- Добавить водяной знак с помощью Aspose.Cells
- Добавить нижний колонтитул с помощью Aspose.Cells
