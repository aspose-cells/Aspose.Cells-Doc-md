Aspose.Cells for SharePoint поддерживает поиск в документах, хранящихся на сайтах SharePoint. Aspose.Cells для Share Point поддерживает следующие форматы ввода для предварительного просмотра документов:

- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- XML - Microsoft Excel SpreadsheetML

Для поиска в документе:

1. Выберите документ или папку, в которой хотите выполнить поиск.
1. Щелкните правой кнопкой мыши и выберите **Поиск в документе с помощью Aspose.Cells**. 
1. Задайте критерии поиска. 
   Результаты поиска отображаются.
