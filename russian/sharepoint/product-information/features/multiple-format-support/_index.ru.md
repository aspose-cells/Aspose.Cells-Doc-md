{{% alert color="primary" %}}

С помощью Aspose.Cells for SharePoint вы можете конвертировать документы между многими популярными форматами офисных документов из библиотеки документов SharePoint. Большинство преобразований выполняются с высокой точностью и точностью.

{{% /alert %}}

## **Поддерживаемые форматы ввода**

Aspose.Cells for SharePoint поддерживает следующие форматы ввода:

- XLS – Microsoft Excel 97~2003
- XLSX — Office Open XML
- XLSM, XLTX, XLTM — Office Open XML
- CSV - Рабочий лист значений, разделенных запятыми
- ODS — OpenDocument
- XML — Microsoft Excel SpreadsheetML

 Для создания документов Aspose.Cells for SharePoint использует встроенную версию[Aspose.Cells for .NET](https://products.aspose.com/cells/net/)лидирующий на рынке компонент обработки электронных таблиц от Aspose. Microsoft Excel не используется для создания документов.

## **Поддерживаемые форматы вывода**

Полный список выходных форматов, поддерживаемых Aspose.Cells for SharePoint:

- PDF — переносимый документ Adobe
- XLS – Microsoft Excel 97 – 2003
- XLSX — Office Open XML
- XLSM — Office Open XML для макросов включен
- TXT — обычный текст
- XML — Microsoft Excel SpreadsheetML
- ODS — OpenDocument
- CSV — Значения, разделенные запятыми
- XLSB – Microsoft Двоичный файл Excel 2007
- MHTML - Многоцелевое расширение Интернет-почты HTML
- XPS — Спецификация документа XML
- TIFF – Формат файла изображения с тегами
- SVG – Масштабируемая векторная графика
- DIF — Формат обмена чертежами
