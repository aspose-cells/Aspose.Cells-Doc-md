{{% alert color="primary" %}}

С помощью Aspose.Cells для SharePoint вы можете преобразовывать документы между многими популярными форматами офисных документов из библиотеки документов SharePoint. Большинство преобразований выполняются с высокой точностью и точностью.

{{% /alert %}}

## **Поддерживаемые форматы ввода**

Aspose.Cells для SharePoint поддерживает следующие форматы ввода:

- XLS – Microsoft Excel 97~2003
- XLSX — офисный Open XML
- XLSM, XLTX, XLTM — Office Open XML
- CSV — рабочий лист значений, разделенных запятыми
- СОД – OpenDocument
- XML — Microsoft Электронная таблица ExcelML

 Для создания документов Aspose.Cells для SharePoint использует встроенную версию[Aspose.Cells for .NET](https://products.aspose.com/cells/net/), лидирующий на рынке компонент обработки электронных таблиц от Aspose. Microsoft Excel не используется для создания документов.

## **Поддерживаемые форматы вывода**

Полный список выходных форматов, поддерживаемых Aspose.Cells для SharePoint:

- PDF — переносимый документ Adobe
- XLS — Microsoft Эксель 97 — 2003
- XLSX — офисный Open XML
- XLSM — Office Open XML для макросов включен
- TXT — обычный текст
- XML — Microsoft Электронная таблица ExcelML
- СОД – OpenDocument
- CSV — значения, разделенные запятыми
- XLSB — Microsoft Двоичный файл Excel 2007 г.
- MHTML - многоцелевое расширение Интернет-почты HTML
- XPS — спецификация документа XML
- TIFF — формат файла изображения с тегами
- SVG — масштабируемая векторная графика
- DIF - Формат обмена чертежами
