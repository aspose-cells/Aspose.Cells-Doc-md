{{% alert color="primary" %}}

С помощью Aspose.Cells for SharePoint вы можете преобразовывать документы между многими популярными форматами офисных документов из библиотеки документов SharePoint. Большинство конверсий выполняются с высокой точностью и качеством.

{{% /alert %}}

## **Поддерживаемые форматы ввода**

Aspose.Cells for SharePoint поддерживает следующие форматы ввода:

- XLS – Microsoft Excel 97~2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- CSV – документ таблицы значений, разделенных запятыми
- ODS – OpenDocument
- XML – Microsoft Excel SpreadsheetML

Для генерации документов Aspose.Cells for SharePoint полагается на встроенную версию [Aspose.Cells for .NET](https://products.aspose.com/cells/net/), ведущий компонент по обработке электронных таблиц от Aspose. Microsoft Excel не используется для генерации документов.

## **Поддерживаемые форматы вывода**

Полный список поддерживаемых форматов вывода Aspose.Cells for SharePoint:

- PDF – Переносимый документ формата Adobe
- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM – Office Open XML for macros enabled
- TXT – Plain Text
- XML – Microsoft Excel SpreadsheetML
- ODS – OpenDocument
- CSV – Comma Separated Values
- XLSB – Microsoft Excel Binary 2007
- MHTML – Multipurpose Internet Mail Extension HTML
- XPS – XML Paper Specification
- TIFF – Tagged Image File Format
- SVG – Scalable Vector Graphics
- DIF – Drawing Interchange Format
