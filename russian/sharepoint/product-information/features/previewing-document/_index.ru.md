Aspose.Cells для SharePoint поддерживает предварительный просмотр документов непосредственно из SharePoint. Поддерживаются следующие форматы предпросмотра:

- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- XML – Microsoft Excel SpreadsheetML
