{{% alert color="primary" %}} 

Чтобы сделать процесс конвертации как можно более гибким, Aspose.Cells for SharePoint позволяет вам установить опции, которые определяют, как будет обработан преобразованный файл.

{{% /alert %}} 
### **Опции**
Следующие опции доступны во время конвертации:
#### **Конкатенировать файлы**
Выберите **Комбинировать файлы**, чтобы объединить файлы, которые изначально находились в разных форматах, в один общий документ. 

Эта опция доступна, когда вы выбираете папку или несколько отдельных файлов для конвертации. Перемещайте документы в списке вверх или вниз, чтобы установить порядок, в котором вы хотите объединить документы.

#### **Преобразовать файлы в подпапках**
Выберите **Преобразовать файлы в подпапках**, чтобы рекурсивно конвертировать все документы и подпапки в выбранной папке. 

Существующая структура подпапок создается в папке назначения.

{{% alert color="primary" %}} 

Эта опция доступна только в SharePoint 2007. В SharePoint 2010 подпапки обрабатываются по умолчанию. Чтобы обработать только файлы в корневой папке в SharePoint 2010, выберите отдельные файлы вместо папки и не выбирайте подпапки.

{{% /alert %}} 
#### **Перезаписать существующие файлы в папке назначения**
Если папка назначения уже содержит файлы с таким же именем, как у тех, которые вы хотите конвертировать, у вас есть два варианта. Выберите перезаписать существующие файлы, чтобы заменить уже существующие файлы новыми сконвертированными. Снимите флажок, чтобы пропустить обработку файлов с тем же именем.
