Чтобы сделать процесс преобразования максимально гибким, Aspose.Cells для SharePoint позволяет установить параметры, определяющие, как будет обрабатываться преобразованный файл.

{{% /alert %}} 
### **Опции**
Во время конвертации доступны следующие опции:
#### **Объединить файлы**
 Выбирать**Объединить файлы** для объединения файлов, которые изначально были в разных форматах, в один документ.

Эта опция доступна, когда вы выбираете папку или несколько отдельных файлов для преобразования. Переместите документы в списке вверх или вниз, чтобы установить порядок, в котором вы хотите объединить документы.

#### **Преобразование файлов в подпапках**
 Выбирать**Преобразование файлов в подпапках** для рекурсивного преобразования всех документов и подпапок в выбранной папке.

Существующая структура вложенных папок создается в папке назначения.

{{% alert color="primary" %}} 

Этот параметр доступен только в SharePoint 2007. В SharePoint 2010 вложенные папки обрабатываются по умолчанию. Чтобы обрабатывать только файлы в корневой папке в SharePoint 2010, выберите отдельные файлы вместо папки и не выбирайте вложенные папки.

{{% /alert %}} 
#### **Перезаписать существующие файлы в папке назначения**
Если папка назначения уже содержит файлы с теми же именами, что и те, которые вы хотите преобразовать, у вас есть два варианта. Выберите, чтобы перезаписать существующие файлы, чтобы заменить уже существующие файлы только что преобразованными. Снимите флажок, чтобы пропустить обработку файлов с таким же именем.
