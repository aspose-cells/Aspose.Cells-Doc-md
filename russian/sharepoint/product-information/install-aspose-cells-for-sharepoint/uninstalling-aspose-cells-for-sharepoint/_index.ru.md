{{% alert color="primary" %}} 

Чтобы удалить Aspose.Cells для SharePoint, запустите программу установки. Если Aspose.Cells для SharePoint уже установлен, программа установки предлагает удалить или восстановить его.

{{% /alert %}} 
### **Удаление**
 Во время удаления программа установки деактивирует Aspose.Cells для SharePoint для всех семейств веб-сайтов и отзывает решение из фермы серверов.

**Удаление Aspose.Cells** 

![дело:изображение_альтернативный_текст](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

 Обратите внимание, что запуск**Ремонт** параметр не активирует повторно Aspose.Cells для SharePoint в семействах веб-сайтов. Вместо этого эту функцию необходимо активировать вручную после повторного развертывания с помощью**Ремонт** вариант.

 Удаление с помощью**Удалять** вариант, а затем повторная установка может быть более удобной, поскольку она позволяет выбрать целевые веб-приложения и семейства веб-сайтов во время установки.

{{% /alert %}}
