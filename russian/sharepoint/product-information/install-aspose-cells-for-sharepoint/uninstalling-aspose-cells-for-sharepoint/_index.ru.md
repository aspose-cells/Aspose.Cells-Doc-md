{{% alert color="primary" %}} 

Чтобы удалить Aspose.Cells for SharePoint, запустите программу установки. Если Aspose.Cells for SharePoint уже установлен, программа установки предлагает удалить или исправить его.

{{% /alert %}} 
### **Деинсталляция**
Во время удаления программа установки деактивирует Aspose.Cells for SharePoint для всех коллекций сайтов и отзывает решение из фермы серверов. 

**Деинсталляция Aspose.Cells** 

![todo:image_alt_text](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

Обратите внимание, что выполнение опции **Восстановление** не активирует Aspose.Cells for SharePoint в коллекциях сайтов. Вместо этого функция должна быть активирована вручную после повторного развертывания с помощью опции **Восстановление**.

Деинсталляция с использованием опции **Удалить** и затем повторная установка может быть более удобной, поскольку она позволяет выбирать целевые веб-приложения и коллекции сайтов во время установки. 

{{% /alert %}}
