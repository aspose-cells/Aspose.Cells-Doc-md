{{% alert color="primary" %}} 

Aspose.Cells for SharePoint можно загрузить в виде архива Aspose.Cells.SharePoint.zip.

{{% /alert %}} 
### **Содержимое архива**
Архив Aspose.Cells.SharePoint.zip содержит:

- Aspose.Cells.SharePoint.wsp — файл решения SharePoint. Aspose.Cells for SharePoint упакован как решение SharePoint для облегчения развертывания/отзыва и активации/деактивации функций на ферме серверов.
- Aspose_LicenseAgreement.rtf — Лицензионное соглашение с конечным пользователем
- Aspose.Cells for SharePoint.pdf – Документация пользователя
- Aspose.Cells for SharePoint Documentation.chm — пользовательская документация с общедоступной ссылкой API
- setup.exe — программа установки
- setup.exe.config — файл конфигурации установки

Программа установки проверяет следующие условия перед продолжением установки:

- WSS 3.0, MOSS 2007 или SharePoint 2010 установлены.
- У пользователя есть разрешение на установку решений SharePoint.
- База данных SharePoint подключена к сети.
- Служба администрирования WSS запущена.
- Служба таймера WSS запущена.

Служба администрирования WSS и служба таймера необходимы, так как некоторые действия по настройке зависят от задания таймера для распространения на все серверы в ферме серверов.
#### **Для установки Aspose.Cells for SharePoint**
1. Распакуйте ZIP-файл Aspose.Cells.SharePoint на локальный диск сервера MOSS 7.0 или WSS 3.0.
1. Запустите setup.exe и следуйте инструкциям на экране.

Программа установки выполняет следующие действия:

1.  Проверьте предварительные условия установки. Установка не будет продолжена, если какая-либо проверка не пройдена.

   **Проверка системы** 

![дело:изображение_альтернативный_текст](installing-aspose-cells-for-sharepoint_1.png)




1.  Показать лицензионное соглашение с конечным пользователем. Пользователь должен принять соглашение, чтобы продолжить.

   **ЛСКП** 

![дело:изображение_альтернативный_текст](installing-aspose-cells-for-sharepoint_2.png)




1.  Показать диалоговое окно выбора цели развертывания. Пользователь выбирает веб-приложения и семейства веб-сайтов, в которых функция должна быть активирована. См. рисунок ниже.

   **Цели развертывания** 

![дело:изображение_альтернативный_текст](installing-aspose-cells-for-sharepoint_3.png)




1.  Разверните компонент на ферме серверов.

   **Запуск установки** 

![дело:изображение_альтернативный_текст](installing-aspose-cells-for-sharepoint_4.png)




1. Активируйте функцию для выбранных семейств веб-сайтов и настройте их родительские веб-приложения.
1. Отобразите список веб-приложений и семейств веб-сайтов, в которых эта функция была развернута и активирована.

   **Установка завершена**
