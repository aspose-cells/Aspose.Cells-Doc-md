{{% alert color="primary" %}} 

Клиенты, разрабатывающие свои собственные решения для SharePoint, могут получить доступ к нашему движку конвертации с использованием публичного API. Это включает все поддерживаемые форматы файлов и параметры конвертации. Подробные спецификации интерфейса в нашем [Справочнике API](https://reference.aspose.com/) сопровождаются различными образцами на C# и VB.

{{% /alert %}}
