Сайты SharePoint предназначены для совместной работы, обмена информацией и хранения контента для удовлетворения определенных бизнес-потребностей. Aspose.Cells for SharePoint был разработан для выполнения определенных операций с документами Microsoft Excel, доступными в библиотеках документов SharePoint. ### **Краткая заметка о библиотеках документов SharePoint**
Содержание на сайтах SharePoint представлено в виде документов, размещенных в централизованном месте, называемом библиотекой документов SharePoint (иногда называемой "общей библиотекой документов"). Авторизованные пользователи могут загружать документы с локальной машины на сайт SharePoint. Эти документы хранятся в двоичном формате в базе данных SQL Server, используемой сайтом SharePoint. ### **Преимущества**
Следующий список подчеркивает несколько преимуществ использования Aspose.Cells for SharePoint:

- Сервер сайта SharePoint не имеет встроенной возможности прямого преобразования файлов Microsoft Excel на сервере. Aspose.Cells for SharePoint добавляет эту функцию на сервер сайта SharePoint.
- Aspose.Cells for SharePoint не требует наличия Microsoft Excel для работы. Он установлен на сервере, поэтому пользователям SharePoint не нужен Microsoft Excel на их локальных системах для преобразования документов Microsoft Excel из одного формата в другой.
- Aspose.Cells for SharePoint не требует никаких предварительных требований, кроме тех, которые требуются для создания сайта SharePoint.
- Aspose.Cells for SharePoint [легко разворачивается на сервере сайта SharePoint](/cells/ru/sharepoint/installing-aspose-cells-for-sharepoint/).
- Aspose.Cells for SharePoint поддерживает [ряд входных и выходных форматов](/cells/ru/sharepoint/multiple-format-support/).
- Aspose.Cells for SharePoint предоставляет общедоступный API для разработчиков, которым необходимо разрабатывать свои собственные решения SharePoint.
### **Преобразование документов Microsoft Excel с использованием Aspose.Cells for SharePoint**
После установки Aspose.Cells for SharePoint на сервере SharePoint появляется опция **Преобразовать с помощью Aspose.Cells.SharePoint** в контекстном меню: 

**Поиск опции Преобразовать с...** 

Чтобы преобразовать документ Microsoft Excel из библиотеки документов SharePoint в другие форматы, выполните следующие действия:

1. Щелкните по нужному документу Microsoft Excel, а затем щелкните Преобразовать с помощью Aspose.Cells for SharePoint
   Отображается экран настроек конвертации. 

   **Принятие решения о преобразовании** 


1. Выберите выходной формат.
   Здесь вы также можете изменить имя выходного файла и папку назначения.
1. Нажмите **Преобразовать**. 

   **Выбор выходного формата** 



1. По завершении преобразования отображается сообщение об успешной операции. 

   **Успешное преобразование** 



1. Нажмите **Вернуться**, чтобы вернуться в библиотеку документов.
1. Преобразованный документ отображается в библиотеке документов SharePoint. 

   **Преобразованный файл отображается в библиотеке документов** 


### **Сводка**
{{% alert color="primary" %}} 

Установка и использование Aspose.Cells for SharePoint легки. Он не только добавляет функцию преобразования файлов Microsoft Excel непосредственно на сервере SharePoint, но также устраняет необходимость установки Microsoft Excel на клиентских компьютерах. Кроме того, разработчики могут использовать общедоступный API Aspose.Cells for SharePoint для разработки собственных решений SharePoint.

Другими словами, это выгодно для администраторов сайтов SharePoint, разработчиков и конечных пользователей.
