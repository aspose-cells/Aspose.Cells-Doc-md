Для конвертации всех файлов в папке: 

1. Выберите **Конвертировать с помощью Aspose.Cells** из ECB вашей желаемой папки, находясь в родительской папке.

Для конвертации нескольких файлов: 

1. Выберите несколько файлов в библиотеке и выберите **Конвертировать с помощью Aspose.Cells**. 

   **Конвертировать все файлы в подпапке**

1. Выберите элемент файла в списке документов и переместите его вверх или вниз, чтобы изменить порядок конвертации.
