{{% alert color="primary" %}} 

Одной из самых распространенных задач, для которой пользователи используют Aspose.Cells, является конвертация отдельных файлов. 

{{% /alert %}} 
### **Конвертирование отдельных файлов**
Для конвертации конкретного документа в библиотеке документов: 

1. Выберите **Конвертировать с помощью Aspose.Cells.SharePoint** в блоке управления редактирования документа (ECB). 

   **Выбор элемента “Конвертировать через Aspose.Cells” в ECB документа** 

![todo:image_alt_text](convert-one-file_1.png)



Отображается экран настроек конвертации. Здесь вы можете точно настроить конвертацию:

1. Выберите формат файла назначения из списка **Конвертировать в**.
1. Введите имя выходного файла в поле **Файл назначения**.
   По умолчанию, Aspose.Cells for SharePoint предварительно заполняет это поле именем исходного файла. 
   Расширение добавляется автоматически в соответствии с выбранным форматом.
1. Начните конвертацию, нажав **Преобразовать**.
   (Или вернитесь в библиотеку, отменяя ваши настройки, нажав **Отмена**.)
