{{% alert color="primary" %}} 

 Одной из наиболее распространенных задач, для которой пользователи используют Aspose.Cells, является преобразование отдельных файлов.

{{% /alert %}} 
### **Преобразование отдельных файлов**
 Чтобы преобразовать определенный документ в библиотеке документов:

1.  Выбирать**Преобразование с помощью Aspose.Cells.SharePoint** в блоке управления редактированием документа (ECB).

   **Выбор пункта «Конвертировать через Aspose.Cells» в ECB документа** 

![дело:изображение_альтернативный_текст](convert-one-file_1.png)



Отображается экран настроек преобразования. Здесь вы можете настроить преобразование:

1.  Выберите формат файла назначения из**Преобразовать в** список.
1.  Введите имя выходного файла в**Целевой файл** поле.
 По умолчанию Aspose.Cells для SharePoint предварительно заполняет это поле именем исходного файла.
 Расширение добавляется автоматически в соответствии с выбранным форматом.
1.  Начните преобразование, нажав**Конвертировать**.
 (Или вернитесь в библиотеку, отменив настройки, нажав**Отмена**.)

**Экран настроек преобразования для преобразования одного файла**
