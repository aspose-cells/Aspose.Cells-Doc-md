Если вы хотите добавить водяной знак в несколько файлов Excel, выберите «Добавить водяной знак с Aspose.Cells» на панели ленты.

![дело:изображение_альтернативный_текст](add-watermark-to-multiple-documents_1.png)

Получите все файлы Excel из папки источника данных и создайте таблицу списка файлов.

 Выберите файл, на который нужно добавить водяной знак, нажмите**Добавить водяной знак** кнопка, чтобы добавить водяной знак для выбранных файлов.

![дело:изображение_альтернативный_текст](add-watermark-to-multiple-documents_2.png)
