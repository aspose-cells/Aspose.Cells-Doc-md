Если вы хотите добавить водяной знак в несколько файлов Excel, выберите опцию «Добавить водяной знак с помощью Aspose.Cells» на панели Ribbon.

![todo:image_alt_text](add-watermark-to-multiple-documents_1.png)

Получить все файлы Excel из исходной папки и создать таблицу списка файлов.

Выберите файл, к которому нужно добавить водяной знак, нажмите кнопку **Добавить водяной знак** для выбранных файлов.

![todo:image_alt_text](add-watermark-to-multiple-documents_2.png)
