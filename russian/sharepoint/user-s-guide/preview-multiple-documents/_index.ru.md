Вы можете просматривать несколько документов Excel в библиотеке документов. 

Чтобы просмотреть все файлы в папке: 

1. Выберите **Предварительный просмотр документа с помощью Aspose.Cells** в контекстном меню редактирования вашей нужной папки, находясь в родительской папке.

Для предварительного просмотра нескольких файлов из папки:

1. Выберите несколько файлов.
1. При выбранных файлах выберите **Предварительный просмотр документа с помощью Aspose.Cells**.

**Предварительный просмотр документа в SharePoint** 

![todo:image_alt_text](preview-multiple-documents_1.png)

Когда вы просматриваете несколько документов, область данных показывает первый документ в списке. Список документов показывает все просматриваемые документы. Чтобы просмотреть другой документ:

1. Выберите документ в списке.
1. Нажмите **Просмотр**.
