Если вы хотите добавить водяной знак в файл Excel, выберите параметр «Добавить водяной знак с номером Aspose.Cells» в блоке управления редактированием документа (ECB).

![дело:изображение_альтернативный_текст](add-watermark-to-a-document_1.png)

Во время настройки добавления водяного знака доступны следующие параметры:

![дело:изображение_альтернативный_текст](add-watermark-to-a-document_2.png)

**Файл водяного знака**

Файл должен быть файлом изображения.
