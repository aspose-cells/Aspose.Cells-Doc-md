Если вы хотите добавить водяной знак в файл Excel, выберите опцию «Добавить водяной знак с помощью Aspose.Cells» в контрольном блоке редактирования документа (ECB).

![todo:image_alt_text](add-watermark-to-a-document_1.png)

Во время настройки добавления водяного знака доступны следующие варианты:

![todo:image_alt_text](add-watermark-to-a-document_2.png)

**Файл водяного знака**

Файл должен быть файлом изображения.
