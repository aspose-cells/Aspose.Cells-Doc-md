Чтобы выполнить поиск всех файлов в папке:

1. Выберите **Поиск в документе с помощью Aspose.Cells** в контекстном меню выбранной папки, находясь в родительской папке.

Чтобы выполнить поиск нескольких файлов сразу:

1. Выберите несколько файлов.
1. При выбранных файлах выберите **Поиск документа с помощью Aspose.Cells**.

**Поиск документов** 

![todo:image_alt_text](searching-in-a-document_1.png)

Доступны следующие варианты:

- **Параметры поиска**: Уточните ваш поиск, выполнив поиск в ячейках, для формул или поиск формул, содержащих определенное выражение.
- **С учетом регистра**: Когда включена опция учета регистра, поиск соответствует символам точно, включая строчные и прописные буквы.
