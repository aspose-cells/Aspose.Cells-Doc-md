## **Что такое диаграмма акций?**

Графики акций — это специальный вид графика, используемый для отслеживания изменений цен торгуемых активов, таких как товары, акции и криптовалюты. Они позволяют видеть максимальные и минимальные значения за определённый период, а также начальные и финальные значения на одном графике. Aspose.Cells for JavaScript через C++ предлагает четыре типа графиков акций, и для их использования необходимо иметь правильные наборы данных и выбирать столбцы в правильном порядке.

Следующий набор данных показывает ежедневную торговую информацию по акции. Мы используем эти данные для создания четырех типов графиков акций: графика High-Low-Close (HLC), графика Open-High-Low-Close (OHLC), графика Volume-High-Low-Close (VHLC) и Volume-Open-High-Low-Close (VOHLC).
