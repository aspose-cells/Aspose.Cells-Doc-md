##  **Возможные сценарии использования**
 Aspose.Cells позволяет вставлять объект OLE внутри листа. Пожалуйста, используйте[Рабочий лист->GetOleObjects()->Добавить()](https://reference.aspose.com/cells/cpp/aspose.cells.drawing/oleobjectcollection/add/)метод для этой цели. Вам понадобится массив байтов изображения, который будет использоваться для вставки объекта OLE в рабочий лист, и байты данных объекта Ole, которые будут вашим фактическим объектом. Чтобы вставить объект Ole в рабочий лист.
