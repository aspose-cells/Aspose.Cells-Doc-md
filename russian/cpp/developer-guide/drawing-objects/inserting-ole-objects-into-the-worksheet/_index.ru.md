## **Возможные сценарии использования**
 Aspose.Cells позволяет вставить объект OLE внутрь рабочего листа. Пожалуйста, используйте[IWorksheet->GetIOleObjects()->Добавить()](https://reference.aspose.com/cells/cpp/class/aspose.cells.drawing.i_ole_object_collection#af230dd65a00cefabcc4b9f165b5dc7ba)метод для этой цели. Вам понадобится массив байтов изображения, который будет использоваться для вставки объекта OLE в рабочий лист, и байты данных объекта Ole, которые будут вашим фактическим объектом. Чтобы вставить объект Ole в рабочий лист.
