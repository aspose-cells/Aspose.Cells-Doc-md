## **Возможные сценарии использования**
Microsoft Excel поддерживает множество форматов, таких как XLS, XLSX, XLSM, XLSB, CSV и т. д. Aspose.Cells также поддерживает многие из этих форматов. В этой статье объясняется, как читать и записывать файл Excel в формате CSV, используя Aspose.Cells.
