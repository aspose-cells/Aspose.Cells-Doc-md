## **Возможные сценарии использования**
Майкрософт Эксель поддерживает множество форматов, таких как XLS, XLSX, XLSM, XLSB, CSV и другие. Aspose.Cells также поддерживает многие из этих форматов. В этой статье объясняется, как читать и записывать файл Excel в формате XLSM с помощью Aspose.Cells.
## **Чтение и запись файлов в формате XLSM**
В следующем примере загружается [исходный файл XLSM](23166990.xlsm), считывается его ячейка A1, затем его содержимое копируется в ячейку C4 и сохраняется как [выходной файл XLSM](23166991.xlsm).
