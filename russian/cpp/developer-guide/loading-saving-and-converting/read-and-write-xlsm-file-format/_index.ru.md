## **Возможные сценарии использования**
Microsoft Excel поддерживает многие форматы, такие как XLS, XLSX, XLSM, XLSB, CSV и т. д. Aspose.Cells также поддерживает многие из этих форматов. В этой статье объясняется, как читать и записывать файл Excel в формате XLSM, используя Aspose.Cells.
## **Чтение и запись формата файла XLSM**
 Следующий пример кода загружает[исходный XLSM-файл](23166990.xlsm)и читает свою ячейку A1, а затем копирует ее содержимое в ячейку C4 и сохраняет ее как[выходной файл XLSM](23166991.xlsm).
