## **Возможные сценарии использования**
Microsoft Excel поддерживает множество форматов, таких как XLS, XLSX, XLSM, XLSB, CSV и т. д. Aspose.Cells также поддерживает многие из этих форматов. В этой статье объясняется, как читать и записывать файл Excel в формате XLSB, используя Aspose.Cells.
## **Чтение и запись XLSB Формат файла**
 Следующий пример кода загружает[исходный файл XLSB](23166992.xlsb)и читает свою ячейку A1, а затем копирует ее содержимое в ячейку C4 и сохраняет ее как[выходной файл XLSB](23166993.xlsb).
