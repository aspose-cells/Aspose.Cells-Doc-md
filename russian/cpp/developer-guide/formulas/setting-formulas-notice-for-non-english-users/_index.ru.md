{{% alert color="primary" %}} 

Aspose.Cells поддерживает большинство формул/функций, которые являются частью Microsoft Excel. Разработчики могут использовать эти формулы как через API, так и [дизайнерские таблицы](/cells/ru/cpp/what-is-a-designer-spreadsheet/). Aspose.Cells поддерживает большой набор математических, строковых, булевых, дат/времени, статистических, баз данных, поиска и ссылочных формул. Формулы должны задаваться с использованием английского (США) стиля.

{{% /alert %}} 

## **Примечание для пользователей, не говорящих на английском языке**
Есть два совета, которые не англоговорящим пользователям нужно соблюдать при создании формул с помощью Aspose.Cells:

1. Формулы должны вводиться на английском языке. Например, используйте английское "=SUM()", а не немецкое "=SUMME()".
1. Всегда используйте запятую (,) для разделения параметров функции. Для некоторых языковых настроек разделитель параметров функции — точка с запятой, однако Aspose.Cells использует английский стиль — запятую. Например, используйте "=IF (C5=0,0,C3/C4)" вместо "=IF(C5=0;0;C3/C4)".
