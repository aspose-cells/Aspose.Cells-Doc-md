## **Введение**
Пожалуйста, используйте метод [PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) для создания сводной таблицы внутри листа. После создания сводной таблицы, можно работать с ней, используя класс [PivotTable](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/).
## **Создать сводную таблицу**
