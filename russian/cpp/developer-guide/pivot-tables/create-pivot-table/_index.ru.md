##  **Введение**
 Пожалуйста, используйте[PivotTableCollection.Добавить()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) метод для создания сводной таблицы внутри листа. После создания сводной таблицы с ней можно работать с помощью[Сводная таблица](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/)сорт.
