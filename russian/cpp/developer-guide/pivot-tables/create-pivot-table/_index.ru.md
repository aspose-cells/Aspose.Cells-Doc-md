## **Введение**
 Пожалуйста, используйте[IPivotTableCollection.Добавить()](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table_collection#a903a0eb3d7ef995c370f4a4385b25111) способ создания сводной таблицы внутри рабочего листа. После того, как сводная таблица создана, вы можете работать с ней, используя[IPivotTable](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table)учебный класс.
