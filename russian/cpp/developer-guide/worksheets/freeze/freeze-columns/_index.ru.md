## **Введение**

В этой статье мы узнаем, как заморозить левый(е) столбец(ы). Когда у вас большое количество данных в строке, вы не можете видеть левый(е) столбец(ы), когда прокручиваете лист по горизонтали. Вы можете заморозить и заблокировать первый(е) столбец(ы), чтобы видеть зафиксированную часть даже при прокрутке остальной части данных. Вы легко увидите заголовки в левых столбцах.

## **Заморозить столбцы в Excel**

**![Заморозить левые столбцы в Excel](freeze-columns.png)**

1. Если хотите заморозить левый(е) столбец(ы), сначала выберите столбец под столбцом, который необходимо зафиксировать.
2. Нажмите Вид > Заморозка областей.
3. В выпадающем меню нажмите Заморозить первый столбец.
4. Если вы прокрутите вниз, первый столбец всегда останется слева.

**![Замороженный столбец](frozen-columns.png)**

Как видите, 1-й столбец зафиксирован, он всегда закреплен вверху при горизонтальной прокрутке.

Заморозка столбцов позволяет просматривать длинные данные, не теряя из виду первые столбцы.

## **Заморозить столбцы с помощью Aspose.Cells for C++**
Очень просто заморозить первый(е) столбец(ы) с помощью Aspose.Cells for C++. 
Пожалуйста, используйте метод [**Worksheet.FreezePanes**](https://reference.aspose.com/cells/cpp/aspose.cells/worksheet/freezepanes/) для заморозки столбцов в выбранном столбце.
1. Создать рабочую книгу для открытия файла или создать пустой файл.
2. Заморозить первый столбец с помощью метода Worksheet.FreezePanes().
3. Сохранить файл.

```c++
#include <iostream>
#include "Aspose.Cells.h"
using namespace Aspose::Cells;

int main()
{
    Aspose::Cells::Startup();

    // Instantiate a new Workbook
    Workbook workbook(u"Freeze.xlsx");

    // Freezing panes at the second column
    workbook.GetWorksheets().Get(0).FreezePanes(u"B1", 0, 1);

    // Saving the file
    workbook.Save(u"frozen.xlsx");

    std::cout << "Panes frozen successfully!" << std::endl;

    Aspose::Cells::Cleanup();
}
```
