Aspose.Cells for C++ — это собственная библиотека C++, которая позволяет разработчикам C++ программно создавать электронные таблицы, манипулировать ими и преобразовывать их, не требуя автоматизации Office или приложения Excel Microsoft.

## Среда разработки

Вы можете использовать Aspose.Cells for C++ при разработке приложений для Windows или Linux.

### Windows

 Aspose.Cells for C++ может использоваться для разработки приложений в любой среде разработки, которая поддерживает[Microsoft Набор инструментов платформы Visual Studio v142](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017?view=msvc-160), но явно поддерживаются среды, перечисленные в следующей таблице:

### Дополнительная зависимость от Linux

 Aspose.Cells for C++ в Linux зависит от[libgdiplus](https://github.com/mono/libgdiplus//) исполняемые файлы как динамическая библиотека, так и инструмент. Пожалуйста, установите его перед использованием:

- Установка libgdiplus в Ubuntu или Debian</br>
`sudo apt-get install libgif-dev autoconf libtool automake build-essential gettext libglib2.0-dev libcairo2-dev libtiff-dev libexif-dev`
