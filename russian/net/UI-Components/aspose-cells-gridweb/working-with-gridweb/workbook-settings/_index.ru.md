Есть некоторые настройки, которые мы можем указать с помощью set GridWorkbookSettings:

 
- **[GridWorkbookSettings] (https://reference.aspose.com/cells/net/aspose.cells.gridweb.data/GridWorkbookSettings)**

|**Имя** |**Описание** |
|:- |:- |
| Макситерация| Получает или задает максимальное количество итераций для разрешения циклической ссылки, значение по умолчанию — 100.|
| Итерация| Получает или задает, использовать ли итерацию для разрешения циклических ссылок.|
| ForceFullРассчитать| Получает или задает, выполняет ли полный расчет каждый раз, когда запускается вычисление.|
| CreateCalcChain|Получает или задает необходимость создания цепочки вычисляемых формул. Значение по умолчанию — ложь.|
| ReCalculateOnOpen| Получает или задает, следует ли пересчитывать все формулы при открытии файла.|
| PrecisionAsDisplayed| Значение true, если вычисления в этой книге будут выполняться только с точностью отображаемых чисел.|
| Дата1904| Получает или задает значение, указывающее, использует ли книга систему дат 1904 года.|
| CheckCustomNumberFormat| Получает или задает, проверяется ли пользовательский числовой формат при установке Style.Custom.|
| Автор| Получает и задает автора файла.|
