{{% alert color="primary" %}} 

Как и Aspose.Cells.GridDesktop, элемент управления Aspose.Cells.GridWeb может открывать и загружать файлы Microsoft Excel - с данными, форматированием, диаграммами, изображениями и т. д. - но в веб-приложениях. В этой статье объясняется, как это сделать.

{{% /alert %}} 
## **Импорт файлов Excel**
### **Импорт из файла**
Открытие файла Excel с помощью элемента управления Aspose.Cells.GridWeb:

1. Добавьте элемент управления Aspose.Cells.GridWeb на веб-форму.
1. Импортируйте файл Excel, указав путь к файлу.
1. Запустите приложение.

{{% alert color="primary" %}} 

Если вы не знаете, как добавить элемент управления на веб-форму, см. [Добавление GridWeb в веб-форму](/cells/ru/net/aspose-cells-gridweb/add-gridweb-to-web-form/).

{{% /alert %}} 

Когда элемент управления Aspose.Cells.GridWeb добавлен на веб-форму, элемент управления автоматически создается и добавляется на форму с размером по умолчанию. Вам не нужно создавать объект элемента управления Aspose.Cells.GridWeb, вам просто нужно перетащить элемент управления и начать его использовать.

Однако чтобы загрузить содержимое из файла Excel в элемент управления Aspose.Cells.GridWeb, вам нужно вызвать метод ImportExcelFile для указания пути к файлу Excel. После этого элемент управления Aspose.Cells.GridWeb автоматически найдет файл по указанному пути и отобразит его содержимое. Ниже приведен фрагмент кода, который загружает содержимое файла Excel.

### **Импорт из потока**
Также как открытие файлов Excel из файла, элемент управления Aspose.Cells.GridWeb может загружать файлы Excel из потока. Использование файла как потока является лучшим подходом для предотвращения любого вида проблем с доступом к файлам или нарушениями совместного использования, потому что этот подход гарантирует закрытие всех подключений к файлам путем закрытия потока.
