Эта тема предоставляет базовое пошаговое руководство для начинающих, чтобы помочь им создавать и использовать элемент управления Aspose.Cells.GridWeb в веб-приложениях.

{{% /alert %}} 
## **Создание и использование элемента управления Aspose.Cells.GridWeb**
### **Шаг 1: Создание проекта веб-приложения**
Сначала создайте проект веб-приложения, в котором будет использоваться элемент управления Aspose.Cells.GridWeb:

1. Откройте Visual Studio.
1. В меню **Файл** выберите **Новый** и затем **Проект**. 

![todo:image_alt_text](add-gridweb-to-web-form_1.png)



Появится диалоговое окно Новый проект.

1. Выберите **ASP.NET Веб-приложение** для нужного языка. 

![todo:image_alt_text](add-gridweb-to-web-form_2.png)

1. Выберите шаблон **Веб-формы**. 

![todo:image_alt_text](add-gridweb-to-web-form_3.png)

1. Добавьте новую веб-форму в проект.
### **Шаг 2: Встраивание элемента управления в веб-форму**
Перетащите элемент управления Aspose.Cells.GridWeb из панели инструментов Visual Studio на веб-форму. 

![todo:image_alt_text](add-gridweb-to-web-form_4.png)

{{% alert color="primary" %}} 

Чтобы узнать, как добавить элементы управления Aspose.Cells Grid в панель инструментов Visual Studio, прочтите [Интеграция элементов управления Aspose.Cells.Grid с Visual Studio.NET](/cells/ru/net/aspose-cells-gridweb/integrate-aspose-cells-grid-controls-with-visual-studio-net/).

{{% /alert %}} 

Когда элемент управления был добавлен на форму, он отображается таким образом: 

![todo:image_alt_text](add-gridweb-to-web-form_5.png)
### **Шаг 3: Изменение размера элемента управления**
Форма отображается в размере по умолчанию. Измените размер, перетянув границы или углы. 

![todo:image_alt_text](add-gridweb-to-web-form_6.png)
### **Шаг 4: Установка свойств элемента управления**
Панель управления Aspose.Cells.GridWeb также может быть настроена с использованием различных свойств. 

![todo:image_alt_text](add-gridweb-to-web-form_7.png)

Возможно настроить множество свойств панели управления с помощью диалогового окна Свойства. Основные свойства включают высоту, ширину, цвет и визуальные стили. Расширенные свойства включают режим редактирования, режим сеанса и режим двойного щелчка. Более того, возможно установить настраиваемые обработчики событий в диалоговом окне Свойства.

Также существуют дополнительные инструменты конфигурации для Aspose.Cells.GridWeb, которые можно увидеть внизу диалогового окна Свойства как гиперссылки или щелкните правой кнопкой мыши на панели управления GridWeb, чтобы найти их. Эти инструменты конфигурации включают:

- Настраиваемые кнопки команд
#### **Настраиваемые кнопки команд**
Для открытия редактора настраиваемых кнопок команд:
Щелкните правой кнопкой мыши панель управления GridWeb и выберите **Настраиваемые кнопки команд**. 

![todo:image_alt_text](add-gridweb-to-web-form_8.png)



Отображается диалоговое окно редактора коллекции настраиваемых кнопок команд. 

![todo:image_alt_text](add-gridweb-to-web-form_9.png)

Диалоговое окно позволяет разработчикам добавлять и удалять настраиваемые кнопки команд в панели управления GridWeb.


### **Важно**
Aspose.Cells.GridWeb также предоставляет свои ресурсные файлы с панелью управления. Папка "acw_client" - это папка (@ ваш каталог установки), которая содержит файлы, и Aspose.Cells.GridWeb использует эту папку для управления своей внутренней конфигурацией и другой функциональностью, в ней есть скриптовые файлы, файлы изображений и другие файлы для определения поведения GridWeb и выполнения других операций. Файл конфигурации используется для управления встроенными клиентскими ресурсами (изображения, скрипты и т. д.). Более того, когда вам нужно развернуть веб-приложение с панелью управления GridWeb, вы также скопируете каталог "acw_client" в папку вашего проекта, иначе ваше веб-приложение (размещенное на сервере) не сможет его найти. Вы всегда можете указать папку ресурсов, добавив следующие строки кода в раздел конфигурации (например, в файл web.config в вашем проекте VS.NET):



|<p>{{< highlight java >}}</p><p> <appSettings></p><p>  <add key="aspose.cells.gridweb.acw_client_path" value="/grid/acw_client/"/> </p><p></appSettings></p><p>{{< /highlight >}}</p>|
| :- |


{{% alert color="primary" %}}

Путь всегда связан с каталогом проекта. Не следует использовать каталоги, находящиеся вне каталога проекта. Поэтому необходимо скопировать каталог "acw_client" (@ ваш каталог установки GridWeb) в каталог/подкаталог проекта.

{{% /alert %}}
### **Шаг 5: Запуск веб-приложения**
Запустите приложение, нажав Ctrl+F5 или нажав кнопку **Запуск**. 

Когда приложение запускается в браузере, отображается страница WebForm1.aspx, теперь содержащая пустую панель управления Aspose.Cells.GridWeb. Добавьте значения в ячейки, кликая на них. Также можно выполнять другие задачи, такие как изменение высоты строки или ширины столбца, копирование (Ctrl+C) или вырезание (Ctrl+X) данных ячейки в буфер обмена и вставка (Ctrl+V) данных в ячейку. Чтобы выполнить дополнительные операции, щелкните правой кнопкой мыши на панели управления, чтобы увидеть полный список опций.
