{{% alert color="primary" %}} 

 Aspose.Cells.GridWeb содержит специальные кнопки, такие как**Представлять на рассмотрение**, **Сохранять** а также**Отменить**. Все эти кнопки выполняют определенные задачи для Aspose.Cells.GridWeb.
Также можно добавить настраиваемые кнопки, выполняющие настраиваемые задачи. В этом разделе объясняется, как использовать эту функцию.

{{% /alert %}} 
## **Создание пользовательских командных кнопок**
Чтобы создать пользовательскую командную кнопку в Aspose.Cells.GridWeb:

1. Добавьте элемент управления Aspose.Cells.GridWeb в веб-форму.
1. Доступ к рабочему листу.
1. Создайте экземпляр класса CustomCommandButton.
1. Установите команду кнопки на некоторое значение. Это значение используется в обработчике событий кнопки.
1. Установите текст кнопки.
1. Установите URL-адрес изображения кнопки.
1. Наконец, добавьте объект CustomCommandButton в коллекцию CustomCommandButtons элемента управления GridWeb.

{{% alert color="primary" %}} 

Пользовательские командные кнопки также можно добавить в режиме WYSIWYG с помощью диалогового окна свойств Visual Studio.

{{% /alert %}}

### **Обработка событий пользовательской командной кнопки**
Наиболее важным аспектом пользовательских командных кнопок является действие, которое они выполняют при нажатии. Чтобы задать действие, создайте обработчик событий для события CustomCommand элемента управления GridWeb.

Событие CustomCommand всегда срабатывает при нажатии пользовательской командной кнопки. Таким образом, обработчик событий должен идентифицировать конкретную настраиваемую кнопку команды, к которой он применяется, с помощью набора команд при добавлении кнопки в элемент управления GridWeb. Наконец, добавьте пользовательский код, который выполняется при нажатии кнопки.
