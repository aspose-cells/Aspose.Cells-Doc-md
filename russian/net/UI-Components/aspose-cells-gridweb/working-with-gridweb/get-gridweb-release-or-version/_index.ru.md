{{% alert color="primary" %}} 

Используйте метод Aspose.Cells.GridWeb.GridWeb.GetVersion(), чтобы получить выпуск или текущую версию Aspose.Cells.GridWeb. Эта версия поможет вам найти версию GridWeb во время выполнения с помощью кода.

{{% /alert %}} 
## **Получите выпускную или текущую версию Aspose.Cells.GridWeb**
Следующий пример кода находит работающую версию Aspose.Cells.GridWeb и печатает ее на ASP.NET Label Control.

## **Найдите номер версии клиентского файла сценария GridWeb.**
Вы также можете найти номер версии acwmain.js, который является файлом сценария на стороне клиента GridWeb. Он содержит переменную acw_version, которая содержит номер версии файла acwmain.js. Вы можете распечатать номер версии файла acwmain.js на консоли следующим образом.
