## **Возможные сценарии использования**
По умолчанию Aspose.Cells.GridWeb отображает числовые данные в соответствии с языковыми/региональными настройками компьютера. Вы можете программно изменить десятичный разделитель с цифровой клавиатуры, используя Aspose.Cells.GridWeb API. Таким образом, когда файл импортируется в матрицу GridWeb или вы вводите некоторые числовые данные (с цифровой клавиатуры) в новую ячейку листа, он должен иметь желаемое десятичное число. сепаратор устанавливается визуально.
## **Изменить десятичный разделитель с цифровой клавиатуры**
