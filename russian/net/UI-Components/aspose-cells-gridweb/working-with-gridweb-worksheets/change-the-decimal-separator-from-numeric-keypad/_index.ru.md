## **Возможные сценарии использования**
По умолчанию Aspose.Cells.GridWeb отображает числовые данные соответственно настройкам локали/региона на компьютере. Вы можете programmatically изменить разделитель десятичной части с цифровой клавиатуры, используя API Aspose.Cells.GridWeb. Таким образом, когда файл импортируется в матрицу GridWeb или вы вводите числовые данные (с цифровой клавиатуры) в новую ячейку листа, он должен иметь установленный визуальный десятичный разделитель. 
## **Изменить разделитель десятичной части с цифровой клавиатуры**

Пожалуйста, обратите внимание, что изменение десятичного разделителя предназначено только для визуального восприятия пользователем в GridWeb. Когда вы редактируете и сохраняете свою книгу, числовые значения (в электронной таблице) все равно будут храниться в соответствии с вашим локальным/региональным разделителем десятичной части.
