В этом разделе объясняется, как использовать Aspose.Cells.GridWeb в приложениях ASP.NET с помощью Visual Studio.NET 2005. Этот раздел полезен для разработчиков начального уровня, работающих с Aspose.Cells.GridWeb.

## **Работа с Aspose.Cells.GridWeb с помощью Visual Studio 2013**
В этом разделе показано, как использовать Aspose.Cells.GridWeb, создав образец веб-сайта в Visual Studio 2013. Процесс разбит на этапы.
### **Шаг 1: Создание нового веб-сайта**
1. Откройте Visual Studio 2013.
1.  От**Файл** меню, выберите**Новое меню** , тогда**Веб-сайт**. 

![дело:изображение_альтернативный_текст](working-with-visual-studio_1.png)


 Откроется диалоговое окно Новый веб-сайт.

1.  Выбирать**ASP.NET Сайт веб-форм** из установленных шаблонов Visual Studio.
1.  Выберите режим HTTP для местоположения веб-сайта.

![дело:изображение_альтернативный_текст](working-with-visual-studio_2.png)




1.  Укажите место, где будут создаваться и храниться файлы веб-сайта.
 1. Нажмите**Просматривать** в диалоговом окне Новый веб-сайт.

![дело:изображение_альтернативный_текст](working-with-visual-studio_3.png)



 Отображается диалоговое окно «Выбор местоположения».

1.  Нажмите на**Локальный IIS** вкладка
Отображаются все папки и веб-приложения, хранящиеся в корневой папке IIS (например, C:\Inetpub\wwwroot).

![дело:изображение_альтернативный_текст](working-with-visual-studio_4.png)




1. Теперь создайте новое веб-приложение в вашем локальном IIS, где будут храниться файлы веб-сайта.
 Диалоговое окно Choose Location позволяет создавать и удалять веб-приложения или виртуальные каталоги в вашем локальном IIS. Чтобы создать веб-приложение, нажмите кнопку, как показано ниже на рисунке.

![дело:изображение_альтернативный_текст](working-with-visual-studio_5.png)



 Будет создано новое веб-приложение с именем по умолчанию WebSite.

1. Переименуйте веб-приложение. Мы переименовали его в GridWebOn2013.
1.  Нажмите**Открытым**. 

![дело:изображение_альтернативный_текст](working-with-visual-studio_6.png)



 Вы вернетесь в диалоговое окно «Новый веб-сайт». Путь к местоположению веб-сайта установлен на<http://localhost/GridWebOn2013>. 

1.  Нажмите**ХОРОШО** чтобы позволить Visual Studio создать веб-сайт.

![дело:изображение_альтернативный_текст](working-with-visual-studio_7.png)
### **Шаг 2. Проверка представлений исходного кода и дизайна веб-страницы**
 Веб-сайт по умолчанию будет создан Visual Studio 2013. Он содержит веб-страницу default.aspx с некоторым фиктивным текстом и разметкой.

**Исходное представление страницы default.aspx** 

![дело:изображение_альтернативный_текст](working-with-visual-studio_8.png)



Все веб-страницы (включая ASP.NET) можно открывать в двух режимах. Одним из них является представление исходного кода, которое позволяет разработчикам получать доступ к исходному коду и изменять его. Второй режим — это представление дизайна, которое можно использовать для разработки веб-страниц в стиле WYSIWYG. На приведенном выше снимке экрана показано исходное представление веб-страницы default.aspx. Чтобы просмотреть представление дизайна, нажмите**Дизайн**. 

**Представление «Дизайн» страницы default.aspx** 

![дело:изображение_альтернативный_текст](working-with-visual-studio_9.png)




Удалите страницу Default.aspx, добавленную Visual Studio, и добавьте новую пустую страницу Default.aspx.

![дело:изображение_альтернативный_текст](working-with-visual-studio_10.png)

### **Шаг 4: Измените тег <!DOCTYPE>**
1.  Переключитесь на исходный вид и найдите следующее**<!ДОКТИП>** тег в исходном коде:

**ASP.NET**

{{< highlight "csharp" >}}



<!DOCTYPE html>



{{< /highlight >}}

1.  Выберите полный тег.

![дело:изображение_альтернативный_текст](working-with-visual-studio_13.png)




1.  Сохранить, изменить или удалить<!DOCTYPE>ярлык.
1.  Или изменить<!DOCTYPE> тег со следующим:

{{< highlight "csharp" >}}



<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.0 Transitional//EN">



{{< /highlight >}}
### **Шаг 5. Изменение размера элемента управления Aspose.Cells.GridWeb**
 Вы можете изменить ширину и высоту элемента управления GridWeb после его перетаскивания на веб-сайт.

 В режиме конструктора вы можете изменить ширину и высоту GridWeb.

![дело:изображение_альтернативный_текст](working-with-visual-studio_14.png)



### **Шаг 6: Настройка свойств Aspose.Cells.GridWeb**
 Настройте свойства Aspose.Cells.GridWeb в режиме WYSIWYG, щелкнув значок**Характеристики** кнопку в правой части интегрированной среды разработки Visual Studio 2013.
 Отображается диалоговое окно свойств.

![дело:изображение_альтернативный_текст](working-with-visual-studio_15.png)



Панель «Свойства» позволяет настроить внешний вид GridWeb и некоторые другие свойства для управления поведением GridWeb.
### **Шаг 7. Запуск вашего первого веб-сайта, содержащего Aspose.Cells.GridWeb**
 Создайте и запустите веб-сайт.

1.  Запустите веб-сайт непосредственно из Visual Studio, нажав Ctrl+F5 или щелкнув**Начать отладку**. 

![дело:изображение_альтернативный_текст](working-with-visual-studio_16.png)

 Теперь вы можете начать играть с элементом управления GridWeb.
