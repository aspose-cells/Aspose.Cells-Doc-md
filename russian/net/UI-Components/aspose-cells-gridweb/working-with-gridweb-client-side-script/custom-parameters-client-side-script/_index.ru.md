{{% alert color="primary" %}} 

 Разработчики могут задать другое значение параметра инициализации для выполнения другого поведения элемента управления Aspose.Cells.GridWeb в acwmain.js.

{{% /alert %}} 
 
### **Параметры**
 
|**Параметр**|**Описание**|
|:- |:- |
|потребностьInitAlignmentAdjust|следует ли выполнять вертикальное выравнивание для содержимого ячейки при инициализации, это потребует некоторого времени для выполнения работы по выравниванию, если рабочий лист имеет большие ячейки, производительность будет низкой, если пользователю не важно вертикальное выравнивание, тогда он может установить его на быть ложным, значение по умолчанию истинно|
|фокус внутри| следует ли фокусироваться внутри диапазона ячеек, значение по умолчанию — true|
|копировать_с_стиль|следует ли копировать со стилем, значение по умолчанию — false, что означает копирование только содержимого ячейки|
|useESCAsLeave|поведение по умолчанию, когда нажатие esc работает как отмена операции редактирования в ячейке, если мы установим это значение в true, мы будем просто рассматривать его как короткую клавишу, чтобы покинуть ячейку, не возвращаясь к предыдущему значению, и это также изменит способ внутреннего редактирования для быстрого редактирования, значение по умолчанию - false|
|нужно проверить все|следует ли проверять все проверки на активном листе при проверке (на странице управления aspx установите ForceValidation="True") . Значение по умолчанию неверно|
|scrollToInvalidate|следует ли прокручивать и отображать первую недействительную ячейку, когда для параметра needValidateall установлено значение true. Значение по умолчанию — true.|
