{{% alert color="primary" %}} 

Разработчики могут устанавливать различные значения параметров инициализации для выполнения различных поведений управления Aspose.Cells.GridWeb в acwmain.js.  

{{% /alert %}} 

### **Параметры**

|**Параметр**|**Описание**|
| :- | :- |
|needInitAlignmentAdjust| необходимость вертикального выравнивания содержимого ячейки при инициализации, это потребует времени для выполнения работы по выравниванию, если рабочий лист содержит большие ячейки, производительность будет плохой, если пользователю не важно вертикальное выравнивание, то он может установить значение false, значение по умолчанию - true |
|focusinside| необходимость фокусировки внутри ячейки, значение по умолчанию - true |
|copy_with_style| необходимость копирования со стилем, значение по умолчанию - false, что означает только копирование содержимого ячейки|
|useESCAsLeave| поведение по умолчанию при нажатии esc работает как отмена операции редактирования ячейки, если установить это значение в true, мы будем просто считать его коротким ключом для выхода из ячейки без возвращения к предыдущему значению, это также изменит способ редактирования на быстрый способ, значение по умолчанию - false|
|needValidateall| необходимость проверки всех проверок на активном листе при выполнении проверки (на странице управления aspx установить ForceValidation="True"). Значение по умолчанию - false|
|scrollToInvalidate| необходимость прокрутки и вывода первой недопустимой ячейки в вид при установке needValidateall в true. Значение по умолчанию - true.|
