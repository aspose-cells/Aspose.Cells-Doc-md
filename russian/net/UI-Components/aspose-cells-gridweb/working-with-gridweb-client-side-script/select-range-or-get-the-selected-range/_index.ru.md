{{% alert color="primary" %}} 

Вы можете использовать следующие клиентские функции для выбора диапазона или получения выбранного диапазона в GridWeb с помощью JavaScript.

- getSelectRange()
- setSelectRange()
- clearSelections()

getSelectRange() возвращает последний выбранный диапазон. setSelectRange() выбирает указанный диапазон. clearSelections() очищает все выделения, исключая текущую активную ячейку.
