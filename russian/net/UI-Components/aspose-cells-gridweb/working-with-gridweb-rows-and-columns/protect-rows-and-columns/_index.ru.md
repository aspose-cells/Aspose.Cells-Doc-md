В этом разделе обсуждается несколько методов защиты ячеек в строках и столбцах от любых действий, выполняемых конечными пользователями. Разработчики могут реализовать эту защиту двумя способами: сделав ячейки в строках и столбцах доступными только для чтения или ограничив параметры контекстного меню Aspose.Cells.GridWeb. Оба эти метода обсуждаются ниже с помощью примеров.

{{% /alert %}} 
## **Защита Cells в строках и столбцах**
### **Создание строк и столбцов только для чтения**
Один из способов защиты строк и столбцов на листе — сделать ячейки доступными только для чтения. Затем они не могут быть удалены конечными пользователями.

Чтобы сделать строки и столбцы только для чтения:

1. Добавьте элемент управления Aspose.Cells.GridWeb в веб-форму.
1. Получите доступ к GridWorksheet в коллекции.
1. Установите нужные ячейки в строках или столбцах только для чтения.

### **Ограничение параметров контекстного меню**
Aspose.Cells.GridWeb предоставляет контекстное меню, которое конечные пользователи могут использовать для выполнения операций над элементом управления. Меню предоставляет множество опций для управления ячейками, строками и столбцами.

**Полные контекстные опции** 

![дело:изображение_альтернативный_текст](protect-rows-and-columns_1.png)

Можно ограничить любые операции на стороне клиента со строками и столбцами, ограничив параметры, доступные в контекстном меню. Это можно сделать, задав для свойств EnableClientColumnOperations и EnableClientRowOperations элемента управления GridWeb значение false. Также можно запретить пользователям замораживать строки и столбцы, задав для свойства EnableClientFreeze элемента управления GridWeb значение false.
