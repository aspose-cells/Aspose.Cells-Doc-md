## **Aspose.Cells.GridJs**
### **Функции импорта/экспорта**
- Импорт из и экспорт в электронные таблицы Excel.

### **Особенности листа**
- Добавление/удаление/переименование листов.


### **Функции строк и столбцов**
- Добавление, вставка и удаление строк и столбцов на листе.
- Скрытие и отображение строк и столбцов.
- Закрепление и открепление строк и столбцов.

### **Функции ячеек**
- Редактирование нескольких ячеек .
- Поддержка настройки формата числа, даты, валюты и времени.
- Поддержка настройки пользовательского формата.
- Поддержка настройки цвета фона.
- Поддержка автозаполнения.
- Управление выравниванием текста.
- Управление переносом текста.
- Управление границами.
- Управление настройками шрифта.
- Объединение или разделение ячеек.
### **Функции форматирования**
- Создание веб-приложений с интерактивной таблицей, которые выглядят и ощущаются как настольные приложения для электронных таблиц.
- Поддержка форматирования ячеек.

### **Утилитарные функции**
- Поддержка создания выпадающих списков и проверок.
- Добавление/удаление/поиск комментариев.
- Поворот/изменение размеров/перемещение изображений/форм.

- Навигация с клавиатуры.
### **Расширенные функции**
- Поддержка создания и вычисления формул на веб-страницах.
- Предоставляет 80+ предопределенных функций, связанных с математикой, текстом и данными, датой и временем, а также логикой.

### **Поддерживаемые браузеры**
- Microsoft Internet Explorer
- Mozilla FireFox
- Opera
- Google Chrome
- Microsoft Edge
### **Поддерживаемые Microsoft Excel**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **Поддерживаемые платформы**
- ASP.NET core
