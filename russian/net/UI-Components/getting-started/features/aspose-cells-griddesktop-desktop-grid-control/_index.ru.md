## **Aspose.Cells.GridDesktop – Элемент управления рабочего стола**
### **Функции импорта/экспорта**
- Импорт содержимого из электронных таблиц Excel.
- Импорт данных непосредственно из объекта DataTable.
- Экспорт содержимого в электронные таблицы Excel.
### **Особенности листа**
- Добавление, вставка или удаление рабочих листов.
- Переименование рабочих листов.
- Сортировка данных рабочего листа.
### **Функции строк и столбцов**
- Добавление, вставка или удаление строк и столбцов.
- Скрыть или показать строки и столбцы.
- Замораживание или размораживание строк и столбцов.
### **Функции ячеек**
- Поддержка настроек числового формата.
- Управление выравниванием текста.
- Управление настройками границ.
- Управление настройками шрифта.
- Объединение или разделение ячеек.
### **Функции форматирования**
- Управление поведением, внешним видом сетки.
- Применение стиля и шрифта активной ячейки к другим ячейкам с помощью Копира формата.
- Легкое управление данными и стилями ячеек.
### **Утилитарные функции**
Добавление или удаление комментариев ячеек.
Вставка или удаление изображений.
Добавление или удаление гиперссылок.
Добавление или удаление правил проверки.
Добавление или удаление элементов управления (например, флажков, комбинированных списков и т. д.) в ячейках.
Навигация с клавиатуры по ячейкам.
### **Функции безопасности**
Защита ваших ячеек путем блокировки или скрытия их.
### **Расширенные функции**
Поддерживает создание и вычисление формул.
Выполнение желаемых действий при различных событиях сетки.
### **Поддерживаемые версии Microsoft Excel**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **Поддерживаемые платформы**
- Windows Forms
