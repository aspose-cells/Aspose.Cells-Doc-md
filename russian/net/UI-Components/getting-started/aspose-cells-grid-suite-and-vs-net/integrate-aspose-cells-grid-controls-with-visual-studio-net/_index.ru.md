{{% alert color="primary" %}} 

 Разработчики Visual Studio.NET могут легко перетаскивать элементы управления из**Ящик для инструментов** на Windows или веб-форму. Aspose.Cells Пакет Grid можно загрузить с помощью установщика MSI или в виде набора пакетов DLL. В этой статье объясняется, что нужно сделать, чтобы убедиться, что элементы управления Aspose.Cells.Grid можно использовать в Visual Studio.NET, если он установлен с помощью библиотек DLL вместо установщика.

{{% /alert %}} 
## **Интеграция элементов управления Grid Aspose.Cells с Visual Studio.NET**
Чтобы интегрировать элементы управления Grid Aspose.Cells с Visual Studio.NET:

1. Откройте панель инструментов.
1. Выберите вкладку «Общие» (или любую другую вкладку, на которую вы хотите добавить элементы управления).
1. Щелкните правой кнопкой мыши вкладку Общие.
1.  В Visual Studio.NET 2003: выберите**Добавить/удалить элементы** из меню.
1. В Visual Studio.NET 2005 выберите**Выбрать товары** из меню. Появится диалоговое окно Customize Toolbox (этот процесс более или менее одинаков для более новых IDE VS.NET (например, VS.NET 2013/2015 или новее)).
1.  Нажмите**Просматривать** и найдите файлы Aspose.Cells.GridDesktop.dll и Aspose.Cells.GridWeb.dll.
1.  Выберите библиотеки DLL и нажмите**Открытым**. Диалоговое окно Customize Toolbox теперь будет содержать элементы управления из Aspose.Cells Grid Suite. Новые добавленные элементы управления будут выделены диалоговым окном.
1.  Нажмите**ХОРОШО** чтобы добавить элементы управления в панель инструментов Visual Studio.NET.

 элементы управления сеткой Aspose.Cells будут добавлены на панель инструментов**Общий** вкладка Только элемент управления GridWeb не активен. Это потому, что мы работаем над приложением Windows Forms. GridWeb доступен только при работе с веб-формами, тогда как GridDesktop можно использовать только с формами Windows.
