{{% alert color="primary" %}} 

При создании книги с большими наборами данных или чтении большого файла Excel от Microsoft обязательно потребуется больше времени и ресурсов. Объем памяти, который процесс займет, всегда вызывает опасения. Существуют меры, которые могут быть приняты для решения этой проблемы. GridWeb Aspose.Cells предоставляет некоторые соответствующие параметры и API для снижения и оптимизации использования памяти. Также это может помочь процессу работать более эффективно и быстро. Для листа, содержащего большие наборы данных ячеек, можно загружать набор данных асинхронно, что может улучшить общую производительность для пользовательского опыта.

{{% /alert %}} 

Используйте параметр GridWeb.EnableAsync для оптимизации использования памяти и производительности для наборов данных ячеек. При построении большого набора данных для ячеек, если вы установите параметр в true, загрузка данных будет основана только на текущей видимой области окна. Как видно из краткого описания, при прокрутке наборов данных ячеек на листе GridWeb будут загружены новые данные окна только на основе текущего положения прокрутки.
