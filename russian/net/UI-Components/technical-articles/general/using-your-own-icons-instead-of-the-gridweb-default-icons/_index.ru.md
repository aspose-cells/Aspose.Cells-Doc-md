{{% alert color="primary" %}} 

Иногда может потребоваться использовать собственные значки (изображения) вместо значков по умолчанию элемента управления Aspose.Cells.GridWeb. В этой статье объясняется, как это сделать.

{{% /alert %}} 

Значки элемента управления по умолчанию расположены в пути URL-адреса "/acw_client/". Путь к файлу может быть следующим: "C:\Program Files\Aspose\Aspose.Cells for .NET\acw_client» по умолчанию. В этой папке вы найдете такие файлы, как submit.gif, save.gif и т. д. Если вы хотите заменить эти изображения своими собственными, добавьте раздел конфигурации в файл web.config вашего веб-приложения.

**XML**

{{< highlight "csharp" >}}

 <appSettings>

 <add key="Aspose.Cells.GridWeb.acw_client_path" value="/acw_client/" />

</appSettings>



{{< /highlight >}}

Возможно, вы заметили, что эта конфигурация влияет только на путь к образам элементов управления и не влияет на путь клиентских скриптов элемента управления. Например, если вы запустите свою страницу с помощью элемента управления GridWeb и проверите исходный файл в браузере, вы можете обнаружить, что acw_ клиент_path элемента DIV сетки по-прежнему говорит: «/yourApp/webform1.aspx/». В некоторых случаях может потребоваться переопределить путь к клиентскому скрипту. Чтобы заставить элемент управления использовать переопределенный путь к изображению в качестве пути к клиентскому скрипту, добавьте еще один параметр конфигурации в раздел appSettings.
**XML**

{{< highlight "csharp" >}}

 <add key="Aspose.Cells.GridWeb.force_script_path" value="true" />



{{< /highlight >}}

{{% alert color="primary" %}} 

Эта конфигурация вступит в силу только с лицензированным элементом управления.

{{% /alert %}}
