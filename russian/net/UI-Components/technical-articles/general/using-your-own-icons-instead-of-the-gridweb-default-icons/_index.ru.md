{{% alert color="primary" %}} 

Иногда вам может потребоваться использовать свои собственные значки (изображения) вместо значков элемента управления Aspose.Cells.GridWeb. В этой статье объясняется, как это сделать.

{{% /alert %}} 

Значки элемента управления по умолчанию находятся по адресу URL "/acw_client/". Путь к файлу по умолчанию может быть: "C:\Program Files\Aspose\Aspose.Cells for .NET\acw_client". Вы найдете такие файлы, как submit.gif, save.gif и т.д. в этой папке. Если вы хотите заменить эти изображения на свои собственные, добавьте раздел конфигурации в файл web.config вашего веб-приложения.

**XML**

{{< highlight csharp >}}

 <appSettings>

 <add key="Aspose.Cells.GridWeb.acw_client_path" value="/acw_client/" />

</appSettings>



{{< /highlight >}}

Вы могли заметить, что эта конфигурация влияет только на путь изображений элемента управления и не влияет на путь клиентских скриптов элемента управления. Например, если вы запустите вашу страницу с элементом управления GridWeb и проверите исходный файл в браузере, вы можете обнаружить, что свойство acw_client _path элемента DIV сетки по-прежнему говорит: "/yourApp/webform1.aspx/". В некоторых случаях вам может потребоваться переопределить путь клиентского скрипта. Чтобы заставить элемент управления использовать переопределенный путь изображения в качестве пути клиентского скрипта, добавьте еще одну настройку конфигурации в раздел appSettings
**XML**

{{< highlight csharp >}}

 <add key="Aspose.Cells.GridWeb.force_script_path" value="true" />



{{< /highlight >}}

{{% alert color="primary" %}} 

Эта конфигурация будет применяться только с лицензированным элементом управления.

{{% /alert %}}
