{{% alert color="primary" %}} 

Иногда вам нужно найти простой способ для ячеек, содержащих формулы. У вас должен быть подход, чтобы быстро прочитать все формулы, чтобы проверить наличие ошибок. Это может помочь вам отслеживать данные, используемые в формуле.

Aspose.Cells.GridWeb предоставляет функцию, которая называется «Показать формулы» в Microsoft Excel.

{{% /alert %}} 
## **Использование функции «Показать формулы»**
Все, что вам нужно использовать сочетание клавиш «CTRL + ~», чтобы отобразить все формулы на листе.

Например, мы загружаем файл шаблона в GridWeb с формулами в столбце C, вот код, который мы просто использовали для загрузки файла.
