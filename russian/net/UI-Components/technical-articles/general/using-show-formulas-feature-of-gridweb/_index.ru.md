Иногда вам может потребоваться найти простой способ для ячеек, содержащих формулы. Вам нужен подход, позволяющий быстро просматривать все формулы для проверки на наличие ошибок. Это может помочь вам отслеживать данные, используемые в формуле.

Aspose.Cells.GridWeb предоставляет функцию, которая называется Показать формулы в Microsoft Excel.

{{% /alert %}}
