 Aspose.Cells.GridWeb предоставляет некоторые встроенные командные кнопки, такие как**Представлять на рассмотрение** а также**Сохранять**. Используйте эти кнопки для выполнения связанных задач.

В этой статье показано, как отправлять данные на сервер, а не просто щелкнуть встроенную в GridWeb**Сохранять** командную кнопку, а нажав обычную кнопку ASP.NET (веб-управление). Цель этой статьи — показать гибкость Aspose.Cells.GridWeb. Кроме того, в этой статье также используются специальные функции, предоставленные Aspose.Cells.GridWeb для использования в сценарии на стороне клиента.

{{% /alert %}} 
## **Отправка данных сетки с помощью кнопки ASP.NET**
Aspose.Cells.GridWeb предоставляет три встроенные кнопки (**Представлять на рассмотрение**, **Сохранять** а также**Отменить** ). После редактирования в GridWeb пользователь может щелкнуть значок**Представлять на рассмотрение** или же**Сохранять** кнопку на панели вкладок, чтобы разрешить GridWeb отправлять данные на сервер. Если пользователь щелкает вкладку листа, элемент управления GridWeb выполняет ту же задачу, что и встроенные командные кнопки. Aspose.Cells.GridWeb также поддерживает добавление этой функции в общий элемент управления ASP.NET Button, но вам потребуется добавить в приложение дополнительный код.
### **1. Создание тестового приложения**
Откройте интегрированную среду разработки Visual Studio.NET и создайте новый проект веб-приложения ASP.NET. После создания приложения в проект будет добавлена стандартная страница WebForm1.aspx. Перетащите элемент управления GridWeb из панели инструментов в веб-форму. Если вы не можете найти элемент управления GridWeb на панели инструментов, перейдите на эту страницу:[Интеграция элементов управления Grid Aspose.Cells с Visual Studio.NET](/cells/ru/net/integrate-aspose-cells-grid-controls-with-visual-studio-net/) чтобы узнать больше об этом. После добавления элемента управления GridWeb в веб-форму также добавьте веб-элемент управления Button из панели инструментов в веб-форму.
### **2. Добавление кода в событие Page_Load**
Теперь пришло время добавить код на страницу_Событие загрузки веб-формы. Вы можете дважды щелкнуть веб-форму в режиме конструктора, и VS.NET IDE автоматически перенесет вас на страницу_Загрузите обработчик событий, в котором вам потребуется использовать коллекцию атрибутов кнопки для переопределения ее события OnClick.

{{% alert color="primary" %}} 

ASP.NET Элемент управления Button — это элемент управления на стороне сервера. Всякий раз, когда он нажимается, запускается событие на стороне сервера, но если вы хотите использовать этот элемент управления Button для выполнения некоторого кода на стороне клиента (обычно с использованием javascript), вам необходимо изменить его атрибут onclick в событии Page_Load. Aspose.Cells.GridWeb предоставляет некоторые функции, которые можно вызывать в javascript для работы с элементом управления GridWeb со стороны клиента. В приведенном ниже примере мы использовали функции updateData и validateAll (которые являются клиентскими функциями) GridWeb для обновления и проверки данных Grid.

### **3. Запуск приложения**
Теперь вы можете скомпилировать и запустить свое приложение, нажав Ctrl+F5, и страница откроется в новом окне браузера. Давайте добавим некоторые значения в GridWeb и нажмем кнопку «Отправить данные Grid на сервер», и вы увидите, что происходит обратная передача для обновления и проверки данных GridWeb.
