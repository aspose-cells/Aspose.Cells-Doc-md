## **Есть ли ограничения в оценочной версии элементов управления Aspose.Cells Grid?**
Нет, в оценочной версии этих контролов нет ограничений функциональности.

Оценочная версия предоставляет все функции лицензированной версии, за исключением того, что добавляется дополнительный лист (содержащий **Предупреждение об оценочном авторском праве**) в выходные файлы.


## **На рынке доступно множество элементов управления Grid. Зачем мне покупать элементы управления Grid Aspose.Cells?**
Ну, элементы управления Grid Aspose.Cells имеют очень разумную цену, доступную для всех типов пользователей. За очень разумную цену,

они предоставляют вам набор из двух контролов для работы в приложениях Windows & Web. 

Более того, это не просто обычная таблица, это ваш **Просмотрщик, редактор и создатель электронных таблиц** одновременно. 

Вы можете привязать его не только к любому виду источника данных (как к обычным элементам управления таблицей), но и создавать и управлять файлами Excel. Он также предоставляет мощный и богатый **Расчетный движок формул** для вычисления не только встроенных функций (поддерживаемых элементами управления Grid Aspose.Cells), но и пользовательских формул, определенных вами. Есть еще много других функций набора элементов управления Grid Aspose.Cells, которые здесь не могут быть охвачены, пожалуйста, обратитесь к странице типов редакций для более подробного списка функций.

## **Как я могу развернуть свой веб-проект/решение на основе Aspose.Cells.GridWeb на сервере?**
Если вам нужно развернуть веб-приложение с управлением GridWeb, вам следует скопировать каталог "acw_client" в корневую папку вашего проекта, иначе ваше веб-приложение (развернутое на сервере) не сможет найти его.

Вы всегда можете указать путь сценария, добавив следующие строки кода в раздел конфигурации (например, в файл web.config вашего проекта VS.NET). "acw_client" - это папка, содержащая файлы, которые использует Aspose.Cells.GridWeb для управления внутренней конфигурацией. В этой папке находятся файлы сценариев, файлы изображений и другие файлы для определения поведения GridWeb и выполнения других операций. Файл конфигурации используется для предотвращения использования элементов управления встроенных ресурсов клиента (изображения, сценарии и т. Д.), что полезно в некоторых случаях / сценариях.

**XML**

{{< highlight csharp >}}

 <appSettings>

  <add key="aspose.cells.gridweb.acw_client_path" value="/grid/acw_client/"/>

  <add key="aspose.cells.gridweb.force_script_path" value="true"/>

  <add key="aspose.cells.gridweb.forcepath" value="true"/>

 </appSettings>

{{< /highlight >}}

{{% alert color="primary" %}} 

Путь всегда связан с каталогом проекта. Не следует использовать каталог, находящийся за пределами каталога проекта. Поэтому необходимо скопировать каталог "acw_client" (@ ваш каталог установки GridWeb) в каталог проекта.

{{% /alert %}}
