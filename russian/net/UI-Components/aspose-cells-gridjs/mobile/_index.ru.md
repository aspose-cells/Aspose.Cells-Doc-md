## Используйте GridJs на мобильном устройстве

 Вы можете без проблем использовать GridJs на мобильном устройстве.

GridJs может адаптировать мобильный экран.

 Запустите[ демонстрация GridJs](https://github.com/aspose-cells/Aspose.Cells-for-.NET/tree/master/Examples_GridJs "demo of GridJs ")

Порт демонстрационного проекта по умолчанию — 24262.

 Вам необходимо получить доступ к странице через IP-адрес на мобильном терминале http://localhost:24262/GridJs2/List.

Вы можете использовать небольшой инструмент, например: iisexpress proxy.

Установите с помощью следующей команды:

*npm установить -g iisexpress-прокси*

Запустите команду, чтобы завершить сопоставление портов:

*iisexpress-прокси 24262 до 82*

![todo:команда iisexpress](iisproxy_port_map.png)

Затем мы можем открыть демонстрационную веб-страницу в мобильном браузере.

Перейдите на страницу http://yourip:82/GridJs2/List и выберите файл для открытия.

![todo:навигация по экрану на мобильном телефоне](gridjs_edit_on_mobile.png)

Затем мы можем выполнять операции редактирования, как на ПК.
