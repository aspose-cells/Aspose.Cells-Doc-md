## Основы GridJ

 Aspose.Cells.GridJs — это стандартная библиотека .NET, которая позволяет пользователям разрабатывать веб-приложения для быстрого и простого отображения/редактирования электронных таблиц.

Aspose.Cells.GridJs поддерживает импорт популярных форматов файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, CSV, SpreadsheetML, ODS).

Он также позволяет экспортировать файлы Excel в PDF, HTML и т. д. Ниже приведены основные этапы процесса разработки веб-приложения GridJ.

- Реализуйте GridCacheForStream, чтобы написать собственную бизнес-логику для хранения кэша.
- Настройте действие контроллера для получения json из файла электронной таблицы. Вы можете использовать API-интерфейсы GridJsWorkbook.ImportExcelFile и GridJsWorkbook.ExportToJson, GridJ автоматически сохранит файл распространения в кеше.
- Настройте действие контроллера, чтобы получить json для операции обновления. Вы можете использовать GridJsWorkbook.UpdateCell API, GridJs выполнит операцию обновления в кеше и вернет обновленный json.
- Настройте действие контроллера, чтобы получить URL-адрес файлов изображений/фигур в электронной таблице, GridJs автоматически заархивирует все изображения/фигуры в кеше. Он будет использовать GridCacheForStream.GetFileUrl API.
- Настройте действие контроллера для получения файла в кеше, таким образом, мы можем получить zip-файл изображений/фигур или файл электронной таблицы в кеше. Он будет использовать GridCacheForStream.LoadStream API.
- Настройте действие контроллера для загрузки электронной таблицы. Вы можете использовать GridJsWorkbook.SaveToCacheWithFileName API.
