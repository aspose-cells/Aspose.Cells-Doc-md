Чтобы облегчить разработчикам обновление, мы поддерживаем документ, описывающий информацию, важную для существующих пользователей, особенно для тех, кто использовал старую версию Aspose.Grid.Desktop и теперь нуждается в обновлении до новой или объединенной версии Aspose.Cells.GridDesktop.

В этом разделе представлено базовое пошаговое руководство для начинающих, которое поможет им обновить Aspose.Grid.Desktop до элемента управления Aspose.Cells.GridDesktop, а также создать и использовать элемент управления Aspose.Cells.GridDesktop в приложениях Windows.

{{% /alert %}} 
## **Обновление с Aspose.Grid.Desktop до Aspose.Cells.GridDesktop Control**
Разработчики, работавшие с Aspose.Grid.Desktop, могут столкнуться с проблемами при использовании нового Aspose.Cells.GridDesktop при обновлении до него. Ниже приведено пространство имен, которое было переименовано:

1. **Пространство имен Aspose.Grid.Desktop было переименовано в пространство имен Aspose.Cells.GridDesktop.**
## **Создание и использование элемента управления Aspose.Cells.GridDesktop**
### **Создание проекта приложения Windows**
Прежде всего, создайте проект приложения Windows, в котором будет использоваться элемент управления Aspose.Cells.GridDesktop. Выполните следующие шаги;

1. **Откройте интегрированную среду разработки Visual Studio.NET.**
1. **В меню «Файл» выберите «Создать», «Проект».**
1. **Выберите приложение Windows в диалоговом окне «Новый проект».**

После выполнения вышеуказанных действий будет создано оконное приложение с пустой формой.
### **Добавление элемента управления GridDesktop в форму**
 Перетащите элемент управления Aspose.Cells.GridDesktop с панели элементов на форму. После этого на форме появится элемент управления, над которым можно выполнить ряд действий, например**изменить размер**, **изменить его свойства**, **так далее**.
### **Запустите приложение**
 Наконец, запустите приложение, нажав**Ctrl+F5** или нажав на**Начинать** кнопка. Когда приложение запускается, мы видим форму, содержащую пустой элемент управления Aspose.Cells.GridDesktop. Теперь вы можете добавлять значения в ячейки, щелкая их. На следующем снимке экрана показано, как ваше приложение будет выглядеть в конечном итоге.

![дело:изображение_альтернативный_текст](how-to-use-aspose-cells-griddesktop_1.png)

В элемент управления можно не только добавлять значения, но и выполнять другие задачи, такие как изменение высоты строки или ширины столбца, копирование (Ctrl+C) или вырезание (Ctrl+X) данных ячейки в буфер обмена и вставка (Ctrl+V) данных в ячейку и т. д. Чтобы выполнить дополнительные операции, щелкните правой кнопкой мыши элемент управления, чтобы увидеть его контекстное меню.

{{% alert color="primary" %}} 

Aspose.Grid.Desktop был переименован и стал частью Aspose.Cells, поэтому больше нельзя использовать старую версию Aspose.Grid.Desktop. Просто подчеркну, что для обновления до последней версии компонента Aspose.Cells.GridDesktop требуется не так уж много работы. В API нет никаких изменений, поскольку классы с элементами, структурами, перечислениями и т. д. остаются прежними. Единственное изменение, которое было сделано, касается пространства имен элемента управления.

{{% /alert %}}
