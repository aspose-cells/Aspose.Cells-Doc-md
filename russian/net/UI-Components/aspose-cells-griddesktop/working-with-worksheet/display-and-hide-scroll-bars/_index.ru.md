{{% alert color="primary" %}}

Полосы прокрутки удобны для навигации по содержимому широких и глубоких электронных таблиц, то есть содержащих много строк и столбцов. Большинство приложений поддерживают два типа полосы прокрутки:

- Вертикальная полоса прокрутки
- Горизонтальная полоса прокрутки

Оба они находятся в Microsoft Excel.

Aspose.Cell GridDesktop API предоставляет горизонтальные и вертикальные полосы прокрутки для прокрутки содержимого рабочего листа. С помощью API-интерфейсов Aspose.Cells.GridDesktop разработчики могут контролировать видимость обеих этих полос прокрутки.

{{% /alert %}}

## **Управление видимостью полосы прокрутки**

Чтобы управлять видимостью полосы прокрутки в GridDesktop, используйте свойства IsVerticalScrollBarVisible и IsHorizontalScrollBarVisible. В приведенных ниже примерах показано, как скрыть и показать полосы прокрутки.

### **Примеры программирования: скрытие полос прокрутки**
