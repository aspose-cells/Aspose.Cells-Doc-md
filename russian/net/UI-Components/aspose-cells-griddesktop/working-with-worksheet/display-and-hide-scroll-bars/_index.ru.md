{{% alert color="primary" %}}

Полосы прокрутки полезны для навигации по содержимому расширенных таблиц, имеющих много строк и столбцов. Большинство приложений поддерживают два типа полос прокрутки:

- Вертикальная полоса прокрутки
- Горизонтальная полоса прокрутки

Оба типа присутствуют в Microsoft Excel.

API GridDesktop Aspose.Cells предоставляет горизонтальные и вертикальные полосы прокрутки для прокрутки содержимого листа. С помощью API GridDesktop Aspose.Cells разработчики могут контролировать видимость обеих этих полос прокрутки.

{{% /alert %}}
