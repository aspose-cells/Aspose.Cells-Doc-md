Эта тема обсуждает способ изменения имени листа при использовании Aspose.Cells.GridDesktop.

{{% /alert %}} 
## **Пример**
Переименование листа - это простая задача, которая может быть очень полезной. Например, если вы хотите пометить листы, чтобы было понятно, что каждый из них содержит. Например, у модели может быть 12 листов, каждый назван по месяцам, чтобы хранить информацию о расходах.

Чтобы переименовать лист с помощью элемента управления Aspose.Cells.GridDesktop:

1. Добавьте элемент управления Aspose.Cells.GridDesktop на форму.
1. Получите ссылку на нужный лист.
1. Установите имя листа, используя его ссылку.
