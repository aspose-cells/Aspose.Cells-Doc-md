{{% alert color="primary" %}} 

Aspose.Cells.GridDesktop - это графический контроль, который позволяет пользователям работать с данными в режиме WYSIWYG. Мы много изучали API Aspose.Cells.GridDesktop для выполнения различных задач. Мы также обсудили управление настройками форматирования сетки, листов, строк и столбцов и ячеек программно с помощью надежного и удобного API Aspose.Cells.GridDesktop. Но всё это было с точки зрения программирования. В этой теме мы обсудим, как простой конечный пользователь может контролировать настройки форматирования Aspose.Cells.GridDesktop с помощью её **Диалога Стиля**.

{{% /alert %}} 
## **Использование диалога стиля**
Чтобы использовать **Диалог Стиля** в Aspose.Cells.GridDesktop, пожалуйста, следуйте следующим шагам:

- **Щелкните правой кнопкой мыши** на сетке. Перед вами появится контекстное меню с предложением некоторых вариантов
- Выберите вариант **Формат ячеек...** из контекстного меню.

После выбора варианта **Формат ячеек...**, перед вами появится **Диалог Стиля**. **Диалог Стиля** состоит из нескольких вкладок, включая:* **Число**

- **Выравнивание**
- **Граница**
- **Шрифт**
- **Защита**

Давайте обсудим и рассмотрим каждую вкладку **Диалога Стиля** по отдельности.
### **Вкладка Число**
**Вкладка Число** используется для установки настроек **Формата числа** ячейки. Вы можете выбрать категорию формата числа, установить количество десятичных знаков для значения, выбрать набор символов, а также выбрать, как показывать отрицательное значение в ячейке.
### **Вкладка Выравнивание**
С помощью вкладки **Выравнивание** пользователи могут задать горизонтальное и вертикальное выравнивание текста. Они также могут использовать другие параметры управления текстом, такие как автоперенос текста, сжатие ячейки под текст и объединение ячеек.
### **Вкладка Граница**
Вкладка **Граница** позволяет пользователям рисовать границы любого типа и цвета всего лишь несколькими кликами. Все, что им нужно сделать, - это выбрать несколько параметров, чтобы нарисовать границу по своему желанию.
### **Вкладка Шрифт**
Используя вкладку **Шрифт**, пользователи могут изменить название, размер, цвет и стиль своего шрифта и т. д. Пользователи также могут предварительно просмотреть свои настройки шрифта перед их применением к ячейкам. Таким образом, это упрощает задачу пользователей в форматировании ячеек **прямо вовремя**.
### **Вкладка защиты**
Вкладка **Защита** просто позволяет пользователям защищать свои ячейки, блокируя или скрывая их. 

{{% alert color="primary" %}} 

Разработчики также могут программно открыть **Диалоговое окно стиля**, вызвав метод **ShowStyleDialog** объекта **GridDesktop**
