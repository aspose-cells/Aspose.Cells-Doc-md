{{% alert color="primary" %}} 

Aspose.Cells.GridDesktop — это элемент управления на основе графического интерфейса, который позволяет пользователям работать со своими данными в режиме WYSIWYG. Мы изучили API из Aspose.Cells.GridDesktop для выполнения различных задач. Мы также обсудили управление настройками форматирования сетки, рабочих листов, строк и столбцов и Cells программно, используя надежный и простой API Aspose.Cells.GridDesktop. Но все это было с точки зрения программирования. В этом разделе мы обсудим, как простой конечный пользователь может управлять настройками форматирования Aspose.Cells.GridDesktop, используя его**Диалог стиля**.

{{% /alert %}} 
## **Использование диалогового окна стиля**
 Использовать**Диалог стиля** в Aspose.Cells.GridDesktop выполните следующие действия:

- **Щелкните правой кнопкой мыши** на сетке. Перед тем, как вы предложите некоторые варианты, появится контекстное меню.
-  Выбирать**Формат Cells...** вариант из контекстного меню.

 После выбора**Формат Cells...** вариант,**Диалог стиля** появится перед вами.**Диалог стиля** состоит из нескольких вкладок, среди которых:* **Число**

- **Выравнивание**
- **Граница**
- **Шрифт**
- **Защита**

 Давайте обсудим и рассмотрим каждую вкладку**Диалог стиля** по одному.
### **Вкладка «Число»**
**Число** вкладка используется для установки**Формат номера** настройки ячейки. Вы можете выбрать категорию числового формата, установить десятичные разряды для значения, выбрать набор символов, а также выбрать способ отображения отрицательного значения в ячейке.
### **Вкладка «Выравнивание»**
 С использованием**Выравнивание** на вкладке пользователи могут установить горизонтальное и вертикальное выравнивание текста. Даже они также могут использовать другие параметры управления текстом, такие как перенос текста, сжатие ячейки для соответствия тексту и объединение ячеек.
### **Вкладка «Граница»**
**Граница** вкладка позволяет пользователям рисовать границы любого типа и цвета всего за несколько кликов. Все, что им нужно сделать, это просто выбрать несколько вариантов, чтобы нарисовать границу в соответствии с их желаниями.
### **Вкладка «Шрифт»**
 С использованием**Шрифт** на вкладке пользователи могут изменить имя шрифта, размер, цвет, стили и т. д. Пользователи также могут предварительно просмотреть настройки шрифта, прежде чем применять их к ячейкам. Таким образом, это упрощает работу пользователей по форматированию ячеек.**Вовремя**.
### **Вкладка «Защита»**
**Защита** tab просто позволяет пользователям защищать свои ячейки, блокируя или скрывая их.
