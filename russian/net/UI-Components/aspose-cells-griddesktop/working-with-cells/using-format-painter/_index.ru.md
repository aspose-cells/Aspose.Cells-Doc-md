{{% alert color="primary" %}} 

Редактор форматов — это функция MS Excel, которая была адаптирована в Aspose.Cells.GridDesktop. Это очень хорошая функция. Для тех людей, которые еще не использовали эту функцию, рисовальщик форматов позволяет пользователям применять настройки форматирования любой выделенной ячейки к другой ячейке. Реализация этой функции очень проста. В этой теме мы рассмотрим и это.

{{% /alert %}} 
## **Использование редактора форматов**
 Особенность**Формат Painter** требует от пользователей выбрать ячейку, параметры форматирования которой вы хотите применить к другим ячейкам, а затем вызвать**StartFormatPainter** метод**GridDesktop**. Существует два режима форматного рисовальщика:

- **Однократное использование Format Painter**
- **Всегда использовать формат по образцу**
### **Однократное использование Format Painter**
 Если разработчики хотят использовать функцию рисования формата только один раз только для применения параметров форматирования сфокусированной ячейки к одной ячейке, они могут сделать это, вызвав**StartFormatPainter** метод и передача**ЛОЖЬ** ценность для него. Этот**ЛОЖЬ** значение навсегда запретит рисунку формата рисовать.

### **Всегда использовать формат по образцу**
Использование форматного рисовальщика всегда является полезной функцией, когда нам нужно применить настройки форматирования более чем к одной ячейке. Разработчики могут реализовать эту функцию, просто позвонив**StartFormatPainter** метод и передача**истинный** ценность для него.
