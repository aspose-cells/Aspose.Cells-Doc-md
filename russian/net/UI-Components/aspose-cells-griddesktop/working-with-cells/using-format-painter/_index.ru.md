{{% alert color="primary" %}} 

Копирование формата - это функция MS Excel, которая была адаптирована в Aspose.Cells.GridDesktop. Это очень удобная функция. Для тех, кто еще не использовал эту функцию, копирование формата позволяет пользователям применять настройки форматирования любой выбранной ячейки к другой ячейке. Реализация этой функции очень проста. В этой теме мы также рассмотрим это.

{{% /alert %}} 
## **Использование копирования формата**
Функция **Копирование формата** требует от пользователей выбрать ячейку, настройки форматирования которой они хотят применить к другим ячейкам, а затем вызвать метод **StartFormatPainter** **GridDesktop**. Существует два режима копирования формата:

- **Использовать копирование формата один раз**
- **Использовать копирование формата всегда**
### **Использовать копирование формата один раз**
Если разработчики хотят использовать функцию копирования формата только один раз, чтобы применить настройки форматирования выбранной ячейки к одной ячейке, они могут сделать это, вызвав метод **StartFormatPainter** и передав в него значение **false**. Это значение **false** запрещает копирование формата вечно.

### **Использовать копирование формата всегда**
Использование копирования формата всегда - полезная функция, когда нам нужно применить настройки форматирования на несколько ячеек. Разработчики могут использовать эту функцию, просто вызвав метод **StartFormatPainter** и передав в него значение **true**.

Этот вид копирования формата продолжает работать вечно, пока мы его не остановим. Поэтому, чтобы остановить копирование формата от работы всегда, мы можем просто вызвать метод **EndFormatPainter** **GridDesktop**.
