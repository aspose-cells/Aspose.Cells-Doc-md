## **Доступ и изменение значения Cell с помощью Aspose.Cells.GridDesktop**
 Прежде чем получить доступ и изменить значение ячейки, мы должны знать, как получить доступ к ячейкам. Существует три подхода к доступу к ячейкам рабочего листа. Подробнее об этих трех подходах см.[Доступ к Cells на рабочем листе.](/cells/ru/net/accessing-cells-in-a-worksheet/)

Каждая ячейка имеет свойство с именем Value . Таким образом, после доступа к ячейке разработчики могут получить доступ к содержимому свойства Value и изменить его, чтобы получить доступ и изменить значение ячейки.

**ВАЖНЫЙ:**Использование свойства Value ячейки для изменения ее значения является хорошим подходом для установки значения одной или нескольких ячеек. Если вам нужно установить значения многих ячеек, то производительность этого подхода не будет хорошей. Итак, чтобы установить значения многих ячеек, вы должны использовать**Сетцеллвалуе** метод ячейки для повышения производительности ваших приложений.
