## **Вступление**
Aspose.Cells Grid Controls поддерживает множество встроенных функций. Эти функции относятся к следующим категориям:

- **Математические функции** , предоставляет огромный набор функций, которые можно использовать в многочисленных математических вычислениях.
- **Функции текста и данных** , предоставляет различные функции для выполнения различных операций с текстом и данными.
- **Функции даты и времени** , предоставляет некоторые полезные функции, связанные с данными и временем
- **Логические функции** , предоставляет небольшой набор логических функций для обработки некоторых логических операций с данными.
