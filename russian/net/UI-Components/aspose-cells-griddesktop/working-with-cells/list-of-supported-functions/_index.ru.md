## **Введение**
Aspose.Cells.GridDesktop Control поддерживает множество встроенных функций. Эти функции относятся к следующим категориям:

- **Математические функции**, предоставляют большой набор функций, которые можно использовать во многих математических расчетах
- **Функции текста и данных**, предоставляют различные функции для выполнения различных операций с текстом и данными
- **Функции даты и времени**, предоставляют полезные функции, связанные с датой и временем
- **Логические функции**, предоставляют небольшой набор логических функций для обработки логических операций над данными
