**Добро пожаловать в Aspose.Cells for .NET**

Aspose.Cells for .NET - это программная библиотека, позволяющая разработчикам программного обеспечения обрабатывать файлы электронных таблиц в их собственных приложениях.

В сочетании с API и графическими элементами управления Aspose.Cells for .NET ускоряет программирование и конвертацию Microsoft Excel®.

Aspose.Cells for .NET поддерживает популярные форматы файла электронной таблицы (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые используются вашим бизнесом ежедневно. Он также позволяет экспортировать файлы Excel в форматы PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, JSON, обычный текст и популярные форматы изображений, включая TIFF, JPG, PNG, BMP и SVG.
