Aspose.Cells for .NET — это библиотека для программирования электронных таблиц, которая позволяет разработчикам программного обеспечения манипулировать файлами электронных таблиц и преобразовывать их из своих собственных приложений. Aspose.Cells for .NET поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) и позволяет экспортировать файлы Excel в форматы PDF, XPS, HTML, MHTML, обычный текст, TIFF, JPG, 0761103611, 0481112 и 3,481112

 Последний номер API можно найти по адресу[Aspose.Cells for .NET API Ссылка](https://reference.aspose.com/cells/net).
