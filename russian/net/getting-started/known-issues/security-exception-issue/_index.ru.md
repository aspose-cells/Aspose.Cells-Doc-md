## **Проблема с исключением безопасности**
Некоторые пользователи могут получать ошибку "Исключение безопасности" при попытке использовать Aspose.Cells. Эта проблема обычно возникает в веб-приложении.
### **Объяснение**
Aspose.Cells должен вызывать некоторые **API Win32 GDI**, чтобы предоставить некоторые важные функции. Если у веб-сервера установлен строгий уровень доверия, может возникнуть это исключение безопасности.
### **Решение**
Попробуйте создать новый набор разрешений, чтобы дать разрешение на безопасность Aspose.Cells.dll с включенной опцией "Разрешить вызовы к неподконтрольным сборкам".
