{{% alert color="primary" %}} 

Aspose.Cells - это чистый компонент .NET, который может упростить процесс развертывания, используя развертывание XCOPY. Для установки Aspose.Cells вы можете просто скопировать сборку компонента (Aspose.Cells.dll) в каталог для вашего приложения и оно сможет сразу начать его использовать. Это возможно благодаря самоописывающей природе компонентов .NET. Такой тип развертывания также полностью отсутствует влияние на процесс установки.

{{% /alert %}} 
## **Развертывание**
Aspose.Cells поддерживает как 32-разрядные, так и 64-разрядные окружения. При установке Aspose.Cells for .NET компонента с помощью установщика Aspose.Cells MSI, различные DLL добавляются в разные папки в папке установки Aspose.Cells ${installation_Path}. Смотрите описание в таблице, какая папка содержит сборки, которые вам нужно использовать с определенной версией .NET Framework.

|**Папка**|**Описание**|
| :- | :- |
|net2.0|Содержит сборки для использования с .NET Framework 2.0, 3.0, 3.5, 4.0 и Mono. Это сборки, которые обычно следует использовать в 32-разрядных и 64-разрядных окружениях.|
|net2.0_AuthenticodeSigned|То же самое, но сборки подписаны цифровой подписью Authenticode. Подписанные сборки могут загружаться медленнее, чем без Authenticode|
|net3.5_ClientProfile|Содержит сборки для использования с .NET Framework 3.5 или 4.0 Client Profile.|
|net3.5_ClientProfile_AuthenticodeSigned|То же самое, но сборки подписаны цифровой подписью Authenticode. Подписанные сборки могут загружаться медленнее, чем без Authenticode.|
|net3.5|Содержит сборки для использования с .NET Framework 3.5 или 4.0.|
|net3.5_AuthenticodeSigned|То же самое, но сборки подписаны цифровой подписью Authenticode. Подписанные сборки могут загружаться медленнее, чем без Authenticode.|
|net4.0|Содержит сборки для использования с .NET Framework 4.0 и 4.5.|
|netStandard|Содержит сборки для использования с .Net Standard 2.0|
|netcoreapp2.1|Содержит сборки для использования с .Net core 2.1|
|Xamarin.iOS|Содержит сборки для использования с Xamarin.iOS|
|Xamarin.Android|Содержит сборки для использования с Xamarin.Android|
|net5.0|Содержит сборки для использования с .net5.0.|
|net6.0|Содержит сборки для использования с .net6.0.|
|net8.0|Включает сборки для использования с .net8.0.|
|net9.0|Включает сборки для использования с .net9.0.|

{{% alert color="primary" %}} 

В VS.NET (например 2005, 2008, 2010, 2012 и т. д.) при добавлении ссылки на Aspose.Cells, диалоговое окно Добавить ссылку ссылается на файлы Aspose.Cells.dll в папке net2.0 или net3.5 соответственно. (Для дальнейшей справки читайте Референсинг Aspose.Cells из .NET проекта.) Вы можете изменить ссылку на библиотеку в соответствии с вашим окружением. Обратите внимание, что если целевая версия фреймворка проекта - .NET Framework 3.5/4 Client Profile, используйте файл компонента Aspose.Cells.dll, расположенный в папке net_ClientProfile.
