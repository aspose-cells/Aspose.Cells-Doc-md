##  **Поддерживаемые платформы**
Aspose.Cells for .NET Продукт охватывает большинство популярных сред разработки и платформ развертывания.

|**Особенность**|**Описание**|
| :- | :- |
|[.NET Каркас](/cells/ru/net/system-requirements/#net-frameworks)|Используйте Aspose.Cells for .NET для сборки ASP.NET, веб-служб, WinForms или любого другого типа приложения для любой .NET Framework версии 2.0 или более поздней, включая версии профиля клиента. Полная поддержка 32-битных и 64-битных версий.|
|[.NET Стандарт 2.0](/cells/ru/net/system-requirements/#net-standard)|Папка netStandard содержит сборки для использования с платформой, реализующей .NET Standard 2.0, например .NET Core 2.1/3.1. Последняя версия Aspose.Cells API также поддерживает .NET 5.0, .NET 6.0, .NET 7.0 и .NET8.0.|
|[COM](/cells/ru/net/system-requirements/)|Может оказаться спасением, если вам приходится разрабатывать технологии, предшествующие .NET.|
