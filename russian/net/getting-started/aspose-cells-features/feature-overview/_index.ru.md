## **Поддерживаемые платформы**
Продукт Aspose.Cells for .NET охватывает большинство популярных сред разработки и платформ развертывания. 

|**Функция**|**Описание**|
| :- | :- |
|[.NET Framework](/cells/ru/net/system-requirements/#net-frameworks)|Используйте Aspose.Cells for .NET для создания ASP.NET, веб-сервисов, WinForms или любого другого типа приложений для любых версий .NET Framework 2.0 и выше, включая версии Client Profile. Полная поддержка 32-битных и 64-битных систем.|
|[.NET Standard 2.0](/cells/ru/net/system-requirements/#net-standard)|Папка 'netStandard' содержит сборку для использования с платформой, реализующей .NET Standard 2.0, такой как .NET Core 2.1/3.1. Последняя версия Aspose.Cells API также поддерживает .NET 5.0, .NET 6.0, .NET 7.0 и .NET8.0.|
|[COM](/cells/ru/net/system-requirements/)|Может оказаться спасением, если вам приходится разрабатывать с использованием старых технологий до .NET.|

|[Windows Azure](/cells/ru/net/system-requirements/)|Aspose.Cells for .NET также работает в облаке Windows Azure.|
