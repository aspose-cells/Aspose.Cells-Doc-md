{{% alert color="primary" %}} 

Aspose.Cells можно использовать для сохранения книг Excel в формате HTML. Экспортированные рабочие книги HTML Microsoft ориентированы на Excel, т.е. они аналогичны тем, которые экспортируются Microsoft Excel. Вы также можете создавать рабочие книги из HTML, но этот HTML должен быть ориентирован на Excel.

{{% /alert %}} 
## **HTML в Aspose.Cells**
Решение Aspose полностью поддерживает следующие функции документа при преобразовании в HTML:

- Cell Форматирование данных и текста
- Диаграммы и сводные таблицы
- Верхние/нижние колонтитулы
- Границы, Цвета границ
- Изображения, текстовые поля и фигуры рисования
- Гиперссылки
