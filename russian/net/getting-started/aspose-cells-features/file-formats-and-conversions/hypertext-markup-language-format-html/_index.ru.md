{{% alert color="primary" %}} 

Aspose.Cells можно использовать для сохранения ваших книг Excel в формат HTML. Экспортированные книги HTML ориентированы на Microsoft Excel, то есть они аналогичны экспортированным из Microsoft Excel. Вы также можете создавать книги из HTML, но это HTML должно быть ориентировано на Excel.

{{% /alert %}} 
## **HTML в Aspose.Cells**
Решение Aspose полностью поддерживает следующие возможности документа при преобразовании в HTML:

- Данные ячейки и форматирование текста
- Диаграммы и сводные таблицы
- Заголовки/подвалы
- Границы, цвета границ
- Изображения, текстовые поля и геометрические фигуры
- Гиперссылки
