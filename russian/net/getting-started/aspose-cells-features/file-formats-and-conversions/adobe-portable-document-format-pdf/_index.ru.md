{{% alert color="primary" %}} 

Любой документ, загруженный в Aspose.Cells, может быть преобразован в PDF, соответствующий [PDF 1.5](https://docs.fileformat.com/pdf/) или [PDF/A-1b](https://docs.fileformat.com/pdf/a/). Aspose.Cells приложил максимум усилий, чтобы верно преобразовать документы Microsoft Excel в PDF. В большинстве случаев внешний вид конечного PDF-документа будет неотличим от исходного документа Microsoft Excel.

{{% /alert %}} 
## **PDF в Aspose.Cells**
Решение Aspose полностью поддерживает следующие возможности документа при преобразовании в PDF:

- Данные ячейки и форматирование текста
- Диаграммы и сводные таблицы
- Заголовки/подвалы
- Условное форматирование
- Границы, цвета границ
- Изображения, текстовые поля и геометрические фигуры
- Закладки
- Гиперссылки
- Объединенные ячейки, переносы страниц, заголовки страниц

Вы также можете просмотреть [подробную спецификацию](https://docs.aspose.com/cells/net/convert-excel-workbook-to-pdf/) о том, как Aspose.Cells преобразует документ в PDF.
