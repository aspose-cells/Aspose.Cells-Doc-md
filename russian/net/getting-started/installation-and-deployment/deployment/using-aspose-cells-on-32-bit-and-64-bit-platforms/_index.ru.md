{{% alert color="primary" %}} 

Aspose.Cells — это чистый компонент .NET, который может упростить процесс развертывания с помощью развертывания XCOPY. Чтобы установить Aspose.Cells, вы можете просто скопировать сборку компонента (Aspose.Cells.dll) в каталог вашего приложения: приложение сразу же может начать его использовать. Это возможно из-за самоописываемого характера компонентов .NET. Этот тип развертывания также не влияет на процесс установки.

{{% /alert %}} 
## **Развертывание**
Aspose.Cells поддерживает как 32-разрядные, так и 64-разрядные среды. При установке компонента Aspose.Cells for .NET с помощью установщика MSI Aspose.Cells разные библиотеки DLL добавляются в разные папки в папках Aspose.Cells ${installation_Path}. См. описание в таблице, в какой папке находятся сборки, необходимые для использования с той или иной версией .NET Framework.

|**Папка**|**Описание**|
|:- |:- |
|сеть2.0|Содержит сборки для использования с .NET Framework 2.0, 3.0, 3.5, 4.0 и Mono. Это сборки, которые обычно следует использовать как для 32-разрядных, так и для 64-разрядных сред.|
|net2.0_AuthenticodeSigned|То же, что и выше, но сборки имеют цифровую подпись с помощью Authenticode. Подписанные сборки могут загружаться медленнее, чем без Authenticode.|
|net3.5_ClientProfile|Содержит сборки для использования с клиентским профилем .NET Framework 3.5 или 4.0.|
|нетто3.5_профиль клиента_AuthenticodeSigned|То же, что и выше, но сборки имеют цифровую подпись с помощью Authenticode. Подписанные сборки могут загружаться медленнее, чем без Authenticode.|
|нетто3.5|Содержит сборки для использования с .NET Framework 3.5 или 4.0.|
|net3.5_AuthenticodeSigned|То же, что и выше, но сборки имеют цифровую подпись с помощью Authenticode. Подписанные сборки могут загружаться медленнее, чем без Authenticode.|
|сеть4.0|Содержит сборки для использования с .NET Framework 4.0 и 4.5.|
|чистыйСтандарт|Содержит сборки для использования с .Net Standard 2.0.|
|сетевое приложение2.1|Содержит сборки для использования с .Net core 2.1.|
|Xamarin.iOS|Содержит сборки для использования с Xamarin.iOS.|
|Xamarin.Android|Содержит сборки для использования с Xamarin.Android.|
|сеть5.0|Содержит сборки для использования с .net5.0.|
|сеть6.0|Содержит сборки для использования с .net6.0.|
{{% alert color="primary" %}} 

В проектах VS.NET (например, 2005, 2008, 2010, 2012 и т. д.) при добавлении ссылки на Aspose.Cells диалоговое окно «Добавить ссылку» ссылается на файлы Aspose.Cells.dll в папках net2.0 или net3.5 соответственно. (Для получения дополнительной справки см. Ссылку Aspose.Cells из проекта .NET.) Вы можете изменить ссылку на библиотеку в соответствии с вашей средой. Обратите внимание, что если целевой платформой проекта является клиентский профиль .NET Framework 3.5/4, используйте файл компонента Aspose.Cells.dll, расположенный в папке net_ClientProfile.

{{% /alert %}}
