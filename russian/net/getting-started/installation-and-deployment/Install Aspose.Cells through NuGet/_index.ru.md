## **Установите Aspose.Cells for .NET через NuGet**
NuGet - самый простой способ загрузить и установить API Aspose для .NET. Откройте Microsoft Visual Studio и менеджер пакетов NuGet. Поиском "aspose" найдите нужное API Aspose. Нажмите "Установить", выбранное API будет загружено и добавлено в ваш проект.

Примечание: Вы можете посетить веб-страницу Nuget для aspose.cells для получения дополнительной информации: 
[Пакет NuGet Aspose.Cells for .NET](https://www.nuget.org/packages/Aspose.Cells/)

### **Установка Aspose.Cells с помощью графического интерфейса управления пакетами**
Следуйте этим шагам, чтобы добавить компонент Aspose.Cells с помощью графического интерфейса управления пакетами:

- Откройте ваше решение/проект в Visual Studio.
- Нажмите **Инструменты** -> **Менеджер пакетов библиотек** -> **Управление пакетами NuGet** из Решения. Вы также можете легко получить доступ к той же опции через Обозреватель решений. Щелкните правой кнопкой мыши по решению или проекту и выберите **Управление пакетами NuGet** из контекстного меню.
- Выберите **Онлайн** из левого меню и введите "Aspose.Cells" в поле поиска, чтобы найти пакет Aspose.Cells .NET.
- Нажмите кнопку **Установить** рядом с записью Aspose.Cells for .NET, чтобы установить последнюю версию в ваш проект.
### **Установите Aspose.Cells с помощью Консоли Менеджера Пакетов**
Вы можете следовать нижеприведенным шагам для ссылки на компонент Aspose.Cells с помощью консоли менеджера пакетов:

- Откройте ваше решение/проект в Visual Studio.
- Выберите **Инструменты** -> **Менеджер пакетов библиотек** -> **Консоль менеджера пакетов** из меню, чтобы открыть консоль менеджера пакетов.
  - Введите команду “Install-Package Aspose.Cells” и нажмите Enter, чтобы установить последний полный релиз в ваше приложение. В качестве альтернативы вы можете добавить суффикс "-prerelease" к команде, чтобы указать, что должен быть установлен также последний релиз с исправлениями ошибок.
- Вы увидите, что появится совет "Загрузка Aspose.Cells..." внизу левого окна, указывающий на то, что загрузка происходит.
- После загрузки вы увидите следующие подтверждающие сообщения. Если вы не знакомы с EULA Aspose, то рекомендуется прочитать лицензию, описанную по URL.
- Теперь вы должны увидеть, что Aspose.Cells успешно добавлен и ссылается в вашем приложении.
## **Ссылка на Aspose.Cells из проекта .NET**
Для использования Aspose.Cells в приложении добавьте к нему ссылку. Чтобы добавить ссылку с помощью Visual Studio:

1. В **Обозревателе решений** разверните узел проекта, для которого требуется добавить ссылку.
1. Щелкните правой кнопкой мыши на узле **Ссылки** для проекта и выберите **Добавить ссылку** из меню.
1. В диалоговом окне **Добавить ссылку** выберите вкладку .NET (выбрана по умолчанию). Если вы установили с помощью установщика MSI, то Aspose.Cells появится в верхней панели.
1. Выберите его и нажмите **Выбрать**.

Если вы загрузили или распаковали только DLL:

1. Найдите файл Aspose.Cells.dll с помощью кнопки **Обзор**. Aspose.Cells должен появиться в панели **Выбранные компоненты** диалогового окна.
1. Нажмите **OK**. Ссылка на Aspose.Cells появится под узлом **Ссылки** проекта.
### **Ссылка на компонент из проекта VS.NET 2010 Client Profile**
Если целевая платформа вашего проекта - .NET Framework 3.5/4 Client Profile, используйте файл компонента Aspose.Cells.dll, находящийся в папке net_ClientProfile вашего каталога установки. Например:

- В **Обозревателе решений** VS.NET 2010 для вашего проекта щелкните правой кнопкой мыши **Ссылки** и выберите **Добавить ссылку**.
- Выберите вкладку **Поиск** в диалоговом окне и щелкните на списке **Поиск**.
- Найдите и выберите файл компонента Aspose.Cells.dll в вашем каталоге установки, например: .../Program Files/Aspose/Aspose.Cells for .NET/Bin/net_ClientProfile/ **(Убедитесь, что вы установили продукт, используя установщик MSI на вашем компьютере.)**
- Щелкните **OK**, чтобы закрыть диалоговое окно

{{% alert color="primary" %}} 

Если целевая платформа вашего проекта VS.NET 2010 - ".NET Framework 4" или другая, просто используйте файл компонента Aspose.Cells.dll, расположенный в папке net4.0/net2.0.

## **Ссылка на элементы управления Aspose.Cells Grid из проекта .NET**
Чтобы использовать элемент управления таблицей в вашем приложении, добавьте ссылку на него. Чтобы ссылаться на элемент управления таблицей в Visual Studio, выполните следующее:

- В **Обозревателе решений** разверните узел проекта, к которому вы хотите добавить ссылку.
- Щелкните правой кнопкой мыши узел **Ссылки** для проекта и выберите **Добавить ссылку** в меню.
- В диалоговом окне **Добавить ссылку** выберите **вкладку .NET** (выбрана по умолчанию). Если вы использовали установщик MSI для установки Aspose.Cells for .NET, Aspose.Cells.GridDesktop и Aspose.Cells.GridWeb появляются в верхнем панели.
- Выберите их и щелкните **Выбрать**.
- Ссылки на Aspose.Cells.GridDesktop и Aspose.Cells.GridWeb появляются под узлом Ссылки проекта.

Если вы скачали и распаковали только DLL:

- Найдите файлы Aspose.Cells.GridDesktop.dll и Aspose.Cells.GridWeb.dll, используя кнопку Обзор. Набор компонентов Aspose.Cells Grid должен быть ссылкой и появиться в диалоговом окне **Выбранные компоненты**.
- Щелкните **OK**.
## **Удаление Aspose.Cells for .NET**
Если вы использовали установщик MSI для развертывания Aspose.Cells for .NET, следуйте этим шагам, чтобы полностью удалить компонент и элементы управления, связанные демонстрации и документацию:

- Из меню **Пуск** выберите **Параметры**, а затем **Панель управления**.
- Нажмите **Добавить/Удалить программы**.
- Выберите Aspose.Cells for .NET (версию).
- Нажмите **Изменить/Удалить**, чтобы удалить Aspose.Cells.
