## **Среды разработки**
Вы можете использовать Aspose.Cells for .NET для разработки приложений в любой среде разработки, предназначенной для платформы .NET, но явным образом поддерживаются следующие среды:

- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- Microsoft Visual Studio 2022

{{% alert color="primary" %}} 

Чтобы использовать Aspose.Cells, на машине не нужно устанавливать Microsoft Excel, так как сама библиотека является механизмом управления электронными таблицами. Однако для просмотра файла Excel, созданного компонентом, на вашем компьютере должна быть установлена программа просмотра Excel не ниже Microsoft.

{{% /alert %}} 
## **Aspose.Cells Системные требования Grid Suite**
Aspose.Cells Grid Suite состоит из двух элементов управления:

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb

Системные требования обоих элементов управления описаны отдельно.

{{% alert color="primary" %}} 

Чтобы использовать эти два элемента управления, на машине с запущенными Aspose.Cells.GridDesktop и Aspose.Cells.GridWeb не требуется, чтобы Microsoft Excel был установлен как Aspose.Cells.GridDesktop и Aspose.Cells.GridWeb может открывать и сохранять электронные таблицы Excel без необходимости в Microsoft Excel.

{{% /alert %}}
