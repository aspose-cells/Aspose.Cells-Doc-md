## **Среды разработки**
Вы можете использовать Aspose.Cells for .NET для создания приложений в любой среде разработки, которая ориентирована на платформу .NET, но явно поддерживаются следующие среды:

- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- Microsoft Visual Studio 2022

{{% alert color="primary" %}} 

Для использования Aspose.Cells машина не должна иметь установленный Microsoft Excel, так как библиотека сама является механизмом управления электронными таблицами. Однако, для просмотра файла Excel, сгенерированного компонентом, на вашей машине должен быть установлен по крайней мере просмотрщик Microsoft Excel.

{{% /alert %}} 
## **Требования к системе для Aspose.Cells Grid Suite**
Aspose.Cells Grid Suite состоит из трех элементов управления:

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb
- Aspose.Cells.GridJs

Требования системы для этих элементов управления описаны отдельно.

{{% alert color="primary" %}} 

Для использования этих элементов управления машина для запуска Aspose.Cells.GridDesktop и Aspose.Cells.GridWeb не требует установленного Microsoft Excel, так как Aspose.Cells.GridDesktop и Aspose.Cells.GridWeb могут открывать и сохранять электронные таблицы Excel без необходимости Microsoft Excel.
