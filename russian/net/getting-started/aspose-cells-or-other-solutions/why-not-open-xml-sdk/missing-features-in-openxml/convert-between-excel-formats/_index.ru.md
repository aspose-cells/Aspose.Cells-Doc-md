## **Преобразование Excel в PDF**

**PDF** файлы широко используются для обмена документами между организациями, государственными секторами и частными лицами. Это стандартный формат документов, и разработчиков программного обеспечения часто просят найти способ конвертировать Microsoft файлы Excel в**PDF** документы.
**Aspose.Cells** поддерживает преобразование файлов Excel в формат PDF и обеспечивает высокую визуальную точность преобразования.

Aspose.Cells for .NET поддерживает преобразование из электронных таблиц в PDF независимо от другого программного обеспечения. Сохраните файл Excel по адресу PDF, используя метод Save класса Workbook. Метод Save предоставляет элемент перечисления SaveFormat.Pdf, который преобразует собственные файлы Excel в формат PDF.

**Преобразование** непосредственно из электронной таблицы на PDF, вместо использования стороннего инструмента или внешнего API, имеет несколько**преимущества**:

1. Прямое преобразование не требует временных файлов, поскольку весь процесс может выполняться в памяти.
1. XML-файл не требуется, поэтому большие файлы можно легко преобразовать.
1. Скорость преобразования намного выше.

**Чтобы преобразовать файлы в PDF:**

1.  Создать экземпляр объекта**Рабочая тетрадь** class, вызвав его пустой конструктор.
1.  Вы можете**открыть/загрузить** существующий файл шаблона или пропустите этот шаг, если вы создаете книгу с нуля.
1. Выполните необходимую работу (введите данные, примените форматирование, задайте формулы, вставьте изображения или другие объекты рисования и т. д.) в электронной таблице, используя API-интерфейсы Aspose.Cells'.
1.  Когда код электронной таблицы будет готов, вызовите**Метод сохранения класса рабочей книги** чтобы сохранить электронную таблицу. Формат файла должен быть PDF, поэтому выберите Pdf (предустановленное значение) из перечисления SaveFormat, чтобы сгенерировать окончательный документ PDF.

{{< highlight "csharp" >}}

  Workbook workbook = new Workbook(openFileDialog1.FileName);

  workbook.Save(saveFileDialog1.FileName, SaveFormat.Pdf);

{{< /highlight >}}

## **Преобразование Excel в MHTML**

**MHTML** объединяет обычный HTML с внешними ресурсами (то есть контентом, который обычно связан, например, изображениями, анимацией, звуком и т. д.) в один файл. Они используются для электронных писем с расширением файла .mht.
Aspose.Cells поддерживает чтение и запись файлов MHTML.

{{< highlight "csharp" >}}

  Workbook workbook = new Workbook(openFileDialog1.FileName);

  //Specify the HTML Saving Options

  HtmlSaveOptions sv = new HtmlSaveOptions(SaveFormat.MHtml);

  workbook.Save(saveFileDialog1.FileName, sv);

{{< /highlight >}}

## **Преобразование Excel в XPS**

Иногда вы хотите преобразовать или сохранить книгу с несколькими листами в текстовом формате. Для текстовых форматов (например, TXT, TabDelim, CSV и т. д.) по умолчанию как Microsoft Excel, так и Aspose.Cells сохраняют содержимое только активного рабочего листа.

{{< highlight "csharp" >}}

  Workbook workbook = new Workbook(openFileDialog1.FileName);

 workbook.Save(saveFileDialog1.FileName, SaveFormat.CSV);

{{< /highlight >}}
