Aspose.Cells уникальная функция, обеспечивающая гибкость преобразования версий без ущерба для работы.
SaveFormat — это перечисление, которое может преобразовать документ в расширения, указанные ниже в таблице.

|**Имя члена** |**Ценность** |**Описание** |
|:- |:- |:- |
| CSV|1 | Представляет файл CSV.|
| XLSX|6 | Представляет файл xlsx.|
| XLSM|7 | Представляет файл xlsm, который включает макросы.|
| XLTX|8 | Представляет xltx-файл.|
| XLTM|9 | Представляет файл xltm, который включает макросы.|
| TabDelimited|11 | Представляет текстовый файл с разделителями табуляцией.|
| HTML|12 | Представляет html-файл.|
| МХтмл|17 | Представляет файл mhtml.|
| ОРВ|14 | Представляет файл ods.|
| Excel97To2003|5 | Представляет файл Excel97-2003 xls.|
| Электронная таблицаML|15 | Представляет XML-файл Excel 2003.|
| XLSB|16 | Представляет файл xlsb.|
| Авто|0 |При сохранении файла на диск формат файла соответствует расширению имени файла.<br> При сохранении файла в поток используется формат xlsx.|
| Неизвестный|255 | Представляет нераспознанный формат, не может быть сохранен.|
| PDF|13 | Представляет файл PDF.|
| XPS|20 | Представляет файл XPS.|
| ТИФФ|21 | Представляет файл TIFF.|
| SVG|22 | Представляет файл SVG.|
| Диф|30 | Формат обмена данными.|
Ниже приведен фрагмент кода, показывающий преобразование из xls в xlsx, вы также можете сделать это и наоборот.

{{< highlight "csharp" >}}

 Workbook workbook = new Workbook("Sample.xls");

workbook.Save("Converted.xlsx", SaveFormat.Xlsx);

{{< /highlight >}}
