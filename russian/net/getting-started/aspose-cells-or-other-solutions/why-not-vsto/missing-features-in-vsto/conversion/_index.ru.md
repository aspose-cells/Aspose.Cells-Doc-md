Уникальная функция Aspose.Cells, обеспечивающая гибкость при конвертации версий без влияния на работу.
SaveFormat - это перечисление, которое может преобразовать документ в указанные ниже расширения в таблице.

|**Имя члена** |**Значение** |**Описание** |
| :- | :- | :- |
|CSV |1 |Представляет файл CSV. |
|Xlsx |6 |Представляет файл xlsx. |
|Xlsm |7 |Представляет файл xlsm, который позволяет макросы. |
|Xltx |8 | Представляет файл xltx. |
|Xltm |9 | Представляет файл xltm, который поддерживает макросы. |
|TabDelimited |11 | Представляет файл текста с разделителями табуляции. |
|Html |12 | Представляет файл html. |
|MHtml |17 | Представляет файл mhtml. |
|ODS |14 | Представляет файл ods. |
|Excel97To2003 |5 | Представляет файл Excel97-2003 xls. |
|SpreadsheetML |15 | Представляет файл Excel 2003 xml. |
|Xlsb |16 | Представляет файл xlsb. |
|Auto |0 | Если сохранить файл на диск, формат файла будет соответствовать расширению имени файла. <br> Если сохранить файл в потоке, формат файла будет xlsx. |
|Unknown |255 | Представляет нераспознаваемый формат, который нельзя сохранить. |
|Pdf |13 | Представляет файл Pdf. |
|XPS |20 | Представляет файл XPS. |
|TIFF |21 | Представляет файл TIFF. |
|SVG |22 | Представляет файл SVG. |
|Dif |30 | Формат обмена данными. |
Ниже приведен фрагмент кода, показывающий преобразование из xls в xlsx, вы также можете это сделать в обратном порядке

{{< highlight csharp >}}

 Workbook workbook = new Workbook("Sample.xls");

workbook.Save("Converted.xlsx", SaveFormat.Xlsx);

{{< /highlight >}}
