## **Справочная информация о продукте**

Компонент Report Canvas позволяет пользователю создавать визуальные отчеты на основе предварительно загруженного набора данных. Пользователь может добавлять различные компоненты на свой холст, включая изображения, текстовые поля, диаграммы и таблицы, а затем указывать данные и способ их агрегирования. Затем пользователь может переупорядочивать и изменять размер объектов в соответствии со своей страницей. Пользователь может задавать цветовые палитры и сохранять шаблоны для использования в будущем. Aspose.Cells используется для экспорта всех объектов на холсте в Excel с правильными данными. Компонент написан с помощью VB.Net в Visual Studio 2008.

## **Сценарий требований**

Мы выбрали Aspose.Cells из-за его почти полных возможностей экспорта в Excel. Наиболее важным для нас является возможность экспорта диаграмм и таблиц, особенно в Microsoft Excel 2007 — этого не хватало в других сторонних компонентах.

## **Внедрение решения**

У каждого объекта на холсте отчета есть функция, которой передается экземпляр рабочего листа Aspose.Cells и который добавляет себя на рабочий лист. Когда пользователь запрашивает экспорт, рабочая книга и рабочие листы инициализируются, и каждый объект на холсте отчета вызывает эту функцию.

## **Преимущества**

Aspose.Cells позволил нам создать книгу Excel полностью независимо от Excel, а затем сохранить книгу в формате, выбранном пользователем. Это сэкономило часы отладки взаимодействия при использовании взаимодействия с Excel и реализации различных подпрограмм для сохранения в разных версиях Excel.

## **Будущая реализация**

Скорее всего, мы будем использовать Aspose.Cells для загрузки и сохранения всех файлов Excel. Это будет включать загрузку шаблонов данных и экспорт результатов.
