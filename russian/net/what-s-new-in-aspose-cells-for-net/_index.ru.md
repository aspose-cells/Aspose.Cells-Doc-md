На этой странице описаны самые интересные новые функции Aspose.Cells, представленные в последних выпусках.

##  Aspose.Cells for .NET 23.5

* Поддержка обновления внешних ссылок данных внешнего листа на локальный лист текущей книги.
* Поддержка отображения элемента управления Checkbox как интерактивного элемента управления вместо статического изображения.
* Поддержка выбора PivotArea и ее форматирования.
* Поддержка добавления водяных знаков при конвертации в pdf

##  Aspose.Cells for .NET 23.4

* Поддержка вычисления функции ENCODEURL
* Поддержка завершения построения формы уравнения путем пошаговой вставки соответствующих узлов.
* Поддержка вычисления выражения формулы в виде формулы массива без установки его в объект ячейки
* Поддержка сохранения файла xlsb в режиме LightCells
* Поддержка проверки того, установлен ли или доступен один шрифт
* Поддержка установки формулы в указанную ячейку в ListObject
* Поддержка сохранения книги как xhtml
* Поддержка рендеринга элемента управления ActiveX и элемента управления формы для GridWeb.

##  Aspose.Cells for .NET 23.3

* Поддержка установки/чтения/сохранения новых функций: LET, ENCODEURL
* Поддержка привязки источников данных к внешним ссылкам при вызове расчета формулы

##  Aspose.Cells for .NET 23.2

* Поддержка установки/чтения/сохранения новых функций: SCAN, LAMBDA
* Поддержка экспорта стиля в json

##  Aspose.Cells for .NET 23.1

* Поддержка новых функций: CHOOSEROWS, CHOOSECOLS, HSTACK, VSTACK.
* Поддержка получения автора и даты и времени из редакций

##  Aspose.Cells for .NET 22.12

* Поддержка экспорта Excel в JSON с многоуровневой иерархической структурой

##  Aspose.Cells for .NET 22.11

* Поддержка различия между формулой динамического массива и устаревшей формулой массива.
* Поддержка чтения/сохранения формул динамического массива для формата файла xlsb

##  Aspose.Cells for .NET 22.10

* Поддержка использования SkiaSharp в качестве графической библиотеки для NET6.
* Поддержка добавления и обновления диаграммы с помощью файла шаблона CRTX.

##  Aspose.Cells for .NET 22.9

* Поддержка установки формулы таблицы для ячеек
* Поддержка установки формулы динамического массива с пользовательской функцией и механизмом расчета
* Поддержка установки предустановленных настроек градиента для формата градиентной заливки.
* Поддержка применения границ контура с цветами темы для диапазона

##  Aspose.Cells for .NET 22.8

* Поддержка экспорта комментариев и заметок, отображаемых на листе, в html.
* Поддержка указания рабочих листов для вывода при преобразовании в pdf/xps

##  Aspose.Cells for .NET 22.7

* Поддержка экспорта html в один файл с верхним/нижним колонтитулом страницы.
* Поддержка получения абсолютной позиции и размера диапазона (в пунктах)

##  Aspose.Cells for .NET 22.6

* Улучшение расчета и отслеживания формул с цепочкой расчета формул.

##  Aspose.Cells for .NET 22.5

* Улучшение производительности памяти при настройке формулы ячеек при сохранении книги в режиме LightCells.

##  Aspose.Cells for .NET 22.4

* Расширенные возможности управления пустыми ячейками для экспорта CSV.
* Поддержка сложного разделения и обновления условного форматирования для операций вставки/удаления.

##  Aspose.Cells for .NET 22.3

* Поддержка пересчета ячеек, которые зависят от определенного имени, когда определенное имя изменено и включена цепочка вычислений рабочей книги.
* Поддержка рекурсивного получения листьев ячейки в соответствии с цепочкой вычислений формул

##  Aspose.Cells for .NET 22.2

* Поддержка извлечения определенных имен из NameCollection с помощью заданного фильтра области.

##  Aspose.Cells for .NET 22.1

* Поддержка создания редактируемого pptx
* Поддержка импорта больших CSV-файлов на несколько рабочих листов, когда количество строк/столбцов превышает лимит.
* Поддержка импорта csv с LightCells
* Поддержка удаления указанной внешней ссылки
* Поддержка сохранения формул при удалении внешних ссылок

##  Aspose.Cells for .NET 21.12

* Поддержка преобразования Excel в сценарии SQL.
* Поддержка импорта и экспорта простых XML-данных.
* Поддержка чтения как можно большего количества данных из поврежденного файла xls.
* Поддержка прерывания процесса SmartMarker
* Повышение производительности при добавлении диапазонов для условного форматирования и проверки.
* Установите приоритет как самый высокий для вновь добавленного условия формата, как поведение ms excel

##  Aspose.Cells for .NET 21.11

* Поддержка вычисления формулы/функции XLookup.
* Интегрируйте сохранение json и изображений в сохранение книги.
* Улучшите производительность при преобразовании Excel в HTML.
