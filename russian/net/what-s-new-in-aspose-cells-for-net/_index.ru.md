На этой странице описаны наиболее интересные новые функции Aspose.Cells, представленные в последних выпусках.

##  Aspose.Cells for .NET 23.12

* Поддержка соответствия PDF/A-2 и PDF/A-3 при преобразовании в PDF.
* Поддержка расчета дискретной группировки сводной таблицы.
* Поддержка экспорта данных ячеек в общий список.
* Поддержка импорта перекрестных ссылок с использованием связанных таблиц для интеллектуальных маркеров.
* Поддержка функции MAKEARRAY для формул.
* Поддержка рендеринга текста ячейки с углом поворота для GridJS.
* Поддержка дополнительных параметров импорта/экспорта данных для GridDesktop.
* Поддержка рендеринга изображения с вращением для GridDesktop.
* Поддержка настройки порядка листов для GridDesktop.
*  Поддержка опции отображения/скрытия предупреждающего сообщения о формуле для GridDesktop.

* Поддержка отображения градиентной заливки фона ячейки для GridJS.
* Поддержка копирования/удаления/вставки/переименования рабочего листа для GridDesktop.
* Поддержка автоматического расширения максимального количества строк/столбцов при выполнении операции вставки для GridDesktop.
* Поддержка SheetPrintingPreview для GridDesktop
*  Предоставьте .NET Базовую сборку GridDesktop.
