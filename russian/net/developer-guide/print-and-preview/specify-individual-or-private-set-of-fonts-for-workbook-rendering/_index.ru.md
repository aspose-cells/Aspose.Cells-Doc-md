## **Возможные сценарии использования**

Как правило, вы указываете каталог шрифтов или список шрифтов для всех книг, но иногда вам нужно указать индивидуальный или конкретный набор шрифтов для ваших книг. Aspose.Cells предоставляет класс [**IndividualFontConfigs**](https://reference.aspose.com/cells/net/aspose.cells/individualfontconfigs), который можно использовать для указания индивидуального или конкретного набора шрифтов для вашей книги.
