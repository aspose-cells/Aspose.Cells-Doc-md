{{% alert color="primary" %}} 

Иногда вы хотите указать формат пикселей при рендеринге рабочего листа в формат изображения. По умолчанию Aspose.Cells использует 32 бита на пиксель. Aspose.Cells позволяет настраивать формат пикселей (разрядность) с помощью параметров визуализируемого изображения.

{{% /alert %}}
