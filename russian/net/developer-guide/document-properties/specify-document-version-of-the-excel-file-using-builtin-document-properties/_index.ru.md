## **Возможные сценарии использования**

 Вы можете изменить**Номер версии** файла Excel, щелкнув файл правой кнопкой мыши и выбрав Свойства > Сведения, а затем отредактировав**Номер версии** поле. Пожалуйста, используйте[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion)свойство, чтобы изменить его программно с помощью API Aspose.Cells.

## **Укажите версию документа файла Excel с помощью встроенных свойств документа**

 Следующий пример кода создает книгу и изменяет ее встроенные свойства документа, в том числе название, авторов и номер версии.
