## **Возможные сценарии использования**

Вы можете изменить **Номер версии** файла Excel, щелкнув правой кнопкой мыши на файле, затем выбрав Свойства > Детали и затем редактировать поле **Номер версии**. Пожалуйстa, используйте свойство [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion), чтобы изменить его программно с использованием API Aspose.Cells.

## **Указание версии документа Excel с использованием встроенных свойств документа**

В следующем образце кода создается книга и изменяются ее встроенные свойства документа, которые включают в себя название, авторов и номер версии. Пожалуйстa, посмотрите [выходной файл Excel](64716811.xlsx), сгенерированный кодом, и снимок экрана, который показывает измененный номер версии с использованием свойства [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion).
