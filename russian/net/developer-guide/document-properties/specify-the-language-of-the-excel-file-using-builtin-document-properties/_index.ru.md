## **Возможные сценарии использования**

Вы можете изменить язык файла Excel, щелкнув файл правой кнопкой мыши и выбрав «Свойства» > «Подробности», а затем отредактировав поле «Язык». Пожалуйста, используйте[**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/language)свойство, чтобы изменить его программно с помощью API Aspose.Cells.

 Следующий пример кода создает книгу и изменяет ее встроенное свойство документа с именем Language.
