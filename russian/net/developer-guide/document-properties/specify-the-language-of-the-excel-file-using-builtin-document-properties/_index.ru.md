## **Возможные сценарии использования**

Вы можете изменить язык файла Excel, нажав правой кнопкой мыши на файле, затем выбрав Свойства > Детали и затем редактировать поле язык. Пожалуйстa, используйте свойство [**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/language), чтобы изменить его программно с использованием API Aspose.Cells.

## **Указание языка файла Excel с использованием встроенных свойств документа**

В следующем образце кода создается книга и изменяется ее встроенное свойство документа с именем Language. Пожалуйста, посмотрите [файл Excel вывода](64716891.xlsx), сгенерированный кодом, и скриншот, который показывает измененное поле Language по свойству [**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/language).
