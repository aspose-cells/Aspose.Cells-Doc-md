{{% alert color="primary" %}}

 Если вы хотите установить значение «Нет» для заполнения текста легенды диаграммы, чтобы он не отображался внутри легенды диаграммы, установите параметр[**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/net/aspose.cells.charts/legendentry/properties/istextnofill) на *истину**.
