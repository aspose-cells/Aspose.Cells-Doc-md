{{% alert color="primary" %}}

 Если вы хотите, чтобы текст записи легенды диаграммы не заполнялся, чтобы он не отображался внутри легенды диаграммы, установите[**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/net/aspose.cells.charts/legendentry/properties/istextnofill) к**истинный**.

{{% /alert %}}

 Следующий пример кода задает для текста второй записи легенды диаграммы значение none. Пожалуйста, загрузите[образец эксель файла](5115219.xlsx) используется в этом коде и[выходной файл excel](5115217.xlsx) созданный им для вашей справки.
