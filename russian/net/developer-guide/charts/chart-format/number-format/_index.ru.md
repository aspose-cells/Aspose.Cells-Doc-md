## **Возможные сценарии использования**
Вы можете установить код формата значений серии диаграммы с помощью[Серия.ValuesFormatCode](https://reference.aspose.com/cells/net/aspose.cells.charts/series/properties/valuesformatcode)имущество. Это свойство полезно не только для ряда, основанного на диапазоне внутри листа, но также хорошо работает для ряда, созданного с помощью массива значений.
## **Установите код формата значений серии диаграммы**
Следующий пример кода добавляет серию в пустую диаграмму, в которой раньше не было серий. Он добавляет серию, используя массив значений. После добавления серии форматирует ее кодом $#,##0 с помощью[Серия.ValuesFormatCode](https://reference.aspose.com/cells/net/aspose.cells.charts/series/properties/valuesformatcode)недвижимость, а число 10000 становится 10000 долларов. На снимке экрана показано влияние кода на[образец файла Excel](51740712.xlsx) а также[выходной файл Excel](51740713.xlsx) после казни.
