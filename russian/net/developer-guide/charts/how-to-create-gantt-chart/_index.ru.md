## Что такое диаграмма Ганта

Диаграмма Ганта помогает вам планировать задачи проекта, а затем отслеживать ваш прогресс.

## Добавить диаграмму Ганта в Excel

Необходимо показать статус для простого графика проекта с помощью диаграммы Ганта? Хотя в Excel нет предопределенного типа диаграммы Ганта, вы можете смоделировать ее, настраивая столбчатую диаграмму для отображения дат начала и завершения задач, например, так:

![todo:image_alt_text](0.png)

## Как создать

- Выберите данные, которые вы хотите построить. В нашем примере это ячейки B1:B7, а затем вставьте диаграмму **столбчатую**.

![todo:image_alt_text](1.png)

- Выберите график, **Выбрать данные** -> **Добавить**, установите **Название ряда** и **Значения ряда** в соответствии с нижеуказанным

![todo:image_alt_text](2.png)

- Выберите диаграмму, отредактируйте **Горизонтальные оси (категории)**

![todo:image_alt_text](3.png)

- **Формат оси**, выберите ось Y, выберите **Категории в обратном порядке**
- Выберите **Синюю серию** и установите **Заливка->Без заливки**
- **Формат оси**, выберите ось X, установите **Минимум и Максимум** (1/5/2019:43470, 1/30/2019:43494)

![todo:image_alt_text](4.png)

- **Добавьте метки данных** к диаграмме
Теперь у вас есть диаграмма Ганта.
