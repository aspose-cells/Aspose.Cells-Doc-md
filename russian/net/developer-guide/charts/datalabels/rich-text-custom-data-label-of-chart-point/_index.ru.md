{{% alert color="primary" %}}

 Вы можете использовать Aspose.Cells для создания пользовательской метки данных с форматированным текстом для точки диаграммы. Aspose.Cells обеспечивает[**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) способ вернуть[**Настройка шрифта**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) объект, который можно использовать для установки свойств шрифта текста, таких как его цвет, жирность и т. д.

{{% /alert %}}
