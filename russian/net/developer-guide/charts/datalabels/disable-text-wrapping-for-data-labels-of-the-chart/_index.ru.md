{{% alert color="primary" %}}

Microsoft Excel 2013 позволяет пользователям переносить или разворачивать текст внутри меток данных диаграммы. По умолчанию текст внутри меток данных диаграммы находится в обернутом состоянии.

 Aspose.Cells предоставляет[**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) свойство, для которого можно установить значение True или False для включения или отключения переноса текста меток данных соответственно.

{{% /alert %}}
