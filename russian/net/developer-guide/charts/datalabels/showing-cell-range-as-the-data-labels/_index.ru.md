В Microsoft Excel 2013 вы можете отобразить диапазон ячеек для меток данных. Aspose.Cells поддерживает эту функцию.

{{% /alert %}}

## **Флажок для отображения диапазона Cell в виде меток данных**

Чтобы отобразить диапазон ячеек в виде меток данных в Microsoft Excel:

1. Выберите метки данных серии и щелкните правой кнопкой мыши, чтобы открыть контекстное меню.
1.  Выбирать**Форматировать метки данных**. Отобразятся параметры метки.
1.  Выберите или снимите флажок**Этикетка содержит — значение от Cells**.

 Приведенный ниже пример кода обращается к меткам данных ряда диаграммы и устанавливает[**DataLabels.ShowCellRange**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/showcellrange) собственность на**истинный** выбрать**Этикетка содержит — значение от Cells** вариант.
