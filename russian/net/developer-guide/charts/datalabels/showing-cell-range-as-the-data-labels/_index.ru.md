{{% alert color="primary" %}}

В Microsoft Excel 2013 вы можете отображать диапазон ячеек для меток данных. Aspose.Cells поддерживает эту функцию.

{{% /alert %}}

##  **Установите флажок для отображения диапазона Cell в виде меток данных.**

Чтобы отобразить диапазон ячеек в виде меток данных в Excel Microsoft:

1. Выберите метки данных серии и щелкните правой кнопкой мыши, чтобы открыть контекстное меню.
1. Выберите *Форматировать метки данных**. Отобразятся параметры метки.
1. Установите или снимите флажок *Ярлык содержит значение от Cells**.
