В этой теме мы покажем вам, как установить китайский регион для диаграммы.

{{% /alert %}}

##  **Определяет класс наследования**

 На первом этапе вам необходимо определить класс «ChartChineseSettings», который наследуется от[**Настройки глобализации диаграммы**](https://reference.aspose.com/cells/net/aspose.cells.charts/chartglobalizationsettings/). 
Затем, переписав связанные функции, вы сможете задать текст элементов диаграммы на своем родном языке.
Пример кода:
{{< gist "aspose-cells-gists" "59a1901d62ea9ceb08456a818431a898" "ChartChineseSetttings.cs" >}}

##  **Настройка китайского языка для диаграммы**

На этом этапе вы будете использовать класс «ChartChineseSettings», определенный на предыдущем шаге.
Пример кода:

```
	Workbook wb = new Workbook("Chinese.xls");
	wb.Settings.GlobalizationSettings.ChartSettings = new ChartChineseSetttings();
	Chart chart0 = wb.Worksheets[0].Charts[0];
	chart0.ToImage("Output.png");
```

Затем вы сможете увидеть эффект на выходном изображении, элементы диаграммы будут отображаться в соответствии с вашими настройками.

##  **Заключение**

В этом примере, если для диаграммы не задан китайский регион, следующие элементы диаграммы могут отображаться на языке по умолчанию, например английском.
После описанной выше операции мы можем получить выходную диаграмму с китайским регионом.
