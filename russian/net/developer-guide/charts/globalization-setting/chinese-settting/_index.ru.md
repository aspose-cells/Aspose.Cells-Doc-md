В этом разделе мы покажем вам, как установить китайский регион для диаграммы.

{{% /alert %}}

##  **Определяет класс наследования**

Первый шаг, вам нужно определить класс «ChartChineseSettings», который наследуется от[**ChartGlobalizationSettings**](https://reference.aspose.com/cells/net/aspose.cells.charts/chartglobalizationsettings/). 
Затем, переписав соответствующие функции, вы сможете задать текст элементов диаграммы на своем родном языке.
Пример кода:
{{< gist "aspose-cells-gists" "59a1901d62ea9ceb08456a818431a898" "ChartChineseSetttings.cs" >}}

##  **Настройка китайского языка для графика**

На этом шаге вы будете использовать класс ChartChineseSettings, определенный на предыдущем шаге.
Пример кода:

```
	Workbook wb = new Workbook("Chinese.xls");
	wb.Settings.GlobalizationSettings.ChartSettings = new ChartChineseSetttings();
	Chart chart0 = wb.Worksheets[0].Charts[0];
	chart0.ToImage("Output.png");
```

Затем вы можете увидеть эффект на выходном изображении, элементы на диаграмме будут отображаться в соответствии с вашими настройками.

##  **Заключение**

В этом примере, если для диаграммы не задан китайский регион, следующие элементы диаграммы могут отображаться на языке по умолчанию, например на английском.
После вышеуказанной операции мы можем получить выходную диаграмму с китайским регионом.

|**Поддерживаемые элементы**|**Значение в этом примере**|**значение по умолчанию в английской среде**|
| :- | :- | :- |

|Название серии|系列|Ряд|
