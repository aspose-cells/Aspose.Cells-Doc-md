## **Возможные сценарии использования**
 Иногда вам нужно, чтобы формула в вашей таблице или объекте списка автоматически распространялась на новые строки при вводе новых данных. Это поведение по умолчанию для Microsoft Excel. Чтобы добиться того же с Aspose.Cells, используйте[СписокКолонка.Формула](https://reference.aspose.com/cells/net/aspose.cells.tables/listcolumn/properties/formula)имущество.
