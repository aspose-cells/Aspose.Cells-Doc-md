{{% alert color="primary" %}} 

 Aspose.Cells поддерживает большинство формул/функций, входящих в состав Microsoft Excel. Разработчики могут использовать эти формулы с номером API или[дизайнерские таблицы](/cells/ru/net/what-is-a-designer-spreadsheet/). Aspose.Cells поддерживает огромный набор математических, строковых, логических, дат/времени, статистических, баз данных, поисковых и справочных формул. Формулы должны быть указаны в английском (США) стиле.

{{% /alert %}} 
## **Уведомление для пользователей, не владеющих английским языком**
Есть два совета, которым должны следовать пользователи, не владеющие английским языком, при создании формул с Aspose.Cells:

1. Формулы необходимо вводить на английском языке. Например, используйте английское «=СУММ()», а не немецкое «=СУМММ()».
1. Всегда используйте запятую (,) для разделения параметров функции. Для некоторых языковых опций или настроек разделителем функциональных параметров является точка с запятой, но Aspose.Cells использует запятую в английском стиле. Например, используйте «=ЕСЛИ (C5=0,0,C3/C4)», а не «=ЕСЛИ(C5=0;0;C3/C4)».
