{{% alert color="primary" %}} 

Aspose.Cells поддерживает большинство формул/функций, которые являются частью Microsoft Excel. Разработчики могут использовать эти формулы с помощью API или дизайнерских электронных таблиц. Aspose.Cells поддерживает большой набор математических, строковых, логических, даты/времени, статистических, базы данных, поисковых и ссылочных формул. Формулы должны быть указаны с использованием английского (US) стиля.

{{% /alert %}} 
## **Примечание для пользователей, не говорящих на английском языке**
Есть два совета, которые не англоговорящим пользователям нужно соблюдать при создании формул с помощью Aspose.Cells:

1. Формулы должны вводиться на английском языке. Например, используйте английское "=SUM()", а не немецкое "=SUMME()".
1. Всегда используйте запятую (,) для разделения параметров функции. Для некоторых языковых опций или настроек разделителем для параметров функции является точка с запятой, но Aspose.Cells использует английский стиль с запятой. Например, используйте "=IF (C5=0,0,C3/C4)", а не "=IF(C5=0;0;C3/C4)"
