## **Открытие CSV-файлов**

Файлы со значениями, разделенными запятыми (CSV), содержат записи, в которых значения разделены запятыми. Данные хранятся в виде таблицы, в которой каждый столбец отделяется запятой и заключен в двойные кавычки. Если значение поля содержит символ двойной кавычки, оно экранируется парой символов двойной кавычки. Вы также можете использовать Microsoft Excel для экспорта данных электронной таблицы в CSV.

## **Использование предпочтительного парсера**

Это не всегда необходимо, чтобы использовать настройки парсера по умолчанию для открытия файлов CSV. Иногда импорт CSV-файла не приводит к ожидаемому результату, например формат даты не соответствует ожидаемому или пустые поля обрабатываются по-другому. Для этого**TxtLoadOptions.PreferredParsers**доступен для предоставления собственного предпочтительного синтаксического анализатора для анализа различных типов данных в соответствии с требованиями. Следующий пример кода демонстрирует использование предпочтительного синтаксического анализатора.
