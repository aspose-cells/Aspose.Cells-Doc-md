Логические структурные возможности PDF предоставляют механизм для включения информации о структуре содержимого документа в файл PDF. Aspose.Cells сохраняет информацию о структуре из документа Microsoft Excel, такую ​​как ячейка, строка, таблица, лист, изображение, форма, колонтитул/подвал и т. д.

С помощью опции [PdfSaveOptions.ExportDocumentStructure](https://reference.aspose.com/cells/net/aspose.cells/pdfsaveoptions/exportdocumentstructure/) можно сохранить в размеченном PDF с экспортированной структурой документа.
