{{% alert color="primary" %}}

Aspose.Cells поддерживает сохранение файла ODS (**Электронная таблица OpenDocument**) в ФРО (**Формат OpenDocument** спецификации 1.1 и 1.2. Aspose.Cells есть[**Одсавеоптионс.IsStrictSchema11**](https://reference.aspose.com/cells/net/aspose.cells/odssaveoptions/properties/isstrictschema11) свойство, указывающее использование спецификации ODF 1.1 для сохранения файлов ODS. Значение по умолчанию для этого свойства**ЛОЖЬ**, поэтому файл ODS, сохраненный без этого параметра, использует спецификации 1.2.

{{% /alert %}}

В приведенном ниже примере кода создается объект рабочей книги, добавляется некоторое значение в ячейку A1 на первом рабочем листе, а затем сохраняется файл ODS в спецификациях ODF 1.1 и 1.2. По умолчанию файл ODS сохраняется в спецификации ODF 1.2.
