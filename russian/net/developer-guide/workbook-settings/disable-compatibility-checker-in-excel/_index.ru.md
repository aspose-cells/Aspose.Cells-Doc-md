##  Отключить проверку совместимости в таблицах Excel по номеру C#.

{{% alert color="primary" %}}

Microsoft Флаги средства проверки совместимости Excel при сохранении файла в более раннем формате могут вызвать проблемы с функциональностью или потерю точности. Средство проверки совместимости — это функция Microsoft Office Excel 2007 и Microsoft Excel 2010.

Когда вы сохраняете книгу в предыдущей версии (от Excel 97 до Excel 2003), из Excel 2007 или Excel 2010, средство проверки совместимости сканирует книгу, чтобы определить, содержит ли она функции, не поддерживаемые более ранней версией. Чтобы помочь вам принять решение о том, как решать проблемы совместимости, средство проверки совместимости отображает диалоговые окна с параметрами. Его также можно использовать для создания отчета о любых проблемах в книге или отключения этой функции.

Иногда вам необходимо отключить проверку совместимости для определенной таблицы. С помощью API-интерфейсов Aspose.Cells вы можете сделать это программно, чтобы пользователи не расстраивались и не путались из-за появления диалогового окна «Проверка совместимости», когда они повторно сохраняют файл в Excel Microsoft вручную.

{{% /alert %}}

##  **Как отключить проверку совместимости с помощью Microsoft Excel**

Чтобы отключить проверку совместимости в Excel Microsoft (например, Microsoft Excel 2007/2010):

-  (Excel 2007) На кнопке Office нажмите**Подготовьте**, затем **Запустите проверку совместимости**, а затем снимите флажок **Проверять совместимость при сохранении этой книги.** вариант.
-  (Excel 2010) На вкладке Файл нажмите кнопку**Информация**, затем **Проверить наличие проблем**, нажмите **Проверить совместимость** и, наконец, снимите флажок **Проверять совместимость при сохранении этой книги.** вариант.

##  **Как отключить проверку совместимости с помощью API Aspose.Cells**

 Установить[**Workbook.Settings.CheckComptilibility**](https://reference.aspose.com/cells/net/aspose.cells/workbooksettings/properties/checkcompatibility) собственность**ЛОЖЬ** чтобы отключить Microsoft Средство проверки совместимости Excel.
