{{% alert color="primary" %}}

 Часто вам нужно знать номер версии приложения, создавшего документ Excel Microsoft. Aspose.Cells обеспечивает[**Workbook.BuiltInDocumentProperties.Version**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/version) имущество для этой цели.

{{% /alert %}}

 Следующий пример кода демонстрирует использование[**Workbook.BuiltInDocumentProperties.Version**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/version)имущество. Он загружает файлы Excel, созданные с помощью Microsoft Excel 2003, 2007, 2010 и 2013, и печатает номер версии приложения, создавшего эти документы Excel.
