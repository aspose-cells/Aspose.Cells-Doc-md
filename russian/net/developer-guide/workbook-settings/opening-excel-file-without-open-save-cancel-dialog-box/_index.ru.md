{{% alert color="primary" %}} 

В этом документе объясняется, как открыть файл Excel Microsoft в браузере без отображения диалогового окна «Открыть-Сохранить-Отмена».

 Здесь следует отметить, что ограничение безопасности, запрещающее прямую загрузку файла, применяется Microsoft (или другими поставщиками браузеров), а не Aspose. Оно налагается для блокировки и ограничения загрузки потенциально опасных файлов на локальные машины. .

Для локальной системы клиента рискованно разрешать загрузку без отображения диалогового окна «Открыть-Сохранить-Отмена» с запросом на загрузку. По телефону Aspose нет доступных вариантов или обходных путей, так как это будет очень большой риск для безопасности.

{{% /alert %}} 
## **Почему угроза безопасности?**
На следующем изображении показано диалоговое окно Open-Save-Cancel, отображаемое Internet Explorer при попытке загрузить файл.

Как объяснялось выше, разрешение открывать или запускать файл в вашей системе без подтверждения того, что вы действительно этого хотите, представляет собой угрозу безопасности. Некоторые файлы содержат вирусы, а некоторые сайты попытаются загрузить вредоносные файлы на ваш компьютер без запроса. Поэтому не рекомендуется разрешать загрузку файла без запроса на загрузку, чтобы пользователи могли проверить файл и его источник перед загрузкой или запуском. Отключение диалогового окна загрузки делает систему уязвимой для вирусов, троянов и хакеров, которые могут незаметно повлиять на вашу систему.
## **Открытие файла без диалогового окна «Открыть-Сохранить-Отмена»**
 Несмотря на то, что это большая проблема безопасности, Microsoft по-прежнему предоставляет настройки Internet Explorer, которые позволяют пользователям отключать запрос Open-Save-Cancel для загрузки файла.

В проводнике Windows:

1.  На**Инструменты** меню, выберите**Опции папки**.
1. Перейдите на вкладку «Типы файлов» в диалоговом окне «Параметры папки».
1. Выберите тип файла с расширением XLS.
1.  Нажмите**Передовой**. 
Отображается диалоговое окно. Внизу есть три варианта.
1.  Снимите флажок**Подтвердить открытие после загрузки**.
1.  Выберите третий вариант -**Просмотр в том же окне** - для просмотра файла Excel в Internet Explorer без запуска Microsoft Excel в автономном режиме.

Этот параметр позволяет запускать файлы непосредственно в веб-браузере без отображения диалогового окна «Открыть-Сохранить-Отмена» при загрузке или открытии файла.
