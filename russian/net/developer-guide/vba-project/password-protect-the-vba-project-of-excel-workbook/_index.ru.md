## **Защита паролем проекта VBA книги Excel в C#**

Вы можете защитить паролем проект рабочей книги VBA (Visual Basic для приложений) с помощью Aspose.Cells, используя[**VbaProject.Protect()**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/methods/protect)метод.

## **Образец кода**

 Следующий пример кода загружает[образец файла Excel](43352067.xlsm), получает доступ к своему проекту VBA и защищает его паролем. Наконец, он сохраняет его как[выходной файл Excel](43352068.xlsm).
