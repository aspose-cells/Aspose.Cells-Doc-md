{{% alert color="primary" %}}

 Вы можете проверить, подписан ли ваш проект VBA или нет, используя Microsoft Excel через**Инструменты > Цифровые подписи...** команда меню. Точно так же вы можете проверить это программно, используя Aspose.Cells.[**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned) имущество.

{{% /alert %}}
