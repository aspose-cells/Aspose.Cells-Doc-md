## **Вырезать и вставить ячейки**

Aspose.Cells предоставляет возможность вырезать и вставить ячейки в пределах рабочего листа с помощью метода [**InsertCutCells**](https://reference.aspose.com/cells/net/aspose.cells/cells/methods/insertcutcells) коллекции [**Cells**](https://reference.aspose.com/cells/net/aspose.cells/cells). Параметры [**InsertCutCells**](https://reference.aspose.com/cells/net/aspose.cells/cells/methods/insertcutcells) принимают следующие параметры.

- [**Range**](https://reference.aspose.com/cells/net/aspose.cells/range): Диапазон ячеек для вырезания.
- Индекс строки: Индекс строки для вставки ячеек.
- Индекс столбца: Индекс столбца для вставки ячеек.
- [**ShiftType**](https://reference.aspose.com/cells/net/aspose.cells/shifttype): Направление сдвига столбцов.
