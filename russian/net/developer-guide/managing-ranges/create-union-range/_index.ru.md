## **Создать объединенный диапазон**
Aspose.Cells предоставляет возможность создания объединенного диапазона с использованием метода [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/net/aspose.cells/worksheetcollection/methods/createunionrange). Метод [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/net/aspose.cells/worksheetcollection/methods/createunionrange) принимает два параметра: адрес для создания объединенного диапазона и индекс рабочего листа. Метод [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/net/aspose.cells/worksheetcollection/methods/createunionrange) возвращает объект [UnionRange](https://reference.aspose.com/cells/net/aspose.cells/unionrange).

В следующем фрагменте кода демонстрируется создание объединенного диапазона с использованием метода [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/net/aspose.cells/worksheetcollection/methods/createunionrange). Сгенерированный код файла вложен в качестве ссылки.
