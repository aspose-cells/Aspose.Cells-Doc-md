##  **Введение**
Если в файлах Excel слишком много определенных имен или именованных диапазонов, нам придется очистить некоторые из них, поскольку они больше не будут использоваться.

##  **Удалить именованный диапазон в MS Excel**

Чтобы удалить именованный диапазон из Excel, вы можете выполнить следующие действия:
1. Откройте Microsoft Excel и откройте книгу, содержащую именованный диапазон.
2. Перейдите на вкладку «Формулы» на ленте Excel.
3. Нажмите кнопку «Диспетчер имен» в группе «Определенные имена». Откроется диалоговое окно «Диспетчер имен».
4. В диалоговом окне «Диспетчер имен» выберите именованный диапазон, который хотите удалить.
5. Нажмите кнопку «Удалить». Подтвердите удаление при появлении соответствующего запроса.
6. Нажмите кнопку «Закрыть», чтобы закрыть диалоговое окно «Диспетчер имен».
7. Сохраните книгу, чтобы сохранить изменения.
