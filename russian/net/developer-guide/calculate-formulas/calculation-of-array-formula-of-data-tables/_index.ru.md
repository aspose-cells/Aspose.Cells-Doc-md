{{% alert color="primary" %}}

 Вы можете создать таблицу данных в Microsoft Excel, используя Данные > Анализ «что, если» > Таблица данных.... Aspose.Cells теперь позволяет вычислить формулу массива таблицы данных. Пожалуйста, используйте[**Рабочая книга.ВычислитьФормулу()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/calculateformula) как обычно для расчета любых типов формул.

{{% /alert %}}
