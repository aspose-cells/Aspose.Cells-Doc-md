##  **Возможные сценарии использования**

Microsoft Окно просмотра Excel — это полезный инструмент для удобного просмотра значений ячеек и их формул в окне. Вы можете открыть*Окно просмотра* используя Microsoft Excel, нажав кнопку*Формулы > Смотреть* *Окно*. Он имеет *Добавить часы*кнопка, которую можно использовать для добавления ячеек для проверки. Аналогичным образом вы можете использовать[**Рабочий лист.CellWatches.Add()**](https://reference.aspose.com/cells/net/aspose.cells/cellwatchcollection/methods/add/index) метод добавления ячеек в*Окно просмотра*используя Aspose.Cells API.
