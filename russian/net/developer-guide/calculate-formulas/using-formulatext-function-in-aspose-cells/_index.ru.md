{{% alert color="primary" %}} 

FormulaText — это функция Excel 2013 и более поздних версий. Он не поддерживается предыдущими версиями, такими как Excel 2010, 2007 и т. д. Как следует из названия, он печатает текст формулы, которая присутствует в данной ячейке. Эта статья покажет вам, как использовать эту функцию с помощью Aspose.Cells.

{{% /alert %}}
