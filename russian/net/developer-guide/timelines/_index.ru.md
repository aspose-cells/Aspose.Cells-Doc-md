## **Возможные сценарии использования**

Вместо того чтобы настраивать фильтры для отображения дат, вы можете использовать временную линию сводной таблицы - это динамический вариант фильтра, который позволяет легко фильтровать по дате/времени и масштабировать период, который вам нужен, с помощью элемента управления ползунком. Microsoft Excel позволяет создавать временную линию, выбирая сводную таблицу, а затем нажимая *Вставка > Временная линия*. Aspose.Cells также позволяет создавать временную линию с помощью метода [**Worksheet.Timelines.Add()**](https://reference.aspose.com/cells/net/aspose.cells.timelines/timelinecollection/methods/index).
