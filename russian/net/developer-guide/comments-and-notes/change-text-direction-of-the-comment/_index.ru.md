{{% alert color="primary" %}}

Microsoft Excel позволяет пользователям добавлять комментарии к ячейкам, чтобы добавить дополнительную информацию и выделить данные. Разработчикам может потребоваться настроить комментарий, чтобы указать параметры выравнивания и направление текста. Aspose.Cells предоставляет API для выполнения этой задачи.

{{% /alert %}}

 Aspose.Cells предоставляет[**Форма.TextDirection**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textdirection)свойство для установки направления текста для комментария. Следующий пример кода демонстрирует использование[**Форма.TextDirection**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textdirection)свойство для установки направления текста для комментария.
