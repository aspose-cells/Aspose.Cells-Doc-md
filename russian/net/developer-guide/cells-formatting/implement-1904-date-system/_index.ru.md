{{% alert color="primary" %}} 

Microsoft Excel поддерживает две системы дат: систему дат 1900 года (система дат по умолчанию, реализованная в Excel для Windows) и систему дат 1904 года. Система дат 1904 года обычно используется для обеспечения совместимости с файлами Macintosh Excel и является системой по умолчанию, если вы используете Excel для Macintosh. Вы можете установить систему дат 1904 года для файлов Excel, используя Aspose.Cells.

{{% /alert %}} 

Чтобы внедрить систему дат 1904 года в Microsoft Excel (например, Microsoft Excel 2003):

1.  От**Инструменты** меню, выберите**Параметры** и выберите**Расчет** вкладка
1.  Выберите**Система дат 1904 г.** вариант.
1.  Нажмите**ХОРОШО**.
