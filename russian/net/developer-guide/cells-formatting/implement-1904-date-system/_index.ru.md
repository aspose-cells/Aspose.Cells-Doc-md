{{% alert color="primary" %}} 

Microsoft Excel поддерживает две системы дат: систему дат 1900 года (система дат по умолчанию, реализованная в Excel для Windows) и систему дат 1904 года. Система дат 1904 обычно используется для обеспечения совместимости с файлами Excel для Macintosh и является системой по умолчанию, если вы используете Excel для Macintosh. Вы можете установить систему дат 1904 года для файлов Excel, используя Aspose.Cells.

{{% /alert %}} 

Чтобы реализовать систему дат 1904 года в Excel Microsoft (например, Microsoft Excel 2003):

1.  Из**Инструменты** меню, выберите**Параметры** и выберите **Расчет.** вкладка.
1.  Выберите**Система дат 1904 года** вариант.
1. Нажмите *ОК**.
