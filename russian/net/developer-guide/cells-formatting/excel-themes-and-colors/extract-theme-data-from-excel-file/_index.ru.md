Aspose.Cells позволяет пользователям извлекать данные, связанные с темой, из файла Excel. Например, вы можете извлечь имя темы, примененное к книге, и цвет темы, примененный к ячейке или границам ячейки и т. д.

Вы можете применить тему к своей книге с помощью Microsoft Excel с помощью команды «Макет страницы» > «Темы».

{{% /alert %}}
