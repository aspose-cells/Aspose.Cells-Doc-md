{{% alert color="primary" %}}

Aspose.Cells позволяет загружать легковесную версию книги в память для редактирования информации о метаданных. Используйте класс [**WorkbookMetadata**](https://reference.aspose.com/cells/net/aspose.cells.metadata/workbookmetadata) для загрузки книги.

{{% /alert %}}

После открытия книги с помощью класса [**Workbook**](https://reference.aspose.com/cells/net/aspose.cells/workbook), вы сможете читать свойства документа.
