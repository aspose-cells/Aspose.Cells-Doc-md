## **Введение**

Aspose.Cells поддерживает подписание книг с помощью подписи XAdES. Для этого API предоставляет класс [**DigitalSignature**](https://reference.aspose.com/cells/net/aspose.cells.digitalsignatures/digitalsignature) и перечисление [**XAdESType**](https://reference.aspose.com/cells/net/aspose.cells.digitalsignatures/xadestype).
