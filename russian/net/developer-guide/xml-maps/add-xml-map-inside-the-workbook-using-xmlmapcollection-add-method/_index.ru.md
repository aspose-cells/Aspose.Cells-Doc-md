## **Возможные сценарии использования**

 Aspose.Cells предоставляет[**XmlMapCollection.Добавить()**](https://reference.aspose.com/cells/net/aspose.cells/xmlmapcollection/methods/add) метод, который вы можете использовать для импорта вашей XML-карты в книгу.

## **Добавьте XML-карту в книгу с помощью метода XmlMapCollection.Add.**

 Следующий пример кода добавляет XML-карту в книгу с помощью[**XmlMapCollection.Добавить()**](https://reference.aspose.com/cells/net/aspose.cells/xmlmapcollection/methods/add) метод и сохраняет его как[выходной файл excel](5115434.xlsx) . На снимке экрана показана XML-карта, импортированная из[образец.xml](5115433.xml) внутри выходного файла Excel.
