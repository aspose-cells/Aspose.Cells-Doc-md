Microsoft Excel позволяет копировать спарклайн, указав диапазон данных и расположение группы спарклайнов. Aspose.Cells поддерживает эту функцию.

{{% /alert %}}

Чтобы скопировать спарклайн в другие ячейки в Microsoft Excel:

1. Выберите ячейку, содержащую спарклайн.
1.  Выбирать**Изменить данные** от**Спарклайн** раздел**Дизайн** вкладка
1.  Выбирать**Изменить местоположение и данные группы**.
1. Укажите диапазон данных и местоположение.
1.  Нажмите**ХОРОШО**.

Aspose.Cells предоставляет метод SparklineCollection.Add(dataRange, row, column) для указания диапазона данных и расположения группы спарклайнов. Следующий пример кода загружает исходный файл Excel, как показано на снимке экрана выше, затем обращается к первой группе спарклайнов и добавляет диапазоны данных и местоположения в группу спарклайнов. Наконец, он записывает выходной файл Excel на диск, который также показан на снимке экрана выше.
