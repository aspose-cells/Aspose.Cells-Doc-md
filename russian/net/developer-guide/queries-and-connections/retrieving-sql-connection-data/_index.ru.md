{{% alert color="primary" %}}

 Aspose.Cells может помочь вам получить данные соединения SQL. Сюда входят любые и все данные, необходимые для подключения к серверу SQL, например,**URL-адрес сервера**, **имя пользователя**, **имя таблицы**, **полный SQL-запрос**, **тип запроса**, **расположение стола** , и**имя именованного диапазона** связанные с ним.

{{% /alert %}}

В Microsoft Excel подключитесь к базе данных:

1.  Нажав на**Данные** меню и выбор**Из других источников** с последующим**Из SQL-сервера**.
1.  Затем выберите**Данные** с последующим**Соединения**.
1. Используйте мастер подключений для подключения к базе данных и создания запроса к базе данных.

Aspose.Cells предоставляет свойство Workbook.DataConnections для получения внешних подключений. Он возвращает коллекцию объектов ExternalConnection в книге.

Если объект ExternalConnection содержит данные соединения SQL, его тип можно привести к объекту DBConnection, а его свойства можно использовать для получения команды базы данных, типа команды, описания соединения, информации о соединении, учетных данных и т. д.
