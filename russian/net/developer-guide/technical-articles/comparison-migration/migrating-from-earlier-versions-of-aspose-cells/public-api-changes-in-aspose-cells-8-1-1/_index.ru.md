В этом документе описаны изменения в Aspose.Cells API с версии 8.1.0 до 8.1.1, которые могут представлять интерес для разработчиков модулей/приложений. Он включает в себя не только новые и обновленные общедоступные методы, но и описание любых изменений в поведении за кулисами в Aspose.Cells.

{{% /alert %}} 
## **Добавлено свойство HtmlSaveOptions.PresentationPreference.**
Класс HtmlSaveOptions предоставляет свойство PresentationPreference, которое можно использовать для визуализации результатов с лучшим макетом при экспорте электронных таблиц в HTML или MHTML. Значение по умолчанию неверно. тогда как если установлено значение true, Aspose.Cells API будет экспортировать содержимое рабочего листа с лучшим представлением.

{{% alert color="primary" %}} 

 Пожалуйста, ознакомьтесь с подробной статьей о[Используйте опцию PresentationPreference для лучшего макета](/cells/ru/net/excel-to-html-use-presentationpreference-option-for-better-layout/)

## **Добавлена поддержка сценариев рабочего листа.**
 Сценарий называется моделью «что, если», которая включает переменные входные ячейки, связанные вместе одной или несколькими формулами соответственно. Aspose.Cells API предоставил свойство Worksheet.Scenarios вместе со следующими классами, чтобы облегчить пользователям создание, управление и удаление сценариев из рабочих листов,

1. Сценарий: представляет отдельный сценарий.
1. ScenarioCollection: представляет набор сценариев.
1. ScenarioInputCellCollection: представляет список входных ячеек для определенного сценария.
1. ScenarioInputCell: представляет входную ячейку из коллекции входных ячеек для определенного сценария.

 Пожалуйста, ознакомьтесь с подробной статьей о[Как создавать, манипулировать или удалять сценарии из рабочих листов](/cells/ru/net/create-manipulate-or-remove-scenarios-from-worksheets/).
