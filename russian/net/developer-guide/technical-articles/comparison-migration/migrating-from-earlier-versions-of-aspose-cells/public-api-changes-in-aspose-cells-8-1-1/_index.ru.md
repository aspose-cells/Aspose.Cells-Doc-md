Этот документ описывает изменения в API Aspose.Cells от версии 8.1.0 до 8.1.1, которые могут быть интересны модульным/приложенческим разработчикам. Здесь приведены не только новые и обновленные публичные методы, но также описание любых изменений в поведении внутри Aspose.Cells.

{{% /alert %}} 
## **Добавлено свойство HtmlSaveOptions.PresentationPreference**
Класс HtmlSaveOptions предоставил свойство PresentationPreference, которое может использоваться для визуализации результатов с лучшим макетом при экспорте электронных таблиц в форматы HTML или MHTML. Значение по умолчанию - false. Если установить в true, то API Aspose.Cells будет экспортировать содержимое листа с лучшей презентацией.

Пожалуйста, ознакомьтесь с подробной статьей по адресу [Использование опции PresentationPreference для лучшего макета](/cells/ru/net/excel-to-html-use-presentationpreference-option-for-better-layout/)

## **Добавлена поддержка для Сценариев листа**
Сценарий представляет собой модель "что-будет-если", которая включает в себя переменные входные ячейки, связанные между собой одной или более формулами соответственно. API Aspose.Cells выделил свойство Worksheet.Scenarios вместе с следующими классами, чтобы облегчить пользователям создание, изменение и удаление сценариев с рабочих листов. 

1. Scenario: Представляет отдельный сценарий.
1. ScenarioCollection: Представляет коллекцию сценариев.
1. ScenarioInputCellCollection: Представляет список входных ячеек для конкретного сценария.
1. ScenarioInputCell: Представляет входную ячейку из коллекции входных ячеек для конкретного сценария.
