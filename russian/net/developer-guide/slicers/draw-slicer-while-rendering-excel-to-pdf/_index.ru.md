## **Draw Slicer при рендеринге Excel в PDF**
Если у вас есть файл Excel, к которому применен слайсер, и вы хотите экспортировать Excel в PDF с настройками слайсера, Aspose.Cells теперь поддерживает это по умолчанию. Вы просто экспортируете файл Excel со слайсером в PDF, сгенерированный PDF-файл покажет примененный слайсер.

 Следующий пример кода загружает[образец файла Excel](94044165.xlsx) который содержит существующий слайсер. Затем он сохраняет книгу как[выходной PDF-файл](94044166.pdf). На следующем снимке экрана сравнивается исходный файл Excel и сгенерированный файл PDF.
