## **Удалить элемент управления ActiveX**

Aspose.Cells предоставляет возможность удаления элементов управления ActiveX из книг. Для этого API предоставляет метод [**Shape.RemoveActiveXControl**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/methods/removeactivexcontrol).
