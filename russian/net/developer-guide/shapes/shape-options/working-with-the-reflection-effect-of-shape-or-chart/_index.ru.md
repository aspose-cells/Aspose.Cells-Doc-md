## **Возможные сценарии использования**
Aspose.Cells предоставляет свойство [Shape.Reflection](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/reflection) вместе с классом [ReflectionEffect](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect) для работы с эффектом отражения формы или диаграммы. Класс [ReflectionEffect](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect) содержит следующие свойства, которые можно установить для достижения различных результатов в соответствии с требованиями приложения.

- [Размытие](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect/properties/blur)
- [Направление](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect/properties/direction)
- [Расстояние](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect/properties/distance)
- [Направление затухания](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect/properties/fadedirection)
- [Поворот с формой](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect/properties/rotwithshape)
- [Размер](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect/properties/size)
- [Прозрачность](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect/properties/transparency)
- [Тип](https://reference.aspose.com/cells/net/aspose.cells.drawing/reflectioneffect/properties/type)
