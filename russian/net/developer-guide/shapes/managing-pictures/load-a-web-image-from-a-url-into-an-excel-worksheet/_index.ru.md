## Загрузка изображения из URL в рабочий лист Excel

API Aspose.Cells for .NET предоставляет простой и удобный способ загрузки изображений из URL в рабочие листы Excel. В этой статье объясняется загрузка данных изображения в поток, а затем его вставка в рабочий лист с помощью API Aspose.Cells. С использованием этого метода изображения становятся частью файла Excel и не загружаются каждый раз при открытии рабочего листа.
