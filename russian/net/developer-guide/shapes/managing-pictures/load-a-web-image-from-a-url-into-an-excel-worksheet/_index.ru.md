## Загрузить изображение из URL-адреса на лист Excel

Aspose.Cells for .NET API предоставляет простой и удобный способ загрузки изображений с URL-адресов в рабочие листы Excel. В этой статье объясняется, как загрузить данные изображения в поток, а затем вставить их в рабочий лист с помощью Aspose.Cells API. При использовании этого метода изображения становятся частью файла Excel и не загружаются при каждом открытии рабочего листа.
