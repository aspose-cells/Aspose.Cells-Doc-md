## **Получить Cell Проверка в файлах ODS**

С помощью Aspose.Cells вы можете применить проверку к ячейке в файлах ODS. Для этого API предоставляет[**Получитьпроверку**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) метод[**Cell**](https://reference.aspose.com/cells/net/aspose.cells/cell) учебный класс.

В следующем примере кода показано использование[**Получитьпроверку**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) метод, загрузив[источник ОРВ](101089354.ods) файл и чтение проверки ячейки A9.
