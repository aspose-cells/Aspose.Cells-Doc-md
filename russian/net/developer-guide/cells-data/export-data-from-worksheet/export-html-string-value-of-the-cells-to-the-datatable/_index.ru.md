## **Возможные сценарии использования**
Используя API-интерфейсы Aspose.Cells, разработчики могут заполнять DataTable соответствующими строками HTML (на основе значений ячеек) при экспорте данных рабочего листа. В этом документе объясняется, как экспортировать строки HTML для данных отформатированных ячеек для заполнения DataTable с помощью API Aspose.Cells.
