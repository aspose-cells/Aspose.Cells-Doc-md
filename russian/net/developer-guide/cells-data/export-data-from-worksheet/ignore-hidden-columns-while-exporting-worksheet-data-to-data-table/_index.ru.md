{{% alert color="primary" %}}

 Иногда вы хотите игнорировать скрытые столбцы при экспорте данных листа в таблицу данных. Вы можете добиться этого, используя Aspose.Cells, установив[**ExportTableOptions.PlotVisibleColumns**](https://reference.aspose.com/cells/net/aspose.cells/exporttableoptions/properties/plotvisiblecolumns) к**истинный**. По умолчанию его значение **false**, поэтому вам необходимо установить его **true.** игнорировать скрытые столбцы.

{{% /alert %}}
