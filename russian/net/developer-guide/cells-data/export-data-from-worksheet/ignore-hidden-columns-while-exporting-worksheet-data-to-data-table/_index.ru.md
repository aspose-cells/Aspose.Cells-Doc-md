{{% alert color="primary" %}}

 Иногда вы хотите игнорировать скрытые столбцы при экспорте данных рабочего листа в таблицу данных. Вы можете добиться этого, используя Aspose.Cells, установив[**Экспорттаблеоптионс.PlotVisibleColumns**](https://reference.aspose.com/cells/net/aspose.cells/exporttableoptions/properties/plotvisiblecolumns) к**истинный** . По умолчанию его значение равно**ЛОЖЬ** , так что вам нужно установить его**истинный** игнорировать скрытые столбцы.

{{% /alert %}}
