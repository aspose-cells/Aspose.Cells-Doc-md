{{% alert color="primary" %}} 

Заполнение электронной таблицы данными сначала по строкам, а затем по столбцам повышает общую производительность.

{{% /alert %}} 

Размещение данных в последовательности A1, B1, A2, B2 происходит быстрее, чем A1, A2, B1, B2. Если на листе много ячеек и вы следуете второй последовательности, то есть заполняете данные строка за строкой, этот совет может значительно ускорить работу программы.
