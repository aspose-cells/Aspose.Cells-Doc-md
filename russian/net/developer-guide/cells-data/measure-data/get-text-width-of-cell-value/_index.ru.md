## **Получить ширину текста Cell Значение**

 Иногда разработчикам может потребоваться вычислить ширину значения ячейки для размещения данных в каком-либо макете. Aspose.Cells обеспечивает[**CellsHelper.GetTextWidth**](https://reference.aspose.com/cells/net/aspose.cells/cellshelper/methods/gettextwidth) метод, который позволяет разработчикам получить ширину текста значения ячейки. В следующем примере кода показано, как использовать[**CellsHelper.GetTextWidth**](https://reference.aspose.com/cells/net/aspose.cells/cellshelper/methods/gettextwidth)для доступа к ширине текста значения ячейки.
