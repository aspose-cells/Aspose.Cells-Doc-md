##  **Возможные сценарии использования**

Aspose.Cells позволяет сместить первую строку вниз при вставке таблицы на лист. В этом документе объясняется, как можно выполнить эту задачу с помощью API Aspose.Cells.

##  **Сдвиньте первую строку вниз при вставке Cells строк таблицы данных.**

В следующем примере кода показано, как сдвинуть первую строку вниз при вставке таблицы на лист. Для демонстрации этой функции мы используем простой шаблон файла Excel в коде. Вы можете использовать эту функцию, установив логическое значение[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown) приписывать**Верно**/**Неверно** чтобы лучше понять это. Пожалуйста, ознакомьтесь с[образец файла Excel](45056031.xlsx), [выходной файл Excel False](45056032.xlsx) , и[выходной файл Excel True](45056033.xlsx)для вашей справки.
