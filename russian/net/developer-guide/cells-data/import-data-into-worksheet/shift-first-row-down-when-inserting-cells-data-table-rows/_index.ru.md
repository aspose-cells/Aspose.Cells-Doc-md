## **Возможные сценарии использования**

Aspose.Cells позволяет сместить первую строку вниз при вставке таблицы на лист. В этом документе объясняется, как выполнить эту задачу с помощью API Aspose.Cells.

## **Сдвинуть первую строку вниз при вставке Cells строк таблицы данных**

В следующем примере кода показано, как сместить первую строку вниз при вставке таблицы на лист. Мы используем простой файл шаблона Excel в коде, чтобы продемонстрировать эту функцию. Вы можете использовать эту функцию, установив логическое значение[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown) приписывать**Истинный**/**ЛОЖЬ** чтобы лучше понять это. Пожалуйста, смотрите[образец файла Excel](45056031.xlsx), [выходной файл Excel False](45056032.xlsx) , и[выходной файл Excel True](45056033.xlsx)для вашей справки.
