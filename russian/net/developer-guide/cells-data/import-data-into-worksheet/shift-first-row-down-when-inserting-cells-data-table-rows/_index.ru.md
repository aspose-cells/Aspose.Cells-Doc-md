## **Возможные сценарии использования**

Aspose.Cells позволяет сдвигать первую строку вниз при вставке таблицы на лист. Этот документ объясняет, как вы можете выполнить эту задачу, используя API Aspose.Cells.

## **Сдвинуть первую строку вниз при вставке строк таблицы данных ячеек**

Приведенный ниже образец кода иллюстрирует, как сдвинуть первую строку вниз при вставке таблицы в рабочий лист. Мы используем простой шаблонный файл Excel в коде, чтобы продемонстрировать функцию. Вы можете проверить функцию, установив булевый атрибут [**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown) в **True**/**False**, чтобы лучше понять его. Пожалуйста, ознакомьтесь с [образцом файла Excel](45056031.xlsx), [выходным файлом Excel False](45056032.xlsx) и [выходным файлом Excel True](45056033.xlsx) для вашего ознакомления.
