## **Пользовательская сортировка в сводной таблице**
Используя Aspose.Cells API, вы можете сортировать сводные таблицы по значениям полей. Следующий фрагмент кода загружает образец файла Excel и добавляет три сводные таблицы. Первая сводная таблица не имеет пользовательской сортировки, вторая сводная таблица отсортирована по значениям поля строки «SeaFood», а третья сводная таблица отсортирована по значениям поля столбца «28/07/2000».
