## **Пользовательская сортировка в сводной таблице**
Используя API Aspose.Cells, вы можете сортировать сводные таблицы по значениям поля. В следующем фрагменте кода загружается пример файла Excel и добавляются три сводные таблицы. Первая сводная таблица без пользовательской сортировки, вторая сводная таблица отсортирована по значениям области "Морепродукты", а третья сводная таблица отсортирована по значениям поля столбца "28/07/2000".
