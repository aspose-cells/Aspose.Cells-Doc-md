{{% alert color="primary" %}}

 Aspose.Cells предоставляет функцию удаления или удаления сводной таблицы из рабочего листа. Вы можете удалить сводную таблицу, используя объект сводной таблицы или позицию сводной таблицы. Пожалуйста, используйте[**Рабочий лист.Сводные таблицы.Удалить()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottablecollection/methods/remove) метод удаления сводной таблицы с использованием объекта сводной таблицы и[**Рабочий лист.Сводные таблицы.RemoveAt()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottablecollection/methods/removeat) метод для удаления объекта сводной таблицы, используя его положение внутри коллекции сводных таблиц.

{{% /alert %}}

 Следующий пример кода удаляет две сводные таблицы с листа. Сначала он удаляет сводную таблицу, используя[**Рабочий лист.Сводные таблицы.Удалить()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottablecollection/methods/remove) метод, а затем удаляет сводную таблицу, используя[**Рабочий лист.Сводные таблицы.RemoveAt()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottablecollection/methods/removeat) метод
