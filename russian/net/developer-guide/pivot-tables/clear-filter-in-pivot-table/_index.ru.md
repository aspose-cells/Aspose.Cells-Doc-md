##  **Возможные сценарии использования**
 Когда вы создаете сводную таблицу с известными данными и хотите отфильтровать сводную таблицу, вам нужно изучить и использовать фильтр. Это может помочь вам эффективно отфильтровать нужные данные. Используя Aspose.Cells API, вы можете фильтровать значения полей в сводных таблицах.

##  **Очистить фильтр в сводной таблице Excel**
Очистите фильтр в сводной таблице в Excel, выполните следующие действия:

1.  Выберите сводную таблицу, для которой вы хотите очистить фильтр.
2. Щелкните стрелку раскрывающегося списка для фильтра, который вы хотите очистить в сводной таблице.
3. Выберите «Очистить фильтр» в раскрывающемся меню.
<img src="1.png" width=80% />
4. Если вы хотите очистить все фильтры из сводной таблицы, вы также можете нажать кнопку «Очистить фильтры» на вкладке «Анализ сводной таблицы» на ленте в Excel.
<img src="2.png" width=80% />

##  **Очистить фильтр в сводной таблице с помощью C#**
 Очистите фильтр в сводной таблице, используя Aspose.Cells. См. следующий пример кода.
1.  Установите данные и создайте на их основе сводную таблицу.
2. Добавьте фильтр в поле строки сводной таблицы.
 3. Сохраните книгу в[вывод XLSX](out_add.xlsx) формат. После выполнения примера кода на рабочий лист добавляется сводная таблица с фильтром top10.
 4. Очистите фильтр для определенного сводного поля. После выполнения кода для очистки фильтра фильтр для определенного сводного поля будет очищен. Пожалуйста, проверьте[вывод XLSX](out_delete.xlsx).
