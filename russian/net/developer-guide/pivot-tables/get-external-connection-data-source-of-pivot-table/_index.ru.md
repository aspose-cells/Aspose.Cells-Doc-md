## **Получить внешний источник подключения сводной таблицы**

Aspose.Cells предоставляет возможность получить внешний источник данных подключения сводной таблицы. Для этого API предоставляет свойство [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) класса [**PivotTable**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable). Свойство [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) возвращает объект [**ExternalConnection**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection). Приведенный ниже фрагмент кода демонстрирует использование свойства [**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) для получения внешнего источника данных подключения сводной таблицы.
