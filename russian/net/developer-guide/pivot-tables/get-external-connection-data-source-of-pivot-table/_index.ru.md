## **Получить источник данных внешнего подключения сводной таблицы**

Aspose.Cells предоставляет возможность получить внешний источник данных соединения сводной таблицы. Для этого API предоставляет[**Внешний источник данных соединения**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)собственность[**сводная таблица**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) учебный класс.[**Внешний источник данных соединения**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)свойство возвращает[**Внешнее соединение**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) объект. Следующий фрагмент кода демонстрирует использование[**Сводная таблица.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) свойство, чтобы получить внешний источник данных соединения сводной таблицы.
