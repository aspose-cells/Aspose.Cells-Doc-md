Aspose.Cells предоставляет возможность сохранять сводную таблицу в файлах ODS. Для этого просто конвертируйте книгу с существующей сводной таблицей или создайте новую сводную таблицу и сохраните файл в формате ODS. Обязательно вызовите вызов [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata) перед сохранением, чтобы быть уверенными, что сводная таблица отображается в выходном файле ODS. Нижеприведенный фрагмент кода демонстрирует сохранение сводной таблицы в файле ODS.
