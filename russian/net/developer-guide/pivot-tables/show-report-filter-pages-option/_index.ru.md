## **Опция отображения страниц фильтров отчета**
Excel поддерживает создание сводных таблиц, добавление параметров фильтрации отчетов и включение параметра "Показывать страницы фильтра отчета". Aspose.Cells также поддерживает эту функцию для включения параметра "Показывать страницы фильтра отчета" для созданной сводной таблицы. На экране показан параметр "Показывать страницы фильтра отчета" в Excel.
