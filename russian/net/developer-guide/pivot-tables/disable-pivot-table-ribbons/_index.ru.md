{{% alert color="primary" %}}

Отчеты на основе сводных таблиц полезны, но подвержены ошибкам, если целевые пользователи не обладают подробными знаниями Excel для настройки этих отчетов. В этих обстоятельствах организации могут захотеть запретить пользователям изменять отчет на основе сводной таблицы. Общие функции сводной таблицы, такие как добавление дополнительных фильтров, срезов, полей или изменение порядка определенных вещей в отчете, в основном не рекомендуются для каждого пользователя. С другой стороны, эти пользователи также должны иметь возможность обновлять отчет и использовать существующие фильтры или слайсеры. Aspose.Cells предоставил разработчикам возможность запретить пользователям изменять эти отчеты при создании этих отчетов. Для этой цели Excel предоставляет функцию отключения ленты сводной таблицы, и то же самое предоставляется Aspose.Cells, т.е. разработчик может отключить ленту, которая содержит элементы управления для изменения этих отчетов.

{{% /alert %}}
