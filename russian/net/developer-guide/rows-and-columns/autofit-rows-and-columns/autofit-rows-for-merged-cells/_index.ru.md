{{% alert color="primary" %}}

Microsoft В Excel предусмотрена функция, позволяющая автоматически изменять высоту ячейки в соответствии с ее содержимым. Эта функция называется автоматическим подбором строк. Microsoft Excel изначально не устанавливает операцию автоматического подгонки для объединенных ячеек. Иногда эта функция становится жизненно важной для пользователя, которому действительно необходимо реализовать автоматическое подгонку строк и в объединенных ячейках.

{{% /alert %}}

##  **Как использовать AutoFitMergedCellsType для автоматической подгонки строк**
 Aspose.Cells поддерживает эту функцию через[**AutoFitterOptions.AutoFitMergedCellsType**](https://reference.aspose.com/cells/net/aspose.cells/autofitmergedcellstype/)API. Используя этот API, можно автоматически подогнать строки на листе, включая объединенные ячейки. Вот список всех возможных типов автоматической подгонки объединенных ячеек:

- Никто
- Первая линия
- Последняя линия
- Каждая строка
