## **Возможные сценарии использования**
Высота строки автоматически соответствует шрифту содержимого, но когда высота кэшированной строки не соответствует высоте содержимого в файле, MS Excel автоматически настраивает высоту строки при загрузке файла, в то время как Aspose.Cells не будет автоматически ее настраивать для повышения производительности. Если вам нужно использовать программу Aspose.Cells для автоматической подгонки высоты строк при загрузке файлов, вы можете достичь этой цели через параметр [LoadOptions.AutoFitterOptions.OnlyAuto](https://reference.aspose.com/cells/net/aspose.cells/autofitteroptions/onlyauto/).

Пожалуйста, обратитесь к следующим данным изображения. Мы видим, что высота кэшированной строки в строке 11 равна 15, но Excel автоматически подгоняет высоту строки при загрузке файла.

## **Настройка высоты строки с помощью Aspose.Cells**
Если вы загружаете файл непосредственно и сохраняете его в формате PDF, данные не будут полностью отображены в PDF, потому что высота кэшированной строки равна только 15.
<br>
<img src="2.png" width=70% />
<br>
Если вы установите параметр [LoadOptions.AutoFitterOptions.OnlyAuto](https://reference.aspose.com/cells/net/aspose.cells/autofitteroptions/onlyauto/) в true при загрузке файла, то Aspose.Cells автоматически подгонит высоту строки. Настроенная высота строки может эффективно соответствовать требованиям отображения текста.
