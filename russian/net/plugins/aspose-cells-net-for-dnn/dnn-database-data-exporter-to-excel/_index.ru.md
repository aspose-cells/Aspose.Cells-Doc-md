## **Вступление**

Aspose .NET Модуль экспорта данных DNN в Excel позволяет пользователям экспортировать данные непосредственно из локальных или удаленных таблиц базы данных, представлений и по пользовательскому запросу в электронные таблицы Microsoft Excel или OpenOffice. Этот модуль демонстрирует мощную функцию построения электронных таблиц, предоставляемую Aspose.Cells.

### **Возможности модуля**

Эта первоначальная версия модуля обогащена следующими интересными функциями, которые делают процесс экспорта простым и удобным в использовании.

- Разрешить подключение к локальной базе данных MS SQL Server
- Разрешить подключение к удаленной базе данных MS SQL Server
- Заполнить все таблицы из подключенной базы данных
- Заполнить все представления из подключенной базы данных
- Разрешить писать пользовательский запрос
- Автоподбор столбцов по длине содержимого.
- Разрешить пропускать строку более 32 КБ в ячейках Excel (LoadOptions)
- Применить форматирование столбца заголовка как полужирный текст
- Разрешить использовать в качестве источника данных (таблица, представления, пользовательский запрос)
- Экспорт данных в документы Excel Microsoft (.xls, .xlsx и .xlsb)
- Экспорт данных в текстовый документ с разделителями табуляции (*.txt)
- Экспорт данных в CSV (с разделителями-запятыми) (*.csv)
- Экспорт данных в электронную таблицу OpenDocument (*.ods)
- Возможность выбора желаемого выходного формата перед экспортом.
- Экспортированный документ автоматически отправляется в браузер для загрузки.

## **Установка**

После загрузки выполните следующие действия, чтобы установить модуль на свой веб-сайт DNN:

1. Войдите на свой сайт либо как хост, либо как другая учетная запись уровня суперпользователя.
1.  Перейдите к**Хозяин** меню и выберите**Расширения**.
1.  Нажмите**Мастер установки расширений**.
1. Как указано, перейдите к местоположению загруженного zip-файла, выберите его и нажмите**Открытым**.
1.  Нажмите**Следующий** , принять лицензию, продолжить установку. Когда закончите, нажмите кнопку**Возвращаться** кнопка.

 пожалуйста, проверьте[видео по установке этого модуля](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) от DNN для более подробной информации.

## **С использованием**

После того, как вы установили модуль Aspose .NET Database Data Expoeter для DNN, начать использовать его на своем веб-сайте очень просто. Пожалуйста, выполните следующие простые шаги, чтобы начать:

1. Убедитесь, что вы вошли в DNN с учетной записью уровня хоста или администратора.
1. Перейдите на страницу, на которую вы хотите добавить модуль экспорта.
1.  Выбирать**Модули** с последующим**Добавить новый модуль** из верхней ленты.
1. Из списка выберите «Aspose.NET Средство экспорта данных DNN в Excel с использованием Aspose.Cells» и перетащите его в нужное место.
