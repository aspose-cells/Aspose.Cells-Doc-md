## **Введение**

Модуль экспорта данных Aspose .NET DNN в Excel позволяет пользователям экспортировать данные непосредственно из локальных или удаленных таблиц баз данных, представлений и пользовательских запросов в документ Microsoft Excel или электронную таблицу OpenOffice. Этот модуль демонстрирует мощную функцию построения электронных таблиц, предоставляемую Aspose.Cells.

### **Возможности модуля**

Эта первая версия модуля обогащена следующими функциями, чтобы сделать процесс экспорта простым и удобным в использовании

- Разрешить подключение локальной базы данных MS SQL Server
- Разрешить подключение удаленной базы данных MS SQL Server
- Заполнить все таблицы из подключенной базы данных
- Заполнить все представления из подключенной базы данных
- Разрешить написать пользовательский запрос
- Автоматическая подгонка столбцов под длину содержимого.
- Разрешить пропускать строку более 32k в ячейках Excel (параметры загрузки)
- Применять форматирование заголовка столбца как полужирный текст
- Разрешить использовать в качестве источника данных (таблица, представления, пользовательский запрос)
- Экспорт данных в документы Microsoft Excel (.xls, .xlsx и .xlsb)
- Экспорт данных в текстовый документ с разделителями табуляции (*.txt)
- Экспорт данных в CSV (с разделителями-запятыми) (*.csv)
- Экспорт данных в OpenDocument Spreadsheet (*.ods)
- Возможность выбора желаемого формата вывода перед экспортом.
- Экспортируемый документ автоматически отправляется в браузер для загрузки.

## **Установка**

После загрузки, пожалуйста, выполните следующие шаги для установки модуля на свой сайт DNN:

1. Войдите на сайт под учетной записью либо хоста, либо другой учетной записью с супер-пользовательским уровнем.
1. Перейдите в меню **Хост** и выберите **Расширения**.
1. Нажмите **Установить мастер расширений**.
1. По указаниям перейдите в место загруженного zip-файла, выберите его, затем нажмите **Открыть**.
1. Нажмите **Далее**, примите лицензию, продолжайте установку. По завершении нажмите кнопку **Вернуться**.

Пожалуйста, посмотрите [это видео установки модуля](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) от DNN для получения более подробной информации.

## **Использование**

После установки модуля Aspose .NET Database Data Expoeter для DNN начать использовать его на вашем сайте очень просто. Пожалуйста, следуйте этим простым шагам, чтобы начать:

1. Убедитесь, что вы вошли в систему в DNN с учетной записью уровня хоста или администратора.
1. Перейдите на страницу, где хотите добавить модуль экспорта.
1. Выберите **Модули**, затем **Добавить новый модуль** в верхней панели.
1. Из списка выберите ‘Aspose.NET DNN Data Exporter to Excel Using Aspose.Cells’ и перетащите его на нужное место.

![todo:image_alt_text](picture2.png)

Вы успешно добавили экспортер данных базы данных Aspose .NET в Excel для модуля DNN на вашу страницу.

![todo:image_alt_text](picture3.png)

1. Введитe или используйте предварительно заполненную строку подключения к MS SQL Server.
1. Выберитe тип источника данных (Таблица, Представление, Пользовательский запрос)
1. Выберитe или введите источник данных (Таблица, Представление, Пользовательский запрос)
1. Выберитe тип экспорта
1. Нажмите Экспортировать данные
1. Желаемый файл будет загружен автоматически.
