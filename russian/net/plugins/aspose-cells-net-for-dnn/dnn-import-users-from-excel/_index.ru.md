## **Введение**
Модуль Aspose .NET Импорт пользователей из Excel позволяет разработчикам импортировать массовых пользователей из файла Excel. Этот модуль демонстрирует мощную функцию электронных таблиц Aspose.Cells.
### **Возможности модуля**
Эта начальная версия модуля обогащена следующими удивительными функциями, чтобы сделать процесс импорта простым и удобным в использовании:

- Позволяет загрузить файл Excel и автоматически фильтрует существующих пользователей.
- Автоматически пропускает существующих пользователей, чтобы избежать путаницы.
- Генерирует динамические пароли для вновь созданных пользователей.
- Назначает роли безопасности «Зарегистрированные пользователи» и «Подписчики» вновь созданным пользователям.

## **Установка**
После загрузки, пожалуйста, выполните следующие шаги для установки модуля на свой сайт DNN:

1. Войдите на сайт под учетной записью либо хоста, либо другой учетной записью с супер-пользовательским уровнем.
1. Перейдите в меню **Хост** и выберите **Расширения**.
1. Нажмите **Установить мастер расширений**.
1. По указаниям перейдите в место загруженного zip-файла, выберите его, затем нажмите **Открыть**.
1. Нажмите **Далее**, примите лицензию, продолжайте установку. По завершении нажмите кнопку **Вернуться**.

Пожалуйста, посмотрите [это видео установки модуля](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) от DNN для получения более подробной информации.
## **Использование**
После установки модуля Aspose .NET Cells Import Users from Excel для DNN очень просто начать его использовать на вашем сайте. Пожалуйста, следуйте этим простым шагам, чтобы начать:

1. Убедитесь, что вы вошли в систему в DNN с учетной записью уровня хоста или администратора.
1. Перейдите на страницу, где хотите добавить модуль экспорта.
1. Выберите **Модули**, затем **Добавить новый модуль** в верхней панели. 

1. Из списка выберите 'Aspose Импорт пользователей из Excel для DNN' и перетащите его на позицию.

Вы успешно добавили модуль Aspose .NET Cells Import Users from Excel для DNN на свою страницу. Элементы управления загрузки файлов будут видны вместе с ссылкой "Файл шаблона". Вы можете загрузить файл шаблона и отформатировать свое содержимое в соответствии с файлом шаблона. После загрузки вашего файла Excel он автоматически обработает всех пользователей и отфильтрует имеющихся пользователей. Вы можете выбрать пользователей, которых хотите импортировать, и нажать кнопку "Импортировать пользователей".
