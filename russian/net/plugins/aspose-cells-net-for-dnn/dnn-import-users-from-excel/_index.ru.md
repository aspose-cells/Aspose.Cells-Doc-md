## **Введение**
Aspose .NET Модуль импорта пользователей из Excel позволяет разработчикам массово импортировать пользователей из файла Excel. Этот модуль демонстрирует мощную функцию электронных таблиц по номеру Aspose.Cells.
### **Возможности модуля**
Эта первоначальная версия модуля обогащена следующими интересными функциями, которые делают процесс импорта простым и легким в использовании:

- Позволяет загружать файл Excel и автоматически отфильтровывает существующих пользователей.
- Автоматически пропускает существующих пользователей, чтобы избежать путаницы.
- Создайте динамические пароли для вновь созданных пользователей.
- Назначает вновь созданным пользователям роли безопасности «Зарегистрированные пользователи» и «Подписчики».

## **Установка**
После загрузки выполните следующие действия, чтобы установить модуль на свой веб-сайт DNN:

1. Войдите на свой сайт либо как хост, либо как другая учетная запись уровня суперпользователя.
1.  Перейдите к**Хозяин** меню и выберите**Расширения**.
1.  Нажмите**Мастер установки расширений**.
1. Как указано, перейдите к местоположению загруженного zip-файла, выберите его и нажмите**Открытым**.
1.  Нажмите**Следующий** , принять лицензию, продолжить установку. Когда закончите, нажмите кнопку**Возвращаться** кнопка.

 пожалуйста, проверьте[видео по установке этого модуля](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) от DNN для более подробной информации.
## **С использованием**
После того, как вы установили модуль Aspose .NET Cells Импорт пользователей из Excel для DNN, начать использовать его на своем веб-сайте очень просто. Пожалуйста, выполните следующие простые шаги, чтобы начать:

1. Убедитесь, что вы вошли в DNN с учетной записью уровня хоста или администратора.
1. Перейдите на страницу, на которую вы хотите добавить модуль экспорта.
1.  Выбирать**Модули** с последующим**Добавить новый модуль** из верхней ленты.

1. Из списка выберите «Aspose Импорт пользователей из модуля Excel для DNN» и перетащите его в нужное место.

Вы успешно добавили модуль Aspose .NET Cells Импорт пользователей из Excel для модуля DNN на свою страницу. Элементы управления загрузкой файлов будут видны вместе со ссылкой «Файл шаблона». Вы можете загрузить файл шаблона и отформатировать содержимое в соответствии с файлом шаблона. Как только вы загрузите свой файл Excel, он автоматически обработает всех пользователей и отфильтрует существующих пользователей. Вы можете выбрать пользователей, которых хотите импортировать, и нажать кнопку «Импортировать пользователей».
