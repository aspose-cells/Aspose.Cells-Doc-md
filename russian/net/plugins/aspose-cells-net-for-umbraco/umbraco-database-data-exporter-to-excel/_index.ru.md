## **Вступление**
Aspose .NET Экспортер данных базы данных в Excel для модуля Umbraco позволяет пользователям экспортировать данные непосредственно из локальных или удаленных таблиц базы данных, представлений и по пользовательскому запросу в Microsoft Excel или электронную таблицу OpenOffice. Этот модуль демонстрирует мощную функцию построения электронных таблиц, предоставленную Aspose.Cells. Эта первоначальная версия модуля обогащена следующими интересными функциями, чтобы сделать процесс экспорта простым и легким в использовании.
### **Возможности модуля**
Эта начальная версия дополнения имеет следующие особенности:

- Разрешить подключение к локальной базе данных MS SQL Server
- Разрешить подключение к удаленной базе данных MS SQL Server
- Заполнить все таблицы из подключенной базы данных
- Заполнить все представления из подключенной базы данных
- Разрешить писать пользовательский запрос
- Автоподбор столбцов по длине содержимого.
- Разрешить пропускать строку более 32 КБ в ячейках Excel (LoadOptions)
- Применить форматирование столбца заголовка как полужирный текст
- Разрешить использовать в качестве источника данных (таблица, представления, пользовательский запрос)
- Экспорт данных в документы Excel Microsoft (.xls, .xlsx и .xlsb)
- Экспорт данных в текстовый документ с разделителями табуляции (*.txt)
- Экспорт данных в CSV (с разделителями-запятыми) (*.csv)
- Экспорт данных в электронную таблицу OpenDocument (*.ods)
- Возможность выбора желаемого выходного формата перед экспортом.
- Экспортированный документ автоматически отправляется в браузер для загрузки.

## **Системные требования и поддерживаемые платформы**
### **Системные Требования**
Чтобы настроить модуль экспорта данных базы данных Aspose .NET в Excel для Umbraco, необходимо выполнить следующие требования:

- Версии Umbraco 6.2.5 и Umbraco 6
- Umbraco с MS SQL Server
- Microsoft .Net Framework 4.0

**Примечание:** Umbraco 7 и выше не поддерживаются в этом выпуске. Мы с нетерпением ждем ваших отзывов и добавим поддержку Umbraco 7 в следующей версии.
### **Поддерживаемые платформы**
Модуль поддерживается всеми версиями

- Umbraco 6.0 работает на ASP.NET 4.0

## **Установка**
После загрузки выполните следующие действия, чтобы установить этот пакет на свой веб-сайт Umbraco:

1.  Войти в Umbraco**Разработчик** раздел, например `http://www.myblog.com/umbraco`
1.  Из дерева разверните**Пакеты** папка.
1.  Отсюда есть два способа установить пакет: выберите**Установить локальный пакет** или просмотрите**Репозиторий пакетов Umbraco.**
1. Если вы установите**локальный пакет**, не разархивируйте пакет, а загрузите zip в Umbraco.
1. Следуйте инструкциям на экране.

**Примечание:** При установке может появиться ошибка «Превышена максимальная длина запроса». Вы можете легко решить эту проблему, обновив значение maxRequestLength в файле Umbraco web.config.
<httpRuntime requestValidationMode="2.0" enableVersionHeader="false" maxRequestLength="25000" />
## **С использованием**
После того, как вы установили Aspose .NET Модуль экспорта данных базы данных в Excel для Umbraco, начать использовать его на своем веб-сайте очень просто. Пожалуйста, следуйте этим простым шагам, чтобы начать

1.  Убедитесь, что вы вошли в систему Umbraco.**Разработчик** раздел, например `http://www.myblog.com/umbraco/`
1.  Нажмите**Настройки** в списке разделов в левом нижнем углу экрана.
1.  Развернуть**Шаблоны** node и выберите шаблон, который вы хотите добавить, например Textpage.
1. Выберите позицию в выбранном шаблоне, куда вы хотите добавить кнопку экспорта. Обычно вы можете добавить его в правый верхний угол страницы или в нижнюю часть страницы.
1.  Нажмите**Вставить макрос** на верхней ленте.
1.  От**Выберите макрос** (Aspose .NET Средство экспорта данных базы данных в Excel для Umbraco), выберите недавно установленный макрос Aspose .NET Средство экспорта данных базы данных в Excel для Umbraco и нажмите**ХОРОШО**.
