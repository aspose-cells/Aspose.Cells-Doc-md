## **Введение**
Модуль Aspose .NET Database Data Exporter to Excel для Umbraco позволяет пользователям экспортировать данные непосредственно из локальных или удаленных таблиц, представлений и пользовательского запроса в Microsoft Excel или OpenOffice Spreadsheet. Этот модуль демонстрирует мощную возможность создания электронных таблиц, предоставленную Aspose.Cells. Эта первоначальная версия модуля обогащена следующими удивительными функциями, чтобы сделать процесс экспорта простым и удобным для использования
### **Возможности модуля**
Эта первоначальная версия дополнения имеет следующие функции:

- Разрешить подключение локальной базы данных MS SQL Server
- Разрешить подключение удаленной базы данных MS SQL Server
- Заполнить все таблицы из подключенной базы данных
- Заполнить все представления из подключенной базы данных
- Разрешить написать пользовательский запрос
- Автоматическая подгонка столбцов под длину содержимого.
- Разрешить пропускать строку более 32k в ячейках Excel (параметры загрузки)
- Применять форматирование заголовка столбца как полужирный текст
- Разрешить использовать в качестве источника данных (таблица, представления, пользовательский запрос)
- Экспорт данных в документы Microsoft Excel (.xls, .xlsx и .xlsb)
- Экспорт данных в текстовый документ с разделителями табуляции (*.txt)
- Экспорт данных в CSV (с разделителями-запятыми) (*.csv)
- Экспорт данных в OpenDocument Spreadsheet (*.ods)
- Возможность выбора желаемого формата вывода перед экспортом.
- Экспортируемый документ автоматически отправляется в браузер для загрузки.

## **Требования к системе и поддерживаемые платформы**
### **Системные требования**
Для настройки Aspose .NET Database Data Exporter to Excel для модуля Umbraco вам необходимо выполнить следующие требования:

- Версии Umbraco 6.2.5 и Umbraco 6
- Umbraco с MS SQL Server
- Microsoft .Net Framework 4.0

**Примечание:** Umbraco 7 и выше не поддерживаются в этом релизе. Мы будем рады услышать ваш отзыв и добавим поддержку Umbraco 7 в следующей версии.
### **Поддерживаемые платформы**
Модуль поддерживается на всех версиях

- Umbraco 6.0, работающем на ASP.NET 4.0

## **Установка**
После загрузки, выполните следующие шаги для установки этого пакета на свой веб-сайт Umbraco:

1. Войдите в раздел Umbraco **Developer**, например, `http://www.myblog.com/umbraco`
1. В раскрытом дереве выберите папку **Пакеты**.
1. Здесь есть два способа установить пакет: выберите **Установить локальный пакет** или просмотрите **Хранилище пакетов Umbraco.**
1. Если вы устанавливаете **локальный пакет**, не распаковывайте его, а загрузите zip-файл в Umbraco.
1. Следуйте инструкциям на экране.

**Примечание:** При установке может возникнуть ошибка «Превышена максимальная длина запроса». Эту проблему легко исправить, обновив значение «maxRequestLength» в файле конфигурации веб-сайта Umbraco.
<httpRuntime requestValidationMode="2.0" enableVersionHeader="false" maxRequestLength="25000" />
## **Использование**
После установки модуля Aspose .NET Database Data Exporter to Excel для Umbraco очень просто начать его использовать на вашем веб-сайте. Пожалуйста, выполните эти простые шаги для начала работы

1. Убедитесь, что вы вошли в раздел Umbraco **Developer**, например, `http://www.myblog.com/umbraco/`
1. Нажмите **Настройки** в списке разделов в нижнем левом углу экрана.
1. Разверните узел **Шаблоны** и выберите шаблон, в который вы хотите добавить, например, Textpage.
1. Выберите позицию в выбранном шаблоне, куда вы хотите добавить кнопку экспорта. Обычно вы можете захотеть добавить ее в верхнюю правую часть страницы или в нижнюю часть страницы.
1. Нажмите **Вставить макрос** на верхней ленте.
1. Из **Выбрать макрос** (Aspose .NET Database Data Exporter to Excel для Umbraco) выберите недавно установленный макрос Aspose .NET Database Data Exporter to Excel для Umbraco и нажмите **OK**.
