# **Что такое Microsoft Dynamics CRM?**
Microsoft Dynamics CRM - это программный пакет управления взаимоотношениями с клиентами, разработанный компанией Microsoft. Изначально продукт фокусировался в основном на секторах продаж, маркетинга и обслуживания (служба поддержки), но Microsoft начал рекламировать Dynamics CRM как платформу XRM и стимулировал партнеров использовать его проприетарную (.NET-основанную) среду для настройки. Этот продукт входит в семейство бизнес-приложений Microsoft Dynamics.

Dynamics CRM - это серверно-клиентское приложение, которое, подобно Microsoft SharePoint, в первую очередь является веб-приложением на основе IIS, поддерживающим также обширные интерфейсы веб-сервисов. Клиенты могут получить доступ к Dynamics CRM с помощью браузера или через толстый клиентский плагин для Microsoft Outlook.
# **Aspose.Cells for .Net**
Aspose.Cells for .NET обеспечивает самую гибкую группу компонентов, позволяющих .NET-приложению создавать и управлять электронными таблицами Excel без необходимости установки Microsoft Excel на сервере. Aspose.Cells является одним из ведущих компонентов для манипулирования всеми типами форматов Excel.

Aspose.Cells поддерживает не только создание электронных таблиц и другие основные функции форматирования файлов, но также поддерживает ряд продвинутых функций. Эти продвинутые функции упрощают работу разработчиков по манипулированию содержимым электронных таблиц, форматированию ячеек и защите файлов. Aspose.Cells также позволяет импортировать данные в электронные таблицы из различных источников данных, добавлять общие и сложные математические, даты, финансовые, текстовые формулы/функции, поддерживать манипуляции с диаграммами, изображениями, комментариями, объектами рисования и элементами управления и др.
