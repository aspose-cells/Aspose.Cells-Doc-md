# **Что такое Microsoft Dynamics CRM?**
Microsoft Dynamics CRM — это пакет программного обеспечения для управления взаимоотношениями с клиентами, разработанный Microsoft. По умолчанию продукт ориентирован в основном на сектора продаж, маркетинга и обслуживания (справочная служба), но Microsoft продвигал Dynamics CRM как платформу XRM и был поощрять партнеров использовать его проприетарную (на основе .NET) структуру для ее настройки. Это часть семейства бизнес-приложений Dynamics Microsoft.

Dynamics CRM — это клиент-серверное приложение, которое, как и Microsoft SharePoint, в первую очередь представляет собой веб-приложение на основе IIS, которое также поддерживает расширенные интерфейсы веб-служб. Клиенты получают доступ к Dynamics CRM либо с помощью браузера, либо с помощью подключаемого модуля толстого клиента к Microsoft Outlook.
# **Aspose.Cells для .Net**
Aspose.Cells for .NET предоставляет наиболее гибкую группу компонентов, которые позволяют .NET Приложение создавать электронные таблицы Excel и управлять ими, не требуя установки Microsoft Excel на сервере. Aspose.Cells — один из ведущих компонентов для всех типов работы с форматами Excel.

Aspose.Cells не только поддерживает создание электронных таблиц и другие основные функции форматирования файлов, но также поддерживает ряд дополнительных функций. Эти расширенные функции значительно упрощают разработчикам работу с содержимым электронных таблиц, форматированием ячеек и защитой файлов. Aspose.Cells также позволяет импортировать данные в электронные таблицы из разных источников данных, добавлять общие и сложные математические, датовые, финансовые, текстовые формулы / функции, поддерживает манипулирование диаграммами, изображениями, комментариями, объектами рисования и элементами управления и т. д.
# **Aspose.Cells для CRM**
Aspose .NET можно использовать с Dynamics CRM несколькими способами. Когда вы работаете с электронными таблицами, будь то создание или обработка электронных таблиц, вам нужно взглянуть на API-интерфейсы Aspose, которые могут предоставить быстрые результаты с минимальным количеством строк кода.
