Ниже приведены пошаговые инструкции снимков экрана, которые нам нужно выполнить для использования инструмента:

![todo:image_alt_text](using-and-configuring-crm-update-import-tool_1)

- Чтобы создать соединение с CRM, нажмите Войти в CRM и добавьте информацию об организации CRM 

![todo:image_alt_text](using-and-configuring-crm-update-import-tool_2)

- Как только соединение установлено, выберите Сущность и выберите список желаемых атрибутов. 

![todo:image_alt_text](using-and-configuring-crm-update-import-tool_3)

- **Получить из CRM:** Нажмите здесь, чтобы показать данные из CRM в сетке
- **Экспорт в Excel:** Нажмите здесь, чтобы экспортировать данные из CRM в файл Excel, указав файл.
- **Получить из Excel:** Нажмите здесь и выберите исходный файл Excel, и он отобразит данные из файла Excel в сетке.
- **Экспорт в CRM:** Нажмите здесь и выберите исходный файл Excel. Он импортирует данные из Excel обратно в CRM в выбранную Сущность. (Обратите внимание, что имена столбцов в файле Excel должны соответствовать имени схемы в CRM)
