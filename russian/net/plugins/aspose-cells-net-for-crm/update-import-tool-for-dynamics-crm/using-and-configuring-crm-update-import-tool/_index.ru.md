Ниже приведены шаги на снимке экрана, которые нам нужно выполнить, чтобы использовать инструмент:

![дело:изображение_альтернативный_текст](using-and-configuring-crm-update-import-tool_1)

-  Чтобы создать соединение с CRM, нажмите «Войти в CRM» и добавьте информацию об организации CRM.

![дело:изображение_альтернативный_текст](using-and-configuring-crm-update-import-tool_2)

-  Как только соединение будет установлено, выберите Entity и выберите список нужных атрибутов.

![дело:изображение_альтернативный_текст](using-and-configuring-crm-update-import-tool_3)

- **Получите из CRM:** Нажмите здесь, чтобы отобразить данные из CRM в сетке
- **Экспорт в Excel:** Нажмите здесь, чтобы экспортировать данные из CRM в файл Excel, предоставив файл.
- **Получить из Excel:** Нажмите здесь и выберите исходный файл Excel, и он отобразит данные из файла Excel в сетке.
- **Экспорт в CRM:** Нажмите здесь и выберите исходный файл Excel, и он импортирует данные из Excel обратно в CRM в выбранном объекте. (Обратите внимание, что имена столбцов в файле Excel должны совпадать с именем схемы в CRM)
