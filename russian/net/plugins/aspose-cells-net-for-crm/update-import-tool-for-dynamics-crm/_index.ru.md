## **Введение**
Инструмент обновления Aspose представляет собой бесплатное дополнение для использования в Microsoft Dynamic CRM. Это дополнение способно автоматически обновлять и/или экспортировать данные из CRM в электронные таблицы Aspose.Cells. Мы не должны ничего делать вручную, устанавливать это дополнение и делать нашу работу удобнее и быстрее.

Это дополнение предоставляет функциональность для автоматического обновления и/или экспортирования из CRM. Одним щелчком мыши можно импортировать/экспортировать запись в/из CRM.

Основные возможности этого дополнения:

- Установление связи с желаемым CRM.
- Разрешение пользователю выбора желаемой сущности и полей.
- Получение записи из CRM в виде сетки.
- Экспорт записи в электронную таблицу Aspose.Cells.
- Получение записи из электронной таблицы Aspose.Cells в виде сетки.
- Экспорт записи из сетки в CRM.
## **Требования к системе и поддерживаемые платформы**
### **Системные требования**
Для установки и использования Инструмента Импорта Aspose .Net для Microsoft Dynamics CRM вам нужно иметь установленную одну из следующих версий CRM

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.

Пожалуйста, не стесняйтесь связаться с нами, если вы обнаружите какие-либо проблемы при установке или использовании этого дополнения.
### **Поддерживаемые платформы**
Это дополнение будет работать со всеми версиями Microsoft Dynamics CRM:

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.
