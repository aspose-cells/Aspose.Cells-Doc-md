## **Получить проверку ячейки в файлах ODS**  

С API Aspose.Cells for Node.js via C++ вы можете получать проверку, примененную к ячейке в файлах ODS. Для этого API предоставляет метод [**Cell.getValidation()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getValidation--) класса [**Cell**](https://reference.aspose.com/cells/nodejs-cpp/cell/).  

Следующий пример кода демонстрирует использование метода [**Cell.getValidation()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getValidation--), загружая исходный файл ODS и читая проверку ячейки A9.
