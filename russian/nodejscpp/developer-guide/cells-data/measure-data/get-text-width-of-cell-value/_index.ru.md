## **Получить ширину текста значения ячейки**

Иногда разработчикам может понадобиться вычислить ширину значения ячейки для размещения данных в каком-либо макете. Aspose.Cells for Node.js via C++ предоставляет метод [**CellsHelper.getTextWidth(string, Font, number)**](https://reference.aspose.com/cells/nodejs-cpp/cellshelper/#getTextWidth-string-font-number-), который позволяет получать ширину текста значения ячейки. Следующий пример показывает, как использовать [**CellsHelper.getTextWidth(string, Font, number)**](https://reference.aspose.com/cells/nodejs-cpp/cellshelper/#getTextWidth-string-font-number-) для доступа к ширине текста значения ячейки.
