## **Опция отображения страниц фильтров отчета**
Excel поддерживает создание сводных таблиц, добавление фильтров отчетов и включение опции "Показать страницы фильтров отчета". Aspose.Cells for Node.js via C++ также поддерживает эту функцию для активации опции "Показать страницы фильтров отчета" в созданной сводной таблице.
