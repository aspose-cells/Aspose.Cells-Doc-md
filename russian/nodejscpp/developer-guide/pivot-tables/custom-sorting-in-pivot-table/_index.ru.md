## **Как установить индивидуальную сортировку в сводной таблице с помощью библиотеки Aspose.Cells for Node.js via C++**
Используя API Aspose.Cells for Node.js via C++, вы можете сортировать сводные таблицы по значениям полей. Следующий пример кода загружает пример файла Excel и добавляет три сводных таблицы. Первая сводная таблица без индивидуальной сортировки, вторая отсортирована по строковому полю "Морепродукты", а третья — по столбцовому полю "28/07/2000".
