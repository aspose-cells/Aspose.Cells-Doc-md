## **Как изменить макет сводной таблицы в MS-Excel**
Microsoft Excel позволяет изменять компоновку сводной таблицы с помощью команд меню *Инструменты сводной таблицы > Оформление > Оформление отчета*. Вы можете изменить компоновку в следующих формах:

- Показать в компактной форме
- Показать в контурной форме
- Показать в табличной форме

## **Как изменить макет сводной таблицы с помощью Aspose.Cells for Node.js via C++**
Библиотека Aspose.Cells for Node.js via C++ также предоставляет методы [**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#showInCompactForm--), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#showInOutlineForm--) и [**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#showInTabularForm--) для изменения макета сводной таблицы в этих трех формах.
