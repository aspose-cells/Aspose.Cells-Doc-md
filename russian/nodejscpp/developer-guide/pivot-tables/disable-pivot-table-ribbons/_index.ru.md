{{% alert color="primary" %}}

Отчеты на основе сводных таблиц полезны, но могут быть ошибочными, если целевые пользователи не имеют подробных знаний об Excel для настройки этих отчетов. В таких случаях организации захотят ограничить возможность пользователей изменять отчет, основанный на сводной таблице. Обычно не рекомендуется для каждого пользователя такие функции, как добавление дополнительных фильтров, слайсеров, полей или изменение порядка некоторых элементов в отчете. С другой стороны, пользователи должны иметь возможность обновлять отчет и использовать существующие фильтры или слайсеры. Aspose.Cells for Node.js via C++ предоставил разработчикам возможность ограничивать изменение этих отчетов при их создании. Для этой цели Excel предоставляет функцию отключения ленты сводной таблицы, которая также реализована через Aspose.Cells for Node.js via C++, то есть разработчик может отключить ленту, содержащую элементы управления для изменения этих отчетов.

{{% /alert %}}
