## **Как получить источник данных внешнего подключения сводной таблицы**

Aspose.Cells for Node.js via C++ предоставляет возможность получения внешнего источника данных сводной таблицы. Для этого API предоставляет метод [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) класса [**PivotTable**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/). Метод [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) возвращает объект [**ExternalConnection**](https://reference.aspose.com/cells/nodejs-cpp/externalconnection/). Следующий пример демонстрирует использование метода [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) для получения внешнего источника данных сводной таблицы.
