{{% alert color="primary" %}}

Вы можете установить различные параметры сводной таблицы с помощью Aspose.Cells for Node.js via C++. Один из таких параметров — "Показать для пустых ячеек". Установив этот параметр, все пустые ячейки сводной таблицы отображаются как заданная строка.

{{% /alert %}}

## **Установка параметров сводной таблицы в Microsoft Excel**

Чтобы найти и установить этот параметр в Microsoft Excel:

1. Выберите сводную таблицу и щелкните правой кнопкой мыши.
1. Выберите **Параметры сводной таблицы**.
1. Выберите вкладку **Макет и формат**.
1. Выберите опцию **Показывать пустые ячейки** и укажите строку.

## **Настройка параметров сводной таблицы с помощью Aspose.Cells for Node.js via C++**

Aspose.Cells for Node.js via C++ предоставляет свойства [**PivotTable.setDisplayNullString**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#setDisplayNullString-boolean-) и [**PivotTable.setNullString**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#setNullString-string-) для установки параметра "Показать для пустых ячеек" в сводной таблице.
