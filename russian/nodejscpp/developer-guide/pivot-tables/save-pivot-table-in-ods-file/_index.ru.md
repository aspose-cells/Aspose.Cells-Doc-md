Aspose.Cells for Node.js via C++ предоставляет возможность сохранять сводные таблицы в файлах ODS. Для этого просто преобразуйте рабочую книгу с существующей сводной таблицей или создайте новую и сохраните файл в формате ODS. Перед сохранением обязательно вызовите [**PivotTable.calculateData**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#calculatedata--), чтобы убедиться, что сводная таблица отрисована в итоговом файле ODS. Следующий пример демонстрирует сохранение сводной таблицы в файле ODS.
