## **Возможные сценарии использования**

Aspose.Cells for Node.js via C++ поддерживает пользовательское форматирование шаблона *DBNum*. Например, если значение вашей ячейки 123 и вы задаёте его пользовательское форматирование как [DBNum2][$-804]General, оно будет отображаться как 壹佰贰拾叁. Вы можете задать пользовательский формат ячейки с помощью методов [**Cell.getStyle()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getStyle--) и [**Style.setCustom(string)**](https://reference.aspose.com/cells/nodejs-cpp/style/#setCustom-string-).
