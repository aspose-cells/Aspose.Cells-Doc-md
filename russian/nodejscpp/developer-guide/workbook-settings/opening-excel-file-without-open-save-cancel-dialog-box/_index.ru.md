{{% alert color="primary" %}} 

В этом документе объясняется, как открыть файл Microsoft Excel в браузере без отображения окна диалога Открыть-Сохранить-Отмена. 

Здесь следует отметить, что ограничение безопасности, которое не позволяет прямое скачивание файла, обеспечивается Microsoft (или другими поставщиками браузеров), а не Aspose. Оно введено для блокирования и ограничения потенциально вредоносных файлов, которые могут быть загружены на локальные компьютеры. 

Рискованно разрешать клиентской локальной системе загружать файл без отображения окна диалога Открыть-Сохранить-Отмена для подтверждения загрузки. От Aspose нет никаких вариантов или обходных решений, поскольку это будет большим риском для безопасности.

{{% /alert %}} 

## **Почему это риск для безопасности?**
На следующем изображении показано окно диалога Открыть-Сохранить-Отмена, отображаемое браузером Internet Explorer при попытке загрузить файл.

 Как объяснялось выше, разрешить файлу открываться или запускаться на вашей системе без подтверждения, что вы действительно хотите его открыть, — это риск безопасности. Некоторые файлы содержат вирусы, а некоторые сайты попытаются загрузить вредоносные файлы на ваш компьютер без предупреждения. Поэтому не рекомендуется разрешать загрузку файла без диалогового окна загрузки, чтобы пользователи могли проверить файл и его источник перед загрузкой или запуском. Отключение диалогового окна загрузки делает систему уязвимой к вирусам, троянам и хакерам, которые могут тихо воздействовать на вашу систему. 

## **Открытие файла без окна диалога Открыть-Сохранить-Отмена**
Хотя это большая проблема безопасности, Microsoft все же предоставляет настройки Internet Explorer, позволяющие пользователям отключить запрос на открытие-сохранение-отмену при скачивании файла. 

В Проводнике Windows:

1. На меню **Инструменты** выберите **Параметры папок**.
1. Нажмите вкладку **Типы файлов** в диалоговом окне Параметры папок.
1. Выберите тип файла с расширением XLS.
1. Нажмите **Дополнительно**. 
   Отображается диалоговое окно. Внизу имеется три варианта.
1. Снимите флажок с **Подтверждать открытие после загрузки**.
1. Выберите третий вариант - **Просматривать в том же окне** - чтобы просматривать файл Excel в Internet Explorer без запуска отдельного приложения Microsoft Excel.
