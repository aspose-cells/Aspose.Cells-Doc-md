**Добро пожаловать в Aspose.Cells for Java**

 Aspose.Cells for Java — это библиотека классов программирования, которая позволяет разработчикам программного обеспечения манипулировать и обрабатывать файлы электронных таблиц в своих собственных приложениях.

В сочетании с API и элементами управления графическим интерфейсом Aspose.Cells for Java ускоряет программирование и преобразование Microsoft Excel®.

Aspose.Cells for Java поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые ваша компания использует каждый день. Он также позволяет экспортировать файлы Excel в PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, JSON, обычный текст и популярные форматы изображений, включая TIFF, JPG, PNG, BMP и SVG.
