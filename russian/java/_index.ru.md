**Добро пожаловать в Aspose.Cells for Java**

Aspose.Cells for Java - это классовая библиотека, которая позволяет разработчикам программного обеспечения манипулировать и обрабатывать файлы электронных таблиц в своих собственных приложениях.

В сочетании с API и графическими элементами управления Aspose.Cells for Java ускоряет программирование и преобразование Microsoft Excel®.

Aspose.Cells for Java поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые используются в вашем бизнесе каждый день. Он также позволяет экспортировать файлы Excel в форматы PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, простой текст и популярные форматы изображений, включая TIFF, JPG, PNG, BMP и SVG.
