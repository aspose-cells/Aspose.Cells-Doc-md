{{% alert color="primary" %}}

Microsoft Excel позволяет пользователям добавлять комментарии к ячейкам, чтобы добавить дополнительную информацию и выделить данные. Разработчикам может потребоваться настроить комментарий, чтобы указать параметры выравнивания, направление текста, цвет шрифта и т. д. Aspose.Cells предоставляет API для выполнения этой задачи.

{{% /alert %}}

Aspose.Cells предоставляет[**Shape.TextBody**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#TextBody)свойство цвета шрифта комментария. Следующий пример кода демонстрирует использование[**Shape.TextBody**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#TextBody)свойство для установки направления текста для комментария.
