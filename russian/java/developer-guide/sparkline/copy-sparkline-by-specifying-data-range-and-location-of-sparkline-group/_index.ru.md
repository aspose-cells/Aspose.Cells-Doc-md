Скопируйте спарклайн, указав диапазон данных и расположение группы спарклайнов в MS Excel

Microsoft Excel позволяет копировать спарклайны, указав диапазон данных и расположение группы спарклайнов. Выполните следующие действия, чтобы скопировать Sparkline в другие ячейки.

- Выберите ячейку, содержащую вашу спарклайн.
-  Выбирать**Изменить данные** от**Спарклайн** раздел внутри**Дизайн** вкладка
-  выберите**Изменить местоположение и данные группы...**
- Укажите диапазон данных и местоположение и нажмите OK.
