Копирование мини-графиков, указывая диапазон данных и расположение группы мини-графиков в MS Excel

Microsoft Excel позволяет копировать мини-график, указав диапазон данных и расположение группы мини-графиков. Следуйте этим шагам, чтобы скопировать ваш мини-график в другие ячейки.

- Выберите ячейку, содержащую ваш мини-график.
- Выберите **Редактирование данных** из раздела **Мини-графики** внутри вкладки **Дизайн**
- Выберите **Редактировать расположение и данные группы...**
- Укажите диапазон данных и расположение, а затем нажмите ОК.

## Пример

Aspose.Cells предоставляет метод [**SparklineCollection.add(dataRange, row, column)**](https://reference.aspose.com/cells/java/com.aspose.cells/SparklineCollection) для указания диапазона данных и расположения группы мини-графиков.
