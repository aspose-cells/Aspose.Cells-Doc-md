{{% alert color="primary" %}}

Microsoft Excel предоставляет возможность добавлять **Строку подписи** в рабочие книги Excel. Вы можете добавить строку подписи, нажав на вкладку **Вставка** и затем выбрав **Строка подписи** из группы **Текст**.

Aspose.Cells также предоставляет эту функцию и предоставляет свойство [**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) для этой цели. В этой статье будет объяснено, как использовать это свойство для добавления строки подписи с помощью Aspose.Cells.

{{% /alert %}}
