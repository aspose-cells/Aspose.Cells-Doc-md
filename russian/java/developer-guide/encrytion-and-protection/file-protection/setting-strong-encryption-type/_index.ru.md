Microsoft Excel (97-2007/2010) позволяет шифровать электронные таблицы и защищать их паролем. Он использует алгоритмы, предоставляемые поставщиком услуг шифрования. Поставщик криптографических услуг (или CSP) — это набор криптографических алгоритмов с различными свойствами. CSP по умолчанию — «Совместимость с Office 97/2000». Это CSP с некоторыми общеизвестными проблемами безопасности. Электронные таблицы, защищенные с помощью «слабого шифрования (XOR)» или типа шифрования «Совместимость с Office 97/2000», могут быть легко взломаны.

Чтобы решить эту проблему, используйте один из надежных типов шифрования, предоставляемых Microsoft Excel. Вы можете изменить тип шифрования на самый надежный доступный CSP. Для надежного шифрования требуется минимальная длина ключа 128 бит, например «Microsoft Strong Cryptographic Provider».

Вы также можете зашифровать и защитить паролем файлы Excel с надежным типом шифрования, используя код Aspose.Cells API.

{{% /alert %}}

## **Применение шифрования с помощью Microsoft Excel**

Чтобы внедрить шифрование файлов в Microsoft Excel (например, 2007):

1.  От**Инструменты** меню, выберите**Опции**.
1.  Выберите**Безопасность** вкладка
1.  Введите значение для**Пароль для открытия** поле.
1.  Нажмите**Передовой**.
1. Выберите тип шифрования и подтвердите пароль.
