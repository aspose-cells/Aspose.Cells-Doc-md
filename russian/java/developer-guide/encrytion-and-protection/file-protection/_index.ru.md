{{% alert color="primary" %}}
Microsoft Excel (97 - 365) позволяет зашифровать / защитить паролем свои электронные таблицы. Он использует алгоритмы, предоставленные поставщиком криптографических услуг (CSP). Поставщик криптографических услуг (CSP) - это набор криптографических алгоритмов с различными свойствами. По умолчанию CSP является "Office 97/2000 Compatible" или "Слабое шифрование (XOR)". Также важно выбрать подходящую длину ключа шифрования. Некоторые поставщики криптографических услуг не поддерживают больше чем 40 или 56 бит. Это считается слабым типом шифрования. Но для сильного типа шифрования требуется минимальная длина ключа 128 бит. Microsoft Windows содержит поставщиков криптографических услуг, предлагающих сильные типы шифрования, например, 'Microsoft Strong Cryptographic Provider'. Для представления идеи, для шифрования соединения с Интернет-банковскими системами используется шифрование на 128 бит. Aspose.Cells позволяет вам зашифровывать / защищать паролем ваши файлы Excel с использованием вашего выбранного типа шифрования.

{{% /alert %}}

## **Используя MS Excel**

В MS Excel (например, MS Excel 2003), для реализации настроек шифрования файла, вы можете попробовать:

- Из меню **Сервис** выберите **Параметры**, затем выберите вкладку **Безопасность**.
- Введите **Пароль для открытия** и нажмите **Дополнительно**.
-Выберите тип шифрования и подтвердите пароль.
