##  **Как преобразовать книгу Excel в JSON**

 Не нужно задаваться вопросом, как преобразовать книгу Excel в JSON, потому что библиотека Aspose.Cells Java имеет лучшее решение. Aspose.Cells Java API обеспечивает поддержку преобразования электронных таблиц в формат JSON. Чтобы экспортировать книгу на номер JSON, введите[**СохранитьФормат.JSON**](https://reference.aspose.com/cells/java/com.aspose.cells/SaveFormat) в качестве второго параметра[**Рабочая книга.Сохранить**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20int) ) метод. Вы также можете использовать[**JsonSaveOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonSaveOptions) class, чтобы указать дополнительные настройки для экспорта листа в JSON.
