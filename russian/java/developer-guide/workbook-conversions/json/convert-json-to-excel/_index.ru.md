## **Преобразование JSON в книгу Excel**

 Не нужно задаваться вопросом, как преобразовать JSON в файл Excel, потому что библиотека Aspose.Cells Java имеет лучшее решение. Aspose.Cells Java API обеспечивает поддержку преобразования формата JSON в электронные таблицы. Вы можете использовать[**JsonLoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLoadOptions) class, чтобы указать дополнительные параметры для импорта JSON в Workbook.
