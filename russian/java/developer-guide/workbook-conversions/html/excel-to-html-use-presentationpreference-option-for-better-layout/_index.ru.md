 Aspose.Cells предоставляет полезное свойство,[**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference) который разработчики могут использовать для улучшения макета при сохранении файла Excel Microsoft в формате HTML или MHT. Значение свойства по умолчанию равно**ЛОЖЬ** . Мы рекомендуем установить для этого свойства значение**истинный** чтобы получить более привлекательные презентации из отчета Excel.
