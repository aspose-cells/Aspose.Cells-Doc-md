{{% alert color="primary" %}}

Aspose.Cells предоставляет полезное свойство, [**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference), которое разработчики могут использовать для получения лучшего макета при сохранении файла Microsoft Excel в формате HTML или MHT. Значение по умолчанию свойства **false**. Мы рекомендуем установить это свойство на **true** для получения более красивых презентаций из отчета Excel.

{{% /alert %}}
