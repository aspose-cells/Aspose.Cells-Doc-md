## **Конвертировать Excel в HTML с заголовками**

Aspose.Cells предоставляет возможность экспортировать заголовки при преобразовании Excel в HTML. Для этого вы можете установить свойство [**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings) в **true**. Значение по умолчанию свойства [**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings) - **false**.
