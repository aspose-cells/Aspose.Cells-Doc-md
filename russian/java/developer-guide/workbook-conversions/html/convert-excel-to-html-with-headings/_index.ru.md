## **Преобразование Excel в HTML с заголовками**

Aspose.Cells позволяет экспортировать заголовки при преобразовании Excel в HTML. Для этого можно установить[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings)собственность на**истинный**. Значение по умолчанию[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings)собственность**ЛОЖЬ**.
