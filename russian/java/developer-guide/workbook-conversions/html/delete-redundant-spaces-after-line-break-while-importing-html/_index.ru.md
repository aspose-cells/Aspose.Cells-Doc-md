{{% alert color="primary" %}} 

 Пожалуйста, используйте[HtmlLoadOptions. DeleteRedundantSpaces](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#DeleteRedundantSpaces) свойство и установить его**истинный** чтобы удалить все лишние пробелы, идущие после тега разрыва строки. По умолчанию это свойство**ЛОЖЬ** и избыточные пробелы сохраняются в выходных файлах Excel.

{{% /alert %}}
