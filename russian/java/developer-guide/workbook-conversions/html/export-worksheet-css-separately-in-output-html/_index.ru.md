## **Возможные сценарии использования**

Aspose.Cells предоставляет возможность отдельно экспортировать CSS рабочего листа при преобразовании файла Excel в HTML. Для этой цели используйте свойство HtmlSaveOptions.ExportWorksheetCSSSeparately и установите для него значение true при сохранении файла Excel в формате HTML.

## **Экспорт рабочего листа CSS отдельно в выходной HTML**

В следующем примере кода создается файл Excel, добавляется текст в ячейку B5 красного цвета, а затем сохраняется в формате HTML с использованием свойства HtmlSaveOptions.ExportWorksheetCSSSeparately. ## **Экспорт книги с одним листом в HTML**

Когда книга с несколькими листами преобразуется в HTML с помощью Aspose.Cells, создается файл HTML вместе с папкой, содержащей файлы CSS и несколько файлов HTML. Когда этот файл HTML открывается в браузере, вкладки видны. Такое же поведение требуется для книги с одним листом при ее преобразовании в HTML. Раньше для книг с одним листом не создавалась отдельная папка, а создавался только HTML-файл. Такой HTML-файл не отображает вкладку при открытии в браузере. Excel также создает правильную папку и HTML для отдельных листов, и, следовательно, такое же поведение реализуется с использованием Aspose.Cells.
