{{% alert color="primary" %}} 

Обычно макет тегов div игнорируется при загрузке HTML в объект книги Excel. Однако, если вы хотите, чтобы макет тегов div не игнорировался, установите свойство [HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag) в **true**. Значение по умолчанию этого свойства - **false**.

{{% /alert %}}
