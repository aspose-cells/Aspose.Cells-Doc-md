## Возможные сценарии использования

Это очень распространенный сценарий, когда нам нужно экспортировать в HTML только область печати, т. е. выбранный диапазон ячеек, а не весь лист. Эта функция уже доступна для рендеринга PDF, однако теперь вы можете выполнять эту задачу и для HTML. Сначала установите область печати в объекте настройки страницы рабочего листа. Позднее использование[**Хтмлсавеоптионс. Экспортпринтареаонли**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportPrintAreaOnly)свойство для экспорта только выбранного диапазона.
