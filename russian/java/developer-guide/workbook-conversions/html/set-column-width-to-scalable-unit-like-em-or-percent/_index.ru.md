Создание HTML-файла из электронной таблицы является довольно распространенным. Размеры столбцов определены в "pt", которые работают во многих случаях. Однако может возникнуть ситуация, когда этот фиксированный размер не требуется. Например, если ширина контейнерной панели равна 600 пикселям, где отображается данная HTML-страница. В этом случае может появиться горизонтальная полоса прокрутки, если ширина созданной таблицы больше. Требовалось, чтобы этот фиксированный размер был изменен на масштабируемую единицу, такую как em или процент, для более качественного представления. В следующем образце кода можно использовать значение [**HtmlSaveOptions.WidthScalable**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#WidthScalable), установленное в **true**, для создания масштабируемой ширины.
