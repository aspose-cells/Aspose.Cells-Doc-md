{{% alert color="primary" %}} 

Aspose.Cells теперь поддерживает расширение текста справа налево при экспорте файла Excel в HTML. Эта функция реализована начиная с версии 8.9.0.0. Теперь, если ваш исходный файл Excel содержит какой-либо текст, который расширяется справа налево, то Aspose.Cells правильно экспортирует его в HTML.

{{% /alert %}} 
## **Расширение текста справа налево при экспорте файла Excel в HTML**
 Следующий пример кода преобразует[образец эксель файла](5472562.xlsx) в HTML. На этом снимке экрана показано, как выглядит образец Excel в Microsoft Excel 2013.

![дело:изображение_альтернативный_текст](expanding-text-from-right-to-left-while-exporting-excel-file-to-html_1.png)

 На этом снимке экрана показано[вывод HTML, созданный в более старой версии](5472570).

![дело:изображение_альтернативный_текст](expanding-text-from-right-to-left-while-exporting-excel-file-to-html_2.png)

 На этом снимке экрана показано[вывод HTML, сгенерированный с более новой версией](5472563).

![дело:изображение_альтернативный_текст](expanding-text-from-right-to-left-while-exporting-excel-file-to-html_3.png)

Как вы можете видеть на снимках экрана, в более новой версии выровненный по правому краю текст правильно расширяется по левому краю, как Microsoft Excel.
