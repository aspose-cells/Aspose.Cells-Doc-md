Вы можете экспортировать условное форматирование данных, цветовую шкалу и набор значков при преобразовании файла Excel в HTML. Эта функция частично поддерживается Microsoft Excel, но полностью поддерживается Aspose.Cells.

{{% /alert %}} 
## **Экспорт условного форматирования данных, цветовой шкалы и набора значков при преобразовании Excel в HTML**
На следующем снимке экрана показан [образец файла Excel](5473447.xlsx) с условным форматированием данных, цветовой шкалой и набором значков. Вы можете загрузить [образец файла Excel](5473447.xlsx) по указанной ссылке.

![todo:image_alt_text](conversion_1.png)

На следующем снимке экрана показан сгенерированный файл HTML Aspose.Cells, отображающий условное форматирование данных, цветовую шкалу и набор значков. Как видно, он выглядит точно так же, как [образец файла Excel](5473447.xlsx).
