HTML может иметь различные форматы тегов для пустых тегов, таких как \<td>\</td> или же \<td/>. Aspose.Cells теперь поддерживает оба этих формата, тогда как раньше он поддерживал только \<td>\</td>как теги. Эту функцию можно протестировать, преобразовав прикрепленный образец HTML-файла в файл Excel. Образец HTML-файла и выходные файлы можно загрузить для тестирования по следующим ссылкам.
