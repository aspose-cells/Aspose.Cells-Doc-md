Средства PDF логической структуры обеспечивают механизм включения информации, касающейся структуры содержимого документа, в файл PDF. Aspose.Cells сохраняет информацию о структуре из документа Excel Microsoft, такую как ячейка, строка, таблица, рабочий лист, изображение, фигура, верхний/нижний колонтитул и т. д.

 С опцией[PdfSaveOptions.ExportDocumentStructure](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setExportDocumentStructure-boolean-), вы можете сохранить в файл с тегом PDF с экспортированной структурой документа.
