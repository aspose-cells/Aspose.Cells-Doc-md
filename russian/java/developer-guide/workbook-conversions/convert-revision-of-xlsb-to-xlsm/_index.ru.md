{{% alert color="primary" %}} 

Aspose.Cells теперь поддерживает полное преобразование версий файла XLSB в файл XLSM. Ревизии находятся внутри пути \xl\revisions. Вы можете просмотреть их, изменив расширение файла XLSB на ZIP. Путь \xl\revisions содержит файлы с расширением .bin.

Когда вы конвертируете файл XLSB в файл XLSM с помощью Aspose.Cells, эти файлы .bin успешно преобразуются в файлы .xml, как показано на этих двух снимках экрана.

{{% /alert %}}
