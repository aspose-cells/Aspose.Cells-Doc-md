{{% alert color="primary" %}} 

Aspose.Cells поддерживает изменение существующего подключения к данным SQL. В статье объясняется, как использовать Aspose.Cells для изменения различных свойств подключения к данным SQL.

Вы можете добавить или просмотреть соединения данных внутри Microsoft Excel, выполнив следующие действия.**Данные > Соединения** команда меню.

Точно так же Aspose.Cells предоставляет средства для доступа и изменения подключений к данным с помощью коллекции Workbook.getDataConnections().
