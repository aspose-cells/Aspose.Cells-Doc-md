{{% alert color="primary" %}} 

Иногда может потребоваться получить информацию о внешних источниках данных. Например, вы можете захотеть увидеть данные соединения SQL. Эта информация может включать в себя любые типы данных, необходимые для подключения к серверу SQL, например URL-адрес сервера, имя пользователя, имя базовой таблицы, запрос SQL, тип запроса, расположение таблицы и т. д.

Aspose.Cells предоставляют несколько полезных вызовов для получения таких сведений из корпоративных баз данных, связанных с шаблонами файлов Excel.

{{% /alert %}} 
## **Получение сведений о внешних источниках данных**
В следующем примере показано, как получить подключение к базе данных и другие сведения. В примере используется простой файл Excel, содержащий ссылки на внешний источник данных (SQL Server).

Когда код запускается, сведения о подключении выводятся на консоль.
