{{% alert color="primary" %}} 

Иногда вам может потребоваться извлечь информацию о внешних источниках данных. Например, вам может понадобиться посмотреть данные соединения с SQL. Эта информация может включать любые типы данных, необходимых для подключения к серверу SQL, например URL сервера, имя пользователя, имя таблицы, SQL запрос, тип запроса, расположение таблицы и т. д.

Aspose.Cells предоставляет несколько полезных вызовов для извлечения таких деталей из корпоративных баз данных, связанных с шаблонами файлов Excel.

{{% /alert %}} 
## **Извлечение деталей внешних источников данных**
Следующий пример показывает, как получить подробности о подключении к базе данных и другие детали. Пример использует простой файл Excel, содержащий ссылки на внешний источник данных (SQL Server).

При запуске кода детали соединения печатаются в консоль.
