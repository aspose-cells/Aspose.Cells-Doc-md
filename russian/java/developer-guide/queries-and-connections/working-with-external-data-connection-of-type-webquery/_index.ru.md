{{% alert color="primary" %}} 

Вы можете получить доступ к внешнему соединению данных любого типа, используя коллекцию [Workbook.DataConnections](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections). Одним из таких типов соединения данных является WebQuery. В этой статье показано, как работать с соединением данных WebQuery. Вы можете создать соединение данных WebQuery в Microsoft Excel, используя меню **Данные** > **Из Интернета**.

{{% /alert %}}
