{{% alert color="primary" %}} 

 Вы можете получить доступ к внешнему подключению для передачи данных любого типа, используя[Workbook.DataConnections](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections) коллекция. Одним из типов такого подключения к данным является WebQuery. В этой статье показано, как работать с подключением к данным WebQuery. Вы можете создать подключение к данным WebQuery в Microsoft Excel, используя**Данные** > **Из Интернета** меню.
