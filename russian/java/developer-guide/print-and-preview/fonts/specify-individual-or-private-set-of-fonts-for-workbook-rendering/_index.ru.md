## **Возможные сценарии использования**

Как правило, вы указываете каталог шрифтов или список шрифтов для всех книг, но иногда вам нужно указать индивидуальный или частный набор шрифтов для ваших книг. Aspose.Cells предоставляет[**Индивидуальные настройки шрифтов**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs)class, который можно использовать для указания индивидуального или закрытого набора шрифтов для вашей книги.

## **Укажите индивидуальный или частный набор шрифтов для визуализации рабочей книги**
