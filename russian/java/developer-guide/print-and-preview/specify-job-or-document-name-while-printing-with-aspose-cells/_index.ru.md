{{% alert color="primary" %}}

 Вы можете указать имя задания или документа при печати книги или рабочего листа с помощью[**Рабочая книгаВизуализация**](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookRender) или же[**Листрендеринг**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender) объекты. Aspose.Cells обеспечивает[**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ) и[**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) методы, которые можно использовать для указания имени задания при печати книги или рабочего листа.

{{% /alert %}}

## **Укажите имя задания или документа при печати с помощью Aspose.Cells.**

 Пример кода загружает исходный файл Excel, а затем отправляет его на принтер, указав имя задания или документа с помощью[**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ) и[**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) методы. На снимке экрана показано, как имя задания выглядит в очереди печати.
