{{% alert color="primary" %}}

Иногда вам нужно представить изображения рабочего листа в приложениях или на веб-страницах. Например, вам может понадобиться вставить изображения в документ Word, файл PDF, презентацию PowerPoint или какой-либо другой документ. По сути, вы хотите визуализировать рабочий лист как изображение, чтобы его можно было вставить в другие приложения. Aspose.Cells API-интерфейсы позволяют преобразовывать Microsoft рабочие листы Excel в изображения.

{{% /alert %}}

[**Листрендеринг**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender)class способен преобразовывать рабочий лист в файл изображения с любыми заданными атрибутами, например форматом изображения, листами с разбивкой на страницы и т. д. Поддерживается несколько форматов изображений, включая BMP, GIF, JPG, TIFF и EMF.

 Когда вы используете функцию преобразования листа в изображение, выходное изображение по умолчанию имеет белое/пустое пространство, то есть границу. Вы можете удалить это. Установите верхнее, левое, нижнее и правое поля настройки страницы для исходного листа на 0 и укажите[**Имажеорпринтоптионс**](https://reference.aspose.com/cells/java/com.aspose.cells/ImageOrPrintOptions)атрибуты соответственно.
