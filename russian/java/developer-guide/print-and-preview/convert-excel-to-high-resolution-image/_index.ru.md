С ростом популярности экранов высокого разрешения изображения, создаваемые по умолчанию с разрешением 96 DPI, выглядят размытыми и нечеткими. Чтобы обеспечить четкость на дисплеях высокого разрешения, важно генерировать изображения с более высоким DPI. Aspose.Cells позволяет устанавливать [**ImageOrPrintOptions.HorizontalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setHorizontalResolution-int-) и [**ImageOrPrintOptions.VerticalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setVerticalResolution-int-), что дает возможность создавать изображения высокого качества из Excel-файлов, которые выглядят четко на дисплеях высокой четкости.
