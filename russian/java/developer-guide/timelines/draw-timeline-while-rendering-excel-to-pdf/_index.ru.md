## **Отображение временной шкалы при преобразовании Excel в PDF**
Если у вас есть файл Excel, к которому применена временная шкала и вы хотите экспортировать Excel в PDF с настройками временной шкалы, Aspose.Cells for Java теперь поддерживает эту функцию по умолчанию. Просто экспортируйте файл Excel с временной шкалой в PDF, и сгенерированный PDF покажет примененную временную шкалу.

Приведенный ниже образец кода загружает [образец Excel-файла](input.xlsx), содержащий существующую временную шкалу. Затем он сохраняет книгу как [файл PDF на выходе](out.pdf). На следующем снимке экрана сравниваются исходный файл Excel и сгенерированный PDF-файл.
