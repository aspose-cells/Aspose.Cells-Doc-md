## **Возможные сценарии использования**

Вместо настройки фильтров для отображения дат можно использовать временную шкалу сводной таблицы - динамический параметр фильтра, который позволяет легко фильтровать по дате/времени и увеличивать интересующий период с помощью элемента управления ползунком. Microsoft Excel позволяет создавать временную шкалу, выбрав сводную таблицу, а затем щелкнув *Вставить > Временная шкала*. Aspose.Cells для Java также позволяет создавать временную шкалу с помощью метода [**Worksheet.getTimelines.add()**].
