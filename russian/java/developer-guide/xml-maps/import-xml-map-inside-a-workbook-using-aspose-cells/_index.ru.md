{{% alert color="primary" %}}

 Aspose.Cells позволяет импортировать XML-карту внутри книги с помощью[**Книга.importXml()**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#importXml(java.lang.String,%20java.lang.String,%20int,%20int)) метод. Вы можете импортировать XML-карту с помощью Excel Microsoft, выполнив следующие действия.

-  Выбирать**Разработчик** вкладка
-  Нажмите**импорт** в разделе XML и выполните необходимые действия.

 Вам нужно будет предоставить данные XML для завершения импорта. Вот[образец XML-данных](5472501.txt) которые вы можете использовать для тестирования.

{{% /alert %}}
