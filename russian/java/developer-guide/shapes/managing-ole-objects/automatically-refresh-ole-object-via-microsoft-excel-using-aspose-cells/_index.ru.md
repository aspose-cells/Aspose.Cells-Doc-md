{{% alert color="primary" %}} 

Aspose.Cells предоставляет свойство [OleObject.AutoLoad](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad) для обновления объекта OLE при открытии файла Excel в Microsoft Excel. Благодаря этому свойству объект OLE будет отображать правильное изображение OLE, сгенерированное Microsoft Excel.

{{% /alert %}} 
## **Автоматическое обновление объекта OLE через Microsoft Excel с помощью Aspose.Cells**
В следующем образце кода загружается [образец файла Excel](5473423.xlsx), который содержит нереальное изображение OLE. Фактически объект OLE является документом Microsoft Word, но в образце файла Excel отображается изображение животного вместо изображения Microsoft Word. Однако, если вы откроете [результирующий файл Excel](5473429.xlsx), вы увидите, что Microsoft Excel отображает правильное изображение OLE.
