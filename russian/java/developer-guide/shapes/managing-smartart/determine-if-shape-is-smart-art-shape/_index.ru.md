## **Возможные сценарии использования**

Формы Smart Art - это специальные формы в Microsoft Excel, которые позволяют вам автоматически создавать сложные диаграммы. Вы можете определить, является ли форма формой Smart Art или обычной формой с помощью свойства [**Shape.IsSmartArt**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#IsSmartArt).

## **Определение, является ли форма формой Smart Art**

Приведенный ниже образец кода загружает содержащий форму умного изображения образец Excel-файл, показанный на этом снимке экрана. Затем он печатает значение свойства [**Shape.IsSmartArt**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#IsSmartArt) для первой формы.
