Aspose.Cells предоставляет обширные возможности для управления формами в электронных таблицах. Иногда требуется получить точки соединения формы для выравнивания или размещения форм в нужном месте. Для этой цели требуются все точки соединения. Следующий код может быть использован для получения списка точек соединения формы, где используется функция [**Shape.GetConnectionPoints()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ConnectionPoints).
