## **Возможные сценарии использования**

 Aspose.Cells позволяет установить поля любой формы или комментария с помощью[**Форма.TextBody.TextAlignment**](https://reference.aspose.com/cells/java/com.aspose.cells/fontsettingcollection#TextAlignment) имущество. Это свойство возвращает объект[**Выравнивание шейптекста**](https://reference.aspose.com/cells/java/com.aspose.cells/ShapeTextAlignment) класс, который имеет разные свойства, например[**TopMarginPt**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#TopMarginPt), [**LeftMarginPt**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#LeftMarginPt), [**Нижняя маржаPt**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#BottomMarginPt), [**RightMarginPt**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#RightMarginPt)и т. д., которые можно использовать для установки верхнего, левого, нижнего и правого полей.
