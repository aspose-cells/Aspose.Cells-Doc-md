## **Возможные сценарии использования**
Вы можете добавить словесное искусство текста с встроенными стилями с помощью Aspose.Cells. Пожалуйста, используйте метод [ShapeCollection.AddWordArt()](https://reference.aspose.com/cells/java/com.aspose.cells/shapecollection#addWordArt\(int,%20java.lang.String,%20int,%20int,%20int,%20int,%20int,%20int\)) для этой цели.
