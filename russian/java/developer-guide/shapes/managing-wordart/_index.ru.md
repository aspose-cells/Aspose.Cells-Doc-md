{{% alert color="primary" %}}

Используйте WordArt, чтобы добавить специальные текстовые эффекты в электронные таблицы. Например, растяните заголовок по верхней части файла, украсьте текст и сделайте его подходящим для предустановленной формы или примените текст к листу Excel в качестве фонового водяного знака. WordArt становится объектом, который можно перемещать или позиционировать в электронных таблицах, чтобы добавить украшение.

{{% /alert %}}
