## **Возможные сценарии использования**
Текстовые поля могут иметь теги, которые можно заменить некоторым текстом во время выполнения, чтобы настроить их в соответствии с требованиями. Теги могут быть некоторой меткой, заключенной в угловые скобки '<' and '>. В одном текстовом поле может быть несколько тегов. Образец файла можно скачать по следующей ссылке.
