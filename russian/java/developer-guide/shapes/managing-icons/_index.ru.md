## Добавить значки на рабочий лист в Aspose.Cells

 Если вам нужно использовать[Aspose.Cells](https://products.aspose.com/cells/) чтобы добавить «значки» в файл Excel, этот документ может оказать вам некоторую помощь.

Интерфейс Excel, соответствующий операции вставки значка, выглядит следующим образом:

![](1.png)

- Выберите положение значка, который будет вставлен на рабочий лист.
-  Щелчок левой кнопкой мыши*Вставлять*->*Иконки*
- В открывшемся окне выберите значок в красном прямоугольнике на рисунке выше
-  Щелчок левой кнопкой мыши*Вставлять*, он будет вставлен в файл Excel.

Эффект следующий:

![](2.png)
