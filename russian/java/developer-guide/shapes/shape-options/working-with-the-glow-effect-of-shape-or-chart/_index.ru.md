## **Возможные сценарии использования**
Aspose.Cells предоставляет свойство Shape.Glow для работы с эффектом свечения формы или диаграммы. Оно содержит следующие подсвойства, которые можно установить для достижения различных результатов в соответствии с вашими требованиями.

- GlowEffect.Size
- GlowEffect.Transparency
