## **Возможные сценарии использования**
Aspose.Cells предоставляет свойство Shape.Glow для работы с эффектом свечения фигуры или диаграммы. Он содержит следующие подсвойства, которые вы можете настроить для достижения различных результатов в соответствии с вашими требованиями.

- GlowEffect.Size
- GlowEffect.Прозрачность

На следующем снимке экрана показан интерфейс Excel Microsoft для установки эффекта свечения формы.
