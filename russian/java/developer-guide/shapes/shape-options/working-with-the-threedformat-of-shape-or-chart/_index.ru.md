## **Возможные сценарии использования**
Aspose.Cells предоставляет свойство [Shape.ThreeDFormat](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ThreeDFormat) для работы с трехмерным форматом формы или диаграммы. Оно содержит различные подсвойства, которые можно установить, чтобы достичь различных результатов в соответствии с вашими требованиями.

На следующем снимке экрана показан интерфейс Microsoft Excel для установки трехмерного формата формы.
