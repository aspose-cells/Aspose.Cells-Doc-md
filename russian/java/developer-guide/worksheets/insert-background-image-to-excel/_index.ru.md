{{% alert color="primary" %}} 

Вы можете сделать рабочий лист более привлекательным, добавив изображение в качестве фона рабочего листа. Эта функция может быть весьма эффективной, если у вас есть специальная корпоративная графика, которая добавляет намек на фон, не заслоняя данные на листе. Вы можете установить фоновое изображение для листа, используя Aspose.Cells API.

{{% /alert %}} 

## **Настройка фона листа в Microsoft Excel**

Чтобы установить фоновое изображение листа в Microsoft Excel (например, Microsoft Excel 2019):

1.  От**Макет страницы** меню, найти**Настройка страницы** вариант, а затем нажмите кнопку**Фон** вариант.
1. Выберите изображение, чтобы установить фоновое изображение листа.
