## **Возможные сценарии использования**
 Вы можете защитить паролем проект рабочей книги VBA (Visual Basic для приложений) с помощью Aspose.Cells, используя[VbaProject.protect()](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#protect\(boolean,%20java.lang.String\)) метод.
## **Образец кода**
 Следующий пример кода загружает[образец файла Excel](43352069.xlsm), получает доступ к своему проекту VBA и защищает его паролем. Наконец, он сохраняет его как[выходной файл Excel](43352070.xlsm).
