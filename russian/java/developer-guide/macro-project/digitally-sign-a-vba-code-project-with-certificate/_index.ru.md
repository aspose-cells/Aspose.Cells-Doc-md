{{% alert color="primary" %}}

 Вы можете подписать свой проект кода VBA цифровой подписью, используя Aspose.Cells с его[**Книга.VbaProject.Sign()**](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#sign(com.aspose.cells.DigitalSignature)) метод. Выполните следующие действия, чтобы проверить, имеет ли ваш файл Excel цифровую подпись сертификата.

-  Нажмите**Visual Basic** от**Разработчик** вкладка, чтобы открыть**Visual Basic для приложений IDE**
-  Нажмите**Инструменты** > **Цифровые подписи...** из**Visual Basic для приложений IDE**

 и он покажет**Форма цифровой подписи** показывает, подписан ли документ цифровой подписью с сертификатом или нет.

{{% /alert %}}
