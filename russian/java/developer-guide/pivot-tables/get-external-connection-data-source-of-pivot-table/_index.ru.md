## **Получить источник данных внешнего подключения сводной таблицы**

Aspose.Cells предоставляет возможность получить внешний источник данных соединения сводной таблицы. Для этого API предоставляет[**Внешний источник данных соединения**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)собственность[**сводная таблица**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable)учебный класс.[**Внешний источник данных соединения**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)свойство возвращает[**Внешнее соединение**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection)объект. Следующий фрагмент кода демонстрирует использование[**Сводная таблица.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) свойство, чтобы получить внешний источник данных соединения сводной таблицы.
