{{% alert color="primary" %}}

Вы можете установить различные параметры сводной таблицы, используя Aspose.Cells. Одним из таких параметров является «Для отображения пустых ячеек». Установка этого параметра означает, что все пустые ячейки в сводной таблице отображаются с указанной строкой.

Чтобы найти и установить этот параметр в Microsoft Excel:

1. Выберите сводную таблицу и щелкните ее правой кнопкой мыши.
1.  Выбирать**Параметры сводной таблицы**.
1.  Выберите**Макет и формат** вкладка
1.  Выберите**Для пустых ячеек показать** вариант и укажите строку.

![дело:изображение_альтернативный_текст](setting-pivot-table-option-for-empty-cells-show_1.png)

{{% /alert %}}

Aspose.Cells обеспечивает[**Сводная таблица.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString) а также[**Сводная таблица.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString)свойства для установки опции сводной таблицы "Для пустых ячеек показывать".
