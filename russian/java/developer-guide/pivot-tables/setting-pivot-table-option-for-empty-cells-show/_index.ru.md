## **Введение**

Вы можете устанавливать различные параметры сводной таблицы с помощью Aspose.Cells. Одним из таких параметров является "Показывать пустые ячейки". Установка этого параметра означает, что все пустые ячейки в сводной таблице отображаются с указанной строкой.

Чтобы найти и установить этот параметр в Microsoft Excel:

1. Выберите сводную таблицу и щелкните правой кнопкой мыши.
1. Выберите **Параметры сводной таблицы**.
1. Выберите вкладку **Макет и формат**.
1. Выберите опцию **Показывать пустые ячейки** и укажите строку.

## **Как настроить параметры сводной таблицы - Показывать пустые ячейки**

Aspose.Cells предоставляет свойства [**PivotTable.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString) и [**PivotTable.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString) для установки параметра сводной таблицы "Показывать пустые ячейки".
