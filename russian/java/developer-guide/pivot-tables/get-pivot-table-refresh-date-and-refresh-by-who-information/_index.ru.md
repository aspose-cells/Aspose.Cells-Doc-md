{{% alert color="primary" %}}

Aspose.Cells теперь поддерживает получение даты обновления и информации о том, кем она обновлялась из книги.

{{% /alert %}}

[**PivotTable.getRefreshDate**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshDate) возвращает дату последнего обновления сводного отчета. Аналогично свойство [**PivotTable.getRefreshByWho**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshedByWho) возвращает имя пользователя, который последний раз обновлял отчет. Приведенный ниже пример демонстрирует эту функцию, и образец файла можно загрузить по следующей ссылке.
