Aspose.Cells теперь поддерживает получение даты обновления и информации об обновлении кем из книги.

{{% /alert %}}

[**сводная таблица.getRefreshDate**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshDate) возвращает дату последнего обновления отчета сводной таблицы. Сходным образом[**Сводная таблица.getRefreshByWho**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshedByWho) Свойство возвращает имя пользователя, который обновлял отчет в последний раз.
