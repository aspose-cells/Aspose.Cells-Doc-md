Aspose.Cells предоставляет возможность сохранения сводной таблицы в файлах ODS. Для этого достаточно конвертировать книгу с существующей сводной таблицей или создать новую сводную таблицу и сохранить файл в формате ODS. Обязательно вызовите [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData--) перед сохранением, чтобы убедиться, что сводная таблица отображается в выходном файле ODS. В следующем фрагменте кода демонстрируется сохранение сводной таблицы в файле ODS.
