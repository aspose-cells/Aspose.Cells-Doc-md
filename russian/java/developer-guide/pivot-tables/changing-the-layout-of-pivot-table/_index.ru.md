{{% alert color="primary" %}}

Microsoft Excel позволяет вам изменять макет сводной таблицы с использованием команд *PivotTable Tools > Design > Report* Layout в меню. Вы можете изменить макет в этих трех формах

- Показать в компактной форме
- Показать в контурной форме
- Показать в табличной форме

Aspose.Cells также предоставляет методы [**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm--), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm--) и [**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm--) для изменения макета сводной таблицы в этих трех формах.

{{% /alert %}}
