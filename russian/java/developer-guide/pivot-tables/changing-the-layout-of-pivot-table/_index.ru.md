{{% alert color="primary" %}}

 Microsoft Excel позволяет изменить макет сводной таблицы с помощью*Инструменты сводной таблицы > Дизайн > Отчет* Команды меню макета. Вы можете изменить макет в этих трех формах

- Показать в компактной форме
- Показать в форме схемы
- Показать в табличной форме

 Aspose.Cells также предоставляет[**Сводная таблица.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm()), [**Сводная таблица.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm() ) и[**Сводная таблица.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm()) методы изменения макета сводной таблицы в этих трех формах.

{{% /alert %}}
