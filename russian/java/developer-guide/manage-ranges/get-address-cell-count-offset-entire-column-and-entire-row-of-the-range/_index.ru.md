## **Возможные сценарии использования**

Aspose.Cells предоставляет объект [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/Range), который имеет различные утилитарные методы, облегчающие пользователю работу с диапазонами Excel. В этой статье приведено описание использования следующих методов или свойств объекта [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/Range).

- **Адрес**

Получает адрес диапазона.

- **Количество ячеек**

Получает общее количество ячеек в диапазоне.

- **Смещение**

Получает диапазон смещения.

- **Весь столбец**

Получает объект Range, представляющий весь столбец (или столбцы), содержащий указанный диапазон.

- **Вся строка**

Получает объект Range, представляющий всю строку (или строки), содержащий указанный диапазон.
