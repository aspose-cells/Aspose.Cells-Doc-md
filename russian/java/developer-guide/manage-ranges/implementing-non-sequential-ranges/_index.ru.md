Обычно [именованные диапазоны] (/cells/ru/java/named-ranges/) являются прямоугольными, с непрерывными и смежными ячейками. Но иногда может потребоваться использовать не последовательный диапазон ячеек, в котором ячейки не являются смежными. Aspose.Cells поддерживает создание именованного диапазона с несмежными ячейками. Для этого API предоставляет свойство [**Name.RefersTo**](https://reference.aspose.com/cells/java/com.aspose.cells/name#RefersTo).
