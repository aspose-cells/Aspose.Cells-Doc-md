 Обычно,[именованные диапазоны](/cells/ru/java/named-ranges/) имеют прямоугольную форму с непрерывными ячейками, примыкающими друг к другу. Но иногда вам может понадобиться использовать непоследовательный диапазон ячеек, в котором ячейки не являются смежными. Aspose.Cells поддерживает создание именованного диапазона с несмежными ячейками. Для этого API предоставляет[**Имя.RefersTo**](https://reference.aspose.com/cells/java/com.aspose.cells/name#RefersTo) имущество.
