*ФормулаТекст* является функцией Excel 2013 и более поздних версий. Он не поддерживается предыдущими версиями, такими как Excel 2010 или 2007 и т. д. Как следует из названия, он печатает текст формулы, присутствующей в данной ячейке. Эта статья покажет вам, как использовать эту функцию, используя Aspose.Cells.

{{% /alert %}}
