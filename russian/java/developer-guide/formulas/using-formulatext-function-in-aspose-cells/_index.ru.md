{{% alert color="primary" %}} 

*FormulaText* - это функция Excel 2013 и более поздних версий. Она не поддерживается предыдущими версиями, такими как Excel 2010 или 2007 и т. д. Как следует из названия, она печатает текст формулы, которая присутствует в заданной ячейке. В этой статье будет показано, как использовать эту функцию с использованием Aspose.Cells.

{{% /alert %}}
