## **Возможные сценарии использования**

Aspose.Cells поддерживает пользоватский шаблон форматирования *DBNum*. Например, если значение вашей ячейки равно 123, и вы указываете его пользовательское форматирование как [DBNum2][$-804]General, то оно будет отображаться как 壹佰贰拾叁. Вы можете указать свое пользовательское форматирование ячейки, используя методы [**Cell.getStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getStyle--) и [**Style.setCustom()**](https://reference.aspose.com/cells/java/com.aspose.cells/style/#setCustom-java.lang.String-).
