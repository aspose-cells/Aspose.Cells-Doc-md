{{% alert color="primary" %}} 

Условные форматирования 2-цветной шкалы и 3-цветной шкалы добавляются таким же образом, за исключением того, что они отличаются свойством [ FormatCondition.ColorScale.Is3ColorScale](https://reference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale). Это свойство равно **false** для 2-цветной шкалы и **true** для условных форматирований 3-цветной шкалы.

{{% /alert %}}
