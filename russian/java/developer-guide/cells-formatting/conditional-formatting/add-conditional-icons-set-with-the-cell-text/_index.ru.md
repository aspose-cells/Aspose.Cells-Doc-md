{{% alert color="primary" %}} 

Иногда вы хотите добавить условные значки рядом с текстом в ячейке, чтобы сделать данные более значимыми для читателей. Вы хотите использовать некоторые типы значков условного форматирования, но без применения условного форматирования к ячейкам. Aspose.Cells поддерживает эту функцию.

{{% /alert %}} 

Ниже приведен пример, который создает файл XLSX с нуля, добавляя условные значки в ячейки с текстом без применения условного форматирования.

Когда код выполняется, изображения из условного набора значков добавляются в область ячеек «B2:C4», как показано ниже.
