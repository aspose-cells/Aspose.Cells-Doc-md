{{% alert color="primary" %}} 

Иногда вам нужно добавить условные значки рядом с текстом в ячейке, чтобы сделать данные более понятными для читателей. Вы хотите использовать некоторые типы значков условного форматирования, но без применения условного форматирования к ячейкам. Aspose.Cells поддерживает эту функцию.

{{% /alert %}} 

Ниже приведен образец, который создает файл XLSX с нуля, добавляя условные значки к ячейкам с текстом без применения условного форматирования.

При выполнении кода изображения из набора условных значков добавляются в область ячейки "B2:C4", как показано ниже.
