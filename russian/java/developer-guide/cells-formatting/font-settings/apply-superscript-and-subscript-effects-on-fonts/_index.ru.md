## **Работа с надстрочным и подстрочным индексами**

Применяйте эффект надстрочного, используя свойство [**setSuperscript**](https://reference.aspose.com/cells/java/com.aspose.cells/font#IsSuperscript) объекта [**Font**](https://reference.aspose.com/cells/java/com.aspose.cells/Font). Для применения подстрочного используйте метод [**setSubscript**](https://reference.aspose.com/cells/java/com.aspose.cells/font#IsSubscript) объекта [**Font**](https://reference.aspose.com/cells/java/com.aspose.cells/Font).
