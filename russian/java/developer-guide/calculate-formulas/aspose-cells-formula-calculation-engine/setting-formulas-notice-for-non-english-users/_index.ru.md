{{% alert color="primary" %}} 

Aspose.Cells поддерживает большинство формул/функций, входящих в состав Microsoft Excel. Разработчики могут использовать эти формулы с помощью API или [дизайнерских электронных таблиц](/cells/ru/java/what-is-a-designer-spreadsheet/). Aspose.Cells поддерживает огромный набор математических, строковых, булевых, дата/время, статистических, базы данных, поисковых и ссылочных формул. Формулы должны быть указаны с использованием английского (US) стиля.

Есть два совета, которые не англоговорящим пользователям нужно соблюдать при создании формул с помощью Aspose.Cells:

1. Формулы должны быть введены на английском языке.
   Например, используйте английское "=SUM()", а не немецкое "=SUMME()".
1. Всегда используйте запятую (,) для разделения параметров функций.
   Для некоторых языковых параметров или настроек разделитель для параметров функций является точка с запятой, но Aspose.Cells использует английский стиль запятой. Например, используйте "=IF (C5=0,0,C3/C4)", а не "=IF(C5=0;0;C3/C4)"
