## **Возможные сценарии использования**

Когда вы делитесь книгой, вы получаете опцию, указывающую ***Сохранить историю изменений на N дней***, как показано на следующем скриншоте. Вы можете обновить количество дней для сохранения истории с помощью Aspose.Cells с помощью свойства [**WorksheetCollection.RevisionLogs.DaysPreservingHistory**](https://reference.aspose.com/cells/java/com.aspose.cells/revisionlogcollection#DaysPreservingHistory).
