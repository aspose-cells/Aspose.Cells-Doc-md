 Aspose.Cells предоставляет[**Workbook.ContentTypeProperties.Add**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypepropertycollection#add(java.lang.Object)) для добавления пользовательских свойств ContentTypeProperties в файл Excel. Вы также можете сделать это свойство необязательным, установив[**ContentTypeProperty.IsNillable**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypeproperty#IsNillable) собственность на**истинный**. В следующем фрагменте кода показано добавление необязательных настраиваемых свойств ContentTypeProperties в файл Excel. На следующем изображении показаны оба свойства, которые были добавлены в примере кода.
