{{% alert color="primary" %}}

Aspose.Cells поддерживает преобразование Microsoft, ориентированного на Excel, HTML в формат XLS/XLSX. Это означает, что HTML, сгенерированный Microsoft Excel, можно преобразовать обратно в формат XLS/XLSX, используя Aspose.Cells.

 Точно так же, если есть какой-то простой HTML, Aspose.Cells может преобразовать его в HTML Rich Text. Aspose.Cells предоставляет[**Cell.setHtmlString()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#HtmlString) свойство, которое может взять такой простой HTML и преобразовать его в форматированный текст ячейки.

{{% /alert %}}
