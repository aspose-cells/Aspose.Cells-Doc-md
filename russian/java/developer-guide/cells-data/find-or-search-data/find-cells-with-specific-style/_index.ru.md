## Использование Microsoft Excel

Для выполнения поиска ячеек с определенными стилями в MS Excel необходимы следующие шаги.

1. Выберите **Найти и выбрать** во вкладке **Главная**.
1. Выберите **Найти**.
1. Нажмите **Дополнительно**, если расширенные параметры не видны.
1. Выберите **Выбор формата из ячейки...** из выпадающего списка **Формат**.
1. Выберите ячейку с тем стилем, который вы хотите найти.
1. Нажмите **Найти все**, чтобы найти все ячейки со стилем, аналогичным вашей выбранной ячейке.

## Использование Aspose.Cells for Java

Aspose.Cells for Java предоставляет возможность находить ячейки в листе Excel с определенным стилем. Для этого API предоставляет свойство [**FindOptions.setStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/findoptions#Style).
