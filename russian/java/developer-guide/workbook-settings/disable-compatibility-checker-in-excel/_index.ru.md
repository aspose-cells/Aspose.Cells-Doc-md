{{% alert color="primary" %}}

Проверка совместимости Microsoft Excel сигнализирует, что при сохранении файла в более раннем формате возможны проблемы с функциональностью или потеря точности. Проверка совместимости - это функция Microsoft Office Excel 2007, 2010 и 2013.

Когда вы сохраняете книгу в предыдущей версии, Excel 97 through Excel 2003, из Excel 2007 или Excel 2010, проверка совместимости сканирует книгу, чтобы увидеть, содержит ли она функции, не поддерживаемые более ранней версией. Чтобы помочь вам с принятием решений о том, как обрабатывать проблемы совместимости, проверка совместимости отображает диалоговые окна с выбором. Ее также можно использовать для создания отчета о любых проблемах в книге или отключения функции.

Иногда вам нужно отключить проверку совместимости для конкретной электронной таблицы. С помощью API Aspose.Cells вы можете сделать это динамически, чтобы пользователи не сталкивались с диалоговым окном проверки совместимости, появляющимся при повторном сохранении файла в Microsoft Excel вручную.

{{% /alert %}}

## **Использование Microsoft Excel**

Для отключения проверки совместимости в Microsoft Excel (например, Microsoft Excel 2007/2010):

- (Excel 2007) На кнопке Office нажмите **Подготовка**, затем **Запустить проверку совместимости** и снимите флажок с параметра **Проверять совместимость при сохранении этой книги**.
- (Excel 2010 и 2013) На вкладке **Файл** нажмите **Информация**, затем **Проверить наличие проблем**, нажмите **Проверить совместимость** и, наконец, снимите флажок с параметра **Проверять совместимость при сохранении этой книги**.

## **Использование API Aspose.Cells**

Установите свойство [**WorkbookSettings.CheckComptiliblity**](https://reference.aspose.com/cells/java/com.aspose.cells/workbooksettings#CheckComptiliblity) в **False**, чтобы отключить проверку совместимости Microsoft Excel.

Предположим, у нас есть шаблонный файл XLS. При сохранении или повторном сохранении пользовательского диалогового окна MS Excel 2007/2010/2013 будет отображено диалоговое окно проверки совместимости, как показано на скриншоте ниже.
