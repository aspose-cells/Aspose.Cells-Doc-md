## **Возможные сценарии использования**

Вы можете установить код формата значений серии диаграммы с помощью[**Серия.ValuesFormatCode**](https://reference.aspose.com/cells/java/com.aspose.cells/series#ValuesFormatCode)имущество. Это свойство полезно не только для рядов, основанных на диапазоне внутри рабочего листа, но также хорошо работает для рядов, созданных с помощью массива значений.

## **Установите код формата значений серии диаграммы**

Следующий пример кода добавляет серию в пустую диаграмму, в которой раньше не было серий. Он добавляет серию, используя массив значений. После добавления серии форматирует ее кодом $#,##0 с помощью[**Серия.ValuesFormatCode**](https://reference.aspose.com/cells/java/com.aspose.cells/series#ValuesFormatCode)недвижимость, а число 10000 становится 10000 долларов. На снимке экрана показано влияние кода на[образец файла Excel](51740736.xlsx)и[выходной файл Excel](51740735.xlsx)после казни.
