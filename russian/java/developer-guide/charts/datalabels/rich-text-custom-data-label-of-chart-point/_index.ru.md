{{% alert color="primary" %}}

 Вы можете использовать Aspose.Cells для создания пользовательской метки данных с форматированным текстом для точки диаграммы. Aspose.Cells обеспечивает[**DataLabels.символы()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#characters(int,%20int) ) метод для возврата[**Настройка шрифта**](https://reference.aspose.com/cells/java/com.aspose.cells/FontSetting)объект, который можно использовать для установки свойств шрифта текста, таких как его цвет, жирность и т. д.

{{% /alert %}}

## **Пользовательская метка данных Rich Text для точки диаграммы**

Следующий код получает доступ к первой точке диаграммы первой серии, устанавливает ее текст, а затем устанавливает шрифт первых 10 символов, устанавливая его цвет на красный и жирность на true.
