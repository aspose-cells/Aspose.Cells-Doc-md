## Показать диапазон ячеек в виде меток данных в MS Excel

В Microsoft Excel 2013 вы можете отобразить диапазон Cell для меток данных. Вы можете выбрать этот параметр, выполнив следующие действия.

- Выберите Метки данных серии и щелкните правой кнопкой мыши, чтобы открыть всплывающее меню.
-  Нажмите на**Форматировать метки данных...** и это покажет**Параметры метки**.
-  Установите или снимите флажок**Этикетка содержит — значение от Cells**.
