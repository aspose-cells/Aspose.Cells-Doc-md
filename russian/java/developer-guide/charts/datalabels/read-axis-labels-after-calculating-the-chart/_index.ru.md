## **Возможные сценарии использования**

Вы можете прочитать метки осей вашего графика после вычисления его значений с помощью[**Диаграмма.рассчитать()**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#calculate()) метод. Пожалуйста, используйте[**Axis.AxisLabels**](https://reference.aspose.com/cells/java/com.aspose.cells/axis#AxisLabels)свойство для этой цели, которое вернет список меток осей.
