## **Возможные сценарии использования**
Вы можете создать диаграмму PDF с желаемым размером страницы, используя Aspose.Cells, и указать, как вы хотите выровнять диаграмму внутри страницы. Вы можете выровнять график по верхнему, нижнему, центральному, левому, правому краям и т. д. Кроме того, выходной график можно создать в потоке или на диске.
## **Создайте диаграмму PDF с желаемым размером страницы**
См. следующий пример кода, который загружает[образец файла Excel](64716912.xlsx), получает доступ к первой диаграмме внутри рабочего листа, а затем преобразует ее в[вывод PDF](64716911.pdf)с желаемым размером страницы. На следующем снимке экрана показано, что размер страницы в выводе PDF составляет 7x7, как указано в коде, а диаграмма выровнена по центру как по горизонтали, так и по вертикали.
