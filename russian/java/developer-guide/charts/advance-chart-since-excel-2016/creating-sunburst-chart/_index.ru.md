##  **Возможные сценарии использования**
Древовидные диаграммы хороши для сравнения пропорций внутри иерархии, однако древовидные диаграммы не очень хороши для отображения иерархических уровней между крупнейшими категориями и каждой точкой данных. Диаграмма солнечных лучей гораздо лучше наглядно демонстрирует это. Диаграмма солнечных лучей идеально подходит для отображения иерархических данных. Каждый уровень иерархии представлен одним кольцом или кругом, причем самый внутренний круг является вершиной иерархии. Диаграмма солнечных лучей без каких-либо иерархических данных (один уровень категорий) похожа на кольцевую диаграмму. Однако диаграмма солнечных лучей с несколькими уровнями категорий показывает, как внешние кольца соотносятся с внутренними кольцами. Диаграмма солнечных лучей наиболее эффективно показывает, как одно кольцо разбивается на составляющие части, тогда как другой тип иерархической диаграммы, древовидная диаграмма, идеально подходит для сравнения относительных размеров.
