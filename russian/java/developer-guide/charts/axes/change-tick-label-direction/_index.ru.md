## **Изменить направление метки деления**

Aspose.Cells предоставляет вам возможность изменить направление метки деления графика с помощью[**TickLabels.DirectionType**](https://reference.aspose.com/cells/java/com.aspose.cells/ticklabels#DirectionType) имущество.[**TickLabels.DirectionType**](https://reference.aspose.com/cells/java/com.aspose.cells/ticklabels#DirectionType)собственность принимает[**ChartTextDirectionType**](https://reference.aspose.com/cells/java/com.aspose.cells/ChartTextDirectionType) перечисляемое значение.[**ChartTextDirectionType**](https://reference.aspose.com/cells/java/com.aspose.cells/ChartTextDirectionType)перечисление предоставляет следующие члены

- [**ГОРИЗОНТАЛЬНЫЙ**](https://reference.aspose.com/cells/java/com.aspose.cells/charttextdirectiontype#HORIZONTAL)
- [**ВЕРТИКАЛЬНЫЙ**](https://reference.aspose.com/cells/java/com.aspose.cells/charttextdirectiontype#VERTICAL)
- [**ROTATE_90**](https://reference.aspose.com/cells/java/com.aspose.cells/charttextdirectiontype#ROTATE_90)
- [**ROTATE_270**](https://reference.aspose.com/cells/java/com.aspose.cells/charttextdirectiontype#ROTATE_270)
- [**УСТАНОВЛЕН**](https://reference.aspose.com/cells/java/com.aspose.cells/charttextdirectiontype#STACKED)
