## **Возможные сценарии использования**
Ранние версии Aspose.Cells не могли правильно обрабатывать автоматические единицы оси диаграммы, когда диаграмма отображалась в изображение или PDF. Теперь Aspose.Cells поддерживает обработку автоматических единиц оси диаграммы. Код не меняется. Просто преобразуйте диаграмму в изображение или PDF, и она отобразит ось диаграммы точно так же, как Microsoft Excel.
