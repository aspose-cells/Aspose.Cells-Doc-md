##  **Возможные сценарии использования**
После создания диаграммы на листе вы можете установить для нее ось категорий. В этой статье мы покажем вам, как установить ось категорий для диаграммы Excel, используя Aspose.Cells с примером кода.

##  **Шаги в примере кода**

1. Создайте новую книгу.

2. Создайте новую диаграмму на первом листе.

3. Добавьте значения в ячейки на первом листе.

4. Теперь вы можете установить ось категорий двумя способами: с помощью данных ячейки или напрямую с помощью строк, оба из которых показаны в примере кода.

5. Установите ось значений, сохраните книгу, чтобы просмотреть результат.
