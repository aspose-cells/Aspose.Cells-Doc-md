## **Возможные сценарии использования**

Aspose.Cells позволяет сдвинуть первую строку вниз при вставке таблицы в лист. В этом документе объясняется, как вы можете выполнить эту задачу с использованием API Aspose.Cells.

## **Сдвинуть первую строку вниз при вставке строк таблицы данных ячеек**

Приведенный образец кода иллюстрирует, как сдвинуть первую строку вниз при вставке таблицы в лист. Мы используем простой шаблон файла Excel в коде, чтобы продемонстрировать функцию. Вы можете проверить функцию, установив атрибут boolean [**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/java/com.aspose.cells/importtableoptions#ShiftFirstRowDown) на **True/False**, чтобы лучше понять ее. Пожалуйста, ознакомьтесь с [образцовым файлом Excel](45056043.xlsx), [выходным файлом Excel при значении False](45056044.xlsx) и [выходным файлом Excel при значении True](45056045.xlsx) для вашего ознакомления.
