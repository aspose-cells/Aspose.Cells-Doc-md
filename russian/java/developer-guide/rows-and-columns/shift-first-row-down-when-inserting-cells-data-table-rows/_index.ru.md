## **Возможные сценарии использования**

Aspose.Cells позволяет сместить первую строку вниз при вставке таблицы на лист. В этом документе объясняется, как выполнить эту задачу с помощью API Aspose.Cells.

## **Сдвинуть первую строку вниз при вставке Cells строк таблицы данных**

 В следующем примере кода показано, как сместить первую строку вниз при вставке таблицы на лист. Мы используем простой файл шаблона Excel в коде, чтобы продемонстрировать эту функцию. Вы можете использовать эту функцию, установив логическое значение[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/java/com.aspose.cells/importtableoptions#ShiftFirstRowDown) приписывать**Верно/Неверно** чтобы лучше понять это. Пожалуйста, смотрите[образец файла Excel](45056043.xlsx), [выходной файл Excel False](45056044.xlsx) , и[выходной файл Excel True](45056045.xlsx)для вашей справки.
