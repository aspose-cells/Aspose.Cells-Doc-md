 Aspose.Cells позволяет загрузить облегченную версию книги в память для редактирования ее метаданных. Пожалуйста, используйте[com.aspose.cells.WorkbookMetadata](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookMetadata) класс для загрузки книги.

## **Использование метаданных рабочей книги**
 В следующем примере кода используется[WorkbookМетаданные](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookMetadata) класс для редактирования пользовательских свойств документа книги. После того, как вы откроете книгу с помощью[Рабочая тетрадь](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook) class, вы сможете прочитать свойства документа.
