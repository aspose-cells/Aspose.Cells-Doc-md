## **Возможные сценарии использования**
Иногда вы хотите встроить формулу в поле смарт-маркера. В этой статье описывается, как использовать параметр «Формула» для встраивания формулы в поле интеллектуального маркера.
## **Использование параметра Formula в поле Smart Marker**
 В следующем примере кода формула встраивается в переменную интеллектуального маркера с именем Test, а ее источник данных также имеет имя Test, поэтому полное поле с параметром формулы выглядит следующим образом:**&=$Тест(формула)** и после выполнения кода[окончательный выходной файл Excel](47153156.xlsx) будут иметь формулы в ячейках от A1 до A5.
