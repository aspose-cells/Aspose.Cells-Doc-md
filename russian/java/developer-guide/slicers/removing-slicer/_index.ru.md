## **Возможные сценарии использования**
Если вы хотите удалить слайсер в Microsoft Excel, просто выберите его и нажмите кнопку*Удалить*кнопка. Точно так же, если вы хотите удалить его с помощью Aspose.Cells API программно, используйте[Рабочий лист.getSlicers().remove()](https://reference.aspose.com/cells/java/com.aspose.cells/slicercollection#remove\(com.aspose.cells.Slicer\)) метод. Это удалит слайсер с рабочего листа.
## **Удаление слайсера**
Следующий пример кода загружает[образец файла Excel](67338504.xlsx)который содержит существующий слайсер. Он обращается к слайсерам, а затем удаляет их. Наконец, он сохраняет книгу как[выходной файл Excel](67338502.xlsx). На следующем снимке экрана показан слайсер, который будет удален после выполнения примера кода.
