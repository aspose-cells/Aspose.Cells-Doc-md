## **Нарисуйте фильтр при рендеринге Excel в PDF**
Если у вас есть файл Excel, к которому применен фильтр, и вы хотите экспортировать Excel в PDF с настройками фильтра, Aspose.Cells теперь поддерживает это по умолчанию. Просто экспортируйте файл Excel с фильтром в PDF, и сгенерированный PDF покажет примененный фильтр.

Следующий образец кода загружает [образец файла Excel](SampleSlicerChart.xlsx), содержащий существующий фильтр. Затем сохраняет книгу в виде [выходного PDF файла](SampleSlicerChart.pdf). На следующем скриншоте сравниваются исходный файл Excel и сгенерированный PDF файл.
