## **Этот формат файла не поддерживается или вы указали неправильный формат**
Если пользователь указал формат файла при создании книги из файла шаблона, обычно эта ошибка возникает из-за того, что указанный формат файла не является фактическим форматом файла шаблона. Если пользователь не указал формат файла, обычно это происходит потому, что расширение имени файла не представляет фактический формат файла этого файла, и формат файла не может быть определен автоматически, например файл csv/tsv, который не имеет каких-либо специальных идентификаторов. .
