## **Этот формат файла не поддерживается или вы не указываете правильный формат**
Если при создании рабочей книги из шаблона пользователь указал формат файла, часто эта ошибка возникает из-за того, что указанный формат файла не совпадает с фактическим форматом файла шаблона. Если формат файла не был указан, обычно это связано с тем, что расширение имени файла не соответствует фактическому формату файла, и он не может быть определен автоматически, например, CSV/TSV файл без специальных идентификаторов. Конечно, форматы файлов, которые не поддерживаются Cells, также будут выдавать эту ошибку.
