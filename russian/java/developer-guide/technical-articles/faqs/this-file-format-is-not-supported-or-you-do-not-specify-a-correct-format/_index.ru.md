## **Этот формат файла не поддерживается или вы не указываете правильный формат**
Если пользователь указал формат файла при создании книги из файла шаблона, часто это происходит потому, что указанный формат файла не является фактическим форматом файла шаблона. Если пользователь не указал формат файла, это обычно происходит потому, что расширение имени файла не соответствует фактическому формату этого файла, и формат файла не может быть обнаружен автоматически, например, файл csv/tsv, у которого нет специальных идентификаторов.
