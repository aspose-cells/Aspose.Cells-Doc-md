{{% alert color="primary" %}} 

 Aspose.Cells — это компонент обработки электронных таблиц, который позволяет создавать, читать, изменять и записывать электронные таблицы Excel® без использования Microsoft Excel®.

- Aspose.Cells for Java — библиотека классов Java, предназначенная для работы в приложениях Java.

{{% /alert %}} 

 Хотя специальных версий Aspose.Cells для других платформ не существует, можно использовать Aspose.Cells во многих других языках программирования, включая (но не ограничиваясь):

|**Язык программирования** |**Aspose.Cells для использования** |**Технология взаимодействия** |
|:- |:- |:- |
| Перл|Java | Встроенный:: Java|
| PHP|Java |Java Мост|
|Python |Java | JPype|
 Из приведенной выше таблицы видно, что для некоторых языков программирования можно использовать Aspose.Cells for Java. Следующая таблица может помочь вам выбрать:

|**Сценарий** |**Когда использовать** |
|:- |:- |
| Aspose.Cells for Java via Java Мост или встроенный::Java|<p>**Плюсы**:</p><p>- Доступно на любой платформе, где доступен номер Java.</p><p>- Легко вызывать статические методы, конструкторы с параметрами, перегруженные методы и использовать перечисляемые значения.</p>|
||<p>**Минусы**: </p><p>- Имеет менее частые выпуски и меньше функций, чем Aspose.Cells for .NET.</p>|
 Обратите внимание, что Aspose.Cells for Java предназначен для приложений for Java, и его использование на других языках программирования связано с некоторыми техническими ограничениями или обходными путями.
