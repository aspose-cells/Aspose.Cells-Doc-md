{{% alert color="primary" %}} 

Aspose.Cells - это компонент обработки электронных таблиц, который позволяет генерировать, читать, изменять и записывать электронные таблицы Excel® без использования Microsoft Excel®. 

- Aspose.Cells for Java - библиотека классов Java и предназначена для работы в приложениях Java.

{{% /alert %}} 

Хотя нет специальных версий Aspose.Cells для других платформ, возможно использовать Aspose.Cells на многих других языках программирования, включая (но не ограничиваясь ими): 

|**Язык программирования** |**Aspose.Cells для использования** |**Технология взаимодействия** |
| :- | :- | :- |
|Perl |Java |Inline::Java |
|PHP |Java |Java Bridge |
|Python |Java |JPype |
Из таблицы выше видно, что для некоторых языков программирования возможно использование Aspose.Cells for Java. Следующая таблица может помочь вам выбрать: 

|**Сценарий** |**Когда использовать** |
| :- | :- |
|Aspose.Cells for Java via Java Bridge или Inline::Java |<p>**Преимущества**:</p><p>- Доступен на любой платформе, где доступен Java.</p><p>- Легко вызывать статические методы, конструкторы с параметрами, перегруженные методы и использовать перечисленные значения.</p>|
| |<p>**Недостатки**: </p><p>- Имеет реже обновления и меньше функций, чем Aspose.Cells for .NET.</p>|
Обратите внимание, что Aspose.Cells for Java предназначен для приложений Java, и его использование в других языках программирования подвержено некоторым техническим ограничениям или обходным путям.
