## **Хотите ускорить разработку?**

{{% alert color="primary" %}}

Вы можете использовать Aspose.Cells для создания Excel-файлов с нуля с помощью его гибкого API. Одним из наиболее распространенных способов использования Aspose.Cells является создание и заполнение Excel-файлов данными, которые могут поступать из существующей базы данных, от пользователя или из других источников динамически. Помимо добавления данных в Excel-файлы, вам также понадобится применять форматирование (внешний вид и стиль).

Форматирование занимает время. Что может ускорить разработку?

{{% /alert %}}

## **Решение - Дизайнерские электронные таблицы**

Чтобы ускорить вашу разработку, Aspose.Cells предлагает захватывающую возможность - дизайнерские электронные таблицы.

### **Что такое дизайнерская электронная таблица?**

Дизайнерская электронная таблица - это файл Excel, созданный с использованием Microsoft Excel, в котором уже применены все форматирования, и содержимое может быть определено на этапе дизайна. Это больше похоже на файл шаблона, который не содержит никаких данных, а только форматирование, как показано ниже:

### **Как ей пользоваться?**

После создания дизайнерской электронной таблицы используйте Aspose.Cells для чтения дизайнерской электронной таблицы и добавления данных в указанные позиции (согласно заранее определенным настройкам формата) во время выполнения. Например, на скриншоте выше вы можете видеть, что мы уже решили на этапе дизайна добавить столбцы **Имя продукта**, **Имя категории**, **Количество в упаковке** и **Единицы на складе**. Они еще не содержат данных: они будут добавлены во время выполнения. На приведенной выше фигуре показана очень простая таблица, но дизайнерские электронные таблицы помогают вам экономить время даже при работе с сложными электронными таблицами.

### **Умные маркеры**

Кроме того, Aspose.Cells позволяет добавлять [умные маркеры](/cells/ru/java/smart-markers/) в дизайнерскую электронную таблицу. Умные маркеры обрабатываются во время выполнения с использованием метода Process класса [**WorkbookDesigner**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookdesigner). Это может помочь вам писать меньше кода и все равно получать отличный результат.

[Узнайте больше о умных маркерах](/cells/ru/java/smart-markers/).
