{{% alert color="primary" %}} 

В некоторых случаях вы можете задаться вопросом, какую версию продукта у вас установлена. Мы часто создаем новые исправления (исправления ошибок для пользовательских сценариев, на которые они указывают) и публикуем их на форумах по их срочной потребности. Номер версии состоит из основного номера версии, номера минорной версии и номера исправления. Все определенные компоненты должны быть целыми числами больше или равными 0. Формат номера версии следующий:

основной.минорный.исправление, мы можем увеличить одну часть на 1 и создать новую версию. Обычно мы увеличиваем последнюю часть на 1 и создаем новое исправление, чтобы опубликовать его на форумах для пользователей.

Этот документ описывает некоторые способы проверки установленной версии компонента на вашей системе.

{{% /alert %}}

Если у вас установлена версия/исправление Java (Aspose.Cells for Java), вы можете распаковать файл jar библиотеки Aspose.Cells, открыть файл MANIFEST блокнотом и найти строку "Specification-Version:" для проверки ее значения.
