{{% alert color="primary" %}}

 Вы можете легко загрузить Aspose.Cells для ознакомления с[страница загрузки](https://releases.aspose.com/cells/java/). Загрузка ознакомительной версии аналогична загрузке полной версии: ознакомительная версия становится лицензионной, когда вы применяете лицензию.

{{% /alert %}}

Ознакомительная версия Aspose.Cells (то есть Aspose.Cells без применения лицензии) предоставляет большую часть функциональных возможностей продукта. Одним из ограничений является то, что он всегда вводит дополнительный рабочий лист с оценочным водяным знаком при открытии/сохранении любого документа электронной таблицы.
