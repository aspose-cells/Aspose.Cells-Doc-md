## **Добро пожаловать в документацию Aspose.Cells**
**Aspose.Cells** является наиболее гибким компонентом, который позволяет приложениям Java создавать электронные таблицы Excel и управлять ими без необходимости**Microsoft Excel**для установки в систему. В отличие от аналогичных продуктов других поставщиков, Aspose.Cells не только поддерживает создание электронных таблиц и другие основные функции форматирования файлов, но также поддерживает ряд дополнительных функций. Эти расширенные функции значительно упрощают разработчикам работу с содержимым электронных таблиц, форматированием ячеек и защитой файлов. Aspose.Cells также позволяет импортировать данные в электронные таблицы из разных источников данных, добавлять общие и сложные математические, даты и времени, финансовые, текстовые формулы / функции, поддерживает манипулирование диаграммами, изображениями, комментариями, объектами рисования и элементами управления и т. д.

Aspose.Cells for Java также предоставляет комплексное грид-решение для веб-клиентов; он имеет элемент управления на основе графического интерфейса следующим образом:

- Aspose.Cells.GridWeb, который поддерживает веб-приложения.

Используя Aspose.Cells.GridWeb для управления сеткой JAVA, разработчики могут легко импортировать и экспортировать электронные таблицы Excel®. Любой пользователь, знакомый с Microsoft Excel, может управлять внешним видом сетки. Aspose.Cells предлагает богатый набор функций, таких как управление дизайном и макетом сетки, управление несколькими рабочими листами, управление ячейками, строками и столбцами рабочего листа, создание и вычисление формул и т. д.

## **Описание продукта**
Aspose.Cells можно использовать с любым приложением, будь то веб-приложение Java или настольное приложение Java. Кроме того, библиотеку можно также использовать с некоторыми другими решениями, такими как PHP и т. д. Aspose.Cells предоставляет полнофункциональные демонстрации и рабочие примеры, написанные на Java, чтобы разработчики могли лучше понять функции, предоставляемые компонентом.

Помимо поддержки общих функций, таких как создание или чтение собственных файлов Excel, Aspose.Cells также предлагает множество дополнительных функций, таких как сохранение и открытие файлов Excel в потоках и из них, отправка вывода в клиентский браузер, импорт и экспорт данных из**Набор результатов** или**Множество**, импорт изображений и диаграмм, создание изображений и диаграмм по телефону API, импорт формул из электронной таблицы конструктора и настройка параметров страницы по номеру API.

Кроме того, есть также ряд уникальных функций, которые разработчики могут найти только в Aspose.Cells, таких как поддержка форматов CSV и SpreadsheetML, добавление копии существующего рабочего листа (с полным содержимым, изображениями и диаграммами и т. д.) в файл, установка градиентного фона для диаграмм до API, создание комментариев, автоматических фильтров и разрывов страниц до API, установка сложных формул, поддержка условного форматирования, поддержка параметров защиты, представленных в Microsoft Excel XP или выше, управление именованными диапазонами, поиск API, поддержка пользовательской диаграммы API и механизм расчета формулы и т. д.
## **Особенности документа**
- Открывайте обычные или зашифрованные файлы Excel (Excel 95, Excel97, Excel2007/2010/2013/2016/2019, Office365) из разных источников.
- Открывайте файлы Plain Numbers (созданные приложением Apple Numbers) из разных источников.
- Сохраняйте файлы Excel (Excel97-Excel2007/2010/2013/2016/2019) в разных форматах (XLS, XLSX/XLSM, XLTX, XLTM, XLSB, CSV, SpreadsheetML, Tab Delimited/TSV, HTML и т. д.)
- Преобразование файла Excel в документ PDF
- Преобразование электронной таблицы Excel в документ XPS
-  Преобразование в тегированный формат файла изображения**(ТИФФ)**
-  Прочти и напиши**Электронная таблица OpenDocument (ODS)** формат
- Изменить свойства документа файлов Excel

### **Защита рабочего листа**
- **Защита содержимого, объектов и сценариев** рабочего листа

-  Установлен**заголовки и колонтитулы**
-  Установлен**область печати**, **печатать заголовок** и т.п.
### **Разрывы страниц**

### **Функции строк и столбцов**
