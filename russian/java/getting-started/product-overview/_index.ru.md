## **Добро пожаловать в документацию Aspose.Cells**
**Aspose.Cells** - самый гибкий компонент, который позволяет приложениям Java создавать и управлять электронными таблицами Excel без необходимости установки **Microsoft Excel** на системе. В отличие от аналогичных продуктов других поставщиков, Aspose.Cells поддерживает не только создание электронных таблиц и другие базовые возможности форматирования файлов, но также поддерживает ряд продвинутых функций. Эти продвинутые функции делают намного удобнее для разработчиков манипулировать содержимым электронных таблиц, форматирование ячеек и защиту файлов. Aspose.Cells также позволяет импортировать данные в электронные таблицы из различных источников данных, добавлять обычные и сложные математические, даты, финансовые, текстовые формулы/функции, поддерживает манипуляцию диаграммами, изображениями, комментариями, объектами рисования и элементами управления и т. д.

Aspose.Cells for Java также предоставляет полное решение для веб-клиентов; у него есть графический контроль следующим образом:

- Aspose.Cells.GridWeb, который поддерживает веб-приложения

Используя Aspose.Cells.GridWeb для интерфейса управления JAVA-сеткой, разработчики могут легко импортировать и экспортировать электронные таблицы Excel®. Любой пользователь, знакомый с Microsoft Excel, может контролировать внешний вид и ощущение сетки. Aspose.Cells предлагает богатый набор функций, таких как управление дизайном и компоновкой сетки, управление несколькими рабочими книгами, управление ячейками, строками и столбцами рабочего листа, создание и вычисление формул и т. д.

## **Описание продукта**
Aspose.Cells может быть использован с любым типом приложения, будь то веб-приложение Java или настольное приложение Java. Более того, библиотеку можно также использовать с некоторыми другими решениями, такими как PHP и т. д. Aspose.Cells предоставляет полностью функциональные демонстрации и рабочие примеры, написанные на Java, чтобы разработчики могли лучше понять возможности, предоставляемые компонентом.

Помимо поддержки общих функций, таких как создание или чтение файлов Excel, Aspose.Cells также предлагает множество расширенных функций, таких как сохранение и открытие файлов Excel в потоки, отправка вывода в браузер клиента, импорт и экспорт данных из **ResultSet** или **Array**, импорт изображений и диаграмм, создание изображений и диаграмм через API, импорт формул из дизайнера электронной таблицы и настройка параметров страницы через API.

Более того, есть также ряд уникальных функций, которые разработчики могут найти только в Aspose.Cells, таких как поддержка форматов CSV и SpreadsheetML, добавление копии существующего рабочего листа (с полным содержимым, изображениями и диаграммами и т. д.) в файл, установка градиентного фона для диаграмм через API, создание комментариев, автофильтры и разрывы страниц через API, установка сложных формул, поддержка условного форматирования, поддержка параметров защиты, представленных в Microsoft Excel XP или выше, манипулирование именованными диапазонами, поиск API, поддержка пользовательского API для создания диаграмм и вычисления формул и т. д.
## **Особенности документа**
- Открыть обычные или зашифрованные файлы Excel (Excel 95, Excel97, Excel2007/2010/2013/2016/2019, Office365) из разных источников
- Открыть обычные файлы Numbers (созданные приложением Apple's "Numbers") из разных источников
- Сохранить файлы Excel (Excel97- Excel2007/2010/2013/2016/2019) в различных форматах (XLS, XLSX/XLSM, XLTX, XLTM, XLSB, CSV, SpreadsheetML, разделенные табуляции/TSV, HTML и т. д.)
- Преобразовать файл Excel в документ PDF
- Преобразование электронной таблицы Excel в документ XPS
- Преобразование в формат файлов изображений с тегами **(TIFF)**
- Чтение и запись формата **OpenDocument Spreadsheet (ODS)**
- Изменение свойств документа Excel
## **Особенности листа**
### **Опции отображения**
- **Скрыть или показать** лист
- Отображение или скрытие **вкладок, полос прокрутки, сетки и заголовков строк и столбцов**
- **Предварительный просмотр переноса на новую страницу**
- Установка уровня **масштабирования** для листа
- Оставление выбранных данных видимыми при прокрутке с помощью **замораживания панелей**
### **Защита листа**
- **Защита содержимого, объектов и сценариев** листа
### **Опции настройки страницы**
- Установка ориентации страницы, масштабирования, размера бумаги, качества печати и т. д.
- Установка **полей страницы**, таких как верхнее, левое, правое, нижнее и т. д.
- Установка **верхних и нижних колонтитулов**
- Установка **области печати**, **заголовка печати** и т. д.
### **Разрывы страниц**
- Вставка **горизонтальных** и **вертикальных разрывов страниц** в указанной ячейке
### **Функции строк и столбцов**
- **Настройка ширины столбца** по мере необходимости
- Автоматическое изменение ширины и высоты ячеек в соответствии с их содержимым
- **Вставка или удаление** строк и столбцов
- **Скрыть или показать** строки и столбцы
- **Группирование или разгруппирование** строк и столбцов
### **Функции управления данными**
- Добавление данных в нужные ячейки во время выполнения
- **Импортирование данных в листы** из различных источников данных
- Извлечение данных из ячеек в соответствии с их типами данных
- **Экспорт данных ячеек в массив**
- Поиск или поиск конкретных данных в ячейках
- Установка **формул с встроенными или дополнительными функциями**
- Установка формулы со ссылкой **на массив** и стиль **R1C1**
- Добавление различных видов гиперссылок
- Создание или доступ к именованным диапазонам
- Применение **проверки данных** и **автофильтрации**
- Применение **подытогов** к таблицам листа
### **Функции форматирования**
- Применение **настроек формата чисел** для ячеек
- Форматирование ячейки как дата
- Установка **выравнивания текста**
- Объединение нескольких ячеек
- Применение различных видов **настроек шрифта** для ячеек
- Применение различных цветов к ячейкам, шрифтам, сеткам, графическим объектам и т. д.
- Установка **форматированного текста** в одной ячейке
- Применение различных **настроек границ** на ячейки
- Применение различных **фоновых узоров** на ячейки
- Применение **настроек формата** на лист, строку или столбец и т. д.
- Применить **условное форматирование** к ячейкам
## **Функции графиков и графики**
- Легко создавать различные виды диаграмм
- Настроить **диаграммы**, установив свойства
- Динамически установить **данные для диаграмм**
- Добавить **пользовательские диаграммы** в листы
- Добавить **изображения** на листы во время выполнения
- Создать и добавить комментарии к ячейкам
- Преобразовать диаграмму в файл изображения
- Преобразовать лист в файл изображения
- Печать листов
## **Расширенные функции**

- Изменить **исходные данные сводной таблицы** во время выполнения
- Создание и управление сводными таблицами
- Манипуляции кодом VBA или макросами.
