Иногда возникает вопрос: почему мы должны использовать продукты Aspose, а не xlsx4j? На этот вопрос легко ответить: функциональность и возможности.

## **xlsx4j**
xlsx4j - это библиотека Java с открытым исходным кодом (ASLv2) для создания и манипулирования файлами Microsoft Open XML Excel xlsx, аналогичная docx4j для манипулирования файлами Microsoft Open XML Word docx.
Она похожа на OpenXML SDK от Microsoft, но для Java. docx4j использует JAXB для создания объектного представления в памяти.
Вам потребуется времени, чтобы понять JAXB и структуру файлов Open XML.

Узнайте больше на [docx4java.org](https://www.docx4java.org/trac/docx4j)
## **Aspose.Cells for Java**
Aspose.Cells for Java - это награжденный компонент электронных таблиц Excel, который позволяет разработчикам Java встраивать возможность чтения, записи и манипулирования электронными таблицами Excel® (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF и изображениями в их собственные Java-приложения без необходимости полагаться на Microsoft Excel®.

Aspose.Cells for Java - это зрелый, масштабируемый и функционально насыщенный компонент, который предлагает множество функций, выходящих далеко за пределы простых возможностей экспорта данных других поставщиков. С Aspose.Cells for Java разработчики могут экспортировать данные, форматировать электронные таблицы на самом мельчайшем уровне, импортировать изображения, создавать диаграммы, применять и вычислять сложные формулы, потоковые данные Excel®, сохранять в различных форматах и многое другое - все это без необходимости использования Microsoft Excel® или автоматизации Microsoft Office.

Узнайте больше на [aspose.com](https://products.aspose.com/cells/java/)

## **Почему не xlsx4j**
xlsx4j может только прочитывать и сохранять форматы файлов Microsoft Open XML Excel xlsx, Aspose.Cells for Java может обрабатывать все форматы файлов Excel (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS) и поддерживает различные преобразования форматов файлов.

## **Aspose.Cells Java for xlsx4j**
Проект Aspose.Cells для xlsx4j показывает, как можно выполнять различные задачи с использованием API Aspose.Cells для Java по сравнению с xlsx4j. Проект также охватывает функции, которые доступны только в API Aspose.Cells, но не в xlsx4j для работы с электронными таблицами.

Этот проект полезен для разработчиков, которые хотят сравнить xlsx4j с Aspose.Cells ИЛИ перейти от xlsx4j к Aspose.Cells.
## **Требования к системе и поддерживаемые платформы**
### **Системные требования**
Ниже приведены системные требования для выполнения Aspose.Cells Java для xlsx4j:

- Установленный Java 1.4 или выше.
- Загруженный компонент Aspose.Cells.
- Загруженные библиотеки docx4j.
### **Поддерживаемые платформы**
Ниже приведены поддерживаемые платформы:

- Aspose.Cells 8.3.0 или выше.
- Библиотеки docx4j 3.1.0 или выше.
- Java IDE (Eclipse, NetBeans, IntelliJ, ...)

Ниже приведены релизы запущенных примеров, доступные для скачивания на всех указанных ниже сайтах социального кодирования:

### **GitHub**

Чтобы настроить **Aspose.Cells for Java API**, включите URL репозитория Aspose и зависимость maven Aspose.Cells for Java API в свой **pom.xml** следующим образом:
### **Maven-репозиторий Aspose**
{{< highlight java >}}

 <repositories>

    <repository>

        <id>aspose-maven-repository</id>

        <name>Aspose Maven Repository</name>

        <url>http://repository.aspose.com/repo/</url>

    </repository>

</repositories>

{{< /highlight >}}
### **Зависимость Maven Aspose.Cells for Java**
{{< highlight java >}}

 <dependencies>

    <dependency>

        <groupId>com.aspose</groupId>

        <artifactId>aspose-cells</artifactId>

        <version>8.6.0</version>

    </dependency>

</dependencies>

{{< /highlight >}}

**Примечание:** Пожалуйста, используйте последнюю доступную версию Aspose API.
