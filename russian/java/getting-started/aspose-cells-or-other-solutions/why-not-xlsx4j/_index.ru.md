## **xlsx4j**
xlsx4j — это библиотека с открытым исходным кодом (ASLv2) Java для создания и управления файлами Microsoft Open XML Excel xlsx, аналогичная docx4j для управления файлами Microsoft Open XML Word docx.
Он похож на SDK OpenXML Microsoft, но for Java. docx4j использует JAXB для создания представления объекта в памяти.
Вам потребуется время, чтобы понять JAXB и файловую структуру Open XML.

 Подробнее на[docx4java.org](https://www.docx4java.org/trac/docx4j)
## **Aspose.Cells for Java**
Aspose.Cells for Java is an award-winning Excel Spreadsheet component that allows Java developers to embed the ability to read, write and manipulate Excel® spreadsheets (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF и форматы файлов изображений в свои собственные Java приложения без необходимости полагаться на Microsoft Excel®.

Aspose.Cells for Java — это зрелый, масштабируемый и многофункциональный компонент, который предлагает множество функций, выходящих далеко за рамки простых возможностей экспорта данных других поставщиков. С помощью Aspose.Cells for Java разработчики могут экспортировать данные, форматировать электронные таблицы с максимальной детализацией, импортировать изображения, создавать диаграммы, применять и вычислять сложные формулы, передавать данные Excel® в потоковом режиме, сохранять в различных форматах и многое другое — и все это без использования Microsoft Excel®. или Microsoft Автоматизация офиса.

 Подробнее на[aspose.com](https://products.aspose.com/cells/java/)

## **Почему не xlsx4j**
xlsx4j only can parsing and saving Microsoft Open XML Excel xlsx files,Aspose.Cells for Java can process all Excel file formats(XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS) and supports various file format conversions.

## **Aspose.Cells Java для xlsx4j**
Проект Aspose.Cells для xlsx4j показывает, как различные задачи могут выполняться с использованием API Aspose.Cells Java по сравнению с xlsx4j. Проект также охватывает функции, которые доступны только в API-интерфейсах Aspose.Cells, но не в xlsx4j для работы с электронными таблицами.

Этот проект полезен для разработчиков, которые хотят сравнить xlsx4j с Aspose.Cells ИЛИ перейти с xlsx4j на Aspose.Cells.
## **Системные требования и поддерживаемые платформы**
### **Системные Требования**
Ниже приведены системные требования для выполнения Aspose.Cells Java для xlsx4j:

- Java Установлена версия 1.4 или выше.
- Скачал компонент Aspose.Cells.
- Скачал библиотеки docx4j.
### **Поддерживаемые платформы**
Ниже приведены поддерживаемые платформы:

- Aspose.Cells 8.3.0 или выше.
- docx4j 3.1.0 или выше библиотеки.
- Java IDE (Eclipse, NetBeans, IntelliJ, ...).

    </dependency>

</dependencies>
