## **Aspose.Cells - Преобразование XLS в PDF**
Документы в формате PDF широко используются в качестве стандартного формата обмена документами между организациями, государственными секторами и частными лицами. Разработчиков программного обеспечения часто просят найти способ легко конвертировать Microsoft файлы Excel в документы PDF. Aspose.Cells поддерживает эту функцию.

**Java**

{{< highlight "java" >}}

 Workbook workbook = new Workbook(dataDir + "workbook.xls");

//Save the document in PDF format

workbook.save(dataDir + "AsposeConvert.pdf", SaveFormat.PDF);

{{< /highlight >}}
