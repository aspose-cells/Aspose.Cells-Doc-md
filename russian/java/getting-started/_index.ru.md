## **Системные Требования**

Aspose.Cells for Java поддерживает следующие версии Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- или выше

[Подробнее](/cells/ru/java/system-requirements/)

## **Монтаж**

### **Установить Aspose Cells for Java из репозитория Maven**

Maven — это самый простой способ загрузить и установить Aspose.Cells for Java.

1.  Укажите конфигурацию/местоположение репозитория Aspose Maven в файле Maven pom.xml.
 2. Определите зависимость Aspose.Cells for Java API в файле pom.xml.

[Больше шагов для деталей](/cells/ru/java/installation/)

### **Установить Aspose Cells for Java вручную**

1. [Скачать aspose-cells-{версия}-java.zip](https://downloads.aspose.com/cells/java/)
1. Разархивируйте пакет, войдите в каталог \JDK 1.6\, разархивируйте aspose-cells-{версия}-java.zip.
1. Войдите в каталог \JDK 1.6\aspose-cells-{version}-java\lib\, установите пакеты JAR вручную с помощью Eclipse или Intellij.
