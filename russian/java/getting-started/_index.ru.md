## **Системные требования**

Aspose.Cells for Java поддерживает следующие версии Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- или выше

[Подробнее](/cells/ru/java/system-requirements/)

## **Установка**

### **Установите Aspose Cells for Java из репозитория Maven**

Maven - самый простой способ загрузить и установить Aspose.Cells for Java. 

1. Укажите конфигурацию/местоположение репозитория Aspose в вашем файле pom.xml для Maven. 
2. Определите зависимость API Aspose.Cells for Java в вашем файле pom.xml. 

[Подробные шаги](/cells/ru/java/installation/)

### **Установите Aspose Cells for Java вручную**

1. [Скачайте файлы aspose-cells jars](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) в папку.
1. Войдите в папку, установите пакеты JAR вручную с помощью Eclipse или Intellij.
