##  **Системные Требования**

Aspose.Cells for Java поддерживает следующие версии Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1,7)
- или выше

[Подробнее](/cells/ru/java/system-requirements/)

##  **Монтаж**

###  **Установить Aspose Cells for Java из репозитория Maven.**

 Maven — это самый простой способ загрузить и установить Aspose.Cells for Java.

1.  Укажите Aspose Maven Конфигурацию/расположение репозитория в вашем pom.xml Maven.
 2. Определите зависимость Aspose.Cells for Java API в вашем pom.xml.

[Дополнительный шаг для получения подробной информации](/cells/ru/java/installation/)

###  **Установите Aspose Cells for Java вручную.**

1. [Загрузите банки aspose-cells](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) в папку.
1. Войдите в папку, установите пакеты JAR вручную с помощью Eclipse или Intellij.
