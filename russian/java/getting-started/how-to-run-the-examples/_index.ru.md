## **Скачать с GitHub**
Все примеры Aspose.Cells for Java размещены на [GitHub](https://github.com/aspose-cells/Aspose.Cells-for-java). Вы можете клонировать репозиторий, используя ваш клиент GitHub или скачать ZIP-файл отсюда (https://github.com/aspose-cells/Aspose.Cells-for-java/archive/master.zip).

Извлеките содержимое ZIP-файла в любую папку на вашем компьютере. Все примеры находятся в папке **Примеры**.

Папка **Ресурсы** содержит входные документы или файлы, используемые в примерах.

Вы можете запустить/выполнить эти примеры в любой среде разработки по вашему выбору, однако, для демонстрационных целей мы используем **Eclipse**. Используйте опцию **Файл -> Импорт**, чтобы импортировать пример по вашему выбору и указать путь.

Добавьте необходимые файлы jar (библиотеки) в проект.

Каждый пример представлен в виде отдельного исполняемого класса. Щелкните правой кнопкой мыши по файлу и выберите пункт Запустить файл.

{{% alert color="primary" %}} 

Примеры представлены в виде **Проекта Eclipse**. Пожалуйста, откройте Примеры в Eclipse, используя команду **Файл > Открыть проект из файловой системы...** и следуйте инструкциям. Теперь вы должны сможете успешно выполнить примеры.

{{% /alert %}} 
## **Запустите пример**
Как только вы откроете Примеры в среде Eclipse IDE, выберите любой файл .java. Например, выберите этот.

TilePictureAsTextureInsideShape.java

найден внутри

Примеры>src>AsposeCellsExamples>ОбъектыРисования каталог

или внутри пакета

AsposeCellsExamples.DrawingObjects

и запустите его.

## **Внести вклад**
Если вы хотите добавить или улучшить пример, мы призываем вас внести свой вклад в проект. Все примеры и проекты-витрины в этом репозитории являются открытыми и могут быть свободно использованы в ваших собственных приложениях.

Для внесения вклада вы можете форкнуть репозиторий, отредактировать исходный код и создать запрос на включение изменений. Мы рассмотрим изменения и включим их в репозиторий, если они будут полезны.
