## **Скачать с GitHub**
 Все примеры Aspose.Cells for Java размещены на[Гитхаб](https://github.com/aspose-cells/Aspose.Cells-for-java) . Вы можете клонировать репозиторий с помощью своего любимого клиента GitHub или загрузить ZIP-файл с[здесь](https://github.com/aspose-cells/Aspose.Cells-for-java/archive/master.zip).

 Извлеките содержимое ZIP-файла в любую папку на вашем компьютере. Все примеры находятся в**Примеры** папка.

**Ресурсы**папка содержит входные документы или файлы, используемые в примерах.

 Вы можете запускать/выполнять эти примеры с любой IDE по вашему выбору, однако для демонстрационных целей мы используем**Затмение** . Использовать**Файл -> Импорт** возможность импортировать пример по вашему выбору и указать путь.

Добавьте в проект необходимые файлы jar (библиотеки).

Каждый пример предоставляется как отдельный исполняемый класс. Щелкните файл правой кнопкой мыши и выберите «Выполнить файл».

{{% alert color="primary" %}} 

 Примеры в виде**Затмение Проект** . Пожалуйста, откройте примеры в Eclipse с помощью**Файл > Открыть проекты из файловой системы...** команду меню и следуйте инструкциям. Теперь вы сможете нормально запускать примеры.

{{% /alert %}} 
## **Запустите пример**
Один раз вы открываете примеры в Eclipse IDE, затем выбираете любой файл .java. Например, выберите этот.

TilePictureAsTextureInsideShape.java

найден внутри

Примеры>источник>AsposeCellsExamples>Каталог DrawingObjects

или внутри упаковки

AsposeCellsExamples.DrawingObjects

и запустить его.

## **Делать вклад**
Если вы хотите добавить или улучшить пример, мы рекомендуем вам внести свой вклад в проект. Все примеры и демонстрационные проекты в этом репозитории имеют открытый исходный код и могут свободно использоваться в ваших собственных приложениях.

Чтобы внести свой вклад, вы можете разветвить репозиторий, отредактировать исходный код и создать запрос на извлечение. Мы рассмотрим изменения и включим их в репозиторий, если сочтем их полезными.
