## **Встраивайте редактор электронных таблиц HTML5 в любое место**

Редактор электронных таблиц HTML5 можно встроить в любой веб-сайт, блог или форум для обмена электронными таблицами в Интернете. Его можно встроить как отдельный редактор или загрузить с помощью файла электронной таблицы.

**Встроить как отдельный редактор**

Для встраивания в качестве отдельного редактора используйте HTML-тег IFRAME для добавления на веб-сайт. Например:

{{< highlight "html" >}}

 <iframe src="http://spreadsheet-editor.aspose.com/" width="800" height="600">

    Your web browser does not support IFRAMEs

</iframe>

{{< /highlight >}}

**Встроить с помощью электронной таблицы**

 Чтобы загрузить электронную таблицу во встроенный редактор**URL** параметр. Например:

{{< highlight "html" >}}

 <iframe src="http://spreadsheet-editor.aspose.com/?url=http://example.com/Sample.xlsx" width="800" height="600">

    Your web browser does not support IFRAMEs

</iframe>

{{< /highlight >}}
