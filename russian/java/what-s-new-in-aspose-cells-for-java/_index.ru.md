Эта страница описывает наиболее интересные новые функции Aspose.Cells, представленные в последних релизах.

## Aspose.Cells for Java 24.7
* Поддержка функции DROP в формулах
* Улучшение производительности вычисления функций LET и MAKEARRAY со сложными параметрами
* Поддержка контура звездных символов в диаграмме
* Поддержка создания объекта как декоративного
* Поддержка перемещения серий
* Улучшение отображения карты дерева
* Поддержка преобразования изображения "перекрытой ячейки" в "в ячейке"
* Поддержка загрузки локального шрифта облачного офиса по умолчанию
* Поддержка сохранения схемы JSON при импорте JSON-данных
* Поддержка экспорта в формат файлов AZW3
* Поддержка автоматической установки высоты после установки переноса для ячейки для GridJS
* Поддержка генерации JSON с использованием способа записи потока для GridJS
