На этой странице описаны наиболее интересные новые функции Aspose.Cells, представленные в последних выпусках.

##  Aspose.Cells for Java 23.12

* Поддержка соответствия PDF/A-2 и PDF/A-3 при преобразовании в PDF.
* Поддержка расчета дискретной группировки сводной таблицы.
* Поддержка экспорта данных ячеек в общий список.
* Поддержка импорта перекрестных ссылок с использованием связанных таблиц для интеллектуальных маркеров.
* Поддержка функции MAKEARRAY для формул.

##  Aspose.Cells for Java 23.11

* Поддержка преобразования электронных таблиц в файл PDF версии 1.7 (ISO32000-1:2008).

##  Aspose.Cells for Java 23.10

* Поддержка установки гиперссылок непосредственно на диапазон.
* Поддержка одновременного копирования нескольких листов
* Поддержка проверки того, пуст ли один диапазон для данных ячеек.

##  Aspose.Cells for Java 23.9

* Поддержка расчета новых функций по формуле: LAMBDA,LET,MAP,SCAN.
* Поддержка импорта/экспорта файлов EPUB.
* Поддержка фильтрации серий диаграмм.

##  Aspose.Cells for Java 23.8

* Поддержка сохранения пространства заполнения в формулах.
* Поддержка замены/вставки текста с форматированием расширенного текста ячейки.

##  Aspose.Cells for Java 23.7

* Улучшено определение формата файлов для некоторых новых форматов, таких как файл visio, файл пакета OneNote.

##  Aspose.Cells for Java 23.6

* Поддержка экспорта и выделения журналов изменений.
* Поддержка перемещения существующих объектов Row из RowCollection в обратном порядке.
* Поддержка обновления нескольких диапазонов консолидации сводной таблицы.

##  Aspose.Cells for Java 23.5

* Поддержка обновления внешних ссылок данных внешнего листа на локальный лист текущей книги.
* Поддержка отображения элемента управления Checkbox как интерактивного элемента управления вместо статического изображения в GridWeb.
* Поддержка выбора PivotArea и ее форматирования.
* Поддержка добавления водяного знака при конвертации в PDF.

##  Aspose.Cells for Java 23.4

* Поддержка расчета функции ENCODEURL
* Поддержка завершения построения формы уравнения путем пошаговой вставки соответствующих узлов.
* Поддержка расчета выражения формулы как формулы массива без установки его в объект ячейки.
* Поддержка сохранения файла xlsb в режиме LightCells.
* Поддержка проверки того, установлен или доступен ли один шрифт.
* Поддержка установки формулы в указанную ячейку в ListObject.
* Поддержка сохранения книги в формате xhtml.
* Поддержка рендеринга элементов управления activex и элементов управления формой для GridWeb.

##  Aspose.Cells for Java 23.3

* Поддержка установки/чтения/сохранения новых функций: LET, ENCODEURL.
* Поддержка связывания источников данных с внешними ссылками при вызове расчета по формуле.

##  Aspose.Cells for Java 23.2

* Поддержка установки/чтения/сохранения новых функций: СКАНИРОВАНИЕ, ЛЯМБДА.
* Поддержка экспорта стиля в json

##  Aspose.Cells for Java 23.1

* Поддержка новых функций: CHOOSEROWS, CHOOSECOLS, HSTACK, VSTACK.
* Поддержка получения автора и даты и времени из редакций.

##  Aspose.Cells for Java 22.12

* Поддержка экспорта Excel в номер JSON с многоуровневой иерархической структурой.

##  Aspose.Cells for Java 22.11

* Поддержка различия между формулой динамического массива и формулой устаревшего массива.
* Поддержка чтения/сохранения формул динамического массива для формата файла xlsb.

##  Aspose.Cells for Java 22.10

* Поддержка добавления и обновления диаграммы с помощью файла шаблона CRTX.

##  Aspose.Cells for Java 22.9

* Поддержка установки формулы таблицы для ячеек.
* Поддержка установки формулы динамического массива с настраиваемой функцией и механизмом вычислений.
* Поддержка установки предустановленных настроек градиента для формата градиентной заливки.
* Поддержка применения контурных границ с цветами темы для диапазона.

##  Aspose.Cells for Java 22.8

* Поддержка экспорта комментариев и заметок, отображаемых на листе, в HTML.
* Поддержка указания листов для вывода при преобразовании в pdf/xps.

##  Aspose.Cells for Java 22.7

* Поддержка чтения файлов шаблонов формата OOXML, которые содержат ненормальные данные zip и не могут быть обработаны Java ZipInputStream.
* Поддержка экспорта HTML в один файл с верхним/нижним колонтитулом страницы.
* Поддержка получения абсолютной позиции и размера диапазона (в пунктах).

##  Aspose.Cells for Java 22.6

* Улучшение расчета и отслеживания формул с помощью цепочки вычислений формул.

##  Aspose.Cells for Java 22.5

* Улучшение производительности памяти при настройке формулы ячеек при сохранении книги в режиме LightCells.

##  Aspose.Cells for Java 22.4

* Расширенные возможности управления пустыми ячейками для экспорта CSV.
* Поддержка сложного разделения и обновления условного форматирования для операций вставки/удаления.

##  Aspose.Cells for Java 22.3

* Поддержка повторного расчета ячеек, которые зависят от определенного имени, когда определенное имя изменено и включена цепочка вычислений книги.
* Поддержка рекурсивного получения листов ячейки в соответствии с цепочкой вычислений формул.

##  Aspose.Cells for Java 22.2

* Поддержка извлечения определенных имен из NameCollection с помощью указанного фильтра области.

##  Aspose.Cells for Java 22.1

* Поддержка создания редактируемого pptx
* Поддержка импорта большого CSV-файла в несколько листов, когда строка/столбец данных превышает лимит.
* Поддержка импорта CSV с помощью LightCells
* Поддержка удаления указанной внешней ссылки
* Поддержка сохранения формул при удалении внешних ссылок.

##  Aspose.Cells for Java 21.12

* Поддержка преобразования Excel в сценарии SQL.
* Поддержка импорта и экспорта данных в формате XML.
* Поддержка чтения как можно большего количества данных из поврежденного файла xls.
* Поддержка прерывания процесса SmartMarker
* Повышение производительности при добавлении диапазонов для условного форматирования и проверки.
* Установите наивысший приоритет для вновь добавленного условия форматирования, как и в случае с MS Excel.

##  Aspose.Cells for Java 21.11

* Поддержка вычисления формулы/функции XLookup.
* Интегрируйте сохранение JSON и изображений в сохранение книги.
* Повысьте производительность при преобразовании Excel в HTML.
