## Основы GridJs

Aspose.Cells.GridJs - это библиотека .NET Standard, которая позволяет пользователям разрабатывать веб-приложения для быстрого и простого отображения/редактирования электронных таблиц. 

Aspose.Cells.GridJs поддерживает импорт популярных форматов файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, CSV, SpreadsheetML, ODS).

Также позволяет экспортировать файлы Excel в PDF, HTML и т. д. Ниже приведены основные этапы разработки веб-приложения GridJs.

- Реализация GridCacheForStream для написания собственной бизнес-логики для кэширования.
- Настройка действия контроллера для получения JSON из файла электронной таблицы. Вы можете использовать API GridJsWorkbook.ImportExcelFile и GridJsWorkbook.ExportToJson, GridJs автоматически сохранит файл электронной таблицы в кэше.
- Настройка действия контроллера для получения JSON для операции обновления. Вы можете использовать API GridJsWorkbook.UpdateCell, GridJs выполнит операцию обновления в кэше и вернет обновленный JSON.
- Настройка действия контроллера для получения URL файлов изображений/форм в электронной таблице, GridJs автоматически создаст архив всех изображений/форм в кэше. Будет использовано API GridCacheForStream.GetFileUrl.
- Настройка действия контроллера для загрузки файла из кэша, таким образом, можно получить архив изображений/форм или файл электронной таблицы в кэше. Будет использовано API GridCacheForStream.LoadStream.
- Настройка действия контроллера для загрузки электронной таблицы. Вы можете использовать API GridJsWorkbook.SaveToCacheWithFileName.
