{{% alert color="primary" %}} 

Aspose.Cells for Java - это библиотека классов, которая позволяет вашим приложениям выполнять широкий спектр задач по обработке таблиц. Aspose.Cells for Java поддерживает популярные форматы файлов таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) и позволяет экспортировать файлы Excel в форматы PDF, XPS, HTML, MHTML, обычный текст, TIFF, JPG, PNG, BMP и SVG.

Последнее Справочное руководство по API можно найти по адресу [Справочное руководство по API Aspose.Cells for Java](https://reference.aspose.com/cells/java).
