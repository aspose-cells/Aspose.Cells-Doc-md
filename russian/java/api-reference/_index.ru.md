{{% alert color="primary" %}} 

Aspose.Cells for Java — это библиотека классов, которая позволяет вашим приложениям выполнять широкий спектр задач по обработке электронных таблиц. Aspose.Cells for Java поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) и позволяет экспортировать файлы Excel в PDF, XPS, HTML, MHTML, обычный текст, TIFF, JPG, PNG, BMP и SVG.

Последний номер API можно найти по адресу[Aspose.Cells for Java API Ссылка](https://reference.aspose.com/cells/java).
