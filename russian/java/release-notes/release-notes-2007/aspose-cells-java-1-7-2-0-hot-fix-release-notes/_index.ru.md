 Эта страница содержит примечания к выпуску для[Aspose.Cells Java 1.7.2.0 Оперативное исправление](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.7.2.0-hot-fix/)

 Дорогие клиенты,

 Мы выпустили Aspose.Cells for Java V1.7.2! Включены следующие улучшения и исправления:

- Поддерживает добавление изображения в верхний/нижний колонтитул страницы настройки.
 Поддерживает импорт настроек страницы, имени, защиты, проверки данных и свойств документа из файла SpreadsheetML.
 Поддерживает экспорт настроек страницы, имени, защиты, проверки данных и свойств документа в файл SpreadsheetML.
 Поддерживает вычисление большего количества встроенных функций
 Исправлена ошибка установки автоматического цвета шрифта на графике.
