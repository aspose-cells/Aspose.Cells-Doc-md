 Эта страница содержит примечания к выпуску для[Aspose.Cells for Java 7.1.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.0/)

{{% /alert %}} 

 Мы рады объявить Aspose.Cells for Java v7.1.0!

 Улучшения

- Удалить файлы шаблонов после чтения файлов XLSX/ODS с помощью инструмента Apache zip

 Исключения

 -Cell.setHtmlString() выдает исключение NullPointerException

 Ошибки

 -Cell содержимое исчезло из сгенерированного файла PDF

 -Сообщение об ошибке: «Сводная таблица была создана в более поздней версии Excel…» при открытии файла в MS Excel.

 ` ` - Масштаб листа изменяется при открытии файла шаблона XLS.

 -Cell.getStringValue() дает неверное значение для типов double

 - При сохранении в PDF правая часть текста фигуры обрезается, а часть изображения теряется.

 -Формат номера теряется при сохранении в формате файла ODS
