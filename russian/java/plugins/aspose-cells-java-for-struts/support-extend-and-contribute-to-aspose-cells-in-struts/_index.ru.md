## **Поддержка**
- Если вы хотите увидеть известные / сообщенные проблемы (пользователями или командой Q.A.) в приложении.
- Или вы хотите сообщить о любой проблеме, которую нашли в приложении.
- Если у вас есть предложение по улучшению или вы хотите сделать запрос на добавление новой функции

**Пожалуйста, используйте любой из указанных инструментов для отслеживания проблем в проекте:**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **Расширение и вклад**
Aspose.Cells Java для Struts 1.3 является открытым исходным кодом, и его исходный код доступен на основных веб-сайтах социального кодирования, указанных ниже. Разработчиков призывают скачать исходный код и внести свой вклад, предлагая или добавляя новые функции или улучшая существующие, чтобы другие могли также воспользоваться им.
