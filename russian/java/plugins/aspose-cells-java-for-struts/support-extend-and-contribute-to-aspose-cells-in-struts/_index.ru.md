## **Поддерживать**
- Если вы хотите видеть в приложении известные/сообщенные проблемы (пользователями или командой контроля качества).
- Или вы хотите сообщить о любой проблеме, которую вы обнаружили в приложении
- У вас есть предложения по улучшению или вы хотели бы сделать запрос на какую-либо функцию

**Пожалуйста, используйте любой из следующих трекеров проекта:**

- [Гитхаб](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **Расширяйте и вносите свой вклад**
Aspose.cells Java для Struts 1.3 является открытым исходным кодом, и его исходный код доступен на основных веб-сайтах социального кодирования, перечисленных ниже. Разработчикам рекомендуется загружать исходный код и вносить свой вклад, предлагая или добавляя новые функции или улучшая существующие, чтобы другие также могли извлечь из этого пользу.
