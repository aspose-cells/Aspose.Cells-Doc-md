## **Веб-фреймворк Apache Struts**
Веб-инфраструктура Apache Struts — это бесплатное решение с открытым исходным кодом для создания веб-приложений Java.

Веб-приложения отличаются от обычных веб-сайтов тем, что веб-приложения могут создавать динамический отклик. Многие веб-сайты предоставляют только статические страницы. Веб-приложение может взаимодействовать с базами данных и механизмами бизнес-логики для настройки ответа.

Одним из способов разделения задач в программном приложении является использование архитектуры Model-View-Controller (MVC). Модель представляет бизнес-код или код базы данных, Представление представляет код дизайна страницы, а Контроллер представляет навигационный код. Платформа Struts призвана помочь разработчикам создавать веб-приложения, использующие архитектуру MVC.

[Учить больше](http://struts.apache.org/birdseye.html)
### **Aspose.Cells for Java**
Aspose.Cells for Java — это расширенная библиотека классов for Java, которая позволяет выполнять широкий спектр задач по обработке документов непосредственно в вашем Java
Приложения.

Aspose.Cells for Java поддерживает обработку Cells (DOC, DOCX, OOXML, RTF) HTML, OpenDocument, PDF, EPUB, XPS, SWF и все форматы изображений. С Aspose.Cells вы можете
создавать, изменять и преобразовывать документы без использования Microsoft Cells.
## **Системные Требования**
**Ниже приведены системные требования для Aspose.Cells Java для веб-приложения Struts 1.3:**

- Java Установлена версия 1.6 или выше.
- Apache Tomcat Server 7 или выше
- Maven 3
- Apache Struts 1.3
- Aspose.Cells компонент.
## **Поддерживаемые платформы**
**Ниже приведены поддерживаемые платформы:**

- Aspose.Cells 8.5.0 или выше.
- Java IDE (Eclipse, NetBeans, IntelliJ, ...).
