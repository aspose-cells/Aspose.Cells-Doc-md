## **Веб-фреймворк Apache Struts**
Веб-фреймворк Apache Struts - бесплатное решение с открытым исходным кодом для создания веб-приложений на Java.

Веб-приложения отличаются от обычных веб-сайтов тем, что веб-приложения могут создавать динамический ответ. Многие сайты предоставляют только статические страницы. Веб-приложение может взаимодействовать с базами данных и бизнес-логикой для настройки ответа.

Один из способов разделения задач в программном приложении - использование архитектуры Model-View-Controller (MVC). Модель представляет бизнес-или базовый код, Представление представляет код дизайна страницы, а контроллер представляет навигационный код. Фреймворк Struts создан для помощи разработчикам в создании веб-приложений, использующих архитектуру MVC.

[Узнать больше](http://struts.apache.org/birdseye.html)
### **Aspose.Cells for Java**
Aspose.Cells for Java - это продвинутая библиотека классов для Java, позволяющая выполнять широкий спектр задач по обработке документов напрямую в Java
приложениях.

Aspose.Cells for Java поддерживает обработку ячеек (DOC, DOCX, OOXML, RTF) HTML, OpenDocument, PDF, EPUB, XPS, SWF и все форматы изображений. С помощью Aspose.Cells вы можете
создавать, изменять и преобразовывать документы, не используя пакет Microsoft Office.
## **Системные требования**
**Ниже приведены системные требования для Aspose.Cells Java для веб-приложения Struts 1.3:**

- Установлен Java 1.6 или выше.
- Сервер Apache Tomcat 7 или выше
- Maven 3
- Apache Struts 1.3
- Компонент Aspose.Cells.
## **Поддерживаемые платформы**
**Ниже приведены поддерживаемые платформы:**

- Aspose.Cells 8.5.0 или выше.
- Java IDE (Eclipse, NetBeans, IntelliJ, ...)
