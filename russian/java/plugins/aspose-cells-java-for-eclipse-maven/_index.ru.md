## **Введение**
### **Eclipse IDE**
Среда Eclipse IDE - известное интегрированное средство разработки (IDE) для Java. Сегодня это ведущая среда разработки для Java с долей рынка около 60%.

Среда разработки Eclipse IDE может быть расширена дополнительными программными компонентами. Eclipse называет эти компоненты программными дополнениями. Несколько проектов с открытым исходным кодом и компаний расширили среду Eclipse IDE или создали автономные приложения (Eclipse RCP) на основе среды Eclipse.
### **Aspose.Cells for Java**
[Aspose.Cells for Java](http://goo.gl/gtvhlQ) - это награжденный компонент таблицы Excel, который позволяет разработчикам Java встраивать возможность чтения, записи и манипулирования электронными таблицами Excel® (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF и изображениями в свои Java-приложения без необходимости полагаться на Microsoft Excel®.

[Aspose.Cells for Java](http://goo.gl/gtvhlQ) - это зрелый, масштабируемый и функционально насыщенный компонент, который предлагает множество функций, выходящих за рамки простых возможностей экспорта данных других поставщиков. С помощью Aspose.Cells for Java разработчики могут экспортировать данные, форматировать электронные таблицы на самом мелком уровне, импортировать изображения, создавать диаграммы, применять и вычислять сложные формулы, передавать данные Excel®, сохранять в различных форматах и многое другое - все это без необходимости использования Microsoft Excel® или Microsoft Office Automation.
### **Aspose.Cells Java for Eclipse (Maven)**
- Aspose.Cells Java для Eclipse (Maven) - это Плагин для **среды Eclipse IDE** от **Aspose.** Этот Плагин предназначен для разработчиков, использующих платформу Maven для разработки на Java и желающих использовать [Aspose.Cells for Java](http://goo.gl/gtvhlQ) в своих проектах. Плагин позволяет создавать проекты maven для использования [API Aspose.Cells for Java](http://goo.gl/gtvhlQ) и также загружать [Примеры кода](https://goo.gl/uxznrb) API. 
- Плагин предоставляет следующие возможности для работы с [API Aspose.Cells for Java](http://goo.gl/gtvhlQ) в **среде Eclipse IDE** удобно:

Плагин содержит два мастера-----

**       **Мастер создания проекта Maven Aspose.Cells (мастер)**

- Этот мастер создания нового проекта позволяет разработчикам создать проект Maven для использования [Aspose.Cells for Java](http://goo.gl/gtvhlQ) из **Новый -> Проект -> Maven-> Aspose.Cells Maven Project.**
- Ссылка на maven зависимость от [API Aspose.Cells for Java](http://goo.gl/gtvhlQ) автоматически извлекается из [Репозитория Aspose Cloud Maven](https://repository.aspose.com/webapp/) и добавляется в pom.xml.
- Созданный проект всегда будет содержать самую последнюю доступную версию зависимости Maven для [API Aspose.Cells for Java](http://goo.gl/gtvhlQ).
- Шаги мастера также предлагают опцию для загрузки [Примеров кода](https://goo.gl/uxznrb) для использования [API Aspose.Cells for Java](http://goo.gl/gtvhlQ).

**       **Пример кода Aspose.Cells (мастер)**

- Мастер создания нового файла позволяет скопировать загруженные [Примеры кода](https://goo.gl/uxznrb) в ваш проект для использования [Aspose.Cells for Java](http://goo.gl/gtvhlQ) из **New -> Other -> Java -> Пример кода Aspose.Cells.** 
- Доступные образцы отображаются в виде древовидной структуры, где пользователь может выбрать их категорически.
- Все примеры в выбранной категории будут скопированы в папку "**com.aspose.cells.examples**" проекта вместе с необходимыми ресурсами в папке "**src/main/resources**", необходимыми для запуска примеров.
- [Примеры кода](https://goo.gl/uxznrb) API [Aspose.Cells for Java](http://goo.gl/gtvhlQ) предназначены для демонстрации различных возможностей API.
- Мастер также будет искать и обновлять доступные новые [Примеры кода](https://goo.gl/uxznrb) из [репозитория примеров Aspose.Cells for Java.](https://goo.gl/uxznrb)
## **Требования к системе и поддерживаемые платформы**
### **Системные требования**
- **Системная память:** 2 ГБ или более (рекомендуется)
- **ОС:** Любая операционная система, поддерживающая Java VM (виртуальная машина)
- **Интернет-соединение:** 2 МБ или быстрее (рекомендуется)
### **Поддерживаемые платформы**
- Рекомендуется Eclipse Mars.1 (4.5.1)
- Eclipse Juno или более поздняя версия.

### **Как применить лицензию Aspose?**
Этот плагин использует оценочную версию Aspose.Cells. Когда вы будете довольны оценкой, вы сможете приобрести лицензию на [веб-сайте Aspose](https://purchase.aspose.com/buy).
Для удаления сообщения об оценке и ограничений функций необходимо применить лицензионный ключ продукта. После приобретения продукта вы получите файл лицензии. Пожалуйста, выполните следующие шаги для применения лицензии

- Убедитесь, что файл лицензии называется **Aspose.Cells.Java.lic**
- Разместите файл **Aspose.Cells.Java.lic** в папке, содержащей Aspose.Cells.jar
- Используйте следующий код для активации лицензии:

{{< highlight java >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **Как настроить исходный код Aspose.Cells Java для Eclipse (Maven)**
Ниже простые шаги плавно приведут к успешной настройке **"Aspose.Cells Java для Eclipse (Maven)"** исходного кода плагина в среде разработки Eclipse

1. Загрузите / Клонируйте исходный код.
1. Выберите **Файл** > **Импорт > Общее > Существующие проекты в рабочей области**
1. Перейдите к последнему исходному коду проекта, который вы загрузили
1. Выберите проект Eclipse, который вы хотите импортировать
1. Нажмите **Завершить**
1. Плагин Aspose.Cells Java для Eclipse теперь готов для улучшения.
