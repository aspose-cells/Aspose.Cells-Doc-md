## **Вступление**
### **Затмение IDE**
Eclipse IDE — известная Java интегрированная среда разработки (IDE). IDE, безусловно, является самым известным продуктом проекта Eclipse с открытым исходным кодом. На сегодняшний день это ведущая среда разработки for Java с долей рынка около 60%.

Eclipse IDE может быть дополнен дополнительными программными компонентами. Eclipse называет эти программные компоненты подключаемыми модулями. Несколько проектов и компаний с открытым исходным кодом расширили Eclipse IDE или создали автономные приложения (Eclipse RCP) поверх среды Eclipse.
### **Aspose.Cells for Java**
[Aspose.Cells for Java ](http://goo.gl/gtvhlQ)is an award-winning Excel Spreadsheet component that allows Java developers to embed the ability to read, write and manipulate Excel® spreadsheets (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF and image форматы файлов в свои собственные Java приложения без необходимости полагаться на Microsoft Excel®.

[Aspose.Cells for Java](http://goo.gl/gtvhlQ)является зрелым, масштабируемым и многофункциональным компонентом, который предлагает множество функций, выходящих далеко за рамки простых возможностей экспорта данных других поставщиков. С помощью Aspose.Cells for Java разработчики могут экспортировать данные, форматировать электронные таблицы с максимальной детализацией, импортировать изображения, создавать диаграммы, применять и вычислять сложные формулы, передавать данные Excel® в потоковом режиме, сохранять в различных форматах и многое другое — и все это без использования Microsoft Excel®. или Microsoft Автоматизация офиса.
### **Aspose.Cells Java для Eclipse (Maven)**
-  Aspose.Cells Java для Eclipse (Maven) — это плагин для**Затмение IDE** к**Aspose.** Этот плагин предназначен для разработчиков, использующих разработки платформы Maven for Java и желающих использовать[Aspose.Cells for Java](http://goo.gl/gtvhlQ) в своих проектах. Плагин позволяет создавать проекты maven для использования[Aspose.Cells for Java API](http://goo.gl/gtvhlQ) а также скачать[Примеры кода](https://goo.gl/uxznrb)API.

**ВОЛШЕБНИКИ**:
Плагин содержит два мастера -----

**       ****Aspose.Cells Maven Проект (мастер)**

- Этот мастер создания нового проекта позволяет разработчикам создавать**Maven** проект по использованию[Aspose.Cells for Java](http://goo.gl/gtvhlQ) от**Новый -> Проект -> Maven-> Aspose.Cells Maven Проект.**
- Ссылка[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)maven зависимость автоматически извлекается из[Aspose Облако Maven Репозиторий](https://repository.aspose.com/webapp/)и добавляется в pom.xml.
-  Созданный проект всегда будет содержать самую последнюю доступную версию**Maven** Зависимость для[Aspose.Cells for Java API](http://goo.gl/gtvhlQ).
-  Шаги мастера также предоставляют возможность загрузки[Примеры кода](https://goo.gl/uxznrb)для использования[Aspose.Cells for Java API](http://goo.gl/gtvhlQ).

**       ****Aspose.Cells Пример кода (мастер)**

-  Этот мастер создания нового файла позволяет скопировать загруженный[Примеры кода](https://goo.gl/uxznrb) в свой проект для использования[Aspose.Cells for Java](http://goo.gl/gtvhlQ) от**Новый -> Другое -> Java -> Aspose.Cells Пример кода.** 
- Доступные образцы отображаются в виде дерева, из которого пользователь может выбрать их по категориям.
- Все примеры в выбранной категории будут скопированы в "**com.aspose.cells.examples**" папка пакета вместе с необходимыми ресурсами внутри "**источник/основной/ресурсы**" папка, необходимая для запуска примеров.
- [Примеры кода](https://goo.gl/uxznrb) из[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)предназначены для демонстрации различных API.
- Мастер также будет искать и обновлять новые доступные[Примеры кода](https://goo.gl/uxznrb)от[Aspose.Cells for Java хранилище примеров.](https://goo.gl/uxznrb)
## **Системные требования и поддерживаемые платформы**
### **Системные Требования**
- **Системная память:**2 ГБ или более (рекомендуется)
- **ОПЕРАЦИОННЫЕ СИСТЕМЫ:**Любая операционная система, поддерживающая виртуальную машину Java (виртуальная машина).
- **Интернет-соединение:**2 МБ или больше (рекомендуется)
### **Поддерживаемые платформы**
- Затмение Марс.1 (4.5.1) - рекомендуется
- Затмение Юнона или более поздняя версия.

### **Как применить лицензию Aspose?**
 Этот подключаемый модуль использует ознакомительную версию Aspose.Cells. Если вы удовлетворены своей ознакомительной версией, вы можете приобрести лицензию в[Aspose сайт](https://purchase.aspose.com/buy).
Чтобы удалить оценочное сообщение и ограничения функций, необходимо применить лицензию на продукт. Вы получите файл лицензии после покупки продукта. Чтобы применить лицензию, выполните следующие действия.

-  Убедитесь, что файл лицензии называется**Aspose.Cells.Java.lic**
-  Место**Aspose.Cells.Java.lic** файл в папке, содержащей Aspose.Cells.jar
- Используйте следующий код для активации лицензии:

{{< highlight "java" >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **Как настроить исходный код Aspose.Cells Java для Eclipse (Maven)**
Ниже приведены простые шаги, которые плавно приведут к успешной настройке**"Aspose.Cells Java для Eclipse (Maven)"** исходный код плагина в Eclipse IDE

1. Скачать/клонировать исходный код.
1. выберите**Файл** > **Импорт > Общие > Существующие проекты в рабочую область**
1. Перейдите к последнему исходному коду проекта, который вы загрузили.
1. Выберите проект Eclipse, который вы хотите импортировать.
1. Нажмите**Заканчивать**
1. Aspose.Cells Java для кода подключаемого модуля Eclipse теперь можно улучшить.
