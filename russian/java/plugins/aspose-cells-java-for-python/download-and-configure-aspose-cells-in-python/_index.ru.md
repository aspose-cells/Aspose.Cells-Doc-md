## **Скачать необходимые библиотеки**
Загрузите необходимые библиотеки, указанные ниже. Они необходимы для выполнения примеров Aspose.Cells Java для Python.

- **Aspose:** [Aspose.Cells for Java Компонент](https://downloads.aspose.com/cells/java/)
- [JPype](https://pypi.org/project/JPype1/)
## **Загрузите примеры с сайтов социального кодирования**
Следующие выпуски работающих примеров доступны для загрузки на указанных ниже сайтах социального кодирования:

-----
### **Гитхаб**
- **Aspose.Cells Java для Python Примеры** 
  - [Aspose.Cells Java для Python](https://github.com/aspose-cells/Aspose.Cells-for-Java/releases/tag/Aspose.Cells_Java_for_Python-v1.0)
## **Как настроить исходный код**
Пожалуйста, следуйте этим простым шагам, чтобы открыть и расширить исходный код при использовании:

 \1. Устанавливайте только Python 2.7.xx из:<https://www.python.org/downloads/>
 \2. Установите JPype из:<https://sourceforge.net/projects/jpype/files/JPype/>
 \3. Установите Java JDK из:<https://www.oracle.com/java/technologies/downloads/>
 \4. Загрузите Aspose.Cells for Java API-интерфейсы, которые вы хотите использовать в своих проектах Python, по адресу:<https://downloads.aspose.com/cells/java> и скопируйте загруженные API-интерфейсы .jars в папку %Python%/lib.
 \5. Клон[Aspose.Cells Java для Python Примеры](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) с гитхаба.
