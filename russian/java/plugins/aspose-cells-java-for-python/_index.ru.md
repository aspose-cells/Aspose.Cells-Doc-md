## **JPype**

JPype - это усилие для предоставления полного доступа программ Python к библиотекам классов Java. Это достигается не через переосуществление Python, как сделано в Jython/JPython, а скорее через взаимодействие на уровне нативных машин в обоих виртуальных машинах.

Узнать больше на [readthedocs.org](https://jpype.readthedocs.io/en/latest/userguide.html)

## **Aspose.Cells for Java**

Aspose.Cells for Java - это продвинутая библиотека классов для Java, которая позволяет выполнять широкий спектр задач по обработке документов непосредственно в ваших приложениях Java.

Aspose.Cells for Java поддерживает обработку файлов Excel (XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM), CSV (значения, разделенные запятыми), TSV (значения, разделенные табуляцией), TabDelimited (текстовый файл с разделением табулятором), Html, MHtml, ODS (документ OpenDocument Spreadsheet), SpreadsheetML и Numbers (документ, созданный приложением Apple "Numbers"). С помощью Aspose.Cells вы можете создавать, изменять и преобразовывать документы без использования Microsoft Excel.

## **Aspose.Cells Java for Jython**

Aspose.Cells Java для Python - проект, демонстрирующий/предоставляющий примеры использования API Aspose.Cells for Java на Python.
