 Линии сетки настраиваются в**Aspose.Cells.ReportingServices.xml** таким образом, вы можете управлять отображением линий сетки в экспортированном Microsoft файле Excel. Значение по умолчанию**выключенный**, что означает, что линии сетки не отображаются.

{{< highlight "java" >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
