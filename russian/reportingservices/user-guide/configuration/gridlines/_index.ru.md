Настройки линий сетки настраиваются в **Aspose.Cells.ReportingServices.xml**, поэтому вы можете контролировать отображение линий сетки в экспортированном файле Microsoft Excel. Значение по умолчанию - **выключено**, что означает, что линии сетки не отображаются.

{{< highlight java >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
