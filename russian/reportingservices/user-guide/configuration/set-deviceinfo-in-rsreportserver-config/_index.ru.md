- **FileExtension** 
  Когда значение равно null, название расширения экспортируемого файла отчета будет значением по умолчанию. Когда значение не равно null, расширение экспортируемого файла отчета будет этим значением.
- **SimplePageHeaders** 
  Когда значение равно true, он рендерит элемент заголовка отчета на заголовке страницы Excel. Значение по умолчанию false.
- **SimplePageFooters** 
  Когда значение равно true, он рендерит элемент нижнего колонтитула отчета на нижнем колонтитуле Excel. Значение по умолчанию true.
- **PutoutHeader** 
  Когда значение равно true, отчет заголовка будет экспортирован. Когда значение равно false, отчет заголовка не будет экспортирован. Значение по умолчанию true. Значение поддерживается только расширением Excel2007Xlsx(Только данные).
- **PutoutFooter** 
  Когда значение равно true, отчет футера будет экспортирован. Когда значение равно false, отчет футера не будет экспортирован. Значение по умолчанию true. Значение поддерживается только расширением Excel2007Xlsx(Только данные).
- **FillTableGroupHeaderForSimpleOutPut** 
  Значение по умолчанию - false. Значение поддерживает только расширение Excel2007Xlsx (только данные).
- **NoOutPutTotalForSimpleOutPut** 
  Значение по умолчанию - false. Значение поддерживает только расширение Excel2007Xlsx (только данные).
- **NoOutPutGroupForSimpleOutPut** 
  Значение по умолчанию - false. Значение поддерживает только расширение Excel2007Xlsx (только данные).
- **NoDoGroupPageForSimpleOutPut** 
  Значение по умолчанию - true. Значение поддерживает только расширение Excel2007Xlsx (только данные).
- **NoDoPageForSimpleOutPut** 
  Значение по умолчанию - true. Значение поддерживает только расширение Excel2007Xlsx (только данные).
- **FieldDelimiter** 
  Устанавливает разделители полей. Значение поддерживает расширения CSV и TXT.
