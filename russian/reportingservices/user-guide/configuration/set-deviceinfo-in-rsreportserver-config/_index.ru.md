- **Расширение файла** 
 Если значение равно null, расширение экспортированного файла отчета будет значением по умолчанию. Если значение не равно null, имя расширения экспортированного файла отчета равно value.
- **SimplePageHeaders** 
 Когда значение равно true, элемент заголовка отчета отображается в заголовок страницы Excel. Значение по умолчанию неверно.
- **SimplePageFooters** 
 Если значение равно true, элемент нижнего колонтитула отчета отображается в нижнем колонтитуле страницы Excel. Значение по умолчанию верно.
- **PutoutHeader** 
 Когда значение равно true, он экспортирует элемент заголовка отчета. Если значение равно false, элемент заголовка отчета не экспортируется. Значение по умолчанию верно. Это значение поддерживает только расширение Excel2007Xlsx (только данные).
- **PutoutFooter** 
Если значение равно true, будет экспортирован элемент нижнего колонтитула отчета. Если значение равно false, элемент нижнего колонтитула отчета не экспортируется. Значение по умолчанию верно. Это значение поддерживает только расширение Excel2007Xlsx (только данные).
- **Филтаблеграупхеадерфорсимплеаутпут** 
 Значение по умолчанию неверно. Это значение поддерживает только расширение Excel2007Xlsx (только данные).
- **Ноаутпуттоталфорсимплеаутпут** 
 Значение по умолчанию неверно. Это значение поддерживает только расширение Excel2007Xlsx (только данные).
- **ноаутпутграупфорсимплеаутпут** 
 Значение по умолчанию неверно. Это значение поддерживает только расширение Excel2007Xlsx (только данные).
- **Нодограуппажефорсимплеаутпут** 
 Значение по умолчанию верно. Это значение поддерживает только расширение Excel2007Xlsx (только данные).
- **Нодопажефорсимплеаутпут** 
 Значение по умолчанию верно. Это значение поддерживает только расширение Excel2007Xlsx (только данные).
- **Разделитель полей** 
 Он устанавливает разделители полей. Значение поддерживает расширения CSV и TXT.
