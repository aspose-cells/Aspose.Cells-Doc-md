 Раздел DeviceInfo в**rsreportserver.config** принимает следующие параметры, влияющие на поведение Aspose.Cells':

- **Расширение файла** : когда значение**нулевой**, расширение экспортированного файла отчета является значением по умолчанию. Если значение не равно null, расширение отчета устанавливается равным этому значению.
- **SimplePageHeaders** : когда значение**истинный** , элемент заголовка отчета отображается как заголовок страницы Microsoft Excel. Значение по умолчанию**ЛОЖЬ**.
- **SimplePageFooters** : когда**истинный** , элемент нижнего колонтитула отчета отображается как нижний колонтитул страницы Microsoft Excel. Значение по умолчанию**истинный**.
- **PutoutHeader** : когда**истинный** (по умолчанию) элемент заголовка отчета экспортируется. Когда**ЛОЖЬ**, элемент заголовка отчета не экспортируется. Поддерживает только расширение Excel 2007 XLSX (только данные).
- **PutoutFooter** : когда**истинный** (по умолчанию) элемент нижнего колонтитула отчета экспортируется. Когда**ЛОЖЬ**нет, это не так. Поддерживает только расширение Excel 2007 XLSX (только данные).
- **Филтаблеграупхеадерфорсимплеаутпут**: **ЛОЖЬ** по умолчанию. Это значение поддерживает только расширение Excel 2007 XLSX (только данные).
- **Ноаутпуттоталфорсимплеаутпут**: **ЛОЖЬ** по умолчанию. Это значение поддерживает только расширение Excel 2007 XLSX (только данные).
- **ноаутпутграупфорсимплеаутпут**: **ЛОЖЬ** по умолчанию. Это значение поддерживает только расширение Excel 2007 XLSX (только данные).
- **Нодограуппажефорсимплеаутпут**: **истинный** по умолчанию. Это значение поддерживает только расширение Excel 2007 XLSX (только данные).
- **Нодопажефорсимплеаутпут**: **истинный** по умолчанию. Это значение поддерживает только расширение Excel 2007 XLSX (только данные).
- **Разделитель полей**: устанавливает разделители полей. Значение поддерживает расширения CSV и TXT.
