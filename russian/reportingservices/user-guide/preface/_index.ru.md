{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services в основном содержит два компонента: Aspose.Cells.Report.Designer и Aspose.Cells.Renderer для Reporting Services. Первый используется для проектирования отчетов напрямую в Microsoft Excel, а второй отвечает за отображение RDL-отчета в Microsoft Excel. 

{{% /alert %}} 
### **Проектирование отчета с помощью конструктора отчетов**
Основные шаги проектирования отчета с использованием Aspose.Cells.Report.Designer:

1. **Создание источников данных и запросов**.
   Microsoft Query интегрирован с Aspose.Cells.Report.Designer и используется в качестве графического инструмента для создания источников данных и запросов. Пользователи также могут использовать существующий файл RDL, в котором источники данных и запросы доступны для операций.
1. **Сопоставление параметров**.
   Если запросы SQL включают параметры, пользователи должны создать параметры отчета и сопоставить SQL-параметры с параметрами отчета. В Aspose.Cells.Report.Designer вы можете указать допустимые значения для параметра отчета.
1. **Проектирование содержимого, стилей и форматов шаблона отчета Microsoft Excel**.
   Шаблон отчета Aspose.Cells может содержать любое количество следующих типов элементов отчета: 
   1. Таблица
   1. Сводная таблица
   1. Диаграмма
   1. Текстовое поле
   1. Матрица
      Обычно запрос (то есть DataSet) используется в качестве источника данных для элемента отчета. Вы также можете использовать параметры Reporting Services, формулы и глобальные переменные как источник данных для некоторых типов элементов отчета. Стили и форматы элементов отчета указываются просто установкой стилей и форматов ячеек, составляющих элементы отчета.
1. **Опубликовать отчет**.
   После выполнения вышеуказанных шагов отчет готов к публикации. Пользователи могут указать каталог, в который будет опубликован отчет. При необходимости вы можете назначить общий источник данных на сервере отчетов в качестве источника данных для отчета.
1. **Предварительный просмотр отчета**.
   При выборе отчета для предварительного просмотра на сервере отчетов вам будет предложено указать, в какой формат файла экспортировать его (например, бинарный формат XLS Microsoft Excel 97-2003, SpreadsheetML или формат XLSX Microsoft Excel 2007), а также любые входные параметры отчета, созданные во время разработки отчета. После этого отчет будет заполнен данными, предоставленными службами отчетов.
