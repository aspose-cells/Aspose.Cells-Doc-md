{{% alert color="primary" %}} 

Когда вы разработали отчет со всеми необходимыми элементами, следующим шагом является его публикация на сервере отчетов, чтобы вы и ваши коллеги могли его выполнять.

{{% /alert %}} 

Чтобы опубликовать отчет на сервере отчетов, выполните следующие шаги:

1. Нажмите **Опубликовать** 

![todo:image_alt_text](publishing-report_1.png)

на панели инструментов Отчета Aspose.Cells.Report.Designer и нажмите кнопку **Да** для сохранения отчета. 

![todo:image_alt_text](publishing-report_2.png)




1. В диалоговом окне введите URL-адрес сервера отчетов и нажмите кнопку **Обновить**

![todo:image_alt_text](publishing-report_3.png)

1. Выберите целевую папку и введите название отчета. Нажмите кнопку **Настройка источника данных**, чтобы настроить источники данных. 

![todo:image_alt_text](publishing-report_4.png)



Отображается диалоговое окно Настройка источника данных.

1. В диалоговом окне введите информацию об источнике данных. Нажмите **Выбрать**, чтобы выбрать общие источники данных (если они вам нужны, общие источники данных должны быть основным источником данных отчета). Нажмите **OK**, чтобы завершить работу.

![todo:image_alt_text](publishing-report_5.png)

1. Нажмите **Опубликовать**, чтобы опубликовать отчет на сервере отчетов. 

![todo:image_alt_text](publishing-report_6.png)



кнопку для завершения работы.
