{{% alert color="primary" %}} 

Когда вы разработали отчет со всеми элементами, которые вы хотите, следующим шагом будет его публикация на сервере отчетов, чтобы вы и ваши коллеги могли его запустить.

{{% /alert %}} 

Чтобы опубликовать отчет на сервере отчетов, выполните следующие действия:

1.  Нажмите**Публиковать** 

![дело:изображение_альтернативный_текст](publishing-report_1.png)

 на панели инструментов Aspose.Cells.Report.Designer и нажмите кнопку**Да** кнопка сохранения отчета.

![дело:изображение_альтернативный_текст](publishing-report_2.png)




1.  В диалоговом окне введите URL-адрес сервера отчетов и нажмите кнопку**Обновить** кнопка.

![дело:изображение_альтернативный_текст](publishing-report_3.png)

1.  Выберите целевую папку и введите имя отчета. Нажмите на**Настройка источника данных** кнопка для настройки источников данных.

![дело:изображение_альтернативный_текст](publishing-report_4.png)



Отображается диалоговое окно «Настройка источника данных».

1.  В диалоговом окне введите информацию об источнике данных. Нажмите**Выбирать** для выбора общих источников данных. (Если они вам нужны, общие источники данных должны быть основным источником данных отчета). Нажмите на**ХОРОШО** кнопку для завершения работы.

![дело:изображение_альтернативный_текст](publishing-report_5.png)

1.  Нажмите**Публиковать** для публикации отчета на сервере отчетов.

![дело:изображение_альтернативный_текст](publishing-report_6.png)



кнопку для завершения работы.
