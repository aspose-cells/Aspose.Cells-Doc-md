{{% alert color="primary" %}} 

Определение отчета содержит информацию о выборке данных и макет для отчета. Язык определения отчетов (RDL) представляет собой XML-представление определения отчета. RDL - это открытая схема: разработчики могут расширять RDL добавлением дополнительных атрибутов и элементов. Стандартным расширением файлов определения отчетов является .rdl. Его MIME-тип - text/xml. 

Шаблон Aspose.Cells - это книга Microsoft Excel, содержащая информацию о макете отчета и сохраненная в файлы RDL (формат) после завершения проектирования шаблона. При открытии файла RDL, не содержащего шаблона Aspose.Cells, доступны только его источники данных и запросы для использования в Aspose.Cells.Report.Designer. При открытии файла RDL, содержащего шаблон Aspose.Cells, шаблон извлекается и открывается как книга в Microsoft Excel.

{{% /alert %}}
