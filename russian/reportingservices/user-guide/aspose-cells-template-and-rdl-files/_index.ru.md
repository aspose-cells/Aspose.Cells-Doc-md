Определение отчета содержит сведения о поиске данных и макете отчета. Язык определения отчетов (RDL) представляет собой XML-представление для определения отчета. RDL — это открытая схема: разработчики могут расширять RDL дополнительными атрибутами и элементами. Стандартное расширение файлов определений отчетов — .rdl. Его тип MIME — text/xml.

Шаблон Aspose.Cells — это рабочая книга Excel Microsoft, которая содержит информацию о макете отчета и сохраняется в файлах RDL (формат) после завершения работы над шаблоном. При открытии файла RDL, не содержащего шаблона Aspose.Cells, для использования в Aspose.Cells.Report.Designer доступны только его источники данных и запросы. Когда файл RDL, содержащий шаблон Aspose.Cells, открывается, шаблон извлекается и открывается как рабочая книга в Microsoft Excel.
