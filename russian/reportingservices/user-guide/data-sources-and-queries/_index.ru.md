{{% alert color="primary" %}} 

Первый этап процедуры проектирования отчета в Aspose.Cells.Report.Designer - создание источников данных и запросов. Существует два способа сделать это. В первом подходе мы можем использовать существующий файл определения отчета RDL, созданный другими инструментами, например, используя Visual Studio, и в файле содержатся источники данных и запросы, необходимые для проектирования отчета и извлечения данных. Второй способ - создать новые источники и запросы в Aspose.Cells.Report.Designer.
