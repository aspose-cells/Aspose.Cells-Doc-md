Первым шагом процедуры разработки отчета в Aspose.Cells.Report.Designer является создание источников данных и запросов. Есть два способа сделать это. В первом подходе мы можем использовать существующий файл определения отчета RDL, созданный другими инструментами, например, с использованием Visual Studio, и файл содержит источники данных и запросы, необходимые для разработки отчета и извлечения данных. Второй способ — создать новые источники и запросы в Aspose.Cells.Report.Designer.
