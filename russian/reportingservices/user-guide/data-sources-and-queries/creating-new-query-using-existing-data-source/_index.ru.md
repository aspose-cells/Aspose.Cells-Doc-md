Для создания нового запроса с использованием существующего источника данных:

1. Откройте Microsoft Excel.
1. Нажмите **Построить набор данных** в конструкторе отчетов Aspose.Cells (

![todo:image_alt_text](creating-new-query-using-existing-data-source_1.png)

).

1. В диалоговом окне выберите один из источников данных в списке слева. Например, **SqlServer**.
1. Нажмите **Добавить**. 

   **Выбор источника данных** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_2.png)




1. В диалоговом окне введите новое имя набора данных, например SalesEmps.
1. Нажмите **Далее**. 

   **Дать имя новому набору данных** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_3.png)



Aspose.Cells.Report.Designer запускает Microsoft Query и подключается к выбранному источнику данных. Теперь можно создать новый запрос с помощью Microsoft Query. 

**Создание нового запроса** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_4.png)




1. В меню **Файл** Microsoft Query выберите **Вернуться в Aspose.Cells.Report.Designer**. 

   **Возвращение к конструктору отчетов** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_5.png)



Источник данных SqlServer содержит два набора данных, один из которых является новым набором данных SalesEmps.
