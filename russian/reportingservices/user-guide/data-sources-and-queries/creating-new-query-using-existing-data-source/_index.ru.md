Чтобы создать новый запрос с использованием существующего источника данных:

1. Откройте Microsoft Excel.
1.  Нажмите**Построить набор данных** в Aspose.Cells.Отчет.Конструктор (

![дело:изображение_альтернативный_текст](creating-new-query-using-existing-data-source_1.png)

).

1.  В диалоговом окне выберите один из источников данных, перечисленных на левой панели. Например,**SQLServer**.
1.  Нажмите**Добавлять**. 

   **Выбор источника данных** 

![дело:изображение_альтернативный_текст](creating-new-query-using-existing-data-source_2.png)




1. В диалоговом окне введите имя нового набора данных, например SalesEmps.
1.  Нажмите**Следующий**. 

   **Именование нового набора данных** 

![дело:изображение_альтернативный_текст](creating-new-query-using-existing-data-source_3.png)



 Aspose.Cells.Report.Designer запускает Microsoft Запрос и подключается к выбранному источнику данных. Теперь можно создать новый запрос с помощью Microsoft Query.

**Создание нового запроса** 

![дело:изображение_альтернативный_текст](creating-new-query-using-existing-data-source_4.png)




1.  От Microsoft Запросы**Файл** меню, выберите**Вернуться к Aspose.Cells.Отчет.Дизайнер**. 

   **Вернуться к дизайнеру отчетов** 

![дело:изображение_альтернативный_текст](creating-new-query-using-existing-data-source_5.png)



 Источник данных SqlServer содержит два набора данных, одним из которых является новый набор данных SalesEmps.
