Чтобы отредактировать запрос в Aspose.Cells.Report.Designer:

1. Откройте Microsoft Excel.
1.  Нажмите**Построить набор данных**на панели инструментов Aspose.Cells.Отчет.Конструктор (

![дело:изображение_альтернативный_текст](editing-query_1.png)

).

1.  В диалоговом окне выберите набор данных, например,**EmpSalesDetail**.
1.  Нажмите**Редактировать**. 

   **Выбор набора данных** 

![дело:изображение_альтернативный_текст](editing-query_2.png)




1.  Aspose.Cell.Report.Designer запускает Microsoft Query и открывает запрос для редактирования.

   **Редактирование запроса в Microsoft Query** 

![дело:изображение_альтернативный_текст](editing-query_3.png)




1.  От Microsoft Запросы**Файл** меню, выберите**Вернуться к Aspose.Cells.Отчет.Дизайнер** 

   **Вернуться к дизайнеру отчетов** 

![дело:изображение_альтернативный_текст](editing-query_4.png)
