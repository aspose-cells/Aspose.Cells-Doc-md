Для редактирования запроса в Aspose.Cells.Report.Designer:

1. Откройте Microsoft Excel.
1. Нажмите **Построить набор данных** на панели инструментов Aspose.Cells.Report.Designer (

![todo:image_alt_text](editing-query_1.png)

).

1. В диалоговом окне выберите набор данных, например, **EmpSalesDetail**.
1. Нажмите **Изменить**. 

   **Выбор набора данных** 

![todo:image_alt_text](editing-query_2.png)




1. Aspose.Cell.Report.Designer запускает Microsoft Query и открывает запрос для редактирования. 

   **Редактирование запроса в Microsoft Query** 

![todo:image_alt_text](editing-query_3.png)




1. В меню **Файл** Microsoft Query выберите **Вернуться в Aspose.Cells.Report.Designer** 

   **Возвращение к конструктору отчетов** 

![todo:image_alt_text](editing-query_4.png)
