{{% alert color="primary" %}} 

Вы можете создавать источники данных и запросы в Visual Studio и сохранять их в RDL-файл. Когда файл RDL открыт в Aspose.Cells.Report.Designer, его источники данных и запросы доступны для использования. Например, мы можем создать запрос с именем EmpSalesDetail в Visual Studio и сохранить его как файл RDL.

**RDL-файл EmpSalesDetail** 

![дело:изображение_альтернативный_текст](using-data-sources-and-queries-created-in-visual-studio_1.png)

В этой статье показано, как работать с RDL-файлом в Aspose.Cells.Report.Designer.

{{% /alert %}} 

Вы можете открывать файлы RDL с помощью Aspose.Cells.Report.Designer и работать с ними:

1. Откройте Microsoft Excel.
1.  На панели инструментов Aspose.Cells.Report.Designer выберите**Открыть отчет**. 

   **Кнопка Открыть отчет** 

![дело:изображение_альтернативный_текст](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Выберите файл RDL и откройте его.
 После открытия файла RDL в Aspose.Cells.Report.Designer запросы появляются в**Набор данных** список.

   **Доступные наборы данных** 

![дело:изображение_альтернативный_текст](using-data-sources-and-queries-created-in-visual-studio_3.png)
