{{% alert color="primary" %}} 

Вы можете создавать источники данных и запросы в Visual Studio и сохранять их в файле RDL. При открытии файла RDL в конструкторе отчетов Aspose.Cells, его источники данных и запросы становятся доступными для использования. Например, мы можем создать запрос с именем EmpSalesDetail в Visual Studio и сохранить его как файл RDL.

**Файл RDL EmpSalesDetail** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_1.png)

В этой статье показано, как работать с файлом RDL в конструкторе отчетов Aspose.Cells.

{{% /alert %}} 

Вы можете открывать файлы RDL с помощью конструктора отчетов Aspose.Cells и работать с ними:

1. Откройте Microsoft Excel.
1. Из панели инструментов конструктора отчетов Aspose.Cells выберите **Открыть отчет**. 

   **Кнопка Открыть отчет** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Выберите файл RDL и откройте его.
   После открытия файла RDL в конструкторе отчетов Aspose.Cells запросы появляются в списке **Наборы данных**. 

   **Доступные наборы данных** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_3.png)
