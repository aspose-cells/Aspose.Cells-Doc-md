Чтобы добавить элемент подотчета:

1. Выберите ячейку, которая будет ссылаться на элемент подотчета.
1. Нажмите **Подотчет** (

![todo:image_alt_text](add-sub-report-item_1.png)

).
Отображается форма подотчета. 

**Форма подотчета** 

![todo:image_alt_text](add-sub-report-item_2.png)




1. Нажмите **Добавить подотчет**.
   Отображается форма Извлечения отчета.
1. Выберите отчет. 

   **Выбор отчета** 

![todo:image_alt_text](add-sub-report-item_3.png)




1. Нажмите **Закрыть**. 

   **Выбран подотчет** 

![todo:image_alt_text](add-sub-report-item_4.png)




1. Дважды щелкните список параметров, чтобы отобразить диалоговое окно Изменить формулу. 

   **Диалог редактирования формул** 

![todo:image_alt_text](add-sub-report-item_5.png)




1. Измените и/или обновите значение параметра. 

   **Наименование подотчета** 

![todo:image_alt_text](add-sub-report-item_6.png)




1. Нажмите **Вставить подотчет**. 

   **Подотчет вставлен в таблицу отчета** 

![todo:image_alt_text](add-sub-report-item_7.png)
