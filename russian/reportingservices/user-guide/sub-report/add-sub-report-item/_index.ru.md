Чтобы добавить элемент вложенного отчета:

1. Выберите ячейку, которая будет ссылаться на элемент вложенного отчета.
1.  Нажмите**Подотчет** (

![дело:изображение_альтернативный_текст](add-sub-report-item_1.png)

).
 Появится форма подотчета.

**Форма подчиненного отчета** 

![дело:изображение_альтернативный_текст](add-sub-report-item_2.png)




1.  Нажмите**Добавить подотчет**.
 Появится форма «Получить отчет».
1.  Выберите отчет.

   **Выбор отчета** 

![дело:изображение_альтернативный_текст](add-sub-report-item_3.png)




1.  Нажмите**Закрывать**. 

   **Подотчет выбран** 

![дело:изображение_альтернативный_текст](add-sub-report-item_4.png)




1.  Дважды щелкните список параметров, чтобы открыть диалоговое окно «Редактировать формулу».

   **Диалоговое окно «Редактировать формулу»** 

![дело:изображение_альтернативный_текст](add-sub-report-item_5.png)




1.  Отредактируйте и/или обновите значение параметра.

   **Наименование вложенного отчета** 

![дело:изображение_альтернативный_текст](add-sub-report-item_6.png)




1.  Нажмите**Вставить подотчет**. 

   **Подотчет, вставленный в таблицу отчета** 

![дело:изображение_альтернативный_текст](add-sub-report-item_7.png)
