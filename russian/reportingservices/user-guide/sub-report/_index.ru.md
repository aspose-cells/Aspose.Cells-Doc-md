{{% alert color="primary" %}} 

Подотчет может быть встроен в элемент таблицы. Формат следующий: &=subreport{ReportName=название вашего отчета; имя параметра 1 = значение параметра 1; имя параметра2 = значение параметра2; ...}

**Подотчет в определении отчета** 

![дело:изображение_альтернативный_текст](sub-report_1.png)

В примере имя подотчета — «Сведения о заказе на продажу». Он имеет один параметр, SalesOrderNumber. Значение параметра — EmpSalesDetail.SalesOrderNumber.
### **Ограничения для подотчетов**
1. Подотчет должен быть разработан с помощью Aspose.Cells.ReportingServices Designer.
1. Подотчет может быть встроен только в строку группы таблицы, а строка группы не может содержать никаких элементов, кроме подотчета. Встраивание подотчета в строки сведений таблицы или строки нижнего колонтитула не допускается.
1. В настоящее время вложенность более чем на один уровень не поддерживается. Подотчет не может содержать встроенный отчет.
