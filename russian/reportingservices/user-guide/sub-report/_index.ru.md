{{% alert color="primary" %}} 

Подотчет можно встроить в элемент таблицы. Формат: &=subreport{ReportName=название вашего отчета; наименование параметра1 = значение параметра1; наименование параметра2 = значение параметра2; ...}

**Подотчет в определении отчета** 

![todo:image_alt_text](sub-report_1.png)

В данном примере имя подотчета «Деталь заказа на продажу». У него есть один параметр, SalesOrderNumber. Значение параметра - EmpSalesDetail.SalesOrderNumber.
### **Ограничения на подотчеты**
1. Подотчет должен быть разработан с помощью Aspose.Cells.ReportingServices Designer.
1. Подотчет может быть встроен только в строку группы таблицы, и строка группы не может содержать никаких элементов, кроме подотчета. Встраивание подотчета в строки деталей или подвала таблицы запрещено.
1. В настоящее время не поддерживается вложенность более одного уровня. Подотчет не может содержать встроенный отчет.
