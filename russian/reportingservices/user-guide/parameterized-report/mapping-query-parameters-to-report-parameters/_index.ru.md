1. Откройте Microsoft Excel.
1.  Нажмите**Построить набор данных**на панели инструментов Aspose.Cells.Отчет.Конструктор (

![дело:изображение_альтернативный_текст](mapping-query-parameters-to-report-parameters_1.png)

).

1.  Выберите набор данных на левой панели, например**EmpSalesDetail**. 

   **Выбор набора данных** 

![дело:изображение_альтернативный_текст](mapping-query-parameters-to-report-parameters_2.png)




1. Щелкните вкладку Параметры на правой панели, чтобы вывести список параметров набора данных.
1.  Нажмите на**Связанный параметр отчета** столбец.
 Перечислены все параметры отчета, созданные заранее.
1.  Выберите параметр.

   **Параметры сопоставления** 

![дело:изображение_альтернативный_текст](mapping-query-parameters-to-report-parameters_3.png)




1.  Повторите, чтобы сопоставить все параметры запроса с параметрами отчета.

   **Сопоставление завершено** 

![дело:изображение_альтернативный_текст](mapping-query-parameters-to-report-parameters_4.png)

1.  Нажмите**Закрывать**.
