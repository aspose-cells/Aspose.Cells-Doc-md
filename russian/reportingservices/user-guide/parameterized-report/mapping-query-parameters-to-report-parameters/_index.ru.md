1. Откройте Microsoft Excel.
1. Нажмите **Построить набор данных** на панели инструментов Aspose.Cells.Report.Designer (

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_1.png)

).

1. Выберите набор данных в левой панели, например **EmpSalesDetail**. 

   **Выбор набора данных** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_2.png)




1. Нажмите вкладку Параметры в правой панели, чтобы перечислить параметры набора данных.
1. Нажмите на столбец **Привязанный параметр отчета**.
   Все заранее созданные параметры отчета перечислены. 
1. Выберите параметр. 

   **Параметры сопоставления** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_3.png)




1. Повторите, чтобы сопоставить все параметры запроса с параметрами отчета. 

   **Сопоставление завершено** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_4.png)

1. Нажмите **Закрыть**.
