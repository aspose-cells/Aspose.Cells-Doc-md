Чтобы создать параметр отчета в Aspose.Cells.Report.Designer:

1. Откройте Microsoft Excel.
1.  Нажмите**Параметр отчета** на панели инструментов Aspose.Cells.Отчет.Конструктор. (

![дело:изображение_альтернативный_текст](creating-report-parameters_1.png)

)
 В диалоговом окне все параметры отчета перечислены на левой панели.

1.  Нажмите**Добавлять**. 

   **Форма настройки параметров отчета** 

![дело:изображение_альтернативный_текст](creating-report-parameters_2.png)




1.  В диалоговом окне введите информацию о параметрах отчета.

   **Добавление информации о параметрах** 

![дело:изображение_альтернативный_текст](creating-report-parameters_3.png)



Допустимые значения для параметра отчета могут исходить из набора данных или группы пользовательских значений. Параметр отчета имеет два поля данных: метку и значение. Значение метки может быть нулевым.

1.  Нажмите**Настраивать** настроить группу значений для параметра отчета.
1.  Добавьте значения для**Этикетка** и**Стоимость** поля.

   **Добавление метки и значения** 

![дело:изображение_альтернативный_текст](creating-report-parameters_4.png)




1.  Нажмите**Добавлять**.
 Повторите, чтобы добавить другие значения параметра.
1.  Нажмите**Закрывать**.
1.  Нажмите**Сохранять** для сохранения параметра.
1.  Новый параметр отчета отображается на левой панели.

   **Новый параметр** 

![дело:изображение_альтернативный_текст](creating-report-parameters_5.png)
