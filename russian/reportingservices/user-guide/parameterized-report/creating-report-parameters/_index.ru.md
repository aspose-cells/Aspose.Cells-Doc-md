Чтобы создать параметр отчета в конструкторе отчетов Aspose.Cells:

1. Откройте Microsoft Excel.
1. Нажмите **Параметр отчета** на панели инструментов конструктора отчетов Aspose.Cells. (

![todo:image_alt_text](creating-report-parameters_1.png)

)
В диалоговом окне все параметры отчета перечислены в левой панели. 

1. Нажмите **Добавить**. 

   **Форма конфигурации параметра отчета** 

![todo:image_alt_text](creating-report-parameters_2.png)




1. В диалоговом окне введите информацию о параметре отчета. 

   **Добавление информации о параметре** 

![todo:image_alt_text](creating-report-parameters_3.png)



Допустимые значения для параметра отчета могут быть получены из набора данных или группы пользовательских значений. У параметра отчета есть два поля данных: метка и значение. Значение метки может быть пустым. 

1. Нажмите **Настроить**, чтобы установить группу значений для параметра отчета.
1. Добавьте значения для полей **Метка** и **Значение**. 

   **Добавление метки и значения** 

![todo:image_alt_text](creating-report-parameters_4.png)




1. Нажмите **Добавить**.
   Повторите для добавления других значений для параметра.
1. Нажмите **Закрыть**.
1. Нажмите **Сохранить**, чтобы сохранить параметр. 
1. Новый параметр отчета перечислен в левой панели. 

   **Новый параметр** 

![todo:image_alt_text](creating-report-parameters_5.png)
