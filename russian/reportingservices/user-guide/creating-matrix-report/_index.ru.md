{{% alert color="primary" %}} 

 Aspose.Cells for Reporting Services позволяет создать матрицу в Microsoft Excel.

{{% /alert %}} 
### **Шаблон матрицы**
В шаблоне отчета Aspose.Cells матрица состоит из углов, групп строк, групп столбцов и частей данных. Образец матрицы показан ниже.

**Образец матрицы** 

![дело:изображение_альтернативный_текст](creating-matrix-report_1.png)

- **Угол матрицы**расположен в верхнем левом углу или в верхнем правом углу для макетов с письмом справа налево (RTL). Эта область создается автоматически при добавлении групп строк и групп столбцов в область матричных данных. В этой области вы можете объединить ячейки, встроенные в элемент отчета текстового поля.
- **Область групп столбцов матрицы**: расположен в правом верхнем углу (верхний левый угол для макета справа налево). Эта область создается автоматически при добавлении группы столбцов. Ячейки в этой области представляют элементы иерархии групп столбцов и отображают значения экземпляров группы столбцов. На рисунке ячейки, отображающие OrderYear, представляют собой вложенную группу столбцов, а ячейка, отображающая OrderQtr, является смежной группой столбцов.
- **Область групп строк матрицы**: расположен в нижнем левом углу (нижний правый для макета справа налево). Эта область создается автоматически при добавлении группы строк. Ячейки в этой области представляют элементы иерархии групп строк и отображают значения экземпляров групп строк. На рисунке эти ячейки представляют собой вложенные группы строк.
- **Область данных матрицы**: расположен в правом нижнем углу (нижний левый для макета справа налево). Данные матрицы отображают подробные и сгруппированные данные. В этом примере используются только агрегированные данные. По умолчанию ячейки в строке или столбце группы, содержащие простые выражения, не включающие агрегатную функцию, оцениваются как первое значение в группе. На рисунке в ячейках отображаются агрегированные итоги по строкам для всех заказов на продажу.
#### **Создание шаблона матрицы**
 Перед созданием матричного отчета создайте источники данных, наборы данных и параметры отчета (необязательно). (Следуйте инструкциям в[Источники данных и запросы](/cells/ru/reportingservices/data-sources-and-queries/) если вам нужна помощь.) В примере мы используем образец базы данных AdventureWorks, который поставляется с SQL Server Reporting Services 2008.

Чтобы создать новую матрицу:

1. Откройте Microsoft Excel.
1.  Нажмите**Открыть отчет** чтобы открыть файл отчета RDL, который содержит источники данных, наборы данных и параметры отчета, созданные заранее.
После того, как файл был успешно открыт, вся его информация доступна для использования, например, его наборы данных перечислены в**набор данных** список.
1.  Откройте рабочий лист Excel Microsoft и выберите набор данных.

![дело:изображение_альтернативный_текст](creating-matrix-report_2.png)




1.  Задайте группы строк и группы столбцов с помощью**Установить группу**. 

![дело:изображение_альтернативный_текст](creating-matrix-report_3.png)




1. Объедините ячейки, чтобы установить угол матрицы.

![дело:изображение_альтернативный_текст](creating-matrix-report_4.png)




1.  Установите угол матрицы, вставив формулу.

![дело:изображение_альтернативный_текст](creating-matrix-report_5.png)




![дело:изображение_альтернативный_текст](creating-matrix-report_6.png)




1.  Нажмите**Установить атрибут** установить атрибут матрицы.

![дело:изображение_альтернативный_текст](creating-matrix-report_7.png)



Он состоит из имени, диапазона, группы и порядка.

1. При нажатии кнопки «Изменить атрибут» проверяются и изменяются все матричные атрибуты текущего рабочего листа.

![дело:изображение_альтернативный_текст](creating-matrix-report_8.png)




1. Сохраните, опубликуйте и просмотрите отчет.
