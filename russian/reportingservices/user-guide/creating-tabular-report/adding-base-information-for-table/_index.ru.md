Для создания новой таблицы:

1. Откройте Microsoft Excel.
1. Нажмите **Открыть отчет** 

![todo:image_alt_text](adding-base-information-for-table_1)

` `на панели инструментов Aspose.Cells.Report.Designer, чтобы открыть файл отчета RDL, который содержит заранее созданные источники данных, наборы данных и параметры отчета. Как только файл будет успешно открыт, вся его информация будет доступна для использования, например, его наборы данных будут перечислены в соответствующем поле выбора DataSet. 

![todo:image_alt_text](adding-base-information-for-table_2.png)

1. Откройте электронную таблицу Microsoft Excel и разработайте заголовок таблицы следующим образом:

   **Выбор набора данных**
