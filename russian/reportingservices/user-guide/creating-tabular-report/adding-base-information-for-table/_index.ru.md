Чтобы создать новую таблицу:

1. Откройте Microsoft Excel.
1.  Нажмите**Открыть отчет** 

![дело:изображение_альтернативный_текст](adding-base-information-for-table_1)

 `  ` на панели инструментов Aspose.Cells.Report.Designer, чтобы открыть файл отчета RDL, который содержит источники данных, наборы данных и параметры отчета, созданные заранее. После успешного открытия файла вся его информация доступна для использования, например, его наборы данных перечислены в соответствующем поле со списком DataSet.

![дело:изображение_альтернативный_текст](adding-base-information-for-table_2.png)

1. Откройте электронную таблицу Excel Microsoft и создайте заголовок таблицы следующим образом:

   **Оформление шапки таблицы** 

![дело:изображение_альтернативный_текст](adding-base-information-for-table_3.png)

1.  Выберите набор данных «EmpSalesDetail» из соответствующего поля со списком набора данных.

   **Выбор набора данных** 

![дело:изображение_альтернативный_текст](adding-base-information-for-table_4.png)

1.  Все поля набора данных «EmpSalesDetail» перечислены в поле со списком полей. Выберите ячейку в столбце OrderNumber Cell, а затем выберите поле «SalesOrderNumber» в поле со списком полей, чтобы вставить маркер данных в ячейку.

   **Вставка поля** 

![дело:изображение_альтернативный_текст](adding-base-information-for-table_5.png)



**Код поля показывает, где было вставлено поле.** 

![дело:изображение_альтернативный_текст](adding-base-information-for-table_6.png)

1.  Повторите шаг 5, чтобы вставить поля ProdCat и SubCat.

   **Все поля вставлены**
