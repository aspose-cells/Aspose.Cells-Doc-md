{{% alert color="primary" %}} 

Для создания табличного отчета необходимо установить атрибуты отчета перед публикацией отчета на сервере отчетов. Атрибуты отчета содержат информацию о имени отчета, диапазоне, группировке данных, сортировке и т. д.

{{% /alert %}} 

Для настройки атрибутов табличного отчета:

1. Нажмите **Установить атрибуты** на панели инструментов Aspose.Cells.Report.Designer (

![todo:image_alt_text](setting-report-attributes_1.png)

).

1. Введите имя отчета в диалоговом окне. 

   **Установка имени отчета** 

![todo:image_alt_text](setting-report-attributes_2.png)




1. Нажмите **Далее**.
1. Выберите диапазон для отчета.
   Диапазон начинается с первой ячейки в строке данных таблицы и заканчивается последней ячейкой в строке футера или, если футеров нет, то последней ячейкой данных группы.
   Индексы начальной и конечной ячеек автоматически отображаются в поле **Диапазон**. Здесь диапазон включает ячейки с D4 по H8. 

   **Определение диапазона отчета** 

![todo:image_alt_text](setting-report-attributes_3.png)




1. Щелкните **Далее**, чтобы продолжить.
   Все поля набора данных перечислены в левой панели в диалоговом окне. 
1. Выберите поля для группировки данных. 

   **Выбор полей для группировки данных** 

![todo:image_alt_text](setting-report-attributes_4.png)




1. Щелкните **Далее**, чтобы продолжить.
1. Выберите поля для упорядочения данных. 

   **Выбор полей для упорядочивания данных** 

![todo:image_alt_text](setting-report-attributes_5.png)




1. Нажмите **Далее**.

   Отображается сводка отчета.
   **Сводка отчета** 

![todo:image_alt_text](setting-report-attributes_6.png)

1. Щелкните **Готово**.
