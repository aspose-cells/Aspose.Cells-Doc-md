{{% alert color="primary" %}} 

Чтобы создать табличный отчет, необходимо установить атрибуты отчета перед его публикацией на сервере отчетов. Атрибуты отчета содержат информацию об имени отчета, диапазоне, группировке данных, сортировке и т. д.

{{% /alert %}} 

Чтобы настроить атрибуты табличного отчета:

1.  Нажмите**Установить атрибут**на панели инструментов Aspose.Cells.Отчет.Конструктор (

![дело:изображение_альтернативный_текст](setting-report-attributes_1.png)

).

1.  Введите имя отчета в диалоговое окно.

   **Установка имени отчета** 

![дело:изображение_альтернативный_текст](setting-report-attributes_2.png)




1.  Нажмите**Следующий**.
1. Выберите диапазон для отчета.
 Диапазон начинается с первой ячейки строки данных таблицы и заканчивается последней ячейкой строки нижнего колонтитула или, если колонтитулы отсутствуют, последней ячейкой данных группы.
 Индекс начальной и конечной ячеек отображается в**Диапазон** поле автоматически. Здесь диапазон включает ячейки от D4 до H8.

   **Определение диапазона отчета** 

![дело:изображение_альтернативный_текст](setting-report-attributes_3.png)




1.  Нажмите**Следующий** продолжить.
Все поля набора данных перечислены на левой панели диалогового окна.
1.  Выберите поля для группировки данных.

   **Выбор полей для группировки данных** 

![дело:изображение_альтернативный_текст](setting-report-attributes_4.png)




1.  Нажмите**Следующий** продолжить.
1.  Выберите поля для упорядочения данных.

   **Выбор полей для упорядочения данных по** 

![дело:изображение_альтернативный_текст](setting-report-attributes_5.png)




1.  Нажмите**Следующий**.

 Отображается сводка отчета.
   **Сводка отчета** 

![дело:изображение_альтернативный_текст](setting-report-attributes_6.png)

1.  Нажмите**Заканчивать**.
