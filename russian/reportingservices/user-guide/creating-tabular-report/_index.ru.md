Таблица в шаблоне отчета Aspose.Cells состоит из заголовка, строк данных таблицы, групп строк и нижних колонтитулов. Образец таблицы показан ниже.

**Пример таблицы** 

![дело:изображение_альтернативный_текст](creating-tabular-report_1.png)
#### **Заголовок таблицы**
Заголовок таблицы обычно содержит заголовок для каждого столбца таблицы и другие текстовые элементы, такие как статический текст, параметры отчета, глобальные переменные отчета и т. д. Заголовок таблицы является необязательным. При использовании заголовка тег заголовка должен быть помещен слева от первого столбца данных таблицы, чтобы указать, что строка является заголовком.
#### **Строка данных таблицы**
Строка данных таблицы — это ряд ячеек, содержащих интеллектуальные маркеры. Таблица может содержать только одну строку данных.
#### **Группа строк**
Группа строк следует за строкой данных таблицы и состоит из двух частей: тега группы и строки данных группы.

Тег группы должен быть помещен слева от первого столбца данных таблицы, чтобы указать, что строка является строкой данных группы строк. Формат группового тега — ##group{GroupColumn}, например ##group{SalesOrderNumber}, где SalesOrderNumber — это одно из имен столбцов набора данных. Таблица может содержать только одну группу строк, но группа строк может содержать более одной строки данных группы. Тег группы можно разместить только в первой строке данных, как показано в примере выше.

Тег группы удаляется из выходного файла Excel Microsoft во время рендеринга. Группы строк необязательны.
#### **Нижние колонтитулы**
 Нижние колонтитулы идут после группы строк и состоят из трех частей: тега нижнего колонтитула, строки данных нижнего колонтитула и текстовой области нижнего колонтитула.

Тег нижнего колонтитула должен быть помещен слева от первого столбца столбца данных таблицы, который указывает, что строка является нижним колонтитулом таблицы. Таблица может содержать более одной строки данных нижнего колонтитула, и каждая строка нижнего колонтитула должна быть помечена тегом нижнего колонтитула.

Текстовая область нижнего колонтитула может содержать статический текст, параметры отчета и глобальные переменные отчета, как показано в примере выше.

Тег нижнего колонтитула удаляется из выходного файла Excel Microsoft во время рендеринга. Нижние колонтитулы необязательны.
