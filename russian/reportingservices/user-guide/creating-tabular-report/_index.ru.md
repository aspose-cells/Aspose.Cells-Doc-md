Таблица в шаблоне отчета Aspose.Cells состоит из заголовка, строк данных таблицы, групп строк и нижнего колонтитула. Пример таблицы показан ниже.

**Пример таблицы** 

![todo:image_alt_text](creating-tabular-report_1.png)
#### **Заголовок таблицы**
Заголовок таблицы обычно содержит заголовок для каждой колонки таблицы и другие текстовые элементы, такие как статический текст, параметры отчета, глобальные переменные отчета и т. д. Заголовок таблицы является необязательным. При использовании заголовка тег заголовка должен быть размещен слева от первой колонки данных таблицы, чтобы указать, что строка является заголовком.
#### **Строка данных таблицы**
Строка данных таблицы - это строка ячеек, содержащих умные маркеры. Таблица может содержать только одну строку данных.
#### **Группа строк**
Группа строк следует сразу после строки данных таблицы и состоит из двух частей: тег группы и строка данных группы. 

Тег группы должен быть размещен слева от первой колонки данных таблицы, чтобы указать, что строка является строкой данных группы. Формат тега группы - ##group{ИмяКолонкиГруппы}, например ##group{НомерЗаказаНаПродажу}, где НомерЗаказаНаПродажу - это одно из имен колонок набора данных. Таблица может содержать только одну группу строк, но группа строк может содержать более одной строки данных группы. Тег группы может быть размещен только в первой строке данных, как показано в примере выше.

Тег группы удаляется из файла Microsoft Excel при рендеринге. Группы строк являются необязательными.
#### **Футеры**
Футеры идут после группы строк и включают три части: тег футера, строка данных футера и область текста футера. 

Тег футера должен быть размещен слева от первой колонки данных таблицы, чтобы указать, что строка является футером таблицы. Таблица может содержать более одной строки данных футера, и каждая строка футера должна быть помечена тегом футера. 

Область текста футера может содержать статический текст, параметры отчета и глобальные переменные отчета, как показано в примере выше.

Тег футера удаляется из файла Microsoft Excel при рендеринге. Футеры являются необязательными.
