Чтобы добавить группу таблиц, выполните следующие действия:



1) Выберите ячейку тега группы, которая находится слева от первого столбца столбца данных таблицы, под строкой данных таблицы, как показано ниже.

![дело:изображение_альтернативный_текст](adding-table-group_1.png)







 2) Выберите «SalesOrderNumber» в**Установить группу** поле со списком, чтобы установить групповое поле.



![дело:изображение_альтернативный_текст](adding-table-group_2.png)



Результат показан ниже.



![дело:изображение_альтернативный_текст](adding-table-group_3.png)



3) Добавьте строку данных группы в соответствии с инструкциями, описанными выше. Строка данных группы может содержать статический текст, маркеры данных, формулы службы отчетов и т. д. В примере мы добавляем строку группы, содержащую объединенную ячейку статического текста и ячейку с формулой служб Reporting Services, как показано ниже.



![дело:изображение_альтернативный_текст](adding-table-group_4.png)



4) В таблице допускается наличие более одной строки групповых данных. Мы можем снова добавить строку данных группы, как показано ниже.



![дело:изображение_альтернативный_текст](adding-table-group_5.png)
