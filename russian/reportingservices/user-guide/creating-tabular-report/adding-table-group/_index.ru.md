Чтобы добавить группировку таблицы, выполните следующие шаги:



1) Выберите ячейку тега группы, находящуюся слева от первого столбца столбца данных таблицы, под строкой данных таблицы, как показано ниже.

![todo:image_alt_text](adding-table-group_1.png)







2) Выберите “SalesOrderNumber” в комбо-боксе **Set Group** для установки поля группировки.



![todo:image_alt_text](adding-table-group_2.png)



Результат показан ниже.



![todo:image_alt_text](adding-table-group_3.png)



3) Добавьте строку данных группировки в соответствии с инструкциями, описанными выше. Строка данных группировки может содержать статический текст, маркеры данных, формулы отчетности и т. д. В примере мы добавляем строку группировки, содержащую объединенную статическую ячейку текста и ячейку формулы отчетности, как показано ниже.



![todo:image_alt_text](adding-table-group_4.png)



4) В таблице допускается более одной строки данных группировки. Мы можем снова добавить строку данных группировки, как показано ниже.



![todo:image_alt_text](adding-table-group_5.png)
