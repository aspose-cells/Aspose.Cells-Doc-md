Чтобы изменить атрибуты отчета:

1. Нажмите **Изменить атрибут** на панели инструментов Aspose.Cells.Report.Designer (

![todo:image_alt_text](modifying-report-attributes_1.png)

).
Доступные отчеты перечислены в левой панели. 

1. Выберите отчет и нажмите **Изменить**. 

   **Выбор отчета** 

![todo:image_alt_text](modifying-report-attributes_2.png)




1. Измените атрибуты отчета. 

   **Изменение атрибутов** 

![todo:image_alt_text](modifying-report-attributes_3.png)




1. Нажмите **Применить** для завершения.
