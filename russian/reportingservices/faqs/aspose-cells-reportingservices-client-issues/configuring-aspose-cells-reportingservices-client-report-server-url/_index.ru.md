Чтобы настроить URL-адрес клиентского сервера отчетов:

1.  Нажмите на**Открыть удаленный отчет** кнопку и получить**Получить отчет** диалоговое окно.

![дело:изображение_альтернативный_текст](configuring-aspose-cells-reportingservices-client-report-server-url_1.png)

1.  Нажмите на**Конфигурация** кнопку и получить**Настройка URL-адреса сервера отчетов** диалог.

![дело:изображение_альтернативный_текст](configuring-aspose-cells-reportingservices-client-report-server-url_2.png)

1. Введите информацию о конфигурации сервера отчетов (URL, имя пользователя и пароль).
 1. URL-адрес сервера отчетов должен иметь формат http://host/ReportServer. Например, http://172.16.204.11/ReportServer.
1. Имя пользователя — это имя пользователя Windows Server. Например, Администратор.
 1. Пароль — это пароль пользователя Windows Server. Например, 123456.
 1. Значение домена по умолчанию равно null.
1.  Нажмите на**Тест** кнопку для проверки информации о конфигурации.
1.  Нажмите**Сохранять** и сохраните информацию о конфигурации сервера отчетов.

![дело:изображение_альтернативный_текст](configuring-aspose-cells-reportingservices-client-report-server-url_3.png)




1.  Нажмите**Изменить** кнопка для изменения информации о конфигурации.
