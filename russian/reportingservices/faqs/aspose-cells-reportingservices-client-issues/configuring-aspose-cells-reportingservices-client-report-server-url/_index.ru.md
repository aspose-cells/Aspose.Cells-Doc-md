Чтобы настроить URL-адрес клиентского сервера отчетов:

1. Нажмите кнопку **Открыть удаленный отчет** и получите диалоговое окно **Извлечь отчет**. 

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_1.png)

1. Нажмите кнопку **Config** и получите диалоговое окно **Настройка URL сервера отчетов**. 

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_2.png)

1. Введите информацию о конфигурации сервера отчетов (URL, имя пользователя и пароль).
   1. URL сервера отчетов должен иметь формат http://host/ReportServer. Например, http://172.16.204.11/ReportServer.
   1. Имя пользователя - это имя пользователя Window Server. Например, Администратор.
   1. Пароль - это пароль пользователя Window Server. Например, 123456.
   1. Значение по умолчанию для домена - null.
1. Нажмите кнопку **Test**, чтобы проверить информацию о конфигурации.
1. Нажмите **Сохранить** и сохраните информацию о конфигурации сервера отчетов.

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_3.png)




1. Нажмите кнопку **Изменить**, чтобы изменить информацию о конфигурации.
