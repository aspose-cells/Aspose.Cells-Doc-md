**Операторы**
Арифметические операторы используются для объединения чисел, числовых переменных, числовых полей и числовых функций, чтобы получить другое число. Операторы сравнения обычно используются для сравнения операндов на условие в управляющей структуре, такой как оператор If. Логические операторы обычно используются с операторами сравнения для создания условий для управляющих структур.

|**Имя набора**|**Имя формулы**|**Описание**|
| :- | :- | :- |
|Арифметический | ^ | Возведение в степень, например 2^3. |
| |* | Умножение, например 2*3. |
| | / | Деление, например 2/3. |
| | \\ | Целочисленное деление, например 2\\\3. |
| | Mod | Остаток от деления, например 4 Mod 3. |
| | + | Сложение, например 4 + 3. |
| | - | Вычитание, например 4 - 3. |
|Сравнение | < | Меньше, например 4 < 3 false. |
| | <= | Меньше или равно, например 4 <= 3 false. |
| | > | Больше, например 4 > 3 true. |
| | >= | Больше или равно, например 4 >= 3 true. |
| | = | Равно, например 4 = 3 false. |
| | <> | Не равно, например 4 <> 3 true. |
| | Like | Сравнивает строку с образцом. Например result = string Like pattern. |
| | Is | Сравнивает две ссылочные переменные объекта, например asp Is aspose. |
|Конкатенация | & | Генерирует объединение строк двух выражений. |
| | + | Складывает два числа или возвращает положительное значение числового выражения. Также может быть использован для объединения двух строковых выражений. |
|Логический/Побитовый |And | Выполняет логическое умножение двух булевых выражений или побитовое умножение двух числовых выражений. |
| |Не |Выполняет логическое отрицание на логическом выражении или побитовое отрицание на числовом выражении. |
| |Или |Выполняет логическое сложение на двух логических выражениях или побитовое сложение на двух числовых выражениях. |
| |Исключающее ИЛИ |Выполняет операцию логического исключения на двух логических выражениях или побитовое исключающее ИЛИ на двух числовых выражениях. |
| |И |Выполняет логическое И на двух выражениях с короткозамыканием. |
| |ИЛИ |Выполняет включающее логическое ИЛИ на двух выражениях с короткозамыканием. |
|Побитовый сдвиг |>> |Выполняет арифметический сдвиг влево для битовой последовательности. |
| |<< |Выполняет арифметический сдвиг вправо для битовой последовательности. |
