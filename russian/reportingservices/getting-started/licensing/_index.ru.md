{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services доступен в виде бесплатной ограниченной по времени оценки на [Странице загрузок](https://downloads.aspose.com/cells/reportingservices). Версии оценки и лицензированные версии продукта идентичны.

Когда вы довольны оценочной версией, приобретите лицензию. Убедитесь, что вы понимаете и соглашаетесь с условиями подписки.

{{% /alert %}} 
### **Получение и использование лицензии**
Лицензия будет доступна для загрузки с страницы заказа после оплаты заказа.

Файл лицензии - это файл XML, подписанный цифровой подписью. Файл лицензии содержит информацию, такую как имя клиента, приобретенный продукт и тип лицензии. Не изменяйте содержимое файла лицензии, так как это делает его недействительным.

Скачайте лицензию на свой компьютер и скопируйте ее в одну из следующих папок.

- $[Папка Установки]\Лицензия\
- $[MSSQL Server Reporting Services]\ReportServer\Bin\
- $[MSSQL Server Reporting Services]\ReportServer\

 Имя файла лицензии может быть одним из следующих, в зависимости от типа вашей подписки:

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic", "Aspose.Total.lic
- Aspose.Cells.Reporting.Services - Developer Site Subscription.lic
- Aspose.Cells.Reporting.Services - Подписка для разработчиков.lic
- Aspose.Cells.Reporting.Services - Подписка для множественного развертывания.lic
- Aspose.Cells.Reporting.Services - Подписка для развертывания на одном сервере.lic

Для проверки правильности установки лицензионного файла экспортируйте любой отчет в книгу Microsoft Excel. Если в конце книги нет листа с водяным знаком, лицензионный файл был успешно активирован.

Aspose.Cells for Reporting Services вставляет лист с водяным знаком при работе в режиме оценки. Когда установлен действительный лицензионный файл, например, когда **Aspose.Cells.ReportingServices.lic** находится в папке $[Папка установки]\License\, лист с водяным знаком отсутствует.
