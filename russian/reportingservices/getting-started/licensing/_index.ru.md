{{% alert color="primary" %}} 

 Aspose.Cells for Reporting Services доступен в качестве бесплатной, неограниченной по времени пробной версии от[Страница загрузки](https://downloads.aspose.com/cells/reportingservices). Ознакомительная и лицензионная версии продукта одинаковы.

Когда вас устроит ознакомительная версия, купите лицензию. Убедитесь, что вы понимаете и согласны с условиями подписки.

{{% /alert %}} 
### **Получение и использование лицензии**
Лицензия будет доступна для скачивания со страницы заказа после оплаты заказа.

Файл лицензии представляет собой обычный текстовый XML-файл с цифровой подписью. Файл лицензии содержит такую информацию, как имя клиента, приобретенный продукт и тип лицензии. Не изменяйте содержимое файла лицензии, так как это сделает его недействительным.

Загрузите лицензию на свой компьютер и скопируйте ее в папку, которая должна быть одной из следующих папок.

- $[Папка установки]\Лицензия\
- $[Службы отчетов MSSQL Server]\ReportServer\Bin\
- $[Службы отчетов MSSQL Server]\ReportServer\

Имя файла лицензии может быть одним из следующих в зависимости от типа вашей подписки:

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic", "Aspose.Total.lic
- Aspose.Cells.Reporting.Services — Подписка на сайт разработчика.lic
- Aspose.Cells.Reporting.Services - Подписка для разработчиков.lic
- Aspose.Cells.Reporting.Services — Подписка на множественное развертывание.lic
- Aspose.Cells.Reporting.Services — подписка на развертывание на одном сервере.lic

Чтобы проверить правильность установки файла лицензии, экспортируйте любой отчет в виде книги Excel Microsoft. Если рабочая книга не содержит водяного знака в конце, файл лицензии был успешно активирован.

 Aspose.Cells for Reporting Services вводит оценочный водяной знак при работе в ознакомительном режиме. Когда действующий файл лицензии, например, когда**Aspose.Cells.ReportingServices.lic** находится в папке $[Installation Folder]\License\, лист оценки водяного знака отсутствует.
