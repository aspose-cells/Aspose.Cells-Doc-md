{{% alert color="primary" %}} 

Пользователи могут разрабатывать и предоставлять отчеты непосредственно в Microsoft Excel с использованием Aspose.Cells.Report.Designer - простого и удобного инструмента. Они не обязаны использовать Visual Studio для создания отчетов, что позволяет сэкономить деньги на дополнительных лицензиях Visual Studio и не тратить время на изучение сложной среды разработки.

Поскольку Aspose.Cells.Report.Designer использует Microsoft Excel в качестве шаблона отчета, пользователи могут воспользоваться всеми мощными функциями и возможностями Microsoft Excel, включая сводные таблицы, интерактивные диаграммы, VBA, макросы и формулы. Не все эти функции поддерживаются стандартным форматом Microsoft Excel, используемым в отчетах Reporting Services.

{{% /alert %}}
