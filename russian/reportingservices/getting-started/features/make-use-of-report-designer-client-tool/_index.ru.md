{{% alert color="primary" %}} 

Пользователи могут создавать и отправлять отчеты в Microsoft Excel непосредственно с помощью Aspose.Cells.Report.Designer, простого и удобного в использовании инструмента. Им не нужно использовать Visual Studio для создания отчетов, но они могут сэкономить на покупке дополнительных лицензий Visual Studio и не должны тратить время на изучение того, как использовать сложную среду разработки.

Поскольку Aspose.Cells.Report.Designer использует Microsoft Excel в качестве шаблона отчета, пользователи могут воспользоваться всеми мощными функциями и функциями Microsoft Excel, включая сводные таблицы, интерактивные диаграммы, VBA, макросы и формулы. Не все эти функции поддерживаются стандартным форматом Excel Microsoft, предоставляемым службами Reporting Services.

{{% /alert %}} 
- **Панель инструментов Aspose.Cells.Отчет.Конструктор**
**Пользователи могут использовать панель инструментов Aspose.Cells.Report.Designer для управления файлом отчета RDL.**
