Некоторые эффективные характеристики Aspose.Cells for Reporting Services перечислены ниже

- Aspose.Cells for Reporting Services поддерживает таблицы с группированными строками и нижними колонтитулами. Группировка строк и нижних колонтитулов может быть настроена на этапе проектирования, подобно другим элементам отчетов.
- Aspose.Cells for Reporting Services поддерживает параметры и глобальные переменные в качестве источников данных отчета.
- Пользователи могут использовать Microsoft Query для создания источников данных и запросов. Microsoft Query - широко используемый инструмент, с которым большинство пользователей должны быть знакомы.
- Aspose.Cells for Reporting Services поддерживает как отчетные источники данных, так и общие источники данных, созданные заранее на сервере отчетов. Более того, возможно использование отчетных источников данных на этапе проектирования и указание общих источников данных как источников данных отчета при публикации отчета.
- Aspose.Cells для отчетов интегрируется с Microsoft Reporting Services безупречно, так что пользователи могут полностью воспользоваться управлением отчетами, безопасностью и опциями доставки Reporting Services.
