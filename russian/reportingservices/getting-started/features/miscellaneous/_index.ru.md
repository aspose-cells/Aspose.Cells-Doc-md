Некоторые эффективные характеристики Aspose.Cells для Reporting Services перечислены ниже.

- Aspose.Cells для служб Reporting Services поддерживает таблицы с сгруппированными строками и нижними колонтитулами. Группировку строк и нижних колонтитулов можно настроить во время разработки, как и другие элементы отчета.
- Aspose.Cells для служб Reporting Services поддерживает параметры и глобальные переменные в качестве источников данных отчета.
- Пользователи могут использовать Microsoft Query для создания источников данных и запросов. Microsoft Запрос — это широко используемый инструмент, который должен быть знаком большинству пользователей.
- Aspose.Cells для служб Reporting Services поддерживает как источники данных для конкретных отчетов, так и общие источники данных, созданные на сервере отчетов заранее. Более того, можно использовать источники данных для конкретного отчета во время разработки и указывать общие источники данных в качестве источников данных отчета при публикации отчета.
- Aspose.Cells для Reporting Service легко интегрируется с Microsoft Reporting Services, поэтому пользователи могут в полной мере использовать возможности Reporting Services по управлению, безопасности и доставке отчетов.

{{% /alert %}}
