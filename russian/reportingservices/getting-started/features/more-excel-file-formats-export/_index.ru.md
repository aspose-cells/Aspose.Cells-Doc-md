{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services позволяет доставлять собственные Microsoft отчеты Excel через Microsoft SQL Server 2005 и 2008 Reporting Services. Unlike the built-in renderer for Microsoft Excel-based reports, Aspose.Cells for Reporting Services provides an extended feature set that convert RDL reports to: Excel 97-2003 binary XLS format, CSV, SpreadsheetML, Excel 2007 XLSX , Excel 2007 XLSM, Excel 2007 XLSB, Формат HTML и ODS.

{{% /alert %}}
