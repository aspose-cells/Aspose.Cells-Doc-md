{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services позволяет предоставлять отчеты в форматах нативного Microsoft Excel с помощью Microsoft SQL Server 2005 и 2008 Reporting Services. В отличие от встроенного преобразователя для отчетов на основе Microsoft Excel, Aspose.Cells for Reporting Services предоставляет расширенный набор функций, который конвертирует отчеты RDL в: двоичный формат XLS Excel 97-2003, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 XLSB, HTML и формат ODS. 

{{% /alert %}}
