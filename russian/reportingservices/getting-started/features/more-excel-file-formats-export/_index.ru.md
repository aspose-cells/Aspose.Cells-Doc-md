{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services позволяет доставлять собственные Microsoft отчеты Excel через Microsoft SQL Server 2005 и 2008 Reporting Services. В отличие от встроенного средства визуализации для отчетов на основе Excel Microsoft, Aspose.Cells for Reporting Services предоставляет расширенный набор функций, которые преобразуют отчеты RDL в: двоичный формат Excel 97-2003 XLS, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 XLSB, Формат HTML и ODS.

{{% /alert %}}
