{{% alert color="primary" %}} 

Aspose.Cells для служб Reporting Services позволяет вам предоставлять исходные отчеты Excel Microsoft через Microsoft SQL Server 2005 и 2008 Reporting Services. В отличие от встроенного средства подготовки отчетов Microsoft для отчетов на основе Excel, Aspose.Cells для служб Reporting Services предоставляет расширенный набор функций, которые преобразуют отчеты RDL в двоичный формат Excel 97-2003 XLS, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007. Формат XLSB, HTML и ODS.

{{% /alert %}}
