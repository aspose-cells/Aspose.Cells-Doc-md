{{% alert color="primary" %}} 

- Если у вас уже установлена предыдущая версия Aspose.Cells for Reporting Services, сначала удалите ее. См. тему [Удаление Aspose.Cells для Reporting Services](/cells/ru/reportingservices/uninstalling-aspose-cells-for-reporting-services/) о том, как удалить компонент.
- [Посетите страницу загрузки Aspose.Cells for Reporting Services](https://downloads.aspose.com/cells/reportingservices) и загрузите установщик MSI Aspose.Cells for Reporting Services с последней версией компонента.
    - Установщик Msi Aspose.Cells for ReportingServices поддерживает только продукты Microsoft SQL Server 2008 Reporting Services и более поздние.
- Запустите загруженный файл и следуйте инструкциям установщика.

{{% /alert %}} 
**Процесс установки**
Чтобы установить Aspose.Cells с помощью установщика MSI:

1. Экран приветствия - это первый экран в установке. Нажмите на **Далее**.

![todo:image_alt_text](using-msi-installer_1.png)




1. Прочтите и примите лицензионное соглашение и нажмите **Далее**.

![todo:image_alt_text](using-msi-installer_2.png)




1. Выберите папку установки и нажмите **Далее**. 

![todo:image_alt_text](using-msi-installer_3.png)




1. Подтвердите установку Aspose.Cells for Reporting Services, нажав **Установить**. 

![todo:image_alt_text](using-msi-installer_4.png)



Прогресс установки отображается на экране. 

![todo:image_alt_text](using-msi-installer_5.png)




1. После завершения установки нажмите **Завершить**, чтобы закрыть установщик.

![todo:image_alt_text](using-msi-installer_6.png)
