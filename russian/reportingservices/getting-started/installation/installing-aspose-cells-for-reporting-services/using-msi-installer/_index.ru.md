{{% alert color="primary" %}} 

- Если у вас установлена предыдущая версия Aspose.Cells for Reporting Services, сначала удалите ее. Видеть[Удаление Aspose.Cells for Reporting Services](/cells/ru/reportingservices/uninstalling-aspose-cells-for-reporting-services/) Тема о том, как удалить компонент.
- [Посетите страницу загрузки Aspose.Cells for Reporting Services](https://downloads.aspose.com/cells/reportingservices) и загрузите установщик MSI Aspose.Cells for Reporting Services с последней версией компонента.
 - Aspose.Cells для ReportingServices Msi поддерживает только Microsoft SQL Server 2008 Reporting Services и более поздние продукты.
- Запустите загруженный файл и следуйте инструкциям установщика.

{{% /alert %}} 
**Процесс установки**
Чтобы установить Aspose.Cells с помощью установщика MSI:

1.  Экран приветствия — это первый экран установки. Нажмите**Следующий**.

![дело:изображение_альтернативный_текст](using-msi-installer_1.png)




1.  Прочитайте и примите лицензионное соглашение и нажмите**Следующий**.

![дело:изображение_альтернативный_текст](using-msi-installer_2.png)




1.  Выберите папку установки и нажмите**Следующий**. 

![дело:изображение_альтернативный_текст](using-msi-installer_3.png)




1.  Подтвердите, что хотите установить Aspose.Cells for Reporting Services, нажав**Установить**. 

![дело:изображение_альтернативный_текст](using-msi-installer_4.png)



 Ход установки отображается на экране.

![дело:изображение_альтернативный_текст](using-msi-installer_5.png)




1.  После завершения установки нажмите**Заканчивать** чтобы закрыть программу установки.

![дело:изображение_альтернативный_текст](using-msi-installer_6.png)
