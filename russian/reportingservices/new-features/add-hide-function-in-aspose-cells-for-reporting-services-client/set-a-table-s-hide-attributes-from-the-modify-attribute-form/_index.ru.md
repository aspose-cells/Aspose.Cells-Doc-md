 Установите атрибуты скрытия табличного отчета из**Изменить атрибут**. Параметры включают:

- **Тип** представляет тип скрытого объекта. Он включает в себя таблицу, строку и столбец.
- **Показатель**представляет позицию скрытого объекта. Если тип — рабочий лист, значение равно null. Значение индекса начинается с 0.
- **Выражение** представляет значение состояния скрытия. Значение true, false или формула Excel. Выражение поддерживает формулу Reporting Services.
