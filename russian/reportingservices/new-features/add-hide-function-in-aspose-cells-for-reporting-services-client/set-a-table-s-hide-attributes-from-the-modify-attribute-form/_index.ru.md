Установите атрибуты скрытия таблицы отчета из **Модификации атрибутов**. Параметры включают:

- **Тип** представляет тип объекта скрытия. Он включает таблицу, строку и столбец.
- **Индекс** представляет позицию скрываемого объекта. Когда тип - рабочий лист, значение равно нулю. Индексация начинается с 0.
- **Выражение** представляет значение состояния скрытия. Значение может быть true, false или Excel-формула. Выражение поддерживает формулу служб отчетов.
