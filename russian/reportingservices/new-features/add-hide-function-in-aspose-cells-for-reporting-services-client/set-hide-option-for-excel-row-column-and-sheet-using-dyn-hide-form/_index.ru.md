Aspose.Cells для служб Reporting Services скрывает любую указанную строку, столбец и лист на основе набора в параметрах скрытия. Параметры включают в себя:

- **имя листа** представляет рабочий лист, который указывает скрытый объект.
- **Тип** представляет тип скрытого объекта. Он включает рабочий лист, строку и столбец.
- **Индекс строки/столбца**представляет указанный скрытый объект. Если тип — рабочий лист, значение равно null. Индекс столбца поддерживает символы. Значение индекса начинается с 1.
- **Скрыть состояние** представляет значение состояния скрытия. Значение true, false или формула Excel. Этот параметр поддерживает только параметры отчетов служб Reporting Services.
