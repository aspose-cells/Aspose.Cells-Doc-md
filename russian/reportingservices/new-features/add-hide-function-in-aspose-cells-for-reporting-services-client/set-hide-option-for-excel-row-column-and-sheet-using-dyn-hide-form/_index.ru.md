Aspose.Cells for Reporting Services скрывает любую указанную строку, столбец и лист на основе заданных параметров скрытия. Параметры включают:

- **SheetName** представляет собой лист, который указывает объект скрытия.
- **Тип** представляет тип объекта скрытия. Включает лист, строку и столбец.
- **Индекс строки/столбца** представляет указанный объект скрытия. Когда тип - лист, значение равно null. Индекс столбца поддерживает символ. Значение индекса начинается с 1.
- **Состояние скрытия** представляет значение состояния скрытия. Значение true, false или формула Excel. Параметр поддерживает только параметры отчетов служб отчетов.
