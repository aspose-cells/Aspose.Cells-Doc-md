{{% alert color="primary" %}} 

Мы включили поддержку встраивания подотчета в строку группы таблицы. Формат следующий:

&=subreport{ReportName=your report name; parameter1 name = parameter1 value; parameter2 name = parameter2 value;......} 

{{% /alert %}} 
### **Пример**

В примере имя подотчета - “Детализация заказа на продажу”. У него есть один параметр, *НомерЗаказаНаПродажу*. Значение параметра - *EmpSalesDetail.SalesOrderNumber.*
#### **Ограничения при использовании подотчетов**
- Подотчет должен быть разработан с помощью инструмента Aspose.Cells.Reporting Services Designer.
- Подотчет можно встраивать только в строку группы таблицы, и в группе не должно быть других элементов, кроме подотчета. Встраивание подотчета в строки деталей таблицы или строки нижнего колонтитула запрещено.
- В настоящее время не поддерживается вложенность на более чем один уровень. Подотчет не может содержать вложенный отчет.
