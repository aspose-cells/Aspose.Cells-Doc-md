{{% alert color="primary" %}} 

Мы включили поддержку встраивания подотчета в строку группы таблицы. Формат:

&=subreport{ReportName=название вашего отчета; имя параметра 1 = значение параметра 1; имя параметра2 = значение параметра2;......} 

{{% /alert %}} 
### **Пример**
**Подотчет в таблице** 

![дело:изображение_альтернативный_текст](sub-reports_1.png)

 В примере имя подотчета — «Сведения о заказе на продажу». Он имеет один параметр,*номер заказа продаж* . Значение параметра*EmpSalesDetail.SalesOrderNumber.*
#### **Ограничения на использование подотчетов**
- Подотчет должен быть разработан с помощью инструмента Aspose.Cells.Reporting Services Designer.
- Вложенный отчет может быть встроен только в строку группы таблицы, и строка группы не может содержать другие элементы, кроме вложенного отчета. Встраивание подотчета в строки сведений таблицы или строки нижнего колонтитула не допускается.
- В настоящее время вложенность более чем на один уровень не поддерживается. Подотчет не может содержать встроенный отчет.
