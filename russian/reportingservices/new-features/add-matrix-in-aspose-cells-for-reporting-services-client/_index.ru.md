{{% alert color="primary" %}} 

Мы добавили поддержку создания элемента отчета матрица в отчете. Контекст матрицы включает угол, группировки столбцов, группировки строк и строки матрицы. Установите атрибуты отчета матрицы, которые включают имя, диапазон и т. д., через кнопку **Установить атрибут** на панели команд Aspose.Cells в Excel.

Например, на скриншоте ниже диапазон матрицы - E9:G11.
