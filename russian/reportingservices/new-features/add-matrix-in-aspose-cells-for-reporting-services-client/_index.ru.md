{{% alert color="primary" %}} 

 Мы включили поддержку для создания элемента матричного отчета в отчете. Контекст матрицы включает углы, группы столбцов, группы строк и строки матрицы. Установите атрибуты матричного отчета, которые включают имя, диапазон и т. д.**Установить атрибут** на панели команд Aspose.Cells' в Excel.

Например, на скриншоте ниже диапазон матрицы равен E9:G11.
