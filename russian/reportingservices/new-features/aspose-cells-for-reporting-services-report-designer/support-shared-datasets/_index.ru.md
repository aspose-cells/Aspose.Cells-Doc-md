Aspose.Cells for Reporting Services поддерживает общие наборы данных SSRS 2008R2. Когда конструктор отчетов Aspose.Cells открывает файл определения отчета, который включает в себя общие наборы данных, он может автоматически получить информацию о наборе данных, который включает поля, с сервера отчетов.
