Aspose.Cells for Reporting Services поддерживает общие наборы данных SSRS 2008R2. Когда дизайнер отчетов Aspose.Cells открывает файл определения отчета, который включает общие наборы данных, он может автоматически получить информацию о наборе данных, включая поля из сервера отчетов.
