Aspose.Cells for Reporting Services Дизайнер отчетов сохраняет текущий URL-адрес сервера отчетов, к которому был осуществлен доступ. При повторном открытии диалогового окна «Диспетчер сервера отчетов» в конструкторе отчетов Aspose.Cells диалоговая форма автоматически загружает сохраненную информацию о сервере отчетов.
