Aspose.Cells для конструктора отчетов служб Reporting Services сохраняет текущий URL-адрес сервера отчетов, к которому осуществляется доступ. При повторном открытии диалогового окна диспетчера сервера отчетов в дизайнере отчетов Aspose.Cells диалоговая форма автоматически загружает сохраненную информацию о сервере отчетов.
