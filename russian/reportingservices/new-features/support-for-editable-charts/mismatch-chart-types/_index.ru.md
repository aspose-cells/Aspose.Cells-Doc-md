{{% alert color="primary" %}} 

Некоторые типы диаграмм в файле определения отчета SQL Server Reporting Services 2008 недоступны в качестве типов диаграмм Excel:

- Линейный шаг
- Диапазон-равнина
- Диапазон-гладкий
- Диапазон-столбец
- Диапазон-бар
- Range-ErrorBar
- Range-BoxPlot
- Форма-воронка

 При отображении элементов диаграммы, которые не имеют эквивалентных параметров в Microsoft Excel, Aspose.Cells выводит их в изображения.

{{% /alert %}}
