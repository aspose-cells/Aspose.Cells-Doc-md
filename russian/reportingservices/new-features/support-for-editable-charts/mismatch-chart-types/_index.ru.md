{{% alert color="primary" %}} 

Некоторые типы диаграмм в файле определения отчета SQL Server Reporting Services 2008 недоступны в качестве типов диаграмм Excel:

- Линия-ступенька
- Диапазон-Простой
- Диапазон-Сглаженный
- Диапазон-Колонка
- Диапазон-Линейка
- Диапазон-Ошибка
- Диапазон-Ящиковая диаграмма
- Фигура-Воронка

При визуализации элементов диаграмм, которые не имеют эквивалентных параметров в Microsoft Excel, Aspose.Cells выводит их в изображения. 

{{% /alert %}}
