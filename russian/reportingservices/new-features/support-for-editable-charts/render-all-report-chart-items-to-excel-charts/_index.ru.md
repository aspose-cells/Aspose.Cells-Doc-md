{{% alert color="primary" %}} 

Для визуализации всех элементов диаграмм в отчете как редактируемые диаграммы Excel:

1. Откройте файл **Aspose.Cells.ReportingServices.xml**.
1. Измените параметры конфигурации следующим образом: 

{{< highlight java >}}

 <Chart value="all">

</Chart>

{{< /highlight >}}

Когда значение установлено на «all», все элементы диаграмм визуализируются как редактируемые диаграммы.
