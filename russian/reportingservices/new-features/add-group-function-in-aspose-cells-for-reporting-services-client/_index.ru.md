Он устанавливает групповые атрибуты элемента отчета таблицы с помощью изменения атрибутов. Параметры включают Тип, Имя, Уровень и Скрыть.

- Тип представляет тип группы. Он включает в себя группу строк и группу столбцов.
- Имя представляет имя группы.
- Уровень представляет уровень группы.
- Скрыть означает групповое скрытие.

Изменение отчета включает следующие шаги:
Откройте диалоговое окно «Изменить отчет» и выберите вкладку «Структура».

![дело:изображение_альтернативный_текст](add-group-function-in-aspose-cells-for-reporting-services-client_1.jpg)


 Нажмите**Изменить** и обновите Скрыть конфигурацию.

![дело:изображение_альтернативный_текст](add-group-function-in-aspose-cells-for-reporting-services-client_2.jpg)


 Нажмите**Совершить** кнопку и сохраните скрыть конфиги.

![дело:изображение_альтернативный_текст](add-group-function-in-aspose-cells-for-reporting-services-client_3.jpg)
