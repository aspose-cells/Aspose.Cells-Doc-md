Устанавливает атрибуты группы элемента отчета типа таблица через Изменить атрибуты. Параметры включают тип, имя, уровень и скрыть.

- Тип представляет собой тип группы. Он включает Группу строк и Группу столбцов.
- Имя представляет собой имя группы.
- Уровень представляет собой уровень группы.
- Скрыть представляет собой скрытие группы.

Изменение отчета включает следующие шаги:
Откройте диалоговое окно "Изменить отчет" и выберите вкладку Структура.

![todo:image_alt_text](add-group-function-in-aspose-cells-for-reporting-services-client_1.jpg)


Нажмите кнопку **Изменить** и обновите конфигурацию скрытия. 

![todo:image_alt_text](add-group-function-in-aspose-cells-for-reporting-services-client_2.jpg)


Нажмите кнопку **Применить** и сохраните конфигурации скрытия. 

![todo:image_alt_text](add-group-function-in-aspose-cells-for-reporting-services-client_3.jpg)
