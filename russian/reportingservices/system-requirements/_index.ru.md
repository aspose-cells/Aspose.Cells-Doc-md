{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services (включая клиент Aspose.Cells.ReportingServices, надстройки для Microsoft Excel и сервер Aspose.Cells.ReportingServices) должен быть правильно установлен. Кроме того, перед установкой Aspose.Cells for Reporting Services должен быть установлен Microsoft SQL Server.

{{% /alert %}} 
### **Система Office**
Поддерживаются следующие версии Microsoft Excel. Кроме того, вам необходимо установить дополнительные исправления, чтобы исправить возможные проблемы для Microsoft Office 2000, 2002, XP или 2003.

- Microsoft Excel 2000
  Установите **lockbackRegKey.msi**, чтобы исправить проблему CLR на машинах с Microsoft Office старше, чем Office 2003.
- Microsoft Excel 2002
  Установите **lockbackRegKey.msi** для исправления проблемы CLR на машинах с Microsoft Office старше, чем Office 2003.
- Microsoft Excel 2003
  Всегда обновляйтесь до Office 2003 (KB907417). Это обновление исправляет потенциальную проблему, при которой надстройка, умный тег или умный документ могут не загружаться, если они используют Microsoft .NET Framework 2.0.
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
- Microsoft Excel 2016
- Microsoft Excel 2019
- Microsoft Excel 365
