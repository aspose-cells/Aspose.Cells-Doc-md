 Эта страница содержит примечания к выпуску для[Aspose.Cells для служб Reporting Services 1.1.0.0](https://downloads.aspose.com/cells/reportingservices/new-releases/aspose.cells-for-reporting-services-1.1.0.0/)

{{% /alert %}} 

 Дорогие клиенты,



Мы выпустили Aspose.Cells.ReportingServices V1.1.0.0.



 Что нового?



-  ` `Мы предоставляем клиентскую надстройку — Aspose Cells Дизайнер отчетов, которая позволяет пользователям использовать MS Excel в качестве инструмента для разработки отчетов. С помощью конструктора отчетов Aspose Cells пользователи могут создавать шаблоны отчетов в Excel вместо Visual Studio или другого средства создания отчетов. Шаблон отчета встроен в RDL-файл служб Reporting Services и может быть опубликован на сервере отчетов. Это делает разработку отчетов легкой и быстрой задачей.
-  Aspose.Cells для рендеринга сервера служб Reporting Services значительно улучшен. Работая с шаблоном отчета, пользователи могут использовать все расширенные функции Excel в службах Reporting Services, включая встроенные диаграммы Excel, расширенные формулы, сводные таблицы и многое другое.
