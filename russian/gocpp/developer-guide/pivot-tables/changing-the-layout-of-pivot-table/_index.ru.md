{{% alert color="primary" %}}

Microsoft Excel позволяет изменять макет сводной таблицы с помощью команд меню *PivotTable Tools > Design > Report Layout*. Вы можете изменить макет в трех формах:

- Показать в компактной форме
- Показать в контурной форме
- Показать в табличной форме

Aspose.Cells также предоставляет методы [**PivotTable.ShowInCompactForm()**](https://reference.aspose.com/cells/go-cpp/pivottable/showincompactform/), [**PivotTable.ShowInOutlineForm()**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/showinoutlineform/) и [**PivotTable.ShowInTabularForm()**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/showintabularform/) для изменения макета сводной таблицы в этих трех формах.

{{% /alert %}}
