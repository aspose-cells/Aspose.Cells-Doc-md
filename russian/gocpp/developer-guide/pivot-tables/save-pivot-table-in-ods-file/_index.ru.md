Aspose.Cells предоставляет возможность сохранять сводные таблицы в файлах ODS. Для этого вы можете либо конвертировать рабочую книгу с уже существующей сводной таблицей, либо создать новую сводную таблицу и сохранить файл в формате ODS. Обязательно вызовите [**PivotTable::CalculateData()**](https://reference.aspose.com/cells/go-cpp/pivottable/calculatedata/) перед сохранением, чтобы сводная таблица была отображена в итоговом файле ODS. Следующий пример показывает, как сохранить сводную таблицу в файл ODS.
