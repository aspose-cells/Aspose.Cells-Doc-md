{{% alert color="primary" %}}

Aspose.Cells теперь поддерживает получение даты обновления и информации о том, кем она обновлялась из книги.

{{% /alert %}}

[**PivotTable.GetRefreshDate()**](https://reference.aspose.com/cells/go-cpp/pivottable/getrefreshdate/) возвращает дату последнего обновления отчёта PivotTable. Аналогично, свойство [**PivotTable.GetRefreshedByWho()**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/getrefreshedbywho/) возвращает имя пользователя, который последний обновлял отчёт. Следующий пример демонстрирует эту функцию, и пример файла можно скачать по следующей ссылке.
