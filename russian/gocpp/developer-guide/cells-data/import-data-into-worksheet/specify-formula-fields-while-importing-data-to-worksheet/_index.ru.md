## **Возможные сценарии использования**

Вы можете указать формульные поля при импорте данных в свой рабочий лист, используя [**ImportTableOptions.GetIsFormulas()**](https://reference.aspose.com/cells/go-cpp/importtableoptions/getisformulas/). Это свойство принимает булевый массив, где значение **True** означает, что поле является формульным. Например, если третье поле является формульным полем, то третье значение в массиве будет **True**.
