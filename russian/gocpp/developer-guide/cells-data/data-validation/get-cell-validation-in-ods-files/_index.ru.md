## **Получить проверку ячейки в файлах ODS**

С помощью API Aspose.Cells for C++ можно получить примененную проверку к ячейке в файлах ODS. API предоставляет метод [**GetValidation**](https://reference.aspose.com/cells/go-cpp/cell/getvalidation/) класса [**Cell**](https://reference.aspose.com/cells/cpp/aspose.cells/cell/).

Следующий пример демонстрирует использование метода [**GetValidation**](https://reference.aspose.com/cells/go-cpp/cell/getvalidation/), загружая [исходный файл ODS](101089354.ods) и читаюа проверку ячейки A9.
