{{% alert color="primary" %}}

Вы можете проверить, подписан ли ваш VBA-проект или нет, используя Microsoft Excel через меню **Инструменты > Цифровые подписи...**. Аналогично, это можно сделать программно с помощью Aspose.Cells [**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/go-cpp/vbaproject/issigned/).

{{% /alert %}}
