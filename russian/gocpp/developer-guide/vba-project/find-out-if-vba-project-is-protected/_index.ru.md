## **Выясните, защищен ли проект VBA на C++**

Вы можете определить, защищен ли проект VBA (Visual Basic Applications) вашего файла Excel, используя свойство [**VbaProject.IsProtected**](https://reference.aspose.com/cells/go-cpp/vbaproject/isprotected/) Aspose.Cells.

## **Образец кода**

Следующий пример создает рабочую книгу и проверяет, защищен ли ее проект VBA, затем защищает проект VBA и снова проверяет, защищен ли его проект VBA. Ознакомьтесь с выводом этой программы в консоли в качестве примера. Перед защитой [**VbaProject.IsProtected**](https://reference.aspose.com/cells/go-cpp/vbaproject/isprotected/) возвращает **false**, но после защиты — **true**.
