## **Возможные сценарии использования**
Вы можете установить заданный стиль WordArt для текста фигуры с помощью Aspose.Cells. Пожалуйста, используйте [FontSetting.SetWordArtStyle()](https://reference.aspose.com/cells/go-cpp/fontsetting/setwordartstyle/) или [FontSettingCollection.SetWordArtStyle()](https://reference.aspose.com/cells/cpp/aspose.cells.drawing.texts/fontsettingcollection/setwordartstyle/) для этой цели.
