## **Возможные сценарии использования**

Вместо настройки фильтров для отображения дат, вы можете использовать временную шкалу PivotTable — динамический фильтр, который позволяет легко фильтровать по дате/времени и увеличивать нужный период с помощью ползунка. Microsoft Excel позволяет создавать временную шкалу, выбирая сводную таблицу и щелкая по **Вставка > Временная шкала**. Aspose.Cells также позволяет создавать временную шкалу с помощью метода [**Worksheet.Timelines.Add()**](https://reference.aspose.com/cells/go-cpp/timelinecollection/add_pivottable_int_int_string/).
