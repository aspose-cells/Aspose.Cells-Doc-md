{{% alert color="primary" %}}

Условное форматирование двухцветной шкалы и трехцветной шкалы добавляются одинаковым образом, за исключением свойства [**FormatCondition.GetIs3ColorScale()**](https://reference.aspose.com/cells/go-cpp/colorscale/getis3colorscale/). Это свойство **false** для двухцветной шкалы и **true** для условного форматирования трехцветной шкалы.
