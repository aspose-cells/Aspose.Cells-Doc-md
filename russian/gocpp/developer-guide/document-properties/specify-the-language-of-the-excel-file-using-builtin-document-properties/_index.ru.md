## **Возможные сценарии использования**

Вы можете изменить язык файла Excel, щелкнув по файлу правой кнопкой мыши и выбрав Свойства > Детали, а затем отредактировав поле Язык. Для этого используйте свойство [**BuiltInDocumentPropertyCollection.GetLanguage()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getlanguage/) и API Aspose.Cells.

## **Указание языка файла Excel с использованием встроенных свойств документа**

Следующий пример кода создает рабочую книгу и изменяет ее встроенное свойство документа, называемое Язык.
