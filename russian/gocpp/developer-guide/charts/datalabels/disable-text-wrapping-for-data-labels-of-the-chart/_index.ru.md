{{% alert color="primary" %}}

Microsoft Excel 2013 позволяет пользователям включать или отключать перенос текста в метках данных диаграммы. По умолчанию текст в метках данных диаграммы находится в перенесенном состоянии.

Aspose.Cells предоставляет свойство [**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/go-cpp/datalabels/istextwrapped/), которое можно установить в True или False для включения или отключения переноса текста меток данных соответственно.
