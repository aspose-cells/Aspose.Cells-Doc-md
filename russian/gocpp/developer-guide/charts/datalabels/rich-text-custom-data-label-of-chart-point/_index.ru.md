{{% alert color="primary" %}}

Вы можете использовать Aspose.Cells для создания богатых текстовых пользовательских меток точек графика. Aspose.Cells предоставляет метод [DataLabels.Characters()](https://reference.aspose.com/cells/go-cpp/charttextframe/characters/), который возвращает объект [FontSetting](https://reference.aspose.com/cells/cpp/aspose.cells/fontsetting/), используемый для установки свойств шрифта текста, таких как его цвет, жирность и т.д.

{{% /alert %}}
