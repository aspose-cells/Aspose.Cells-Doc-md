## **Создание настраиваемых диаграмм**

На данный момент, когда мы говорим о графиках, мы рассматриваем стандартные графики с их стандартными настройками форматирования. Мы только указываем источник данных, задаем несколько свойств, и график создается с настройками по умолчанию. Но API Aspose.Cells также поддерживают создание настраиваемых графиков, позволяющих разработчикам создавать графики со своими собственными настроить.

Разработчики могут создавать пользовательские графики во время выполнения с использованием Aspose.Cells.

График состоит из серии данных. Каждая серия данных в Aspose.Cells представлена объектом [**Series**](https://reference.aspose.com/cells/go-cpp/series/), в то время как объект [**SeriesCollection**](https://reference.aspose.com/cells/cpp/aspose.cells.charts/seriescollection/) служит коллекцией объектов [**Series**](https://reference.aspose.com/cells/go-cpp/series/). При создании пользовательского графика разработчики имеют свободу использовать различные типы графиков для различных серий данных (собранных в объекте [**SeriesCollection**](https://reference.aspose.com/cells/cpp/aspose.cells.charts/seriescollection/)).
