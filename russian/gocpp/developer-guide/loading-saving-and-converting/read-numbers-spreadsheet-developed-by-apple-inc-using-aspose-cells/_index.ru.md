## **Возможные сценарии использования**

Numbers — это приложение для работы с таблицами, разработанное компанией Apple Inc. Aspose.Cells может читать таблицы Numbers, но не поддерживает редактирование. Он может читать данные, стили и формулы таблицы Numbers.

## **Чтение таблицы чисел, разработанной Apple Inc. с использованием Aspose.Cells**

Следующий пример кода загружает [образец таблицы Numbers](sampleNumbersByAppleInc.numbers) и преобразует его в [вывод в формате PDF](outputNumbersByAppleInc.pdf). Для успешной загрузки необходимо использовать класс **[LoadOptions](https://reference.aspose.com/cells/go-cpp/loadoptions/)** и указать **[LoadFormat::Numbers](https://reference.aspose.com/cells/go-cpp/loadformat/)** в качестве параметра конструктора. Скачать оба файла по приведённым ссылкам. Вы можете попробовать код с любой таблицей Numbers. Также рекомендуется ознакомиться с комментариями в коде для получения дополнительной информации.
