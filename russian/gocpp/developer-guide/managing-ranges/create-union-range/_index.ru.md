## **Создать объединенный диапазон**
Aspose.Cells позволяет создавать объединённый диапазон с помощью метода [CreateUnionRange](https://reference.aspose.com/cells/go-cpp/worksheetcollection/createunionrange/). Метод [CreateUnionRange](https://reference.aspose.com/cells/go-cpp/worksheetcollection/createunionrange/) принимает два параметра: адрес для создания объединённого диапазона и индекс листа. Метод [CreateUnionRange](https://reference.aspose.com/cells/go-cpp/worksheetcollection/createunionrange/) возвращает объект [UnionRange](https://reference.aspose.com/cells/cpp/aspose.cells/unionrange/).

Следующий пример демонстрирует создание объединённого диапазона с помощью метода [CreateUnionRange](https://reference.aspose.com/cells/go-cpp/worksheetcollection/createunionrange/). Выходной файл, созданный этим кодом, приложен для справки.
