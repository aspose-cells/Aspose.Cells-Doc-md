{{% alert color="primary" %}} 

Обычно макет тегов DIV игнорируется при загрузке HTML в объект рабочей книги Excel. Однако, если вы хотите сохранить макет тегов DIV, установите свойство [GetSupportDivTag()](https://reference.aspose.com/cells/go-cpp/htmlloadoptions/getsupportdivtag/) в значение **true**. Значение по умолчанию — **false**.

{{% /alert %}}
