## **Преобразование JSON в книгу Excel** 
Нет необходимости гадать, как преобразовать JSON в файл Excel, потому что библиотека Aspose.Cells for C++ предлагает лучшее решение. API Aspose.Cells поддерживает конвертацию формата JSON в таблицы. Вы можете использовать класс [JsonLoadOptions](https://reference.aspose.com/cells/go-cpp/jsonloadoptions/), чтобы указать дополнительные параметры для импорта JSON в Workbook. 

В следующем примере кода демонстрируется импорт JSON в книгу Excel. Пожалуйста, ознакомьтесь с кодом для преобразования [исходного файла](sample.json) в файл xlsx, сгенерированный кодом для справки.
