Для совместимости со старыми версиями Aspose.Cells по умолчанию отображает линии сетки пунктирными при преобразовании Excel в PDF. Однако современные версии Excel отображают линии сетки как сплошные линии.

С опцией [PdfSaveOptions.PaginatedSaveOptions(PaginatedSaveOptions_Impl* impl)](https://reference.aspose.com/cells/go-cpp/paginatedsaveoptions/~paginatedsaveoptions/), Aspose.Cells также может отображать линии сетки как сплошные линии.
