{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells per .Net V4.5.1 Aggiornamento rapido](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-v4.5.1-hotfix/)

{{% /alert %}} 

 Cosa c'è di nuovo?

- Supporta la crittografia/decrittografia di file xlsx Excel2007
 Supporta la lettura/scrittura di grafici e forme nel file xlsx di Excel2007
 Supporta per ottenere e impostare lo stile del gradiente della cella nel file Excel2007
 Supporta la lettura/scrittura della formattazione condizionale nel file Excel2003 SpreadML
 Supporta la lettura/scrittura della barra dei dati, della scala dei colori, della formattazione condizionale dell'icona Imposta nel file xlsx di Excel2007

 Cosa è migliorato?



- Conversione Xls2Pdf migliorata
 Conversione migliorata di Chart2Image
 Conversione Worksheet2Image avanzata e foglio di lavoro per la stampa
 Foglio di lavoro per la copia migliorato

 Cosa è stato risolto?

- Risolti alcuni bug durante la lettura/scrittura del file Excel 97-2003xls in alcuni casi speciali
 Risolto un bug durante l'inserimento di righe/colonne
Risolti alcuni bug durante il calcolo delle formule
 Risolto un bug di ordinamento dei dati
 Risolto un bug di copia dei grafici
 Risolti alcuni bug della funzione di adattamento automatico
