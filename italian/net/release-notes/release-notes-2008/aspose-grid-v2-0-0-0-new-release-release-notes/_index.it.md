 Questa pagina contiene le note di rilascio per[Aspose.Grid V2.0.0.0 Nuova versione](https://downloads.aspose.com/cells/net/new-releases/aspose.grid-v2.0.0.0-new-release/)

{{% /alert %}} 

 Cari clienti, abbiamo appena rilasciato Aspose.Grid 2.0.0!

 Cosa c'è di nuovo?
 Aspose.Grid.Web1. GridWeb è ora compilato in entrambi i framework 1.0 e 2.0.2. Supporta completamente .NET 2.0 framework.3. Supporta .NET 2.0 MEDIUM TRUST LEVEL.4. Supporta .NET 2.0 DataSourceControl Binding.5. Nuovo metodo API client resizeColumnToFit.6. Supporta Windows Vista.

Aspose.Grid.Desktop1. Nuova proprietà GridCell.Protected, indica se la cella è protetta. Se il valore è "true", l'utente non può modificare la cella tramite l'interfaccia utente.2. Nuova proprietà ComboBox.IsInputable, indica se l'utente può inserire testo nella casella combinata. Il testo immesso verrà salvato come valore della cella.3. Nuovo evento Desktop.CommentDataChanged, si verifica quando i dati del commento sono cambiati..4. La nuova proprietà Validation.CustomMsgForStopEdit consente all'utente di ottenere o impostare il messaggio personalizzato da visualizzare quando l'utente immette un valore non valido.5. Il nuovo metodo Worksheet.SetFirstVisibleColumn()/SetFirstVisibleRow() consente all'utente di impostare il primo indice di colonna/riga visibile della vista foglio.6. Il nuovo metodo Worksheet.GetFirstVisiableColumn()/GetFirstVisiableRow() consente all'utente di ottenere il primo indice di colonna/riga visibile della vista foglio.7. Nuovo metodo Foglio di lavoro. AddSelectedRange() consente all'utente di aggiungere un nuovo intervallo selezionato al foglio di lavoro..8. Nuova proprietà Worksheet.ShowGridlines, abilita l'utente a mostrare/nascondere le linee della griglia.

 Cosa è migliorato?
Aspose.Grid.Web1. Supporto XHTML migliorato.2. Ordinamento migliorato. Null Cells rimarrà sempre nella parte posteriore.3. Copia/Incolla migliorata.

 Cosa è stato risolto?
 Aspose.Grid.Web1. Risolto il bug dei pulsanti del filtro automatico che non venivano riposizionati dopo l'adattamento automatico di una colonna.2. Risolto un bug di Excel Exporting.3. Risolto un bug di Impossibile ottenere l'oggetto Cell di un collegamento ipertestuale di tipo comando in XhtmlMode. 4. Risolto il bug dell'aggiornamento automatico AJAX che non funzionava in un modulo utilizzando la pagina principale.

Aspose.Grid.Desktop1. Risolto un bug di visualizzazione dei commenti.2. Risolto un problema di input: nella versione precedente, in alcune condizioni speciali, premere il tasto Invio in una cella di modifica pulirà il contenuto della cella.
