 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for .NET V4.4.2!

 Formato Xlsm 2007 supportato

 Cosa è migliorato?

- Motore di calcolo delle formule migliorato
Conversione xls2pdf migliorata
 Conversione chart2Image migliorata

 Cosa è stato risolto?

- Risolti bug nel metodo CopyRow/CopyColumn
 Risolto un bug nella proprietà Cell.StringValue
 Risolto un bug nel metodo FreezePanes
