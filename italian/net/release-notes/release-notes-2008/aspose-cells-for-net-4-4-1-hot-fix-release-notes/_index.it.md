 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for .NET V4.4.1!

 Cosa c'è di nuovo?

- Rileva automaticamente il tipo di file all'apertura di un file Excel

 Cosa è migliorato?

- Supporto ottimizzato di Excel2007 Xlsx
Conversione xls2pdf migliorata
 Supporto della tavola pivot migliorato
 Supportate più funzioni nel motore di calcolo delle formule

 Cosa è stato risolto?

- Errore nell'apertura di alcuni file Excel speciali dell'estremo oriente
 Errore nell'analisi e nel calcolo delle formule
 Bug nelle impostazioni di configurazione della pagina
 Bug nell'import/export di alcuni file di testo
 Bug nell'elaborazione dei marcatori intelligenti
 Problema nell'installazione su vista
