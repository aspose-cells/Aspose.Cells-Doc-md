{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for .NET 4.9.1](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-4.9.1/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for .NET v!



 Cosa è cambiato in Aspose.Cells:

- Include l'assieme Aspose.Cells for .NET Framework Client Profile
- Migliora le prestazioni del motore di calcolo delle formule
- Migliora le prestazioni di apertura dei file xlsx
- Supporta la creazione di un intervallo denominato in base all'indirizzo
- 23 correzioni



 Cosa è cambiato in Aspose.Cells.GridWeb:

- Supporta l'impostazione della visibilità delle barre di scorrimento orizzontali o verticali
- 1 correzione



 Cosa è cambiato in Aspose.Cells.GridDesktop:

- Esegue il copia e incolla delle celle con il controllo ComboBox
- Gestisce l'evento Popup di ContextMenu
- Importa un foglio di lavoro da un file Excel
- Recupera tutti gli intervalli selezionati nel foglio di lavoro
- 2 miglioramenti
