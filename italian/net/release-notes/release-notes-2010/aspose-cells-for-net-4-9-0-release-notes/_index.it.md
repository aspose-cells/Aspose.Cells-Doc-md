{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for .NET 4.9.0](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-4.9.0/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for .NET v!

 Cosa è cambiato in Aspose.Cells:

- Calcola le formule che superano il limite di riga/colonna di Excel2003
- Supporta gli sparkline di MS Excel 2010.
- Implementa effetti 3D sui grafici
- Rimuove i valori duplicati
- Supporta Consenti agli utenti di modificare gli intervalli
- Il motore di calcolo delle formule è stato migliorato
- Rileva e rimuove le connessioni a dati esterni
- Imposta l'immagine collegata nel file xlsx
- Include ICustomTypeDesciptor come origine dati di marcatori intelligenti
- I dati di origine collegati esterni nella funzione Grafico a immagine sono supportati
- Sono supportate le opzioni di configurazione della pagina esterna di MS Excel 2007 nel file xls
- Specifica il contenuto HTML come valore della cella o nota del commento
- 112 correzioni e altri miglioramenti.

 (Nota: la classe Aspose.Cells.Charts è stata rinominata in Aspose.Cells.Classe ChartCollection.)

 Cosa è cambiato in Aspose.Cells.GridWeb:

- Aggiunge nuovi formati di data: "M/g/aa", "MM/gg/aaaa" e "gg.mm.aaaa"
- Imposta il messaggio di errore nel titolo della cella quando la convalida non riesce sul lato client
- Ottiene il nome della colonna per una cella sul lato client
- Salva i file Excel per lo streaming o carica i file Excel dallo stream
- Supporta nuove funzioni: INDICE, CERCA, CONFRONTA
- 8 correzioni e altri miglioramenti.

 Cosa è cambiato in Aspose.Cells.GridDesktop:

- Imposta i formati di data "MM/GG/AAAA" e "M/G/AAAA" con la finestra di dialogo Formato Cells
- Importa il formato di file CSV
- Supporta formule: CERCA, MATCH
- 3 correzioni e altri miglioramenti.
