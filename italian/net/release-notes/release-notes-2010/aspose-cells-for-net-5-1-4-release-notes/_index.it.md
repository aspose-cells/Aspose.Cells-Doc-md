{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for .NET 5.1.4](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-5.1.4/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for .NET v5.1.4!

 Cosa è cambiato in Aspose.Cells:

- Supporta l'esportazione di file crittografati PDF.
- 45 miglioramenti e correzioni.

 Cosa è cambiato in Aspose.Cells.GridWeb:

- Obsolete: le proprietà NumberType, Custom e IsLocked di Aspose.Cells.GridWeb.Data.WebCell sono obsolete. Queste proprietà vengono ora aggiunte a Aspose.Cells.GridWeb.TableItemStyle.
- Obsoleto: l'attributo di stile di Aspose.Cells.GridWeb.Data.WebCell è obsoleto. Le alternative consigliate utilizzano i metodi Aspose.Cells.GridWeb.Data.WebCell.GetStyle() e Aspose.Cells.GridWeb.Data.WebCell.SetStyle().
- 1 correzione e 1 miglioramento.



 Cosa è cambiato in Aspose.Cells.GridDesktop:

- Supporta l'importazione/esportazione di filtri automatici nei file Excel.
- 1 correzione e 1 miglioramento.
