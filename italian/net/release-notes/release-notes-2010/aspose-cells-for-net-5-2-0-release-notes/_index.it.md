 Questa pagina contiene le note di rilascio per[Aspose.Cells for .NET 5.2.0](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-5.2.0/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for .NET v5.2.0!

 Cosa è cambiato in Aspose.Cells:

- Supporta la firma e la convalida delle firme digitali nei file XLSX.
 Supporta tabelle pivot R/W nei file XLS e XLSX.
Calcola i dati della tabella pivot in celle.
 31 miglioramenti e correzioni.

 Cosa è cambiato in Aspose.Cells.GridWeb:

- Supporta l'importazione/esportazione di immagini di MS Excel.



 Cosa è cambiato in Aspose.Cells.GridDesktop:

- Supporta intervalli denominati.
 1 correzione e 1 miglioramento.
