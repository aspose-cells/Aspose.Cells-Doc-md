{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for .NET v4.8.0](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-v4.8.0/)

{{% /alert %}} 

 Siamo lieti di annunciare Aspose.Cells for .NET v4.8.0.0!

 Cosa è cambiato:

 l Unisci la suite Aspose.Grid in Aspose.Cells

 l Ottimizza le prestazioni dei ricalcoli delle formule

 l Migliorare le prestazioni di inserimento/eliminazione di righe/colonne

 l Migliora le prestazioni della funzione Adatta automaticamente righe/colonne

 l Aggiungi forme WordArt al file XLS

 Leggi gli effetti di riempimento sfumato per il grafico

 l Creare fogli di calcolo con attributi Prima pagina o Pari/Dispari Intestazione/Piè di pagina per Imposta pagina

 l Supporta più aree di stampa nella funzione da foglio di lavoro a immagine

 l Aggiunto il supporto per il calcolo delle formule per alcune nuove funzioni integrate di MS Excel

 l Impostare l'immagine di sfondo per un foglio di lavoro di tipo grafico

 l Fornire supporto per l'impostazione delle Opzioni immagine nelle funzioni Grafico a immagine e Foglio di lavoro a immagine

 l 101 correzioni e altri miglioramenti
