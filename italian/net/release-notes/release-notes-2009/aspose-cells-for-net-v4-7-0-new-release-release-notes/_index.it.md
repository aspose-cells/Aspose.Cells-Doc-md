 Questa pagina contiene le note di rilascio per[Aspose.Cells per .Net V4.7.0 Nuova versione](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-v4.7.0-new-release/)

{{% /alert %}}

Siamo felici di annunciare Aspose.Cells for .NET 4.7.0!

Cosa è cambiato:

- Supporta la conversione diretta di file Excel in PDF.
 Supporta l'opzione Gridline dell'impostazione di impostazione della pagina durante la stampa del foglio di lavoro.
 Supporta l'inserimento di un file WAV come oggetto Ole.
Supporta le caselle di controllo R/W nei file Excel 2007.
 Supporta l'utilizzo di tipi anonimi o oggetti personalizzati come origine dati in Smart marker
 Supporta l'impostazione dell'opzione di visualizzazione del valore zero.
 Supporta l'unione e l'intersezione di due intervalli.
 Supporta la conservazione dei dati dopo aver salvato la cartella di lavoro.
 Supporta l'opzione di inserire o meno righe vuote nella tabella pivot.
 Migliora il motore di calcolo e supporta più funzioni.
 45 correzioni e miglioramenti.
