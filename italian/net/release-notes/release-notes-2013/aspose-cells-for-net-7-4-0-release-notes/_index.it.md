 Questa pagina contiene le note di rilascio per[Aspose.Cells for .NET 7.4.0](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-7.4.0/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells per .NETv7.4.0!



\1) Aspose.Cells 



 Miglioramenti

- Rendering della formattazione condizionale avanzata nella funzione Sheet-to-Image
- Il righello è invisibile nella visualizzazione Layout di pagina



 Prestazione

- Impiccato durante il salvataggio in formato PDF



 Eccezioni

- Eccezione all'apertura di un file Open XML



 Insetti

- Bugin usando la formula come parametro di una funzione personalizzata
- Problema di aggiornamento e calcolo dei dati della tabella pivot
- CERCA.ORIZZ con bug delle formule INDIRETTE
- Il foglio di lavoro non protetto non riesce per i file salvati con Excel 2013
- Problema del numero di pagina del piè di pagina nell'esportazione in PDF
- AcceptAllRevisions non viene applicato correttamente durante il salvataggio della cartella di lavoro nei flussi

 -Caselle di controllo danneggiate rese al salvataggio

- La decrittografia di un file protetto da password danneggia il file

 - Intervalli di nomi corrotti al salvataggio

- Le impostazioni locali non funzionano per una cella con una formattazione DataTime
- Errore nella formattazione di un documento con citazione semplice nella colonna della scheda Intestazione
- La formattazione di caratteri specifici in una stringa non funziona per un file XLSX



 \2) Aspose.Cells.GridWeb



 Nuove caratteristiche

 41269 - Carica file Excel con elenchi a discesa - Convalida dati elenco.

 41312 - Supporta l'evento clic per disegnare oggetti/forme



 Miglioramenti

 41301 - Digitare direttamente il valore in una cella non è supportato su Chrome, FireFox e Safari



 Insetti

 41279 - L'elenco dei menu di convalida di GridWeb fallirà sulle voci di menu che contengono il simbolo del dollaro -"$"

 41280- L'elenco del menu di convalida di GridWeb viene aggiornato dopo l'aggiornamento del valore della cella di riferimento

 41294 - GridWeb non funzionerà su FireFox per i file che contengono celle con elenco di convalida

41308 - Durante la modifica di un valore di cella, premendo il tasto Invio si perderà l'interruzione di riga originale in IE

- Problema di copia-incolla nella griglia Web che non funziona per Chrome, FireFox ecc.

 Miglioramenti

 41291 - Problema durante l'utilizzo della tastiera per passare a qualsiasi cella per inserire un valore
