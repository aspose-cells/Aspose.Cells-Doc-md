- Risolti 2 bug minori nell'impostazione delle formule
- Risolto un bug nel calcolo delle formule
- Risolto il problema dell'utilizzo di Aspose.Cells su Windows Vista RC2. Questo problema è causato da obfucator.
- Risolto un bug di orientamento della pagina durante la conversione in file pdf
- Risolto un bug di copia delle righe con formule condivise
