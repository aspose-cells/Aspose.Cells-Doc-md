 Questa pagina contiene le note di rilascio per[Aspose.Grid .Net 1.7.0.0 Nuova versione](https://downloads.aspose.com/cells/net/new-releases/aspose.grid-.net-1.7.0.0-new-release/)

{{% /alert %}} 

Gentili clienti, abbiamo appena rilasciato Aspose.Grid 1.7! Questa versione include molte nuove entusiasmanti funzionalità: Aspose.Grid.Web 1.71.Nuovo METAL-LIGHT Look & Feel.2.Aggiunto supporto completamente DataBinding. Ora supporta fogli di rilegatura per DataSet, DataTable, DataView o qualsiasi oggetto di raccolta. Il controllo carica i dati dal DataSource e aggiorna automaticamente il DataSource. 3. Aggiunto il nuovo "Worksheets Designer" nell'IDE vs.net. Puoi usarlo per creare potenti applicazioni di associazione di database con pochi clic del mouse e poche righe di codice. 4. Aggiunti nuovi tipi di convalida: FreeList, Number, Integer, Date, DateTime e CustomFunction. 5. Effetto visivo e usabilità migliorati in None-IE browsers.6.Aggiunte 6 nuove proprietà di colore per consentire la regolazione di ActiveCell/SelectCell/SelectHeader's Color/BackgroundColor.7.Aggiunta nuova proprietà "Visibile" a WebWorksheet. Puoi usarlo per nascondere un foglio di lavoro dalla TabBar. 8. Aggiunto il supporto dei pulsanti di comando personalizzati. È possibile creare i propri pulsanti nella TabBar del controllo. 9. Ora è possibile utilizzare la convalida dell'espressione regolare con una delle altre convalide contemporaneamente. 10. Aggiunto il metodo WebWorksheet.Copy (WebWorksheet), che consente di copiare il contenuto di un foglio di lavoro in un altro foglio di lavoro anche se si trovano in due diversi controlli GridWeb. 11. Aggiunto supporto per eventi di invio/convalida lato client. È possibile scrivere il proprio script client per gestire l'evento di errore di convalida o di invio lato client. 12. Aggiunte funzioni formula "MIN", "MAX". 13. Risolto il bug della perdita di tipi di dati durante l'importazione da un oggetto DataView. Risolti alcuni problemi in VS.NET 2005.15. Prestazioni ottimizzate quando è presente un numero elevato di convalide in un foglio.

Aspose.Grid.Desktop 1.71.Raggruppamento di righe o colonne supportato.2.Nascondere l'intestazione della colonna di riga supportata.3.Risolto il problema con il pulsante o il ComboBox di una cella focalizzata che nascondeva una parte del testo della cella.4.Risolto il bug del metodo AddRow.5 .Risolto il bug per cui il controllo non può essere creato istanza in VS.NET 2002.6.I progetti demo supportano VS.NET 2002.7.Risolto il bug della proprietà Dock.Fill del controllo.8.Prestazioni ottimizzate del metodo ImportExcelFile e ExportExcelFile.9.Aggiunto metodo CreateRange alla classe del foglio di lavoro. 10. Aggiunto il metodo Copy alla classe CellRange.
