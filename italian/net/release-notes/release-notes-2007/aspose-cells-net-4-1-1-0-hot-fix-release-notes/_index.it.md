{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells .Net 4.1.1.0 correzione rapida](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-.net-4.1.1.0-hot-fix/)

{{% /alert %}} 

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for .NET v4.1.1!

- Demo WebForm migliorate
- F0007: funzione VAN supportata nel motore di calcolo delle formule
- F0009: supportata per leggere la stringa delle caselle di testo nei file modello
- F0015: crittografia BIFF7 supportata in alcuni file protetti
- F0016: funzione INDIRETTO, VAL.VUOTO, VAL.ERRORE, ISNA, VAL.NUMERO supportata nel motore di calcolo delle formule
- F0017: stringa di formato datetime tedesca supportata
- F0019: supportato per impostare la cella attiva in un foglio di lavoro
- B0001: Corretto un bug nella proprietà Cells.MaxDataRowInColumn
- B0002: Corretto un bug durante il caricamento di file csv con stringhe vuote
- B0003: Risolto un bug durante il caricamento di commenti con contenuto di stringa superiore a 8k
- B0004: Risolto un bug per caricare un file con protezione della cartella di lavoro
- B0005: Risolto un bug nel metodo InsertRows che poteva causare l'errore "Formula non valida ..." per alcuni file
- B0006: Risolto un bug per la distorsione dell'immagine durante l'esecuzione con una risoluzione dello schermo di 120 DPI.
- B0007: Risolto un bug nel metodo CalculateFormula durante il confronto tra due valori DateTime
