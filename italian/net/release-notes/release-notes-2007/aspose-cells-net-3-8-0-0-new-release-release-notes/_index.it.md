 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for .NET v3.8!

- Supporta la creazione di caselle di controllo in fase di esecuzione
- Supporta la creazione di oggetti OLE in fase di esecuzione
- Supporta la protezione/rimozione della protezione delle cartelle di lavoro
- Consente di modificare l'origine dati dei collegamenti esterni
- Elaborazione avanzata dei marcatori intelligenti
- Motore di calcolo delle formule migliorato
- Risolti bug in DeleteRows e InsertRows
- Risolto un bug di apertura e salvataggio ripetuti di un file con la versione di valutazione
- Risolto bug AddCopy durante la copia di alcune formule
- Altri miglioramenti e correzioni dalla v3.7
