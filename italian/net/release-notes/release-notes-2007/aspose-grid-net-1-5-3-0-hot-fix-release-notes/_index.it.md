 Questa pagina contiene le note di rilascio per[Aspose.Grid .Net 1.5.3.0 correzione rapida](https://downloads.aspose.com/cells/net/new-releases/aspose.grid-.net-1.5.3.0-hot-fix/)

{{% /alert %}} 

 Abbiamo appena rilasciato Aspose.Grid 1.5.3!

 Miglioramento e correzione di bug in questa versione:

 Aspose.Grid.Desktop

 Migliorato: 1. Convalida colonna supportata. 2. Controllo colonna supportato. 3. Stile di riferimento R1C1 supportato. 4. Ordinamento supportato.

 Aspose.Grid.Web

 Bug risolti: 1. Quando il numero di righe è maggiore di circa 100, FreezePanes (Fixed Rows/Columns) a volte scompariva. 2. Quando si seleziona più un intervallo di celle e si incollano i dati, le celle di sola lettura vengono modificate. 3. Celle di tipo DropDownList può essere cancellato.4.La larghezza della colonna non è corretta in alcune impostazioni della regione.

Migliorato: 1. Supporto per l'impostazione di didascalie personalizzate per colonne/righe (WebCells.SetRowCaption, WebCells.SetColumnCaption). 2. Supporto per l'ordinamento di un intervallo di celle, sia dall'alto verso il basso che da sinistra a destra (WebCells.Sort). 3. Supporto per l'esportazione celle dati a un oggetto DataTable (WebCells.Export).4.Aspose.Grid.Web UI Designer. Funziona dal menu di avvio del sistema o dall'ambiente di Visual Studio (di gran lunga supporta solo .net 1.0/1.1).
