 Questa pagina contiene le note di rilascio per[Aspose.Cells .Net 4.0.0.0 Nuova versione](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-.net-4.0.0.0-new-release/)

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for .NET V4.0!

- Modifica del nome della classe: ExcelHelper->CellsHelper, ExcelException->CellsException
- Metodo AutoFitRow modellato per renderlo più preciso
- Aggiunto il metodo Worksheets.SortNames per risolvere un problema durante l'aggiunta di una grande quantità di intervalli denominati.
- Aggiunte nuove demo di grafici
- Risolto un bug nell'impostazione della formula
- Risolto un bug durante il salvataggio del file in streaming con il formato CSV
