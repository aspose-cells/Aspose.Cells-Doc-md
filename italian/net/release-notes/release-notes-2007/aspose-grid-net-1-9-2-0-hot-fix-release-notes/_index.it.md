Questa pagina contiene le note di rilascio per Aspose.Grid .Net 1.9.2.0 Hot Fix

{{% /alert %}}

Cari clienti,

Abbiamo appena rilasciato Aspose.Grid v1.9.2!

introduzione

Questa nuova versione include tutte le nuove funzionalità e correzioni dalla v1.9.0.

Cosa c'è di nuovo?

Aspose.Grid.Web

1. Supporta il rendering delle specifiche XHTML 1.0.

2. Supporta la serializzazione dei fogli di lavoro. Ora gli utenti possono utilizzare StateServer o SQLServer come archivio di sessione.

3. Supporta sia il filtro automatico che il filtro personalizzato.

4. Supporta la conformità CLS.

5. Supporta il trascinamento e lo scorrimento.

6. Supporta la visualizzazione della barra dei comandi nella parte superiore del controllo.

7. Supporta la visualizzazione di una casella "Caricamento in corso..." quando PostBack.

Cosa è migliorato?

Aspose.Grid.Web

1. Scorrimento delle schede del foglio migliorato.

2. Notevolmente migliorate le prestazioni in presenza di un numero elevato di convalide di List/FreeList/DropdownList.

3. Prestazioni di rendering XHTML notevolmente migliorate (rispetto a 1.9.1.0).

Aspose.Grid.Desktop

1. Nuovo motore di formula, notevolmente migliorato le prestazioni di esecuzione della formula.

Cosa è stato risolto?

Aspose.Grid.Web

1. Risolto un problema di stile di cella errato dopo un'operazione di ordinamento.

\2. Risolto un problema di visualizzazione errata di DropdownList nella modalità di rendering XHTML.

Aspose.Grid.Desktop

1. Risolto il bug di incollare una cella da un'altra applicazione.
