 Questa pagina contiene le note di rilascio per[Aspose.Cells .Net 3.9.0.0 Nuova versione](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-.net-3.9.0.0-new-release/)

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for .NET v3.9!

- Per alcuni motivi esterni, ora la classe Excel viene rinominata in Workbook e la classe ExcelDesigner viene rinominata in WorkbookDesigner. Tutte le proprietà e il metodo di queste due classi non vengono modificati.
- Supporta l'aggiunta di immagini EMF nel loro formato raw
- Viene aggiunto un nuovo marcatore intelligente
- Supporta il caricamento di file di testo con separatore personalizzato
- Risolto un bug nell'impostazione dei collegamenti ipertestuali ai file locali
- Risolti bug nel metodo InsertRows
- Risolto un bug nelle impostazioni di convalida dei dati
- Risolto un bug nell'apertura di alcuni file Excel
