 Questa pagina contiene le note di rilascio per[Aspose.Grid .Net 1.5.1.0 correzione rapida](https://downloads.aspose.com/cells/net/new-releases/aspose.grid-.net-1.5.1.0-hot-fix/)

 Cari clienti,

 Abbiamo rilasciato Aspose.Grid.Web v1.5.1!

 In questa nuova correzione, apportiamo i seguenti miglioramenti e correzioni:

 Aspose.Grid.Desktop: ottimizza le prestazioni e l'utilizzo della memoria Supporta righe, stili di colonna Risolto: bug del valore della barra di scorrimento durante il blocco di righe o colonne Risolto: collegamento ipertestuale, commento, indice delle immagini non vengono aggiornati durante l'inserimento, l'eliminazione di righe o colonne

Aspose.Grid.Web: modifica il nome della classe del controllo da "ExcelWeb" a "GridWeb".Supporta la barra dell'intestazione nascostaSupporta la barra di scorrimento nascostaSupporta il percorso relativo per i file di utilizzo lato client.Supporta lo stile dell'immagine di sfondo della cella/tabella.Supporta il salvataggio/caricamento del file binario definito Aspose ( AGWF-Aspose Grid Web Format). Aggiunta nuova proprietà "AGWClientPath", è possibile utilizzare questa proprietà per impostare il percorso per i file di utilizzo lato client (script, immagini, ecc.). Risolto bug di colore durante l'esportazione in file excel. Risolto bug di carattere predefinito .
