## **Piattaforme Supportate**
Il prodotto Aspose.Cells for .NET copre la maggior parte degli ambienti di sviluppo popolari e delle piattaforme di distribuzione. 

|**Funzionalità**|**Descrizione**|
| :- | :- |
|[.NET Framework](/cells/it/net/system-requirements/#net-frameworks)|Usa Aspose.Cells for .NET per costruire ASP.NET, servizi Web, WinForms o qualsiasi altro tipo di applicazione per qualsiasi versione di .NET Framework 2.0 o successiva, inclusa la versione Client Profile. Supporto completo per 32-bit e 64-bit.|
|[.NET Standard 2.0](/cells/it/net/system-requirements/#net-standard)|La cartella 'netStandard' contiene l'assembly da utilizzare con framework che implementano .NET Standard 2.0 come .NET Core 2.1/3.1. L'ultima versione dell'API Aspose.Cells supporta anche .NET 5.0, .NET 6.0, .NET 7.0 e .NET8.0.|
|[COM](/cells/it/net/system-requirements/)|Può essere un salvavita se devi sviluppare utilizzando tecnologie pre-.NET.|
|[Mono](/cells/it/net/system-requirements/)|È possibile utilizzare Aspose.Cells for .NET per creare applicazioni con Mono.|
|[Windows Azure](/cells/it/net/system-requirements/)|Aspose.Cells for .NET è inoltre eseguito su Windows Azure.|
