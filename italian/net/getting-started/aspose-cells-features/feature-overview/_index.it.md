Il prodotto Aspose.Cells for .NET copre la maggior parte degli ambienti di sviluppo e delle piattaforme di distribuzione più diffusi.

|**Caratteristica**|**Descrizione**|
| :- | :- |
|[.NET Quadro](/cells/it/net/system-requirements/#net-frameworks)|Utilizzare Aspose.Cells for .NET per creare ASP.NET, servizi Web, WinForms o qualsiasi altro tipo di applicazione per qualsiasi versione .NET Framework 2.0 o successiva, incluse le versioni del profilo client. Supporto completo per 32 bit e 64 bit.|
|[.NET Norma 2.0](/cells/it/net/system-requirements/#net-standard)|La cartella 'netStandard' contiene l'assembly da utilizzare con il framework che implementa .NET Standard 2.0 come .NET Core 2.1/3.1. L'ultima versione di Aspose.Cells API supporta anche .NET 5.0, .NET 6.0, .NET 7.0 e .NET8.0.|
|[COM](/cells/it/net/system-requirements/)|Può essere un vero toccasana se devi sviluppare utilizzando tecnologie precedenti a .NET.|
|[Mono](/cells/it/net/system-requirements/)|È possibile utilizzare Aspose.Cells for .NET per creare applicazioni con Mono.|
|[Windows Azzurro](/cells/it/net/system-requirements/)|Aspose.Cells for .NET funziona anche su Windows Azure.|
