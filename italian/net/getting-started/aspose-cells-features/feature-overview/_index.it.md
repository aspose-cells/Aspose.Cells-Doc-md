## **Piattaforme supportate**
Il prodotto Aspose.Cells for .NET copre la maggior parte degli ambienti di sviluppo e delle piattaforme di distribuzione più diffusi.

|**Caratteristica**|**Descrizione**|
|:- |:- |
|[.NET Framework](/cells/it/net/system-requirements/#net-frameworks)|Utilizzare Aspose.Cells for .NET per creare ASP.NET, servizi Web, WinForms o qualsiasi altro tipo di applicazione per qualsiasi versione di .NET Framework 2.0 o successiva, comprese le versioni del profilo client. Pieno supporto per 32 bit e 64 bit.|
|[Standard .NET 2.0](/cells/it/net/system-requirements/#net-standard)|La cartella "netStandard" contiene l'assembly da usare con framework che implementano .NET Standard 2.0 come .NET Core 2.1/3.1. L'ultima versione dell'API Aspose.Cells supporta anche .NET 5.0 e .NET 6.0.|
|Xamarin.Android|Usa Aspose.Cells for .NET nelle tue applicazioni Android native.|
|Novell. iOS|Usa Aspose.Cells for .NET nelle tue applicazioni iOS native.|
|[COM](/cells/it/net/system-requirements/)|Può essere un vero toccasana se devi sviluppare utilizzando tecnologie pre-.NET.|
|[Mono](/cells/it/net/system-requirements/)|È possibile utilizzare Aspose.Cells for .NET per creare applicazioni con Mono.|
|[Windows Azure](/cells/it/net/system-requirements/)|Aspose.Cells for .NET funziona anche su Windows Azure.|
