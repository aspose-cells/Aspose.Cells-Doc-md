{{% alert color="primary" %}} 

Aspose.Cells può essere utilizzato per salvare i tuoi workbook di Excel in formato HTML. I workbook HTML esportati sono orientati a Microsoft Excel cioè sono simili a quelli esportati da Microsoft Excel. Puoi anche creare workbook da HTML ma questo HTML dovrebbe essere orientato a Excel.

{{% /alert %}} 
## **HTML in Aspose.Cells**
La soluzione Aspose supporta completamente le seguenti caratteristiche del documento durante la conversione in HTML:

- Dati delle celle e formattazione del testo
- Grafici e tabelle pivot
- Intestazioni/Piè di pagina
- Bordi, colori dei bordi
- Immagini, caselle di testo e forme disegnate
- Collegamenti ipertestuali
