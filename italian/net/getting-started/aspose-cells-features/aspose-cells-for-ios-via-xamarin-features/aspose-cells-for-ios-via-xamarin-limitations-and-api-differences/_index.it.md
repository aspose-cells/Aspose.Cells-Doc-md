## L'ultima versione di Aspose.Cells per iOS tramite Xamarin potrebbe non funzionare con la versione precedente di Xamarin.iOS
Tieni presente che Aspose.Cells per iOS tramite Xamarin viene sempre compilato usando le versioni stabili più recenti delle piattaforme Xamarin e Xamarin.iOS. In caso di problemi durante l'uso di Aspose.Cells per iOS tramite Xamarin nell'applicazione Xamarin.Android, assicurarsi di avere installato le versioni più recenti di Xamarin e Xamarin.iOS. A volte Aspose.Cells per iOS tramite Xamarin viene compilato usando l'ultima versione di Xamarin (Xamarin.iOS) che non funziona con le versioni precedenti di Xamarin.
## Aspose.Cells per iOS tramite limitazioni Xamarin
- Inserimento di immagini - Non supportato.
- Creazione di grafici: non supportata.
- Impostazione dello sfondo sfumato - Non supportato.
- Aggiunta di commenti alle celle: non supportata.
- Implementazione delle convalide dei dati - Non supportata.
- Creazione di interruzioni di pagina personalizzate: non supportata.
- Implementazione di marcatori intelligenti - Non supportato.
- Protezione/non protezione dei fogli di lavoro - Non supportata.
- Specifica delle opzioni di protezione avanzate introdotte in Excel XP e versioni successive - Non supportato.
- Inserimento di controlli del modulo e altre forme/oggetti di disegno - Non supportato.
- Creazione di tabelle pivot e grafici pivot - Non supportato.
- Mantenimento o rimozione di un componente aggiuntivo, VBA, macro: non supportato.
- Implementazione delle opzioni di trasposizione - Non supportato.
- Creazione di grafici personalizzati: non supportata.
- Aggiunta, conservazione o estrazione di oggetti OLE dai fogli di calcolo - Non supportato.
- Implementazione delle sparkline di Microsoft Excel 2010 - Non supportata.
- Crittografia dei file: non supportata.
## Differenze nell'API pubblica (spazio dei nomi).
