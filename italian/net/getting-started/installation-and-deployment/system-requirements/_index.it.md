{{% alert color="primary" %}} 

I componenti Aspose.Cells for .NET non richiedono che Microsoft Excel sia installato né sul lato client né su quello del server.

{{% /alert %}}

## **Ambienti di sviluppo**
Puoi utilizzare Aspose.Cells for .NET per sviluppare applicazioni in qualsiasi ambiente di sviluppo che mira alla piattaforma .NET, ma gli ambienti seguenti sono esplicitamente supportati:

- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- Microsoft Visual Studio 2022

{{% alert color="primary" %}} 

Per utilizzare Aspose.Cells, la macchina non ha bisogno di avere Microsoft Excel installato poiché la libreria stessa è un motore di gestione dei fogli di calcolo. Tuttavia, per visualizzare il file excel generato dal componente, è necessario avere almeno Microsoft Excel Viewer installato sulla tua macchina.

{{% /alert %}} 
## **Requisiti di sistema Aspose.Cells Grid Suite**
Aspose.Cells Grid Suite è composto da tre controlli:

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb
- Aspose.Cells.GridJs

I requisiti di sistema dei controlli sono descritti separatamente.

{{% alert color="primary" %}} 

Per utilizzare questi controlli, la macchina su cui eseguire Aspose.Cells.GridDesktop e Aspose.Cells.GridWeb non ha bisogno di avere Microsoft Excel installato in quanto Aspose.Cells.GridDesktop e Aspose.Cells.GridWeb possono aprire e salvare fogli elettronici Excel senza bisogno di Microsoft Excel.
