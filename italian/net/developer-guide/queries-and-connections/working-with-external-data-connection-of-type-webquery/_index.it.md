 È possibile accedere a una connessione dati esterna di qualsiasi tipo utilizzando la raccolta Workbook.DataConnections. Un tipo di tale connessione dati è WebQuery. Questo articolo ti mostrerà come lavorare con la connessione dati WebQuery. È possibile creare una connessione dati WebQuery in Microsoft Excel utilizzando il file**Dati** > **Dal web** menù.
