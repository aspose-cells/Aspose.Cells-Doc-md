{{% alert color="primary" %}}

 Puoi verificare se il tuo progetto VBA è firmato o meno utilizzando Microsoft Excel tramite**Strumenti > Firme digitali...** comando di menù. Allo stesso modo, puoi controllarlo a livello di codice usando Aspose.Cells[**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned) proprietà.

{{% /alert %}}
