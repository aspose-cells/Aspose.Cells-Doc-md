## Verifica se il Progetto VBA è Protetto e Bloccato per la Visualizzazione in C#

Aspose.Cells ti permette di verificare se il Progetto VBA (Visual Basic for Applications) del file Excel è protetto e bloccato per la visualizzazione. Per questo, l'API fornisce la proprietà [**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing). Se è bloccato per la visualizzazione, allora la proprietà [**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) restituisce **true**.
