Microsoft Excel 2007 e versioni successive (2010/2013/2016) forniscono alcune funzionalità avanzate per la formattazione condizionale. Ad esempio, ti consente di applicare ombreggiatura delle celle, bordi, icone colorate, frecce, bandiere e formattazione dei caratteri, ecc., che è diventato piuttosto sofisticato.

{{% /alert %}} 
##  **Applica la formattazione condizionale avanzata ai file Excel Microsoft**
La formattazione condizionale può:

- Aggiungi barre dati ombreggiate per migliorare graficamente i numeri sottostanti incorporando un semplice grafico a barre nelle celle.
- Ombreggia automaticamente le celle con scale di colore in base alla loro relazione con i valori nelle altre celle dell'intervallo. Le impostazioni predefinite ombreggiano il valore più basso in rosso spostandosi fino al valore più alto in verde.
- Utilizza i set di icone in modo simile alle scale di colori, ma invece di ombreggiare le celle aggiunge piccole icone, come frecce e semafori alle celle.

Aspose.Cells supporta completamente la formattazione condizionale fornita da Microsoft Excel 2007 e versioni successive nel formato XLSX sulle celle in fase di esecuzione. Questo esempio mostra un esercizio per tipi di formattazione condizionale avanzati tra cui IconSets, DataBars, Color Scales, TimePeriods, Top/Bottom e altre regole con diversi set di attributi.
