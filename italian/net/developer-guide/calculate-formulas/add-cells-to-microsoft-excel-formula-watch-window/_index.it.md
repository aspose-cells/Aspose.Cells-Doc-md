##  **Possibili scenari di utilizzo**

Microsoft Excel Watch Window è uno strumento utile per osservare comodamente i valori delle celle e le relative formule in una finestra. Puoi aprire il*Guarda la finestra* utilizzando Microsoft Excel facendo clic su*Formule > Guarda* *Finestra*. Ha il *Aggiungi orologio*pulsante che può essere utilizzato per aggiungere le celle per l'ispezione. Allo stesso modo, puoi usare[**Foglio di lavoro.CellWatches.Add()**](https://reference.aspose.com/cells/net/aspose.cells/cellwatchcollection/methods/add/index) metodo per aggiungere celle*Guarda la finestra*utilizzando Aspose.Cells API.
