{{% alert color="primary" %}} 

**Office OpenXML**(conosciuto anche come**OOXML**o**Microsoft Apri XML**(MOX)) è un formato di file basato su XML sviluppato da Microsoft per rappresentare documenti d'ufficio come fogli di calcolo, grafici, presentazioni e documenti di elaborazione testi.

{{% /alert %}} 



Aspose.Cells fornisce un modo per rilevare il formato del file crittografato**Microsoft Apri XML** File. Per identificare il tipo di file, utilizzare l'estensione[FileFormatUtil.DetectFileFormat](https://reference.aspose.com/cells/net/aspose.cells/fileformatutil/methods/detectfileformat/index) metodo come mostrato di seguito nell'esempio di codice.
