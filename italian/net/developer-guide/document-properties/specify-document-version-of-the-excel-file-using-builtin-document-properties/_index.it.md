## **Possibili Scenari di Utilizzo**

Puoi modificare il **numero di versione** del file Excel facendo clic con il tasto destro del mouse sul file e selezionando Proprietà > Dettagli e modificando il campo **Numero di versione**. Si prega di utilizzare la proprietà [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion) per modificarla programmaticamente utilizzando le API di Aspose.Cells.
