## **Possibili scenari di utilizzo**

 Puoi cambiare il**Numero della versione** del file Excel facendo clic con il pulsante destro del mouse sul file e quindi selezionando Proprietà > Dettagli e quindi modificando il file**Numero della versione** campo. Si prega di utilizzare[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion)property per modificarlo a livello di codice utilizzando le API Aspose.Cells.
