{{% alert color="primary" %}}

Microsoft Excel consente agli utenti di aggiungere commenti alle celle per aggiungere ulteriori informazioni ed evidenziare i dati. Gli sviluppatori potrebbero dover personalizzare il commento per specificare le impostazioni di allineamento e la direzione del testo. Aspose.Cells fornisce API per eseguire l'attività.

{{% /alert %}}

 Aspose.Cells fornisce a[**Shape.TextDirection**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textdirection)proprietà per impostare la direzione del testo per un commento.
