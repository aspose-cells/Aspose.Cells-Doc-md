{{% alert color="primary" %}}

Aspose.Cells consente di importare la mappatura XML all'interno del foglio di calcolo utilizzando il metodo [**Workbook.ImportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/importxml/index). Puoi importare la mappa XML utilizzando Microsoft Excel con i seguenti passaggi

- Seleziona la scheda **Sviluppo**
- Fai clic su **Importa** nella sezione XML e segui i passaggi richiesti.

Dovrai fornire i tuoi dati XML per completare l'importazione. Ecco un [esempio di dati XML](5115037.txt) che puoi utilizzare per i test.
