Il codice di esempio seguente esporta i dati XML di tutti i mapping XML dalla cartella di lavoro uno per uno. Si prega di controllare[file excel di esempio](5115497.xlsx) utilizzato in questo codice e il[dati XML esportati della prima mappa XML](5472487.xml).
