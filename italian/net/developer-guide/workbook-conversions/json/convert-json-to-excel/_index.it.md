## **Converti JSON in Excel Workbook**
Non c'è bisogno di chiedersi come convertire un file JSON in un file Excel, poiché la libreria Apose.Cells per .NET ha la migliore soluzione. L'API di Aspose.Cells fornisce supporto per la conversione del formato JSON in fogli di calcolo. È possibile utilizzare la classe [**JsonLoadOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonLoadOptions) per specificare impostazioni aggiuntive per l'importazione di JSON in Workbook.

L'esempio di codice seguente dimostra l'importazione di JSON in Excel Workbook. Si prega di consultare il codice per convertire [file di origine](sample.json) nel file xlsx generato dal codice per riferimento.
