## **Converti Excel in HTML con intestazioni**

Aspose.Cells offre la possibilità di esportare le intestazioni durante la conversione di Excel in HTML. Per questo, puoi impostare il file[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/exportheadings) proprietà a**VERO**. Il valore predefinito di[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/exportheadings) la proprietà è**falso**.
