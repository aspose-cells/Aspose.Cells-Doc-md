## **Converti Excel in HTML con intestazioni**

Aspose.Cells fornisce l'opzione di esportare le intestazioni durante la conversione di Excel in HTML. Per questo, è possibile impostare la proprietà [**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/exportheadings) su **true**. Il valore predefinito della proprietà [**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/exportheadings) è **false**.
