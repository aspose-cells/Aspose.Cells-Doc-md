{{% alert color="primary" %}} 

Aspose.Cells fornisce un'utile proprietà HtmlSaveOptions.PresentationPreference per gli sviluppatori che devono eseguire il rendering di un layout migliore durante il salvataggio di un file Excel Microsoft nel formato HTML o MHT. Il valore predefinito della proprietà è false. Si consiglia di impostare questa proprietà su true per ottenere una presentazione più attraente dei report di Excel.

{{% /alert %}}
