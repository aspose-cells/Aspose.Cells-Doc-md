## **Possibili Scenari di Utilizzo**

Quando salvi il file di Excel in HTML, i commenti non vengono esportati. Tuttavia, Aspose.Cells fornisce questa funzionalità utilizzando la proprietà [**HtmlSaveOptions.IsExportComments**](https://docs.aspose.com/cells/net/export-comments-while-saving-excel-file-to/). Se la imposti su **true**, allora l'HTML mostrerà anche i commenti presenti nel tuo file di Excel.
