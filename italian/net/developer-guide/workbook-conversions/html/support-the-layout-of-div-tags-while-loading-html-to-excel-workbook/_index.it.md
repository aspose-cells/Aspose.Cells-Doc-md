{{% alert color="primary" %}} 

Normalmente, il layout dei tag div viene ignorato durante il caricamento dell'HTML in un oggetto cartella di lavoro Excel. Tuttavia, se desideri che il layout dei tag div non venga ignorato, imposta il file[HTMLLoadOptions.SupportDivTag](https://reference.aspose.com/cells/net/aspose.cells/htmlloadoptions/properties/supportdivtag) proprietà a**VERO** . Il valore predefinito di questa proprietà è**falso**.
