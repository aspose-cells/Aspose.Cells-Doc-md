##  **Ottieni la convalida Cell nei file ODS**

Con Aspose.Cells puoi ottenere la convalida applicata a una cella nei file ODS. Per questo lo API mette a disposizione il[**OttieniValidazione**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) metodo del[**Cell**](https://reference.aspose.com/cells/net/aspose.cells/cell) classe.

Nell'esempio di codice seguente viene illustrato l'utilizzo di[**OttieniValidazione**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) metodo caricando il file[fonte ODS](101089354.ods)file e leggendo la validazione della cella A9.
