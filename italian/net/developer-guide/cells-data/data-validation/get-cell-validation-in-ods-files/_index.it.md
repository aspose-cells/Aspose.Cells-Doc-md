## **Ottieni la Convalida Cellulare nei File ODS**

Con Aspose.Cells, è possibile ottenere la convalida applicata a una cella nei file ODS. A questo scopo, l'API fornisce il metodo [**GetValidation**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) della classe [**Cell**](https://reference.aspose.com/cells/net/aspose.cells/cell).

Il seguente esempio di codice dimostra l'uso del metodo [**GetValidation**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) caricando il [file ODS di origine](101089354.ods) e leggendo la convalida della cella A9.
