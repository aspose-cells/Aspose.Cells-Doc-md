## **Possibili scenari di utilizzo**
Utilizzando le API Aspose.Cells, gli sviluppatori possono compilare un DataTable con le stringhe HTML corrispondenti (basate sui valori delle celle) durante l'esportazione dei dati del foglio di lavoro. Questo documento spiega come esportare le stringhe HTML per i dati delle celle formattate per riempire un DataTable usando le API Aspose.Cells.
