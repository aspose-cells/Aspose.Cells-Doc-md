##  **Possibili scenari di utilizzo**
Utilizzando le API Aspose.Cells, gli sviluppatori possono riempire un DataTable con le stringhe HTML corrispondenti (in base ai valori delle celle) durante l'esportazione dei dati del foglio di lavoro. Questo documento spiega come esportare le stringhe HTML per i dati delle celle formattate per riempire un DataTable utilizzando le API Aspose.Cells.
