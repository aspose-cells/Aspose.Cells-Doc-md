{{% alert color="primary" %}}

 A volte è necessario ignorare le colonne nascoste durante l'esportazione dei dati del foglio di lavoro in una tabella dati. Puoi ottenerlo utilizzando Aspose.Cells impostando il[**ExportTableOptions.PlotVisibleColumns**](https://reference.aspose.com/cells/net/aspose.cells/exporttableoptions/properties/plotvisiblecolumns) A**VERO**. Per impostazione predefinita, il suo valore è **false**, quindi è necessario impostarlo **true** per ignorare le colonne nascoste.
