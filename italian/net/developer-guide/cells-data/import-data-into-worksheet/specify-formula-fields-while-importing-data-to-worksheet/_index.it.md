##  **Possibili scenari di utilizzo**

Puoi specificare i campi formula quando importi i dati nel foglio di lavoro utilizzando il file[**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas) . Questa proprietà accetta l'array booleano in cui si trova il valore**VERO**significa che il campo è un campo formula. Ad esempio, se il terzo campo è un campo formula, il terzo valore nell'array sarà *true**.
