## **Possibili Scenari di Utilizzo**

In generale, si specifica la directory dei font o l'elenco dei font per tutti i workbook ma a volte è necessario specificare un insieme individuale o privato di font per i propri workbook. Aspose.Cells fornisce la classe [**IndividualFontConfigs**](https://reference.aspose.com/cells/net/aspose.cells/individualfontconfigs) che può essere utilizzata per specificare l'insieme individuale o privato di font per il proprio workbook.
