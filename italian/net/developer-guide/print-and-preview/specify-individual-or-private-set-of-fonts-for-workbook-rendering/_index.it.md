## **Possibili scenari di utilizzo**

In genere, si specifica la directory dei caratteri o l'elenco dei caratteri per tutte le cartelle di lavoro, ma a volte è necessario specificare un set di caratteri individuale o privato per le cartelle di lavoro. Aspose.Cells fornisce la classe ****[IndividualFontConfigs](https://reference.aspose.com/cells/net/aspose.cells/individualfontconfigs)** che può essere utilizzata per specificare il set di caratteri individuale o privato per la cartella di lavoro.
