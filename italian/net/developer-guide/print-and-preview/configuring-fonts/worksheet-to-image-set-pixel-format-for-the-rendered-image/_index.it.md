{{% alert color="primary" %}} 

A volte si desidera specificare il formato pixel durante il rendering di un foglio di lavoro in formato immagine. Per impostazione predefinita, Aspose.Cells utilizza 32 bit per pixel. Aspose.Cells consente di personalizzare il formato dei pixel (profondità di bit) utilizzando le opzioni per l'immagine renderizzata.
