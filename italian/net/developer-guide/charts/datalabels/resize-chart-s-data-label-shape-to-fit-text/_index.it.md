 L'applicazione Excel fornisce il file**Ridimensiona la forma per adattarla al testo** opzione per le DataLabel del grafico per aumentare la dimensione della forma in modo che il testo si adatti al suo interno.

{{% /alert %}}

##  **Come ridimensionare la forma dell'etichetta dati del grafico per adattarla al testo in Microsoft Excel**

È possibile accedere a questa opzione dall'interfaccia di Excel selezionando una qualsiasi delle etichette dati sul grafico. Fare clic con il tasto destro e selezionare**Formatta etichette dati** menù. SU**Dimensioni e proprietà** scheda, espandere**Allineamento** per rivelare le proprietà correlate, incluso il**Ridimensiona la forma per correggere il testo** opzione.

##  **Come ridimensionare la forma dell'etichetta dati del grafico per adattarla al testo utilizzando Aspose.Cells for .NET**

Per imitare la funzionalità di Excel di ridimensionare le forme delle etichette dati per adattarle al testo, le API Aspose.Cells hanno esposto il tipo booleano[**DataLabels.IsResizeShapeToFitText**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/properties/isresizeshapetofittext) proprietà.
