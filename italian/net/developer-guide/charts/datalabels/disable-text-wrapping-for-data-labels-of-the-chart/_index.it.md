{{% alert color="primary" %}}

Microsoft Excel 2013 consente agli utenti di avvolgere o scartare il testo all'interno delle etichette dati del grafico. Per impostazione predefinita, il testo all'interno delle etichette dati del grafico è nello stato a capo.

 Aspose.Cells fornisce a[**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) proprietà che è possibile impostare True o False rispettivamente per abilitare o disabilitare il ritorno a capo del testo delle etichette dati.
