{{% alert color="primary" %}}

 È possibile utilizzare Aspose.Cells per creare un'etichetta dati personalizzata RTF del punto del grafico. Aspose.Cells fornisce il[**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) metodo per restituire il[**Impostazione carattere**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) oggetto che può essere utilizzato per impostare le proprietà del carattere del testo come colore, grassetto, ecc.

{{% /alert %}}
