{{% alert color="primary" %}}

Puoi utilizzare Aspose.Cells per creare etichette dati personalizzate di testo ricco del punto del grafico. Aspose.Cells fornisce il metodo [**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) per restituire l'oggetto [**FontSetting**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) che può essere utilizzato per impostare le proprietà del carattere del testo come il colore, il corsivo, ecc.
