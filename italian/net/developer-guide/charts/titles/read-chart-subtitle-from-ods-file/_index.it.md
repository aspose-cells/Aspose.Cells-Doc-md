##  **Leggi i sottotitoli del grafico dal file ODS**

Aspose.Cells ti offre la possibilità di leggere i sottotitoli delle carte nei file ODS utilizzando il comando[**Grafico.Sottotitolo**](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/properties/subtitle) proprietà. Il codice di esempio seguente carica il file[file campione ODS](89620481.ods) e legge il sottotitolo del grafico utilizzando[**Grafico.Sottotitolo**](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/properties/subtitle) proprietà e la stampa nella finestra della console. Si prega di consultare l'output della console del codice fornito di seguito come riferimento.
