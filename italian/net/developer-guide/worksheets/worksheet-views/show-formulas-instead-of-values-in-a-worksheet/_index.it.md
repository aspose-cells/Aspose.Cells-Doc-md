{{% alert color="primary" %}}

 È possibile mostrare formule invece di valori calcolati in Microsoft Excel utilizzando il file**Mostra formule** opzione dal**Formule**nastro. Quando vengono visualizzate le formule, Microsoft Excel visualizza le formule nel foglio di lavoro. Puoi ottenere la stessa cosa usando Aspose.Cells.

{{% /alert %}}

 Aspose.Cells fornisce a[**Worksheet.ShowFormulas**](https://reference.aspose.com/cells/net/aspose.cells/worksheet/properties/showformulas) proprietà. Imposta questo su**VERO**per impostare Microsoft Excel per visualizzare le formule.
