È possibile impostare il commento dell'oggetto tabella o elenco all'interno del foglio di lavoro utilizzando la proprietà [**ListObject.Comment**](https://reference.aspose.com/cells/net/aspose.cells.tables/listobject/properties/comment). Il commento sarà visibile all'interno del file xl/tables/tableName.xml.
