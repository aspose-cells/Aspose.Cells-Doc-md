## **Disegna la Timeline durante la rappresentazione di Excel in PDF**
Se hai un file Excel a cui è stata applicata una linea temporale e desideri esportare l'Excel in PDF con le impostazioni della linea temporale, Aspose.Cells ora supporta questo per impostazione predefinita. Basta esportare il file Excel con la linea temporale in PDF, il PDF generato mostrerà la linea temporale applicata.
