 Aspose.Cells offre la possibilità di salvare la tabella pivot nei file ODS. Per questo, basta convertire una cartella di lavoro con una tabella pivot esistente o creare una nuova tabella pivot e salvare il file nel formato ODS. Assicurati di chiamare[**Tabella pivot.CalculateData()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata) prima di salvare per assicurarsi che la tabella pivot venga visualizzata nel file di output ODS.
