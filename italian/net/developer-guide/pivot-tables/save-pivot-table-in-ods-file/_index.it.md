Aspose.Cells fornisce la possibilità di salvare la tabella pivot nei file ODS. A tal fine, è sufficiente convertire un workbook con una tabella pivot esistente o creare una nuova tabella pivot e salvare il file nel formato ODS. Assicurati di chiamare [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata) prima di salvare per garantire che la tabella pivot sia resa nel file ODS in uscita.
