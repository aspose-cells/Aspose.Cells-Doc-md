{{% alert color="primary" %}}

 Microsoft Excel consente di modificare il layout della tabella pivot utilizzando*Strumenti tabella pivot > Progettazione > Layout report* comandi del menu. È possibile modificare il layout in questi tre moduli

- Mostra in forma compatta
- Mostra in forma di contorno
- Mostra in forma tabellare

 Aspose.Cells fornisce anche[**Tabella pivot.ShowInCompactForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showincompactform), [**Tabella pivot.ShowInOutlineForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showinoutlineform) e[**Tabella pivot.ShowInTabularForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showintabularform) metodi per modificare il layout della tabella pivot in queste tre forme.
