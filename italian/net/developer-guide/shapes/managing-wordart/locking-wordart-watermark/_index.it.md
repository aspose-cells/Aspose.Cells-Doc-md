 Aspose.Cells Le API consentono di aggiungere filigrane WordArt al foglio di lavoro in modo che la WordArt diventi un oggetto che può essere spostato e posizionato sul foglio di lavoro. È anche possibile bloccare l'oggetto WordArt per qualsiasi interazione come modifica, movimento e selezione. Questo articolo spiega l'utilizzo di[**Shape.SetLockedProperty**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/methods/setlockedproperty)metodo per bloccare alcuni aspetti della filigrana.

{{% /alert %}}

Aspose.Cells Le API consentono di bloccare alcuni aspetti della filigrana in modo che l'interazione dell'utente possa essere limitata o completamente bloccata.
