## **Possibili Scenari di Utilizzo**

Microsoft Excel consente di modificare l'etichetta di visualizzazione dell'oggetto Ole come mostrato nello screenshot seguente. È anche possibile accedere o modificare l'etichetta di visualizzazione dell'oggetto Ole utilizzando le API di Aspose.Cells con la proprietà [**OleObject.Label**](https://reference.aspose.com/cells/net/aspose.cells.drawing/oleobject/properties/label).

![todo:image_alt_text](access-and-modify-the-display-label-of-the-linked-ole-object_1.png)

## **Accesso e modifica dell'etichetta di visualizzazione dell'oggetto Ole collegato**

Si prega di vedere il seguente codice di esempio, carica il [file Excel di esempio](64716810.xlsx) che contiene l'oggetto Ole.
