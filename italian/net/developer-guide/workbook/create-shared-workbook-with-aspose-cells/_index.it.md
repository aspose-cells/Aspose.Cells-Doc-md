## **Possibili Scenari di Utilizzo**

Microsoft Excel consente di condividere il libro di lavoro come mostrato nella seguente schermata. Quando si condivide il libro di lavoro, più di un utente può modificare il libro di lavoro sulla rete. Aspose.Cells consente di creare un libro di lavoro condiviso con la proprietà [**Workbook.Settings.Shared**](https://reference.aspose.com/cells/net/aspose.cells/workbooksettings/properties/shared).

![todo:image_alt_text](create-shared-workbook-with-aspose-cells_1.png)
