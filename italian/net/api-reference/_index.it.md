{{% alert color="primary" %}} 

Aspose.Cells for .NET è una libreria di programmazione per fogli di calcolo che consente agli sviluppatori di software di manipolare e convertire file di fogli di calcolo dall'interno delle proprie applicazioni. Aspose.Cells for .NET supporta i formati di file di fogli di calcolo più diffusi (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) e consente di esportare file Excel in PDF, XPS, HTML, MHTML, testo normale, TIFF, JPG, PNG, BMP e SVG.

L'ultimo riferimento API è disponibile all'indirizzo[Aspose.Cells for .NET Riferimento API](https://reference.aspose.com/cells/net).
