{{% alert color="primary" %}} 

Aspose.Cells for .NET è una libreria di programmazione dei fogli di calcolo che consente ai programmatori di software di manipolare e convertire file di fogli di calcolo all'interno delle proprie applicazioni. Aspose.Cells for .NET supporta i popolari formati di file di fogli di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) e consente di esportare file Excel in PDF, XPS, HTML, MHTML, testo normale, TIFF, JPG, PNG, BMP e SVG.

Il riferimento API più recente si trova su [Riferimento API Aspose.Cells for .NET](https://reference.aspose.com/cells/net).
