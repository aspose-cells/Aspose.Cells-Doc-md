{{% alert color="primary" %}} 

Aspose.Cells for .NET è una libreria di programmazione di fogli di calcolo che consente agli sviluppatori di software di manipolare e convertire file di fogli di calcolo dall'interno delle proprie applicazioni. Aspose.Cells for .NET supporta i formati di file più diffusi del foglio di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) e consente di esportare file Excel in PDF, XPS, HTML, MHTML, Testo semplice, TIFF, JPG, PNG, 07611 13481 e SVG.

 L'ultimo riferimento API può essere trovato all'indirizzo[Aspose.Cells for .NET API Riferimento](https://reference.aspose.com/cells/net).
