**Benvenuti allo Aspose.Cells for .NET**

Aspose.Cells for .NET è una libreria di classi di programmazione che consente agli sviluppatori di software di manipolare ed elaborare file di fogli di calcolo all'interno delle proprie applicazioni.

In combinazione con API e controlli GUI, Aspose.Cells for .NET accelera la programmazione e la conversione di Microsoft Excel®.

Aspose.Cells for .NET supporta i popolari formati di file foglio di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) utilizzati ogni giorno dalla tua azienda. Consente inoltre di esportare file Excel in PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, JSON, testo normale e formati di immagine più diffusi tra cui TIFF, JPG, PNG, BMP and SVG.
