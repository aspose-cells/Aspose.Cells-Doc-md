**Benvenuti in Aspose.Cells for .NET**

Aspose.Cells for .NET è una libreria di classi di programmazione che consente ai software developer di manipolare e elaborare file di fogli di calcolo all'interno delle proprie applicazioni.

Combinato con API e controlli GUI, Aspose.Cells for .NET velocizza la programmazione e la conversione di Microsoft Excel®.

Aspose.Cells for .NET supporta i popolari formati file di fogli di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) utilizzati quotidianamente dalle aziende. Consente inoltre di esportare file Excel in PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, JSON，testo normale e formati di immagine popolari, inclusi TIFF, JPG, PNG, BMP e SVG.
