## **Introduzione**
Aspose .NET Database Data Exporter to Excel for Umbraco Module consente agli utenti di esportare i dati direttamente dalle tabelle del database locale o remoto, dalle viste e tramite query personalizzate in Microsoft Excel o Fogli di calcolo OpenOffice. Questo modulo dimostra la potente funzionalità di creazione di fogli elettronici fornita da Aspose.Cells. Questa versione iniziale del modulo è arricchita con le seguenti fantastiche funzionalità per rendere il processo di esportazione semplice e facile da usare
### **Caratteristiche del Modulo**
Questa versione iniziale dell'Add-on ha le seguenti funzionalità:

- Permette di connettersi al database del server MS SQL locale
- Permette di connettersi al database del server MS SQL remoto
Popola tutte le tabelle dal database connesso
Popola tutte le viste del database connesso
- Permette di scrivere una query personalizzata
- Adatta automaticamente le colonne alla lunghezza del contenuto.
- Permette di saltare stringhe più lunghe di 32k nelle celle di Excel (Opzioni di caricamento)
- Applica la formattazione del testo in grassetto come intestazione della colonna
- Consente di utilizzare come origine dati (Tabella, Viste, Query personalizzata)
- Esporta dati in documenti Microsoft Excel (.xls, .xlsx e .xlsb)
- Esporta dati in documento di testo delimitato da tabulazione (*.txt)
- Esporta dati in CSV (delimitato da virgole) (*.csv)
- Esporta dati in Foglio di calcolo OpenDocument (*.ods)
- Opzione per selezionare il formato di output desiderato prima dell'esportazione
- Il documento esportato viene automaticamente inviato al browser per il download

## **Requisiti di sistema e piattaforme supportate**
### **Requisiti di sistema**
Per configurare Aspose .NET Database Data Exporter to Excel per il modulo Umbraco è necessario soddisfare i seguenti requisiti:

- Umbraco 6.2.5 e versioni di Umbraco 6
- Umbraco con MS SQL Server
- Microsoft .Net Framework 4.0

**Nota:** Umbraco 7 e versioni successive non sono supportate in questa versione. Attendiamo con interesse i vostri feedback e aggiungeremo il supporto per Umbraco 7 nella prossima versione.
### **Piattaforme Supportate**
Il modulo è supportato su tutte le versioni di

- Umbraco 6.0 in esecuzione su ASP.NET 4.0

## **Installazione**
Una volta scaricato, seguire questi passaggi per installare questo pacchetto nel sito web Umbraco:

1. Accedere alla sezione **Sviluppatore** di Umbraco, per esempio `http://www.myblog.com/umbraco`
1. Dall'albero, espandere la cartella **Pacchetti**.
1. Da qui ci sono due modi per installare un pacchetto: selezionare **Installa pacchetto locale** o sfogliare il **Repository dei pacchetti Umbraco.**
1. Se si installa un **pacchetto locale**, non decomprimere il pacchetto ma caricare il file zip in Umbraco.
1. Seguire le istruzioni a schermo.

**Nota:** Potresti ricevere un errore di 'Superamento della lunghezza della richiesta massima' durante l'installazione. Puoi risolvere facilmente questo problema aggiornando il valore 'maxRequestLength' nel file web.config di Umbraco.
<httpRuntime requestValidationMode="2.0" enableVersionHeader="false" maxRequestLength="25000" />
## **Utilizzo**
Dopo aver installato Aspose .NET Database Data Exporter to Excel per il modulo Umbraco è davvero semplice iniziare a usarlo sul tuo sito web. Seguire questi semplici passaggi per iniziare

1. Assicurati di essere loggato nella sezione Sviluppatore di Umbraco, ad esempio `http://www.myblog.com/umbraco/`
1. Fai clic su **Impostazioni** nell'elenco delle sezioni in basso a sinistra dello schermo.
1. Espandi il nodo **Modelli** e seleziona il modello a cui desideri aggiungere, ad esempio Textpage.
1. Seleziona la posizione nel modello selezionato dove desideri che il pulsante di esportazione venga aggiunto. Di solito potresti volerlo aggiungere in alto a destra della pagina, o in basso alla pagina.
1. Fai clic su **Inserisci Macro** nella barra degli strumenti superiore.
1. Da **Scegli un macro** (Aspose .NET Database Data Exporter to Excel per Umbraco), seleziona il macro Aspose .NET Database Data Exporter to Excel per Umbraco installato di recente e fai clic su **OK**.

Si prega di controllare lo screenshot sottostante per i dettagli. 

![todo:image_alt_text](umbraco-database-data-exporter-to-excel_2)

Hai aggiunto con successo il modulo Aspose .NET Database Data Exporter to Excel alla tua pagina.

![todo:image_alt_text](umbraco-database-data-exporter-to-excel_1)

1. Inserire o utilizzare la stringa di connessione del server MS SQL pre-popolata
1. Selezionare il tipo di origine dati (Tabella, Vista, Query personalizzata)
1. Seleziona o inserisci l'origine dati (Tabella, Vista, Query personalizzata)
1. Seleziona il tipo di esportazione
1. Fare clic su Esporta dati
1. Il file desiderato verrà scaricato automaticamente.
