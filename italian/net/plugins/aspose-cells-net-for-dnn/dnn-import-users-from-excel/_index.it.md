## **Introduzione**
Il modulo di importazione utenti Aspose .NET da Excel consente agli sviluppatori di importare utenti in blocco da un file Excel. Questo modulo dimostra la potente funzionalità del foglio di calcolo di Aspose.Cells.
### **Caratteristiche del Modulo**
Questa versione iniziale del modulo è arricchita con le seguenti fantastiche funzionalità per rendere il processo di importazione semplice e facile da utilizzare:

- Consente di caricare un file Excel e filtra automaticamente gli utenti esistenti.
- Salta automaticamente gli utenti esistenti per evitare qualsiasi confusione.
- Genera password dinamiche per gli utenti appena creati.
- Assegna ruoli di sicurezza "Utenti registrati" e "Iscritti" agli utenti appena creati.

## **Installazione**
Una volta scaricato, segui questi passaggi per installare il modulo sul tuo sito DNN:

1. Accedi al tuo sito come Host o con un altro account di livello super-user.
2. Naviga nel menu **Host** e seleziona **Estensioni**.
1. Fare clic su **Installa Wizard Estensione**.
1. Come indicato, passare alla posizione del file zip scaricato, selezionarlo e fare clic su **Apri**.
1. Fare clic su **Avanti**, accettare la licenza, continuare con l'installazione. Quando finito, fare clic sul pulsante **Indietro**.

Si prega di controllare [questo video di installazione del modulo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) da DNN per ulteriori dettagli.
## **Utilizzo**
Dopo aver installato il modulo Aspose .NET Cells Import Users from Excel per DNN, è davvero semplice iniziare a utilizzarlo sul proprio sito web. Seguire questi semplici passaggi per iniziare:

1. Assicurati di essere loggato a DNN con un account di livello Host o Admin.
1. Passare alla pagina in cui si desidera aggiungere il modulo di esportazione.
1. Selezionare **Moduli** seguito da **Aggiungi nuovo modulo** nella parte superiore del menu a schede. 

1. Dall'elenco, seleziona 'Aspose Import Users from Excel module for DNN' e trascinalo nella posizione desiderata.

Hai aggiunto con successo il modulo di importazione utenti Aspose .NET Cells da Excel per il modulo DNN alla tua pagina. I controlli di caricamento del file saranno visibili insieme al link 'File modello'. È possibile scaricare il file modello e formattare i propri contenuti secondo il file modello. Una volta caricato il file Excel, processerà automaticamente tutti gli utenti e filtra quelli esistenti. È possibile selezionare gli utenti che si desidera importare e premere il pulsante 'Importa utenti'.
