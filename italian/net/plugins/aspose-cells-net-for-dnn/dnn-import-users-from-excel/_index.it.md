## **introduzione**
Aspose .NET Importa utenti da Excel Il modulo consente agli sviluppatori di importare utenti in blocco da file Excel. Questo modulo dimostra la potente funzionalità del foglio di calcolo di Aspose.Cells.
### **Caratteristiche del modulo**
Questa versione iniziale del modulo è arricchita con le seguenti fantastiche funzionalità per rendere il processo di importazione semplice e facile da usare:

- Consente di caricare file Excel e filtra automaticamente gli utenti esistenti.
- Salta automaticamente gli utenti esistenti per evitare qualsiasi confusione.
- Genera password dinamiche per gli utenti appena creati.
- Assegna i ruoli di sicurezza "Utenti registrati" e "Iscritti" agli utenti appena creati.

## **Installazione**
Una volta scaricato, segui questi passaggi per installare il modulo nel tuo sito Web DNN:

1. Accedi al tuo sito come Host o con un altro account a livello di superutente.
1.  Naviga verso il**Ospite** menu e selezionare**Estensioni**.
1.  Clic**Installazione guidata estensione**.
1. Come indicato, sfoglia fino alla posizione del file zip scaricato, selezionalo e fai clic**Aprire**.
1.  Clic**Prossimo** , accettare la licenza, continuare con l'installazione. Al termine, fare clic su**Ritorno** pulsante.

 si prega di controllare[questo video di installazione del modulo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) da DNN per maggiori dettagli.
## **Usando**
Dopo aver installato il modulo Aspose .NET Cells Import Users from Excel for DNN, è davvero semplice iniziare a utilizzarlo sul tuo sito web. Segui questi semplici passaggi per iniziare:

1. Assicurati di aver effettuato l'accesso a DNN con un account a livello di host o amministratore.
1. Passare alla pagina in cui si desidera aggiungere il modulo Esporta.
1.  Selezionare**Moduli** seguito da**Aggiungi nuovo modulo** dal nastro superiore.

1. Dall'elenco, seleziona "Aspose Import Users from Excel module for DNN" e trascinalo in posizione.

Hai aggiunto correttamente Aspose .NET Cells Importa utenti dal modulo Excel per il modulo DNN alla tua pagina. I controlli per il caricamento dei file saranno visibili insieme al collegamento "File modello". Puoi scaricare il file modello e formattare i tuoi contenuti come da file modello. Una volta caricato il file Excel, elaborerà automaticamente tutti gli utenti e filtrerà gli utenti esistenti. È possibile selezionare gli utenti che si desidera importare e premere il pulsante "Importa utenti".
