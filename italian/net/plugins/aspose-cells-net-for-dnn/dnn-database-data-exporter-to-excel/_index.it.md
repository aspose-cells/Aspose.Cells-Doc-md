## **introduzione**

Aspose .NET DNN Data Exporter to Excel Module consente agli utenti di esportare i dati direttamente da tabelle di database locali o remoti, viste e tramite query personalizzate in Microsoft Excel o OpenOffice Spreadsheet. Questo modulo dimostra la potente funzionalità di creazione di fogli di calcolo fornita da Aspose.Cells.

### **Caratteristiche del modulo**

Questa versione iniziale del modulo è arricchita con le seguenti interessanti funzionalità per rendere il processo di esportazione semplice e facile da usare

- Consenti la connessione al database locale di MS SQL Server
- Consenti la connessione al database MS SQL Server remoto
- Popola tutte le tabelle dal database connesso
- Popola tutte le viste dal database connesso
- Consenti di scrivere query personalizzate
- Adatta automaticamente le colonne alla lunghezza del contenuto.
- Consenti di saltare la stringa più di 32k nelle celle Excel (LoadOptions)
- Applicare la formattazione della colonna di intestazione come testo in grassetto
- Consenti utilizzo come origine dati (tabella, viste, query personalizzata)
- Esporta dati in documenti Excel Microsoft (.xls, .xlsx e .xlsb)
- Esporta dati in un documento di testo delimitato da tabulazioni (*.txt)
- Esporta dati in CSV (delimitato da virgola) (*.csv)
- Esporta dati in un foglio di calcolo OpenDocument (*.ods)
- Opzione per selezionare il formato di output desiderato prima dell'esportazione.
- Il documento esportato viene inviato automaticamente al browser per il download.

## **Installazione**

Una volta scaricato, segui questi passaggi per installare il modulo nel tuo sito Web DNN:

1. Accedi al tuo sito come Host o con un altro account a livello di superutente.
1.  Naviga verso il**Ospite** menu e selezionare**Estensioni**.
1.  Clic**Installazione guidata estensione**.
1. Come indicato, sfoglia fino alla posizione del file zip scaricato, selezionalo e fai clic**Aprire**.
1.  Clic**Prossimo** , accettare la licenza, continuare con l'installazione. Al termine, fare clic su**Ritorno** pulsante.

 si prega di controllare[questo video di installazione del modulo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) da DNN per maggiori dettagli.

## **Usando**

Dopo aver installato il modulo Aspose .NET Database Data Expoeter for DNN, è davvero semplice iniziare a utilizzarlo sul tuo sito web. Segui questi semplici passaggi per iniziare:

1. Assicurati di aver effettuato l'accesso a DNN con un account a livello di host o amministratore.
1. Passare alla pagina in cui si desidera aggiungere il modulo Esporta.
1.  Selezionare**Moduli** seguito da**Aggiungi nuovo modulo** dal nastro superiore.
1. Dall'elenco, seleziona "Aspose.NET DNN Data Exporter to Excel Using Aspose.Cells" e trascinalo in posizione.

![cose da fare:immagine_alt_testo](picture2.png)

Hai aggiunto correttamente il modulo Aspose .NET Database Data Exporter a Excel per DNN alla tua pagina.

![cose da fare:immagine_alt_testo](picture3.png)

1. Immettere o utilizzare la stringa di connessione MS SQL Server precompilata
1. Tipo di origine dati Seletec (tabella, vista, query personalizzata)
1. Seleziona o inserisci l'origine dati (tabella, vista, query personalizzata)
1. Seleziona Tipo di esportazione
1. Fare clic su Esporta dati
1. Il file desiderato verrà scaricato automaticamente.
