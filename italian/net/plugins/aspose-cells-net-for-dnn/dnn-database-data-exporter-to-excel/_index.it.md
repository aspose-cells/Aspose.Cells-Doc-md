## **Introduzione**

Aspose .NET DNN Data Exporter to Excel Module consente agli utenti di esportare dati direttamente da tabelle del database locale o remoto, viste e tramite query personalizzate in Microsoft Excel o OpenOffice Spreadsheet. Questo modulo dimostra la potente funzionalità di creazione di fogli di calcolo fornita da Aspose.Cells.

### **Caratteristiche del Modulo**

Questa versione iniziale del modulo è arricchita con le seguenti interessanti funzionalità per rendere il processo di esportazione semplice e facile da usare

- Permette di connettersi al database del server MS SQL locale
- Permette di connettersi al database del server MS SQL remoto
Popola tutte le tabelle dal database connesso
Popola tutte le viste del database connesso
- Permette di scrivere una query personalizzata
- Adatta automaticamente le colonne alla lunghezza del contenuto.
- Permette di saltare stringhe più lunghe di 32k nelle celle di Excel (Opzioni di caricamento)
- Applica la formattazione del testo in grassetto come intestazione della colonna
- Consente di utilizzare come origine dati (Tabella, Viste, Query personalizzata)
- Esporta dati in documenti Microsoft Excel (.xls, .xlsx e .xlsb)
- Esporta dati in documento di testo delimitato da tabulazione (*.txt)
- Esporta dati in CSV (delimitato da virgole) (*.csv)
- Esporta dati in Foglio di calcolo OpenDocument (*.ods)
- Opzione per selezionare il formato di output desiderato prima dell'esportazione
- Il documento esportato viene automaticamente inviato al browser per il download

## **Installazione**

Una volta scaricato, segui questi passaggi per installare il modulo sul tuo sito DNN:

1. Accedi al tuo sito come Host o con un altro account di livello super-user.
2. Naviga nel menu **Host** e seleziona **Estensioni**.
1. Fare clic su **Installa Wizard Estensione**.
1. Come indicato, passare alla posizione del file zip scaricato, selezionarlo e fare clic su **Apri**.
1. Fare clic su **Avanti**, accettare la licenza, continuare con l'installazione. Quando finito, fare clic sul pulsante **Indietro**.

Si prega di controllare [questo video di installazione del modulo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) da DNN per ulteriori dettagli.

## **Utilizzo**

Dopo aver installato il modulo Aspose .NET Database Data Expoeter per DNN, è davvero semplice iniziare a utilizzarlo sul tuo sito web. Segui questi semplici passaggi per iniziare:

1. Assicurati di essere loggato a DNN con un account di livello Host o Admin.
1. Passare alla pagina in cui si desidera aggiungere il modulo di esportazione.
1. Selezionare **Moduli** seguito da **Aggiungi nuovo modulo** nella parte superiore del menu a schede.
1. Dall'elenco, seleziona 'Esportatore di dati Aspose.NET DNN su Excel utilizzando Aspose.Cells' e trascinalo nella posizione desiderata.

![todo:image_alt_text](picture2.png)

Hai aggiunto con successo il modulo Aspose .NET Database Data Exporter to Excel for DNN alla tua pagina.

![todo:image_alt_text](picture3.png)

1. Inserire o utilizzare la stringa di connessione del server MS SQL pre-popolata
1. Selezionare il tipo di origine dati (Tabella, Vista, Query personalizzata)
1. Seleziona o inserisci l'origine dati (Tabella, Vista, Query personalizzata)
1. Seleziona il tipo di esportazione
1. Fare clic su Esporta dati
1. Il file desiderato verrà scaricato automaticamente.
