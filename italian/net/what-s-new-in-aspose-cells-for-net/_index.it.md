Questa pagina descrive le nuove e più interessanti funzionalità di Aspose.Cells introdotte nelle versioni recenti.

## Aspose.Cells for .NET 24.7
* Supporto funzione DROP nelle formule
* Migliorare le prestazioni del calcolo delle funzioni LET e MAKEARRAY con parametri complessi
* Supporto per il contorno dei caratteri stella nel grafico
* Supporto per rendere un oggetto decorativo
* Supporto per lo scambio di serie
* Migliorare la visualizzazione del grafico ad albero TreeMap
* Supporto per la conversione dell'immagine della cella 'sovrapposta' in 'nella cella'
* Supporto per caricare di default il font locale Office cloud
* Supporto per mantenere lo schema json durante l'importazione dei dati json
* Supporto per esportare in formato file AZW3
* Supporto per impostare automaticamente l'altezza dopo aver impostato il wrap per la cella in GridJS
* Supporto per la generazione json utilizzando il metodo stream writer per GridJS

## Aspose.Cells for .NET 24.6
* Miglioramenti per il rendering: regolazione della saturazione del colore e della temperatura del colore, duotoni, immagini Webp
* Miglioramenti per PivotTable: unione intervallo, "Mostra valori come" di PivotField
* Supporto Range.ToHtml()/ToImage() per la comodità dell'utente nella manipolazione dei dati dell'intervallo

## Aspose.Cells for .NET 24.5
* Supporto delle nuove funzioni: XMATCH, XOR
* Supporto per la trasposizione dei dati dell'intervallo
* Supporto per ottenere le proprietà delle impostazioni del filtro avanzato
* Supporto per rendere l'immagine inserita nella cella in pdf/immagine
* Supporto per copiare stili per riempimento automatico per GridJS
* Supporto impostazione colori grafico per GridJS

## Aspose.Cells for .NET 24.4
* Fornire implementazioni semplici del monitor di interruzione per la comodità dell'utente
* Supporto per incorporare allegati durante la conversione dell'oggetto OLE in pdf
* Supporto per rendere la forma automatica di ChartX in immagine
* Supporto per incorporare un'immagine nella cella (operazione "Inserisci nella cella" per l'immagine in ms excel)
* Miglioramenti per l'importazione del file Numbers: celle unite, righe/colonne nascoste, ...ecc.

## Aspose.Cells for .NET 24.3
* Supporto per l'importazione di file AZW3
* Supporto per salvare ods con standard odf 1.3
* Supporto per calcolare la funzione VALUTA
* Supporto filtri per GridDesktop: Top10, Data, Testo
* Aggiungere più formule nel menu di inserimento e suggerimenti per l'uso di GridJS
* Aggiungere la funzione di categorizzazione al menu delle formule per GridJS
* Impostare automaticamente la cella al formato data per GridJS quando il risultato calcolato è una data e un'ora

## Aspose.Cells for .NET 24.2
* Supporto alle nuove funzioni: TESTODOPO,TESTOPRIMA,SUDDIVISIONETESTO,A RIGA,A COLONNA
* Supporto per il rilevamento del formato oxps
* Miglioramento delle prestazioni per la classificazione di grandi set di dati
* Miglioramento per le operazioni sui filtri per GridDesktop: filtro numerico, ricerca per parole chiave, cancella filtro.

## Aspose.Cells for .NET 24.1

* Supporto NET8
* Supporto per l'importazione di dati csv con righe/colonne di intestazione specificate dall'utente e righe/colonne totali per foglio
* Supporto per l'esportazione html con stili personalizzati
