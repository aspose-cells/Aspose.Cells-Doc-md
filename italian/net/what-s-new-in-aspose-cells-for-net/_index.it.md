In questa pagina vengono descritte le novità più interessanti di Aspose.Cells introdotte nelle versioni recenti.

##  Aspose.Cells for .NET 24.3
* Supporto per importare file AZW3
* Supporto per salvare le quote con lo standard odf 1.3
* Supporto per calcolare la funzione VALUTAZIONE
* Filtri supportati per GridDesktop: Top10, Data, Testo
* Aggiungi più formule nel menu Inserisci e suggerimenti di aiuto per GridJS
* Aggiungi la funzionalità di categorizzazione al menu delle formule per GridJS
* Imposta automaticamente il formato della cella fino alla data per GridJS quando il risultato calcolato è data/ora

##  Aspose.Cells for .NET 24.2
* Supporta nuove funzioni: TEXTAFTER,TEXTBEFORE,TEXTSPLIT,TOROW,TOCOL
* Supporta il rilevamento del formato oxps
* Migliorare le prestazioni per l'ordinamento di set di dati di grandi dimensioni
* Miglioramenti per le operazioni di filtro per GridDesktop: filtro numerico, ricerca per parole chiave, cancella filtro.

##  Aspose.Cells for .NET 24.1

* Supporta NET8
* Supporto per importare dati CSV con righe/colonne di intestazione specificate dall'utente e righe/colonne totali per foglio
* Supporto per esportare HTML con stili personalizzati
* Supporta il riempimento automatico per le aree unite
* Supporta il riempimento automatico dei dati da destra a sinistra e dal basso verso l'alto

##  Aspose.Cells for .NET 23.12

* Supporto per la conformità PDF/A-2 e PDF/A-3 durante la conversione in PDF
* Supporta il calcolo del raggruppamento discreto della tabella pivot
* Supporto per esportare i dati delle celle in un elenco generico
* Supporto per importare riferimenti incrociati utilizzando tabelle collegate per marcatori intelligenti
* Supporta la funzione MAKEEARRAY per le formule
* Supporto per il rendering del testo della cella con angolo di rotazione per GridJS
* Supporta più opzioni per importare/esportare datatable per GridDesktop
* Supporta l'immagine di rendering con rotazione per GridDesktop
* Supporto per regolare l'ordine dei fogli di lavoro per GridDesktop
*  Opzione di supporto per mostrare/nascondere il messaggio di avviso della formula per GridDesktop

##  Aspose.Cells for .NET 23.11

* Supporto per convertire i fogli di calcolo nel file PDF versione 1.7 (ISO32000-1:2008)
* Supporto per inserire controlli del modulo per GridJS
* Supporto per mostrare il riempimento sfumato dello sfondo della cella per GridJS
* Supporto per copiare/eliminare/inserire/rinominare il foglio di lavoro per GridDesktop
* Supporta l'estensione automatica del numero massimo di righe/colonne durante l'operazione di incollaggio per GridDesktop
* Supporto SheetPrintingPreview per GridDesktop
*  Fornire .NET build principale di GridDesktop

##  Aspose.Cells for .NET 23.10

* Supporta l'operazione di incollaggio adattivo quando la dimensione dell'intervallo di destinazione è diversa dall'intervallo di origine
* Supporto per impostare collegamenti ipertestuali direttamente a un intervallo
* Supporto per copiare più fogli di lavoro contemporaneamente
* Supporto per verificare se un intervallo è vuoto per i dati delle celle

##  Aspose.Cells for .NET 23.9

* Supporto per importare/esportare file EPUB
* Supporto per filtrare le serie di grafici
* Supporta la localizzazione per GridDesktop
* Fornire GridWeb per la piattaforma .net7-windows

##  Aspose.Cells for .NET 23.8

* Supporto per sostituire/inserire testo con formattazione per richtext della cella
* Supporto per il rendering del controllo ActiveX e del controllo del modulo per il pulsante di selezione in GridJs
