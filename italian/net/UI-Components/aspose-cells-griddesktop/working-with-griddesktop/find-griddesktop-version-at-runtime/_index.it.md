## **Possibili scenari di utilizzo**
 A volte è importante trovare la versione di GridDesktop in fase di esecuzione. Ad esempio, vuoi assicurarti di utilizzare la versione corretta o stai eseguendo il debug della tua applicazione e vuoi conoscere la versione di GridDesktop attualmente in uso. GridDesktop fornisce il metodo Aspose.Cells.GridDesktop.GridDesktop.GetVersion() che restituisce la versione di GridDesktop in fase di esecuzione.
