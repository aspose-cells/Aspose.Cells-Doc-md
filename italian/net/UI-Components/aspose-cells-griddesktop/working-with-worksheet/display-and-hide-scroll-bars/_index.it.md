{{% alert color="primary" %}}

Le barre di scorrimento sono utili per navigare i contenuti dei fogli di calcolo che sono ampi e profondi, cioè che hanno molte righe e colonne. La maggior parte delle applicazioni supporta due tipi di barre di scorrimento:

- Barra di scorrimento verticale
- Barra di scorrimento orizzontale

Entrambe queste sono presenti in Microsoft Excel.

L'API GridDesktop di Aspose.Cells fornisce barre di scorrimento orizzontali e verticali per scorrere i contenuti di un foglio di lavoro. Con le API GridDesktop di Aspose.Cells, gli sviluppatori possono controllare la visibilità di entrambe queste barre di scorrimento.

{{% /alert %}}
