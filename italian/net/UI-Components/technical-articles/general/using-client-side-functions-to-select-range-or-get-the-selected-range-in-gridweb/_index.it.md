{{% alert color="primary" %}} 

È possibile utilizzare le seguenti funzioni lato client per selezionare un intervallo o per ottenere l'intervallo selezionato in un GridWeb utilizzando JavaScript.

- getSelectRange()
- setSelectRange()
- cancellaSelezioni()

getSelectRange() restituisce l'ultimo intervallo selezionato. setSelectRange() seleziona l'intervallo specificato. clearSelections() cancella tutta la selezione esclusa la cella attiva corrente.
