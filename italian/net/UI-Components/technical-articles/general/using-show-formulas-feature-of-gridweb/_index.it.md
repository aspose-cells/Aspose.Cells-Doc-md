{{% alert color="primary" %}} 

A volte è necessario trovare un modo semplice per le celle che contengono formule. È necessario avere un approccio per leggere rapidamente tutte le formule per verificare gli errori. Ciò può aiutarti a tracciare i dati utilizzati in una formula.

Aspose.Cells.GridWeb fornisce la funzionalità chiamata Mostra Formule in Microsoft Excel.

{{% /alert %}} 
## **Usare la funzionalità Mostra Formule**
Basta usare una scorciatoia "CTRL + ~" per visualizzare tutte le formule nel foglio di lavoro.

Ad esempio, carichiamo un file modello nella GridWeb che ha formule nella colonna C, ecco il codice che abbiamo usato per caricare il file.
