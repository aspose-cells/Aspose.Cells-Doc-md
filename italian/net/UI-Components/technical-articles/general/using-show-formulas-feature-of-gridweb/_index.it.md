{{% alert color="primary" %}} 

volte, è necessario trovare un modo semplice per le celle contenenti formule. Devi avere un approccio per leggere rapidamente tutte le formule per verificare la presenza di errori. Questo può aiutarti a tracciare i dati utilizzati in una formula.

Aspose.Cells.GridWeb fornisce la funzione che viene chiamata Mostra formule in Microsoft Excel.

{{% /alert %}} 
## **Utilizzo della funzione Mostra formule**
Tutto ciò che serve per utilizzare una scorciatoia "CTRL + ~" per visualizzare tutte le formule nel foglio di lavoro.

Ad esempio, carichiamo un file modello nel GridWeb che ha delle formule nella colonna C, ecco il codice che abbiamo semplicemente usato per caricare il file.
