{{% alert color="primary" %}} 

Aspose.Cells.GridWeb è un controllo web basato su GUI che può essere integrato nei moduli web ASP.NET come altri controlli web forniti dal Framework Microsoft .NET. La cosa più importante di un controllo web è fornire supporto multipiattaforma. Aspose.Cells.GridWeb fornisce supporto multipiattaforma.

{{% /alert %}} 
## **Confronto**
Aspose.Cells.GridWeb è costruito utilizzando il Framework Microsoft .NET quindi è pienamente supportato su Internet Explorer di Microsoft (IE). Tuttavia, su altri browser ha alcune limitazioni minori. Questo argomento fornisce un confronto dettagliato di quali funzionalità sono supportate su browser diversi.
