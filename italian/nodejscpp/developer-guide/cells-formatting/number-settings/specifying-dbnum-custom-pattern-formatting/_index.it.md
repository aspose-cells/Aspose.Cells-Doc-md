## **Possibili Scenari di Utilizzo**

Aspose.Cells for Node.js via C++ supporta il formato di pattern personalizzato *DBNum*. Ad esempio, se il valore della cella è 123 e si specifica il suo formato personalizzato come [DBNum2][$-804]General, verrà visualizzato come 壹佰贰拾叁. È possibile specificare il formato personalizzato della cella usando i metodi [**Cell.getStyle()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getStyle--) e [**Style.setCustom(string)**](https://reference.aspose.com/cells/nodejs-cpp/style/#setCustom-string-).
