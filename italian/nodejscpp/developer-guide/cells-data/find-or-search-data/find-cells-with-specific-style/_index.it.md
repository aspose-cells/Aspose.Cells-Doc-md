{{% alert color="primary" %}}

A volte, devi trovare celle con uno stile particolare applicato. Puoi usare Aspose.Cells for Node.js via C++ per trovare tutte le celle con uno stile comune. Aspose.Cells fornisce il metodo [**FindOptions.setStyle(Style)**](https://reference.aspose.com/cells/nodejs-cpp/findoptions/#setStyle-style-) che puoi usare per specificare lo stile da cercare nelle celle.

{{% /alert %}}

Il codice in questo esempio trova tutte le celle che hanno lo stesso stile di quella della cella A1. Dopo che il codice è stato eseguito, tutte le celle che hanno lo stesso stile di A1 contengono il testo "Trovato".
