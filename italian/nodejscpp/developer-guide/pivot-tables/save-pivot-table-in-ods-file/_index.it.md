Aspose.Cells for Node.js via C++ consente di salvare le tabelle pivot in file ODS. Assicurarsi di chiamare [**PivotTable.calculateData**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#calculatedata--) prima di salvare per garantire che la tabella pivot venga renderizzata nel file ODS di output.
