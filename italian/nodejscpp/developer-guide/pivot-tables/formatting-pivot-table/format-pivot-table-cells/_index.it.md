{{% alert color="primary" %}}

A volte, si desidera formattare le celle delle tabelle pivot. Per esempio, si desidera applicare un colore di sfondo alle celle pivot. Aspose.Cells for Node.js via C++ fornisce due metodi [**PivotTable.formatAll(style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#formatAll-style-) e [**PivotTable.format(row, column, style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#format-number-number-style-), che puoi usare a questo scopo.

[**PivotTable.formatAll(style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#formatAll-style-) applica lo stile all'intera tabella pivot mentre [**PivotTable.format(row, column, style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#format-number-number-style-) applica lo stile a una singola cella della tabella pivot.

{{% /alert %}}
