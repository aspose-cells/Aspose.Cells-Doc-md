## **Come ottenere la fonte dati della connessione esterna della tabella pivot**

Aspose.Cells for Node.js via C++ fornisce la possibilità di ottenere la fonte dati di connessione esterna della tabella pivot. A tal fine, l'API fornisce il metodo [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) della classe [**PivotTable**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/). Il metodo [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) restituisce l'oggetto [**ExternalConnection**](https://reference.aspose.com/cells/nodejs-cpp/externalconnection/). Il seguente frammento di codice dimostra come usare il metodo [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) per ottenere la fonte dati di connessione esterna della tabella pivot.
