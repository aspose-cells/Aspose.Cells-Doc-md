{{% alert color="primary" %}}

Puoi impostare diverse opzioni della tabella pivot usando Aspose.Cells for Node.js via C++. Una di queste opzioni è "Mostra per le celle vuote". Impostando questa opzione, tutte le celle vuote in una tabella pivot vengono visualizzate come una stringa specificata.

{{% /alert %}}

## **Impostazione dell'opzione della tabella pivot in Microsoft Excel**

Per trovare e impostare questa opzione in Microsoft Excel:

1. Seleziona una tabella pivot e fai clic con il pulsante destro del mouse.
1. Seleziona **Opzioni tabella pivot**.
1. Seleziona la scheda **Layout e Formato**.
1. Seleziona l'opzione **Per le celle vuote mostra** e specifica una stringa.

## **Impostare l'opzione della tabella pivot usando Aspose.Cells for Node.js via C++**

Aspose.Cells for Node.js via C++ fornisce le proprietà [**PivotTable.setDisplayNullString**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#setDisplayNullString-boolean-) e [**PivotTable.setNullString**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#setNullString-string-) per impostare l'opzione "Mostra per le celle vuote" nella tabella pivot.
