## **Come impostare un ordinamento personalizzato nella tabella pivot utilizzando la libreria Aspose.Cells for Node.js via C++**
Utilizzando l'API Aspose.Cells for Node.js via C++, puoi ordinare le tabelle pivot sui valori dei campi. Il seguente frammento di codice carica il file excel di esempio e aggiunge tre tabelle pivot.
