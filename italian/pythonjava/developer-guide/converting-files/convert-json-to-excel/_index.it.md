Aspose.Cells for Python via Java supporta la conversione di un file Json (JavaScript Object Notation) in un foglio di lavoro Excel.

{{% /alert %}}

## **Converti JSON in Excel Workbook**
Non c'è bisogno di chiedersi come convertire JSON in un file Excel, perché la libreria Aspose.Cells for Python via Java offre la migliore soluzione. L'API Aspose.Cells for Python via Java fornisce supporto per la conversione del formato JSON in fogli elettronici. Puoi utilizzare la classe [**JsonLoadOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonLoadOptions) per specificare impostazioni aggiuntive per l'importazione di JSON nel foglio di lavoro.

L'esempio di codice seguente mostra come importare JSON in un foglio di lavoro Excel. Si prega di consultare il codice per convertire [il file di origine](sample.json) nel file generato dal codice a titolo di riferimento.
