Aspose.Cells for Python via Java supporta la conversione di un file Json (JavaScript Object Notation) in una cartella di lavoro di Excel.

{{% /alert %}}

## **Converti JSON in cartella di lavoro di Excel**
 Non c'è bisogno di chiedersi come convertire JSON in file Excel, perché la libreria Aspose.Cells for Python via Java ha la decisione migliore. Il Aspose.Cells for Python via Java API fornisce supporto per la conversione del formato JSON in fogli di calcolo. Puoi usare[**JsonLoadOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonLoadOptions) class per specificare impostazioni aggiuntive per l'importazione di JSON nella cartella di lavoro.
