## **Converti Excel in immagine**
Aspose.Cells per Python tramite Java supporta la conversione di file Excel in diverse immagini di moduli immagine. Per questo, lo API fornisce il[FoglioRendering](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)e[ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions) classi. Questa classe rappresenta il foglio di lavoro di cui verrà eseguito il rendering in un'immagine. Il[FoglioRendering](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)la classe fornisce il[immaginare()](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage\(int,%20java.lang.String\)) metodo per convertire un foglio di lavoro in un file immagine. Sono supportati i formati BMP, PNG, JPEG, TIFF e EMF.
