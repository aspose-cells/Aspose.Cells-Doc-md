## **Convertire Excel in Immagine**
Aspose.Cells per Python via Java supporta la conversione di file Excel in diverse forme di immagini. A tal scopo, l'API fornisce le classi [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender) e [ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions). Questa classe rappresenta il foglio di lavoro che verr√† convertito in un'immagine. La classe [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender) fornisce il metodo [toImage()](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage\(int,%20java.lang.String\)) per convertire un foglio di lavoro in un file immagine. Sono supportati i formati BMP, PNG, JPEG, TIFF e EMF.
