Aspose.Cells for Python via Java supporta la conversione di file Excel (.xls, xlsx, .xlsb,xlsm ), CSV e OpenOffice( .ods) in file Word.

{{% /alert %}}

## **Converti la cartella di lavoro di Excel in Word**

Non c'è bisogno di chiedersi come convertire la cartella di lavoro di Excel in Word, perché la libreria Aspose.Cells for Python via Java ha la decisione migliore. L'API Aspose.Cells for Python via Java fornisce supporto per la conversione di fogli di calcolo in formato Word. Per esportare la cartella di lavoro in Word, passare[**SalvaFormato.DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) come secondo parametro di[**Cartella di lavoro.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) metodo. Puoi anche usare[**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions) class per specificare impostazioni aggiuntive per l'esportazione del foglio di lavoro nel file .docx.
