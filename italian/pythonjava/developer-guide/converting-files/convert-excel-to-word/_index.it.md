Aspose.Cells per Python via Java supporta la conversione di file Excel (.xls, xlsx, .xlsb, xlsm), CSV e OpenOffice (.ods) in file Word.

{{% /alert %}}

## **Convertire un foglio di lavoro Excel in Word**

Non c'è bisogno di chiedersi come convertire un foglio di lavoro Excel in Word, perché la libreria Aspose.Cells for Python via Java offre la migliore soluzione. L'API Aspose.Cells for Python via Java fornisce supporto per la conversione di fogli elettronici nel formato Word. Per esportare il foglio di lavoro in Word, passare [**SaveFormat.DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) come secondo parametro del metodo [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)). È anche possibile utilizzare la classe [**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions) per specificare impostazioni aggiuntive per l'esportazione del foglio di lavoro in un file .docx.
