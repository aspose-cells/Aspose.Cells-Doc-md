Aspose.Cells per Python via Java supporta la conversione di un Workbook in un file Json (JavaScript Object Notation).

{{% /alert %}}

## **Converti Workbook Excel in JSON**

Non c'è bisogno di chiedersi come convertire il Workbook Excel in JSON, perché la libreria Aspose.Cells per Python via Java ha la migliore soluzione. L'API Aspose.Cells per Python via Java fornisce il supporto per la conversione dei fogli di calcolo nel formato JSON. Per esportare il workbook in JSON, passare [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) come secondo parametro del metodo [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)). È inoltre possibile utilizzare la classe [**JsonSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonSaveOptions) per specificare impostazioni aggiuntive per l'esportazione del foglio di lavoro in JSON.

L'esempio di codice seguente mostra l'esportazione del foglio di lavoro di Excel in JSON. Si prega di consultare il codice per convertire il [file di origine](sample.xlsx) nel file JSON generato dal codice per riferimento.
