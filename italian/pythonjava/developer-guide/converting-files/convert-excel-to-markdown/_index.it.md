## **Converti Excel in Markdown**
Aspose.Cells per Python via Java supporta la conversione dei file di Excel nel formato Markdown. Per esportare il foglio di lavoro attivo in Markdown, passare [SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN) come secondo parametro del metodo [Workbook.Save](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)).
