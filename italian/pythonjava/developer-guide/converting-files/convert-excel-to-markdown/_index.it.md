## **Converti Excel in Markdown**
Aspose.Cells for Python via Java supporta la conversione di file Excel in formato Markdown. Per esportare il foglio di lavoro attivo in Markdown, passare[SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN)come secondo parametro del[Cartella di lavoro.Salva](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)) metodo. Puoi anche usare il[MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions)class per specificare impostazioni aggiuntive per l'esportazione di fogli di lavoro in Markdown.
