## **Esportare l'intervallo dell'area di stampa in HTML**
Aspose.Cells per Python via Java supporta l'esportazione solo dell'area di stampa, cioè una gamma selezionata di celle anziché dell'intero foglio su HTML. A tal fine, l'API fornisce la proprietà [HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly). Impostando questa proprietà su **True** verrà esportata solo l'area di stampa.
