## **Esporta l'intervallo dell'area di stampa in HTML**
Aspose.Cells for Python via Java supporta l'esportazione in HTML solo dell'area di stampa, cioè l'intervallo di celle selezionato invece dell'intera shest. Per questo, l'API fornisce il[HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) proprietà. Impostando questa proprietà su**Vero** esporterà solo l'area di stampa.
