## **Converti Excel in HTML con intestazioni**
Aspose.Cells fornisce l'opzione di esportare le intestazioni di righe e colonne durante la conversione di Excel in HTML. Questo può essere realizzato utilizzando la proprietà [HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) fornita dall'API. Il valore predefinito di [HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) è **False**. Passare **True** come parametro per visualizzare le intestazioni nel file HTML di output. L'immagine seguente mostra il file di output generato dal codice seguente.
