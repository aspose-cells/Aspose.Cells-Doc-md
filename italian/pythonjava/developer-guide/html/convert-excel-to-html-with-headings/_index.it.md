## **Converti Excel in HTML con intestazioni**
Aspose.Cells offre la possibilità di esportare intestazioni di righe e colonne durante la conversione di Excel in HTML. Ciò può essere ottenuto utilizzando[HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings)proprietà fornita da API. Il valore predefinito di[HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) è**Falso** . Passaggio**Vero** come parametro per il rendering delle intestazioni nel file HTML di output. L'immagine seguente mostra il file di output generato dal codice seguente.
