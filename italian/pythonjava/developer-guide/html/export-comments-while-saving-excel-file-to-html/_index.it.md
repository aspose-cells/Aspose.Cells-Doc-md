## **Esporta commenti durante il salvataggio del file Excel in HTML**
Quando Excel viene convertito in HTML, i commenti non vengono esportati. Aspose.Cells for Python via Java fornisce la funzione per esportare i commenti durante la conversione da Excel a HTML. Per raggiungere questo obiettivo, lo API fornisce il[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments)proprietà. Impostazione del valore di[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments) proprietà a**Vero** esporterà i commenti nell'output HTML.
