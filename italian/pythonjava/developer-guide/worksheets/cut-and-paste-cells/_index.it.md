## **Taglia e incolla Cells**
Aspose.Cells per Python tramite Java offre la possibilità di tagliare e incollare celle. Per questo, lo API fornisce il[insertCutCells](https://reference.aspose.com/cells/python/asposecells.api/cells#insertCutCells\(com.aspose.cells.Range,%20int,%20int,%20int\)) metodo del[Cells](https://reference.aspose.com/cells/python/asposecells.api/Cells)collezione. Il[insertCutCells](https://reference.aspose.com/cells/python/asposecells.api/cells#insertCutCells\(com.aspose.cells.Range,%20int,%20int,%20int\)) accetta i seguenti parametri.

- [Gamma](https://reference.aspose.com/cells/python/asposecells.api/Range)l'intervallo di celle da tagliare.
- Indice riga: l'indice della riga in cui inserire le celle.
- Indice colonna: l'indice della colonna in cui inserire le celle.
- [Tipo di spostamento](https://reference.aspose.com/cells/python/asposecells.api/ShiftType): La direzione di spostamento delle colonne.
