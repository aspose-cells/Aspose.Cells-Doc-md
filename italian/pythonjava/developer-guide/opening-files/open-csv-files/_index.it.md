## **Apri i file CSV**
file CSV (Comma Separated Values) contengono record i cui valori sono delimitati o separati da virgole. È possibile utilizzare Microsoft Excel per salvare il foglio di calcolo in formato CSV.

Per aprire i file CSV, utilizzare il file[LoadOptions](https://reference.aspose.com/cells/python/asposecells.api/LoadOptions)classe e selezionare il[CSV](https://reference.aspose.com/cells/python/asposecells.api/fileformattype#CSV)valore, predefinito nel[FileFormatType](https://reference.aspose.com/cells/python/asposecells.api/FileFormatType)enumerazione.
