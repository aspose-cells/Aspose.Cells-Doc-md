{{% alert color="primary" %}} 

Aspose.Cells è stato creato per aiutare gli sviluppatori a creare e gestire i fogli di calcolo Excel Microsoft a livello di programmazione. Aspose.Cells non comprende solo fogli di calcolo in Microsoft Excel Sheet Format (XLS/XLSX) ma anche molti altri formati di file come Comma Separated Values (CSV), Tab Delimited/TSV (Text), SpreadsheetML ecc.

Aspose.Cells fornisce la maggior parte delle funzionalità disponibili in Microsoft Excel, ma la cosa interessante è che Aspose.Cells può elaborare i file Excel in modo dinamico, il che consente agli sviluppatori di elaborare migliaia o milioni di fogli di calcolo molto rapidamente.

Questo articolo illustra scenari comuni di utilizzo di Aspose.Cells.

{{% /alert %}} 
## **Possibili scenari di utilizzo**
Spetta agli sviluppatori come utilizzare Aspose.Cells, ma ci sono pochi scenari comuni in cui Aspose.Cells può essere d'aiuto. L'uso più comune di Aspose.Cells è la distribuzione sul server Web o nei servizi Web per l'elaborazione lato server. Di seguito sono descritti alcuni dei possibili scenari di utilizzo di Aspose.Cells.
## **Gestione automatizzata dei fogli di calcolo**
1. Gli utenti caricano fogli di calcolo creati manualmente su un server.
1. Aspose.Cells viene utilizzato per aprire file e recuperare dati.
 Dopo che i dati sono stati recuperati:
 1. I dati vengono salvati in un database.
 1. Alcuni dati sono presentati nelle pagine web.
 1. Alcuni aspetti dei fogli di calcolo vengono modificati e salvati nuovamente in file Excel Microsoft.
## **Applicazione web**
1. Gli utenti inseriscono i dati nelle pagine web.
1. I dati vengono raccolti dal server web.
1. Aspose.Cells viene utilizzato per popolare i dati in file nuovi di zecca o file modello predefiniti.
1. Dopo che il file Excel è stato creato, può essere inviato all'utente come report ecc.

{{% alert color="primary" %}} 

Abbiamo discusso alcuni degli scenari più comuni di utilizzo di Aspose.Cells in un ambiente basato sul web. Ma Aspose.Cells è ugualmente utile anche nelle applicazioni basate su desktop.
