## **Impostare il commento della tabella o dell'oggetto elenco all'interno del foglio di lavoro**
Aspose.Cells per Python tramite Java supporta l'aggiunta del commento di List Object. Per questo, lo API fornisce il[ListObject.Comment](https://reference.aspose.com/cells/python/asposecells.api/listobject#Comment)proprietà. Il commento aggiunto dal[ListObject.Comment](https://reference.aspose.com/cells/python/asposecells.api/listobject#Comment)proprietà sarà visibile all'interno del*xl/tabelle/nometabella.xml* file.
