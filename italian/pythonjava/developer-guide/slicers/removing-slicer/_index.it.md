## **Rimozione dell'affettatrice**
 Per rimuovere l'affettatrice in Microsoft Excel, è sufficiente selezionare l'affettatrice e premere il tasto*Elimina* pulsante. Per ottenere il salvataggio utilizzando Aspose.Cells for Python via Java, utilizzare il metodo Worksheet.getSlicers().remove(). Rimuoverà l'affettatrice dal foglio di lavoro.

Il seguente frammento di codice carica il file[esempio di file Excel](106364970.xlsx)che contiene un'affettatrice esistente. Accede all'affettatrice, lo rimuove e salva il file[file Excel di output](106364971.xlsx). Lo screenshot seguente mostra l'affettatrice che verrà rimossa.
