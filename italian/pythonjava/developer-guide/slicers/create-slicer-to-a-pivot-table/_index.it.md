## **Possibili scenari di utilizzo**
I filtri dei dati vengono utilizzati per filtrare rapidamente i dati. Possono essere utilizzati per filtrare i dati sia in una tabella che in una tabella pivot. Microsoft Excel consente di creare un'affettatrice selezionando una tabella o una tabella pivot e quindi facendo clic su*Inserisci > Affettatrice*Aspose.Cells for Python via Java fornisce il metodo Worksheet.getSlicers().add() per creare lo slicer.
