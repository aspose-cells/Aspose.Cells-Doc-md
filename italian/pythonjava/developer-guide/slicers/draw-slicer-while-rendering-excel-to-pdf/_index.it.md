## **Disegna lo slicer durante il rendering di Excel in PDF**
Aspose.Cells for Python via Java supporta l'esportazione di Excel in PDF con impostazioni del filtro. L'API supporta questo di default, è sufficiente esportare il file Excel con filtro in PDF, il PDF generato mostrerà il filtro applicato. Il seguente frammento di codice carica il [file Excel di esempio](106365058.xlsx) che contiene un filtro e salva il workbook come [file PDF di output](106365052.xlsx). La schermata seguente confronta il file Excel di origine e il file PDF generato.
