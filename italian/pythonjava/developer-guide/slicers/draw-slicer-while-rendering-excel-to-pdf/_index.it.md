## **Disegna Slicer durante il rendering di Excel a PDF**
Aspose.Cells for Python via Java supporta l'esportazione di Excel in PDF con impostazioni slicer. Lo API lo supporta per impostazione predefinita, è sufficiente esportare il file Excel con l'affettatrice in PDF, lo PDF generato mostrerà l'affettatrice applicata. Il seguente frammento di codice carica il file[esempio di file Excel](106365058.xlsx)che contiene un'affettatrice e salva la cartella di lavoro come file[output PDF file](106365052.xlsx).
