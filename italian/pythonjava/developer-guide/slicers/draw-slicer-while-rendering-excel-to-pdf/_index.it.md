## **Disegna Slicer durante il rendering di Excel in PDF**
Aspose.Cells per Python tramite Java supporta l'esportazione da Excel a PDF con impostazioni slicer. Lo API lo supporta per impostazione predefinita, è sufficiente esportare il file Excel con affettatrice in PDF, il PDF generato mostrerà l'affettatrice applicata. Il seguente frammento di codice carica il file[esempio di file Excel](106365058.xlsx)che contiene un'affettatrice e salva la cartella di lavoro come file[file di output PDF](106365052.xlsx).
