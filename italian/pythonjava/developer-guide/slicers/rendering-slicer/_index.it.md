## **Rendering del filtro**
Aspose.Cells for Python via Java supporta il rendering della forma filtro. Il seguente frammento di codice carica il [file Excel di esempio](106364974.xlsx) che contiene un filtro. Converte quindi il foglio di lavoro in un'immagine impostando l'area di stampa che copre solo il filtro. L'immagine risultante è la [immagine di output](106364975.png) che mostra il filtro renderizzato. Come puoi vedere, il filtro è stato correttamente renderizzato e appare uguale a come nel file Excel di esempio.
