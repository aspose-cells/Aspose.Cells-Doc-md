## **Affettatrice di rendering**
Aspose.Cells per Python tramite Java supporta il rendering della forma dell'affettatrice. Il seguente frammento di codice carica il file[esempio di file Excel](106364974.xlsx)che contiene un'affettatrice. Converte il foglio di lavoro in un'immagine impostando l'area di stampa che copre solo l'affettatrice. L'immagine che scorre è il[immagine in uscita](106364975.png)che mostra l'affettatrice renderizzata. Come puoi vedere, l'affettatrice è stata renderizzata correttamente e ha lo stesso aspetto del file Excel di esempio.
