## **Requisiti di sistema**
Aspose.Cells for Python via Java è un'API multi-piattaforma e può essere utilizzata su qualsiasi piattaforma (Windows, Linux e MacOS) dove è installato [Python](https://www.python.org/downloads/). La macchina deve avere installata Java 8 o versioni successive prima di impostare l'installazione.

## **Versione di Python**
- Python 3.5 o superiore
## **Versione di Java**
- Java 1.8 o superiore

## **Installazione**
### **Windows:**
- **Installare Java e impostare le variabili di ambiente JAVA_HOME e PATH**
Per esempio:
{{< highlight java >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}

- **Installare Aspose.Cells for Python via Java da pypi**
È possibile utilizzare facilmente Aspose.Cells for Python via Java da [pypi](https://pypi.org/project/aspose-cells/) con il seguente comando.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}

### **Linux:**
- **Installare Java**

- **Installare Aspose.Cells for Python via Java da pypi**
È possibile utilizzare facilmente Aspose.Cells for Python via Java da [pypi](https://pypi.org/project/aspose-cells/) con il seguente comando.

### **macOS:**

- **Installare Aspose.Cells for Python via Java da pypi**
