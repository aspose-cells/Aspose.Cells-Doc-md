### **API pubblica e modifiche non compatibili con le versioni precedenti**
Di seguito è riportato un elenco di tutte le modifiche apportate all'API pubblica come membri aggiunti, rinominati, rimossi o deprecati, nonché qualsiasi modifica non compatibile con le versioni precedenti apportata a Aspose.Cells for JasperReports. il forum di supporto Aspose.Cells.
#### **Supporta JasperServer V6.3.0**
Abbiamo rimosso quelle versioni che supportano le vecchie versioni di JasperReports e manteniamo solo quella che supporta JasperServer v6.3.0 nell'archivio delle versioni. Se gli utenti devono lavorare con versioni precedenti di JasperReports, scaricare le versioni precedenti di Aspose.Cells for JasperReports.
