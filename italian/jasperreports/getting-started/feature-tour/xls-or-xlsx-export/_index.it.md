{{% alert color="primary" %}}

 Aspose.Cells for JasperReports consente di consegnare report nel formato XLS e XLSX senza perdita di layout o formattazione. Per creare fogli di calcolo nei formati Excel Microsoft, Aspose.Cells for JasperReports si basa su una versione integrata di[Aspose.Cells for Java](https://products.aspose.com/cells/java/), una delle principali librerie di elaborazione di fogli di calcolo da Aspose.

Microsoft Excel non è necessario per generare documenti con Aspose.Cells.
