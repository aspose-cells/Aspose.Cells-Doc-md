{{% alert color="primary" %}}

Aspose.Cells for JasperReports ti consente di consegnare report nei formati XLS e XLSX senza perdita di layout o formattazione. Per creare fogli di calcolo nei formati Microsoft Excel, Aspose.Cells for JasperReports si basa su una versione integrata di [Aspose.Cells for Java](https://products.aspose.com/cells/java/), una nota libreria di elaborazione fogli di calcolo di Aspose.

Não è necessario Microsoft Excel per generare documenti con Aspose.Cells.
