{{% alert color="primary" %}} 

Alcune caratteristiche efficaci di Aspose.Cells for JasperReports sono elencate di seguito:

1. Aspose.Cells for JasperReports supporta grafici Microsoft Excel in modo che i grafici vengano esportati come grafici modificabili anziché immagini statiche. In questo modo, gli utenti possono modificare manualmente i grafici in Microsoft Excel se necessario.
2. Aspose.Cells for JasperReports è [facilmente integrato con JasperServer](/cells/it/jasperreports/integration-with-jasperserver/).

{{% /alert %}}
