{{% alert color="primary" %}}

Aspose.Cells for JasperReports è un prodotto per esportare report da JasperReports e JasperServer nel formato Microsoft Excel. Aspose.Cells for JasperReports eredita tutte le funzionalità dell'esportatore integrato e compensa alcune delle debolezze del renderer XLS originale di JasperReports. Questa sezione fornisce una descrizione dettagliata delle caratteristiche di Aspose.Cells for JasperReports. Fare clic sui rispettivi link nella tabella per saperne di più sulle caratteristiche.
