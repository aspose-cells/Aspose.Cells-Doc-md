{{% alert color="primary" %}}

Aspose.Cells for JasperReports è un prodotto per l'esportazione di report da JasperReports e JasperServer in formato Microsoft Excel. Aspose.Cells for JasperReports eredita tutte le funzionalità dell'esportatore integrato e compensa alcuni dei punti deboli del renderer XLS originale di JasperReports. Questa sezione fornisce una descrizione dettagliata delle funzionalità di Aspose.Cells for JasperReports. Fare clic sui rispettivi collegamenti nella tabella per ulteriori informazioni sulle funzionalità.

{{% /alert %}}

|**Caratteristica** |**Descrizione** |
|:- |:- |
|[Esportazione XLS o XLSX](/cells/it/jasperreports/xls-or-xlsx-export/)| Esporta i report in formato Microsoft Excel XLS dalla tua applicazione senza utilizzare Microsoft Excel.|
|[Elevata fedeltà al design del report](/cells/it/jasperreports/high-fidelity-to-the-report-design/)| Il layout e l'aspetto di un report XLS/XLSX corrispondono all'originale, simile a PDF o altri formati.|
