{{% alert color="primary" %}}

 Aspose.Cells for JasperReports è un esportatore esterno per[JasperReports](https://community.jaspersoft.com/project/jasperreports-library).

 Aspose.Cells for JasperReports viene fornito come un singolo file JAR e può essere facilmente installato su computer che eseguono JasperReports.[Scaricalo per iniziare una prova](https://downloads.aspose.com/cells/jasperreports).

[Per installare l'esportatore](/cells/it/jasperreports/installation/), **aspose.cells.jasperreports.jar** dovrebbe essere copiato in**JasperReports\lib** directory.

{{% /alert %}}
