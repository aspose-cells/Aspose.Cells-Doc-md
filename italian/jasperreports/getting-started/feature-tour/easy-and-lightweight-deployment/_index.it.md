{{% alert color="primary" %}}

Aspose.Cells for JasperReports è un esportatore esterno per [JasperReports](https://community.jaspersoft.com/project/jasperreports-library).

Aspose.Cells for JasperReports è fornito come un singolo file JAR e può essere facilmente installato su computer che eseguono JasperReports. [Scaricalo per avviare una prova](https://downloads.aspose.com/cells/jasperreports).

[Per installare l'esportatore](/cells/it/jasperreports/installation/), **aspose.cells.jasperreports.jar** dovrebbe essere copiato nella directory **JasperReports\lib**.

{{% /alert %}}
