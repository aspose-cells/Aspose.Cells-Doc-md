Per installare Aspose.Cells for JasperReports sul tuo sistema, il tuo sistema deve soddisfare i seguenti requisiti:

- Piattaforma Java 2 Standard Edition 1.6 o superiore.
- JasperReports 6.3.0 o superiore.
