Siamo lieti di annunciare la prima versione pubblica di Aspose.Cells per PHP tramite Java. Le API incorporano le funzionalità di elaborazione dei fogli di calcolo MS Excel in PHP (un linguaggio di scripting open source ampiamente utilizzato). Aspose.Cells per PHP tramite Java è un componente di sottoinsieme che include tutte le funzionalità importanti e utili presenti nella sua versione nativa[Aspose.Cells for Java](https://products.aspose.com/cells/java/). Il nuovo API è altrettanto robusto e ricco di funzionalità. Supporta conversioni di formati di file ad alta fedeltà da e verso XLS, XLSX, XLSM, SpreadsheetML, CSV, delimitato da tabulazioni, HTML, MHTML e OpenDocument Spreadsheet in PHP. Per favore visita[Pagina di confronto delle caratteristiche del prodotto](/cells/it/java/aspose-cells-for-php-via-java-features/)per tua referenza. Inoltre, si prega di consultare il documento su come[Ambiente di configurazione e altre linee guida per l'installazione](/cells/it/java/setup-and-installation-guidelines/)per tua referenza.
