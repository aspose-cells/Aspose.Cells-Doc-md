**![Aspose.Cells for .NET Logo del prodotto](cells-for-net.png)**

**Benvenuti in Aspose.Cells for .NET**

Aspose.Cells for .NET è una libreria di programmazione per fogli di calcolo che consente ai programmatori di software di creare, leggere, convertire, aggiornare e stampare file di fogli di calcolo all'interno delle proprie applicazioni. Una combinazione di API e controlli GUI, Aspose.Cells for .NET velocizza la programmazione e la conversione di Microsoft Excel®. Aspose.Cells for .NET supporta i formati di file di fogli di calcolo più popolari (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) usati quotidianamente dalle aziende. Consente anche di esportare file Excel in PDF, XPS, HTML, MHTML, testo semplice e formati di immagine popolari tra cui TIFF, JPG, PNG, BMP e SVG.
