**Benvenuti a Aspose.Cells for .NET**

Aspose.Cells for .NET è una libreria di programmazione per fogli di calcolo che consente agli sviluppatori di software di creare, leggere, convertire, aggiornare e stampare file di fogli di calcolo dall'interno delle proprie applicazioni. Una combinazione di API e controlli GUI, Aspose.Cells for .NET velocizza la programmazione e la conversione di Microsoft Excel®. Aspose.Cells for .NET supporta i popolari fogli di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) che la tua azienda utilizza ogni giorno. Consente inoltre di esportare file Excel in PDF, XPS, HTML, MHTML, testo normale e formati di immagini popolari tra cui TIFF, JPG, PNG, BMP e SVG.

**Benvenuti allo Aspose.Cells per C++**

Aspose.Cells per C++ è una libreria C++ nativa che consente agli sviluppatori C++ di creare, manipolare e convertire fogli di calcolo a livello di codice senza richiedere l'applicazione Office Automation o Microsoft Excel. Aspose.Cells per C++ supporta formati di fogli di calcolo come MS Excel 97-2003 (XLS), MS Excel 2007-2013/2016 (XLSX, XLSM, XLSB, ecc.), Open Office XML e altri formati. È inoltre possibile eseguire il rendering di immagini da fogli di lavoro e convertire fogli di calcolo MS Excel in formato PDF. Il API consente inoltre di creare, manipolare e rendere grafici e forme. Inoltre, i calcoli delle formule sono un altro aspetto positivo fornito dal componente e puoi utilizzarlo.

Poiché Aspose.Cells.Report.Designer utilizza Microsoft Excel come modello di report, gli utenti possono sfruttare le potenti funzionalità disponibili in Microsoft Excel.
