## **Cos'è un grafico azionario?**

I grafici azionari sono un grafico specifico utilizzato per monitorare le variazioni del prezzo di asset trattati come commodities, azioni e criptovalute. Permettono di vedere valori massimi e minimi nel tempo, insieme ai valori di apertura e chiusura in un unico grafico. Aspose.Cells for JavaScript tramite C++ offre quattro tipi di grafici azionari, e per utilizzarli, è necessario avere set di dati corretti disponibili, e scegliere le colonne nell'ordine giusto.

Il dataset seguente mostra le informazioni di trading giornaliere di un'azione. Useremo questi dati per creare quattro tipi di grafici di stock: grafico di stock High-Low-Close (HLC), grafico Open-High-Low-Close (OHLC), volume-high-low-close (VHLC) e volume-open-high-low-close (VOHLC).
