**Benvenuti nella sezione documentazione Aspose.Cells for C++**

Aspose.Cells for C++ è una libreria nativa C++ che consente ai programmatori C++ di creare, manipolare e convertire fogli di calcolo in modo programmatico senza richiedere l'automazione di Office o l'applicazione Microsoft Excel.

Aspose.Cells for C++ supporta formati di fogli di calcolo come il formato di file di Excel 97-2003 e i formati di file Open Office XML.

{{% /alert %}}

## **Vantaggi**

Utilizzare Aspose.Cells for C++ nelle tue applicazioni ti offre i seguenti vantaggi:

- Funzionalità avanzate
- Indipendenza dalla piattaforma
- Prestazioni e scalabilità

### **Funzionalità avanzate**

Aspose.Cells for C++ offre una vasta gamma di funzionalità per la creazione, conversione e manipolazione di fogli di calcolo:

- Proprietà incorporate
- Proprietà personalizzate
- Temi
- Stili e formattazione
- Convalida dei dati
- Formattazione condizionale
- Collegamento ipertestuale
- Filtro automatico
- Impostazioni di pagina
- Lettura, scrittura e calcolo delle formule
- Raggruppare righe e colonne
- Tabella pivot
- Tabella

### **Indipendenza dalla piattaforma**

Usa Aspose.Cells for C++ per creare qualsiasi tipo di applicazione C++ a 32 o 64 bit.

### **Prestazioni e scalabilità**

Aspose.Cells for C++ è progettato per funzionare bene sia sul server che sul lato client. Aspose.Cells for C++ è un'assemblaggio nativo che può essere distribuito semplicemente copiandolo. Non è necessario preoccuparsi di altri servizi o moduli.
