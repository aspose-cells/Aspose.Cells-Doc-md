## **Installazione di Aspose.Cells for C++ in Windows**
### **Utilizzo di Nuget Package Manager**
1. Nel Nuget Package Manager, cerca Aspose.Cells. 
   1. Qui c'è il [Link al Pacchetto NuGet Aspose.Cells.Cpp](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. Fai riferimento a [Installare Aspose Cells tramite NuGet] per i dettagli.(https://docs.aspose.com/cells/cpp/getting-started/).  

Puoi anche scaricare il pacchetto ZIP dalla pagina di [download di Aspose.Cells for C++](https://downloads.aspose.com/cells/cpp/). L'archivio contiene anche la documentazione API offline.
### **Utilizzo delle Cartelle Include e lib**
1. Scarica/Installa i [file più recenti](https://downloads.aspose.com/cells/cpp/).
1. Decomprimere la cartella nell'ambiente di produzione.
1. Per utilizzare Aspose.Cells for C++, fai riferimento alla directory "Include" e "lib / dll" nel tuo progetto.
