## **Installazione di Aspose.Cells for C++ in Windows**
### **Utilizzo di Nuget Package Manager**
1. In Gestione pacchetti Nuget cercare Aspose.Cells.
1. Ecco il[Collegamento al pacchetto NuGet Aspose.Cells.Cpp](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. Fare riferimento a [Installare Aspose Cells tramite NuGet] per i dettagli. (https://docs.aspose.com/cells/cpp/getting-started/).

Puoi anche scaricare il pacchetto ZIP dal Aspose.Cells for C++[pagina di download](https://downloads.aspose.com/cells/cpp/). L'archivio contiene anche la documentazione dell'API offline.
### **Utilizzo delle cartelle Include e lib**
1.  Scarica/Installa il file[file più recenti](https://downloads.aspose.com/cells/cpp/).
1. Decomprimere la cartella nell'ambiente di produzione.
1. Per utilizzare Aspose.Cells for C++, fare riferimento alla directory "Include" e "lib" nel progetto.

## **Installazione di Aspose.Cells for C++ su Linux**
1.  Scarica Aspose.Cells for C++ dalla seguente pagina:[Scarica Aspose.Cells for C++ (Linux)](https://downloads.aspose.com/cells/cpp)
2. Decomprimi il pacchetto e troverai una Demo che spiega come usare Aspose.Cells for C++ per Linux.
3. Esegui "cd Demo" nella riga di comando di Linux
4. Esegui "rm -rf build;mkdir build;cd build"
5. Esegui "cmake .." creerà un Makefile da CMakeLists.txt nella cartella Demo
6. Eseguire "make" per compilare
 7. Esegui "./demo" vedrai il risultato
