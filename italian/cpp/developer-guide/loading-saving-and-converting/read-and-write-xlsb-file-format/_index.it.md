##  **Possibili scenari di utilizzo**
Microsoft Excel supporta molti formati come XLS, XLSX, XLSM, XLSB, CSV, ecc. Anche Aspose.Cells supporta molti di questi formati. Questo articolo spiega come leggere e scrivere il file Excel avente formato XLSB utilizzando Aspose.Cells.

 Il codice di esempio seguente carica il file[file sorgente XLSB](23166992.xlsb) legge la sua cella A1, quindi ne copia il contenuto nella cella C4 e lo salva come[file di output XLSB](23166993.xlsb).
