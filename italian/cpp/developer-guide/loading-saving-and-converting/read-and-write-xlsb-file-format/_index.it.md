## **Possibili scenari di utilizzo**
Microsoft Excel supporta molti formati come XLS, XLSX, XLSM, XLSB, CSV, ecc. Aspose.Cells supporta anche molti di questi formati. Questo articolo spiega come leggere e scrivere il file excel con formato XLSB usando Aspose.Cells.
## **Leggere e scrivere in formato file XLSB**
 Il codice di esempio seguente carica il file[file XLSB di origine](23166992.xlsb) e legge la sua cella A1 e quindi copia il suo contenuto nella cella C4 e lo salva come[file XLSB di output](23166993.xlsb).
