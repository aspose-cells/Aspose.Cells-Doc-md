##  **introduzione**
 Si prega di utilizzare[PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) metodo per creare una tabella pivot all'interno del foglio di lavoro. Una volta creata la tabella pivot, puoi lavorarci utilizzando il file[Tabella pivot](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/)classe.
