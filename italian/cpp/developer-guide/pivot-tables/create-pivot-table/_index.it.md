## **introduzione**
 Si prega di utilizzare[IPivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table_collection#a903a0eb3d7ef995c370f4a4385b25111) metodo per creare una tabella pivot all'interno del foglio di lavoro. Una volta creata la tabella pivot, puoi lavorarci usando il file[IPivotTable](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table)classe.
