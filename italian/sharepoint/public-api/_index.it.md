I clienti che sviluppano le proprie soluzioni SharePoint possono accedere al nostro motore di conversione utilizzando l'API pubblica. Questo include tutti i formati file supportati e le opzioni di conversione. Le specifiche dell'interfaccia dettagliata nel nostro [Riferimento API](https://reference.aspose.com/) sono accompagnate da diversi esempi in C# e VB.

{{% /alert %}}
