### **Requisiti del server**
Per installare Aspose.Cells for SharePoint è necessario un computer che esegua uno dei seguenti sistemi operativi:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft SharePoint Server 2010
- Microsoft SharePoint Server 2013

{{% alert color="primary" %}} 

Raccomandiamo di installare il Service Pack 2 o successivo per questi prodotti.

{{% /alert %}}
### **Requisiti del Client**
È necessario un browser per accedere ai siti SharePoint. Aspose.Cells for SharePoint è testato con:

- Internet Explorer di Microsoft
- Mozilla FireFox
- Google Chrome
- Safari
