### **Requisiti del server**
Per installare Aspose.Cells for SharePoint è necessario un computer che esegue uno dei seguenti:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft SharePoint Server 2010
- Microsoft SharePoint Server 2013

{{% alert color="primary" %}} 

Si consiglia di installare Service Pack 2 o successivo per questi prodotti.

{{% /alert %}}
### **Requisiti del cliente**
È necessario un browser per accedere ai siti di SharePoint. Aspose.Cells for SharePoint è testato con:

- Microsoft Internet Explorer
- Mozilla Firefox
- Google Cromo
- Safari
