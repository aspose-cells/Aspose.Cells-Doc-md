{{% alert color="primary" %}} 

[Durante l'installazione](/cells/it/sharepoint/installing-aspose-cells-for-sharepoint/), Aspose.Cells for SharePoint viene [attivato](/cells/it/sharepoint/deployment-and-activation/) per tutte le raccolte di siti selezionate. Dopo l'installazione, utilizzare il menu **Azioni sito** nel sito Web radice di una raccolta per attivare e disattivare Aspose.Cells for SharePoint.
