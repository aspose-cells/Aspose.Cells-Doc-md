[Durante l'installazione](/cells/it/sharepoint/installing-aspose-cells-for-sharepoint/) , Aspose.Cells per SharePoint è[attivato](/cells/it/sharepoint/deployment-and-activation/) per tutte le raccolte siti selezionate. Dopo l'installazione, utilizzare il**Azioni del sito** menu nel sito Web radice di una raccolta siti per attivare e disattivare Aspose.Cells per SharePoint.
