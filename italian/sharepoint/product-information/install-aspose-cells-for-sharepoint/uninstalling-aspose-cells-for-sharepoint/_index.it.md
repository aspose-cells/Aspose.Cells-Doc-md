{{% alert color="primary" %}} 

Per disinstallare Aspose.Cells for SharePoint, esegui il programma di installazione. Se Aspose.Cells for SharePoint è già installato, il programma di installazione suggerisce di rimuoverlo o ripararlo.

{{% /alert %}} 
### **Disinstallazione**
Durante la rimozione, il programma di installazione disattiva Aspose.Cells for SharePoint per tutte le raccolte siti e ritrae la soluzione dal server farm. 

**Disinstallazione di Aspose.Cells** 

![todo:image_alt_text](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

Si noti che l'esecuzione dell'opzione **Ripara** non riattiverà Aspose.Cells for SharePoint sulle raccolte siti. Invece, la funzionalità deve essere attivata manualmente dopo il ridispiegamento con l'opzione **Ripara**.

Disinstallare utilizzando l'opzione **Rimuovi** e successivamente reinstallare potrebbe essere più conveniente in quanto consente di selezionare applicazioni Web di destinazione e raccolte siti durante l'installazione. 

{{% /alert %}}
