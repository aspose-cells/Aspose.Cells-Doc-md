{{% alert color="primary" %}} 

Per rendere il processo di conversione il più flessibile possibile, Aspose.Cells for SharePoint consente di impostare delle opzioni che definiscono come il file convertito verrà trattato.

{{% /alert %}} 
### **Opzioni**
Le seguenti opzioni sono disponibili durante la conversione:
#### **Concatenare i File**
Seleziona **Combinare i File** per unire file che erano originariamente in formati diversi in un unico documento. 

Questa opzione è disponibile quando si seleziona una cartella o diversi file individuali per la conversione. Sposta i documenti nell'elenco su o giù per impostare l'ordine in cui desideri che i documenti siano combinati.

**Combinare i file**

#### **Convertire i File nelle Sottocartelle**
Seleziona **Convertire i file nelle sottocartelle** per convertire in modo ricorsivo tutti i documenti e le sottocartelle in una cartella selezionata. 

La struttura della sottocartella esistente viene creata nella cartella di destinazione.

{{% alert color="primary" %}} 

Questa opzione è disponibile solo in SharePoint 2007. In SharePoint 2010, le sottocartelle sono elaborate per impostazione predefinita. Per elaborare solo i file in una cartella radice in SharePoint 2010, seleziona i file individuali anziché la cartella e non selezionare alcuna sottocartella.

{{% /alert %}} 
#### **Sovrascrivi i File Esistenti nella Cartella di Destinazione**
Se la cartella di destinazione contiene già file con lo stesso nome di quelli che si desidera convertire, hai due opzioni. Seleziona per sovrascrivere i file esistenti per sostituire i file già esistenti con quelli appena convertiti. Deseleziona l'opzione per omettere il trattamento dei file con lo stesso nome.
