{{% alert color="primary" %}} 

Per rendere il processo di conversione il più flessibile possibile, Aspose.Cells per SharePoint consente di impostare opzioni che definiscono come verrà trattato il file convertito.

{{% /alert %}} 
### **Opzioni**
Durante la conversione sono disponibili le seguenti opzioni:
#### **Concatena file**
 Selezionare**Combina file** per combinare file originariamente in diversi formati in un unico documento.

Questa opzione è disponibile quando si seleziona una cartella o più singoli file per la conversione. Spostare i documenti nell'elenco in alto o in basso per impostare l'ordine in cui si desidera combinare i documenti.

#### **Converti file in sottocartelle**
 Selezionare**Converti file in sottocartelle** per convertire in modo ricorsivo tutti i documenti e le sottocartelle in una cartella selezionata.

La struttura di sottocartelle esistente viene creata nella cartella di destinazione.

{{% alert color="primary" %}} 

Questa opzione è disponibile solo in SharePoint 2007. In SharePoint 2010, le sottocartelle vengono elaborate per impostazione predefinita. Per elaborare solo i file in una cartella radice in SharePoint 2010, selezionare i singoli file anziché la cartella e non selezionare alcuna sottocartella.

{{% /alert %}} 
#### **Sovrascrivi file esistenti nella cartella di destinazione**
Se la cartella di destinazione contiene già file con lo stesso nome di quelli che vuoi convertire, hai due possibilità. Selezionare per sovrascrivere i file esistenti per sostituire i file già esistenti con quelli appena convertiti. Deselezionare l'opzione per ignorare l'elaborazione dei file con lo stesso nome.
