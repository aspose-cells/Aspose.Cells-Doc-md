{{% alert color="primary" %}}

Con Aspose.Cells per SharePoint puoi convertire documenti tra molti formati di documenti per ufficio popolari da una raccolta documenti di SharePoint. La maggior parte delle conversioni viene eseguita con alta fedeltà e precisione.

{{% /alert %}}

## **Formati di input supportati**

Aspose.Cells per SharePoint supporta i seguenti formati di input:

- XLS – Microsoft Excel 97~2003
- XLSX: Office Open XML
- XLSM, XLTX, XLTM: Office Open XML
- CSV - Documento del foglio di lavoro con valori separati da virgola
- ODS – OpenDocument
- XML – Microsoft Foglio di calcolo ExcelML

 Per generare documenti, Aspose.Cells per SharePoint si basa su una versione integrata di[Aspose.Cells for .NET](https://products.aspose.com/cells/net/), il componente di elaborazione di fogli di calcolo leader di mercato da Aspose. Microsoft Excel non viene utilizzato per generare documenti.

## **Formati di output supportati**

L'elenco completo dei formati di output supportati da Aspose.Cells per SharePoint è:

- PDF – Documento portatile Adobe
- XLS – Microsoft Excel 97 - 2003
- XLSX: Office Open XML
- XLSM: Office Open XML per macro abilitato
- TXT – Testo semplice
- XML – Microsoft Foglio di calcolo ExcelML
- ODS – OpenDocument
- CSV – Valori separati da virgola
- XLSB – Microsoft Excel binario 2007
- MHTML - HTML multiuso per l'estensione della posta Internet
- XPS – Specifica carta XML
- TIFF: formato file immagine con tag
- SVG: grafica vettoriale scalabile
- DIF – Formato di interscambio disegni

**Vengono elencati i formati di conversione supportati disponibili per un file**
