Aspose.Cells for SharePoint supporta ricerche nei documenti archiviati nei siti SharePoint. Aspose.Cells per SharePoint supporta i seguenti formati di input per anteprima documenti:

- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- XML - Microsoft Excel SpreadsheetML

Per cercare in un documento:

1. Seleziona un documento o una cartella in cui desideri cercare.
1. Fai clic con il pulsante destro e seleziona **Cerca nel documento con Aspose.Cells**. 
1. Imposta i criteri di ricerca. 
   I risultati della ricerca vengono visualizzati.
