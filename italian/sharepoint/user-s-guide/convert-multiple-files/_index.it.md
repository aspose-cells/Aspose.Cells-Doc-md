Per convertire tutti i file in una cartella: 

1. Seleziona **Converti con Aspose.Cells** dal ECB della cartella desiderata rimanendo nel genitore.

Per convertire più file: 

1. Seleziona diversi file nella libreria e seleziona **Converti con Aspose.Cells**. 

   **Converti tutti i file nella sottocartella** 

![todo:image_alt_text](convert-multiple-files_1.png)




1. Seleziona l'elemento file nell'elenco dei documenti e spostalo su o giù per cambiare l'ordine di conversione. 

   **Cambiare l'ordine di conversione** 

![todo:image_alt_text](convert-multiple-files_2.png)
### **Utilizzo del pulsante Server Ribbon in SharePoint 2010**
In SharePoint 2010, il pulsante Server Ribbon è disponibile invece del menu di azione personalizzato utilizzato con SharePoint 2007 / WSS 3.
