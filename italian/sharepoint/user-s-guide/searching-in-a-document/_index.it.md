Per cercare tutti i file in una cartella:

1. Seleziona **Cerca nel Documento con Aspose.Cells** nella barra dei comandi estesa (ECB) della cartella desiderata rimanendo nel livello superiore.

Per cercare diversi file contemporaneamente:

1. Seleziona più file.
1. Con i file ancora selezionati, seleziona **Cerca Documento con Aspose.Cells**.

**Ricerca documenti** 

![todo:image_alt_text](searching-in-a-document_1.png)

Le seguenti opzioni sono disponibili:

- **Opzioni di ricerca**: Focalizza la tua ricerca cercando nelle celle, nelle formule o trovando formule che contengono una specifica espressione.
- **Maiuscole e minuscole**: Quando l'opzione maiuscole e minuscole è selezionata, la ricerca corrisponde ai caratteri esattamente, inclusi lettere minuscole e maiuscole.
