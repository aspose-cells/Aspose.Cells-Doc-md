I siti SharePoint sono pensati per la collaborazione, la comunicazione e lo storage dei contenuti al fine di soddisfare particolari esigenze aziendali. Aspose.Cells for SharePoint è stato introdotto per eseguire determinate operazioni sui documenti di Microsoft Excel disponibili nelle librerie documenti di SharePoint. Questo articolo copre i seguenti argomenti:

### **Breve nota sulle librerie documenti di SharePoint**
I contenuti sui siti SharePoint sono sotto forma di documenti condivisi in una posizione centralizzata chiamata libreria documenti di SharePoint (talvolta nota come "libreria documenti condivisa"). Gli utenti autorizzati possono caricare documenti dal loro computer locale al sito SharePoint. Questi documenti vengono memorizzati in formato binario in un database SQL Server utilizzato dal sito SharePoint. I documenti diventano disponibili agli altri utenti di SharePoint sotto forma di elenco di collegamenti ipertestuali come mostrato di seguito:

**Una libreria documenti** 


### **Vantaggi**
L'elenco seguente mette in evidenza alcuni benefici nell'uso di Aspose.Cells for SharePoint:

- Il server del sito SharePoint non ha la capacità integrata di convertire direttamente i file di Microsoft Excel sul server. Aspose.Cells for SharePoint aggiunge questa funzionalità al server del sito SharePoint.
- Aspose.Cells for SharePoint non richiede Microsoft Excel per funzionare. È installato sul server in modo che gli utenti di SharePoint non abbiano bisogno di Microsoft Excel sui loro sistemi locali per convertire i documenti di Microsoft Excel da un formato all'altro.
- Aspose.Cells for SharePoint non richiede alcun prerequisito eccetto quelli richiesti per creare un sito SharePoint.
- Aspose.Cells for SharePoint è [facile da implementare su un server del sito SharePoint](/cells/it/sharepoint/installing-aspose-cells-for-sharepoint/).
- Aspose.Cells for SharePoint supporta [una serie di formati di input e output](/cells/it/sharepoint/multiple-format-support/).
- Aspose.Cells for SharePoint espone un'API pubblica per gli sviluppatori che hanno bisogno di sviluppare le proprie soluzioni SharePoint.
### **Conversione di documenti Microsoft Excel con Aspose.Cells for SharePoint**
Una volta installato Aspose.Cells for SharePoint sul server SharePoint, l'opzione **Converti con Aspose.Cells.SharePoint** è disponibile nel menu contestuale: 

**Trova l'opzione Converti con...** 

Per convertire un documento Microsoft Excel dalla libreria documenti di SharePoint in altri formati, seguire i passaggi di seguito:

1. Fare clic sul documento Microsoft Excel desiderato e quindi fare clic su Converti con Aspose.Cells for SharePoint
   Viene visualizzata la schermata Impostazioni conversione. 

   **Decidere di convertire** 


1. Seleziona il formato di output.
   Qui è anche possibile modificare il nome del file di output e la cartella di destinazione.
1. Fare clic su **Converti**. 

   **Selezione del formato di output** 



1. Quando la conversione è completata, viene visualizzato un messaggio di successo. 

   **Conversione riuscita** 



1. Fare clic su **Torna** per tornare alla libreria documenti.
1. Il documento convertito è elencato nella libreria documenti di SharePoint. 

   **Il file convertito è elencato nella libreria documenti** 


### **Sommario**
{{% alert color="primary" %}} 

Installare e utilizzare Aspose.Cells for SharePoint è facile. Non solo aggiunge una funzionalità per convertire i file Microsoft Excel direttamente sul server SharePoint, ma elimina anche la necessità di installare Microsoft Excel sulle macchine client. Inoltre, gli sviluppatori possono utilizzare l'API pubblica di Aspose.Cells for SharePoint per sviluppare le proprie soluzioni SharePoint.

In altre parole, è vantaggioso per gli amministratori del sito di SharePoint, sviluppatori e utenti finali.
