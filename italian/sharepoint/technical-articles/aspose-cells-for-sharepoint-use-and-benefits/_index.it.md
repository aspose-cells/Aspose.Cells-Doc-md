Aspose.Cells per SharePoint è stato introdotto per eseguire determinate operazioni sui documenti Excel Microsoft disponibili nelle raccolte documenti di SharePoint. ### **Una breve nota sulle librerie di documenti di SharePoint**
I contenuti dei siti di SharePoint sono sotto forma di documenti condivisi in una posizione centralizzata denominata raccolta documenti di SharePoint (a volte nota come "raccolta documenti condivisa"). Gli utenti autorizzati possono caricare documenti dal proprio computer locale al sito di SharePoint. Questi documenti vengono archiviati in formato binario in un database di SQL Server utilizzato dal sito di SharePoint. I documenti diventano disponibili per altri utenti di SharePoint sotto forma di un elenco di collegamenti ipertestuali come mostrato di seguito:

**Una biblioteca di documenti** 


### **Benefici**
L'elenco seguente evidenzia alcuni vantaggi dell'utilizzo di Aspose.Cells per SharePoint:

- Il server del sito di SharePoint non dispone della funzionalità incorporata per convertire i file Excel Microsoft direttamente sul server. Aspose.Cells per SharePoint aggiunge questa funzionalità al server del sito di SharePoint.
- Aspose.Cells per SharePoint non richiede Microsoft Excel per funzionare. È installato sul server in modo che gli utenti di SharePoint non abbiano bisogno di Microsoft Excel sui loro sistemi locali per convertire i documenti Microsoft Excel da un formato all'altro.
- Aspose.Cells per SharePoint non richiede alcun prerequisito se non quelli richiesti per la creazione di un sito SharePoint.
-  Aspose.Cells per SharePoint è[facile da distribuire su un server del sito di SharePoint](/cells/it/sharepoint/installing-aspose-cells-for-sharepoint/).
-  Aspose.Cells per supporti SharePoint[diversi formati di input e output](/cells/it/sharepoint/multiple-format-support/).
- Aspose.Cells per SharePoint espone un API pubblico per gli sviluppatori che devono sviluppare le proprie soluzioni SharePoint.
### **Conversione di documenti Excel Microsoft con Aspose.Cells per SharePoint**
 Dopo aver installato Aspose.Cells per SharePoint nel server SharePoint, l'opzione**Converti con Aspose.Cells.SharePoint** è disponibile nel menu contestuale:

**Trovare l'opzione Converti con...** 

Per convertire un documento Excel Microsoft dalla raccolta documenti di SharePoint in altri formati, attenersi alla seguente procedura:

1. Fare clic sul documento Excel Microsoft desiderato, quindi fare clic su Converti con Aspose.Cells per SharePoint
 Viene visualizzata la schermata Impostazioni di conversione.

   **Decidere di convertirsi** 


1. Seleziona il formato di output.
 Qui puoi anche modificare il nome del file di output e la cartella di destinazione.
1.  Clic**Convertire**. 

   **Selezione del formato di output** 



1.  Al termine della conversione, viene visualizzato un messaggio di successo.

   **Conversione riuscita** 



1.  Clic**Ritorno** per tornare alla libreria dei documenti.
1.  Il documento convertito viene elencato nella raccolta documenti di SharePoint.

   **Il file convertito viene elencato nella raccolta documenti** 


### **Riepilogo**
{{% alert color="primary" %}} 

Installare e utilizzare Aspose.Cells per SharePoint è facile. Non solo aggiunge una funzionalità per convertire i file Excel Microsoft direttamente sul server SharePoint, ma elimina anche la necessità di installare Excel Microsoft sui computer client. Inoltre, gli sviluppatori possono utilizzare Aspose.Cells per lo API pubblico di SharePoint per sviluppare le proprie soluzioni SharePoint.

 In altre parole, è vantaggioso per gli amministratori, gli sviluppatori e gli utenti finali dei siti di SharePoint.
