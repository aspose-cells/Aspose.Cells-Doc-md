I siti di SharePoint sono pensati per la collaborazione, la comunicazione e l'archiviazione di contenuti al fine di soddisfare particolari esigenze aziendali. Aspose.Cells for SharePoint è stato introdotto per eseguire determinate operazioni sui documenti Microsoft Excel disponibili nelle raccolte documenti di SharePoint. ### **Una breve nota sulle librerie di documenti di SharePoint**
contenuti dei siti di SharePoint sono sotto forma di documenti condivisi in una posizione centralizzata denominata raccolta documenti di SharePoint (a volte nota come "raccolta documenti condivisa"). Gli utenti autorizzati possono caricare documenti dal proprio computer locale al sito di SharePoint. Questi documenti vengono archiviati in formato binario in un database di SQL Server utilizzato dal sito di SharePoint. I documenti diventano disponibili per altri utenti di SharePoint sotto forma di un elenco di collegamenti ipertestuali come mostrato di seguito:

**Una biblioteca di documenti** 


### **Benefici**
Il seguente elenco evidenzia alcuni vantaggi dell'utilizzo di Aspose.Cells for SharePoint:

- Il server del sito di SharePoint non dispone della funzionalità incorporata per convertire i file di Microsoft Excel direttamente nel server. Aspose.Cells for SharePoint aggiunge questa funzionalità al server del sito di SharePoint.
- Aspose.Cells for SharePoint non richiede Microsoft Excel per funzionare. È installato sul server in modo che gli utenti di SharePoint non abbiano bisogno di Microsoft Excel sui loro sistemi locali per convertire i documenti di Microsoft Excel da un formato all'altro.
- Aspose.Cells for SharePoint non richiede alcun prerequisito se non quelli richiesti per la creazione di un sito SharePoint.
-  Aspose.Cells for SharePoint è[facile da distribuire su un server del sito di SharePoint](/cells/it/sharepoint/installing-aspose-cells-for-sharepoint/).
-  Aspose.Cells for SharePoint supporti[diversi formati di input e output](/cells/it/sharepoint/multiple-format-support/).
- Aspose.Cells for SharePoint espone un'API pubblica per gli sviluppatori che devono sviluppare le proprie soluzioni SharePoint.
### **Conversione di documenti Microsoft Excel con Aspose.Cells for SharePoint**
 Dopo aver installato Aspose.Cells for SharePoint sul server SharePoint, l'opzione**Converti con Aspose.Cells.SharePoint** è disponibile nel menu contestuale:

**Trovare l'opzione Converti con...** 

Per convertire un documento di Microsoft Excel dalla raccolta documenti di SharePoint in altri formati, attenersi alla seguente procedura:

1. Fare clic sul documento Microsoft Excel desiderato e quindi su Converti con Aspose.Cells for SharePoint
 Viene visualizzata la schermata Impostazioni di conversione.

   **Decidere di convertirsi** 


1. Seleziona il formato di output.
Qui puoi anche modificare il nome del file di output e la cartella di destinazione.
1.  Clic**Convertire**. 

   **Selezione del formato di output** 



1.  Al termine della conversione, viene visualizzato un messaggio di successo.

   **Conversione riuscita** 



1.  Clic**Ritorno** per tornare alla libreria dei documenti.
1.  Il documento convertito viene elencato nella raccolta documenti di SharePoint.

   **Il file convertito viene elencato nella raccolta documenti** 


### **Riepilogo**
{{% alert color="primary" %}} 

Installare e utilizzare Aspose.Cells for SharePoint è facile. Non solo aggiunge una funzionalità per convertire i file Microsoft Excel direttamente sul server SharePoint, ma elimina anche la necessità di installare Microsoft Excel sui computer client. Inoltre, gli sviluppatori possono utilizzare l'API pubblica di Aspose.Cells for SharePoint per sviluppare le proprie soluzioni SharePoint.

 In altre parole, è vantaggioso for SharePoint amministratori del sito, sviluppatori e utenti finali.
