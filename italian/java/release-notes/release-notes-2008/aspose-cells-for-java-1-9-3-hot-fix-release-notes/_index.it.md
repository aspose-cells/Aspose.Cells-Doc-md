 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 1.9.3 Correzione rapida](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-1.9.3-hot-fix/)

{{% /alert %}} 

Cari clienti, abbiamo rilasciato Aspose.Cells for Java V1.9.3!

 Cosa c'è di nuovo?

- Supporta l'aggiunta di arco, ovale, rettangolo
 Supporta l'aggiunta di file Word ed Excel come oggetto OLE
 Supporta solo il caricamento dei dati e della formattazione nel file modello
 Supporta la copia dello stile quando si inseriscono righe e colonne
 Supporta per ottenere il tipo di cella in formato numerico
 Supporta il filtraggio di celle non vuote/vuote nel filtro automatico

 Cosa è migliorato?

- Fornisce metodi per l'oggetto Shapes per aggiungere qualsiasi forma

 Cosa è stato risolto?

- Risolto un bug nel calcolo delle formule
 Risolto un bug per ottenere la colonna minima della riga
 Risolto un bug relativo all'importazione di una raccolta nelle celle
 Risolto un bug di lettura del file
 Risolto un bug di copia delle formule
 Risolto un bug di copia dell'intervallo di celle
 Risolto un bug di formattazione del numero con E-esponente

 Commenti

- Per aggiungere textBox, nella nuova versione, i metodi add(...) sono validi solo nel foglio di lavoro. Il metodo addInChart() è valido solo nel foglio grafico.
