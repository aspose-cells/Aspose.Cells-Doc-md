{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java V1.9.4 correzione rapida](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-v1.9.4-hot-fix/)

{{% /alert %}} 

 Cosa c'è di nuovo?

- Supporta l'aggiunta di rich text per forme come Arc, Oval, Rectangle, Comment, TextBox
 Supporta tutti i tipi di posizionamento per la forma
 Supporta la lettura di tutte le forme dal modello
 Supporta l'impostazione dell'array in SmartMarker

 Cosa è migliorato?

- Aggiunti nuovi metodi per manipolare la posizione e le dimensioni della forma e deprecati alcuni vecchi metodi
 Aggiungi alcuni codici demo e modelli per il nuovo API
 Sostituisci l'invocazione obsoleta API con la nuova API nelle demo

 Cosa è stato risolto?

- Risolti alcuni bug di calcolo della formula
 Risolto un bug nell'esportazione del formato Excel2007 XLSX
 Risolto un bug di copia dello stile
 Risolto un bug di copia della formula
 Risolto un bug nel salvataggio delle dimensioni dell'immagine

 Commenti

- Per i grafici importati da file modello, la manipolazione dei relativi dati non è supportata, attualmente supporta solo l'operazione sulle proprietà della forma, come posizione, posizionamento e dimensione
