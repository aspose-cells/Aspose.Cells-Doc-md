 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java V2.0.1 Aggiornamento rapido](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-v2.0.1-hotfix/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java 2.0.1!

 Cosa è cambiato:

 l Supporta R/W Open Document Spreadsheet file (ODS): Cell dati e stile, stile riga/colonna, unione Cells, proprietà foglio di lavoro/cartella di lavoro, proprietà documento, formula/nome.

 l Supporta l'impostazione di formule condivise.

 Supporta la manipolazione dei controlli: casella di controllo, pulsante, pulsante di opzione, casella combinata, casella di riepilogo, casella di gruppo, barra di scorrimento, spinner, etichetta.

 l Supporta l'impostazione della pagina Intestazioni e piè di pagina in xls2pdf.

 l Supporta la copia del grafico durante la copia di un foglio di lavoro/cartella di lavoro

 l Supporta la copia di oggetti correlati a Shape

 l Supporta il salvataggio del foglio di lavoro come html per lo streaming

 l Supporta il grafico pivot

 l Supporta l'ottenimento di dati immagine di Picture e OleObject

 l Supporta il tema colore R/W per il formato file 2007

 l Supporta il tipo di crittografia: Microsoft Forte provider di crittografia

 l 27 correzioni e miglioramenti.
