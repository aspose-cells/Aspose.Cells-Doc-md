 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java v2.1.0 (nuova versione)](https://releases.aspose.com/cells/java/new-releases/aspose.cells-for-java-v2.1.0-new-release/)

{{% /alert %}} 

Siamo lieti di annunciare Aspose.Cells for Java v2.1.0!

Cosa è cambiato:

l Aggiunte nuove funzionalità per il file Open Document Spreadsheet (ODS): collegamento ipertestuale, opzioni di impostazione della pagina, struttura del gruppo, formattazione condizionale, convalida dei dati, filtro automatico, interruzione di pagina, righe/colonne del titolo di stampa, oggetti correlati alle forme (commento, arco, pulsante, casella di controllo, casella combinata , GroupBox, GroupShape, Label, Line, ListBox, OleObject, Oval, Picture, RadioButton, Rectangle, ScrollBar, Spinner, TextBox), Chart.

l Fornire nuove funzionalità per il formato di file MS Excel 2007: inserimento di immagini, aggiunta di grafici.

l Supporto per l'aggiornamento dei controlli tramite le celle collegate e viceversa: CheckBox, ComboBox, ListBox, RadioButton, ScrollBar, Spinner.

l Accedere alle proprietà di Shape: Name, ZOrderPosition.

l Applicare totali parziali alle tabelle del foglio di lavoro.

l Funzione Calcola collegamento ipertestuale.

l Elimina nomi definiti dall'utente.

Includi gli attributi del filtro automatico durante la copia della cartella di lavoro/foglio di lavoro.

l Supporto per collegamenti ipertestuali nella conversione xls2pdf.

l Estrai Ole Objects nel file modello.

l Risolti alcuni bug per R/W ODS.

l 48 correzioni e altri miglioramenti.
