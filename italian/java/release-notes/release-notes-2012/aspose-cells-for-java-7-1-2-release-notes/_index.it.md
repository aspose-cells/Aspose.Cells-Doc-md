 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 7.1.2](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.2/)

{{% /alert %}} 

Noi siamo
 felice di annunciare Aspose.Cells for Java v7.1.2!

 Nuove caratteristiche

 – Supporta il metodo Cells.importCustomObjects() per l'importazione di POJO

 Miglioramenti

- Sono stati apportati miglioramenti ai metodi autoFitColumns()
- Prestazioni per la funzione Foglio-immagine con griglia attiva

 Eccezioni

- NullPointerException si è verificata durante la chiamata al metodo Cell.getDisplayStyle()

 Insetti

- Il valore massimo per il grafico a dispersione non può essere impostato correttamente
- Il testo nelle forme è stato perso o spostato al di fuori dell'intervallo delle forme nel file PDF generato
- Il nome della serie non è stato mostrato per le etichette della data nel grafico
