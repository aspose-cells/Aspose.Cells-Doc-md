{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 7.1.1](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.1/)

{{% /alert %}} 

Noi siamo
 felice di annunciare Aspose.Cells for Java v7.1.1!

 Nuove caratteristiche

- Controlla se una colonna è vuota

 Miglioramenti

- Salvataggio in XLSX con LightCells API eseguendo il rendering diretto in OutputStream

 Eccezioni

- Il metodo Cell.setHtmlString() genera NullPointerException
- La lettura di un file modello XLSX genera NullPointerException

 Insetti

- L'asse delle categorie secondarie del grafico a dispersione diventa l'asse del testo
- I contenuti nella casella di testo non sono stati visualizzati correttamente
- La funzione ROUND non può arrotondare un valore maggiore di 922337,20
- Il grafico non è stato letto correttamente dal file ODStemplate
- Formato numero personalizzato: "gg.MM.aaaa" non è stato salvato correttamente per il file ODS
- LineShape si accorcia nel documento PDF generato
