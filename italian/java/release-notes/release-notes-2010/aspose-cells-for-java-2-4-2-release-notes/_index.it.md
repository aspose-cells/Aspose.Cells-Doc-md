 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 2.4.2](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.4.2/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java 2.4.2!

 Cosa è cambiato:

-  Supporta i parametri dell'array (

Macro sconosciuta: {...}

) per impostare le formule.

- Fornisce supporto per aggiungere immagini collegate.
- Supporta il salvataggio dei dati delle celle direttamente nel flusso o nel file senza creare un modello di dati completo nella memoria.
- Controlla se la licenza è stata impostata, include il supporto per ottenere la data di scadenza del tuo abbonamento.
- Le operazioni di lettura/salvataggio dei file di Excel 2007 sono migliorate.
- La funzione da grafico a immagine è stata migliorata.
- La funzione diretta da Excel a PDF è stata migliorata.
- La funzione di formattazione dei valori Cell è stata migliorata.
- Le operazioni di lettura/salvataggio dei file Excel 97-2003 sono migliorate.
- 21 correzioni e miglioramenti.
