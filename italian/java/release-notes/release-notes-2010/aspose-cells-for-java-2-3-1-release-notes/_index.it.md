{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 2.3.1](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.3.1/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java 2.3.1!

 Cosa è cambiato:

- Fornisce il supporto per il tipo di crittografia AES per i file Excel 2007.
- Supporta la lettura/scrittura di tabelle pivot per file Excel 2007.
- Aggiunge il supporto del formato immagine svg/svgz per la funzione Grafico a immagine.
- Supporta più aree di stampa per le opzioni PageSetup.
- Imposta formati numerici personalizzati per PivotField.
- Include nuove proprietà del documento integrate per il file Excel 2007.
- Combina più cartelle di lavoro.
- Supporta il salvataggio di PDF in Stream per la funzione diretta da Excel a PDF.
- È stato apportato un miglioramento per la lettura/scrittura di file Excel 2007.
- È stato apportato un miglioramento per la lettura/scrittura di file Excel 2003.
- È stato apportato un miglioramento per la lettura dei file modello XLS.
- 25 correzioni e altri miglioramenti.
