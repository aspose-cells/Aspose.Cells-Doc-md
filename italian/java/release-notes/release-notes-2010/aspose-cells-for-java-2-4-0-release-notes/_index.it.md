 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 2.4.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.4.0/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java v2.4.0!

 Cosa è cambiato:

- Include le funzioni Da foglio a immagine, Foglio da stampare e Cartella di lavoro da stampare
- Supporta la formattazione condizionale per la funzione da Excel a Pdf
- Fornisce il supporto per il raggruppamento dei campi della tabella pivot nei file Excel2007
- Mantiene le macro durante la conversione di file Excel97-2003 in file Excel2007
- Rende le forme durante la conversione in file Pdf, ad esempio TextBox, Rectangle, Oval, Arc
- Supporta per ottenere/impostare il nome completo del file sorgente per l'oggetto Ole
- È stato apportato un miglioramento per la lettura/salvataggio della formattazione condizionale
- È stato apportato un miglioramento per la funzione Grafico a immagine
- Il miglioramento è stato apportato per la funzione diretta da Excel a Pdf
- Il miglioramento è fatto per la formattazione dei valori delle celle
- È stato apportato un miglioramento per la rimozione di intestazione/piè di pagina di PageSetup
- È stato apportato un miglioramento per la copia delle celle
- 22 correzioni e miglioramenti.
