 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 2.4.1](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.4.1/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java v2.4.1!

 Cosa è cambiato:

- Supporta un modello più flessibile per il calcolo delle funzioni personalizzate.
- Include opzioni di controllo degli errori per le celle.
- È stato aggiunto il calcolo matematico dei parametri Range nelle formule.
- Fornisce il supporto per impostare celle collegate per frame di grafici come Titolo, Etichette dati.
- Supporta la manipolazione della posizione della finestra e delle proprietà delle dimensioni per la cartella di lavoro.
- Le operazioni di lettura/salvataggio dei file di Excel 2007 sono migliorate.
- La funzione da grafico a immagine è stata migliorata.
- La funzione diretta da Excel a PDF è stata migliorata.
- La formattazione dei valori Cell è migliorata.
- L'operazione di lettura dei file Excel 97-2003 è migliorata.
- Il modulo di salvataggio dei grafici è stato migliorato.
- Il motore di calcolo delle formule è stato migliorato.
- 21 correzioni e miglioramenti.
