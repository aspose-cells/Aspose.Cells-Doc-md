 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 2.4.3](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.4.3/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java 2.4.3!

 Cosa è cambiato:

- Supporta per ottenere i colori del tema di Excel 2007.
- Esporta grafici/forme come immagini per la conversione di file HTML.
- Supporta la lettura/scrittura di diverse intestazioni/piè di pagina per le pagine prima/pari/dispari.
- Corregge alcuni bug interni di lettura/salvataggio di file Excel 2007.
- Corregge alcuni bug interni di lettura/salvataggio dei file ODS.
- Corregge alcuni bug interni di lettura/salvataggio di file Excel 97-2003.
- È stato apportato un miglioramento per la funzione Grafico a immagine.
- Il miglioramento è stato apportato per Direct Excel alla funzione PDF.
- 14 correzioni e miglioramenti.
