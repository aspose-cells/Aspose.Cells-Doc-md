 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 2.3.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.3.0/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java 2.3.0!

 Cosa è cambiato:

- Supporta il salvataggio diretto dei file PDF senza utilizzare Aspose.Pdf for Java.
- Include il supporto del formato immagine EMF per la funzione Chart-to-Image.
- Importa l'attributo CalculationMode dai file modello MS Excel 2007.
- L'operazione di copia del foglio di lavoro è stata migliorata.
- Migliora la copia delle formule.
- Migliora la lettura dei file ODS con ODF 1.2.
- Migliora la lettura dei grafici con effetto di riempimento sfumato.
- La lettura dei file modello XLS è ottimizzata.
- 31 correzioni e miglioramenti.
