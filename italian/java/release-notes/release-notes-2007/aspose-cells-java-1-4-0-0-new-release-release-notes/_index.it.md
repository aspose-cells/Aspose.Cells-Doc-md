 Questa pagina contiene le note di rilascio per[Aspose.Cells Java 1.4.0.0 Nuova versione](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.4.0.0-new-release/)

{{% /alert %}} 

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.4.0.0!

 Sono inclusi i seguenti miglioramenti e correzioni:

- Aggiunto il metodo Cell.setValue(Object).
 Aggiunto il metodo Cell.getStringValue()
 Ottimizzato l'utilizzo della memoria
 Risolto un bug nell'importazione dei formati numerici del grafico

 Cambio di nome

 Si prega di notare che Aspose.Excel for Java è stato rinominato in Aspose.Cells for Java.

Se stai aggiornando da Aspose.Excel for Java, devi sostituire Aspose.Excel.jar con Aspose.Cells.jar nel CLASSPATH.
