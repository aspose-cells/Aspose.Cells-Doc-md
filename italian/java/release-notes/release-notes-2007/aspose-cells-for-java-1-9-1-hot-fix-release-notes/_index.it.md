 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 1.9.1 Correzione rapida](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-1.9.1-hot-fix/)

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.9.1!

 Cosa c'è di nuovo?

- Supporta l'esportazione e l'importazione da file Excel2007xlsx.
- Supporta l'ordinamento dei dati dell'intervallo.
- Supporta l'utilizzo con PHP.

 Cosa è migliorato?

-  Supporta l'impostazione della scala e dell'unità dell'asse delle categorie della scala temporale.

 Cosa è stato risolto?

-  Risolto un bug nel calcolo delle formule
-  Risolto un bug di formattazione del numero.
-  Risolto un bug nell'aggiunta di immagini.
-  Risolto un bug di salvataggio del file.
