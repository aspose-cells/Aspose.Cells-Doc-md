 Questa pagina contiene le note di rilascio per[Aspose.Cells Correzione rapida Java 1.7.1.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.7.1.0-hot-fix/)

{{% /alert %}} 

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.7.1! Sono inclusi i seguenti miglioramenti e correzioni:

- Supporta l'importazione di contenuti e stili delle celle dal file SpreadsheetML
 Supporta l'esportazione di contenuti e stili delle celle nel file SpreadsheetML
 Supporta il motore di calcolo della formula
 Supporta l'impostazione della formula R1C1
 Supporta altri due tipi di origine dati di tabella pivot
 Corregge alcuni bug relativi all'aggiunta di PivotTable .

Nota: abbiamo rimosso il const CellValueType.FORMULA e aggiunto il metodo Cell.isFormula() per decidere se la cella contiene una formula.
