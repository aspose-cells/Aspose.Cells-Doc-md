 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.4.2.0!

 Sono inclusi i seguenti miglioramenti e correzioni:

- Intervallo denominato supportato
 Aggiunto il metodo Worksheets.addCopy(),Worksheet.copy(Worksheet).
 Supportato l'aggiunta di immagini
 Impostazione del filtro automatico supportata
 Aggiunto il metodo Worksheets.addSheet(int),Worksheets.addSheet(int,String) per aggiungere il tipo di foglio grafico.
