 Questa pagina contiene le note di rilascio per[Aspose.Cells Correzione rapida Java 1.4.3.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.4.3.0-hot-fix/)

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.4.3.0! Sono inclusi i seguenti miglioramenti e correzioni:

- Supporta l'aggiunta di caselle di testo, commenti in fase di esecuzione
- Supporti per le proprietà del documento
- Impostazione Punto grafico
- Impostazione contorno su un foglio di lavoro
- Impostazione della posizione e delle dimensioni dell'area del grafico e dell'area del tracciato del grafico
- Impostazione posizione, riempimento, colore e dimensione della forma
- Importazione di dati dall'array e dalla raccolta
- Risolti bug nelle impostazioni del filtro automatico
- Risolto un bug del formato numerico personalizzato
- Risolto un bug di inserimento/cancellazione di righe e colonne
