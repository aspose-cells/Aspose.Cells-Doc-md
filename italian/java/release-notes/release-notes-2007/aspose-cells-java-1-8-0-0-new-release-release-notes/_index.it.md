 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.8.0!

 Sono inclusi i seguenti miglioramenti e correzioni:

 Supporta l'impostazione di fogli e celle attivi. Risolve alcuni bug nell'impostazione delle formule. Risolti alcuni bug di calcolo delle formule. Risolve un bug nell'apertura del file. Risolve un bug nel coping del foglio di lavoro. Risolve un bug della tabella pivot. Risolve un bug di validazione. Risolve un bug di filtri automatici. Risolve un bug di freezePanes.

Nota: per Aspose.Cells for Java 1.4, abbiamo incorporato tutte le classi di retrotranslator-runtime e backport-util-concurrent. Quindi non è necessario fare riferimento ai due jar.
