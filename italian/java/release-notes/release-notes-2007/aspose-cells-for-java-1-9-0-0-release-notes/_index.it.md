 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 1.9.0.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-1.9.0.0/)

{{% /alert %}} 

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.9.0!

 Cosa c'è di nuovo?

- Supporta l'esportazione nel file xml Aspose.Pdf
 Supporta la copia di commenti e intervalli denominati durante la copia del foglio di lavoro
 Supporta l'interfaccia ICustomFunction per il calcolo delle formule
Supporti per rintracciare precedenti e dipendenti di una cella

 Cosa è migliorato?

- Supporta solo per calcolare tutte le formule in un foglio di lavoro
 Supporta POJO e POJO con marcatori intelligenti

 Cosa è stato risolto?

- Risolto un bug per ottenere intervalli denominati del foglio di lavoro
 Risolto un bug di creazione della tabella pivot
 Risolto un bug di protezione del file
 Risolto un bug nel calcolo delle formule
 Risolto un bug di salvataggio del file.
