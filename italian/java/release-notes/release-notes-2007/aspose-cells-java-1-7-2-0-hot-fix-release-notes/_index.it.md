 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.7.2! Sono inclusi i seguenti miglioramenti e correzioni:

- Supporta l'aggiunta di immagini all'intestazione/piè di pagina dell'impostazione della pagina
Supporta l'importazione di Imposta pagina, Nome, Protezione, Convalida dati e Proprietà documento dal file SpreadsheetML
 Supporta l'esportazione di impostazioni di pagina, nome, protezione, convalida dei dati e proprietà del documento nel file SpreadsheetML
 Supporta il calcolo di più funzioni integrate
 Corregge il bug dell'impostazione automatica del colore del carattere nel grafico.
