 Questa pagina contiene le note di rilascio per[Aspose.Cells Correzione rapida Java 1.7.4.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.7.4.0-hot-fix/)

{{% /alert %}} 

 Cari clienti,

 Abbiamo rilasciato Aspose.Cells for Java V1.7.4! Sono inclusi i seguenti miglioramenti e correzioni:

- Aggiunge i metodi Column.applyStyle e Row.applyStyle.
 Corregge alcuni bug di impostazione e calcolo delle formule.
 Risolve un bug della riga di coping.
 Risolve un bug di congelamento dei riquadri.
 Risolve un bug nel salvataggio dei grafici.
 Risolve un bug nel salvataggio delle tabelle pivot.
 Corregge un bug del metodo cells.importResultSet().

Nota: il metodo getNumerOfSheets di Workbook/Worksheets è stato rinominato come bean getNumberOfSheets. Ed è meglio chiamare Worksheets.size() per ottenere il numero di fogli.
