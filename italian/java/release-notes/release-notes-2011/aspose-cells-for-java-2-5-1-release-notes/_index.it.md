 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 2.5.1](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.5.1/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java 2.5.1!

 Cosa è cambiato:

· Supporta l'interruzione del processo di apertura/salvataggio mentre si lavora con file di grandi dimensioni.

 · Fornisce supporto per impostare e calcolare più formule.

 · Supporti per ottenere/impostare la posizione assoluta della forma.

 · Risolve bug di lettura/salvataggio di file Excel 2007.

 · Corregge i bug di lettura/salvataggio dei file ODS.

 · Risolve bug di lettura/salvataggio di file Excel 97-2003.

 · È stato apportato un miglioramento alle funzioni Grafico a immagine, Foglio a immagine ed Excel a PDF.

 · Il miglioramento è fatto per l'analisi delle formule.

 · Il miglioramento è fatto per il calcolo delle formule.

 · 22 correzioni e miglioramenti.
