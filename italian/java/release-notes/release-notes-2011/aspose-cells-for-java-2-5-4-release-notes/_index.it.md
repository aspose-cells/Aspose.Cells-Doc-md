## **Siamo lieti di annunciare Aspose.Cells for Java v2.5.4!**
### **Nuove caratteristiche**
25051 - Supporta tipi di filtri automatici avanzati: MultipleFilter, DynamicFilter, IconFilter

 25742 - Formatta l'intera tabella pivot o le celle nel rapporto tabella pivot

 26627 - Impostare le proprietà dell'asse, come il valore Max/Min come "Automatico"

 26655 - Esporta immagini GIF animate per file HTML

 26607 - Supporto per il calcolo della funzione EOMONTH
### **Miglioramenti**
 26655 - Importa/Esporta stili di data e bordo per file ODS

 27317 - Mantieni la miniatura del grafico risalvato uguale al file originale quando incorpori XLS in PPT

 27214 - Rende nascosti/mostrati gli elementi del campo pivot

 26626 - Leggi/Salva gli stili della tabella pivot
### **Prestazione**
 27041 - La lettura dei file ODS causa problemi di memoria
### **Eccezioni**
 26710 - Ricevi un'eccezione durante la lettura dei controlli del modulo dal file XLSX

 27128 - Gestire "NumberFormatException" durante la lettura del file ODS il cui indice di riga del riquadro diviso supera il limite di Short

27578 - Si è verificata un'eccezione "IllegalArgumentException" durante la lettura di file ODS la cui definizione della larghezza della colonna supera il limite del file Excel

 27773 - Gestione di "ArrayIndexOutOfBoundsException" durante la lettura di collegamenti ipertestuali da file ODS

 27780 - Ottieni "NumberFormatException" durante il rendering del file PDF
### **Insetti**
 26563 - Problema con la scala Axis nella funzione Chart-to-Image

 27793 - Stile del bordo perso durante il nuovo salvataggio del file XLS

 26610 - Calcola le funzioni EVEN, ODD, ROUNDUP/ROUNDDOWN

 27445 - Problema relativo alla scala dell'asse X/Y per la funzionalità da grafico a immagine

 26605 - Calcola le funzioni CERCA.ORI, CERCA.VERT/CERCA

 26156 - Problema nel calcolo della funzione SOMMA

 26986 - Visualizza le serie di dati del grafico per chart2image

 26735 - Problema nel rendering del colore delle etichette dei dati per la funzionalità Grafico a immagine

 26600 - Rendering del grafico a torta esploso per la funzione Grafico a immagine

 27754 - Il file HTML generato genera il problema "File non trovato..."

26599 - Problema con il rendering delle linee della griglia per la funzione da grafico a immagine

 27699 - La creazione di un foglio nascosto può causare problemi di protezione

 27040 - Problema con la lettura dello stile e delle formule del bordo

 26151 - Problema con la lettura delle date precedenti al 1900-02-28 dal file modello ODS

 27426 - Problema con la lettura delle forme nel grafico

 26626 - Aggiorna/calcola le tabelle pivot a livello di programmazione

 26604 - Eseguire il rendering di una serie di celle nascoste per la funzionalità da grafico a immagine

 26561 - Problema con il corretto allineamento di numeri grandi per la funzione Foglio-immagine

 27792 - Salva il formato data personalizzato nel file ODS

 27445 - Problema di scala dell'asse nella funzione Grafico a immagine

 26666 - Imposta lo stile di una cartella di lavoro sulle celle di un'altra cartella di lavoro
