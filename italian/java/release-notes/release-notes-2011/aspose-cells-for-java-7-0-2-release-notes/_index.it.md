{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 7.0.2](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.0.2/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java v7.0.2!

 Caratteristiche generali / Miglioramenti

 Supporta LightCellsDataProvider per il salvataggio di file XLSX

 Supporta la versione J2SE7.0

 Sono stati apportati miglioramenti alla funzione Foglio-immagine

 Sono stati apportati miglioramenti per la funzionalità da Excel a PDF



 Tutte le correzioni



 Miglioramenti

 30504 - Sincronizza il tipo di valore della proprietà Cells.StandardHeightPixels

 30738 - Supporto per impostare il foglio di lavoro come nascosto in fase di esecuzione anche se è l'unico foglio nella cartella di lavoro

 29151 - Supporta la funzionalità di raggruppamento dei dati negli indicatori intelligenti

 30673 - Supporta la funzione di ordinamento dei dati negli Smart Marker

 30809 - Includi più tipi di condizioni di formato (per la funzione di formattazione condizionale) durante il salvataggio del file XLS

 31192 - Supporta il tipo di crittografia: RC4, Microsoft Enhanced Cryptographic Provider 1.0 a 128 bit

 Eccezioni

30733 - Ottenere un'eccezione quando si utilizzano valori scientifici per la formula di convalida

 30748 - La lettura del file modello Excel2007 crittografato causa un'eccezione

 30971 - Ottenere un'eccezione per il metodo Cell.getStringValue()

 30994 - Ricevi un'eccezione durante il salvataggio di XLSX con le API LightCells

 Insetti

 30616 - Immagine persa nella funzione da XLS a PDF

 30695 - Asse X/Y perso nella funzione Grafico a immagine

 30477 - Valori "#NA" persi nella funzione Chart-to-Image

 30697 - L'immagine è stata creata in modo errato per il grafico a piramide

 30776 - Un file PDF non valido è stato generato dalla funzione XLS in PDF

 28545 - Il valore di Cell non è stato reso correttamente con la proprietà "shrink to fit"

 30777 - Il testo è stato allineato al centro anziché a sinistra nel file PDF risultante

 30911 - TextBox.setHtmlText() non ha analizzato correttamente il rich text

 30479 - È stata aggiunta una parentesi aggiuntiva per il valore della cella nel file PDF generato

 30899 - Il valore ridimensionato dell'immagine appena aggiunta è stato modificato nel file XLS salvato

30972 - Il contenuto della casella di testo non è stato visualizzato correttamente nel file PDF

 31050 - Cell.getStringValue() non ha fornito il doppio valore corretto

 31007 - Le formule sono state visualizzate come "#NAME" nel file XLS generato

 31194 - La formula MROUND non è stata calcolata correttamente

 31147 - La formula è stata calcolata come "#VALUE" -1

 31237 - La formula è stata calcolata come "#VALUE" -2

 31005 - Le parentesi sono state perse per i valori negativi nel file PDF generato

 31312 - La formula ROUND è stata calcolata in modo errato
