{{% alert color="primary" %}} 

 Questa pagina contiene le note di rilascio per[Aspose.Cells for Java 7.0.1](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.0.1/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Java v7.0.1!

 Miglioramenti

 30556 Migliora la funzione da foglio di lavoro a immagine

 30557 Migliora la conversione da Excel a PDF

30458 Aggiunta di nuove API: Worksheet.calculateFormula(), Row.getFirstCell(), Row.getLastCell()

 30487 Supporto Aspose.Total licenza

 29838 Eseguire il rendering dell'immagine nella grafica personalizzata dell'utente

 30558 Includere il metodo FormatCondition.getPriority()

 30505 Includere il metodo Cells.checkColumn(int).

 30504 Includi getter/setter per gli attributi Cells.StandardHeightPixels/StandardWidthPixels

 30459 Includere i metodi WorkbookSettings.setLocale(Locale) e Protection.getPasswordHash()

 30469 Includi getter/setter per WorkbookSettings.WindowHeightInch/WindowWidthInch

 Eccezioni

 29285 Ottieni un'eccezione durante il salvataggio di PDF con locale francese

 30459 Ricevi un'eccezione durante la lettura del file XLS con le impostazioni locali predefinite prive di prefisso internazionale

 30489 Worksheet.autoFitColumn() e Chart.toImage() possono causare "java.awt.HeadlessException"

 30587 Ricevi un'eccezione durante la lettura del file modello XLS

 Insetti

30103 I caratteri del grafico sono stati modificati dopo il nuovo salvataggio del file XLSX

 28858 L'ordinamento dei nomi provoca la modifica della cella collegata del controllo

 30486 L'allineamento verticale è stato modificato nel file XLSX nuovamente salvato
