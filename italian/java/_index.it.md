**Benvenuti su Aspose.Cells for Java**

Aspose.Cells for Java è una libreria di classi di programmazione che consente agli sviluppatori software di manipolare e elaborare file di fogli di calcolo all'interno delle proprie applicazioni.

Unito alle API e ai controlli GUI, Aspose.Cells for Java accelera la programmazione e la conversione di Microsoft Excel®.

Aspose.Cells for Java supporta i formati di file di fogli di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) popolari utilizzati quotidianamente nel tuo business. Consente inoltre di esportare file Excel in PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, testo normale e formati di immagine popolari tra cui TIFF, JPG, PNG, BMP e SVG.
