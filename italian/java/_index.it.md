**Benvenuti a Aspose.Cells for Java**

 Aspose.Cells for Java è una libreria di classi di programmazione che consente agli sviluppatori di software di manipolare ed elaborare file di fogli di calcolo all'interno delle proprie applicazioni.

Combinato con API e controlli GUI, Aspose.Cells for Java velocizza la programmazione e la conversione di Microsoft Excel®.

Aspose.Cells for Java supporta i popolari fogli di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) formati di file che la tua azienda utilizza ogni giorno. Consente inoltre di esportare file Excel in PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, testo normale e formati di immagine popolari tra cui TIFF, JPG, PNG, BMP and SVG.
