## **Scarica da GitHub**
 Tutti gli esempi di Aspose.Cells for Java sono ospitati su[Git Hub](https://github.com/aspose-cells/Aspose.Cells-for-java) . Puoi clonare il repository utilizzando il tuo client GitHub preferito o scaricare il file ZIP da[qui](https://github.com/aspose-cells/Aspose.Cells-for-java/archive/master.zip).

 Estrai il contenuto del file ZIP in qualsiasi cartella sul tuo computer. Tutti gli esempi si trovano in**Esempi** cartella.

 Il**Risorse**La cartella contiene documenti di input o file utilizzati negli esempi.

 Puoi eseguire/eseguire questi esempi con qualsiasi IDE di tua scelta, tuttavia, a scopo dimostrativo, stiamo usando**Eclisse** . Utilizzare il**File -> Importa** opzione per importare l'esempio di tua scelta e specificare il percorso.

Aggiungere i file jar richiesti (librerie) nel progetto.

Ogni esempio viene fornito come classe eseguibile separata. Fare clic con il pulsante destro del mouse sul file e scegliere Esegui file.

{{% alert color="primary" %}} 

 Gli esempi sono sotto forma di**Progetto Eclipse** . Si prega di aprire gli esempi in Eclipse utilizzando il file**File > Apri progetti dal file system...** comando di menu e seguire i passaggi. Ora dovresti essere in grado di eseguire correttamente gli esempi.

{{% /alert %}} 
## **Eseguire l'esempio**

## **Contribuire**
Se desideri aggiungere o migliorare un esempio, ti invitiamo a contribuire al progetto. Tutti gli esempi e i progetti vetrina in questo repository sono open source e possono essere utilizzati liberamente nelle tue applicazioni.

Per contribuire, puoi eseguire il fork del repository, modificare il codice sorgente e creare una richiesta pull. Esamineremo le modifiche e le includeremo nel repository se ritenuto utile.
