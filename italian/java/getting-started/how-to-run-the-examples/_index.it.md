## **Scarica da GitHub**
Tutti gli esempi di Aspose.Cells for Java sono ospitati su [GitHub](https://github.com/aspose-cells/Aspose.Cells-for-java). Puoi clonare il repository usando il tuo client GitHub preferito o scaricare il file ZIP da [qui](https://github.com/aspose-cells/Aspose.Cells-for-java/archive/master.zip).

Estrai i contenuti del file ZIP in una qualsiasi cartella sul tuo computer. Tutti gli esempi sono situati nella cartella **Esempi**.

La cartella **Risorse** contiene documenti di input o file utilizzati negli esempi.

Puoi eseguire/eseguire questi esempi con qualsiasi IDE a tua scelta, tuttavia, per fini dimostrativi, stiamo usando **Eclipse**. Utilizza l'opzione **File -> Importa** per importare l'esempio di tua scelta e specificare il percorso.

Aggiungi i file jar (Librerie) necessari al progetto.

Ogni esempio è fornito come una classe eseguibile separata. Fai clic con il pulsante destro del mouse sul file e scegli Esegui File.

{{% alert color="primary" %}} 

Gli Esempi sono sotto forma di **Progetto Eclipse**. Si prega di aprire gli Esempi in Eclipse utilizzando il comando di menu **File > Apri Progetti da File System...** e seguire i passaggi. Ora, dovresti essere in grado di eseguire gli esempi correttamente.

{{% /alert %}} 
## **Esegui l'Esempio**
Una volta aperti gli Esempi nell'IDE Eclipse, seleziona un file .java qualsiasi. Ad esempio, seleziona questo.

TilePictureAsTextureInsideShape.java

trovato all'interno della

Esempi>src>AsposeCellsExamples>directory OggettiDisegno

o all'interno del pacchetto

AsposeCellsExamples.DrawingObjects

e eseguilo.

## **Contribuisci**
Se ti piace aggiungere o migliorare un esempio, ti incoraggiamo a contribuire al progetto. Tutti gli esempi e i progetti di showcase in questo repository sono open source e possono essere liberamente utilizzati nelle proprie applicazioni.

Per contribuire, è possibile fare il fork del repository, modificare il codice sorgente e creare una pull request. Esamineremo le modifiche e le includeremo nel repository se trovate utili.
