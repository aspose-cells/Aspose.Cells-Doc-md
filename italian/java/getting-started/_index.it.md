##  **Requisiti di sistema**

Aspose.Cells for Java supporta le seguenti versioni Java:

- J2SE6.0 (1.6)
- J2SE7.0 (1.7)
- o sopra

[Più dettagli](/cells/it/java/system-requirements/)

##  **Installazione**

###  **Installa Aspose Cells for Java dal repository Maven**

 Maven è il modo più semplice per scaricare e installare Aspose.Cells for Java.

1.  Specificare la configurazione/posizione del repository Aspose Maven nel file Maven pom.xml.
 2. Definisci la dipendenza Aspose.Cells for Java API nel tuo pom.xml.

[Ulteriori passaggi per i dettagli](/cells/it/java/installation/)

###  **Installa Aspose Cells for Java manualmente**

1. [Scarica i vasetti di aspose-cells](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) in una cartella.
1. Entra nella cartella, installa manualmente i pacchetti JAR con Eclipse o Intellij.
