## **Requisiti di sistema**

Aspose.Cells for Java supporta le seguenti versioni di Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- o superiore

[Maggiori dettagli](/cells/it/java/system-requirements/)

## **Installazione**

### **Installare Aspose Cells for Java dal repository Maven**

Maven è il modo più semplice per scaricare ed installare Aspose.Cells for Java. 

1. Specifica la configurazione/localizzazione del repository Maven Aspose nel tuo file pom.xml di Maven. 
2. Definisci la dipendenza API di Aspose.Cells for Java nel tuo file pom.xml. 

[Altri passaggi per i dettagli](/cells/it/java/installation/)

### **Installare Aspose Cells for Java manualmente**

1. [Scarica i file jar di aspose-cells](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) in una cartella.
1. Entra nella cartella, installa manualmente i pacchetti JAR con Eclipse o Intellij.
