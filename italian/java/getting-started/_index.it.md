## **Requisiti di sistema**

Aspose.Cells for Java supporta le seguenti versioni Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- o sopra

[Più dettagli](/cells/it/java/system-requirements/)

## **Installazione**

### **Installa Aspose Cells for Java dal repository Maven**

 Maven è il modo più semplice per scaricare e installare Aspose.Cells for Java.

1.  Specificare Aspose Maven Configurazione/posizione del repository nel file Maven pom.xml.
 2. Definisci la dipendenza Aspose.Cells for Java API nel tuo pom.xml.

[Più passo per i dettagli](/cells/it/java/installation/)

### **Installa Aspose Cells for Java manualmente**

1. [Scarica aspose-cells-{versione}-java.zip](https://downloads.aspose.com/cells/java/)
1. Decomprimere il pacchetto, accedere alla directory \JDK 1.6\, decomprimere aspose-cells-{version}-java.zip.
1. Immettere la directory \JDK 1.6\aspose-cells-{versione}-java\lib\, installare i pacchetti JAR manualmente con Eclipse o Intellij.
