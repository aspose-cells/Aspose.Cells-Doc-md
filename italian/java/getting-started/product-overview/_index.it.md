## **Benvenuti nella documentazione di Aspose.Cells**
**Aspose.Cells** è il componente più flessibile che consente alle applicazioni Java di creare e gestire fogli di calcolo Excel senza richiedere l'installazione di **Microsoft Excel** sul sistema. A differenza di prodotti simili di altri fornitori, Aspose.Cells non supporta solo la generazione di fogli di calcolo e altre funzionalità di formattazione di base, ma supporta anche numerose funzionalità avanzate. Queste caratteristiche avanzate rendono molto più conveniente per gli sviluppatori manipolare i contenuti del foglio di calcolo, la formattazione delle celle e la protezione dei file. Aspose.Cells consente anche di importare dati nei fogli di calcolo da diverse fonti di dati, aggiungere formule/matematiche, data/ora, finanziarie, testuali, supporta la manipolazione di grafici, immagini, commenti, oggetti disegno e controlli, ecc.

Aspose.Cells for Java fornisce anche una soluzione Grid completa per i clienti web; ha un controllo basato su GUI come segue:

- Aspose.Cells.GridWeb, che supporta le applicazioni web

Utilizzando Aspose.Cells.GridWeb per il controllo Grid JAVA, gli sviluppatori possono importare ed esportare facilmente fogli di calcolo Excel®. Qualsiasi tipo di utente che è familiare con Microsoft Excel può controllare l'aspetto della griglia. Aspose.Cells offre un ricco set di funzionalità come il controllo del design e del layout della griglia, la gestione di più fogli di lavoro, la gestione di celle, righe e colonne di un foglio di lavoro, la creazione e il calcolo di formule, ecc.

## **Descrizione del prodotto**
Aspose.Cells può essere utilizzato con qualsiasi tipo di applicazione, sia che si tratti di un'applicazione Web Java o di un'applicazione Desktop Java. Inoltre, la libreria può essere utilizzata anche con altre soluzioni come PHP, ecc. Aspose.Cells fornisce demo completamente funzionali ed esempi di lavoro scritti in Java per consentire agli sviluppatori di avere una migliore comprensione delle funzionalità esposte dal componente.

Oltre a supportare funzionalità comuni come la creazione o la lettura di file Excel nativi, Aspose.Cells offre molte funzionalità avanzate come il salvataggio e l'apertura di file Excel da e verso flussi, l'invio di output a un browser client, l'importazione ed esportazione di dati da un **ResultSet** o un **Array**, l'importazione di immagini e grafici, la creazione di immagini e grafici tramite API, l'importazione di formule da un foglio di calcolo di progettazione e la configurazione del setup di pagina tramite API.

Inoltre, ci sono anche diverse funzionalità uniche che gli sviluppatori possono trovare solo in Aspose.Cells, come il supporto dei formati CSV e SpreadsheetML, l'aggiunta della copia di un foglio di lavoro esistente (con tutti i contenuti, immagini e grafici ecc.) a un file, la configurazione di uno sfondo a sfumature per i grafici tramite API, la creazione di commenti, filtri automatici e interruzioni di pagina tramite API, la configurazione di formule complesse, il supporto della formattazione condizionale, il supporto delle opzioni di protezione introdotte in Microsoft Excel XP o successivo, la manipolazione di intervalli denominati, la ricerca di API, il supporto di API di grafici personalizzati e del motore di calcolo delle formule, ecc.
## **Caratteristiche del documento**
- Aprire file Excel Plain o crittografati (Excel 95, Excel97, Excel2007/2010/2013/2016/2019, Office365) da diverse fonti
- Aprire file di numeri Plain (creati dall'applicazione "Numbers" di Apple) da diverse fonti
- Salvare file Excel (Excel97-Excel2007/2010/2013/2016/2019) in diversi formati (XLS, XLSX/XLSM, XLTX, XLTM, XLSB, CSV, SpreadsheetML, Tab delimited/TSV, HTML ecc.)
- Convertire file Excel in documento PDF
- Convertire foglio di calcolo Excel in documento XPS
- Convertire in formato di file immagine **(TIFF)**
- Leggere e scrivere nel formato **OpenDocument Spreadsheet (ODS)**
- Modificare le proprietà del documento dei file Excel
## **Caratteristiche del foglio di lavoro**
### **Opzioni di visualizzazione**
- **Nascondere o mostrare** un foglio di lavoro
- Visualizzare o nascondere **schede, barre di scorrimento, griglie e intestazioni delle righe e colonne**
- **Anteprima interruzioni di pagina**
- Impostare il **livello di zoom** per il foglio di lavoro
- Rendere visibili i dati selezionati durante lo scorrimento utilizzando **pannelli bloccati**
### **Protezione del foglio di lavoro**
- **Proteggere i contenuti, gli oggetti e gli scenari** di un foglio di lavoro
### **Opzioni di impostazione pagina**
- Impostare l'**orientamento della pagina, la scala, le dimensioni della carta, la qualità di stampa** ecc.
- Impostare i **margini della pagina** come superiore, sinistro, destro, inferiore ecc.
- Imposta **intestazioni e piè di pagina**
- Imposta **area di stampa**, **titolo di stampa** ecc.
### **Interruzioni di pagina**
- Inserisci **interruzioni di pagina orizzontali** e **verticali** in una cella specifica
### **Funzionalità righe e colonne**
- **Regola la larghezza delle colonne** secondo necessità
- Ridimensiona automaticamente la larghezza e l'altezza delle celle in base ai loro contenuti
- **Inserisci o elimina** righe e colonne
- **Nascondi o mostra** righe e colonne
- **Raggruppa o sepaprugroupa** righe e colonne
### **Funzionalità di gestione dei dati**
- Aggiungi dati alle celle desiderate in tempo reale
- **Importa dati nei fogli di lavoro** da diverse fonti di dati
- Recupera dati dalle celle in base ai loro tipi di dati
- **Esporta dati delle celle in un array**
- Trova o ricerca dati specifici nelle celle
- Imposta **formule con funzioni integrate o aggiuntive**
- Imposta la formula di riferimento **array** e **R1C1**
- Aggiungi diversi tipi di collegamenti ipertestuali
- Crea o accedi a intervalli nominati
- Applica **validazione dei dati** e **filtro automatico**
- Applica **subtotali** alle tabelle dei fogli di lavoro
### **Caratteristiche di formattazione**
- Applica **impostazioni di formato numero** alle celle
- Formattare la cella come una data
- Impostare l'**allineamento del testo**
- Unire più celle
- Applicare diversi tipi di **impostazioni del carattere** per le celle
- Applicare colori diversi alle celle, caratteri, griglie, oggetti grafici, ecc.
- Impostare il **testo arricchito** in una singola cella
- Applicare diverse **impostazioni dei bordi** alle celle
- Applicare diversi **schemi di sfondo** alle celle
- Applicare **impostazioni di formato** su un foglio di lavoro, riga o colonna, ecc.
- Applicare la **formattazione condizionale** alle celle
## **Funzionalità di grafici e grafica**
- Creare diversi tipi di grafici facilmente
- Personalizzare **grafici** impostando le proprietà
- Impostare i **dati per i grafici** in modo dinamico
- Aggiungere **grafici personalizzati** ai fogli di lavoro
- Aggiungere **immagini** ai fogli di lavoro durante l'esecuzione
- Creare e aggiungere commenti alle celle
- Convertire il grafico in un file immagine
- Convertire il foglio di lavoro in un file immagine
- Stampare i fogli di lavoro
## **Funzionalità avanzate**
- Supporta il **calcolo delle formule** utilizzando un robusto motore di calcolo delle formule
- Cambiare i **dati di origine di una tabella pivot** durante l'esecuzione
- Creare e manipolare tabelle pivot
- Manipolare il codice VBA o le macro.
## **Piattaforme supportate**
