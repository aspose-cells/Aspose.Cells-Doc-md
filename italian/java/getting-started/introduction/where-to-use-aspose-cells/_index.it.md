{{% alert color="primary" %}} 

Aspose.Cells è stato creato per aiutare gli sviluppatori a creare e gestire fogli di calcolo di Microsoft Excel in modo programmatico. Aspose.Cells non comprende solo fogli di calcolo nel formato Microsoft Excel (XLS/XLSX), ma anche molti altri formati di file come valori separati da virgola (CSV), delimitato da tabulazioni/TSV (testo), SpreadsheetML, ecc.

Aspose.Cells offre la maggior parte delle funzionalità disponibili in Microsoft Excel, ma la cosa eccitante è che Aspose.Cells può elaborare i file Excel in modo dinamico, il che consente agli sviluppatori di elaborare migliaia o milioni di fogli di calcolo molto rapidamente.

Questo articolo discute i casi d'uso comuni di Aspose.Cells.

{{% /alert %}} 
## **Possibili Scenari di Utilizzo**
È compito degli sviluppatori decidere come utilizzare Aspose.Cells, ma ci sono alcuni casi d'uso comuni in cui Aspose.Cells può essere utile. L'uso più comune di Aspose.Cells consiste nel distribuirlo sul server Web o nei servizi Web per l'elaborazione lato server. Alcuni dei possibili scenari d'uso di Aspose.Cells sono descritti di seguito.
## **Gestione automatizzata dei fogli di calcolo**
1. Gli utenti caricano manualmente i fogli di calcolo creati su un server.
1. Aspose.Cells viene utilizzato per aprire file e recuperare dati.
   Dopo che i dati sono recuperati:
   1. I dati vengono salvati in un database.
   1. Alcuni dati sono presentati nelle pagine web.
   1. Alcuni aspetti dei fogli di calcolo vengono modificati e salvati di nuovo nei file Microsoft Excel.
## **Applicazione web**
1. Gli utenti inseriscono dati nelle pagine web.
1. I dati vengono raccolti dal server web.
1. Aspose.Cells viene utilizzato per popolare i dati in nuovi file o file modello predefiniti.
1. Dopo che il file Excel viene creato, può essere inviato all'utente come report, ecc.
