## **xlsx4j**
xlsx4j è una libreria Java open source (ASLv2) per la creazione e la manipolazione dei file Microsoft Open XML Excel xlsx simile a docx4j per la manipolazione dei file Microsoft Open XML Word docx.
È simile all'OpenXML SDK di Microsoft, ma per Java. docx4j utilizza JAXB per creare la rappresentazione degli oggetti in memoria.
Dovrai prenderti il tempo per comprendere JAXB e la struttura del file Open XML.

Leggi di più su [docx4java.org](https://www.docx4java.org/trac/docx4j)
## **Aspose.Cells for Java**
Aspose.Cells for Java è un premiato componente di fogli di calcolo Excel che consente ai programmatori Java di incorporare la capacità di leggere, scrivere e manipolare fogli di calcolo Excel® (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF e formati di file immagine nei propri applicazioni Java senza dover fare affidamento su Microsoft Excel®.

Aspose.Cells for Java è un componente maturo, scalabile e ricco di funzionalità che offre molte funzioni che vanno ben oltre le semplici capacità di esportazione dati di altri venditori. Con Aspose.Cells for Java, i programmatori possono esportare dati, formattare fogli di calcolo al livello più granulare, importare immagini, creare grafici, applicare e calcolare formule complesse, trasmettere dati da Excel®, salvare in vari formati e molto altro, tutto ciò senza la necessità di Microsoft Excel® o Automazione di Microsoft Office.

Leggi di più su [aspose.com](https://products.aspose.com/cells/java/)

## **Perché non xlsx4j**
xlsx4j può solo analizzare e salvare file Excel xlsx Microsoft Open XML, Aspose.Cells for Java può elaborare tutti i formati di file Excel (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS) e supporta varie conversioni di formati di file.

## **Aspose.Cells Java for xlsx4j**
Il progetto Aspose.Cells per xlsx4j mostra come eseguire diverse attività utilizzando le API Aspose.Cells Java in confronto con xlsx4j. Il progetto copre anche le funzionalità disponibili solo nelle API Aspose.Cells ma non in xlsx4j per gestire i fogli di calcolo.

Questo progetto è utile per gli sviluppatori che desiderano confrontare xlsx4j con Aspose.Cells O migrare da xlsx4j ad Aspose.Cells.
## **Requisiti di sistema e piattaforme supportate**
### **Requisiti di sistema**
Di seguito i requisiti di sistema per eseguire Aspose.Cells Java per xlsx4j:

- Java 1.4 o versione successiva installato.
- Componente Aspose.Cells scaricato.
- Librerie docx4j scaricate.
### **Piattaforme Supportate**
Di seguito le piattaforme supportate:

- Aspose.Cells 8.3.0 o versione successiva.
- Librerie docx4j 3.1.0 o versione successiva.
- IDE Java (Eclipse, NetBeans, IntelliJ, ...).

Per configurare **Aspose.Cells for Java API**, includere l'URL del repository di Aspose e la dipendenza maven di Aspose.Cells for Java API nel vostro **pom.xml** come segue:
### **Repository Maven di Aspose**
{{< highlight java >}}

 <repositories>

    <repository>

        <id>aspose-maven-repository</id>

        <name>Aspose Maven Repository</name>

        <url>http://repository.aspose.com/repo/</url>

    </repository>

</repositories>

{{< /highlight >}}
### **Dipendenza Maven di Aspose.Cells for Java**
{{< highlight java >}}

 <dependencies>

    <dependency>

        <groupId>com.aspose</groupId>

        <artifactId>aspose-cells</artifactId>

        <version>8.6.0</version>

    </dependency>

</dependencies>

{{< /highlight >}}

**Nota:** Si prega di utilizzare la versione più recente disponibile dell'API di Aspose.
