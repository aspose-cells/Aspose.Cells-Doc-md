## **Scarica le librerie richieste**
Scarica le librerie richieste menzionate di seguito. Questi sono i requisiti per l'esecuzione degli esempi Aspose.Cells Java for Python.

- **Aspose:** [Aspose.Cells for Java Componente](https://downloads.aspose.com/cells/java/)
- [JPype](https://pypi.org/project/JPype1/)
## **Scarica esempi dai siti di codifica sociale**

## **Come configurare il codice sorgente**
Si prega di seguire questi semplici passaggi per aprire ed estendere il codice sorgente durante l'utilizzo:

 \1. Installa solo Python 2.7.xx da:<https://www.python.org/downloads/>
 \2. Installa JPype da:<https://sourceforge.net/projects/jpype/files/JPype/>
 \3. Installa Java JDK da:<https://www.oracle.com/java/technologies/downloads/>
 \4. Scarica le API Aspose.Cells for Java che desideri utilizzare nei tuoi progetti Python da:<https://downloads.aspose.com/cells/java> e copia i file .jars delle API scaricate nella cartella %Python%/lib.
 \5. Clone[Aspose.Cells Java for Python Esempi](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) da github.
