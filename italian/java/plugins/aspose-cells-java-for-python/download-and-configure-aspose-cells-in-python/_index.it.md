## **Come configurare il codice sorgente**
Si prega di seguire questi semplici passaggi per aprire ed estendere il codice sorgente durante l'uso:

\1. Install only Python 2.7.x.x from: <https://www.python.org/downloads/>
\2. Install JPype from: <https://sourceforge.net/projects/jpype/files/JPype/>
\3. Install Java JDK from: <https://www.oracle.com/java/technologies/downloads/>
\4. Download Aspose.Cells for Java APIs you want to use in your python projects from: <https://downloads.aspose.com/cells/java> and copy downloaded APIsâ€™s .jars into %Python%/lib folder.
