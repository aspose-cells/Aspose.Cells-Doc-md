## **Framework Web Apache Struts**
Il framework web Apache Struts è una soluzione gratuita e open-source per la creazione di applicazioni web Java.

Le applicazioni web differiscono dai siti web convenzionali nel senso che le applicazioni web possono creare una risposta dinamica. Molti siti web forniscono solo pagine statiche. Un'applicazione web può interagire con database e motori di logica aziendale per personalizzare una risposta.

Un modo per separare le preoccupazioni in un'applicazione software è utilizzare un'architettura Model-View-Controller (MVC). Il Modello rappresenta il codice aziendale o del database, la Vista rappresenta il codice di design della pagina e il Controller rappresenta il codice di navigazione. Il framework Struts è progettato per aiutare gli sviluppatori a creare applicazioni web che utilizzano un'architettura MVC.

[Per saperne di più](http://struts.apache.org/birdseye.html)
### **Aspose.Cells for Java**
Aspose.Cells for Java è una libreria di classi avanzata per Java che consente di eseguire una vasta gamma di attività di elaborazione documenti direttamente all'interno del tuo Java
applicazioni.

Aspose.Cells for Java supporta l'elaborazione di Cells (DOC, DOCX, OOXML, RTF) HTML, OpenDocument, PDF, EPUB, XPS, SWF e tutti i formati di immagine. Con Aspose.Cells è possibile
generare, modificare e convertire documenti senza utilizzare Microsoft Cells.
## **Requisiti di sistema**
**Di seguito sono indicati i requisiti di sistema per Aspose.Cells Java per l'applicazione web Struts 1.3:**

- Java 1.6 o superiore installato.
- Server Apache Tomcat 7 o superiore
- Maven 3
- Apache Struts 1.3
- Componente Aspose.Cells.
## **Piattaforme Supportate**
**Di seguito sono indicate le piattaforme supportate:**

- Aspose.Cells 8.5.0 o successivo.
- IDE Java (Eclipse, NetBeans, IntelliJ, ...).
