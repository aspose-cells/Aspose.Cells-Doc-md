## **Costruire Aspose.Cells Java per Struts 1.3 dai codici sorgente**
Dopo aver verificato i codici sorgente da uno qualsiasi dei repository di cui sopra, applica i seguenti comandi mvn:

{{< highlight "java" >}}

 $ mvn -U clean package 

{{< /highlight >}}

Questo creerà "Strutsbookapp.war" nel cassiere di destinazione.

Per distribuire il file .war basta copiarlo nella directory webapp del server Apache tomcat in esecuzione.
