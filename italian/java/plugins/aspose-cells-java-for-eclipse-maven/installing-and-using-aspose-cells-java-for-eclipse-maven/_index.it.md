## **Installazione**

Il plugin **Aspose.Cells Java per Eclipse (Maven)** può essere facilmente installato dalla scheda **Plugin** disponibile nella finestra di dialogo Plugin.

- Per aprirlo, seleziona **Plugin** dal menu **Strumenti** in Eclipse.  

![todo:image_alt_text](installing-and-using-aspose-cells-java-for-eclipse-maven_1)

- Questo aggiunge il **Progetto Maven Aspose.Cells** nella procedura guidata Nuovo Progetto e il **Esempio di Codice Aspose.Cells** nella procedura guidata Nuovo Altro dell'IDE Eclipse.  

![todo:image_alt_text](project_1.png)

## **Utilizzo**

### **Progetto Maven Aspose.Cells (procedura guidata)**

Per creare il **Progetto Maven** tramite la procedura guidata per utilizzare l'[API Aspose.Cells for Java](https://products.aspose.com/cells/java/):

1. Seleziona **Nuovo -> Progetto**.
1. Seleziona **Progetto Maven Aspose.Cells** nella categoria **Maven**.
1. Fare clic su **Avanti**.

![todo:image_alt_text](project_2.png)

1. Fornisci **Nome Progetto, Posizione, GroupId, ArtifactId** e **Versione** per il tuo Progetto Maven e fai clic su **Fine.**

![todo:image_alt_text](project_3.png)

1. Ciò recupererà il riferimento **Maven Dependency** all'[API Aspose.Cells for Java](https://products.aspose.com/cells/java/) più recente dal [Repository Maven di Aspose Cloud](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) e lo configurerà nel file **pom.xml**. Se hai scelto anche di scaricare gli **Esempi di Codice**, inizierà anche il download degli **Esempi di Codice** dal [Repository degli Esempi di API Aspose.Cells for Java](https://github.com/aspose-cells/Aspose.Cells-for-Java).
1. Al termine della procedura guidata, verrà creato il **Progetto Maven** seguente sul tuo **Eclipse IDE**:  

![todo:image_alt_text](project_4.png)

1. Il **Progetto Maven** creato è configurato per utilizzare l'**API Aspose.Cells for Java** ed è pronto per essere migliorato in base alle esigenze del progetto.
   Se hai scelto di scaricare gli [Esempi di Codice](https://github.com/aspose-cells/Aspose.Cells-for-Java), puoi utilizzare la **Procedura guidata Esempio di Codice Aspose.Cells** per importare gli **Esempi di Codice** necessari dell'**API Aspose.Cells for Java** nel tuo progetto.

### **Procedura guidata Esempio di Codice Aspose.Cells**

La **Procedura guidata Esempio di Codice Aspose.Cells** ti consente di provare molti esempi forniti per l'**API Aspose.Cells for Java**.

{{% alert color="primary" %}}

Per poter utilizzare comodamente il **Aspose.Cells Code Example wizard**: è consigliabile selezionare sempre **Scarica anche gli esempi di codice** durante la creazione del **Progetto Maven** sul **Aspose.Cells Maven Project wizard**,

{{% /alert %}}

Per utilizzare gli esempi, basta:

1. Fare clic su **Nuovo -> Altro** in **Eclipse**.
1. Selezionare **Aspose.Cells Code Example** nella categoria **Java**.
1. Fare clic su **Avanti**.  

![todo:image_alt_text](example_1.png)

1. Scegliere il progetto e poi espandere l'albero per selezionare la categoria **Code Example** e fare clic su **Fine**.

![todo:image_alt_text](example_2.png)

1. Questo copierà i file Java della categoria selezionata **Esempi di codice** nel progetto sotto il pacchetto **com.aspose.cells.examples**. Anche tutte le risorse necessarie per gli Esempi di codice verrano copiate nella cartella **src/main/resources**, come mostrato di seguito:

![todo:image_alt_text](example_3.png)

1. Rivedere il codice di esempio, compilare ed eseguire.
1. Ora puoi testare altri esempi e iniziare a costruire la tua applicazione utilizzando l'API Aspose.Cells for Java.
