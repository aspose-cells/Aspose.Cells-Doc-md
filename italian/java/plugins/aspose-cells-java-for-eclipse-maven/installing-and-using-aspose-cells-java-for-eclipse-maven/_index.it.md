## **Installazione**

**Aspose.Cells Java per Eclipse (Maven)** plugin può essere facilmente installato dal file Available**Collegare** scheda nella finestra di dialogo Plugin.

- Per aprirlo, seleziona**Plugin**dal**Strumenti** menu in Eclipse.

![cose da fare:immagine_alt_testo](installing-and-using-aspose-cells-java-for-eclipse-maven_1)

-  Questo aggiunge il**Aspose.Cells Maven Progetto** nella procedura guidata Nuovo progetto e**Aspose.Cells Codice Esempio** in New Other procedura guidata di Eclipse IDE.

![cose da fare:immagine_alt_testo](project_1.png)

## **Usando**

### **Aspose.Cells Maven Progetto (procedura guidata)**

 Creare**Maven Progetto** dalla procedura guidata per l'utilizzo[Aspose.Cells for Java API](https://products.aspose.com/cells/java/):

1. Selezionare**Nuovo -> Progetto**.
1. Selezionare**Aspose.Cells Maven Progetto**nel**Maven**categoria.
1. Clic**Prossimo**.

![cose da fare:immagine_alt_testo](project_2.png)

1. Fornire**Nome progetto, Posizione, GroupId, ArtifactId**e**Versione**per il tuo progetto Maven e clicca**Fine.**

![cose da fare:immagine_alt_testo](project_3.png)

1. Questo recupererà il file[Aspose.Cells for Java](https://products.aspose.com/cells/java/) ultimo[Maven Dipendenza](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) riferimento da[Aspose Nuvola Maven Deposito](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo) e configurarlo in**pom.xml**. Se hai optato per**Scarica anche esempi di codice,** il download del**Esempi di codice** inizierà anche dal[Aspose.Cells for Java API Repository di esempi.](https://github.com/aspose-cells/Aspose.Cells-for-Java)
1. Seguente**Maven** Il progetto verrà creato sul tuo**Eclipse IDE** al termine della procedura guidata:

![cose da fare:immagine_alt_testo](project_4.png)

1.  Il creato**Maven Progetto** è configurato per l'uso**Aspose.Cells for Java API** e pronto per essere migliorato secondo i requisiti del tuo progetto.
Se hai scelto di scaricare[Esempi di codice](https://github.com/aspose-cells/Aspose.Cells-for-Java) , Puoi usare**Aspose.Cells Codice Esempio (procedura guidata)**per l'importazione necessaria**Esempi di codice** di Aspose.Cells for Java API nel tuo progetto.

### **Aspose.Cells Codice Esempio (procedura guidata)**

**Aspose.Cells Procedura guidata di esempio di codice** permette di provare molti campioni forniti per Aspose.Cells for Java API.

{{% alert color="primary" %}}

 Essere in grado di utilizzare**Aspose.Cells Procedura guidata di esempio di codice**comodamente: si consiglia di selezionare sempre**Scarica anche esempi di codice** durante la creazione**Maven Progetto**Su**Aspose.Cells Maven Progetto** **procedura guidata**,

{{% /alert %}}

Per usare esempi, basta:

1. Clic**Nuovo -> Altro** in**Eclisse**.
1. Selezionare**Aspose.Cells Codice Esempio**nel**Java**categoria.
1. Clic**Prossimo**.  

![cose da fare:immagine_alt_testo](example_1.png)

1.  Scegli il tuo progetto, quindi espandi l'albero per selezionarlo**Esempio di codice** categoria e fare clic**Fine**.

![cose da fare:immagine_alt_testo](example_2.png)

1.  Questo copierà la categoria selezionata**Esempi di codice**Java file nel progetto sotto**com.aspose.cells.examples**pacchetto. Verranno copiate anche tutte le risorse richieste dagli esempi di codice**src/principale/risorse**cartella, come mostrato di seguito:

![cose da fare:immagine_alt_testo](example_3.png)

1. Rivedere il codice di esempio, compilare ed eseguire.
1. Ora puoi testare altri esempi e iniziare a creare la tua applicazione utilizzando Aspose.Cells for Java API.
