## **introduzione**
### **Eclipse IDE**
Eclipse IDE è un famoso ambiente di sviluppo integrato (IDE) Java. L'IDE è sicuramente il prodotto più conosciuto del progetto Eclipse Open Source. Oggi è l'ambiente di sviluppo leader for Java con una quota di mercato di circa il 60%.

L'IDE Eclipse può essere esteso con componenti software aggiuntivi. Eclipse chiama questi componenti software Plug-in. Diversi progetti e aziende Open Source hanno esteso l'IDE Eclipse o creato applicazioni autonome (Eclipse RCP) in aggiunta al framework Eclipse.
### **Aspose.Cells for Java**
[Aspose.Cells for Java ](http://goo.gl/gtvhlQ)is an award-winning Excel Spreadsheet component that allows Java developers to embed the ability to read, write and manipulate Excel® spreadsheets (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF and image formati di file nelle proprie applicazioni Java senza dover fare affidamento su Microsoft Excel®.

[Aspose.Cells for Java](http://goo.gl/gtvhlQ)è un componente maturo, scalabile e ricco di funzionalità che offre molte funzioni che vanno ben oltre le semplici capacità di esportazione dei dati di altri fornitori. Con Aspose.Cells for Java gli sviluppatori possono esportare dati, formattare fogli di calcolo al livello più granulare, importare immagini, creare grafici, applicare e calcolare formule complesse, eseguire lo streaming di dati Excel®, salvare in vari formati e molto altro - tutto questo senza bisogno di Microsoft Excel® o Microsoft Domotica.
### **Aspose.Cells Java per Eclipse (Maven)**
-  Aspose.Cells Java per Eclipse (Maven) è un plug-in per**Eclipse IDE** di**Aspose.** Questo plugin è destinato agli sviluppatori che utilizzano gli sviluppi della piattaforma Maven for Java e desiderano utilizzare[Aspose.Cells for Java](http://goo.gl/gtvhlQ) nei loro progetti. Il plugin ti consente di creare maven progetti da utilizzare[Aspose.Cells for Java API](http://goo.gl/gtvhlQ) e anche scaricare[Esempi di codice](https://goo.gl/uxznrb)dello API.
-  Il plug-in fornisce le seguenti funzionalità con cui lavorare[Aspose.Cells for Java API](http://goo.gl/gtvhlQ) entro**Eclipse IDE** comodamente:

**MAGO**:
Il plugin contiene due procedure guidate -----

**       ****Aspose.Cells Maven Progetto (procedura guidata)**

- Questa procedura guidata Nuovo progetto consente agli sviluppatori di creare**Maven** progetto per l'utilizzo[Aspose.Cells for Java](http://goo.gl/gtvhlQ) a partire dal**Nuovo -> Progetto -> Maven-> Aspose.Cells Maven Progetto.**
- Il riferimento di[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)maven la dipendenza viene recuperata automaticamente da[Aspose Nuvola Maven Deposito](https://repository.aspose.com/webapp/)e viene aggiunto in pom.xml.
-  Il progetto creato conterrà sempre la versione disponibile più recente**Maven** Dipendenza per[Aspose.Cells for Java API](http://goo.gl/gtvhlQ).
-  I passaggi della procedura guidata presentano anche l'opzione per il download[Esempi di codice](https://goo.gl/uxznrb)per usare[Aspose.Cells for Java API](http://goo.gl/gtvhlQ).

**       ****Aspose.Cells Codice Esempio (procedura guidata)**

-  Questa procedura guidata Nuovo file consente di copiare il file scaricato[Esempi di codice](https://goo.gl/uxznrb) nel tuo progetto per l'utilizzo[Aspose.Cells for Java](http://goo.gl/gtvhlQ) a partire dal**Nuovo -> Altro -> Java -> Aspose.Cells Codice Esempio.** 
- I campioni disponibili vengono visualizzati in un formato ad albero da cui l'utente può selezionarli in modo categorico.
- Tutti gli esempi all'interno della categoria selezionata verranno copiati nella "**com.aspose.cells.examples**" cartella del pacchetto insieme alle risorse richieste all'interno di "**src/principale/risorse**" cartella necessaria per eseguire gli esempi.
- [Esempi di codice](https://goo.gl/uxznrb) di[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)sono pensati per dimostrare i vari del API.
- La procedura guidata cercherà anche gli aggiornamenti per i nuovi disponibili[Esempi di codice](https://goo.gl/uxznrb)partire dal[Aspose.Cells for Java repository di esempi.](https://goo.gl/uxznrb)

## **Download**
### **Scarica Eclipse IDE**
Dovrai prima installare Eclipse IDE prima di scaricare il plug-in Aspose.Cells Java per Eclipse (Maven).

Per scaricare Eclipse IDE

1.  Vai a[https://eclipse.org](https://eclipse.org/).
1. Scarica e installa gli sviluppatori Eclipse IDE for Java SE / EE consigliati.
### **Scarica Aspose.Cells Java per Eclipse (Maven)**
Di seguito sono riportati tre metodi consigliati per il corretto download e installazione del plug-in Aspose.Cells Java per Eclipse (Maven):

-  Trascina e rilascia l'installazione da[Mercato Eclipse](https://marketplace.eclipse.org/content/asposecells-java-maven-eclipse) nell'area di lavoro di Eclipse.
-  Oppure vai a**Aiuto** > **Installa nuovo software...**> Inserisci il seguente URL del sito di aggiornamento in**Lavorare con**:
 cdn.aspose.com/marketplace/eclipse/java/maven/asposecells
 Quindi selezionare "Aspose.Cells Java per Eclipse (Maven)" e**Fine**. Accetta il contratto di licenza e installa il plug-in.

### **Come richiedere la licenza Aspose?**
 Questo plugin utilizza una versione di valutazione di Aspose.Cells. Una volta che sei soddisfatto della tua valutazione, puoi acquistare una licenza al[Aspose sito web](https://purchase.aspose.com/buy).
Per rimuovere il messaggio di valutazione e le limitazioni delle funzionalità, è necessario applicare la licenza del prodotto. Riceverai un file di licenza dopo aver acquistato il prodotto. Si prega di seguire i passaggi seguenti per applicare la licenza

-  Assicurati che il file di licenza sia denominato come**Aspose.Cells.Java.lic**
-  Posto**Aspose.Cells.Java.lic** file nella cartella che contiene Aspose.Cells.jar
- Utilizzare il seguente codice per l'attivazione della licenza:

{{< highlight "java" >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **Come configurare il codice sorgente di Aspose.Cells Java per Eclipse (Maven)**
Di seguito semplici passaggi porteranno senza problemi alla corretta configurazione di**"Aspose.Cells Java per Eclipse (Maven)"** codice sorgente del plug-in nell'IDE di Eclipse

1. Scarica / clona il codice sorgente.
1. Scegliere**File** > **Importa > Generale > Progetti esistenti in Workspace**
1. Sfoglia fino all'ultima fonte del progetto che hai scaricato
1. Seleziona il progetto Eclipse che desideri importare
1. Clic**Fine**
1. Aspose.Cells Java per il codice del plug-in Eclipse è ora pronto per essere migliorato.
