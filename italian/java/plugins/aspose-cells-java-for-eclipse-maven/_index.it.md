## **Introduzione**
### **Eclipse IDE**
L'IDE Eclipse è un famoso ambiente di sviluppo integrato (IDE) per Java. L'IDE è sicuramente il prodotto più conosciuto del progetto Open Source Eclipse. Oggi è l'ambiente di sviluppo leader per Java con una quota di mercato di circa il 60%.

L'IDE Eclipse può essere esteso con componenti software aggiuntivi. Eclipse chiama questi componenti software Plug-in. Diversi progetti open source e aziende hanno esteso l'IDE Eclipse o creato applicazioni standalone (Eclipse RCP) sopra il framework Eclipse.
### **Aspose.Cells for Java**
[Aspose.Cells for Java ](http://goo.gl/gtvhlQ)è un componente premiato per i fogli di calcolo Excel che consente ai programmatori Java di incorporare la capacità di leggere, scrivere e manipolare fogli di calcolo Excel® (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF e formati di file immagine nelle proprie applicazioni Java senza dover fare affidamento su Microsoft Excel®.

[Aspose.Cells for Java](http://goo.gl/gtvhlQ) è un componente maturo, scalabile e ricco di funzionalità che offre molte funzioni che vanno ben oltre le semplici capacità di esportazione di dati di altri fornitori. Con Aspose.Cells for Java i programmatori possono esportare dati, formattare fogli di calcolo al livello più granulare, importare immagini, creare grafici, applicare e calcolare formule complesse, trasmettere i dati di Excel®, salvare in vari formati e molto altro ancora - tutto questo senza la necessità di Microsoft Excel® o dell'automazione di Microsoft Office.
### **Aspose.Cells Java for Eclipse (Maven)**
- Aspose.Cells Java for Eclipse (Maven) è un Plugin per **Eclipse IDE** di **Aspose.** Questo Plugin è destinato ai programmatori che utilizzano la piattaforma Maven per lo sviluppo Java e desiderano utilizzare [Aspose.Cells for Java](http://goo.gl/gtvhlQ) nei loro progetti. Il Plugin consente di creare progetti Maven per utilizzare [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) e scaricare anche [Esempi di Codice](https://goo.gl/uxznrb) dell'API. 
- Il plugin fornisce le seguenti funzionalità per lavorare con [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) all'interno di **Eclipse IDE** in modo confortevole:

**WIZARDS**:
Il plugin contiene due wizard:

**Aspose.Cells Maven Project (wizard)**

- Questo wizard di Nuovo Progetto consente agli sviluppatori di creare un progetto **Maven** per utilizzare [Aspose.Cells for Java](http://goo.gl/gtvhlQ) da **Nuovo -> Progetto -> Maven -> Aspose.Cells Maven Project.**
- Il riferimento alla dipendenza maven di [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) viene automaticamente recuperato da [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/) e aggiunto nel pom.xml.
- Il progetto creato conterrà sempre la versione più recente disponibile della dipendenza **Maven** per [Aspose.Cells for Java API](http://goo.gl/gtvhlQ).
- I passaggi del wizard presentano anche l'opzione per scaricare gli [Esempi di Codice](https://goo.gl/uxznrb) per utilizzare [Aspose.Cells for Java API](http://goo.gl/gtvhlQ).

**Aspose.Cells Code Example (wizard)**

- Questo wizard di Nuovo File ti permette di copiare gli [Esempi di Codice](https://goo.gl/uxznrb) scaricati nel tuo progetto per utilizzare [Aspose.Cells for Java](http://goo.gl/gtvhlQ) da **Nuovo -> Altro -> Java -> Aspose.Cells Code Example.** 
- Gli esempi disponibili sono visualizzati in formato albero da cui l'utente può selezionarli per categoria.
- Tutti gli esempi all'interno della categoria selezionata verranno copiati nella cartella del pacchetto "**com.aspose.cells.examples**" del progetto insieme alle risorse necessarie nella cartella "**src/main/resources**" per eseguire gli esempi.
- Gli [Esempi di Codice](https://goo.gl/uxznrb) di [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) servono a mostrare le varie funzionalità dell'API.
- Il wizard cercherà anche gli aggiornamenti per i nuovi [Esempi di Codice](https://goo.gl/uxznrb) disponibili dal [repositorio degli esempi Aspose.Cells for Java](https://goo.gl/uxznrb).

### **Come applicare la licenza Aspose?**
Questo plugin utilizza una versione di valutazione di Aspose.Cells. Una volta soddisfatto della tua valutazione, puoi acquistare una licenza sul [sito web di Aspose](https://purchase.aspose.com/buy).
Per rimuovere il messaggio di valutazione e le limitazioni delle funzionalità, la licenza del prodotto deve essere applicata. Riceverai un file di licenza dopo aver acquistato il prodotto. Segui i passaggi seguenti per applicare la licenza

- Assicurati che il file di licenza sia chiamato **Aspose.Cells.Java.lic**
- Posiziona il file **Aspose.Cells.Java.lic** nella cartella che contiene Aspose.Cells.jar
- Utilizza il codice seguente per attivare la licenza:

{{< highlight java >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **Come configurare il codice sorgente di Aspose.Cells Java per Eclipse (Maven)**
Di seguito i semplici passaggi che condurranno senza intoppi alla corretta configurazione del plugin **"Aspose.Cells Java per Eclipse (Maven)"** nell'IDE Eclipse

1. Scaricare / Clonare il codice sorgente.
1. Scegliere **File** > **Import > General > Existing Projects into Workspace**
1. Navigare fino all'ultimo progetto sorgente scaricato
1. Selezionare il progetto Eclipse da importare
1. Fare clic su **Fine**
1. Il codice del plugin Aspose.Cells Java per Eclipse è ora pronto per essere migliorato.
