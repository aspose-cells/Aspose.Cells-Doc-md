## **introduzione**

### **IDEA IntelliJ**

**IDEA IntelliJ** è un ambiente di sviluppo integrato (IDE) Java per gli sviluppatori di software. È sviluppato da JetBrains. Originariamente creato for Java, attualmente supporta un'ampia varietà di linguaggi, inclusi JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure e Kotlin.

 In un 2010**Infomondo** rapporto, IntelliJ ha ricevuto il punteggio più alto del centro di test tra i quattro principali strumenti di programmazione Java: Eclipse, IntelliJ IDEA, NetBeans e Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) for Java è un componente di reporting del foglio di calcolo Excel® che consente alle applicazioni Java di creare e gestire fogli di calcolo Excel® senza utilizzare Microsoft Excel®. Aspose.Cells for Java è un componente incredibilmente ricco di funzionalità che offre molto di più delle semplici funzioni di esportazione dei dati di base.

### **Aspose.Cells Java per IntelliJ IDEA-Maven**

- Aspose.Cells Java per IntelliJ IDEA (Maven) è un**Plug-in IntelliJ IDEA** che consente di creare progetti maven (ovvero il progetto creato contiene la dipendenza Aspose.Cells for Java API maven) e scaricare esempi di codice sorgente di API per l'utilizzo**Aspose.Cells for Java API**.

-  Questo plugin ti aiuta a lavorare con**Aspose.Cells for Java API** all'interno della tua IDEA IntelliJ preferita.

**MAGO**:
Il plugin contiene due procedure guidate -----

**Mago 1:** Aspose.Cells Maven Progetto

-  Plugin**Aspose Maven Procedura guidata di progetto** consente agli sviluppatori di creare il progetto Aspose.Cells for Java Maven.
-  Il riferimento di**Aspose.Cells for Java API** maven la dipendenza viene recuperata automaticamente da[Aspose Nuvola Maven Deposito](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) e viene aggiunto in pom.xml.
- Il progetto creato conterrà sempre la versione più recente disponibile Maven Dipendenza per Aspose.Cells for Java.
-  I passaggi della procedura guidata sono presenti anche con l'opzione per il download[Codice sorgente Esempi per l'utilizzo di Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

**Mago 2:** Aspose.Cells Esempi

- **Aspose.Cells Procedura guidata esempi** ti consente di creare / copiare scaricato[Codice sorgente Esempi di Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) nel tuo progetto. Tutti gli esempi per la categoria selezionata verranno copiati nella cartella del pacchetto "com.aspose.cells.examples" del progetto insieme alle risorse richieste all'interno della cartella "src/main/resources" necessarie per eseguire gli esempi.

- [Codice sorgente Esempi di Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) hanno lo scopo di dimostrare i diversi usi dello API.

## **Requisiti di sistema e piattaforme supportate**

### **Requisiti di sistema**

- Microsoft Windows 8/7/Vista/2003/XP (incl.64-bit) / Linux GNOME o desktop KDE / Mac OS X 10.5 o superiore
- Almeno 1 GB di RAM, consigliati 2 GB di RAM
- 300 MB di spazio su disco rigido + almeno 1 G per le cache
- Risoluzione minima dello schermo 1024x768
- JDK 1.7 o superiore consigliato
- IntelliJ IDEA CE/UE (13.0.1, 13.1.2 o 14.0.2)
- Connessione Internet (consigliati 2 MB o più veloci)

### **Piattaforme supportate**

Il plug-in supporta le seguenti versioni di IntelliJ IDEA:

- 13.0.1 (Community/Ultimate Edition)
- 13.1.2 (CE/UE)
- 14.0.2 (CE/UE)

## **Download**

### **Download di IntelliJ IDEA**

Dovrai prima installare IntelliJ IDEA CE / UE prima di scaricare il file**Aspose.Cells Java per IntelliJ IDEA (Maven)** collegare.

Per scaricare IntellliJ IDEA

1.  Vai a<https://www.jetbrains.com/idea/download/>
1. Seleziona il tuo sistema operativo in esecuzione
1. Fare clic su download UE o CE.

### **Download di Aspose.Cells Java per il plug-in IntelliJ IDEA (Maven)**

-  Il plugin può essere scaricato direttamente dal[URL di hosting del plug-in JetBrains](https://goo.gl/R4pysl) - su IntelliJ IDEA Plugin Repository
 e versioni successive possono essere installate utilizzando l'opzione IDE:
 - Vai a**File** > **Impostazioni**...>**Plugin**
 - Selezionare "**Installa plug-in da disco"**
- Oppure può essere installato direttamente (senza download manuale dall'URL di hosting del plug-in JetBrains) utilizzando le opzioni IDE:
 - Vai a**File** > **Impostazioni**...>**Plugin**
 - Selezionare**"Sfoglia repository"**
 - Nella casella di ricerca digitare: "Aspose"
 - Selezionare "**Aspose Cells Java per IntelliJ IDEA Maven**" plug-in da installare.
