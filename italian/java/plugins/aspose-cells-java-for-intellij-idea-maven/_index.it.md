## **Introduzione**

### **IntelliJ IDEA**

**IntelliJ IDEA** è un ambiente di sviluppo integrato (IDE) per sviluppatori software. È sviluppato da JetBrains. Creato originariamente per Java, attualmente supporta una vasta varietà di linguaggi, tra cui JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure e Kotlin.

In un rapporto del 2010 di **Infoworld**, IntelliJ ha ricevuto il punteggio più alto nel centro di test dei quattro migliori strumenti di programmazione Java: Eclipse, IntelliJ IDEA, NetBeans e Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) per Java è un componente di reporting di fogli elettronici Excel® che consente alle applicazioni Java di creare e gestire fogli elettronici Excel® senza utilizzare Microsoft Excel®. Aspose.Cells for Java è un componente incredibilmente ricco di funzionalità che offre molto più che semplici funzioni di esportazione dei dati.

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java per IntelliJ IDEA (Maven) è un **Plugin IntelliJ IDEA** che consente di creare progetti Maven (ossia il progetto creato contiene la dipendenza Maven API Aspose.Cells for Java) e scaricare esempi di codice sorgente dell'API per utilizzare **Aspose.Cells for Java API**.

- Questo plugin ti aiuta a lavorare con **Aspose.Cells for Java API** nel tuo amato IntelliJ IDEA.

**WIZARDS**:
Il plugin contiene due wizard -----

**Wizard 1:** Progetto Maven Aspose.Cells

- Il **Aspose Maven Project Wizard** del plugin consente ai developer di creare un progetto Maven Aspose.Cells for Java.
- Il riferimento della dipendenza maven di **Aspose.Cells for Java API** viene automaticamente recuperato da [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) e viene aggiunto nel pom.xml.
- Il progetto creato conterrà sempre la versione più recente della dipendenza Maven disponibile per Aspose.Cells for Java.
- I passaggi del wizard presentano anche l'opzione per scaricare [Esempi di codice sorgente per utilizzare l'API Aspose.Cells for Java](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

**Wizard 2:** Esempi di Aspose.Cells

- Il **Aspose.Cells Examples Wizard** ti permette di creare/copiare gli [Esempi di codice sorgente dell'API Aspose.Cells for Java](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) scaricati nel tuo progetto. Tutti gli esempi della categoria selezionata verranno copiati nella cartella del pacchetto del progetto "com.aspose.cells.examples" insieme alle risorse necessarie nella cartella "src/main/resources" per eseguire gli esempi.

- Gli [Esempi di codice sorgente dell'API Aspose.Cells for Java](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) sono destinati a dimostrare le diverse modalità di utilizzo dell'API.

## **Requisiti di sistema e piattaforme supportate**

### **Requisiti di sistema**

- Microsoft Windows 8/7/Vista/2003/XP (incl.64-bit) / desktop Linux GNOME o KDE / Mac OS X 10.5 o superiore
- 1 GB di RAM minimo, 2 GB di RAM consigliati
- 300 MB di spazio su disco rigido + almeno 1 GB per le cache
- Risoluzione dello schermo minima di 1024x768
- JDK 1.7 o superiore consigliato
- IntelliJ IDEA CE/UE (13.0.1, 13.1.2 o 14.0.2)
- Connessione Internet (si consiglia 2 MB o più veloce)

### **Piattaforme Supportate**

Il plugin supporta le seguenti versioni di IntelliJ IDEA:

- 13.0.1 (Community / Ultimate Edition)
- 13.1.2  (CE / UE)
- 14.0.2 (CE/ UE)

## **Download**

### **Scaricando IntelliJ IDEA**

Prima di scaricare il plugin **Aspose.Cells Java per IntelliJ IDEA (Maven)**, sarà necessario installare IntelliJ IDEA CE/UE.

Per scaricare IntellliJ IDEA

1. Go to <https://www.jetbrains.com/idea/download/>
1. Seleziona il tuo sistema operativo in esecuzione
1. Fai clic su download UE o CE.

### **Scaricando il plugin Aspose.Cells Java per IntelliJ IDEA (Maven)**

- Il plugin può essere scaricato direttamente dall'URL di hosting del plugin JetBrains - su IntelliJ IDEA Plugin Repository
  e successivamente può essere installato utilizzando l'opzione IDE:
  - Vai a **File** > **Impostazioni**...>**Plugin**
  - Seleziona "**Installa plugin da disco"**
- Oppure può essere installato direttamente (senza scaricare manualmente dall'URL di hosting del plugin JetBrains) utilizzando le opzioni IDE:
  - Vai a **File** > **Impostazioni**...>**Plugin**
  - Seleziona **"Sfoglia repository"**
  - Nella casella di ricerca digita: "Aspose"
  - Seleziona "**Aspose Cells Java for IntelliJ IDEA Maven**" per installare il plugin.
