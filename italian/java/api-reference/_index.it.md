Aspose.Cells for Java è una libreria di classi che consente alle applicazioni di eseguire una vasta gamma di attività di elaborazione di fogli di calcolo. Aspose.Cells for Java supporta i più diffusi fogli di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) e consente l'esportazione di file Excel in PDF, XPS, HTML, MHTML, testo normale, TIFF, JPG, PNG, BMP e SVG.

L'ultimo riferimento API è disponibile all'indirizzo[Aspose.Cells for Java API Riferimento](https://reference.aspose.com/cells/java).
