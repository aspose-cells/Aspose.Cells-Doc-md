Aspose.Cells for Java è una libreria di classi che consente alle applicazioni di eseguire una vasta gamma di attività di elaborazione di fogli di calcolo. Aspose.Cells for Java supporta i formati di file di fogli di calcolo più diffusi (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) e consente di esportare file Excel in PDF, XPS, HTML, MHTML, testo normale, TIFF, JPG, PNG, BMP e SVG.

L'ultimo riferimento API è disponibile all'indirizzo[Aspose.Cells for Java Riferimento API](https://reference.aspose.com/cells/java).
