{{% alert color="primary" %}} 

È possibile eliminare tutte le righe e colonne vuote da un foglio di lavoro. Questo è utile, ad esempio, quando si genera un file PDF da un file Microsoft Excel e si desidera convertire solo le righe e le colonne che contengono dati.

Utilizzare i seguenti metodi Aspose.Cells per eliminare le righe e le colonne vuote:

1. Per eliminare righe vuote, utilizzare il metodo [Cells.deleteBlankRows()](https://reference.aspose.com/cells/java/com.aspose.cells/cells#deleteBlankRows--)
1. Per eliminare colonne vuote, utilizzare il metodo [Cells.deleteBlankColumns()](https://reference.aspose.com/cells/java/com.aspose.cells/cells#deleteBlankColumns--)
