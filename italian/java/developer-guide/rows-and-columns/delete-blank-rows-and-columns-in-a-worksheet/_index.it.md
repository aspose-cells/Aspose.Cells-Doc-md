{{% alert color="primary" %}} 

È possibile eliminare tutte le righe e le colonne vuote da un foglio di lavoro. Ciò è utile, ad esempio, quando si genera un file PDF da un file Microsoft Excel e si desidera convertire solo righe e colonne che contengono dati.

Utilizzare i seguenti metodi Aspose.Cells per eliminare righe e colonne vuote:

1.  Per eliminare le righe vuote, utilizzare il[Cells.deleteBlankRows()](https://reference.aspose.com/cells/java/com.aspose.cells/cells#deleteBlankRows\(\)) metodo.
1.  Per eliminare le colonne vuote, utilizzare il[Cells.deleteBlankColumns()](https://reference.aspose.com/cells/java/com.aspose.cells/cells#deleteBlankColumns\(\)) metodo.
