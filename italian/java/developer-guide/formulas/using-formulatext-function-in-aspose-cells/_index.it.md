*FormulaTesto* è una funzione di Excel 2013 e versioni successive. Non è supportato da versioni precedenti come Excel 2010 o 2007 ecc. Come suggerisce il nome, stampa il testo della formula presente in una determinata cella. Questo articolo ti mostrerà come utilizzare questa funzione utilizzando Aspose.Cells.
