{{% alert color="primary" %}}

Gli utenti di Microsoft Excel possono aggiungere commenti alle celle nei fogli di calcolo per aggiungere ulteriori informazioni alle celle ed evidenziare i dati. Gli sviluppatori potrebbero dover personalizzare il commento per specificare le impostazioni di allineamento e la direzione del testo. Aspose.Cells fornisce API per eseguire l'attività.

{{% /alert %}}

 Aspose.Cells fornisce a[**Shape.setTextDirection()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#TextDirection)metodo per impostare la direzione del testo desiderata per un commento.
