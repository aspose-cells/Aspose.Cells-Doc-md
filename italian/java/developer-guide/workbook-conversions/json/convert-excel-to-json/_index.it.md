{{% /alert %}}

## **Come convertire un Workbook Excel in JSON**

Non c'è bisogno di chiedersi come convertire un Workbook Excel in JSON, perché la libreria Aspose.Cells Java ha la migliore soluzione. L'API Java Aspose.Cells fornisce il supporto per la conversione dei fogli di calcolo in formato JSON. Per esportare il workbook in JSON, passa [**SaveFormat.JSON**](https://reference.aspose.com/cells/java/com.aspose.cells/SaveFormat) come secondo parametro del metodo [**Workbook.save**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save-java.lang.String-int-). È anche possibile utilizzare la classe [**JsonSaveOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonSaveOptions) per specificare impostazioni aggiuntive per esportare il foglio di lavoro in JSON.

L'esempio di codice seguente mostra l'esportazione del foglio di lavoro di Excel in JSON. Si prega di consultare il codice per convertire il [file di origine](sample.xlsx) nel file JSON generato dal codice per riferimento.
