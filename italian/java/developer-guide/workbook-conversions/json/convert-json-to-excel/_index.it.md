Aspose.Cells supporta la conversione di un file Json (JavaScript Object Notation) in un Excel Workbook.

{{% /alert %}}

## **Converti JSON in Excel Workbook**

Non c'è bisogno di chiedersi come convertire JSON in file Excel, perché la libreria Aspose.Cells Java ha la soluzione migliore. L'API Aspose.Cells Java fornisce il supporto per la conversione del formato JSON in fogli di calcolo. Puoi utilizzare la classe [**JsonLoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLoadOptions) per specificare le impostazioni aggiuntive per l'importazione del JSON nel Workbook.
