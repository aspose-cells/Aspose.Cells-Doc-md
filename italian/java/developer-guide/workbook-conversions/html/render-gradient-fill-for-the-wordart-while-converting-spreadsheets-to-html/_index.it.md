## **Possibili scenari di utilizzo**
In precedenza, le API Aspose.Cells non erano in grado di eseguire il rendering del riempimento sfumato delle forme WordArt quando i fogli di calcolo venivano convertiti nel formato HTML. Ora le API Aspose.Cells supportano il rendering del riempimento sfumato delle forme WordArt.
