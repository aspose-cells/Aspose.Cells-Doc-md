## **Possibili Scenari di Utilizzo**

Aspose.Cells consente di prefissare gli stili degli elementi della tabella con la proprietà [**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId). Supponiamo di impostare questa proprietà con un qualche valore come *MyTest_TableCssId*, allora troverai gli stili degli elementi della tabella come mostrato di seguito

{{< highlight java >}}

table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}
