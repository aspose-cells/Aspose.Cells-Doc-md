{{% alert color="primary" %}} 

 Normalmente, il layout dei tag div viene ignorato durante il caricamento di HTML in un oggetto cartella di lavoro Excel. Tuttavia, se desideri che il layout dei tag div non venga ignorato, imposta il file[HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag) proprietà a**VERO** . Il valore predefinito di questa proprietà è**falso**.
