{{% alert color="primary" %}} 

Normalmente, il layout dei tag div viene ignorato durante il caricamento di HTML in un oggetto del foglio di calcolo di Excel. Tuttavia, se si desidera che il layout dei tag div non venga ignorato, impostare la proprietà [HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag) su **true**. Il valore predefinito di questa proprietà è **falso**.
