## **Converti Excel in HTML con intestazioni**

Aspose.Cells fornisce l'opzione di esportare le intestazioni durante la conversione di Excel in HTML. Per fare ciò, è possibile impostare la proprietà [**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings) su **true**. Il valore predefinito della proprietà [**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings) è **false**.
