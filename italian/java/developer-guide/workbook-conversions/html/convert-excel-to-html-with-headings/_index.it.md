## **Converti Excel in HTML con intestazioni**

Aspose.Cells offre la possibilità di esportare le intestazioni durante la conversione di Excel in HTML. Per questo, puoi impostare il file[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings)proprietà a**VERO**. Il valore predefinito di[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings)la proprietà è**falso**.
