{{% alert color="primary" %}}

Aspose.Cells fornisce una proprietà utile, [**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference), che gli sviluppatori possono utilizzare per ottenere una migliore disposizione quando si salvano un file Microsoft Excel in formato HTML o MHT. Il valore predefinito della proprietà è **false**. Consigliamo di impostare questa proprietà su **true** per ottenere presentazioni più belle da un report Excel.

{{% /alert %}}
