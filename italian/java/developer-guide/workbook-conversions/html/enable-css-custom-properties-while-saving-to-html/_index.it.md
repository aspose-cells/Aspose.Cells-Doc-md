## **Possibili Scenari di Utilizzo**

Quando salvi il tuo file Excel in HTML, per lo scenario in cui ci sono più occorrenze di un'immagine base64, con proprietà personalizzate i dati dell'immagine devono essere salvati una sola volta, migliorando così le prestazioni dell'html risultante. Si prega di usare [**HtmlSaveOptions.EnableCssCustomProperties**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions/properties/#EnableCssCustomProperties) proprietà e impostarla su **true** durante il salvataggio in HTML.
