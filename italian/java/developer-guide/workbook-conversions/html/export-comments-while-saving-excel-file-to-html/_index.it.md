## **Possibili Scenari di Utilizzo**

Quando si salva il file Excel in HTML, i commenti non vengono esportati. Tuttavia, Aspose.Cells fornisce questa funzionalità utilizzando la proprietà [**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#IsExportComments). Se la imposti su **true**, allora l'HTML visualizzerà anche i commenti presenti nel tuo file Excel.
