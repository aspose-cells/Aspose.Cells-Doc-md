## **Possibili scenari di utilizzo**

Quando salvi il tuo file Excel in HTML, i commenti non vengono esportati. Tuttavia, Aspose.Cells fornisce questa funzione utilizzando il[**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#IsExportComments)proprietà. Se lo imposti**VERO**quindi HTML visualizzerà anche i commenti presenti nel file Excel.
