Puoi esportare le formattazioni condizionali DataBar, ColorScale e IconSet durante la conversione del tuo file Excel in HTML. Questa funzionalità è supportata solo parzialmente da Microsoft Excel, ma Aspose.Cells la supporta completamente.

{{% /alert %}} 
## **Esportazione di formattazioni condizionali DataBar, ColorScale e IconSet durante la conversione da Excel in HTML**
Lo screenshot seguente mostra il [file Excel di esempio](5473447.xlsx) con formattazioni condizionali DataBar, ColorScale e IconSet. È possibile scaricare il [file Excel di esempio](5473447.xlsx) dal link fornito.

![todo:image_alt_text](conversion_1.png)

Lo screenshot seguente mostra il file HTML di output di Aspose.Cells che mostra le formattazioni condizionali DataBar, ColorScale e IconSet. Come puoi vedere, appare esattamente come il [file Excel di esempio](5473447.xlsx).
