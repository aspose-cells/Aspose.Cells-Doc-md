Puoi esportare la formattazione condizionale DataBar, ColorScale e IconSet durante la conversione del file Excel in HTML. Questa funzione è parzialmente supportata da Microsoft Excel ma Aspose.Cells la supporta completamente.

{{% /alert %}}
