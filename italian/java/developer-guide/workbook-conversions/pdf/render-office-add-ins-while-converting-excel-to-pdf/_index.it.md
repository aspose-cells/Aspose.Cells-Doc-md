## **Possibili scenari di utilizzo**
In precedenza, Aspose.Cells non poteva eseguire il rendering dei componenti aggiuntivi di Office quando il file Excel veniva salvato nel formato PDF. Ora Aspose.Cells lo rende bene. Non è necessario utilizzare alcun metodo o proprietà speciale per eseguire il rendering dei componenti aggiuntivi di Office nell'output PDF. Basta salvare il file Excel nel formato PDF e verrà eseguito il rendering dei componenti aggiuntivi di Office.
## **Eseguire il rendering dei componenti aggiuntivi di Office durante la conversione di Excel in PDF**
Il codice di esempio seguente salva il file[esempio di file Excel](60489783.xlsx)che contiene i componenti aggiuntivi di Office. Si prega di consultare il[output PDF generato con la versione precedente ovvero 17.11](60489781.pdf)e il[output PDF generato con la versione più recente, ovvero 17.12 e successive](60489782.pdf)L'output della versione precedente PDF è vuoto, ma l'output della versione più recente PDF mostra il componente aggiuntivo di Office.
