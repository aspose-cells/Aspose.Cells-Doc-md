## **Possibili scenari di utilizzo**
In precedenza, Aspose.Cells non poteva eseguire il rendering dei componenti aggiuntivi di Office quando il file Excel veniva salvato in formato PDF. Ora Aspose.Cells lo rende bene. Non è necessario utilizzare alcun metodo o proprietà speciale per eseguire il rendering dei componenti aggiuntivi di Office nel PDF di output. Basta salvare il file Excel in formato PDF e visualizzerà i componenti aggiuntivi di Office.
## **Renderizza i componenti aggiuntivi di Office durante la conversione di Excel in PDF**
Il codice di esempio seguente salva il file[esempio di file Excel](60489783.xlsx)che contiene i componenti aggiuntivi di Office. Si prega di consultare il[PDF di output generato con la versione precedente, ovvero 17.11](60489781.pdf)e il[PDF di output generato con la versione più recente, ovvero 17.12 e successive](60489782.pdf). Il PDF di output della versione precedente è vuoto, ma il PDF di output della versione più recente mostra il componente aggiuntivo di Office.
