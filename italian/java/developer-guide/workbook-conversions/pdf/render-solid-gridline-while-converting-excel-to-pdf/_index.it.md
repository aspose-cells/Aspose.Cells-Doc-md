Per la compatibilità con le versioni precedenti, Aspose.Cells rende la griglia come linea tratteggiata per impostazione predefinita durante la conversione di Excel in PDF. Tuttavia, Excel moderno rende la griglia come linea continua ai giorni nostri.

Con l'opzione [PdfSaveOptions.GridlineTypes](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setGridlineType-int-), Aspose.Cells può anche rappresentare la griglia come linea solida.
