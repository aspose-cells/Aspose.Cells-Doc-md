Per compatibilità con le versioni più vecchie, Aspose.Cells renderizza le linee di griglia come linee tratteggiate di default durante la conversione di Excel in PDF. Tuttavia, le versioni moderne di Excel rendono le linee di griglia come linee solide attualmente.

Con l'opzione [PdfSaveOptions.GridlineTypes](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setGridlineType-int-), Aspose.Cells può anche rendere le linee di griglia come linee solide.
