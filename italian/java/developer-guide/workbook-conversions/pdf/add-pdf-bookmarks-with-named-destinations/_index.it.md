## **Possibili scenari di utilizzo**

Le destinazioni con nome sono tipi speciali di segnalibri o collegamenti in PDF che non dipendono dalle pagine PDF. Significa che se le pagine vengono aggiunte o eliminate dal PDF, i segnalibri potrebbero diventare non validi ma le destinazioni denominate rimarranno intatte. Per creare Destinazione con nome, impostare il file[**PdfBookmarkEntry.DestinationName**](https://reference.aspose.com/cells/java/com.aspose.cells/pdfbookmarkentry#DestinationName)proprietà.
