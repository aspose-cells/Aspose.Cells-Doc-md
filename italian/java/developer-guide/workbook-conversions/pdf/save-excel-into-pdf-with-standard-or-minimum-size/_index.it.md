{{% alert color="primary" %}} 

 Per impostazione predefinita, Aspose.Cells salva Excel in PDF con dimensione Standard. Tuttavia puoi anche salvarlo con Dimensione minima usando il file[PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions#OptimizationType) proprietà. Accetta i seguenti valori

- [PdfOptimizationType.STANDARD](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#STANDARD)
- [PdfOptimizationType.MINIMUM_SIZE](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#MINIMUM_SIZE)
