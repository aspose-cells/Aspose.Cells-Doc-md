{{% alert color="primary" %}}

Aspose.Cells supporta la conversione di file XLS che contengono immagini e grafici in documenti PDF. Aspose.Cells for Java può lavorare indipendentemente per convertire un foglio di calcolo in PDF: le API di Aspose.PDF non sono necessarie per la conversione.

{{% /alert %}}

Il processo può essere eseguito in memoria poiché il processo non dipende da file XML temporanei o intermedi. Ciò significa che i file di Excel di grandi dimensioni, ad esempio, quelli contenenti immagini, grafici e altri oggetti disegnati, possono essere convertiti rapidamente ed efficientemente.

{{% alert color="primary" %}}

Se il foglio di calcolo contiene formule, è meglio chiamare il metodo [**Workbook.calculateFormula**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#calculateFormula--) subito prima di rendere il PDF. In questo modo si garantisce che i valori dipendenti dalle formule vengano ricalcolati e i valori corretti siano resi nel PDF.

{{% /alert %}}
