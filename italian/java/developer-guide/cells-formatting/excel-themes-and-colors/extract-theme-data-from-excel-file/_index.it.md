{{% alert color="primary" %}}

Aspose.Cells consente agli utenti di estrarre dati correlati al tema da un file Excel. Ad esempio, è possibile estrarre il Nome del tema applicato alla cartella di lavoro e il Colore del tema applicato alla cella o ai bordi della cella, ecc.

È possibile applicare un tema alla propria cartella di lavoro utilizzando Microsoft Excel tramite il comando Layout Pagina > Temi.
