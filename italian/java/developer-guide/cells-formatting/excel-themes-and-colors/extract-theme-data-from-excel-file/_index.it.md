{{% alert color="primary" %}}

Aspose.Cells consente agli utenti di estrarre i dati relativi al tema da un file Excel. Ad esempio, puoi estrarre il nome del tema applicato alla cartella di lavoro e il colore del tema applicato alla cella o ai bordi della cella, ecc.

È possibile applicare il tema alla cartella di lavoro utilizzando Microsoft Excel tramite il comando Layout di pagina > Temi.
