{{% alert color="primary" %}} 

**Scala a 2 colori** e**Scala a 3 colori** Le formattazioni condizionali vengono aggiunte allo stesso modo tranne che differiscono per[FormatCondition.ColorScale.Is3ColorScale](https://reference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale) proprietà. Questa proprietà è**falso** per scala a 2 colori e**VERO** per le formattazioni condizionali della scala a 3 colori.
