{{% alert color="primary" %}} 

Le formattazioni condizionali **2-Colori Scala** e **3-Colori Scala** sono aggiunte allo stesso modo, eccetto che sono differenziate dalla proprietà [FormatCondition.ColorScale.Is3ColorScale](https://reference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale). Questa proprietà è **false** per la formattazione condizionale a 2 colori e **true** per la formattazione condizionale a 3 colori.

{{% /alert %}}
