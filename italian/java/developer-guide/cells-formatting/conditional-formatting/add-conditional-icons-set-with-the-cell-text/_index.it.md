{{% alert color="primary" %}} 

A volte, vuoi aggiungere icone condizionali accanto al testo in una cella per rendere i dati più significativi per i lettori. Si desidera utilizzare alcuni dei tipi di icone di formattazione condizionale, ma senza applicare la formattazione condizionale alle celle. Aspose.Cells supporta la funzione.

{{% /alert %}} 

Di seguito è riportato un esempio che crea un file XLSX da zero, aggiungendo icone condizionali alle celle con testo senza applicare la formattazione condizionale.

Quando il codice viene eseguito, le immagini del set di icone condizionali vengono aggiunte nell'area della cella "B2:C4" come mostrato di seguito.
