## **Imposta il commento della tabella o dell'oggetto lista all'interno del foglio di lavoro**

Imposta il commento della Tabella o dell'Oggetto Elenco all'interno del foglio di calcolo utilizzando la proprietà [**ListObject.Comment**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment). Il commento sarà visibile all'interno del file xl/tables/tableName.xml.

Il codice di esempio seguente carica il [file excel di origine](5472561.xlsx), imposta il commento del primo oggetto Tabella o Elenco all'interno del foglio di calcolo e la schermata mostra il commento creato da questo codice nel rettangolo rosso.
