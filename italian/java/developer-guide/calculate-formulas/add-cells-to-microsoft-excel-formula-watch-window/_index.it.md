## **Possibili scenari di utilizzo**

Microsoft Excel Watch Window è uno strumento utile per guardare i valori delle celle e le sue formule comodamente in una finestra. Puoi aprire il*Guarda la finestra*utilizzando Microsoft Excel facendo clic su*Formule > Guarda* *Finestra*. Ha il*Aggiungi orologio*pulsante che può essere utilizzato per aggiungere le celle per l'ispezione. Allo stesso modo, puoi usare[**Foglio di lavoro.getCellWatches().add()**](https://reference.aspose.com/cells/java/com.aspose.cells/cellwatchcollection#add(int,%20int)) metodo per aggiungere celle in*Guarda la finestra*utilizzando Aspose.Cells API.
