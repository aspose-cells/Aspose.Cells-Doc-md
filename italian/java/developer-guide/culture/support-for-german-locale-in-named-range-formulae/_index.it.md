Le formule in inglese vengono scritte nella regione nominata. Questo file Excel può essere aperto in un ambiente in cui il sistema è configurato per la localizzazione tedesca, tuttavia le formule in inglese devono essere tradotte nella lingua tedesca. L'esempio seguente dimostra questa funzionalità, ma richiede che Excel sia installato in lingua tedesca e che la localizzazione del sistema sia impostata anche in tedesco.
