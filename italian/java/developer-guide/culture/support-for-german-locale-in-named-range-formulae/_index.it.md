Le formule inglesi sono scritte nella regione denominata. Questo file Excel può essere aperto in un ambiente in cui il sistema è configurato sulla lingua tedesca, tuttavia la formula inglese deve essere tradotta nella lingua tedesca. L'esempio seguente dimostra questa funzionalità, tuttavia, richiede che Excel sia installato in lingua tedesca e che anche le impostazioni locali del sistema siano impostate su tedesco.
