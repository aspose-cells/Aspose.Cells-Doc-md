{{% alert color="primary" %}} 

A volte potresti voler recuperare informazioni sulle fonti dati esterne. Ad esempio, potresti voler vedere i dati della connessione SQL. Queste informazioni possono includere qualsiasi tipo di dati necessari per connettersi a un server SQL, ad esempio l'URL del server, il nome utente, il nome della tabella sottostante, la query SQL, il tipo di query, la posizione della tabella, ecc.

Aspose.Cells fornisce alcune chiamate utili per recuperare tali dettagli dai database aziendali collegati ai file Excel di modello.

{{% /alert %}} 
## **Recupero dei dettagli delle fonti dati esterne**
L'esempio seguente mostra come ottenere la connessione al database e altri dettagli. L'esempio utilizza un semplice file Excel che contiene collegamenti a una fonte dati esterna (SQL Server).

Quando il codice viene eseguito, i dettagli della connessione vengono stampati sulla console.
