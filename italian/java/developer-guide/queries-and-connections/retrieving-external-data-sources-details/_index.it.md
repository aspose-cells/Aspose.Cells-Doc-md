volte, potresti voler recuperare informazioni su origini dati esterne. Ad esempio, potresti voler visualizzare i dati di connessione SQL. Queste informazioni possono includere qualsiasi tipo di dati necessari per connettersi a un server SQL, ad esempio l'URL del server, il nome utente, il nome della tabella sottostante, la query SQL, il tipo di query, la posizione della tabella e così via.

Aspose.Cells forniscono alcuni utili richiami per il recupero di tali dettagli dai database aziendali collegati ai file template Excel.
