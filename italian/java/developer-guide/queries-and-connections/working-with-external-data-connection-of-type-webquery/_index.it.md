{{% alert color="primary" %}} 

È possibile accedere alla connessione a dati esterni di qualsiasi tipo utilizzando la [Workbook.DataConnections](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections) raccolta. Un tipo di connessione dati di questo tipo è WebQuery. Questo articolo ti mostrerà come lavorare con la connessione dati WebQuery. Puoi creare una connessione dati WebQuery in Microsoft Excel utilizzando il **Dati** > **Da Web** menu.

{{% /alert %}}
