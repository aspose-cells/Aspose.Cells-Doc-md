{{% alert color="primary" %}} 

 È possibile accedere alla connessione dati esterna di qualsiasi tipo utilizzando il file[Cartella di lavoro.Connessioni dati](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections) collezione. Un tipo di tale connessione dati è WebQuery. Questo articolo ti mostrerà come lavorare con la connessione dati WebQuery. È possibile creare una connessione dati WebQuery in Microsoft Excel utilizzando il file**Dati** > **Dal web** menù.
