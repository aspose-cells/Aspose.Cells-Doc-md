## **Possibili scenari di utilizzo**

Quando aggiungi un nuovo foglio a una cartella di lavoro, contiene le impostazioni di Imposta pagina predefinite. Potrebbero esserci momenti in cui è necessario trasferire le impostazioni ([**Impostazione della pagina**](https://reference.aspose.com/cells/java/com.aspose.cells/PageSetup)da un foglio di lavoro a un altro foglio di lavoro. Questo documento spiega come copiare le impostazioni di Imposta pagina da un foglio di lavoro a un altro utilizzando le API Aspose.Cells.
