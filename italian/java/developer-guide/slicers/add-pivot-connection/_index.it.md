## **Possibili Scenari di Utilizzo**

Se vuoi associare slicer e tabella pivot in Excel, devi fare clic con il pulsante destro del mouse su slicer e selezionare l'elemento "Connessioni report...". Nell'elenco delle opzioni, puoi operare sulla casella di controllo. Allo stesso modo, se vuoi associare slicer e tabella pivot usando Aspose.Cells Java API in modo programmato, utilizza il metodo [**Slicer.addPivotConnection(PivotTable pivot)**](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#addPivotConnection-com.aspose.cells.PivotTable-).
