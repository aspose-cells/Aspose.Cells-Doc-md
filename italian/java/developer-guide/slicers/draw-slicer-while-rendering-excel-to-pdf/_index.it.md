## **Disegna lo slicer durante il rendering di Excel in PDF**
Se hai un file Excel a cui è stato applicato un slicer e vuoi esportare l'Excel in PDF con le impostazioni dello slicer, Aspose.Cells ora supporta questo per impostazione predefinita. Devi semplicemente esportare il file Excel con lo slicer in PDF, il PDF generato mostrerà lo slicer applicato.

Il seguente codice di esempio carica il [file Excel di esempio](SampleSlicerChart.xlsx) che contiene uno slicer esistente. Quindi salva il workbook come [output file PDF](SampleSlicerChart.pdf). La schermata seguente confronta il file Excel di origine e il file PDF generato.
