## **Disegna Slicer durante il rendering di Excel in PDF**
Se si dispone di un file Excel a cui è stato applicato un filtro dei dati e si desidera esportare Excel in PDF con le impostazioni del filtro dei dati, Aspose.Cells ora lo supporta per impostazione predefinita. Devi semplicemente esportare il file Excel con l'affettatrice in PDF, il PDF generato mostrerà l'affettatrice applicata.

Il codice di esempio seguente carica il file[esempio di file Excel](SampleSlicerChart.xlsx)che contiene un'affettatrice esistente. Quindi salva la cartella di lavoro come[file di output PDF](SampleSlicerChart.pdf). Lo screenshot seguente confronta il file Excel di origine e il file PDF generato.
