{{% alert color="primary" %}}

 È possibile specificare il nome del lavoro o del documento durante la stampa della cartella di lavoro o del foglio di lavoro utilizzando il file[**Workbook Render**](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookRender) o[**FoglioRendering**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender) oggetti. Aspose.Cells fornisce il[**WorkbookRender.toPrinter(nomestampante, nomelavoro)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ) e[**SheetRender.toPrinter(nomestampante, nomelavoro)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) metodi che è possibile utilizzare per specificare il nome del lavoro durante la stampa della cartella di lavoro o del foglio di lavoro

{{% /alert %}}

## **Specificare il nome del lavoro o del documento durante la stampa con Aspose.Cells**

 Il codice di esempio carica il file Excel di origine e quindi lo invia alla stampante specificando il nome del lavoro o del documento utilizzando l'estensione[**WorkbookRender.toPrinter(nomestampante, nomelavoro)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ) e[**SheetRender.toPrinter(nomestampante, nomelavoro)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)metodi. Lo screenshot mostra l'aspetto del nome del lavoro nella coda della stampante.
