Con la crescente diffusione degli schermi ad alta risoluzione, le immagini generate con il default 96 DPI spesso appaiono sfocate e non chiare. Per garantire la chiarezza sugli schermi ad alta risoluzione, è essenziale generare immagini a una risoluzione più elevata. Aspose.Cells offre la funzionalità di impostare [**ImageOrPrintOptions.HorizontalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setHorizontalResolution-int-) e [**ImageOrPrintOptions.VerticalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setVerticalResolution-int-), permettendoti di creare immagini di alta qualità da file Excel che si presentano nitide sui display ad alta risoluzione.
