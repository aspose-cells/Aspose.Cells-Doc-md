{{% alert color="primary" %}}

 Puoi utilizzare Aspose.Cells per esportare il foglio di lavoro o il grafico in un'immagine con la larghezza e l'altezza desiderate. Fornisce[**ImageOrPrintOptions.setDesiredSize()**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#setDesiredSize(int,%20int)) per impostare la larghezza e l'altezza desiderate dell'immagine esportata. La larghezza e l'altezza sono specificate nell'unità di pixel.

{{% /alert %}}
