## **Possibili Scenari di Utilizzo**

Generalmente si specifica la directory dei font o l'elenco dei font per tutti i fogli di lavoro, ma a volte è necessario specificare un insieme individuale o privato di font per i propri fogli di lavoro. Aspose.Cells fornisce la classe [**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs) che può essere utilizzata per specificare l'insieme individuale o privato di font per il proprio foglio di lavoro.
