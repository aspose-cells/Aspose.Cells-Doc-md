Aspose.Cells fornisce la funzione per sostituire il testo in una cartella di lavoro utilizzando un'espressione regolare. Per questo, lo API fornisce[**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)proprietà del[**SostituisciOpzioni**](https://reference.aspose.com/cells/java/com.aspose.cells/ReplaceOptions)classe. Impostazione del[**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)a**VERO**indica che la chiave cercata sarà un'espressione regolare.

Il seguente frammento di codice illustra l'uso di[**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)proprietà utilizzando il[file excel di esempio](SampleRegexReplace.xlsx). Il[file di uscita](RegexReplace_out.xlsx)generato dal seguente frammento di codice è allegato per riferimento.
