Aspose.Cells offre la possibilità di sostituire il testo in un libro di lavoro utilizzando un'espressione regolare. A tal fine, l'API fornisce la proprietà [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) della classe [**ReplaceOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/ReplaceOptions). Impostando [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) su **true** si indica che la chiave cercata sarà un'espressione regolare.

Il seguente snippet di codice dimostra l'utilizzo della proprietà [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) utilizzando il [file Excel di esempio](SampleRegexReplace.xlsx).
