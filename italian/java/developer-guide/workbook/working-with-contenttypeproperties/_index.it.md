Aspose.Cells fornisce il metodo [**Workbook.ContentTypeProperties.Add**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypepropertycollection#add-java.lang.Object-) per aggiungere custom ContentTypeProperties a un file Excel. È anche possibile rendere facoltativa la proprietà impostando la proprietà [**ContentTypeProperty.IsNillable**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypeproperty#IsNillable) su **true**. Il seguente snippet di codice dimostra l'aggiunta facoltativa di custom ContentTypeProperties a un file Excel.
