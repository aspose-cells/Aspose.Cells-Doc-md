{{% alert color="primary" %}}

 Puoi utilizzare Aspose.Cells per unire o dividere un intervallo di celle. Aspose.Cells fornisce il[**Intervallo.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge() ) e[**Intervallo.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge()) metodi a tal fine. Questo articolo spiega come unire un intervallo di celle in una singola cella.

{{% /alert %}}

 Il codice di esempio seguente crea innanzitutto un intervallo, A1:D4, quindi unisce le celle nell'intervallo in una singola cella utilizzando il[**Intervallo.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge()) metodo.
 Allo stesso modo, è possibile dividere le celle creando un intervallo e chiamando il metodo[**Intervallo.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge()) metodo.

L'immagine seguente mostra il file Excel di output generato con il codice di esempio. Come puoi vedere, l'intervallo A1:D4 è stato unito in una singola cella.
