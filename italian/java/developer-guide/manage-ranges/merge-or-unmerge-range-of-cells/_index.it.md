{{% alert color="primary" %}}

Puoi utilizzare Aspose.Cells per unire o dividere un intervallo di celle. Aspose.Cells fornisce i metodi [**Range.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge--) e [**Range.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge--) a questo scopo. Questo articolo spiega come unire un intervallo di celle in una singola cella.

{{% /alert %}}

Il seguente codice di esempio crea prima un intervallo - A1:D4 - e poi unisce le celle nell'intervallo in una singola cella utilizzando il metodo [**Range.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge--).
Allo stesso modo, è possibile dividere le celle creando un intervallo e chiamando il metodo [**Range.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge--).

La seguente immagine mostra il file Excel generato con il codice di esempio. Come puoi vedere, l'intervallo A1:D4 è stato unito in una singola cella.
