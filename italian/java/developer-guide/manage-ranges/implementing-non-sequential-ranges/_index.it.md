 Normalmente,[intervalli denominati](/cells/it/java/named-ranges/) sono rettangolari con celle continue e adiacenti l'una all'altra. Ma a volte, potrebbe essere necessario utilizzare un intervallo di celle non sequenziale in cui le celle non sono adiacenti. Aspose.Cells supporta la creazione di un intervallo denominato con celle non adiacenti. Per questo, l'API fornisce il[**Name.RefersTo**](https://reference.aspose.com/cells/java/com.aspose.cells/name#RefersTo) proprietà.
