{{% alert color="primary" %}}

In Microsoft Excel, puoi aggiungere un riferimento di libreria al progetto VBA cliccando manualmente su **Strumenti > Riferimenti...**. Si aprirà la seguente finestra di dialogo che ti aiuterà a selezionare riferimenti esistenti o a sfogliare la tua libreria.

![todo:image_alt_text](add-a-library-reference-to-vba-project-in-workbook_1.png)

Ma a volte è necessario aggiungere o registrare il riferimento di libreria al progetto VBA tramite codice. Puoi farlo utilizzando il metodo Aspose.Cells [**VbaProject.getReferences().addRegisteredReference()**](https://reference.aspose.com/cells/java/com.aspose.cells/vbaprojectreferencecollection#addRegisteredReference-java.lang.String-java.lang.String-).

{{% /alert %}}
