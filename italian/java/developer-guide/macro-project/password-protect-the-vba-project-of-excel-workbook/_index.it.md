## **Possibili scenari di utilizzo**
 È possibile proteggere con password il progetto VBA (Visual Basic for Applications) della cartella di lavoro con Aspose.Cells utilizzando[VbaProject.protect()](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#protect\(boolean,%20java.lang.String\)) metodo.
## **Codice di esempio**
 Il codice di esempio seguente carica il file[esempio di file Excel](43352069.xlsm), accede al suo progetto VBA e lo protegge con una password. Infine, lo salva come file[file Excel di output](43352070.xlsm).
