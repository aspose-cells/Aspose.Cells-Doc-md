{{% alert color="primary" %}} 

**Office OpenXML**(conosciuto anche come**OOXML**o**Microsoft Apri XML**(MOX)) è un formato di file basato su XML sviluppato da Microsoft per rappresentare documenti d'ufficio come fogli di calcolo, grafici, presentazioni e documenti di elaborazione testi.

{{% /alert %}} 



Aspose.Cells fornisce un modo per rilevare il formato del file crittografato**Microsoft Apri XML**File. Per identificare il tipo di file, utilizzare l'estensione[FileFormatUtil.detectFileFormat](https://reference.aspose.com/cells/java/com.aspose.cells/fileformatutil#detectFileFormat\(java.lang.String,%20java.lang.String\)) come mostrato di seguito nell'esempio di codice.
