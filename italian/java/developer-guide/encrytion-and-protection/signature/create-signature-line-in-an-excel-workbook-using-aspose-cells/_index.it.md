{{% alert color="primary" %}}

Microsoft Excel fornisce la funzionalità di aggiungere **Linea firma** in workbook Excel. È possibile aggiungere una linea di firma facendo clic sulla scheda **Inserisci** e quindi selezionando **Linea firma** dal gruppo **Testo**.

Aspose.Cells fornisce anche questa funzionalità ed ha esposto la proprietà [**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) a questo scopo. Questo articolo spiegherà come utilizzare questa proprietà per aggiungere una linea di firma utilizzando Aspose.Cells.

{{% /alert %}}
