{{% alert color="primary" %}}

 Microsoft Excel fornisce una funzionalità da aggiungere**Linea Firma** nelle cartelle di lavoro di Excel. È possibile aggiungere una riga della firma facendo clic su**Inserire** Tab e quindi selezionando**Linea Firma** dal**Testo** gruppo.

 Aspose.Cells fornisce anche questa funzione e ha esposto il[**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) proprietà a tale scopo. Questo articolo spiegherà come utilizzare questa proprietà per aggiungere una riga della firma utilizzando Aspose.Cells.

{{% /alert %}}
