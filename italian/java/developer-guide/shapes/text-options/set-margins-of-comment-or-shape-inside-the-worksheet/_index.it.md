## **Possibili scenari di utilizzo**

 Aspose.Cells consente di impostare i margini di qualsiasi forma o commento utilizzando il[**Shape.TextBody.TextAlignment**](https://reference.aspose.com/cells/java/com.aspose.cells/fontsettingcollection#TextAlignment) proprietà. Questa proprietà restituisce l'oggetto di[**ShapeTextAlignment**](https://reference.aspose.com/cells/java/com.aspose.cells/ShapeTextAlignment) classe che ha proprietà diverse, ad es[**TopMarginPt**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#TopMarginPt), [**Margine SinistroPt**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#LeftMarginPt), [**BottomMarginPt**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#BottomMarginPt), [**Margine destroPt**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#RightMarginPt), ecc. che possono essere utilizzati per impostare i margini superiore, sinistro, inferiore e destro.
