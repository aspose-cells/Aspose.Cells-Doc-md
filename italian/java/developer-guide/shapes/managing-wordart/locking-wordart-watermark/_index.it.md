Le API di Aspose.Cells consentono di aggiungere filigrane WordArt nel foglio di lavoro in modo che il WordArt diventi un oggetto che può essere spostato e posizionato nel foglio di lavoro. È inoltre possibile bloccare l'oggetto WordArt per qualsiasi interazione come editing, spostamento e selezione. Questo articolo spiega l'uso del metodo [**Shape.setLockedProperty**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#setLockedProperty-int-boolean-) per bloccare alcuni aspetti della filigrana.

{{% /alert %}}
