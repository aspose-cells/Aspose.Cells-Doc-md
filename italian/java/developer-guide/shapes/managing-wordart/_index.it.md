{{% alert color="primary" %}}

Utilizza WordArt per aggiungere effetti di testo speciali ai fogli di calcolo. Ad esempio, allunga un titolo nella parte superiore del file, decora il testo e adatta il testo a una forma preimpostata oppure applica il testo a un foglio Excel come filigrana di sfondo. La WordArt diventa un oggetto che puoi spostare o posizionare nei fogli di calcolo per aggiungere decorazioni.

{{% /alert %}}

##  **Aggiunta della filigrana WordArt al foglio di lavoro**

Nell'esempio seguente viene illustrato come aggiungere una forma WordArt per impostare una filigrana di sfondo per un foglio di lavoro.

Dopo aver eseguito il codice, il file di output contiene una filigrana WordArt rosso pallido.
