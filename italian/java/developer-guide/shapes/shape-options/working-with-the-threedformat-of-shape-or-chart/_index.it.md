## **Possibili Scenari di Utilizzo**
Aspose.Cells fornisce la proprietà [Shape.ThreeDFormat](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ThreeDFormat) per lavorare con il Formato 3D della forma o del grafico. Contiene diverse sottoproprietà che è possibile impostare per ottenere risultati diversi in base alle proprie esigenze.

La seguente schermata mostra l'interfaccia di Microsoft Excel per impostare il Formato 3D della forma.
