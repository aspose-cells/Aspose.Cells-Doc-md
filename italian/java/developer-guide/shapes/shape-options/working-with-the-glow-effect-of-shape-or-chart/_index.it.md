## **Possibili scenari di utilizzo**
Aspose.Cells fornisce la proprietà Shape.Glow per lavorare con l'effetto bagliore della forma o del grafico. Contiene le seguenti proprietà secondarie che è possibile impostare per ottenere risultati diversi in base alle proprie esigenze.

- GlowEffect.Size
- GlowEffect.Transparency
