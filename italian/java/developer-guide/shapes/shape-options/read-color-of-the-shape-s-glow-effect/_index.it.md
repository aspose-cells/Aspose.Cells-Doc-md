## Possibili scenari di utilizzo

Se si desidera leggere il colore dell'effetto di bagliore di una qualsiasi forma, si prega di utilizzare la proprietà [**Shape.Glow.Color**](https://reference.aspose.com/cells/java/com.aspose.cells/gloweffect#Color). Questo vi aiuterà a trovare le varie proprietà relative al colore dell'effetto di bagliore applicato a una forma.
