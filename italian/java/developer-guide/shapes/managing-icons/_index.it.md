## Aggiungi icone al foglio di lavoro in Aspose.Cells

 Se hai bisogno di usare[Aspose.Cells](https://products.aspose.com/cells/) per aggiungere "icone" in un file Excel, questo documento può fornirti un aiuto.

L'interfaccia di Excel corrispondente all'operazione di inserimento dell'icona è la seguente:

![](1.png)

- Selezionare la posizione dell'icona da inserire nel foglio di lavoro
-  Clic sinistro*Inserire*->*Icone*
- Nella finestra che si apre, seleziona l'icona nel rettangolo rosso nella figura sopra
-  Clic sinistro*Inserire*, verrà inserito nel file Excel.

L'effetto è il seguente:

![](2.png)
