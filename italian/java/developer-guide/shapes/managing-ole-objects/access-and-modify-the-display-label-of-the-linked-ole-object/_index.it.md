## **Possibili Scenari di Utilizzo**

Microsoft Excel consente di modificare l'etichetta di visualizzazione dell'oggetto Ole come mostrato nella seguente schermata. È inoltre possibile accedere o modificare l'etichetta di visualizzazione dell'oggetto Ole utilizzando le API di Aspose.Cells con la proprietà [**OleObject.Label**](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#Label).

![todo:image_alt_text](access-and-modify-the-display-label-of-the-linked-ole-object_1.png)

## **Accesso e modifica dell'etichetta di visualizzazione dell'oggetto Ole collegato**

Si prega di vedere il seguente codice di esempio, carica il [file Excel di esempio](64716833.xlsx) che contiene l'oggetto Ole. Il codice accede all'oggetto Ole e ne cambia l'etichetta da API di esempio a API di Aspose. Si prega di vedere l'output della console qui sotto che mostra l'effetto del codice di esempio sul file Excel di esempio per un riferimento.
