Aspose.Cells fornisce funzionalità avanzate per gestire le forme nel foglio di calcolo. A volte c'è bisogno di ottenere i punti di connessione di una forma per allinearle o posizionarle nel posto appropriato. A questo scopo, sono richiesti tutti i punti di connessione. Il seguente codice può essere utilizzato per ottenere l'elenco dei punti di connessione di una forma in cui è utilizzata la funzione [**Shape.GetConnectionPoints()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ConnectionPoints).
