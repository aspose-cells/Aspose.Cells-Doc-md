{{% alert color="primary" %}}

È possibile impostare diverse opzioni della tabella pivot utilizzando Aspose.Cells. Una di queste opzioni è "Per le celle vuote mostra". L'impostazione di questa opzione significa che tutte le celle vuote nella tabella pivot vengono visualizzate con una stringa specificata.

Per trovare e impostare questa opzione in Microsoft Excel:

1. Seleziona una tabella pivot e fai clic con il pulsante destro del mouse su di essa.
1.  Selezionare**Opzioni tabella pivot**.
1.  Seleziona il**Disposizione e formato** scheda.
1.  Seleziona il**Per le celle vuote mostra** opzione e specificare una stringa.

![cose da fare:immagine_alt_testo](setting-pivot-table-option-for-empty-cells-show_1.png)

{{% /alert %}}

Aspose.Cells fornisce il[**Tabella pivot.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString) e[**Tabella pivot.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString)proprietà per l'impostazione dell'opzione tabella pivot "Per le celle vuote mostra".
