## **Ottieni la fonte di dati della connessione esterna della tabella pivot**

Aspose.Cells fornisce la possibilità di ottenere la connessione dati esterna della tabella pivot. A questo scopo, l'API fornisce la proprietà [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) della classe [**PivotTable**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable). La proprietà [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) restituisce l'oggetto [**ExternalConnection**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection). Il seguente frammento di codice dimostra l'uso della proprietà [**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) per ottenere la connessione dati esterna della tabella pivot.
