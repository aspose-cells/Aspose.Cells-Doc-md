{{% alert color="primary" %}}

 Microsoft Excel consente di modificare il layout della tabella pivot utilizzando*Strumenti tabella pivot > Progettazione > Report* Comandi del menu Layout. È possibile modificare il layout in questi tre moduli

- Mostra in forma compatta
- Mostra in forma di contorno
- Mostra in forma tabellare

 Aspose.Cells fornisce anche[**Tabella pivot.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm()), [**Tabella pivot.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm() ) e[**Tabella pivot.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm()) metodi per modificare il layout della tabella pivot in queste tre forme.

{{% /alert %}}
