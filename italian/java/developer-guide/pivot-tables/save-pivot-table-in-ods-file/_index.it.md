 Aspose.Cells offre la possibilità di salvare la tabella pivot nei file ODS. Per questo, basta convertire una cartella di lavoro con la tabella povot esistente o creare una nuova tabella pivot e salvare il file nel formato ODS. Assicurati di chiamare[**Tabella pivot.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData()prima di salvare per garantire che la tabella pivot venga visualizzata nel file di output ODS.
