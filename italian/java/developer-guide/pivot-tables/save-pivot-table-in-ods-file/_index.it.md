Aspose.Cells fornisce la possibilità di salvare la tabella pivot nei file ODS. Per fare ciò, è sufficiente convertire un workbook con una tabella pivot esistente o creare una nuova tabella pivot e salvare il file nel formato ODS. Assicurati di chiamare [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData--) prima di salvare per garantire che la tabella pivot venga visualizzata nel file ODS di output.
