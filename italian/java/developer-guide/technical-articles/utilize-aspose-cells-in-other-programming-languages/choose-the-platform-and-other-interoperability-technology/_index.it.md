{{% alert color="primary" %}} 

Aspose.Cells è un componente di elaborazione del foglio di calcolo che consente di generare, leggere, modificare e scrivere fogli di calcolo Excel® senza utilizzare Microsoft Excel®. 

- Aspose.Cells for Java - una libreria di classi Java progettata per funzionare nelle applicazioni Java.

{{% /alert %}} 

Anche se non ci sono versioni speciali di Aspose.Cells per altre piattaforme, è possibile utilizzare Aspose.Cells in molti altri linguaggi di programmazione, inclusi (ma non limitati a): 

|**Linguaggio di Programmazione** |**Aspose.Cells da Utilizzare** |**Tecnologia di Interoperabilità** |
| :- | :- | :- |
|Perl |Java |Inline::Java |
|PHP |Java |Java Bridge |
|Python |Java |JPype |
Dalla tabella sopra puoi vedere che per alcuni linguaggi di programmazione è possibile utilizzare Aspose.Cells for Java. La tabella seguente può aiutarti a scegliere: 

|**Scenario** |**Quando Utilizzare** |
| :- | :- |
|Aspose.Cells for Java via Java Bridge o Inline::Java |<p>**Vantaggi**:</p><p>- Disponibile su qualsiasi piattaforma dove è disponibile Java.</p><p>- Facile chiamare metodi statici, costruttori con parametri, metodi sovraccaricati e utilizzare valori enumerati.</p>|
| |<p>**Svantaggi**: </p><p>- Ha rilasci meno frequenti e meno funzionalità rispetto a Aspose.Cells for .NET.</p>|
Si prega di notare che Aspose.Cells for Java è progettato per applicazioni Java e utilizzarlo in altri linguaggi di programmazione è soggetto a alcune limitazioni tecniche o soluzioni alternative.
