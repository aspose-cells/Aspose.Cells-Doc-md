{{% alert color="primary" %}} 

 Aspose.Cells è un componente di elaborazione di fogli di calcolo che consente di generare, leggere, modificare e scrivere fogli di calcolo Excel® senza utilizzare Microsoft Excel®.

- Aspose.Cells for Java - una libreria di classi Java ed è progettata per funzionare in applicazioni Java.

{{% /alert %}} 

 Sebbene non esistano versioni speciali di Aspose.Cells per altre piattaforme, è possibile utilizzare Aspose.Cells in molti altri linguaggi di programmazione, inclusi (ma non limitati a):

|**Linguaggio di programmazione** |**Aspose.Cells da utilizzare** |**Tecnologia di interoperabilità** |
|:- |:- |:- |
| Perle| Giava| Inline::Java|
| PHP| Giava|Ponte di Giava|
| Pitone| Giava| JPype|
 Dalla tabella sopra puoi vedere che per alcuni linguaggi di programmazione è possibile utilizzare Aspose.Cells for Java. La tabella seguente può aiutarti nella scelta:

|**Scenario** |**Quando usare** |
|:- |:- |
| Aspose.Cells for Java via Java Bridge o Inline::Java|<p>**Professionisti**:</p><p>- Disponibile su qualsiasi piattaforma in cui è disponibile Java.</p><p>- Metodi statici facili da chiamare, costruttori con parametri, metodi sovraccaricati e utilizzo di valori enumerati.</p>|
||<p>**Contro**: </p><p>- Ha versioni meno frequenti e meno funzionalità rispetto a Aspose.Cells for .NET.</p>|
 Si prega di notare che Aspose.Cells for Java è progettato per applicazioni for Java e il suo utilizzo in altri linguaggi di programmazione è soggetto ad alcune limitazioni tecniche o soluzioni alternative.
