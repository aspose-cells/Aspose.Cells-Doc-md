## **Questo formato di file non è supportato o non si specifica un formato corretto**
Se l'utente ha specificato il formato del file durante la creazione della cartella di lavoro dal file modello, in genere questo errore è dovuto al fatto che il formato file specificato non è il formato file effettivo del file modello. Se l'utente non ha specificato il formato del file, in genere è perché l'estensione del nome file non rappresenta il formato file effettivo di questo file e il formato file non può essere rilevato automaticamente, come il file csv/tsv che non ha identificatori speciali .
