## **Questo formato file non è supportato o non hai specificato un formato corretto**
Se l'utente ha specificato il formato file durante la creazione del file di lavoro dal file modello, comunemente questo errore è dovuto al fatto che il formato file specificato non corrisponde al formato effettivo del file modello. Se l'utente non ha specificato il formato file, comunemente è perché l'estensione del nome file non rappresenta il formato effettivo di questo file e il formato file non può essere rilevato automaticamente, come nel caso di file csv/tsv che non hanno speciali identificatori.
