A volte è necessario combinare più fogli di lavoro in un unico foglio di lavoro. Ciò può essere facilmente ottenuto utilizzando Aspose.Cells API. Questo articolo mostrerà un esempio di codice che legge una cartella di lavoro di origine e combina i dati di tutti i fogli di lavoro di origine in un unico foglio di lavoro all'interno di una cartella di lavoro di destinazione.

{{% /alert %}}
