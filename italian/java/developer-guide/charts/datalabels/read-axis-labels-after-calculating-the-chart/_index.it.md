##  **Possibili scenari di utilizzo**

Puoi leggere le etichette degli assi del tuo grafico dopo aver calcolato i suoi valori utilizzando[**Grafico.calcola()**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/#calculate--)metodo. Si prega di utilizzare il[**Axis.getAxisTexts()**](https://reference.aspose.com/cells/java/com.aspose.cells/axis/#getAxisTexts--)metodo a questo scopo che restituirà l'elenco delle etichette degli assi.
