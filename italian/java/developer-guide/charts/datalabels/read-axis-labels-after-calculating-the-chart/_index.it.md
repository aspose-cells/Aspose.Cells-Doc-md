## **Possibili scenari di utilizzo**

Puoi leggere le etichette degli assi del tuo grafico dopo aver calcolato i suoi valori usando il[**Grafico.calcola()**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#calculate()) metodo. Si prega di utilizzare il[**Axis.AxisLabels**](https://reference.aspose.com/cells/java/com.aspose.cells/axis#AxisLabels)proprietà per questo scopo che restituirà l'elenco delle etichette degli assi.
