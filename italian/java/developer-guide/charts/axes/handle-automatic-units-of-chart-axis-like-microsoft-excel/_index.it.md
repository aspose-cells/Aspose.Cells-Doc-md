## **Possibili scenari di utilizzo**
Le versioni precedenti di Aspose.Cells non erano in grado di gestire correttamente le unità automatiche dell'asse del grafico quando il grafico veniva visualizzato come immagine o PDF. Ora, Aspose.Cells supporta la gestione delle unità automatiche dell'asse del grafico. Non c'è nessun cambio di codice. Basta convertire il grafico in immagine o PDF e renderà l'asse del grafico proprio come lo rende Microsoft Excel.
