##  **Possibili scenari di utilizzo**
Dopo aver creato un grafico in un foglio di lavoro, puoi impostare l'asse delle categorie per esso. In questo articolo ti mostreremo come impostare l'asse delle categorie per il grafico Excel, utilizzando Aspose.Cells con codice di esempio.

##  **I passaggi nel codice di esempio**

1. Crea una nuova cartella di lavoro.

2. Crea un nuovo grafico nel primo foglio di lavoro.

3. Aggiungi alcuni valori alle celle nel primo foglio di lavoro.

4. Ora puoi impostare l'asse delle categorie, ci sono due modi: utilizzando i dati della cella o utilizzando direttamente le stringhe, entrambi mostrati nel codice di esempio.

5. Imposta l'asse dei valori, salva la cartella di lavoro per visualizzare il risultato.
