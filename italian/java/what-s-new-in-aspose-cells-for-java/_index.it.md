Questa pagina descrive le nuove e più interessanti funzionalità di Aspose.Cells introdotte nelle versioni recenti.

## Aspose.Cells for Java 24.6
* Miglioramenti per il rendering: regolazione della saturazione del colore e della temperatura del colore, duotoni, immagini Webp
* Miglioramenti per PivotTable: unione intervallo, "Mostra valori come" di PivotField
* Supporto Range.ToHtml()/ToImage() per la comodità dell'utente nella manipolazione dei dati dell'intervallo

## Aspose.Cells for Java 24.5
* Supporto delle nuove funzioni: XMATCH, XOR
* Supporto per la trasposizione dei dati dell'intervallo
* Supporto per ottenere le proprietà delle impostazioni del filtro avanzato
* Supporto per rendere l'immagine inserita nella cella in pdf/immagine

## Aspose.Cells for Java 24.4
* Fornire implementazioni semplici del monitor di interruzione per la comodità dell'utente
* Supporto per incorporare allegati durante la conversione dell'oggetto OLE in pdf
* Supporto per rendere la forma automatica di ChartX in immagine
* Supporto per incorporare un'immagine nella cella (operazione "Inserisci nella cella" per l'immagine in ms excel)
* Miglioramenti per l'importazione del file Numbers: celle unite, righe/colonne nascoste, ...ecc.

## Aspose.Cells for Java 24.3
* Supporto per l'importazione di file AZW3
* Supporto per salvare ods con standard odf 1.3
* Supporto per calcolare la funzione VALUTA

## Aspose.Cells for Java 24.2
* Supporto alle nuove funzioni: TESTODOPO,TESTOPRIMA,SUDDIVISIONETESTO,A RIGA,A COLONNA
* Supporto per il rilevamento del formato oxps
* Miglioramento delle prestazioni per la classificazione di grandi set di dati

## Aspose.Cells for Java 24.1

* Supporto jakarta.servlet per GridWeb
* Supporto per l'importazione di dati csv con righe/colonne di intestazione specificate dall'utente e righe/colonne totali per foglio
* Supporto per l'esportazione html con stili personalizzati
* Supporto per il riempimento automatico delle aree unite
* Supporto per il riempimento automatico dei dati da destra a sinistra e dal basso verso l'alto

## Aspose.Cells for Java 23.12

* Supporto per la conformità PDF/A-2 e PDF/A-3 durante la conversione in pdf
* Supporto per il calcolo della raggruppamento discreto della tabella pivot
* Supporto per l'esportazione dei dati delle celle in una lista generica
* Supporto per l'importazione di riferimenti incrociati utilizzando tabelle collegate per intelligenti marcatori
* Supporta la funzione MAKEARRAY per le formule

## Aspose.Cells for Java 23.11

* Supporto per convertire i fogli di calcolo in file PDF versione 1.7 (ISO32000-1: 2008)

## Aspose.Cells for Java 23.10

* Supporto per impostare collegamenti ipertestuali direttamente su un intervallo
* Supporto per copiare più fogli di lavoro contemporaneamente
* Supporto per verificare se un intervallo è vuoto per i dati delle celle

## Aspose.Cells for Java 23.9

* Supporto per calcolare nuove funzioni in formula: LAMBDA,LET,MAP,SCAN
* Supporto per importare/esportare file EPUB
* Supporto per filtrare le serie del grafico

## Aspose.Cells for Java 23.8

* Supporto per conservare gli spazi di riempimento nelle formule
* Supporto per sostituire/inserire testo con formattazione per il testo ricco della cella

## Aspose.Cells for Java 23.7

* Rilevamento migliorato del formato file per alcuni nuovi formati come file visio, file pacchetto OneNote.

## Aspose.Cells for Java 23.6

* Supporto per esportare ed evidenziare i log delle revisioni
* Supporto per attraversare gli oggetti riga esistenti da RowCollection in ordine inverso
* Supporto per aggiornare più intervalli di consolidamento di tavole pivot contemporaneamente

## Aspose.Cells for Java 23.5

* Supporto per aggiornare riferimenti esterni dei dati della scheda esterna alla scheda locale del documento corrente
* Supporto per rendere il controllo casella di controllo come controllo interattivo invece di immagine statica in GridWeb
* Supporto per selezionare PivotArea e formattarlo
* Supporto per aggiungere watermark durante la conversione in pdf

## Aspose.Cells for Java 23.4

* Supporto per calcolare la funzione ENCODEURL
* Supporto per completare la costruzione di una forma di equazione inserendo nodi rilevanti passo dopo passo
* Supporto per calcolare l'espressione della formula come formula matriciale senza impostarla su un oggetto cella
* Supporto per salvare file xlsb in modalità LightCells
* Supporto per verificare se un font è installato o disponibile
* Supporto per impostare la formula su cella specificata nell'oggetto ListObject
* Supporto per salvare il documento di lavoro come xhtml
* Supporto per rendere il controllo activex e il controllo del modulo per GridWeb

## Aspose.Cells for Java 23.3

* Supporto per impostare/leggere/salvare nuove funzioni: LET, ENCODEURL
* Supporto per collegare origini dati a collegamenti esterni durante il calcolo della formula

## Aspose.Cells for Java 23.2

* Supporto per impostare/leggere/salvare nuove funzioni: SCAN, LAMBDA
* Supporto per esportare lo stile in json

## Aspose.Cells for Java 23.1

* Supporto delle nuove funzioni: CHOOSEROWS, CHOOSECOLS, HSTACK, VSTACK
* Supporto per ottenere autore e data e ora dalle revisioni

## Aspose.Cells for Java 22.12

* Supporto per esportare excel in JSON con struttura ad albero a più livelli

## Aspose.Cells for Java 22.11

* Supporto per distinguere tra formula di matrice dinamica e formula di matrice legacy

## Aspose.Cells for Java 22.10

## Aspose.Cells for Java 22.9

* Supporto per impostare formula della tabella per celle
* Supporto per impostare formula matriciale dinamica con funzione personalizzata e motore di calcolo
* Supporto per impostare impostazioni di gradiente predefinite per il formato di riempimento gradienre
* Supporto per applicare bordi esterni con colori del tema per intervallo

## Aspose.Cells for Java 22.8

* Supporto per esportare commenti e note come visualizzati sul foglio in html
* Supporto per specificare fogli da esportare durante la conversione in pdf/xps

## Aspose.Cells for Java 22.7

* Supporto per leggere file modello OOXML che contengono dati zip anomali e non possono essere elaborati da ZipInputStream di Java
* Supporto per esportare l'html in un unico file con intestazione e piè di pagina

## Aspose.Cells for Java 22.6

* Miglioramento per il calcolo e il tracciamento delle formule con la catena di calcolo della formula

## Aspose.Cells for Java 22.5

* Miglioramento delle prestazioni della memoria nel settaggio delle formule delle celle durante il salvataggio del foglio di lavoro in modalità LightCells

## Aspose.Cells for Java 22.4

* Opzioni potenziate per controllare le celle vuote per l'esportazione in CSV.

## Aspose.Cells for Java 22.1

## Aspose.Cells for Java 21.11

* Supporto per il calcolo della formula/funzione XLookup.
