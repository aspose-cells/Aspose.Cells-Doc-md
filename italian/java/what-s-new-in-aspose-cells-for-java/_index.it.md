In questa pagina vengono descritte le novità più interessanti di Aspose.Cells introdotte nelle versioni recenti.

##  Aspose.Cells for Java 23.12

* Supporto per la conformità PDF/A-2 e PDF/A-3 durante la conversione in PDF
* Supporta il calcolo del raggruppamento discreto della tabella pivot
* Supporto per esportare i dati delle celle in un elenco generico
* Supporto per importare riferimenti incrociati utilizzando tabelle collegate per marcatori intelligenti
* Supporta la funzione MAKEEARRAY per le formule

##  Aspose.Cells for Java 23.11

* Supporto per convertire i fogli di calcolo nel file PDF versione 1.7 (ISO32000-1:2008)

##  Aspose.Cells for Java 23.10

* Supporto per impostare collegamenti ipertestuali direttamente a un intervallo
* Supporto per copiare più fogli di lavoro contemporaneamente
* Supporto per verificare se un intervallo è vuoto per i dati delle celle

##  Aspose.Cells for Java 23.9

* Supporto per calcolare nuove funzioni nella formula: LAMBDA,LET,MAP,SCAN
* Supporto per importare/esportare file EPUB
* Supporto per filtrare le serie di grafici

##  Aspose.Cells for Java 23.8

* Supporto per preservare lo spazio di riempimento nelle formule
* Supporto per sostituire/inserire testo con formattazione per richtext della cella

##  Aspose.Cells for Java 23.7

* Rilevamento del formato file migliorato per alcuni nuovi formati come file Visio e file del pacchetto OneNote.

##  Aspose.Cells for Java 23.6

* Supporto per esportare ed evidenziare i registri di revisione
* Supporto per attraversare oggetti Row esistenti da RowCollection in ordine inverso
* Supporto per aggiornare più intervalli di consolidamento di tabella pivot

##  Aspose.Cells for Java 23.5

* Supporto per aggiornare i riferimenti esterni dei dati del foglio esterno al foglio locale della cartella di lavoro corrente
* Supporto per eseguire il rendering del controllo Checkbox come controllo interattivo anziché come immagine statica in GridWeb
* Supporto per selezionare PivotArea e formattarla
* Supporto per aggiungere filigrana durante la conversione in PDF

##  Aspose.Cells for Java 23.4

* Supporto per calcolare la funzione ENCODEURL
* Supporto per completare la costruzione di una forma di equazione inserendo passo dopo passo i nodi rilevanti
* Supporto per calcolare l'espressione della formula come formula di matrice senza impostarla su un oggetto cella
* Supporto per salvare file xlsb in modalità LightCells
* Supporto per verificare se un font è installato o disponibile
* Supporto per impostare la formula sulla cella specificata in ListObject
* Supporto per salvare la cartella di lavoro come xhtml
* Supporto per il rendering del controllo ActiveX e del controllo del modulo per GridWeb

##  Aspose.Cells for Java 23.3

* Supporto per impostare/leggere/salvare nuove funzioni: LET, ENCODEURL
* Supporto per collegare origini dati a collegamenti esterni quando si chiama il calcolo della formula

##  Aspose.Cells for Java 23.2

* Supporto per impostare/leggere/salvare nuove funzioni: SCAN, LAMBDA
* Supporto per esportare lo stile in JSON

##  Aspose.Cells for Java 23.1

* Supporta nuove funzioni: CHOOSEROWS, CHOOSECOLS, HSTACK, VSTACK
* Supporto per ottenere l'autore e la data dalle revisioni

##  Aspose.Cells for Java 22.12

* Supporto per esportare Excel a JSON con struttura gerarchica a più livelli

##  Aspose.Cells for Java 22.11

* Supporto per distinguere tra formula di matrice dinamica e formula di matrice legacy
* Supporto per leggere/salvare formule di array dinamici per il formato file xlsb

##  Aspose.Cells for Java 22.10

* Supporto per aggiungere e aggiornare il grafico con il file modello CRTX

##  Aspose.Cells for Java 22.9

* Supporto per impostare la formula della tabella per le celle
* Supporto per impostare la formula di matrice dinamica con funzione personalizzata e motore di calcolo
* Supporto per impostare le impostazioni del gradiente preimpostate per il formato di riempimento gradiente
* Supporto per applicare i bordi del contorno con i colori del tema per Intervallo

##  Aspose.Cells for Java 22.8

* Supporta l'esportazione di commenti e note come visualizzati sul foglio in HTML
* Supporta la specifica dei fogli di lavoro da produrre durante la conversione in pdf/xps

##  Aspose.Cells for Java 22.7

* Supporto per leggere file modello in formato OOXML che contengono dati zip anomali e non possono essere elaborati da ZipInputStream di Java
* Supporto per esportare HTML in un singolo file con intestazione/piè di pagina
* Supporto per ottenere la posizione assoluta e la dimensione dell'intervallo (in punti)

##  Aspose.Cells for Java 22.6

* Miglioramento per il calcolo e il tracciamento delle formule con la catena di calcolo delle formule

##  Aspose.Cells for Java 22.5

* Miglioramento delle prestazioni della memoria nell'impostazione della formula delle celle durante il salvataggio della cartella di lavoro in modalità LightCells

##  Aspose.Cells for Java 22.4

* Opzioni migliorate per controllare le celle vuote per l'esportazione CSV.
* Supporta la suddivisione complicata e l'aggiornamento della formattazione condizionale per le operazioni di inserimento/eliminazione

##  Aspose.Cells for Java 22.3

* Supporto per ricalcolare le celle che dipendono dal nome definito quando il nome definito viene modificato e la catena di calcolo della cartella di lavoro è abilitata.
* Supporto per ottenere ricorsivamente le foglie della cella secondo la catena di calcolo delle formule

##  Aspose.Cells for Java 22.2

* Supporto per estrarre nomi definiti da NameCollection tramite filtro di ambito specificato

##  Aspose.Cells for Java 22.1

* Supporto per generare pptx modificabile
* Supporto per importare CSV di grandi dimensioni in più fogli di lavoro quando la riga/colonna di dati supera il limite
* Supporto per importare CSV con LightCells
* Supporto per rimuovere il collegamento esterno specificato
* Supporto per mantenere le formule rimuovendo i collegamenti esterni

##  Aspose.Cells for Java 21.12

* Supporta la conversione di Excel in script SQL
* Supporta l'importazione e l'esportazione di dati XML semplici
* Supporto per leggere quanti più dati possibile dal file xls danneggiato
* Supporto per interrompere il processo di SmartMarker
* Migliorare le prestazioni per l'aggiunta di intervalli per la formattazione condizionale e la convalida
* Imposta la priorità più alta per la condizione di formato appena aggiunta, proprio come il comportamento di MS Excel

##  Aspose.Cells for Java 21.11

* Supporta il calcolo della formula/funzione XLookup.
* Integra il salvataggio di JSON e immagini nel salvataggio della cartella di lavoro.
* Migliora le prestazioni durante la conversione di Excel in HTML.
