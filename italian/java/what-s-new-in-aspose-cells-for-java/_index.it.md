In questa pagina vengono descritte le novità più interessanti di Aspose.Cells introdotte nelle ultime release.

## Aspose.Cells for Java 22.12
* Supporto per esportare Excel in JSON con struttura gerarchica a più livelli

## Aspose.Cells for Java 22.11

Supporto per distinguere tra formula di matrice dinamica e formula di matrice legacy
* Supporto per leggere/salvare formule di matrice dinamica per il formato di file xlsb

## Aspose.Cells for Java 22.10

* Supporto per aggiungere e aggiornare il grafico con il file modello CRTX

## Aspose.Cells for Java 22.9

* Supporto per impostare la formula della tabella per le celle
* Supporto per impostare la formula dell'array dinamico con funzione personalizzata e motore di calcolo
* Supporto per impostare le impostazioni del gradiente preimpostate per il formato di riempimento del gradiente
* Supporto per applicare i bordi del contorno con i colori del tema per l'intervallo

## Aspose.Cells for Java 22.8

* Supporta l'esportazione di commenti e note come visualizzati sul foglio in html
* Supporto per specificare i fogli di lavoro da produrre durante la conversione in pdf/xps

## Aspose.Cells for Java 22.7

* Supporto per la lettura di file modello in formato OOXML che contengono dati zip anomali e non possono essere elaborati da ZipInputStream di java
* Supporto per esportare html in un singolo file con intestazione/piè di pagina
* Supporto per ottenere la posizione assoluta e la dimensione dell'intervallo (in punti)

## Aspose.Cells for Java 22.6

* Miglioramento per il calcolo e la tracciatura delle formule con la catena di calcolo delle formule

## Aspose.Cells for Java 22.5

Miglioramento delle prestazioni della memoria nell'impostazione della formula delle celle durante il salvataggio della cartella di lavoro in modalità LightCells

## Aspose.Cells for Java 22.4

* Opzioni avanzate per controllare le celle vuote per l'esportazione di CSV.
* Supporta la divisione complicata e l'aggiornamento della formattazione condizionale per le operazioni di inserimento/cancellazione

## Aspose.Cells for Java 22.3

* Supporto per ricalcolare le celle che dipendono dal nome definito quando il nome definito viene modificato e la catena di calcolo della cartella di lavoro è abilitata.
* Supporto per ottenere le foglie delle celle in modo ricorsivo secondo la catena di calcolo delle formule

## Aspose.Cells for Java 22.2

* Supporto per estrarre nomi definiti da NameCollection mediante filtro ambito specificato

## Aspose.Cells for Java 22.1

* Supporto per generare pptx modificabili
* Supporto per importare csv di grandi dimensioni in più fogli di lavoro quando la riga/colonna di dati supera il limite
* Supporto per importare csv con LightCells
* Supporto per rimuovere il collegamento esterno specificato
* Supporto per mantenere le formule durante la rimozione dei collegamenti esterni

## Aspose.Cells for Java 21.12

* Supporta la conversione di Excel in script SQL
* Supporta l'importazione e l'esportazione di dati XML semplici
Supporto per leggere quanti più dati possibile dal file xls danneggiato
* Supporto per interrompere il processo di SmartMarker
* Migliora le prestazioni per l'aggiunta di intervalli per la formattazione condizionale e la convalida
* Imposta la priorità come massima per la condizione di formato appena aggiunta, proprio come il comportamento di ms excel

## Aspose.Cells for Java 21.11

* Supporto per il calcolo della formula/funzione XLookup.
* Integra il salvataggio di json e immagini per salvare la cartella di lavoro.
* Migliora le prestazioni durante la conversione di Excel in HTML.
