Innanzitutto assicurati di aver installato il bundle ADT e di aver scaricato l'ultima versione di Aspose.Cells for Android via Java.

- Estrarre il**aspose-cells-XXXjar**dall'archivio dell'ultima versione e aggiungerlo a una nuova cartella sotto l'esempio che si desidera eseguire. Aggiungi il file in una cartella chiamata**librerie**.
- Apri il progetto nel tuo spazio di lavoro utilizzando Eclipse con ADT.
- Clic**Correre**per avviare e scegliere l'emulatore appropriato su cui eseguire il pacchetto.
