Assicurarsi innanzitutto di avere il pacchetto ADT installato e di aver scaricato l'ultima versione di Aspose.Cells per Android via Java.

- Estrarre il file **aspose-cells-X.X.X.jar** dall'archivio dell'ultima versione e aggiungerlo a una nuova cartella sotto l'esempio che si desidera eseguire. Aggiungere il file in una cartella chiamata **libs**.
- Aprire il progetto nel proprio workspace utilizzando Eclipse con ADT.
- Fare clic su **Esegui** per avviare e scegliere l'emulatore appropriato su cui eseguire il pacchetto.
