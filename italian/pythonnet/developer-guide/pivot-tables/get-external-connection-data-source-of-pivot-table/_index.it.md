## **Come ottenere la fonte dati della connessione esterna della tabella pivot**

Aspose.Cells per Python via .NET fornisce la possibilità di ottenere la fonte dati della connessione esterna della tabella pivot. A questo scopo, l'API fornisce la proprietà [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) della classe [**PivotTable**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/). La proprietà [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) restituisce l'oggetto [**ExternalConnection**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/). Il seguente frammento di codice dimostra l'uso della proprietà [**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) per ottenere la fonte dati della connessione esterna della tabella pivot.
