##  **Ottieni l'origine dati della connessione esterna della tabella pivot**

Aspose.Cells for Python via .NET offre la possibilità di ottenere l'origine dati della connessione esterna della tabella pivot. Per questo lo API mette a disposizione il[**origine_dati_connessione_esterna**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) proprietà del[**Tabella pivot**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/) classe. IL[**origine_dati_connessione_esterna**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/)la proprietà restituisce il file[**Connessioneesterna**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/) oggetto. Il seguente frammento di codice illustra l'uso di[**Tabella pivot.origine_dati_connessione_esterna**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) proprietà per ottenere l'origine dati della connessione esterna della tabella pivot.
