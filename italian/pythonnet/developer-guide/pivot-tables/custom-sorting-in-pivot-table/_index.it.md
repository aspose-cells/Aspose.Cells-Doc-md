##  **Ordinamento personalizzato nella tabella pivot**
Utilizzando Aspose.Cells for Python via .NET API, è possibile ordinare le tabelle pivot in base ai valori dei campi. Il seguente frammento di codice carica il file Excel di esempio e aggiunge tre tabelle pivot. La prima tabella pivot non ha un ordinamento personalizzato, la seconda tabella pivot è ordinata in base ai valori del campo riga "Pesce" e la terza tabella pivot è ordinata in base ai valori del campo colonna "28/07/2000".
