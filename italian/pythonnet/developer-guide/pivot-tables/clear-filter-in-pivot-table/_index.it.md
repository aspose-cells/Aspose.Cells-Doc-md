##  **Possibili scenari di utilizzo**
 Quando crei una tabella pivot con dati noti e desideri filtrare la tabella pivot, devi imparare e utilizzare il filtro. Può aiutarti a filtrare i dati che desideri in modo efficace. Utilizzando Aspose.Cells for Python via .NET API, è possibile utilizzare il filtro sui valori dei campi nelle tabelle pivot.

##  **Cancella filtro nella tabella pivot in Excel**
Cancella il filtro nella tabella pivot in Excel, procedi nel seguente modo:

1. Seleziona la tabella pivot su cui vuoi cancellare il filtro.
2. Fare clic sulla freccia a discesa relativa al filtro che si desidera cancellare nella tabella pivot.
3. Seleziona "Cancella filtro" dal menu a discesa.
<img src="1.png" width=80% />
4. Se desideri cancellare tutti i filtri dalla tabella pivot, puoi anche fare clic sul pulsante "Cancella filtri" nella scheda Analizza tabella pivot sulla barra multifunzione in Excel.
<img src="2.png" width=80% />
