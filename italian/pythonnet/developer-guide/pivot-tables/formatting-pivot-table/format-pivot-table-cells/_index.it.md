{{% alert color="primary" %}}

 A volte, vuoi formattare le celle della tabella pivot. Ad esempio, desideri applicare il colore di sfondo alle celle della tabella pivot. Aspose.Cells for Python via .NET prevede due metodi[**Tabella pivot.format_all(stile)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format_all/) E[**Tabella pivot.format(riga, colonna, stile)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format/), che puoi utilizzare per questo scopo.

[**Tabella pivot.format_all(stile)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format_all/) applica lo stile all'intera tabella pivot mentre[**Tabella pivot.format(riga, colonna, stile)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format/) applica lo stile a una singola cella della tabella pivot.
