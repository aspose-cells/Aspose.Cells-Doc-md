{{% alert color="primary" %}}

I report basati su tabelle pivot sono utili ma soggetti a errori se gli utenti destinatari non hanno conoscenze dettagliate di Excel per configurare questi report. In queste circostanze, le organizzazioni vorranno limitare agli utenti la possibilità di modificare un report basato su tabelle pivot. Le funzionalità comuni delle tabelle pivot come l'aggiunta di filtri aggiuntivi, slicer, campi o la modifica dell'ordine di determinate cose nel report non sono generalmente consigliate per ogni utente. D'altro canto, questi utenti devono anche essere in grado di aggiornare il report e utilizzare i filtri o gli slicer esistenti. Aspose.Cells for Python via .NET ha fornito questa capacità agli sviluppatori per limitare agli utenti la modifica di questi report durante la creazione di questi report. A questo scopo, Excel fornisce una funzione per disabilitare la barra delle tabelle pivot e lo stesso è fornito da Aspose.Cells for Python via .NET, cioè lo sviluppatore può disabilitare la barra che contiene i controlli per modificare questi report.

{{% /alert %}}
