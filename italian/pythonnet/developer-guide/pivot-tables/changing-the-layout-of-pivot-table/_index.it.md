{{% alert color="primary" %}}

 Microsoft Excel consente di modificare il layout della tabella pivot utilizzando*Strumenti tabella pivot > Progettazione > Layout report* comandi del menu. Puoi modificare il layout in questi tre moduli

- Mostra in forma compatta
- Mostra in forma struttura
- Mostra in forma tabellare

 Aspose.Cells fornisce anche[**Tabella pivot.show_in_compact_form()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/show_in_compact_form/#), [**Tabella pivot.show_in_outline_form()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/show_in_outline_form/#) E[**Tabella pivot.show_in_tabular_form()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/show_in_tabular_form/#) metodi per modificare il layout della tabella pivot in queste tre forme.
