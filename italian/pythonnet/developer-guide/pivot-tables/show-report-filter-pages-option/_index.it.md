##  **Mostra l'opzione delle pagine di filtro del report**
Excel supporta la creazione di tabelle pivot, l'aggiunta di filtri di report e l'abilitazione dell'opzione "Mostra pagine filtro report". Aspose.Cells for Python via .NET supporta anche questa funzione per abilitare l'opzione "Mostra pagine filtro report" sulla tabella pivot creata. Di seguito è riportata la schermata che mostra l'opzione "Mostra pagine filtro report" in Excel.
