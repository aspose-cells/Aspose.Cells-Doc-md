## **Come mostrare l'opzione delle pagine di filtro del report**
Excel supporta la creazione di tabelle pivot, l'aggiunta di filtri del report e abilita l'opzione "Mostra pagine del filtro del report". Anche Aspose.Cells per Python via .NET supporta questa funzione per abilitare l'opzione "Mostra pagine del filtro del report" sulla tabella pivot creata. Di seguito è riportata la schermata che mostra l'opzione "Mostra pagine del filtro del report" in Excel.
