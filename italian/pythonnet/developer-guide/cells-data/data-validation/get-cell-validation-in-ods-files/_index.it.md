## **Ottieni la Convalida Cellulare nei File ODS**

Con Aspose.Cells, è possibile ottenere la convalida applicata a una cella nei file ODS. A questo scopo, l'API fornisce il metodo [**get_validation**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) della classe [**Cell**](https://reference.aspose.com/cells/python-net/aspose.cells/cell).
