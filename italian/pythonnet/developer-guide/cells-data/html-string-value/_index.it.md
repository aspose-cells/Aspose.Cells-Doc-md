{{% alert color="primary" %}}

Aspose.Cells for Python via .NET supporta la conversione di HTML orientato a Microsoft Excel in formato XLS/XLSX. Significa che l'HTML generato da Microsoft Excel può essere convertito nuovamente in formato XLS/XLSX utilizzando Aspose.Cells for Python via .NET.

Allo stesso modo, se c'è dell'HTML semplice, Aspose.Cells for Python via .NET può convertirlo in Testo Ricco HTML. Aspose.Cells for Python via .NET fornisce la proprietà [**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/) che può prendere un HTML semplice e convertirlo in testo di cella formattato.
