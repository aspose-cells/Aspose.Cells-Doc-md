{{% alert color="primary" %}}

A volte devi trovare celle con uno stile particolare applicato. Puoi utilizzare Aspose.Cells per Python via .NET per trovare tutte le celle con uno stile comune. Aspose.Cells per Python via .NET fornisce la proprietà [**FindOptions.style**](https://reference.aspose.com/cells/python-net/aspose.cells/findoptions/style/) che puoi utilizzare per specificare lo stile per cercare le celle.

{{% /alert %}}

Il codice in questo esempio trova tutte le celle che hanno lo stesso stile di quella della cella A1. Dopo che il codice è stato eseguito, tutte le celle che hanno lo stesso stile di A1 contengono il testo "Trovato".
