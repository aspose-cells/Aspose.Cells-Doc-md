{{% alert color="primary" %}}

Aspose.Cells per Python via .NET supporta la modifica della connessione dati SQL esistente. L'articolo spiegherà come usare Aspose.Cells per Python via .NET per modificare le diverse proprietà della connessione dati SQL.

È possibile aggiungere o visualizzare le connessioni dati all'interno di Microsoft Excel seguendo il comando di menu **Dati > Connessioni**.

Analogamente, Aspose.Cells per Python via .NET fornisce i mezzi per accedere e modificare le Connessioni Dati utilizzando la collezione Workbook.DataConnections.

{{% /alert %}}
