## **Possibili Scenari di Utilizzo**

In generale, specifichi la directory dei font o l’elenco dei font per tutti i workbook, ma a volte devi specificare font individuali o privati. Aspose.Cells per Python via .NET fornisce la classe [**IndividualFontConfigs**](https://reference.aspose.com/cells/python-net/aspose.cells/individualfontconfigs) che può essere usata per specificare i font individuali o privati del tuo workbook.
