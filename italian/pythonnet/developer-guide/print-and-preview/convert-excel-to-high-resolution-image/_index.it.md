Con l’aumentare della diffusione di schermi ad alta risoluzione, le immagini generate con il DPI di default 96 spesso appaiono sfocate e poco chiare. Per garantire la chiarezza sugli schermi ad alta risoluzione, è essenziale generare immagini ad un DPI più alto. Aspose.Cells per Python via .NET offre la funzionalità di impostare [**ImageOrPrintOptions.horizontal_resolution**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/horizontal_resolution/) e [**ImageOrPrintOptions.vertical_resolution**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/vertical_resolution/), consentendoti di creare immagini di alta qualità da file Excel che risultano nitide su display ad alta risoluzione.
