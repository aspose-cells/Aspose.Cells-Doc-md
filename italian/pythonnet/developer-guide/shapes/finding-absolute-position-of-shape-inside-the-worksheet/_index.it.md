{{% alert color="primary" %}}

A volte, è necessario conoscere la posizione assoluta di una forma in un foglio di lavoro. Aspose.Cells per Python via .NET fornisce le proprietà [**Shape.left_to_corner**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/left_to_corner) e [**Shape.top_to_corner**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/top_to_corner) per questo scopo. Queste proprietà restituiscono la posizione assoluta della forma all'interno del foglio di lavoro in pixel.
