## **Possibili Scenari di Utilizzo**

A volte si desidera visualizzare il testo in carattere di lingue dell'Estremo Oriente ad esempio giapponese, cinese, thailandese, ecc. Aspose.Cells per Python via .NET fornisce la proprietà [**TextOptions.far_east_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/far_east_name) che può essere usata per specificare il nome del carattere della lingua dell'Estremo Oriente. Inoltre, puoi anche specificare il nome del carattere latino usando la proprietà [**TextOptions.latin_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/latin_name).
