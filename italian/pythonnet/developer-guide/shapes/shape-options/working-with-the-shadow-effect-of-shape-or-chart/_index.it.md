## **Possibili Scenari di Utilizzo**
Aspose.Cells per Python via .NET fornisce la proprietà [Shape.shadow_effect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/shadow_effect) insieme alla classe [ShadowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect) per lavorare con l'effetto ombra di forma o grafico. La classe [ShadowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect) contiene le seguenti proprietà che possono essere impostate per ottenere risultati diversi in base ai requisiti dell'applicazione.

- [angolo](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/angle)
- [sfocatura](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/blur)
- [colore](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/color)
- [distanza](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/distance)
- [tipo_preset](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/preset_type)
- [dimensione](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/size)
- [trasparenza](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/transparency)
