## Verifica se il progetto VBA è protetto e bloccato in modalità di visualizzazione in Python

Aspose.Cells per Python via .NET ti permette di verificare se il progetto VBA (Visual Basic for Applications) di un file Excel è protetto e bloccato per la visualizzazione. Per questo, l'API fornisce la proprietà [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/). Se è bloccato per la visualizzazione, allora la proprietà [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/) restituisce **true**.
