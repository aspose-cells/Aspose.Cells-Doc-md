{{% alert color="primary" %}} 

Aspose.Cells for Python via .NET ora supporta la piena conversione delle revisioni dei file XLSB in file XLSM. Le revisioni si trovano all'interno del percorso \xl\revisions. Puoi visualizzarle cambiando l'estensione del file XLSB in ZIP. Il percorso \xl\revisions contiene file con estensione .bin.

Quando converti il tuo file XLSB in un file XLSM utilizzando Aspose.Cells for Python via .NET, questi file .bin vengono convertiti con successo in file .xml come mostrato in queste due schermate.
