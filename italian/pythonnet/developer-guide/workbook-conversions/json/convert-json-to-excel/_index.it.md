Aspose.Cells for Python via .NET supporta la conversione di un file Json (JavaScript Object Notation) in una cartella di lavoro Excel.

{{% /alert %}}

##  **Converti JSON in cartella di lavoro Excel**
Non c'è bisogno di chiedersi come convertire JSON in un file Excel, perché la libreria Aspose.Cells for Python via .NET ha la decisione migliore. Aspose.Cells for Python via .NET API fornisce supporto per la conversione del formato JSON in fogli di calcolo. Puoi usare**[JsonLoadOptions](https://reference.aspose.com/cells/python-net/aspose.cells/JsonLoadOptions)** class per specificare impostazioni aggiuntive per l'importazione di JSON nella cartella di lavoro.
