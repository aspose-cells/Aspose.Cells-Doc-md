Se desideri esportare il tuo file Excel in HTML con le linee della griglia, utilizza la proprietà [HtmlSaveOptions.export_grid_lines](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_grid_lines/) e impostala su **true**.
