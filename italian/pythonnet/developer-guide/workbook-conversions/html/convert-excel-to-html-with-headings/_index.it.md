## **Converti Excel in HTML con intestazioni**

Aspose.Cells per Python via NET offre l'opzione di esportare le intestazioni durante la conversione di Excel in HTML. Per fare ciò, è possibile impostare la proprietà [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) su **true**. Il valore predefinito della proprietà [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) è **false**.
