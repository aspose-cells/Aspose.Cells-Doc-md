##  **Converti Excel in HTML con intestazioni**

Aspose.Cells for Python tramite NET offre la possibilità di esportare intestazioni durante la conversione di Excel in HTML. Per questo, è possibile impostare il[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)proprietà su *true**. Il valore predefinito di[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)la proprietà è *false**.
