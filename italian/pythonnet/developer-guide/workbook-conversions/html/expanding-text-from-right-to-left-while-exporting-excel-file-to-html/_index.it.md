{{% alert color="primary" %}} 

Aspose.Cells per Python via .NET supporta ora l'espansione del testo da destra a sinistra durante l'esportazione di un file Excel in HTML. Questa funzione è stata implementata dallv8.9.0.0. Ora, se il tuo file Excel di origine contiene testo che si espande da destra a sinistra, Aspose.Cells lo esporta correttamente in HTML.

{{% /alert %}}
