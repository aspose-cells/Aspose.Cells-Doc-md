## **Possibili Scenari di Utilizzo**

In precedenza, Aspose.Cells for Python via .NET non poteva renderizzare gli add-in di Office quando il file di Excel veniva salvato nel formato PDF. Ora Aspose.Cells for Python via .NET lo renderizza correttamente. Non è necessario utilizzare alcun metodo o proprietà speciale per renderizzare gli add-in di Office nel PDF di output. Basta salvare il file di Excel in formato PDF e renderà gli add-in di Office.
