In precedenza, Aspose.Cells for Python via .NET non era in grado di eseguire il rendering dei componenti aggiuntivi di Office quando il file Excel veniva salvato nel formato PDF. Ora Aspose.Cells for Python via .NET lo rende perfetto. Non è necessario utilizzare alcun metodo o proprietà speciale per eseguire il rendering dei componenti aggiuntivi di Office nell'output PDF. Basta salvare il file Excel nel formato PDF e verrà eseguito il rendering dei componenti aggiuntivi di Office.
