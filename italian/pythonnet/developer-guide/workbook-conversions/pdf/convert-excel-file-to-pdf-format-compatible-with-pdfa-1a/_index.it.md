##  **Possibili scenari di utilizzo**

PDF/A è un gusto unico di PDF pensato per la conservazione a lungo termine dei documenti. PDF/A è una versione standardizzata ISO del Portable Document Format (PDF) che è un formato di archivio di PDF che incorpora tutti i caratteri utilizzati nel documento all'interno del file PDF. PDF/A differisce da PDF vietando funzionalità quali il collegamento dei caratteri (in contrapposizione all'incorporamento dei caratteri) e la crittografia. Aspose.Cells for Python via .NET consente di salvare i file Excel in file PDF conformi a PDF (sono supportati sia PdfA1a che PdfA1b). In questo argomento viene descritto come salvare la cartella di lavoro di Excel nel file PDF compatibile con PDF/A (PdfA1a).

##  **Converti file Excel nel formato PDF compatibile con PDFA-1a**

Gli sviluppatori possono utilizzare il file**[PdfSaveOptions](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/)**classe per impostare attributi diversi per la conversione. Impostazione di diverse proprietà di**[PdfSaveOptions](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/)**ti dà il controllo sulle impostazioni di stampa, carattere, sicurezza e compressione per l'output PDF. La proprietà più importante è**[PdfSaveOptions.compliance](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/compliance/)**che consente di salvare i file Excel in file PDF conformi a PDF/A.
