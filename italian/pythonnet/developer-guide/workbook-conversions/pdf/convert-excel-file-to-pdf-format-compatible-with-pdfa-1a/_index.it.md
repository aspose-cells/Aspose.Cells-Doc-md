## **Possibili Scenari di Utilizzo**

PDF/A è una particolare variante del PDF progettata per la conservazione a lungo termine di documenti. PDF/A è una versione standardizzata ISO del formato di documento portatile (PDF) che è un formato archivistico del PDF che incorpora tutti i font utilizzati nel documento all'interno del file PDF. PDF/A differisce dal PDF proibendo funzionalità come il collegamento dei font (invece che l'incorporamento) e la crittografia. Aspose.Cells per Python via .NET consente di salvare i file Excel in file PDF conformi a PDF/A (sia PdfA1a che PdfA1b sono supportati). Questo argomento descrive come salvare il quaderno Excel in un file PDF/A conforme (PdfA1a) di PDF.

## **Converti il file Excel nel Formato PDF Compatibile con PDFA-1a**

I programmatori possono utilizzare la classe [**PdfSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/) per impostare diversi attributi per la conversione. Impostare diverse proprietà della classe [**PdfSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/) ti consente di controllare le impostazioni di stampa, font, sicurezza e compressione per il PDF di output. La proprietà più importante è [**PdfSaveOptions.compliance**](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/compliance/) che ti consente di salvare i file Excel in file PDF/A conformi.
