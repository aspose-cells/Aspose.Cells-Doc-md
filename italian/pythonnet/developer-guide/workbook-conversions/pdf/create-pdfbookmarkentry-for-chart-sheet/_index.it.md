## **Possibili Scenari di Utilizzo**

In precedenza, Aspose.Cells per Python via .NET avrebbe creato [**PdfBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) per un foglio normale. Ma ora Aspose.Cells per Python via .NET può anche creare [**PdfBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) per un foglio grafico. Poiché il foglio grafico non ha altre celle oltre alla cella A1, creerà [**PdfBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) solo per la cella A1.
