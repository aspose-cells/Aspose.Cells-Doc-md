Aspose.Cells for Python via .NET supporta la conversione di file XLS che contengono immagini e grafici in documenti PDF. Aspose.Cells for Python via .NET API può funzionare in modo indipendente per convertire un foglio di calcolo in PDF: Aspose.PDF for .NET non è richiesto per la conversione. Il processo può essere eseguito in memoria poiché non dipende da file XML temporanei o intermedi. Ciò significa che file Excel di grandi dimensioni, ad esempio quelli contenenti immagini, grafici e altri oggetti di disegno, possono essere convertiti in modo rapido ed efficiente.
