Le strutture della struttura logica PDF forniscono un meccanismo per incorporare informazioni, riguardanti la struttura del contenuto del documento, in un file PDF. Aspose.Cells for Python via .NET conserva le informazioni sulla struttura di un documento Excel Microsoft, ad esempio cella, riga, tabella, foglio di lavoro, immagine, forma, intestazione/piè di pagina, ecc.

Con l'opzione [PdfSaveOptions.export_document_structure]https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/export_document_structure/), è possibile salvare in un tag PDF con la struttura del documento esportata.
