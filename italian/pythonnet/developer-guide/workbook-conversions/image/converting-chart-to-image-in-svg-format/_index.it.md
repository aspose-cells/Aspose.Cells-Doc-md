Scalable Vector Graphics (SVG) è un formato di immagine vettoriale basato su XML per grafica bidimensionale che supporta anche interattività e animazione. La specifica SVG è uno standard aperto sviluppato dal World Wide Web Consortium (W3C) a partire dal 1999.

SVG le immagini e i relativi comportamenti sono definiti in file di testo XML. Ciò significa che possono essere cercati, indicizzati, scriptati e compressi. Come file XML, le immagini SVG possono essere create e modificate con qualsiasi editor di testo, ma più spesso vengono create con software di disegno.

Aspose.Cells for Python via .NET può salvare il grafico in immagini in vari formati come BMP, JPEG, PNG, GIF, SVG ecc. Questo articolo spiega come salvare un grafico nel formato SVG.
