{{% alert color="primary" %}}

 Aspose.Cells for Python via .NET supporta il salvataggio di un file ODS (**OpenDocument Spreadsheet**) nelle specifiche ODF (**OpenDocument Format**) 1.1 e 1.2. Aspose.Cells for Python via .NET ha[**OdsSaveOptions.is_strict_schema11**](https://reference.aspose.com/cells/python-net/aspose.cells/odssaveoptions/is_strict_schema11/)proprietà che specifica l'uso della specifica ODF 1.1 per il salvataggio dei file ODS. Il valore predefinito di questa proprietà è *false**, quindi il file ODS salvato senza questa impostazione utilizza le specifiche 1.2.

{{% /alert %}}

Il codice di esempio seguente crea un oggetto cartella di lavoro, aggiunge un valore alla cella A1 nel primo foglio di lavoro e quindi salva il file ODS nelle specifiche ODF 1.1 e 1.2. Per impostazione predefinita, il file ODS viene salvato nella specifica ODF 1.2.
