{{% alert color="primary" %}}

È possibile mostrare le formule anziché i valori calcolati in Microsoft Excel utilizzando l'opzione **Mostra Formule** nella scheda **Formule**. Quando le formule sono mostrate, Microsoft Excel visualizza le formule nel foglio di lavoro. Puoi ottenere lo stesso risultato utilizzando Aspose.Cells per Python via .NET.

{{% /alert %}}

Aspose.Cells per Python via .NET fornisce una proprietà [**Worksheet.show_formulas**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/show_formulas). Imposta questa a **true** per impostare Microsoft Excel per mostrare le formule.
