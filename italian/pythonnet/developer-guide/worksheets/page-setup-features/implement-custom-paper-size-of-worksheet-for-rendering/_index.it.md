## **Possibili Scenari di Utilizzo**

Non è disponibile un'opzione diretta per creare formati carta personalizzati in MS Excel, tuttavia puoi impostare la dimensione carta personalizzata dei tuoi fogli desiderati durante la conversione del file Excel in formato file PDF. Questo documento spiega come impostare una dimensione carta personalizzata di un foglio utilizzando le API Aspose.Cells per Python via .NET.

## **Implementare un formato carta personalizzato del foglio di lavoro per il rendering**

Aspose.Cells per Python via .NET ti permette di implementare la dimensione carta desiderata del foglio. Puoi utilizzare il metodo [**custom_paper_size**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup/custom_paper_size/#float-float) della classe [**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup) per specificare una dimensione pagina personalizzata. Il codice di esempio seguente illustra come specificare una dimensione carta personalizzata per il primo foglio di lavoro.
