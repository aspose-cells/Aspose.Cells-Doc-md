## **Possibili Scenari di Utilizzo**

Non esiste un'opzione diretta per creare dimensioni personalizzate della carta in MS Excel, tuttavia puoi impostare una dimensione personalizzata della carta per i fogli di lavoro desiderati durante la conversione del file Excel in formato PDF. Questo documento spiega come impostare una dimensione personalizzata della carta di un foglio di lavoro utilizzando le API Aspose.Cells per Python via .NET.

## **Implementare un formato carta personalizzato del foglio di lavoro per il rendering**

Aspose.Cells per Python via .NET permette di implementare la dimensione desiderata del foglio di lavoro. Puoi utilizzare il metodo [**custom_paper_size**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup/custom_paper_size/#float-float) della classe [**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup) per specificare una dimensione personalizzata della pagina. Il seguente esempio di codice illustra come specificare una dimensione personalizzata della carta per il primo foglio di lavoro nel workbook.
