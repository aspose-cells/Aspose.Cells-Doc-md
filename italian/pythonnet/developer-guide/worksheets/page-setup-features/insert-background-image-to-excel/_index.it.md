Puoi rendere un foglio di lavoro più attraente aggiungendo un'immagine come sfondo del foglio. Questa funzione può essere molto efficace se hai un grafico aziendale speciale che aggiunge un tocco di sfondo senza oscurare i dati del foglio. Puoi impostare l'immagine di sfondo per un foglio usando le API Aspose.Cells per Python via .NET.

{{% /alert %}} 

## **Come impostare lo sfondo del foglio in Microsoft Excel**

Impostare un'immagine di sfondo di un foglio in Microsoft Excel (ad esempio, Microsoft Excel 2019):

1. Dal menu **Layout di pagina**, individuare l'opzione **Imposta pagina** e fare clic sull'opzione **Sfondo**.
1. Seleziona un'immagine da impostare come sfondo del foglio.
