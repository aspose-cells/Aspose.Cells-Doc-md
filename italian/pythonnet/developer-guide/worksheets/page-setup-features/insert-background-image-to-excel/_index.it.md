{{% alert color="primary" %}} 

È possibile rendere un foglio di lavoro più accattivante aggiungendo un'immagine come sfondo del foglio. Questa funzionalità può essere molto efficace se si dispone di una grafica aziendale speciale che aggiunge un suggerimento dello sfondo senza oscurare i dati presenti nel foglio. Puoi impostare un'immagine di sfondo per un foglio utilizzando l'API Aspose.Cells per Python via .NET.

{{% /alert %}} 

## **Come Impostare Sfondo Foglio in Microsoft Excel**

Impostare un'immagine di sfondo di un foglio in Microsoft Excel (ad esempio, Microsoft Excel 2019):

1. Dal menu **Layout di pagina**, individuare l'opzione **Imposta pagina** e fare clic sull'opzione **Sfondo**.
1. Seleziona un'immagine da impostare come sfondo del foglio.
