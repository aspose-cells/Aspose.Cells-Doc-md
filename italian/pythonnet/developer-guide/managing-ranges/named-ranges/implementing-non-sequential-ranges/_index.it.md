## **Possibili Scenari di Utilizzo**

Normalmente, i range nominati sono rettangolari con celle continue e adiacenti tra loro. Ma a volte potresti aver bisogno di utilizzare un intervallo di celle non sequenziali in cui le celle non sono adiacenti. Aspose.Cells per Python via .NET supporta la creazione di un range nominato con celle non adiacenti.
