{{% alert color="primary" %}}

Microsoft Excel consente agli utenti di aggiungere commenti alle celle per aggiungere informazioni aggiuntive e evidenziare dati. Gli sviluppatori potrebbero aver bisogno di personalizzare il commento per specificare le impostazioni di allineamento e la direzione del testo. Aspose.Cells per Python via .NET fornisce API per eseguire questo compito.

{{% /alert %}}

Aspose.Cells per Python via .NET fornisce una proprietà [**Shape.text_direction**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_direction) per impostare la direzione del testo di un commento.
