Rimuovere righe duplicate è una delle molte funzionalità utili di Microsoft Excel. Consente agli utenti di rimuovere righe duplicate in un foglio di lavoro, è possibile scegliere quali colonne devono essere controllate per informazioni duplicate.

Aspose.Cells per Python via .NET fornisce il metodo [**Cells.remove_duplicates()**](https://reference.aspose.com/cells/python-net/aspose.cells/cells/remove_duplicates/) per questo scopo. È inoltre possibile impostare startRow, startColumn, endRow, endColumn per selezionare le colonne.
