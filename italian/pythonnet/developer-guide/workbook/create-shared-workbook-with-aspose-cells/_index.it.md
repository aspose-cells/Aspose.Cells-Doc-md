## **Possibili Scenari di Utilizzo**

Microsoft Excel consente di condividere il workbook come mostrato nello screenshot seguente. Quando condividi il workbook, più di un utente può modificarlo sulla rete. Aspose.Cells per Python via .NET permette di creare un workbook condiviso con la proprietà [**Workbook.settings.shared**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/shared).
