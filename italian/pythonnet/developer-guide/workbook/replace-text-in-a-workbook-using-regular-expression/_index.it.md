Aspose.Cells per Python via .NET offre la funzionalità di sostituire il testo in un workbook usando un’espressione regolare. Per questo, l’API fornisce la proprietà [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) della classe [**ReplaceOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions). Impostare il [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) a **true** indica che la chiave cercata sarà un’espressione regolare.

Il frammento di codice seguente dimostra l'uso della proprietà [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) utilizzando il [file excel di esempio](101089318.xlsx).
