{{% alert color="primary" %}} 

**Office Open XML** (noto anche come **OOXML** o **Microsoft Open XML** (MOX)) è un formato file basato su XML sviluppato da Microsoft per rappresentare documenti di ufficio come fogli di calcolo, grafici, presentazioni e documenti di elaborazione testi.

{{% /alert %}} 



Aspose.Cells per Python via .NET fornisce un modo per rilevare il formato del file di file criptati **Microsoft Open XML**. Per identificare il tipo di file, usa il metodo [FileFormatUtil.detect_file_format](https://reference.aspose.com/cells/python-net/aspose.cells/fileformatutil/detect_file_format/#str) come mostrato nell’esempio di codice.
