## **Possibili Scenari di Utilizzo**

Puoi specificare il nome dell'autore mentre proteggi in scrittura il tuo workbook usando l'API di Aspose.Cells per Python via .NET. Utilizza la proprietà [**Workbook.settings.write_protection.author**](https://reference.aspose.com/cells/python-net/aspose.cells/writeprotection/author/) a questo scopo.
