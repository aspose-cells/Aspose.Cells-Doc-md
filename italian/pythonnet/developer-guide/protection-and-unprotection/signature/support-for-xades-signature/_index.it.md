## **Introduzione**

Aspose.Cells per Python via .NET supporta la firma di workbooks con la firma XAdES. Per questo, l'API fornisce le classi [**DigitalSignature**](https://reference.aspose.com/cells/python-net/aspose.cells.digitalsignatures/digitalsignature) e le enumerazioni [**XAdESType**](https://reference.aspose.com/cells/python-net/aspose.cells.digitalsignatures/xadestype).
