## **Possibili Scenari di Utilizzo**

Aspose.Cells per Python via .NET supporta il formato personalizzato *DBNum*. Ad esempio, se il valore della cella è 123 e si specifica il formato personalizzato come [DBNum2][$-804]General, verrà visualizzato come 壹佰贰拾叁.
