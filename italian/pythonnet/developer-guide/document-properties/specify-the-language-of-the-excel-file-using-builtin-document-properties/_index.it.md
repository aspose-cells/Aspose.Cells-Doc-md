## **Possibili Scenari di Utilizzo**

Puoi cambiare la lingua del file Excel cliccando con il tasto destro sul file, scegliendo Proprietà > Dettagli e modificando il campo Lingua. Usa la proprietà [**BuiltInDocumentPropertyCollection.language**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/language) per modificarla programmaticamente usando Aspose.Cells for Python via .NET.
