## **Cos'è un grafico azionario?**

I grafici azionari sono un grafico specifico utilizzato per tracciare le variazioni nel prezzo degli asset negoziati, come commodity, azioni e criptovalute. Permettono di vedere valori massimi e minimi nel tempo, insieme ai valori di apertura e chiusura in un unico grafico. Aspose.Cells per Python via .NET offre 4 grafici azionari e per utilizzarli, devi avere i giusti set di dati e selezionare le colonne nell'ordine corretto.

Il seguente set di dati mostra le informazioni di trading giornaliere per un'azione. Utilizzeremo questi dati per creare quattro tipi di grafici azionari: grafico azionario Alto-Basso-Chiuso (HLC), grafico Open-High-Low-Close (OHLC), grafico azionario Volume-Alto-Basso-Chiuso (VHLC) e grafico azionario Volume-Open-High-Low-Close (VOHLC).
