## **Cambia la direzione delle etichette di graduazione**

Aspose.Cells per Python via .NET ti permette di cambiare la direzione delle etichette degli intervalli dell'asse del grafico usando la proprietà [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type). La proprietà [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type) accetta il valore dell'enumerazione [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/). L'enumerazione [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/) fornisce i seguenti membri

- ORIZZONTALE
- VERTICALE
- ROTATE90
- ROTATE270
- IMPILATI
