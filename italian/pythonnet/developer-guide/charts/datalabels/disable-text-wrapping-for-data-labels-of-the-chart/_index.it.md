{{% alert color="primary" %}}

Microsoft Excel 2013 consente agli utenti di abilitare o disabilitare il testo all'interno delle etichette dati del grafico. Per impostazione predefinita, il testo all'interno delle etichette dati del grafico è nello stato di testo a capo.

Aspose.Cells per Python via .NET fornisce una proprietà [**DataLabels.is_text_wrapped**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/datalabels/is_text_wrapped) che puoi impostare su True o False per abilitare o disabilitare l'avanzo del testo delle etichette dati rispettivamente.

{{% /alert %}}
