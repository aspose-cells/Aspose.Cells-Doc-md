## **Leggi il sottotitolo del grafico dal file ODS**

Aspose.Cells per Python via .NET ti consente di leggere i sottotitoli dei grafici nei file ODS utilizzando la proprietà [**Chart.sub_title**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/chart/sub_title). Il seguente esempio di codice carica il [file ODS di esempio](89620481.ods) e legge il sottotitolo del grafico usando la proprietà [**Chart.sub_title**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/chart/sub_title) e lo stampa nella Finestra di Console.
