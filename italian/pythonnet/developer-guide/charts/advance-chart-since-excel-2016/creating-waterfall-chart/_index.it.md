{{% alert color="primary" %}}

Un grafico a cascata è un tipo speciale di grafico che viene normalmente utilizzato per mostrare come la posizione iniziale aumenta o diminuisce. Microsoft Excel ha molti tipi di grafici predefiniti, tra cui colonne, linee, torte, barre, radar, ecc. ma il grafico a cascata va oltre i grafici di base e può essere creato utilizzando i tipi di grafico esistenti con una personalizzazione minima o maggiore.

{{% /alert %}} 

Le API di Aspose.Cells per Python via .NET permettono di creare un grafico a cascata utilizzando un grafico a linee. L'API consente anche di personalizzare l'aspetto del grafico per conferirgli la forma di una cascata impostando le proprietà [**Series.up_bars**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/series/up_bars) e [**Series.down_bars**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/series/down_bars).
