##  **Possibili scenari di utilizzo**

È possibile formattare l'affettatrice in Microsoft Excel impostandone il numero di colonne o impostandone lo stile, ecc. Aspose.Cells for Python via .NET ti consente anche di farlo utilizzando il comando[**Affettatrice.numero_di_colonne**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/number_of_columns/)E[**Slicer.style_type**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/style_type/)proprietà.
