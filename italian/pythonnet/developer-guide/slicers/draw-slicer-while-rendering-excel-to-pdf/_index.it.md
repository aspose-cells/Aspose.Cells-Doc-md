## **Come disegnare il slicer durante il rendering di Excel in PDF utilizzando la libreria Aspose.Cells per Python Excel.**
Se hai un file Excel a cui è stato applicato un slicer e vuoi esportare l'Excel in PDF con le impostazioni del slicer, Aspose.Cells per Python via .NET ora supporta questo di default. Devi semplicemente esportare il file Excel con slicer in PDF e il PDF generato mostrerà il slicer applicato.
