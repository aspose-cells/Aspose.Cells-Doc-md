##  **Disegna Slicer durante il rendering di Excel su PDF**
Se disponi di un file Excel a cui è applicata l'affettatrice e desideri esportare Excel in PDF con le impostazioni dell'affettatrice, Aspose.Cells for Python via .NET ora lo supporta per impostazione predefinita. È sufficiente esportare il file Excel con l'affettatrice su PDF, lo PDF generato mostrerà l'affettatrice applicata.

 Il codice di esempio seguente carica il file[file Excel di esempio](94044165.xlsx) che contiene un'affettatrice esistente. Quindi salva la cartella di lavoro come[file di output PDF](94044166.pdf). Lo screenshot seguente confronta il file Excel di origine e il file PDF generato.
