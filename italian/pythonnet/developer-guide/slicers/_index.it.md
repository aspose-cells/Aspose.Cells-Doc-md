## **Possibili Scenari di Utilizzo**

Un slicer è utilizzato per filtrare rapidamente i dati. Può essere utilizzato per filtrare i dati sia in una tabella che in una tabella pivot. Microsoft Excel ti permette di creare uno slicer selezionando una tabella o tabella pivot e quindi cliccando su *Inserisci > Slicer*. Anche Aspose.Cells per Python via .NET ti consente di creare uno slicer utilizzando il metodo [**Worksheet.slicers.add()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/add/#aspose.cells.pivot.PivotTable-int-int-aspose.cells.pivot.PivotField).
