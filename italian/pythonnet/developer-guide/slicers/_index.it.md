##  **Possibili scenari di utilizzo**

Un'affettatrice viene utilizzata per filtrare rapidamente i dati. Può essere utilizzato per filtrare i dati sia in una tabella che in una tabella pivot. Microsoft Excel consente di creare un'affettatrice selezionando una tabella o una tabella pivot e quindi facendo clic su *Inserisci > Affettatrice*. Aspose.Cells for Python via .NET consente inoltre di creare uno slicer utilizzando il comando[**Foglio di lavoro.slicers.add()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/add/#aspose.cells.pivot.PivotTable-int-int-aspose.cells.pivot.PivotField)metodo.
