##  **Possibili scenari di utilizzo**

 Se desideri rimuovere l'affettatrice in Microsoft Excel, selezionalo e premi il pulsante*Eliminare* pulsante. Allo stesso modo, se desideri rimuoverlo utilizzando Aspose.Cells for Python via .NET API a livello di codice, utilizza il comando[**Foglio di lavoro.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/)metodo. Rimuoverà l'affettatrice dal foglio di lavoro.
