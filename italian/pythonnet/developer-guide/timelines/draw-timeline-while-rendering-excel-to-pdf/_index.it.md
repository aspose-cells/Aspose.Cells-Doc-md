##  **Disegna la sequenza temporale durante il rendering di Excel su PDF**
Se disponi di un file Excel a cui è applicata la sequenza temporale e desideri esportare Excel in PDF con le impostazioni della sequenza temporale, Aspose.Cells for Python via .NET ora lo supporta per impostazione predefinita. Esporta semplicemente il file Excel con la sequenza temporale su PDF, lo PDF generato mostrerà la sequenza temporale applicata.

 Il codice di esempio seguente carica il file[file Excel di esempio](input.xlsx) che contiene una sequenza temporale esistente. Quindi salva la cartella di lavoro come[file di output PDF](out.pdf). Lo screenshot seguente confronta il file Excel di origine e il file PDF generato.
