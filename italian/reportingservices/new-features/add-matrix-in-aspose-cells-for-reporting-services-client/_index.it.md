{{% alert color="primary" %}} 

Abbiamo incorporato il supporto per creare un elemento di report di matrice in un report. Il contesto della matrice include raggruppamenti angolo, di colonne, di righe e righe della matrice. Imposta attributi del report di matrice che includono nome, intervallo e così via tramite il pulsante **Imposta attributo** nella barra comandi di Aspose.Cells in Excel.

Ad esempio, nello screenshot sottostante, il intervallo di matrice è E9:G11.
