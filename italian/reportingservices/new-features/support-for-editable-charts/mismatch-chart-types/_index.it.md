{{% alert color="primary" %}} 

Alcuni tipi di grafico nel file di definizione del report di SQL Server Reporting Services 2008 non sono disponibili come tipi di grafico di Excel:

- Linea a gradini
- Range-Plain
- Gamma-Liscio
- Intervallo-Colonna
- Range-Bar
- Range-ErrorBar
- Range-BoxPlot
- Forma-imbuto

 Quando si esegue il rendering di elementi del grafico che non dispongono di opzioni equivalenti in Microsoft Excel, Aspose.Cells li restituisce alle immagini.

{{% /alert %}}
