{{% alert color="primary" %}} 

Alcuni tipi di grafico nel file di definizione del report di SQL Server Reporting Services 2008 non sono disponibili come tipi di grafico di Excel:

- Linea-Gradino
- Intervallo-Piano
- Intervallo-Regolare
- Intervallo-Colonna
- Intervallo-Barra
- Intervallo-Barre di errore
- Intervallo-BoxPlot
- Forma-Filtro

Quando si rendono gli elementi del grafico che non dispongono di opzioni equivalenti in Microsoft Excel, Aspose.Cells li trasforma in immagini. 

{{% /alert %}}
