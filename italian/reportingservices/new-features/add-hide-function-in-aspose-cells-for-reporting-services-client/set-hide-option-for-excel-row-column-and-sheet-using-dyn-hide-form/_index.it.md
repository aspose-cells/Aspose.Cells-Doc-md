Aspose.Cells for Reporting Services nasconde qualsiasi riga, colonna e foglio specificati in base ai parametri di nascondere. I parametri includono:

- **NomeFoglio** rappresenta un foglio di lavoro che specifica un oggetto nascosto.
- **Tipo** rappresenta il tipo di oggetto nascosto. Include foglio di lavoro, riga e colonna.
- **Indice Riga/Colonna** rappresenta l'oggetto nascosto specificato. Quando il tipo è foglio di lavoro, il valore è nullo. L'indice colonna supporta i caratteri. Il valore dell'indice parte da 1.
- **Stato di nascondere** rappresenta il valore dello stato di nascondere. Il valore è true, false o una formula Excel. Il parametro supporta solo i parametri del report di Reporting Services.
