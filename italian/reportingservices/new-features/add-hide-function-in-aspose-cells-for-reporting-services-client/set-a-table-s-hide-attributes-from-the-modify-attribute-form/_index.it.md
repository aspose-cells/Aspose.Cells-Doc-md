Imposta gli attributi di nascondere della tabella dal **Modifica attributo**. I parametri includono:

- **Tipo** rappresenta il tipo di oggetto nascosto. Include tabella, riga e colonna.
- **Indice** rappresenta la posizione dell'oggetto nascosto. Quando il tipo è foglio di lavoro, il valore è nullo. Il valore dell'indice parte da 0.
- **Espressione** rappresenta il valore dello stato nascosto. Il valore è true, false o una formula Excel. L'espressione supporta la formula Reporting Services.
