 Imposta gli attributi Nascondi da un rapporto tabella**Modifica attributo**. I parametri includono:

- **Tipo** rappresenta il tipo di oggetto Nascondi. Include tabella, riga e colonna.
- **Indice**rappresenta la posizione dell'oggetto nascosto. Quando type è foglio di lavoro, il valore è null. Il valore dell'indice parte da 0.
- **Espressione** rappresenta il valore dello stato nascosto. Il valore è vero, falso o una formula di Excel. L'espressione supporta la formula di Reporting Services.
