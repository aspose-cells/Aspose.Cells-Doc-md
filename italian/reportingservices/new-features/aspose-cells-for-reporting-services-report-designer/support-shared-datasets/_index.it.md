Aspose.Cells for Reporting Services supporta i set di dati condivisi SSRS 2008R2. Quando Aspose.Cells Report Designer apre un file di definizione del report che include set di dati condivisi, può ottenere automaticamente le informazioni sul set di dati che includono i campi del server di report.
