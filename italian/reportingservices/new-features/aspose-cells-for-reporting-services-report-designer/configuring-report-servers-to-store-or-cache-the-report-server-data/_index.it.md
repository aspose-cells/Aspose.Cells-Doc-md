Il Aspose.Cells for Reporting Services Report Designer salva l'URL del server dei report attualmente accessato. Quando si apre nuovamente il modulo del Gestore del server dei report nel Aspose.Cells Report Designer, il modulo carica automaticamente le informazioni del server dei report salvate.
