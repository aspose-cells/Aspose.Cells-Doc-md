Per modificare una query in Aspose.Cells.Report.Designer:

1. Aprire Microsoft Excel.
1. Fare clic su **Genera DataSet** sulla barra degli strumenti di Aspose.Cells.Report.Designer (

![todo:image_alt_text](editing-query_1.png)

).

1. Nella finestra di dialogo, selezionare un set di dati, ad esempio, **EmpSalesDetail**.
1. Fare clic su **Modifica**. 

   **Selezione di un set di dati** 

![todo:image_alt_text](editing-query_2.png)




1. Aspose.Cell.Report.Designer avvia Microsoft Query e apre la query per la modifica. 

   **Modifica di una query in Microsoft Query** 

![todo:image_alt_text](editing-query_3.png)




1. Dal menu **File** di Microsoft Query, selezionare **Torna a Aspose.Cells.Report.Designer** 

   **Torna al Designer del Report** 

![todo:image_alt_text](editing-query_4.png)
