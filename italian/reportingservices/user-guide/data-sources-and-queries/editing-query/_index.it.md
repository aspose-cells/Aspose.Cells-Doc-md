Per modificare una query in Aspose.Cells.Report.Designer:

1. Apri Microsoft Excel.
1.  Clic**Crea set di dati**sulla barra degli strumenti Aspose.Cells.Report.Designer (

![cose da fare:immagine_alt_testo](editing-query_1.png)

).

1.  Nella finestra di dialogo, seleziona un set di dati, ad esempio,**EmpSalesDetail**.
1.  Clic**Modificare**. 

   **Selezione di un set di dati** 

![cose da fare:immagine_alt_testo](editing-query_2.png)




1.  Aspose.Cell.Report.Designer avvia Microsoft Query e apre la query per la modifica.

   **Modifica di una query in Microsoft Query** 

![cose da fare:immagine_alt_testo](editing-query_3.png)




1.  Da Microsoft Query**File** menù, selezionare**Torna a Aspose.Cells.Report.Designer** 

   **Ritorna al Report Designer** 

![cose da fare:immagine_alt_testo](editing-query_4.png)
