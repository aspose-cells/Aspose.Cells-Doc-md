{{% alert color="primary" %}} 

È possibile creare origini dati e query in Visual Studio e salvarle in un file RDL. Quando il file RDL viene aperto in Aspose.Cells.Report.Designer, le relative origini dati e query sono disponibili per l'uso. Ad esempio, possiamo creare una query denominata EmpSalesDetail in Visual Studio e salvarla come file RDL.

**Il file RDL EmpSalesDetail** 

![cose da fare:immagine_alt_testo](using-data-sources-and-queries-created-in-visual-studio_1.png)

Questo articolo illustra come lavorare con un file RDL in Aspose.Cells.Report.Designer.

{{% /alert %}} 

Puoi aprire i file RDL con Aspose.Cells.Report.Designer e lavorarci:

1. Apri Microsoft Excel.
1.  Dalla barra degli strumenti Aspose.Cells.Report.Designer selezionare**Apri rapporto**. 

   **Il pulsante Apri rapporto** 

![cose da fare:immagine_alt_testo](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Seleziona il file RDL e aprilo.
 Una volta aperto il file RDL in Aspose.Cells.Report.Designer, le query vengono visualizzate nella**Set di dati** elenco.

   **Set di dati disponibili** 

![cose da fare:immagine_alt_testo](using-data-sources-and-queries-created-in-visual-studio_3.png)
