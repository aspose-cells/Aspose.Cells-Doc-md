{{% alert color="primary" %}} 

È possibile creare fonti dati e query in Visual Studio e salvarle in un file RDL. Quando il file RDL viene aperto in Aspose.Cells.Report.Designer, le sue fonti dati e query sono disponibili per l'uso. Ad esempio, possiamo creare una query chiamata EmpSalesDetail in Visual Studio e salvarla come file RDL.

**Il file RDL EmpSalesDetail** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_1.png)

Questo articolo mostra come lavorare con un file RDL in Aspose.Cells.Report.Designer.

{{% /alert %}} 

È possibile aprire file RDL con Aspose.Cells.Report.Designer e lavorare con essi:

1. Aprire Microsoft Excel.
1. Dalla barra degli strumenti di Aspose.Cells.Report.Designer, selezionare **Apri Report**. 

   **Il pulsante Apri Report** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Seleziona il file RDL e aprilo.
   Una volta che il file RDL è stato aperto in Aspose.Cells.Report.Designer, le query appaiono nell'elenco **Dataset**. 

   **Set di dati disponibili** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_3.png)
