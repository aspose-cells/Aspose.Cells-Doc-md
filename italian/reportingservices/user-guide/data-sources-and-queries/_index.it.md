{{% alert color="primary" %}} 

Il primo passaggio nella procedura per la progettazione di un report in Aspose.Cells.Report.Designer consiste nel creare origini dati e query. Ci sono due modi per farlo. Nel primo approccio, potremmo utilizzare un file di definizione del report RDL esistente creato da altri strumenti, ad esempio utilizzando Visual Studio e il file contiene origini dati e query necessarie per la progettazione del report e il recupero dei dati. Il secondo modo consiste nel creare nuove fonti e query in Aspose.Cells.Report.Designer.
