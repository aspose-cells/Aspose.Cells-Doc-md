Per creare una nuova tabella:

1. Apri Microsoft Excel.
1.  Clic**Apri rapporto** 

![cose da fare:immagine_alt_testo](adding-base-information-for-table_1)

 `  ` sulla barra degli strumenti Aspose.Cells.Report.Designer per aprire un file di report RDL che contiene le origini dati, i set di dati e i parametri di report creati in precedenza. Una volta che il file è stato aperto con successo, tutte le sue informazioni sono disponibili per l'uso, ad esempio, i suoi set di dati sono elencati nella relativa casella combinata DataSet.

![cose da fare:immagine_alt_testo](adding-base-information-for-table_2.png)

1. Apri un foglio di calcolo Microsoft Excel e progetta l'intestazione della tabella come segue:

   **Progettazione dell'intestazione della tabella** 

![cose da fare:immagine_alt_testo](adding-base-information-for-table_3.png)

1.  Seleziona un set di dati "EmpSalesDetail" dal controllo della casella combinata del set di dati pertinente.

   **Selezione di un set di dati** 

![cose da fare:immagine_alt_testo](adding-base-information-for-table_4.png)

1.  Tutti i campi del set di dati "EmpSalesDetail" sono elencati nella casella combinata dei campi. Selezionare la cella sotto la colonna OrderNumber Cell e quindi selezionare il campo "SalesOrderNumber" dalla casella combinata dei campi per inserire un indicatore di dati nella cella.

   **Inserimento di un campo** 

![cose da fare:immagine_alt_testo](adding-base-information-for-table_5.png)



**Il codice del campo mostra dove è stato inserito un campo.** 

![cose da fare:immagine_alt_testo](adding-base-information-for-table_6.png)

1.  Ripetere il passaggio 5 per inserire i campi ProdCat e SubCat.

   **Tutti i campi inseriti** 

![cose da fare:immagine_alt_testo](adding-base-information-for-table_7.png)
