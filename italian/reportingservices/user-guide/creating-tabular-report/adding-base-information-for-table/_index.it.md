Per creare una nuova tabella:

1. Aprire Microsoft Excel.
1. Fare clic su **Apri Rapporto** 

![todo:image_alt_text](adding-base-information-for-table_1)

` `nella barra degli strumenti Aspose.Cells.Report.Designer per aprire un file di rapporto RDL che contiene le origini dati, i set di dati e i Parametri di report creati in anticipo. Una volta aperto con successo, tutte le sue informazioni sono disponibili per l'uso, ad esempio, i suoi set di dati sono elencati nella casella combinata DataSet pertinente. 

![todo:image_alt_text](adding-base-information-for-table_2.png)

1. Aprire un foglio di calcolo di Microsoft Excel e progettare l'intestazione della tabella come segue: 

   **Progettazione dell'intestazione della tabella** 

![todo:image_alt_text](adding-base-information-for-table_3.png)

1. Selezionare un set di dati ”EmpSalesDetail” dal controllo casella combinata set di dati pertinente. 

   **Selezione di un set di dati** 

![todo:image_alt_text](adding-base-information-for-table_4.png)

1. Tutti i campi del dataset 'EmpSalesDetail' sono elencati nella casella degli strumenti dei campi. Seleziona la cella sotto la colonna Ordernumber e quindi seleziona il campo 'SalesOrderNumber' dalla casella degli strumenti dei campi per inserire un marcatore dati nella cella. 

   **Inserimento di un campo** 

![todo:image_alt_text](adding-base-information-for-table_5.png)



**Il codice del campo mostra dove è stato inserito un campo.** 

![todo:image_alt_text](adding-base-information-for-table_6.png)

1. Ripeti il passaggio 5 per inserire i campi ProdCat e SubCat. 

   **Tutti i campi inseriti** 

![todo:image_alt_text](adding-base-information-for-table_7.png)
