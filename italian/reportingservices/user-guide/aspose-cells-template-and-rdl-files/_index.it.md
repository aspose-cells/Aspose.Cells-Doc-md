{{% alert color="primary" %}} 

Una definizione di report contiene il recupero dei dati e le informazioni sul layout per un report. Report Definition Language (RDL) è una rappresentazione XML per la definizione di report. RDL è uno schema aperto: gli sviluppatori possono estendere RDL con attributi ed elementi aggiuntivi. L'estensione file standard per i file di definizione del report è .rdl. Il suo tipo MIME è text/xml.

Un modello Aspose.Cells è una cartella di lavoro di Microsoft Excel che contiene informazioni sul layout del report e viene salvata in file RDL (formato) al termine del lavoro di progettazione del modello. Quando viene aperto un file RDL che non contiene un modello Aspose.Cells, solo le relative origini dati e query sono disponibili per l'uso in Aspose.Cells.Report.Designer. Quando viene aperto un file RDL che contiene un modello Aspose.Cells, il modello viene recuperato e aperto come cartella di lavoro in Microsoft Excel.
