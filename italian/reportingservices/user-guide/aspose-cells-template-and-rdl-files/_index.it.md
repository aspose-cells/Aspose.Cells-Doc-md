{{% alert color="primary" %}} 

Una definizione di report contiene informazioni di recupero dati e layout per un report. Il Linguaggio di Definizione Report (RDL) è una rappresentazione XML per la definizione di report. RDL è uno schema aperto: gli sviluppatori possono estendere RDL con attributi ed elementi aggiuntivi. L'estensione file standard per i file di definizione di report è .rdl. Il suo tipo MIME è text/xml. 

Un modello Aspose.Cells è un foglio di calcolo di Microsoft Excel che contiene informazioni sul layout del report e viene salvato in File RDL (Formato) quando il lavoro di progettazione del modello è completo. Quando un file RDL che non contiene un modello Aspose.Cells viene aperto, sono disponibili solo le sue origini dati e le query per l'uso in Aspose.Cells.Report.Designer. Quando un file RDL che contiene un modello Aspose.Cells viene aperto, il modello viene recuperato e aperto come un foglio di lavoro in Microsoft Excel.

{{% /alert %}}
