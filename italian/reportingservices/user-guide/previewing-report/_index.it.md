{{% alert color="primary" %}} 

L'anteprima di un report ti aiuta a essere sicuro che sia il report che desideri.

{{% /alert %}} 

Per visualizzare l'anteprima di un report:

1. Fare clic su **Visualizza** 

![todo:image_alt_text](previewing-report_1.png)

nella barra degli strumenti Aspose.Cells.Report.Designer.

1. Immettere l'URL del Server dei report e fare clic sul pulsante **Aggiorna**.

![todo:image_alt_text](previewing-report_2.png)




1. Selezionare un report e fare clic sul pulsante **Anteprima**. 

![todo:image_alt_text](previewing-report_3.png)

1. Nella finestra di dialogo, immettere i parametri del report e selezionare il formato del file di esportazione. Fare clic sul pulsante **OK** per esportare il report.

![todo:image_alt_text](previewing-report_4.png)
