 Le griglie sono configurate in**Aspose.Cells.ReportingServices.xml** così puoi controllare se un file Microsoft Excel esportato mostra le linee della griglia. Il valore predefinito è**spento**, il che significa che le linee della griglia non vengono visualizzate.

{{< highlight "java" >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
