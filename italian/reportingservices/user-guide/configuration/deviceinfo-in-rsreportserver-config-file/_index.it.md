 La sezione DeviceInfo in**rsreportserver.config** accetta i seguenti parametri che influenzano il comportamento di Aspose.Cells:

- **Estensione del file** : quando il valore è**nullo**, l'estensione del file di report esportato è il valore predefinito. Quando il valore non è nullo, l'estensione del report viene impostata sul valore.
- **SimplePageHeaders** : quando il valore è**VERO** , l'elemento di intestazione del report viene visualizzato come intestazione di pagina Excel Microsoft. Il valore predefinito è**falso**.
- **SimplePageFooters** : quando**VERO** , l'elemento del piè di pagina del report viene visualizzato come piè di pagina di Excel Microsoft. Il valore predefinito è**VERO**.
- **PutoutHeader** : quando**VERO** (impostazione predefinita), l'elemento dell'intestazione del report viene esportato. quando**falso**, l'elemento dell'intestazione del report non viene esportato. Supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **PutoutFooter** : quando**VERO** (impostazione predefinita), viene esportato l'elemento a piè di pagina del report. quando**falso**non è. Supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **FillTableGroupHeaderForSimpleOutPut**: **falso** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **NoOutPutTotalForSimpleOutPut**: **falso** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **NoOutPutGroupForSimpleOutPut**: **falso** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **NoDoGroupPageForSimpleOutPut**: **VERO** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **NoDoPageForSimpleOutPut**: **VERO** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **Delimitatore di campo**: imposta i delimitatori di campo. Il valore supporta le estensioni CSV e TXT.
