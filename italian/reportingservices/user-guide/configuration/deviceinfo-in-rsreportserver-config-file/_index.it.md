La sezione DeviceInfo in **rsreportserver.config** prende i seguenti parametri che influenzano il comportamento di Aspose.Cells:

- **FileExtension**: quando il valore è **nullo**, l'estensione del file del report esportato è il valore predefinito. Quando il valore non è nullo, l'estensione del report viene impostata sul valore.
- **SimplePageHeaders**: quando il valore è **true**, l'oggetto intestazione del report viene reso come intestazione di pagina di Microsoft Excel. Il valore predefinito è **false**.
- **SimplePageFooters**: quando è **true**, l'oggetto piè di pagina del report è reso come piè di pagina di Microsoft Excel. Il valore predefinito è **true**.
- **PutoutHeader**: quando è **true** (predefinito), l'oggetto intestazione del report viene esportato. Quando è **false**, l'oggetto intestazione del report non viene esportato. Supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **PutoutFooter**: quando è **true** (predefinito), l'oggetto piè di pagina del report viene esportato. Quando è **false**, non lo è. Supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **FillTableGroupHeaderForSimpleOutPut**: **false** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **NoOutPutTotalForSimpleOutPut**: **false** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **NoOutPutGroupForSimpleOutPut**: **false** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **NoDoGroupPageForSimpleOutPut**: **true** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **NoDoPageForSimpleOutPut**: **true** per impostazione predefinita. Il valore supporta solo l'estensione Excel 2007 XLSX (solo dati).
- **FieldDelimiter**: imposta i delimitatori di campo. Il valore supporta le estensioni CSV e TXT.
