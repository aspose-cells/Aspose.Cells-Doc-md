- **FileExtension** 
  Quando il valore è nullo, il nome dell'estensione del file del report esportato sarà il valore predefinito. Quando il valore non è nullo, l'estensione del nome del file del report esportato è il valore.
- **SimplePageHeaders** 
  Quando il valore è true, renderizza l'elemento di intestazione del report nell'intestazione della pagina di Excel. Il valore predefinito è false.
- **SimplePageFooters** 
  Quando il valore è vero, rende l'elemento del piè di pagina del report nel piè di pagina di Excel. Il valore predefinito è vero.
- **PutoutHeader** 
  Quando il valore è vero, esporta l'elemento dell'intestazione del report. Quando il valore è falso, non esporta l'elemento dell'intestazione del report. Il valore predefinito è vero. Il valore supporta solo l'estensione Excel2007Xlsx (solo dati).
- **PutoutFooter** 
  Quando il valore è vero, esporta l'elemento del piè di pagina del report. Quando il valore è falso, non esporta l'elemento del piè di pagina del report. Il valore predefinito è vero. Il valore supporta solo l'estensione Excel2007Xlsx (solo dati).
- **FillTableGroupHeaderForSimpleOutPut** 
  Il valore predefinito è falso. Il valore supporta solo l'estensione Excel2007Xlsx (solo dati).
- **NoOutPutTotalForSimpleOutPut** 
  Il valore predefinito è falso. Il valore supporta solo l'estensione Excel2007Xlsx (solo dati).
- **NoOutPutGroupForSimpleOutPut** 
  Il valore predefinito è falso. Il valore supporta solo l'estensione Excel2007Xlsx (solo dati).
- **NoDoGroupPageForSimpleOutPut** 
  Il valore predefinito è vero. Il valore supporta solo l'estensione Excel2007Xlsx (solo dati).
- **NoDoPageForSimpleOutPut** 
  Il valore predefinito è vero. Il valore supporta solo l'estensione Excel2007Xlsx (solo dati).
- **FieldDelimiter** 
  Imposta i delimitatori campi. Il valore supporta le estensioni CSV e TXT.
