- **Estensione del file** 
 Quando il valore è null, il nome dell'estensione del file di report esportato sarà il valore predefinito. Quando il valore non è nullo, il nome dell'estensione del file di report esportato è valore.
- **SimplePageHeaders** 
 Quando il valore è true, esegue il rendering dell'elemento dell'intestazione del report nell'intestazione della pagina di Excel. Il valore predefinito è falso.
- **SimplePageFooters** 
 Quando il valore è true, esegue il rendering dell'elemento del piè di pagina del report nel piè di pagina di Excel. Il valore predefinito è vero.
- **PutoutHeader** 
 Quando il valore è true, esporterà l'elemento dell'intestazione del report. Quando il valore è false, non esporta l'elemento dell'intestazione del report. Il valore predefinito è vero. Il valore supporta solo l'estensione Excel2007Xlsx(solo dati).
- **PutoutFooter** 
Quando il valore è true, esporterà l'elemento a piè di pagina del report. Quando il valore è false, non esporta l'elemento a piè di pagina del report. Il valore predefinito è vero. Il valore supporta solo l'estensione Excel2007Xlsx(solo dati).
- **FillTableGroupHeaderForSimpleOutPut** 
 Il valore predefinito è falso. Il valore supporta solo l'estensione Excel2007Xlsx(solo dati).
- **NoOutPutTotalForSimpleOutPut** 
 Il valore predefinito è falso. Il valore supporta solo l'estensione Excel2007Xlsx(solo dati).
- **NoOutPutGroupForSimpleOutPut** 
 Il valore predefinito è falso. Il valore supporta solo l'estensione Excel2007Xlsx(solo dati).
- **NoDoGroupPageForSimpleOutPut** 
 Il valore predefinito è vero. Il valore supporta solo l'estensione Excel2007Xlsx(solo dati).
- **NoDoPageForSimpleOutPut** 
 Il valore predefinito è vero. Il valore supporta solo l'estensione Excel2007Xlsx(solo dati).
- **Delimitatore di campo** 
 Imposta i delimitatori di campo. Il valore supporta le estensioni CSV e TXT.
