 Cosa è migliorato?

 Cosa è stato risolto?

- Risolto il bug di installazione quando più istanze di SQL Server Reporting Serverices sono installate nella stessa macchina
- Risolto il bug della formula di inserimento quando non si sceglie un set di dati
