 introduzione

 Aspose.Cells per Reporting Service v1.0.1 include miglioramenti delle funzionalità e correzioni di bug da

 la prima uscita.

 Cosa c'è di nuovo?

- Supportato per esportare la matrice
- Supportato per esportare file CSV e TabDelimited
- Supportata l'immagine di sfondo.

 Cosa è stato risolto?

- Risolto il bug quando image.ImageData è nullo.
- Risolto il bug quando il valore della casella di testo contiene il formato numerico.
- Risolto il bug quando la matrice è raggruppata.
