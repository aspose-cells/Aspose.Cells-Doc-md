 Questa pagina contiene le note di rilascio per[Aspose.Cells for Reporting Services 1.5.2](https://downloads.aspose.com/cells/reportingservices/new-releases/aspose.cells-for-reporting-services-1.5.2/)

{{% /alert %}} 

 Siamo felici di annunciare Aspose.Cells for Reporting Services 1.5.2!

 Cosa è cambiato?



-  Ottimizza il rendering per report di dati di grandi dimensioni.
- Risolve un problema di installazione per Windows 7.
-  Risolve il paging per l'elemento del report Tablix.
-  Gestisce il rendering di più gruppi per Tablix.
-  Riduce il tasso di utilizzo della memoria.
