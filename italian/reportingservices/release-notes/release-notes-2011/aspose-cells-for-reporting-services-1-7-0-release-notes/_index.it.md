 Questa pagina contiene le note di rilascio per[Aspose.Cells per Reporting Services 1.7.0](https://downloads.aspose.com/cells/reportingservices/new-releases/aspose.cells-for-reporting-services-1.7.0/)

{{% /alert %}} 

7.0! 

 Cosa c'è di nuovo?

- Aspose.Cells per Reporting Services Designer supporta la creazione di modelli di matrice in Excel.
- Supporta la formattazione condizionale per gli elementi del report tabella e matrice.
- Aggiunge il rendering dell'elemento del report Tablix solo per i formati CSV e TXT in SQL Server 2008 Reporting Services.

 Cosa è cambiato?

- Ottimizza la configurazione della crittografia nel file Aspose.Cells.ReportingServices.xml.

 Cosa è stato risolto?



- Risolto il problema Aspose.Cells per il progettista di report di Reporting Services e il bug del formato del report.
- Risolto il bug relativo alla fila nascosta.
- Corretto l'algoritmo di unione delle righe.
