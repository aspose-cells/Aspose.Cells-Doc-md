{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services è disponibile come valutazione gratuita e senza limiti di tempo dalla [pagina di download](https://downloads.aspose.com/cells/reportingservices). La versione di valutazione e le versioni con licenza del prodotto sono le stesse.

Quando sei soddisfatto della versione di valutazione, acquista una licenza. Assicurati di comprendere e accettare i termini e le condizioni dell'abbonamento.

{{% /alert %}} 
### **Ottenere e utilizzare la licenza**
La licenza sarà disponibile per il download dalla pagina dell'ordine quando l'ordine verrà pagato.

Il file della licenza è un file XML in chiaro firmato digitalmente. Il file della licenza contiene informazioni come il nome del cliente, il prodotto acquistato e il tipo di licenza. Non modificare il contenuto del file della licenza in quanto ciò lo rende non valido.

Scarica la licenza sul tuo computer e copiala nella cartella che deve essere una delle seguenti cartelle.

- $[Cartella di Installazione]\Licenza\
- $[MSSQL Server Reporting Services]\ReportServer\Bin\
- $[MSSQL Server Reporting Services]\ReportServer\

Il nome del file della licenza può essere uno dei seguenti in base al tipo di abbonamento:

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic", "Aspose.Total.lic
- Aspose.Cells.Reporting.Services - Abbonamento Sito Sviluppatore.lic
- Aspose.Cells.Reporting.Services - Abbonamento Sviluppatore.lic
- Aspose.Cells.Reporting.Services - Abbonamento Multi-deploy.lic
- Aspose.Cells.Reporting.Services - Abbonamento Single Server Deployment.lic

Per verificare che il file della licenza sia stato installato correttamente, esporta qualsiasi report come un documento di Microsoft Excel. Se il documento non contiene alcun foglio di lavoro con watermark alla fine, il file della licenza è stato attivato correttamente.

Aspose.Cells for Reporting Services inserisce un foglio di lavoro con watermark di valutazione quando si lavora in modalità di valutazione. Quando è presente un file di licenza valido, ad esempio quando **Aspose.Cells.ReportingServices.lic** si trova nella cartella $[Installation Folder]\Licenza\, non è presente alcun foglio di lavoro con watermark di valutazione.
