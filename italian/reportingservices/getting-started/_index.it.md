Aspose.Cells for Reporting Services è composto da **due parti** come segue:

- ***Aspose.Cells.Report.Designer***

&nbsp;&nbsp;&nbsp;&nbsp; Questo è uno strumento di progettazione di report integrato con MS Excel in formato di Add-Ins… . Con Aspose.Cells.Report.Designer gli utenti possono progettare un modello di report in MS Excel anziché in Visual Studio. Il modello di report viene quindi incorporato in un file RDL dei Reporting Services e può essere pubblicato nel Report Server. Gli utenti possono anche visualizzare in anteprima ed esportare il report in Aspose.Cells.Report.Designer.

- ***Aspose.Cells.Renderer for Reporting Services***

&nbsp;&nbsp;&nbsp;&nbsp; Aspose.Cells.Renderer for Reporting Services è effettivamente un'implementazione dell'Estensione del Rendering dei Reporting Services di Microsoft. Può generare report RDL progettati sia con Visual Studio che con Aspose.Cells.Report.Designer.
