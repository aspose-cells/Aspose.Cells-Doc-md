 Aspose.Cells per Reporting Services è costituito**due parti** come segue:

- ***Aspose.Cells.Report.Designer***

&nbsp;&nbsp;&nbsp;&nbsp;Si tratta di uno strumento di progettazione di report integrato con MS Excel sotto forma di componenti aggiuntivi…. Con Aspose.Cells.Report.Designer gli utenti possono progettare un modello di report in MS Excel anziché in Visual Studio. Il modello di report viene quindi incorporato nel file RDL di Reporting Services e può essere pubblicato nel server di report. Gli utenti possono anche visualizzare in anteprima ed esportare il report in Aspose.Cells.Report.Designer.

- ***Aspose.Cells.Renderer per Reporting Services***

&nbsp;&nbsp;&nbsp;&nbsp;Aspose.Cells.Renderer per Reporting Services è in realtà un'implementazione di Microsoft Reporting Services Rendering Extension. Può eseguire il rendering di report RDL progettati sia da Visual Studio che da Aspose.Cells.Report.Designer.
