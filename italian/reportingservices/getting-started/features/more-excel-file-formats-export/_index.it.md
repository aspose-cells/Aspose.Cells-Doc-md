{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services consente di fornire report Microsoft Excel nativi tramite Microsoft SQL Server 2005 e 2008 Reporting Services. A differenza del renderer integrato per i report basati su Microsoft Excel, Aspose.Cells for Reporting Services fornisce un set di funzionalità esteso che converte i report RDL in: formato XLS binario di Excel 97-2003, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 XLSB, Formato HTML e ODS.

{{% /alert %}} 

Aspose.Cells for Reporting Services supporta più formati di file rispetto agli esportatori nativi. Di seguito sono riportate schermate di report XLS di esempio esportati con Aspose.Cells.
