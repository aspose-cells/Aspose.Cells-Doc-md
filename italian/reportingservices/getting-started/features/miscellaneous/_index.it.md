{{% alert color="primary" %}} 

Di seguito sono elencate alcune caratteristiche efficaci di Aspose.Cells for Reporting Services

- Aspose.Cells for Reporting Services supporta tabelle con righe raggruppate e piè di pagina. Il raggruppamento di righe e piè di pagina può essere configurato durante la progettazione come altri elementi del report.
- Aspose.Cells for Reporting Services supporta parametri e variabili globali come sorgenti dati del report.
- Gli utenti possono utilizzare Microsoft Query per creare sorgenti dati e query. Microsoft Query è uno strumento ampiamente utilizzato che la maggior parte degli utenti dovrebbe conoscere e familiarizzare.
- Aspose.Cells for Reporting Services supporta sia sorgenti dati specifiche del report che sorgenti dati condivise create in anticipo sul Server dei report. Inoltre, è possibile utilizzare sorgenti dati specifiche del report durante la progettazione e specificare sorgenti dati condivise come sorgenti dati del report durante la pubblicazione di un report.
- Aspose.Cells for Reporting Service è integrato in modo trasparente con Microsoft Reporting Services, quindi gli utenti possono sfruttare appieno la gestione dei report, la sicurezza e le opzioni di distribuzione di Reporting Services.

{{% /alert %}}
