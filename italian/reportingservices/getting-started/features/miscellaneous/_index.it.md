{{% alert color="primary" %}} 

Di seguito sono elencate alcune caratteristiche efficaci di Aspose.Cells for Reporting Services

- Aspose.Cells for Reporting Services supporta tabelle con righe raggruppate e piè di pagina. Il raggruppamento di righe e piè di pagina può essere impostato in fase di progettazione come altri elementi del report.
- Aspose.Cells for Reporting Services supporta parametri e variabili globali come origini dati del report.
- Gli utenti possono utilizzare Microsoft Query per creare origini dati e query. Microsoft Query è uno strumento ampiamente utilizzato che la maggior parte degli utenti dovrebbe conoscere e conoscere.
- Aspose.Cells for Reporting Services supporta sia le origini dati specifiche del report sia le origini dati condivise create in precedenza nel server di report. Inoltre, è possibile utilizzare origini dati specifiche del report in fase di progettazione e specificare origini dati condivise come origini dati del report durante la pubblicazione di un report.
- Aspose.Cells per Reporting Service è perfettamente integrato con Microsoft Reporting Services e quindi gli utenti possono sfruttare appieno le opzioni di gestione, sicurezza e consegna dei report di Reporting Services.
