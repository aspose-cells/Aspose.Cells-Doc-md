Una volta installato, Aspose.Cells for Reporting Services è registrato come componente a 32 bit per impostazione predefinita. Non può caricare correttamente i componenti aggiuntivi di Excel in Microsoft Excel x64. Quindi è necessario registrare nuovamente Aspose.Cells.ReportingServices.dll come componente/i a 64 bit.
