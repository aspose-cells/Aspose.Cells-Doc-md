Quando installato, Aspose.Cells for Reporting Services viene registrato come componente a 32 bit per impostazione predefinita. Non è in grado di caricare correttamente gli add-in di Excel in Microsoft Excel x64. È quindi necessario registrare nuovamente il file Aspose.Cells.ReportingServices.dll come componente a 64 bit.
