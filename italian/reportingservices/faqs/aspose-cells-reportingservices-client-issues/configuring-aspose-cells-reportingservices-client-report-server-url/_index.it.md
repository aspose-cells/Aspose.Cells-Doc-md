Per configurare un URL del server di report client:

1. Fai clic sul pulsante **Apri report remoto** e ottieni la finestra di dialogo **Recupera report**. 

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_1.png)

1. Fai clic sul pulsante **Configura** e ottieni la finestra di dialogo **Impostazioni dell'URL del server di report**. 

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_2.png)

1. Inserisci le informazioni di configurazione del server di report (URL, nome utente e password).
   1. L'URL del server di report deve essere nel formato http://host/ReportServer. Ad esempio http://172.16.204.11/ReportServer.
   1. Il nome utente è il nome utente del server Windows. Ad esempio, Amministratore.
   1. La password è la password dell'utente del server Window. Per esempio, 123456.
   1. Il valore predefinito del dominio è nullo.
1. Fare clic sul pulsante **Test** per verificare le informazioni di configurazione.
1. Fare clic su **Salva** e salvare le informazioni di configurazione del server dei report.

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_3.png)




1. Fare clic sul pulsante **Modifica** per modificare le informazioni di configurazione.
