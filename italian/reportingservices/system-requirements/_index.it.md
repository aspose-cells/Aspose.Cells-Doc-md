{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services (incluso il client Aspose.Cells.ReportingServices, componenti aggiuntivi per Microsoft Excel e server Aspose.Cells.ReportingServices) deve essere installato correttamente. Inoltre, Microsoft SQL Server dovrebbe essere installato prima di installare Aspose.Cells for Reporting Services.

{{% /alert %}} 
### **Sistema Office**
Le seguenti versioni di Microsoft Excel sono supportate. Inoltre, è necessario installare patch aggiuntive per superare eventuali problemi per Microsoft Office 2000, 2002, XP o 2003.

- Microsoft Excel 2000
  Installare **lockbackRegKey.msi** per correggere un problema CLR su macchine Microsoft Office più datate rispetto a Office 2003.
- Microsoft Excel 2002
  Installare **lockbackRegKey.msi** per correggere un problema CLR su macchine Microsoft Office più datate rispetto a Office 2003.
- Microsoft Excel 2003
  Aggiornare sempre a Office 2003 (KB907417). Questo aggiornamento corregge un potenziale problema in cui un componente aggiuntivo, una smart tag o un smart document potrebbero non caricarsi se utilizza il Microsoft .NET Framework 2.0.
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
- Microsoft Excel 2016
- Microsoft Excel 2019
- Microsoft Excel 365
