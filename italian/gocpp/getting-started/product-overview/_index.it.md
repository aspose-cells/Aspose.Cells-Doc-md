**Benvenuto nella sezione documentazione Aspose.Cells for Go via C++**

Aspose.Cells for Go via C++ è una libreria Go nativa che permette agli sviluppatori di Go di creare, manipolare e convertire fogli di calcolo programmaticamente senza necessità di Office Automation o Microsoft Excel.

Il supporto di Aspose.Cells for Go via C++ include formati di fogli di calcolo come il formato Workbook di Excel 97-2003 e i formati Open Office XML.

{{% /alert %}}

## **Vantaggi**

Utilizzando Aspose.Cells for Go via C++ nelle tue applicazioni, ottieni i seguenti vantaggi:

- Funzionalità avanzate
- Indipendenza dalla piattaforma
- Prestazioni e scalabilità

### **Funzionalità avanzate**

Aspose.Cells for Go via C++ offre un’ampia gamma di funzionalità per creare, convertire e manipolare fogli di calcolo:

- Proprietà incorporate
- Proprietà personalizzate
- Temi
- Stili e formattazione
- Convalida dei dati
- Formattazione condizionale
- Collegamento ipertestuale
- Filtro automatico
- Impostazioni di pagina
- Lettura, scrittura e calcolo delle formule
- Raggruppare righe e colonne
- Tabella pivot
- Tabella

### **Indipendenza dalla piattaforma**

Usa Aspose.Cells for Go via C++ per sviluppare qualsiasi tipo di applicazione Go a 64 bit.

### **Prestazioni e scalabilità**

Aspose.Cells for Go via C++ è progettato per funzionare altrettanto bene lato server o client. Aspose.Cells for Go via C++ è un’assemblaggio nativo che può essere distribuito semplicemente copiandolo. Non devi preoccuparti di altri servizi o moduli.
