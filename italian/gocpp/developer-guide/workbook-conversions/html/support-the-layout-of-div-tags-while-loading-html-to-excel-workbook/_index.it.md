{{% alert color="primary" %}} 

Normalmente, il layout dei tag DIV viene ignorato durante il caricamento di HTML in un oggetto workbook Excel. Tuttavia, se vuoi che il layout dei tag DIV venga preservato, imposta la proprietà [GetSupportDivTag()](https://reference.aspose.com/cells/go-cpp/htmlloadoptions/getsupportdivtag/) su **true**. Il valore predefinito di questa proprietà è **false**.
