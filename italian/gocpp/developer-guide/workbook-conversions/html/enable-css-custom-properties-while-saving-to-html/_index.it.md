## **Possibili Scenari di Utilizzo**

Quando salvi il tuo file Excel in HTML, nel caso in cui ci siano più occorrenze di un'immagine base64, con proprietà personalizzate i dati dell'immagine devono essere salvati una sola volta così da migliorare le performance dell'HTML risultante. Usa la proprietà [**HtmlSaveOptions.GetEnableCssCustomProperties()**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/getenablecsscustomproperties/) e impostala su **true** durante il salvataggio in HTML.
