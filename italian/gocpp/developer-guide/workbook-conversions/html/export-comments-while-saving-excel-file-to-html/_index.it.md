## **Possibili Scenari di Utilizzo**

Quando salvi il tuo file Excel in HTML, i commenti non vengono esportati. Tuttavia, Aspose.Cells offre questa funzionalità usando la proprietà [**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/isexportcomments/). Se la imposti su **true**, l'HTML mostrerà anche i commenti presenti nel file Excel.
