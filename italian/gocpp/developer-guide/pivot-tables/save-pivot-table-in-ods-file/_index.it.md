Aspose.Cells offre la possibilità di salvare le tabelle pivot in file ODS. Per questo, puoi convertire un workbook con una tabella pivot esistente o creare una nuova tabella pivot e salvare il file nel formato ODS. Assicurati di chiamare [**PivotTable::CalculateData()**](https://reference.aspose.com/cells/go-cpp/pivottable/calculatedata/) prima di salvare per garantire che la tabella pivot venga renderizzata nel file ODS di output.
