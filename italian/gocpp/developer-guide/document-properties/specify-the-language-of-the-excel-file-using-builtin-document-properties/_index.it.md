## **Possibili Scenari di Utilizzo**

Puoi cambiare la lingua di un file Excel facendo clic destro sul file, selezionando Proprietà > Dettagli e modificando il campo Lingua. Usa la proprietà [**BuiltInDocumentPropertyCollection.GetLanguage()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getlanguage/) per cambiarla programmaticamente usando le API di Aspose.Cells.
