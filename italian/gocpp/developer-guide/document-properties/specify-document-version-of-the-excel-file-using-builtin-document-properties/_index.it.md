## **Possibili Scenari di Utilizzo**

Puoi modificare **Numero di versione** di un file Excel facendo clic destro sul file, selezionando Proprietà > Dettagli e modificando il campo **Numero di versione**. Usa la proprietà [**BuiltInDocumentPropertyCollection.GetDocumentVersion()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getdocumentversion/) per cambiarlo programmaticamente usando le API di Aspose.Cells.
