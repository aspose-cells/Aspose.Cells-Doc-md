{{% alert color="primary" %}}

Microsoft Excel 2013 consente agli utenti di abilitare o disabilitare il testo all'interno delle etichette dati del grafico. Per impostazione predefinita, il testo all'interno delle etichette dati del grafico è nello stato di testo a capo.

Aspose.Cells fornisce una proprietà [**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/go-cpp/datalabels/istextwrapped/) che è possibile impostare su True o False per abilitare o disabilitare il testo a capo delle etichette dati rispettivamente.
