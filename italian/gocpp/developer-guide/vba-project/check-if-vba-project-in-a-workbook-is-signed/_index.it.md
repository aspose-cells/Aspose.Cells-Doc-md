{{% alert color="primary" %}}

Puoi verificare se il tuo progetto VBA è firmato o meno usando Microsoft Excel tramite il comando del menu **Strumenti > Firma Digitale...**. Allo stesso modo, puoi verificarlo programmaticamente usando Aspose.Cells [**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/go-cpp/vbaproject/issigned/).

{{% /alert %}}
