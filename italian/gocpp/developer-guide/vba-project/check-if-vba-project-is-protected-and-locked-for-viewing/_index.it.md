## Verifica se il Progetto VBA è Protetto e Bloccato per la Visualizzazione in C++

Aspose.Cells permette di verificare se il Progetto VBA (Visual Basic for Applications) di un file Excel è protetto e bloccato per la visualizzazione. Per questo, l’API fornisce la proprietà [**VbaProject.GetIslockedForViewing()**](https://reference.aspose.com/cells/go-cpp/vbaproject/getislockedforviewing/). Se è bloccato per la visualizzazione, allora la proprietà [**VbaProject.GetIslockedForViewing()**](https://reference.aspose.com/cells/go-cpp/vbaproject/getislockedforviewing/) restituisce **true**.
