{{% alert color="primary" %}}

**Le formattazioni condizionali a scala di 2 colori** e **a scala di 3 colori** vengono aggiunte allo stesso modo tranne che si differenziano per la proprietà [**FormatCondition.GetIs3ColorScale()**](https://reference.aspose.com/cells/go-cpp/colorscale/getis3colorscale/). Questa proprietà è **false** per le formattazioni condizionali a scala di 2 colori e **true** per le formattazioni condizionali a scala di 3 colori.
