1. I Microsoft Excel, klicka på **Ändra attribut** på Aspose.Cells kommandofält.
   Ändra Rapportdialogen visas.
1. Välj en tabellrapportpost.
1. Klicka på fliken **Dölj**. 

![todo:image_alt_text](working-with-table-hide-attributes_1.png)




1. Klicka på **Ändra** för att göra fliken redigerbar. 

![todo:image_alt_text](working-with-table-hide-attributes_2.png)




1. Redigera döljinformationen.
1. Klicka på **Fx**.
   **Redigera Formel** dialog visas.

![todo:image_alt_text](working-with-table-hide-attributes_3.png)





1. Redigera uttrycksvärdet.
1. Klicka på **OK** för att spara formeln i Ändra Rapportdialogen. 

![todo:image_alt_text](working-with-table-hide-attributes_4.png)




1. Redigera annan döljinformation.
1. Klicka på **Lägg till** för att lägga till döljinformationen. 

![todo:image_alt_text](working-with-table-hide-attributes_5.png)




1. Klicka på **Verkställ** för att lägga till döljinformationen i rapportcachen.
1. Klicka på **Stäng** för att stänga dialogrutan. 

![todo:image_alt_text](working-with-table-hide-attributes_6.png)




1. Spara och publicera rapporten.
