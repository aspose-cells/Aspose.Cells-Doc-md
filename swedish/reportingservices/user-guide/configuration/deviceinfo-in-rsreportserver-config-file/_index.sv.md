 Avsnittet DeviceInfo i**rsreportserver.config** tar följande parametrar som påverkar Aspose.Cells' beteende:

- **FileExtension** : när värdet är**null**, är den exporterade rapportens filtillägg standardvärdet. När värdet inte är null, ställs rapportens tillägg till värdet.
- **SimplePageHeaders** : när värdet är**Sann** renderas rapporthuvudet som ett Microsoft Excel-sidhuvud. Standardvärdet är**falsk**.
- **SimplePageFooters** : när**Sann** , renderas rapportsidfoten som en Microsoft Excel-sidfot. Standardvärdet är**Sann**.
- **PutoutHeader** : när**Sann** (standard) exporteras rapporthuvudet. När**falsk**, är rapporthuvudet inte exporterat. Stöder endast tillägget Excel 2007 XLSX (endast data).
- **PutoutFooter** : när**Sann** (standard) exporteras rapportsidfoten. När**falsk**, det är det inte. Stöder endast tillägget Excel 2007 XLSX (endast data).
- **FillTableGroupHeaderForSimpleOutPut**: **falsk** som standard. Värdet stöder endast tillägget Excel 2007 XLSX (endast data).
- **NoOutPutTotalForSimpleOutPut**: **falsk** som standard. Värdet stöder endast tillägget Excel 2007 XLSX (endast data).
- **NoOutPutGroupForSimpleOutPut**: **falsk** som standard. Värdet stöder endast tillägget Excel 2007 XLSX (endast data).
- **NoDoGroupPageForSimpleOutPut**: **Sann** som standard. Värdet stöder endast tillägget Excel 2007 XLSX (endast data).
- **NoDoPageForSimpleOutPut**: **Sann** som standard. Värdet stöder endast tillägget Excel 2007 XLSX (endast data).
- **Fältavgränsare**: ställer in fältavgränsare. Värdet stöder tilläggen CSV och TXT.
