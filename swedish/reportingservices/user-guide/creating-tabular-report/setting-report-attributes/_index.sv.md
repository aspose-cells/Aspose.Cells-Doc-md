{{% alert color="primary" %}} 

För att skapa en tabellerapport måste du ställa in rapportattribut innan du publicerar rapporten till rapportservern. Rapportattributen innehåller information om rapportens namn, omfattning, data gruppering, sortering och så vidare.

{{% /alert %}} 

För att ställa in attributen för en tabellerapport:

1. Klicka på **Ange attribut** på verktygsfältet Aspose.Cells.Report.Designer (

![todo:image_alt_text](setting-report-attributes_1.png)

).

1. Mata in rapportnamnet i dialogrutan. 

   **Ange rapportnamnet** 

![todo:image_alt_text](setting-report-attributes_2.png)




1. Klicka på **Nästa**.
1. Välj en omfattning för rapporten.
   Omfattningen börjar med den första cellen i tabellens datarad och slutar vid den sista cellen i sidfotsraden eller, om det inte finns några sidfötter, den sista cellen i gruppdataläget.
   Start- och slutcellernas index visas automatiskt i fältet **Omfattning**. Här inkluderar omfattningen celler från D4 till H8. 

   **Definiera rapportomfattningen** 

![todo:image_alt_text](setting-report-attributes_3.png)




1. Klicka på **Nästa** för att fortsätta.
   Alla datasetens fält listas i vänster panel i dialogrutan. 
1. Välj fält för att gruppera datan efter. 

   **Välja fält för att gruppera data** 

![todo:image_alt_text](setting-report-attributes_4.png)




1. Klicka på **Nästa** för att fortsätta.
1. Välj fält för att sortera datan efter. 

   **Välja fält för att sortera data efter** 

![todo:image_alt_text](setting-report-attributes_5.png)




1. Klicka på **Nästa**.

   En sammanfattning av rapporten visas.
   **Rapportsammanfattning** 

![todo:image_alt_text](setting-report-attributes_6.png)

1. Klicka på **Slutför**.
