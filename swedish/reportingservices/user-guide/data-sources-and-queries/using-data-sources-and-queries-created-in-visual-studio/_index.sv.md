{{% alert color="primary" %}} 

Du kan skapa datakällor och frågor i Visual Studio och spara dem till en RDL-fil. När RDL-filen öppnas i Aspose.Cells.Report.Designer är dess datakällor och frågor tillgängliga för användning. Till exempel kan vi skapa en fråga med namnet EmpSalesDetail i Visual Studio och spara den som en RDL-fil.

**RDL-filen EmpSalesDetail** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_1.png)

Den här artikeln visar hur man arbetar med en RDL-fil i Aspose.Cells.Report.Designer.

{{% /alert %}} 

Du kan öppna RDL-filer med Aspose.Cells.Report.Designer och arbeta med dem:

1. Öppna Microsoft Excel.
1. Från verktygsfältet Aspose.Cells.Report.Designer väljer du **Öppna rapport**. 

   **Knappen Öppna rapport** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Välj RDL-filen och öppna den.
   När RDL-filen är öppnad i Aspose.Cells.Report.Designer visas frågorna i **Datamängd**-listan. 

   **Tillgängliga datamängder** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_3.png)
