{{% alert color="primary" %}} 

- Om du har en tidigare version av Aspose.Cells for Reporting Services installerad, vänligen avinstallera den först. Ser[Avinstallerar Aspose.Cells for Reporting Services](/cells/sv/reportingservices/uninstalling-aspose-cells-for-reporting-services/) ämne om hur man tar bort komponenten.
- [Besök nedladdningssidan Aspose.Cells for Reporting Services](https://downloads.aspose.com/cells/reportingservices) och ladda ner Aspose.Cells for Reporting Services MSI-installationsprogrammet med den senaste versionen av komponenten.
 - Aspose.Cells för ReportingServices Msi stöder endast Microsoft SQL Server 2008 Reporting Services och senare produkter.
- Kör den nedladdade filen och följ installationsanvisningarna.

{{% /alert %}} 
**Installationsprocess**
Så här installerar du Aspose.Cells med MSI-installationsprogrammet:

1.  Välkomstskärmen är den första skärmen i installationen. Klick**Nästa**.

![todo:image_alt_text](using-msi-installer_1.png)




1.  Läs och acceptera licensavtalet och klicka**Nästa**.

![todo:image_alt_text](using-msi-installer_2.png)




1.  Välj installationsmappen och klicka**Nästa**. 

![todo:image_alt_text](using-msi-installer_3.png)




1.  Bekräfta att du vill installera Aspose.Cells for Reporting Services genom att klicka på**Installera**. 

![todo:image_alt_text](using-msi-installer_4.png)



 Installationsförloppet visas på skärmen.

![todo:image_alt_text](using-msi-installer_5.png)




1.  Klicka på när installationen är klar**Avsluta** för att stänga installationsprogrammet.

![todo:image_alt_text](using-msi-installer_6.png)
