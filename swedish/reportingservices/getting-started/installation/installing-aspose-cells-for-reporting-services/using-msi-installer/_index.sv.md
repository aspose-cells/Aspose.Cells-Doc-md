{{% alert color="primary" %}} 

- Om du har en tidigare version av Aspose.Cells for Reporting Services installerad, avinstallera den först. Se [Avinstallera Aspose.Cells for Reporting Services](/cells/sv/reportingservices/uninstalling-aspose-cells-for-reporting-services/) ämne om hur du tar bort komponenten.
- [Besök Aspose.Cells for Reporting Services nedladdningssida](https://downloads.aspose.com/cells/reportingservices) och ladda ner Aspose.Cells for Reporting Services MSI-installatören med den senaste versionen av komponenten.
    - Aspose.Cells för ReportingServices Msi stöder endast Microsoft SQL Server 2008 Reporting Services och senare produkter.
- Kör den nedladdade filen och följ installatörsanvisningarna.

{{% /alert %}} 
**Installationsprocess**
För att installera Aspose.Cells med MSI-installatören:

1. Välkommen skärmen är den första skärmen i installationen. Klicka på **Nästa**.

![todo:image_alt_text](using-msi-installer_1.png)




1. Läs och acceptera licensavtalet och klicka på **Nästa**.

![todo:image_alt_text](using-msi-installer_2.png)




1. Välj installationsmappen och klicka på **Nästa**. 

![todo:image_alt_text](using-msi-installer_3.png)




1. Bekräfta att du vill installera Aspose.Cells for Reporting Services genom att klicka på **Installera**. 

![todo:image_alt_text](using-msi-installer_4.png)



Installationsframstegen visas på skärmen. 

![todo:image_alt_text](using-msi-installer_5.png)




1. När installationen är klar, klicka på **Slutför** för att stänga installatören.

![todo:image_alt_text](using-msi-installer_6.png)
