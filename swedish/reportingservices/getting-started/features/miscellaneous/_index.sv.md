Några effektiva egenskaper hos Aspose.Cells for Reporting Services listas nedan

- Aspose.Cells for Reporting Services stöder tabeller med grupperade rader och sidfot. Gruppering av rader och sidfot kan ställas in vid design-tid precis som andra rapportobjekt.
- Aspose.Cells for Reporting Services stödjer parametrar och globala variabler som rapportdatakällor.
- Användare kan använda Microsoft Query för att skapa datakällor och frågor. Microsoft Query är ett väl använt verktyg som de flesta användare bör vara bekanta med.
- Aspose.Cells for Reporting Services stödjer både rapportspecifika datakällor och delade datakällor som skapats på Rapportservern i förväg. Dessutom är det möjligt att använda rapportspecifika datakällor vid design-tid och ange delade datakällor som rapportdatakällor vid publicering av en rapport.
- Aspose.Cells för Reporting Service är integrerat med Microsoft Reporting Services sömlöst, så användare kan dra full nytta av Reporting Services rapporthantering, säkerhet och leveransalternativ.
