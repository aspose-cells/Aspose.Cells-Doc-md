Några effektiva egenskaper hos Aspose.Cells for Reporting Services listas nedan

- Aspose.Cells for Reporting Services stöder tabeller med grupperade rader och sidfötter. Grupperingen av rader och sidfötter kan ställas in vid designtillfället som andra rapportobjekt.
- Aspose.Cells for Reporting Services stöder parametrar och globala variabler som rapportdatakällor.
- Användare kan använda Microsoft Query för att skapa datakällor och frågor. Microsoft Query är ett flitigt använt verktyg som de flesta användare borde vara och bekanta med.
- Aspose.Cells for Reporting Services stöder både rapportspecifika datakällor och delade datakällor som skapats på rapportservern i förväg. Dessutom är det möjligt att använda rapportspecifika datakällor vid designtillfället och ange delade datakällor som rapportdatakällor samtidigt som man publicerar en rapport.
- Aspose.Cells för Reporting Service är integrerat med Microsoft Reporting Services sömlöst och så att användare kan dra full nytta av Reporting Services rapporthantering, säkerhet och leveransalternativ.
