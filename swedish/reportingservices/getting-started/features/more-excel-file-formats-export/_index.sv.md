{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services gör det möjligt att leverera nativa Microsoft Excel-rapporter genom Microsoft SQL Server 2005 och 2008 Rapporterings tjänster. Till skillnad från den inbyggda renderaren för Microsoft Excel-baserade rapporter, erbjuder Aspose.Cells for Reporting Services en utökad funktionsuppsättning som konverterar RDL-rapporter till: Excel 97-2003 binär XLS-format, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 XLSB, HTML och ODS-format. 

{{% /alert %}} 

Aspose.Cells for Reporting Services stöder fler filformat än de nativa exportörerna. Nedan visas skärmbilder från prov XLS-rapporter exporterade med Aspose.Cells.
