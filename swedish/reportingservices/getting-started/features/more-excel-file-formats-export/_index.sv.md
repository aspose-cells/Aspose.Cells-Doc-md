{{% alert color="primary" %}} 

Aspose.Cells för Reporting Services låter dig leverera inbyggda Microsoft Excel-rapporter genom Microsoft SQL Server 2005 och 2008 Reporting Services. Till skillnad från den inbyggda renderaren för Microsoft Excel-baserade rapporter, tillhandahåller Aspose.Cells för Reporting Services en utökad funktionsuppsättning som konverterar RDL-rapporter till: Excel 97-2003 binärt XLS-format, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 XLSB, HTML och ODS format.

{{% /alert %}} 

Aspose.Cells för Reporting Services stöder fler filformat än de inhemska exportörerna. Nedan finns skärmdumpar från exempel på XLS-rapporter som exporterats med Aspose.Cells.
