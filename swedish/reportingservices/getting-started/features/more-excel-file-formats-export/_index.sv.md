{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services låter dig leverera inbyggda Microsoft Excel-rapporter genom Microsoft SQL Server 2005 och 2008 Reporting Services. Unlike the built-in renderer for Microsoft Excel-based reports, Aspose.Cells for Reporting Services provides an extended feature set that convert RDL reports to: Excel 97-2003 binary XLS format, CSV, SpreadsheetML, Excel 2007 XLSX , Excel 2007 XLSM, Excel 2007 XLSB, HTML och ODS format.

{{% /alert %}}
