{{% alert color="primary" %}} 

Aspose.Cells för Reporting Services stöder fullt ut RDL-specifikationen. Detta innebär att du enkelt kan porta befintliga RDL-rapporter till en Aspose-lösning.

{{% /alert %}} 
**RDL-element som stöds**
 Aspose.Cells stöder följande RDL-element:

- Sida, sidhuvuden, sidfötter
- Textrutor
- Bilder
- Diagram
- Listor
- Tabeller
- Matriser
- Stilar
- Rektanglar, linjer
- Delrapporter
- Tablix
- Mätare
