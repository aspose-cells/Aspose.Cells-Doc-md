{{% alert color="primary" %}} 

Vissa diagramtyper i SQL Server Reporting Services 2008-rapportdefinitionsfilen är inte tillgängliga som Excel-diagramtyper:

- Linjesteg
- Range-Plain
- Räckvidd-Smidig
- Range-Column
- Range-Bar
- Range-ErrorBar
- Range-BoxPlot
- Form-tratt

När du renderar diagramobjekt som inte har motsvarande alternativ i Microsoft Excel, matar Aspose.Cells ut dem till bilder.

{{% /alert %}}
