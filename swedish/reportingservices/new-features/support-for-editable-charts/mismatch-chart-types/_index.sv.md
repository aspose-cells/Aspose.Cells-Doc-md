{{% alert color="primary" %}} 

Vissa diagramtyper i SQL Server Reporting Services 2008-rapportdefinitionsfiler är inte tillgängliga som Excel-diagramtyper:

- Linje-steg
- Intervall-Plan
- Intervall-Slät
- Intervall-kolumn
- Intervall-stapeldiagram
- Intervall-felstapeldiagram
- Intervall-lådplot
- Form-tratt

När diagramobjekt som inte har motsvarande alternativ i Microsoft Excel renderas exporterar Aspose.Cells dem som bilder. 

{{% /alert %}}
