Aspose.Cells for Reporting Services stöder SSRS 2008R2 delade datauppsättningar. När Aspose.Cells Report Designer öppnar en rapportdefinitionsfil som inkluderar delade datauppsättningar, kan den automatiskt få datauppsättningsinformation som inkluderar fält från rapportservern.
