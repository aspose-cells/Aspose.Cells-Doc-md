Aspose.Cells for Reporting Services stödjer delade dataset för SSRS 2008R2. När Aspose.Cells Rapport Designer öppnar en rapportdefinieringsfil som inkluderar delade dataset kan den automatiskt hämta datasetinformation som inkluderar fälten från rapportservern.
