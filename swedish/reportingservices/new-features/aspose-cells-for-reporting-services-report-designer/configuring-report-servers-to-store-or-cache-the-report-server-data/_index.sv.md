Aspose.Cells for Reporting Services Report Designer sparar den aktuella åtkomstadressen till rapportservern. När du öppnar dialogrutan Rapportserverhanterare i Aspose.Cells Report Designer igen, laddar dialogformuläret automatiskt den sparade rapportserverinformationen.
