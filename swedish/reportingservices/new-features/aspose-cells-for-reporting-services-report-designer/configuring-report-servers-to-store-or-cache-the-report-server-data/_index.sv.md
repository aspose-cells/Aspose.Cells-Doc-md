Aspose.Cells for Reporting Services Rapportdesignern sparar den aktuella åtkomstadressen för rapportservern. När du öppnar dialogrutan för Rapportserverhanteraren i Aspose.Cells Rapportdesignern, laddas formulärdialogen automatiskt med den sparade rapportserverinformationen.
