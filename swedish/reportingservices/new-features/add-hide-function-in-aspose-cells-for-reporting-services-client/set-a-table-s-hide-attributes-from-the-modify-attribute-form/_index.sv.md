Ange en tabellrapports dolda attribut från **Ändra attribut**. Parametrar inkluderar:

- **Typ** representerar dolt objektstyp. Det inkluderar tabell, rad och kolumn.
- **Index** representerar den dolda objektets position. När typen är kalkylblad är värdet tomt. Indexvärdet börjar från 0.
- **Uttryck** representerar dolt statiskt värde. Värdet är sant, falskt eller en Excel-formel. Uttrycket stödjer formulär för rapporteringstjänster.
