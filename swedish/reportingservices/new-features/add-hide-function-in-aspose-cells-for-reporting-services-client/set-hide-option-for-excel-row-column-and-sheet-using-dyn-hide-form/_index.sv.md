Aspose.Cells for Reporting Services döljer valfri rad, kolumn och ark baserat på inställningarna för dolda parametrar. Parametrarna inkluderar:

- **SheetName** representerar ett kalkylblad som specificerar ett dolt objekt.
- **Typ** representerar dolt objektstyp. Det inkluderar kalkylblad, rad och kolumn.
- **Rad/Kolumn Index** representerar det angivna dolda objektet. När typen är kalkylblad är värdet tomt. Kolumnindex stöder tecken. Indexvärdet börjar från 1.
- **Dolt tillstånd** representerar doldt tillståndsvärde. Värdet är sant, falskt eller en Excel-formel. Parametern stödjer endast rapportparametrar för rapporteringstjänster.
