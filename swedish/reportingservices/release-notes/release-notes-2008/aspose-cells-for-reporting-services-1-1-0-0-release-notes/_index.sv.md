 Kära kunder,



Vi har släppt Aspose.Cells.ReportingServices V1.1.0.0.



 Vad är nytt?



-  ` `Vi tillhandahåller ett klienttilläggsverktyg - Aspose Cells Rapportdesigner som tillåter användare att använda MS Excel som ett rapportdesignverktyg. Med Aspose Cells kan Report Designer-användare designa en rapportmall i Excel istället för Visual Studio eller annat rapportbyggarverktyg. Rapportmallen är inbäddad i Reporting Services RDL-fil och kan publiceras på Report Server. Det gör rapportdesign till en enkel och snabb uppgift.
-  Aspose.Cells för Reporting Services-serverrenderingen är avsevärt förbättrad. Genom att arbeta med rapportmallen kan användare använda alla avancerade Excel-funktioner i Reporting Services, inklusive inbyggda Excel-diagram, avancerade formler, pivottabeller och mer.
