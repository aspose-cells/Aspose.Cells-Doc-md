 Introduktion

 Aspose.Cells för Reporting Service v1.0.1 inkluderar funktionsförbättringar och buggfixar sedan

 den första utgåvan.

 Vad är nytt?

- Stöds för att exportera matris
- Stöds för att exportera CSV och TabDelimited fil
- Stödde bakgrundsbilden.

 Vad är fixat?

- Fixade felet när image.ImageData är null.
- Fixade felet när textbox-värdet innehåller nummerformat.
- Fixade felet när matrisen grupperades.
