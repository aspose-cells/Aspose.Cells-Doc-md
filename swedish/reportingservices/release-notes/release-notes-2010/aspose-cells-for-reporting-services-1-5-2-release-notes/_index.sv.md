 Vi är glada att meddela Aspose.Cells for Reporting Services 1.5.2!

 Vad har förändrats?



-  Optimerar renderingen för stora datarapporter.
- Åtgärdar ett installationsproblem för Windows 7.
-  Löser personsökning för Tablix-rapportobjekt.
-  Hanterar rendering av fler grupper för Tablix.
-  Minskar minnesanvändningshastigheten.
