För att konfigurera en klients Rapport Server URL:

1. Klicka på knappen **Öppna fjärrrapport** och få dialogrutan **Hämta rapport**. 

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_1.png)

1. Klicka på knappen **Konfig** och få dialogrutan **Rapport Server URL Setup**. 

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_2.png)

1. Ange rapportserverkonfigurationsinformation (URL, användarnamn och lösenord).
   1. Rapport Server URL måste vara i formatet http://värd/ReportServer. Till exempel http://172.16.204.11/ReportServer.
   1. Användarnamnet är användarnamnet för fönster Server. Till exempel, administratör.
   1. Lösenordet är användarlösenordet för fönster Server. Till exempel, 123456.
   1. Standardvärdet för domänen är null.
1. Klicka på **Test**-knappen för att kontrollera konfigurationsinformationen.
1. Klicka på **Spara** och spara rapportserverkonfigurationsinformationen.

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_3.png)




1. Klicka på **Ändra**-knappen för att ändra konfigurationsinformationen.
