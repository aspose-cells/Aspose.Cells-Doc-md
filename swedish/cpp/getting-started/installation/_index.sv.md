## **Installera Aspose.Cells for C++ i Windows**
### **Använda Nuget Package Manager**
1. I Nuget Package Manager, sök efter Aspose.Cells. 
   1. Här är [Länk till NuGet Package Aspose.Cells.Cpp](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. Se [Installera Aspose Cells genom NuGet] för detaljer.(https://docs.aspose.com/cells/cpp/getting-started/).  

Du kan också ladda ner ZIP-paket från Aspose.Cells for C++ [nedladdningssida](https://downloads.aspose.com/cells/cpp/). Arkivet innehåller också offline-API-dokumentation.
### **Användning av Include- och lib-mappar**
1. Ladda ner/Installera [senaste filerna](https://downloads.aspose.com/cells/cpp/).
1. Packa upp mappen till produktionsmiljön.
1. För att använda Aspose.Cells for C++, hänvisa till 'Include' och 'lib / dll' -mappen i ditt projekt.
