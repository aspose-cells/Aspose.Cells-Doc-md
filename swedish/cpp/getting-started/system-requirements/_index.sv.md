Aspose.Cells för C++ är ett inbyggt C++-bibliotek som gör det möjligt för C++-utvecklare att skapa, manipulera och konvertera kalkylblad programmatiskt utan att behöva Office Automation eller Microsoft Excel-program.

## Utvecklingsmiljö

Du kan använda Aspose.Cells för C++ när du utvecklar applikationer för Windows eller Linux.

### Windows

Aspose.Cells för C++ kan användas för att utveckla applikationer i alla utvecklingsmiljöer som stöder[Microsoft Visual Studio v142 Platform Toolset](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017?view=msvc-160), men de miljöer som anges i följande tabell stöds uttryckligen:

<table>  
	<tr>
			<td style="font-weight: bold; width:800px">Utvecklingsmiljöer</td>
		</tr>
  <tr>
			<td><ul><li>Microsoft Visual Studio 2017</li><li>Microsoft Visual Studio 2019</li></ul></td>
			</tr>
</table>

### Linux

Aspose.Cells för C++ kan användas för att utveckla applikationer i utvecklingsmiljön som stöder C++11 eller högre, men följande kompilator och verktyg stöds uttryckligen:

	<tr>

			</tr>
</table>

### Ytterligare beroende av Linux

 Aspose.Cells för C++ på Linux beror på[libgdiplus](https://github.com/mono/libgdiplus//) binärer både dynamiskt bibliotek och verktyg. Installera det innan du använder:

- Installera libgdiplus på Ubuntu eller Debian</br>
`sudo apt-get install libgif-dev autoconf libtool automake build-essential gettext libglib2.0-dev libcairo2-dev libtiff-dev libexif-dev`
