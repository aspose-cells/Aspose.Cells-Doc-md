{{% alert color="primary" %}} 

Använd WordArt för att lägga till speciella text- eller bildeffekter i kalkylblad. Till exempel, sträck ut en titel över filens överkant, dekorera text och få text att passa en förinställd form, eller använd text på ett Excel-ark som en bakgrundsvattenstämpel. WordArt blir ett objekt som du kan flytta eller placera i kalkylblad för att lägga till dekoration.

{{% /alert %}}
