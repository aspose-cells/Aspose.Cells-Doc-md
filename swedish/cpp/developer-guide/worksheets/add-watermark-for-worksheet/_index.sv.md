{{% alert color="primary" %}} 

Använd WordArt för att lägga till särskild text eller bild effekter till kalkylblad. Till exempel, sträck en rubrik överst på filen, dekorera text och få text att passa en förinställd form, eller tillämpa text på ett Excel-kalkylblad som en bakgrundsvattenstämpel. WordArt blir ett objekt som du kan flytta eller positionera i kalkylblad för att lägga till dekoration.

{{% /alert %}}
