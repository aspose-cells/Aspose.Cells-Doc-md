##  **Introduktion**
 Snälla använd[PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) metod för att skapa en pivottabell i kalkylbladet. När pivottabellen har skapats kan du arbeta med den med hjälp av[Pivottabell](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/)klass.
