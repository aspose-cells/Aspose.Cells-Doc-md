## **Introduktion**
Använd [PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) metoden för att skapa en pivottabell i arbetsbladet. När pivottabellen har skapats kan du arbeta med den med hjälp av [PivotTable](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/) klassen.
