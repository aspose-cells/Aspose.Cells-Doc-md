## **Introduktion**
Snälla använd[IPivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table_collection#a903a0eb3d7ef995c370f4a4385b25111) metod för att skapa en pivottabell i kalkylbladet. När pivottabellen har skapats kan du arbeta med den med hjälp av[IPivotTable](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table)klass.
