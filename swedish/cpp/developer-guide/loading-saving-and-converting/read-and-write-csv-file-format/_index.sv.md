## **Möjliga användningsscenarier**
Microsoft Excel stöder många format som XLS, XLSX, XLSM, XLSB, CSV, etc. Aspose.Cells stöder också många av dessa format. Den här artikeln förklarar hur man läser och skriver excel-filen med CSV-format med Aspose.Cells.
## **Läs och skriv CSV-filformat**
 Följande exempelkod laddar[käll-CSV-fil](23166994.csv) och läser dess cell A1 och kopierar sedan dess innehåll till cell C4 och sparar den som[utdata CSV-fil](23166994.csv).
