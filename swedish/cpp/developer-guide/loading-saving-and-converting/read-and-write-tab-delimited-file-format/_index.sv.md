## **Möjliga användningsscenarier**
Microsoft Excel stöder många format som XLS, XLSX, XLSM, XLSB, CSV, tabbavgränsad, etc. Aspose.Cells stöder också många av dessa format. Den här artikeln förklarar hur du läser och skriver excel-filen med tabbavgränsat format med Aspose.Cells.
## **Läs och skriv tabbavgränsat filformat**
 Följande exempelkod laddar[källtabbavgränsad fil](23166997.txt)och läser dess cell A1 och kopierar sedan dess innehåll till cell C4 och sparar den som[output tabbavgränsad fil](23166996.txt).
