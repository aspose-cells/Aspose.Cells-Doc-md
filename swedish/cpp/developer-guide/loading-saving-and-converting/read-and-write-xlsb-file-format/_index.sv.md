##  **Möjliga användningsscenarier**
Microsoft Excel stöder många format som XLS, XLSX, XLSM, XLSB, CSV, etc. Aspose.Cells stöder också många av dessa format. Den här artikeln förklarar hur du läser och skriver excel-filen med XLSB-format med Aspose.Cells.
##  **Läs och skriv XLSB Filformat**
 Följande exempelkod laddar[källfil XLSB](23166992.xlsb)och läser dess cell A1 och kopierar sedan dess innehåll till cell C4 och sparar den som[utgång XLSB fil](23166993.xlsb).
