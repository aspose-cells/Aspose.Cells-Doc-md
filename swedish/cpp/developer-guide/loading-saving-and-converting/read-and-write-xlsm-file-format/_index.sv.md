## **Möjliga användningsscenarier**
Microsoft Excel stöder många format som XLS, XLSX, XLSM, XLSB, CSV, etc. Aspose.Cells stöder också många av dessa format. Den här artikeln förklarar hur man läser och skriver excel-filen med XLSM-format med Aspose.Cells.
## **Läs och skriv XLSM-filformat**
 Följande exempelkod laddar[XLSM källfil](23166990.xlsm)och läser dess cell A1 och kopierar sedan dess innehåll till cell C4 och sparar den som[utdata XLSM-fil](23166991.xlsm).
