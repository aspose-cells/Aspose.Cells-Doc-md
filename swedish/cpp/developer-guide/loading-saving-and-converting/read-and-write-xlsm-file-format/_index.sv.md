## **Möjliga användningsscenario**
Microsoft Excel stöder många format som XLS, XLSX, XLSM, XLSB, CSV, etc. Aspose.Cells stöder också många av dessa format. Den här artikeln förklarar hur du läser och skriver excelfil med XLSM-format med hjälp av Aspose.Cells.
## **Läs och skriv XLSM-filformat**
Följande exempelkod laddar den [källa XLSM-filen](23166990.xlsm), läser dess cell A1 och kopierar sedan dess innehåll till cell C4 och sparar den som [utdata XLSM-fil](23166991.xlsm).
