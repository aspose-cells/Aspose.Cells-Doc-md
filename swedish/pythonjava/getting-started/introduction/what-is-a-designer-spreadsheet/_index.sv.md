## **Vill du snabba upp utvecklingen?**

{{% alert color="primary" %}}

Du kan använda Aspose.Cells för att skapa Excel-filer från grunden med hjälp av dess flexibla API. Ett av de vanligaste användningsområdena för Aspose.Cells är att skapa och fylla Excel-filer med data, vilket kan komma från en befintlig databas, en användare eller andra källor, dynamiskt. Förutom att lägga till data i Excel-filer behöver du också skriva för att tillämpa formatering (utseende och känsla).

Formatering tar tid. Vad kan snabba upp utvecklingen?

{{% /alert %}}

## **Lösningen - Designer-kalkylblad**

För att snabba på utvecklingen erbjuder Aspose.Cells den spännande funktionen designer-kalkylblad.

### **Vad är ett Designer-kalkylblad?**

Ett designer-kalkylblad är en Excel-fil skapad med Microsoft Excel där all formatering redan har tillämpats och innehållet kan beslutas vid designstadiet. Det är mer som en mallfil som inte innehåller några data utan bara formateringen, som visas nedan:

**Designerkalkylblad**

![todo:image_alt_text](what-is-a-designer-spreadsheet_1.png)

### **Hur man använder det?**

Efter att ha skapat ett designer-kalkylblad använder du Aspose.Cells för att läsa av designer-kalkylbladet och lägga till data på dess angivna positioner (enligt de fördefinierade formatinställningarna) vid körning. Till exempel, i skärmbilden ovan kan du se att vi redan bestämde vid designstiden att lägga till **Produktnamn**, **Kategorinamn**, **Antal per enhet** och **Enheter i lager** kolumner. De har ännu ingen data: detta läggs till vid körning. Ovanstående figur visar en mycket enkel kalkylblad men designer-kalkylblad sparar tid till och med på komplexa kalkylblad.

### **Smart Markers**

Dessutom låter Aspose.Cells dig lägga till [smarta markörer](/cells/sv/python-java/smart-markers/) i designer-kalkylbladet. Smarta markörer bearbetas vid körning med hjälp av [**WorkbookDesigner**](https://reference.aspose.com/cells/python-java/asposecells.api/WorkbookDesigner)-klassens Process-metod. Detta kan hjälpa dig att skriva mindre kod och ändå få fantastiska resultat.

[Läs mer om smarta markörer](/cells/sv/python-java/smart-markers/).
