## **Systemkrav**
Aspose.Cells för Python via Java är en plattformsoberoende API och kan användas på vilken plattform som helst (Windows, Linux och MacOS) där [Python](https://www.python.org/downloads/) är installerat. Datorn måste ha Java 8 eller senare versioner innan installationen är klar.

## **Python Version**
- Python 3.5 eller nyare
## **Java Version**
- Java 1.8 eller högre

## **Installation**
### **Windows:**
- **Installera Java och ange JAVA_HOME och PATH miljövariabler.**
Exempelvis:
{{< highlight java >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}

- **Installera Aspose.Cells för Python via Java från pypi**
Du kan enkelt använda Aspose.Cells för Python via Java från [pypi](https://pypi.org/project/aspose-cells/) med följande kommando.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}
