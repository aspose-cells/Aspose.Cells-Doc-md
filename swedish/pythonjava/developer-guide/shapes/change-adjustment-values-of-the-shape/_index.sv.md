{{% alert color="primary" %}}

Aspose.Cells för Python via Java tillhandahåller [**Shape.getGeometry().getShapeAdjustValues()**](https://reference.aspose.com/cells/python-java/asposecells.api/geometry#ShapeAdjustValues) metod för att göra ändringar i justeringspunkter med former. I Microsoft Excel UI visas justeringar som gula diamantnoder. Till exempel:

- Avrundad rektangel har en justering för att ändra buen
- Triangel har en justering för att ändra platsen för punkten
- Trapets har en justering för att ändra bredden på toppen
- Pilar har två justeringar för att ändra formen på huvudet och svansen

Den här artikeln förklarar användningen av [**Shape.getGeometry().getShapeAdjustValues()**](https://reference.aspose.com/cells/python-java/asposecells.api/geometry#ShapeAdjustValues) metoden för att ändra justeringsvärdet för de olika formerna.
