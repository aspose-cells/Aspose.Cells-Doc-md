## **Öppna CSV-filer**
Comma Separated Values (CSV) -filer innehåller poster vars värden är avgränsade eller separerade av kommatecken. Du kan använda Microsoft Excel för att spara din kalkylblad i CSV-format.

För att öppna CSV-filer, använd klassen [LoadOptions](https://reference.aspose.com/cells/python/asposecells.api/LoadOptions) och välj värdet [CSV](https://reference.aspose.com/cells/python/asposecells.api/fileformattype#CSV), fördefinierad i uppräkningen [FileFormatType](https://reference.aspose.com/cells/python/asposecells.api/FileFormatType).
