## **Exportera utskriftsområde till HTML**
Aspose.Cells för Python via Java stöder endast export av utskriftsområde dvs. det markerade cellområdet istället för hela bladet till HTML. För detta tillhandahåller API:et egenskapen [HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly). Genom att ange denna egenskap till **Sant** kommer endast utskriftsområdet att exporteras.

Följande kodexempel demonstrerar användningen av egenskapen [HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) för att endast exportera utskriftsområdet till HTML.
