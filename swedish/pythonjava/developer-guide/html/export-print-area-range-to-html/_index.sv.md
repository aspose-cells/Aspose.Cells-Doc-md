## **Exportera utskriftsområde till HTML**
Aspose.Cells for Python via Java stöder export av endast utskriftsområdet, dvs. valt cellområde istället för hela she's till HTML. För detta tillhandahåller API[HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) fast egendom. Ställer in den här egenskapen till**Sann** kommer bara att exportera utskriftsområdet.
