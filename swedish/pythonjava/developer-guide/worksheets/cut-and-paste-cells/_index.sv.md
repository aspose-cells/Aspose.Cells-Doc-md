## **Klipp ut och klistra Cells**
Aspose.Cells for Python via Java ger möjlighet att klippa och klistra celler. För detta tillhandahåller API[insertCutCells](https://reference.aspose.com/cells/python/asposecells.api/cells#insertCutCells\(com.aspose.cells.Range,%20int,%20int,%20int\)) metod för[Cells](https://reference.aspose.com/cells/python/asposecells.api/Cells)samling. De[insertCutCells](https://reference.aspose.com/cells/python/asposecells.api/cells#insertCutCells\(com.aspose.cells.Range,%20int,%20int,%20int\)) metod accepterar följande parametrar.

- [Räckvidd](https://reference.aspose.com/cells/python/asposecells.api/Range)Området av celler som ska skäras.
- Radindex: Indexet för raden för att infoga celler.
- Kolumnindex: Indexet för kolumnen för att infoga celler.
- [ShiftType](https://reference.aspose.com/cells/python/asposecells.api/ShiftType): Kolumnernas förskjutningsriktning.
