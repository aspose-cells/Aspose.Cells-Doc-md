## **Konvertera Excel-arbetsbok till Word**

 Du behöver inte undra hur man konverterar Excel Workbook till Word, eftersom Aspose.Cells for Python via Java biblioteket har bästa beslutet. Aspose.Cells for Python via Java API ger stöd för att konvertera kalkylblad till Word-format. För att exportera arbetsboken till Word, godkänn[**SparaFormat.DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) som den andra parametern för[**Arbetsbok.spara**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) metod. Du kan också använda[**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions)klass för att ange ytterligare inställningar för export av kalkylblad till .docx-fil.
