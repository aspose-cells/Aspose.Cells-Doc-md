## **Konvertera Excel arbetsbok till Word**

Behöver inte fundera på hur man konverterar Excel arbetsbok till Word, eftersom Aspose.Cells för Python via Java-biblioteket har det bästa beslutet. Aspose.Cells för Python via Java-API:et stödjer att konvertera kalkylblad till Word-format. För att exportera arbetsboken till Word, skicka [**SaveFormat.DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) som andra parameter till [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)) metoden. Du kan också använda [**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions)-klassen för att specificera ytterligare inställningar för att exportera kalkylbladet till .docx-fil.
