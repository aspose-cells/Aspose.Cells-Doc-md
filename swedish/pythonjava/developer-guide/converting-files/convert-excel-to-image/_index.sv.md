## **Konvertera Excel till bild**
Aspose.Cells för Python via Java stödjer konvertering av Excel-filer till olika bildformer. För detta tillhandahåller API:et [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)- och [ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions)-klasserna. Denna klass representerar kalkylarket som ska renderas till en bild. [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)-klassen tillhandahåller metoden [toImage()](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage\(int,%20java.lang.String\)) för att konvertera ett kalkylblad till en bildfil. BMP, PNG, JPEG, TIFF och EMF-format stöds.
