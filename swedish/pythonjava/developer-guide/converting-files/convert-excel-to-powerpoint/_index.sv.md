## **Konvertera Excel-arbetsbok till PPT**

Du behöver inte undra hur man konverterar Excel Workbook till PowerPoint, eftersom biblioteket Aspose.Cells for Python via Java har bästa beslutet. Aspose.Cells for Python via Java API ger stöd för att konvertera kalkylblad till formatet PowerPoint. För att exportera arbetsboken till PowerPoint, godkänn[**SparaFormat.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) som den andra parametern för[**Arbetsbok.spara**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) metod. Du kan också använda[**PptxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) klass för att ange ytterligare inställningar för export av kalkylblad till .pptx-fil.
