## **Konvertera Excel arbetsbok till PPT**

Behöver inte fundera på hur man konverterar Excel arbetsbok till PowerPoint, eftersom Aspose.Cells för Python via Java-biblioteket har det bästa beslutet. Aspose.Cells för Python via Java-API:et stödjer att konvertera kalkylblad till PowerPoint-format. För att exportera arbetsboken till PowerPoint, skicka [**SaveFormat.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) som andra parameter till [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)) metoden. Du kan också använda [**PptxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions)-klassen för att specificera ytterligare inställningar för att exportera kalkylbladet till .pptx-fil.
