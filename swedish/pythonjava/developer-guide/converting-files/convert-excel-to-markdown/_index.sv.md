## **Konvertera Excel till Markdown**
Aspose.Cells för Python via Java stödjer konvertering av Excel-filer till Markdown-format. För att exportera det aktiva kalkylarket till Markdown, ange [SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN) som andra parameter för [Workbook.Save](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\))-metoden. Du kan också använda [MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions)-klassen för att specificera ytterligare inställningar för att exportera kalkylblad till Markdown.
