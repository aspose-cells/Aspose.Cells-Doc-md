## **Konvertera Excel till SVG**
SVG(**Skalbar vektorgrafik)**är en specifikation baserad på XML-standarder för tvådimensionell vektorgrafik. Det är en öppen standard som har varit under utveckling av World Wide Web Consortium (W3C) sedan 1999.

Aspose.Cells for Python via Java stöder konvertering av Excel-filer till SVG bilder. För detta tillhandahåller API[SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender), [ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions), och[WorkbookRender](https://reference.aspose.com/cells/python/asposecells.api/WorkbookRender)klasser.
