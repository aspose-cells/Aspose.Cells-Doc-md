## **Ställ in kommentaren för tabell eller listobjekt i kalkylbladet**
Aspose.Cells for Python via Java stöder tillägg av kommentaren för listobjekt. För detta tillhandahåller API[ListObject.Comment](https://reference.aspose.com/cells/python/asposecells.api/listobject#Comment)fast egendom. Kommentaren tillagd av[ListObject.Comment](https://reference.aspose.com/cells/python/asposecells.api/listobject#Comment)egendom kommer att vara synlig inuti*xl/tabeller/tabellnamn.xml* fil.
