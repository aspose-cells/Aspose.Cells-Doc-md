## **Rita Slicer vid rendering av Excel till PDF**
Aspose.Cells för Python via Java stödjer export av Excel till PDF med slicerinställningar. API:et stödjer detta som standard, du exportera helt enkelt Excel-filen med slicer till PDF, den genererade PDF:en kommer att visa en tillämpad slicer. Följande kodsnutt laddar in [prov Excel-filen](106365058.xlsx) som innehåller en slicer och sparar arbetsboken som [utdata PDF-fil](106365052.xlsx).
