## **Rita Slicer medan du renderar Excel till PDF**
Aspose.Cells for Python via Java stöder export av Excel till PDF med skivinställningar. API stöder detta som standard, du exporterar helt enkelt Excel-filen med slicer till PDF, den genererade PDF kommer att visa slicer tillämpad. Följande kodavsnitt laddar[exempel på Excel-fil](106365058.xlsx)som innehåller en slicer och sparar arbetsboken som[utgång PDF fil](106365052.xlsx). Följande skärmdump jämför källfilen i Excel och den genererade PDF-filen.
