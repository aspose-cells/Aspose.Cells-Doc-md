## **Formatera skärva**
Med Microsoft Excel kan du formatera slicern genom att ställa in dess antal kolumner, stilar osv. Aspose.Cells för Python via Java erbjuder egenskaperna Slicer.NumberOfColumns och Slicer.StyleType för att uppnå detta.

Följande kodsnutt laddar in [prov Excel-filen](106364964.xlsx) som innehåller en slicer. Den får åtkomst till slicern och ställer in dess antal kolumner och stiltyp och sparar den som [utdata Excel-fil](106364965.xlsx). Skärmbilden visar hur slicern ser ut efter att exempelkoden har körts.
