## **Hur man sparar pivottabell i ODS-fil**

Aspose.Cells för Python via .NET ger möjlighet att spara pivottabell i ODS-filer. För detta konverterar du bara en arbetsbok med befintlig pivottabell eller skapar en ny pivottabell och sparar filen i ODS-format. Se till att anropa [**PivotTable.calculate_data**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/calculate_data/#) innan du sparar för att säkerställa att pivottabellen visas i den resulterande ODS-filen.
