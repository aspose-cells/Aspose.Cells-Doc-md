 Aspose.Cells for Python via .NET ger möjlighet att spara pivottabell i ODS-filer. För detta konverterar du bara en arbetsbok med befintlig pivottabell eller skapar en ny pivottabell och sparar filen i formatet ODS. Se till att ringa[**PivotTable.calculate_data**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/calculate_data/#)innan du sparar för att säkerställa att pivottabellen återges i utdatafilen ODS.
