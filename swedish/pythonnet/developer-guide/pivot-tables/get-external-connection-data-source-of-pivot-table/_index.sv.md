##  **Hämta extern anslutningsdatakälla för pivottabell**

Aspose.Cells for Python via .NET ger möjlighet att hämta den externa anslutningsdatakällan för pivottabellen. För detta tillhandahåller API[**extern_anslutning_datakälla**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) egendom av[**Pivottabell**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/) klass. De[**extern_anslutning_datakälla**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/)egendom returnerar[**Extern anslutning**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/) objekt. Följande kodavsnitt visar användningen av[**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) egenskap för att hämta den externa anslutningsdatakällan för pivottabellen.
