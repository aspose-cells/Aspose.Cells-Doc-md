## **Hur man får extern anslutningsdatakälla för pivot-tabell**

Aspose.Cells for Python via .NET ger möjlighet att få den externa anslutningsdatakällan för pivot-tabellen. För detta tillhandahåller API:et egenskapen [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) för klassen [**PivotTable**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/). Egenskapen [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) returnerar objektet [**ExternalConnection**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/). Följande kodsnutt visar användningen av egenskapen [**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) för att hämta den externa anslutningsdatakällan för pivot-tabellen.
