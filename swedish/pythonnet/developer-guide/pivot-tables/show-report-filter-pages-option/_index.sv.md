## **Hur man visar filterkällsalternativ**
Excel stöder skapandet av pivot tabeller, lägga till rapportfilter och aktivera alternativet "Visa filterrapportsidorna". Aspose.Cells för Python via .NET stöder också denna funktion för att aktivera alternativet "Visa filterrapportsidorna" på den skapade pivot tabellen.
