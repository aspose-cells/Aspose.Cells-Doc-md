{{% alert color="primary" %}}

 Microsoft Excel låter dig ändra layouten för pivottabellen med*Pivottabellverktyg > Design > Rapportlayout* menykommandon. Du kan ändra layouten i dessa tre former

- Visa i kompakt form
- Visa i dispositionsformulär
- Visa i tabellform

 Aspose.Cells tillhandahåller också[**PivotTable.show_in_compact_form()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/show_in_compact_form/#), [**PivotTable.show_in_outline_form()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/show_in_outline_form/#) och[**PivotTable.show_in_tabular_form()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/show_in_tabular_form/#) metoder för att ändra layouten för pivottabellen i dessa tre former.

{{% /alert %}}
