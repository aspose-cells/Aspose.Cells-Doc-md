{{% alert color="primary" %}}

Rapporter baserade på pivot-tabeller är användbara men benägna att fel om målgrupperna inte har detaljerad kunskap om Excel för att konfigurera dessa rapporter. Under sådana omständigheter vill organisationer begränsa användare från att kunna ändra en rapport baserad på pivot-tabell. Vanliga pivot-tabellfunktioner som att lägga till ytterligare filter, snedstreck, fält eller ändra ordningen på vissa saker i rapporten rekommenderas oftast inte för varje användare. Å andra sidan ska dessa användare även kunna uppdatera rapporten och använda befintliga filter eller snedstreck. Aspose.Cells för Python via .NET har tillhandahållit denna förmåga till utvecklare för att begränsa användare från att ändra dessa rapporter samtidigt som de skapar dessa rapporter. För detta ändamål tillhandahåller Excel en funktion för att inaktivera pivot-tabellfliken och samma funktion tillhandahålls av Aspose.Cells för Python via .NET, dvs. utvecklaren kan inaktivera fliken som innehåller kontroller för att modifiera dessa rapporter.

{{% /alert %}}
