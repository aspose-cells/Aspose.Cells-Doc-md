## **Möjliga användningsscenario**

Numbers är en kalkylprogramvara utvecklad av Apple Inc. Aspose.Cells för Python via .NET kan läsa Numbers-kalkylblad men stöder inte att skriva till det. Den kan läsa data, stil och formler från Numbers-kalkylblad.

## **Läs Numbers-kalkylblad utvecklat av Apple Inc. med Aspose.Cells för Python via .NET**

Följande kodexempel laddar [Provkalkylblad från Apple Inc.](sampleNumbersByAppleInc.numbers) och konverterar det till [Utgång PDF-format](outputNumbersByAppleInc.pdf). Du måste använda [**LoadOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/loadoptions) -klassen och ange [**LoadFormat.NUMBERS**](https://reference.aspose.com/cells/python-net/aspose.cells/loadformat) som parameter i dess konstruktor för att ladda det framgångsrikt. Vänligen ladda ner båda från de angivna länkarna. Du kan prova koden med vilket kalkylblad som helst. Läs även kommentarerna till koden för mer hjälp.
