## **Konvertera Excel-arbetsbok till JSON**

Behöver du inte fundera på hur man konverterar Excel Arbetsbok till JSON, eftersom Aspose.Cells för Python via .NET bibliotek har det bästa beslutet. Aspose.Cells API:et erbjuder stöd för att konvertera kalkylblad till JSON-format. För att exportera arbetsboken till JSON, skicka [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) som andra parameter av [**Workbook.save**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) metoden. Du kan även använda [**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) klassen för att ange ytterligare inställningar för att exportera kalkylbladet till JSON.
