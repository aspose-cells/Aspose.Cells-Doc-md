## **Konvertera Excel-arbetsbok till JSON**

Aspose.Cells for Python via .NET API tillhandahåller stöd för att konvertera kalkylblad till JSON-format. För att exportera arbetsboken till JSON, skicka [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) som andra parametern av [**Workbook.save**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) metoden. Du kan också använda [**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) klass för att ange ytterligare inställningar för att exportera kalkylblad till JSON.

Följande kodexempel visar hur man exporterar aktivt kalkylblad till Json genom att använda [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) -uppräkningsmedlem. Se koden för att konvertera [källfil](book1.xlsx) till [utdata-Json-filen](book1.Json) som genererats av koden som referens.
