## **Möjliga användningsscenario**

Aspose.Cells låter dig prefixa tabelldatastilar med [**HtmlSaveOptions.table_css_id**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/table_css_id)-egenskapen. Anta att du anger denna egenskap med något värde som **MyTest_TableCssId**, kommer du att hitta tabelldatastilar som visas nedan

{{< highlight java >}}

 table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}
