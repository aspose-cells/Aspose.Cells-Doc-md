##  **Konvertera Excel till HTML med rubriker**

Aspose.Cells for Python via NET ger möjlighet att exportera rubriker samtidigt som Excel konverteras till HTML. För detta kan du ställa in[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) egendom till *sant**. Standardvärdet för[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)egenskapen är *falsk**.
