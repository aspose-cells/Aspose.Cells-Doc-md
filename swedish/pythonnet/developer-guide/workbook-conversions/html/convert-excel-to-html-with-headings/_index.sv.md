## **Konvertera Excel till HTML med rubriker**

Aspose.Cells for Python via NET ger möjligheten att exportera rubriker vid konvertering av Excel till HTML. För detta kan du sätta [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) egenskapen till **true**. Standardvärdet för [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)egenskapen är **false**.
