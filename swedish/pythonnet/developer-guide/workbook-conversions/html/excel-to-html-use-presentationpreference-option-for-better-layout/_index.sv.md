{{% alert color="primary" %}} 

Aspose.Cells för Python via .NET tillhandahåller en användbar HtmlSaveOptions.PresentationPreference-egenskap för utvecklare som behöver bättre layout när de sparar en Microsoft Excel-fil till HTML eller MHT-format. Standardvärdet för egenskapen är false. Vi rekommenderar att sätta detta värde till true för att få en mer attraktiv presentation av Excel-rapporter.
