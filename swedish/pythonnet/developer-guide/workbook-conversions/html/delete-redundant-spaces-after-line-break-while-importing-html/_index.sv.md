{{% alert color="primary" %}}

Använd egenskapen [**HTMLLoadOptions.delete_redundant_spaces**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlloadoptions/delete_redundant_spaces/) och ange den som **true** för att ta bort alla överflödiga mellanslag som kommer efter radbrytningstaggen. Som standard är den här egenskapen **false** och överflödiga mellanslag bevaras i de resulterande Excel-filerna.

{{% /alert %}}
