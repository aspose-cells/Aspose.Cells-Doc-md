Aspose.Cells för Python via .NET stöder nu fullständig konvertering av revideringar från XLSB-fil till XLSM-fil. Revideringarna finns inuti sökvägen \xl\revisions. Du kan se dem genom att ändra filändelsen på din XLSB-fil till ZIP. Sökvägen \xl\revisions innehåller filer med .bin-förlängningar.

När du konverterar din XLSB-fil till XLSM-fil med Aspose.Cells för Python via .NET, konverteras dessa .bin-filer framgångsrikt till .xml-filer som visas i dessa två skärmbilder.
