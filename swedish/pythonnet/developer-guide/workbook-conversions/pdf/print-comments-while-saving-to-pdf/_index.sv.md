Microsoft Excel låter dig skriva ut kommentarer medan du skriver ut eller sparar till PDF-format med följande alternativ

- Ingen
- I slutet av arket
- Som visas på arket

 Aspose.Cells for Python via .NET tillhandahålla[**PrintCommentsType**](https://reference.aspose.com/cells/python-net/aspose.cells/printcommentstype/) enum för att stödja samma funktion. De[**PrintCommentsType**](https://reference.aspose.com/cells/python-net/aspose.cells/printcommentstype/)enum har följande medlemmar

- Skriv ut Inga kommentarer
- PrintInPlace
- PrintSheetEnd
