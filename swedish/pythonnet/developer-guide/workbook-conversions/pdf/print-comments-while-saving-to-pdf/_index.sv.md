{{% alert color="primary" %}}

Microsoft Excel låter dig skriva ut kommentarer när du skriver ut eller sparar till PDF-format med följande alternativ

- Ingen
- I slutet av bladet
- Enligt visad på kalkylbladet

Aspose.Cells för Python via .NET tillhandahåller [**PrintCommentsType**](https://reference.aspose.com/cells/python-net/aspose.cells/printcommentstype/) enum för att stödja samma funktion. [**PrintCommentsType**](https://reference.aspose.com/cells/python-net/aspose.cells/printcommentstype/) enum har följande medlemsfunktioner

- PrintNoComments
- PrintInPlace
- PrintSheetEnd
