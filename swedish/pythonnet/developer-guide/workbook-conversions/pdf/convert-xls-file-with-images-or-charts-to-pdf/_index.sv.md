Aspose.Cells for Python via .NET stöder konvertering av XLS-filer som innehåller bilder och diagram till PDF-dokument. Aspose.Cells for Python via .NET API kan arbeta självständigt för att konvertera ett kalkylblad till PDF: Aspose.PDF for .NET krävs inte för konverteringen. Processen kan göras i minnet eftersom processen inte är beroende av temporära eller mellanliggande XML-filer. Detta innebär att stora Excel-filer, till exempel sådana som innehåller bilder, diagram och andra ritobjekt, kan konverteras snabbt och effektivt.
