## **Möjliga användningsscenario**

Tidigare kunde inte Aspose.Cells för Python via .NET rendera Office-tillägg när Excelfilen sparas i PDF-format. Nu renderar Aspose.Cells för Python via .NET det bra. Du behöver inte använda någon speciell metod eller egenskap för att rendera Office-tillägg i utdata-PDF. Spara bara din Excelfil i PDF-format och det kommer att rendera Office-tillägg.

## **Rendera Office-tillägg vid konvertering av Excel till PDF**

Följande exempelkod sparar den [exempel Excel-filen](60489769.xlsx) som innehåller Office Add-Ins. Vänligen se [utdata PDF som genererats med föregående version dvs 17.11](60489770.pdf) och [utdata PDF som genererats med nyare version dvs 17.12 och framåt](60489771.pdf). Utdata PDF för föregående version är tom men utdata PDF för nyare version visar Office Add-In.
