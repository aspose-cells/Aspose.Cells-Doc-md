{{% alert color="primary" %}}

 Aspose.Cells for Python via .NET stöder att spara en ODS-fil (**OpenDocument Spreadsheet**) i ODF-specifikationerna (**OpenDocument Format**) 1.1 och 1.2. Aspose.Cells for Python via .NET har[**OdsSaveOptions.is_strict_schema11**](https://reference.aspose.com/cells/python-net/aspose.cells/odssaveoptions/is_strict_schema11/)egenskap som anger användningen av ODF 1.1-specifikationen för att spara ODS-filer. Standardvärdet för den här egenskapen är *false**, så filen ODS som sparas utan denna inställning använder 1.2-specifikationerna.

{{% /alert %}}

Exempelkoden nedan skapar ett arbetsboksobjekt, lägger till ett visst värde till cell A1 på det första kalkylbladet och sparar sedan filen ODS i ODF 1.1- och 1.2-specifikationer. Som standard sparas filen ODS i ODF 1.2-specifikationen.
