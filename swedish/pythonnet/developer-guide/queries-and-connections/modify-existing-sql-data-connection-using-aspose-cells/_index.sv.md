Aspose.Cells för Python via .NET stöder att modifiera befintlig SQL Data Connection. Artikeln förklarar hur man använder Aspose.Cells för Python via .NET för att ändra olika egenskaper för SQL Data Connection.

Du kan lägga till eller se dataanslutningar inne i Microsoft Excel genom att följa menyn **Data > Connections**.

På liknande sätt ger Aspose.Cells för Python via .NET möjlighet att komma åt och ändra dataanslutningar med hjälp av Workbook.DataConnections samling.

{{% /alert %}}
