## **Användningsscenarie**

Det kan finnas fall där datakällfiler flyttas och Excel-filen inte kan lokalisera filen. I sådana fall erbjuder Aspose.Cells för Python via .NET API:n möjlighet att uppdatera Power Query Formel-objektet genom att använda klass [*PowerQueryFormulaItem*](https://reference.aspose.com/cells/python-net/aspose.cells.querytables/powerqueryformulaitem) för att uppdatera platsen för källfilen.
