{{% alert color="primary" %}}

Aspose.Cells för Python via .NET stöder konvertering av Microsoft Excel-orienterad HTML till XLS/XLSX-format. Det innebär att HTML genererad av Microsoft Excel kan konverteras tillbaka till XLS/XLSX-format med Aspose.Cells för Python via .NET.

På samma sätt, om det finns enkel HTML kan Aspose.Cells för Python via .NET omvandla den till HTML Rich Text. Aspose.Cells för Python via .NET tillhandahåller [**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/) egenskap som kan ta en sådan enkel HTML och konvertera den till formaterad celltext.
