{{% alert color="primary" %}}

Du kan använda Aspose.Cells for Python via .NET för att få validering som har tillämpats på en cell. Aspose.Cells for Python via .NET tillhandahåller metoden [**Cell.get_validation()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) för detta ändamål. Om det inte finns någon validering som har tillämpats på cellen returnerar den null.

På liknande sätt kan du använda metod [**Worksheet.validations.get_validation_in_cell**](https://reference.aspose.com/cells/python-net/aspose.cells/validationcollection/get_validation_in_cell/#int-int) för att få validering som har tillämpats på en cell genom att ange dess rad- och kolumnindex.
