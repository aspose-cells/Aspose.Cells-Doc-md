{{% alert color="primary" %}} 

Hyperlänkar är användbara för att komma åt information på andra kalkylblad eller på webbplatser. Microsoft Excel låter användare lägga till hyperlänkar på text i celler och på bilder. Bildhyperlänkar kan underlätta navigering i ett kalkylblad, till exempel som knappar för nästa och föregående, eller logotyper som länkar till specifika platser. Detta dokument förklarar hur man infogar bildhyperlänkar i ett kalkylblad med hjälp av Aspose.Cells for Python via .NET.

{{% /alert %}} 

Aspose.Cells for Python via .NET tillåter dig att lägga till hyperlänkar till bilder i kalkylblad vid körning. Det är möjligt att ange och ändra länkens skärmtips och adress. Följande provkod illustrerar hur man lägger till en bildhyperlänk i ett kalkylblad.
