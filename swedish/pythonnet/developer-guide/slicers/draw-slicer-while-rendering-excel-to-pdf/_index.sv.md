## **Hur man ritar slicern vid rendering av Excel till PDF med Aspose.Cells för Python Excel-bibliotek**
Om du har en Excel-fil som har en slicer som är applicerad på den och du vill exportera Excel till PDF med slicerinställningar, stöder nu Aspose.Cells för Python via .NET detta som standard. Du exporterar helt enkelt Excel-filen med slicer till PDF, den genererade PDF:en kommer att visa den applicerade slicern.
