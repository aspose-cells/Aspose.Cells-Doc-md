## **Möjliga användningsscenario**

Om du vill koppla bort slicern och pivot-tabellen i Excel, behöver du högerklicka på slicern och välja ”Rapportanslutningar…” objektet. I alternativlistan kan du arbeta med kryssrutan. På samma sätt, om du vill koppla bort slicern och pivot-tabellen använda Aspose.Cells för Python via .NET API programmatiskt, använd [**Slicer.remove_pivot_connection(pivot)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/remove_pivot_connection/#aspose.cells.pivot.PivotTable) -metoden. Den kommer att koppla bort slicern och pivot-tabellen.

## **Hur man kopplar bort slicern och pivot-tabellen med Aspose.Cells för Python Excel-bibliotek**

Följande provkod laddar in [provmappen](remove-pivot-connection.xlsx) som innehåller en befintlig slicer. Den går igenom slicerna och sedan avassocierar den slicern och pivottabellen. Slutligen sparar den arbetsboken som [output Excel-fil](remove-pivot-connection-out.xlsx).
