## **Möjliga användningsscenario**

Om du vill ta bort slicern i Microsoft Excel, välj den bara och tryck på *Delete*-knappen. På samma sätt, om du vill ta bort den med Aspose.Cells för Python via .NET API programmatiskt, använd [**Worksheet.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/) -metoden. Det kommer att ta bort slicern från kalkylbladet.
