##  **Möjliga användningsscenarier**

 Om du vill ta bort slicer i Microsoft Excel, välj bara den och tryck på*Radera* knapp. På samma sätt, om du vill ta bort det med Aspose.Cells for Python via .NET API programmatiskt, använd[**Worksheet.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/)metod. Det kommer att ta bort skivaren från kalkylbladet.
