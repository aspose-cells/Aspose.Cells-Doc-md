## **Möjliga användningsscenario**

Du kan formatera slicern i Microsoft Excel genom att ange dess antal kolumner eller genom att ange dess stil osv. Aspose.Cells för Python via .NET tillåter dig också att göra detta med hjälp av [**Slicer.number_of_columns**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/number_of_columns/) och [**Slicer.style_type**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/style_type/)-egenskaper.
