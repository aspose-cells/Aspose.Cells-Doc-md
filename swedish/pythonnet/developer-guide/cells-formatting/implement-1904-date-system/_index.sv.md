{{% alert color="primary" %}} 

Microsoft Excel stöder två datorsystem: 1900-datumsystemet (standard datorsystem som implementeras i Excel för Windows) och 1904-datumsystemet. 1904-datumsystemet används normalt för att ge kompatibilitet med Macintosh Excel-filer och är standard när du använder Excel för Macintosh. Du kan ställa in 1904-datumsystemet för Excel-filer med Aspose.Cells för Python via .NET.

{{% /alert %}} 

För att implementera 1904 datumsystemet i Microsoft Excel (till exempel Microsoft Excel 2003):

1. Från menyn **Verktyg** väljer du **Alternativ** och väljer fliken **Beräkning**.
1. Välj alternativet **1904 datumssystem**.
1. Klicka på **OK**.
