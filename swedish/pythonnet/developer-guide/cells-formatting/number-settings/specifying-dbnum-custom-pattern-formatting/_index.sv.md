## **Möjliga användningsscenario**

Aspose.Cells för Python via .NET stöder *DBNum*-anpassat mönsterformat. Till exempel, om ditt cellvärde är 123 och du anger dess anpassade formatering som [DBNum2][$-804]General, kommer det att visas som 壹佰贰拾叁. Du kan ange din anpassade formatering av cellen med metoden [**Cell.get_style()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_style/#) och egenskapen [**Style.custom**](https://reference.aspose.com/cells/python-net/aspose.cells/style/custom).
