## **Möjliga användningsscenario**

Du kan använda Aspose.Cells för Python via .NET för att sammanfoga eller dela upp en område av celler. Aspose.Cells för Python via .NET tillhandahåller [**Range.merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/merge/#) och [**Range.un_merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/un_merge/#) metoder för detta ändamål. Den här artikeln förklarar hur man sammanfogar ett område av celler till en enda cell.


## **Hur man sammanfogar eller avsammanfogar område av celler**

Följande exempelkod skapar först ett intervall - A1:D4 - och slår sedan samman cellerna i intervallet till en enda cell med hjälp av [**Range.merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/merge/#)-metoden. På liknande sätt kan du dela upp celler genom att skapa ett intervall och anropa [**Range.un_merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/un_merge/#)-metoden.
