## **Möjliga användningsscenario**

Normalt sett är namngivna intervaller rektangulära med celler kontinuerliga och intilliggande till varandra. Men ibland kan du behöva använda ett icke-sekventiellt cellintervall där celler inte ligger intill varandra. Aspose.Cells för Python via .NET stödjer skapandet av namngivet intervall med icke-intilliggande celler.
