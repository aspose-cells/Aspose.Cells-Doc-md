## **Möjliga användningsscenario**

Generellt anger du typsnitts-katalogen eller listan av typsnitt för alla arbetsböcker, men ibland måste du specificera individuella eller privata typsnitt. Aspose.Cells för Python via .NET erbjuder [**IndividualFontConfigs**](https://reference.aspose.com/cells/python-net/aspose.cells/individualfontconfigs) klassen som kan användas för att specificera enskilda eller privata typsnitt för ditt arbetsbok.
