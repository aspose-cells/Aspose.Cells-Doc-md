Med den ökande användningen av högupplösta skärmar visas bilder genererade vid standard 96 DPI ofta suddiga och oklara. För att säkerställa tydlighet på högupplösta skärmar är det viktigt att generera bilder med högre DPI. Aspose.Cells för Python via .NET erbjuder funktionalitet för att ställa in [**ImageOrPrintOptions.horizontal_resolution**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/horizontal_resolution/) och [**ImageOrPrintOptions.vertical_resolution**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/vertical_resolution/), vilket gör att du kan skapa högkvalitativa bilder från Excel-filer som ser skarpa ut på högupplösta skärmar.
