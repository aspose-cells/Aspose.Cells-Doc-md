Du kan använda Aspose.Cells för Python via .NET för att exportera ditt arbetsblad eller diagram till en bild med önskad bredd och höjd. Det tillhandahåller [**ImageOrPrintOptions.set_desired_size()**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/set_desired_size) metoden för att ställa in önskad bredd och höjd på den exporterade bilden. Bredden och höjden anges i pixlar.
