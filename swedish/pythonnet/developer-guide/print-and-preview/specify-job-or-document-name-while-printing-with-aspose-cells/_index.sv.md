Du kan ange Jobb- eller Dokumentnamn när du skriver ut ditt arbetsbok eller arbetsblad med hjälp av WorkbookRender- eller SheetRender-objekten. Aspose.Cells för Python via .NET tillhandahåller metoderna WorkbookRender.ToPrinter(printerName, jobName) och SheetRender.ToPrinter(printerName, jobName) vilka du kan använda för att specificera Jobbnamn vid utskrift.

{{% /alert %}}
