## **Möjliga användningsscenario**

*Sheet.SheetId* egenskapen finns inom *DocumentFormat.OpenXml.Spreadsheet* namnområdet och är en del av OpenXml. Du kan se denna egenskap och dess värde i *workbook.xml* som visas på följande skärmdump. Aspose.Cells för Python via .NET ger den ekvivalenta egenskapen som [**Worksheet.tab_id**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/tab_id).
