{{% alert color="primary" %}}

Det är möjligt att visa formler istället för beräknade värden i Microsoft Excel genom att använda alternativet **Visa formler** från fliken **Formler**. När formler visas visar Microsoft Excel formler i arbetsboken. Du kan uppnå samma sak med Aspose.Cells för Python via .NET.

{{% /alert %}}

Aspose.Cells for Python via .NET tillhandahåller en egenskap [**Worksheet.show_formulas**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/show_formulas). Sätt detta till **true** för att ställa in Microsoft Excel att visa formler.
