{{% alert color="primary" %}} 

Aspose.Cells for Python via .NET gör att du kan ändra färgen på individuella arbetsbladstabbar för att göra dem framträdande från resten. Till exempel kan du göra kostnader röda, försäljning gröna, tillgångar blå, osv.

{{% /alert %}} 

## **Hur man Ställer in Arbetsbladstabellfärg med Microsoft Excel**
1. Högerklicka på en flik i flikarket längst ned på det aktuella arbetsbladet.
1. Välj **Flikfärg**.
1. Välj en färg från paletten.
1. Klicka på **OK**.
