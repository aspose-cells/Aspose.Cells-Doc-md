Detta avsnitt kommer att berätta för dig hur man skapar ett lagerdiagram, vilket inkluderar fyra typer:
- **HLC** – Hög-Låg-Stäng.
- **OHLC** – Öppen-Hög-Låg-Stäng.
- **VHLC** – Volym-Hög-Låg-Stäng.
- **VOHLC** – Volym-Öppen-Hög-Låg-Stäng.

## **Vad är lagerdiagram?**

Aktiekurvor är ett specifikt diagram som används för att spåra prisförändringar på trivade tillgångar. Tillgångar kan vara råvaror, aktier och kryptovalutor. De gör det möjligt att se höga och låga värden över tid, tillsammans med öppnings- och stängningsvärden i ett diagram. Aspose.Cells för Python via .NET erbjuder 4 aktiekurvor och för att använda dessa måste du ha rätt data tillgänglig och välja kolumnerna i rätt ordning.

Vi kommer att använda dessa data för att skapa fyra typer av aktie-diagram: Hög-Låg-Stäng (HLC) aktie-diagram, Öppen-Hög-Låg-Stäng (OHLC) diagram, Volym-Hög-Låg-Stäng (VHLC) aktie-diagram och Volym-Öppen-Hög-Låg-Stäng (VOHLC) aktie-diagram.
