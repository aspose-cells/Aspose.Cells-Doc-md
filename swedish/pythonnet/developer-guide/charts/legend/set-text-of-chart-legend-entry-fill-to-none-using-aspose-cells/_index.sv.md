{{% alert color="primary" %}}

Om du vill ställa in texten för diagrammets legendpostfyllnad till none så att den inte ska visas i diagrammets legend, ställ in [**LegendEntry.is_text_no_fill**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/legendentry/is_text_no_fill) till **true**.

{{% /alert %}}
