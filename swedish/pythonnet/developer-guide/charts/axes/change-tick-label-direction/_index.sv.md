## **Ändra riktning för ticketiketter**

Aspose.Cells för Python via .NET ger dig möjlighet att ändra diagrammets ticketikett Riktning genom att använda egenskapen [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type). Egenskapen [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type) accepterar värdet från [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/) enum. [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/) enum tillhandahåller följande medlemmar

- HORISONTELL
- VERTIKAL
- ROTERA90
- ROTERA270
- STAPLAT
