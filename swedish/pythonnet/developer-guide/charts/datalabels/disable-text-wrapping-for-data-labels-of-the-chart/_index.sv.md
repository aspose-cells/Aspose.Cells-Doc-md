{{% alert color="primary" %}}

Microsoft Excel 2013 tillåter användare att vicka eller avvicka text inne i dataetiketterna för diagrammet. Som standard är texten inne i dataetiketterna för diagrammet i det vikta läget.

Aspose.Cells för Python via .NET tillhandahåller en [**DataLabels.is_text_wrapped**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/datalabels/is_text_wrapped)-egenskap som du kan sätta till True eller False för att Aktivera eller Avaktivera Textbrytning av datamärkningar respektive.
