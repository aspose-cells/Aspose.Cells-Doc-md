Du kan använda Aspose.Cells för Python via .NET för att skapa rik text anpassade datamärkningar för diagrampunkterna. Aspose.Cells för Python via .NET tillhandahåller [**DataLabels.characters()**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextframe/characters)-metoden för att returnera [**FontSetting**](https://reference.aspose.com/cells/python-net/aspose.cells/fontsetting)-objektet som kan användas för att ställa in teckensnittsegenskaper som färg, fetstil, etc.
