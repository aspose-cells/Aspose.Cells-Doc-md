{{% alert color="primary" %}}

Microsoft Excel låter användare lägga till kommentarer i celler för att tillföra ytterligare information och markera data. Utvecklare kan behöva anpassa kommentaren för att specificera inställningar för blandning, textriktning, teckensnittfärg etc. Aspose.Cells för Python via .NET tillhandahåller API:er för att genomföra detta.

{{% /alert %}}

Aspose.Cells för Python via .NET tillhandahåller en [**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body)-egenskap för teckensnittfärgen i kommentaren. Följande exempel visar användningen av [**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body)-egenskapen för att ställa in textens riktning för en kommentar.
