{{% alert color="primary" %}}

Microsoft Excel tillåter användare att lägga till kommentarer i celler för att lägga till ytterligare information och markera data. Utvecklare kan behöva anpassa kommentaren för att specificera justeringsinställningar och textriktning. Aspose.Cells för Python via .NET tillhandahåller API:er för att utföra uppgiften.

{{% /alert %}}

Aspose.Cells för Python via .NET tillhandahåller en [**Shape.text_direction**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_direction) egenskap för att ställa in textriktning för en kommentar.
