Ta bort dubblettrader är en av Microsoft Excels många användbara funktioner. Det gör det möjligt för användare att ta bort dubblettrader i ett arbetsblad, du kan välja vilka kolumner som ska kontrolleras för dubbel information.

Aspose.Cells för Python via .NET tillhandahåller metoden [**Cells.remove_duplicates()**](https://reference.aspose.com/cells/python-net/aspose.cells/cells/remove_duplicates/) för detta ändamål. Du kan även ställa in startRad, startKolumn, slutRad, slutKolumn för att välja kolumner.
