{{% alert color="primary" %}}

Aspose.Cells for Python via .NET tillhandahåller egenskapen [**Shape.geometry.shape_adjust_values**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/geometry/shape_adjust_values/) för att göra ändringar i justeringspunkterna med former. I Microsoft Excel UI visas justeringar som gula diamantnoder. Till exempel:

- Avrundad rektangel har en justering för att ändra buen
- Triangel har en justering för att ändra platsen för punkten
- Trapets har en justering för att ändra bredden på toppen
- Pilar har två justeringar för att ändra formen på huvudet och svansen

Den här artikeln förklarar användningen av [**Shape.geometry.shape_adjust_values**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/geometry/shape_adjust_values/) egenskapen för att ändra justeringsvärdet för olika former.
