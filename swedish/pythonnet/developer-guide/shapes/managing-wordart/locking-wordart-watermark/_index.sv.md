Aspose.Cells för Python via .NET API:erna tillåter att lägga till WordArt-vattenstämplar på arbetsbladet så att WordArt blir ett objekt som kan flyttas och positioneras på arbetsbladet. Det är också möjligt att låsa WordArt-objektet för interaktion såsom redigering, flyttning och urval. Den här artikeln förklarar användningen av [**Shape.set_locked_property**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/set_locked_property)-metoden för att låsa vissa aspekter av vattenstämplen.

{{% /alert %}}
