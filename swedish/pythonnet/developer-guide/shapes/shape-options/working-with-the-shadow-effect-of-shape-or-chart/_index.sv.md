## **Möjliga användningsscenario**
Aspose.Cells för Python via .NET tillhandahåller egenskapen [Shape.shadow_effect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/shadow_effect) tillsammans med [ShadowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect) klass för att arbeta med skuggeffekten av form eller diagram. [ShadowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect)-klassen innehåller följande egenskaper som kan ställas in för att uppnå olika resultat.

- [vinkel](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/angle)
- [oskarpa](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/blur)
- [färg](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/color)
- [avstånd](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/distance)
- [förinställningstyp](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/preset_type)
- [storlek](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/size)
- [genomskinlighet](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/transparency)
