## **Möjliga användningsscenario**
Aspose.Cells för Python via .NET tillhandahåller egenskapen [Shape.reflection](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/reflection) tillsammans med [ReflectionEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect) klass för att arbeta med reflektionseffekten av form eller diagram. [ReflectionEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect)-klassen innehåller följande egenskaper som kan ställas in för att uppnå olika resultat enligt applikationskrav.

- [oskarpa](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/blur)
- [riktning](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/direction)
- [avstånd](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/distance)
- [faderiktning](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/fade_direction)
- [tot_med_form](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/rot_with_shape)
- [storlek](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/size)
- [genomskinlighet](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/transparency)
- [typ](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/type)
