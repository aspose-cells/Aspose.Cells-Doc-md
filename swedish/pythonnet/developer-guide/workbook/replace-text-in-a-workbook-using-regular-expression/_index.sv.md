Aspose.Cells för Python via .NET har funktionen att ersätta text i en arbetsbok med hjälp av ett reguljärt uttryck. För detta tillhandahåller API:et [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key)-egenskapen av klassen [**ReplaceOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions). Att sätta [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) till **true** indikerar att nyckeln som söks är ett reguljärt uttryck.

Följande kodsnutt visar användningen av [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) egenskapen genom att använda den [exempel excel-fil](101089318.xlsx). [Utdatafilen](101089319.xlsx) som genereras av följande kodsnutt bifogas som referens.
