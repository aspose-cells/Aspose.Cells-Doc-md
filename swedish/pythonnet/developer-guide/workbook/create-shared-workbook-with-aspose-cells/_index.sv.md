## **Möjliga användningsscenario**

Microsoft Excel tillåter att du delar arbetsboken som visas i följande skärmbild. När du delar arbetsboken kan mer än en användare redigera den på nätverket. Aspose.Cells för Python via .NET gör det möjligt att skapa en delad arbetsbok med hjälp av egenskapen [**Workbook.settings.shared**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/shared).

![todo:image_alt_text](create-shared-workbook-with-aspose-cells_1.png)

## **Skapa delad arbetsbok med Aspose.Cells för Python via .NET**

Följande exempelkod skapar en delad arbetsbok genom att ställa in egenskapen [**Workbook.settings.shared**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/shared) som *true*. När du öppnar [utdata excel-fil](55541786.xlsx) i Microsoft Excel kommer du att se **Delad** med utdata arbetsboksnamnet enligt skärmdumpen.
