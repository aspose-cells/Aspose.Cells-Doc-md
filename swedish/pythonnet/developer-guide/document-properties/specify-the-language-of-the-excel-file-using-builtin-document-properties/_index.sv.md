## **Möjliga användningsscenario**

Du kan ändra språket på Excel-filen genom att högerklicka på filen och välja Egenskaper > Detaljer och sedan redigera Språk-fältet. Använd [**BuiltInDocumentPropertyCollection.language**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/language)-egenskapen för att programatiskt ändra det med Aspose.Cells för Python via .NET API:er.
