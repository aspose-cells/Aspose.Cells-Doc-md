## Kontrollera om VBA-projekt är skyddat och låst för visning i Python

Aspose.Cells för Python via .NET tillåter dig att kontrollera om VBA (Visual Basic for Applications)-projekt i Excel-fil är skyddat och låst för visning. För detta erbjuder API:et egenskapen [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/). Om det är låst för visning, returnerar egenskapen [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/) **true**.
