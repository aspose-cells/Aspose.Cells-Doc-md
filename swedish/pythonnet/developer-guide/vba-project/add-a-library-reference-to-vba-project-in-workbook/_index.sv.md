Ibland behöver du lägga till eller registrera bibliotekreferensen i VBA-projektet via kod. Detta kan göras med Aspose.Cells för Python via .NET [**VbaProject.references.add_registered_reference()**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaprojectreferencecollection/add_registered_reference/#str-str)-metoden.

{{% /alert %}}

## **Lägg till en biblioteksreferens till VBA-projekt i Microsoft Excel**

I Microsoft Excel kan du lägga till en biblioteksreferens till VBA-projektet genom att klicka på **Verktyg > Referenser...** manuellt.
