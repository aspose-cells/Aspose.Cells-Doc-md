## **Lösenordsskydda VBA-projekt i Excel-arbetsbok i Python**

Du kan lösenordsskydda VBA (Visual Basic for Applications)-projektet i en arbetsbok med Aspose.Cells för Python via .NET med hjälp av [**VbaProject.protect()**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/protect)-metoden.
