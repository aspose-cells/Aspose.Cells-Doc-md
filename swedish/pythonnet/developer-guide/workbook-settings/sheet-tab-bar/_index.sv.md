## **Möjliga användningsscenario**
När du behöver justera visningen av Excel-bladfältet måste du veta hur man kontrollerar flikfältet, till exempel att dölja eller visa flikfältet, ändra bredden på flikfältet, ange det första synliga fliket och så vidare. Aspose.Cells för Python via .NET stöder dessa funktioner. Aspose.Cells för Python via .NET tillhandahåller följande egenskaper och metoder för att hjälpa dig att uppnå dina mål.

- [**WorkbookSettings.show_tabs**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/show_tabs/)
- [**WorkbookSettings.sheet_tab_bar_width**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/sheet_tab_bar_width/)
- [**WorkbookSettings.first_visible_tab**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/first_visible_tab/)
