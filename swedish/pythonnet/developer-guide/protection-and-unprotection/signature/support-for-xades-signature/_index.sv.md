## **Introduktion**

Aspose.Cells för Python via .NET stödjer att signera arbetsböcker med XAdES-signatur. För detta tillhandahåller API:et klassen [**DigitalSignature**](https://reference.aspose.com/cells/python-net/aspose.cells.digitalsignatures/digitalsignature) och uppräkningen [**XAdESType**](https://reference.aspose.com/cells/python-net/aspose.cells.digitalsignatures/xadestype).
