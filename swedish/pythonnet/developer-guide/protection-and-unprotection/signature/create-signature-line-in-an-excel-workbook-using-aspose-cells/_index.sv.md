## **Introduktion**

Microsoft Excel tillhandahåller en funktion för att lägga till **Signature Line** i Excel-arbetsböcker. Du kan lägga till en signaturlinje genom att klicka på fliken **Infoga** och sedan välja **Signaturlinje** från gruppen **Text**.

## **Hur man skapar en signeringsrad för Excel**

Aspose.Cells för Python via .NET tillhandahåller också denna funktion och har exponerat egenskapen [**Picture.signature_line**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/picture/signature_line) för detta ändamål. Denna artikel förklarar hur man använder denna egenskap för att lägga till en signaturlinje med Aspose.Cells för Python via .NET.

Följande kodexempel lägger till en signaturlinje med hjälp av egenskapen [**Picture.signature_line**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/picture/signature_line) och sparar arbetsboken.
