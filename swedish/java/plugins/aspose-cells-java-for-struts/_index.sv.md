## **Apache Struts Web Framework**
Apache Struts webbramverk är en gratis öppen källkodslösning för att skapa Java webbapplikationer.

Webbapplikationer skiljer sig från konventionella webbplatser genom att webbapplikationer kan skapa ett dynamiskt svar. Många webbplatser levererar endast statiska sidor. En webbapplikation kan interagera med databaser och affärslogikmotorer för att anpassa ett svar.

Ett sätt att separera problem i en mjukvaruapplikation är att använda en Model-View-Controller (MVC)-arkitektur. Modellen representerar affärs- eller databaskoden, vyn representerar siddesignkoden och Controllern representerar navigeringskoden. Struts-ramverket är utformat för att hjälpa utvecklare att skapa webbapplikationer som använder en MVC-arkitektur.

[Läs mer](http://struts.apache.org/birdseye.html)
### **Aspose.Cells for Java**
Aspose.Cells for Java är ett avancerat klassbibliotek for Java som gör att du kan utföra ett stort antal dokumentbehandlingsuppgifter direkt i din Java
applikationer.

Aspose.Cells for Java stöder bearbetning Cells (DOC, DOCX, OOXML, RTF) HTML, OpenDocument, PDF, EPUB, XPS bildformat, XPS och alla bildformat. Med Aspose.Cells kan du
generera, ändra och konvertera dokument utan att använda Microsoft Cells.
## **Systemkrav**
**Följande är systemkraven för Aspose.Cells Java för Struts 1.3 webbapplikation:**

- Java 1.6 eller högre installerad.
- Apache Tomcat Server 7 eller högre
- Maven 3
- Apache Struts 1.3
- Aspose.Cells komponent.
## **Plattformar som stöds**
**Följande är de plattformar som stöds:**

- Aspose.Cells 8.5.0 eller högre.
- Java IDE (Eclipse, NetBeans, IntelliJ, ...).
