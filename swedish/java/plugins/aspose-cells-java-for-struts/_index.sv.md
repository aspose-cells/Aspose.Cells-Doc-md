## **Apache Struts Web Framework**
Apache Struts web framework är en gratis open-source-lösning för att skapa Java-webbapplikationer.

Webbapplikationer skiljer sig från vanliga webbplatser i det att webbapplikationer kan skapa en dynamisk respons. Många webbplatser levererar endast statiska sidor. En webbapplikation kan interagera med databaser och affärslogikmotorer för att anpassa en respons.

Ett sätt att separera bekymmer i en programvara är att använda en Model-View-Controller (MVC) arkitektur. Modellen representerar verksamhets- eller databaskoden, Vyn representerar sidodesignskoden, och Kontrollanten representerar den navigationskoden. Struts-ramverket är utformat för att hjälpa utvecklare att skapa webbapplikationer som utnyttjar en MVC-arkitektur.

## **Systemkrav**
**Följande är systemkraven för Aspose.Cells Java för Struts 1.3 webbapplikation:**

- Java 1.6 eller senare installerad.
- Apache Tomcat Server 7 eller senare
- Maven 3
- Apache Struts 1.3
- Aspose.Cells-komponent.
## **Stödda plattformar**
**Följande plattformar stöds:**

- Aspose.Cells 8.5.0 eller senare.
- Java IDE (Eclipse, NetBeans, IntelliJ, ...).
