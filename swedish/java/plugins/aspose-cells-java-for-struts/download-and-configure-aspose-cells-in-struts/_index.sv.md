## **Byggnad Aspose.Cells Java för Struts 1.3 från källkoder**
Efter att ha checkat ut källkoder från något av ovanstående arkiv, använd följande mvn-kommandon:

{{< highlight "java" >}}

 $ mvn -U clean package 

{{< /highlight >}}

Detta kommer att bygga "Strutsbookapp.war" i måltellern.

För att distribuera .war-filen, kopiera den bara till den körande Apache tomcat-serverns webbappkatalog.
