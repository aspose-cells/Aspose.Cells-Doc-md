## **Bygga Aspose.Cells Java för Struts 1.3 från källkoderna**
Efter att ha kontrollerat källkoderna från någon av ovanstående repositories, tillämpa följande mvn-kommandon:

{{< highlight java >}}

 $ mvn -U clean package 

{{< /highlight >}}

Detta kommer att bygga "Strutsbookapp.war" i målmappen.

För att distribuera .war-filen kopierar du den bara till körande Apache Tomcat-serverns webbappkatalog.
