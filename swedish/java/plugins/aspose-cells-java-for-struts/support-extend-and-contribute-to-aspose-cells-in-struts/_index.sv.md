## **Support**
- Om du vill se kända/rapporterade problem (av användarna eller Q.A-teamet) i applikationen.
- Eller om du vill rapportera något problem du hittat i applikationen
- Har du något förbättringsförslag eller vill göra en funktionförfrågan.

**Använd någon av följande projektspårare för ärenden:**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **Utvidga och Bidra**
Aspose.Cells Java för Struts 1.3 är öppen källkod och dess källkod finns tillgänglig på de största sociala kodningswebbplatserna som listas nedan. Utvecklare uppmanas att ladda ner källkoden och bidra genom att föreslå eller lägga till nya funktioner eller förbättra befintliga, så att andra också kan dra nytta av det.
