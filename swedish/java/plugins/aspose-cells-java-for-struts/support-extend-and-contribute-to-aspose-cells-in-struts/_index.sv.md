## **Stöd**
- Om du vill se kända/rapporterade problem (av användarna eller QA-teamet) i applikationen.
- Eller så vill du rapportera alla problem du hittat i applikationen
- Har några förbättringsförslag eller vill göra någon funktionsbegäran

**Använd någon av följande projekts problemspårare:**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **Utöka och bidra**
Aspose.cells Java for Struts 1.3 är öppen källkod och dess källkod är tillgänglig på de stora sociala kodningswebbplatserna nedan. Utvecklare uppmuntras att ladda ner källkoden och bidra genom att föreslå eller lägga till nya funktioner eller förbättra de befintliga, så att andra också kan dra nytta av den.
