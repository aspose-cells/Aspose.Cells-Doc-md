## **Installation**

**Aspose.Cells Java för Eclipse (Maven)** -plugin kan enkelt installeras från den tillgängliga **Plugin** -fliken i Plugin-dialogrutan.

- För att öppna den, välj **Plugins** från **Verktyg** -menyn i Eclipse.  

![todo:image_alt_text](installing-and-using-aspose-cells-java-for-eclipse-maven_1)

- Detta lägger till **Aspose.Cells Maven Project** i Nytt projekt-trollkarlen och **Aspose.Cells Code Example** i Ny annan trollkarl av Eclipse IDE.  

![todo:image_alt_text](project_1.png)

## **Användning**

### **Aspose.Cells Maven Project (trollkarl)**

Att skapa **Maven-projekt** med hjälp av guiden för att använda [Aspose.Cells for Java API](https://products.aspose.com/cells/java/):

1. Välj **Nytt -> Projekt**.
1. Välj **Aspose.Cells Maven Project** under **Maven** -kategorin.
1. Klicka på **Nästa**.

![todo:image_alt_text](project_2.png)

1. Ange **Projektnamn, Plats, GroupId, ArtifactId** och **Version** för ditt Maven-projekt och klicka på **Slutför**.

![todo:image_alt_text](project_3.png)

1. Detta kommer att hämta den senaste [Maven Dependency](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells)-referensen från [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo) och konfigurera den i **pom.xml**. Om du har valt **Ladda även ner kodexempel,** kommer även nedladdningen av **kodexempel** att börja från [Aspose.Cells for Java API Examples Repository.](https://github.com/aspose-cells/Aspose.Cells-for-Java)
1. Följande **Maven**-projekt kommer att skapas i din **Eclipse IDE** vid slutförandet av guiden:  

![todo:image_alt_text](project_4.png)

1. Det skapade **Maven-projektet** är konfigurerat för att använda **Aspose.Cells for Java API** och redo att utökas enligt dina projektkrav.
   Om du har valt att ladda ner [kodexempel](https://github.com/aspose-cells/Aspose.Cells-for-Java), kan du använda **Aspose.Cells Code Example (trollkarl)** för att importera nödvändiga **kodexempel** av Aspose.Cells for Java API till ditt projekt.

### **Aspose.Cells Code Example (trollkarl)**

**Aspose.Cells Code Example-trollkarlen** låter dig prova många exempel som tillhandahålls för Aspose.Cells for Java API.

{{% alert color="primary" %}}

För att kunna använda **Aspose.Cells Code Example-trollkarlen** bekvämt: det rekommenderas att alltid välja **Ladda även ner kodexempel** vid skapande av **Maven-projekt** på **Aspose.Cells Maven Project**-guiden.

{{% /alert %}}

För att använda exempel, bara:

1. Klicka på **Nytt -> Annat** i **Eclipse**.
1. Välj **Aspose.Cells Code Example** under **Java** -kategorin.
1. Klicka på **Nästa**.  

![todo:image_alt_text](example_1.png)

1. Välj ditt projekt och expandera sedan trädet för att välja **Kodexemepl** -kategorin och klicka på **Slutför**.

![todo:image_alt_text](example_2.png)

1. Detta kommer att kopiera de valda kategori **Kodexempel** Java-filer till projektet under **com.aspose.cells.examples** -paketet. Eventuella krävda resurser som behövs av kodexemplen kommer att kopieras in i **src/main/resources** -mappen, som visas nedan:

![todo:image_alt_text](example_3.png)

1. Granska exempelkoden, kompilera och kör.
1. Du kan nu testa andra exempel och börja bygga din egen applikation med hjälp av Aspose.Cells for Java API.
