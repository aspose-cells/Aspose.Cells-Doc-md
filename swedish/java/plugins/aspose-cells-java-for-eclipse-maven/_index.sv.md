## **Introduktion**
### **Eclipse IDE**
Eclipse IDE är en känd Java Integrated Development Environment (IDE). IDE är definitivt den mest kända produkten i Eclipse Open Source-projektet. Idag är det den ledande utvecklingsmiljön för Java med en marknadsandel på cirka 60%.

Eclipse IDE kan utökas med ytterligare programvarukomponenter. Eclipse kallar dessa programvarukomponenter för tillägg. Flera öppna källprojekt och företag har utökat Eclipse IDE eller skapat fristående applikationer (Eclipse RCP) ovanpå Eclipse-ramverket.
### **Aspose.Cells for Java**
[Aspose.Cells for Java](http://goo.gl/gtvhlQ) är en prisbelönt Excel-kalkylbladskomponent som tillåter Java-utvecklare att bädda in förmågan att läsa, skriva och manipulera Excel®-kalkylblad (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF och bildfilformat i sina egna Java-applikationer utan att behöva förlita sig på Microsoft Excel®.

[Aspose.Cells for Java](http://goo.gl/gtvhlQ) är en mogen, skalbar och funktion rik komponent som erbjuder många funktioner som är långt bortom de enkla dataexportmöjligheterna hos andra leverantörer. Med Aspose.Cells for Java kan utvecklare exportera data, formatera kalkylblad till den mest detaljerade nivån, importera bilder, skapa diagram, tillämpa och beräkna komplexa formler, strömma Excel® data, spara i olika format och mycket mer - allt detta utan behov av Microsoft Excel® eller Microsoft Office Automation.

## **Nedladdning**
### **Ladda ner Eclipse IDE**
Du måste först installera Eclipse IDE innan du laddar ner Aspose.Cells Java for Eclipse (Maven) plugin.

För att ladda ner Eclipse IDE

1. Gå till [https://eclipse.org](https://eclipse.org/).
1. Ladda ner och installera den rekommenderade Eclipse IDE för Java SE / EE-utvecklare.
### **Ladda ner Aspose.Cells Java for Eclipse (Maven)**
Här är tre rekommenderade metoder för att framgångsrikt ladda ner och installera Aspose.Cells Java for Eclipse (Maven) plugin:

- Dra och släpp installation från [Eclipse Marketplace](https://marketplace.eclipse.org/content/asposecells-java-maven-eclipse) till din Eclipse arbetsyta.
- Eller gå till **Hjälp** > **Installera ny programvara...** > Skriv in följande uppdateringssida url i **Arbeta med**:
 cdn.aspose.com/marketplace/eclipse/java/maven/asposecells
  Välj sedan "Aspose.Cells Java for Eclipse (Maven)" och **Slutför**. Godkänn licensavtalet och installera pluginet.

### **Hur man tillämpar Aspose License?**
Denna plugin använder en utvärderingsversion av Aspose.Cells. När du är nöjd med din utvärdering kan du köpa en licens på [Aspose webbplats](https://purchase.aspose.com/buy).
För att ta bort utvärderingsmeddelandet och begränsningarna för funktioner måste produkten licensieras. Du kommer att få en licensfil efter att du har köpt produkten. Följ stegen nedan för att tillämpa licensen

- Se till att licensfilen är namngiven som **Aspose.Cells.Java.lic**
- Placera filen **Aspose.Cells.Java.lic** i mappen som innehåller Aspose.Cells.jar
- Använd följande kod för att aktivera licensen:

{{< highlight java >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **Hur man konfigurerar källkoden för Aspose.Cells Java för Eclipse (Maven)**
Nedan enkla steg kommer smidigt leda till en framgångsrik konfiguration av  **"Aspose.Cells Java för Eclipse (Maven)"** plugin-källkoden i Eclipse IDE

1. Ladda ner / Klonen källkoden.
1. Välj **Fil** > **Import > Allmänt > Befintliga projekt i arbetsytan**
1. Bläddra till den senaste projekt källkod som du har laddat ner
1. Välj det Eclipse-projekt du vill importera
1. Klicka på **Färdig**
1. Aspose.Cells Java för Eclipse plugin-koden är nu klar att förbättras.
