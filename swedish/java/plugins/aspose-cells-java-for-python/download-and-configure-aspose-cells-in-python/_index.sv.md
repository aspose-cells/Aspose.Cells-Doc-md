## **Hämta nödvändiga bibliotek**
Ladda ner nödvändiga bibliotek som anges nedan. Dessa krävs för att köra Aspose.Cells Java för Python exempel.

- **Aspose:** [Aspose.Cells for Java Component](https://downloads.aspose.com/cells/java/)
- [JPype](https://pypi.org/project/JPype1/)

## **Hur man konfigurerar källkoden**
Följ dessa enkla steg för att öppna och utöka källkoden medan du använder:

\1. Install only Python 2.7.x.x from: <https://www.python.org/downloads/>
\2. Install JPype from: <https://sourceforge.net/projects/jpype/files/JPype/>
\3. Install Java JDK from: <https://www.oracle.com/java/technologies/downloads/>
\4. Download Aspose.Cells for Java APIs you want to use in your python projects from: <https://downloads.aspose.com/cells/java> and copy downloaded APIs’s .jars into %Python%/lib folder.
\ 5. Klon [Aspose.Cells Java för Python-exempel](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) från github.
