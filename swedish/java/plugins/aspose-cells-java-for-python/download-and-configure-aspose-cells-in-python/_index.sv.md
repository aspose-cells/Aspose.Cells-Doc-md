## **Hur man konfigurerar källkoden**
Följ dessa enkla steg för att öppna och utöka källkoden medan du använder:

 \1. Installera endast Python 2.7.xx från:<https://www.python.org/downloads/>
 \2. Installera JPype från:<https://sourceforge.net/projects/jpype/files/JPype/>
 \3. Installera Java JDK från:<https://www.oracle.com/java/technologies/downloads/>
 \4. Ladda ner Aspose.Cells for Java API:er som du vill använda i dina pythonprojekt från:<https://downloads.aspose.com/cells/java> och kopiera nedladdade API:ers .jars till mappen %Python%/lib.
 \5. Klona[Aspose.Cells Java for Python Exempel](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) från github.
