## **Installerar**
### **Hur man installerar Aspose.Cells Java för IntelliJ IDEA (Maven)**
 Stegen nedan visar hur du installerar**Aspose.Cells Java för IntelliJ IDEA (Maven)** plugin.

 Insticksprogrammet är listat på IntelliJ IDEA:s CE[officiella plugin-webbplats](https://goo.gl/R4pysl) - plugin-förråd som**Aspose.Cells Java för IntelliJ IDEA Maven**, så du behöver inte ladda ner det manuellt och installera det från disk, men det här extra alternativet finns fortfarande i IntelliJ IDEA.

**Aspose.Cells Java för IntelliJ IDEA (Maven)** plugin kan enkelt installeras (utan manuell nedladdning) från IntelliJ IDEA. Att göra det:

1.  Välj**inställningar** från**Fil** meny.
1.  Klick**Plugins** , bläddra sedan i arkiven i IntelliJ IDEA.

## **Använder sig av**
### **Aspose.Cells Maven Project Wizard**
 För att skapa Aspose.Cells Maven-projekt för användning[Aspose.Cells för Java API](http://goo.gl/c1eSD2) inom IntelliJ IDEA måste du ha internetuppkoppling.

Följ sedan stegen nedan:

 \1. Välj**Nytt projekt**.
 \2. Välj**Aspose.Cells Maven Project** 
 \3. Klick**Nästa**. 

![todo:image_alt_text](m1du9a1.jpg)


 \4. Tillhandahålla en**GroupId, ArtifactId** och**Version** för ditt maven-projekt och klicka**Nästa.**

![todo:image_alt_text](khijrce.jpg)


`    `Du kan markera kryssrutan "Hämta även exempel källkod" för att ladda ner exempel på användning av API:t (som visas i ögonblicksbilden ovan)

 \6. Tillhandahålla en**Projektnamn** och**Plats** precis som när du skapar vilken Java-applikation som helst

![todo:image_alt_text](0gszuiu.jpg)


 \7. Klick**Avsluta**.
 `    `Detta kommer att hämta[Aspose.Cells för Java API](http://goo.gl/c1eSD2)senaste maven-beroendereferens från Aspose Cloud Maven Repository och konfigurera det i pom.xml
 `     `Om du har markerat kryssrutan "Hämta även exempel källkod". Nedladdning av exemplen börjar också från[Aspose.Cells för Java API GitHub Repository](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

![todo:image_alt_text](eezoq3s.jpg)

![todo:image_alt_text](bujsm8v.jpg)

 `     `Grattis!{{< emoticons/cross >}} Du har framgångsrikt skapat Aspose.Cells Maven-projekt för användning[Aspose.Cells för Java API](http://goo.gl/c1eSD2).

![todo:image_alt_text](2oon4vh.jpg)

`      `Det skapade Aspose.Cells för Java API Maven-projektet är redo att modifieras enligt ditt projekts krav.

 `      `Om du har valt att ladda ner[Exempel på källkoder](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples), kan du använda Aspose.Cells Exempel Wizard för att kopiera/skapa de nedladdade exemplen till ditt projekt.
### **Aspose.Cells Exempel Wizard**
 Att skapa/kopiera[ Exempel på källkoder](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) in i projektet med hjälp av**Aspose.Cells Exempel Wizard**, måste du först ladda ner exemplen genom att välja "Hämta även exempel källkod" på Aspose.Cells Maven Project Wizard.

Följ sedan stegen nedan på IntelliJ IDEA:

 \1. Välj**Fil** => **Ny -** Enligt nedanstående:

![todo:image_alt_text](n8tt9q0.jpg)


`    `-ELLER-**Högerklicka** (på projektfönstret) =>**Ny** - Enligt nedanstående:

![todo:image_alt_text](aubwkhp.jpg)


 \2. Välj**Aspose.Cells Exempel** Alternativ från menyn - Som visas nedan:

![todo:image_alt_text](g4nwlem.jpg)


`  `Guiden kommer att uppdatera/söka efter det senaste tillgängliga[Aspose.Cells för Java API-exempel](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) 

![todo:image_alt_text](5pzwsuq.jpg)


\3. **Aspose.Cells Exempelguide** dialogrutan kommer att visas, där du kan välja valfri exempelkategori för att skapa/kopiera exemplen till ditt projekt
 `    `Välj valfri exempelkategori och klicka**Skapa**: 

![todo:image_alt_text](bc05c5v.jpg)


 \5. Detta kommer att skapa/kopiera de valda kategoriexemplen till projektet under**com.aspose.cells.examples** paket.
 `    `Alla nödvändiga resurser för exemplen kommer också att kopieras till**src/main/resources** mapp - Som visas nedan:

![todo:image_alt_text](jyxdo4d.jpg)

`       `3. Du har skapat/kopierat Exempel källkoder till ditt projekt med hjälp av Aspose.Cells Exempel Wizard.
