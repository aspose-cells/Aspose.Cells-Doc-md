## **Introduktion**

### **IntelliJ IDEA**

**IntelliJ IDEA** är en integrerad utvecklingsmiljö (IDE) för Java för mjukvaruutvecklare. Den är utvecklad av JetBrains. Ursprungligen skapad för Java, stöder den för närvarande ett brett utbud av språk, inklusive JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure och Kotlin.

I en 2010 **Infoworld** rapport fick IntelliJ det högsta testscores i testcentret av de fyra främsta Java-programvarorna: Eclipse, IntelliJ IDEA, NetBeans och Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) för Java är en Excel®-kalkylbladsrapporteringskomponent som möjliggör att Java-applikationer kan skapa och hantera Excel®-kalkylblad utan att använda Microsoft Excel®. Aspose.Cells for Java är en otroligt funktionell komponent som erbjuder mycket mer än bara grundläggande datamarknadsföringsfunktioner.

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java för IntelliJ IDEA (Maven) är en **IntelliJ IDEA Plugin** som låter dig skapa mavenprojekt (dvs. skapat projekt innehåller Aspose.Cells for Java API mavenberoende) och ladda ner exempel på källkoden för API för att använda **Aspose.Cells for Java API**.

- Denna plugin hjälper dig att arbeta med **Aspose.Cells for Java API** inne i din favorit IntelliJ IDEA.

**WIZARDS**:
Pluginet innehåller två guider -----

**Guide 1:** Aspose.Cells Maven-projekt

- Pluginets **Aspose Maven-projektsguide** låter utvecklare skapa Aspose.Cells for Java Maven-projekt.
- Referensen till **Aspose.Cells for Java API** mavenberoende hämtas automatiskt från [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) och läggs till i pom.xml.
- Det skapade projektet kommer alltid att innehålla den senaste tillgängliga versionen Maven Dependency för Aspose.Cells for Java.
- Guiden innehåller också alternativ för att ladda ner [Källkodsexempel för att använda Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

**Guide 2:** Aspose.Cells Exempel

- **Aspose.Cells Exempelguiden** låter dig skapa / kopiera nedladdade [Källkodsexempel för Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) till ditt projekt. Alla exempel för den valda kategorin kommer att kopieras till projektets mapp "com.aspose.cells.examples" tillsammans med de nödvändiga resurserna inom mappen "src/main/resources" som behövs för att köra exemplen.

- [Källkodsexempel för Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) är avsedda för att demonstrera de olika användningarna av API:et.

## **Nedladdning**

### **Laddar ner IntelliJ IDEA**

Du kommer först att behöva installera IntelliJ IDEA CE / UE innan du laddar ner **Aspose.Cells Java for IntelliJ IDEA (Maven)** pluginet.

För att ladda ner IntellliJ IDEA

1. Go to <https://www.jetbrains.com/idea/download/>
1. Välj ditt operativsystem
1. Klicka på UE eller CE nedladdning.

### **Laddar ner Aspose.Cells Java för IntelliJ IDEA plugin (Maven)**

- Pluginet kan direkt laddas ner från [JetBrains plugin hosting URL](https://goo.gl/R4pysl) - på IntelliJ IDEA Plugin Repository
  och kan senare installeras med hjälp av IDE-alternativet:
  - Gå till **Fil** > **Inställningar**...> **Plugins**
  - Välj "**Installera plugin från disk**"
- Eller så kan det direkt installeras (utan manuell nedladdning från JetBrains plugin hosting URL) med hjälp av IDE-alternativen:
  - Gå till **Fil** > **Inställningar**...> **Plugins**
  - Välj **"Bläddra i Repositories"**
  - I sökrutan skriv: "Aspose"
  - Välj "**Aspose Cells Java for IntelliJ IDEA Maven**" plugin att installera.
