## **Systemkrav**

Aspose.Cells för Java stöder följande Java-versioner:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- eller ovanför

[Fler detaljer](/cells/sv/java/system-requirements/)

## **Installation**

### **Installera Aspose Cells för Java från Maven Repository**

 Maven är det enklaste sättet att ladda ner och installera Aspose.Cells för Java.

1.  Ange Aspose Maven Repository konfiguration/plats i din Maven pom.xml.
 2. Definiera Aspose.Cells för Java API-beroende i din pom.xml.

[Fler steg för detaljer](/cells/sv/java/installation/)

### **Installera Aspose Cells för Java manuellt**

1. [Ladda ner aspose-cells-{version}-java.zip](https://downloads.aspose.com/cells/java/)
1. Packa upp paketet, ange katalogen \JDK 1.6\, packa upp aspose-cells-{version}-java.zip.
1. Ange katalogen \JDK 1.6\aspose-cells-{version}-java\lib\, installera JAR-paketen manuellt med Eclipse eller Intellij.
