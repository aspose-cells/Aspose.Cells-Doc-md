{{% alert color="primary" %}} 

Aspose.Cells för Java kräver inte att Microsoft Excel är installerat, eftersom Aspose.Cells i sig är en motor för att skapa, konvertera och rendera kalkylblad i Excel. För att se Excel-dokument producerade av Aspose.Cells måste du ha minst Microsoft Excel Viewer installerat.

{{% /alert %}} 
## **Operativsystem som stöds**
Aspose.Cells för Java stöder alla operativsystem som kör Java-runtime inklusive, men inte begränsat:
### **Windows**
- Microsoft Windows 2000 ( x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows Server 2003 ( x64, x86)
- Microsoft Windows Server 2008 (x64, x86)
- Microsoft Windows Server 2012 (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8 (x64, x86)
- Microsoft Windows 10 (x64, x86)
### **Linux**
- Linux (Ubuntu, openSUSE, CentOS och andra)
### **Mac**
- Mac OS X osv.
## **Java-versioner som stöds**
Aspose.Cells för Java stöder följande Java-versioner:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- eller ovanför
