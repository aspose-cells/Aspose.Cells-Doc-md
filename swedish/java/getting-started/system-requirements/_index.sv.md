{{% alert color="primary" %}} 

Aspose.Cells for Java kräver inte att Microsoft Excel är installerat, eftersom Aspose.Cells själv är en motor för skapande, konvertering och renderering av Excel-kalkylblad. För att visa Excel-dokument producerade av Aspose.Cells måste du ha åtminstone Microsoft Excel Viewer installerat.

{{% /alert %}} 
## **Stödda operativsystem**
Aspose.Cells for Java stöder alla operativsystem som körs Java-runtime inklusive, men inte begränsat till:
### **Windows**
- Microsoft Windows 2000 ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows Server 2003 ( x64, x86)
- Microsoft Windows Server 2008 ( x64, x86)
- Microsoft Windows Server 2012 ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
### **Linux**
- Linux (Ubuntu, openSUSE, CentOS och andra)
### **Mac**
- Mac OS X etc.
## **Stödda Java-versioner**
Aspose.Cells for Java stöder följande Java-versioner:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- eller högre
