 Den här sidan innehåller release notes för[Aspose.Cells for Java V1.9.4 Hot Fix](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-v1.9.4-hot-fix/)

 Vad är nytt?

- Stöder för att lägga till rik text för former som Arc, Oval, Rectangle, Comment, TextBox
 Stöder alla typer av placering för form
 Stöder för att läsa alla former från mall
 Stöder för att ställa in array i SmartMarker

 Vad är förbättrat?

- Lägg till nya metoder för att manipulera formens position och storlek, och avskaffade några gamla metoder
 Lägg till några demokoder och mallar för nya API
 Ersätt föråldrat API-anrop med nytt API i demos

 Vad är fixat?

- Fixade några buggar i beräkningsformeln
 Fixade en bugg med att exportera Excel2007 XLSX-format
 Fixat en bugg med kopieringsstil
 Fixat en bugg med kopieringsformel
 Fixade en bugg med att spara bildstorlek

 Kommentarer

- För diagram som importeras från mallfiler stöds inte manipulering av deras data, för närvarande stöds endast för att arbeta på formegenskaper, såsom position, placering och storlek
