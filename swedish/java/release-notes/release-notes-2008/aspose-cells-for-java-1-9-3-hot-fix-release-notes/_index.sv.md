 Den här sidan innehåller release notes för[Aspose.Cells för Java 1.9.3 Hot Fix](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-1.9.3-hot-fix/)

Kära kunder, Vi har släppt Aspose.Cells för Java V1.9.3!

 Vad är nytt?

- Stöder för att lägga till Arc, Oval, Rectangle
 Stöder för att lägga till Word- och Excel-filer som OLE-objekt
 Stöder att endast ladda data och formatering i mallfilen
 Stöder för att kopiera stil när du infogar rader och kolumner
 Stöder för att få nummerformat typ av cell
 Stöder för att filtrera icke-tomma/tomma celler i AutoFilter

 Vad är förbättrat?

- Tillhandahåller metoder för Shapes-objekt för att lägga till valfri form

 Vad är fixat?

- Fixat en bugg med beräkningsformler
 Fixade en bugg med att få minsta rad kolumn
 Fixade ett fel med att importera en samling till celler
 Fixat en bugg med att läsa filen
 Fixat en bugg med kopiering av formler
 Fixat en bugg med kopiering av cellintervall
 Fixat en bugg med formateringsnummer med E-exponent

 Kommentarer

- För att lägga till textBox, i den nya versionen, är add(...)-metoder endast giltiga i workheet.addInChart()-metoden är endast giltig i diagramarket.
