 Den här sidan innehåller release notes för[Aspose.Cells for Java V2.0.1 Hotfix](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-v2.0.1-hotfix/)

 Vi är glada att meddela Aspose.Cells for Java 2.0.1!

 Vad har ändrats:

 l Stöd för R/W Open Document Spreadsheet-fil (ODS): Cell data och stil, Rad/Kolumn-stil, Sammanfogad Cells, Egenskaper för kalkylblad/arbetsbok, Dokumentegenskaper, Formel/Namn.

 l Stöd för att ställa in delad formel.

 Stöd för att manipulera kontroller: Kryssruta, knapp, alternativknapp, kombinationsruta, listruta, gruppruta, rullningslist, spinner, etikett.

 l Stöder sidinställningar Sidhuvud(er) och sidfot(er) i xls2pdf.

 l Stöder kopiering av diagram vid kopiering av arbetsblad/arbetsbok

 l Stöd för att kopiera Shape-relaterade objekt

 l Stöder för att spara kalkylblad som html för att streama

 l Stöder PivotChart

 l Stöder för att få bilddata av Picture och OleObject

 l Stöder till R/W färgtema för 2007 filformat

 l Stöder krypteringstyp: Microsoft Stark kryptografisk leverantör

 l 27 korrigeringar och förbättringar.
