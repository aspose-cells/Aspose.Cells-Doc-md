 Den här sidan innehåller release notes för[Aspose.Cells for Java 7.1.1](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.1.1/)

Vi är
 glad att meddela Aspose.Cells for Java v7.1.1!

 Nya egenskaper

- Kontrollera om en kolumn är tom

 Förbättringar

- Spara till XLSX med LightCells API genom att direktrendera till OutputStream

 Undantag

- Cell.setHtmlString()-metoden kastar NullPointerException
- Läsning av en XLSX-mall ger NullPointerException

 Buggar

- Sekundär kategoriaxel för SCATTER-diagram blir textaxel
- Innehållet i Textbox har inte renderats korrekt
- RUNDA-funktionen kan inte avrunda ett värde som är större än 922337,20
- Diagrammet lästes inte korrekt från ODStemplate-filen
- Anpassat nummerformat: "dd.MM.åååå" sparades inte korrekt för ODS-filen
- LineShape blir kortare i det genererade PDF-dokumentet
