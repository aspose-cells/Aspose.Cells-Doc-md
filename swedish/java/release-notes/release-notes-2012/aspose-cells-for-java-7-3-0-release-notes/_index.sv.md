 glad att meddela Aspose.Cells for Java v7.3.0!

 Vad är nytt

 och Skriv MHT-filer

 XML-kartor

 MS Excel 2007/2010 teman till diagram



Andra anmärkningsvärda funktioner, förbättringar och fixar ges nedan.

 Nya egenskaper

 – Stöder inställningar för utvärdering av övergångsformel

 Förbättringar

- Formatera cellvärden med användarspecificerad lokal

 Undantag

- Teckensnittsfiler som inte stöds kan orsaka att spara till PDF misslyckades med undantag

 Buggar

- Metoden Cell.setR1C1Formula() fungerade inte korrekt utan kolumnförskjutning
- Namngivna intervall skapades två gånger
- sortNames()-metoden orsakade undantag när en XLSM-fil sparas
- Talet formaterades inte korrekt för fraktion
- Tom PDF genererad för PrintOrderType.OVER_SEDAN_NER
- Bakgrundsfärg och rutnät är felaktiga i den genererade PDF-filen
- Intercept- och Slope-funktioner beräknades inte korrekt
- Ta bort 33 000 gräns för pivotfältobjekt för Excel 2007-filformat
- Notation 1:1 stöds inte i IFfunction
- DATEDIF-formeln beräknades felaktigt
- Felaktig uppslagning av celler vid namngiven rad
