 Vi är glada att kunna meddela Aspose.Cells för Java v7.1.0!

 Förbättringar

- Ta bort mallfiler efter att ha läst XLSX/ODS-filer med Apaches zip-verktyg

 Undantag

 -Cell.setHtmlString() kastar NullPointerException

 Buggar

 -Cell innehåll försvann från den genererade PDF-filen

 -Felmeddelande: "Pivottabellen skapades i en senare version av Excel..." när filen öppnades i MS Excel

 ` `- Arkets zoom ändras när en XLS-mallfil öppnas

 -Cell.getStringValue() ger felaktigt värde för dubbla typer

 -När du sparar till PDF, trunkeras den högra delen av formens text och en del av bilden går förlorad

 -Nummerformat förlorat när du sparar till ODS-filformat
