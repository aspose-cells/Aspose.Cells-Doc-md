## **Offentlig API och bakåtinkompatibla ändringar**

Följande är en lista över alla ändringar som gjorts för allmänheten API, såsom tillagda, bytt namn, borttagna eller utfasade medlemmar samt alla icke-bakåtkompatibla ändringar som gjorts till Aspose.Cells for Java. Om du har frågor om någon ändring som anges, vänligen ta upp den på supportforumet Aspose.Cells.

### **Lägger till egenskapen PasteOptions.OperationType och PasteOperationType enum .**

 Hämtar och ställer in operationstypen när intervall klistras in.

### **Lägger till metoden PivotFormatCondition.AddColumnAreaCondition(PivotField columnField).**

Lägger till PivotTable-villkorlig formatgräns i kolumnfälten.

### **Lägger till metoden PivotFormatCondition.AddColumnAreaCondition(String fieldName) .**

Lägger till PivotTable-villkorlig formatgräns i kolumnfälten.

### **Lägger till metoden PivotFormatCondition.AddRowAreaCondition(PivotField rowField).**

Lägger till PivotTable-villkorlig formatgräns i radfälten.

### **Lägger till metoden PivotFormatCondition.AddRowAreaCondition(String fieldName) .**

Lägger till PivotTable-villkorlig formatgräns i radfälten.

### **Lägger till metoden PivotFormatCondition.AddDataAreaCondition(PivotField dataField).**

Lägger till PivotTable-villkorlig formatgräns i datafälten.

### **Lägger till metoden PivotFormatCondition.AddDataAreaCondition(String fieldName) .**

Lägger till PivotTable-villkorlig formatgräns i datafälten.

### **Lägger till metoden PivotFormatCondition.SetConditionalAreas().**

Ställer in villkorliga områden för PivotFormatCondition-objektet.

### **Lägger till SeriesCollection.Add(boolesk, boolesk) metod.**

Lägger till serier med ett intervall.

### **Lägger till metoden SetSeriesNames().**

Ställer in ett intervall som namn på serien.
