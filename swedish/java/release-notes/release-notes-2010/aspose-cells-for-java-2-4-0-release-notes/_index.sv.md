 Vi är glada att kunna meddela Aspose.Cells for Java v2.4.0!

 Vad har ändrats:

- Inkluderar funktioner från ark till bild, ark till utskrift och arbetsbok till utskrift
- Stöder villkorlig formatering för Excel till Pdf-funktionen
- Ger stöd för att gruppera pivottabellfält i Excel2007-filer
- Behåller makron vid konvertering av Excel97-2003-filer till Excel2007-filer
- Återger former vid konvertering till Pdf-filer, t.ex. TextBox, Rectangle, Oval, Arc
- Stöder för att få/ställa in fullständigt namn på källfilen för Ole-objektet
- Förbättring görs för att läsa/spara villkorlig formatering
- Förbättring har gjorts för diagram till bild-funktionen
- Förbättring görs för direkt Excel till Pdf-funktion
- Förbättring görs för formatering av cellvärden
- Förbättring görs för att ta bort sidhuvud/sidfot i PageSetup
- Förbättring görs för att kopiera celler
- 22 korrigeringar och förbättringar.
