 Vi är glada att kunna meddela Aspose.Cells för Java v2.4.1!

 Vad har ändrats:

- Stöder mer flexibel modell för beräkning av anpassade funktioner.
- Inkluderar felkontrollalternativ för celler.
- Matematisk beräkning av intervallparametrar i formler läggs till.
- Ger stöd för att ställa in länkade celler för diagramramar som titel, dataetiketter.
- Stöder för att manipulera fönsterposition och storleksegenskaper för arbetsbok.
- Att läsa/spara Excel 2007-filer förbättras.
- Diagram till bild-funktionen är förbättrad.
- Direkt Excel till PDF-funktionen är förbättrad.
- Formatering av Cell-värden förbättras.
- Att läsa Excel 97-2003-filer förbättras.
- Modulen för att spara diagram är förbättrad.
- Formelberäkningsmotorn är förbättrad.
- 21 korrigeringar och förbättringar.
