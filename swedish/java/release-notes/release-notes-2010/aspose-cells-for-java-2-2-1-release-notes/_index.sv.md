 Vi är glada att kunna meddela Aspose.Cells för Java 2.2.1!

 Vad har ändrats:

-  Inkluderar stöd för avancerad villkorlig formatering tillhandahållen av MS Excel 2007
-  Stöder utökade register över villkorlig formatering sparad av MS Excel 2007 för XLS-fil
- Behåller formler som inte stöds vid läsning av MS Excel 2007-filer
-  Läser bilder från sidhuvud/sidfot i MS Excel 2007 mallfil
-  Förbättringar görs för att läsa mallfiler
-  17 korrigeringar och andra förbättringar.
