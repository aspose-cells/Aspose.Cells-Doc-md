 Vi är glada att meddela Aspose.Cells for Java 2.4.2!

 Vad har ändrats:

-  Stöder arrayparametrar (

Okänt makro: {...}

) för att ställa in formler.

- Ger stöd för att lägga till länkade bilder.
- Stöder för att spara celldata direkt i ström eller fil utan att bygga en komplett datamodell i minnet.
- Kontrollera om licensen har ställts in, inkluderar support för att få utgångsdatum för din prenumeration.
- Att läsa/spara Excel 2007-filer förbättras.
- Diagram till bild-funktionen är förbättrad.
- Direkt Excel till PDF-funktionen är förbättrad.
- Funktionen för att formatera Cell värden är förbättrad.
- Att läsa/spara Excel 97-2003-filer förbättras.
- 21 korrigeringar och förbättringar.
