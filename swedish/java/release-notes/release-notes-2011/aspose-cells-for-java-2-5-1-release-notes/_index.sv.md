 Vi är glada att meddela Aspose.Cells for Java 2.5.1!

 Vad har ändrats:

· Stöder för att avbryta öppna/spara-processen när du arbetar med stora filer.

 · Ger stöd för att ställa in och beräkna fler formler.

 · Stöd för att få/ställa in formens absoluta position.

 · Fixar buggar med att läsa/spara Excel 2007-filer.

 · Fixar buggar för att läsa/spara ODS-filer.

 · Fixar buggar med att läsa/spara Excel 97-2003-filer.

 · Förbättringar görs för funktionerna diagram till bild, ark till bild och Excel till PDF.

 · Förbättring görs för att tolka formler.

 · Förbättring görs för att beräkna formler.

 · 22 korrigeringar och förbättringar.
