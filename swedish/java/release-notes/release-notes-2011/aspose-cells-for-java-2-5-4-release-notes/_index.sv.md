## **Vi är glada att meddela Aspose.Cells for Java v2.5.4!**
### **Nya egenskaper**
25051 - Stöd för avancerade autofiltertyper: MultipleFilter, DynamicFilter, IconFilter

 25742 - Formatera hela pivottabellen eller cellerna i pivottabellsrapporten

 26627 - Ställ in axelegenskaper, såsom Max/Min värde som "Automatisk"

 26655 - Exportera animerade GIF-bilder för HTML-fil

 26607 - Stöd för att beräkna EOMONTH-funktionen

 26655 - Importera/exportera datum- och kantstilar för ODS-fil

 27317 - Behåll miniatyrbilden av återsparat diagram samma som originalfilen när du bäddar in XLS i PPT

 27214 - Gör pivotfältets objekt dolda/visade

 26626 - Läs/spara pivottabellstilar
