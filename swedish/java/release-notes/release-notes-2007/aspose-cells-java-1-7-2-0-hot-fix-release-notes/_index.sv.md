 Den här sidan innehåller release notes för[Aspose.Cells Java 1.7.2.0 Hot Fix](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.7.2.0-hot-fix/)

 Kära kunder,

 Vi har släppt Aspose.Cells for Java V1.7.2!Följande förbättringar och korrigeringar ingår:

- Stöder för att lägga till bild i sidhuvudet/sidfoten i sidinställningarna
 Stöder import av sidinställningar, namn, skydd, datavalidering och dokumentegenskaper från filen SpreadsheetML
 Stöder export av sidinställningar, namn, skydd, datavalidering och dokumentegenskaper till SpreadsheetML-fil
 Stöder för att beräkna fler inbyggda funktioner
 Åtgärdar buggen med att ställa in automatisk färg på teckensnittet i diagrammet.
