 Kära kunder,

 Vi har släppt Aspose.Cells för Java V1.7.4!Följande förbättringar och korrigeringar ingår:

- Lägger till metoderna Column.applyStyle och Row.applyStyle.
 Fixar några buggar med inställning och beräkning av formler.
 Fixar en bugg i coping row.
 Åtgärdar en bugg med frysta rutor.
 Fixar en bugg med att spara diagram.
 Fixar en bugg med att spara pivottabeller.
 Fixar en bugg i cells.importResultSet()-metoden.

Obs: Metoden getNumerOfSheets of Workbook/Worksheets har döpts om till getNumberOfSheets. Och det är bättre att anropa Worksheets.size() för att få antalet ark.
