 Den här sidan innehåller release notes för[Aspose.Cells for Java 1.9.0.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-1.9.0.0/)

 Kära kunder,

 Vi har släppt Aspose.Cells for Java V1.9.0!

 Vad är nytt?

- Stöder export till Aspose.Pdf xml-fil
 Stöder för att kopiera kommentarer och namngivna intervall när du hanterar kalkylblad
 Stöder ICustomFunction-gränssnitt för beräkningsformler
Stöder för att spåra predents och dependents av en cell

 Vad har förbättrats?

- Stöder att endast beräkna alla formler i ett kalkylblad
 Stöder POJO och POJO med smarta markörer

 Vad är fixat?

- Fixade en bugg med att få namngivna intervall för kalkylbladet
 Fixade en bugg med att skapa pivottabellen
 Fixat ett fel med att skydda filen
 Fixat en bugg med beräkningsformler
 Fixade en bugg med att spara filen.
