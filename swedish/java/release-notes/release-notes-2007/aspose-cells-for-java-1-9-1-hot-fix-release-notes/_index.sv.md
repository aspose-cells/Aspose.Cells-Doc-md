 Kära kunder,

 Vi har släppt Aspose.Cells for Java V1.9.1!

 Vad är nytt?

- Stöder export till och import från Excel2007xlsx-fil.
- Stöder för att sortera data i intervallet.
- Stöder för att användas med PHP.

 Vad har förbättrats?

-  Stöder för att ställa in skalan och enheten för Time Scale Category-axeln.

 Vad är fixat?

-  Fixat en bugg med beräkningsformler
-  Fixat en bugg med formateringsnummer.
-  Fixade en bugg med att lägga till bild.
-  Fixat ett fel med att spara filen.
