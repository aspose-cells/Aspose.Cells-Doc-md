 Den här sidan innehåller release notes för[Aspose.Cells Java 1.7.3.0 Hot Fix](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.7.3.0-hot-fix/)

 Kära kunder,

 Vi har släppt Aspose.Cells for Java V1.7.3!Följande förbättringar och korrigeringar ingår:

- Stöder för att infoga/ta bort ett intervall.
 Stöder för att öppna och spara krypterad Excel-fil.
 Förbättrar metoden Cell.getStringValue() för att stödja fler datum-/talformat.
 Uppdaterade webbdemos.
 Stöder för att beräkna formler med JDK1.4.
 Fixar en bugg av License.setLicense()-metoden.
 Fixar en bugg med att ställa in anpassat nummerformat.
Fixar en bugg med att ta bort ett kalkylblad från arbetsboken.
