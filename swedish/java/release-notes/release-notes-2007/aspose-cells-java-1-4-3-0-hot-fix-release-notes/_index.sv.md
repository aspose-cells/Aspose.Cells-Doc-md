 Kära kunder,

 Vi har släppt Aspose.Cells för Java V1.4.3.0!Följande förbättringar och korrigeringar ingår:

- Stöder för att lägga till textrutor, kommentarer vid körning
- Stöd för dokumentegenskaper
- Ställa in sjökortspunkt
- Ange disposition på ett kalkylblad
- Ställa in position och storlek på sjökortsområdet och sjökortsområdet
- Ställa in position, fyllning, färg och storlek på formen
- Importera data från array och samling
- Fixade buggar i autofilterinställningar
- Fixat en bugg med anpassat nummerformat
- Fixade en bugg med att infoga/ta bort rader och kolumner
