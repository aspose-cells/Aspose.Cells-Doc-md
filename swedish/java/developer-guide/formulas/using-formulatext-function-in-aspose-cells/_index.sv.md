{{% alert color="primary" %}} 

*FormulaText* är en funktion för Excel 2013 och senare. Den stöds inte av tidigare versioner som Excel 2010 eller 2007 osv. Som namnet antyder skriver den ut texten i formeln som finns i en given cell. Den här artikeln kommer att visa dig hur du kan använda denna funktion med hjälp av Aspose.Cells.

{{% /alert %}}
