 I vanliga fall,[namngivna intervall](/cells/sv/java/named-ranges/) är rektangulära med celler kontinuerliga och intill varandra. Men ibland kan du behöva använda ett icke-sekventiellt cellområde där cellerna inte är intill varandra. Aspose.Cells stöder att skapa ett namngivet område med icke-intilliggande celler. För detta tillhandahåller API[**Name.RefersTo**](https://reference.aspose.com/cells/java/com.aspose.cells/name#RefersTo) fast egendom.
