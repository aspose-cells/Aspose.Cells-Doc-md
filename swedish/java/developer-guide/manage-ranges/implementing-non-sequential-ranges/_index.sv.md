Normalt är [namngivna områden](/cells/sv/java/named-ranges/) rektangulära med sammanhängande celler intill varandra. Men ibland kan det behövas använda ett icke-sequentiellt cellområde där celler inte är intilliggande. Aspose.Cells stödjer skapandet av ett namngivet område med icke-intilliggande celler. För detta tillhandahåller API:et egenskapen [**Name.RefersTo**](https://reference.aspose.com/cells/java/com.aspose.cells/name#RefersTo).
