## **Introduktion**

I Excel kan du välja ett intervall, sedan infoga ett intervall och skifta andra data höger eller ned.

**![Skiftalternativ](InsertRange.png)**

## **Infoga Intervall med Aspose.Cells**

Aspose.Cells ger metoden [Cells.InsertRange](https://reference.aspose.com/cells/java/com.aspose.cells/cells#insertRange-com.aspose.cells.CellArea-int-) för att infoga ett område.
