## **Skapa unionsspann**
Aspose.Cells ger möjligheten att skapa unionsområden genom att använda metoden [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/java/com.aspose.cells/worksheetcollection#createUnionRange-java.lang.String-int-). Metoden accepterar två parametrar: adressen för att skapa unionområdet och indexet för arket. Metoden returnerar ett [UnionRange](https://reference.aspose.com/cells/java/com.aspose.cells/UnionRange) objekt.

Följande kodexempel visar hur man skapar ett unionområde med hjälp av metoden [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/java/com.aspose.cells/worksheetcollection#createUnionRange-java.lang.String-int-). Den genererade output-filen finns bifogad för referens.
