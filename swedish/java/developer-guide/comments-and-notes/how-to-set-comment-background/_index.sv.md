Kommentarer läggs till i celler för att spela in kommentarer, allt från detaljer om hur en formel fungerar, var en värde kommer från eller frågor från granskare. Kommentarer spelar en extremt viktig roll när flera personer diskuterar eller granskar samma dokument vid olika tillfällen. Hur man skiljer olika personers kommentarer åt? Ja, vi kan ställa in en annan bakgrundsfärg för varje kommentar. Men när vi behöver behandla många dokument och många kommentarer, är det att göra det manuellt en katastrof. Lyckligtvis tillhandahåller [**Aspose.Cells**](https://products.aspose.com/cells/java/) ett API som låter dig göra detta i kod.
