Kommentarer läggs till celler för att registrera kommentarer, allt från detaljerna om hur en formel fungerar, var ett värde kommer ifrån eller frågor från granskare. Kommentarer spelar en extremt viktig roll när flera personer diskuterar eller granskar samma dokument vid olika tidpunkter. Hur särskiljer man olika personers kommentarer? Ja, vi kan ställa in olika bakgrundsfärger för varje kommentar. Men när vi behöver behandla många dokument och många kommentarer är det en katastrof att göra det manuellt. som tur är[**Aspose.Cells**](https://products.aspose.com/cells/java/) tillhandahåller ett API som låter dig göra detta i kod.
