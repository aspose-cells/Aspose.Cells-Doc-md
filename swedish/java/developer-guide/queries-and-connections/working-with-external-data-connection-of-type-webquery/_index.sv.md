{{% alert color="primary" %}} 

Du kan komma åt extern dataanslutning av vilken typ som helst genom att använda [Workbook.DataConnections](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections) samlingen. En typ av en sådan dataanslutning är WebQuery. I den här artikeln visas hur du kan arbeta med WebQuery-dataanslutningen. Du kan skapa WebQuery-dataanslutning i Microsoft Excel genom att använda menyobjektet **Data** > **Från webben**.

{{% /alert %}}
