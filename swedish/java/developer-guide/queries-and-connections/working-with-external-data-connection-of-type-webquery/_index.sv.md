{{% alert color="primary" %}} 

 Du kan komma åt den externa dataanslutningen av vilken typ som helst med hjälp av[Arbetsbok.Dataanslutningar](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections) samling. En typ av sådan dataanslutning är WebQuery. Den här artikeln visar hur du arbetar med WebQuery-dataanslutningen. Du kan skapa WebQuery-dataanslutning i Microsoft Excel med hjälp av**Data** > **Från webben** meny.

{{% /alert %}}
