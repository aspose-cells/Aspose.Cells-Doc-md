{{% alert color="primary" %}} 

Ibland vill du kanske hämta information om externa datakällor. Till exempel kan du vilja se SQL-anslutningsdata. Denna information kan inkludera alla typer av data som krävs för att ansluta till en SQL-server, t.ex. server-URL, användarnamn, underliggande tabellnamn, SQL-fråga, frågetyp, tabellens plats osv.

Aspose.Cells tillhandahåller några användbara anrop för att hämta sådana detaljer från företagsdatabaser som är länkade till mall-excelfiler.

{{% /alert %}}
