{{% alert color="primary" %}} 

Ibland behöver du beräkna bredd och höjd av cellvärdet för att passa cellvärdet inuti cellen. Aspose.Cells tillhandahåller metoder [Cell.getWidthOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getWidthOfValue\(\)) och [Cell.getHeightOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getHeightOfValue\(\)) för detta ändamål. Genom att använda dessa metoder kan du beräkna bredd och höjd av cellvärdet och sedan ställa in kolumnens bredd och radens höjd för den cellen respektive, vilket sedan justerar eller passar cellvärdet inuti cellen.

Alternativt kan du även [anpassa rader och kolumner för din cell eller cellintervall](/cells/sv/java/autofit-rows-and-columns/) med hjälp av Aspose.Cells API:er.
