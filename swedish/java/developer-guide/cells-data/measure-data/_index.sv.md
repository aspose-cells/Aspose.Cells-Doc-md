{{% alert color="primary" %}} 

 Ibland behöver du beräkna bredden och höjden på cellvärdet för att passa cellvärdet inuti cellen. Aspose.Cells tillhandahåller[Cell.getWidthOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getWidthOfValue\(\) ) och[Cell.getHeightOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getHeightOfValue\(\)metoder för detta ändamål. Genom att använda dessa metoder kan du beräkna bredd och höjd på cellvärdet och sedan ställa in bredden på kolumnen och höjden på raden i den cellen respektive och detta kommer sedan att justera eller passa cellvärdet inuti cellen.

 Alternativt kan du också[autopassa rader och kolumner i din cell eller cellintervall](/cells/sv/java/autofit-rows-and-columns/) använder Aspose.Cells API:er.

{{% /alert %}}
