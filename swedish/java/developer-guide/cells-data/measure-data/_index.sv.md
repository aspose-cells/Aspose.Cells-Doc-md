{{% alert color="primary" %}} 

Ibland måste du beräkna bredden och höjden på cellvärdet för att passa cellens innehåll. Aspose.Cells tillhandahåller metoder [Cell.getWidthOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getWidthOfValue--) och [Cell.getHeightOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getHeightOfValue--) för detta ändamål. Med hjälp av dessa metoder kan du beräkna bredd och höjd på cellvärdet och sedan ställa in kolumnens bredd och radens höjd för den cellen, vilket då justerar eller passar cellinnehållet.

Alternativt kan du även [anpassa rader och kolumner för din cell eller cellintervall](/cells/sv/java/autofit-rows-and-columns/) med hjälp av Aspose.Cells API:er.
