{{% alert color="primary" %}}

Aspose.Cells stöder konvertering av Microsoft Excel-orienterad HTML till XLS/XLSX-format. Det innebär att den HTML som genereras av Microsoft Excel kan konverteras tillbaka till XLS/XLSX-format med hjälp av Aspose.Cells.

På samma sätt kan, om det finns enkel HTML, Aspose.Cells konvertera den till HTML Rich Text. Aspose.Cells tillhandahåller egenskapen [**Cell.setHtmlString()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#HtmlString) som kan ta sådan enkel HTML och konvertera den till formaterad celltext.

{{% /alert %}}
