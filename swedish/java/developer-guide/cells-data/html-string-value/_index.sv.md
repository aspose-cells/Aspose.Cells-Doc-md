{{% alert color="primary" %}}

Aspose.Cells stöder konvertering av Microsoft Excel-orienterad HTML till XLS/XLSX-format. Det betyder att HTML som genereras av Microsoft Excel kan konverteras tillbaka till XLS/XLSX-format med Aspose.Cells.

 På samma sätt, om det finns någon enkel HTML, kan Aspose.Cells konvertera den till HTML Rich Text. Aspose.Cells tillhandahåller[**Cell.setHtmlString()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#HtmlString) egenskap som kan ta en så enkel HTML och konvertera den till formaterad celltext.

{{% /alert %}}
