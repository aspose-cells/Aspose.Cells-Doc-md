## **Möjliga användningsscenario**

Läs siffror kalkylblad utvecklade av Apple Inc. med hjälp av Aspose.Cells

## **Läsa Numbers-kalkylblad utvecklat av Apple Inc. med Aspose.Cells**

Följande kodexempel laddar [Exempel Numbers kalkylblad](sampleNumbersByAppleInc.numbers) och konverterar det till [Output PDF Format](outputNumbersByAppleInc.pdf). Du måste använda [**LoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/LoadOptions) klassen och ange [**LoadFormat.NUMBERS*](https://reference.aspose.com/cells/java/com.aspose.cells/loadformat#NUMBERS) som en parameter i dess konstruktor för att ladda det framgångsrikt. Se till att ladda ner båda filerna från de angivna länkarna. Du kan prova koden med valfritt Numbers kalkylblad. Läs även kommentarerna till koden för mer hjälp.
