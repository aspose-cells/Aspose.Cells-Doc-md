## **Hämta extern anslutningsdatakälla för pivottabell**

Aspose.Cells ger möjlighet att hämta extern datakälla för pivottabellen. För detta tillhandahåller API:et egenskapen [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) av klassen [**PivotTable**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable). Egenskapen [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) returnerar objektet [**ExternalConnection**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection). Den följande kodsnutten visar användningen av egenskapen [**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) för att hämta extern datakälla för pivottabellen.
