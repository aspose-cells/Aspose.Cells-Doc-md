Aspose.Cells ger möjligheten att spara pivottabell i ODS-filer. För detta konverterar du bara en arbetsbok med en befintlig pivottabell eller skapar en ny pivottabell och sparar filen i ODS-formatet. Se till att anropa [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData--) innan du sparar för att säkerställa att pivottabellen renderas i den slutgiltiga ODS-filen. Följande kodsnutt demonstrerar att spara pivottabell i ODS-fil.
