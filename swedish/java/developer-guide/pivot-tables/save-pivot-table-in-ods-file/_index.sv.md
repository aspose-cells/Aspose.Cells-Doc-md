 Aspose.Cells ger möjlighet att spara pivottabell i ODS-filer. För detta konverterar du bara en arbetsbok med befintlig povot-tabell eller skapar en ny pivottabell och sparar filen i ODS-formatet. Se till att ringa[**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData()innan du sparar för att säkerställa att pivottabellen återges i ODS-utdatafilen. Följande kodavsnitt visar hur pivottabellen sparas i ODS-fil.
