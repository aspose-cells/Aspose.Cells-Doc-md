{{% alert color="primary" %}}

 Microsoft Excel låter dig ändra layouten för pivottabellen med*Pivottabellverktyg > Design > Rapport* Layoutmenykommandon. Du kan ändra layouten i dessa tre former

- Visa i kompakt form
- Visa i dispositionsformulär
- Visa i tabellform

 Aspose.Cells tillhandahåller också[**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm()), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm() ) och[**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm()) metoder för att ändra layouten för pivottabellen i dessa tre former.
