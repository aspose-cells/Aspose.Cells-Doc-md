{{% alert color="primary" %}}

Microsoft Excel gör det möjligt för dig att ändra layouten i Pivot-tabell med *PivotTable Tools > Design > Rapport* Layout-menyalternativ. Du kan ändra layouten på dessa tre former

- Visa i kompakt form
- Visa i kontursform
- Visa i tabellform

Aspose.Cells tillhandahåller också [**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm--), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm--) och [**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm--) metoder för att ändra layouten för pivot-tabell i dessa tre former.
