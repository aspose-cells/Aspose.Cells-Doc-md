## **Konvertera JSON till Excel-arbetsbok**

Behöver du inte undra över hur du konverterar JSON till excelfil, eftersom Aspose.Cells Java-bibliotek har bästa lösningen. Aspose.Cells Java API tillhandahåller stöd för konvertering av JSON-format till kalkylblad. Du kan använda klassen [**JsonLoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLoadOptions) för att ange ytterligare inställningar för import av JSON till arbetsbok.
