{{% alert color="primary" %}}

Aspose.Cells stöder konvertering av XLS-filer som innehåller bilder och diagram till PDF-dokument. Aspose.Cells for Java kan arbeta självständigt för att konvertera en kalkylblad till PDF: Aspose.PDF-API:er behövs inte för konverteringen.

{{% /alert %}}

Processen kan göras i minnet eftersom processen inte är beroende av temporära eller mellanliggande XML-filer. Det innebär att stora Excel-filer, till exempel de som innehåller bilder, diagram och andra ritobjekt, kan konverteras snabbt och effektivt.

{{% alert color="primary" %}}

Om kalkylbladet innehåller formler är det bäst att anropa metoden [**Workbook.calculateFormula**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#calculateFormula--) strax före rendering till PDF. Genom att göra det säkerställs att formelberoende värden omberäknas och de korrekta värdena visas i PDF:n.
