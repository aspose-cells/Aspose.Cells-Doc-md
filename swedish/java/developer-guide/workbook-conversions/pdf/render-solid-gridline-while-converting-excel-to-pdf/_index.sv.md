För kompatibilitet med äldre versioner renderar Aspose.Cells rutnätslinjen som prickad linje som standard vid konvertering av Excel till PDF. Moderna Excel-versioner visar dock rutnätslinjen som solid linje nuförtiden.

Med alternativet [PdfSaveOptions.GridlineTypes](https://reference.as pose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setGridlineType-int-), kan Aspose.Cells även rendera rutnätslinjen som solid linje.
