PDF logiska strukturfaciliteter tillhandahåller en mekanism för att införliva information om dokumentinnehållsstrukturen i en PDF-fil. Aspose.Cells bevarar information om strukturen från ett Microsoft Excel-dokument, som cell, rad, tabell, kalkylblad, bild, form, sidhuvud/sidfot, etc.

 Med option[PdfSaveOptions.ExportDocumentStructure](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setExportDocumentStructure-boolean-), kan du spara till en taggad PDF med dokumentstruktur exporterad.
