{{% alert color="primary" %}} 

 Som standard sparar Aspose.Cells Excel i PDF med standardstorlek. Men du kan också spara den med minsta storlek med hjälp av[PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions#OptimizationType) fast egendom. Den accepterar följande värden

- [PdfOptimizationType.STANDARD](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#STANDARD)
- [PdfOptimizationType.MINIMUM_SIZE](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#MINIMUM_SIZE)

{{% /alert %}}
