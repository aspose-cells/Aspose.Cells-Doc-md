{{% alert color="primary" %}} 

Som standard sparar Aspose.Cells Excel som PDF med standardstorlek. Du kan dock också spara den med minsta storlek genom att använda [PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions#OptimizationType)-egenskapen. Den accepterar följande värden

- [PdfOptimizationType.STANDARD](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#STANDARD)
- [PdfOptimizationType.MINIMUM_SIZE](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#MINIMUM-SIZE)

{{% /alert %}}
