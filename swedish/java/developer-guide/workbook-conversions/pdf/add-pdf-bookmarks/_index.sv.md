{{% alert color="primary" %}} 

Den här artikeln innehåller information om hur du infogar PDF-bokmärken när du konverterar ett kalkylblad till PDF.

Aspose.Cells låter dig lägga till bokmärken under körning. PDF bokmärken kan drastiskt förbättra navigerbarheten för långa dokument. När du lägger till bokmärkeslänkar till ett PDF-dokument kan du ha exakt kontroll över den exakta vy du vill ha, du är inte begränsad till att länka till en sida. Du kan ställa in den exakta vyn genom att placera målsidan och sedan skapa bokmärket.

{{% /alert %}}
