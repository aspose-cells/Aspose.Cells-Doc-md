Liksom PDF är XPS ett dokumentformat med fast layout utformat för att bevara dokumenttrohet och ge enhetsoberoende dokumentutseende.|

## **XPS i Aspose.Cells**
Alla dokument som laddas in i Aspose.Cells kan konverteras till XPS.

Aspose.Cells använder samma sidlayout och renderingsmotor med hög kvalitet för att producera utdata i alla dess format med fast layout som stöds, t.ex. PDF, XPS etc.
