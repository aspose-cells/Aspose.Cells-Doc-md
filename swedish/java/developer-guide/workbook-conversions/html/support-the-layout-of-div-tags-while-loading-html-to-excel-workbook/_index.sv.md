{{% alert color="primary" %}} 

Normalt ignoreras layouten av div-taggar vid laddning av HTML till ett Excel-arbetsboksobjekt. Om du vill att layouten av div-taggar inte ska ignoreras, ställ in egenskapen [HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag) till **true**. Standardvärdet för denna egenskap är **false**.

{{% /alert %}}
