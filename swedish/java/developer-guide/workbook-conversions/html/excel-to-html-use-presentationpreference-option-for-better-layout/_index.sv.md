{{% alert color="primary" %}}

Aspose.Cells tillhandahåller en användbar egenskap, [**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference), som utvecklare kan använda för att få bättre layout när de sparar en Microsoft Excel-fil till HTML- eller MHT-format. Standardvärdet för egenskapen är **false**. Vi rekommenderar att du sätter denna egenskap till **true** för att få bättre utseende presentationer från en Excel-rapport.

{{% /alert %}}
