{{% alert color="primary" %}}

Aspose.Cells tillhandahåller en användbar egenskap,[**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference) , som utvecklare kan använda för att få bättre layout när de sparar en Microsoft Excel-fil till HTML eller MHT-format. Standardvärdet för fastigheten är**falsk** . Vi rekommenderar att du ställer in den här egenskapen till**Sann** för att få snyggare presentationer från en Excel-rapport.
