## **Möjliga användningsscenario**

Aspose.Cells låter dig förprefixa tabellens elementstilar med [**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId)egenskap. Antag att du ställer in denna egenskap med ett värde som *MyTest_TableCssId*, då kommer du hitta tabellens elementstilar som visas nedan

{{< highlight java >}}

table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}
