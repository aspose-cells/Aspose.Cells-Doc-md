## **Möjliga användningsscenarier**

Aspose.Cells låter dig prefixa tabellelementstilar med[**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId)fast egendom. Anta att du ställer in den här egenskapen med något värde som*MyTest_TableCssId*, då hittar du tabellelementstilar som visas nedan

{{< highlight "java" >}}

table#MyTest_TableCssId

# MyTest_TableCssId tr

# MyTest_TableCssId col

# MyTest_TableCssId br

etc.

{{< /highlight >}}
