{{% alert color="primary" %}}

Aspose.Cells stöder sparande (**OpenDocument Kalkylblad**) ODS-fil i (**OpenDocument Format** ) Specifikationer för ODF 1.1 och ODF 1.2. Aspose.Cells har lagt till[**OdsSaveOptions.setStrictSchema11()**](https://reference.aspose.com/cells/java/com.aspose.cells/odssaveoptions#IsStrictSchema11) egenskap för att använda ODF 1.1-specifikationen för att spara ODS-filer. Standardvärdet för den här egenskapen är**falsk**, så ODS-fil som sparats utan dessa speciella inställningar kommer att använda 1.2-specifikationen.

{{% /alert %}}

Exempelkoden nedan skapar arbetsboksobjektet, lägger till ett visst värde i cell A1 i det första kalkylbladet och sparar sedan ODS-filen i ODF 1.1- och 1.2-specifikationer. Som standard sparas ODS-filen i ODF 1.2-specifikationen.
