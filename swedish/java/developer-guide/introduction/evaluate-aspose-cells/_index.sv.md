{{% alert color="primary" %}}

 Du kan enkelt ladda ner Aspose.Cells för utvärdering från[nedladdningssida](https://releases.aspose.com/cells/java/). Nedladdningen av utvärderingen är densamma som nedladdningen för den fullständiga versionen: utvärderingen blir licensierad när du ansöker om en licens.

{{% /alert %}}

Utvärderingsversionen av Aspose.Cells (det vill säga Aspose.Cells utan tillämplig licens) tillhandahåller det mesta av produktfunktionaliteten. En begränsning är att det alltid injicerar ett extra kalkylblad med utvärderingsvattenstämpel på öppna/spara alla kalkylarksdokument.
