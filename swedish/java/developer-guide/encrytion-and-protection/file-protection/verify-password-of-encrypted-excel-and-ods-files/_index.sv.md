Om Excel (xlsx, xlsb, xls, xlsm) och Open office (ODS)-filer är låsta med lösenord, stöder Aspose.Cells for Java enkel lösenordsverifiering utan att tolka specifik data för filerna.
{{% /alert %}}

## **Verifiera lösenordet för den krypterade filen**

 För att verifiera lösenordet för den krypterade filen tillhandahåller Aspose.Cells for Java[**Verifiera lösenord**](https://reference.aspose.com/cells/java/com.aspose.cells/fileformatutil#verifyPassword(java.io.InputStream,%20java.lang.String))metod. Metoderna accepterar två parametrar, filströmmen och lösenordet som måste verifieras.
 Följande kodavsnitt visar användningen av[**Verifiera lösenord**](https://reference.aspose.com/cells/java/com.aspose.cells/fileformatutil#verifyPassword(java.io.InputStream,%20java.lang.String)) metod för att verifiera om det angivna lösenordet är giltigt eller inte.
