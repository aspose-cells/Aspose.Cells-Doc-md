## **Möjliga användningsscenarier**

Du kan ange författarens namn samtidigt som du skrivskyddar din arbetsbok med Aspose.Cells API. Snälla använd[**Workbook.Settings.WriteProtection.Author**](https://reference.aspose.com/cells/java/com.aspose.cells/writeprotection#Author)egendom för detta ändamål.
