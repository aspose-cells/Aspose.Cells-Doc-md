## [**Encrypt and decrypt Excel files**](/cells/sv/java/encrypt-and-decrypt-excel-files/)
För att förhindra andra från att komma åt data i Excel-filer, skydda Excel-filen med ett lösenord.

## [**Protect and unprotect workbook**](/cells/sv/java/protect-and-unprotect-workbook-structure/)
För att förhindra att andra användare ser dolda kalkylblad, lägger till, flyttar, tar bort eller gömmer kalkylbladen, och döper om kalkylbladen, kan du skydda strukturen för din Excel-arbetsbok med ett lösenord.

## [**Protect and unprotect worksheet**](/cells/sv/java/protect-and-unprotect-worksheet/)
För att förhindra att andra användare avsiktligt eller oavsiktligt ändrar, flyttar eller tar bort data i ett kalkylblad kan du låsa cellerna i ditt Excel-kalkylblad och sedan skydda kalkylbladet med ett lösenord. 

## [**Protect and unprotect shared workbook**](/cells/sv/java/password-protect-or-unprotect-the-shared-workbook/)
Skydda och avskydda den delade arbetsboken.

## [**Digital signature in Office files**](/cells/sv/java/assign-and-validate-digital-signatures/)
Skydda dokumentets äkthet, du kan lägga till en digital signatur.
