## **Möjliga användningsscenario**
Du kan skapa en diagram PDF med önskad sidstorlek med Aspose.Cells och ange hur du vill justera diagrammet på sidan. Du kan justera diagrammet till toppen, botten, mitten, vänster, höger, osv. Dessutom kan utmatningsdiagrammet skapas i en ström eller på disk.
## **Skapa diagram-PDF med önskad sidstorlek**
Se följande kodexempel som laddar in [provexempelexcelfilen](64716912.xlsx), får åtkomst till det första diagrammet i kalkylarket och konverterar det sedan till [utmatnings-PDF](64716911.pdf) med önskad sidstorlek. Skärmbilden nedan visar att sidstorleken i utmatnings-PDF är 7x7 enligt angivet i koden och diagrammet är centerjusterat både horisontellt och vertikalt.
