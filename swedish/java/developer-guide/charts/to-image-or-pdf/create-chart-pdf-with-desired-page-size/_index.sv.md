## **Möjliga användningsscenarier**
Du kan skapa en diagram-PDF med önskad sidstorlek med Aspose.Cells och ange hur du vill justera diagrammet inuti sidan. Du kan justera diagrammet till toppen, botten, mitten, vänster, höger, etc. Dessutom kan utdatadiagrammet skapas i en ström eller på disk.
## **Skapa diagram PDF med önskad sidstorlek**
Se följande exempelkod som laddar[exempel på Excel-fil](64716912.xlsx)kommer åt det första diagrammet i kalkylbladet och konverterar det sedan till[mata ut PDF](64716911.pdf)med önskad sidstorlek. Följande skärmdump visar att sidstorleken i den utgående PDF-filen är 7x7 som specificerats i koden och diagrammet är mittjusterat både horisontellt och vertikalt.
