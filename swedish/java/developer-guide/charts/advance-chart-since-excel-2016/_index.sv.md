## **Möjliga användningsscenario**
Aspose.Cells stöder läsning och hantering av Microsoft Excel 2016-diagram som inte finns i Microsoft Excel 2013 eller äldre versioner.
## **Läs och hantera Excel 2016-diagram**
Den följande exempelkoden laddar [käll-Excel-filen](23166980.xlsx) som innehåller Microsoft Excel 2016-diagram i den första arbetsbladet. Den läser alla diagram ett efter ett och ändrar deras titel enligt deras diagrumtyp. Den följande skärmbilden visar käll-Excel-filen innan körningen av koden. Som du kan se är diagramtiteln densamma för alla diagram.

![todo:image_alt_text](read-and-manipulate-excel-2016-charts_1.png)

Den följande skärmbilden visar den [utmatade Excel-filen] (23166978.xlsx) efter körningen av koden. Som du kan se har diagramtiteln ändrats enligt deras diagramtyp.

![todo:image_alt_text](read-and-manipulate-excel-2016-charts_2.png)
