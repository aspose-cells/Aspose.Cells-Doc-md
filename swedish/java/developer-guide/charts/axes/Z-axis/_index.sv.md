##  **Möjliga användningsscenarier**
För vissa 3D-diagram som 3D-kolumn, 3D-kon eller 3D-pyramid som har en djupaxel (serieaxel), även känd som Z-axeln, som du kan ändra. Du kan ange intervallet mellan bockmarkeringar, axeletiketter och annan operation.
##  **Hantera primära och andra axeln som Microsoft Excel**
 Se följande exempelkod som skapar en ny Excel-fil och anger värden för diagrammet i det första kalkylbladet. Sedan lägger vi till ett diagram och ställer in diagramtypen på Column3D, då kan du se Z-axeln även kallad Depth Axis.
