I Excel-diagram finns det 3 typer av axlar:
1. Horisontell (Kategori) Axel : X-axel
2. Vertikal (Värde) Axel : Y-axel
3. Djup (Serie) Axel : Z-axel



{{% /alert %}}

## **Axelealternativ**
Aspose.Cells tillåter även att hantera diagrammets axlar vid körning，med [Axel](https://reference.aspose.com/cells/java/com.aspose.cells/Axis)-objekt kan du ändra alla axelalternativ som görs i Excel.
