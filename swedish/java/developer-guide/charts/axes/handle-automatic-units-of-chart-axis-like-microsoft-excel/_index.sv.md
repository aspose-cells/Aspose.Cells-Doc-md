## **Möjliga användningsscenario**
Tidiga versioner av Aspose.Cells kunde inte hantera automatiska enheter för diagramaxeln på rätt sätt när diagrammet renderades till bild eller PDF. Nu stöder Aspose.Cells hanteringen av automatiska enheter för diagramaxeln. Det krävs ingen kodändring. Konvertera bara ditt diagram till bild eller PDF och det kommer att rendera diagramaxeln precis som Microsoft Excel renderar den.
## **Hantera automatiska enheter för diagramaxeln som Microsoft Excel**
Nedanstående exempelkod laddar [inmatnings Excel-filen](61767769.xlsx) och genererar [utdata PDF-diagram](61767768.pdf). Skärmbilden visar de automatiska enheterna för diagramaxeln i röda rektanglar och jämför också inmatnings Excel-filsdiagrammet med utdata PDF-diagrammet. Båda är exakt likadana.
