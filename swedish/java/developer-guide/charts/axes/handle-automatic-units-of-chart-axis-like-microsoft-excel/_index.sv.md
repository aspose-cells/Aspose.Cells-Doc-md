## **Möjliga användningsscenarier**
Tidiga versioner av Aspose.Cells kunde inte hantera automatiska enheter av sjökortsaxeln korrekt när sjökortet renderades till bild eller PDF. Nu stöder Aspose.Cells hanteringen av automatiska enheter för sjökortsaxel. Det finns ingen kodändring. Konvertera bara ditt diagram till bild eller PDF och det kommer att återge diagramaxeln precis som Microsoft Excel återger den.
## **Hantera automatiska enheter för diagramaxeln som Microsoft Excel**
Följande exempelkod laddar[exempel på Excel-fil](61767769.xlsx)och genererar[utgång PDF diagram](61767768.pdf)Skärmdumpen visar de automatiska enheterna för diagramaxeln i röda rektanglar och jämför även exemplet på Excel-fildiagrammet med utgångsdiagrammet PDF. Båda är exakt lika.
