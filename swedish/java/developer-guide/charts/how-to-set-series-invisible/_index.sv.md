## Hur man ställer in serie som osynlig i Excel-diagram

I Excel-diagram kan du högerklicka på ett diagram, klicka på "Välj data", och i popup-fönstret kan du ställa in om en serie är synlig genom att markera eller avmarkera den.
Du kan ladda ner följande [exempel fil](SeriesFiltered.xlsx) och arbeta med den i Excel som visas i figuren för att uppnå denna funktion. Nästa kommer vi att visa hur du gör detta med Aspose.Cells-biblioteket.
