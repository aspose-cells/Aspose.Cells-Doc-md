##  **Möjliga användningsscenarier**
Kombinationsdiagram i Excel låter dig utnyttja det här alternativet eftersom du enkelt kan kombinera två eller flera diagramtyper för att göra dina data begripliga. Kombinationsdiagram är användbara när din data innehåller flera typer av värden inklusive pris och volym. Dessutom är kombinationsdiagram möjliga när dina datasiffror ändras kraftigt från serie till serie.
 Med följande datauppsättning som exempel kan vi observera att dessa data är ganska lika de data som nämns i[**VHCL**](https://docs.aspose.com/cells/java/create-volume-high-low-close-stock-chart/).
