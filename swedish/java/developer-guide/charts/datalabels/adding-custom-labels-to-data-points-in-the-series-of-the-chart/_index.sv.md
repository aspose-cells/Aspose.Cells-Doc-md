{{% alert color="primary" %}} 

 Du kan lägga till anpassade etiketter till datapunkter i diagramserien. Aspose.Cells tillhandahåller[ChartPoint.getDataLabels().setText()](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#Text)egendom för att lägga till dessa anpassade etiketter. Den här artikeln kommer att förklara hur du använder den här egenskapen för att lägga till anpassade etiketter till datapunkter i diagramserien.

{{% /alert %}} 
## **Lägga till anpassade etiketter till datapunkter i diagramserien**
Följande kod skapar spridningsdiagram anslutna med linjer med datamarkörer och lägger sedan till anpassade etiketter till datapunkterna i diagramserien. Varje anpassad etikett visar serienamn och punktnamn. Du kan använda vilken annan text som helst istället för den. Efter exekvering av koden skapas följande Excel-fil. Som du kan se i diagrammet har varje DataPoint anpassad etikettuppsättning.
