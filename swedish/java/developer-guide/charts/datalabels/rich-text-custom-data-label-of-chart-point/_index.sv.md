{{% alert color="primary" %}}

 Du kan använda Aspose.Cells för att skapa en anpassad dataetikett med rik text för diagrampunkten. Aspose.Cells tillhandahåller[**DataLabels.characters()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#characters(int,%20int) ) metod för att returnera[**FontSetting**](https://reference.aspose.com/cells/java/com.aspose.cells/FontSetting)objekt som kan användas för att ställa in teckensnittsegenskaperna för texten som dess färg, djärvhet, etc.

{{% /alert %}}

## **Rich Text anpassad dataetikett för diagrampunkt**

Följande kod kommer åt den första diagrampunkten i den första serien, ställer in dess text och ställer sedan in teckensnittet för de första 10 tecknen genom att inställningarna är färgen till röd och fetstilt till sann.
