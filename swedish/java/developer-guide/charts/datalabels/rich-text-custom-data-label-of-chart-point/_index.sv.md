{{% alert color="primary" %}}

Du kan använda Aspose.Cells för att skapa en rik anpassad textdatamarkör för diagrammets punkt. Aspose.Cells tillhandahåller metoden [**DataLabels.characters()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#characters-int-int-) för att returnera [**FontSetting**](https://reference.aspose.com/cells/java/com.aspose.cells/FontSetting) objektet som kan användas för att ställa in textens typsnittsegenskaper, som dess färg, fetstil, osv.

{{% /alert %}}

## **Riktextanpassade datamärken för diagrammet**

Följande kod får åtkomst till den första punkten i den första serien, ställer in dess text och ställer sedan in typsnittet för de första 10 tecknen genom att ställa in dess färg till röd och fetstil till sann.
