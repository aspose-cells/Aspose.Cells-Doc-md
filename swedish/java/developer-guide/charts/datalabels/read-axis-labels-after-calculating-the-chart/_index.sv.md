##  **Möjliga användningsscenarier**

Du kan läsa axeletiketter för ditt diagram efter att ha beräknat dess värden med hjälp av[**Chart.calculate()**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/#calculate--)metod. Vänligen använd[**Axis.getAxisTexts()**](https://reference.aspose.com/cells/java/com.aspose.cells/axis/#getAxisTexts--)metod för detta ändamål som kommer att returnera listan med axeletiketter.
