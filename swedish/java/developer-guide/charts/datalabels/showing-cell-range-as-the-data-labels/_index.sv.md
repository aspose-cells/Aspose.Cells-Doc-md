## Visa cellintervall som datamarkörer i MS Excel

I Microsoft Excel 2013 kan du visa cellintervall för datamarkörer. Du kan välja det här alternativet genom att följa dessa steg

- Välj Datamarkörer för Serier och högerklicka för att öppna snabbmenyn.
- Klicka på **Formatera datamarkörer...** och det kommer att visa **Markeringalternativ**.
- Markera eller avmarkera kryssrutan **Markering innehåller - Värde från celler**.

## Visa cellintervall som datamarkörer med Aspose.Cells

Aspose.Cells tillhandahåller metoden [**DataLabels.setShowCellRange()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#ShowCellRange) för att markera eller avmarkera kryssrutan **Markering innehåller - Värde från celler** enligt bilden ovan.

## Java-kod för att visa cellintervall som datamarkörer

Det här exempelkoden nedan att nå Datamarkörer för diagramserier och sedan ställa in [**DataLabels.setShowCellRange()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#ShowCellRange)-metod för att markera **Markering innehåller - Värde från celler** alternativet till true.
