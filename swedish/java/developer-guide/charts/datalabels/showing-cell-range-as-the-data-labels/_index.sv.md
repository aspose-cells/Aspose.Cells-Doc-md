## Visa cellintervall som dataetiketter i MS Excel

I Microsoft Excel 2013 kan du visa Cell intervall för dataetiketter. Du kan välja det här alternativet genom att följa dessa steg

- Välj Dataetiketter i serien och högerklicka för att öppna popup-menyn.
-  Klicka på**Formatera dataetiketter...** och det kommer att synas**Etikettalternativ**.
-  Markera eller avmarkera kryssrutan**Etikett innehåller - Värde från Cells**.
