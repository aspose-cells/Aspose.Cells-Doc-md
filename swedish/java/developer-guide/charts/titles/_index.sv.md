## **Ställa in titlar på diagram eller axlar**
Du kan använda Microsoft Excel för att ställa in titlarna på ett diagram och dess axlar i en WYSIWYG-miljö. Aspose.Cells tillåter också utvecklare att ställa in titlarna på ett diagram och dess axlar under körning. Alla diagram och deras axlar innehåller en[Titel](https://reference.aspose.com/cells/java/com.aspose.cells/Title) objekt som kan användas för att ställa in deras titlar som visas nedan i ett exempel.
