I Excel-diagram finns det 2 typer av titlar:
1. Diagramtitel 
1. Axeltitlar

## **Tillval för titlar**
Aspose.Cells låter också dig hantera titlar på diagram vid körning, med [Title](https://reference.aspose.com/cells/java/com.aspose.cells/Title)-objektet kan du ändra text, typsnitt och fyllningsformat för titlar.

|![todo:image_alt_text](chart_title.png)|

## **Inställning av diagram- eller axeltitlar**
Du kan använda Microsoft Excel för att ställa in titlar på ett diagram och dess axlar i en WYSIWYG-miljö. Aspose.Cells låter också utvecklare ställa in titlar på diagram och dess axlar vid körning. Alla diagram och deras axlar innehåller ett [Title](https://reference.aspose.com/cells/java/com.aspose.cells/Title)-objekt som kan användas för att ställa in deras titlar enligt exemplet nedan.
