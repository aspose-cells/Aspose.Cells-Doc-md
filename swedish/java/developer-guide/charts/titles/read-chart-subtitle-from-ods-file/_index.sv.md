## **Läs diagramunderrubrik från ODS-fil**

Aspose.Cells ger dig möjlighet att läsa diagramrubriker i ODS-filer genom att använda [**Chart.SubTitle**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#SubTitle)-egenskapen. Följande exempelkod laddar den [exempelfil ODS](SampleChart.ods) och läser diagramrubriken med hjälp av [**Chart.SubTitle**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#SubTitle)-egenskapen och skriver ut den i Konsolfönstret. Vänligen se konsoloutput av koden nedan för referens.
