## **Läs diagramunderrubrik från ODS fil**

Aspose.Cells ger dig möjlighet att läsa diagramundertexter i ODS-filer genom att använda[**Diagram.Undertitel**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#SubTitle)fast egendom. Följande exempelkod laddar[exempel ODS fil](SampleChart.ods)och läser diagrammets undertext med hjälp av[**Diagram.Undertitel**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#SubTitle)egenskapen och skriver ut den i konsolfönstret. Se konsolutgången för koden nedan för referens.
