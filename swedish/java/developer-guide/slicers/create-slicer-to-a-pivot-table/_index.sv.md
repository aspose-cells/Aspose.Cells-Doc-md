## **Möjliga användningsscenario**
Slicern används för att snabbt filtrera data. Den kan användas för att filtrera data både i en tabell eller pivottabell. Microsoft Excel tillåter dig att skapa slicern genom att välja en tabell eller pivottabell och sedan klicka på *Infoga > Slicer*. Aspose.Cells tillåter också att du skapar slicer med hjälp av {0} -metoden.
## **Skapa skärva till en pivot-tabell**
Var god se följande exempelkod. Den laddar in den [exempel-Excel-filen](67338498.xlsx) som innehåller pivottabellen. Den skapar sedan slicern baserat på den första baspivottabellfältet.
