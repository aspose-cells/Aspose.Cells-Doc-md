## **Möjliga användningsscenarier**
Om du vill ta bort slicer i Microsoft Excel, välj bara den och tryck på*Radera*knapp. På samma sätt, om du vill ta bort det med Aspose.Cells API programmatiskt, använd[Worksheet.getSlicers().remove()](https://reference.aspose.com/cells/java/com.aspose.cells/slicercollection#remove\(com.aspose.cells.Slicer\)) metod. Det kommer att ta bort skivaren från kalkylbladet.
