## **Möjliga användningsscenario**
Du kan formatera slicern i Microsoft Excel genom att ange dess antal kolumner eller genom att ange dess stil osv. Aspose.Cells tillåter dig också att göra detta med hjälp av egenskaperna [Slicer.NumberOfColumns](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#NumberOfColumns) och [Slicer.StyleType](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#StyleType).
## **Formatera skärva**
Vänligen se den följande koden, den laddar [provmappen](67338501.xlsx) som innehåller en slicer. Den går igenom slicern, ställer in dess antal kolumner och stiltyp och sparar den som [output Excel-fil](67338500.xlsx). Skärmbilden visar hur slicern ser ut efter körningen av provkoden.
