## **Rita tidslinjen medan du renderar Excel till PDF**
Om du har en Excel-fil som har en tidslinje tillämpad på den och du vill exportera Excel till PDF med tidslinjeinställningarna, stöder Aspose.Cells for Java det nu som standard. Du exporterar helt enkelt Excel-filen med tidslinje till PDF, den genererade PDF-filen visar tidslinjen som tillämpas.

 Följande exempelkod laddar[exempel på Excel-fil](input.xlsx) som innehåller en befintlig tidslinje. Den sparar sedan arbetsboken som[utdata PDF-fil](out.pdf). Följande skärmdump jämför källfilen i Excel och den genererade PDF-filen.
