## **Rita tidslinje vid rendering av Excel till PDF**
Om du har en Excel-fil som har en tidslinje tillämpad och du vill exportera Excel till PDF med tidslinjeinställningarna, stöder nu Aspose.Cells for Java det som standard. Du exporterar helt enkelt Excel-filen med tidslinje till PDF, den genererade PDFen kommer att visa tidslinjen som tillämpats.

Följande exempelkod laddar in den [exempel-Excel-filen](input.xlsx) som innehåller en befintlig tidslinje. Sedan sparar den arbetsboken som [utmatnings-PDF-filen](out.pdf). Följande skärmbild jämför käll-Excel-filen och den genererade PDF-filen.
