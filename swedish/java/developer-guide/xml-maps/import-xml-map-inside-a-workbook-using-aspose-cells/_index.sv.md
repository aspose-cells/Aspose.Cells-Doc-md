{{% alert color="primary" %}}

Aspose.Cells ger dig möjlighet att importera XML-karta i arbetsboken med hjälp av [**Workbook.importXml()**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#importXml-java.lang.String-java.lang.String-int-int-) metoden. Du kan importera XML-karta med Microsoft Excel med följande steg

- Välj **Utvecklare**-fliken
- Klicka på **Importera** i XML-avsnittet och följ de nödvändiga stegen.

Du behöver tillhandahålla din XML-data för att slutföra importen. Här är ett [prov-XML-data](5472501.txt) som du kan använda för testning.
