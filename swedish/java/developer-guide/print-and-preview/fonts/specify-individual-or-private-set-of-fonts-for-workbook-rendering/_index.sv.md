## **Möjliga användningsscenarier**

I allmänhet anger du typsnittskatalogen eller listan med teckensnitt för alla arbetsböcker, men ibland måste du ange individuella eller privata teckensnitt för dina arbetsböcker. Aspose.Cells tillhandahåller[**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs)klass som kan användas för att specificera den individuella eller privata uppsättningen teckensnitt för din arbetsbok.

## **Ange individuell eller privat uppsättning teckensnitt för arbetsbokrendering**

Följande exempelkod laddar[exempel på Excel-fil](67338304.xlsx)med sin individuella eller privata uppsättning teckensnitt som specificeras med hjälp av[**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs)klass. Vänligen se[exempel på teckensnitt](67338302.zip)används i koden såväl som i[utgång PDF](67338303.pdf)genereras av det. Följande skärmdump visar hur utgången PDF ser ut om teckensnittet hittas.
