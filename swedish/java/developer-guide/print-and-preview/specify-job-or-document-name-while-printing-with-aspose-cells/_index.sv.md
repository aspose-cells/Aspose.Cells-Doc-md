{{% alert color="primary" %}}

Du kan ange jobb- eller dokumentnamn vid utskrift av din arbetsbok eller kalkylblad med hjälp av objekten [**WorkbookRender**](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookRender) eller [**SheetRender**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender). Aspose.Cells tillhandahåller metoderna [**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String)) och [**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) som du kan använda för att ange jobbnamn vid utskrift av din arbetsbok eller kalkylblad.

{{% /alert %}}

## **Ange jobb- eller dokumentnamn vid utskrift med Aspose.Cells**

Exempelkoden laddar den angivna Excel-filen och skickar sedan den till skrivaren genom att ange jobb- eller dokumentnamn med hjälp av metoderna [**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String)) och [**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)). Skärmskottet visar hur jobbnamnet ser ut i skrivarkön.
