{{% alert color="primary" %}}

Du kan använda Aspose.Cells för att exportera ditt kalkylblad eller diagram till en bild med önskad bredd och höjd. Det ger[**ImageOrPrintOptions.setDesiredSize()**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#setDesiredSize(int,%20int)) för att ställa in önskad bredd och höjd på den exporterade bilden. Bredden och höjden anges i enheten pixlar.
