{{% alert color="primary" %}}

Du kan använda Aspose.Cells för att exportera din arbetsbok eller diagram till en bild med önskad bredd och höjd. Det finns en [**ImageOrPrintOptions.setDesiredSize()**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#setDesiredSize-int-int-)-metod för att ange den önskade bredden och höjden på den exporterade bilden. Bredden och höjden anges i pixelenheter.
