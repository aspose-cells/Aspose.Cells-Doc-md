 Aspose.Cells tillhandahåller[**Workbook.ContentTypeProperties.Add**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypepropertycollection#add(java.lang.Object)) metod för att lägga till anpassade ContentTypeProperties till en excel-fil. Du kan också göra egenskapen valfri genom att ställa in[**ContentTypeProperty.IsNillable**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypeproperty#IsNillable) egendom till**Sann**.
