Aspose.Cells tillhandahåller funktionen för att ersätta text i en arbetsbok med ett reguljärt uttryck. För detta tillhandahåller API[**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)egendom av[**Ersätt Alternativ**](https://reference.aspose.com/cells/java/com.aspose.cells/ReplaceOptions)klass. Ställa in[**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)till**Sann**indikerar att den sökta nyckeln kommer att vara ett reguljärt uttryck.

Följande kodavsnitt visar användningen av[**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)egendom genom att använda[exempel på excel-fil](SampleRegexReplace.xlsx). De[utdatafil](RegexReplace_out.xlsx)som genereras av följande kodavsnitt bifogas som referens.
