Aspose.Cells tillhandahåller funktionen att ersätta text i en arbetsbok med hjälp av ett reguljärt uttryck. För detta tillhandahåller API:et [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) egenskapen för [**ReplaceOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/ReplaceOptions) klassen. Genom att ställa [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) till **true** anger att den sökta nyckeln är ett reguljärt uttryck.

Den följande kodsnutten visar användningen av [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) egenskapen genom att använda den [exempel-excel-filen](SampleRegexReplace.xlsx). [Utmatningsfilen](RegexReplace_out.xlsx) som genereras av följande kodsnutt bifogas som referens.
