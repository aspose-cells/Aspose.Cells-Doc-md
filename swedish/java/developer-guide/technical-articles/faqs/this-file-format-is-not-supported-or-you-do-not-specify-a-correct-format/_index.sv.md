## **Det här filformatet stöds inte eller så anger du inte ett korrekt format**
Om användaren har angett filformatet när arbetsboken skapades från mallfil, beror detta fel vanligtvis på att det angivna filformatet inte är det faktiska filformatet för mallfilen. Om användaren inte har angett filformatet beror det vanligtvis på att filnamnstillägget inte representerar det faktiska filformatet för denna fil och filformatet inte kan upptäckas automatiskt, till exempel csv/tsv-filen som inte har några speciella identifierare .
