## **Detta filformat stöds inte eller så har du inte angett ett korrekt format**
Om användaren har angett filformatet när han skapade arbetsboken från mallfilen, är det vanligtvis ett fel eftersom det angivna filformatet inte är det verkliga filformatet för mallfilen. Om användaren inte har angett filformatet beror det vanligtvis på att filnamnstillägget inte representerar det verkliga filformatet för denna fil och filformatet inte kan detekteras automatiskt, som till exempel en csv/tsv-fil som inte har några speciella identifierare.
