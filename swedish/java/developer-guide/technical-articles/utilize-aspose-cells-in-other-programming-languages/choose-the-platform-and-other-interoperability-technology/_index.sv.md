{{% alert color="primary" %}} 

 Aspose.Cells är en kalkylbladsbearbetningskomponent som låter dig generera, läsa, ändra och skriva Excel®-kalkylblad utan att använda Microsoft Excel®.

- Aspose.Cells for Java - ett Java klassbibliotek och det är designat för att fungera i Java applikationer.

{{% /alert %}} 

 Även om det inte finns några speciella Aspose.Cells-versioner för andra plattformar, är det möjligt att använda Aspose.Cells på många andra programmeringsspråk, inklusive (men inte begränsat till):

|**Programmeringsspråk** |**Aspose.Cells att använda** |**Interoperabilitetsteknik** |
|:- |:- |:- |
| Perl|Java | Inline::Java|
| PHP|Java |Java Bro|
|Python |Java | JPype|
 Från tabellen ovan kan du se att för vissa programmeringsspråk är det möjligt att använda Aspose.Cells for Java. Följande tabell kan hjälpa dig att välja:

|**Scenario** |**När du ska använda** |
|:- |:- |
| Aspose.Cells for Java via Java Bridge eller Inline::Java|<p>**Fördelar**:</p><p>- Tillgänglig på alla plattformar där Java är tillgänglig.</p><p>- Lätt att anropa statiska metoder, konstruktörer med parametrar, överbelastade metoder och använda uppräknade värden.</p>|
||<p>**Nackdelar**: </p><p>- Har mindre frekventa utgåvor och färre funktioner än Aspose.Cells for .NET.</p>|
 Observera att Aspose.Cells for Java är designad for Java applikationer och att använda den i andra programmeringsspråk är föremål för vissa tekniska begränsningar eller lösningar.
