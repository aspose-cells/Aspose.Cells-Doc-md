## **Möjliga användningsscenario**

Aspose.Cells stöder det *DBNum*	anpassade mönsterformatet. Till exempel, om cellvärdet är 123 och du specificerar dess anpassade formatering som [DBNum2][$-804]Allmänt, då kommer det att visas som 壹佰贰拾叁. Du kan ange din anpassade formatering av cellen med metoderna [**Cell.getStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getStyle--) och [**Style.setCustom()**](https://reference.aspose.com/cells/java/com.aspose.cells/style/#setCustom-java.lang.String-).
