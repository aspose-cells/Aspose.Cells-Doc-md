Aspose.Cells tillhandahåller funktionalitet för att tillämpa överstruken (text ovanför baslinjen) och understreckande (text under baslinjen) effekter på text.

{{% /alert %}}

## **Arbeta med överstruken och understreck**

Applicera överstreckseffekten med hjälp av [**Font**](https://reference.aspose.com/cells/java/com.aspose.cells/Font) objektets [**setSuperscript**](https://reference.aspose.com/cells/java/com.aspose.cells/font#IsSuperscript) egenskap. För att applicera understreck, använd [**Font**](https://reference.aspose.com/cells/java/com.aspose.cells/Font) objektets [**setSubscript**](https://reference.aspose.com/cells/java/com.aspose.cells/font#IsSubscript) metod.
