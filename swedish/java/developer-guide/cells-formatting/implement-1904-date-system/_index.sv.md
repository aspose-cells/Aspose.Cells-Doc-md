{{% alert color="primary" %}} 

Microsoft Excel stöder två datumsystem: 1900-datumsystemet (standarddatumsystemet implementerat i Excel för Windows) och 1904-datumsystemet. Datumsystemet 1904 används normalt för att tillhandahålla kompatibilitet med Macintosh Excel-filer och är standardsystemet om du använder Excel för Macintosh. Du kan ställa in 1904-datumsystemet för Excel-filer med Aspose.Cells.

{{% /alert %}} 

Så här implementerar du 1904-datumsystemet i Microsoft Excel (till exempel Microsoft Excel 2003):

1.  Från**Verktyg** menyn, välj**alternativ** , och välj**Beräkning** flik.
1.  Välj**1904 års datumsystem** alternativ.
1.  Klick**OK**.

   **Väljer 1904 datumsystem i Microsoft Excel**
