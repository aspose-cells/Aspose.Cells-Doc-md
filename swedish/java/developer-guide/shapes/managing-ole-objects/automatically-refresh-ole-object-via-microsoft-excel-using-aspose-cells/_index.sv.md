{{% alert color="primary" %}} 

 Aspose.Cells tillhandahåller[OleObject.AutoLoad](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad) egenskap för att uppdatera OLE-objektet när excel-filen öppnas i Microsoft Excel. På grund av den här egenskapen kommer OLE-objektet att visa den korrekta OLE-bilden genererad av Microsoft Excel.

{{% /alert %}} 
## **Uppdatera OLE-objekt automatiskt via Microsoft Excel med Aspose.Cells**
 Följande exempelkod laddar[exempel på excel-fil](5473423.xlsx) som har en icke-äkta OLE-bild. OLE-objektet är egentligen ett Microsoft Word-dokument men exemplet i Excel-filen visar djurbilden istället för Microsoft Word-bilden. Men om du öppnar[output excel-fil](5473429.xlsx), kommer du att se Microsoft Excel visar rätt OLE-bild.
