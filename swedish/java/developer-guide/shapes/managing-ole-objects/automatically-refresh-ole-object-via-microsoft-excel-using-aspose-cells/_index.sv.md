{{% alert color="primary" %}} 

Aspose.Cells tillhandahåller egenskapen [OleObject.AutoLoad](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad) för att uppdatera OLE-objektet när Excelfilen öppnas i Microsoft Excel. På grund av denna egenskap kommer OLE-objektet att visa rätt OLE-bild som genererats av Microsoft Excel.

{{% /alert %}} 
## **Automatisk uppdatering av OLE-objekt via Microsoft Excel med Aspose.Cells**
Nedanstående exempelkod laddar [exempel excelfilen](5473423.xlsx) som har en icke-reell OLE-bild. OLE-objektet är faktiskt ett Microsoft Word-dokument men exemplet excelfilen visar djurbilden istället för Microsoft Word-bilden. Men om du öppnar [utdata excelfilen](5473429.xlsx), kommer du att se att Microsoft Excel visar rätt OLE-bild.
