## **Möjliga användningsscenario**
Aspose.Cells tillhandahåller egenskapen [Shape.ThreeDFormat](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ThreeDFormat) för att arbeta med 3D-formatet i formen eller diagrammet. Det innehåller olika underegenskaper som du kan ställa in för att uppnå olika resultat enligt dina behov.

Följande skärmbild visar Microsoft Excels gränssnitt för att ställa in 3D-formatet för formen.
