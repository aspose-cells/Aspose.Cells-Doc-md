## **Möjliga användningsscenarier**
 Aspose.Cells tillhandahåller[Shape.ThreeDFormat](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ThreeDFormat)egenskap för att arbeta med 3D-formatet för form eller diagram. Den innehåller olika underegenskaper som du kan ställa in för att uppnå olika resultat enligt dina krav.

Följande skärmdump visar Microsoft Excel-gränssnittet för att ställa in 3D-formatet för form.
