Aspose.Cells tillhandahåller rika funktioner för att hantera former i kalkylarket. Ibland behövs anslutningspunkterna för en form för att justera eller placera formerna på rätt plats. För detta ändamål krävs alla anslutningspunkter. Följande kod kan användas för att få en lista över anslutningspunkter för en form där funktionen [**Shape.GetConnectionPoints()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ConnectionPoints) används.
