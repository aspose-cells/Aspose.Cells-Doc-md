 Aspose.Cells tillhandahåller omfattande funktioner för att hantera former i kalkylarket. Ibland finns det ett behov av att få anslutningspunkterna för en form för att rikta in eller placera formerna på lämplig plats. För detta ändamål krävs alla anslutningspunkter. Följande kod kan användas för att få listan över anslutningspunkter för en form var[**Shape.GetConnectionPoints()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ConnectionPoints) funktionen används.
