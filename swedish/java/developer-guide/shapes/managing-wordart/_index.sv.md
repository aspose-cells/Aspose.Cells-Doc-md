{{% alert color="primary" %}}

Använd WordArt för att lägga till speciella texteffekter i kalkylblad. Till exempel, sträck ut en titel över filens överkant, dekorera text och få texten att passa en förinställd form, eller använd texten på ett Excel-ark som en bakgrundsvattenstämpel. WordArt blir ett objekt som du kan flytta eller placera i kalkylblad för att lägga till dekoration.

{{% /alert %}}
