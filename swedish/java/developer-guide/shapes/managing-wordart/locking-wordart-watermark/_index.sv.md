Aspose.Cells API:er gör det möjligt att lägga till WordArt-vattenstämplar på kalkylbladet på ett sätt så att WordArt blir ett objekt som kan flyttas och placeras på kalkylbladet. Det är också möjligt att låsa WordArt-objektet för alla interaktioner som redigering, rörelse och markering. Den här artikeln förklarar användningen av[**Shape.setLockedProperty**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#setLockedProperty(int,%20boolean)) metod för att låsa några aspekter av vattenstämpeln.

{{% /alert %}}

## **Låsning av WordArt vattenstämpel**

Aspose.Cells API:er tillåter låsning av vissa aspekter av vattenstämpeln så att användarinteraktionen kan begränsas eller helt blockeras. Följande kodavsnitt visar användningen av Aspose.Cells för Java API för att skapa ett vattenmärke för varje kalkylblad i det laddade kalkylarket och låsa val, förflyttning, redigering och ändra storlek på vattenstämpeln.
