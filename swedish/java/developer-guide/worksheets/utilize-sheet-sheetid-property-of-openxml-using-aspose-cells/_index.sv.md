## **Möjliga användningsscenario**

*Sheet.SheetId*egenskapen finns inuti *DocumentFormat.OpenXml.Spreadsheet*namespace och är en del av OpenXml. Du kan se denna egenskap och dess värde inuti *workbook.xml* som visas i följande skärmdump. Aspose.Cells tillhandahåller motsvarande egenskap som [**Worksheet.TabId**](https://reference.aspose.com/cells/java/com.aspose.cells/worksheet#TabId).
