Aspose.Cells låter användaren kontrollera om VBA-kodprojektet är signerat eller inte. Använd [**Workbook.getVbaProject().isSigned()**](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#IsSigned) metoden för att kontrollera om VBA-kodprojektet är signerat eller inte.
