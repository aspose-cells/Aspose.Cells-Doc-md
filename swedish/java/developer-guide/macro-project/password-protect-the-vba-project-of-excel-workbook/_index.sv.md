## **Möjliga användningsscenario**
Du kan skydda VBA (Visual Basic for Applications) -projektet för arbetsbok med Aspose.Cells genom att använda metoden [VbaProject.protect()](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#protect\(boolean,%20java.lang.String\)).
## **Exempelkod**
Följande exempelkod laddar den [prov Excel-fil](43352069.xlsm), har åtkomst till dess VBA-projekt och skyddar det med ett lösenord. Slutligen sparar den den som en [utdata Excel-fil](43352070.xlsm).
