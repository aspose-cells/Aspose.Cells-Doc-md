## **Möjliga användningsscenarier**

Du kan ändra*Språk*av Excel-filen genom att högerklicka på filen och sedan välja*Egenskaper > Detaljer*och sedan redigera*Språk*fält. Snälla använd[**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#Language)egenskap för att ändra den programmatiskt med Aspose.Cells API:er.
