## **Möjliga användningsscenario**

Du kan ändra *Språk* för Excel-filen genom att högerklicka på filen och sedan välja *Egenskaper > Detaljer* och sedan redigera fältet *Språk*. Använd [**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#Language)-egenskapen för att ändra det programmatiskt med hjälp av Aspose.Cells API:er.
