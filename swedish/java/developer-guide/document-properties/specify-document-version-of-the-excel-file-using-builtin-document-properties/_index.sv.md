## **Möjliga användningsscenarier**

Du kan ändra*Versionsnummer*av Excel-filen genom att högerklicka på filen och sedan välja*Egenskaper > Detaljer*och sedan redigera*Versionsnummer*fält. Snälla använd[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#DocumentVersion)egenskap för att ändra den programmatiskt med Aspose.Cells API:er.

Följande exempelkod skapar en arbetsbok och ändrar dess inbyggda dokumentegenskaper som inkluderar*Titel*, *Författare* och*Versionsnummer*.
