{{% alert color="primary" %}} 

Aspose.Cells for Java är en klassbibliotek som gör det möjligt för dina program att utföra en stor mängd kalkylbladsbehandling. Aspose.Cells for Java stöder de populära kalkylbladsfilformaten (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) och tillåter export av Excel-filer till PDF, XPS, HTML, MHTML, oformaterad text, TIFF, JPG, PNG, BMP och SVG.

Den senaste API-referensen finns på [Aspose.Cells for Java API-referens](https://reference.aspose.com/cells/java).

{{% /alert %}}
