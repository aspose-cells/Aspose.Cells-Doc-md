Aspose.Cells för .NET är ett kalkylbladsprogrammeringsbibliotek som låter mjukvaruutvecklare manipulera och konvertera kalkylbladsfiler från sina egna applikationer. Aspose.Cells för .NET stöder de populära kalkylarket (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) filformat och tillåter export av Excel-filer till PDF, XPS, HTML, MHTML, vanlig text, TIFF, JPG, PNG, BMP och SVG.

 Den senaste API-referensen finns på[Aspose.Cells för .NET API-referens](https://reference.aspose.com/cells/net).
