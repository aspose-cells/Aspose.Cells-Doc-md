Aspose.Cells for .NET är ett kalkylbladsprogrammeringsbibliotek som tillåter programutvecklare att manipulera och konvertera kalkylarksfiler från sina egna applikationer. Aspose.Cells for .NET stöder de populära kalkylarket (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) filformat och tillåter export av Excel-filer till PDF, XPS, HTML, MHTML, vanlig text, TIFF, JPG, PNG, BMP och SVG.

Den senaste API Referensen finns på[Aspose.Cells for .NET API Referens](https://reference.aspose.com/cells/net).
