{{% alert color="primary" %}} 

Aspose.Cells for .NET är ett kalkylbladsprogrammeringsbibliotek som tillåter programutvecklare att manipulera och konvertera kalkylarksfiler från sina egna applikationer. Aspose.Cells for .NET stöder de populära kalkylarksfilformaten (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) och tillåter export av Excel-filer till PDF, XPS, HTML, MHTML, oformaterad text, 619, 4013, 7013, 7013, 7013, 8, 8 761113481 och SVG.

 Den senaste API Referensen finns på[Aspose.Cells for .NET API Referens](https://reference.aspose.com/cells/net).
