{{% alert color="primary" %}} 

Aspose.Cells for .NET är ett kalkylbladsprogrammeringsbibliotek som gör att programutvecklare kan manipulera och konvertera kalkylbladsfiler inom sina egna applikationer. Aspose.Cells for .NET stödjer de populära kalkylbladsfilformaten (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) och möjliggör export av excelfiler till PDF, XPS, HTML, MHTML, ren text, TIFF, JPG, PNG, BMP och SVG.

Den senaste API-referensen finns på [Aspose.Cells for .NET API-referens](https://reference.aspose.com/cells/net).

{{% /alert %}}
