 Den här sidan innehåller release notes för[Aspose.Cells för .Net V4.7.0 Ny utgåva](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-v4.7.0-new-release/)

{{% /alert %}}

Vi är glada att meddela Aspose.Cells for .NET 4.7.0!

Vad har ändrats:

- Stöder direktkonvertering av Excel-filer till PDF.
 Stöder Gridline-alternativet för sidinställningarna vid utskrift av kalkylblad.
 Stöder för att infoga en WAV-fil som Ole-objekt.
Stöder R/W kryssrutor i Excel 2007-filer.
 Stöder att använda anonyma typer eller anpassade objekt som datakälla i Smart markör
 Stöder för att ställa in displayen nollvärde alternativ.
 Stöder för att förena och skära två områden.
 Stöder bevarande av data efter att ha sparat arbetsboken.
 Stöder alternativet att infoga tomma rader i pivottabellen eller inte.
 Förbättrar beräkningsmotorn och stöder fler funktioner.
 45 korrigeringar och förbättringar.
