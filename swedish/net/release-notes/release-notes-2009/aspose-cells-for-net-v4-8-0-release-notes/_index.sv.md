 Den här sidan innehåller release notes för[Aspose.Cells för .NET v4.8.0](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-v4.8.0/)

{{% /alert %}} 

 Vi är glada att kunna meddela Aspose.Cells för .NET v4.8.0.0!

 Vad har ändrats:

 l Slå ihop svit Aspose.Grid till Aspose.Cells

 l Optimera prestandan för formelomräkningar

 l Förbättra prestandan för att infoga/ta bort rader/kolumner

 l Förbättra prestandan för funktionen för automatisk anpassning av rader/kolumner

 l Lägg till WordArt-former i XLS-filen

 l Läs Gradient Fill-effekter för diagrammet

 Skapa kalkylblad med attribut för första sidan eller jämn/udda sidhuvud/sidfot för sidinställningar

 l Stöd för flera utskriftsområden i kalkylblad-till-bild-funktionen

 l Lägg till stöd för formelberäkning för några nya inbyggda MS Excel-funktioner

 l Ställ in bakgrundsbild för ett arbetsblad av diagramtyp

 l Tillhandahålla stöd för att ställa in bildalternativ i diagram-till-bild- och kalkylblad-till-bild-funktionerna

 l 101 korrigeringar och andra förbättringar
