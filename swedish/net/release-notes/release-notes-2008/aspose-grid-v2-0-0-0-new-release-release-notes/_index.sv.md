 Den här sidan innehåller release notes för[Aspose.Grid V2.0.0.0 Ny utgåva](https://downloads.aspose.com/cells/net/new-releases/aspose.grid-v2.0.0.0-new-release/)

{{% /alert %}} 

 Kära kunder, Vi har precis släppt Aspose.Grid 2.0.0!

 Vad är nytt?
 Aspose.Grid.Web1. GridWeb är nu kompilerad under både 1.0 och 2.0 ramverk.2. Stöder fullt ut .NET 2.0 framework.3. Stöder .NET 2.0 MEDIUM TRUST LEVEL.4. Stöder .NET 2.0 DataSourceControl Binding.5. Ny klient-API-metod resizeColumnToFit.6. Stöder Windows Vista.

Aspose.Grid.Skrivbord1. Ny egenskap GridCell.Protected, indikerar om cellen är skyddad. Om värdet är "true" kan användaren inte ändra cellen via användargränssnittet.2. Ny egenskap ComboBox.IsInputable, indikerar om användaren kan mata in text i kombinationsrutan. Den inmatade texten kommer att sparas som cellens värde.3. Ny händelse Desktop.CommentDataChanged, inträffar när kommentarsdata har ändrats..4. Ny egenskap Validation.CustomMsgForStopEdit, gör att användaren får eller ställer in det anpassade meddelandet att visa när användaren anger ogiltigt värde.5. Ny metod Worksheet.SetFirstVisibleColumn()/SetFirstVisibleRow() gör det möjligt för användaren att ställa in det första synliga kolumn-/radindexet i arkvyn.6. Ny metod Worksheet.GetFirstVisiableColumn()/GetFirstVisiableRow() gör det möjligt för användaren att få det första synliga kolumn-/radindexet i arkvyn.7. Ny metod Arbetsblad. AddSelectedRange() gör det möjligt för användaren att lägga till ett nytt valt område till kalkylbladet...8. Ny egenskap Worksheet.ShowGridlines, gör det möjligt för användaren att visa/dölja rutnätslinjerna.

 Vad är förbättrat?
Aspose.Grid.Web1. Förbättrat XHTML-stöd.2. Förbättrad sortering. Null Cells kommer alltid att stanna på baksidan.3. Förbättrad kopiera/klistra in.

 Vad är fixat?
 Aspose.Grid.Web1. Fixat buggen med AutoFilter-knappar kommer inte att flyttas om efter AutoAnpassa en kolumn.2. Fixat en bugg i Excel Exporting.3. Fixade en bugg av Kan inte hämta objektet Cell av en kommandotypshyperlänk i XhtmlMode. 4. Fixat buggen med AJAX automatisk uppdatering fungerar inte i en form som använder Master Page.

Aspose.Grid.Skrivbord1. Fixat en bugg av kommentar display.2. Fixat ett inmatningsproblem: I den tidigare versionen, i något speciellt tillstånd, trycker du på enter-tangenten i en redigeringscell för att rensa cellens innehåll.
