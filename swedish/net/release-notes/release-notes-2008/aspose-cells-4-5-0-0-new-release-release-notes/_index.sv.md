 Den här sidan innehåller release notes för[Aspose.Cells 4.5.0.0 Ny utgåva](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-4.5.0.0-new-release/)

{{% /alert %}} 

 Vad är nytt?



- ` `Konvertera arbetsblad till bild.
- ` `Stöd för att konvertera Excel-fil till HTML filformat.
- ` `Stöd för att sortera data från vänster till höger.



 Vad är förbättrat?



- ` `Stödde fler funktioner i formelberäkningsmotorn
- ` `Stödde fler anpassade filter i autofilter.
- ` `Förbättrad Excel2007 Xlsx-stöd
- ` `Förbättrad SpreadsheetML-support
- ` `Förbättrad xls2Pdf-konvertering
- ` `Enhanced Charts2Image-konvertering.
- ` `Förbättrad smart markör
- ` `Stöds för att hämta länkad cell av radioknapp från mallfil.



 Vad är fixat?



- ` `Bug i läs- och skrivdiagram.
- ` `Bug i beräkningsformler
- ` `Bug i att öppna och spara några speciella xls-filer.
- ` `Bugg in Cells.ImportDataTable-metod
- ` `Bug i uppdatering av diagramreferenser vid infogning av rader/kolumner.
