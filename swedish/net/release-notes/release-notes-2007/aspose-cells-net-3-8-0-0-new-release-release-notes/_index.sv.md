 Kära kunder,

 Vi har släppt Aspose.Cells for .NET v3.8!

- Stöder för att skapa kryssruta vid körning
- Stöder för att skapa OLE-objekt under körning
- Stöder för att skydda/avskydda arbetsböcker
- Tillåter att ändra externa länkars datakälla
- Förbättrad smart markörbearbetning
- Förbättrad formelberäkningsmotor
- Fixade buggar i DeleteRows och InsertRows
- Fixade en bugg med att upprepade gånger öppna och spara en fil med utvärderingsversion
- Fixade AddCopy-bugg när du kopierade några formler
- Andra förbättringar och korrigeringar sedan v3.7
