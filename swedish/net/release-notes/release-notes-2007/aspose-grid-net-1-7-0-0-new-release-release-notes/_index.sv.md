Kära kunder, Vi har precis släppt Aspose.Grid 1.7!Denna version inkluderar många nya spännande funktioner:Aspose.Grid.Web 1.71.New METAL-LIGHT Look & Feel.2.Added Fully DataBinding Support. Nu stöder den bindning av ark till DataSet, DataTable, DataView eller andra samlingsobjekt. Kontrollen laddar data från datakällan och uppdaterar datakällan automatiskt. 3. Ny "Worksheets Designer" har lagts till i vs.net IDE. Du kan använda den för att skapa kraftfulla databasbindningsapplikationer med några få musklick och några rader kod.4.Ladda nya valideringstyper: FreeList, Number, Integer, Date, DateTime och CustomFunction.5.Förbättrad visuell effekt och användbarhet i None-IE browsers.6.Lade till 6 nya färgegenskaper för att du ska kunna justera ActiveCell/SelectCell/SelectHeaders Color/BackgroundColor.7.Lade till ny egenskap "Visible" till WebWorksheet. Du kan använda den för att dölja ett kalkylblad från TabBar.8.Added Custom Command Buttons Support. Du kan skapa dina egna knappar i kontrollens TabBar.9.Nu kan du använda RegularExpression Validation med en av andra valideringar samtidigt.10.Added WebWorksheet.Copy(WebWorksheet) metod, vilket gör att du kan kopiera ett kalkylblads innehåll till ett annat kalkylblad även om de är i två olika GridWeb control.11.Added Client-Side Submit/Validation event Support. Du kan skriva ditt eget klientskript för att hantera inlämnings- eller valideringsfelhändelsen på klientsidan. 12. Tillagda formelfunktioner "MIN", "MAX". 13. Fixade buggen med förlorade datatyper vid import från ett DataView-objekt.14. Fixade några problem i VS.NET 2005.15.Optimerad prestanda när det finns ett stort antal valideringar i ett ark.
