 Den här sidan innehåller release notes för[Aspose.Grid .Net 1.7.2.0 Hot Fix](https://downloads.aspose.com/cells/net/new-releases/aspose.grid-.net-1.7.2.0-hot-fix/)

Kära kunder, Vi har släppt Aspose.Grid 1.7.2! Fixar och förbättringar i denna version: Aspose.Grid.Web1. Lade till funktionen "Hierarkisk vy" i databindningsläge.2. Fixade några formateringsproblem när du körde med inga engelska språkinställningar.3. Lade till egenskapen GridWeb.DisplayCellTip för att låta dig aktivera/inaktivera standardverktygstipsen för celler.4. Lade till GridWeb.NoScroll property.5. Förbättrad prestanda för att lägga till/ta bort rader/kolumner.6. Skapa hyperlänkar i kolumnrubriker supported.7. Skapa cell-kommando i bindande kolumner supported.8. Stöder klickning och sortering av kolumnrubriker.

Aspose.Grid.Skrivbord1. "Klicka på startcell och skift och/eller ctrl klicka på slutcell" stöds.2. SelectedCellRangeChange-händelsen stöds.3. Fixat den vanliga stilfonten som inte stöds.4. SetStyle med bitmask stödd.5. Formelstil "$A$1" stöds.6. Fixade de formelstilar som inte stöds i ImportExcelFile.7. Fixat teckensnitt och teckensnittsfärg bug.8. Dra rullning med mus stöds.9. "Vertikal text" stöds.10. Fixade buggen med att gruppera samma intervall mer än två gånger.
