 Den här sidan innehåller release notes för[Aspose.Grid .Net 1.5.3.0 Hot Fix](https://downloads.aspose.com/cells/net/new-releases/aspose.grid-.net-1.5.3.0-hot-fix/)

 Vi har precis släppt Aspose.Grid 1.5.3!

 Förbättring och buggfix i denna utgåva:

 Aspose.Grid.Skrivbord

 Förbättrad:1.Kolumnvalidering stöds.2.Kolumnkontroll stöds.3.R1C1-referensstil stöds.4.Sortering stöds.

 Aspose.Grid.Webb

 Fixade buggar: 1. När antalet rader är större än cirka 100 försvann ibland FreezePanes (Fixed Rows/Columns). kan tas bort. 4. Kolumnens bredd är inte korrekt vid vissa regioninställningar.

Förbättrad: 1.Support inställning av anpassade kolumn-/radrubriktexter (WebCells.SetRowCaption, WebCells.SetColumnCaption).2.Stöd sortering av en rad celler, både uppifrån och ned och från vänster till höger (WebCells.Sort).3.Support export celler data till ett DataTable-objekt (WebCells.Export).4.Aspose.Grid.Web UI Designer. Körs från systemstartmenyn eller Visual Studio-miljön (endast stöd för .net 1.0/1.1 överlägset).
