 Den här sidan innehåller release notes för[Aspose.Cells .Net 4.1.0.0 Ny version](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-.net-4.1.0.0-new-release/)

 Kära kunder,

 Vi har släppt Aspose.Cells för .NET v4.1!

- Tillhandahållit API:er för att skapa villkorliga formateringar från början
- Tillhandahöll API:er för att skapa pivottabeller från början
- Lade till LinkedCell-egenskap för CheckBox-klassen
- Lade till nya C# snabbstartsdemos
- Fixade 2 mindre buggar i inställningsformler
- Fixade en bugg i beräkningsformler
- Fixade problemet med att använda Aspose.Cells på Windows Vista RC2. Detta problem orsakas av obfucator.
- Fixade ett sidorienteringsfel vid konvertering till pdf-fil
- Fixat ett fel med kopiering av rader med delade formler
