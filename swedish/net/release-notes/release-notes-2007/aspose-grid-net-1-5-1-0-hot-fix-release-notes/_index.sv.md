 Kära kunder,

 Vi har släppt Aspose.Grid.Web v1.5.1!

 I den här nya korrigeringen gör vi följande förbättringar och korrigeringar:

 Aspose.Grid.Skrivbord: Optimera prestanda och minnesanvändning. Stödrad, kolumnstilar Fixat: Bugg för rullningslistvärde vid frysning av rader eller kolumner.

Aspose.Grid.Webb:Ändra kontrollens klassnamn från "ExcelWeb" till "GridWeb". Stöd för att dölja rubrikfältStöd för att dölja rullningsfältSupport relativ sökväg för klientsidans användningsfiler. Stöd för cell/tabellbakgrundsbildstil. Stöd spara/ladda Aspose definierad binär fil( AGWF-Aspose Grid Web Format). Lade till ny egenskap "AGWClientPath", du kan använda den här egenskapen för att ange sökvägen för klientsidans användningsfiler (skript, bilder, etc.). Fixat färgbugg vid export till Excel-filer. Fixat standardteckensnittsbugg .
