 Den här sidan innehåller release notes för[Aspose.Cells .Net 4.2.0.0 Ny version](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-.net-4.2.0.0-new-release/)

 Kära kunder:

 Vi har släppt Aspose.Cells för .NET v4.2!

 den här nya versionen läggs några viktiga nya funktioner till:

- Nu stöder Aspose.Cells helt att manipulera diagram, bilder och andra ritobjekt i mallfiler
- Chart2Image-funktionen stöds. För närvarande stöds endast kolumn-, linje-, cirkel- och stapeldiagram. Och endast 2D-diagram stöds. Vi kommer att stödja fler diagramtyper i framtida versioner.
- Xls2Pdf-konvertering är förbättrad och optimerad.
- Formelberäkningsmotorn är förbättrad och optimerad.
