 Vi släpper gärna Aspose.Cells for .NET v5.0.0!

 Vad har ändrats i Aspose.Cells:

- Inkluderar stöd för MS Excel 2007 XLSB-filformat.
 Ger specialfunktion för kopiera och klistra in.
 Stöder sortering av smarta markörer.
37 förbättringar och korrigeringar.

 Vad har ändrats i Aspose.Cells.GridWeb:

- Stöder Hiding Context Menu.
 3 förbättringar och korrigeringar.



 Vad har ändrats i Aspose.Cells.GridDesktop:

- Stöder kopiering av ComboBox och tillhandahåller en händelsehanterare för att jämföra objekt och värden för de två ComboBoxarna.
 Inkluderar textrotationsfunktion för celldata.
 Hämtar cellposition genom klientens koordinater/punkter.

 Problem lösta i Aspose.Cells for .NET v5.0.0.







 Anmärkningsvärda förändringar för användarna:

 Samtidigt som vi försöker hålla API så enkel och tydlig som möjligt, har vi beslutat att erkänna och respektera plattformens gemensamma utvecklingsmetoder; vi har omarrangerat API Structure/ Namespaces.

Med den här versionen, dvs. Aspose.Cells for .NET v5.0.0, har vi omorganiserat API-klasserna för komponenten Aspose.Cells. Denna förändring har några viktiga aspekter som vi följer. Vi har lagt till nya namnutrymmen. Hela API (klasser, gränssnitt, uppräkningar, strukturer etc.) fanns tidigare i namnområdet Aspose.Cells. Nu har vissa uppsättningar av API flyttats till sina relativa namnområden, vilket gör förhållandet mellan klasser (med deras medlemmar) och namnutrymmen tydligt och förenklat.

 Alla klasser i Aspose.Cells-biblioteket är uppdelade i lämpliga namnområden för enkel åtkomst. Namnutrymmet Aspose.Cells är naturligtvis rotnamnutrymmet som tillhandahåller klasser för att generera, konvertera, modifiera, rendera och skriva ut Microsoft Excel-dokument utan att använda Microsoft Excel. Komponenten Aspose.Cells har följande namnrymder:

- Aspose.Cells
 Aspose.Cells. Diagram
 Aspose.Cells.Teckning
 Aspose.Cells.Rendering
 Aspose.Cells. Pivot
 Aspose.Cells.Tabell
 Aspose.Cells.Egenskaper
