\1) Aspose.Cells 

 Förbättringar

- Flera delsummor lägger till extra rad

- Text omgiven av citattecken i filen CSV

- WorkSheetCollection.AddCopy uppdaterar inte referenser
- Problem när du tar bort datakällan i pivottabellen
- Problem med kopiering av pivottabell (formatering)
- Att spara med Excel97-2003-formatet förstör filen
- Problem med att rensa partiellt innehåll i en tabell
- Det gick inte att öppna filen som skapats av OXML SDK
