## **Introduktion**
Aspose .NET Database Data Exporter till Excel för Umbraco-modulen gör att användare kan exportera data direkt från lokala eller fjärrdatabastabeller, vyer och med anpassad fråga till Microsoft Excel eller OpenOffice Spreadsheet. Denna modul demonstrerar den kraftfulla kalkylbladsbyggnadsfunktionen som tillhandahålls av Aspose.Cells. Denna inledande version av modulen är berikad med följande coola funktioner för att göra exportprocessen enkel och lätt att använda.
### **Modulfunktioner**
Denna första version av tillägget har följande funktioner:

- Tillåt att ansluta Lokal MS SQL Server-databas
- Tillåt att ansluta Fjärr MS SQL Server-databas
- Fyll i alla tabeller från ansluten databas
- Fyll i alla vyer från ansluten databas
- Tillåt att skriva anpassad fråga
- Auto Fit kolumner till innehållslängd.
- Tillåt att hoppa över sträng mer än 32k i excelceller (LoadOptions)
- Applicera Header Column formating som Fet Text
- Tillåt att använda som datakälla (tabell, vyer, anpassad fråga)
- Exportera data till Microsoft Excel-dokument (.xls, .xlsx och .xlsb)
- Exportera data till tabbavgränsat textdokument (*.txt)
- Exportera data till CSV (kommaseparerad) (*.csv)
- Exportera data till OpenDocument Spreadsheet (*.ods)
- Alternativ för att välja önskat utdataformat innan exportering.
- Exporterat dokument skickas automatiskt till webbläsaren för nedladdning.

## **Systemkrav och stödda plattformar**
### **Systemkrav**
För att installera Aspose .NET Database Data Exporter till Excel för Umbraco-modulen måste du uppfylla följande krav:

- Umbraco 6.2.5 & Umbraco 6 versioner
- Umbraco med MS SQL Server
- Microsoft .Net Framework 4.0

**Observera:** Umbraco 7 och högre stöds inte i denna version. Vi ser fram emot att höra dina åsikter och lägga till stöd för Umbraco 7 i nästa version.
### **Stödda plattformar**
Modulen stöds på alla versioner av

- Umbraco 6.0 som körs på ASP.NET 4.0

## **Installation**
När du har laddat ner, följ dessa steg för att installera detta paket på din Umbraco-webbplats:

1. Logga in på Umbraco **Utvecklare** avsnittet, till exempel `http://www.myblog.com/umbraco`
1. I trädet, expandera **Paket**-mappen.
1. Här finns två sätt att installera ett paket: välj **Installera lokalt paket** eller bläddra i **Umbraco Package Respository**.
1. Om du installerar ett **lokalt paket**, packa inte upp paketet utan ladda in zip-filen i Umbraco.
1. Följ instruktionerna på skärmen.

**Observera:** Du kan få ett fel med ‘Maximalt förfrågningslängdsöverskridande’ när du installerar. Du kan enkelt lösa detta genom att uppdatera värdet ‘maxRequestLength’ i din Umbraco web.config-fil.
<httpRuntime requestValidationMode="2.0" enableVersionHeader="false" maxRequestLength="25000" />
## **Användning**
När du har installerat Aspose .NET Database Data Exporter till Excel för Umbraco-modulen är det väldigt enkelt att börja använda den på din hemsida. Följ dessa enkla steg för att komma igång

1. Se till att du är inloggad i Umbraco **Utvecklare** avsnittet, till exempel `http://www.myblog.com/umbraco/`
1. Klicka på **Inställningar** i listan över avsnitt längst ner till vänster på skärmen.
1. Expandera **Mallar** noden och välj den mall du vill lägga till, till exempel Textsida.
1. Välj positionen i den valda mallen där du vill exportera knappen att läggas till. Vanligtvis vill du lägga till den längst upp till höger på sidan eller längst ner på sidan.
1. Klicka på **Infoga Makro** på den övre menyfliken.
1. Från **Välj ett makro** (Aspose .NET-databasdataskapare till Excel för Umbraco), välj det nyligen installerade Aspose .NET-databasdataskapare till Excel för Umbraco-makrot och klicka på **OK**.

Vänligen kontrollera skärmdumpen nedan för detaljer. 

![todo:image_alt_text](umbraco-database-data-exporter-to-excel_2)

Du har framgångsrikt lagt till Aspose .NET-databasdataskapare till Excel-modulen på din sida.

![todo:image_alt_text](umbraco-database-data-exporter-to-excel_1)

1. Ange eller använd förifylld MS SQL-serveranslutningssträng
1. Välj Datakälla Typ (Tabell, Vy, Anpassad fråga)
1. Välj eller ange Datakälla (Tabell, Vy, Anpassad fråga)
1. Välj exporttyp
1. Klicka på Exportera Data
1. Önskad fil kommer att laddas ner automatiskt.
