## **Introduktion**
Aspose Update Import Tool är ett tillägg med öppen källkod som ska användas i Microsoft Dynamic CRM. Detta tillägg kan automatiskt uppdatera och/eller exportera data från CRM till Aspose.Cells Kalkylblad. Vi behöver inte göra något manuellt och installera detta tillägg och göra vårt arbete enklare och snabbare.

Detta tillägg ger funktionalitet för att automatiskt uppdatera och/eller exportera från CRM. Ett klick importerar/exporterar posten till/från CRM.

Huvudfunktionerna i detta tillägg är:

- Upprätta anslutningen till önskat CRM.
- Tillåt användaren att välja önskad enhet och fält.
- Hämta posten från CRM-visning i rutnät.
- Exportera posten till Aspose.Cells kalkylblad.
- Hämta posten från Aspose.Cells Kalkylbladsvisning i rutnät.
- Exportera posten från grid till CRM.
## **Systemkrav och plattformar som stöds**
### **Systemkrav**
För att installera och använda Aspose .Net Update Import Tool för Microsoft Dynamics CRM måste du ha en av följande CRM-versioner installerad

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.

Kontakta oss gärna om du hittar några problem med att installera eller använda detta tillägg.
### **Plattformar som stöds**
Detta tillägg kommer att fungera med alla versioner av Microsoft Dynamics CRM:

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.
