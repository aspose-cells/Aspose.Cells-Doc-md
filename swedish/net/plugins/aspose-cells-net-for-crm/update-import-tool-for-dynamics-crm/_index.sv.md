## **Introduktion**
Aspose Update Import Tool är ett öppen källkods-tillägg som ska användas i Microsoft Dynamic CRM. Detta tillägg är kapabelt att automatiskt uppdatera och/eller exportera data från CRM till Aspose.Cells kalkylblad. Vi behöver inte göra något manuellt och installera detta tillägg och göra vårt arbete enklare och snabbare.

Detta tillägg ger funktionalitet för automatisk uppdatering och/eller export från CRM. Ett klick kommer att importera/exportera posten till/från CRM.

De största funktionerna i detta tillägg är:

- Upprätta anslutningen till önskad CRM.
- Tillåt användaren att välja önskad enhet och fält.
- Hämta posten från CRM och visa den i rutnätet.
- Exportera posten till Aspose.Cells Spreadsheet.
- Hämta posten från Aspose.Cells Spreadsheet och visa den i rutnätet.
- Exportera posten från rutnätet till CRM.
## **Systemkrav och stödda plattformar**
### **Systemkrav**
För att installera och använda Aspose .Net Update Import Tool för Microsoft Dynamics CRM måste du ha en av följande CRM-versioner installerade.

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.

Tveka inte att kontakta oss om du hittar några problem med att installera eller använda detta tillägg.
### **Stödda plattformar**
Detta tillägg fungerar med alla versioner av Microsoft Dynamics CRM:

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.
