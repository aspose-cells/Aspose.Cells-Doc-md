## **Introduktion**

Aspose .NET DNN-dataexportör till Excel-modulen tillåter användare att exportera data direkt från lokala eller externa databastabeller, vyer och genom anpassad fråga till Microsoft Excel eller OpenOffice Spreadsheet. Denna modul visar den kraftfulla kalkylarksbyggnadsfunktionen som tillhandahålls av Aspose.Cells.

### **Modulfunktioner**

Den här första versionen av modulen är rik på följande coola funktioner för att göra exportprocessen enkel och användarvänlig

- Tillåt att ansluta Lokal MS SQL Server-databas
- Tillåt att ansluta Fjärr MS SQL Server-databas
- Fyll i alla tabeller från ansluten databas
- Fyll i alla vyer från ansluten databas
- Tillåt att skriva anpassad fråga
- Auto Fit kolumner till innehållslängd.
- Tillåt att hoppa över sträng mer än 32k i excelceller (LoadOptions)
- Applicera Header Column formating som Fet Text
- Tillåt att använda som datakälla (tabell, vyer, anpassad fråga)
- Exportera data till Microsoft Excel-dokument (.xls, .xlsx och .xlsb)
- Exportera data till tabbavgränsat textdokument (*.txt)
- Exportera data till CSV (kommaseparerad) (*.csv)
- Exportera data till OpenDocument Spreadsheet (*.ods)
- Alternativ för att välja önskat utdataformat innan exportering.
- Exporterat dokument skickas automatiskt till webbläsaren för nedladdning.

## **Installation**

När du har laddat ner den, följ dessa steg för att installera modulen i din DNN-webbplats:

1. Logga in på din webbplats som antingen Host eller något annat superanvändar-konto.
1. Navigera till **Host**-menyn och välj **Tillägg**.
1. Klicka på **Installera tilläggsvägvisaren**.
1. Välj åtkomstväggvsaren, bläddra till platsen för den nerladdade zip-filen, välj den och klicka på **Öppna**.
1. Klicka på **Nästa**, acceptera licensen, fortsätt installationen. När den är klar klickar du på **Return**-knappen.

Kontrollera [denna modulinstallationvideo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) från DNN för mer information.

## **Användning**

När du har installerat Aspose .NET Database Data Expoeter för DNN-modulen är det väldigt enkelt att börja använda den på din webbplats. Följ dessa enkla steg för att komma igång:

1. Se till att du är inloggad på DNN med ett Host- eller administratörskonto.
1. Navigera till sidan där du vill lägga till exportmodulen.
1. Välj **Moduler** följt av **Lägg till ny modul** i den övre bågen.
1. Från listan väljer du 'Aspose.NET DNN Data Exporter till Excel med användning av Aspose.Cells' och drar den till positionen.

![todo:image_alt_text](picture2.png)

Du har nu lagt till Aspose .NET Database Data Exporter till Excel för DNN-modulen på din sida.

![todo:image_alt_text](picture3.png)

1. Ange eller använd förifylld MS SQL-serveranslutningssträng
1. Välj Datakälla Typ (Tabell, Vy, Anpassad fråga)
1. Välj eller ange Datakälla (Tabell, Vy, Anpassad fråga)
1. Välj exporttyp
1. Klicka på Exportera Data
1. Önskad fil kommer att laddas ner automatiskt.
