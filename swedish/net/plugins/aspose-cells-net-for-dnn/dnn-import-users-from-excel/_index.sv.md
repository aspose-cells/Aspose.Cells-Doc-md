## **Introduktion**
Aspose .NET Import Users from Excel-modulen tillåter utvecklare att importera bulkanvändare från Excelfil. Den här modulen demonstrerar den kraftfulla kalkylarksfunktionen med Aspose.Cells.
### **Modulfunktioner**
Denna första version av modulen är berikad med följande coola funktioner för att göra importprocessen enkel och lätt att använda:

- Tillåter dig att ladda upp Excelfil och automatiskt filtrerar ut befintliga användare.
- Hoppas automatiskt över befintliga användare för att undvika förvirring.
- Genererar dynamiska lösenord för de nyskapade användarna.
- Tilldelar säkerhetsroller "Registrerade användare" och "Prenumeranter" till nysskapade användare.

## **Installation**
När du har laddat ner den, följ dessa steg för att installera modulen i din DNN-webbplats:

1. Logga in på din webbplats som antingen Host eller något annat superanvändar-konto.
1. Navigera till **Host**-menyn och välj **Tillägg**.
1. Klicka på **Installera tilläggsvägvisaren**.
1. Välj åtkomstväggvsaren, bläddra till platsen för den nerladdade zip-filen, välj den och klicka på **Öppna**.
1. Klicka på **Nästa**, acceptera licensen, fortsätt installationen. När den är klar klickar du på **Return**-knappen.

Kontrollera [denna modulinstallationvideo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) från DNN för mer information.
## **Användning**
När du har installerat Aspose .NET Cells Import Users från Excel för DNN-modulen är det riktigt enkelt att börja använda den på din webbplats. Följ dessa enkla steg för att komma igång:

1. Se till att du är inloggad på DNN med ett Host- eller administratörskonto.
1. Navigera till sidan där du vill lägga till exportmodulen.
1. Välj **Moduler** följt av **Lägg till ny modul** i den övre bågen. 

1. Från listan väljer du 'Aspose Import Users from Excel module for DNN' och drar den till position.

Du har lagt till Aspose .NET Cells Import Users från Excel-modul för DNN-modul på din sida. Filuppladdningskontroller kommer att vara synliga tillsammans med länken "Mallfil". Du kan ladda ner mallfilen och formatera ditt innehåll enligt mallfilen. När du laddar upp din excelfil behandlas automatiskt alla användare och filtrerar ut befintliga användare. Du kan välja de användare du vill importera och trycka på "Import Users"-knappen.
