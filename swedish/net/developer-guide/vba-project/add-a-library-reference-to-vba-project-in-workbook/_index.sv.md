{{% alert color="primary" %}}

 Ibland måste du lägga till eller registrera biblioteksreferensen till VBA-projektet genom kod. Du kan göra det med Aspose.Cells[**VbaProject.References.AddRegisteredReference()**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaprojectreferencecollection/methods/addregisteredreference) metod.

{{% /alert %}}

## **Lägg till en biblioteksreferens till VBA-projektet i Microsoft Excel**

 I Microsoft Excel kan du lägga till en biblioteksreferens till VBA-projektet genom att klicka på**Verktyg > Referenser...** manuellt.
