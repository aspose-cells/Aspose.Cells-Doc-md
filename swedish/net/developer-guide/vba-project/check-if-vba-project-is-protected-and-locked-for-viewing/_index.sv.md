## Kontrollera om VBA-projektet är skyddat och låst för visning i C#

Aspose.Cells låter dig kontrollera om VBA (Visual Basic for Applications)-projektet för Excel-filen är skyddat och låst för visning. För detta tillhandahåller API:et [**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) egenskapen. Om det är låst för visning returnerar [**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) egenskapen **true**.
