## **Lösenordsskydda VBA-projektet i Excel-arboken i C#**

Du kan lösenordsskydda VBA (Visual Basic for Applications)-projektet i arbetsboken med Aspose.Cells med [**VbaProject.Protect()**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/methods/protect) metoden.

## **Exempelkod**

Följande exempelkod laddar den [provexelfilen](43352067.xlsm), får åtkomst till dess VBA-projekt och skyddar det med ett lösenord. Slutligen sparas det som [utdataexelfilen](43352068.xlsm).
