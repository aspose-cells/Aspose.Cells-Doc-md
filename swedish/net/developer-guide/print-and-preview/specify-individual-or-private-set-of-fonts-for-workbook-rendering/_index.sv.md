## **Möjliga användningsscenario**

Vanligtvis specificerar du katalogen med typsnitten eller listan med typsnitt för alla arbetsböcker, men ibland måste du ange individuella eller privata uppsättningar med typsnitt för dina arbetsböcker. Aspose.Cells tillhandahåller [**IndividualFontConfigs**](https://reference.aspose.com/cells/net/aspose.cells/individualfontconfigs)-klassen som kan användas för att ange individuella eller privata uppsättningar med typsnitt för din arbetsbok.

## **Ange individuella eller privata uppsättningar typsnitt för arbetsbokpresentation**

Följande kodexempel laddar den [prov-Excel-filen](67338268.xlsx) med sina individuella eller privata uppsättningar med typsnitt som anges med [**IndividualFontConfigs**](https://reference.aspose.com/cells/net/aspose.cells/individualfontconfigs)-klassen. Se också den [provtypsnitt](67338271.zip) som används i koden samt den [utmatade PDF](67338269.pdf) som genereras av den.
