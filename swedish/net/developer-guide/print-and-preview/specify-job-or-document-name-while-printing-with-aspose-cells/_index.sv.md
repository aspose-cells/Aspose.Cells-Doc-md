{{% alert color="primary" %}}

Du kan specificera jobb- eller dokumentnamn vid utskrift av din arbetsbok eller arbetsblad med hjälp av objekten WorkbookRender eller SheetRender. Aspose.Cells tillhandahåller metoderna WorkbookRender.ToPrinter(printerName, jobName) och SheetRender.ToPrinter(printerName, jobName) som du kan använda för att ange jobbnamn vid utskrift av din arbetsbok eller arbetsblad

{{% /alert %}}

## Specificera jobb- eller dokumentnamn vid utskrift med Aspose.Cells

Kodexemplet laddar den käll-Excel-filen och skickar den sedan till skrivaren genom att ange jobb- eller dokumentnamnet med metoderna WorkbookRender.ToPrinter(printerName, jobName) och SheetRender.ToPrinter(printerName, jobName).
