## **Möjliga användningsscenario**

Ibland vill du visa text i teckensnitt för fjärröstliga språk, t.ex. japanska, kinesiska, thailändska, osv. Aspose.Cells tillhandahåller egenskapen [**TextOptions.FarEastName**](https://reference.aspose.com/cells/net/aspose.cells.drawing.texts/textoptions/properties/fareastname) som kan användas för att ange teckensnittsnamnet för fjärröstligt språk. Dessutom kan du också ange det latinska teckensnittsnamnet med hjälp av egenskapen [**TextOptions.LatinName**](https://reference.aspose.com/cells/net/aspose.cells.drawing.texts/textoptions/properties/latinname).

## **Ange det fjärrösterländska och latinska namnet på teckensnittet i texternas alternativ för Shape**
