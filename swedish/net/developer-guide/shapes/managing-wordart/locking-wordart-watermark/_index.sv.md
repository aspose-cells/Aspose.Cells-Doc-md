{{% alert color="primary" %}}

Aspose.Cells API:erna tillåter att lägga till WordArt-vattenstämplar på kalkylarket på ett sätt så att WordArt blir ett objekt som du kan flytta och positionera på kalkylarket. Det är också möjligt att låsa WordArt-objektet för interaktion såsom redigering, rörelse och markering. Den här artikeln förklarar användning av [**Shape.SetLockedProperty**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/methods/setlockedproperty) metoden för att låsa några aspekter av vattenstämpeln.

{{% /alert %}}

Aspose.Cells API:erna tillåter att låsa vissa aspekter av vattenstämpeln så att användarinteraktionen kan begränsas eller helt blockeras. Följande kodsnutt visar användningen av Aspose.Cells for .NET API för att låsa val av, rörelse, redigering och storleksändring av vattenstämpeln genom att skapa ett kalkylblad från grunden.
