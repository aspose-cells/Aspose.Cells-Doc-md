## Ladda en bild från en URL till ett Excel-kalkylblad

Aspose.Cells för .NET API ger ett enkelt och enkelt sätt att ladda bilder från webbadresser till Excel-kalkylblad. Den här artikeln förklarar hur du laddar ner bilddata till en ström och sedan infogar den i kalkylbladet med Aspose.Cells API. Med den här metoden blir bilderna en del av excel-filen och laddas inte ner varje gång kalkylbladet öppnas.
