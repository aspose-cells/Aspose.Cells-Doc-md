## Ladda en bild från en URL till ett Excel-kalkylblad

Aspose.Cells for .NET API ger ett enkelt och enkelt sätt att ladda bilder från URL: er till Excel-kalkylblad. Den här artikeln förklarar nedladdning av bilddata till en ström och sedan infoga den i kalkylbladet med hjälp av Aspose.Cells API. Genom att använda den här metoden blir bilderna en del av excel-filen och laddas inte ner varje gång kalkylbladet öppnas.
