{{% alert color="primary" %}}

Aspose.Cells låter användarna extrahera temarelaterade data från Excel-fil. Till exempel kan du extrahera Temanamn som tillämpas på arbetsbok och Temafärg som tillämpas på cell eller cellens kanter, etc.

Du kan tillämpa tema på din arbetsbok med Microsoft Excel via kommandot Sidlayout > Teman.

{{% /alert %}}
