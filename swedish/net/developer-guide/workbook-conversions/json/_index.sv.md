## **Konvertera Excel-arbetsbok till JSON**

Aspose.Cells API ger stöd för att konvertera kalkylblad till JSON-format. För att exportera arbetsboken till JSON, ange [**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat) som andra parameter för [**Workbook.Save**](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/3) -metoden. Du kan också använda [**JsonSaveOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonSaveoptions) -klassen för att ange ytterligare inställningar för att exportera kalkylblad till JSON.

Följande kodexempel visar hur man exporterar aktivt kalkylblad till Json genom att använda [**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat) -uppräkningsmedlem. Se koden för att konvertera [källfil](book1.xlsx) till [utdata-Json-filen](book1.Json) som genererats av koden som referens.
