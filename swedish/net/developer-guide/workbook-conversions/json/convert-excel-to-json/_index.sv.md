## **Konvertera Excel-arbetsbok till JSON**

Det finns ingen anledning att undra över hur man konverterar Excel Arbetsbok till JSON, eftersom Apose.Cells for .NET-biblioteket har det bästa beslutet. Aspose.Cells API:et ger stöd för att konvertera kalkylblad till JSON-format. För att exportera arbetsboken till JSON, skicka [**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat) som andra parameter av [**Workbook.Save**](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/3)-metoden. Du kan också använda [**JsonSaveOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonSaveoptions)-klassen för att ange ytterligare inställningar för att exportera kalkylbladet till JSON.
