{{% alert color="primary" %}} 

Som standard sparar Aspose.Cells Excel som PDF med standardstorlek. Du kan dock också spara det med minsta storlek med hjälp av egenskapen [PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/net/aspose.cells/pdfsaveoptions/properties/optimizationtype). Den accepterar följande värden

- PdfOptimizationType.Standard
- PdfOptimizationType.MinimumSize
