{{% alert color="primary" %}}

Aspose.Cells stöder att spara en ODS-fil (**OpenDocument Kalkylblad**) i ODF (**OpenDocument Format** ) 1.1 och 1.2 specifikationer. Aspose.Cells har[**OdsSaveOptions.IsStrictSchema11**](https://reference.aspose.com/cells/net/aspose.cells/odssaveoptions/properties/isstrictschema11) egenskap som anger användningen av ODF 1.1-specifikationen för att spara ODS-filer. Standardvärdet för den här egenskapen är**falsk**, så filen ODS som sparas utan denna inställning använder 1.2-specifikationerna.

{{% /alert %}}

Exempelkoden nedan skapar ett arbetsboksobjekt, lägger till ett visst värde till cell A1 på det första kalkylbladet och sparar sedan filen ODS i ODF 1.1- och 1.2-specifikationer. Som standard sparas filen ODS i ODF 1.2-specifikationen.
