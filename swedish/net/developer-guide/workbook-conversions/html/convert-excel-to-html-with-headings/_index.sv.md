## **Konvertera Excel till HTML med rubriker**

Aspose.Cells ger möjlighet att exportera rubriker vid omvandling av Excel till HTML. För detta kan du ställa in egenskapen [**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/exportheadings) som **true**. Standardvärdet för egenskapen [**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/exportheadings) är **false**.
