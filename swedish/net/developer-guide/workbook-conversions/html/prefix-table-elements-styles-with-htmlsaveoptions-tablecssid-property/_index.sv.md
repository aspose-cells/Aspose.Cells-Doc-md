## **Möjliga användningsscenario**

Aspose.Cells låter dig prefixa tabelldatastilar med [**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/tablecssid)-egenskapen. Anta att du anger denna egenskap med något värde som **MyTest_TableCssId**, kommer du att hitta tabelldatastilar som visas nedan

{{< highlight java >}}

 table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}
