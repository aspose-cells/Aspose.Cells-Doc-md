{{% alert color="primary" %}}

Microsoft Excel gör det möjligt för användare att lägga till kommentarer i celler för att lägga till ytterligare information och markera data. Utvecklare kan behöva anpassa kommentaren för att specificera inriktning, textinriktning, teckensnittsfärg osv. Aspose.Cells tillhandahåller API:er för att uppnå detta.

{{% /alert %}}

Aspose.Cells tillhandahåller en [**Shape.TextBody**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textbody)-egenskap för att ställa in textfärgen för kommentaren. Följande provkod visar användningen av [**Shape.TextBody**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textbody)-egenskapen för att ställa in textriktning för en kommentar.
