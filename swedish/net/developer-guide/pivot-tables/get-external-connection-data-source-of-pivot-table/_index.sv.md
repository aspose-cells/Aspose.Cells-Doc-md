## **Hämta extern anslutningsdatakälla för pivottabell**

Aspose.Cells ger möjlighet att hämta den externa anslutningsdatakällan för pivottabellen. För detta tillhandahåller API[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)egendom av[**Pivottabell**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) klass. De[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)egendom returnerar[**Extern anslutning**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) objekt. Följande kodavsnitt visar användningen av[**Pivottabell.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) egenskap för att hämta den externa anslutningsdatakällan för pivottabellen.
