Aspose.Cells tillhandahåller funktionen att byta ut text i en arbetsbok med hjälp av ett reguljärt uttryck. För detta tillhandahåller API [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) egenskapen i [**ReplaceOptions**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions) klassen. Om [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) är inställt på **true** indikerar det att den sökta nyckeln kommer att vara ett reguljärt uttryck.

Följande kodsnutt visar användningen av [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) egenskapen genom att använda den [exempel excel-fil](101089318.xlsx). [Utdatafilen](101089319.xlsx) som genereras av följande kodsnutt bifogas som referens.
