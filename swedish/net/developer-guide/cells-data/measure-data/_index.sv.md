{{% alert color="primary" %}}

Ibland behöver du beräkna bredden och höjden på cellvärdet för att passa cellvärdet inuti cellen. Aspose.Cells tillhandahåller [**Cell.GetWidthOfValue()**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getwidthofvalue) och [**Cell.GetHeightOfValue()**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getheightofvalue) metoder för detta ändamål. Genom att använda dessa metoder kan du beräkna bredden och höjden på cellvärdet och sedan ställa in bredden på kolumnen och höjden på raden för den cellen respektive och detta kommer sedan att justera eller passa cellvärdet inuti cellen.

Alternativt kan du också [automatiskt anpassa rader och kolumner för din cell eller serie av celler](/cells/sv/net/autofit-rows-and-columns/) med hjälp av Aspose.Cells API:er.
