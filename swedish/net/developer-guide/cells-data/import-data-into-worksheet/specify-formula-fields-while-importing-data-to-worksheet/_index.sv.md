##  **Möjliga användningsscenarier**

Du kan ange formelfält när du importerar data till ditt kalkylblad med hjälp av[**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas) . Den här egenskapen tar den booleska arrayen där värdet**Sann**betyder att fältet är ett formelfält. Till exempel, om det tredje fältet är ett formelfält, kommer det tredje värdet i matrisen att vara *true**.
