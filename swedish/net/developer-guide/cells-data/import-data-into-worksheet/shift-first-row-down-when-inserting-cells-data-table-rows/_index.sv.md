## **Möjliga användningsscenarier**

Aspose.Cells låter dig flytta den första raden nedåt när du infogar en tabell i kalkylbladet. Det här dokumentet förklarar hur du kan utföra uppgiften med Aspose.Cells API:er.

## **Flytta första raden nedåt när du infogar Cells datatabellrader**

Följande exempelkod illustrerar hur du flyttar den första raden nedåt när du infogar en tabell i kalkylbladet. Vi använder en enkel mall Excel-fil i kod för att demonstrera funktionen. Du kan utöva funktionen genom att ställa in boolean[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown) tillskriva**Sann**/**Falsk** för att bättre förstå det. Vänligen se[exempel på Excel-fil](45056031.xlsx), [output Excel False-fil](45056032.xlsx) , och[output Excel True-fil](45056033.xlsx)för din kännedom.
