{{% alert color="primary" %}}

Ibland vill du ignorera dolda kolumner vid export av kalkylbladsdata till en datatabell. Du kan åstadkomma det med hjälp av Aspose.Cells genom att ställa [**ExportTableOptions.PlotVisibleColumns**](https://reference.aspose.com/cells/net/aspose.cells/exporttableoptions/properties/plotvisiblecolumns) till **true**. Som standard är värdet **false**, så du måste ställa in det till **true** för att ignorera de dolda kolumnerna.
