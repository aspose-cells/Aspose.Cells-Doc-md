{{% alert color="primary" %}}

Aspose.Cells stöder konvertering av Microsoft Excel-orienterad HTML till XLS/XLSX-format. Det innebär att den HTML som genereras av Microsoft Excel kan konverteras tillbaka till XLS/XLSX-format med hjälp av Aspose.Cells.

På samma sätt kan, om det finns enkel HTML, Aspose.Cells konvertera den till HTML Rich Text. Aspose.Cells tillhandahåller egenskapen [**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/properties/htmlstring) som kan ta sådan enkel HTML och konvertera den till formaterad celltext.

{{% /alert %}}
