## **Exportera XML-data som är länkad till XML-karta inuti arbetsboken**

Följande exempelkod exporterar XML-data från alla XML-kartor från arbetsboken en efter en. Vänligen kontrollera [sample excel file](5115497.xlsx) som används i denna kod och [exported XML data of the first XML Map](5472487.xml).
