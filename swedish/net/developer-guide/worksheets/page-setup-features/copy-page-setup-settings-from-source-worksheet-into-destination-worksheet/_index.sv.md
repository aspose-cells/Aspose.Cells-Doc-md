## **Möjliga användningsscenarier**

 När du lägger till ett nytt ark i en arbetsbok innehåller det standarden*Utskriftsinställningar*. Det kan finnas tillfällen då du behöver överföra inställningarna ([**Utskriftsformat**](https://reference.aspose.com/cells/net/aspose.cells/pagesetup)) från ett kalkylblad till ett annat kalkylblad. Det här dokumentet förklarar hur du kopierar inställningar för sidinställningar från ett kalkylblad till ett annat med hjälp av Aspose.Cells API:er.
