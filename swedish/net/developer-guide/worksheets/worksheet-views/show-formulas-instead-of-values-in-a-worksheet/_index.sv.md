{{% alert color="primary" %}}

 Det är möjligt att visa formler istället för beräknade värden i Microsoft Excel med hjälp av**Visa formler** alternativ från**Formler**band. När formler visas visar Microsoft Excel formler i kalkylbladet. Du kan uppnå samma sak med Aspose.Cells.

{{% /alert %}}

 Aspose.Cells tillhandahåller en[**Arbetsblad.ShowFormulas**](https://reference.aspose.com/cells/net/aspose.cells/worksheet/properties/showformulas) fast egendom. Ställ in detta på**Sann**för att ställa in Microsoft Excel för att visa formler.
