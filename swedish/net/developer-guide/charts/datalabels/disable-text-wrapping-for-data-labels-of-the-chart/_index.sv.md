{{% alert color="primary" %}}

Microsoft Excel 2013 tillåter användare att radbryta eller packa upp text i diagrammets dataetiketter. Som standard är texten inuti dataetiketterna i diagrammet i lindat läge.

 Aspose.Cells tillhandahåller en[**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) egenskapen som du kan ställa in True eller False för att aktivera eller inaktivera textinbrytning av dataetiketter.
