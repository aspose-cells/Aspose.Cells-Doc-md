{{% alert color="primary" %}}

 Du kan använda Aspose.Cells för att skapa en anpassad dataetikett med rik text för diagrampunkten. Aspose.Cells tillhandahåller[**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) metod för att returnera[**FontSetting**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) objekt som kan användas för att ställa in teckensnittsegenskaperna för texten som dess färg, djärvhet, etc.

{{% /alert %}}
