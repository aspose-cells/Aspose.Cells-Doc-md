{{% alert color="primary" %}}

Du kan använda Aspose.Cells för att skapa anpassade riktmärken med rik text för diagram punkten. Aspose.Cells tillhandahåller metoden [**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) för att returnera objektet [**FontSetting**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) som kan användas för att ställa in egenskaperna för teckensnittet som dess färg, fetstil, osv.

{{% /alert %}}
