{{% alert color="primary" %}}

 Ibland vill du komma åt ett kalkylblad från ett diagrams referens. Aspose.Cells tillhandahåller[**Chart.Worksheet**](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/properties/worksheet) egenskap som returnerar referensen till kalkylbladet som innehåller diagrammet.

{{% /alert %}}

 Följande exempel visar hur du använder[**Chart.Worksheet**](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/properties/worksheet) fast egendom. Koden skriver först ut namnet på kalkylbladet och kommer sedan åt det första diagrammet på kalkylbladet. Den skriver sedan ut kalkylbladets namn igen med hjälp av[**Chart.Worksheet**](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/properties/worksheet)fast egendom.
