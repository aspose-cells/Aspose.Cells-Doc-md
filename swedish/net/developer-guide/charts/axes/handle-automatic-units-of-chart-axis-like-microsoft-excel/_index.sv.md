## **Möjliga användningsscenarier**
Tidiga versioner av Aspose.Cells kunde inte hantera automatiska enheter av sjökortsaxeln korrekt när diagrammet renderas till bild eller PDF. Nu stöder Aspose.Cells hanteringen av automatiska enheter för sjökortsaxeln. Det finns ingen kodändring. Konvertera bara ditt diagram till bild eller PDF och det kommer att återge diagramaxeln precis som Microsoft Excel återger den.
## **Hantera automatiska enheter för diagramaxeln som Microsoft Excel**
 Följande exempelkod laddar[exempel på Excel-fil](61767755.xlsx) och genererar[output PDF-diagram](61767752.pdf). Skärmdumpen visar de automatiska enheterna för diagramaxeln i röda rektanglar och jämför även exemplet på Excel-fildiagrammet med det utgående PDF-diagrammet. Båda är exakt lika.
