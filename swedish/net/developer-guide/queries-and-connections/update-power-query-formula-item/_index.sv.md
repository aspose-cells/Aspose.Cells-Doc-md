## **Användningsscenarie**

Det kan finnas fall där datakällans filer flyttas och excelfilen inte kan hitta filen. I sådana fall tillhandahåller Aspose.Cells API alternativet att uppdatera Power Query-formelobjektet genom att använda klassen [*PowerQueryFormulaItem*](https://reference.aspose.com/cells/net/aspose.cells.querytables/powerqueryformulaitem) för att uppdatera platsen för källfilen.
