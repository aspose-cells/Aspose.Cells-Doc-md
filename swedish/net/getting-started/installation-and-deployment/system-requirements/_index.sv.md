## **Utvecklingsmiljöer**
Du kan använda Aspose.Cells for .NET för att utveckla applikationer i alla utvecklingsmiljöer som riktar sig mot .NET-plattformen, men följande miljöer stöds explicit:

- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- Microsoft Visual Studio 2022

{{% alert color="primary" %}} 

För att använda Aspose.Cells behöver inte maskinen ha Microsoft Excel installerat eftersom biblioteket i sig är en spreadsheet-hanteringsmotor. Men för att visa excelfilen som genereras av komponenten måste du ha åtminstone Microsoft Excel Viewer installerat på din maskin.

{{% /alert %}} 
## **Aspose.Cells Grid Suite Systemkrav**
Aspose.Cells Grid Suite är sammansatt av tre kontroller:

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb
- Aspose.Cells.GridJs

Kontrollernas systemkrav beskrivs separat.

{{% alert color="primary" %}} 

För att använda dessa kontroller behöver inte maskinen som kör Aspose.Cells.GridDesktop och Aspose.Cells.GridWeb ha Microsoft Excel installerat eftersom Aspose.Cells.GridDesktop och Aspose.Cells.GridWeb kan öppna och spara Excel-kalkylblad utan behov av Microsoft Excel.
