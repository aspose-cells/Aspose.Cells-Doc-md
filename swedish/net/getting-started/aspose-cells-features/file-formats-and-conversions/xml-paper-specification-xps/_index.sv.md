## **Acerca de XPS**

|![todo:imagen_alternativa_texto](xml-paper-specification-xps_1.png)| los[Especificación de papel XML](https://en.wikipedia.org/wiki/XML_Paper_Specification) es un lenguaje de descripción de página y un formato de documento fijo desarrollado originalmente por Microsoft. Al igual que PDF, XPS es un formato de documento de diseño fijo diseñado para preservar la fidelidad del documento y proporcionar una apariencia de documento independiente del dispositivo.|
|:- |:- |
## **XPS en Aspose.Cells**
Cualquier documento cargado en Aspose.Cells se puede convertir a XPS. Aspose.Cells utiliza el mismo diseño de página de alta fidelidad y motor de renderizado para generar resultados en todos los formatos de diseño fijo compatibles, por ejemplo, PDF, XPS, etc.

Vale la pena señalar que Aspose.Cells genera XPS directamente (incluido el subconjunto de fuentes TrueType), sin depender de las clases de Presentation Foundation Windows que se encuentran en .NET Framework 3.5, esto permite que Aspose.Cells produzca XPS en máquinas que ejecutan .NET de versiones anteriores o en plataformas completamente diferentes.
