{{% alert color="primary" %}} 

Aspose.Cells kan användas för att spara dina Excel-arbetsböcker i HTML-format. De exporterade HTML-arbetsböckerna är Microsoft Excel-orienterade dvs. de är liknande som exporteras av Microsoft Excel. Du kan också skapa arbetsböcker från HTML men detta HTML bör vara Excel-orienterat.

{{% /alert %}} 
## **HTML i Aspose.Cells**
Aspose-lösningen stöder noggrant följande dokumentfunktioner vid konvertering till HTML:

- Celledata och textformatering
- Diagram och pivottabeller
- Huvuden/Fötter
- Kantlinjer, kantfärger
- Bilder, textlådor och ritningsformer
- Hyperlänkar
