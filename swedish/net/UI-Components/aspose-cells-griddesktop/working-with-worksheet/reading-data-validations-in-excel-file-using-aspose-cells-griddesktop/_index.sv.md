{{% alert color="primary" %}} 

Aspose.Cells.GridDesktop kan läsa och tillämpa **Data Valideringar** i din Excel-fil. Ladda din Excel-fil i GridDesktop och alla Data Valideringar i den kommer att börja fungera som de gör i Microsoft Excel.

{{% /alert %}}
