## **Möjliga användningsscenarier**
 Ibland är det viktigt att hitta GridDesktop-versionen vid körning. Du vill till exempel försäkra dig om att du använder rätt version eller så felsöker du din applikation och du vill veta vilken version av GridDesktop som används för närvarande. GridDesktop tillhandahåller metoden Aspose.Cells.GridDesktop.GridDesktop.GetVersion() som returnerar GridDesktop-versionen vid körning.
