## **Möjliga användningsscenario**
Ibland är det viktigt att hitta GridDesktop-versionen vid körning. Till exempel vill du se till att du använder rätt version eller felsöka din applikation och vill veta versionen av GridDesktop som för närvarande används. GridDesktop tillhandahåller metoden Aspose.Cells.GridDesktop.GridDesktop.GetVersion() som returnerar GridDesktop-versionen vid körning.
