{{% alert color="primary" %}} 

Du kan använda följande klientens sidofunktioner för att välja ett område eller hämta det valda området i GridWeb med JavaScript.

- getSelectRange()
- setSelectRange()
- clearSelections()

getSelectRange() returnerar det senast valda området. setSelectRange() väljer det angivna området. clearSelections() rensar all markering förutom den aktuella aktiva cellen.
