Använd metoden Aspose.Cells.GridWeb.GridWeb.GetVersion() för att hämta versionen eller den körande versionen av Aspose.Cells.GridWeb. Denna version hjälper dig att hitta GridWeb-versionen på körtid genom kod.

{{% /alert %}}
