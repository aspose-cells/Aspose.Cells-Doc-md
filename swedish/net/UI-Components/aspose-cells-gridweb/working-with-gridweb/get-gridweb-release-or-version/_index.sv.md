Använd metoden Aspose.Cells.GridWeb.GridWeb.GetVersion() för att få utgivnings- eller körversionen av Aspose.Cells.GridWeb. Denna version hjälper dig att hitta GridWeb-versionen under körning genom kod.

{{% /alert %}}

Följande exempelkod hittar den körande versionen av Aspose.Cells.GridWeb och skriver ut den på ASP.NET Label Control.

## **Hitta versionsnumret för klientens skriptfil av GridWeb**
