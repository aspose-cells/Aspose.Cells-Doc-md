{{% alert color="primary" %}}

Med Aspose.Cells for SharePoint kan du konvertera dokument mellan många populära kontorsdokumentformat från en SharePoint-dokumentbibliotek. De flesta konverteringarna görs med hög fidelity och precision.

{{% /alert %}}

## **Stödda ingångsformat**

Aspose.Cells for SharePoint stöder följande ingångsformat:

- XLS – Microsoft Excel 97~2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- CSV - Komma Separerade Värden Kalkyldokument
- ODS – OpenDocument
- XML – Microsoft Excel SpreadsheetML

För att generera dokument förlitar sig Aspose.Cells for SharePoint på en inbyggd version av [Aspose.Cells for .NET](https://products.aspose.com/cells/net/), den marknadsledande kalkylbladsbehandlingskomponenten från Aspose. Microsoft Excel används inte för att generera dokument.

## **Stödda utdataformat**

Fullständig lista över utdataformat som stöds av Aspose.Cells for SharePoint är:

- PDF - Adobe Portable Document
- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM – Office Open XML for macros enabled
- TXT – Plain Text
- XML – Microsoft Excel SpreadsheetML
- ODS – OpenDocument
- CSV – Comma Separated Values
- XLSB – Microsoft Excel Binary 2007
- MHTML – Multipurpose Internet Mail Extension HTML
- XPS – XML Paper Specification
- TIFF – Tagged Image File Format
- SVG – Scalable Vector Graphics
- DIF – Drawing Interchange Format
