{{% alert color="primary" %}} 

För att avinstallera Aspose.Cells för SharePoint, kör installationsprogrammet. Om Aspose.Cells för SharePoint redan är installerat, föreslår installationsprogrammet att du tar bort eller reparerar det.

{{% /alert %}} 
### **Avinstallation**
 Under borttagning avaktiverar installationsprogrammet Aspose.Cells för SharePoint för alla webbplatssamlingar och drar tillbaka lösningen från serverfarmen.

**Avinstallerar Aspose.Cells** 

![todo:image_alt_text](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

 Observera att körning av**Reparera** alternativet återaktiverar inte Aspose.Cells för SharePoint på webbplatssamlingar. Istället måste funktionen aktiveras manuellt efter ominstallation med**Reparera** alternativ.

 Avinstallera med hjälp av**Ta bort** alternativet och sedan ominstallera kan vara bekvämare eftersom det låter dig välja målwebbapplikationer och webbplatssamlingar under installationen.

{{% /alert %}}
