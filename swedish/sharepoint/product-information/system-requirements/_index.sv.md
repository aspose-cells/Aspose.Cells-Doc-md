### **Serverkrav**
För att installera Aspose.Cells för SharePoint behöver du en dator som kör något av följande:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft SharePoint Server 2010
- Microsoft SharePoint Server 2013

{{% alert color="primary" %}} 

VI rekommenderar att du installerar Service Pack 2 eller senare för dessa produkter.

{{% /alert %}}
### **Kundkrav**
Du behöver en webbläsare för att komma åt SharePoint-webbplatser. Aspose.Cells för SharePoint testas med:

- Microsoft Internet Explorer
- Mozilla FireFox
- Google Chrome
- Safari
