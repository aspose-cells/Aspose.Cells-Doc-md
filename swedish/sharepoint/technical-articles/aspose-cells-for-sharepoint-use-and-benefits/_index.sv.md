{{% alert color="primary" %}} 

SharePoint-sidor är avsedda för samarbete, kommunikation och lagring av innehåll för att uppfylla specifika affärsbehov. Aspose.Cells for SharePoint har införts för att utföra vissa operationer på Microsoft Excel-dokument som finns i SharePoint-dokumentbibliotek. Denna artikel omfattar följande:

- [En kort anteckning om SharePoint-dokumentbibliotek](/cells/sv/sharepoint/aspose-cells-for-sharepoint-use-and-benefits/).
- [Fördelarna med Aspose.Cells for SharePoint](/cells/sv/sharepoint/aspose-cells-for-sharepoint-use-and-benefits/).
- [Hur man konverterar Microsoft Excel-dokument till andra format](/cells/sv/sharepoint/aspose-cells-for-sharepoint-use-and-benefits/).

{{% /alert %}} 
### **En Kort Anteckning om SharePoint-Dokumentbibliotek**
Innehållet på SharePoint-sidor är i form av dokument delade på en centraliserad plats som kallas SharePoint-dokumentbibliotek (ibland känt som "delat dokumentbibliotek"). Behöriga användare kan ladda upp dokument från sina lokala maskiner till SharePoint-sidan. Dessa dokument lagras i binärformat i en SQL Server-databas som används av SharePoint-sidan. Dokument blir tillgängliga för andra SharePoint-användare i form av en lista med hyperlänkar enligt nedan:

**Ett dokumentbibliotek** 


### **Fördelar**
Följande lista framhäver några fördelar med att använda Aspose.Cells for SharePoint:

- SharePoint-sideservern har inte inbyggd förmåga att konvertera Microsoft Excel-filer direkt på servern. Aspose.Cells for SharePoint lägger till den här funktionen till SharePoint-sideservern.
- Aspose.Cells for SharePoint kräver inte att Microsoft Excel används. Det är installerat på servern så SharePoint-användare behöver inte Microsoft Excel på sina lokala system för att konvertera Microsoft Excel-dokument från ett format till ett annat.
- Aspose.Cells for SharePoint kräver inga förkunskaper förutom de som krävs för att skapa en SharePoint-sida.
- Aspose.Cells for SharePoint är [lätt att distribuera på en SharePoint sideserver](/cells/sv/sharepoint/installing-aspose-cells-for-sharepoint/).
- Aspose.Cells for SharePoint stödjer [ett antal inmatnings- och utmatningsformat](/cells/sv/sharepoint/multiple-format-support/).
- Aspose.Cells for SharePoint exponerar ett offentligt API för utvecklare som behöver utveckla sina egna SharePoint-lösningar.
### **Konvertera Microsoft Excel-dokument med Aspose.Cells for SharePoint**
När Aspose.Cells for SharePoint är installerat på SharePoint-sidan finns alternativet **Konvertera med Aspose.Cells.SharePoint** tillgängligt i snabbmenyn: 

**Hitta alternativet Konvertera med...** 

För att konvertera ett Microsoft Excel-dokument från ditt SharePoint-dokumentbibliotek till andra format, följ nedanstående steg:

1. Klicka på önskat Microsoft Excel-dokument och klicka sedan på Konvertera med Aspose.Cells for SharePoint
   Konverteringsinställningarna visas. 

   **Besluta om att konvertera** 


1. Välj utdataformat.
   Här kan du också ändra utdatafilnamn och destinationsmapp.
1. Klicka på **Konvertera**. 

   **Välja utdataformat** 



1. När konverteringen är klar visas ett framgångsmeddelande. 

   **Lyckad konvertering** 



1. Klicka på **Återvänd** för att komma tillbaka till dokumentbiblioteket.
1. Det konverterade dokumentet listas i SharePoint-dokumentbiblioteket. 

   **Den konverterade filen listas i dokumentbiblioteket** 


### **Sammanfattning**
{{% alert color="primary" %}} 

Att installera och använda Aspose.Cells for SharePoint är enkelt. Det lägger inte bara till en funktion för att konvertera Microsoft Excel-filer direkt på SharePoint-servern, utan eliminerar också behovet av att Microsoft Excel ska vara installerat på klientmaskiner. Dessutom kan utvecklare använda Aspose.Cells for SharePoint's offentliga API för att utveckla sina egna SharePoint-lösningar.

Med andra ord är det fördelaktigt för SharePoint-webbplatsadministratörer, utvecklare och slutanvändare.
