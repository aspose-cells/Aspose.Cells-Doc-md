{{% alert color="primary" %}} 

En av de vanligaste uppgifterna som användare använder Aspose.Cells för är att konvertera individuella filer. 

{{% /alert %}} 
### **Konvertera individuella filer**
För att konvertera ett särskilt dokument i en dokumentmapp: 

1. Välj **Konvertera med Aspose.Cells.SharePoint** i dokumentets Redigera kontrollblock (ECB). 

   **Välj “Konvertera via Aspose.Cells” objektet i dokumentets ECB** 

![todo:image_alt_text](convert-one-file_1.png)



Konverteringsinställningsskärmen visas. Här kan du finjustera konverteringen:

1. Välj destinationsfilformatet från listan **Konvertera till**.
1. Ange utdatafilens namn i fältet **Destinationsfil**.
   Som standard förifyller Aspose.Cells for SharePoint detta fält med namnet på källfilen. 
   Förlängningen läggs till automatiskt enligt det valda formatet.
1. Starta konverteringen genom att klicka på **Konvertera**.
   (Eller gå tillbaka till biblioteket, avbryt dina inställningar, genom att klicka på **Avbryt**.)
