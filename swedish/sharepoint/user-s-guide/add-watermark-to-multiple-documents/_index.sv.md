Om du vill lägga till vattenstämpel i flera Excel-filer, välj alternativet "Lägg till vattenmärke med Aspose.Cells" i menyfliksområdet.

![todo:image_alt_text](add-watermark-to-multiple-documents_1.png)

Hämta alla Excel-filer från datakällmappen och skapa en fillistatabell.

 Välj fil som behöver lägga till vattenstämpel, klicka**Lägg till vattenstämpel** för att lägga till vattenstämpel för de valda filerna.

![todo:image_alt_text](add-watermark-to-multiple-documents_2.png)
