Om du vill lägga till vattenstämpel i flera Excel-filer, välj alternativet ”Lägg till vattenstämpel med Aspose.Cells” på menyraden.

![todo:image_alt_text](add-watermark-to-multiple-documents_1.png)

Hämta alla Excel-filer från datakälla mapp och skapa en filtabell.

Välj filen som behöver lägga till vattenstämpel, klicka på **Lägg till vattenstämpel** knappen för att lägga till vattenstämpel för de valda filerna.

![todo:image_alt_text](add-watermark-to-multiple-documents_2.png)
