## **Möjliga användningsscenario**

Aspose.Cells for Node.js via C++ stöder det anpassade mönstret *DBNum*. Till exempel, om ditt cellvärde är 123 och du anger dess anpassade formatering som [DBNum2][$-804]General, kommer det att visas som 壹佰贰拾叁. Du kan ange ditt anpassade cellformat med hjälp av metoderna [**Cell.getStyle()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getStyle--) och [**Style.setCustom(string)**](https://reference.aspose.com/cells/nodejs-cpp/style/#setCustom-string-).
