## **Hämta cellvalidering i ODS-filer**  

Med Aspose.Cells for Node.js via C++ kan du hämta validering som tillämpats på en cell i ODS-filer. För detta erbjuder API:n metoden [**Cell.getValidation()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getValidation--) i klassen [**Cell**](https://reference.aspose.com/cells/nodejs-cpp/cell/).  

Följande kodexempel demonstrerar användningen av metoden [**Cell.getValidation()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getValidation--) genom att ladda [källa ODS](101089354.ods) filen och läsa valideringen av cell A9.
