## **Få textens bredd av cellvärdet**

Ibland kan utvecklare behöva beräkna bredden på cellens värde för att arrangera data i en viss layout. Aspose.Cells for Node.js via C++ tillhandahåller [**CellsHelper.getTextWidth(string, Font, number)**](https://reference.aspose.com/cells/nodejs-cpp/cellshelper/#getTextWidth-string-font-number-)-metoden, som låter utvecklare få textbredden för cellens värde. Följande exempel kod illustrerar hur man använder [**CellsHelper.getTextWidth(string, Font, number)**](https://reference.aspose.com/cells/nodejs-cpp/cellshelper/#getTextWidth-string-font-number-) för att få åtkomst till textbredden av cellens värde.
