{{% alert color="primary" %}}  

Ibland behöver du beräkna bredden och höjden på cellvärdet för att passa cellvärdet inuti cellen. Aspose.Cells tillhandahåller [**Cell.getWidthOfValue()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getWidthOfValue--) och [**Cell.getHeightOfValue()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getHeightOfValue--) metoder för detta ändamål. Genom att använda dessa metoder kan du beräkna bredden och höjden på cellvärdet och sedan ställa in bredden på kolumnen och höjden på raden för den cellen respektive och detta kommer sedan att justera eller passa cellvärdet inuti cellen.  

Alternativt kan du också [autopassa rader och kolumner för din cell eller cellintervall](/cells/sv/nodejs-cpp/autofit-rows-and-columns/) med Aspose.Cells API:er.
