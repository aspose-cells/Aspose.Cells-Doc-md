{{% alert color="primary" %}} 

Hyperlänkar är användbara för att få tillgång till information på andra kalkblad eller på webbplatser. Microsoft Excel låter användare lägga till hyperlänkar på text i celler och på bilder. Bildhyperlänkar kan göra navigering i ett kalkblad enklare, exempelvis, som knappar för nästa och föregående, eller logotyper som länkar till specifika sidor. Denna dokument förklarar hur man infogar bildhyperlänkar i ett kalkblad med hjälp av Aspose.Cells for Node.js via C++.

{{% /alert %}} 

Aspose.Cells for Node.js via C++ låter dig lägga till hyperlänkar till bilder i kalkblad vid körning. Det är möjligt att sätta och ändra länkens visningstips och adress. Följande exempel visar hur man lägger till en bildhyperlänk i ett kalkblad.
