{{% alert color="primary" %}}

Aspose.Cells stödjer konvertering av Microsoft Excel-inriktad HTML till XLS/XLSX-format. Det innebär att HTML som genereras av Microsoft Excel kan konverteras tillbaka till XLS/XLSX med hjälp av Aspose.Cells.

Likadant, om det finns enkel HTML, kan Aspose.Cells konvertera den till HTML Rich Text. Aspose.Cells erbjuder [**Cell.setHtmlString(string)**](https://reference.aspose.com/cells/nodejs-cpp/cell/#setHtmlString-string-)-metoden som kan ta sådan enkel HTML och konvertera den till formaterad celltext.

{{% /alert %}}
