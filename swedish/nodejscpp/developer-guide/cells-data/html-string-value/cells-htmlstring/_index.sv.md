## **Möjliga användningsscenario**
När du behöver ställa in formaterad data för en specifik cell kan du tilldela en HTML-sträng till cellen. Självklart kan du också få HTML-strängen för cellen. Aspose.Cells for Node.js via C++ erbjuder denna funktion. Aspose.Cells ger följande egenskaper och metoder för att hjälpa dig att nå dina mål.
- [**Cell.setHtmlString(string)**](https://reference.aspose.com/cells/nodejs-cpp/cell/#setHtmlString-string-)

## **Hämta och sätt HTML-sträng med Aspose.Cells for Node.js via C++**
Detta exempel visar hur man:

1. Skapa en arbetsbok och lägg till lite data.
1. Hämta den specifika cellen i det första kalkbladet.
1. Sätt HTML-sträng till cellen.
1. Hämta HTML-sträng för cellen.
