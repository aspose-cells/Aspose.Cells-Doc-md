{{% alert color="primary" %}}

 Ibland vill du formatera pivottabellceller. Till exempel, du vill tillämpa bakgrundsfärg på pivottabellceller. Aspose.Cells for Node.js via C++ erbjuder två metoder [**PivotTable.formatAll(style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#formatAll-style-) och [**PivotTable.format(row, column, style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#format-number-number-style-), som du kan använda för detta ändamål.

[**PivotTable.formatAll(style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#formatAll-style-) tillämpar stilen på hela pivot-tabellen medan [**PivotTable.format(row, column, style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#format-number-number-style-) tillämpar stilen på en enskild cell i pivot-tabellen.

{{% /alert %}}
Följande exempelkod läser in [exemplet Excel-fil](pivot_format.xlsx) som innehåller två pivot-tabeller och uppnår operationen att formatera hela pivot-tabellen och formatera individuella celler i pivot-tabellen.
