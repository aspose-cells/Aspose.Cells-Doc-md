## **Hur man ställer in anpassad sortering i pivot-tabell med Aspose.Cells for Node.js via C++-biblioteket**
Genom att använda Aspose.Cells for Node.js via C++ API kan du sortera pivot-tabeller på fältvärden. Följande kodavsnitt laddar ett exempel-Excel och lägger till tre pivot-tabeller. Den första utan anpassad sortering, den andra sorteras på "SeaFood"-radsfältvärden, och den tredje sorteras på "28/07/2000"-kolumnfältvärden.
