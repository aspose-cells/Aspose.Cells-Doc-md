Aspose.Cells for Node.js via C++ ger möjlighet att spara pivottabeller i ODS-filer. För detta konverterar du bara en arbetsbok med befintlig pivottabell eller skapar en ny pivottabell och sparar filen i ODS-format. Se till att anropa [**PivotTable.calculateData**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#calculatedata--) innan du sparar för att säkerställa att pivottabellen renderas i den exporterade ODS-filen.
