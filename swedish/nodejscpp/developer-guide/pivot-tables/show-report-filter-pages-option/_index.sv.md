## **Visa alternativ för rapportsidfiltrering**
Excel stöder att skapa pivottabeller, lägga till rapportfilter och aktivera "Visa rapportfilter-sidor"-alternativet. Aspose.Cells for Node.js via C++ stöder också denna funktion för att aktivera "Visa rapportfilter-sidor"-alternativet för den skapade pivottabellen. Nedan visas skärmbilden som visar "Visa rapportfilter-sidor"-alternativet i Excel.
