 Den här sidan innehåller release notes för[Aspose.Cells for JasperReports 1.1.0](https://downloads.aspose.com/cells/jasperreports/new-releases/aspose.cells-for-jasperreports-1.1.0/)

Vi är glada att kunna meddela Aspose.Cells for JasperReports v1.1.0 för användarna! Vad är nytt:

- Stöder Microsoft Excel 97 till 2003 och Microsoft Excel 2007-format. Mer information kan hittas i dokumentet "Konfigurationsparametrar" i produktdokumentationen.
- Fixar flera beräkningsfel i rapportelementstorlek.
- Stöder JasperReports 3.7.x.
