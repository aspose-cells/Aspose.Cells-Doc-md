 Den här sidan innehåller release notes för[Aspose.Cells för JasperReports 1.0.1](https://downloads.aspose.com/cells/jasperreports/new-releases/aspose.cells-for-jasperreports-1.0.1/)

{{% /alert %}}

Vi är glada att kunna meddela Aspose.Cells för JasperReports 1.0.1!

Vad är nytt:

- Teckenstorleken justeras på lämpligt sätt för att klistra in tecken i Microsoft Excel-format som liknar tecken i PDF-format.
-  Axeletiketter (av diagram) i datumformat renderas korrekt och därefter.
- Gradient Fill-effekter för områdena stöds inklusive diagrammets bakgrund (diagramområde, plotområde etc.), legends bakgrund, staplar och bubblor etc.

Vad är fixat:

- Fixade några buggar med att fylla vägg och golv i 3D-diagram.
- Det gick inte att rendera inbäddat objekt: Filen hittades inte.

- Fixade en bugg med cirkeldiagrametiketter.

- Fixat en bugg angående stöd för jasperserver 3.1.0.
