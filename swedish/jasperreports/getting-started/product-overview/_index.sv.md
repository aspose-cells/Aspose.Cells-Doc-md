### **Välkommen till Aspose.Cells for JasperReports-dokumentationen!**

Aspose.Cells for JasperReports är en bibliotek som fungerar som ett ersättning för [JasperReports](https://community.jaspersoft.com/project/jasperreports-library)'s ursprungliga Microsoft Excel-exportör, med några förbättringar. Den eliminerar beroendet av POI-biblioteket och använder Aspose.Cells for Java på baksidan för att exportera rapporter i Microsoft Excel-format (XLS, XLSX).

{{% /alert %}}

### **Produktbeskrivning**

Aspose.Cells for JasperReports är en flexibel lösning som gör det enkelt att exportera rapporter från JasperReports och JasperServer till Microsoft Excel filformat. Alla rapportfunktioner konverteras till Microsoft Excel-kalkylblad med högsta precision.

Förutom funktionaliteten som tillhandahålls av den ursprungliga POI-baserade Micrsofot Excel-exportören erbjuder denna komponent några värdefulla funktioner. De stora innovationerna relaterade till export till XLS inkluderar:

- Tillhandahålla redigerbara diagram i Microsoft Excel-format istället för statiska bilder.
