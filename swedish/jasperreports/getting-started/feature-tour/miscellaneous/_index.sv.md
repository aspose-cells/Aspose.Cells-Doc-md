Några effektiva funktioner i Aspose.Cells för JasperReports listas nedan:

1. Aspose.Cells för JasperReports stöder Microsoft Excel-diagram så att diagram exporteras som redigerbara diagram snarare än statiska bilder. På så sätt kan användare redigera diagrammen manuellt i Microsoft Excel om de behöver.
 2. Aspose.Cells för JasperReports är[lätt integrerad med JasperServer](/cells/sv/jasperreports/integration-with-jasperserver/).
