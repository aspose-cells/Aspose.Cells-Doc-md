{{% alert color="primary" %}} 

Några effektiva funktioner hos Aspose.Cells for JasperReports listas nedan:

1. Aspose.Cells for JasperReports stöder Microsoft Excel-diagram så att diagram exporteras som redigerbara diagram istället för statiska bilder. På så sätt kan användare redigera diagrammen manuellt i Microsoft Excel om de behöver.
2. Aspose.Cells for JasperReports kan [enkelt integreras med JasperServer](/cells/sv/jasperreports/integration-with-jasperserver/).

{{% /alert %}}
