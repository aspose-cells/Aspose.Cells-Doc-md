{{% alert color="primary" %}}

Aspose.Cells for JasperReports gör att du kan leverera rapporter i XLS- och XLSX-format utan att förlora layout eller formatering. För att skapa kalkylblad i Microsoft Excel-format, förlitar sig Aspose.Cells for JasperReports på en inbyggd version av [Aspose.Cells for Java](https://products.aspose.com/cells/java/), en ledande kalkylbladsbehandlingsbibliotek från Aspose.

Microsoft Excel krävs inte för att skapa dokument med Aspose.Cells.
