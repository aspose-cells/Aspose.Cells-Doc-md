{{% alert color="primary" %}}

 Aspose.Cells för JasperReports låter dig leverera rapporter i XLS- och XLSX-format utan att förlora layout eller formatering. För att skapa kalkylblad i Microsoft Excel-format förlitar sig Aspose.Cells för JasperReports på en inbyggd version av[Aspose.Cells för Java](https://products.aspose.com/cells/java/), ett ledande bibliotek för kalkylbladsbearbetning från Aspose.

Microsoft Excel krävs inte för att generera dokument med Aspose.Cells.
