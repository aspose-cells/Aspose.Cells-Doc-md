{{% alert color="primary" %}}

Aspose.Cells for JasperReports är en produkt för att exportera rapporter från JasperReports och JasperServer till Microsoft Excel-format. Aspose.Cells for JasperReports ärvt alla funktioner i den inbyggda exportören samt kompenserar för vissa brister i den ursprungliga XLS-renderaren i JasperReports. Denna sektion ger en detaljerad beskrivning av Aspose.Cells for JasperReports-funktioner. Klicka på respektive länkar i tabellen för att lära dig mer om funktionerna.

{{% /alert %}}
