{{% alert color="primary" %}}

 Aspose.Cells for JasperReports är en extern exportör för[Jasper rapporterar](https://community.jaspersoft.com/project/jasperreports-library).

 Aspose.Cells for JasperReports tillhandahålls som en enda JAR-fil och kan enkelt installeras på datorer som kör JasperReports.[Ladda ner den för att starta en testversion](https://downloads.aspose.com/cells/jasperreports).

[För att installera exportören](/cells/sv/jasperreports/installation/), **aspose.cells.jasperreports.jar** ska kopieras till**JasperReports\lib** katalog.

{{% /alert %}}
