{{% alert color="primary" %}}

Aspose.Cells for JasperReports är en extern exportör för [JasperReports](https://community.jaspersoft.com/project/jasperreports-library).

Aspose.Cells for JasperReports tillhandahålls som en enda JAR-fil och kan enkelt installeras på datorer som kör JasperReports. [Ladda ner den för att påbörja en testperiod](https://downloads.aspose.com/cells/jasperreports).

[För att installera exportören](/cells/sv/jasperreports/installation/), bör **aspose.cells.jasperreports.jar** kopieras till **JasperReports\lib**-katalogen.

{{% /alert %}}
