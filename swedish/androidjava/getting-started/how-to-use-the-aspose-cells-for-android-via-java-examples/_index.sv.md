Se först till att du har ADT-paketet installerat och att du har laddat ner den senaste versionen av Aspose.Cells for Android via Java.

- Extrahera**aspose-celler-XXXjar**från det senaste versionsarkivet och lägg till det i en ny mapp under exemplet du vill köra. Lägg till filen under en mapp som heter**libs**.
- Öppna projektet i din arbetsyta med Eclipse med ADT.
- Klick**Springa**för att starta och välj lämplig emulator att köra paketet på.
