Se först till att du har ADT-bunteln installerad och att du har laddat ner den senaste versionen av Aspose.Cells för Android via Java.

- Extrahera **aspose-cells-X.X.X.jar** från det senaste versionsarkivet och lägg till det i en ny mapp under exemplet du vill köra. Lägg till filen under en mapp som kallas **libs**.
- Öppna projektet i din arbetsplats med Eclipse med ADT.
- Klicka på **Kör** för att starta och välj den lämpliga emulatorn för att köra paketet.
