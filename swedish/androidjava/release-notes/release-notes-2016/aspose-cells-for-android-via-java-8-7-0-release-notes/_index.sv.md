Aspose.Cells for Android har uppdaterats till version 8.7.0.

Notera
Eftersom kodbasen för Aspose.Cells for Android matchar koden för relevant version .NET och Java, är de flesta ändringar, förbättringar och korrigeringar som ingår i
Aspose.Cells for .NET v8.6.1
, 
Aspose.Cells for .NET v8.6.2
, 
Aspose.Cells for .NET v8.6.3
, 
Aspose.Cells for .NET v8.7.0
, 
Aspose.Cells for Java v8.6.1
, 
Aspose.Cells for Java v8.6.2
, 
Aspose.Cells for Java v8.6.3
och
Aspose.Cells for Java v8.7.0
ingår också i denna Aspose.Cells for Android v8.7.0.
