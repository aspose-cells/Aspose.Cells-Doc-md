**Välkommen till Aspose.Cells för Go**

Aspose.Cells for Go via C++ är ett inbyggt Go-bibliotek som gör det möjligt för Go-utvecklare att skapa, manipulera och konvertera kalkylblad programmässigt utan att kräva Office Automation eller Microsoft Excel-applikationen. Aspose.Cells för Go stöder kalkylbladsformat som MS Excel 97-2003 (XLS), MS Excel 2007-2013/2016 (XLSX, XLSM, XLSB m.m.), Open Office XML och andra format. Du kan också rendera bilder från arbetsblad och konvertera MS Excel-kalkylblad till PDF. API:et tillåter dig också att skapa, manipulera och rendera diagram och former. Dessutom är formelberäkningar en annan bra funktion som erbjuds av komponenten och du kan utnyttja den.
