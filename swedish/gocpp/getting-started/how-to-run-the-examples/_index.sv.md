## **Mjukvarukrav**

Se till att du uppfyller följande krav innan du laddar ner och kör exemplen.

- Go 1.13 eller senare
- Textredigerare eller IDE

## **Importera från GitHub**

Alla exempel på Aspose.Cells for Go via C++ finns på [GitHub](https://github.com/aspose-cells/aspose.cells-go-cpp-examples/).

- Du kan kopiera repositoryt med din favorit-GitHub-klient eller ladda ner ZIP-filen [här](https://github.com/aspose-cells/aspose.cells-go-cpp-examples/archive/master.zip).
- Packa upp innehållet i ZIP-filen till valfri mapp på din dator. Alla exemplen finns i mappen **Exempel**.
- Öppna lösningsfilen i textredigerare eller IDE och bygg projektet.
- Mappen **Data** i rotmappen för **Exempel** innehåller inmatningsfiler. Det är obligatoriskt att ladda ned mappen **Data** tillsammans med exemplen.
- Öppna main.go, alla exempel anropas därifrån.
- Avkommentera de exempel du vill köra från filen.
- go run .

Tveka inte att kontakta oss via vårt forum om du har några problem med att konfigurera eller köra exemplen.

## **Bidra**

Om du vill lägga till eller förbättra ett exempel uppmuntrar vi dig att bidra till projektet. Alla exempel och showcase-projekt i detta repository är öppen källkod och kan fritt användas i dina egna applikationer.

För att bidra kan du forka lagringsplatsen, redigera källkoden och skapa en pull-begäran. Vi kommer att granska ändringarna och inkludera dem i lagringsplatsen om de anses vara hjälpsamma.
