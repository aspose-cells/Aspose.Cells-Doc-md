{{% alert color="primary" %}}

Aspose.Cells tillåter dig att importera XML-kartan inuti arbetsboken med hjälp av [**Workbook.ImportXml()**](https://reference.aspose.com/cells/go-cpp/workbook/importxml_string_string_int_int/)-metoden. Du kan importera XML-karta med Microsoft Excel med följande steg:

- Välj fliken **Developer**.
- Klicka på **Importera** i XML-avsnittet och följ de nödvändiga stegen.

Du måste tillhandahålla dina XML-data för att slutföra importen. Här är ett [prov-XML-data](5115037.txt) som du kan använda för testning.
