{{% alert color="primary" %}}

Aspose.Cells stöder konvertering av Microsoft Excel-orienterad HTML till XLS/XLSX-format. Det innebär att den HTML som genereras av Microsoft Excel kan konverteras tillbaka till XLS/XLSX-format med hjälp av Aspose.Cells.

Likadant, om det finns enkel HTML, kan Aspose.Cells konvertera den till HTML Rich Text. Aspose.Cells erbjuder [**Cell::GetHtmlString**](https://reference.aspose.com/cells/go-cpp/cell/gethtmlstring/)-metoden som kan ta sådan enkel HTML och konvertera den till formaterad celltext.

{{% /alert %}}
