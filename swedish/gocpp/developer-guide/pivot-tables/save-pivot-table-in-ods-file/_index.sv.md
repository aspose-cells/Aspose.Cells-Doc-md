Aspose.Cells ger möjligheten att spara pivottabeller i ODS-filer. För detta kan du antingen konvertera en arbetsbok med en befintlig pivottabell eller skapa en ny pivottabell och spara filen i ODS-format. Se till att anropa [**PivotTable::CalculateData()**](https://reference.aspose.com/cells/go-cpp/pivottable/calculatedata/) innan du sparar för att säkerställa att pivottabellen renderas i utdata-ODS-filen.
