{{% alert color="primary" %}}

Microsoft Excel tillåter dig att ändra layouten för pivottabell med *PivotTable-verktyg > Design > Rapportlayout* menykommandon. Du kan ändra layouten i dessa tre former:

- Visa i kompakt form
- Visa i kontursform
- Visa i tabellform

Aspose.Cells tillhandahåller också [**PivotTable.ShowInCompactForm()**](https://reference.aspose.com/cells/go-cpp/pivottable/showincompactform/), [**PivotTable.ShowInOutlineForm()**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/showinoutlineform/), och [**PivotTable.ShowInTabularForm()**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/showintabularform/)-metoder för att ändra layouten på pivottabellen i dessa tre former.
