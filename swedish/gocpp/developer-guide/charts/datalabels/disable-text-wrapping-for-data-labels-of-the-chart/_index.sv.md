{{% alert color="primary" %}}

Microsoft Excel 2013 tillåter användare att vicka eller avvicka text inne i dataetiketterna för diagrammet. Som standard är texten inne i dataetiketterna för diagrammet i det vikta läget.

Aspose.Cells tillhandahåller en [**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/go-cpp/datalabels/istextwrapped/)-egenskap som du kan ställa in på True eller False för att aktivera eller inaktivera textvaggning för dataetiketterna respektive.
