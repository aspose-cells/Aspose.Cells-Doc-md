{{% alert color="primary" %}}

Du kan använda Aspose.Cells för att skapa rik text anpassad datamärkespunktsbeskrivning. Aspose.Cells tillhandahåller metoden [DataLabels.Characters()](https://reference.aspose.com/cells/go-cpp/charttextframe/characters/) för att returnera objektet [FontSetting](https://reference.aspose.com/cells/cpp/aspose.cells/fontsetting/), vilket kan användas för att ställa in teckensnittsinställningar för texten som dess färg, fetstil, etc.

{{% /alert %}}
