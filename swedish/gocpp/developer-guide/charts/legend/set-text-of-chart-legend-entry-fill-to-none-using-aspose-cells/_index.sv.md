{{% alert color="primary" %}}

Om du vill ställa in texten för diagrammets legendpostfyllnad till none så att den inte ska visas i diagrammets legend, ställ in [**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/go-cpp/legendentry/istextnofill/) till **true**.

{{% /alert %}}

Följande exempelkod ställer in texten för diagrammets andra legendpostfyllnad till none. Vänligen ladda ner den [exempelfil för Excel](5115219.xlsx) som används i denna kod och den [utfärdade Excelfilen](5115217.xlsx) som genererats av den som referens.
