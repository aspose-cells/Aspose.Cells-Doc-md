## **Tillval för titlar**
Aspose.Cells tillåter dig också att hantera diagramtitlar i realtid. Med [Title](https://reference.aspose.com/cells/go-cpp/title/) objektet kan du ändra text, teckensnitt och fyllnadsformat för titlar.

## **Inställning av diagram- eller axeltitlar**
Du kan använda Microsoft Excel för att sätta titlar för ett diagram och dess axlar i en WYSIWYG-miljö. Aspose.Cells tillåter också utvecklare att sätta titlar för ett diagram och dess axlar i realtid. Alla diagram och deras axlar innehåller en [Title](https://reference.aspose.com/cells/go-cpp/title/) egenskap som kan användas för att ställa in deras titlar, som visas i exemplet nedan.
