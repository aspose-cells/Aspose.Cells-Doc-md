Ibland behöver du lägga till eller registrera biblioteksreferensen till VBA-projektet via kod. Du kan göra det med hjälp av Aspose.Cells [**VbaProject.References.AddRegisteredReference()**](https://reference.aspose.com/cells/go-cpp/vbaprojectreferencecollection/addregisteredreference/) -metoden.

{{% /alert %}}

## **Lägg till en biblioteksreferens till VBA-projekt i Microsoft Excel**

I Microsoft Excel kan du lägga till en biblioteksreferens till VBA-projektet genom att klicka på **Verktyg > Referenser...** manuellt.
