{{% alert color="primary" %}}

Aspose.Cells API:erna tillåter att lägga till WordArt-vattenmärken på kalkbladet på ett sätt som gör att WordArt blir ett objekt som du kan flytta och positionera på kalkbladet. Det är också möjligt att låsa WordArt-objektet för alla interaktioner som redigering, flyttning och markering. Denna artikel förklarar användningen av [**Shape.SetLockedProperty**](https://reference.aspose.com/cells/go-cpp/shape/setlockedproperty/) metod för att låsa vissa aspekter av vattenmärket.

{{% /alert %}}
