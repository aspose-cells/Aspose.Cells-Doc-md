## **Konvertera JSON till Excel-arbetsbok** 
Behöver inte fundera på hur man konverterar JSON till en Excelfil, eftersom biblioteket Aspose.Cells for C++ har den bästa lösningen. Aspose.Cells API stöder konvertering av JSON-format till kalkylblad. Du kan använda [JsonLoadOptions](https://reference.aspose.com/cells/go-cpp/jsonloadoptions/) klassen för att specificera ytterligare inställningar för import av JSON till Workbook. 

Följande kodexempel visar import av JSON till Excel-arbetsbok. Se koden för att konvertera [källfilen](sample.json) till den xlsx-fil som genereras av koden som referens.
