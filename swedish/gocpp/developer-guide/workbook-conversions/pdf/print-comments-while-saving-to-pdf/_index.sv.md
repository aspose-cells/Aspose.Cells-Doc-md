{{% alert color="primary" %}}

Microsoft Excel tillåter dig att skriva ut kommentarer när du skriver ut eller sparar till PDF-format med följande alternativ:

- Ingen
- I slutet av bladet
- Enligt visad på kalkylbladet

Aspose.Cells tillhandahåller enum [**PrintCommentsType**](https://reference.aspose.com/cells/go-cpp/printcommentstype/) för att stödja samma funktion. Enum [**PrintCommentsType**](https://reference.aspose.com/cells/go-cpp/printcommentstype/) har följande medlemmar:

- PrintNoComments
- PrintInPlace
- PrintSheetEnd

{{% /alert %}}
