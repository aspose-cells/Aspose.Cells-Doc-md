{{% alert color="primary" %}} 

Som standard sparar Aspose.Cells Excel som PDF med standardstorlek. Du kan också spara den med minsta storlek med egenskapen [PdfSaveOptions.GetOptimizationType()](https://reference.aspose.com/cells/go-cpp/pdfsaveoptions/getoptimizationtype/). Den accepterar följande värden:

- PdfOptimizationType::Standard
- PdfOptimizationType::MinimumSize

{{% /alert %}}
