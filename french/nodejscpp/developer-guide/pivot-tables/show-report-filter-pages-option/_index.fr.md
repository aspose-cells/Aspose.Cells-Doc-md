## **Afficher l'option de pages de filtre de rapport**
Excel supporte la création de tableaux croisés dynamiques, l’ajout de filtres de rapport et l’activation de l’option "Afficher les pages de filtres de rapport". Aspose.Cells for Node.js via C++ supporte également cette fonctionnalité pour activer l’option "Afficher les pages de filtres de rapport" sur le tableau croisé dynamique créé. Voici une capture d’écran montrant l’option "Afficher les pages de filtres de rapport" dans Excel.
