Aspose.Cells for Node.js via C++ offre la possibilité d’enregistrer un tableau croisé dynamique en fichiers ODS. Pour cela, vous pouvez simplement convertir un classeur contenant un tableau croisé dynamique existant ou en créer un nouveau, puis sauvegarder le fichier au format ODS. Assurez-vous d’appeler [**PivotTable.calculateData**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#calculatedata--) avant de sauvegarder pour garantir que le tableau croisé dynamique est bien rendu dans le fichier ODS de sortie.
