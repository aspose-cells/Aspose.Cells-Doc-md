## **Comment obtenir la source de données de connexion externe de la table de données**

Aspose.Cells for Node.js via C++ permet d’obtenir la source de données de connexion externe du tableau croisé dynamique. Pour cela, l’API fournit la méthode [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) de la classe [**PivotTable**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/). La méthode [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) renvoie l’objet [**ExternalConnection**](https://reference.aspose.com/cells/nodejs-cpp/externalconnection/). Le code suivant montre comment utiliser la méthode [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) pour obtenir la source de données de connexion externe du tableau croisé dynamique.
