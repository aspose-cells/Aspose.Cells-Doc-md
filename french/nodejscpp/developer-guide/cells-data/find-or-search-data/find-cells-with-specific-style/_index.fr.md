{{% alert color="primary" %}}

Parfois, vous devez trouver des cellules avec un style particulier appliqué. Vous pouvez utiliser Aspose.Cells for Node.js via C++ pour trouver toutes les cellules avec un style commun. Aspose.Cells propose la méthode [**FindOptions.setStyle(Style)**](https://reference.aspose.com/cells/nodejs-cpp/findoptions/#setStyle-style-) que vous pouvez utiliser pour spécifier le style à rechercher dans les cellules.

{{% /alert %}}

Dans cet exemple, le code trouve toutes les cellules ayant le même style que celle de la cellule A1. Après l'exécution du code, toutes les cellules ayant le même style que A1 contiennent le texte "Trouvé".
