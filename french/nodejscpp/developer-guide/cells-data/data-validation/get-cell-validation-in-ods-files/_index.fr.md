## **Obtenir la validation de la cellule dans les fichiers ODS**  

Avec Aspose.Cells for Node.js via C++, vous pouvez obtenir la validation appliquée à une cellule dans les fichiers ODS. Pour cela, l'API fournit la méthode [**Cell.getValidation()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getValidation--) de la classe [**Cell**](https://reference.aspose.com/cells/nodejs-cpp/cell/).  

 L'exemple de code suivant démontre l'utilisation de la méthode [**Cell.getValidation()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getValidation--) en chargeant le fichier [source ODS](101089354.ods) et en lisant la validation de la cellule A9.
