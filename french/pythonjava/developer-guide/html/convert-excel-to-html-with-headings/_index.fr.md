## **Convertir Excel en HTML avec en-têtes**
Aspose.Cells offre la possibilité d'exporter les en-têtes de lignes et de colonnes lors de la conversion d'Excel en HTML. Ceci peut être réalisé en utilisant[HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings)propriété fournie par le API. La valeur par défaut de[HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) est**Faux** . Passe**Vrai** comme paramètre pour afficher les en-têtes dans le fichier de sortie HTML. L'image suivante montre le fichier de sortie généré par le code suivant.
