## **Convertir Excel en HTML avec des en-têtes**
Aspose.Cells propose l'option d'exporter les en-têtes de lignes et de colonnes lors de la conversion d'Excel en HTML. Cela peut être réalisé en utilisant la propriété [HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) fournie par l'API. La valeur par défaut de [HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) est **False**. Passez **True** en paramètre pour afficher les en-têtes dans le fichier HTML de sortie. L'image suivante montre le fichier généré par le code suivant.
