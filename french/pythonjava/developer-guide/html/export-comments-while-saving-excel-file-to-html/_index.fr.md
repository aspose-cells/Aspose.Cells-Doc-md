## **Exporter les commentaires lors de l'enregistrement d'un fichier Excel en HTML**
Lorsque Excel est converti en HTML, les commentaires ne sont pas exportés. Aspose.Cells pour Python via Java offre la possibilité d'exporter les commentaires lors de la conversion d'Excel en HTML. Pour ce faire, l'API fournit la propriété [HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments). En définissant la valeur de la propriété [HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments) à **True**, les commentaires seront exportés dans le fichier HTML de sortie.
