## **Exporter les commentaires lors de l'enregistrement du fichier Excel au format HTML**
Lorsque Excel est converti en HTML, les commentaires ne sont pas exportés. Aspose.Cells for Python via Java fournit la fonctionnalité permettant d'exporter des commentaires lors de la conversion d'Excel en HTML. Pour ce faire, le API fournit le[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments)propriété. Réglage de la valeur de[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments) propriété à**Vrai** exportera les commentaires dans le HTML de sortie.
