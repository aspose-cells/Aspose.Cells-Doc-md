## **Convertir Excel en SVG**
SVG(**Image Vectorielle)**est une spécification basée sur les normes XML pour les graphiques vectoriels bidimensionnels. Il s'agit d'un standard ouvert développé par le World Wide Web Consortium (W3C) depuis 1999.

Aspose.Cells for Python via Java prend en charge la conversion de fichiers Excel en images SVG. Pour cela, le API fournit le[FeuilleRendu](https://reference.aspose.com/cells/python/asposecells.api/SheetRender), [Options d'image ou d'impression](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions), et[ClasseurRender](https://reference.aspose.com/cells/python/asposecells.api/WorkbookRender)Des classes.
