## **Convertir un classeur Excel en PPT**

Pas besoin de se demander comment convertir Excel Workbook en PowerPoint, car la bibliothèque Aspose.Cells for Python via Java a la meilleure décision. Le Aspose.Cells for Python via Java API prend en charge la conversion des feuilles de calcul au format PowerPoint. Pour exporter le classeur vers PowerPoint, passez[**EnregistrerFormat.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) comme deuxième paramètre de[**Classeur.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) méthode. Vous pouvez également utiliser[**PptxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) classe pour spécifier des paramètres supplémentaires pour l'exportation de la feuille de calcul vers un fichier .pptx.
