## **Convertir le classeur Excel en PPT**

Pas besoin de se demander comment convertir un classeur Excel en PowerPoint, car la bibliothèque Aspose.Cells for Python via Java propose la meilleure solution. L'API Aspose.Cells for Python via Java prend en charge la conversion de feuilles de calcul au format PowerPoint. Pour exporter le classeur au format PowerPoint, passez [**SaveFormat.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) comme second paramètre de la méthode [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)). Vous pouvez également utiliser la classe [**PptxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) pour spécifier des paramètres supplémentaires pour exporter la feuille de calcul au format .pptx.
