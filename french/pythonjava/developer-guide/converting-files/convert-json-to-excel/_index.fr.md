## **Convertir JSON en classeur Excel**
Pas besoin de se demander comment convertir un fichier JSON en fichier Excel, car la bibliothèque Aspose.Cells pour Python via Java propose la meilleure solution. L'API Aspose.Cells pour Python via Java prend en charge la conversion du format JSON en feuilles de calcul. Vous pouvez utiliser la classe [**JsonLoadOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonLoadOptions) pour spécifier des paramètres supplémentaires pour l'importation du JSON dans le classeur.
