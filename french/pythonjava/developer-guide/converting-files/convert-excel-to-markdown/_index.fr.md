## **Convertir Excel en Markdown**
Aspose.Cells for Python via Java prend en charge la conversion de fichiers Excel au format Markdown. Pour exporter la feuille de calcul active au format Markdown, passez [SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN) comme deuxième paramètre de la méthode [Workbook.Save](https://reference.aspose.com/cells/python/asposecells.api/workbook#save(java.lang.String, %20int)). Vous pouvez également utiliser la classe [MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions) pour spécifier des paramètres supplémentaires pour exporter des feuilles de calcul au format Markdown.
