## **Convertir Excel en Markdown**
Aspose.Cells pour Python via Java prend en charge la conversion des fichiers Excel au format Markdown. Pour exporter la feuille de calcul active vers Markdown, passez[SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN)comme deuxième paramètre de la[Classeur.Enregistrer](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)) méthode. Vous pouvez également utiliser le[MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions)class pour spécifier des paramètres supplémentaires pour l'exportation de feuilles de calcul vers Markdown.

L'exemple de code suivant illustre l'exportation d'une feuille de calcul active vers Markdown à l'aide de[SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN)membre de l'énumération.
