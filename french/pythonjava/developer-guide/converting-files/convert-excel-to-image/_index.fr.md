## **Convertir Excel en Image**
Aspose.Cells for Python via Java prend en charge la conversion de fichiers Excel en différentes formes d'images. Pour cela, l'API fournit les classes [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender) et [ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions). Cette classe représente la feuille de calcul qui sera rendue sous forme d'image. La classe [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender) fournit la méthode [toImage()](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage(int, java.lang.String)) pour convertir une feuille de calcul en fichier image. Les formats BMP, PNG, JPEG, TIFF et EMF sont pris en charge.
