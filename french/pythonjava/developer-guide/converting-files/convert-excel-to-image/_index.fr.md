## **Convertir Excel en image**
Aspose.Cells for Python via Java prend en charge la conversion de fichiers Excel en différentes images de formulaires d'image. Pour cela, le API fournit le[FeuilleRendu](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)et[Options d'image ou d'impression](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions) Des classes. Cette classe représente la feuille de calcul qui sera rendue à une image. La[FeuilleRendu](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)la classe fournit la[toImage()](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage\(int,%20java.lang.String\)) méthode de conversion d'une feuille de calcul en fichier image. Les formats BMP, PNG, JPEG, TIFF et EMF sont pris en charge.
