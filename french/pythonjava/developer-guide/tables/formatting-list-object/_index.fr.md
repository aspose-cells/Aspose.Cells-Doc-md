## **Mise en forme d'un objet de liste**
Un tableau est une série de lignes et de colonnes qui contiennent des données associées gérées indépendamment des données des autres lignes et colonnes. Par défaut, chaque colonne du tableau a un filtrage activé dans la ligne d'en-tête afin que vous puissiez filtrer ou trier rapidement les données d'objet de votre liste. Vous pouvez ajouter une ligne de total (une ligne spéciale dans une liste qui fournit une sélection de fonctions d'agrégation utiles pour travailler avec des données numériques) à l'objet de liste qui fournit une liste déroulante de fonctions d'agrégation pour chaque cellule de ligne de total.

Aspose.Cells prend en charge le formatage des objets Liste. Pour cela, le API fournit le[ListObject](https://reference.aspose.com/cells/python/asposecells.api/ListObject) et[Type de style de table](https://reference.aspose.com/cells/python/asposecells.api/TableStyleType) Des classes. La[Type de style de table](https://reference.aspose.com/cells/python/asposecells.api/TableStyleType)La classe contient des constantes qui représentent les styles de tableau intégrés. L'extrait de code suivant crée un nouvel objet de liste et définit son type de style de tableau sur[TABLE_STYLE_MOYEN_10](https://reference.aspose.com/cells/python/asposecells.api/tablestyletype#TABLE_STYLE_MEDIUM_10)
