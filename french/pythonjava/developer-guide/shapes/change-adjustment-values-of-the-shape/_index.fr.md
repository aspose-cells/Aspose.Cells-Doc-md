{{% alert color="primary" %}}

Aspose.Cells pour Python via Java fournit la méthode [**Shape.getGeometry().getShapeAdjustValues()**](https://reference.aspose.com/cells/python-java/asposecells.api/geometry#ShapeAdjustValues) pour apporter des modifications aux points de réglage avec des formes. Dans l’interface utilisateur de Microsoft Excel, les ajustements s’affichent sous forme de nœuds en losange jaune. Par exemple :

- Le rectangle arrondi possède un ajustement pour changer l'arc.
- Le triangle a un ajustement pour changer l'emplacement du point.
- Le trapèze a un ajustement pour changer la largeur du haut
- Les flèches ont deux ajustements pour changer la forme de la tête et de la queue.

Cet article expliquera l'utilisation de la méthode [**Shape.getGeometry().getShapeAdjustValues()**](https://reference.aspose.com/cells/python-java/asposecells.api/geometry#ShapeAdjustValues) pour modifier la valeur de réglage des différentes formes.
