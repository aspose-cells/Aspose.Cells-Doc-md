## **Convertir du texte en colonnes**
Vous pouvez convertir votre texte en colonnes en utilisant Microsoft Excel. Cette fonctionnalité est disponible à partir de*Outils de données*sous le*Données*languette. Afin de diviser le contenu d'une colonne en plusieurs colonnes, les données doivent contenir un délimiteur spécifique tel qu'une virgule (ou tout autre caractère) sur la base duquel Microsoft Excel divise le contenu d'une cellule en plusieurs cellules. Aspose.Cells fournit également cette fonctionnalité via le[TexteVersColonnes](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) méthode. L'extrait de code suivant illustre l'utilisation de[TexteVersColonnes](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) en convertissant le texte en colonnes avec un espace comme délimiteur.
