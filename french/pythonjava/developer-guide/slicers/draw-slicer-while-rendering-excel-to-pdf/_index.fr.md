## **Dessiner un tronçonneur lors du rendu Excel en PDF**
Aspose.Cells pour Python via Java prend en charge l'exportation d'Excel en PDF avec les paramètres du segment de tarte. L'API prend en charge cela par défaut, vous exportez simplement le fichier Excel avec le segment de tarte en PDF, le PDF généré affichera le segment de tarte appliqué. Le code suivant charge le *fichier Excel d'exemple*(106365058.xlsx) qui contient un segment de tarte et enregistre le classeur au format *fichier PDF de sortie*(106365052.xlsx). La capture d'écran suivante compare le fichier Excel source et le fichier PDF généré.
