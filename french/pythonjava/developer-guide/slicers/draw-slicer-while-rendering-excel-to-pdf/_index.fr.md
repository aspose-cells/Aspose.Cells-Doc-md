## **Dessiner Slicer lors du rendu d'Excel au format PDF**
Aspose.Cells pour Python via Java prend en charge l'exportation d'Excel au format PDF avec les paramètres du segment. Le API le supporte par défaut, il vous suffit d'exporter le fichier Excel avec slicer au format PDF, le PDF généré montrera le slicer appliqué. L'extrait de code suivant charge le[exemple de fichier Excel](106365058.xlsx)qui contient un segment et enregistre le classeur en tant que[fichier PDF de sortie](106365052.xlsx). La capture d'écran suivante compare le fichier Excel source et le fichier PDF généré.
