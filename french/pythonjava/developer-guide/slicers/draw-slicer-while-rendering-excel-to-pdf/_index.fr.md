## **Dessinez Slicer tout en rendant Excel à PDF**
Aspose.Cells for Python via Java prend en charge l'exportation d'Excel vers PDF avec les paramètres du segment. Le API le supporte par défaut, il vous suffit d'exporter le fichier Excel avec slicer vers PDF, le PDF généré affichera le slicer appliqué. L'extrait de code suivant charge le[exemple de fichier Excel](106365058.xlsx)qui contient un segment et enregistre le classeur en tant que[fichier de sortie PDF](106365052.xlsx). La capture d'écran suivante compare le fichier Excel source et le fichier PDF généré.
