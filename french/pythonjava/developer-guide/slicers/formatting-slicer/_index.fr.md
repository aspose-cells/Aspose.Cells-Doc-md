## **Formatage du segment**
En utilisant Microsoft Excel, vous pouvez formater le segment en définissant son nombre de colonnes, de styles, etc. Aspose.Cells pour Python via Java fournit les propriétés Slicer.NumberOfColumns et Slicer.StyleType pour y parvenir.

L'extrait de code suivant charge le[exemple de fichier Excel](106364964.xlsx)qui contient un trancheur. Il accède au segment et définit son nombre de colonnes et son type de style et l'enregistre sous[fichier Excel de sortie](106364965.xlsx). La capture d'écran montre à quoi ressemble le segment après l'exécution de l'exemple de code.
