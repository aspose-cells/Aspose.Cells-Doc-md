## **Formatage d'un tronçonneur**
Avec Microsoft Excel, vous pouvez formater le filtre en définissant son nombre de colonnes, ses styles, etc. Aspose.Cells for Python via Java fournit les propriétés Slicer.NumberOfColumns et Slicer.StyleType pour y parvenir.

Le code suivant charge le [fichier Excel exemple](106364964.xlsx) qui contient un filtre. Il accède au filtre, définit son nombre de colonnes et son type de style, puis l'enregistre sous le nom de [fichier Excel de sortie](106364965.xlsx). La capture d'écran montre à quoi ressemble le filtre après l'exécution du code exemple.
