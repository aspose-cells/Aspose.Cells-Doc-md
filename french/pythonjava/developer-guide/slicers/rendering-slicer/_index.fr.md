## **Rendu du filtre**
Aspose.Cells pour Python via Java prend en charge le rendu de la forme du filtre. Le code suivant charge le [fichier Excel exemple](106364974.xlsx) qui contient un filtre. Il convertit la feuille de calcul en une image en définissant la zone d'impression qui ne couvre que le filtre. L'image résultante est la [image de sortie](106364975.png) qui montre le filtre rendu. Comme vous pouvez le voir, le filtre a été correctement rendu, il ressemble au même dans le fichier Excel exemple.
