## **Retrait du trancheur**
 Pour supprimer le trancheur dans Microsoft Excel, il vous suffit de sélectionner le trancheur et d'appuyer sur le*Effacer* bouton. Pour réaliser la sauvegarde en utilisant Aspose.Cells for Python via Java, utilisez la méthode Worksheet.getSlicers().remove(). Cela supprimera le segment de la feuille de calcul.

L'extrait de code suivant charge le[exemple de fichier Excel](106364970.xlsx)qui contient un segment existant. Il accède au slicer, le supprime et enregistre le[fichier Excel de sortie](106364971.xlsx). La capture d'écran suivante montre le segment qui sera supprimé.
