**Bienvenue au Aspose.Cells for Python via Java**

Aspose.Cells for Python via Java permet à vos applications d'effectuer une large gamme de tâches de traitement de feuille de calcul. Aspose.Cells for Python via Java prend en charge les formats de fichier de feuille de calcul populaires (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) que votre entreprise utilise au quotidien. Il permet également d'exporter des fichiers Excel® vers PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, texte brut et formats d'image populaires, notamment TIFF, JPG, PNG, BMP et SVG.
