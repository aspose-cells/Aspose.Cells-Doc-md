## **Configuration requise**
Aspose.Cells for Python via Java est une API indépendante de la plateforme et peut être utilisée sur n'importe quelle plateforme (Windows, Linux et MacOS) où [Python](https://www.python.org/downloads/) est installé. La machine doit avoir Java 8 ou des versions supérieures avant de procéder à l'installation.

## **Version de Python**
- Python 3.5 ou version ultérieure
## **Version Java**
- Java 1.8 ou version ultérieure

## **Installation**
### **Windows:**
- **Installez Java et définissez les variables d'environnement JAVA_HOME et PATH**
Par exemple:
{{< highlight java >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}

- **Installez Aspose.Cells pour Python via Java à partir de pypi**
Vous pouvez facilement utiliser Aspose.Cells pour Python via Java à partir de [pypi](https://pypi.org/project/aspose-cells/) avec la commande suivante.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}
