|**Format**|**Description**|**Charger**|**Sauver**|
|:- |:- |:- |:- |
|[XLS](https://docs.fileformat.com/spreadsheet/xls/)|Enregistre le document dans le classeur Excel Microsoft 97-2003.||{{< emoticons/tick >}}|
|[XLSX](https://docs.fileformat.com/spreadsheet/xlsx/)|Enregistre le document dans Office Open XML SpreadsheetML Classeur ou fichier de modèle, avec ou sans macros.||{{< emoticons/tick >}}|
|[ODS](https://docs.fileformat.com/spreadsheet/ods/)|Enregistre le document au format ODS (feuille de calcul OpenDocument).||{{< emoticons/tick >}}|
|[XML](https://docs.fileformat.com/web/xml/)|Langage de balisage extensible||{{< emoticons/tick >}}|
|[PDF](https://docs.fileformat.com/pdf/)|Format de document portable Adobe.||{{< emoticons/tick >}}|
|[CSV](https://docs.fileformat.com/spreadsheet/csv/)|Enregistre le document dans le fichier CSV (Comma Separated Value).||{{< emoticons/tick >}}|
|[TSV](https://docs.fileformat.com/spreadsheet/tsv/)|Enregistre le document dans le fichier TSV (valeurs séparées par des tabulations).||{{< emoticons/tick >}}|
|TabDelimited|Enregistre le document dans un fichier texte délimité par des tabulations, identique au fichier TSV.||{{< emoticons/tick >}}|
