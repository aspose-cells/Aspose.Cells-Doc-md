 Veuillez utiliser[PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) méthode pour créer un tableau croisé dynamique à l’intérieur de la feuille de calcul.
