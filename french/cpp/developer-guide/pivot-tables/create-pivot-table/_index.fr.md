## **Introduction**
Veuillez utiliser la méthode [PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) pour créer un tableau croisé dynamique dans la feuille de calcul. Une fois le tableau croisé dynamique créé, vous pouvez travailler avec lui en utilisant la classe [PivotTable](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/).
