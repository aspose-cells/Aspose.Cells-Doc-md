## **Le cryptage et le décryptage ne sont pas pris en charge pour Aspose.Cells pour C++ maintenant**
Maintenant, le cryptage et le décryptage ne sont pas pris en charge pour Aspose.Cells pour C++, donc si vous voulez ouvrir un fichier avec un mot de passe, le code lèvera une exception.
