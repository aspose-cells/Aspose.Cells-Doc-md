**Bienvenue dans la section documentation Aspose.Cells pour C++**

Aspose.Cells pour C++ est une bibliothèque native C++ qui permet aux développeurs C++ de créer, manipuler et convertir des feuilles de calcul par programmation sans nécessiter Office Automation ou Microsoft application Excel.

Le Aspose.Cells pour C++ prend en charge les formats de feuille de calcul tels que le format Excel 97-2003 Workbook et les formats Open Office XML.

{{% /alert %}}

## **Avantages**

L'utilisation de Aspose.Cells pour C++ dans vos applications vous offre les avantages suivants :

- Fonctionnalités riches
- Indépendance de la plateforme
- Performances et évolutivité

### **Fonctionnalités riches**

Aspose.Cells pour C++ offre un large éventail de fonctionnalités pour créer, convertir et manipuler des feuilles de calcul :

- Propriétés intégrées
- Propriétés personnalisées
- Thèmes
- Styles et formatage
- La validation des données
- Mise en forme conditionnelle
- Lien hypertexte
- Filtre automatique
- Mise en page
- Lire, écrire et calculer des formules
- Regrouper les lignes et les colonnes
- Tableau croisé dynamique
- Table

### **Indépendance de la plate-forme**

Utilisez Aspose.Cells pour C++ pour créer tout type d'application C++ 32 bits et 64 bits.

### **Performances et évolutivité**

Aspose.Cells pour C++ est conçu pour fonctionner aussi bien côté serveur que côté client. Aspose.Cells pour C++ est un assembly natif qui peut être déployé en le copiant simplement. Vous n'avez pas à vous soucier d'autres services ou modules.

## **Soutien technique**

Aspose fournit un support technique gratuit et illimité pour tous ses produits. L'assistance est disponible pour tous les utilisateurs, y compris l'évaluation.

 Si vous avez besoin d'aide avec Aspose.Cells pour C++[visitez le Aspose.Forums](https://forum.aspose.com/c/cells/9). Postez votre question dans le forum Aspose.Cells pour C++ et elle sera répondue dans quelques heures. Les questions sont répondues directement par l'équipe d'assistance Aspose.Cells pour C++.

Lorsque vous attendez une réponse dans les forums, veuillez tenir compte des différences de fuseau horaire.

Si vous avez un problème avec Aspose.Cells pour C++, suivez ces étapes simples pour vous assurer qu'il est résolu de la manière la plus efficace :

- Assurez-vous d'utiliser la dernière version Aspose.Cells pour C++ avant de signaler le problème.
- Lorsque vous signalez un problème, veuillez inclure le document original et éventuellement un fragment du code à l'origine du problème. Si vous devez joindre plusieurs fichiers, compressez-les en un seul. Il est sécuritaire de joindre vos documents au Aspose.Forums puisque seuls vous et les développeurs du Aspose aurez accès aux fichiers joints.
- Veuillez essayer de signaler un problème par sujet. Si vous avez un autre problème, signalez-le dans un fil séparé.
