{{% alert color="primary" %}} 

- Si vous avez une version précédente de Aspose.Cells for Reporting Services installée, veuillez d'abord la désinstaller. Voir le sujet [Désinstallation de Aspose.Cells for Reporting Services](/cells/fr/reportingservices/desinstallation-aspose-cells-for-reporting-services/) sur la façon de supprimer le composant.
- [Visitez la page de téléchargement de Aspose.Cells for Reporting Services](https://downloads.aspose.com/cells/reportingservices) et téléchargez l'installateur MSI Aspose.Cells for Reporting Services avec la dernière version du composant.
    - L'installateur Msi Aspose.Cells for ReportingServices ne prend en charge que Microsoft SQL Server 2008 Reporting Services et les produits ultérieurs.
- Exécutez le fichier téléchargé et suivez les instructions de l'installateur.

{{% /alert %}} 
**Processus d'installation**
Pour installer Aspose.Cells en utilisant l'installateur MSI :

1. L'écran de bienvenue est le premier écran de l'installation. Cliquez sur **Suivant**.

![todo:image_alt_text](using-msi-installer_1.png)




1. Lisez et acceptez le contrat de licence, puis cliquez sur **Suivant**.

![todo:image_alt_text](using-msi-installer_2.png)




1. Sélectionnez le dossier d'installation, puis cliquez sur **Suivant**. 

![todo:image_alt_text](using-msi-installer_3.png)




1. Confirmez que vous souhaitez installer Aspose.Cells for Reporting Services en cliquant sur **Installer**. 

![todo:image_alt_text](using-msi-installer_4.png)



La progression de l'installation est affichée à l'écran. 

![todo:image_alt_text](using-msi-installer_5.png)




1. Une fois l'installation terminée, cliquez sur **Terminer** pour fermer l'installateur.

![todo:image_alt_text](using-msi-installer_6.png)
