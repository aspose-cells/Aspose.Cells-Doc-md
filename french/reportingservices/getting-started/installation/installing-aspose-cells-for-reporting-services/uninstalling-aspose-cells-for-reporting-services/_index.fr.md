 Si vous avez utilisé le programme d'installation MSI pour déployer Aspose.Cells for Reporting Services, suivez ces étapes pour supprimer le composant, avec les démos et la documentation associées.

1.  Du**Démarrer** menu, sélectionnez**Réglages** et puis**Panneau de commande**.
1.  Cliquez sur**Ajouter/supprimer des programmes**.
1.  Sélectionner**Aspose.Cells.ReportingServices**.
1.  Cliquez sur**Modifier/Supprimer** pour supprimer Aspose.Cells.ReportingServices.
