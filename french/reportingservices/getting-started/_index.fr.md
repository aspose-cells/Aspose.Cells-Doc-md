Aspose.Cells for Reporting Services est composé de **deux parties** comme suit :

- ***Aspose.Cells.Report.Designer***

&nbsp;&nbsp;&nbsp;&nbsp;Il s'agit d'un outil de conception de rapport intégré à MS Excel dans le formulaire des compléments… Avec Aspose.Cells.Report.Designer, les utilisateurs peuvent concevoir un modèle de rapport dans MS Excel au lieu de Visual Studio. Le modèle de rapport est ensuite incorporé dans un fichier RDL de Reporting Services et peut être publié sur le serveur de rapports. Les utilisateurs peuvent également prévisualiser et exporter le rapport dans Aspose.Cells.Report.Designer.

- ***Aspose.Cells.Renderer for Reporting Services***

&nbsp;&nbsp;&nbsp;&nbsp;Aspose.Cells.Renderer for Reporting Services est en fait une implémentation de l'extension de rendu de Microsoft Reporting Services. Il peut générer des rapports RDL conçus à la fois par Visual Studio et Aspose.Cells.Report.Designer.
