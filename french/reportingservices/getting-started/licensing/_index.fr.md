{{% alert color="primary" %}} 

 Aspose.Cells for Reporting Services est disponible en tant qu'évaluation gratuite et illimitée dans le temps auprès du[Page de téléchargement](https://downloads.aspose.com/cells/reportingservices). Les versions d'évaluation et sous licence du produit sont les mêmes.

Lorsque vous êtes satisfait de la version d'évaluation, achetez une licence. Assurez-vous de comprendre et d'accepter les conditions générales d'abonnement.

{{% /alert %}} 
### **Obtention et utilisation de la licence**
La licence sera disponible en téléchargement à partir de la page de commande une fois la commande payée.

Le fichier de licence est un fichier XML en texte clair signé numériquement. Le fichier de licence contient des informations telles que le nom du client, le produit acheté et le type de licence. Ne modifiez pas le contenu du fichier de licence car cela l'invaliderait.

Téléchargez la licence sur votre ordinateur et copiez-la dans le dossier qui doit être l'un des dossiers suivants.

- $[Dossier d'installation]\Licence\
- $[MSSQL Server Reporting Services]\ReportServer\Bin\
- $[MSSQL Server Reporting Services]\ReportServer\

Le nom du fichier de licence peut être l'un des suivants en fonction de votre type d'abonnement :

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic", "Aspose.Total.lic
- Aspose.Cells.Reporting.Services - Abonnement au site du développeur.lic
- Aspose.Cells.Reporting.Services - Developer Subscription.lic
- Aspose.Cells.Reporting.Services - Multiple Deployment Subscription.lic
- Aspose.Cells.Reporting.Services - Abonnement de déploiement de serveur unique.lic

Pour vérifier que le fichier de licence a été installé correctement, exportez n'importe quel rapport en tant que classeur Excel Microsoft. Si le classeur ne contient aucune feuille de calcul en filigrane à la fin, le fichier de licence a été activé avec succès.

 Aspose.Cells for Reporting Services injecte une feuille de calcul de filigrane d'évaluation lorsque vous travaillez en mode d'évaluation. Lorsqu'un fichier de licence valide, par exemple lorsque**Aspose.Cells.ReportingServices.lic** se trouve dans le dossier $[Installation Folder]\License\, il n'y a pas de feuille de calcul de filigrane d'évaluation.
