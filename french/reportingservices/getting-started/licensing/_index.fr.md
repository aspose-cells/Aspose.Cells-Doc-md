{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services est disponible en téléchargement gratuit, sans limite de temps, sur la [page de téléchargement](https://downloads.aspose.com/cells/reportingservices). Les versions d'évaluation et sous licence du produit sont identiques.

Lorsque vous êtes satisfait de la version d'évaluation, achetez une licence. Assurez-vous de comprendre et d'accepter les modalités et conditions d'abonnement.

{{% /alert %}} 
### **Obtention et utilisation de la licence**
La licence sera disponible en téléchargement depuis la page de commande une fois la commande payée.

Le fichier de licence est un fichier XML en clair, signé numériquement. Le fichier de licence contient des informations telles que le nom du client, le produit acheté et le type de licence. Ne modifiez pas le contenu du fichier de licence, car cela l'invalidite.

Téléchargez la licence sur votre ordinateur et copiez-la dans le dossier qui doit être l'un des dossiers suivants.

- $[Dossier d'installation]\Licence\
- $[MSSQL Server Reporting Services]\ReportServer\Bin\
- $[MSSQL Server Reporting Services]\ReportServer\

Le nom du fichier de licence peut être l'un des suivants en fonction de votre type d'abonnement :

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic", "Aspose.Total.lic
- Aspose.Cells.Reporting.Services - Abonnement au site du développeur.lic
- Aspose.Cells.Reporting.Services - Abonnement développeur.lic
- Aspose.Cells.Reporting.Services - Abonnement déploiement multiple.lic
- Aspose.Cells.Reporting.Services - Abonnement déploiement sur un seul serveur.lic

Pour tester si le fichier de licence a été correctement installé, exportez un rapport sous forme de classeur Microsoft Excel. Si le classeur ne contient pas de feuille de filigrane à la fin, le fichier de licence a été activé avec succès. 

Aspose.Cells for Reporting Services injecte une feuille de filigrane d'évaluation lorsqu'il fonctionne en mode d'évaluation. Lorsqu'un fichier de licence valide, par exemple **Aspose.Cells.ReportingServices.lic**, se trouve dans le dossier $[Dossier d'installation]\Licence\, aucune feuille de filigrane d'évaluation n'apparaît.
