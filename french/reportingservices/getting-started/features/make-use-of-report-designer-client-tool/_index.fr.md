{{% alert color="primary" %}} 

Les utilisateurs peuvent concevoir et fournir des rapports dans Microsoft Excel directement à l'aide de Aspose.Cells.Report.Designer, un outil simple et facile à utiliser. Ils n'ont pas besoin d'utiliser Visual Studio pour créer des rapports, mais peuvent économiser le coût d'achat de licences Visual Studio supplémentaires et n'ont pas besoin de passer du temps à apprendre à utiliser un environnement de développement complexe.

Étant donné que Aspose.Cells.Report.Designer utilise Microsoft Excel comme modèle de rapport, les utilisateurs peuvent profiter de toutes les fonctionnalités et fonctionnalités puissantes de Microsoft Excel, y compris les tableaux croisés dynamiques, les graphiques interactifs, VBA, les macros et les formules. Toutes ces fonctionnalités ne sont pas prises en charge par le format Excel standard Microsoft fourni par Reporting Services.
