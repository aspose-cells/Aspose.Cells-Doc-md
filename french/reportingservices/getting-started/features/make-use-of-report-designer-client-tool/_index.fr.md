{{% alert color="primary" %}} 

Les utilisateurs peuvent concevoir et fournir des rapports directement dans Microsoft Excel en utilisant Aspose.Cells.Report.Designer, un outil simple et facile à utiliser. Ils n'ont pas besoin d'utiliser Visual Studio pour créer des rapports mais peuvent économiser le coût d'achat de licences supplémentaires pour Visual Studio et ne pas avoir à passer du temps à apprendre à utiliser un environnement de développement complexe.

Comme Aspose.Cells.Report.Designer utilise Microsoft Excel comme modèle de rapport, les utilisateurs peuvent tirer parti de toutes les fonctionnalités puissantes de Microsoft Excel, y compris les tableaux croisés dynamiques, les graphiques interactifs, VBA, les macros et les formules. Toutes ces fonctionnalités ne sont pas prises en charge par le format standard Microsoft Excel que Reporting Services délivrent.

{{% /alert %}}
