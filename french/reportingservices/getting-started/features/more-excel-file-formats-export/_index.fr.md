{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services vous permet de distribuer des rapports natifs Microsoft Excel grâce aux services de rapports Microsoft SQL Server 2005 et 2008. Contrairement au moteur intégré pour les rapports basés sur Microsoft Excel, Aspose.Cells for Reporting Services fournit un ensemble de fonctionnalités étendu qui convertit les rapports RDL en : format binaire XLS Excel 97-2003, CSV, SpreadsheetML, format XLSX Excel 2007, XLSM Excel 2007, XLSB Excel 2007, HTML et ODS. 

{{% /alert %}} 

Aspose.Cells for Reporting Services prend en charge plus de formats de fichier que les exportateurs natifs. Ci-dessous, des captures d'écran de rapports XLS d'exemple exportés avec Aspose.Cells.
