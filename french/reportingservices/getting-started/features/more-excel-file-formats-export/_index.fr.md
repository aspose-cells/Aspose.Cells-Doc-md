{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services vous permet de fournir des rapports Excel natifs Microsoft via Microsoft SQL Server 2005 et 2008 Reporting Services. Contrairement au rendu intégré pour Microsoft Rapports basés sur Excel, Aspose.Cells for Reporting Services fournit un ensemble de fonctionnalités étendu qui convertit les rapports RDL à: Excel 97-2003 binaire XLS Format, CSV, SpreadsheetML, Excel 2007 XLSX, Exccel, excul Formats HTML et ODS.

{{% /alert %}}
