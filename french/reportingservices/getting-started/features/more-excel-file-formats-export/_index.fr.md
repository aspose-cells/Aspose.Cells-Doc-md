{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services vous permet de fournir des rapports Excel natifs Microsoft via Microsoft SQL Server 2005 et 2008 Reporting Services. Contrairement au moteur de rendu intégré pour les rapports basés sur Excel Microsoft, Aspose.Cells for Reporting Services fournit un ensemble de fonctionnalités étendu qui convertit les rapports RDL au format XLS binaire Excel 97-2003, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 XLSB, Format HTML et ODS.

{{% /alert %}} 

Aspose.Cells for Reporting Services prend en charge plus de formats de fichiers que les exportateurs natifs. Ci-dessous, des captures d'écran d'exemples de rapports XLS exportés avec Aspose.Cells.
