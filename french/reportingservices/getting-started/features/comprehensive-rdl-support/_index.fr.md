{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services prend entièrement en charge la spécification RDL. Cela signifie que vous pouvez facilement transférer les rapports RDL existants vers une solution Aspose.

{{% /alert %}} 
**Éléments RDL pris en charge**
 Aspose.Cells prend en charge les éléments RDL suivants :

- Page, en-têtes, pieds de page
- Zones de texte
- Images
- Graphiques
- Listes
- les tables
- Matrices
- modes
- Rectangles, lignes
- Sous-rapports
- Tableau matriciel
- Jauge
