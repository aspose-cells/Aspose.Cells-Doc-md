**Bienvenue dans la Documentation Aspose.Cells for Reporting Services !**
Aspose.Cells est un composant de feuille de calcul primé qui permet aux développeurs d'intégrer la possibilité de lire, d'écrire et de manipuler des feuilles de calcul Excel dans leurs propres applications sans compter sur Microsoft Excel.


**Présentation du produit**
Aspose.Cells for Reporting Services est une solution pour les développeurs qui ont besoin de fournir des rapports Excel Microsoft dans Reporting Services. Il ajoute des fonctionnalités flexibles et conviviales aux services de rapport via l'outil Aspose.Cells.Report.Designer. Toutes les tâches liées au développement de rapports dans Reporting Services - telles que la configuration de sources de données, la création de requêtes, la conception de rapports et la publication de rapports sur Report Server - peuvent être effectuées. Étant donné que Aspose.Cells.Report.Designer utilise Microsoft Excel comme modèle de rapport, les utilisateurs peuvent tirer parti des fonctionnalités puissantes disponibles dans Microsoft Excel.

Il existe deux façons de concevoir des rapports à l'aide du composant Aspose.Cells for Reporting Services. Commencez avec un modèle Excel Microsoft vide, puis effectuez des tâches, de la configuration des sources de données à la publication de rapports sur Report Server. Ou commencez avec un rapport RDL existant dans lequel les requêtes de base de données sont disponibles. En empruntant cette voie, les utilisateurs doivent se concentrer uniquement sur la conception du rapport. Cette méthode est utile pour les utilisateurs qui souhaitent séparer la conception de rapport de la création de requêtes. Cette dernière est relativement complexe et doit être effectuée par des utilisateurs ayant des compétences en base de données et en SQL.

Aspose.Cells for Reporting Services est composé de deux parties.
- **Aspose.Cells.Report.Designer**
Il s'agit d'un outil de conception de rapports intégré à Microsoft Excel sous forme de compléments. Avec Aspose.Cells.Report.Designer, les utilisateurs peuvent concevoir un modèle de rapport dans Microsoft Excel au lieu de Visual Studio. Le modèle de rapport est ensuite intégré dans un fichier RDL Reporting Services et peut être publié sur Report Server. Les utilisateurs peuvent également prévisualiser et exporter le rapport dans Aspose.Cells.Report.Designer.
- **Aspose.Cells.Renderer for Reporting Services**
Aspose.Cells.Renderer for Reporting Services est une implémentation de l'extension de rendu Reporting Services de Microsoft. Il peut afficher des rapports RDL conçus à la fois par Visual Studio et Aspose.Cells.Report.Designer.
