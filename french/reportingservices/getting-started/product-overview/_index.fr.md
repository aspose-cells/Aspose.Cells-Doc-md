**Bienvenue dans la documentation Aspose.Cells for Reporting Services!**
Aspose.Cells est un composant de feuille de calcul primé qui permet aux développeurs d'intégrer la capacité de lire, écrire et manipuler des feuilles de calcul Excel dans leurs propres applications sans compter sur Microsoft Excel.


**Aperçu du produit**
Aspose.Cells for Reporting Services est une solution pour les développeurs qui ont besoin de fournir des rapports Microsoft Excel dans Reporting Services. Il ajoute des fonctionnalités flexibles et conviviales aux services de reporting grâce à l'outil Aspose.Cells.Report.Designer. Toutes les tâches liées au développement de rapports dans Reporting Services - comme la configuration des sources de données, la création de requêtes, la conception de rapports et la publication de rapports sur le serveur de rapports - peuvent être effectuées. Comme Aspose.Cells.Report.Designer utilise Microsoft Excel comme modèle de rapport, les utilisateurs peuvent tirer parti des puissantes fonctionnalités disponibles dans Microsoft Excel.

Il y a deux façons de concevoir des rapports en utilisant le composant Aspose.Cells for Reporting Services. Commencez avec un modèle Microsoft Excel vide, puis complétez les tâches, de la configuration des sources de données à la publication des rapports sur le serveur de rapports. Ou commencez par un rapport RDL existant dans lequel les requêtes de base de données sont disponibles pour une utilisation. En empruntant cette voie, les utilisateurs doivent se concentrer uniquement sur la conception du rapport. Cette méthode est utile pour les utilisateurs qui souhaitent séparer la conception du rapport de la création de la requête. Cette dernière est relativement complexe et doit être effectuée par les utilisateurs possédant des compétences en bases de données et en SQL.

Aspose.Cells for Reporting Services est composé de deux parties.
- **Aspose.Cells.Report.Designer**
Il s'agit d'un outil de conception de rapport intégré à Microsoft Excel sous forme de compléments. Avec Aspose.Cells.Report.Designer, les utilisateurs peuvent concevoir un modèle de rapport dans Microsoft Excel au lieu de Visual Studio. Le modèle de rapport est ensuite intégré dans un fichier RDL des services de reporting et peut être publié sur le serveur de rapports. Les utilisateurs peuvent également prévisualiser et exporter le rapport dans Aspose.Cells.Report.Designer.
- **Aspose.Cells.Renderer for Reporting Services**
Aspose.Cells.Renderer pour les services de reporting est une implémentation de l'extension de rendu des services de reporting de Microsoft. Il peut rendre les rapports RDL conçus à la fois par Visual Studio et Aspose.Cells.Report.Designer.
