{{% alert color="primary" %}}

 Vous pouvez facilement télécharger Aspose.Cells for Reporting Services pour évaluation à partir du[page de téléchargement](https://downloads.aspose.com/cells/reportingservices) . Le téléchargement d'évaluation est identique au téléchargement acheté. La version d'évaluation devient simplement sous licence après[demander une licence](/cells/fr/reportingservices/licensing/).

{{% /alert %}}

La version d'évaluation de Aspose.Cells for Reporting Services (sans licence spécifiée) fournit des fonctionnalités du produit dans une certaine mesure non complètes. Une limitation est qu'il injecte toujours une feuille de calcul supplémentaire avec un filigrane d'évaluation lors de l'ouverture/de l'enregistrement de tout document de feuille de calcul.

Il injectera une feuille de calcul de filigrane d'évaluation supplémentaire dans le classeur.

![tâche : image_autre_texte](evaluate-aspose-cells_2.png)

**Chiffre:** Avertissement d'évaluation - Une feuille de travail supplémentaire

 Si vous souhaitez tester Aspose.Cells for Reporting Services sans limitation de version d'évaluation, vous pouvez[demander une licence temporaire de 30 jours](https://purchase.aspose.com/temporary-license).
