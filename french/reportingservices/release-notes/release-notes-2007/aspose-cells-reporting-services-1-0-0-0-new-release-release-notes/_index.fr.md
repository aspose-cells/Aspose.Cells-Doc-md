 Introduction

Aspose.Cells pour Reporting Services vous permet de fournir des rapports Excel natifs dans Microsoft SQL Server 2005 Reporting Services. Contrairement au moteur de rendu intégré pour les rapports basés sur Excel, Aspose.Cells pour Reporting Services fournit un ensemble de fonctionnalités étendu et permet de convertir les rapports RDL au format XLS binaire Excel 97-2003, SpreadsheetML ou au format Excel2007 XLSX.
