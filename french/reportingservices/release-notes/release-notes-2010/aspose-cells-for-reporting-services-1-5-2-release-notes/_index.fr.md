 Nous sommes heureux d'annoncer Aspose.Cells for Reporting Services 1.5.2 !

 Qu'est-ce qui a changé ?



-  Optimise le rendu pour les rapports de données volumineux.
- Résout un problème d'installation pour Windows 7.
-  Résout la pagination pour l'élément de rapport Tablix.
-  Gère le rendu de plusieurs groupes pour Tablix.
-  Réduit le taux d'utilisation de la mémoire.
