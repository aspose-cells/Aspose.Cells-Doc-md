{{% alert color="primary" %}} 

La boîte de dialogue "Dyn-Hide" vous permet de masquer dynamiquement des informations dans un rapport. Il vous permet de définir une plage de cellules sur une feuille de calcul que vous souhaitez masquer.

{{% /alert %}} 

Pour masquer des informations :

1. Dans Microsoft Excel, cliquez sur **Dyn-Hide** dans la barre de commandes Aspose.Cells.
   La boîte de dialogue Dyn-Hide s'affiche. 

![todo:image_alt_text](working-with-dyn-hide-form_1.png)




1. Modifier les informations masquées.
1. Cliquez sur **Ajouter** pour ajouter des informations masquées. 

![todo:image_alt_text](working-with-dyn-hide-form_2.png)



La zone définie est masquée dans le rapport.

Pour supprimer des informations masquées :

1. Sélectionnez une feuille dans la liste des zones masquées dans la boîte de dialogue Dyn-Hide.
1. Cliquez sur **Supprimer**.
   La zone sélectionnée est supprimée et s'affiche dans le rapport.
