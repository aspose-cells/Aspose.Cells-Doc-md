{{% alert color="primary" %}} 

La boîte de dialogue "Dyn-Hide" vous permet de masquer dynamiquement les informations d'un rapport. Il vous permet de définir une plage de cellules sur une feuille de calcul que vous souhaitez masquer.

{{% /alert %}} 

Pour masquer des informations :

1.  Dans Microsoft Excel, cliquez sur**Dyn-Masquer** sur la barre de commande Aspose.Cells.
 La boîte de dialogue Dyn-Hide s'affiche.

![tâche : image_autre_texte](working-with-dyn-hide-form_1.png)




1. Modifiez les informations de masquage.
1.  Cliquez sur**Ajouter** pour ajouter des informations masquées.

![tâche : image_autre_texte](working-with-dyn-hide-form_2.png)



La zone définie est masquée dans le rapport.

Pour supprimer les informations masquées :

1. Sélectionnez une feuille dans la liste des zones masquées de la boîte de dialogue Dyn-Hide.
1.  Cliquez sur**Effacer**.
 La zone sélectionnée est supprimée et s'affiche dans le rapport.
