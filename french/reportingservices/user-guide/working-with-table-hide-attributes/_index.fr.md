1. Dans Microsoft Excel, cliquez sur **Modifier l'attribut** dans la barre de commande Aspose.Cells.
   La boîte de dialogue Modifier le rapport apparaît.
1. Sélectionnez un élément de rapport de tableau.
1. Cliquez sur l'onglet **Masquer**. 

![todo:image_alt_text](working-with-table-hide-attributes_1.png)




1. Cliquez sur **Modifier** pour rendre l'onglet modifiable. 

![todo:image_alt_text](working-with-table-hide-attributes_2.png)




1. Modifier les informations masquées.
1. Cliquez sur **Fx**.
   La boîte de dialogue **Modifier la formule** est affichée.

![todo:image_alt_text](working-with-table-hide-attributes_3.png)





1. Modifiez la valeur de l'expression.
1. Cliquez sur **OK** pour enregistrer la formule dans la boîte de dialogue Modifier le rapport. 

![todo:image_alt_text](working-with-table-hide-attributes_4.png)




1. Modifiez toute autre information masquée.
1. Cliquez sur **Ajouter** pour ajouter l'information masquée. 

![todo:image_alt_text](working-with-table-hide-attributes_5.png)




1. Cliquez sur **Valider** pour ajouter l'information masquée dans le cache du rapport.
1. Cliquez sur **Fermer** pour fermer la boîte de dialogue. 

![todo:image_alt_text](working-with-table-hide-attributes_6.png)




1. Enregistrez et publiez le rapport.
