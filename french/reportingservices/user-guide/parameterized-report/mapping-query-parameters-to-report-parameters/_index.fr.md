1. Ouvrez Microsoft Excel.
1. Cliquez sur **Construire un ensemble de données** dans la barre d'outils Aspose.Cells.Report.Designer (

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_1.png)

).

1. Sélectionnez un ensemble de données dans le panneau de gauche, par exemple **EmpSalesDetail**. 

   **Sélection d'un ensemble de données** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_2.png)




1. Cliquez sur l'onglet Paramètres dans le panneau de droite pour répertorier les paramètres de l'ensemble de données.
1. Cliquez sur la colonne **Paramètre de rapport lié**.
   Tous les paramètres de rapport créés à l'avance sont répertoriés. 
1. Sélectionnez un paramètre. 

   **Paramètres de mappage** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_3.png)




1. Répétez pour mapper tous les paramètres de requête aux paramètres de rapport. 

   **Mapping terminé** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_4.png)

1. Cliquez sur **Fermer**.
