Pour créer un paramètre de rapport dans Aspose.Cells.Report.Designer :

1. Ouvrez Microsoft Excel.
1. Cliquez sur **Paramètre de rapport** dans la barre d'outils Aspose.Cells.Report.Designer. (

![todo:image_alt_text](creating-report-parameters_1.png)

)
Dans la boîte de dialogue, tous les paramètres de rapport sont répertoriés dans le volet gauche. 

1. Cliquez sur **Ajouter**. 

   **Le formulaire de configuration des paramètres de rapport** 

![todo:image_alt_text](creating-report-parameters_2.png)




1. Dans la boîte de dialogue, saisissez les informations du paramètre du rapport. 

   **Ajout d'informations de paramètre** 

![todo:image_alt_text](creating-report-parameters_3.png)



Les valeurs valides pour un paramètre de rapport peuvent provenir d'un ensemble de données ou d'un groupe de valeurs définies par l'utilisateur. Un paramètre de rapport a deux champs de données : étiquette et valeur. La valeur de l'étiquette peut être nulle. 

1. Cliquez sur **Configuration** pour configurer un groupe de valeurs pour le paramètre du rapport.
1. Ajoutez des valeurs pour les champs **Étiquette** et **Valeur**. 

   **Ajout d'étiquette et de valeur** 

![todo:image_alt_text](creating-report-parameters_4.png)




1. Cliquez sur **Ajouter**.
   Répétez pour ajouter d'autres valeurs pour le paramètre.
1. Cliquez sur **Fermer**.
1. Cliquez sur **Enregistrer** pour enregistrer le paramètre. 
1. Le nouveau paramètre de rapport est répertorié dans le panneau de gauche. 

   **Le nouveau paramètre** 

![todo:image_alt_text](creating-report-parameters_5.png)
