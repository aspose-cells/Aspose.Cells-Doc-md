Pour créer un paramètre de rapport dans Aspose.Cells.Report.Designer :

1. Ouvrez Microsoft Excel.
1.  Cliquez sur**Paramètre de rapport** dans la barre d'outils Aspose.Cells.Report.Designer. (

![tâche : image_autre_texte](creating-report-parameters_1.png)

)
 Dans la boîte de dialogue, tous les paramètres du rapport sont répertoriés dans le panneau de gauche.

1.  Cliquez sur**Ajouter**. 

   **Le formulaire de configuration des paramètres de rapport** 

![tâche : image_autre_texte](creating-report-parameters_2.png)




1.  Dans la boîte de dialogue, saisissez les informations sur les paramètres du rapport.

   **Ajout d'informations sur les paramètres** 

![tâche : image_autre_texte](creating-report-parameters_3.png)



Les valeurs valides pour un paramètre de rapport peuvent provenir d'un jeu de données ou d'un groupe de valeurs définies par l'utilisateur. Un paramètre de rapport comporte deux champs de données : étiquette et valeur. La valeur de l'étiquette peut être nulle.

1.  Cliquez sur**Installer** pour configurer un groupe de valeurs pour le paramètre de rapport.
1.  Ajouter des valeurs pour le**Étiquette** et**Évaluer** des champs.

   **Ajouter une étiquette et de la valeur** 

![tâche : image_autre_texte](creating-report-parameters_4.png)




1.  Cliquez sur**Ajouter**.
 Répétez l'opération pour ajouter d'autres valeurs pour le paramètre.
1.  Cliquez sur**proche**.
1.  Cliquez sur**Sauver** pour enregistrer le paramètre.
1.  Le nouveau paramètre de rapport est répertorié dans le panneau de gauche.

   **Le nouveau paramètre** 

![tâche : image_autre_texte](creating-report-parameters_5.png)
