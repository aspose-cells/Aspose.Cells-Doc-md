{{% alert color="primary" %}} 

Pour créer un rapport de tableau, vous devez définir les attributs du rapport avant de publier le rapport sur le serveur de rapports. Les attributs du rapport contiennent des informations sur le nom du rapport, la plage, le regroupement des données, le tri, etc.

{{% /alert %}} 

Pour configurer les attributs d'un rapport tabulaire :

1.  Cliquez sur**Définir l'attribut**dans la barre d'outils Aspose.Cells.Report.Designer (

![tâche : image_autre_texte](setting-report-attributes_1.png)

).

1.  Saisissez le nom du rapport dans la boîte de dialogue.

   **Définition du nom du rapport** 

![tâche : image_autre_texte](setting-report-attributes_2.png)




1.  Cliquez sur**Suivant**.
1. Sélectionnez une plage pour le rapport.
 La plage commence par la première cellule de la ligne de données du tableau et se termine à la dernière cellule de la ligne de pied de page ou, s'il n'y a pas de pied de page, à la dernière cellule des données de groupe.
 L'index des cellules de début et de fin apparaît dans le**Intervalle** champ automatiquement. Ici, la gamme comprend des cellules de D4 à H8.

   **Définition de la plage de rapport** 

![tâche : image_autre_texte](setting-report-attributes_3.png)




1.  Cliquez sur**Suivant** continuer.
Tous les champs du jeu de données sont répertoriés dans le panneau de gauche de la boîte de dialogue.
1.  Sélectionnez les champs pour regrouper les données.

   **Sélection des champs pour regrouper les données** 

![tâche : image_autre_texte](setting-report-attributes_4.png)




1.  Cliquez sur**Suivant** continuer.
1.  Sélectionnez les champs pour trier les données.

   **Sélection des champs pour classer les données par** 

![tâche : image_autre_texte](setting-report-attributes_5.png)




1.  Cliquez sur**Suivant**.

 Un résumé du rapport s'affiche.
   **Résumé du rapport** 

![tâche : image_autre_texte](setting-report-attributes_6.png)

1.  Cliquez sur**Finir**.
