{{% alert color="primary" %}} 

Pour créer un rapport tabulaire, il est nécessaire de définir les attributs du rapport avant de le publier sur le serveur de rapports. Les attributs du rapport contiennent des informations sur le nom du rapport, la plage, le regroupement des données, le tri, et ainsi de suite.

{{% /alert %}} 

Pour configurer les attributs d'un rapport tabulaire :

1. Cliquez sur **Définir un attribut** dans la barre d'outils Aspose.Cells.Report.Designer (

![todo:image_alt_text](setting-report-attributes_1.png)

).

1. Entrez le nom du rapport dans la boîte de dialogue. 

   **Définir le nom du rapport** 

![todo:image_alt_text](setting-report-attributes_2.png)




1. Cliquez sur **Suivant**.
1. Sélectionnez une plage pour le rapport.
   La plage commence par la première cellule de la ligne de données du tableau et se termine à la dernière cellule de la ligne de footer, ou, s'il n'y a pas de footer, à la dernière cellule des données du groupe.
   L'index des cellules de début et de fin apparaît automatiquement dans le champ **Plage**. Ici, la plage comprend des cellules de D4 à H8. 

   **Définir la plage du rapport** 

![todo:image_alt_text](setting-report-attributes_3.png)




1. Cliquez sur **Suivant** pour continuer.
   Tous les champs du jeu de données sont répertoriés dans le volet de gauche de la boîte de dialogue. 
1. Sélectionnez les champs pour regrouper les données par. 

   **Sélection des champs pour regrouper les données** 

![todo:image_alt_text](setting-report-attributes_4.png)




1. Cliquez sur **Suivant** pour continuer.
1. Sélectionnez les champs pour trier les données par. 

   **Sélection des champs pour trier les données** 

![todo:image_alt_text](setting-report-attributes_5.png)




1. Cliquez sur **Suivant**.

   Un résumé du rapport est affiché.
   **Résumé du rapport** 

![todo:image_alt_text](setting-report-attributes_6.png)

1. Cliquez sur **Terminer**.
