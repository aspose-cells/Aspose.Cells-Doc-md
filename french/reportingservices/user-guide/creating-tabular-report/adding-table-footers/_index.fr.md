Pour ajouter des pieds de table :

1. Sélectionnez la cellule de balise de pied, directement à gauche de la première colonne de la table, mais sous la ligne de données du groupe de table (ou la ligne de données de la table si la table ne contient pas de groupe de table) : 

   **Sélection de la cellule de balise de pied** 

![todo:image_alt_text](adding-table-footers_1.png)




1. Cliquez sur **Définir pied de page** dans la barre d'outils Aspose.Cells.Report.Designer (

![todo:image_alt_text](adding-table-footers_2.png)

). 

**La balise de pied est définie** 

![todo:image_alt_text](adding-table-footers_3.png)




1. Ajouter la ligne de données de pied de page.
   Les lignes de données de pied de page peuvent contenir du texte statique, des marqueurs de données, des formules de Reporting Services, et ainsi de suite. Ici, nous avons ajouté une ligne de pied de page qui contient une cellule de texte statique fusionnée et une cellule de formule de Reporting Services. 

   **Le pied de page terminé** 

![todo:image_alt_text](adding-table-footers_4.png)



Plus d'une ligne de données de pied de page est autorisée dans un tableau.
