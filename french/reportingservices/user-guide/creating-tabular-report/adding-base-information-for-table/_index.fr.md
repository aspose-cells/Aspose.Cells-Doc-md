Pour créer une nouvelle table:

1. Ouvrez Microsft Excel.
1. Cliquez sur **Ouvrir le Rapport** 

![todo:image_alt_text](adding-base-information-for-table_1)

`  `sur la barre d'outils Aspose.Cells.Report.Designer pour ouvrir un fichier de rapport RDL contenant les sources données, les jeux de données et les Paramètres de Rapport créés à l'avance. Une fois le fichier ouvert avec succès, toutes ses informations sont disponibles pour une utilisation, par exemple, ses jeux de données sont répertoriés dans la zone de liste déroulante DataSet pertinente. 

![todo:image_alt_text](adding-base-information-for-table_2.png)

1. Ouvrez un classeur Microsoft Excel et concevez l'en-tête du tableau comme suit: 

   **Conception de l'en-tête du tableau** 

![todo:image_alt_text](adding-base-information-for-table_3.png)

1. Sélectionnez un jeu de données ”EmpSalesDetail” dans la liste déroulante de jeu de données pertinente. 

   **Sélection d'un ensemble de données** 

![todo:image_alt_text](adding-base-information-for-table_4.png)

1. Tous les champs du jeu de données ”EmpSalesDetail” sont répertoriés dans la zone de liste déroulante des champs. Sélectionnez la cellule sous la colonne OrderNumber Cell puis sélectionnez le champ ”SalesOrderNumber” dans la zone de liste déroulante des champs pour insérer un marqueur de données dans la cellule. 

   **Insertion d'un champ** 

![todo:image_alt_text](adding-base-information-for-table_5.png)



**Le code du champ montre où un champ a été inséré.** 

![todo:image_alt_text](adding-base-information-for-table_6.png)

1. Répétez l'étape 5 pour insérer les champs ProdCat et SubCat. 

   **Tous les champs insérés**
