Pour créer un nouveau tableau :

1. Ouvrez Microsoft Excel.
1.  Cliquez sur**Ouvrir le rapport** 

![tâche : image_autre_texte](adding-base-information-for-table_1)

 `  ` sur la barre d'outils Aspose.Cells.Report.Designer pour ouvrir un fichier de rapport RDL qui contient les sources de données, les jeux de données et les paramètres de rapport créés à l'avance. Une fois que le fichier a été ouvert avec succès, toutes ses informations sont disponibles pour utilisation, par exemple, ses jeux de données sont répertoriés dans la zone de liste déroulante DataSet appropriée.

![tâche : image_autre_texte](adding-base-information-for-table_2.png)

1. Ouvrez une feuille de calcul Excel Microsoft et concevez l'en-tête du tableau comme suit :

   **Conception de l'en-tête du tableau** 

![tâche : image_autre_texte](adding-base-information-for-table_3.png)

1.  Sélectionnez un jeu de données « EmpSalesDetail » dans le contrôle de zone de liste déroulante du jeu de données pertinent.

   **Sélection d'un ensemble de données** 

![tâche : image_autre_texte](adding-base-information-for-table_4.png)

1.  Tous les champs du jeu de données "EmpSalesDetail" sont répertoriés dans la zone de liste déroulante des champs. Sélectionnez la cellule sous la colonne OrderNumber Cell, puis sélectionnez le champ "SalesOrderNumber" dans la zone de liste déroulante des champs pour insérer un marqueur de données dans la cellule.

   **Insertion d'un champ** 

![tâche : image_autre_texte](adding-base-information-for-table_5.png)



**Le code de champ indique où un champ a été inséré.** 

![tâche : image_autre_texte](adding-base-information-for-table_6.png)

1.  Répétez l'étape 5 pour insérer les champs ProdCat et SubCat.

   **Tous les champs insérés** 

![tâche : image_autre_texte](adding-base-information-for-table_7.png)
