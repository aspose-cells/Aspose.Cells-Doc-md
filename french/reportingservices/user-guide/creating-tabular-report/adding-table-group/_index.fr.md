Pour ajouter un groupe de tableau, suivez les étapes ci-dessous :



1) Sélectionnez la cellule de balise de groupe qui se trouve à gauche de la première colonne de la colonne de données de tableau, sous la ligne de données de tableau, comme indiqué ci-dessous.

![todo:image_alt_text](adding-table-group_1.png)







2) Sélectionnez "Numéro de commande de vente" dans la case de combinaison **Définir le groupe** pour définir le champ de groupe.



![todo:image_alt_text](adding-table-group_2.png)



Le résultat est montré dans la suite.



![todo:image_alt_text](adding-table-group_3.png)



3) Ajoutez la ligne de données de groupe selon les instructions décrites ci-dessus. La ligne de données de groupe peut contenir du texte statique, des marqueurs de données, des formules de Reporting Service, etc. Dans l'exemple, nous ajoutons une ligne de groupe qui contient une cellule de texte statique fusionnée et une cellule de formule de Reporting Services, comme indiqué ci-dessous.



![todo:image_alt_text](adding-table-group_4.png)



4) Plus d'une ligne de données de groupe est autorisée dans un tableau. Nous pouvons ajouter à nouveau une ligne de données de groupe, comme indiqué ci-dessous.



![todo:image_alt_text](adding-table-group_5.png)
