Pour ajouter un groupe de tables, suivez les étapes ci-dessous :



1) Sélectionnez la cellule de balise de groupe qui se trouve à gauche de la première colonne de la colonne de données du tableau, sous la ligne de données du tableau, comme indiqué ci-dessous.

![tâche : image_autre_texte](adding-table-group_1.png)







 2) Sélectionnez "SalesOrderNumber" dans le**Définir le groupe** zone de liste déroulante pour définir le champ de groupe.



![tâche : image_autre_texte](adding-table-group_2.png)



Le résultat est montré dans ce qui suit.



![tâche : image_autre_texte](adding-table-group_3.png)



3) Ajoutez la ligne de données de groupe selon les instructions décrites ci-dessus. La ligne de données de groupe peut contenir du texte statique, des marqueurs de données, des formules de service de rapport, etc. Dans l'exemple, nous ajoutons une ligne de groupe qui contient une cellule de texte statique et une cellule de formule Reporting Services fusionnées, comme illustré ci-dessous.



![tâche : image_autre_texte](adding-table-group_4.png)



4) Plusieurs lignes de données de groupe sont autorisées dans un tableau. Nous pouvons à nouveau ajouter une ligne de données de groupe, comme illustré ci-dessous.



![tâche : image_autre_texte](adding-table-group_5.png)
