Pour ajouter des formules de Reporting Services à la cellule de données du tableau, suivez les étapes ci-dessous :

- Sélectionnez la cellule de données du tableau et cliquez sur **Insérer une formule** dans la barre d'outils Aspose.Cells.Report.Designer (

![todo:image_alt_text](adding-reporting-services-formulas_1.png)

).

![todo:image_alt_text](adding-reporting-services-formulas_2.png)




- Sélectionnez ”Ensemble de données” dans le panneau de gauche, sélectionnez ”EmpSalesDetail” dans le panneau du milieu et double-cliquez sur ”Sum(EmpSalesDetail.OrderQty)” dans le panneau de droite. Une expression de formule apparaît dans la boîte d'édition en haut. Cliquez sur le bouton **OK** pour insérer l'expression dans la cellule.
