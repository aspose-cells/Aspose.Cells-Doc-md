Pour ajouter des formules Reporting Services à la cellule de données du tableau, procédez comme suit :

-  Sélectionnez la cellule de données du tableau et cliquez sur**Insérer une formule**dans la barre d'outils Aspose.Cells.Report.Designer (

![tâche : image_autre_texte](adding-reporting-services-formulas_1.png)

).

![tâche : image_autre_texte](adding-reporting-services-formulas_2.png)




-  Sélectionnez "DataSet" dans le panneau de gauche, sélectionnez "EmpSalesDetail" dans le panneau du milieu et double-cliquez sur "Sum(EmpSalesDetail.OrderQty)" dans le panneau de droite. Une expression de formule apparaît dans la zone d'édition supérieure. Cliquez sur**D'ACCORD** bouton pour insérer l'expression dans la cellule.
