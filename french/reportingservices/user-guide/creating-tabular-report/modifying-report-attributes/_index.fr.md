Pour modifier les attributs d'un rapport :

1. Cliquez sur **Modifier l'attribut** dans la barre d'outils Aspose.Cells.Report.Designer (

![todo:image_alt_text](modifying-report-attributes_1.png)

).
Les rapports disponibles sont répertoriés dans le volet de gauche. 

1. Sélectionnez un rapport et cliquez sur **Modifier**. 

   **Sélection d'un rapport** 

![todo:image_alt_text](modifying-report-attributes_2.png)




1. Modifiez les attributs du rapport. 

   **Modifier les attributs** 

![todo:image_alt_text](modifying-report-attributes_3.png)




1. Cliquez sur **Valider** pour terminer.
