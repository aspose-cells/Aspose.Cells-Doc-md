- **Extension de fichier** 
 Lorsque la valeur est nulle, le nom de l'extension du fichier de rapport exporté sera la valeur par défaut. Lorsque la valeur n'est pas nulle, le nom d'extension du fichier de rapport exporté est valeur.
- **En-têtes de page simples** 
Lorsque la valeur est true, elle affiche l'élément d'en-tête de rapport dans l'en-tête de page Excel. La valeur par défaut est faux.
- **Pieds de page simples** 
 Lorsque la valeur est true, il affiche l'élément de pied de page du rapport dans le pied de page Excel. La valeur par défaut est true.
- **PutoutHeader** 
 Lorsque la valeur est true, il exportera l'élément d'en-tête du rapport. Lorsque la valeur est false, il n'exporte pas l'élément d'en-tête du rapport. La valeur par défaut est true. La valeur prend uniquement en charge l'extension Excel2007Xlsx (données uniquement).
- **PutoutFooter** 
 Lorsque la valeur est true, il exporte l'élément de pied de page du rapport. Lorsque la valeur est false, il n'exporte pas l'élément de pied de page du rapport. La valeur par défaut est true. La valeur prend uniquement en charge l'extension Excel2007Xlsx (données uniquement).
- **FillTableGroupHeaderForSimpleOutPut** 
 La valeur par défaut est faux. La valeur prend uniquement en charge l'extension Excel2007Xlsx (données uniquement).
- **NoOutPutTotalForSimpleOutPut** 
 La valeur par défaut est faux. La valeur prend uniquement en charge l'extension Excel2007Xlsx (données uniquement).
- **NoOutPutGroupForSimpleOutPut** 
 La valeur par défaut est faux. La valeur prend uniquement en charge l'extension Excel2007Xlsx (données uniquement).
- **NoDoGroupPageForSimpleOutPut** 
 La valeur par défaut est true. La valeur prend uniquement en charge l'extension Excel2007Xlsx (données uniquement).
- **NoDoPageForSimpleOutPut** 
 La valeur par défaut est true. La valeur prend uniquement en charge l'extension Excel2007Xlsx (données uniquement).
- **Délimiteur de champ** 
 Il définit les délimiteurs de champs. La valeur prend en charge les extensions CSV et TXT.
