Les lignes de grille sont configurées dans **Aspose.Cells.ReportingServices.xml** afin que vous puissiez contrôler si un fichier Microsoft Excel exporté affiche des lignes de grille. La valeur par défaut est **désactivée**, ce qui signifie que les lignes de grille ne sont pas affichées.

{{< highlight java >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
