 Les quadrillages sont configurés dans**Aspose.Cells.ReportingServices.xml** vous pouvez ainsi contrôler si un fichier Excel Microsoft exporté affiche des lignes de grille. La valeur par défaut est**à l'arrêt**, ce qui signifie que le quadrillage n'est pas affiché.

{{< highlight "java" >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
