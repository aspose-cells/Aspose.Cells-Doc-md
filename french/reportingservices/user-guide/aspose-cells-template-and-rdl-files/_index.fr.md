{{% alert color="primary" %}} 

 Une définition de rapport contient des informations d'extraction de données et de mise en page pour un rapport. Le langage de définition de rapport (RDL) est une représentation XML pour la définition de rapport. RDL est un schéma ouvert : les développeurs peuvent étendre RDL avec des attributs et des éléments supplémentaires. L'extension de fichier standard pour les fichiers de définition de rapport est .rdl. Son type MIME est text/xml.

Un modèle Aspose.Cells est un classeur Excel Microsoft qui contient des informations sur la mise en page du rapport et est enregistré dans des fichiers RDL (format) lorsque le travail de conception du modèle est terminé. Lorsqu'un fichier RDL qui ne contient pas de modèle Aspose.Cells est ouvert, seules ses sources de données et ses requêtes sont disponibles pour une utilisation dans Aspose.Cells.Report.Designer. Lorsqu'un fichier RDL contenant un modèle Aspose.Cells est ouvert, le modèle est récupéré et ouvert en tant que classeur dans Microsoft Excel.
