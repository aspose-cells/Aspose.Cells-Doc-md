Pour ajouter un élément de sous-rapport :

1. Sélectionnez la cellule qui fera référence à l'élément de sous-rapport.
1. Cliquez sur **Sous-Rapport** (

![todo:image_alt_text](add-sub-report-item_1.png)

).
Le formulaire de sous-rapport est affiché. 

**Le formulaire de sous-rapport** 

![todo:image_alt_text](add-sub-report-item_2.png)




1. Cliquez sur **Ajouter un sous-rapport**.
   Le formulaire Rapport de récupération est affiché.
1. Sélectionnez un rapport. 

   **Sélection d'un rapport** 

![todo:image_alt_text](add-sub-report-item_3.png)




1. Cliquez sur **Fermer**. 

   **Un sous-rapport a été sélectionné** 

![todo:image_alt_text](add-sub-report-item_4.png)




1. Double-cliquez sur la liste des paramètres pour afficher la boîte de dialogue Modifier la formule. 

   **La boîte de dialogue Modifier la Formule** 

![todo:image_alt_text](add-sub-report-item_5.png)




1. Modifiez et/ou mettez à jour une valeur de paramètre. 

   **Nommer le sous-rapport** 

![todo:image_alt_text](add-sub-report-item_6.png)




1. Cliquez sur **Insérer un sous-rapport**. 

   **Le sous-rapport inséré dans un tableau de rapport** 

![todo:image_alt_text](add-sub-report-item_7.png)
