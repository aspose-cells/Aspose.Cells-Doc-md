Pour ajouter un élément de sous-rapport :

1. Sélectionnez la cellule qui fera référence à l'élément de sous-rapport.
1.  Cliquez sur**Sous-rapport** (

![tâche : image_autre_texte](add-sub-report-item_1.png)

).
 Le formulaire de sous-rapport s'affiche.

**Le formulaire de sous-rapport** 

![tâche : image_autre_texte](add-sub-report-item_2.png)




1.  Cliquez sur**Ajouter un sous-rapport**.
 Le formulaire Récupérer le rapport s'affiche.
1.  Sélectionnez un rapport.

   **Sélection d'un rapport** 

![tâche : image_autre_texte](add-sub-report-item_3.png)




1.  Cliquez sur**proche**. 

   **Un sous-rapport a été sélectionné** 

![tâche : image_autre_texte](add-sub-report-item_4.png)




1.  Double-cliquez sur la liste des paramètres pour afficher la boîte de dialogue Modifier la formule.

   **La boîte de dialogue Modifier la formule** 

![tâche : image_autre_texte](add-sub-report-item_5.png)




1.  Modifier et/ou mettre à jour une valeur de paramètre.

   **Nommer le sous-rapport** 

![tâche : image_autre_texte](add-sub-report-item_6.png)




1.  Cliquez sur**Insérer un sous-rapport**. 

   **Le sous-rapport inséré dans une table de rapport** 

![tâche : image_autre_texte](add-sub-report-item_7.png)
