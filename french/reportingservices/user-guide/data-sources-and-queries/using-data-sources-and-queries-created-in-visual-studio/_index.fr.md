{{% alert color="primary" %}} 

Vous pouvez créer des sources de données et des requêtes dans Visual Studio et les enregistrer dans un fichier RDL. Lorsque le fichier RDL est ouvert dans Aspose.Cells.Report.Designer, ses sources de données et ses requêtes sont disponibles pour une utilisation. Par exemple, nous pouvons créer une requête nommée EmpSalesDetail dans Visual Studio et l'enregistrer en tant que fichier RDL.

**Le fichier RDL EmpSalesDetail** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_1.png)

Cet article explique comment travailler avec un fichier RDL dans Aspose.Cells.Report.Designer.

{{% /alert %}} 

Vous pouvez ouvrir des fichiers RDL avec Aspose.Cells.Report.Designer et travailler avec eux :

1. Ouvrez Microsoft Excel.
1. Dans la barre d'outils Aspose.Cells.Report.Designer, sélectionnez **Ouvrir un rapport**. 

   **Le bouton Ouvrir un rapport** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Sélectionnez le fichier RDL et ouvrez-le.
   Une fois le fichier RDL ouvert dans Aspose.Cells.Report.Designer, les requêtes apparaissent dans la liste **Ensemble de données**. 

   **Ensembles de données disponibles** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_3.png)
