Pour modifier une requête dans Aspose.Cells.Report.Designer :

1. Ouvrez Microsoft Excel.
1.  Cliquez sur**Construire un ensemble de données**dans la barre d'outils Aspose.Cells.Report.Designer (

![tâche : image_autre_texte](editing-query_1.png)

).

1.  Dans la boîte de dialogue, sélectionnez un ensemble de données, par exemple,**EmpSalesDetail**.
1.  Cliquez sur**Éditer**. 

   **Sélection d'un ensemble de données** 

![tâche : image_autre_texte](editing-query_2.png)




1.  Aspose.Cell.Report.Designer démarre Microsoft Query et ouvre la requête pour modification.

   **Modification d'une requête dans la requête Microsoft** 

![tâche : image_autre_texte](editing-query_3.png)




1.  De Microsoft Requêtes**Dossier** menu, sélectionnez**Retour à Aspose.Cells.Report.Designer** 

   **Revenir au concepteur de rapport** 

![tâche : image_autre_texte](editing-query_4.png)
