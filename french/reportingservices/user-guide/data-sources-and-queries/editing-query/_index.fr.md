Pour modifier une requête dans Aspose.Cells.Report.Designer :

1. Ouvrez Microsoft Excel.
1. Cliquez sur **Build DataSet** dans la barre d'outils Aspose.Cells.Report.Designer (

![todo:image_alt_text](editing-query_1.png)

).

1. Dans la boîte de dialogue, sélectionnez un jeu de données, par exemple, **EmpSalesDetail**.
1. Cliquez sur **Éditer**. 

   **Sélection d'un ensemble de données** 

![todo:image_alt_text](editing-query_2.png)




1. Aspose.Cell.Report.Designer démarre Microsoft Query et ouvre la requête pour édition. 

   **Modification d'une requête dans Microsoft Query** 

![todo:image_alt_text](editing-query_3.png)




1. Dans le menu **Fichier** de Microsoft Query, sélectionnez **Retourner à Aspose.Cells.Report.Designer** 

   **Retourner au concepteur de rapport** 

![todo:image_alt_text](editing-query_4.png)
