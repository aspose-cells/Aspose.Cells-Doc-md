Pour créer une nouvelle requête à l'aide d'une source de données existante :

1. Ouvrez Microsoft Excel.
1. Cliquez sur **Build DataSet** dans Aspose.Cells.Report.Designer (

![todo:image_alt_text](creating-new-query-using-existing-data-source_1.png)

).

1. Dans la boîte de dialogue, sélectionnez l'une des sources de données répertoriées dans le volet de gauche. Par exemple, **SqlServer**.
1. Cliquez sur **Ajouter**. 

   **Sélection d'une source de données** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_2.png)




1. Dans la boîte de dialogue, tapez un nouveau nom de jeu de données, par exemple SalesEmps.
1. Cliquez sur **Suivant**. 

   **Nommer un nouveau jeu de données** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_3.png)



Aspose.Cells.Report.Designer démarre Microsoft Query et se connecte à la source de données sélectionnée. Une nouvelle requête peut maintenant être créée avec Microsoft Query. 

**Création d'une nouvelle requête** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_4.png)




1. Dans le menu **Fichier** de Microsoft Query, sélectionnez **Retour à Aspose.Cells.Report.Designer**. 

   **Retourner au concepteur de rapport** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_5.png)



La source de données SqlServer contient deux jeux de données, dont l'un est le nouveau jeu de données SalesEmps.
