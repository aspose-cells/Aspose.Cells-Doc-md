Pour créer une nouvelle requête à l'aide d'une source de données existante :

1. Ouvrez Microsoft Excel.
1.  Cliquez sur**Construire un ensemble de données** dans le Aspose.Cells.Report.Designer (

![tâche : image_autre_texte](creating-new-query-using-existing-data-source_1.png)

).

1.  Dans la boîte de dialogue, sélectionnez l'une des sources de données répertoriées dans le panneau de gauche. Par exemple,**Serveur SQL**.
1.  Cliquez sur**Ajouter**. 

   **Sélection d'une source de données** 

![tâche : image_autre_texte](creating-new-query-using-existing-data-source_2.png)




1. Dans la boîte de dialogue, saisissez un nouveau nom d'ensemble de données, par exemple SalesEmps.
1.  Cliquez sur**Suivant**. 

   **Nommer un nouvel ensemble de données** 

![tâche : image_autre_texte](creating-new-query-using-existing-data-source_3.png)



 Aspose.Cells.Report.Designer démarre Microsoft Query et se connecte à la source de données sélectionnée. Une nouvelle requête peut maintenant être créée avec Microsoft Query.

**Construire une nouvelle requête** 

![tâche : image_autre_texte](creating-new-query-using-existing-data-source_4.png)




1.  De Microsoft Requêtes**Dossier** menu, sélectionnez**Retour à Aspose.Cells.Report.Designer**. 

   **Revenir au concepteur de rapport** 

![tâche : image_autre_texte](creating-new-query-using-existing-data-source_5.png)



 La source de données SqlServer contient deux jeux de données, dont l'un est le nouveau jeu de données SalesEmps.
