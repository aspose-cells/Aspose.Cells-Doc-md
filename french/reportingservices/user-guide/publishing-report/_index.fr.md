{{% alert color="primary" %}} 

Lorsque vous avez conçu un rapport avec tous les éléments souhaités, l'étape suivante consiste à le publier sur le serveur de rapports afin que vous et vos collègues puissiez l'exécuter.

{{% /alert %}} 

Pour publier un rapport sur Report Server, suivez les étapes ci-dessous :

1.  Cliquez sur**Publier** 

![tâche : image_autre_texte](publishing-report_1.png)

 dans la barre d'outils Aspose.Cells.Report.Designer et cliquez sur le**Oui** bouton pour enregistrer le rapport.

![tâche : image_autre_texte](publishing-report_2.png)




1.  Dans la boîte de dialogue, entrez l'URL du serveur de rapports et cliquez sur le**Rafraîchir** bouton.

![tâche : image_autre_texte](publishing-report_3.png)

1.  Sélectionnez un dossier cible et entrez le nom du rapport. Clique le**Configuration de la source de données** bouton pour configurer les sources de données.

![tâche : image_autre_texte](publishing-report_4.png)



La boîte de dialogue Configuration de la source de données s'affiche.

1.  Dans la boîte de dialogue, entrez les informations sur la source de données. Cliquez sur**Sélectionner** pour sélectionner les sources de données partagées. (Si vous en avez besoin, les sources de données partagées doivent être la principale source de données du rapport). Clique le**D'ACCORD** bouton pour terminer le travail.

![tâche : image_autre_texte](publishing-report_5.png)

1.  Cliquez sur**Publier** pour publier le rapport sur Report Server.

![tâche : image_autre_texte](publishing-report_6.png)



bouton pour terminer le travail.
