{{% alert color="primary" %}} 

Lorsque vous avez conçu un rapport avec tous les éléments que vous souhaitez qu'il contienne, l'étape suivante consiste à le publier sur le Serveur de rapports afin que vous et vos collègues puissiez l'exécuter.

{{% /alert %}} 

Pour publier un rapport sur le Serveur de rapports, suivez les étapes ci-dessous :

1. Cliquez sur **Publier** 

![todo:image_alt_text](publishing-report_1.png)

sur la barre d'outils Aspose.Cells.Report.Designer et cliquez sur le bouton **Oui** pour enregistrer le rapport. 

![todo:image_alt_text](publishing-report_2.png)




1. Dans la boîte de dialogue, saisissez l'URL du Serveur de rapports et cliquez sur le bouton **Actualiser**.

![todo:image_alt_text](publishing-report_3.png)

1. Sélectionnez un dossier cible et saisissez le nom du rapport. Cliquez sur le bouton **Configuration de la source de données** pour configurer les sources de données. 

![todo:image_alt_text](publishing-report_4.png)



La boîte de dialogue Configuration de la source de données est affichée.

1. Dans la boîte de dialogue, saisissez les informations de la source de données. Cliquez sur **Sélectionner** pour sélectionner les sources de données partagées (si vous en avez besoin, les sources de données partagées devraient être la principale source de données du rapport). Cliquez sur le bouton **OK** pour terminer le travail.

![todo:image_alt_text](publishing-report_5.png)

1. Cliquez sur **Publier** pour publier le rapport sur le Serveur de rapports. 

![todo:image_alt_text](publishing-report_6.png)



bouton pour terminer le travail.
