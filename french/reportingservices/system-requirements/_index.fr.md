{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services (incluant le client Aspose.Cells.ReportingServices, les compléments pour Microsoft Excel et Aspose.Cells.ReportingServices Server) doit être installé correctement. De plus, Microsoft SQL Server doit être installé avant d'installer Aspose.Cells for Reporting Services.

{{% /alert %}} 
### **Système Office**
Les versions de Microsoft Excel suivantes sont prises en charge. De plus, vous devez installer des correctifs supplémentaires pour résoudre certains problèmes potentiels liés à Microsoft Office 2000, 2002, XP ou 2003.

- Microsoft Excel 2000
  Installer le **lockbackRegKey.msi** pour résoudre un problème CLR sur les machines Microsoft Office antérieures à Office 2003.
- Microsoft Excel 2002
  Installer **lockbackRegKey.msi** pour résoudre un problème CLR sur les machines Microsoft Office antérieures à Office 2003.
- Microsoft Excel 2003
  Toujours mettre à jour vers Office 2003 (KB907417). Cette mise à jour corrige un problème potentiel où un complément, une balise intelligente ou un document intelligent peut ne pas se charger s'il utilise le Microsoft .NET Framework 2.0.
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
- Microsoft Excel 2016
- Microsoft Excel 2019
- Microsoft Excel 365
