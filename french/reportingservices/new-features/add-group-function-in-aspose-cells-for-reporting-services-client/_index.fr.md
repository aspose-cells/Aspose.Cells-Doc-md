Il définit les attributs de groupe de l'élément de rapport de table via Modifier les attributs. Les paramètres incluent le type, le nom, le niveau et le masquage.

- Type représente le type de groupe. Il comprend le groupe de lignes et le groupe de colonnes.
- Le nom représente le nom du groupe.
- Le niveau représente le niveau du groupe.
- Hide représente Group hide.

La modification du rapport implique les étapes suivantes :
Ouvrez la boîte de dialogue "Modifier le rapport" et choisissez l'onglet Plan.

![tâche : image_autre_texte](add-group-function-in-aspose-cells-for-reporting-services-client_1.jpg)


 Cliquez sur**Modifier** bouton et mettre à jour Masquer la configuration.

![tâche : image_autre_texte](add-group-function-in-aspose-cells-for-reporting-services-client_2.jpg)


 Cliquez sur**S'engager** bouton et enregistrez les configurations masquées.

![tâche : image_autre_texte](add-group-function-in-aspose-cells-for-reporting-services-client_3.jpg)
