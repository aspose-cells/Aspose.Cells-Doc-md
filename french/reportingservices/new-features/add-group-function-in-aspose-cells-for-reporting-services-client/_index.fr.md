Établit les attributs de groupe de l'élément de rapport de tableau via la modification des attributs. Les paramètres incluent le type, le nom, le niveau et le masquage.

- Le type représente le type de groupe. Il comprend Groupe de lignes et Groupe de colonnes.
- Le nom représente le nom du groupe.
- Le niveau représente le niveau du groupe.
- Le masquage représente le masquage du groupe.

La modification du rapport implique les étapes suivantes :
Ouvrez la boîte de dialogue « Modifier le rapport » et choisissez l'onglet Vue d'ensemble.

![todo:image_alt_text](add-group-function-in-aspose-cells-for-reporting-services-client_1.jpg)


Cliquez sur le bouton **Modifier** et mettez à jour la configuration de masquage. 

![todo:image_alt_text](add-group-function-in-aspose-cells-for-reporting-services-client_2.jpg)


Cliquez sur le bouton **Valider** et sauvegardez les configurations de masquage. 

![todo:image_alt_text](add-group-function-in-aspose-cells-for-reporting-services-client_3.jpg)
