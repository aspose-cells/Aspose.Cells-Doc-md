Le concepteur de rapports Aspose.Cells for Reporting Services enregistre l'URL actuellement consulté du serveur de rapports. Lorsque vous ouvrez à nouveau la boîte de dialogue du Gestionnaire de serveur de rapports dans le concepteur de rapports Aspose.Cells, le formulaire de dialogue charge automatiquement les informations de serveur de rapports enregistrées.
