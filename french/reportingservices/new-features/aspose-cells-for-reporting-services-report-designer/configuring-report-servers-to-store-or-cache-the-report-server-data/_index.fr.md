Le concepteur de rapports Aspose.Cells for Reporting Services enregistre l'URL du serveur de rapports en cours d'accès. Lorsque vous ouvrez à nouveau la boîte de dialogue Report Server Manager dans le Aspose.Cells Report Designer, le formulaire de boîte de dialogue charge automatiquement les informations enregistrées sur le serveur de rapports.
