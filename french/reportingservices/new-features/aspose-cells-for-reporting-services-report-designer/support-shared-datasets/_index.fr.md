Aspose.Cells pour Reporting Services prend en charge les jeux de données partagés SSRS 2008R2. Lorsque le concepteur de rapports Aspose.Cells ouvre un fichier de définition de rapport qui inclut des ensembles de données partagés, il peut automatiquement obtenir des informations sur l'ensemble de données qui incluent des champs du serveur de rapports.
