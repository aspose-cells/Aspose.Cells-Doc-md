Aspose.Cells for Reporting Services prend en charge les jeux de données partagés SSRS 2008R2. Lorsque le concepteur de rapports Aspose.Cells ouvre un fichier de définition de rapport qui inclut des jeux de données partagés, il peut obtenir automatiquement des informations sur les jeux de données qui incluent des champs provenant du serveur de rapports.
