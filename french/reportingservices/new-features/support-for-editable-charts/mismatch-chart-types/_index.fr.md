{{% alert color="primary" %}} 

Certains types de graphiques dans le fichier de définition de rapport SQL Server Reporting Services 2008 ne sont pas disponibles en tant que types de graphiques Excel :

- Pas de ligne
- Gamme-Plaine
- Gamme-Lisse
- Plage-Colonne
- Barre de gamme
- Plage-Barre d'erreur
- Range-BoxPlot
- Forme-Entonnoir

 Lors du rendu d'éléments de graphique qui n'ont pas d'options équivalentes dans Microsoft Excel, Aspose.Cells les génère en images.

{{% /alert %}}
