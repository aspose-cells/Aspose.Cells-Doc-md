{{% alert color="primary" %}} 

Certains types de graphiques dans le fichier de définition de rapport SQL Server Reporting Services 2008 ne sont pas disponibles en tant que types de graphiques Excel :

- Ligne-Escalier
- Plaine-Écart
- Lisse-Écart
- Colonne-Écart
- Barre-Écart
- Barre-d'erreur-Écart
- Boîte à moustaches-Écart
- Entonnoir-Forme

Lors du rendu des éléments graphiques qui n'ont pas d'options équivalentes dans Microsoft Excel, Aspose.Cells les génère sous forme d'images. 

{{% /alert %}}
