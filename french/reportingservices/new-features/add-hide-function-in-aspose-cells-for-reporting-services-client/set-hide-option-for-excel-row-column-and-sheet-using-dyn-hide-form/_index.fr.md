Aspose.Cells for Reporting Services masque toute ligne, colonne et feuille spécifiée en fonction des paramètres de masquage. Les paramètres incluent :

- **NomFeuille** représente une feuille de calcul qui spécifie un objet masqué.
- **Type** représente le type d'objet masqué. Il inclut la feuille de calcul, la ligne et la colonne.
- **IndiceLigne/Colonne** représente l'objet masqué spécifié. Lorsque le type est une feuille de calcul, la valeur est nulle. L'indice de colonne prend en charge les caractères. La valeur de l'indice commence à partir de 1.
- **ÉtatMasqué** représente la valeur de l'état masqué. La valeur est vrai, faux ou une formule Excel. Le paramètre prend en charge uniquement les paramètres de rapport de Reporting Services.
