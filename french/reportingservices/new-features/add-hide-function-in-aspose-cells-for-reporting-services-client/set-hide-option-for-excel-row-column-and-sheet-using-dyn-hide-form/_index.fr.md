Aspose.Cells pour Reporting Services masque tout ensemble basé sur des lignes, des colonnes et des feuilles spécifié dans les paramètres de masquage. Les paramètres comprennent :

- **NomFeuille** représente une feuille de calcul qui spécifie un objet masqué.
- **Taper** représente le type d'objet masqué. Il comprend une feuille de calcul, une ligne et une colonne.
- **Index de ligne/colonne**représente l'objet masqué spécifié. Lorsque le type est une feuille de calcul, la valeur est nulle. L'index de colonne prend en charge le caractère. La valeur de l'indice commence à 1.
- **Masquer l'état** représente la valeur de l'état masqué. La valeur est vrai, faux ou une formule Excel. Le paramètre prend uniquement en charge les paramètres de rapport Reporting Services.
