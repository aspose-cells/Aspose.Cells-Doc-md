Définir les attributs de masquage d'un rapport de table dans **Modifier les attributs**. Les paramètres incluent :

- **Type** représente le type d'objet masqué. Il inclut la table, la ligne et la colonne.
- **Index** représente la position de l'objet masqué. Lorsque le type est une feuille de calcul, la valeur est nulle. La valeur de l'indice commence à partir de 0.
- **Expression** représente la valeur de l'état masqué. La valeur est vrai, faux ou une formule Excel. L'expression prend en charge la formule de Reporting Services.
