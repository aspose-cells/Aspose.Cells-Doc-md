Définir les attributs de masquage d'un rapport de tableau à partir de**Modifier l'attribut**. Les paramètres incluent :

- **Taper** représente le type d'objet masqué. Il comprend un tableau, une ligne et une colonne.
- **Indice** représente la position de l'objet masqué. Lorsque le type est une feuille de calcul, la valeur est nulle. La valeur de l'indice commence à 0.
- **Expression** représente la valeur de l'état masqué. La valeur est vrai, faux ou une formule Excel. L'expression prend en charge la formule Reporting Services.
