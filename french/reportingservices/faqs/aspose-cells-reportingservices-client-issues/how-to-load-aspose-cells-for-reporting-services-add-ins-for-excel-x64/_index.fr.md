Une fois installé, Aspose.Cells for Reporting Services est enregistré en tant que composant 32 bits par défaut. Il ne peut pas charger correctement les compléments Excel dans Microsoft Excel x64. Vous devez donc réenregistrer Aspose.Cells.ReportingServices.dll en tant que composant(s) 64 bits.
