Pour configurer une URL de serveur de rapports client :

1.  Clique le**Ouvrir le rapport à distance** bouton et obtenez le**Récupérer le rapport** boite de dialogue.

![tâche : image_autre_texte](configuring-aspose-cells-reportingservices-client-report-server-url_1.png)

1.  Clique le**Configuration** bouton et obtenir**Configuration de l'URL du serveur de rapports** dialogue.

![tâche : image_autre_texte](configuring-aspose-cells-reportingservices-client-report-server-url_2.png)

1. Saisissez les informations de configuration du serveur de rapports (URL, nom d'utilisateur et mot de passe).
 1. L'URL du serveur de rapports doit être au format http://hôte/ReportServer. Par exemple http://172.16.204.11/ReportServer.
1. Le nom d'utilisateur est le nom d'utilisateur de Windows Server. Par exemple, Administrateur.
 1. Le mot de passe est le mot de passe utilisateur de Windows Server. Par exemple, 123456.
 1. La valeur par défaut du domaine est null.
1.  Clique le**Test** bouton pour vérifier les informations de configuration.
1.  Cliquez sur**sauvegarder** et enregistrez les informations de configuration du serveur de rapports.

![tâche : image_autre_texte](configuring-aspose-cells-reportingservices-client-report-server-url_3.png)




1.  Cliquez sur**Modifier** bouton pour modifier les informations de configuration.
