Pour configurer une adresse URL de serveur de rapport client :

1. Cliquez sur le bouton **Ouvrir un rapport à distance** et obtenez la boîte de dialogue **Récupérer le rapport**. 

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_1.png)

1. Cliquez sur le bouton **Config**  et obtenez la boîte de dialogue **Configuration de l'URL du serveur de rapport**. 

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_2.png)

1. Entrez les informations de configuration du serveur de rapport (URL, nom d'utilisateur et mot de passe).
   1. L'URL du serveur de rapport doit être au format http://host/ReportServer. Par exemple, http://172.16.204.11/ReportServer.
   1. Le nom d'utilisateur est le nom d'utilisateur du serveur Windows. Par exemple, Administrator.
   1. Le mot de passe est le mot de passe de l'utilisateur du serveur Windows. Par exemple, 123456.
   1. La valeur par défaut du domaine est nulle.
1. Cliquez sur le bouton **Test** pour vérifier les informations de configuration.
1. Cliquez sur **Enregistrer** et enregistrez les informations de configuration du serveur de rapport.

![todo:image_alt_text](configuring-aspose-cells-reportingservices-client-report-server-url_3.png)




1. Cliquez sur le bouton **Modifier** pour modifier les informations de configuration.
