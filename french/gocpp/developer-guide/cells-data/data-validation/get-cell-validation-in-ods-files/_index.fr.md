## **Obtenir la validation de la cellule dans les fichiers ODS**

Avec l'API Aspose.Cells for C++, vous pouvez obtenir la validation appliquée à une cellule dans les fichiers ODS. L'API fournit la méthode [**GetValidation**](https://reference.aspose.com/cells/go-cpp/cell/getvalidation/) de la classe [**Cell**](https://reference.aspose.com/cells/cpp/aspose.cells/cell/).
