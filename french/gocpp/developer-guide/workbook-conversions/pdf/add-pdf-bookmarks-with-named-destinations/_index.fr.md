## **Scénarios d'utilisation possibles**

Les destinations nommées sont un type spécial de signets ou de liens dans le PDF qui ne dépendent pas des pages PDF. Cela signifie que si des pages sont ajoutées ou supprimées du PDF, les signets peuvent devenir invalides mais les destinations nommées resteront intactes. Pour créer une destination nommée, veuillez définir la propriété [**PdfBookmarkEntry.GetDestinationName()**](https://reference.aspose.com/cells/go-cpp/pdfbookmarkentry/getdestinationname/).
