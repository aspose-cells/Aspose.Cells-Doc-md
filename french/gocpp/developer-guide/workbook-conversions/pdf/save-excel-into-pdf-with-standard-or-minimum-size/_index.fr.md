{{% alert color="primary" %}} 

Par défaut, Aspose.Cells enregistre Excel en PDF avec une taille standard. Cependant, vous pouvez également l’enregistrer avec une taille minimale en utilisant la propriété [PdfSaveOptions.GetOptimizationType()](https://reference.aspose.com/cells/go-cpp/pdfsaveoptions/getoptimizationtype/). Elle accepte les valeurs suivantes :

- PdfOptimizationType::Standard
- PdfOptimizationType::TailleMinimale
