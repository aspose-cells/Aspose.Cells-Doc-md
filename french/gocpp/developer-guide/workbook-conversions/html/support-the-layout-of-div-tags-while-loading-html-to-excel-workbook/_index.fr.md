{{% alert color="primary" %}} 

Normalement, la mise en page des balises DIV est ignorée lors du chargement du HTML dans un objet classeur Excel. Cependant, si vous souhaitez préserver la mise en page des balises DIV, définissez la propriété [GetSupportDivTag()](https://reference.aspose.com/cells/go-cpp/htmlloadoptions/getsupportdivtag/) sur **true**. La valeur par défaut de cette propriété est **false**.
