Générer un fichier HTML à partir d'une feuille de calcul est très courant. La taille des colonnes est définie en "pt" ce qui fonctionne dans de nombreux cas. Cependant, il peut arriver qu'une taille fixe ne soit pas requise. Par exemple, si la largeur du panneau conteneur est de 600px où cette page HTML est affichée. Dans ce cas, des barres de défilement horizontales peuvent apparaître si la largeur de la table générée est plus grande. Il était nécessaire que cette taille fixe soit changée en une unité scalable telle que em ou pourcentage pour obtenir une meilleure présentation. Le code d'exemple suivant peut être utilisé où [**HtmlSaveOptions.GetWidthScalable()**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/getwidthscalable/) est défini sur **true** pour créer une largeur scalable.
