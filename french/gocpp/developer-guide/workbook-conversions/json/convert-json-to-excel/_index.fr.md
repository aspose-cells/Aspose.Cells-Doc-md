Aspose.Cells supporte la conversion d'un fichier Json (JavaScript Object Notation) en classe de travail Excel. 

{{% /alert %}} 

## **Convertir JSON en classeur Excel** 
Pas besoin de se demander comment convertir JSON en fichier Excel, car la bibliothèque Aspose.Cells for C++ offre la meilleure solution. L'API Aspose.Cells prend en charge la conversion du format JSON en feuilles de calcul. Vous pouvez utiliser la classe [JsonLoadOptions](https://reference.aspose.com/cells/go-cpp/jsonloadoptions/) pour spécifier des paramètres supplémentaires pour l'importation JSON vers le classeur.
