## **Exporter des données XML liées à la carte XML à l'intérieur du classeur**

Veuillez utiliser la méthode [**Workbook::ExportXml()**](https://reference.aspose.com/cells/go-cpp/workbook/exportxml_string_string/) pour exporter les données XML liées à vos cartographies XML dans votre classeur. Le code d'exemple suivant exporte les données XML de toutes les cartographies XML du classeur une par une. Veuillez vérifier le [fichier Excel d'exemple](5115497.xlsx) utilisé dans ce code et les [données XML exportées de la première cartographie XML](5472487.xml).
