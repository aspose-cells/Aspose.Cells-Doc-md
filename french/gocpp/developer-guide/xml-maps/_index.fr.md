{{% alert color="primary" %}}

Aspose.Cells vous permet d'importer la carte XML dans le classeur en utilisant la méthode [**Workbook.ImportXml()**](https://reference.aspose.com/cells/go-cpp/workbook/importxml_string_string_int_int/). Vous pouvez importer la carte XML en utilisant Microsoft Excel avec les étapes suivantes :

- Sélectionnez l'onglet **Developer**.
- Cliquez sur **Importer** dans la section XML et suivez les étapes requises.

Vous devrez fournir vos données XML pour compléter l'importation. Voici un [exemple de données XML](5115037.txt) que vous pouvez utiliser pour les tests.
