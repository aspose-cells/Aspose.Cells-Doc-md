Aspose.Cells propose la capacité d'enregistrer des tableaux croisés dynamiques dans des fichiers ODS. Pour ce faire, vous pouvez soit convertir un classeur avec un tableau croisé dynamique existant, soit créer un nouveau tableau croisé dynamique et enregistrer le fichier au format ODS. Assurez-vous d'appeler [**PivotTable::CalculateData()**](https://reference.aspose.com/cells/go-cpp/pivottable/calculatedata/) avant l'enregistrement pour garantir que le tableau croisé dynamique est rendu dans le fichier ODS de sortie.
