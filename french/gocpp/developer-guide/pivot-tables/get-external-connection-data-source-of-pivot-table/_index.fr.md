## **Obtenir la source de données de connexion externe du tableau croisé dynamique**

Aspose.Cells permet d’obtenir la source de données de connexion extérieure du tableau croisé dynamique. Pour cela, l’API fournit la propriété [**GetExternalConnectionDataSource()**](https://reference.aspose.com/cells/go-cpp/pivottable/getexternalconnectiondatasource/) de la classe [**PivotTable**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/). La propriété [**GetExternalConnectionDataSource()**](https://reference.aspose.com/cells/go-cpp/pivottable/getexternalconnectiondatasource/) retourne l’objet [**ExternalConnection**](https://reference.aspose.com/cells/cpp/aspose.cells.externalconnections/externalconnection/). Le code ci-dessous montre comment utiliser la propriété [**PivotTable.GetExternalConnectionDataSource()**](https://reference.aspose.com/cells/go-cpp/pivottable/getexternalconnectiondatasource/) pour obtenir la source de connexion extérieure du tableau croisé dynamique.
