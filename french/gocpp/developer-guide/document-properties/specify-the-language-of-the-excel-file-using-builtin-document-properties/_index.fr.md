## **Scénarios d'utilisation possibles**

Vous pouvez changer la langue d'un fichier Excel en cliquant avec le bouton droit sur le fichier, en sélectionnant Propriétés > Détails, puis en modifiant le champ Langue. Utilisez la propriété [**BuiltInDocumentPropertyCollection.GetLanguage()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getlanguage/) pour le changer de manière programmatique avec les API Aspose.Cells.
