## **Scénarios d'utilisation possibles**  

Microsoft Excel vous permet de partager le classeur comme montré dans la capture d'écran suivante. Lorsque vous partagez le classeur, plus d'un utilisateur peut éditer le classeur sur le réseau. Aspose.Cells vous permet de créer un classeur partagé avec la propriété [**Workbook.GetShared()**](https://reference.aspose.com/cells/go-cpp/workbooksettings/getshared/).
