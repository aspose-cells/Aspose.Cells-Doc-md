{{% alert color="primary" %}}

Si vous souhaitez définir le texte du remplissage de l'entrée de légende du graphique afin qu'il ne s'affiche pas dans la légende du graphique, veuillez définir [**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/go-cpp/legendentry/istextnofill/) sur **true**.

{{% /alert %}}
