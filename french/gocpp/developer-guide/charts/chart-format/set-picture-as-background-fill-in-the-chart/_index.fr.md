{{% alert color="primary" %}}

Aspose.Cells vous permet de définir un dégradé, une texture, un motif ou une image comme effets de remplissage pour différents objets, tels que la zone de tracé, la zone du graphique ou la boîte de légende d'un graphique. Ce document montre comment ajouter une image en arrière-plan d'un graphique.

{{% /alert %}}

Pour réaliser cela, Aspose.Cells fournit la propriété [**Chart.GetImageData()**](https://reference.aspose.com/cells/go-cpp/fillformat/getimagedata/). L'exemple de code suivant démontre l'utilisation de la propriété [**Chart.GetImageData()**](https://reference.aspose.com/cells/go-cpp/fillformat/getimagedata/) pour définir une image comme remplissage d'arrière-plan dans le graphique.
