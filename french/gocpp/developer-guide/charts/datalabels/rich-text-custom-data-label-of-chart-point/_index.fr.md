{{% alert color="primary" %}}

Vous pouvez utiliser Aspose.Cells pour créer une étiquette de données personnalisée en texte enrichi pour le point du graphique. Aspose.Cells fournit la méthode [DataLabels.Characters()](https://reference.aspose.com/cells/go-cpp/charttextframe/characters/) pour retourner l’objet [FontSetting](https://reference.aspose.com/cells/cpp/aspose.cells/fontsetting/) qui peut être utilisé pour définir les propriétés de police du texte comme sa couleur, son gras, etc.
