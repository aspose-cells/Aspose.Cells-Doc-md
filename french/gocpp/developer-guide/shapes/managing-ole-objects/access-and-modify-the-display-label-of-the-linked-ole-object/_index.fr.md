## **Scénarios d'utilisation possibles**

Microsoft Excel vous permet de changer le libellé d'affichage de l'objet Ole comme montré dans la capture d'écran suivante. Vous pouvez également accéder ou modifier le libellé d'affichage de l'objet Ole en utilisant les API Aspose.Cells avec les méthodes [**GetLabel()**](https://reference.aspose.com/cells/go-cpp/oleobject/getlabel/) et [**SetLabel(const U16String\& value)**](https://reference.aspose.com/cells/cpp/aspose.cells.drawing/oleobject/setlabel/).
