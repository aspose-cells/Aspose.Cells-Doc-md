Aspose.Cells for Java est une bibliothèque de classes qui permet à vos applications d'effectuer une large gamme de tâches de traitement de feuille de calcul. Aspose.Cells for Java prend en charge les formats de fichier de feuille de calcul populaires (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) et permet d'exporter des fichiers Excel au format PDF, XPS, HTML, MHTML, Texte brut, TIFF, JPG, PNG, BMP et SVG.

La dernière référence API peut être trouvée à[Aspose.Cells for Java API Référence](https://reference.aspose.com/cells/java).
