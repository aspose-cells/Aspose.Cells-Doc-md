## **Introduction**
### **Eclipse IDE**
L'IDE Eclipse est un environnement de développement intégré Java (IDE) très connu. L'IDE est définitivement le produit le plus connu du projet Open Source Eclipse. Aujourd'hui, c'est l'environnement de développement principal pour Java avec une part de marché d'environ 60 %.

L'IDE Eclipse peut être étendu avec des composants logiciels supplémentaires. Eclipse appelle ces composants logiciels des Plug-ins. Plusieurs projets Open Source et entreprises ont étendu l'IDE Eclipse ou créé des applications autonomes (Eclipse RCP) sur la base du framework Eclipse.
### **Aspose.Cells for Java**
[Aspose.Cells for Java] (http://goo.gl/gtvhlQ) est un composant primé de feuille de calcul Excel qui permet aux développeurs Java d'intégrer la capacité de lire, écrire et manipuler des feuilles de calcul Excel® (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF et des formats de fichiers image dans leurs propres applications Java sans avoir besoin de compter sur Microsoft Excel®.

[Aspose.Cells for Java] (http://goo.gl/gtvhlQ) est un composant évolutif, riche en fonctionnalités et mature qui offre de nombreuses fonctions bien au-delà des simples capacités d'exportation de données des autres fournisseurs. Avec Aspose.Cells for Java, les développeurs peuvent exporter des données, formater des feuilles de calcul au niveau le plus granulaire, importer des images, créer des graphiques, appliquer et calculer des formules complexes, diffuser des données Excel®, enregistrer dans différents formats et bien plus encore - tout cela sans avoir besoin de Microsoft Excel® ou de l'automatisation de Microsoft Office.
### **Aspose.Cells Java for Eclipse (Maven)**
- Aspose.Cells Java for Eclipse (Maven) est un plugin pour **Eclipse IDE** par **Aspose.** Ce plugin est destiné aux développeurs utilisant la plateforme Maven pour les développements Java et souhaitant utiliser [Aspose.Cells for Java](http://goo.gl/gtvhlQ) dans leurs projets. Le plugin vous permet de créer des projets Maven pour utiliser l'API [Aspose.Cells for Java](http://goo.gl/gtvhlQ) et de télécharger également les [Exemples de code](https://goo.gl/uxznrb) de l'API. 
- Le plugin fournit les fonctionnalités suivantes pour travailler avec l'API [Aspose.Cells for Java](http://goo.gl/gtvhlQ) dans **Eclipse IDE** confortablement :

**WIZARDS**:
Le plugin contient deux assistants-----

**       ****Projet Maven Aspose.Cells (assistant)**

- Cet assistant de création de nouveau projet permet aux développeurs de créer un projet **Maven** pour utiliser [Aspose.Cells for Java](http://goo.gl/gtvhlQ) à partir de **Nouveau -> Projet -> Maven -> Projet Aspose.Cells Maven.**
- La référence de dépendance Maven de l'API [Aspose.Cells for Java](http://goo.gl/gtvhlQ) est automatiquement récupérée à partir du [Repository Maven Aspose Cloud](https://repository.aspose.com/webapp/) et ajoutée dans le pom.xml.
- Le projet créé contiendra toujours la version la plus récente de la dépendance **Maven** pour l'API [Aspose.Cells for Java](http://goo.gl/gtvhlQ).
- Les étapes de l'assistant présentent également l'option de télécharger les [Exemples de code](https://goo.gl/uxznrb) pour utiliser l'API [Aspose.Cells for Java](http://goo.gl/gtvhlQ).

**       ****Exemple de code Aspose.Cells (assistant)**

- Cet assistant de création de nouveau fichier vous permet de copier les [Exemples de code téléchargés](https://goo.gl/uxznrb) dans votre projet pour utiliser [Aspose.Cells for Java](http://goo.gl/gtvhlQ) à partir de **Nouveau -> Autre -> Java -> Exemple de code Aspose.Cells.** 
- Les échantillons disponibles sont affichés sous forme d'arborescence où l'utilisateur peut les sélectionner par catégorie.
- Tous les exemples dans la catégorie sélectionnée seront copiés dans le dossier "**com.aspose.cells.examples**" du package du projet ainsi que les ressources requises dans le dossier "**src/main/resources**" nécessaires pour exécuter les exemples.
- Les [Exemples de code](https://goo.gl/uxznrb) de l'API [Aspose.Cells for Java](http://goo.gl/gtvhlQ) sont destinés à démontrer diverses fonctionnalités de l'API.
- L'assistant recherchera également et mettra à jour les nouveaux [Exemples de code](https://goo.gl/uxznrb) disponibles à partir du [dépôt d'exemples Aspose.Cells for Java.](https://goo.gl/uxznrb)
## **Configuration requise et plateformes prises en charge**
### **Configuration requise**
- **Mémoire système :** 2 Go ou plus (recommandé)
- **OS :** Tout système d'exploitation prenant en charge la machine virtuelle Java (VM)
- **Connexion Internet :** 2 Mo ou plus rapide (recommandé)

- Eclipse Mars.1 (4.5.1) - Recommandé
- Eclipse Juno ou ultérieur.

### **Comment appliquer la licence Aspose ?**
Ce plugin utilise une version d'évaluation d'Aspose.Cells. Une fois que vous êtes satisfait de votre évaluation, vous pouvez acheter une licence sur le [site Web d'Aspose](https://purchase.aspose.com/buy).
Pour supprimer le message d'évaluation et les limitations des fonctionnalités, la licence du produit doit être appliquée. Vous recevrez un fichier de licence après avoir acheté le produit. Veuillez suivre les étapes ci-dessous pour appliquer la licence

- Assurez-vous que le fichier de licence est nommé **Aspose.Cells.Java.lic**
- Placez le fichier **Aspose.Cells.Java.lic** dans le dossier qui contient Aspose.Cells.jar
- Utilisez le code suivant pour activer la licence :

{{< highlight java >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **Comment configurer le code source d'Aspose.Cells Java pour Eclipse (Maven)**
Les étapes simples ci-dessous conduiront à une configuration réussie du code source du plugin **"Aspose.Cells Java pour Eclipse (Maven)"** dans l'IDE Eclipse

1. Téléchargez / Clonez le code source.
1. Choisissez **Fichier** > **Importation > Général > Projets Existants dans l'Espace de Travail**
1. Parcourez le dernier projet source que vous avez téléchargé
1. Sélectionnez le projet Eclipse que vous souhaitez importer
1. Cliquez sur **Terminer**
1. Le code du plugin Aspose.Cells Java pour Eclipse est maintenant prêt à être amélioré.
