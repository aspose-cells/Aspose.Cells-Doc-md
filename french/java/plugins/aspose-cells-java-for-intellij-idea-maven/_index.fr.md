## **Introduction**

### **IntelliJ IDEA**

**IntelliJ IDEA** est un environnement de développement intégré (IDE) Java pour les développeurs de logiciels. Il est développé par JetBrains. À l'origine créé pour Java, il prend actuellement en charge une grande variété de langages, dont JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure et Kotlin.

Dans un rapport **Infoworld** de 2010, IntelliJ a obtenu le score le plus élevé parmi les quatre meilleurs outils de programmation Java : Eclipse, IntelliJ IDEA, NetBeans et Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) pour Java est un composant de génération de rapports de feuilles de calcul Excel® qui permet aux applications Java de créer et de gérer des feuilles de calcul Excel® sans utiliser Microsoft Excel®. Aspose.Cells for Java est un composant incroyablement riche en fonctionnalités qui offre beaucoup plus que de simples fonctions d'exportation de données.

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java pour IntelliJ IDEA (Maven) est un **plugin IntelliJ IDEA** qui vous permet de créer des projets Maven (c.-à-d. le projet créé contient la dépendance maven API Aspose.Cells for Java) et de télécharger des exemples de code source de l'API pour utiliser l'**API Aspose.Cells for Java**.

- Ce plugin vous aide à travailler avec **l'API Aspose.Cells for Java** dans votre IntelliJ IDEA préféré.

![todo:image_alt_text](kwkgljg.png)

**WIZARDS**:
Le plugin contient deux assistants -----

**Assistant 1 :** Projet Maven Aspose.Cells

- L'**Assistant Projet Maven Aspose** du plugin permet aux développeurs de créer un projet Maven Aspose.Cells for Java.
- La référence de la dépendance maven de l'API Aspose.Cells for Java est automatiquement récupérée depuis [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) et ajoutée dans le pom.xml.
- Le projet créé contiendra toujours la version de la dépendance Maven la plus récente disponible pour Aspose.Cells for Java.
- Les étapes de l'assistant présentent également l'option de télécharger [Exemples de code source pour l'utilisation de l'API Aspose.Cells for Java](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

**Wizard 2 :** Exemples Aspose.Cells

- **L'assistant des exemples Aspose.Cells** vous permet de créer / copier des [Exemples de code source de l'API Aspose.Cells for Java](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) téléchargés dans votre projet. Tous les exemples de la catégorie sélectionnée seront copiés dans le dossier du package "com.aspose.cells.examples" du projet, ainsi que les ressources requises dans le dossier "src/main/resources" nécessaires pour exécuter les exemples.

- Les [Exemples de code source de l'API Aspose.Cells for Java](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) sont destinés à démontrer les différentes utilisations de l'API.

## **Configuration requise et plateformes prises en charge**

### **Configuration requise**

- Microsoft Windows 8/7/Vista/2003/XP (incl.64-bit) / bureau Linux GNOME ou KDE / Mac OS X 10.5 ou supérieur
- 1 Go de RAM minimum, 2 Go de RAM recommandés
- Espace disque dur de 300 Mo + au moins 1 Go pour les caches
- Résolution d'écran minimale 1024x768
- JDK 1.7 ou version ultérieure recommandée
- IntelliJ IDEA CE/UE (13.0.1, 13.1.2 ou 14.0.2)
- Connexion Internet (2 Mo ou plus recommandés)

### **Plateformes prises en charge**

Le plugin prend en charge les versions suivantes d'IntelliJ IDEA :

- 13.0.1 (Édition Communauté / Ultime)
- 13.1.2 (CE/ UE)
- 14.0.2 (CE/ UE)

## **Téléchargement**

### **Téléchargement de IntelliJ IDEA**

Vous devrez d'abord installer IntelliJ IDEA CE/UE avant de télécharger le plugin **Aspose.Cells Java pour IntelliJ IDEA (Maven)**.

Pour télécharger IntellliJ IDEA

1. Go to <https://www.jetbrains.com/idea/download/>
1. Sélectionnez votre OS en cours d'exécution
1. Cliquez sur le téléchargement UE ou CE.

### **Téléchargement du plugin Aspose.Cells Java pour IntelliJ IDEA (Maven)**

- Le plugin peut être téléchargé directement depuis l'[URL d'hébergement de plugin JetBrains](https://goo.gl/R4pysl) - sur le référentiel de plugins IntelliJ IDEA
  et peut ensuite être installé en utilisant l'option de l'IDE :
  - Allez dans **Fichier** > **Paramètres**...>**Plugins**
  - Sélectionnez "**Installer le plugin depuis le disque"**
- Ou il peut être installé directement (sans téléchargement manuel depuis l'URL d'hébergement de plugin JetBrains) en utilisant les options de l'IDE :
  - Allez dans **Fichier** > **Paramètres**...>**Plugins**
  - Sélectionnez **"Parcourir les référentiels"**
  - Dans la zone de recherche, tapez : "Aspose"
  - Sélectionnez le plugin "**Aspose Cells Java pour IntelliJ IDEA Maven**" à installer.
