## **Télécharger les bibliothèques requises**
Téléchargez les bibliothèques requises mentionnées ci-dessous. Ce sont les éléments requis pour exécuter les exemples Aspose.Cells Java for Python.

- **Aspose:** [Aspose.Cells for Java Composant](https://downloads.aspose.com/cells/java/)
- [JPype](https://pypi.org/project/JPype1/)
## **Télécharger des exemples à partir de sites de codage social**

## **Comment configurer le code source**
Veuillez suivre ces étapes simples afin d'ouvrir et d'étendre le code source lors de l'utilisation :

 \1. Installez uniquement Python 2.7.xx à partir de :<https://www.python.org/downloads/>
 \2. Installez JType à partir de :<https://sourceforge.net/projects/jpype/files/JPype/>
 \3. Installez Java JDK à partir de :<https://www.oracle.com/java/technologies/downloads/>
 \4. Téléchargez les API Aspose.Cells for Java que vous souhaitez utiliser dans vos projets Python à partir de :<https://downloads.aspose.com/cells/java> et copiez les .jars des API téléchargées dans le dossier %Python%/lib.
 \5. Cloner[Aspose.Cells Java for Python Exemples](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) de github.
