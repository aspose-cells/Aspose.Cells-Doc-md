## **JPype**

JPype est un effort visant à permettre aux programmes python un accès complet aux bibliothèques de classes Java. Cela est réalisé non pas en réimplémentant Python, comme l'a fait Jython/JPython, mais plutôt en interfaçant au niveau natif dans les deux machines virtuelles.

Lire la suite sur [readthedocs.org](https://jpype.readthedocs.io/en/latest/userguide.html)

## **Aspose.Cells for Java**

Aspose.Cells for Java est une bibliothèque de classes avancée pour Java qui vous permet d'effectuer une grande variété de tâches de traitement de documents directement dans vos applications Java.

Aspose.Cells for Java prend en charge le traitement des fichiers Excel (XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM), CSV (valeurs séparées par des virgules), TSV (valeurs séparées par des tabulations), TabDelimited (fichier texte à tabulation), Html, MHtml, ODS (OpenDocument Spreadsheet), SpreadsheetML et Numbers (document créé par l'application "Numbers" d'Apple). Avec Aspose.Cells, vous pouvez générer, modifier et convertir des documents sans utiliser Microsoft Excel.

## **Aspose.Cells Java for Jython**

Aspose.Cells Java pour Python est un projet qui démontre/fournit des exemples d'utilisation de l'API Aspose.Cells for Java en Python.

**Voici les configurations requises pour utiliser Aspose.Cells Java pour Python :**

- Composant Aspose.Cells téléchargé.
