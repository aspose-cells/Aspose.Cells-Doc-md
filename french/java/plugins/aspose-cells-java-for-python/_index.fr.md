## **JPype**

JPype est un effort pour permettre aux programmes python un accès complet aux bibliothèques de classes Java. Ceci est réalisé non pas en réimplémentant Python, comme l'a fait Jython/JPython, mais plutôt en interfaçant au niveau natif dans les deux machines virtuelles.

 Lire la suite sur[readthedocs.org](https://jpype.readthedocs.io/en/latest/userguide.html)

## **Aspose.Cells for Java**

Aspose.Cells for Java est une bibliothèque de classe avancée for Java qui vous permet d'effectuer une large gamme de tâches de traitement de documents directement dans vos applications Java.

Aspose.Cells for Java prend en charge le traitement Excel (XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM), CSV (valeurs séparées par des virgules), TSV (valeurs séparées par des tabulations), TabDelimited (fichier texte délimité par des tabulations), Html, MHtml, ODS ( OpenDocument Spreadsheet), SpreadsheetML et Numbers (le document est créé par l'application "Numbers" d'Apple). Avec Aspose.Cells, vous pouvez générer, modifier et convertir des documents sans utiliser Microsoft Excel.

## **Aspose.Cells Java pour Jython**

Aspose.Cells Java pour Python est un projet qui démontre/fournit les exemples d'utilisation Aspose.Cells for Java API dans Python.
