## **Insérer un sparkline**
{{% alert color="primary" %}} 

Sparkline est un petit graphique dans une cellule de feuille de calcul qui fournit une représentation visuelle des données. Utilisez des sparklines pour afficher les tendances d'une série de valeurs, telles que les augmentations ou les diminutions saisonnières, les cycles économiques, ou pour mettre en évidence les valeurs maximales et minimales. Positionnez un graphique sparkline près de ses données pour un impact maximal. Il existe trois types de graphique sparkline : Ligne, Colonne et Empilé.
