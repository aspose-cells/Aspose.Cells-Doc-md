## **Scénarios d'utilisation possibles**
Vous pouvez protéger par mot de passe le projet VBA (Visual Basic for Applications) du classeur avec Aspose.Cells en utilisant la méthode [VbaProject.protect()](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#protect-boolean-java.lang.String-).
## **Code d'exemple**
Le code d'exemple suivant charge le [fichier Excel d'exemple](43352069.xlsm), accède à son projet VBA, et le protège avec un mot de passe. Enfin, il l'enregistre en tant que [fichier Excel de sortie](43352070.xlsm).
