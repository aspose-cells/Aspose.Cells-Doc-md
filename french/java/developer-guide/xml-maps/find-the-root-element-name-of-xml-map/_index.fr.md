Vous pouvez trouver le *Nom de l'élément racine de la carte XML* en utilisant Aspose.Cells avec la propriété [**XmlMap.RootElementName**](https://reference.aspose.com/cells/java/com.aspose.cells/xmlmap#RootElementName).
