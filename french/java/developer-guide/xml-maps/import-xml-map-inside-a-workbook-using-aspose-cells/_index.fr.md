{{% alert color="primary" %}}

 Aspose.Cells vous permet d'importer une carte XML dans le classeur à l'aide de la[**Classeur.importXml()**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#importXml(java.lang.String,%20java.lang.String,%20int,%20int)) méthode. Vous pouvez importer une carte XML à l'aide du Microsoft Excel en suivant les étapes suivantes

-  Sélectionner**Développeur** languette
-  Cliquez sur**Importer** dans la section XML et suivez les étapes requises.

 Vous devrez fournir vos données XML pour terminer l'importation. Voici une[exemples de données XML](5472501.txt) que vous pouvez utiliser pour les tests.
