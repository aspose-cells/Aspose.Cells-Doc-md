{{% alert color="primary" %}}

Aspose.Cells vous permet d'importer une carte XML à l'intérieur du classeur à l'aide de la méthode [**Workbook.importXml()**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#importXml-java.lang.String-java.lang.String-int-int-). Vous pouvez importer une carte XML à l'aide de Microsoft Excel en suivant les étapes suivantes

- Sélectionnez l'onglet **Développeur**
- Cliquez sur **Importer** dans la section XML et suivez les étapes requises.

Vous devrez fournir vos données XML pour compléter l'importation. Voici un [exemple de données XML](5472501.txt) que vous pouvez utiliser pour des tests.
