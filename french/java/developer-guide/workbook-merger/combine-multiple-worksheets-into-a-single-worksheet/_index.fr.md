Parfois, vous devez combiner plusieurs feuilles de calcul en une seule. Cela peut facilement être réalisé en utilisant Aspose.Cells API. Cet article vous montrera un exemple de code qui lit un classeur source et combine les données de toutes les feuilles de calcul source en une seule feuille de calcul dans un classeur de destination.

{{% /alert %}}

##  **Comment combiner des feuilles de calcul**

 L'exemple ci-dessous utilise le[**Range.copie()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#copy(com.aspose.cells.Range)) méthode pour copier toutes les feuilles de calcul source sur une seule feuille à l’intérieur d’un classeur de destination.
