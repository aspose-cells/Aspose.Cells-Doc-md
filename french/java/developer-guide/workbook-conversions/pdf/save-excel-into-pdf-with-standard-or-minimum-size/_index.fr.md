{{% alert color="primary" %}} 

 Par défaut, Aspose.Cells enregistre Excel au format PDF avec une taille standard. Cependant, vous pouvez également l'enregistrer avec la taille minimale en utilisant le[PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions#OptimizationType) propriété. Il accepte les valeurs suivantes

- [PdfOptimizationType.STANDARD](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#STANDARD)
- [PdfOptimizationType.MINIMUM_SIZE](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#MINIMUM_SIZE)
