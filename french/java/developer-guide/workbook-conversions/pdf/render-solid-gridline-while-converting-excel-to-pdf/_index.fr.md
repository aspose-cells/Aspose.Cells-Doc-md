Pour la compatibilité avec les anciennes versions, Aspose.Cells rend les lignes de grille comme ligne en pointillé par défaut lors de la conversion d'Excel en PDF. Cependant, de nos jours, Excel moderne rend les lignes de grille comme des lignes pleines.

Avec l'option [PdfSaveOptions.GridlineTypes](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setGridlineType-int-), Aspose.Cells peut également rendre la grille comme une ligne solide.
