## **Scénarios d'utilisation possibles**

Lorsque vous enregistrez votre fichier Excel dans HTML, les commentaires ne sont pas exportés. Cependant, Aspose.Cells fournit cette fonctionnalité en utilisant le[**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#IsExportComments)la propriété. Si vous le réglez**vrai**alors HTML affichera également les commentaires présents dans votre fichier Excel.
