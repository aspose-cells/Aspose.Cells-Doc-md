{{% alert color="primary" %}}

 Aspose.Cells fournit une propriété utile,[**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference) que les développeurs peuvent utiliser pour obtenir une meilleure mise en page lors de l'enregistrement d'un fichier Excel Microsoft au format HTML ou MHT. La valeur par défaut de la propriété est**faux** . Nous vous recommandons de définir cette propriété sur**vrai** pour obtenir de meilleures présentations à partir d'un rapport Excel.
