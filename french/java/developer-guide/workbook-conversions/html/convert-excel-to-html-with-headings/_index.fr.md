## **Convertir Excel en HTML avec en-têtes**

Aspose.Cells offre la possibilité d'exporter les en-têtes lors de la conversion d'Excel en HTML. Pour cela, vous pouvez définir le[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings) propriété à**vrai**. La valeur par défaut de[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings)la propriété est**faux**.
