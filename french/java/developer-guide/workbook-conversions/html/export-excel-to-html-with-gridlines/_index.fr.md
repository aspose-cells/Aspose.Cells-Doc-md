{{% alert color="primary" %}} 

Si vous souhaitez exporter votre fichier Excel au format HTML avec des grilles, veuillez utiliser la propriété [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) et la définir sur **vrai**.

{{% /alert %}} 
## **Exporter Excel au format HTML avec des lignes de grille**
Le code d'exemple suivant crée un classeur et remplit sa feuille de calcul avec certaines valeurs, puis le sauvegarde au format HTML après avoir défini la [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) sur **true**.

La capture d'écran suivante montre le HTML de sortie généré avec ce code d'exemple. Comme vous pouvez le voir, il affiche également des grilles dans le HTML de sortie.
