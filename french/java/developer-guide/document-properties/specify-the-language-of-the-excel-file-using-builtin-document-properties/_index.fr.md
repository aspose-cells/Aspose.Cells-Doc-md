## **Scénarios d'utilisation possibles**

Vous pouvez changer le*Langue*du fichier Excel en cliquant avec le bouton droit sur le fichier, puis en sélectionnant*Propriétés > Détails*puis en éditant le*Langue*domaine. Veuillez utiliser[**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#Language)propriété pour le modifier par programmation à l'aide des API Aspose.Cells.
