## **Scénarios d'utilisation possibles**

Vous pouvez changer le*Numéro de version*du fichier Excel en cliquant avec le bouton droit sur le fichier, puis en sélectionnant*Propriétés > Détails*puis en éditant le*Numéro de version*champ. Veuillez utiliser[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#DocumentVersion)propriété pour le modifier par programmation à l'aide des API Aspose.Cells.
