## **Scénarios d'utilisation possibles**

Numbers est une application de feuille de calcul développée par Apple Inc. Aspose.Cells peut lire la feuille de calcul Numbers mais ne prend pas en charge l'écriture. Il peut lire les données, le style et les formules de la feuille de calcul Numbers.

## **Lire la feuille de calcul des nombres Développé par Apple Inc. en utilisant Aspose.Cells**

 L'exemple de code suivant charge le[Feuille de calcul des nombres d'échantillons](sampleNumbersByAppleInc.numbers) et le convertit en[Format de sortie PDF](outputNumbersByAppleInc.pdf) . Vous devrez utiliser**[LoadOptions](https://reference.aspose.com/cells/java/com.aspose.cells/LoadOptions)** classer et préciser**[LoadFormat.NUMBERS](https://reference.aspose.com/cells/java/com.aspose.cells/loadformat#NUMBERS)**comme paramètre dans son constructeur pour le charger avec succès. Veuillez les télécharger tous les deux à partir des liens indiqués. Vous pouvez essayer le code avec n'importe quelle feuille de calcul Numbers. Veuillez également lire les commentaires du code pour plus d'aide.
