## **Scénarios d'utilisation possibles**

Dans certaines situations, vous souhaiterez peut-être modifier les propriétés du segment, telles que le placement ou la hauteur des lignes. Aspose.Cells vous offre la possibilité de mettre à jour ces propriétés.

## **Modifier les propriétés du segment**

 Veuillez consulter l'exemple de code suivant. Il charge le[exemple de fichier Excel](sampleCreateSlicerToExcelTable.xlsx) qui contient un tableau. Il crée ensuite le segment en fonction de la première colonne et modifie ses propriétés telles que la hauteur de ligne, la largeur, est imprimable, le titre, etc. Il enregistre le classeur sous[outputChangeSlicerProperties.xlsx](outputChangeSlicerProperties.xlsx).
