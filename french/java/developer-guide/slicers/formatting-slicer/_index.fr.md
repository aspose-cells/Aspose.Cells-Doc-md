## **Scénarios d'utilisation possibles**
Vous pouvez formater la trancheuse dans Microsoft Excel en configurant son nombre de colonnes ou en définissant son style, etc. Aspose.Cells vous permet également de le faire en utilisant les propriétés [Slicer.NumberOfColumns](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#NumberOfColumns) et [Slicer.StyleType](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#StyleType).
## **Formatage d'un tronçonneur**
Veuillez consulter le code suivant, il charge le [fichier Excel d'exemple](67338501.xlsx) qui contient une trancheuse. Il accède à la trancheuse, définit son nombre de colonnes et son type de style, puis l'enregistre au format [fichier Excel de sortie](67338500.xlsx).
