{{% alert color="primary" %}} 

Parfois, vous souhaiterez peut-être récupérer des informations sur des sources de données externes. Par exemple, vous souhaiterez peut-être voir les données de connexion SQL. Ces informations peuvent inclure tous les types de données nécessaires pour se connecter à un serveur SQL, par exemple l'URL du serveur, le nom d'utilisateur, le nom de la table sous-jacente, la requête SQL, le type de requête, l'emplacement de la table, etc.

Aspose.Cells fournissent des appels utiles pour récupérer ces détails à partir de bases de données d'entreprise liées à des modèles de fichiers Excel.

{{% /alert %}} 
## **Récupération des détails des sources de données externes**
L'exemple suivant montre comment obtenir une connexion à la base de données et d'autres détails. L'exemple utilise un simple fichier Excel qui contient des liens vers une source de données externe (SQL Server).

Lorsque le code s'exécute, les détails de connexion sont imprimés sur la console.
