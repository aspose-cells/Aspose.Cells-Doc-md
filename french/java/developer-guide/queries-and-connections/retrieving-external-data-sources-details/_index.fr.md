{{% alert color="primary" %}} 

Parfois, vous voudrez peut-être récupérer des informations sur les sources de données externes. Par exemple, vous voudrez peut-être voir les données de connexion SQL. Ces informations peuvent inclure tout type de données requis pour se connecter à un serveur SQL, par exemple l'URL du serveur, le nom d'utilisateur, le nom de la table sous-jacente, la requête SQL, le type de requête, l'emplacement de la table, etc.

Aspose.Cells fournit des appels utiles pour récupérer de telles informations à partir de bases de données d'entreprise liées à des fichiers Excel modèles.

{{% /alert %}} 
## **Récupération des détails des sources de données externes**
L'exemple suivant montre comment obtenir une connexion à une base de données et d'autres détails. L'exemple utilise un simple fichier Excel contenant des liens vers une source de données externe (SQL Server).

Lorsque le code s'exécute, les détails de la connexion sont imprimés dans la console.
