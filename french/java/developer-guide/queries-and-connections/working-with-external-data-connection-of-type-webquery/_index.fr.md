{{% alert color="primary" %}} 

Vous pouvez accéder à la connexion de données externes de n'importe quel type en utilisant la collection [Workbook.DataConnections](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections). Un type de cette connexion de données est WebQuery. Cet article vous montrera comment travailler avec la connexion de données WebQuery. Vous pouvez créer une connexion de données WebQuery dans Microsoft Excel à l'aide du menu **Données** > **A partir du Web**.
