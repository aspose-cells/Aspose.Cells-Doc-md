{{% alert color="primary" %}}

Microsoft Excel fournit une fonctionnalité qui vous permet de dimensionner automatiquement la hauteur d'une cellule en fonction de son contenu. La fonctionnalité est appelée lignes d'ajustement automatique. Microsoft Excel ne définit pas nativement l'opération d'ajustement automatique sur les cellules fusionnées. Parfois, la fonctionnalité devient vitale pour un utilisateur qui a vraiment besoin d'implémenter des lignes d'ajustement automatique sur des cellules fusionnées également.

{{% /alert %}}

 Aspose.Cells prend en charge cette fonctionnalité via le[**AutoFitterOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/AutoFitterOptions)API. À l'aide de ce API, il est possible d'ajuster automatiquement les lignes d'une feuille de calcul comprenant des cellules fusionnées.

Après avoir exécuté le code, Aspose.Cells ajuste automatiquement les lignes pour les cellules fusionnées.
