## Scénarios d'utilisation possibles

Si vous voulez lire la couleur de l'effet de lueur de n'importe quelle forme, veuillez utiliser la propriété [**Shape.Glow.Color**](https://reference.aspose.com/cells/java/com.aspose.cells/gloweffect#Color). Cela vous aidera à trouver les différentes propriétés liées à la couleur de l'effet de lueur appliqué à une forme.
