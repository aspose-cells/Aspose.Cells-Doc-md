## **Scénarios d'utilisation possibles**
 Aspose.Cells fournit le[Shape.ThreeDFormat](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ThreeDFormat)propriété pour travailler avec le format 3D de la forme ou du graphique. Il contient différentes sous-propriétés que vous pouvez définir pour obtenir différents résultats selon vos besoins.

La capture d'écran suivante montre l'interface Excel Microsoft pour définir le format 3D de la forme.
