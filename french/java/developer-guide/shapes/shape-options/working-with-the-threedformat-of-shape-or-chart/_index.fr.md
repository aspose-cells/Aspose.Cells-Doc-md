## **Scénarios d'utilisation possibles**
Aspose.Cells fournit la propriété [Shape.ThreeDFormat](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ThreeDFormat) pour travailler avec le format 3D de la forme ou du graphique. Elle contient différentes sous-propriétés que vous pouvez définir pour obtenir différents résultats selon vos besoins.

La capture d'écran suivante montre l'interface de Microsoft Excel pour définir le format 3D de la forme.
