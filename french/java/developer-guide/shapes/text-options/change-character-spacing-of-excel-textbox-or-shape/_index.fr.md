Vous pouvez modifier l'espacement des caractères d'une zone de texte ou d'une forme Excel en utilisant la propriété [FontSetting.TextOptions.Spacing](https://reference.aspose.com/cells/java/com.aspose.cells/textoptions#Spacing).
