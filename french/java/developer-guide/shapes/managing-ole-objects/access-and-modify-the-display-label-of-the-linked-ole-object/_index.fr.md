## **Scénarios d'utilisation possibles**

Microsoft Excel vous permet de changer l'étiquette d'affichage de l'objet Ole comme le montre la capture d'écran suivante. Vous pouvez également accéder ou modifier l'étiquette d'affichage de l'objet Ole en utilisant les API Aspose.Cells avec la propriété [**OleObject.Label**](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#Label).
