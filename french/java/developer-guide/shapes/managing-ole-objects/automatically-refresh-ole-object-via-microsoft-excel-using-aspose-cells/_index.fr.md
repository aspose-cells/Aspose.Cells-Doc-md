{{% alert color="primary" %}} 

Aspose.Cells fournit la propriété [OleObject.AutoLoad](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad) pour actualiser l'objet OLE lorsque le fichier Excel est ouvert dans Microsoft Excel. En raison de cette propriété, l'objet OLE affichera l'image OLE correcte générée par Microsoft Excel.

{{% /alert %}} 
## **Rafraîchir automatiquement un objet OLE via Microsoft Excel en utilisant Aspose.Cells**
Le code d'exemple suivant charge le [fichier Excel d'exemple](5473423.xlsx) qui contient une image OLE non réelle. L'objet OLE est en réalité un document Microsoft Word mais le fichier Excel d'exemple montre l'image de l'animal au lieu de l'image de Microsoft Word. Mais si vous ouvrez le [fichier Excel de sortie](5473429.xlsx), vous verrez que Microsoft Excel affiche l'image OLE correcte.
