{{% alert color="primary" %}} 

 Aspose.Cells fournit le[OleObject.AutoLoad](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad) propriété pour actualiser l'objet OLE lorsque le fichier Excel est ouvert dans Microsoft Excel. En raison de cette propriété, l'objet OLE affichera l'image OLE correcte générée par Microsoft Excel.

{{% /alert %}} 
## **Actualiser automatiquement l'objet OLE via Microsoft Excel en utilisant Aspose.Cells**
 L'exemple de code suivant charge le[exemple de fichier excel](5473423.xlsx) qui a une image OLE non réelle. L'objet OLE est en fait un document Word Microsoft, mais l'exemple de fichier Excel montre l'image de l'animal au lieu de l'image Word Microsoft. Mais si vous ouvrez le[fichier excel de sortie](5473429.xlsx), vous verrez Microsoft Excel affiche l'image OLE correcte.
