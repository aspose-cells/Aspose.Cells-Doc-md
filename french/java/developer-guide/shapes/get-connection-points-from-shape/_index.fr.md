Aspose.Cells propose des fonctionnalités riches pour gérer les formes dans la feuille de calcul. Parfois, il est nécessaire d'obtenir les points de connexion d'une forme pour aligner ou placer les formes au bon endroit. Dans ce but, tous les points de connexion sont nécessaires. Le code suivant peut être utilisé pour obtenir la liste des points de connexion d'une forme où la fonction [**Shape.GetConnectionPoints()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ConnectionPoints) est utilisée.
