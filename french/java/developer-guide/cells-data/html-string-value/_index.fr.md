{{% alert color="primary" %}}

Aspose.Cells prend en charge la conversion de Microsoft HTML orienté Excel au format XLS/XLSX. Cela signifie que le code HTML généré par Microsoft Excel peut être reconverti au format XLS/XLSX à l'aide de Aspose.Cells.

 De même, s'il existe du code HTML simple, Aspose.Cells peut le convertir en texte enrichi HTML. Aspose.Cells fournit[**Cell.setHtmlString()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#HtmlString) propriété qui peut prendre un code HTML aussi simple et le convertir en texte de cellule formaté.
