## **Obtenir la largeur du texte de la valeur Cell**

Parfois, les développeurs peuvent avoir besoin de calculer la largeur de la valeur de la cellule pour organiser les données dans une mise en page. Aspose.Cells fournit le[**CellsHelper.GetTextWidth**](https://reference.aspose.com/cells/java/com.aspose.cells/cellshelper#getTextWidth(java.lang.String,%20com.aspose.cells.Font,%20double)) qui permet aux développeurs d'obtenir la largeur du texte de la valeur de la cellule. L'exemple de code suivant illustre l'utilisation[**CellsHelper.GetTextWidth**](https://reference.aspose.com/cells/java/com.aspose.cells/cellshelper#getTextWidth(java.lang.String,%20com.aspose.cells.Font,%20double)) pour accéder à la largeur du texte de la valeur de la cellule.
