## **Obtenir des hyperliens dans la plage**

La classe [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/range) fournit un accesseur de propriété [**getHyperlinks()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#Hyperlinks) qui obtient tous les hyperliens dans la plage sélectionnée. Vous pouvez également supprimer le Hyperlink en appelant la méthode [**Hyperlink.delete()**](https://reference.aspose.com/cells/java/com.aspose.cells/hyperlink#delete--).
