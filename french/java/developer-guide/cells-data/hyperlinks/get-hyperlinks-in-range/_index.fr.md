## **Obtenir des hyperliens dans la plage**

La[**Intervalle**](https://reference.aspose.com/cells/java/com.aspose.cells/range) la classe offre une[**getHyperlinks()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#Hyperlinks) propriété getter qui obtient tous les liens hypertexte dans la plage sélectionnée. Vous pouvez également supprimer le lien hypertexte en appelant le[**Lien hypertexte.delete()**](https://reference.aspose.com/cells/java/com.aspose.cells/hyperlink#delete()) méthode.
