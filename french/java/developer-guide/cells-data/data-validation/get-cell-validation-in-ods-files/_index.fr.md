## **Obtenir la validation Cell dans les fichiers ODS**

Avec Aspose.Cells, vous pouvez obtenir la validation appliquée à une cellule dans les fichiers ODS. Pour cela, le API fournit le[**ObtenirValidation**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getValidation()) méthode de la[**Cell**](https://reference.aspose.com/cells/java/com.aspose.cells/Cell)classer.
