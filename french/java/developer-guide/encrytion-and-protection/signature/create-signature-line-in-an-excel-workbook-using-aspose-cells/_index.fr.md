{{% alert color="primary" %}}

Microsoft Excel permet d'ajouter une **ligne de signature** dans les classeurs Excel. Vous pouvez ajouter une ligne de signature en cliquant sur l'onglet **Insertion** puis en sélectionnant **Ligne de signature** dans le groupe **Texte**.

Aspose.Cells propose également cette fonctionnalité et a exposé la propriété [**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) à cette fin. Cet article expliquera comment utiliser cette propriété pour ajouter une ligne de signature en utilisant Aspose.Cells.

{{% /alert %}}
