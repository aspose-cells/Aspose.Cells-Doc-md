## **Scénarios d'utilisation possibles**

Vous pouvez spécifier le nom de l'auteur tout en protégeant votre classeur en écriture en utilisant Aspose.Cells API. Veuillez utiliser[**Workbook.Settings.WriteProtection.Author**](https://reference.aspose.com/cells/java/com.aspose.cells/writeprotection#Author)propriété à cet effet.
