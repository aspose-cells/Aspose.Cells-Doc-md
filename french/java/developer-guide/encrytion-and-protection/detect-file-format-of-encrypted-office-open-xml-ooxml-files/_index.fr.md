**Office XML ouvert**(aussi connu sous le nom**OOXML**ou alors**Microsoft XML ouvert**(MOX)) est un format de fichier basé sur XML développé par Microsoft pour représenter des documents de bureau tels que des feuilles de calcul, des graphiques, des présentations et des documents de traitement de texte.

{{% /alert %}} 



Aspose.Cells fournit un moyen de détecter le format de fichier de crypté**Microsoft XML ouvert**des dossiers. Pour identifier le type de fichier, utilisez le[FileFormatUtil.detectFileFormat](https://reference.aspose.com/cells/java/com.aspose.cells/fileformatutil#detectFileFormat\(java.lang.String,%20java.lang.String\)) comme indiqué ci-dessous dans l'exemple de code.
