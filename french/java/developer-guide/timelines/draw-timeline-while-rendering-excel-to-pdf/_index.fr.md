## **Dessiner une chronologie lors du rendu d'Excel en PDF**
Si vous avez un fichier Excel auquel une chronologie est appliquée et que vous souhaitez exporter l'Excel au format PDF avec les paramètres de chronologie, Aspose.Cells for Java le prend en charge par défaut. Vous exportez simplement le fichier Excel avec la chronologie au format PDF, le PDF généré affichera la chronologie appliquée.

Le code d'exemple suivant charge le [fichier Excel d'exemple](input.xlsx) qui contient une chronologie existante. Il enregistre ensuite le classeur sous la forme de [fichier PDF de sortie](out.pdf). La capture d'écran suivante compare le fichier Excel source et le fichier PDF généré.
