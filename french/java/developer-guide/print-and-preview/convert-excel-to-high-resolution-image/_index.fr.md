Avec la prévalence croissante des écrans haute résolution, les images générées à la résolution par défaut de 96 DPI semblent souvent floues et peu claires. Pour garantir la clarté sur les écrans haute résolution, il est essentiel de générer des images à une DPI plus élevée. Aspose.Cells offre la fonctionnalité pour définir [**ImageOrPrintOptions.HorizontalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setHorizontalResolution-int-) et [**ImageOrPrintOptions.VerticalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setVerticalResolution-int-), ce qui vous permet de créer des images de haute qualité à partir de fichiers Excel qui semblent nets sur les écrans haute résolution.
