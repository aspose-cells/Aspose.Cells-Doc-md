## **Scénarios d'utilisation possibles**

Généralement, vous spécifiez le répertoire des polices ou la liste des polices pour tous les classeurs, mais parfois, vous devez spécifier un ensemble individuel ou privé de polices pour vos classeurs. Aspose.Cells fournit la classe [**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs) qui peut être utilisée pour spécifier l'ensemble individuel ou privé de polices pour votre classeur.
