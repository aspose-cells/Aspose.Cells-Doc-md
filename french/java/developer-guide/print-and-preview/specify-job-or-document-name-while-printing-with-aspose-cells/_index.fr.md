{{% alert color="primary" %}}

 Vous pouvez spécifier le nom du travail ou du document lors de l'impression de votre classeur ou de votre feuille de calcul à l'aide de la[**ClasseurRender**](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookRender) ou[**FeuilleRendu**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender) objets. Aspose.Cells fournit le[**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ) et[**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) méthodes que vous pouvez utiliser pour spécifier le nom du travail lors de l'impression de votre classeur ou feuille de calcul

{{% /alert %}}

## **Spécifiez le nom du travail ou du document lors de l'impression avec Aspose.Cells**

 L'exemple de code charge le fichier Excel source, puis l'envoie à l'imprimante en spécifiant le nom du travail ou du document à l'aide de la[**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ) et[**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) méthodes. La capture d'écran montre à quoi ressemble le nom du travail dans la file d'attente de l'imprimante.
