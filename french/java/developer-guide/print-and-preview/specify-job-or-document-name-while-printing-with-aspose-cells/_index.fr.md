{{% alert color="primary" %}}

Vous pouvez spécifier un nom de tâche ou de document lors de l'impression de votre classeur ou de votre feuille de calcul en utilisant les objets [**WorkbookRender**](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookRender) ou [**SheetRender**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender). Aspose.Cells fournit les méthodes [**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String)) et [**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) que vous pouvez utiliser pour spécifier le nom de la tâche lors de l'impression de votre classeur ou de votre feuille de calcul.

{{% /alert %}}

## **Spécifier le nom du travail ou du document lors de l'impression avec Aspose.Cells**

Le code d'exemple charge le fichier Excel source, puis l'envoie à l'imprimante en spécifiant le nom de la tâche ou du document à l'aide des méthodes [**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String)) et [**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)). La capture d'écran montre à quoi ressemble le nom de la tâche dans la file d'attente de l'imprimante.
