{{% alert color="primary" %}}

Un graphique en cascade est un type spécial de graphique qui est normalement utilisé pour montrer comment la position de départ augmente ou diminue. Microsoft Excel propose de nombreux types de graphiques prédéfinis, notamment les colonnes, les lignes, les secteurs, les barres, les radars, etc., mais le graphique en cascade va au-delà des graphiques de base et peut être créé à l'aide des types de graphiques existants avec peu ou plus de personnalisation.

{{% /alert %}}

## Création d'un graphique en cascade

 Aspose.Cells Les API permettent de créer un graphique en cascade à l'aide d'un graphique linéaire.
