## **Scénarios d'utilisation possibles**
Aspose.Cells prend en charge la lecture et la manipulation des graphiques Microsoft Excel 2016 qui ne sont pas présents dans Microsoft Excel 2013 ou les versions antérieures.
## **Lire et manipuler les graphiques Excel 2016**
Le code d'exemple suivant charge le [fichier excel source](23166980.xlsx) qui contient des graphiques Microsoft Excel 2016 dans la première feuille de calcul. Il lit tous les graphiques un par un et modifie son titre selon son type de graphique. La capture d'écran suivante montre le fichier excel source avant l'exécution du code. Comme vous pouvez le voir, le titre du graphique est le même pour tous les graphiques.

![todo:image_alt_text](read-and-manipulate-excel-2016-charts_1.png)

La capture d'écran suivante montre le [fichier excel de sortie](23166978.xlsx) après l'exécution du code. Comme vous pouvez le voir, le titre du graphique est modifié selon son type de graphique.
