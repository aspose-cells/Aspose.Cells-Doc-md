##  **Scénarios d'utilisation possibles**
Les graphiques combinés dans Excel vous permettent de bénéficier de cette option car vous pouvez facilement combiner deux ou plusieurs types de graphiques pour rendre vos données compréhensibles. Les graphiques combinés sont utiles lorsque vos données contiennent plusieurs types de valeurs, notamment le prix et le volume. De plus, les graphiques combinés sont réalisables lorsque vos numéros de données changent considérablement d'une série à l'autre.
