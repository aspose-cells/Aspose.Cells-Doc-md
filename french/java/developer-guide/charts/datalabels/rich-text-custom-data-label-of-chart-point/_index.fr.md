{{% alert color="primary" %}}

 Vous pouvez utiliser Aspose.Cells pour créer une étiquette de données personnalisée en texte enrichi du point de graphique. Aspose.Cells fournit le[**DataLabels.characters()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#characters(int,%20int) ) méthode pour retourner le[**FontSetting**](https://reference.aspose.com/cells/java/com.aspose.cells/FontSetting)objet qui peut être utilisé pour définir les propriétés de la police du texte comme sa couleur, sa mise en gras, etc.

{{% /alert %}}
