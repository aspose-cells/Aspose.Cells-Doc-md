{{% alert color="primary" %}} 

 Vous pouvez ajouter des étiquettes personnalisées aux points de données dans la série du graphique. Aspose.Cells fournit[ChartPoint.getDataLabels().setText()](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#Text)propriété pour ajouter ces étiquettes personnalisées. Cet article explique comment utiliser cette propriété pour ajouter des étiquettes personnalisées aux points de données dans la série du graphique.

{{% /alert %}} 
## **Ajout d'étiquettes personnalisées aux points de données dans la série du graphique**
Le code suivant crée un graphique en nuage de points connecté par des lignes avec des marqueurs de données, puis ajoute des étiquettes personnalisées aux points de données dans la série du graphique. Chaque étiquette personnalisée affiche le nom de la série et le nom du point. Vous pouvez utiliser n'importe quel autre texte à sa place. Après l'exécution du code, le fichier Excel suivant est créé. Comme vous pouvez le voir à l'intérieur du graphique, chaque point de données a un ensemble d'étiquettes personnalisées.
