{{% alert color="primary" %}} 

Si vous souhaitez définir le texte du remplissage de l'entrée de légende du graphique sur aucun pour qu'il ne s'affiche pas à l'intérieur de la légende du graphique, veuillez définir [LegendEntry.IsTextNoFill](https://reference.aspose.com/cells/java/com.aspose.cells/legendentry#IsTextNoFill) à **true**.

{{% /alert %}}
