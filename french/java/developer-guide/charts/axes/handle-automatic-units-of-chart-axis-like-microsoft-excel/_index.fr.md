## **Scénarios d'utilisation possibles**
Les premières versions de Aspose.Cells n'étaient pas en mesure de gérer correctement les unités automatiques d'axe de graphique lorsque le graphique était rendu en image ou PDF. Désormais, Aspose.Cells prend en charge la gestion des unités automatiques d'axe de graphique. Il n'y a pas de changement de code. Convertissez simplement votre graphique en image ou PDF et il rendra l'axe du graphique comme Microsoft Excel le rend.
