{{% alert color="primary" %}}

Dans les graphiques Excel, il existe 3 types d'axes :
1. Axe Horizontal (Catégorie) : Axe des X
1. Axe Vertical (Valeur) : Axe des Y
1. Axe de Profondeur (Série) : Axe des Z



{{% /alert %}}

## **Options d'Axe**
Aspose.Cells permet également de gérer les axes du graphique en cours d'exécution. Avec l'objet [Axis](https://reference.aspose.com/cells/java/com.aspose.cells/Axis), vous pouvez modifier toutes les options de l'axe comme dans Excel.
