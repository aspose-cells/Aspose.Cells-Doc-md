{{% alert color="primary" %}} 

Microsoft Excel prend en charge deux systèmes de date : le système de date 1900 (le système de date par défaut implémenté dans Excel pour Windows) et le système de date 1904. Le système de date 1904 est normalement utilisé pour assurer la compatibilité avec les fichiers Macintosh Excel et est le système par défaut si vous utilisez Excel pour Macintosh. Vous pouvez définir le système de date 1904 pour les fichiers Excel en utilisant Aspose.Cells.

{{% /alert %}} 

Pour implémenter le système de date 1904 dans Microsoft Excel (par exemple Microsoft Excel 2003) :

1.  Du**Outils** menu, sélectionnez**Choix** , et sélectionnez le**Calcul** languette.
1.  Sélectionnez le**système de date de 1904** option.
1.  Cliquez sur**D'ACCORD**.
