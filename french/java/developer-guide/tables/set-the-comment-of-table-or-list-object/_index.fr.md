## **Définir le commentaire de la table ou de l'objet de liste dans la feuille de calcul**

 Veuillez définir le commentaire du tableau ou de l'objet de liste dans la feuille de calcul à l'aide du[**ListObject.Comment**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment)la propriété. Le commentaire sera visible dans le fichier xl/tables/tableName.xml.

 L'exemple de code suivant charge le[fichier excel source](5472561.xlsx), définit le commentaire du premier objet de table ou de liste à l'intérieur de la feuille de calcul et la capture d'écran montre le commentaire créé par ce code dans le rectangle rouge.
