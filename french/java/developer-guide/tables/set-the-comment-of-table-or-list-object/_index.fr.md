## **Définir le Commentaire d'un Tableau ou Objet Liste à l'intérieur de la Feuille de Calcul**

Veuillez définir le commentaire de la Table ou de l'objet Liste à l'intérieur de la feuille de calcul en utilisant la propriété [**ListObject.Comment**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment). Le commentaire sera visible dans le fichier xl/tables/tableName.xml.

Le code d'exemple suivant charge le [fichier excel source](5472561.xlsx), définit le commentaire de la première table ou de l'objet liste à l'intérieur de la feuille de calcul et la capture d'écran montre le commentaire créé par ce code dans le rectangle rouge.
