{{% alert color="primary" %}}

 Vous pouvez utiliser Aspose.Cells pour fusionner ou diviser une plage de cellules. Aspose.Cells fournit le[**Range.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge() ) et[**Range.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge()) méthodes à cet effet. Cet article explique comment fusionner une plage de cellules en une seule cellule.

{{% /alert %}}

 L'exemple de code suivant crée d'abord une plage - A1:D4 - puis fusionne les cellules de la plage en une seule cellule à l'aide de la[**Range.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge()) méthode.
 De même, il est possible de fractionner des cellules en créant une plage et en appelant le[**Range.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge()) méthode.

L'image suivante montre le fichier Excel de sortie généré avec l'exemple de code. Comme vous pouvez le voir, la plage A1:D4 a été fusionnée en une seule cellule.
