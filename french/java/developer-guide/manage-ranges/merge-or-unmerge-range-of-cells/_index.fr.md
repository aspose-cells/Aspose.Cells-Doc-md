{{% alert color="primary" %}}

Vous pouvez utiliser Aspose.Cells pour fusionner ou diviser une plage de cellules. Aspose.Cells fournit les méthodes [**Range.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge--) et [**Range.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge--) à cette fin. Cet article explique comment fusionner une plage de cellules dans une cellule unique.

{{% /alert %}}

Le code d'exemple suivant crée d'abord une plage - A1:D4 - puis fusionne les cellules de la plage en une seule cellule à l'aide de la méthode [**Range.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge--).
De même, il est possible de scinder les cellules en créant une plage et en appelant la méthode [**Range.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge--).

L'image suivante montre le fichier Excel généré avec le code d'exemple. Comme vous pouvez le voir, la plage A1:D4 a été fusionnée dans une seule cellule.
