Aspose.Cells permet de sauvegarder un tableau croisé dynamique dans des fichiers ODS. Pour ce faire, vous convertissez simplement un classeur avec un tableau croisé dynamique existant ou créez un nouveau tableau croisé dynamique et enregistrez le fichier au format ODS. Assurez-vous d'appeler [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData--) avant d'enregistrer pour vous assurer que le tableau croisé dynamique est rendu dans le fichier ODS de sortie. Le code ci-dessous montre comment enregistrer un tableau croisé dynamique dans un fichier ODS.
