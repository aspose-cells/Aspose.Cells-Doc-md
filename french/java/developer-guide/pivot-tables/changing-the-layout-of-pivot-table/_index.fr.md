{{% alert color="primary" %}}

 Microsoft Excel vous permet de modifier la disposition du tableau croisé dynamique à l'aide*Outils de tableau croisé dynamique > Conception > Rapport* Commandes du menu Disposition. Vous pouvez modifier la mise en page dans ces trois formulaires

- Afficher sous forme compacte
- Afficher sous forme de plan
- Afficher sous forme tabulaire

 Aspose.Cells fournit également[**Tableau croisé dynamique.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm()), [**Tableau croisé dynamique.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm() ) et[**Tableau croisé dynamique.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm()) méthodes pour modifier la disposition du tableau croisé dynamique dans ces trois formulaires.

{{% /alert %}}
