{{% alert color="primary" %}}

Vous pouvez définir différentes options de tableau croisé dynamique à l'aide de Aspose.Cells. L'une de ces options est "Pour l'affichage des cellules vides". La définition de cette option signifie que toutes les cellules vides du tableau croisé dynamique sont affichées avec une chaîne spécifiée.

Pour rechercher et définir cette option dans Microsoft Excel :

1. Sélectionnez un tableau croisé dynamique et cliquez dessus avec le bouton droit.
1.  Sélectionner**Options de tableau croisé dynamique**.
1.  Sélectionnez le**Mise en page et format** languette.
1.  Sélectionnez le**Pour les cellules vides afficher** option et spécifiez une chaîne.

![tâche : image_autre_texte](setting-pivot-table-option-for-empty-cells-show_1.png)

{{% /alert %}}

Aspose.Cells fournit le[**Tableau croisé dynamique.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString) et[**Tableau croisé dynamique.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString)propriétés pour définir l'option de tableau croisé dynamique "Pour les cellules vides afficher".
