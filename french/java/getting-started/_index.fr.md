## **Configuration requise**

Aspose.Cells for Java prend en charge les versions de Java suivantes :

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- ou supérieur

[Plus de détails](/cells/fr/java/system-requirements/)

## **Installation**

### **Installer Aspose Cells pour Java à partir du dépôt Maven**

Maven est le moyen le plus simple de télécharger et d'installer Aspose.Cells for Java. 

1. Spécifiez la configuration/l'emplacement du dépôt Maven Aspose dans votre fichier pom.xml. 
2. Définissez la dépendance à l'API Aspose.Cells for Java dans votre fichier pom.xml. 

[Plus d'étapes pour les détails](/cells/fr/java/installation/)

### **Installer manuellement Aspose Cells pour Java**

1. [Téléchargez les fichiers aspose-cells jars](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) dans un dossier.
1. Accédez au dossier, installez manuellement les packages JAR avec Eclipse ou Intellij.
