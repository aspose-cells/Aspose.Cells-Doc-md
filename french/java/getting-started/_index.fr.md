##  **Configuration requise**

Aspose.Cells for Java prend en charge les versions Java suivantes :

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- ou au-dessus

[Plus de détails](/cells/fr/java/system-requirements/)

##  **Installation**

###  **Installez Aspose Cells for Java à partir du référentiel Maven**

 Maven est le moyen le plus simple de télécharger et d’installer Aspose.Cells for Java.

1.  Spécifiez la configuration/l'emplacement du référentiel Aspose Maven dans votre pom.xml Maven.
 2. Définissez la dépendance Aspose.Cells for Java API dans votre pom.xml.

[Plus d'étape pour plus de détails](/cells/fr/java/installation/)

###  **Installer Aspose Cells for Java manuellement**

1. [Télécharger les bocaux aspose-cells](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) dans un dossier.
1. Entrez dans le dossier, installez les packages JAR manuellement avec Eclipse ou Intellij.
