{{% alert color="primary" %}} 

Aspose.Cells for Java ne nécessite pas l'installation de Microsoft Excel, car Aspose.Cells est lui-même un moteur de création, de conversion et de rendu de feuilles de calcul Excel. Pour visualiser les documents Excel produits par Aspose.Cells, vous devez au moins avoir Microsoft Excel Viewer installé.

{{% /alert %}} 
## **Systèmes d'exploitation pris en charge**
Aspose.Cells for Java prend en charge tout système d'exploitation qui exécute l'environnement d'exécution Java, notamment :
### **Windows**
- Microsoft Windows 2000 (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows Server 2003 (x64, x86)
- Microsoft Windows Server 2008 (x64, x86)
- Microsoft Windows Server 2012 (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8 (x64, x86)
- Microsoft Windows 10 (x64, x86)
### **Linux**
- Linux (Ubuntu, openSUSE, CentOS et autres)
### **Mac**
- Mac OS X, etc.
## **Versions de Java prises en charge**
Aspose.Cells for Java prend en charge les versions de Java suivantes :

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- ou supérieur
