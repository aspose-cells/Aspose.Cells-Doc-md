{{% alert color="primary" %}} 

Aspose.Cells for Java ne nécessite pas l'installation de Microsoft Excel, car Aspose.Cells lui-même est un moteur de création, de conversion et de rendu de feuille de calcul Excel. Pour afficher les documents Excel produits par Aspose.Cells, vous devez avoir au moins Microsoft Excel Viewer installé.

{{% /alert %}} 
## **Systèmes d'exploitation pris en charge**
Aspose.Cells for Java prend en charge tout système d'exploitation qui exécute le runtime Java, y compris, mais sans s'y limiter :
### **Windows**
- Microsoft Windows 2000 (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows Serveur 2003 (x64, x86)
- Microsoft Windows Serveur 2008 (x64, x86)
- Microsoft Windows Serveur 2012 (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8 (x64, x86)
- Microsoft Windows 10 (x64, x86)
### **Linux**
- Linux (Ubuntu, openSUSE, CentOS et autres)
### **Mac**
- Mac OS X etc...
## **Versions Java prises en charge**
Aspose.Cells for Java prend en charge les versions Java suivantes :

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- ou au-dessus
