## **Aspose.Cells - Convertir XLS en PDF**
Les documents PDF sont largement utilisés comme format standard d'échange de documents entre organisations, secteurs gouvernementaux et individus. Les développeurs de logiciels sont souvent invités à concevoir un moyen de convertir facilement des fichiers Microsoft Excel en documents PDF. Aspose.Cells prend en charge cette fonctionnalité.

**Java**

{{< highlight java >}}

 Workbook workbook = new Workbook(dataDir + "workbook.xls");

//Save the document in PDF format

workbook.save(dataDir + "AsposeConvert.pdf", SaveFormat.PDF);

{{< /highlight >}}
