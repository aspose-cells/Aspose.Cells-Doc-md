## **Aspose.Cells - Convertir une feuille de calcul en PDF**
Les documents PDF sont largement utilisés comme format standard d'échange de documents entre les organisations, les secteurs gouvernementaux et les particuliers. Les développeurs de logiciels sont souvent invités à trouver un moyen de convertir facilement des fichiers Excel Microsoft en documents PDF. Aspose.Cells prend en charge ces fonctionnalités.

**Java**

{{< highlight "java" >}}

 Workbook workbook = new Workbook("workbook.xls");

//Save the document in PDF format

workbook.save("AsposeConvert.pdf", SaveFormat.PDF);

{{< /highlight >}}
