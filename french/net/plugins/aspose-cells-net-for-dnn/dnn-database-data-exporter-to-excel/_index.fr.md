## **Introduction**

Le module Aspose .NET DNN Data Exporter to Excel permet aux utilisateurs d'exporter des données directement depuis des tables de base de données locales ou distantes, des vues et par une requête personnalisée dans Microsoft Excel ou OpenOffice Spreadsheet. Ce module démontre la puissante fonctionnalité de création de feuilles de calcul fournie par Aspose.Cells.

### **Fonctionnalités du module**

Cette version initiale du module est enrichie des fonctionnalités suivantes pour rendre le processus d'exportation simple et facile à utiliser

- Permettre de se connecter à la base de données locale de MS SQL Server
- Permettre de se connecter à la base de données distante de MS SQL Server
- Remplir toutes les tables de la base de données connectée
- Remplir toutes les vues de la base de données connectée
- Permettre d'écrire une requête personnalisée
- Ajustement automatique des colonnes à la longueur du contenu.
- Autoriser le saut de chaîne de plus de 32 ko dans les cellules Excel (LoadOptions)
- Appliquer la mise en forme de l'en-tête des colonnes en texte gras
- Autoriser l'utilisation en tant que source de données (tableaux, vues, requête personnalisée)
- Exporter des données vers des documents Microsoft Excel (.xls, .xlsx et .xlsb)
- Exporter des données vers un document texte délimité par des tabulations (*.txt)
- Exporter des données au format CSV (délimité par des virgules) (*.csv)
- Exporter des données vers des feuilles de calcul OpenDocument (*.ods)
- Option pour sélectionner le format de sortie souhaité avant l'exportation.
- Le document exporté est automatiquement envoyé au navigateur pour téléchargement.

## **Installer**

Une fois téléchargé, veuillez suivre ces étapes pour installer le module sur votre site DNN :

1. Connectez-vous à votre site en tant qu'administrateur principal ou un autre compte disposant des autorisations d'administrateur.
1. Accédez au menu **Hôte** et sélectionnez **Extensions**.
1. Cliquez sur **Assistant d'installation d'extension**.
1. Suivez les instructions pour accéder à l'emplacement du fichier zip téléchargé, sélectionnez-le, puis cliquez sur **Ouvrir**.
1. Cliquez sur **Suivant**, acceptez la licence, suivez l'installation. Lorsque c'est terminé, cliquez sur le bouton **Retour**.

Veuillez consulter [cette vidéo d'installation du module](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN pour plus de détails.

## **En utilisant**

Une fois que vous avez installé le module Aspose .NET Database Data Expoeter for DNN, il est très simple de commencer à l'utiliser sur votre site. Veuillez suivre ces étapes simples pour commencer :

1. Assurez-vous d'être connecté à DNN avec un compte de niveau Host ou Admin.
1. Accédez à la page où vous souhaitez ajouter le module d'exportation.
1. Sélectionnez **Modules** suivi de **Ajouter un nouveau module** dans le ruban supérieur.
1. Dans la liste, sélectionnez 'Aspose.NET DNN Data Exporter to Excel Using Aspose.Cells' et faites-le glisser à l'endroit souhaité.

![todo:image_alt_text](picture2.png)

Vous avez ajouté avec succès le module Aspose .NET Database Data Exporter to Excel for DNN à votre page.

![todo:image_alt_text](picture3.png)

1. Entrez ou utilisez une chaîne de connexion MS SQL Server pré-renseignée
1. Sélectionnez le type de source de données (tableau, vue, requête personnalisée)
1. Sélectionnez ou saisissez la source de données (tableau, vue, requête personnalisée)
1. Sélectionnez le type d'exportation
1. Cliquez sur Exporter les données
1. Le fichier souhaité sera téléchargé automatiquement.
