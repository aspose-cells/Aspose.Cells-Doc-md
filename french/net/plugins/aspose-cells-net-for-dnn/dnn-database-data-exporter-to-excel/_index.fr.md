## **Introduction**

Aspose .NET DNN Data Exporter to Excel Module permet aux utilisateurs d'exporter des données directement à partir de tables de base de données locales ou distantes, de vues et par requête personnalisée dans Microsoft Excel ou OpenOffice Spreadsheet. Ce module illustre la puissante fonctionnalité de création de feuille de calcul fournie par Aspose.Cells.

### **Caractéristiques des modules**

Cette version initiale du module est enrichie des fonctionnalités intéressantes suivantes pour rendre le processus d'exportation simple et facile à utiliser

- Autoriser la connexion à la base de données MS SQL Server locale
- Autoriser la connexion à la base de données MS SQL Server distante
- Remplir toutes les tables à partir de la base de données connectée
- Remplir toutes les vues de la base de données connectée
- Autoriser l'écriture d'une requête personnalisée
- Ajuster automatiquement les colonnes à la longueur du contenu.
- Autoriser à ignorer la chaîne de plus de 32k dans les cellules Excel (LoadOptions)
- Appliquer la mise en forme de la colonne d'en-tête en texte gras
- Autoriser l'utilisation en tant que source de données (table, vues, requête personnalisée)
- Exporter des données vers des documents Excel Microsoft (.xls, .xlsx et .xlsb)
- Exporter les données vers un document texte délimité par des tabulations (*.txt)
- Exporter des données vers CSV (délimité par des virgules) (*.csv)
- Exporter des données vers une feuille de calcul OpenDocument (*.ods)
- Option pour sélectionner le format de sortie souhaité avant l'exportation.
- Le document exporté est automatiquement envoyé au navigateur pour téléchargement.

## **Installation**

Une fois téléchargé, veuillez suivre ces étapes pour installer le module sur votre site Web DNN :

1. Connectez-vous à votre site en tant qu'hôte ou en tant que compte de niveau super-utilisateur.
1.  Naviguez vers le**Héberger** menu et sélectionnez**Rallonges**.
1.  Cliquez sur**Assistant d'installation d'extension**.
1. Comme indiqué, accédez à l'emplacement du fichier zip téléchargé, sélectionnez-le, puis cliquez sur**Ouvert**.
1.  Cliquez sur**Prochain** , acceptez la licence, poursuivez l'installation. Lorsque vous avez terminé, cliquez sur le**Revenir** bouton.

 Vérifiez s'il vous plaît[cette vidéo d'installation du module](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN pour plus de détails.

## **Utilisant**

Après avoir installé le module Aspose .NET Database Data Expoeter for DNN, il est très simple de commencer à l'utiliser sur votre site Web. Veuillez suivre ces étapes simples pour commencer :

1. Assurez-vous que vous êtes connecté à DNN avec un compte de niveau hôte ou administrateur.
1. Accédez à la page où vous souhaitez ajouter le module d'exportation.
1.  Sélectionner**Modules** suivie par**Ajouter un nouveau module** du ruban supérieur.
1. Dans la liste, sélectionnez 'Aspose.NET DNN Data Exporter to Excel Using Aspose.Cells' et faites-le glisser en position.

![tâche : image_autre_texte](picture2.png)

Vous avez ajouté avec succès le module Aspose .NET Database Data Exporter to Excel for DNN à votre page.

![tâche : image_autre_texte](picture3.png)

1. Entrez ou utilisez la chaîne de connexion MS SQL Server pré-remplie
1. Type de source de données Seletec (table, vue, requête personnalisée)
1. Sélectionner ou saisir une source de données (table, vue, requête personnalisée)
1. Sélectionnez le type d'exportation
1. Cliquez sur Exporter les données
1. Le fichier souhaité sera téléchargé automatiquement.
