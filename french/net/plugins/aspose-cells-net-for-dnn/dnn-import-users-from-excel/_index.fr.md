## **Introduction**
Le module d'importation des utilisateurs à partir d'Excel Aspose .NET permet aux développeurs d'importer des utilisateurs en masse à partir d'un fichier Excel. Ce module démontre la puissante fonctionnalité de feuille de calcul d'Aspose.Cells.
### **Fonctionnalités du module**
Cette première version du module est enrichie des fonctionnalités suivantes pour rendre le processus d'importation simple et facile à utiliser :

- Vous permet de télécharger un fichier Excel et filtre automatiquement les utilisateurs existants.
- Passe automatiquement les utilisateurs existants pour éviter toute confusion.
- Génère des mots de passe dynamiques pour les utilisateurs nouvellement créés.
- Affecte les rôles de sécurité "Utilisateurs enregistrés" et "Abonnés" aux utilisateurs nouvellement créés.

## **Installer**
Une fois téléchargé, veuillez suivre ces étapes pour installer le module sur votre site DNN :

1. Connectez-vous à votre site en tant qu'administrateur principal ou un autre compte disposant des autorisations d'administrateur.
1. Accédez au menu **Hôte** et sélectionnez **Extensions**.
1. Cliquez sur **Assistant d'installation d'extension**.
1. Suivez les instructions pour accéder à l'emplacement du fichier zip téléchargé, sélectionnez-le, puis cliquez sur **Ouvrir**.
1. Cliquez sur **Suivant**, acceptez la licence, suivez l'installation. Lorsque c'est terminé, cliquez sur le bouton **Retour**.

Veuillez consulter [cette vidéo d'installation du module](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN pour plus de détails.
## **En utilisant**
Après avoir installé le module Aspose .NET Cells Import Users from Excel for DNN, il est vraiment simple de commencer à l'utiliser sur votre site web. Veuillez suivre ces étapes simples pour commencer :

1. Assurez-vous d'être connecté à DNN avec un compte de niveau Host ou Admin.
1. Accédez à la page où vous souhaitez ajouter le module d'exportation.
1. Sélectionnez **Modules** suivi de **Ajouter un nouveau module** dans le ruban supérieur. 

1. Dans la liste, sélectionnez ‘Aspose Import Users from Excel module for DNN’ et faites-le glisser à l'endroit souhaité.

Vous avez ajouté avec succès le module Aspose .NET Cells Import Users from Excel for DNN sur votre page. Les contrôles de téléchargement de fichier seront visibles avec le lien "Fichier modèle". Vous pouvez télécharger un fichier modèle et formater vos contenus selon le fichier modèle. Une fois que vous aurez téléchargé votre fichier Excel, il traitera automatiquement tous les utilisateurs et filtrera les utilisateurs existants. Vous pouvez sélectionner les utilisateurs que vous souhaitez importer et appuyer sur le bouton "Importer les utilisateurs".
