## **Introduction**
Aspose .NET Exporter Module utilisateur et rôles vers Excel permet aux développeurs d'exporter les utilisateurs DNN ainsi que leurs rôles dans Microsoft Excel ou OpenOffice Spreadsheet. Ce module démontre la puissante fonction de création de feuilles de calcul fournie par Aspose.Cells.
### **Fonctionnalités du module**
Cette version initiale du module est enrichie avec les fonctionnalités suivantes pour rendre le processus d'exportation simple et facile à utiliser.

- Exporter les utilisateurs vers des documents Microsoft Excel (.xls, .xlsx et .xlsb)
- Exporter les utilisateurs vers un document texte délimité par des onglets (*.txt)
- Exporter les utilisateurs vers un CSV (délimité par des virgules) (*.csv)
- Exporter les utilisateurs vers une feuille de calcul OpenDocument (*.ods)
- Option pour sélectionner le format de sortie souhaité avant l'exportation.
- Option permettant d'exporter tous les utilisateurs ou les utilisateurs sélectionnés vers un format de document de sortie choisi.
- Le document exporté est automatiquement envoyé au navigateur pour téléchargement.

## **Installer**
Une fois téléchargé, veuillez suivre ces étapes pour installer le module sur votre site DNN :

1. Connectez-vous à votre site en tant qu'administrateur principal ou un autre compte disposant des autorisations d'administrateur.
1. Accédez au menu **Hôte** et sélectionnez **Extensions**.
1. Cliquez sur **Assistant d'installation d'extension**.
1. Suivez les instructions pour accéder à l'emplacement du fichier zip téléchargé, sélectionnez-le, puis cliquez sur **Ouvrir**.
1. Cliquez sur **Suivant**, acceptez la licence, suivez l'installation. Lorsque c'est terminé, cliquez sur le bouton **Retour**.

Veuillez consulter [cette vidéo d'installation du module](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN pour plus de détails.
## **En utilisant**
Après avoir installé le module Aspose .NET Cells Export Users and Roles for DNN, il est vraiment simple de commencer à l'utiliser sur votre site web. Veuillez suivre ces étapes simples pour commencer :

1. Assurez-vous d'être connecté à DNN avec un compte de niveau Host ou Admin.
1. Accédez à la page où vous souhaitez ajouter le module d'exportation.
1. Sélectionnez **Modules** suivi de **Ajouter un nouveau module** dans le ruban supérieur. 

1. Dans la liste, sélectionnez 'Aspose Export Users and Roles to Excel' et faites-le glisser à l'endroit désiré.

Vous avez ajouté avec succès le module Aspose .NET Export Users and Roles to Excel pour DNN à votre page. Tous les utilisateurs dnn seront visibles dans la grille et vous pourrez choisir les utilisateurs à exporter et cliquer sur le bouton "Exporter" après avoir sélectionné le format de sortie souhaité dans la liste déroulante. Le fichier souhaité sera téléchargé automatiquement.
