## **Introduction**
Aspose .NET Database Data Exporter to Excel for Umbraco Module permet aux utilisateurs d'exporter des données directement à partir de tables de base de données locales ou distantes, de vues et par requête personnalisée dans Microsoft Excel ou OpenOffice Spreadsheet. Ce module illustre la puissante fonctionnalité de création de feuille de calcul fournie par Aspose.Cells. Cette version initiale du module est enrichie des fonctionnalités intéressantes suivantes pour rendre le processus d'exportation simple et facile à utiliser
### **Caractéristiques des modules**
Cette version initiale de l'Add-on possède les fonctionnalités suivantes :

- Autoriser la connexion à la base de données MS SQL Server locale
- Autoriser la connexion à la base de données MS SQL Server distante
- Remplir toutes les tables à partir de la base de données connectée
- Remplir toutes les vues de la base de données connectée
- Autoriser l'écriture d'une requête personnalisée
- Ajuster automatiquement les colonnes à la longueur du contenu.
- Autoriser à ignorer la chaîne de plus de 32k dans les cellules Excel (LoadOptions)
- Appliquer la mise en forme de la colonne d'en-tête en texte gras
- Autoriser l'utilisation en tant que source de données (table, vues, requête personnalisée)
- Exporter des données vers des documents Excel Microsoft (.xls, .xlsx et .xlsb)
- Exporter les données vers un document texte délimité par des tabulations (*.txt)
- Exporter des données vers CSV (délimité par des virgules) (*.csv)
- Exporter des données vers une feuille de calcul OpenDocument (*.ods)
- Option pour sélectionner le format de sortie souhaité avant l'exportation.
- Le document exporté est automatiquement envoyé au navigateur pour téléchargement.

## **Configuration système requise et plates-formes prises en charge**
### **Configuration requise**
Afin de configurer Aspose .NET Exportateur de données de base de données vers Excel pour le module Umbraco, vous devez remplir les conditions suivantes :

- Versions Umbraco 6.2.5 et Umbraco 6
- Umbraco avec MS SQL Server
- Microsoft .Net Framework 4.0

**Noter:** Umbraco 7 et supérieur ne sont pas pris en charge dans cette version. Nous sommes impatients d'entendre vos commentaires et d'ajouter la prise en charge d'Umbraco 7 dans la prochaine version.
### **Plates-formes prises en charge**
Le module est pris en charge sur toutes les versions de

- Umbraco 6.0 fonctionnant sur ASP.NET 4.0

## **Installation**
Une fois téléchargé, veuillez suivre ces étapes pour installer ce package sur votre site Web Umbraco :

1.  Connectez-vous à Umbraco**Développeur** section, par exemple `http://www.myblog.com/umbraco`
1.  Dans l'arborescence, développez le**Paquets** dossier.
1.  À partir de là, il existe deux façons d'installer un paquet : sélectionnez**Installer le paquet local** ou parcourez le**Référentiel de paquets Umbraco.**
1. Si vous installez**forfait local**, ne décompressez pas le package mais chargez le zip dans Umbraco.
1. Suivez les instructions à l'écran.

**Noter:** Vous pouvez obtenir une erreur "Longueur maximale de la demande dépassée" lors de l'installation. Vous pouvez facilement résoudre ce problème en mettant à jour la valeur 'maxRequestLength' dans votre fichier Umbraco web.config.
<httpRuntime requestValidationMode="2.0" enableVersionHeader="false" maxRequestLength="25000" />
## **Utilisant**
Après avoir installé le module Aspose .NET Exportateur de données de base de données vers Excel pour Umbraco, il est très simple de commencer à l'utiliser sur votre site Web. Veuillez suivre ces étapes simples pour commencer

1.  Assurez-vous que vous êtes connecté à Umbraco**Développeur** section, par exemple `http://www.myblog.com/umbraco/`
1.  Cliquez sur**Réglages** dans la liste des rubriques en bas à gauche de l'écran.
1.  Élargir le**Modèles** node et sélectionnez le modèle que vous souhaitez ajouter, par exemple Textpage.
1. Sélectionnez la position dans le modèle sélectionné où vous souhaitez exporter le bouton à ajouter. Habituellement, vous voudrez peut-être l'ajouter en haut à droite de la page ou en bas de la page.
1.  Cliquez sur**Insérer une macro** sur le ruban supérieur.
1.  De**Choisissez une macro** (Aspose .NET Database Data Exporter to Excel for Umbraco), sélectionnez la macro Aspose .NET Database Data Exporter to Excel for Umbraco récemment installée et cliquez sur**D'ACCORD**.

 Veuillez vérifier la capture d'écran ci-dessous pour plus de détails.

1. Entrez ou utilisez la chaîne de connexion MS SQL Server pré-remplie
1. Type de source de données Seletec (table, vue, requête personnalisée)
1. Sélectionner ou saisir une source de données (table, vue, requête personnalisée)
1. Sélectionnez le type d'exportation
1. Cliquez sur Exporter les données
1. Le fichier souhaité sera téléchargé automatiquement.
