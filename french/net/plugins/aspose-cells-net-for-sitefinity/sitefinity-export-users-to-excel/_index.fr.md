## **Introduction**
Aspose .NET Exporter des utilisateurs vers Excel pour le module SiteFinity permet aux développeurs d'exporter des utilisateurs SiteFinity vers Microsoft Excel ou une feuille de calcul OpenOffice. Ce module illustre la puissante fonctionnalité de création de feuille de calcul fournie par Aspose.Cells.

## **Configuration système requise et plates-formes prises en charge**
### **Configuration requise**
Afin de configurer Aspose.Cells .NET pour les modules complémentaires Sitefinity, vous devez remplir les conditions suivantes :

- Sitefinity CMS fonctionnant sur ASP.NET 4.0

N'hésitez pas à nous contacter si vous rencontrez des problèmes lors de la configuration de ce module complémentaire Sitefinity.
### **Plates-formes prises en charge**
Le module complémentaire est pris en charge sur toutes les versions de

- Sitefinity CMS fonctionnant sur ASP.NET 4.0

## **Installation et utilisation**
### **Téléchargement**
Vous pouvez télécharger le module Aspose .NET Content Exporter for Sitefinity à partir de l'un des emplacements suivants

- [ GithubGenericName](https://github.com/aspose-cells/Aspose.Cells-for-.NET/releases)
### **Installation**
Une fois téléchargé, veuillez suivre ces étapes pour installer le module complémentaire sur votre site Web Sitefinity :

**Étape 1 : Copiez les fichiers sur votre installation Sitefinity**

Veuillez extraire le fichier ZIP téléchargé. Vous aurez besoin d'un FTP ou d'un accès direct au dossier d'installation de Sitefinity sur le serveur pour effectuer les opérations suivantes :

1.  Copiez Aspose.Cells.dll et Aspose.SiteFinity.ExportUsersToExcel.dll dans le**poubelle** dossier de l'installation de Sitefinity.
1.  Copiez le**Compléments** dossier à la racine de l'installation de Sitefinity où le**poubelle** dossier est localisé.

**Étape 2: Enregistrez le module complémentaire d'exportation de contenu Sitefinity Aspose dans Sitefinity**

1. Connectez-vous à votre CMS Sitefinity avec un '**Administrateur** ' Compte. La page de connexion est accessible par<http://www.mywebsite.com/sitefinity>
1.  Cliquez sur**Administration** et alors**Réglages**.
La page Paramètres de base s'affiche.
1.  Clique le**Avancé** lien.
 La page Paramètres s'affiche.
1.  Dans le volet de gauche, cliquez sur**Boîtes à outils** suivie par**Boîtes à outils** , alors**PageControls**, **Sections** et**ContentToolboxSection** , alors**Outils.**
1.  Cliquez sur**Créer un nouveau**.
 Le formulaire d'enregistrement du widget s'affiche.
1.  Remplissez les champs du formulaire comme suit :
 1. Assurez-vous**Activé** est sélectionné.
 1. Ajouter ~/Addons/Aspose.SiteFinity.ExportUsersToExcel/AsposeExportUsersToExcel.ascx

 1. ` `dans le**Contrôler le type CLR ou le chemin virtuel** champ.
 1. Ajouter**Nom**, **Titre** et**La description** comme suit:
 Aspose.SiteFinity.ExportUsersToExcel
 Aspose Exporter les utilisateurs de SiteFinity vers Excel
 Exporter les utilisateurs de SiteFinity vers Excel
 1. Vous pouvez laisser tous les autres champs tels quels.
1.  Lorsque vous avez terminé, cliquez sur**Sauvegarder les modifications**.
 Le widget est enregistré dans la boîte à outils et peut être utilisé dans Sitefinity.
## **Utilisation et démo vidéo**
### **Utilisant**
Après avoir installé et configuré le module complémentaire Aspose Sitefinity Export Users to Excel, il est très simple de commencer à l'utiliser sur votre site Web. Veuillez suivre ces étapes simples pour commencer :

1. Assurez-vous que vous êtes connecté à Sitefinity avec un compte de niveau administrateur.
1. Accédez à la page où vous souhaitez ajouter le module complémentaire d'exportation. Assurez-vous que la page est ouverte en mode édition.
1.  Du**Faites glisser les widgets** menu de droite, sélectionnez Aspose Exporter les utilisateurs vers Excel et faites-le glisser en position.


Vous avez ajouté avec succès Aspose utilisateurs d'exportation Sitefinity à Excel.
