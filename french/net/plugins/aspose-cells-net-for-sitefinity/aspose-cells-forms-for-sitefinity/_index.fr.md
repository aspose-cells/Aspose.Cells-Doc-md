## **Introduction**

Le module Aspose.Cells Dynamic Forms for Sitefinity permet aux utilisateurs de générer des questionnaires dynamiques et des enquêtes, de sauvegarder les entrées des utilisateurs dans une feuille de calcul Excel et d'exporter les résultats dans Excel, du texte, CSV et un tableau de calcul OpenDocument en utilisant Aspose.Cells. Ce module démontre la puissante fonctionnalité de construction de feuilles de calcul fournie par Aspose.Cells for .NET.

### **Fonctionnalités du module**

Cette version initiale du module est enrichie des fonctionnalités suivantes pour rendre la construction et l'exportation de formulaires simples et faciles à utiliser

- Enregistrer les paramètres des champs du formulaire dans Excel
- Enregistrer les données de saisie de l'utilisateur dans Excel
- Permettre d'ajouter de nouveaux champs au formulaire et de mettre à jour les champs existants
- Permettre d'ajouter des champs de type TextBox, Multiline TextBox, RadioButtons, CheckBox et DropDownList, DropDownList Items
- Permettre d'ajouter/mettre à jour une étiquette pour chaque champ
- Permettre d'afficher/masquer les champs du formulaire
- Ajustement automatique des colonnes à la longueur du contenu et application du format de colonne d'en-tête en texte en gras
- Exporter des données vers des documents Microsoft Excel (.xls, .xlsx et .xlsb)
- Exporter des données vers un document texte délimité par des tabulations (*.txt)
- Exporter des données au format CSV (délimité par des virgules) (*.csv)
- Exporter des données vers des feuilles de calcul OpenDocument (*.ods)
- Option pour sélectionner le format de sortie souhaité avant l'exportation.
- Le document exporté est automatiquement envoyé au navigateur pour téléchargement.

## **Téléchargement et installation**

### **Téléchargement**

Vous pouvez télécharger le module Aspose .NET Content Exporter for Sitefinity depuis l'un des emplacements suivants

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-.NET/tree/master/Plugins/SiteFinity/Aspose.SiteFinity.FormBuilder.ToExcel)

### **Installer**

Une fois téléchargé, veuillez suivre ces étapes pour installer le module complémentaire dans votre site Web Sitefinity :

**Étape 1 : Copiez les fichiers dans votre installation Sitefinity**

Veuillez extraire le fichier ZIP téléchargé. Vous aurez besoin d'un accès FTP ou direct au dossier d'installation de Sitefinity sur le serveur pour effectuer ce qui suit :

1. Copiez **Aspose.Cells.dll** & **Aspose.Sitefinity.FormBuilder.dll** dans le dossier bin de l'installation de Sitefinity.
1. Copiez le dossier **Addons** à la racine de l'installation de Sitefinity où se trouve le dossier **bin**.

**Étape 2 : Enregistrez le module complémentaire Aspose Sitefinity Content Export dans Sitefinity**

1. Log into your Sitefinity CMS with an ‘**Administrator**’ account. The login page can be reached by <http://www.mywebsite.com/sitefinity>
1. Cliquez sur **Administration** puis **Paramètres**.
   La page des paramètres de base apparaît.
1. Cliquez sur le lien **Avancé**.
   La page des paramètres apparaît.
1. Dans le volet de gauche, cliquez sur **Boîtes à outils**, suivi de **Boîtes à outils**, puis **Contrôles de page**, **Sections** et **ContentToolboxSection**, puis **Outils**.
1. Cliquez sur **Créer un nouveau**.
   Le formulaire d'enregistrement du widget apparaît.
1. Remplissez les champs du formulaire comme suit : 
   1. Assurez-vous que **Activé** est sélectionné. 
      1. Dans le champ **Type de CLR ou Chemin virtuel de contrôle**. 
         1. Ajoutez **~/Addons/Aspose.SiteFinity.FormBuilder.ToExcel/Edit.ascx**
      1. Ajoutez **Nom**, **Titre** et **Description** comme suit : 
         1. Aspose  **NomDePage** (comme Edit, View, Export) sous forme de formulaire aux utilisateurs de SiteFinity
         1. Formulaires Aspose **NomDePage** (comme formulaire Aspose Edit, formulaire Aspose View, formulaire Aspose Export)
         1. Constructeur et exportateur de formulaire **NomDePage** pour Sitefinity.
      1. Vous pouvez laisser tous les autres champs tels qu'ils sont.
      1. Lorsque vous avez terminé, cliquez sur **Enregistrer les modifications**.
      1. Le widget est enregistré dans la boîte à outils et peut être utilisé dans Sitefinity. (Voir l'image ci-dessous)

## **Utilisation et démonstration vidéo**

### **En utilisant**

Après avoir installé et configuré le module complémentaire Aspose.Cells Dynamic Forms Builder pour les utilisateurs de Sitefinity, il est vraiment simple de commencer à l'utiliser sur votre site web. Veuillez suivre ces étapes simples pour commencer :

1. Assurez-vous d'être connecté à Sitefinity avec un compte de niveau Administrateur.
1. Accédez à la page où vous souhaitez ajouter le plugin. Assurez-vous que la page est ouverte en mode édition.
1. Dans le menu **Faire glisser les widgets** sur la droite, sélectionnez le formulaire Aspose Edit/View/Export et faites-le glisser à l'endroit souhaité. (Voir l'image ci-dessous)

|<p>![todo:image_alt_text](aspose-cells-forms-for-sitefinity_2)</p><p></p>|
| :- |

Vous avez ajouté avec succès le module Aspose.Cells Dynamic Form Builder à votre page Sitefinity.

#### **Comment appliquer la licence Aspose ?**

Ce plugin utilise une version d'évaluation d'Aspose.Cells. Une fois satisfait de votre évaluation, vous pouvez acheter une licence sur le [site d'achat Aspose](https://purchase.aspose.com/buy).
Pour supprimer le message d'évaluation et les limitations des fonctionnalités, la licence du produit doit être appliquée. Vous recevrez un fichier de licence après avoir acheté le produit. Veuillez suivre les étapes ci-dessous pour appliquer la licence

- Assurez-vous que le fichier de licence est nommé **Aspose.Total.lic**
- Placez le fichier **Aspose.Total.lic** dans le dossier **App_Data** de votre site web Sitefinity, par exemple, "Dossier racine de Sitefinity/App_Data/Aspose.Total.lic"

#### **Paramètres des formulaires dynamiques**

1. Assurez-vous d'être connecté, cliquez sur le menu Page, cliquez sur la première rangée de l'option de bouton **Vue** près de la colonne **Action.**  
1. Cliquez sur le bouton **Modifier** disponible près de l'étiquette d'option.
1. Il y a quelques champs prédéfinis, que vous pouvez modifier/cacher en cliquant simplement sur **Modifier** dans la grille.
1. Vous pouvez créer/supprimer/mettre à jour de nouveaux champs de tous types **(Zone de texte, Zone de texte multi-lignes, Bouton radio, Cases à cocher, Liste déroulante, Titre, Message de réussite)**

#### **Soumettre le formulaire dynamique**

1. Remplissez les champs.
1. Cliquez sur le bouton **Soumettre** pour enregistrer les données.
1. Chaque clic sur le bouton **Soumettre** enregistrera un nouvel enregistrement dans Excel.

#### **Exporter les données enregistrées**

1. Assurez-vous d'être connecté, allez dans le menu Pages et cliquez sur le bouton d'option de vue de la première ligne près de la colonne d'action.
1. Passez la souris sur **l'icône d'exportation** et cliquez sur **onit** la page d'exportation s'ouvrira.
1. Sélectionnez le **Type d'exportation**.
1. Cliquez sur **Exporter les données**.
1. Le fichier de données exporté selon le type d'exportation apparaîtra pour être téléchargé/ouvert.

Vous avez ajouté avec succès Aspose Sitefinity Export Users to Excel.
