### **Introduction**
#### **Qu'est-ce que Sitefinity?**
Sitefinity est un système de gestion de contenu web moderne basé sur ASP.NET (CMS), offrant une gestion intuitive du contenu web et un environnement de développement robuste pour les professionnels de l'entreprise. Voici les projets que nous avons créés pour ce CMS populaire.
#### **Aspose.Cells for .NET**
Aspose.Cells for .NET est une bibliothèque de classes avancée pour .NET qui vous permet d'effectuer un large éventail de tâches de traitement de feuilles de calcul directement dans vos applications .NET.

Avec Aspose.Cells, vous pouvez générer, modifier, convertir, rendre et imprimer des feuilles de calcul sans utiliser Microsoft Excel.

Aspose.Cells for .NET prend en charge les formats XSL, XSLX, OpenOffice et de nombreux autres formats.
