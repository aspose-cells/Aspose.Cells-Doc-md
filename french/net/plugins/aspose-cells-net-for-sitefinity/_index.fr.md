### **Introduction**
#### **Qu'est-ce que Sitefinity ?**
Sitefinity est un système de gestion de contenu Web (CMS) moderne basé sur ASP.NET, offrant une gestion de contenu Web intuitive et un environnement de développement robuste pour les professionnels. Voici les projets que nous avons créés pour ce CMS populaire.
#### **Aspose.Cells for .NET**
Aspose.Cells for .NET est une bibliothèque de classes avancée for .NET qui vous permet d'effectuer un large éventail de tâches de traitement de feuille de calcul directement dans vos applications .NET.

Avec Aspose.Cells, vous pouvez générer, modifier, convertir, rendre et imprimer une feuille de calcul sans utiliser Microsoft Excel.

Aspose.Cells for .NET prend en charge XSL, XSLX, OpenOffice et de nombreux autres formats.
