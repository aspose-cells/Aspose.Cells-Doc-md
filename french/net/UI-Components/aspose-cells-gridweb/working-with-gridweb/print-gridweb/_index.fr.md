Il arrive que les développeurs aient besoin d'imprimer le contenu de GridWeb à partir d'une page Web sans enregistrer le résultat en tant que fichier de feuille de calcul Excel Microsoft. Le contrôle Aspose.Cells.GridWeb prend en charge cette fonctionnalité via la fonction côté client.

{{% /alert %}}
