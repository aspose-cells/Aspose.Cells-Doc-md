{{% alert color="primary" %}}

Les barres de défilement sont utiles pour naviguer dans le contenu des feuilles de calcul qui sont larges et profondes, c'est-à-dire qui ont de nombreuses lignes et colonnes. La plupart des applications prennent en charge deux types de barre de défilement :

- Barre de défilement verticale
- Barre de défilement horizontale

Les deux sont présentes dans Microsoft Excel.

L'API GridDesktop d'Aspose.Cells fournit des barres de défilement horizontales et verticales pour faire défiler le contenu d'une feuille de calcul. Avec les API GridDesktop d'Aspose.Cells, les développeurs peuvent contrôler la visibilité de ces deux barres de défilement.

{{% /alert %}}
