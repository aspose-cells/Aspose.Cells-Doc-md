{{% alert color="primary" %}} 

Il s'agit d'un autre sujet de base pour les développeurs pour expliquer une approche de base consistant à supprimer une ligne de la feuille de calcul. Il s'agit d'une fonctionnalité simple et pratique qui peut être utilisée par les développeurs lorsqu'ils travaillent avec Aspose.Cells.GridDesktop.

{{% /alert %}} 

Pour supprimer une ligne de la feuille de calcul, veuillez suivre les étapes ci-dessous :

-  Accédez à tout**Feuille de travail**
-  Supprimer**Ligne** du**Feuille de travail** en précisant l'index de la ligne à supprimer
