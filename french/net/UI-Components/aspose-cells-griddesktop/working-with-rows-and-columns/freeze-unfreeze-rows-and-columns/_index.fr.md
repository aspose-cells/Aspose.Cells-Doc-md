{{% alert color="primary" %}} 

Cette rubrique traite des informations sur le gel et le dégel des lignes et des colonnes d'une feuille de calcul. Le gel des colonnes ou des lignes permet aux utilisateurs de garder les en-têtes de colonne ou les titres de ligne visibles pendant qu'ils font défiler vers d'autres parties de la feuille de calcul. Cette fonctionnalité est très utile pour faire défiler les feuilles de calcul qui contiennent d'énormes volumes de données. Parce que, dans de tels cas, lorsque les utilisateurs feront défiler les données, seules les données défileront vers le bas et les en-têtes resteront là pour augmenter la lisibilité des données.

{{% /alert %}} 
## **Colonnes de congélation**
Pour figer les colonnes d'une feuille de calcul à l'aide de Aspose.Cells.GridDesktop, veuillez suivre les étapes ci-dessous :

-  Accédez à tout**Feuille de travail**
-  Définissez le nombre de**Colonnes gelées** dans le**Feuille de travail**

## **Colonnes de dégel**
Pour dégeler les colonnes d'une feuille de calcul à l'aide de Aspose.Cells.GridDesktop, veuillez suivre les étapes ci-dessous :

-  Accédez à tout**Feuille de travail**
-  Définissez le nombre de**Colonnes gelées** dans le**Feuille de travail** à zéro (0). Il produira l'effet de colonnes non figées
