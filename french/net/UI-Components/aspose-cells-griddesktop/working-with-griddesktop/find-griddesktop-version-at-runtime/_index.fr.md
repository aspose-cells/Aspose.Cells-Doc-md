## **Scénarios d'utilisation possibles**
 Parfois, il est important de trouver la version de GridDesktop au moment de l'exécution. Par exemple, vous voulez vous assurer que vous utilisez la bonne version ou vous déboguez votre application et vous voulez connaître la version de GridDesktop actuellement utilisée. GridDesktop fournit la méthode Aspose.Cells.GridDesktop.GridDesktop.GetVersion() qui renvoie la version de GridDesktop au moment de l'exécution.
