{{% alert color="primary" %}}

Vous pouvez spécifier le nom du travail ou du document lors de l'impression de votre classeur ou de votre feuille de calcul à l'aide des objets WorkbookRender ou SheetRender. Aspose.Cells fournit les méthodes WorkbookRender.ToPrinter(nomImprimante, nomTravail) et SheetRender.ToPrinter(nomImprimante, nomTravail) que vous pouvez utiliser pour spécifier le nom du travail lors de l'impression de votre classeur ou de votre feuille de calcul

{{% /alert %}}

## Spécifiez le nom du travail ou du document lors de l'impression avec Aspose.Cells

Le code d'exemple charge le fichier Excel source puis l'envoie à l'imprimante en spécifiant le nom du travail ou du document en utilisant les méthodes WorkbookRender.ToPrinter(nomImprimante, nomTravail) et SheetRender.ToPrinter(nomImprimante, nomTravail).
