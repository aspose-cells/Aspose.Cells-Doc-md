## **Scénarios d'utilisation possibles**
En utilisant les APIs Aspose.Cells, les développeurs peuvent remplir un DataTable avec des chaînes HTML correspondantes (en fonction des valeurs des cellules) tout en exportant les données de la feuille de calcul. Ce document explique comment exporter des chaînes HTML pour les données de cellules formatées afin de remplir un DataTable en utilisant les APIs Aspose.Cells.
