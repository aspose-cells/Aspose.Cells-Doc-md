## **Scénarios d'utilisation possibles**
À l'aide des API Aspose.Cells, les développeurs peuvent remplir un DataTable avec les chaînes HTML correspondantes (basées sur les valeurs des cellules) lors de l'exportation des données de la feuille de calcul. Ce document explique comment exporter des chaînes HTML pour les données de cellules formatées afin de remplir un DataTable à l'aide des API Aspose.Cells.
