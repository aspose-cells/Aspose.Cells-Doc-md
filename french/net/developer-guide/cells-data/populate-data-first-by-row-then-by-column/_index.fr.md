{{% alert color="primary" %}} 

Remplir une feuille de calcul avec des données d'abord par ligne, puis par colonne, améliore les performances globales.

{{% /alert %}} 

Mettre les données dans la séquence A1, B1, A2, B2 est plus rapide que A1, A2, B1, B2. S'il y a de nombreuses cellules dans une feuille de calcul et que vous suivez la deuxième séquence, c'est-à-dire que vous remplissez les données ligne par ligne, cette astuce peut rendre le programme beaucoup plus rapide.
