##  **Scénarios d'utilisation possibles**

Vous pouvez spécifier des champs de formule lorsque vous importez des données dans votre feuille de calcul à l'aide de l'outil[**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas) . Cette propriété prend le tableau booléen où la valeur**vrai**signifie que le champ est un champ de formule. Par exemple, si le troisième champ est un champ de formule, alors la troisième valeur du tableau sera *true**.
