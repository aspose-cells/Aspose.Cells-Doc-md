## **Scénarios d'utilisation possibles**

Vous pouvez spécifier les champs de formule lors de l'importation de données dans votre feuille de calcul en utilisant le [**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas). Cette propriété prend le tableau booléen où la valeur **true** signifie que le champ est un champ de formule. Par exemple, si le troisième champ est un champ de formule, alors la troisième valeur dans le tableau sera **true**.
