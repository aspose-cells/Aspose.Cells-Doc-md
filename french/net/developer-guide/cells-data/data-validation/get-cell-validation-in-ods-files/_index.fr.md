## **Obtenir la validation Cell dans les fichiers ODS**

Avec Aspose.Cells, vous pouvez obtenir la validation appliquée à une cellule dans les fichiers ODS. Pour cela, le API fournit le[**ObtenirValidation**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) méthode de la[**Cell**](https://reference.aspose.com/cells/net/aspose.cells/cell) classer.
