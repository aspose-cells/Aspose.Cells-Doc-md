##  **Scénarios d'utilisation possibles**
Lorsque vous devez définir des données stylisées pour un Cell spécifique, vous pouvez attribuer une chaîne HTML au Cell. Bien entendu, vous pouvez également obtenir la chaîne HTML de la cellule. Aspose.Cells offre cette fonctionnalité. Aspose.Cells fournit les propriétés et méthodes suivantes pour vous aider à atteindre vos objectifs.
- [**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/htmlstring)
