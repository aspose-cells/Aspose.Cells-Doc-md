{{% alert color="primary" %}}

 Parfois, vous devez trouver des cellules avec un style particulier appliqué. Vous pouvez utiliser Aspose.Cells pour trouver toutes les cellules avec un style commun. Aspose.Cells fournit le[**FindOptions.StyleFindOptions.Style**](https://reference.aspose.com/cells/net/aspose.cells/findoptions/properties/style) propriété que vous pouvez utiliser pour spécifier le style de recherche des cellules.

{{% /alert %}}

Le code de cet exemple recherche toutes les cellules qui ont le même style que celui de la cellule A1. Une fois le code exécuté, toutes les cellules qui ont le même style que A1 contiennent le texte "Trouvé".
