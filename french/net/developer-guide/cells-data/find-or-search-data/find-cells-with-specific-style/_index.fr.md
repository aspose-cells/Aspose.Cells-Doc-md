{{% alert color="primary" %}}

 Parfois, vous devez rechercher des cellules auxquelles un style particulier est appliqué. Vous pouvez utiliser Aspose.Cells pour rechercher toutes les cellules ayant un style commun. Aspose.Cells fournit le[**RechercherOptions.Style**](https://reference.aspose.com/cells/net/aspose.cells/findoptions/properties/style) propriété que vous pouvez utiliser pour spécifier le style dans lequel rechercher les cellules.

{{% /alert %}}

Le code de cet exemple recherche toutes les cellules ayant le même style que celui de la cellule A1. Une fois le code exécuté, toutes les cellules ayant le même style que A1 contiennent le texte « Trouvé ».
