## **Obtenir des hyperliens dans la plage**

 Le[**Intervalle**](https://reference.aspose.com/cells/net/aspose.cells/range) la classe offre une[**Hyperliens**](https://reference.aspose.com/cells/net/aspose.cells/range/properties/hyperlinks) propriété qui renvoie tous les liens hypertexte dans la plage sélectionnée. Vous pouvez également supprimer le lien hypertexte en appelant le[**Lien hypertexte.Supprimer**](https://reference.aspose.com/cells/net/aspose.cells/hyperlink/methods/delete) méthode.
