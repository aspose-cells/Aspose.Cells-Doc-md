{{% alert color="primary" %}}

 Aspose.Cells vous permet d'importer la carte XML dans le classeur à l'aide de la[**Classeur.ImportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/importxml/index) méthode. Vous pouvez importer une carte XML à l'aide du Microsoft Excel en suivant les étapes suivantes

-  Sélectionner**Développeur** languette
-  Cliquez sur**Importer** dans la section XML et suivez les étapes requises.

 Vous devrez fournir vos données XML pour terminer l'importation. Voici une[exemples de données XML](5115037.txt) que vous pouvez utiliser pour les tests.
