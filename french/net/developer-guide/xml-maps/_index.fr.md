{{% alert color="primary" %}}

Aspose.Cells vous permet d'importer la carte XML à l'intérieur du classeur en utilisant la méthode [**Workbook.ImportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/importxml/index). Vous pouvez importer la carte XML en suivant les étapes suivantes dans Microsoft Excel

- Sélectionnez l'onglet **Développeur**
- Cliquez sur **Importer** dans la section XML et suivez les étapes requises.

Vous devrez fournir vos données XML pour compléter l'importation. Voici un [exemple de données XML](5115037.txt) que vous pouvez utiliser pour les tests.

{{% /alert %}}
