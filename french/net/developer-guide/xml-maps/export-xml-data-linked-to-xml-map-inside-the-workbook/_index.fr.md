## **Exporter les données XML liées à la carte XML dans le classeur**

 Veuillez utiliser le[**Classeur.ExportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/exportxml/index) méthode pour exporter des données XML liées à vos cartes XML dans votre classeur. L'exemple de code suivant exporte les données XML de tous les mappages XML du classeur un par un. S'il vous plaît, vérifiez le[exemple de fichier excel](5115497.xlsx) utilisé dans ce code et le[données XML exportées de la première carte XML](5472487.xml).
