## Vérifiez si le projet VBA est protégé et verrouillé pour la visualisation en C#

Aspose.Cells vous permet de vérifier si le projet VBA (Visual Basic for Applications) d'un fichier Excel est protégé et verrouillé pour la visualisation. Pour cela, l'API fournit la propriété [**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing). Si elle est verrouillée pour la visualisation, alors la propriété [**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) retourne **true**.
