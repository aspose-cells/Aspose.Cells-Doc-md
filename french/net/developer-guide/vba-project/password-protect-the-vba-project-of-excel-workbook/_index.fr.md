## **Protégez par mot de passe le projet VBA du classeur Excel dans C#**

Vous pouvez protéger par mot de passe le projet VBA (Visual Basic pour Applications) du classeur avec Aspose.Cells en utilisant[**VbaProject.Protect()**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/methods/protect)méthode.

 L'exemple de code suivant charge le[exemple de fichier Excel](43352067.xlsm), accède à son projet VBA et le protège avec un mot de passe. Enfin, il l'enregistre en tant que[fichier Excel de sortie](43352068.xlsm).
