## **Protéger par mot de passe le projet VBA du classeur Excel en C#**

Vous pouvez protéger par mot de passe le projet VBA (Visual Basic for Applications) d'un classeur avec Aspose.Cells en utilisant la méthode [**VbaProject.Protect()**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/methods/protect).

## **Code d'exemple**

Le code d'exemple suivant charge le [fichier Excel d'exemple](43352067.xlsm), accède à son projet VBA et le protège avec un mot de passe. Enfin, il l'enregistre en tant que [fichier Excel de sortie](43352068.xlsm).
