{{% alert color="primary" %}}

Vous pouvez définir le commentaire de la table ou de l'objet liste à l'intérieur de la feuille de calcul en utilisant la propriété [**ListObject.Comment**](https://reference.aspose.com/cells/net/aspose.cells.tables/listobject/properties/comment). Le commentaire sera visible dans le fichier xl/tables/tableName.xml.

{{% /alert %}}
