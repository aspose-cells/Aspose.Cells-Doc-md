## **Scénarios d'utilisation possibles**
Auparavant, vous deviez changer les paramètres régionaux de l'ensemble du thread pour gérer les nombres et les dates dans un format culturel particulier, mais maintenant Aspose.Cells fournit la propriété [LoadOptions.CultureInfo](https://reference.aspose.com/cells/net/aspose.cells/loadoptions/properties/cultureinfo) que vous pouvez utiliser pour charger votre classeur avec des paramètres régionaux spécifiques sans changer ceux de l'ensemble du thread.
