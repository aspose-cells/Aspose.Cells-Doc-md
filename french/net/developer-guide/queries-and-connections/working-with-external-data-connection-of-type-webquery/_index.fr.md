{{% alert color="primary" %}}

 Vous pouvez accéder à une connexion de données externe de tout type à l'aide de la collection Workbook.DataConnections. Un type de connexion de données de ce type est WebQuery. Cet article vous montrera comment travailler avec la connexion de données WebQuery. Vous pouvez créer une connexion de données WebQuery dans Microsoft Excel en utilisant le**Données** > **À partir du Web** menu.

{{% /alert %}}
