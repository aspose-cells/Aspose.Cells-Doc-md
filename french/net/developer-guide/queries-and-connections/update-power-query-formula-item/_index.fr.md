## **Scénario d'utilisation**

Il peut arriver que les fichiers source des données soient déplacés et que le fichier Excel ne puisse pas localiser le fichier. Dans de tels cas, l'API Aspose.Cells offre la possibilité de mettre à jour l'élément de formule Power Query en utilisant la classe [*PowerQueryFormulaItem*](https://reference.aspose.com/cells/net/aspose.cells.querytables/powerqueryformulaitem) pour mettre à jour l'emplacement du fichier source.
