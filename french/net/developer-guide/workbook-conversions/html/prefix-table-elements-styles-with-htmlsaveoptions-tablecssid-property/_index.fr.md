## **Scénarios d'utilisation possibles**

Aspose.Cells vous permet de préfixer les styles d'éléments de table avec[**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/tablecssid)propriété. Supposons que vous définissiez cette propriété avec une valeur telle que**MyTest_TableCssId**, vous trouverez alors des styles d'éléments de tableau comme indiqué ci-dessous

{{< highlight "java" >}}

 table#MyTest_TableCssId

# MyTest_TableCssId tr

# MyTest_TableCssId col

# MyTest_TableCssId br

etc.

{{< /highlight >}}
