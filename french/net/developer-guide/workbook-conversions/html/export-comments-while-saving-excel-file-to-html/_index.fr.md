## **Scénarios d'utilisation possibles**

Lorsque vous enregistrez votre fichier Excel au format HTML, les commentaires ne sont pas exportés. Cependant, Aspose.Cells fournit cette fonctionnalité en utilisant le[**HtmlSaveOptions.IsExportComments**](https://docs.aspose.com/cells/net/export-comments-while-saving-excel-file-to/) propriété. Si vous le réglez**vrai**, alors HTML affichera également les commentaires présents dans votre fichier Excel.
