Les installations de structure logique PDF fournissent un mécanisme pour incorporer des informations sur la structure du contenu du document dans un fichier PDF. Aspose.Cells préserve les informations sur la structure d'un document Microsoft Excel, telles que cellule, ligne, table, feuille de calcul, image, forme, en-tête/pied de page, etc.

Avec l'option [PdfSaveOptions.ExportDocumentStructure](https://reference.aspose.com/cells/net/aspose.cells/pdfsaveoptions/exportdocumentstructure/), vous pouvez enregistrer un PDF balisé avec la structure du document exportée.
