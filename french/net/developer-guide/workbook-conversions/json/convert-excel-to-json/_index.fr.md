## **Convertir un classeur Excel en JSON**

Pas besoin de se demander comment convertir Excel Workbook en JSON, car la bibliothèque Apose.Cells for .NET a la meilleure décision. Le Aspose.Cells API prend en charge la conversion des feuilles de calcul au format JSON. Pour exporter le classeur vers JSON, passez[**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat) comme deuxième paramètre de[**Classeur.Enregistrer**](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/3) méthode. Vous pouvez également utiliser[**JsonSaveOptionsJsonSaveOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonSaveoptions) class pour spécifier des paramètres supplémentaires pour l'exportation de la feuille de calcul vers JSON.
