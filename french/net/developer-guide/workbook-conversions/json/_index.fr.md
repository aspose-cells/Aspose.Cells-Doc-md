## **Convertir un classeur Excel en JSON**

L'API Aspose.Cells prend en charge la conversion des feuilles de calcul en format JSON. Pour exporter le classeur en JSON, passez [**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat) comme deuxième paramètre de la méthode [**Workbook.Save**](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/3). Vous pouvez également utiliser la classe [**JsonSaveOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonSaveoptions) pour spécifier des paramètres supplémentaires pour exporter la feuille de calcul en JSON.

L'exemple de code suivant démontre l'exportation de la feuille de calcul active en Json en utilisant le membre d'énumération [**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat). Veuillez consulter le code pour convertir le [fichier source](book1.xlsx) en le [fichier Json de sortie](book1.Json) généré par le code pour référence.
