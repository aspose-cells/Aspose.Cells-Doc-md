{{% alert color="primary" %}}

Aspose.Cells prend en charge l'enregistrement d'un fichier ODS (**Feuille de calcul OpenDocument**) dans l'ODF (**Format OpenDocument** ) Spécifications 1.1 et 1.2. Aspose.Cells a[**OdsSaveOptions.IsStrictSchema11**](https://reference.aspose.com/cells/net/aspose.cells/odssaveoptions/properties/isstrictschema11) propriété qui spécifie l'utilisation de la spécification ODF 1.1 pour enregistrer les fichiers ODS. La valeur par défaut de cette propriété est**faux**, ainsi le fichier ODS enregistré sans ce paramètre utilise les spécifications 1.2.

{{% /alert %}}

L'exemple de code ci-dessous crée un objet classeur, ajoute une valeur à la cellule A1 sur la première feuille de calcul, puis enregistre le fichier ODS dans les spécifications ODF 1.1 et 1.2. Par défaut, le fichier ODS est enregistré dans la spécification ODF 1.2.
