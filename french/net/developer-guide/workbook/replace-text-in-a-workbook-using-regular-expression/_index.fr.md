Aspose.Cells propose la fonctionnalité de remplacer du texte dans un classeur à l'aide d'une expression régulière. Pour cela, l'API fournit la propriété [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) de la classe [**ReplaceOptions**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions). Définir [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) sur **true** indique que la clé recherchée sera une expression régulière.

L'extrait de code suivant démontre l'utilisation de la propriété [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) en utilisant le [fichier Excel d'exemple](101089318.xlsx).
