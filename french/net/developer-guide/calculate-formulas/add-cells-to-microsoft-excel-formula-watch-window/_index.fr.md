##  **Scénarios d'utilisation possibles**

Microsoft Excel Watch Window est un outil utile pour visualiser facilement les valeurs des cellules et leurs formules dans une fenêtre. Vous pouvez ouvrir le*Fenêtre de surveillance* en utilisant Microsoft Excel en cliquant sur le*Formules > Regarder* *Fenêtre*. Il a le *Ajouter une montre*bouton qui peut être utilisé pour ajouter les cellules à inspecter. De la même manière, vous pouvez utiliser[**Feuille de calcul.CellWatches.Add()**](https://reference.aspose.com/cells/net/aspose.cells/cellwatchcollection/methods/add/index) méthode pour ajouter des cellules dans*Fenêtre de surveillance*en utilisant le Aspose.Cells API.
