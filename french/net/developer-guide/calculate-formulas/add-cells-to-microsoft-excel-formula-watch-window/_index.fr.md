## **Scénarios d'utilisation possibles**

Microsoft Excel Watch Window est un outil utile pour observer les valeurs des cellules et ses formules de manière pratique dans une fenêtre. Vous pouvez ouvrir le*Fenêtre de surveillance* en utilisant Microsoft Excel en cliquant sur le*Formules > Regarder* *Fenêtre*. Il a les*Ajouter une montre*bouton qui peut être utilisé pour ajouter les cellules à inspecter. De même, vous pouvez utiliser[**Feuille de calcul.CellWatches.Add()**](https://reference.aspose.com/cells/net/aspose.cells/cellwatchcollection/methods/add/index) méthode pour ajouter des cellules dans*Fenêtre de surveillance*en utilisant Aspose.Cells API.
