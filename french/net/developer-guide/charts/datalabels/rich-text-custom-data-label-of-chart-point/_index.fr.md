{{% alert color="primary" %}}

 Vous pouvez utiliser Aspose.Cells pour créer une étiquette de données personnalisée en texte enrichi du point du graphique. Aspose.Cells fournit le[**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) méthode pour retourner le[**Paramètres de police**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) objet qui peut être utilisé pour définir les propriétés de police du texte comme sa couleur, son gras, etc.

{{% /alert %}}
