{{% alert color="primary" %}}

 Vous pouvez utiliser Aspose.Cells pour créer une étiquette de données personnalisée en texte enrichi du point de graphique. Aspose.Cells fournit le[**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) méthode pour retourner le[**FontSetting**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) objet qui peut être utilisé pour définir les propriétés de la police du texte comme sa couleur, sa mise en gras, etc.

{{% /alert %}}
