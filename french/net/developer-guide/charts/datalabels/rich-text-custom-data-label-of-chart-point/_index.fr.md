{{% alert color="primary" %}}

Vous pouvez utiliser Aspose.Cells pour créer une étiquette de données personnalisée en texte enrichi du point du graphique. Aspose.Cells fournit la méthode [**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) pour renvoyer l'objet [**FontSetting**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) qui peut être utilisé pour définir les propriétés de police du texte comme sa couleur, son épaisseur, etc.

{{% /alert %}}
