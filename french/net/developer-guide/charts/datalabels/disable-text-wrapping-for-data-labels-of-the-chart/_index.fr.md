{{% alert color="primary" %}}

Microsoft Excel 2013 permet aux utilisateurs d'envelopper ou de déballer du texte à l'intérieur des étiquettes de données du graphique. Par défaut, le texte à l’intérieur des étiquettes de données du graphique est à l’état enveloppé.

 Aspose.Cells fournit un[**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) propriété que vous pouvez définir sur True ou False pour activer ou désactiver l'habillage du texte des étiquettes de données respectivement.

{{% /alert %}}
