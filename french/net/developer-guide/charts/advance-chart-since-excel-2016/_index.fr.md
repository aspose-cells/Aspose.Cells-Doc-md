##  **Scénarios d'utilisation possibles**
Aspose.Cells prend désormais en charge la lecture et la manipulation des graphiques Microsoft Excel 2016 qui ne sont pas présents dans Microsoft Excel 2013 ou dans les versions antérieures.
##  **Lire et manipuler des graphiques Excel 2016**
 L'exemple de code suivant charge le[fichier Excel source](22774101.xlsx) qui contient des graphiques Excel 2016 dans la première feuille de calcul. Il lit tous les graphiques un par un et modifie son titre selon son type de graphique. La capture d'écran suivante montre le fichier Excel source avant l'exécution du code. Comme vous pouvez le constater, le titre du graphique est le même pour tous les graphiques.
