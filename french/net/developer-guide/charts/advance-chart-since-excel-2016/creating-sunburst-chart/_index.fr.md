## **Scénarios d'utilisation possibles**
Les diagrammes de treemap sont utiles pour comparer les proportions au sein de la hiérarchie, cependant, les diagrammes de treemap ne sont pas très efficaces pour montrer les niveaux hiérarchiques entre les plus grandes catégories et chaque point de données. Un diagramme sunburst est bien plus efficace pour montrer cela. Le diagramme sunburst est idéal pour afficher des données hiérarchiques. Chaque niveau de la hiérarchie est représenté par un anneau ou un cercle, avec le cercle le plus interne en haut de la hiérarchie. Un diagramme sunburst sans données hiérarchiques (un niveau de catégories) ressemble à un diagramme en anneau. Cependant, un diagramme sunburst avec plusieurs niveaux de catégories montre comment les anneaux extérieurs sont liés aux anneaux intérieurs. Le diagramme sunburst est le plus efficace pour montrer comment un anneau est divisé en ses éléments constitutifs, alors qu'un autre type de diagramme hiérarchique, le diagramme de treemap, est idéal pour comparer les tailles relatives.
