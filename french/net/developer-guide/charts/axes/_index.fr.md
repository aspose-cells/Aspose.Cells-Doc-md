{{% alert color="primary" %}}

Dans les graphiques Excel, il existe 3 types d'axes :
1. Axe horizontal (catégorie) : axe X
1. Axe vertical (valeur) : axe Y
1. Axe de profondeur (série) : axe Z



{{% /alert %}}

##  **Options d'axe**
 Aspose.Cells permet également de gérer les axes du graphique lors de l'exécution, avec[Axe](https://reference.aspose.com/cells/net/aspose.cells.charts/axis/) objet, vous pouvez modifier toutes les options d’Axis comme dans Excel.
