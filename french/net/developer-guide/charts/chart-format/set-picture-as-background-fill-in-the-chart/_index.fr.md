{{% alert color="primary" %}}

Aspose.Cells vous permet de définir un dégradé, une texture, un motif ou une image comme effets de remplissage pour différents objets, tels que la zone de traçage, la zone de graphique ou la boîte de légende d'un graphique. Ce document montre comment ajouter une image à l'arrière-plan d'un graphique.

{{% /alert %}}

Pour ce faire, Aspose.Cells fournit la propriété [**Chart.PlotArea.Area.FillFormat.ImageData**](https://reference.aspose.com/cells/net/aspose.cells.drawing/fillformat/properties/imagedata).
