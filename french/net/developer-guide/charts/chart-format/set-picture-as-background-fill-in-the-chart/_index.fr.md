{{% alert color="primary" %}}

Aspose.Cells vous permet de définir un dégradé, une texture, un motif ou une image comme effets de remplissage pour différents objets, tels que la zone de tracé, la zone de graphique ou la zone de légende d'un graphique. Ce document montre comment ajouter une image à l'arrière-plan d'un graphique.

{{% /alert %}}

Pour y parvenir, le Aspose.Cells met à votre disposition le[**Chart.PlotArea.Area.FillFormat.ImageData**](https://reference.aspose.com/cells/net/aspose.cells.drawing/fillformat/properties/imagedata) propriété.
