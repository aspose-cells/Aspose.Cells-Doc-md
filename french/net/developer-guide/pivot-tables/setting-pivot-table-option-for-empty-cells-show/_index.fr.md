{{% alert color="primary" %}}

Vous pouvez définir différentes options de tableau croisé dynamique à l'aide de Aspose.Cells. L'une de ces options est "Pour l'affichage des cellules vides". En définissant cette option, toutes les cellules vides d'un tableau croisé dynamique sont affichées sous la forme d'une chaîne spécifiée.

{{% /alert %}}

## **Définition de l'option de tableau croisé dynamique dans Microsoft Excel**

Pour rechercher et définir cette option dans Microsoft Excel :

1. Sélectionnez un tableau croisé dynamique et cliquez avec le bouton droit.
1.  Sélectionner**Options de tableau croisé dynamique**.
1.  Sélectionnez le**Mise en page et format** languette.
1.  Sélectionnez le**Pour les cellules vides afficher** option et spécifiez une chaîne.

## **Définition de l'option de tableau croisé dynamique à l'aide de Aspose.Cells**

 Aspose.Cells fournit le[**PivotTable.DisplayNullStringPivotTable.DisplayNullString**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/displaynullstring) et[**PivotTable.NullStringPivotTable.NullString**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/nullstring) propriétés pour définir l'option de tableau croisé dynamique "Pour les cellules vides afficher".
