{{% alert color="primary" %}}

 Microsoft Excel vous permet de modifier la disposition du tableau croisé dynamique à l'aide*Outils de tableau croisé dynamique > Conception > Mise en page du rapport* commandes des menus. Vous pouvez modifier la mise en page dans ces trois formulaires

- Afficher sous forme compacte
- Afficher sous forme de plan
- Afficher sous forme tabulaire

 Aspose.Cells fournit également[**Tableau croisé dynamique.ShowInCompactForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showincompactform), [**Tableau croisé dynamique.ShowInOutlineForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showinoutlineform) et[**Tableau croisé dynamique.ShowInTabularForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showintabularform) méthodes pour changer la disposition du tableau croisé dynamique dans ces trois formes.

{{% /alert %}}
