Aspose.Cells offre la possibilité de sauvegarder un tableau croisé dynamique dans des fichiers ODS. Pour cela, convertissez simplement un classeur avec un tableau croisé dynamique existant ou créez un nouveau tableau croisé dynamique et enregistrez le fichier au format ODS. Assurez-vous d'appeler [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata) avant l'enregistrement pour garantir que le tableau croisé dynamique est rendu dans le fichier ODS de sortie.
