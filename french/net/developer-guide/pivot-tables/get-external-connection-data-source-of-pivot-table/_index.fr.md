## **Obtenir la source de données de connexion externe du tableau croisé dynamique**

Aspose.Cells offre la possibilité d'obtenir la source de données de connexion externe du tableau croisé dynamique. Pour cela, l'API fournit la propriété [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) de la classe [**PivotTable**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable). La propriété [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) renvoie l'objet [**ExternalConnection**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection). L'extrait de code suivant montre l'utilisation de la propriété [**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) pour obtenir la source de données de connexion externe du tableau croisé dynamique.
