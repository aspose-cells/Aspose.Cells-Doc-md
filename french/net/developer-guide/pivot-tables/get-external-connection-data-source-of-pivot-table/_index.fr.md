## **Obtenir la source de données de connexion externe du tableau croisé dynamique**

Aspose.Cells offre la possibilité d'obtenir la source de données de connexion externe du tableau croisé dynamique. Pour cela, le API fournit le[**ExternalConnectionDataSourceExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)propriété de la[**Tableau croisé dynamique**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) classer. La[**ExternalConnectionDataSourceExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)la propriété renvoie le[**ConnexionExterne**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) objet. L'extrait de code suivant illustre l'utilisation de[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) propriété pour obtenir la source de données de connexion externe du tableau croisé dynamique.
