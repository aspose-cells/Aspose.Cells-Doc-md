Supprimer les lignes en double est l'une des nombreuses fonctionnalités utiles d'Excel Microsoft. Il permet aux utilisateurs de supprimer les lignes en double dans une feuille de calcul, vous pouvez choisir les colonnes à vérifier pour les informations en double.

Aspose.Cells fournit la méthode Cells.RemoveDuplicates() à cet effet. Vous pouvez également définir startRow, startColumn, endRow, endColumn pour récupérer les colonnes.
