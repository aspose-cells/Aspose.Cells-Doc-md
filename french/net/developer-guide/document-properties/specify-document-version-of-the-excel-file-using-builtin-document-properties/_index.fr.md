## **Scénarios d'utilisation possibles**

Vous pouvez modifier le **Numéro de version** du fichier Excel en cliquant avec le bouton droit sur le fichier, puis en sélectionnant Propriétés > Détails et en éditant le champ **Numéro de version**. Veuillez utiliser la propriété [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion) pour le modifier de manière programmatique à l'aide des API Aspose.Cells.
