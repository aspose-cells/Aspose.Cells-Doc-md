## **Scénarios d'utilisation possibles**

 Vous pouvez changer le**Numéro de version** du fichier Excel en cliquant avec le bouton droit sur le fichier, puis en sélectionnant Propriétés> Détails, puis en modifiant le**Numéro de version** domaine. Veuillez utiliser[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion)propriété pour le modifier par programmation à l'aide des API Aspose.Cells.
