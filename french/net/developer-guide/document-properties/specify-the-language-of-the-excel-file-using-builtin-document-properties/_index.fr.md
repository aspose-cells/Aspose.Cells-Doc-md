## **Scénarios d'utilisation possibles**

Vous pouvez modifier la langue du fichier Excel en cliquant avec le bouton droit sur le fichier, puis en sélectionnant Propriétés > Détails, puis en modifiant le champ Langue. Veuillez utiliser[**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/language)propriété pour le modifier par programmation à l'aide des API Aspose.Cells.
