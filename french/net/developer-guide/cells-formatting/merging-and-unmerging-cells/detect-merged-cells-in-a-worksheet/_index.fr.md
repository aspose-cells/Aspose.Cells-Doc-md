Cet article fournit des informations sur la façon d'obtenir des zones de cellules fusionnées dans une feuille de calcul.

Aspose.Cells vous permet d'obtenir des zones de cellules fusionnées dans une feuille de calcul. Vous pouvez également les dissocier (diviser). Cet article montre le code le plus simple utilisant**Aspose.Cells API** pour effectuer la tâche.

{{% /alert %}}
