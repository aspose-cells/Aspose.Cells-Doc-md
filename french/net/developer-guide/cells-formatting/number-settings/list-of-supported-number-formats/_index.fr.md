##  **Aspose.Cells**
 Le composant Aspose.Cells propose des formats de nombres intégrés pour configurer les formats d'affichage des nombres et des dates. Ces formats de nombres intégrés peuvent être appliqués en utilisant le**Nombre** propriété du**Style** objet. Tous les formats de nombres intégrés reçoivent des valeurs numériques uniques. Les développeurs peuvent attribuer n'importe quelle valeur numérique souhaitée au*Nombre* propriété de**Style** objet et donc le format d’affichage est appliqué. Cette approche est plus rapide.
