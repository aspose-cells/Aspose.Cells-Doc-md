Aspose.Cells offre des fonctionnalités riches pour gérer les formes dans la feuille de calcul. Parfois, il est nécessaire d'obtenir les points de connexion d'une forme pour aligner ou placer les formes au bon endroit. À cette fin, tous les points de connexion sont requis. Le code suivant peut être utilisé pour obtenir la liste des points de connexion d'une forme en utilisant la propriété [Shape.ConnectionPoints](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/connectionpoints).
