## **Scénarios d'utilisation possibles**
Les zones de texte peuvent avoir des balises qui peuvent être remplacées par du texte au moment de l'exécution pour les configurer en fonction des besoins. Les balises peuvent être des étiquettes entre crochets '<' and '>. Il peut y avoir plusieurs balises dans une même zone de texte.
## **Exemple de code**
L'exemple de code suivant remplace les balises TAG_1 et TAG_2 avec du texte, dites « oui » et « 1 ». Un exemple de fichier pour tester le code ci-dessous peut être téléchargé à partir du lien suivant :
