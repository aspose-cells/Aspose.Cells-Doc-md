## **Scénarios d'utilisation possibles**

Si vous voulez supprimer un filtre dans Microsoft Excel, sélectionnez-le et appuyez sur le bouton *Supprimer*. De même, si vous voulez le supprimer en utilisant le code Aspose.Cells de manière programmée, veuillez utiliser la méthode [**Worksheet.Slicers.Remove()**](https://reference.aspose.com/cells/net/aspose.cells.slicers/slicercollection/methods/remove). Cela supprimera le filtre de la feuille de calcul.
