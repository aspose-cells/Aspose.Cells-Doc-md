## **Scénarios d'utilisation possibles**

 Si vous souhaitez supprimer le trancheur dans Microsoft Excel, sélectionnez-le simplement et appuyez sur le bouton*Effacer* bouton. De même, si vous souhaitez le supprimer à l'aide de Aspose.Cells API par programme, veuillez utiliser le[**Feuille de calcul.Slicers.Remove()**](https://reference.aspose.com/cells/net/aspose.cells.slicers/slicercollection/methods/remove)méthode. Cela supprimera le segment de la feuille de calcul.
