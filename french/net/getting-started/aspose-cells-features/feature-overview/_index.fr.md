Le produit Aspose.Cells for .NET couvre la plupart des environnements de développement et plates-formes de déploiement populaires.

| :- | :- |

Prise en charge complète des versions 32 bits et 64 bits.|
|[.NET Norme 2.0](/cells/fr/net/system-requirements/#net-standard)|Le dossier « netStandard » contient un assembly à utiliser avec un framework qui implémente .NET Standard 2.0 tel que .NET Core 2.1/3.1. |[Windows Azur](/cells/fr/net/system-requirements/)|Aspose.Cells for .NET fonctionne également sur Windows Azure.|
