## **Plateformes prises en charge**
Aspose.Cells for .NET couvre la plupart des environnements de développement populaires et des plates-formes de déploiement. 

|**Fonctionnalité**|**Description**|
| :- | :- |
|[.NET Framework](/cells/fr/net/system-requirements/#net-frameworks)|Utilisez Aspose.Cells for .NET pour construire des applications ASP.NET, des services web, des formulaires Windows ou tout autre type d'application pour n'importe quelle version de .NET Framework 2.0 ou ultérieure, y compris les versions du profil client. Support complet pour les versions 32 bits et 64 bits.|
|[.NET Standard 2.0](/cells/fr/net/system-requirements/#net-standard)|Le dossier 'netStandard' contient l'assembly à utiliser avec un framework qui met en œuvre .NET Standard 2.0 tel que .NET Core 2.1/3.1. La dernière version de l'API Aspose.Cells prend également en charge .NET 5.0, .NET 6.0, .NET 7.0 et .NET8.0.|
|[COM](/cells/fr/net/system-requirements/)|Peut être un atout si vous devez développer en utilisant des technologies antérieures à .NET.|
|[Mono](/cells/fr/net/system-requirements/)|Vous pouvez utiliser Aspose.Cells for .NET pour construire des applications avec Mono.|
|[Windows Azure](/cells/fr/net/system-requirements/)|Aspose.Cells for .NET fonctionne également sur Windows Azure.|
