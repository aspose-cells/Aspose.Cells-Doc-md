## La dernière version de Aspose.Cells pour iOS via Xamarin peut ne pas fonctionner avec l'ancienne version de Xamarin.iOS
Veuillez noter que Aspose.Cells pour iOS via Xamarin est toujours construit en utilisant les dernières versions stables des plates-formes Xamarin et Xamarin.iOS. Si vous rencontrez des problèmes lors de l'utilisation de Aspose.Cells pour iOS via Xamarin dans votre application Xamarin.Android, assurez-vous que les dernières versions de Xamarin et Xamarin.iOS sont installées. Parfois, Aspose.Cells pour iOS via Xamarin est construit en utilisant la dernière version de Xamarin (Xamarin.iOS) qui ne fonctionne pas avec les anciennes versions de Xamarin.
## Aspose.Cells pour iOS via Xamarin Limitations
- Insertion d'images - Non pris en charge.
- Création de graphiques - Non pris en charge.
- Réglage de l'arrière-plan dégradé - Non pris en charge.
- Ajout de commentaires aux cellules - Non pris en charge.
- Implémentation des validations de données - Non pris en charge.
- Création de sauts de page personnalisés - Non pris en charge.
- Implémentation de marqueurs intelligents - Non pris en charge.
- Protéger/déprotéger les feuilles de travail - Non pris en charge.
- Spécification des options de protection avancées introduites dans Excel XP et les versions ultérieures - Non pris en charge.
- Insertion de contrôles de formulaire et d'autres formes/objets de dessin - Non pris en charge.
- Création de tableaux croisés dynamiques et de graphiques croisés dynamiques - Non pris en charge.
- Conservation ou suppression d'un complément, VBA, macros - Non pris en charge.
- Implémentation des options de transposition - Non pris en charge.
- Création de graphiques personnalisés - Non pris en charge.
- Ajout, conservation ou extraction d'objets OLE à partir des feuilles de calcul - Non pris en charge.
- Mise en œuvre des lignes Spark Microsoft Excel 2010 - Non pris en charge.
- Cryptage des fichiers - Non pris en charge.
## Public API (espace de noms) différences
Dans Aspose.Cells pour iOS via Xamarin, l'espace de noms Aspose.Cells.Drawing est utilisé à la place de System.Drawing dans .NET API. La liste des objets concernés est la suivante :

- Aspose.Cells.Drawing.Color
- Aspose.Cells.Drawing.ColorConverter
- Aspose.Cells.Drawing.ColorTranslator
- Aspose.Cells.Drawing.FontStyle
- Aspose.Cells.Drawing.GraphicsUnit
- Aspose.Cells.Drawing.ImageFormatConverter
- Aspose.Cells.Drawing.KnownColor
- Aspose.Cells.Drawing.KnownColors
- Aspose.Cells.Drawing.Locale
- Aspose.Cells.Drawing.SystemColors
- Aspose.Cells.Drawing.Imaging.PixelFormat
- Aspose.Cells.Drawing.Imaging.ImageFormat
- Aspose.Cells.Drawing.Imaging.ImageFlags
- Aspose.Cells.Drawing.Drawing2D.SmoothingMode
- Aspose.Cells.Drawing.Drawing2D.PathPointType
- Aspose.Cells.Drawing.Drawing2D.PathData
- Aspose.Cells.Drawing.Drawing2D
