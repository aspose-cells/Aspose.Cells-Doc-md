{{% alert color="primary" %}} 

Aspose.Cells for .NET est une bibliothèque de programmation de feuilles de calcul qui permet aux développeurs de logiciels de manipuler et de convertir des fichiers de feuilles de calcul à partir de leurs propres applications. Aspose.Cells for .NET prend en charge les formats de fichiers de feuilles de calcul populaires (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) et permet d'exporter des fichiers Excel vers PDF, XPS, HTML, MHTML, texte brut, TIFF, JPG, PNG, 076111. 3481 et SVG.

 La dernière référence API peut être trouvée sur[Aspose.Cells for .NET API Référence](https://reference.aspose.com/cells/net).
