**Bienvenue dans la section de documentation de Aspose.Cells for JavaScript via C++**

Aspose.Cells for JavaScript via C++ est une bibliothèque WebAssembly native qui permet aux développeurs JavaScript/Node.js de créer, manipuler et convertir des feuilles de calcul de manière programmatique sans nécessiter d'automatisation Office ou de Microsoft Excel.

La bibliothèque supporte les principaux formats de feuilles de calcul, y compris Excel (XLS, XLSX, XLSB, XLSM), ODS, CSV et HTML, offrant une solution complète pour le traitement des feuilles de calcul directement dans le navigateur ou en environnement serveur sans dépendances externes.

{{% /alert %}}
