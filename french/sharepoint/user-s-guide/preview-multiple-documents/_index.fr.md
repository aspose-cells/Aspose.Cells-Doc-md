Vous pouvez prévisualiser plusieurs documents Excel dans la bibliothèque de documents. 

Pour prévisualiser tous les fichiers dans un dossier : 

1. Sélectionnez **Prévisualiser le document avec Aspose.Cells** dans le ECB pour le dossier souhaité tout en restant dans le parent.

Pour prévisualiser plusieurs fichiers depuis un dossier :

1. Sélectionnez plusieurs fichiers.
1. Avec les fichiers toujours sélectionnés, sélectionnez **Prévisualiser le document avec Aspose.Cells**.

**Aperçu du document depuis SharePoint** 

![todo:image_alt_text](preview-multiple-documents_1.png)

Lorsque vous prévisualisez plusieurs documents, la zone de données affiche le premier document de la liste. La liste des documents affiche tous les documents prévisualisés. Pour visualiser un autre document :

1. Sélectionnez un document dans la liste.
1. Cliquez sur **Aperçu**.
