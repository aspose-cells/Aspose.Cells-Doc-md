 Vous pouvez prévisualiser plusieurs documents Excel dans la bibliothèque de documents.

 Pour prévisualiser tous les fichiers d'un dossier :

1.  Sélectionner**Aperçu du document avec Aspose.Cells** dans l'ECB pour le dossier de votre choix tout en restant dans le parent.

Pour prévisualiser plusieurs fichiers à partir d'un dossier :

1. Sélectionnez plusieurs fichiers.
1.  Avec les fichiers toujours sélectionnés, sélectionnez**Aperçu du document avec Aspose.Cells**.

**Aperçu du document à partir de SharePoint** 

![tâche : image_autre_texte](preview-multiple-documents_1.png)

Lorsque vous prévisualisez plusieurs documents, la zone de données affiche le premier document de la liste. La liste des documents affiche tous les documents prévisualisés. Pour afficher un autre document :

1. Sélectionnez un document dans la liste.
1.  Cliquez sur**Aperçu**.
