Pour rechercher tous les fichiers dans un dossier :

1. Sélectionnez **Rechercher dans le document avec Aspose.Cells** dans le ECB du dossier souhaité tout en restant dans le parent.

Pour rechercher plusieurs fichiers en même temps :

1. Sélectionnez plusieurs fichiers.
1. Avec les fichiers toujours sélectionnés, sélectionnez **Rechercher un document avec Aspose.Cells**.

**Recherche de documents** 

![todo:image_alt_text](searching-in-a-document_1.png)

Les options suivantes sont disponibles :

- **Options de recherche** : Concentrez votre recherche en recherchant dans les cellules, les formules, ou en trouvant des formules contenant une expression spécifique.
- **Sensible à la casse** : Lorsque l'option sensible à la casse est sélectionnée, la recherche correspond exactement aux caractères, y compris les lettres minuscules et majuscules.
