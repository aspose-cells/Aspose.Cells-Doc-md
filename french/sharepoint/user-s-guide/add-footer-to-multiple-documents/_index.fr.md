Si vous souhaitez ajouter un pied de page dans plusieurs fichiers Excel, veuillez choisir l'option "Ajouter un pied de page avec Aspose.Cells" dans la barre de ruban.

![tâche : image_autre_texte](add-footer-to-multiple-documents_1.png)



Obtenez tous les fichiers Excel du dossier de source de données et créez un tableau de liste de fichiers.

 Sélectionnez le fichier qui doit ajouter un pied de page, cliquez sur**Ajouter un pied de page** bouton pour ajouter un pied de page pour les fichiers sélectionnés.

![tâche : image_autre_texte](add-footer-to-multiple-documents_2.png)



Les options suivantes sont disponibles lors de l'ajout de la configuration du pied de page :

**Section**

Ajouter une position de pied de page : section gauche, section centrale et section droite.

**Texte de pied de page**

Il représente le script de formatage du pied de page. Commandes de script : Commande | Descriptif| &P Numéro de la page actuelle| &N Nombre de pages|&D Date actuelle| &T Heure actuelle &A Nom de la feuille &F Nom du fichier sans chemin &"<FontName>" Nom de la police, par exemple : &"Arial" &"<FontName>, <FontStyle>" Nom et style de police, par exemple : &"Arial,Gras" &<FontSize> Taille de police. Si cette commande est suivie d'un nombre en clair à imprimer dans l'en-tête, il sera séparé de la hauteur de la police par un espace. &G Script d'image Par exemple : "&Arial,Bold&8Footer Note".
