Si vous souhaitez ajouter un pied de page dans plusieurs fichiers Excel, veuillez choisir l'option « Ajouter un pied de page avec Aspose.Cells » dans la barre de ruban.

![todo:image_alt_text](add-footer-to-multiple-documents_1.png)



Obtenez tous les fichiers Excel à partir du dossier source de données et créez une table de liste de fichiers.

Sélectionnez le fichier auquel ajouter un pied de page, cliquez sur le bouton **Ajouter un pied de page** pour ajouter un pied de page aux fichiers sélectionnés. 

![todo:image_alt_text](add-footer-to-multiple-documents_2.png)



Les options suivantes sont disponibles lors de la configuration de l'ajout de pied de page :

**Section**

Position d'ajout du pied de page : Section gauche, Section centrale et Section droite.

**Script de pied de page**

It represents Footer formatting script. Script commands: Command | Description| &P Current page number| &N Page count|&D Current date| &T Current time &A Sheet name &F File name without path &"<FontName>" Font name, for example: &"Arial" &"<FontName>, <FontStyle>" Font name and font style, for example: &"Arial,Bold" &<FontSize> Font size. If this command is followed by a plain number to be printed in the header, it will be separated from the font height with a space character. &G Image script For example: "&Arial,Bold&8Footer Note".
