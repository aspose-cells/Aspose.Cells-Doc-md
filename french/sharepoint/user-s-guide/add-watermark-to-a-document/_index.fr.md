Si vous souhaitez ajouter un filigrane dans un fichier Excel, veuillez choisir l'option « Ajouter un filigrane avec Aspose.Cells » dans le bloc de contrôle d'édition (ECB) du document.

![todo:image_alt_text](add-watermark-to-a-document_1.png)

Les options suivantes sont disponibles lors de la configuration de l'ajout de filigrane :

![todo:image_alt_text](add-watermark-to-a-document_2.png)

**Fichier de filigrane**

Le fichier doit être un fichier image.
