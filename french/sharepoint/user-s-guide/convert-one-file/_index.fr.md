{{% alert color="primary" %}} 

Une des tâches les plus courantes que les utilisateurs effectuent avec Aspose.Cells est de convertir des fichiers individuels. 

{{% /alert %}} 
### **Conversion de fichiers individuels**
Pour convertir un document particulier dans une bibliothèque de documents : 

1. Sélectionnez **Convertir avec Aspose.Cells.SharePoint** dans le Bloc de contrôle de modification (ECB) du document. 

   **Sélection de l'élément “Convertir via Aspose.Cells” dans le ECB du document** 

![todo:image_alt_text](convert-one-file_1.png)



L'écran des paramètres de conversion s'affiche. Ici, vous pouvez affiner la conversion :

1. Sélectionnez le format de fichier de destination dans la liste **Convertir en**.
1. Entrez le nom du fichier de sortie dans le champ **Fichier de destination**.
   Par défaut, Aspose.Cells for SharePoint pré-remplit ce champ avec le nom du fichier source. 
   L'extension est ajoutée automatiquement en fonction du format sélectionné.
1. Lancez la conversion en cliquant sur **Convertir**.
   (Ou retournez à la bibliothèque, en annulant vos paramètres, en cliquant sur **Annuler**.)
