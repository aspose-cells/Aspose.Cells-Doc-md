Pour convertir tous les fichiers dans un dossier: 

1. Sélectionnez **Convertir avec Aspose.Cells** dans le BCE de votre dossier souhaité tout en restant dans le parent.

Pour convertir plusieurs fichiers: 

1. Sélectionnez plusieurs fichiers dans la bibliothèque et sélectionnez **Convertir avec Aspose.Cells**. 

   **Convertir tous les fichiers dans le sous-dossier** 

![todo:image_alt_text](convert-multiple-files_1.png)




1. Sélectionnez l'élément fichier dans la liste des documents et déplacez-le vers le haut ou vers le bas pour changer l'ordre de conversion. 

   **Changer l'ordre de conversion** 

![todo:image_alt_text](convert-multiple-files_2.png)
