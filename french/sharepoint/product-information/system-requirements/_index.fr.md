### **Exigences du serveur**
Pour installer Aspose.Cells for SharePoint, vous avez besoin d'un ordinateur exécutant l'un des éléments suivants :

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft SharePoint Server 2010
- Microsoft SharePoint Server 2013

{{% alert color="primary" %}} 

Nous recommandons d'installer le Service Pack 2 ou une version ultérieure pour ces produits.

{{% /alert %}}
### **Exigences du client**
Vous avez besoin d'un navigateur pour accéder aux sites SharePoint. Aspose.Cells for SharePoint est testé avec :

- Microsoft Internet Explorer
- Mozilla FireFox
- Google Chrome
- Safari
