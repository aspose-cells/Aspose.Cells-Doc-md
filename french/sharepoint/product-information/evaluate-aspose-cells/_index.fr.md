{{% alert color="primary" %}}

La version d'évaluation de Aspose.Cells pour SharePoint n'a pas de limite de temps et un support technique gratuit est également disponible pour les utilisateurs d'évaluation.

{{% /alert %}} {{% alert color="primary" %}}

 Le téléchargement des versions d'évaluation et complètes de Aspose.Cells pour SharePoint est le même. Téléchargez simplement Aspose.Cells pour SharePoint à partir du[page de téléchargement](https://downloads.aspose.com/cells/sharepoint) et[Installez-le](/cells/fr/sharepoint/install-aspose-cells-for-sharepoint/). Aspose.Cells fonctionne en mode d'évaluation par défaut.

{{% /alert %}}

### **Limites de la version d'évaluation**

 Le mode d'évaluation ajoute un avertissement d'évaluation dans tous les documents exportés. Lorsque vous avez acheté une licence, il vous suffit[installer la licence](/cells/fr/sharepoint/installing-aspose-cells-for-sharepoint-license/) et cela fonctionnera en mode licence.
