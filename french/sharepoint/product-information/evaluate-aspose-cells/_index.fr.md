{{% alert color="primary" %}}

La version d'évaluation de Aspose.Cells for SharePoint n'a pas de limite de temps et un support technique gratuit est également disponible pour les utilisateurs en évaluation.

{{% /alert %}} {{% alert color="primary" %}}

Le téléchargement des versions d'évaluation et complètes de Aspose.Cells for SharePoint est le même. Il suffit de télécharger Aspose.Cells for SharePoint à partir de la [page de téléchargement](https://downloads.aspose.com/cells/sharepoint) et de [l'installer](/cells/fr/sharepoint/install-aspose-cells-for-sharepoint/). Aspose.Cells fonctionne en mode d'évaluation par défaut.

{{% /alert %}}

### **Limitations de la version d'évaluation**

Le mode d'évaluation ajoute un avertissement d'évaluation dans tous les documents exportés. Lorsque vous avez acheté une licence, il vous suffit de [installer la licence](/cells/fr/sharepoint/installing-aspose-cells-for-sharepoint-license/) et il fonctionnera en mode sous licence.
