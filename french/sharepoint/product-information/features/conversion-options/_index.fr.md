{{% alert color="primary" %}} 

Pour rendre le processus de conversion aussi flexible que possible, Aspose.Cells for SharePoints vous permet de définir des options définissant la manière dont le fichier converti sera traité.

{{% /alert %}} 
### **Choix**
Les options suivantes sont disponibles lors de la conversion :
#### **Concaténer des fichiers**
 Sélectionner**Combiner des fichiers** pour combiner des fichiers qui étaient à l'origine dans des formats différents en un seul document.

Cette option est disponible lorsque vous sélectionnez un dossier ou plusieurs fichiers individuels à convertir. Déplacez les documents dans la liste vers le haut ou vers le bas pour définir l'ordre dans lequel vous souhaitez que les documents soient combinés.

#### **Convertir des fichiers dans des sous-dossiers**
 Sélectionner**Convertir des fichiers dans des sous-dossiers** pour convertir de manière récursive tous les documents et sous-dossiers d'un dossier sélectionné.

La structure de sous-dossiers existante est créée dans le dossier de destination.

{{% alert color="primary" %}} 

Cette option n'est disponible que sous SharePoint 2007. Dans SharePoint 2010, les sous-dossiers sont traités par défaut. Pour traiter uniquement les fichiers d'un dossier racine dans SharePoint 2010, sélectionnez les fichiers individuels au lieu du dossier et ne sélectionnez aucun sous-dossier.

{{% /alert %}} 
#### **Remplacer les fichiers existants dans le dossier de destination**
Si le dossier de destination contient déjà des fichiers portant le même nom que ceux que vous souhaitez convertir, vous avez deux options. Sélectionnez pour écraser les fichiers existants afin de remplacer les fichiers déjà existants par des fichiers fraîchement convertis. Décochez l'option pour ignorer le traitement des fichiers du même nom.
