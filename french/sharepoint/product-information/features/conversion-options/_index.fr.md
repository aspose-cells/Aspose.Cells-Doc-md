{{% alert color="primary" %}} 

Pour rendre le processus de conversion aussi flexible que possible, Aspose.Cells for SharePoint vous permet de définir des options qui définissent comment le fichier converti sera traité.

{{% /alert %}} 
### **Options**
Les options suivantes sont disponibles lors de la conversion :
#### **Concaténer des fichiers**
Sélectionnez **Combinaison de fichiers** pour combiner des fichiers qui étaient initialement dans différents formats en un seul document. 

Cette option est disponible lorsque vous sélectionnez un dossier, ou plusieurs fichiers individuels, pour la conversion. Déplacez les documents dans la liste vers le haut ou vers le bas pour définir l'ordre dans lequel vous souhaitez que les documents soient combinés.

#### **Convertir les fichiers dans les sous-dossiers**
Sélectionnez **Convertir les fichiers dans les sous-dossiers** pour convertir de manière récursive tous les documents et sous-dossiers dans un dossier sélectionné. 

La structure de sous-dossier existante est créée dans le dossier de destination.

{{% alert color="primary" %}} 

Cette option n'est disponible que sous SharePoint 2007. Dans SharePoint 2010, les sous-dossiers sont traités par défaut. Pour traiter uniquement les fichiers dans un dossier racine dans SharePoint 2010, sélectionnez les fichiers individuels au lieu du dossier, et ne sélectionnez aucun sous-dossier.

{{% /alert %}} 
#### **Écraser les fichiers existants dans le dossier de destination**
Si le dossier de destination contient déjà des fichiers portant le même nom que ceux que vous souhaitez convertir, vous avez deux options. Sélectionnez pour écraser les fichiers existants pour remplacer les fichiers déjà existants par des fichiers fraîchement convertis. Désélectionnez l'option pour ignorer le traitement des fichiers portant le même nom.
