Aspose.Cells for SharePoint fournit une barre d'outils qui inclut certaines options et commandes précieuses :

- Convertir avec Aspose.Cells
- Aperçu du document avec Aspose.Cells
- Recherche dans le document avec Aspose.Cells
- Ajouter un filigrane avec Aspose.Cells
- Ajouter un pied de page avec Aspose.Cells
