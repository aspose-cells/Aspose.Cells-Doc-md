Aspose.Cells pour SharePoint prend en charge les recherches dans les documents stockés dans les sites SharePoint. Aspose.Cells pour Share Point prend en charge les formats d'entrée suivants pour prévisualiser les documents :

- XLS-Microsoft Excel 97-2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- XML - Microsoft Tableur ExcelML

Pour rechercher dans un document :

1. Sélectionnez un document ou un dossier que vous souhaitez rechercher.
1.  Faites un clic droit et sélectionnez**Rechercher dans le document avec Aspose.Cells**. 
1.  Définissez les critères de recherche.
 Les résultats de la recherche s'affichent.
