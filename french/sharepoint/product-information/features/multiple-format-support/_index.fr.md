{{% alert color="primary" %}}

Avec Aspose.Cells for SharePoint, vous pouvez convertir des documents entre de nombreux formats de document bureautique populaires à partir d'une bibliothèque de documents SharePoint. La plupart des conversions sont effectuées avec une grande fidélité et précision.

{{% /alert %}}

## **Formats d'entrée pris en charge**

Aspose.Cells for SharePoint prend en charge les formats d'entrée suivants :

- XLS – Microsoft Excel 97~2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- CSV – Document de feuille de calcul à valeurs séparées par des virgules
- ODS – OpenDocument
- XML – Microsoft Excel SpreadsheetML

Pour générer des documents, Aspose.Cells for SharePoint s'appuie sur une version intégrée de [Aspose.Cells for .NET](https://products.aspose.com/cells/net/), le composant de traitement de feuilles de calcul de premier plan d'Aspose. Microsoft Excel n'est pas utilisé pour générer des documents.

## **Formats de sortie pris en charge**

La liste complète des formats de sortie pris en charge par Aspose.Cells for SharePoint est :

- PDF – Document portable Adobe
- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM – Office Open XML for macros enabled
- TXT – Plain Text
- XML – Microsoft Excel SpreadsheetML
- ODS – OpenDocument
- CSV – Comma Separated Values
- XLSB – Microsoft Excel Binary 2007
- MHTML – Multipurpose Internet Mail Extension HTML
- XPS – XML Paper Specification
- TIFF – Tagged Image File Format
- SVG – Scalable Vector Graphics
- DIF – Drawing Interchange Format
