{{% alert color="primary" %}} 

 Aspose.Cells pour SharePoint est téléchargeable en tant qu'archive Aspose.Cells.SharePoint.zip.

{{% /alert %}} 
### **Contenu des archives**
L'archive Aspose.Cells.SharePoint.zip contient :

- Aspose.Cells.SharePoint.wsp – Fichier de solution SharePoint. Aspose.Cells pour SharePoint est présenté sous la forme d'une solution SharePoint pour faciliter le déploiement/le retrait et l'activation/la désactivation des fonctionnalités sur l'ensemble de la batterie de serveurs.
- Aspose_LicenseAgreement.rtf – Contrat de licence utilisateur final
- Aspose.Cells pour SharePoint.pdf – Documentation utilisateur
- Aspose.Cells pour SharePoint Documentation.chm – Documentation utilisateur avec référence publique API
- setup.exe – Programme d'installation
- setup.exe.config - Fichier de configuration d'installation

Le programme d'installation vérifie les conditions suivantes avant de procéder à l'installation :

- WSS 3.0, MOSS 2007 ou SharePoint 2010 est installé.
- L'utilisateur est autorisé à installer des solutions SharePoint.
- La base de données SharePoint est en ligne.
- Le service d'administration WSS est démarré.
- Le service WSS Timer est démarré.

 Le service d'administration WSS et le service de minuterie sont nécessaires car certaines actions de configuration reposent sur un travail de minuterie pour se propager à tous les serveurs de la batterie de serveurs.
#### **Pour installer Aspose.Cells pour SharePoint**
1. Décompressez Aspose.Cells.Zip SharePoint sur le lecteur local du serveur MOSS 7.0 ou WSS 3.0.
1. Exécutez setup.exe et suivez les instructions à l'écran.

Le programme d'installation effectue les actions suivantes :

1.  Vérifiez les conditions préalables à l'installation. L'installation ne se poursuivra pas si une vérification échoue.

   **Verification du système** 

![tâche : image_autre_texte](installing-aspose-cells-for-sharepoint_1.png)




1.  Afficher le contrat de licence utilisateur final. L'utilisateur doit accepter l'accord pour continuer.

   **Le CLUF** 

![tâche : image_autre_texte](installing-aspose-cells-for-sharepoint_2.png)




1. Afficher la boîte de dialogue de sélection de cible de déploiement. L'utilisateur sélectionne les applications Web et les collections de sites où la fonctionnalité doit être activée. Voir la figure ci-dessous.

   **Cibles de déploiement** 

![tâche : image_autre_texte](installing-aspose-cells-for-sharepoint_3.png)




1.  Déployez la fonctionnalité sur la batterie de serveurs.

   **Installation en cours** 

![tâche : image_autre_texte](installing-aspose-cells-for-sharepoint_4.png)




1. Activez la fonctionnalité pour les collections de sites sélectionnées et configurez leurs applications Web parentes.
1.  Affichez une liste des applications Web et des collections de sites où la fonctionnalité a été déployée et activée.

   **Installation complète** 

![tâche : image_autre_texte](installing-aspose-cells-for-sharepoint_5.png)
