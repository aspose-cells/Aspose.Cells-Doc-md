{{% alert color="primary" %}} 

Pour désinstaller Aspose.Cells pour SharePoint, exécutez le programme d'installation. Si Aspose.Cells pour SharePoint est déjà installé, le programme d'installation suggère de le supprimer ou de le réparer.

{{% /alert %}} 
### **Désinstallation**
 Lors de la suppression, le programme d'installation désactive Aspose.Cells pour SharePoint pour toutes les collections de sites et retire la solution de la batterie de serveurs.

**Désinstallation Aspose.Cells** 

![tâche : image_autre_texte](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

 Veuillez noter que l'exécution du**Réparation** l'option ne réactive pas Aspose.Cells pour SharePoint sur les collections de sites. Au lieu de cela, la fonction doit être activée manuellement après le redéploiement avec le**Réparation** option.

 Désinstallation à l'aide de**Retirer** puis la réinstallation peut être plus pratique car elle vous permet de sélectionner les applications Web et les collections de sites cibles lors de l'installation.

{{% /alert %}}
