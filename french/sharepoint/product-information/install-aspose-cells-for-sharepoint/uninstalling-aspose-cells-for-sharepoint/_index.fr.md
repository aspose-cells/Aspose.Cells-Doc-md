{{% alert color="primary" %}} 

Pour désinstaller Aspose.Cells for SharePoint, exécutez le programme d'installation. Si Aspose.Cells for SharePoint est déjà installé, le programme d'installation suggère de le supprimer ou de le réparer.

{{% /alert %}} 
### **Désinstallation**
Pendant la suppression, le programme de désinstallation désactive Aspose.Cells for SharePoint pour toutes les collections de sites et retire la solution de la ferme de serveurs. 

**Désinstallation Aspose.Cells** 

![todo:image_alt_text](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

Veuillez noter que l'exécution de l'option **Réparer** ne ré-active pas Aspose.Cells for SharePoint sur les collections de sites. Au lieu de cela, la fonction doit être activée manuellement après le retrait avec l'option **Réparer**.

Désinstaller en utilisant l'option **Supprimer** puis réinstaller peut être plus pratique car cela vous permet de sélectionner les applications Web cibles et les collections de sites lors de l'installation. 

{{% /alert %}}
