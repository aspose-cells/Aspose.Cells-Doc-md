{{% alert color="primary" %}} 

[Pendant l'installation](/cells/fr/sharepoint/installing-aspose-cells-for-sharepoint/) , Aspose.Cells for SharePoint est[activé](/cells/fr/sharepoint/deployment-and-activation/) pour toutes les collections de sites sélectionnées. Après l'installation, utilisez le**Actions sur le site** menu à la racine du site Web d'une collection de sites pour activer et désactiver Aspose.Cells for SharePoint.
