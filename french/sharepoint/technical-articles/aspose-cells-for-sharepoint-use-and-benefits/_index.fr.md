Les sites SharePoint sont destinés à la collaboration, à la communication et au stockage de contenu afin de répondre à des besoins commerciaux particuliers. Aspose.Cells pour SharePoint a été introduit pour effectuer certaines opérations sur les documents Excel Microsoft disponibles dans les bibliothèques de documents SharePoint. ### **Une brève note sur les bibliothèques de documents SharePoint**
Le contenu des sites SharePoint se présente sous la forme de documents partagés dans un emplacement centralisé appelé bibliothèque de documents SharePoint (parfois appelée « bibliothèque de documents partagée »). Les utilisateurs autorisés peuvent télécharger des documents depuis leur ordinateur local vers le site SharePoint. Ces documents sont stockés au format binaire dans une base de données SQL Server utilisée par le site SharePoint. Les documents deviennent disponibles pour les autres utilisateurs de SharePoint sous la forme d'une liste d'hyperliens comme indiqué ci-dessous :

**Une bibliothèque de documents** 


### **Avantages**
La liste suivante met en évidence quelques avantages de l'utilisation de Aspose.Cells pour SharePoint :

- Le serveur de site SharePoint n'a pas de capacité intégrée pour convertir les fichiers Excel Microsoft directement sur le serveur. Aspose.Cells pour SharePoint ajoute cette fonctionnalité au serveur de site SharePoint.
- Aspose.Cells pour SharePoint ne nécessite pas Microsoft Excel pour fonctionner. Il est installé sur le serveur afin que les utilisateurs de SharePoint n'aient pas besoin d'Excel Microsoft sur leurs systèmes locaux pour convertir des documents Excel Microsoft d'un format à un autre.
- Aspose.Cells pour SharePoint ne nécessite aucun pré-requis hormis ceux nécessaires à la création d'un site SharePoint.
-  Aspose.Cells pour SharePoint est[facile à déployer sur un serveur de site SharePoint](/cells/fr/sharepoint/installing-aspose-cells-for-sharepoint/).
-  Aspose.Cells pour les supports SharePoint[plusieurs formats d'entrée et de sortie](/cells/fr/sharepoint/multiple-format-support/).
- Aspose.Cells pour SharePoint expose un public API pour les développeurs qui ont besoin de développer leurs propres solutions SharePoint.
### **Conversion de documents Excel Microsoft avec Aspose.Cells pour SharePoint**
 Une fois Aspose.Cells pour SharePoint installé sur le serveur SharePoint, l'option**Convertir avec Aspose.Cells.SharePoint** est disponible dans le menu contextuel :

**Trouver l'option Convertir avec...** 

Pour convertir un document Excel Microsoft de votre bibliothèque de documents SharePoint dans d'autres formats, suivez les étapes ci-dessous :

1. Cliquez sur le document Excel Microsoft souhaité, puis cliquez sur Convertir avec Aspose.Cells pour SharePoint
 L'écran Paramètres de conversion s'affiche.

   **Décider de se convertir** 


1. Sélectionnez le format de sortie.
 Ici, vous pouvez également modifier le nom du fichier de sortie et le dossier de destination.
1.  Cliquez sur**Convertir**. 

   **Sélection du format de sortie** 



1.  Lorsque la conversion est terminée, un message de réussite s'affiche.

   **Conversion réussie** 



1.  Cliquez sur**Revenir** pour revenir à la bibliothèque de documents.
1.  Le document converti est répertorié dans la bibliothèque de documents SharePoint.

   **Le fichier converti est répertorié dans la bibliothèque de documents** 


### **Sommaire**
{{% alert color="primary" %}} 

L'installation et l'utilisation de Aspose.Cells pour SharePoint sont simples. Il ajoute non seulement une fonctionnalité pour convertir les fichiers Excel Microsoft directement sur le serveur SharePoint, mais élimine également le besoin d'installer Excel Microsoft sur les machines clientes. De plus, les développeurs peuvent utiliser Aspose.Cells pour le public API de SharePoint pour développer leurs propres solutions SharePoint.

 En d'autres termes, il est avantageux pour les administrateurs de sites SharePoint, les développeurs et les utilisateurs finaux.
