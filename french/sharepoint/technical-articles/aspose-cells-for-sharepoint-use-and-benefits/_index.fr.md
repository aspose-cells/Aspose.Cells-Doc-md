Les sites SharePoint sont destinés à la collaboration, à la communication et au stockage de contenu afin de répondre à des besoins commerciaux spécifiques. Aspose.Cells for SharePoint a été introduit pour effectuer certaines opérations sur des documents Microsoft Excel disponibles dans les bibliothèques de documents SharePoint. ### **Une brève note sur les bibliothèques de documents SharePoint**
Les contenus sur les sites SharePoint se présentent sous forme de documents partagés au niveau d'un emplacement centralisé appelé une bibliothèque de documents SharePoint (parfois appelée "bibliothèque de documents partagés"). Les utilisateurs autorisés peuvent télécharger des documents de leur machine locale vers le site SharePoint. Ces documents sont stockés sous forme binaire dans une base de données SQL Server utilisée par le site SharePoint. Les documents deviennent accessibles aux autres utilisateurs SharePoint sous forme d'une liste d'hyperliens, comme indiqué ci-dessous :

**Une bibliothèque de documents** 


### **Avantages**
La liste suivante met en évidence quelques avantages d'utiliser Aspose.Cells for SharePoint :

- Le serveur du site SharePoint n'a pas la capacité intégrée de convertir directement des fichiers Microsoft Excel sur le serveur. Aspose.Cells for SharePoint ajoute cette fonctionnalité au serveur du site SharePoint.
- Aspose.Cells for SharePoint ne nécessite pas Microsoft Excel pour fonctionner. Il est installé sur le serveur, de sorte que les utilisateurs SharePoint n'ont pas besoin de Microsoft Excel sur leurs systèmes locaux pour convertir des documents Microsoft Excel d'un format à un autre.
- Aspose.Cells for SharePoint ne nécessite aucune condition préalable sauf celles requises pour la création d'un site SharePoint.
- Aspose.Cells for SharePoint est [facile à déployer sur un serveur de site SharePoint](/cells/fr/sharepoint/installing-aspose-cells-for-sharepoint/).
- Aspose.Cells for SharePoint prend en charge [de nombreux formats d'entrée et de sortie](/cells/fr/sharepoint/multiple-format-support/).
- Aspose.Cells for SharePoint expose une API publique pour les développeurs qui ont besoin de développer leurs propres solutions SharePoint.
### **Conversion de documents Microsoft Excel avec Aspose.Cells for SharePoint**
Une fois Aspose.Cells for SharePoint est installé sur le serveur SharePoint, l'option **Convertir avec Aspose.Cells.SharePoint** est disponible dans le menu contextuel: 

**Trouver l'option Convertir avec...** 

Pour convertir un document Microsoft Excel de votre bibliothèque de documents SharePoint dans d'autres formats, suivez les étapes ci-dessous:

1. Cliquez sur le document Microsoft Excel souhaité, puis cliquez sur Convertir avec Aspose.Cells for SharePoint
   L'écran des paramètres de conversion s'affiche. 

   **Décider de convertir** 


1. Sélectionnez le format de sortie.
   Ici, vous pouvez également modifier le nom du fichier de sortie et le dossier de destination.
1. Cliquez sur **Convertir**. 

   **Sélection du format de sortie** 



1. Lorsque la conversion est terminée, un message de succès s'affiche. 

   **Conversion réussie** 



1. Cliquez sur **Retour** pour revenir à la bibliothèque de documents.
1. Le document converti est répertorié dans la bibliothèque de documents SharePoint. 

   **Le fichier converti est répertorié dans la bibliothèque de documents** 


### **Résumé**
{{% alert color="primary" %}} 

L'installation et l'utilisation de Aspose.Cells for SharePoint est simple. Non seulement elle ajoute une fonctionnalité pour convertir les fichiers Microsoft Excel directement sur le serveur SharePoint, mais elle élimine également le besoin d'installer Microsoft Excel sur les machines clientes. De plus, les développeurs peuvent utiliser l'API publique de Aspose.Cells for SharePoint pour développer leurs propres solutions SharePoint.

En d'autres termes, c'est avantageux pour les administrateurs de site SharePoint, les développeurs et les utilisateurs finaux.
