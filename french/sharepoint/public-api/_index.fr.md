{{% alert color="primary" %}} 

Les clients développant leurs propres solutions SharePoint peuvent accéder à notre moteur de conversion en utilisant l'API publique. Cela comprend tous les formats de fichier pris en charge et les options de conversion. Les spécifications d'interface détaillées dans notre [Référence de l'API](https://reference.aspose.com/) sont accompagnées de différents exemples en C# et VB.

{{% /alert %}}
