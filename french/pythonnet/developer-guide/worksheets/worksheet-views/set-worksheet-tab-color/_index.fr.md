Aspose.Cells pour Python via .NET vous permet de changer la couleur des onglets de chaque feuille de calcul pour les rendre plus visibles. Par exemple, vous pouvez faire en sorte que Dépenses soit rouge, Ventes vert, Actifs bleu, etc.

{{% /alert %}} 

## **Comment définir la couleur de l'onglet de la feuille de calcul avec Microsoft Excel**
1. Cliquez avec le bouton droit sur un onglet dans la feuille d'onglets en bas de la feuille de calcul actuelle.
1. Sélectionnez **Couleur d'onglet**.
1. Sélectionnez une couleur dans la palette.
1. Cliquez sur **OK**.
