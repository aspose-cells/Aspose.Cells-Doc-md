## **Scénarios d'utilisation possibles**

Il n'existe pas d'option directe pour créer des tailles de papier personnalisées dans MS Excel, cependant, vous pouvez définir une taille de papier personnalisée pour vos feuilles de calcul souhaitées lors du rendu d'un fichier Excel au format PDF. Ce document explique comment définir une taille de papier personnalisée d'une feuille de calcul en utilisant les API Aspose.Cells pour Python via .NET.

## **Implémenter une taille de papier personnalisée de la feuille de calcul pour le rendu**

Aspose.Cells pour Python via .NET vous permet de mettre en œuvre la taille de papier souhaitée de la feuille de calcul. Vous pouvez utiliser la méthode [**custom_paper_size**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup/custom_paper_size/#float-float) de la classe [**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup) pour spécifier une taille de page personnalisée. Le code d'exemple suivant illustre comment spécifier une taille de papier personnalisée pour la première feuille de calcul du classeur. Veuillez également consulter le [PDF de sortie](45056028.pdf) généré avec le code suivant pour une référence.
