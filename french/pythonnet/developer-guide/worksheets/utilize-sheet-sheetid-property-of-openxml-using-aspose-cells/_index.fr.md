## **Scénarios d'utilisation possibles**

*Sheet.SheetId* est une propriété située dans l'espace de noms *DocumentFormat.OpenXml.Spreadsheet* et fait partie d'OpenXml. Vous pouvez voir cette propriété et sa valeur dans *workbook.xml* comme illustré dans la capture d'écran suivante. Aspose.Cells pour Python via .NET fournit la propriété équivalente sous forme de [**Worksheet.tab_id**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/tab_id).
