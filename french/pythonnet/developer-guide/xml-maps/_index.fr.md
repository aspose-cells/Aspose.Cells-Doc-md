{{% alert color="primary" %}}

Aspose.Cells pour Python via .NET vous permet d’importer la carte XML à l’intérieur du classeur en utilisant la méthode [**Workbook.import_xml()**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/import_xml/#str-str-int-int). Vous pouvez importer la carte XML en utilisant Microsoft Excel en suivant les étapes suivantes

- Sélectionnez l'onglet **Développeur**
- Cliquez sur **Importer** dans la section XML et suivez les étapes requises.

Vous devrez fournir vos données XML pour compléter l'importation. Voici un [exemple de données XML](5115037.txt) que vous pouvez utiliser pour les tests.
