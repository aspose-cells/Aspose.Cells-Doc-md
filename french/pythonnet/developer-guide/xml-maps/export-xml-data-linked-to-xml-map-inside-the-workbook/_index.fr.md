## **Exporter des données XML liées à la carte XML à l'intérieur du classeur**

Veuillez utiliser la méthode [**Workbook.export_xml()**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/export_xml/#str-str) pour exporter les données XML liées à vos cartes XML à l'intérieur de votre classeur. Le code d'exemple suivant exporte les données XML de toutes les cartes XML du classeur une par une. Veuillez vérifier le [fichier Excel d'exemple](5115497.xlsx) utilisé dans ce code et le [exported XML data of the first XML Map](5472487.xml).
