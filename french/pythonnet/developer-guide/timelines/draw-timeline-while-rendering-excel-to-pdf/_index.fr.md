## **Comment dessiner la chronologie lors du rendu d'Excel en PDF à l'aide de la bibliothèque Excel Aspose.Cells pour Python**
Si vous avez un fichier Excel auquel une chronologie a été appliquée et que vous souhaitez exporter l'Excel en PDF avec les paramètres de chronologie, Aspose.Cells pour Python via .NET le prend désormais en charge par défaut. Il vous suffit d'exporter le fichier Excel avec une chronologie en PDF, le PDF généré affichera la chronologie appliquée.

Le code d'exemple suivant charge le [fichier Excel d'exemple](input.xlsx) qui contient une chronologie existante. Il enregistre ensuite le classeur sous la forme de [fichier PDF de sortie](out.pdf). La capture d'écran suivante compare le fichier Excel source et le fichier PDF généré.
