##  **Dessinez la chronologie lors du rendu d'Excel au PDF**
Si vous disposez d'un fichier Excel auquel une chronologie est appliquée et que vous souhaitez exporter Excel vers PDF avec les paramètres de chronologie, Aspose.Cells for Python via .NET le prend désormais en charge par défaut. Vous exportez simplement le fichier Excel avec la chronologie vers PDF, le PDF généré affichera la chronologie appliquée.

 L'exemple de code suivant charge le[exemple de fichier Excel](input.xlsx) qui contient une chronologie existante. Il enregistre ensuite le classeur sous[sortie du fichier PDF](out.pdf). La capture d'écran suivante compare le fichier Excel source et le fichier PDF généré.
