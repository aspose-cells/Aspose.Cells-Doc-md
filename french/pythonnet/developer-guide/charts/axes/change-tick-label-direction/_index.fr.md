## **Changer la direction des étiquettes des graduations**

Aspose.Cells pour Python via .NET vous permet de changer la direction des étiquettes de tick du graphique en utilisant la propriété [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type). La propriété [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type) accepte la valeur d'énumération [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/). L'énumération [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/) comprend les membres suivants

- HORIZONTAL
- VERTICAL
- ROTATE90
- ROTATE270
- EMPILÉ
