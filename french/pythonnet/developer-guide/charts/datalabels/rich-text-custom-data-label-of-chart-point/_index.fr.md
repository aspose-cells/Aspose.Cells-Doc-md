Vous pouvez utiliser Aspose.Cells pour Python via .NET pour créer une étiquette de donnée personnalisée en texte enrichi pour le point de graphique. Aspose.Cells pour Python via .NET fournit la méthode [**DataLabels.characters()**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextframe/characters) pour retourner l'objet [**FontSetting**](https://reference.aspose.com/cells/python-net/aspose.cells/fontsetting) qui peut être utilisé pour définir les propriétés de police du texte, comme sa couleur, sa graisse, etc.
