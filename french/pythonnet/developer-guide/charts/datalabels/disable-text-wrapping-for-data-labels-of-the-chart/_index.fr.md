{{% alert color="primary" %}}

Microsoft Excel 2013 permet aux utilisateurs de mettre en forme ou de défaire le retour à la ligne à l'intérieur des étiquettes de données du graphique. Par défaut, le texte à l'intérieur des étiquettes de données du graphique est à l'état de retour à la ligne.

Aspose.Cells pour Python via .NET fournit une propriété [**DataLabels.is_text_wrapped**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/datalabels/is_text_wrapped) que vous pouvez définir à True ou False pour activer ou désactiver le saut de ligne des étiquettes de données respectivement.

{{% /alert %}}
