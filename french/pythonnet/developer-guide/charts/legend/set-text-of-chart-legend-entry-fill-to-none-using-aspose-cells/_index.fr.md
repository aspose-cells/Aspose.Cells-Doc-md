{{% alert color="primary" %}}

Si vous souhaitez définir le texte du remplissage de l'entrée de légende du graphique afin qu'il ne s'affiche pas dans la légende du graphique, veuillez définir [**LegendEntry.is_text_no_fill**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/legendentry/is_text_no_fill) sur **true**.
