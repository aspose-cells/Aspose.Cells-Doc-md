## **Scénarios d'utilisation possibles**
Aspose.Cells pour Python via .NET prend désormais en charge la lecture et la manipulation de graphiques Microsoft Excel 2016 qui ne sont pas présents dans Microsoft Excel 2013 ou les versions antérieures.
