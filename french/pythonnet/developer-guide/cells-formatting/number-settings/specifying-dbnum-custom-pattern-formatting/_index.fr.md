## **Scénarios d'utilisation possibles**

Aspose.Cells pour Python via .NET supporte le formatage personnalisé *DBNum*. Par exemple, si la valeur de votre cellule est 123 et que vous spécifiez son format personnalisé comme [DBNum2][$-804]General, elle sera affichée comme 壹佰贰拾叁. Vous pouvez spécifier votre format personnalisé de la cellule en utilisant la méthode [**Cell.get_style()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_style/#) et la propriété [**Style.custom**](https://reference.aspose.com/cells/python-net/aspose.cells/style/custom).
