Les API Aspose.Cells pour Python via .NET permettent d’ajouter des filigranes WordArt sur la feuille de calcul de manière à ce que le WordArt devienne un objet que l’on peut déplacer et positionner sur la feuille. Il est également possible de verrouiller l’objet WordArt pour toute interaction comme la modification, le déplacement et la sélection. Cet article explique l’utilisation de la méthode [**Shape.set_locked_property**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/set_locked_property) pour verrouiller certains aspects du filigrane.
