## **Scénarios d'utilisation possibles**
Aspose.Cells pour Python via .NET fournit la propriété [Shape.three_d_format](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/three_d_format/) ainsi que la classe [ThreeDFormat](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/threedformat) pour manipuler le format 3D d'une forme ou d'un graphique. La classe [ThreeDFormat](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/threedformat) contient différentes propriétés pouvant être configurées selon les résultats souhaités.
