## **Scénarios d'utilisation possibles**
Aspose.Cells pour Python via .NET fournit la propriété [Shape.glow](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/glow) ainsi que la classe [GlowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/gloweffect) pour travailler avec l’effet de halo d’une forme ou d’un graphique. La classe [GlowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/gloweffect) contient les propriétés suivantes qui peuvent être configurées pour obtenir différents résultats selon les besoins de l’application.

- [GlowEffect.taille](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/gloweffect/size)
- [GlowEffect.transparence](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/gloweffect/transparency)
- [GlowEffect.couleur](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/gloweffect/color)
