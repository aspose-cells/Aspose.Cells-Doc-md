## **Scénarios d'utilisation possibles**
Aspose.Cells pour Python via .NET fournit la propriété [Shape.reflection](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/reflection) ainsi que la classe [ReflectionEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect) pour travailler avec l’effet de réflexion d’une forme ou d’un graphique. La classe [ReflectionEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect) contient les propriétés suivantes qui peuvent être configurées pour obtenir différents résultats selon les besoins de l’application.

- [flou](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/blur)
- [direction](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/direction)
- [distance](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/distance)
- [direction_degradation](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/fade_direction)
- [tot_avec_forme](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/rot_with_shape)
- [taille](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/size)
- [transparence](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/transparency)
- [type](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/reflectioneffect/type)
