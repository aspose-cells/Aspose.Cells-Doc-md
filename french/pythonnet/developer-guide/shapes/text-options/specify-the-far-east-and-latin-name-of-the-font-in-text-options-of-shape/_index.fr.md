## **Scénarios d'utilisation possibles**

Parfois, vous souhaitez afficher du texte en police de langue d'Extrême-Orient, par exemple japonais, chinois, thaïlandais, etc. Aspose.Cells pour Python via .NET fournit la propriété [**TextOptions.far_east_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/far_east_name) que vous pouvez utiliser pour spécifier le nom de police de cette langue. De plus, vous pouvez également spécifier le nom de la police latine en utilisant la propriété [**TextOptions.latin_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/latin_name).

## **Spécifier le nom Extrême-Orient et Latin de la police dans les options de texte de la Forme**

Le code d'échantillon suivant crée une zone de texte et ajoute du texte en japonais à l'intérieur. Il spécifie ensuite les noms des polices latine et Extrême-Orient du texte et enregistre le classeur sous la forme de [fichier Excel de sortie](67338274.xlsx).
