{{% alert color="primary" %}}

Aspose.Cells pour Python via .NET fournit la propriété [**Shape.geometry.shape_adjust_values**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/geometry/shape_adjust_values/) pour apporter des modifications aux points d’ajustement avec les formes. Dans l’interface utilisateur de Microsoft Excel, les ajustements s’affichent sous forme de diamants jaunes. Par exemple :

- Le rectangle arrondi possède un ajustement pour changer l'arc.
- Le triangle a un ajustement pour changer l'emplacement du point.
- Le trapèze a un ajustement pour changer la largeur du haut
- Les flèches ont deux ajustements pour changer la forme de la tête et de la queue.

Cet article expliquera l'utilisation de la propriété [**Shape.geometry.shape_adjust_values**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/geometry/shape_adjust_values/) pour changer la valeur d'ajustement des formes différentes.
