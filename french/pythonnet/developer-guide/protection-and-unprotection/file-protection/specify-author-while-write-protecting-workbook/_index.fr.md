## **Scénarios d'utilisation possibles**

Vous pouvez spécifier le nom de l'auteur lors de la protection par écriture de votre classeur en utilisant l'API Aspose.Cells pour Python via .NET. Veuillez utiliser la propriété [**Workbook.settings.write_protection.author**](https://reference.aspose.com/cells/python-net/aspose.cells/writeprotection/author/) à cette fin.
