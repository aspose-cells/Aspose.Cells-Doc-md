## **Introduction**

Microsoft Excel permet d'ajouter une **ligne de signature** dans les classeurs Excel. Vous pouvez ajouter une ligne de signature en cliquant sur l'onglet **Insertion** puis en sélectionnant **Ligne de signature** dans le groupe **Texte**.

## **Comment créer une ligne de signature pour Excel**

Aspose.Cells pour Python via .NET offre également cette fonctionnalité et a exposé la propriété [**Picture.signature_line**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/picture/signature_line) à cette fin. Cet article expliquera comment utiliser cette propriété pour ajouter une ligne de signature avec Aspose.Cells pour Python via .NET.
