##  **Obtenir la source de données de connexion externe du tableau croisé dynamique**

Aspose.Cells for Python via .NET offre la possibilité d'obtenir la source de données de connexion externe du tableau croisé dynamique. Pour cela, le API met à disposition le[**source_de_données_de_connexion_externe**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) propriété du[**Tableau croisé dynamique**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/) classe. Le[**source_de_données_de_connexion_externe**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/)la propriété renvoie le[**ConnexionExterne**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/) objet. L'extrait de code suivant illustre l'utilisation de[**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) propriété pour obtenir la source de données de connexion externe du tableau croisé dynamique.
