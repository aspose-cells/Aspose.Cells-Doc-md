## **Comment enregistrer un tableau croisé dynamique dans un fichier ODS**

Aspose.Cells for Python via .NET offre la possibilité d'enregistrer un tableau croisé dynamique dans des fichiers ODS. Pour cela, vous convertissez simplement un classeur avec un tableau croisé dynamique existant ou créez un nouveau tableau croisé dynamique et enregistrez le fichier au format ODS. Assurez-vous d'appeler [**PivotTable.calculate_data**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/calculate_data/#) avant l'enregistrement pour garantir que le tableau croisé dynamique est rendu dans le fichier ODS de sortie.
