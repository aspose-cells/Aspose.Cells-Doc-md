Aspose.Cells pour Python via .NET fournit une fonctionnalité pour supprimer ou récupérer un tableau croisé dynamique d'une feuille de calcul. Vous pouvez supprimer le tableau croisé dynamique en utilisant un objet tableau croisé dynamique ou une position de tableau croisé dynamique. Veuillez utiliser la méthode [**Worksheet.pivot_tables.remove(pivot_table)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottablecollection/) pour supprimer le tableau croisé dynamique en utilisant l'objet tableau croisé dynamique et la méthode [**Worksheet.pivot_tables.remove_at(index, keep_data)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottablecollection/remove_at/#int-bool) pour supprimer l'objet tableau croisé dynamique en utilisant sa position dans la collection de tableaux croisés dynamiques.
