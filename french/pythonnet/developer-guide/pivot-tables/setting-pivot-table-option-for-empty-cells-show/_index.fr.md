{{% alert color="primary" %}}

Vous pouvez définir différentes options de tableau croisé dynamique en utilisant Aspose.Cells for Python via .NET. L'une de ces options est "Pour afficher les cellules vides". En définissant cette option, toutes les cellules vides d'un tableau croisé dynamique sont affichées sous la forme d'une chaîne spécifiée.

{{% /alert %}}

##  **Définition de l'option de tableau croisé dynamique dans Microsoft Excel**

Pour rechercher et définir cette option dans Excel Microsoft :

1. Sélectionnez un tableau croisé dynamique et cliquez avec le bouton droit.
1. Sélectionnez *Options du tableau croisé dynamique**.
1.  Sélectionnez le**Mise en page et format** languette.
1.  Sélectionnez le**Pour les cellules vides, afficher** option et spécifiez une chaîne.
