##  **Scénarios d'utilisation possibles**
Aspose.Cells for Python via .NET prend en charge le rendu de la forme du slicer. Si vous convertissez votre feuille de calcul en image ou si vous enregistrez votre classeur aux formats PDF ou HTML, vous verrez que les slicers sont rendus correctement.

##  **Trancheur de rendu**
L'exemple de code suivant charge le[exemple de fichier Excel](67338479.xlsx) qui contient un slicer existant. Il convertit la feuille de calcul en image en définissant la zone d'impression qui couvre uniquement le slicer. L'image fluide est la[image de sortie](67338480.png) qui montre le slicer rendu. Comme vous pouvez le voir, le slicer a été rendu correctement et il ressemble à l'exemple de fichier Excel.
