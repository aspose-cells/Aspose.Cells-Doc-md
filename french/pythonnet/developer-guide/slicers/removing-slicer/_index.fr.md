##  **Scénarios d'utilisation possibles**

 Si vous souhaitez supprimer le slicer dans Microsoft Excel, sélectionnez-le simplement et appuyez sur le bouton*Supprimer* bouton. De même, si vous souhaitez le supprimer en utilisant Aspose.Cells for Python via .NET API par programme, veuillez utiliser le[**Feuille de calcul.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/)méthode. Cela supprimera le slicer de la feuille de calcul.
