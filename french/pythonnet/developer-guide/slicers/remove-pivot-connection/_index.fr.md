##  **Scénarios d'utilisation possibles**

 Si vous souhaitez dissocier le slicer et le tableau croisé dynamique dans Excel, vous devez cliquer avec le bouton droit sur le slicer et sélectionner l'élément "Connexions du rapport...". Dans la liste des options, vous pouvez opérer sur la case à cocher. De même, si vous souhaitez dissocier le slicer et le tableau croisé dynamique à l'aide de Aspose.Cells for Python via .NET API par programme, veuillez utiliser le[**Slicer.remove_pivot_connection(pivot)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/remove_pivot_connection/#aspose.cells.pivot.PivotTable)méthode. Il dissociera le slicer et le tableau croisé dynamique.

##  **Dissocier le slicer et le tableau croisé dynamique**

L'exemple de code suivant charge le[exemple de fichier Excel](remove-pivot-connection.xlsx)qui contient un slicer existant. Il accède aux slicers puis dissocie le slicer et le tableau croisé dynamique. Enfin, il enregistre le classeur sous[sortie du fichier Excel](remove-pivot-connection-out.xlsx).
