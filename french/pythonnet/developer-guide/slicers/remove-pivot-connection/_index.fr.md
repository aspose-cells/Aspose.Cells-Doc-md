## **Scénarios d'utilisation possibles**

Si vous souhaitez dissocier la trancheuse et le tableau croisé dynamique dans Excel, vous devez faire un clic droit sur la trancheuse et sélectionner l'élément "Connexions de rapport...". Dans la liste des options, vous pouvez agir sur la case à cocher. De même, si vous souhaitez dissocier la trancheuse et le tableau croisé dynamique en utilisant l'API Aspose.Cells pour Python via .NET de manière programmée, veuillez utiliser la méthode [**Slicer.remove_pivot_connection(pivot)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/remove_pivot_connection/#aspose.cells.pivot.PivotTable). Cela dissociera la trancheuse et le tableau croisé dynamique.
