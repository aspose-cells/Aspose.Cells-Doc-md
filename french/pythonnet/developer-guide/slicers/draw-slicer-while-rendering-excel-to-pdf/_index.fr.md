## **Comment dessiner une trancheuse lors du rendu d'Excel en PDF en utilisant la bibliothèque Aspose.Cells pour Python Excel**
Si vous avez un fichier Excel avec une trancheuse appliquée et que vous souhaitez exporter l'Excel en PDF avec les paramètres de la trancheuse, Aspose.Cells pour Python via .NET prend maintenant en charge cela par défaut. Vous exportez simplement le fichier Excel avec trancheuse en PDF, le PDF généré montrera la trancheuse appliquée.

Le code d'exemple suivant charge le [fichier Excel d'exemple](94044165.xlsx) contenant une trancheuse existante. Ensuite, il enregistre le classeur de travail en tant que [fichier PDF de sortie](94044166.pdf). La capture d'écran suivante compare le fichier Excel source et le fichier PDF généré.
