##  **Dessinez Slicer lors du rendu d'Excel au PDF**
Si vous disposez d'un fichier Excel auquel un slicer est appliqué et que vous souhaitez exporter Excel vers PDF avec les paramètres du slicer, Aspose.Cells for Python via .NET le prend désormais en charge par défaut. Vous exportez simplement le fichier Excel avec le slicer vers PDF, le PDF généré affichera le slicer appliqué.

 L'exemple de code suivant charge le[exemple de fichier Excel](94044165.xlsx) qui contient un slicer existant. Il enregistre ensuite le classeur sous[sortie du fichier PDF](94044166.pdf). La capture d'écran suivante compare le fichier Excel source et le fichier PDF généré.
