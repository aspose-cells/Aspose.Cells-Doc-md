{{% alert color="primary" %}}

Microsoft Excel permet aux utilisateurs d’ajouter des commentaires aux cellules pour fournir des informations supplémentaires et mettre en évidence des données. Les développeurs peuvent avoir besoin de personnaliser le commentaire pour spécifier les paramètres d’alignement, la direction du texte, la couleur de la police, etc. Aspose.Cells pour Python via .NET fournit des API pour accomplir cette tâche.

{{% /alert %}}

Aspose.Cells pour Python via .NET fournit une propriété [**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body) pour la couleur de la police du commentaire. Le code d’exemple ci-dessous montre comment utiliser la propriété [**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body) pour définir la direction du texte pour un commentaire.
