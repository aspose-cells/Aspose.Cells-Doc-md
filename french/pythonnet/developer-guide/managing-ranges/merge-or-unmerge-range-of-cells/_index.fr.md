## **Scénarios d'utilisation possibles**

Vous pouvez utiliser Aspose.Cells pour Python via .NET pour fusionner ou diviser une plage de cellules. Aspose.Cells pour Python via .NET propose les méthodes [**Range.merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/merge/#) et [**Range.un_merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/un_merge/#) à cette fin. Cet article explique comment fusionner une plage de cellules en une seule cellule.


## **Comment fusionner ou séparer une plage de cellules**

Le code d'exemple suivant crée d'abord une plage - A1:D4 - puis fusionne les cellules de la plage en une seule cellule en utilisant la méthode [**Range.merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/merge/#). De même, vous pouvez diviser les cellules en créant une plage et en appelant la méthode [**Range.un_merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/un_merge/#).
