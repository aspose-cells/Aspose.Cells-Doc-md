{{% alert color="primary" %}}

Aspose.Cells pour Python via .NET prend en charge la conversion de HTML orienté Microsoft Excel en format XLS/XLSX. Cela signifie que le HTML généré par Microsoft Excel peut être converti en format XLS/XLSX en utilisant Aspose.Cells pour Python via .NET.

De même, s'il y a un HTML simple, Aspose.Cells pour Python via .NET peut le convertir en texte enrichi HTML. Aspose.Cells pour Python via .NET fournit la propriété [**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/) qui peut prendre un HTML simple et le convertir en texte de cellule formaté.

{{% /alert %}}
