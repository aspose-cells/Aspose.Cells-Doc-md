{{% alert color="primary" %}}

Parfois, vous devez trouver des cellules avec un style particulier appliqué. Vous pouvez utiliser Aspose.Cells pour Python via .NET pour trouver toutes les cellules avec un style commun. Aspose.Cells pour Python via .NET fournit la propriété [**FindOptions.style**](https://reference.aspose.com/cells/python-net/aspose.cells/findoptions/style/) que vous pouvez utiliser pour spécifier le style à rechercher dans les cellules.

{{% /alert %}}
