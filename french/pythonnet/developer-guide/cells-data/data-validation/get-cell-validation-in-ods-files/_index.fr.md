## **Obtenir la validation de la cellule dans les fichiers ODS**

Avec Aspose.Cells, vous pouvez obtenir la validation appliquée à une cellule dans les fichiers ODS. Pour cela, l'API fournit la méthode [**get_validation**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) de la classe [**Cell**](https://reference.aspose.com/cells/python-net/aspose.cells/cell).
