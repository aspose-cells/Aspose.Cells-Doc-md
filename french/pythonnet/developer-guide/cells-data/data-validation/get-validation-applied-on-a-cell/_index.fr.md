{{% alert color="primary" %}}

Vous pouvez utiliser Aspose.Cells pour Python via .NET pour obtenir la validation appliquée à une cellule. Aspose.Cells pour Python via .NET fournit la méthode [**Cell.get_validation()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) à cette fin. S'il n'y a pas de validation appliquée à la cellule, elle renvoie null.

De même, vous pouvez utiliser la méthode [**Worksheet.validations.get_validation_in_cell**](https://reference.aspose.com/cells/python-net/aspose.cells/validationcollection/get_validation_in_cell/#int-int) pour acquérir la validation appliquée à une cellule en fournissant ses indices de ligne et de colonne.
