Avec la prévalence croissante des écrans haute résolution, les images générées avec la résolution par défaut de 96 DPI apparaissent souvent floues et peu nettes. Pour garantir la clarté sur les écrans haute résolution, il est essentiel de générer des images à une résolution DPI plus élevée. Aspose.Cells pour Python via .NET offre la possibilité de définir [**ImageOrPrintOptions.horizontal_resolution**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/horizontal_resolution/) et [**ImageOrPrintOptions.vertical_resolution**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/vertical_resolution/), vous permettant de créer des images de haute qualité à partir de fichiers Excel qui apparaissent nettes sur les écrans haute résolution.
