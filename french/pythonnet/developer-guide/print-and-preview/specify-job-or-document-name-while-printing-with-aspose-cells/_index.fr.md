{{% alert color="primary" %}}

Vous pouvez spécifier le nom du travail ou du document lors de l'impression de votre classeur ou feuille de calcul en utilisant les objets WorkbookRender ou SheetRender. Aspose.Cells pour Python via .NET fournit les méthodes WorkbookRender.ToPrinter(printerName, jobName) et SheetRender.ToPrinter(printerName, jobName) pour cela.

{{% /alert %}}
