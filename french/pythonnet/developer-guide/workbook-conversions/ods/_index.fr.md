##  **À propos d'OpenDocument**
 Le[Format OpenDocument (ODF)](https://en.wikipedia.org/wiki/OpenDocument) est un format de fichier gratuit et ouvert pour les documents bureautiques électroniques développé à l'origine par Sun pour la suite Open Office. OpenDocument Spreadsheet (ODS) est le format de fichier pour les documents Excel. OpenDocument est actuellement une norme OASIS et ISO.
