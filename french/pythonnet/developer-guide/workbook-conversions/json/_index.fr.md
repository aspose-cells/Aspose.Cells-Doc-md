## **Convertir un classeur Excel en JSON**

L'API Aspose.Cells pour Python via .NET prend en charge la conversion de feuilles de calcul au format JSON. Pour exporter le classeur au format JSON, passez [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) en tant que deuxième paramètre de la méthode [**Workbook.save**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions). Vous pouvez également utiliser la classe [**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) pour spécifier des paramètres supplémentaires pour exporter la feuille de calcul au format JSON.

L'exemple de code suivant démontre l'exportation de la feuille de calcul active en Json en utilisant le membre d'énumération [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat). Veuillez consulter le code pour convertir le [fichier source](book1.xlsx) en le [fichier Json de sortie](book1.Json) généré par le code pour référence.
