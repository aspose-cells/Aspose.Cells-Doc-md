## **Convertir JSON en classeur Excel**
Pas besoin de se demander comment convertir du JSON en fichier Excel, car la bibliothèque Aspose.Cells pour Python via .NET offre la meilleure solution. L'API Aspose.Cells pour Python via .NET prend en charge la conversion du format JSON en feuilles de calcul. Vous pouvez utiliser la classe [**JsonLoadOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonLoadOptions) pour spécifier des paramètres supplémentaires pour l'importation de JSON dans Workbook.
