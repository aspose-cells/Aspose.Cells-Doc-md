## **Convertir Excel en HTML avec des en-têtes**

Aspose.Cells pour Python via NET offre la possibilité d'exporter des en-têtes lors de la conversion d'Excel en HTML. Pour cela, vous pouvez définir la propriété [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) sur **true**. La valeur par défaut de la propriété [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) est **false**.
