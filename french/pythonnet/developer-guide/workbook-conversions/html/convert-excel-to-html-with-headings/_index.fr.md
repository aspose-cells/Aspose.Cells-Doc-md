##  **Convertir Excel en HTML avec titres**

Aspose.Cells for Python via NET offre la possibilité d'exporter les titres lors de la conversion d'Excel en HTML. Pour cela, vous pouvez définir le[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)propriété à *true**. La valeur par défaut de[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)la propriété est *fausse**.
