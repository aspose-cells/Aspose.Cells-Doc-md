## **Removing Slicer**
To remove slicer in Microsoft Excel, you simply select the slicer and press the *Delete* button. To achieve the save using Aspose.Cells for Python via Java, use the Worksheet.getSlicers().remove() method. It will remove the slicer from the worksheet. 

The following code snippet loads the [sample Excel file](attachments/106201888/106364970.xlsx) that contains an existing slicer. It accesses the slicer, removes it, and saves the [output Excel file](attachments/106201888/106364971.xlsx). The following screenshot shows the slicer that will be removed.
