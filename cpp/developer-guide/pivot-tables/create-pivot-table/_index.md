## **Introduction**
Please use [IPivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table_collection#a903a0eb3d7ef995c370f4a4385b25111) method to create a pivot table inside the worksheet. Once, the pivot table is created, you can work with it using the [IPivotTable](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table) class.
