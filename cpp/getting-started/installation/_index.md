## **Installing Aspose.Cells for C++**
### **Using Nuget Package Manager**
1. In Nuget Package Manager, search for Aspose.Cells. 
   1. Here is the [Link to NuGet Package Aspose.Cells.Cpp](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. It will list the API as Aspose.Cells.Cpp.
1. Press the Add button to install the package 

You can also download ZIP package from the Aspose.Cells for C++ download page. The archive also contains the offline API documentation.
### **Using Include and lib Folders**
1. Download/Install the latest files.
1. Unzip the folder to the production environment.
1. To use Aspose.Cells for C++, reference Include and lib folders in your project.
