## **متطلبات النظام**
Aspose.Cells لبيثون via Java هو واجهة برمجة تطبيقات مستقلة عن النظام ويمكن استخدامها على أي منصة (ويندوز، لينكس وماك) حيث يتم تثبيت [بيثون](https://www.python.org/downloads/). يجب أن يحتوي الجهاز على جافا 8 أو إصدارات أعلى قبل إعداد التثبيت.

## **إصدار Python**
- Python 3.5 أو أحدث
## **نسخة Java**
- Java 1.8 أو أحدث

## **التثبيت**
### **Windows:**
- **قم بتثبيت Java وتعيين متغيرات البيئة JAVA_HOME وPATH**
على سبيل المثال:
{{< highlight java >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}

- **قم بتثبيت Aspose.Cells for Python via Java من pypi**
يمكنك استخدام بسهولة Aspose.Cells for Python via Java من [pypi](https://pypi.org/project/aspose-cells/) بالأمر التالي.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}

### **Linux:**
- **قم بتثبيت Java**

- **قم بتثبيت Aspose.Cells for Python via Java من pypi**
يمكنك استخدام بسهولة Aspose.Cells for Python via Java من [pypi](https://pypi.org/project/aspose-cells/) بالأمر التالي.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}

### **macOS:**
- **قم بتثبيت Java**

- **قم بتثبيت Aspose.Cells for Python via Java من pypi**
يمكنك استخدام بسهولة Aspose.Cells for Python via Java من [pypi](https://pypi.org/project/aspose-cells/) بالأمر التالي.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}
