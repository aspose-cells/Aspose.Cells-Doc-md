## **تصدير التعليقات أثناء حفظ ملف Excel إلى HTML**
عند تحويل Excel إلى HTML، لا تتم تصدير التعليقات. يوفر Aspose.Cells for Python via Java ميزة تصدير التعليقات أثناء تحويل Excel إلى HTML. لتحقيق ذلك، توفر الواجهة برمجة التطبيقات (API) خاصية IsExportComments (https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments). سيؤدي ضبط قيمة خاصية IsExportComments (https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments) إلى True إلى تصدير التعليقات في ملف HTML الناتج.
