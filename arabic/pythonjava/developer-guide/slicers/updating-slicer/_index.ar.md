## **تحديث القطاعة**
Aspose.Cells for Python via Java يدعم تحديث مقسمات طرق العرض. لهذا الغرض ، يوفر API الخاصية Slicer.SlicerCache.SlicerCacheItems المستخدمة لتحديد عناصر مقسم طريقة العرض أو إلغاء تحديدها. يقوم مقتطف الشفرة التالي بتحميل ملف[نموذج لملف Excel](106365050.xlsx)الذي يحتوي على قطاعة. يقوم بإلغاء تحديد العنصرين الثاني والثالث من أداة التقطيع ويقوم بتحديث أداة التقطيع باستخدام طريقة Slicer.refresh (). ثم يقوم بحفظ المصنف كملف[إخراج ملف Excel](106365051.xlsx). تُظهر لقطة الشاشة التالية تأثير نموذج التعليمات البرمجية على نموذج ملف Excel. كما ترى في لقطة الشاشة ، فإن تحديث أداة التقطيع بالعناصر المحددة أدى أيضًا إلى تحديث الجدول المحوري وفقًا لذلك.
