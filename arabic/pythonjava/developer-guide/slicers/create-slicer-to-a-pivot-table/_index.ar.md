## **سيناريوهات الاستخدام المحتملة**
يُستخدم الجُزار لتصفية البيانات بسرعة. يمكن استخدامها لتصفية البيانات في الجدول أو جدول الأحصاء. يسمح لك برنامج Microsoft Excel بإنشاء جُزار عن طريق تحديد جدول أو جدول الأحصاء ثم النقر فوق *إدراج > جزار*. يوفر Aspose.Cells for Python via Java الطريقة Worksheet.getSlicers().add() لإنشاء الجزار.
## **إنشاء مُقطَّع إلى جدول محوري**
الكود البرمجي التالي يحمل [ملف إكسل عيني](106364966.xlsx) الذي يحتوي على جدول الأحصاء. بعد ذلك، ينشئ الجزار بناءً على أول حقل جدول أساسي. وأخيرًا، يحفظ المصنوع في تنسيق [XLSX مخرج](106364967.xlsx). تُظهر الصورة الملتقطة التالية الجزار الذي أنشأته Aspose.Cells في ملف الاكسل المخرج.
