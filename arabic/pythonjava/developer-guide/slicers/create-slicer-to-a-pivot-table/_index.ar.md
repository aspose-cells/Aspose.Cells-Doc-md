## **سيناريوهات الاستخدام الممكنة**
تُستخدم مقسمات الشرائح لتصفية البيانات بسرعة. يمكن استخدامها لتصفية البيانات سواء في الجدول أو الجدول المحوري. Microsoft يسمح لك Excel بإنشاء أداة تقطيع عن طريق تحديد جدول أو جدول محوري ثم النقر فوق الزر*إدراج> قطاعة شرائح*يوفر Aspose.Cells لـ Python عبر Java طريقة Worksheet.getSlicers (). add () لإنشاء أداة تقطيع الشرائح.
## **إنشاء مقسم إلى جدول محوري**
يقوم مقتطف الشفرة التالي بتحميل ملف[نموذج لملف Excel](106364966.xlsx)الذي يحتوي على الجدول المحوري. ثم يقوم بإنشاء أداة التقطيع بناءً على الحقل المحوري الأساسي الأول. أخيرًا ، يحفظ المصنف بتنسيق[إخراج XLSX](106364967.xlsx)صيغة. تُظهر لقطة الشاشة التالية أداة التقطيع التي تم إنشاؤها بواسطة Aspose.Cells في ملف Excel الناتج.
