## **ارسم Slicer أثناء تحويل Excel إلى PDF**
Aspose.Cells for Python via Java يدعم تصدير Excel إلى PDF بإعدادات تقطيع الشرائح. يدعم API هذا افتراضيًا ، ما عليك سوى تصدير ملف Excel باستخدام أداة تقطيع الشرائح إلى PDF ، وسيُظهر ملف PDF الذي تم إنشاؤه أداة التقطيع المطبقة. يقوم مقتطف الشفرة التالي بتحميل ملف[نموذج لملف Excel](106365058.xlsx)الذي يحتوي على مقسم شرائح ويحفظ المصنف كملف[إخراج ملف PDF](106365052.xlsx). تقارن لقطة الشاشة التالية بين ملف Excel المصدر وملف PDF الذي تم إنشاؤه.
