## **رسم المنقي أثناء تحويل Excel إلى PDF**
تدعم Aspose.Cells for Python via Java تصدير Excel إلى PDF مع إعدادات الجزار. تدعم الواجهة البرمجية هذا افتراضيًا، فقط قم بتصدير ملف الاكسل مع جزار إلى PDF، سيُعرض ملف PDF المولد الجزار المطبق. الشيفرة البرمجية التالية تحمل [ملف إكسل عيني](106365058.xlsx) الذي يحتوي على جزار وتحفظ المصنوع كملف [PDF مخرج](106365052.xlsx). الصورة الملتقطة التالية تقارن ملف الاكسل المصدر وملف PDF المولد.
