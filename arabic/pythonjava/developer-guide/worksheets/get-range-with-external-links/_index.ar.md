## **احصل على النطاق مع الروابط الخارجية**
هناك العديد من الحالات التي تصل فيها ملفات Excel إلى البيانات من ملفات Excel الأخرى باستخدام روابط خارجية. يوفر Aspose.Cells لـ Python عبر Java خيار استرداد هذه الروابط الخارجية باستخدام[الاسم](https://reference.aspose.com/cells/python/asposecells.api/name#getReferredAreas\(boolean\)) طريقة. ال[الاسم](https://reference.aspose.com/cells/python/asposecells.api/name#getReferredAreas\(boolean\)) تقوم بإرجاع مصفوفة من النوع[إحالة](https://reference.aspose.com/cells/python/asposecells.api/ReferredArea). ال[إحالة](https://reference.aspose.com/cells/python/asposecells.api/ReferredArea)فئة توفر[ExternalFileName](https://reference.aspose.com/cells/python/asposecells.api/referredarea#ExternalFileName)الخاصية التي تُرجع اسم الملف الخارجي.
