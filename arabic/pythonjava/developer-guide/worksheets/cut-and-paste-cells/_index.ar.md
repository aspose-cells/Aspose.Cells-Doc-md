## **قص ولصق الخلايا**
توفر Aspose.Cells for Python via Java القدرة على قص ولصق الخلايا. لذلك، توفر واجهة البرنامج الثابت [insertCutCells](https://reference.aspose.com/cells/python/asposecells.api/cells#insertCutCells\(com.aspose.cells.Range,%20int,%20int,%20int\)) للمجموعة [Cells](https://reference.aspose.com/cells/python/asposecells.api/Cells). تقبل الطريقة [insertCutCells](https://reference.aspose.com/cells/python/asposecells.api/cells#insertCutCells\(com.aspose.cells.Range,%20int,%20int,%20int\)) البرمجة الثابتة التالية.

- [Range](https://reference.aspose.com/cells/python/asposecells.api/Range): نطاق الخلايا المراد قصه.
- فهرس الصف: فهرس الصف لإدراج الخلايا.
- فهرس العمود: فهرس العمود لإدراج الخلايا.
- [ShiftType](https://reference.aspose.com/cells/python/asposecells.api/ShiftType): اتجاه التحول للأعمدة.
