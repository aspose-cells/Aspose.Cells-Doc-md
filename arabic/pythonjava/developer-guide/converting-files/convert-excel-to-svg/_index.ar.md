## **تحويل Excel إلى SVG**
SVG (**الرسومات المتجهات قابلة لل)**هي مواصفة تستند إلى معايير XML للرسومات المتجهة ثنائية الأبعاد. إنه معيار مفتوح قيد التطوير من قبل World Wide Web Consortium (W3C) منذ 1999.

Aspose.Cells for Python via Java يدعم تحويل ملفات Excel إلى صور SVG. لهذا ، يوفر API امتداد[عرض الورقة](https://reference.aspose.com/cells/python/asposecells.api/SheetRender), [خيارات ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions)، و[عرض المصنف](https://reference.aspose.com/cells/python/asposecells.api/WorkbookRender)الطبقات.
