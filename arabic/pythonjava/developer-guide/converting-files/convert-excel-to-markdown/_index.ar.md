## **تحويل Excel إلى Markdown**
Aspose.Cells لـ Python عبر Java يدعم تحويل ملفات Excel إلى تنسيق Markdown. لتصدير ورقة العمل النشطة إلى Markdown ، قم بتمرير[SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN)كمعامل ثاني لملف[المصنف](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)) طريقة. يمكنك أيضًا استخدام ملف[Markdown حفظ الخيارات](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions)فئة لتحديد إعدادات إضافية لتصدير أوراق العمل إلى Markdown.

يوضح المثال التالي من التعليمات البرمجية تصدير ورقة العمل النشطة إلى Markdown باستخدام[SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN)عضو التعداد. الرجاء مراجعة[إخراج ملف Markdown](Book1.txt)تم إنشاؤها بواسطة رمز كمرجع.
