## **تحويل إكسل إلى تنسيق  Markdown**
تدعم واجهة برمجة التطبيقات Aspose.Cells for Python via Java تحويل ملفات إكسل إلى تنسيق Markdown. لتصدير الورقة العمل النشطة إلى Markdown، قم بتمرير [SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN) كالمعلمة الثانية لأسلوب [Workbook.Save](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)). يمكنك أيضًا استخدام فئة [MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions)‏ لتحديد إعدادات إضافية لتصدير الأوراق العمل إلى Markdown.

يوضح المثال التالي تصدير ورقة العمل النشطة إلى التنسيق الأساسي باستخدام عضو تعداد [SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN). يرجى الاطلاع على [ملف العلامات التوضيحية Markdown](Book1.txt) الذي تم إنشاؤه بواسطة الكود للرجوع إليه.
