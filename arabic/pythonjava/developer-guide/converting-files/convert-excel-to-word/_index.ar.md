## **تحويل مصنف Excel إلى Word**

 لا داعي للتساؤل عن كيفية تحويل Excel Workbook إلى Word ، لأن مكتبة Aspose.Cells for Python via Java لديها أفضل قرار. يوفر Aspose.Cells for Python via Java API دعمًا لتحويل جداول البيانات إلى تنسيق Word. لتصدير المصنف إلى Word ، قم بتمرير[**حفظ تنسيق DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) كمعامل ثاني لـ[**مصنف**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) طريقة. يمكنك أيضًا استخدام ملفات[**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions)فئة لتحديد إعدادات إضافية لتصدير ورقة العمل إلى ملف docx.

 يوضح المثال التالي من التعليمات البرمجية تصدير مصنف Excel إلى Word. يرجى الاطلاع على الكود للتحويل[مصدر الملف](sample.xlsx) إلى ملف Word الذي تم إنشاؤه بواسطة الرمز كمرجع.
