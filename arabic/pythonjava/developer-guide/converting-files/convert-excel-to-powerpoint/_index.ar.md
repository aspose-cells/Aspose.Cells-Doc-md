## **تحويل مصنف Excel إلى PPT**

لا داعي للتساؤل عن كيفية تحويل Excel Workbook إلى PowerPoint ، لأن مكتبة Aspose.Cells for Python via Java لديها القرار الأفضل. يوفر Aspose.Cells for Python via Java API دعمًا لتحويل جداول البيانات إلى تنسيق PowerPoint. لتصدير المصنف إلى PowerPoint ، قم بتمرير[**حفظ تنسيق PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) كمعامل ثاني لـ[**مصنف**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) طريقة. يمكنك أيضًا استخدام ملفات[**خيارات PptxSaveOpt**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) فئة لتحديد إعدادات إضافية لتصدير ورقة العمل إلى ملف .pptx.

 يوضح المثال التالي من التعليمات البرمجية تصدير مصنف Excel إلى PPT. يرجى الاطلاع على الكود للتحويل[مصدر الملف](sample.xlsx) إلى ملف Word الذي تم إنشاؤه بواسطة الرمز كمرجع.
