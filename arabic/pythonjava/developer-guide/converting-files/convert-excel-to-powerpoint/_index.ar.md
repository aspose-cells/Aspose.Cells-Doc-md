Aspose.Cells للـ Python via Java يدعم تحويل ملفات Excel (.xls، xlsx، .xlsb، xlsm) ، CSV و OpenOffice (.ods) إلى ملف PowerPoint.

{{% /alert %}}

## **تحويل دفتر العمل Excel إلى PPT**

لا حاجة للتساءل حول كيفية تحويل سجل البيانات Excel إلى برنامج PowerPoint، لأن Aspose.Cells for Python via Java التي تقدم أفضل قرار. توفر واجهة برمجة التطبيقات Aspose.Cells for Python via Java الدعم لتحويل جداول البيانات إلى تنسيق PowerPoint. لتصدير سجل البيانات إلى PowerPoint، قم بتمرير [**SaveFormat.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) كالمعلمة الثانية في الطريقة [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)). يمكنك أيضًا استخدام الفئة [**PptxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) لتحديد إعدادات إضافية لتصدير ورقة العمل إلى ملف .pptx.

الكود الواجهي التالي يوضح تصدير دفتر العمل Excel إلى PPT. يرجى الرجوع إلى الكود لتحويل [ملف المصدر](sample.xlsx) إلى ملف الكلمة المولد بواسطة الكود للإشارة.
