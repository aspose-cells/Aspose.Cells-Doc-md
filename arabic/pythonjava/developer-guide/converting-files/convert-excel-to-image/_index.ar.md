## **تحويل ملف إكسل إلى صورة**
تدعم واجهة برمجة التطبيقات Aspose.Cells for Python via Java تحويل ملفات إكسل إلى صور مختلفة. لهذا الغرض، توفر واجهة البرمجة الـ[SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender) و [ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions)‏ فئات. تمثل هذه الفئة الورقة التي سيتم عرضها كصورة. توفر فئة [SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender) الـ[toImage()](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage\(int,%20java.lang.String\)) الميثود لتحويل ورقة عمل إلى ملف صورة. يتم دعم تنسيقات BMP، PNG، JPEG، TIFF، و EMF.

القطعة البرمجية التالية توضح تحويل ورقة عمل إكسل إلى صورة PNG.
