## **تحويل Excel إلى صورة**
Aspose.Cells for Python via Java يدعم تحويل ملفات Excel إلى صور ذات أشكال صور مختلفة. لهذا ، يوفر API امتداد[عرض الورقة](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)و[خيارات ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions) الطبقات. تمثل هذه الفئة ورقة العمل التي سيتم عرضها على صورة. ال[عرض الورقة](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)فئة توفر[toImage ()](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage\(int,%20java.lang.String\)) طريقة لتحويل ورقة العمل إلى ملف صورة. يتم دعم التنسيقات BMP و PNG و JPEG و TIFF و EMF.
