لتثبيت Aspose.Cells for JasperReports على نظامك ، يجب أن يفي نظامك بالمتطلبات التالية:

- Java 2 Platform Standard Edition 1.6 أو أعلى.
- JasperReports 6.3.0 أو أعلى.
