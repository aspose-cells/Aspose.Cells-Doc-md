 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells لـ JasperReports 1.0.1](https://downloads.aspose.com/cells/jasperreports/new-releases/aspose.cells-for-jasperreports-1.0.1/)

يسعدنا أن نعلن 06103481 لـ JasperReports 1.0.1!

ما هو الجديد:

- يتم ضبط أحجام الخطوط بشكل مناسب للصق الأحرف بتنسيق Microsoft Excel المماثل للأحرف بتنسيق PDF.
-  يتم تقديم تسميات المحاور (من المخططات) بتنسيق التاريخ بشكل صحيح ووفقًا لذلك.
- يتم دعم تأثيرات التعبئة المتدرجة للمناطق بما في ذلك خلفية الرسم البياني (منطقة الرسم البياني ومنطقة الرسم وما إلى ذلك) وخلفية وسيلة الإيضاح والأشرطة والفقاعات وما إلى ذلك.

ما تم إصلاحه:

- تم إصلاح بعض الأخطاء الخاصة بملء الجدران والأرضيات بالمخططات ثلاثية الأبعاد.
- تعذر تقديم الكائن المضمن: الملف غير موجود.

- تم إصلاح خطأ في تسميات المخطط الدائري.

- تم إصلاح الخلل المتعلق بدعم jasperserver 3.1.0.
