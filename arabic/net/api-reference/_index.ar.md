Aspose.Cells for .NET هي مكتبة برمجة جداول بيانات تتيح لمطوري البرامج معالجة ملفات جداول البيانات وتحويلها من داخل تطبيقاتهم الخاصة. Aspose.Cells for .NET يدعم تنسيقات ملفات جداول البيانات الشائعة (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) ويسمح بتصدير ملفات Excel إلى PDF، XPS، HTML، MHTML، Plain Text، TIFF، JPG، PNG، 07 61113481 و SVG.

 يمكن العثور على أحدث مرجع API على[Aspose.Cells for .NET API المرجع](https://reference.aspose.com/cells/net).
