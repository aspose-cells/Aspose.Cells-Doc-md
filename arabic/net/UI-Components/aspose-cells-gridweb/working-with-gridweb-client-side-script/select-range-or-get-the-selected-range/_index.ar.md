{{% alert color="primary" %}} 

يمكنك استخدام الوظائف الجانبية العميل التالية لتحديد مدى أو للحصول على المدى المحدد في GridWeb باستخدام JavaScript.

- getSelectRange()
- setSelectRange()
- clearSelections()

تعيد getSelectRange() المدى المحدد الأخير. تحدد setSelectRange() المدى المعطى. تقوم clearSelections() بمسح جميع التحديدات باستثناء الخلية النشطة الحالية.
