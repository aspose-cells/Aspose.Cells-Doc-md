{{% alert color="primary" %}} 

Aspose.Cells.GridWeb هو عنصر تحكم ويب قائم على واجهة المستخدم الرسومية ويمكن تضمينه في نماذج الويب ASP.NET مثل عناصر تحكم الويب الأخرى التي يوفرها Microsoft .NET Framework. أهم شيء في عنصر تحكم الويب هو توفير دعم عبر المتصفحات. Aspose.Cells.GridWeb يوفر الدعم عبر المستعرضات.

{{% /alert %}}
