## **Aspose.Cells.GridDesktop - تحكم سطح المكتب للجدول**
### **ميزات الاستيراد/التصدير**
- استيراد المحتوى من جداول البيانات في Excel.
- استيراد البيانات مباشرة من كائن DataTable.
- تصدير المحتوى إلى جداول بيانات Excel.
### **ميزات الورقة العمل**
- إضافة، إدراج أو إزالة أوراق عمل.
- إعادة تسمية أوراق العمل.
- فرز بيانات ورق العمل.
### **ميزات الصف والعمود**
- إضافة، إدراج أو إزالة الصفوف والأعمدة.
- إخفاء أو عرض الصفوف والأعمدة.
- تجميد أو إلغاء تجميد الصفوف والأعمدة.
### **ميزات الخلية**
- يدعم إعدادات تنسيق الأرقام.
- التحكم في توجيه النص.
- إدارة إعدادات الحدود.
- التحكم في إعدادات الخط.
- دمج أو تقسيم الخلايا.
### **ميزات التنسيق**
- التحكم في سلوك الشبكة ومظهرها.
- تطبيق نمط خلفية الخلية المحددة والخط على الخلايا الأخرى باستخدام لوحة التنسيق.
- تلاعب بالبيانات وأنماط الخلايا بسهولة.
### **ميزات الأدوات**
إضافة أو إزالة تعليقات الخلية.
إدراج أو إزالة الصور.
إضافة أو إزالة الروابط الفائقة.
إضافة أو إزالة قواعد التحقق.
إضافة أو إزالة عناصر التحكم (مثل مربع الاختيار، ومربع الاختيار المتعدد الخيارات الخ.).
تصفح الخلية بواسطة لوحة المفاتيح.
### **ميزات الأمان**
حماية خلاياك من خلال قفلها أو إخفائها.
### **ميزات متقدمة**
يدعم إنشاء وحساب الصيغ.
قم بتنفيذ الإجراءات المطلوبة في إطار أحداث الشبكة المختلفة.
### **إصدارات Microsoft Excel المدعومة**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
