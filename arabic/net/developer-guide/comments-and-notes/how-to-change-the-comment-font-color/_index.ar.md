{{% alert color="primary" %}}

يسمح Microsoft Excel للمستخدمين بإضافة تعليقات إلى الخلايا لإضافة معلومات إضافية وتسليط الضوء على البيانات. قد تحتاج المطورون إلى تخصيص التعليق لتحديد إعدادات المحاذاة واتجاه النص ولون الخط وما إلى ذلك. توفر Aspose.Cells واجهات برمجة التطبيقات لإنجاز المهمة.

{{% /alert %}}

توفر Aspose.Cells خاصية [**Shape.TextBody**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textbody) للون الخط في التعليق. يوضح الكود العينة التالي استخدام الخاصية [**Shape.TextBody**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textbody) لتعيين اتجاه النص للتعليق.
