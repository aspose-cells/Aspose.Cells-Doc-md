{{% alert color="primary" %}}

غالبًا ما تحتاج إلى معرفة رقم الإصدار للتطبيق الذي أنشأ مستند Microsoft Excel. توفر Aspose.Cells الخاصية [**Workbook.BuiltInDocumentProperties.Version**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/version) لهذا الغرض.

{{% /alert %}}

الكود النموذجي التالي يظهر استخدام خاصية [**Workbook.BuiltInDocumentProperties.Version**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/version). يقوم بتحميل ملفات Excel التي تم إنشاؤها باستخدام Microsoft Excel 2003 و 2007 و 2010 و 2013 ويقوم بطباعة رقم الإصدار للتطبيق الذي أنشأ هذه المستندات في Excel.
