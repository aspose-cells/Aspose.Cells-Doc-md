## **سيناريوهات الاستخدام الممكنة**

 يمكنك تغيير**رقم الإصدار** من ملف Excel عن طريق النقر بزر الماوس الأيمن فوق الملف ثم تحديد خصائص> تفاصيل ثم تحرير ملف**رقم الإصدار** مجال. يرجى استخدام[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion)لتغييره برمجيًا باستخدام واجهات برمجة تطبيقات Aspose.Cells.

## **حدد إصدار المستند لملف Excel باستخدام خصائص المستند المضمنة**

 يقوم نموذج التعليمات البرمجية التالي بإنشاء مصنف وتغيير خصائص المستند المضمنة الخاصة به والتي تتضمن العنوان والمؤلفين ورقم الإصدار. الرجاء مراجعة[إخراج ملف Excel](64716811.xlsx) تم إنشاؤها بواسطة الكود ولقطة الشاشة التي تعرض رقم الإصدار المعدل بواسطة[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion)منشأه.
