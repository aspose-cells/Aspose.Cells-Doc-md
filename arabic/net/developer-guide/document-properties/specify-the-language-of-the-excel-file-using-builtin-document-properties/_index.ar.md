## **سيناريوهات الاستخدام المحتملة**

يمكنك تغيير لغة ملف Excel عن طريق النقر بزر الماوس الأيمن على الملف ثم اختيار الخصائص > التفاصيل ومن ثم تحرير حقل اللغة. يرجى استخدام [**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/language) خاصية لتغييره برمجيا باستخدام واجهات تطبيق Aspose.Cells.

## **تحديد لغة ملف إكسل باستخدام الخصائص المدمجة للمستند**

الشيفرة المثالية التالية تنشئ دفتر عمل وتغير الخاصية المدمجة للمستند بإسم اللغة. يرجى مشاهدة [ملف Excel الناتج](64716891.xlsx) من خلال الشيفرة ولقطة شاشة تُظهر الحقل اللغوي المعدل بواسطة [**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/language) خاصية.
