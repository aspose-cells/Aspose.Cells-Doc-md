## **تحويل JSON إلى CSV**

تدعم Aspose.Cells تحويل JSON بسيط ومتداخل إلى CSV. لهذا الغرض، توفر واجهة البرمجة [**JsonLayoutOptions**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonlayoutoptions) وواجهة البرمجة [**JsonUtility**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonutility). توفر واجهة البرمجة [**JsonLayoutOptions**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonlayoutoptions) خيارات التخطيط JSON مثل [**IgnoreArrayTitle**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonlayoutoptions/properties/ignorearraytitle)(يتجاهل العنوان إذا كانت المصفوفة خاصية لكائن) أو [**ArrayAsTable**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonlayoutoptions/properties/arrayastable)(يعالج المصفوفة كجدول). تعالج واجهة البرمجة [**JsonUtility**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonutility) JSON باستخدام خيارات التخطيط المعينة مع واجهة البرمجة [**JsonLayoutOptions**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonlayoutoptions).

يُظهر الكود العيني التالي استخدام واجهة البرمجة [**JsonLayoutOptions**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonlayoutoptions) وواجهة البرمجة [**JsonUtility**](https://reference.aspose.com/cells/net/aspose.cells.utility/jsonutility) لتحميل [ملف JSON المصدر](104398869.json) وتوليد [ملف CSV الناتج](104398870.csv).
