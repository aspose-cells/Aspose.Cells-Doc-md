## **سيناريو الاستخدام**

 قد تكون هناك حالات يتم فيها نقل ملفات مصدر البيانات ويكون ملف Excel غير قادر على تحديد موقع الملف. في مثل هذه الحالات ، يوفر Aspose.Cells API خيار تحديث عنصر صيغة Power Query باستخدام[*PowerQueryFormulaItem*](https://reference.aspose.com/cells/net/aspose.cells.querytables/powerqueryformulaitem) فئة لتحديث موقع الملف المصدر.
