{{% alert color="primary" %}} 

Microsoft يدعم Excel نظامين للتاريخ: نظام التاريخ 1900 (نظام التاريخ الافتراضي المطبق في Excel لـ Windows) ونظام التاريخ 1904. يُستخدم نظام التاريخ 1904 عادةً لتوفير التوافق مع ملفات Macintosh Excel وهو النظام الافتراضي إذا كنت تستخدم Excel لنظام التشغيل Macintosh. يمكنك ضبط نظام التاريخ 1904 لملفات Excel باستخدام Aspose.Cells.

{{% /alert %}} 

لتطبيق نظام التاريخ 1904 في Microsoft Excel (على سبيل المثال Microsoft Excel 2003):

1.  من**أدوات** القائمة، حدد**الخيارات**، وحدد **الحساب** فاتورة غير مدفوعة.
1.  حدد**نظام التاريخ 1904** خيار.
1. انقر فوق موافق**.
