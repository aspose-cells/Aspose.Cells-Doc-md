{{% alert color="primary" %}}

يسمح Aspose.Cells لك بإستيراد خريطة XML داخل الكتيب باستخدام الطريقة [**Workbook.ImportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/importxml/index). يمكنك استيراد خريطة XML باستخدام Microsoft Excel باستخدام الخطوات التالية

- حدد علامة **المطور**
- انقر فوق **استيراد** في القسم XML واتبع الخطوات المطلوبة.

ستحتاج إلى تقديم بياناتك XML لإكمال الاستيراد. إليك [بيانات XML عينية](5115037.txt) يمكنك استخدامها للفحص.
