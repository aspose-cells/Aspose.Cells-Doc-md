## **سيناريوهات الاستخدام الممكنة**

Aspose.Cells يسمح لك بإزاحة الصف الأول لأسفل عند إدراج جدول في ورقة العمل. يشرح هذا المستند كيف يمكنك إنجاز المهمة باستخدام Aspose.Cells APIs.

## **نقل الصف الأول لأسفل عند ادخال Cells صفوف جدول البيانات**

يوضح نموذج التعليمات البرمجية التالي كيفية إزاحة الصف الأول لأسفل عند إدراج جدول في ورقة العمل. نستخدم ملف إكسل قالب بسيط في كود لإظهار الميزة. يمكنك ممارسة هذه الميزة عن طريق ضبط القيمة المنطقية[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown) يعزو إلى**حقيقي**/**خطأ شنيع** لفهمها بشكل أفضل. الرجاء مراجعة[نموذج لملف Excel](45056031.xlsx), [إخراج ملف Excel خطأ](45056032.xlsx) ، و[إخراج ملف Excel صحيح](45056033.xlsx)للرجوع اليها.
