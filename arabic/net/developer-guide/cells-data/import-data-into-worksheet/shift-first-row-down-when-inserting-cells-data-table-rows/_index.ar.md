## **سيناريوهات الاستخدام المحتملة**

يتيح لك Aspose.Cells نقل الصف الأول إلى الأسفل عند إدراج جدول في ورقة العمل. يوضح هذا الوثيقة كيف يمكنك إنجاز المهمة باستخدام واجهات برمجة التطبيقات Aspose.Cells.

## **نقل الصف الأول إلى الأسفل عند إدراج صفوف بيانات الخلايا**

يوضح الكود النموذجي التالي كيفية نقل الصف الأول لأسفل عند إدراج جدول في الورقة العمل. نحن نستخدم ملف Excel قالبي بسيط في الكود لتوضيح الميزة. يمكنك ممارسة الميزة عن طريق ضبط السمة البوليانية [**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown) إلى **True**/**False** لفهمها بشكل أفضل. يرجى الاطلاع على [ملف Excel عيني](45056031.xlsx)، [ملف Excel False الإخراج](45056032.xlsx)، و [ملف Excel True الإخراج](45056033.xlsx) للرجوع إليها.
