{{% alert color="primary" %}}

Aspose.Cells يدعم تحويل Microsoft Excel الموجه HTML إلى تنسيق XLS/XLSX. وهذا يعني أنه يمكن تحويل HTML الذي تم إنشاؤه بواسطة Microsoft Excel مرة أخرى إلى تنسيق XLS/XLSX باستخدام Aspose.Cells.

 وبالمثل، إذا كان هناك بعض HTML البسيط، فيمكن لـ Aspose.Cells تحويله إلى HTML نص منسق. Aspose.Cells يوفر[**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/properties/htmlstring) الخاصية التي يمكن أن تأخذ مثل هذا الرقم البسيط HTML وتحويله إلى نص خلية منسق.

{{% /alert %}}

يوضح لك نموذج التعليمات البرمجية أدناه كيفية إضافة نص منسق HTML داخل الخلية. يرجى الاطلاع على لقطة شاشة لملف Excel الناتج.
