{{% alert color="primary" %}}

 Aspose.Cells يوفر الآن[**Style.QuotePrefix**](https://reference.aspose.com/cells/net/aspose.cells/style/properties/quoteprefix) الخاصية لمعرفة ما إذا كانت قيمة الخلية تبدأ بعلامة اقتباس واحدة. قبل هذه الخاصية، لم تكن هناك طريقة للتمييز بين سلاسل مثل Sample و'sample etc.

{{% /alert %}}

يوضح نموذج التعليمات البرمجية التالي أنه لا يمكن التمييز بين السلاسل مثل Sample و'sample[**Cell.StringValue**](https://reference.aspose.com/cells/net/aspose.cells/cell/properties/stringvalue) ملكية. ولذلك يجب علينا استخدام[**Style.QuotePrefix**](https://reference.aspose.com/cells/net/aspose.cells/style/properties/quoteprefix)خاصية التمييز بينهما.
