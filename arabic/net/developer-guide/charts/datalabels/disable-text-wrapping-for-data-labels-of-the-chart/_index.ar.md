{{% alert color="primary" %}}

Microsoft يتيح Excel 2013 للمستخدمين التفاف النص أو إلغاء تغليفه داخل تسميات البيانات الخاصة بالمخطط. بشكل افتراضي، يكون النص الموجود داخل تسميات البيانات للمخطط في حالة الالتفاف.

 Aspose.Cells يوفر أ[**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) الخاصية التي يمكنك من خلالها تعيين True أو False لتمكين أو تعطيل التفاف النص لتسميات البيانات على التوالي.

{{% /alert %}}
