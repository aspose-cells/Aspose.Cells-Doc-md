{{% alert color="primary" %}}

يسمح Microsoft Excel 2013 للمستخدمين بتضمين أو عدم تضمين النص داخل تسميات البيانات في الرسم البياني. بشكل افتراضي، يكون النص داخل تسميات البيانات في الرسم البياني في حالة التضمين.

يوفر Aspose.Cells [**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) خاصية يمكنك تعيين True أو False لتمكين أو تعطيل تضمين النص بتسميات البيانات على التوالي.

{{% /alert %}}
