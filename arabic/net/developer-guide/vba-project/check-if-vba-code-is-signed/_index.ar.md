{{% alert color="primary" %}}

يتيح Aspose.Cells للمستخدم التحقق مما إذا كان مشروع كود VBA موقع بالتوقيع أم لا. يرجى استخدام خاصية [**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned) للتحقق مما إذا كان مشروع كود VBA موقع بالتوقيع أم لا.

{{% /alert %}}

الكود التالي يوضح كيفية التحقق مما إذا كان كود VBA موقع بالتوقيع أم لا باستخدام خاصية [**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned). يمكنك استخدام أي من ملفات Excel الخاصة بك لاختبار هذا الكود. لأغراض الاختبار، يمكنك استخدام [هذا الملف Excel المستخدم في الكود](5115032.xlsm).
