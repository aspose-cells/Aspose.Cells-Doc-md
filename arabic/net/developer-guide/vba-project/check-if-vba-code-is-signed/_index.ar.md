{{% alert color="primary" %}}

Aspose.Cells يسمح للمستخدم بالتحقق مما إذا كان مشروع كود VBA موقعاً أم لا. الرجاء استخدام[**المصنف. VbaProject.IsSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned) للتحقق مما إذا كان مشروع رمز VBA موقّعًا أم لا.

{{% /alert %}}

 يشرح الكود التالي كيفية التحقق مما إذا كان رمز VBA موقّعًا أم لا يستخدم ملف[**المصنف. VbaProject.IsSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned) خاصية. يمكنك استخدام أي من ملفات Excel الخاصة بك لاختبار هذا الرمز. لأغراض الاختبار ، يمكنك استخدام[ملف Excel هذا المستخدم في الكود](5115032.xlsm).
