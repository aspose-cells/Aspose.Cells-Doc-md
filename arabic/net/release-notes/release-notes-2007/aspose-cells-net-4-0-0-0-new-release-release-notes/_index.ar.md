- تمت إضافة طريقة أسماء النطاقات لحل مشكلة عند إضافة كمية كبيرة من النطاقات المسماة.
- تمت إضافة عروض توضيحية جديدة للرسم البياني
- إصلاح الخلل في إعداد الصيغة
- تم إصلاح خطأ أثناء حفظ الملف للدفق بتنسيق CSV
