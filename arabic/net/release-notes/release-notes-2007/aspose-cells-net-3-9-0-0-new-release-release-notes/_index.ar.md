- تمت إضافة علامة ذكية جديدة
- يدعم تحميل الملفات النصية بفاصل مخصص
- إصلاح الخلل في تعيين الارتباطات التشعبية للملفات المحلية
- إصلاح الخلل في طريقة InsertRows
- إصلاح الخلل في إعدادات التحقق من صحة البيانات
- إصلاح الخلل في فتح بعض ملفات Excel
