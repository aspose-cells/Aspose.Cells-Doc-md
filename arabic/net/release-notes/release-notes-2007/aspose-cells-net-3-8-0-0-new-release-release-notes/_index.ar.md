 لقد أصدرنا Aspose.Cells for .NET v3.8!

- يدعم حماية / إلغاء حماية المصنفات

- محرك حساب الصيغة المحسن

- تم إصلاح الخلل المتمثل في فتح ملف وحفظه بشكل متكرر بإصدار تقييمي
- إصلاح خطأ AddCopy أثناء نسخ بعض الصيغ
- التحسينات والإصلاحات الأخرى منذ الإصدار 3.7
