 l يضيف خصائص العرض والارتفاع للتعليق.

 l يضيف خاصية CommentFont إلى GridDesktop لتعيين خط التعليقات.

 l توفير إصدارات جديدة لقائمة التحميل الزائد للأسلوب Add لفئة CommentCollection لتحديد Width and Height of Comments.

 l إضافة خاصية IsVisible إلى ورقة العمل.

 يدعم قراءة / كتابة خصائص ورقة العمل المخصصة في ملفات إكسل ، وإضافة خاصية CustomProperties للقراءة فقط إلى ورقة العمل.

 ل يدعم وظيفة / صيغة vlookup.

 l يدعم ميزات التراجع / الإعادة عند تغيير قيم الخلايا.

 l يضيف خاصية ContextMenuManager إلى GridDesktop لإدارة قائمة السياق.

 l يضيف حدث RowColumnHiddenChanged.

 ل يدعم الاختيار المتعدد للصفوف / الأعمدة / المناطق.

 ل يدعم استيراد / تصدير الأجزاء المجمدة من / إلى ملفات Excel.
