يسعدنا أن نعلن Aspose.Cells for .NET 4.7.0!

ما الذي تغير:

- يدعم التحويل المباشر لملفات Excel إلى PDF.
 يدعم خيار خط الشبكة لإعداد إعداد الصفحة أثناء طباعة ورقة العمل.
 يدعم إدراج ملف WAV ككائن Ole.
يدعم مربعات الاختيار R / W في ملفات Excel 2007.
 يدعم استخدام أنواع مجهولة أو كائن مخصص كمصدر بيانات في العلامة الذكية
 يدعم ضبط خيار عرض القيمة الصفرية.
 يدعم الاتحاد ويتقاطع نطاقين.
 يدعم حفظ البيانات بعد حفظ المصنف.
 يدعم خيار إدراج صفوف فارغة في الجدول المحوري أم لا.
 يعزز حساب المحرك ويدعم المزيد من الوظائف.
