 يسعدنا أن نعلن Aspose.Cells for .NET v4.8.0.0!

 ما الذي تغير:

 ل دمج مجموعة Aspose.Grid في Aspose.Cells

 l تحسين أداء عمليات إعادة حساب الصيغ

 l تحسين أداء إدراج / حذف الصفوف / الأعمدة

 l تحسين أداء ميزة الملاءمة التلقائية للصفوف / الأعمدة

 l إضافة أشكال WordArt إلى ملف XLS

 قراءة تأثيرات التعبئة المتدرجة للرسم البياني

 l إنشاء جداول بيانات باستخدام سمات الصفحة الأولى أو سمات رأس / تذييل زوجية / فردية لإعداد الصفحة

 l دعم مناطق الطباعة المتعددة في ميزة ورقة العمل إلى الصورة

 l إضافة دعم حساب الصيغة لبعض الوظائف المضمنة الجديدة في MS Excel

 l تعيين صورة الخلفية لورقة عمل من نوع المخطط

 l توفير الدعم لإعداد "خيارات الصورة" في ميزات "مخطط إلى صورة" و "ورقة العمل إلى صورة"
