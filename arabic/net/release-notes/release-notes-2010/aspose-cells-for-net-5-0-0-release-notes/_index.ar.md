 يسعدنا إصدار Aspose.Cells for .NET v5.0.0!

 ما تغير في Aspose.Cells:

- يتضمن دعم تنسيق ملف MS Excel 2007 XLSB.
 يوفر ميزة نسخ ولصق خاص.
يدعم فرز العلامات الذكية.
 37 التحسين والإصلاحات.

 ما تغير في Aspose.Cells.GridWeb:

- يدعم إخفاء قائمة السياق.
 3 تحسينات وإصلاحات.



 ما تغير في Aspose.Cells.GridDesktop:

- يدعم Copying ComboBox ويوفر معالج أحداث لمقارنة العناصر والقيم في اثنين من ComboBox.
 يتضمن ميزة استدارة النص لبيانات الخلايا.
 الحصول على موقع الخلية من خلال إحداثيات / نقاط العميل.

 تم حل المشكلات في Aspose.Cells for .NET v5.0.0.







 تغييرات ملحوظة للمستخدمين:

 أثناء محاولتنا الحفاظ على API واضحًا ومباشرًا قدر الإمكان ، قررنا التعرف على ممارسات التطوير المشتركة للمنصة واحترامها ؛ لقد أعدنا ترتيب API الهيكل / مساحات الأسماء.

مع هذا الإصدار ، أي Aspose.Cells for .NET v5.0.0 ، قمنا بإعادة تنظيم فئات API لمكون Aspose.Cells. هذا التغيير له بعض الجوانب الرئيسية التي نتبعها. لقد أضفنا مساحات أسماء جديدة. كان API بأكمله (الفئات والواجهات والتعدادات والهياكل وما إلى ذلك) موجودًا سابقًا في مساحة الاسم Aspose.Cells. الآن ، تم نقل مجموعات معينة من API إلى مساحات الأسماء النسبية الخاصة بها ، مما يجعل علاقة الفئات (مع أعضائها) ومساحات الأسماء واضحة ومبسطة.

 يتم تقسيم جميع فئات مكتبة Aspose.Cells إلى مساحات أسماء مناسبة لسهولة الوصول إليها. مساحة الاسم Aspose.Cells هي بالطبع مساحة اسم الجذر التي توفر فئات لإنشاء وتحويل وتعديل وعرض وطباعة مستندات Excel Microsoft دون استخدام Microsoft Excel. المكون Aspose.Cells له مساحات الأسماء التالية:

- Aspose.Cells
 Aspose.Cells. الرسوم البيانية
 Aspose.Cells الرسم
 Aspose.Cells التقديم
 Aspose.Cells.Pivot
 Aspose.Cells الجداول
 Aspose.Cells. عقارات
