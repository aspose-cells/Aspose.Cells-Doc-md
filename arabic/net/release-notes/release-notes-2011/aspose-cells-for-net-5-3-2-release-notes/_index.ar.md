- يدعم تشفير MS Excel 2010.
- يوفر دعمًا لتعطيل / إخفاء مربع حوار عملية الطباعة
- يدعم تغيير مصدر الورق أثناء الطباعة.
- 43 تحسينات وإصلاحات.
