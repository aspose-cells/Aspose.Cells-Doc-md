### **API العام والتغييرات غير المتوافقة مع الإصدارات السابقة**

فيما يلي قائمة بأي تغييرات تم إجراؤها على API العام مثل الأعضاء المضافين أو المعاد تسميتهم أو المحذوفون أو المهملون بالإضافة إلى أي تغيير غير متوافق مع الإصدارات السابقة تم إجراؤه على Aspose.Cells for .NET. إذا كانت لديك مخاوف بشأن أي تغيير مدرج ، فيرجى رفعه في منتدى الدعم Aspose.Cells.

يضيف خاصية ImportTableOptions.IsHtmlString.

يشير إلى ما إذا كانت قيمة السلسلة في جدول البيانات تحتوي على علامات html.

يضيف طريقة Workbook.CreateBuiltinStyle (نوع BuiltinStyleType).

ينشئ نمطًا مدمجًا حسب النوع المحدد.

عفا عليها الزمن Cells.End الملكية.

استخدم Cells.LastCell بدلاً من ذلك.

يضيف Cells.ImportGridView (GridView GridView، int firstRow، int firstColumn، ImportTableOptions options).

يستورد طريقة عرض الشبكة لهذه الخلايا بخيارات

إضافة خاصية ImportTableOptions.ConvertGridStyle.

يشير إلى ما إذا كان سيتم تطبيق نمط عرض الشبكة على الخلايا.

 Obsoletes Cells.ImportGridView (GridView GridView، int firstRow، int firstColumn، bool insertRows، bool convertStringToNumber، bool convertStyle) الطريقة.

استخدم Cells.ImportGridView (GridView GridView، int firstRow، int firstColumn، ImportTableOptions options).

يضيف خاصية LoadDataOption.OnlyVisibleWorksheet.

يقوم بتحميل بيانات ورقة العمل المرئية فقط.

أسلوب Obsoletes Worksheet.CopyConditionalFormatting.

استخدم Cells.CopyRows () أو Range.Copy () بدلاً من ذلك.
