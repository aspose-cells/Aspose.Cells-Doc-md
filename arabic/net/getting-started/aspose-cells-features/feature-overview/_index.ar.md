|[.NET ستاندرد 2.0](/cells/ar/net/system-requirements/#net-standard)|يحتوي المجلد "netStandard" على تجميع لاستخدامه مع إطار العمل الذي يطبق .NET Standard 2.0 مثل .NET Core 2.1/3.1. أحدث إصدار من Aspose.Cells API يدعم أيضًا .NET 5.0 و.NET 6.0 و.NET 7.0 و.NET8.0.|
|[COM](/cells/ar/net/system-requirements/)|يمكن أن يكون منقذًا إذا كان عليك التطوير باستخدام تقنيات ما قبل .NET.|
|[Mono](/cells/ar/net/system-requirements/)|يمكنك استخدام Aspose.Cells for .NET لبناء التطبيقات باستخدام Mono.|
|[Windows أزور](/cells/ar/net/system-requirements/)|Aspose.Cells for .NET يعمل أيضًا على Windows Azure.|
