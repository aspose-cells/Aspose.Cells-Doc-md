|[. إطار العمل .NET](/cells/ar/net/system-requirements/#net-frameworks)|استخدم Aspose.Cells for .NET لبناء ASP.NET، خدمات الويب، WinForms أو أي نوع آخر من التطبيقات لأي إصدارات .NET Framework 2.0 أو أحدث بما في ذلك إصدارات ملف العميل. دعم كامل للإصدارات 32 بت و 64 بت.
|[.NET Standard 2.0](/cells/ar/net/system-requirements/#net-standard)|يحتوي مجلد 'netStandard' على التجميعات التي يمكن استخدامها مع الإطار الذي ينفذ .NET Standard 2.0 مثل .NET Core 2.1/3.1. تدعم الإصدار الأخير لواجهة برمجة التطبيقات Aspose.Cells أيضًا .NET 5.0 و .NET 6.0 و .NET 7.0 و .NET8.0

|[ويندوز أزور](/cells/ar/net/system-requirements/)|تعمل Aspose.Cells for .NET أيضًا على Windows Azure.
