## **مقدمة**
يرجى استخدام الطريقة [PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) لإنشاء جدول محوري داخل ورقة العمل. بمجرد إنشاء الجدول المحوري، يمكنك العمل معه باستخدام فئة [PivotTable](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/)
## **إنشاء جدول محوري**
يُظهر الكود المصدري التالي كيفية إنشاء جدول محوري والعمل معه. يُرجى التحقق من ملف Excel الناتج المولد بهذا الكود واللقطة الشاشية التالية التي تُظهر [ملف Excel الناتج](23167010.xlsx) في Microsoft Excel
