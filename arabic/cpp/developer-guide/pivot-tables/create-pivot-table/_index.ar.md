##  **مقدمة**
 الرجاء استخدام[PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) طريقة إنشاء جدول محوري داخل ورقة العمل. بمجرد إنشاء الجدول المحوري، يمكنك العمل معه باستخدام[جدول محوري](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/)فصل.
##  **إنشاء جدول محوري**
 يوضح نموذج التعليمات البرمجية التالي كيفية إنشاء جدول محوري والعمل معه. رجاء تاكد من[إخراج ملف إكسل](23167010.xlsx) تم إنشاؤه باستخدام هذا الرمز ولقطة الشاشة التالية التي توضح[إخراج ملف إكسل](23167010.xlsx) في Microsoft اكسل.
