## **سيناريوهات الاستخدام الممكنة**
Microsoft يدعم Excel العديد من التنسيقات مثل XLS و XLSX و XLSM و XLSB و CSV وما إلى ذلك. كما يدعم Aspose.Cells العديد من هذه التنسيقات. تشرح هذه المقالة كيفية قراءة وكتابة ملف excel بتنسيق CSV باستخدام Aspose.Cells.
## **قراءة وكتابة تنسيق ملف CSV**
