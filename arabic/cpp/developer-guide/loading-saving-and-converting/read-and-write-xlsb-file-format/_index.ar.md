##  **سيناريوهات الاستخدام المحتملة**
Microsoft يدعم Excel العديد من التنسيقات مثل XLS، XLSX، XLSM، XLSB، CSV وغيرها. كما يدعم Aspose.Cells العديد من هذه التنسيقات. تشرح هذه المقالة كيفية قراءة وكتابة ملف Excel بتنسيق XLSB باستخدام Aspose.Cells.
##  **قراءة وكتابة XLSB تنسيق الملف**
 يقوم نموذج التعليمات البرمجية التالي بتحميل ملف[المصدر XLSB الملف](23166992.xlsb)ويقرأ الخلية A1 ثم ينسخ محتوياتها إلى الخلية C4 ويحفظها باسم[ملف الإخراج XLSB](23166993.xlsb).
