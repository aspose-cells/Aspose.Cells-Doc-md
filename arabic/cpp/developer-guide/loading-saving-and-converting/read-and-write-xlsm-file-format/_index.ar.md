## **سيناريوهات الاستخدام الممكنة**
Microsoft يدعم Excel العديد من التنسيقات مثل XLS و XLSX و XLSM و XLSB و CSV وما إلى ذلك. كما يدعم Aspose.Cells العديد من هذه التنسيقات. تشرح هذه المقالة كيفية قراءة وكتابة ملف Excel بتنسيق XLSM باستخدام Aspose.Cells.
## **قراءة وكتابة تنسيق ملف XLSM**
 يقوم نموذج التعليمات البرمجية التالي بتحميل ملف[مصدر ملف XLSM](23166990.xlsm) ويقرأ الخلية A1 الخاصة بها ثم ينسخ محتوياتها إلى الخلية C4 ويحفظها بتنسيق[إخراج ملف XLSM](23166991.xlsm).
