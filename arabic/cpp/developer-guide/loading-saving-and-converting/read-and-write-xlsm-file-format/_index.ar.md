##  **سيناريوهات الاستخدام المحتملة**
Microsoft يدعم Excel العديد من التنسيقات مثل XLS، XLSX، XLSM، XLSB، CSV وغيرها. كما يدعم Aspose.Cells العديد من هذه التنسيقات. تشرح هذه المقالة كيفية قراءة وكتابة ملف Excel بتنسيق XLSM باستخدام Aspose.Cells.
##  **قراءة وكتابة XLSM تنسيق الملف**
 يقوم نموذج التعليمات البرمجية التالي بتحميل ملف[المصدر XLSM الملف](23166990.xlsm) ويقرأ الخلية A1 ثم ينسخ محتوياتها إلى الخلية C4 ويحفظها باسم[ملف الإخراج XLSM](23166991.xlsm).
