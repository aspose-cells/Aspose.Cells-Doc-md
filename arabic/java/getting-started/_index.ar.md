## **متطلبات النظام**

Aspose.Cells for Java يدعم الإصدارات التالية من Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- أو أعلى

[تفاصيل أكثر](/cells/ar/java/system-requirements/)

## **التثبيت**

### **قم بتثبيت Aspose Cells لـ Java من مستودع Maven**

Maven هو أسهل طريقة لتنزيل وتثبيت Aspose.Cells for Java. 

1. حدد تكوين/موقع مستودع Aspose Maven في pom.xml الخاص بك. 
2. قم بتحديد الاعتمادية API الخاصة بـ Aspose.Cells for Java في pom.xml الخاص بك. 

[مزيد من التفاصيل](/cells/ar/java/installation/)

### **قم بتثبيت Aspose Cells لـ Java يدوياً**

1. [قم بتنزيل حزم aspose-cells JAR](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) إلى مجلد.
1. ادخل إلى المجلد، ثم قم بتثبيت حزم JAR يدوياً باستخدام Eclipse أو Intellij.
