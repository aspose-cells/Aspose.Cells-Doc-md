Aspose.Cells for Javaلا تتطلب وجود Microsoft Excel المثبت، حيث يُعتبر Aspose.Cells نفسه مُحرك إنشاء وتحويل وعرض جداول بيانات Excel. لعرض المستندات التي تُنتجها Aspose.Cells، يجب أن تكون لديك على الأقل Microsoft Excel Viewer مثبت.

## **أنظمة التشغيل المدعومة**
Aspose.Cells for Javaيدعم أي نظام تشغيل يعمل على تشغيل Java بما في ذلك ولكن لا يقتصر على:
### **Windows**
- Microsoft Windows 2000 (x64، x86)
- Microsoft Windows XP (x64، x86)
- Microsoft Windows Server 2003 (x64، x86)
- Microsoft Windows Server 2008 (x64، x86)
- Microsoft Windows Server 2012 (x64، x86)
- Microsoft Windows Vista (x64، x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8 (x64, x86)
- Microsoft Windows 10 (x64, x86)
### **Linux**
- لينكس (أوبونتو، openSUSE، CentOS وغيرها)
### **Mac**
- نظام تشغيل ماك OS X الخ
## **إصدارات Java المدعومة**
Aspose.Cells for Java يدعم الإصدارات التالية من Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- أو أعلى
