## **مرحبًا بك في وثائق Aspose.Cells**
**Aspose.Cells** هو العنصر الأكثر مرونة الذي يتيح لتطبيقات Java إنشاء وإدارة أوراق عمل Excel دون الحاجة إلى تثبيت **Microsoft Excel** على النظام. على عكس المنتجات المماثلة من البائعين الآخرين، لا تدعم Aspose.Cells فقط إنشاء أوراق العمل وميزات التنسيق الأساسية الأخرى، ولكنها تدعم أيضًا عددًا من الميزات المتقدمة. تجعل هذه الميزات المتقدمة من الأكثر راحة للمطورين للتلاعب بمحتويات أوراق العمل، وتنسيق الخلايا وحماية الملفات. تسمح Aspose.Cells أيضًا باستيراد البيانات إلى أوراق العمل من مصادر بيانات مختلفة، وإضافة الصيغ والدوال الرياضية الشائعة والمعقدة للنصوص / الوظائف، وتدعم تلاعب الرسوم البيانية والصور والتعليقات وكائنات الرسم والتحكم الخ.

يوفر Aspose.Cells for Java أيضًا حلاً كاملاً للشبكة للعملاء؛ لديه تحكم مبني على واجهة المستخدم الرسومية كما يلي:

- Aspose.Cells.GridWeb، الذي يدعم تطبيقات الويب

عند استخدام Aspose.Cells.GridWeb لتحكم القاعدة الخاص بـ JAVA Grid، يمكن للمطورين استيراد البيانات من وتصديرها إلى أوراق عمل Excel بسهولة. يمكن لأي نوع من المستخدمين الذين يعرفون Microsoft Excel التحكم في شكل ومظهر الشبكة. توفر Aspose.Cells مجموعة غنية من الميزات مثل التحكم في التصميم وتخطيط الشبكة، وإدارة الورقات العمل المتعددة، وإدارة الخلايا والصفوف والأعمدة في ورقة عمل، وإنشاء وحساب الصيغ الرياضية إلخ.

## **وصف المنتج**
يمكن استخدام Aspose.Cells مع أي نوع من التطبيق سواء كان تطبيق Java Web أو تطبيق سطح المكتب في Java. علاوة على ذلك، يمكن استخدام المكتبة أيضًا مع بعض الحلول الأخرى مثل PHP إلخ. يوفر Aspose.Cells نماذج توضيحية ميزة وأمثلة عملية مكتوبة بلغة Java لدى المطورين لديهم فهم أفضل للميزات التي يتم عرضها من قبل العنصر الأساسي.

بالإضافة إلى دعم الميزات الشائعة مثل إنشاء ملفات Excel الأصلية أو قراءتها، تقدم Aspose.Cells أيضًا العديد من الميزات المتقدمة مثل حفظ وفتح ملفات Excel إلى ومن السلاسل، إرسال الناتج إلى متصفح العميل، استيراد وتصدير البيانات من **ResultSet** أو **Array**، استيراد الصور والرسوم البيانية، إنشاء الصور والرسوم البيانية من خلال API، استيراد النصوص النموذجية من ورقة عمل المصمم وتكوين إعداد الصفحة من خلال API.

علاوة على ذلك، هناك أيضًا عدد من الميزات الفريدة التي يمكن للمطورين العثور عليها فقط في Aspose.Cells مثل دعم تنسيقات CSV وSpreadsheetML، إضافة نسخة من ورقة عمل موجودة مسبقًا (بجميع المحتويات والصور والرسوم البيانية إلخ) إلى ملف، تعيين خلفية براعة للرسوم البيانية من خلال API، إنشاء تعليقات، مرشحات تلقائية، كسور الصفحة من خلال API، تعيين صيغ معقدة، دعم التنسيق الشرطي، دعم خيارات الحماية المقدمة في Microsoft Excel XP أو أعلى، التلاعب بالمجالات المسماة، توجيه APIs، دعم API للرسوم البيانية المخصصة ومحرك حساب الصيغ الخ.
## **ميزات الوثيقة**
- فتح الملفات العادية أو المشفرة Excel (Excel 95، Excel97، Excel2007/2010/2013/2016/2019، Office365) من مصادر مختلفة
- فتح ملفات الأرقام العادية (التي تم إنشاؤها بواسطة تطبيق "Numbers" لشركة Apple) من مصادر مختلفة
-  حفظ ملفات Excel (Excel97- Excel2007/2010/2013/2016/2019) بتنسيقات مختلفة (XLS، XLSX/XLSM، XLTX، XLTM، XLSB، CSV، SpreadsheetML، محدد / TSV، HTML إلخ)
- تحويل ملف Excel إلى مستند PDF
- تحويل ورقة عمل Excel إلى مستند XPS.
- التحويل إلى  تنسيق ملف الصور المعلم **(TIFF)**
- قراءة وكتابة **تنسيق جدول جدول الوثيقة (ODS)** الخاص بـ OpenDocument
- تعديل خصائص الوثيقة لملفات Excel
## **ميزات الورقة العمل**
### **خيارات العرض**
- **إخفاء أو عرض** ورقة عمل
- عرض أو إخفاء علامات التبويب، شريط التمرير، خطوط الشبكة وتذييل الصفوف والأعمدة
- **معاينة كسر الصفحة**
- تعيين مستوى **التكبير** لورقة العمل
- تجميد الأجزاء المحددة للبيانات لتظل مرئية أثناء التمرير باستخدام **لوحات التجميد**
### **حماية ورقة العمل**
- **حماية محتويات، أجهزة وسيناريوهات** ورقة العمل
### **خيارات إعداد الصفحة**
- تعيين **اتجاه الصفحة، تحجيم، حجم الورق، جودة الطباعة** الخ.
- تعيين **هوامش الصفحة** مثل الأعلى، اليسار، اليمين، السفل
- تعيين **تذييلات ورؤوس الصفحات**
- تعيين **منطقة الطباعة**، **عنوان الطباعة** الخ.
### **كسرات الصفحة**
- إدراج **كسور صفحة أفقية** و**عمودية** في خلية محددة
### **ميزات الصفوف والأعمدة**
- تعديل عرض العمود كما هو مطلوب
- تكبير عرض وارتفاع الخلايا تلقائياً وفقًا لمحتوياتها
- **إدراج أو حذف** صفوف وأعمدة
- إخفاء أو عرض الصفوف والأعمدة
- تجميع أو فك تجميع الصفوف والأعمدة
### **ميزات إدارة البيانات**
- إضافة بيانات إلى الخلايا المطلوبة أثناء التشغيل
- استيراد البيانات إلى أوراق العمل من مصادر بيانات مختلفة
- استرداد البيانات من الخلايا وفقًا لأنواعها
- تصدير بيانات الخلايا إلى مجموعة
- البحث أو البحث عن بيانات محددة في الخلايا
- تعيين الصيغ باستخدام وظائف مدمجة أو إضافية
- تعيين صيغة مرجعية للمصفوفة و R1C1
- إضافة أنواع مختلفة من الارتباطات التشعبية
- إنشاء أو الوصول إلى مجموعات الأسماء
- تطبيق تحقق البيانات والتصفية التلقائية
- تطبيق المجاميع الفرعية على جداول أوراق العمل
### **ميزات التنسيق**
- تطبيق إعدادات تنسيق الأرقام للخلايا
- تنسيق الخلية كتاريخ
- تعيين محاذاة النص
- دمج عدة خلايا
- تطبيق أنواع مختلفة من إعدادات الخط للخلايا
- قم بتطبيق ألوان مختلفة على الخلايا والخطوط وخطوط الشبكة والكائنات الرسومية الخ.
- ضبط النص الغني في خلية واحدة
- تطبيق إعدادات حدود مختلفة للخلايا
- تطبيق أنماط خلفية مختلفة على الخلايا
- تطبيق إعدادات التنسيق على ورقة عمل أو صف أو عمود الخ
- تطبيق تنسيق مشروط على الخلايا
## **ميزات الرسم البياني والرسومات**
- إنشاء أنواع مختلفة من المخططات بسهولة
- تخصيص المخططات عن طريق تعيين الخصائص
- ضبط البيانات للمخططات بشكل ديناميكي
- إضافة مخططات مخصصة إلى أوراق العمل
- إضافة صور إلى أوراق العمل أثناء التشغيل
- إنشاء وإضافة تعليقات إلى الخلايا
- تحويل المخطط إلى ملف صورة
- تحويل ورقة العمل إلى ملف صورة
- طباعة أوراق العمل
## **ميزات متقدمة**
- يدعم حساب الصيغ باستخدام محرك حساب الصيغ القوي
- تغيير بيانات المصدر لجدول المحور أثناء التشغيل
- إنشاء وتلاعب بجداول البيانات المحورية
- تلاعب برمجيات VBA أو النصوص البرمجية.
