## **xlsx4j**
xlsx4j هي مكتبة Java مفتوحة المصدر (ASLv2) لإنشاء وتلاعب ملفات Microsoft Open XML Excel xlsx على غرار docx4j لتلاعب ملفات Microsoft Open XML Word docx.
إنها مشابهة لتطبيقات Microsoft’s OpenXML SDK، ولكن لـ Java. docx4j يستخدم JAXB لإنشاء التمثيل الداخلي للكائن.
سيتعين عليك أن تأخذ الوقت الكافي لفهم JAXB وهيكل ملف Open XML.

اقرأ المزيد على [docx4java.org](https://www.docx4java.org/trac/docx4j)
## **Aspose.Cells for Java**
Aspose.Cells for Java هو مكون جدول بيانات Excel® الحائز على جوائز يسمح لمطوري Java بتضمين القدرة على قراءة وكتابة وتلاعب بجداول بيانات Excel® (XLS، XLSX، XLSM، XLSB، XLTX، SpreadsheetML، CSV، ODS) وHTML وMHTML و PDF وصيغ ملفات الصور في تطبيقاتهم الخاصة بـ Java دون الحاجة للانتظار على Microsoft Excel®.

Aspose.Cells for Java هو مكون ناضج وقابل للتطبيق بشكل كبير وميزات غنية يقدم العديد من الوظائف التي تتجاوز قدرات تصدير البيانات البسيطة للموردين الآخرين. باستخدام Aspose.Cells for Java، يمكن للمطورين تصدير البيانات، تنسيق الجداول الخاصة بـ أدق التفاصيل، استيراد الصور، إنشاء الرسوم البيانية، تطبيق وحساب الصيغ المعقدة، ستريم بيانات Excel®، حفظ في صيغ مختلفة وأكثر من ذلك بكثير - كل هذا دون الحاجة لـ Microsoft Excel® أو Microsoft Office Automation.

اقرأ المزيد على [aspose.com](https://products.aspose.com/cells/java/)

## **لماذا لا تستخدم xlsx4j**
xlsx4j يمكنها فقط تحليل وحفظ ملفات Microsoft Open XML Excel xlsx، يمكن Aspose.Cells for Java معالجة جميع تنسيقات ملفات Excel (XLS، XLSX، XLSM، XLSB، XLTX، SpreadsheetML، CSV، ODS) ويدعم التحويل بين تنسيقات الملفات المختلفة.

## **Aspose.Cells Java for xlsx4j**
يظهر مشروع Aspose.Cells لـ xlsx4j كيف يمكن تنفيذ مهام مختلفة باستخدام واجهات برمجة التطبيقات Java لـ Aspose.Cells بالمقارنة مع xlsx4j. يغطي المشروع أيضًا الميزات المتوفرة فقط في واجهات برمجة التطبيقات Aspose.Cells وليست متاحة في xlsx4j للتعامل مع جداول البيانات.

هذا المشروع مفيد للمطورين الذين يرغبون في مقارنة xlsx4j مع Aspose.Cells أو الانتقال من xlsx4j إلى Aspose.Cells.
## **متطلبات النظام والمنصات المدعومة**
### **متطلبات النظام**
فيما يلي متطلبات النظام لتشغيل Aspose.Cells Java لـ xlsx4j:

- يجب توفر Java 1.4 أو أحدث.
- تنزيل مكون Aspose.Cells.
- تنزيل مكتبات docx4j.
### **المنصات المدعومة**
فيما يلي المنصات المدعومة:

- Aspose.Cells 8.3.0 أو أحدث.
- مكتبات docx4j 3.1.0 أو أحدث.
- بيئة تطوير Java (Eclipse، NetBeans، IntelliJ، ...).
## **التحميل**
إصدارات تشغيل الأمثلة التي يمكن تنزيلها على جميع المواقع المذكورة أدناه على مواقع التعاون الاجتماعي:

### **GitHub**
- [Aspose.Cells Java for Xlsx4j - v 1.0](https://github.com/aspose-cells/Aspose.Cells-for-Java/releases/tag/Aspose.Cells_Java_for_Xlsx4j-v1.0.0)

لتهيئة **Aspose.Cells for Java API**، قم بتضمين عنوان مستودع Aspose URL والتبعية مافن Aspose.Cells for Java API في ملف **pom.xml** الخاص بك كما يلي:
### **مستودع Aspose Maven**
{{< highlight java >}}

 <repositories>

    <repository>

        <id>aspose-maven-repository</id>

        <name>Aspose Maven Repository</name>

        <url>http://repository.aspose.com/repo/</url>

    </repository>

</repositories>

{{< /highlight >}}
### **تبعية مافن Aspose.Cells for Java**
{{< highlight java >}}

 <dependencies>

    <dependency>

        <groupId>com.aspose</groupId>

        <artifactId>aspose-cells</artifactId>

        <version>8.6.0</version>

    </dependency>

</dependencies>

{{< /highlight >}}
