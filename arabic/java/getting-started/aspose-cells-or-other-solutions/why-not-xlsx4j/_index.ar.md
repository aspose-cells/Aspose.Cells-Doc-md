## **xlsx4j**
xlsx4j هي مكتبة مفتوحة المصدر (ASLv2) Java لإنشاء ومعالجة ملفات Microsoft Open XML Excel xlsx المشابهة لـ docx4j لمعالجة Microsoft Open XML Word docx files.
إنه مشابه لـ Microsoft OpenXML SDK ، لكن for Java. يستخدم docx4j JAXB لإنشاء تمثيل كائن في الذاكرة.
ستحتاج إلى قضاء بعض الوقت في فهم JAXB وهيكل ملف Open XML.

 اقراء المزيد على[docx4java.org](https://www.docx4java.org/trac/docx4j)
## **Aspose.Cells for Java**
Aspose.Cells for Java هو أحد مكونات جداول بيانات Excel الحائز على جوائز والذي يسمح لمطوري Java بتضمين القدرة على قراءة وكتابة ومعالجة جداول بيانات Excel® (XLS ، XLSX ، XLSM ، XLSB ، XLTX ، 07618310481 ، 0761910481 ، SpreadsheetML ، 076191481) وتنسيقات ملفات الصور في تطبيقات Java الخاصة بهم دون الحاجة إلى الاعتماد على Microsoft Excel®.

Aspose.Cells for Java هو مكون ناضج وقابل للتطوير وغني بالميزات يقدم العديد من الوظائف التي تفوق بكثير قدرات تصدير البيانات البسيطة للبائعين الآخرين. مع Aspose.Cells for Java يمكن للمطورين تصدير البيانات وتنسيق جداول البيانات إلى المستوى الأكثر دقة واستيراد الصور وإنشاء المخططات وتطبيق وحساب الصيغ المعقدة ودفق بيانات Excel® وحفظها بتنسيقات مختلفة وغير ذلك الكثير - كل هذا دون الحاجة إلى Microsoft Excel® أو Microsoft أتمتة المكاتب.

 اقراء المزيد على[aspose.com](https://products.aspose.com/cells/java/)

## **لماذا لا xlsx4j**
يمكن لـ xlsx4j فقط تحليل وحفظ ملفات Microsoft Open XML Excel xlsx ، ويمكن Aspose.Cells for Java معالجة جميع تنسيقات ملفات Excel (XLS ، XLSX ، XLSM ، XLSB ، XLTX ، SpreadsheetML ، يدعم تنسيقات ملفات مختلفة CSV ، 0761)

## **Aspose.Cells Java لـ xlsx4j**
يوضح المشروع Aspose.Cells الخاص بـ xlsx4j كيف يمكن تنفيذ المهام المختلفة باستخدام واجهات برمجة التطبيقات Aspose.Cells Java مقارنةً بـ xlsx4j. يغطي المشروع أيضًا الميزات المتوفرة فقط في Aspose.Cells APIs ولكن ليس في xlsx4j للتعامل مع جداول البيانات.

هذا المشروع مفيد للمطورين الذين يريدون مقارنة xlsx4j بـ Aspose.Cells أو الترحيل من xlsx4j إلى Aspose.Cells.
## **متطلبات النظام والأنظمة الأساسية المدعومة**
### **متطلبات النظام**
فيما يلي متطلبات النظام لتنفيذ Aspose.Cells Java لـ xlsx4j:

- Java 1.4 أو أعلى مثبتة.
- تم تحميل Aspose.Cells المكون.
- مكتبات docx4j التي تم تنزيلها.
### **المنصات المدعومة**
فيما يلي المنصات المدعومة:

- Aspose.Cells 8.3.0 أو أعلى.
- docx4j 3.1.0 أو أعلى مكتبات.
- Java IDE (Eclipse ، NetBeans ، IntelliJ ، ...).
## **جارى التحميل**
الإصدارات التالية من الأمثلة قيد التشغيل متاحة للتنزيل على جميع مواقع الترميز الاجتماعي المذكورة أدناه:

### **جيثب**
- [Aspose.Cells Java لـ Xlsx4j - v 1.0](https://github.com/aspose-cells/Aspose.Cells-for-Java/releases/tag/Aspose.Cells_Java_for_Xlsx4j-v1.0.0)

 لتكوين**Aspose.Cells for Java API** ، قم بتضمين عنوان URL للمستودع Aspose و Aspose.Cells for Java API maven التبعية في**pom.xml** كالتالي:
### **مستودع Aspose Maven**
{{< highlight "java" >}}

 <repositories>

    <repository>

        <id>aspose-maven-repository</id>

        <name>Aspose Maven Repository</name>

        <url>http://repository.aspose.com/repo/</url>

    </repository>

</repositories>

{{< /highlight >}}
### **Aspose.Cells for Java Maven التبعية**
{{< highlight "java" >}}

 <dependencies>

    <dependency>

        <groupId>com.aspose</groupId>

        <artifactId>aspose-cells</artifactId>

        <version>8.6.0</version>

    </dependency>

</dependencies>

{{< /highlight >}}

**ملحوظة:**الرجاء استخدام أحدث إصدار متوفر من Aspose API.
