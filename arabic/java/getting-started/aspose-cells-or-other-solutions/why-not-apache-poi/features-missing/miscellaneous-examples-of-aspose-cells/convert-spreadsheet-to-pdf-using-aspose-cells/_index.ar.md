## **أسبوز.خلايا - تحويل ورقة البيانات إلى PDF**
تستخدم وثائق PDF على نطاق واسع كتنسيق قياسي لتبادل الوثائق بين المؤسسات والقطاعات الحكومية والأفراد. غالبًا ما يُطلب من مطوري البرمجيات أن يبتكروا طريقة لتحويل ملفات Microsoft Excel إلى وثائق PDF بسهولة. تدعم Aspose.Cells هذه الميزات.

**Java**

{{< highlight java >}}

 Workbook workbook = new Workbook("workbook.xls");

//Save the document in PDF format

workbook.save("AsposeConvert.pdf", SaveFormat.PDF);

{{< /highlight >}}
