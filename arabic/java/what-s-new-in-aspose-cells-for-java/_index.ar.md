تصف هذه الصفحة الميزات الجديدة الأكثر إثارة في Aspose.Cells التي تم إدخالها في الإصدارات الأخيرة.

## Aspose.Cells for Java 24.7
* دعم وظيفة الإسقاط في الصيغ
* تحسين أداء حساب وظائف LET و MAKEARRAY بمعلمات معقدة
* دعم مخطط لتوضيح حروف النجوم
* دعم جعل كائن مزخرفًا
* دعم تبديل السلاسل
* تحسين عرض رسم بياني لشجرة الخريطة
* دعم تحويل صورة "خلية محددة" إلى "في الخلية"
* دعم تحميل خط أوفيس سحابي محلي بشكل افتراضي
* دعم الاحتفاظ بمخطط json عند استيراد بيانات json
* دعم التصدير إلى ملف بتنسيق AZW3
* دعم تعيين الارتفاع تلقائيًا بعد تعيين الانطباع للخلية في GridJS
* دعم إنشاء json باستخدام طريقة كاتب تيار لـ GridJS

## Aspose.Cells for Java 24.6
* تحسينات للتقديم: تشبع اللون وضبط درجة الحرارة اللونية، صور ويب ب متعددة
* تحسينات لجدول الدوران: دمج النطاق، "عرض القيم كـ" من حقل الدوران
* دعم Range.ToHtml()/ToImage() لراحة المستخدم في التلاعب ببيانات النطاق

## Aspose.Cells for Java 24.5

* دعم عرض الصورة الموجودة في الخلية إلى ملف PDF / صورة

## Aspose.Cells for Java 24.4
* توفير تنفيذات بسيطة لرصد الانقطاع لراحة المستخدم
* دعم تضمين المرفقات عند تحويل كائن OLE إلى PDF
* دعم عرض الشكل التلقائي لـ ChartX إلى صورة
* دعم تضمين الصورة في الخلية (عملية "وضع في الخلية" للصورة في MS Excel)
* تحسينات لاستيراد ملف Numbers: الخلايا المدمجة ، الصفوف / الأعمدة المخفية ، ... الخ.
