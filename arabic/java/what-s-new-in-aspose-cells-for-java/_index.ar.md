تصف هذه الصفحة الميزات الجديدة الأكثر إثارة في Aspose.Cells التي تم إدخالها في الإصدارات الأخيرة.

## Aspose.Cells for Java 25.4
* يدعم تحديد مجموعة الأوراق للتصدير إلى HTML
* يدعم الحصول على لون الشبكة لضبط ورقة العمل
* يدعم تصدير الرسوم البيانية القابلة للتحرير إلى ملفات DOCX
* يدعم قراءة وكتابة علامات الحساسية
* يدعم تصدير المزيد من سمات النمط إلى أرقام
* يدعم استيراد ملفات القالب بصيغة Markdown
* يدعم اكتشاف ما إذا كانت أي وظيفة مخصصة مستخدمة بواسطة الصيغ في `Cell` و`Workbook`

* دعم تحويل إكسل إلى JSON لبنية دفتر العمل
* دعم تصدير (قيمة الخلية، التنسيق، الصيغة والتعليق) إلى ملف Numbers
* دعم وضع التنسيق/التنسيق داخل عناصر HTML عند حفظ HTML
* دعم تمييز علامة التبويب للورقة المحددة في HTML الناتج
* دعم المزيد من مفاتيح الاختصار ل GridJs
* دعم نسخ المحتوى المنسوخ من إكسل للصقه في مناطق متعددة لـ GridJs

## Aspose.Cells for Java 25.2
