تصف هذه الصفحة ميزات Aspose.Cells الجديدة الأكثر إثارة للاهتمام والتي تم تقديمها في الإصدارات الأخيرة.

##  Aspose.Cells for Java 24.2
* دعم وظائف جديدة: TEXTAFTER، TEXTBEFORE، TEXTSPLIT، TOROW، TOCOL
* دعم الكشف عن تنسيق oxps
* تحسين الأداء لفرز مجموعة البيانات الكبيرة

##  Aspose.Cells for Java 24.1

* دعم jakarta.servlet لـ GridWeb
* دعم لاستيراد بيانات CSV مع صفوف/أعمدة الرأس المحددة من قبل المستخدم وإجمالي الصفوف/الأعمدة لكل ورقة
* دعم تصدير HTML مع الأنماط المخصصة
* دعم التعبئة التلقائية للمناطق المدمجة
* دعم تعبئة البيانات تلقائيًا من اليمين إلى اليسار ومن الأسفل إلى الأعلى

##  Aspose.Cells for Java 23.12

* دعم الامتثال PDF/A-2 وPDF/A-3 أثناء التحويل إلى pdf
* دعم حساب التجميع المنفصل للجدول المحوري
* دعم تصدير بيانات الخلايا إلى القائمة العامة
* دعم استيراد المراجع الترافقية باستخدام الجداول المرتبطة للعلامات الذكية
* دعم وظيفة MAKEARRAY للصيغ

##  Aspose.Cells for Java 23.11

* دعم لتحويل جداول البيانات إلى إصدار الملف PDF 1.7 (ISO32000-1:2008)

##  Aspose.Cells for Java 23.10

* دعم لتعيين الارتباطات التشعبية مباشرة إلى النطاق
* دعم لنسخ أوراق عمل متعددة في وقت واحد
* دعم للتحقق مما إذا كان أحد النطاقات فارغًا لبيانات الخلايا

##  Aspose.Cells for Java 23.9

* دعم لحساب وظائف جديدة في الصيغة: LAMBDA، LET، MAP، SCAN
* دعم لاستيراد/تصدير ملفات EPUB
* دعم لتصفية سلسلة الرسم البياني

##  Aspose.Cells for Java 23.8

* دعم للحفاظ على مساحة الحشو في الصيغ
* دعم استبدال/إدراج نص بتنسيق النص المنسق للخلية

##  Aspose.Cells for Java 23.7

* اكتشاف محسّن لتنسيقات الملفات لبعض التنسيقات الجديدة مثل ملف Visio وملف حزمة OneNote.

##  Aspose.Cells for Java 23.6

* دعم لتصدير وتسليط الضوء على سجلات المراجعة
* دعم لاجتياز كائنات الصف الموجودة من RowCollection بترتيب عكسي
* دعم لتحديث نطاقات الدمج المتعددة للمحور المحوري

##  Aspose.Cells for Java 23.5

* دعم لتحديث المراجع الخارجية لبيانات الورقة الخارجية إلى الورقة المحلية للمصنف الحالي
* دعم تقديم عنصر تحكم Checkbox كعنصر تحكم تفاعلي بدلاً من الصورة الثابتة في GridWeb
* دعم لتحديد PivotArea وتهيئته
* دعم لإضافة علامة مائية أثناء التحويل إلى pdf

##  Aspose.Cells for Java 23.4

* دعم لحساب وظيفة ENCODEURL
* دعم لإكمال بناء شكل المعادلة عن طريق إدراج العقد ذات الصلة خطوة بخطوة
* دعم لحساب تعبير الصيغة كصيغة صفيف دون تعيينه على كائن خلية
* دعم لحفظ ملف xlsb في وضع LightCells
* دعم للتحقق مما إذا كان هناك خط واحد مثبت أو متاح
* دعم لتعيين الصيغة للخلية المحددة في ListObject
* دعم لحفظ المصنف كـ XHTML
* دعم لتقديم التحكم activex والتحكم في النموذج لـ GridWeb

##  Aspose.Cells for Java 23.3

* دعم ضبط/قراءة/حفظ الوظائف الجديدة: LET، ENCODEURL

##  Aspose.Cells for Java 23.2
