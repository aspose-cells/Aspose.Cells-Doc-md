 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells for Java 2.3.1](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.3.1/)

 يسعدنا أن نعلن Aspose.Cells for Java 2.3.1!

 ما الذي تغير:

- يوفر دعمًا لنوع تشفير AES لملفات Excel 2007.
- يدعم قراءة / كتابة جداول PivotTable لملفات Excel 2007.
- يضيف دعم تنسيق صورة svg / svgz لميزة الرسم البياني للصورة.
- يدعم مناطق طباعة متعددة لخيارات إعداد الصفحة.
- تعيين تنسيقات الأرقام المخصصة لـ PivotField.
- يتضمن خصائص المستند المضمنة الجديدة لملف Excel 2007.
- يجمع بين مصنفات متعددة.
- يدعم حفظ ملفات PDF في البث المباشر لميزة Excel إلى PDF.
- تم إجراء تحسين لقراءة / كتابة ملفات Excel 2007.
- تم إجراء تحسين لقراءة / كتابة ملفات Excel 2003.
- تم إجراء تحسين لقراءة ملفات قالب XLS.
- 25 إصلاحات وتحسينات أخرى.
