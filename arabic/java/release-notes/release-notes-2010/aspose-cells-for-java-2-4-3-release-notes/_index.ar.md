 يسعدنا أن نعلن Aspose.Cells for Java 2.4.3!

 ما الذي تغير:

- يدعم الحصول على ألوان Excel 2007 Theme.
- يصدر المخططات / الأشكال كصور لتحويل ملف HTML.
- يدعم قراءة / كتابة رؤوس / تذييلات مختلفة للصفحات الأولى / الزوجية / الفردية.
- يعمل على إصلاح بعض الأخطاء الداخلية المتعلقة بقراءة / حفظ ملفات Excel 2007.
- إصلاح بعض الأخطاء الداخلية لقراءة / حفظ ODS ملفات.
- يعمل على إصلاح بعض الأخطاء الداخلية المتعلقة بقراءة / حفظ ملفات Excel 97-2003.
- تم إجراء تحسين لميزة الرسم البياني إلى الصورة.
- تم إجراء تحسين لـ Direct Excel لميزة PDF.
- 14 إصلاحات وتحسينات.
