-  يدعم معلمات الصفيف (

- تحقق مما إذا كان الترخيص قد تم تعيينه ، بما في ذلك الدعم للحصول على تاريخ انتهاء اشتراكك.
- تم تحسين عملية قراءة / حفظ ملفات Excel 2007.
- تم تحسين ميزة الرسم البياني للصورة.
- تم تحسين ميزة Direct Excel to PDF.
- تم تحسين تنسيق قيم Cell.
- تم تحسين عملية قراءة / حفظ ملفات Excel 97-2003.
- 21 إصلاحات وتحسينات.
