 يسعدنا أن نعلن Aspose.Cells for Java 2.2.1!

 ما الذي تغير:

-  يتضمن دعمًا للتنسيق الشرطي المتقدم الذي يوفره MS Excel 2007
-  يدعم السجلات الموسعة للتنسيق الشرطي المحفوظة بواسطة MS Excel 2007 لملف XLS
- يحتفظ بالصيغ غير المدعومة في قراءة ملفات MS Excel 2007
-  يقرأ الصور من رأس / تذييل الصفحة في ملف قالب MS Excel 2007
-  تم إجراء تحسينات لقراءة ملفات القوالب
-  17 إصلاحات وتحسينات أخرى.
