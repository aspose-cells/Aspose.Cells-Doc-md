 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells for Java 2.4.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.4.0/)

 يسعدنا أن نعلن Aspose.Cells for Java v2.4.0!

 ما الذي تغير:

- يتضمن ميزات الورقة إلى الصورة والورقة المراد طباعتها والمصنف المراد طباعته
- يدعم التنسيق الشرطي لميزة Excel إلى Pdf
- يوفر دعمًا لتجميع حقول الجدول المحوري في ملفات Excel2007
- يحتفظ بوحدات الماكرو عند تحويل ملفات Excel97-2003 إلى ملفات Excel2007
- يعرض الأشكال عند التحويل إلى ملفات Pdf ، مثل TextBox و Rectangle و Oval و Arc
- يدعم الحصول على / تعيين الاسم الكامل للملف المصدر لكائن Ole
- تم إجراء تحسين لقراءة / حفظ التنسيق الشرطي
- تم إجراء تحسين لميزة الرسم البياني إلى الصورة
- تم إجراء تحسين لميزة Excel إلى Pdf المباشرة
- تم إجراء تحسين لتنسيق قيم الخلايا
- تم إجراء تحسين لإزالة الرأس / التذييل من PageSetup
- تم إجراء تحسين لنسخ الخلايا
- 22 إصلاحات وتحسينات.
