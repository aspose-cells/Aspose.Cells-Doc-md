 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells for Java 2.3.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-2.3.0/)

{{% /alert %}} 

 يسعدنا أن نعلن Aspose.Cells for Java 2.3.0!

 ما الذي تغير:

- يدعم حفظ ملفات PDF مباشرة دون استخدام Aspose.Pdf for Java.
- يتضمن EMF دعم تنسيق صورة لميزة الرسم البياني للصورة.
- يستورد سمة CalculationMode من ملفات قالب MS Excel 2007.
- تم تحسين عملية نسخ ورقة العمل.
- يحسن نسخ الصيغ.
- يحسن قراءة ODS الملفات باستخدام ODF 1.2.
- يحسن قراءة المخططات مع تأثير تعبئة متدرج.
- تم تحسين قراءة ملفات القوالب XLS.
- 31 إصلاحات وتحسينات.
