نحن
 يسعدني أن نعلن Aspose.Cells for Java v7.1.1!

 ميزات جديدة

- تحقق مما إذا كان العمود فارغًا

 التحسينات

- الحفظ في XLSX باستخدام LightCells API عن طريق العرض المباشر لـ OutputStream

 استثناءات

- يطرح أسلوب Cell.setHtmlString () NullPointerException
- صفوف ملفات قالب Readingan XLSX NullPointerException

 البق

- أصبح محور الفئة الثانوية لمخطط SCATTER محور النص
- لم يتم تقديم المحتويات في Textbox بشكل صحيح
- لا يمكن للدالة ROUND تقريب قيمة أكبر من 922337.20
- لم تتم قراءة المخطط بشكل صحيح من ملف ODStemplate
- تنسيق الأرقام المخصص: لم يتم حفظ "dd.MM.yyyy" بشكل صحيح لملف ODS
- أصبح LineShape أقصر في مستند PDF الذي تم إنشاؤه
