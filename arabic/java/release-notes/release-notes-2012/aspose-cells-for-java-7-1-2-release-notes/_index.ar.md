 يسعدني أن نعلن Aspose.Cells for Java v7.1.2!

 ميزات جديدة

- تم إجراء تحسينات على أساليب autoFitColumns ()

- حدث NullPointerException أثناء استدعاء طريقة Cell.getDisplayStyle ()
