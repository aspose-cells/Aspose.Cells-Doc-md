 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells for Java 7.3.4](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.3.4/)

{{% /alert %}} 

نحن
 يسعدني أن نعلن Aspose.Cells for Java v7.3.4!

 ميزات جديدة

- دعم تنسيق TIFF في ورقة إلى صورة

 التحسينات

- التذييل الأيمن ليس أعلى من الوسط كما هو الحال في MS Excel

 استثناءات

- استثناء نفاد الذاكرة أثناء تحويل Excel إلى PDF
- تعيين "+100" كصيغة شرطية تسبب في استثناء
- الاستثناء: "StackOverflowError" عند حساب الصيغ
- يتم طرح "IllegalArgumentException" عند استدعاءWorkbook.copy ()

 البق

- أصبح النص العربي أحرفًا غير مهمة في ملف CSV المحفوظ باستخدام UTF-8
- خطأ "ربما فقدت البيانات" لملف XLS المعاد حفظه بواسطة Aspose.Cells
- خطأ "عثر Excel على محتوى غير قابل للقراءة ..." للتقرير الذي تم إنشاؤه Aspose.Cells
- Cell.getFormula () لم يميز نطاقات مسماه مختلفة بنفس الاسم ولكن بنطاق مختلف
- العنوان التلقائي لإصدار مخطط PIE
- لم يميز Cell.getR1C1Formula () نطاقات مسماة مختلفة بنفس الاسم ولكن بنطاق مختلف
