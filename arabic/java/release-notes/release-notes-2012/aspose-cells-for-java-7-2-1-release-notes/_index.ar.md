 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells for Java 7.2.1](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.2.1/)

نحن
 يسعدني أن نعلن Aspose.Cells for Java v7.2.1!

 ميزات جديدة

- تنسيق الجدول المحوري بأنماط مخصصة لفئات مختلفة (تعديل النمط السريع للجدول المحوري)

 التحسينات

- Cells.findString () / find () يدعم البحث RegExand في نطاق معين
- دعم Picture.setTitle () / getTitle ()
- حفظ مخططات MS Excel في ملف ODS
- اجعل Aspose.Cells أنشأ ملف XLS متوافقًا مع POI

 استثناءات

- ينتج عن قراءة ملف XLSX: "java.lang.ClassCastException: org.dom4j.Namespace"

 البق

- يعطي ملف XLSX المحفوظ خطأ: "Datamay has been lost"
- الرقم المنسق غير صحيح في ملف PDF الذي تم إنشاؤه (تم فقد ألف حرف مجموعة)
- لم يظهر المخطط الشريطي في ملف PDF الذي تم إنشاؤه لإصدار JDK6
- لا يتم تحديث المراجع عند توسيع النطاق
