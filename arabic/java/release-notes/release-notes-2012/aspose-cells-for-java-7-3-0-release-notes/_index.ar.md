نحن
 يسعدني أن نعلن Aspose.Cells for Java v7.3.0!

 ما هو الجديد

- اقرأ
 واكتب ملفات MHT
- يدعم
 خرائط XML
- يتقدم
 موضوعات MS Excel 2007/2010 إلى المخططات



الميزات والتحسينات والإصلاحات البارزة الأخرى مذكورة أدناه.

 ميزات جديدة

 - يدعم إعدادات تقييم الصيغة الانتقالية

 التحسينات

- تنسيق قيم الخلية بمقياس المستخدم المحدد

 استثناءات

- قد تتسبب ملفات الخطوط غير المدعومة في فشل الحفظ في PDF مع استثناء

 البق

- Cell.setR1C1 لم تعمل طريقة الصيغة () بشكل صحيح بدون إزاحة العمود
- تم إنشاء النطاق المسمى مرتين
- تسببت طريقة sortNames () في استثناء عند حفظ ملف anXLSM
- لم يتم تنسيق الرقم بشكل صحيح مع كسر
- تم إنشاء ملف PDF فارغ لـPrintOrderType.OVER_ومن بعد_أسفل
- لون الخلفية والشبكات غير صحيحين في ملف PDF الذي تم إنشاؤه
- لم يتم حساب وظائف التقاطع والميل بشكل صحيح
- قم بإزالة حد 33 كيلو بايت لعناصر الحقول المحورية لتنسيقات ملفات Excel 2007
- التدوين 1: 1 غير مدعوم في IFfunction
- تم حساب صيغة DATEDIF بشكل غير صحيح
- بحث غير صحيح عن الخلايا في حالة الصف المسمى
