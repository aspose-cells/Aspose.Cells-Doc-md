 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells for Java 7.2.2](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-7.2.2/)

{{% /alert %}} 

نحن
 يسعدني أن نعلن Aspose.Cells for Java v7.2.2!

 ميزات جديدة

- البحث عن نص RegEx عن Cells.Find () طريقة

 التحسينات

- اجعل Aspose.Cells متوافقًا مع الإصدار الأقدم من برطمانات Woodstox
- يتم إخراج ملفات OOXML المضمنة في XLS من الملفات المجمعة ASOLE بدلاً من الملفات غير المضغوطة
- دعم ExportObjectListener لحفظ ملفات HTML
- نسخ التنسيق الشرطي عند نسخ صفوف

 استثناءات

- يتسبب Picture.isPrintable () للصورة inPageSetup في NullPointerException
- يؤدي حفظ المصنف المشفر باستخدام Pivot Tablecauses java.lang.NegativeArraySizeException

 البق

- Cells.importCustomObjects () بتنسيق DateTime المحدد لا يعمل
- نوع المخطط المبعثر غير صحيح
- تفقد القيمة المزدوجة الدقة عند القراءة من ملف القالب CSV
- انقلبت سلسلة الرسوم البيانية رأسًا على عقب عند تحويلها إلى صورة
- إعادة حفظ الملف XLSX يتسبب في حدوث خطأ "Excelfound un readable content ..."
- تسبب الجدول المحوري المحفوظ في ظهور "ProtectionView" عند فتحه في MS Excel
- استخدام التحقق من القائمة بواسطة Aspose.Cellscreates وهو ملف XLS لا يعمل بعد تغيير فاصل قوائم النظام
