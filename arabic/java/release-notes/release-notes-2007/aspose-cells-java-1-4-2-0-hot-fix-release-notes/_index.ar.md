 العملاء الأعزاء ،

 لقد أصدرنا Aspose.Cells for Java V1.4.2.0!

 يتم تضمين التحسينات والإصلاحات التالية:

- النطاق المعتمد المدعوم
 تمت إضافة Worksheets.addCopy () ، طريقة Worksheet.copy (ورقة العمل)
 أيد إضافة الصور
 دعم وضع التصفية التلقائية
 تمت إضافة Worksheets.addSheet (int) ، Worksheets.addSheet (int ، String) لإضافة نوع ورقة الرسم البياني.
