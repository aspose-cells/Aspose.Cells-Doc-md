 العملاء الأعزاء ،

 لقد أصدرنا Aspose.Cells for Java V1.7.4! تم تضمين التحسينات والإصلاحات التالية:

- يضيف أساليب Column.applyStyle و Row.applyStyle.
 يعمل على إصلاح بعض الأخطاء المتعلقة بإعداد الصيغ وحسابها.
 يعمل على إصلاح خطأ في صف المواجهة.
 يعمل على إصلاح خطأ أجزاء التجميد.
 يعمل على إصلاح خطأ حفظ المخططات.
 يعمل على إصلاح خطأ حفظ الجداول المحورية.
 يعمل على إصلاح خطأ في طريقة cell.importResultSet ().

ملاحظة: الطريقة getNumerOfSheets of Workbook / Worksheets تمت إعادة تسميتها باسم getNumberOfSheets. ومن الأفضل استدعاء Worksheets.size () للحصول على عدد الأوراق.
