 العملاء الأعزاء ،

 لقد أصدرنا Aspose.Cells for Java V1.7.1! تم تضمين التحسينات والإصلاحات التالية:

- يدعم استيراد محتوى وأنماط الخلايا من ملف SpreadsheetML
 يدعم تصدير محتوى وأنماط الخلايا إلى ملف SpreadsheetML
 يدعم محرك حساب الصيغة
 يدعم ضبط صيغة R1C1
 يدعم نوعين آخرين من مصادر البيانات من PivotTable
 يعمل على إصلاح بعض الأخطاء المتعلقة بإضافة PivotTable.

ملاحظة: أزلنا الثابت CellValueType.FORMULA وأضفنا طريقة Cell.isFormula () لتحديد ما إذا كانت الخلية تحتوي على صيغة.
