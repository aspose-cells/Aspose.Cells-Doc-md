 العملاء الأعزاء ،

 لقد أصدرنا Aspose.Cells for Java V1.7.2! تم تضمين التحسينات والإصلاحات التالية:

- يدعم إضافة صورة إلى رأس / تذييل إعداد الصفحة
 يدعم استيراد إعداد الصفحة والاسم والحماية والتحقق من صحة البيانات و DocumentProperties من ملف SpreadsheetML
 يدعم تصدير إعداد الصفحة والاسم والحماية والتحقق من صحة البيانات وخصائص المستند إلى ملف SpreadsheetML
 يدعم حساب المزيد من الوظائف المضمنة
 يعمل على إصلاح خطأ تعيين اللون التلقائي للخط في المخطط.
