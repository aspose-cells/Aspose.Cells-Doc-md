 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells Java 1.4.0.0 إصدار جديد](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.4.0.0-new-release/)

 العملاء الأعزاء ،

 لقد أصدرنا Aspose.Cells for Java V1.4.0.0!

 يتم تضمين التحسينات والإصلاحات التالية:

- تمت إضافة طريقة Cell.setValue (كائن)
 تمت إضافة طريقة Cell.getStringValue ()
 تحسين استخدام الذاكرة
 إصلاح الخلل في استيراد تنسيقات أرقام الرسم البياني

 تغيير الاسم

 يرجى ملاحظة أنه تمت إعادة تسمية Aspose.Excel for Java إلى Aspose.Cells for Java.

إذا كنت تقوم بالترقية من Aspose.Excel for Java ، فأنت بحاجة إلى استبدال Aspose.Excel.jar بـ Aspose.Cells.jar في CLASSPATH. تحتاج أيضًا إلى تغيير كافة حزمة "الاستيراد" com.aspose.excel إلى com.aspose.cells في التعليمات البرمجية الخاصة بك.
