 العملاء الأعزاء ،

 لقد أصدرنا Aspose.Cells for Java V1.8.0!

 يتم تضمين التحسينات والإصلاحات التالية:

 يدعم تعيين الورقة والخلية النشطة ، ويصلح بعض الأخطاء المتعلقة بإعداد الصيغ. يعمل على إصلاح بعض الأخطاء في حساب الصيغ. يصلح خطأ في فتح الملف ، ويصلح خطأ في التكيف مع ورقة العمل. يعمل على إصلاح خطأ الجدول المحوري. يصلح خطأ في التحقق من الصحة. يصلح خطأ في المرشحات التلقائية. يصلح خطأ في freezePanes.

ملاحظة: بالنسبة إلى Aspose.Cells for Java 1.4 ، قمنا بتضمين جميع فئات وقت تشغيل مترجم خلفي و backport-util-concurrent. لذلك لا تحتاج إلى الرجوع إلى البرطمانين.
