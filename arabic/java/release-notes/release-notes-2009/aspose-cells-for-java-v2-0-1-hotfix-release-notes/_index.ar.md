 يسعدنا أن نعلن Aspose.Cells for Java 2.0.1!

 ما الذي تغير:

 l يدعم R / W Open Document Spreadsheet file (ODS): Cell البيانات والنمط ، نمط الصف / العمود ، مدمج Cells ، خصائص ورقة العمل / المصنف ، خصائص المستند ، الصيغة / الاسم.

 ل يدعم وضع الصيغة المشتركة.

 يدعم معالجة عناصر التحكم: خانة الاختيار ، الزر ، زر الاختيار ، مربع التحرير والسرد ، مربع القائمة ، مربع المجموعة ، شريط التمرير ، القرص الدوار ، التسمية.

 l يدعم رأس (رؤوس) إعداد الصفحة والتذييل (التذييلات) في xls2pdf.

 ل يدعم نسخ الرسم البياني عند نسخ ورقة العمل / المصنف

 ل يدعم نسخ الكائنات ذات الصلة بالشكل

 ل يدعم حفظ ورقة العمل بصيغة html للدفق

 ل يدعم PivotChart

 ل يدعم الحصول على بيانات الصورة للصورة و OleObject

 ل يدعم R / W Color theme لتنسيق ملف 2007

 ل يدعم نوع التشفير: Microsoft مزود تشفير قوي

 ل 27 الإصلاحات والتحسينات.
