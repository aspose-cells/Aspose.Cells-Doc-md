 تحتوي هذه الصفحة على ملاحظات الإصدار لـ[Aspose.Cells for Java V1.9.4 الإصلاح السريع](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-v1.9.4-hot-fix/)

 ما هو الجديد؟

- يدعم إضافة نص منسق لأشكال مثل Arc و Oval و Rectangle و Comment و TextBox
 يدعم كل أنواع التنسيب للشكل
 يدعم قراءة جميع الأشكال من القالب
 يدعم ضبط مجموعة في SmartMarker

 ما الذي تم تحسينه؟

- أضف طرقًا جديدة للتعامل مع موضع الشكل وحجمه ، وتم إهمال بعض الطرق القديمة
 أضف بعض الأكواد التجريبية والقوالب لـ API الجديد
 استبدل استدعاء API المهمل بـ API جديد في العروض التوضيحية

 ما الذي تم إصلاحه؟

- إصلاح بعض الخلل في حساب الصيغة
 تم إصلاح خطأ في تصدير تنسيق Excel2007 XLSX
 إصلاح الخلل في أسلوب النسخ
 إصلاح الخلل في نسخ الصيغة
 إصلاح الخلل في حفظ حجم الصورة

 تعليقات

- بالنسبة إلى المخططات التي تم استيرادها من ملفات القوالب ، لا يتم دعم معالجة البيانات الخاصة بها ، ولا يدعم حاليًا سوى العمل على خصائص الشكل ، مثل الموضع والموضع والحجم
