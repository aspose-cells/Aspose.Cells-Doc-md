## **يمكنك تضمين محرر جداول بيانات HTML5 في أي مكان**

يمكن تضمين محرر جداول بيانات HTML5 في أي موقع ويب أو مدونة أو منتديات لمشاركة جداول البيانات عبر الإنترنت. يمكن تضمينه كمحرر مستقل أو يمكنك تحميله بملف جدول بيانات.

**تضمين كمحرر مستقل**

للتضمين كمحرر مستقل ، استخدم علامة HTML IFRAME لإضافتها إلى موقع الويب. على سبيل المثال:

{{< highlight "html" >}}

 <iframe src="http://spreadsheet-editor.aspose.com/" width="800" height="600">

    Your web browser does not support IFRAMEs

</iframe>

{{< /highlight >}}

**تضمين مع جدول بيانات**

 لتحميل جدول بيانات في محرر مضمن**عنوان url** معامل. على سبيل المثال:

{{< highlight "html" >}}

 <iframe src="http://spreadsheet-editor.aspose.com/?url=http://example.com/Sample.xlsx" width="800" height="600">

    Your web browser does not support IFRAMEs

</iframe>

{{< /highlight >}}
