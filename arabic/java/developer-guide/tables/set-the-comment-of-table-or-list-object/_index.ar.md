## **ضبط التعليق للجدول أو كائن القائمة داخل ورقة العمل**

يرجى تعيين تعليق لكائن الجدول أو القائمة داخل ورقة العمل باستخدام خاصية [**ListObject.Comment**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment). سيكون التعليق مرئيًا داخل ملف xl/tables/tableName.xml.

الكود النموذجي التالي يحمل [ملف الإكسل الأصلي](5472561.xlsx)، يعين تعليقًا لأول كائن جدول أو قائمة داخل ورقة العمل ويظهر اللقطة المصورة التي تُظهر التعليق الذي أنشأه هذا الكود في المستطيل الأحمر.
