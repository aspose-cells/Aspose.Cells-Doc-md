## **سيناريوهات الاستخدام المحتملة**

توفر Aspose.Cells الطريقة [**XmlMapCollection.Add()**](https://reference.aspose.com/cells/java/com.aspose.cells/xmlmapcollection#add(java.lang.Object)) التي يمكنك استخدامها لاستيراد Map XML داخل دفتر العمل.

## **إضافة خريطة XML داخل الكتيب باستخدام طريقة XmlMapCollection.Add**

تضيف الشفرة النموذجية التالية خريطة XML داخل دفتر العمل باستخدام الـ [**XmlMapCollection.Add()**](https://reference.aspose.com/cells/java/com.aspose.cells/xmlmapcollection#add(java.lang.Object)) وتحفظها كـ [ملف إكسل مُخرَج](5472533.xlsx). يُظهر لقطة الشاشة خريطة الـ XML التي تم استيرادها من ملف الـ [sample.xml](5472532.xml) داخل ملف الإكسل المُخرَج.
