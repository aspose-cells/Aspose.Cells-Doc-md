{{% alert color="primary" %}}

توفر Microsoft Excel ميزة إضافة **سطر توقيع** في سجل Excel. يمكنك إضافة سطر توقيع عن طريق النقر فوق علامة **إدراج** ثم تحديد **سطر توقيع** من مجموعة **نص**.

توفر Aspose.Cells أيضًا هذه الميزة وقد عرضت الخاصية [**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) لهذا الغرض. سيشرح هذا المقال كيفية استخدام هذه الخاصية لإضافة سطر توقيع باستخدام Aspose.Cells.

{{% /alert %}}

## كود Java لإنشاء سطر توقيع في سجل Excel باستخدام Aspose.Cells

الكود العيني التالي يضيف سطر توقيع باستخدام [**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) ويحفظ السجل. تُظهر اللقطة الشاشية كيف يبدو السجل الناتج في Microsoft Excel.
