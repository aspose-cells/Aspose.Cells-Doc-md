 يوفر Microsoft Excel ميزة للإضافة**خط التوقيع** في مصنفات Excel. يمكنك إضافة سطر التوقيع بالنقر فوق**إدراج** انقر ثم اختر**خط التوقيع** من**نص** مجموعة.

 يوفر Aspose.Cells أيضًا هذه الميزة وقد قام بكشف ملف[**Picture.setSignatureLine ()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) خاصية لهذا الغرض. تشرح هذه المقالة كيفية استخدام هذه الخاصية لإضافة سطر توقيع باستخدام Aspose.Cells.

{{% /alert %}}
