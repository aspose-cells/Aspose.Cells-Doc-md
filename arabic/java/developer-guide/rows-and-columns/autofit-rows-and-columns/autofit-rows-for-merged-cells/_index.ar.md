{{% alert color="primary" %}}

يوفر Microsoft Excel ميزة تسمح لك بتغيير حجم ارتفاع الخلية تلقائيًا وفقًا لمحتواها. تسمى الميزة صفوف الاحتواء التلقائي. Microsoft لا يقوم Excel بتعيين عملية الاحتواء التلقائي على الخلايا المدمجة أصلاً. في بعض الأحيان تصبح الميزة حيوية للمستخدم الذي يحتاج حقًا إلى تنفيذ صفوف الاحتواء التلقائي على الخلايا المدمجة أيضًا.

{{% /alert %}}

 Aspose.Cells يدعم هذه الميزة من خلال[**AutoFitterOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/AutoFitterOptions)API. باستخدام هذا API ، من الممكن احتواء الصفوف تلقائيًا في ورقة العمل بما في ذلك الخلايا المدمجة.

بعد تنفيذ الكود ، يقوم Aspose.Cells بملاءمة الصفوف تلقائيًا للخلايا المدمجة.
