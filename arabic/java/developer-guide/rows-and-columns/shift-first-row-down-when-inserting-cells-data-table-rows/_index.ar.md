## **سيناريوهات الاستخدام المحتملة**

يسمح Aspose.Cells لك بنقل الصف الأول إلى الأسفل عند إدراج جدول في ورقة العمل. يشرح هذا المستند كيف يمكنك تحقيق المهمة باستخدام واجهات برمجة التطبيقات لـ Aspose.Cells.

## **نقل الصف الأول إلى الأسفل عند إدراج صفوف بيانات الخلايا**

يوضح الكود المصدري التالي كيفية نقل الصف الأول إلى الأسفل عند إدراج جدول في ورقة العمل. نستخدم ملف إكسل نموذجي بسيط في الكود لتوضيح الميزة. يمكنك ممارسة الميزة بضبط السمة المنطقية [**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/java/com.aspose.cells/importtableoptions#ShiftFirstRowDown) إلى True/False. يرجى الرجوع إلى [ملف الإكسل النموذجي](45056043.xlsx)، [ملف الإكسل المُخرّج False](45056044.xlsx)، و [ملف الإكسل المخرج True](45056045.xlsx) للرجوع إليها.
