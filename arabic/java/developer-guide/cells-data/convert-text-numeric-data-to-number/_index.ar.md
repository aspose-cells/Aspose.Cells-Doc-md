{{% alert color="primary" %}}

 في بعض الأحيان ، تريد تحويل البيانات الرقمية التي تم إدخالها كنص إلى أرقام. يمكنك إدخال أرقام كنص في Microsoft Excel بوضع علامة اقتباس أحادية قبل رقم ، على سبيل المثال**'12345**. ثم يعامل Excel الرقم كسلسلة. Aspose.Cells يسمح لك بتحويل السلاسل إلى أرقام.

{{% /alert %}}

يوفر Aspose.Cells for Java API[**Cells.convertStringToNumericValue ()**](https://reference.aspose.com/cells/java/com.aspose.cells/cells#convertStringToNumericValue()) الطريقة التي يمكن استخدامها لتحويل جميع البيانات الرقمية النصية أو النصية إلى أرقام.

 تُظهر لقطة الشاشة التالية أرقام السلسلة في الخلايا**A1: A17**. يتم محاذاة أرقام السلاسل إلى اليسار.

**ملف الإدخال: الأرقام التي تم إدخالها كسلاسل نصية** 

![ما يجب القيام به: image_بديل_نص](convert-text-numeric-data-to-number_1.png)

تم تحويل أرقام السلسلة هذه إلى أرقام باستخدام[**Cells.convertStringToNumericValue ()**](https://reference.aspose.com/cells/java/com.aspose.cells/cells#convertStringToNumericValue()) في لقطة الشاشة التالية. كما ترون ، هم الآن محاذون لليمين.

**ملف الإخراج: تم تحويل السلاسل إلى أرقام** 

![ما يجب القيام به: image_بديل_نص](convert-text-numeric-data-to-number_2.png)
