{{% /alert %}}

## باستخدام Microsoft Excel

هذه هي الخطوات المطلوبة للبحث عن الخلايا ذات الأنماط المحددة في MS Excel.

1. حدد **البحث والاختيار** في **علامة التبويب الرئيسية**.
1. حدد **بحث**.
1. انقر على **خيارات** إذا لم تكن الخيارات المتقدمة مرئية.
1. حدد **اختيار التنسيق من الخلية...** من القائمة المنسدلة **تنسيق**.
1. حدد الخلية ذات النمط الذي ترغب في البحث عنه.
1. انقر على **العثور على الكل** للعثور على كافة الخلايا ذات النمط المشابه للخلية التي حددتها.

## باستخدام Aspose.Cells for Java

يوفر Aspose.Cells for Java ميزة العثور على الخلايا في ورقة العمل بنمط محدد ما. لهذا، يوفر الـ API خاصية [**FindOptions.setStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/findoptions#Style).
