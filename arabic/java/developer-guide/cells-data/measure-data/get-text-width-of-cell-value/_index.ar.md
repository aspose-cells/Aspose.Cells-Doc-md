## **احصل على عرض نص بقيمة Cell**

في بعض الأحيان ، قد يحتاج المطورون إلى حساب عرض قيمة الخلية لترتيب البيانات في بعض التخطيط. يوفر Aspose.Cells ملف[**CellsHelper.GetTextWidth**](https://reference.aspose.com/cells/java/com.aspose.cells/cellshelper#getTextWidth(java.lang.String,%20com.aspose.cells.Font,%20double)) طريقة تسمح للمطورين بالحصول على عرض نص قيمة الخلية. يوضح نموذج التعليمات البرمجية التالي كيفية الاستخدام[**CellsHelper.GetTextWidth**](https://reference.aspose.com/cells/java/com.aspose.cells/cellshelper#getTextWidth(java.lang.String,%20com.aspose.cells.Font,%20double)) للوصول إلى عرض نص قيمة الخلية.

تم إرفاق ملف المصدر المستخدم في مقتطف الشفرة التالي للرجوع إليه.
