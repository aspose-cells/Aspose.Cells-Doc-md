أحيانًا تحتاج إلى حساب عرض وارتفاع قيمة الخلية لتناسب قيمة الخلية داخل الخلية. Aspose.Cells توفر طرق [Cell.getWidthOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getWidthOfValue\(\)) و [Cell.getHeightOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getHeightOfValue\(\)) لهذا الغرض. باستخدام هذه الطرق يمكنك حساب عرض وارتفاع قيمة الخلية ومن ثم تعيين عرض العمود وارتفاع الصف لتلك الخلية على التوالي وبعد ذلك ستقوم بضبط أو تناسب قيمة الخلية داخل الخلية.

بالإضافة إلى ذلك، يمكنك أيضًا [ضبط تلقائي للصفوف والأعمدة للخلية أو مجموعة الخلايا](/cells/ar/java/autofit-rows-and-columns/) باستخدام واجهات برمجة التطبيقات في Aspose.Cells.
