## **تحويل Excel إلى HTML مع تلميحة**

قد تكون هناك حالات يتم فيها قطع النص في HTML المولد وترغب في عرض النص الكامل كتلميح على حدث التحويم. يدعم Aspose.Cells ذلك عن طريق توفير الخاصية [**HtmlSaveOptions.AddTooltipText**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#AddTooltipText). ضبط الخاصية [**HtmlSaveOptions.AddTooltipText**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#AddTooltipText) إلى **true** سيضيف النص الكامل كتلميح في HTML المولد.

تُظهر الصورة التالية التلميح السريع في ملف HTML المولد.
