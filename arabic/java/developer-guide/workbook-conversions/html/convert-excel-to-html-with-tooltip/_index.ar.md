## **تحويل Excel إلى HTML مع تلميح الأدوات**

قد تكون هناك حالات يتم فيها قطع النص في HTML الذي تم إنشاؤه وتريد عرض النص الكامل كتلميح أداة في حدث التمرير. Aspose.Cells يدعم هذا من خلال توفير**[HtmlSaveOptions.AddTooltipText] (https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#AddTooltipText)**منشأه. وضع**[HtmlSaveOptions.AddTooltipText] (https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#AddTooltipText)**الملكية ل**حقيقي**سيضيف النص الكامل كتلميح أداة في HTML الذي تم إنشاؤه.

توضح الصورة التالية تلميح الأداة في ملف HTML الذي تم إنشاؤه.

![ما يجب القيام به: image_بديل_نص](convert-excel-to-html-with-tooltip_1.jpg)

نموذج التعليمات البرمجية التالي بتحميل[ملف اكسل المصدر](AddTooltipToHtmlSample.xlsx)ويولد ال[إخراج ملف HTML](AddTooltipToHtmlSample_out.zip)مع تلميح الأداة.
