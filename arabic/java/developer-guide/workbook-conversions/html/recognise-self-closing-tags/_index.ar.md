يمكن أن تحتوي HTML على مجموعة متنوعة من تنسيق العلامات للعلامات الفارغة مثل \<td>\</td> أو \<td/>. يدعم Aspose.Cells هذين التنسيقين الآن بينما كان يدعم سابقًا \<td>\</td>مثل العلامات. يمكن اختبار هذه الميزة عن طريق تحويل نموذج ملف HTML المرفق إلى ملف Excel. يمكن تنزيل نموذج ملف HTML وملفات الإخراج من الروابط التالية للاختبار.
