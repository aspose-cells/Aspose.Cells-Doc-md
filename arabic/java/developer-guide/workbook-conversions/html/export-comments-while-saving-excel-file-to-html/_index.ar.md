## **سيناريوهات الاستخدام الممكنة**

عند حفظ ملف Excel في HTML ، لا يتم تصدير التعليقات. ومع ذلك ، يوفر Aspose.Cells هذه الميزة باستخدام امتداد[**HtmlSaveOptions.SExportComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#IsExportComments)خاصية. إذا قمت بتعيينه**حقيقي**، ثم سيعرض HTML أيضًا التعليقات الموجودة في ملف Excel الخاص بك.

## **تصدير التعليقات أثناء حفظ ملف Excel إلى Html**

يشرح نموذج التعليمات البرمجية التالي استخدام[**HtmlSaveOptions.SExportComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#IsExportComments) خاصية. تُظهر لقطة الشاشة تأثير الكود على HTML عند تعيين الخاصية على**حقيقي**. يرجى تنزيل ملف[نموذج لملف Excel](50528270.xlsx)و ال[ولدت HTML](50528269)كمرجع.
