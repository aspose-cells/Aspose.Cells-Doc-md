{{% alert color="primary" %}}

توفر Aspose.Cells خاصية مفيدة، [**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference)، يمكن للمطورين استخدامها للحصول على تخطيط أفضل عند حفظ ملف Microsoft Excel إلى تنسيق HTML أو MHT. القيمة الافتراضية للخاصية هي **false**. نوصي بتعيين هذه الخاصية على **true** للحصول على عروض تقديمية تبدو أفضل من تقرير Excel.
