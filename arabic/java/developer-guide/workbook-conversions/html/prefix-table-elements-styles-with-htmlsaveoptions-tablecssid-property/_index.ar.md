## **سيناريوهات الاستخدام المحتملة**

تتيح Aspose.Cells لك تمهيد أنماط عناصر الجدول بخاصية [**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId). على سبيل المثال، إذا قمت بتعيين هذه الخاصية بقيمة معينة مثل *MyTest_TableCssId*، فستجد أنماط عناصر الجدول كما هو موضح أدناه

{{< highlight java >}}

table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}

اللقطة الشاشية التالية تظهر تأثير استخدام خاصية [**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId) على الإخراج الخاص بـ HTML.
