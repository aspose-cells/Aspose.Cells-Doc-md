{{% alert color="primary" %}} 

تقدم هذه المقالة معلومات حول كيفية إدراج إشارات مرجعية لملف PDF عند تحويل جدول بيانات إلى شكل PDF.

تتيح لك Aspose.Cells إضافة إشارات مرجعية أثناء التشغيل. يمكن لإشارات المرجعية في ملف PDF تحسين تجوال الوثائق الطويلة بشكل كبير. عند إضافة إشارات مرجعية إلى مستند PDF، يمكنك السيطرة بشكل دقيق على العرض الدقيق الذي تريده، فأنت لست مقيدًا بالربط بصفحة. يمكنك إعداد العرض الدقيق بوضع الصفحة المستهدفة، ثم إنشاء الإشارة المرجعية.

{{% /alert %}} 
## **إضافة إشارات مرجعية لملف PDF**
يرجى الرجوع إلى الكود العيني التالي لمعرفة كيفية إضافة إشارات مرجعية لملف PDF. يقوم الكود بإنشاء دفتر عمل بسيط، ويحدد إشارات مرجعية للوجهات ويقوم بإنشاء ملف PDF.
