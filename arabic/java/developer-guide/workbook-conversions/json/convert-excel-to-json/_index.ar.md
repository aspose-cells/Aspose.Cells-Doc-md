##  **كيفية تحويل مصنف Excel إلى JSON**

 لا داعي للتساؤل عن كيفية تحويل Excel Workbook إلى JSON، لأن مكتبة Aspose.Cells Java لديها القرار الأفضل. يوفر Aspose.Cells Java API الدعم لتحويل جداول البيانات إلى تنسيق JSON. لتصدير المصنف إلى JSON قم بالمرور[**حفظ التنسيق.JSON**](https://reference.aspose.com/cells/java/com.aspose.cells/SaveFormat) كمعلمة ثانية[**مصنف.حفظ**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20int) ) طريقة. يمكنك أيضًا استخدام[**JsonSaveOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonSaveOptions) فئة لتحديد إعدادات إضافية لتصدير ورقة العمل إلى JSON.
