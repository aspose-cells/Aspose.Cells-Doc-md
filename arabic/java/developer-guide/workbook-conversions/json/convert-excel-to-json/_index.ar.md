## **تحويل مصنف Excel إلى JSON**

 لا داعي للتساؤل عن كيفية تحويل Excel Workbook إلى JSON ، لأن مكتبة Aspose.Cells Java لديها أفضل قرار. يوفر Aspose.Cells Java API دعمًا لتحويل جداول البيانات إلى تنسيق JSON. لتصدير المصنف إلى JSON ، قم بتمرير[**حفظ تنسيق JSON**](https://reference.aspose.com/cells/java/com.aspose.cells/SaveFormat) كمعامل ثاني لـ[**مصنف**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#save(java.lang.String,%20int) ) طريقة. يمكنك أيضًا استخدام ملفات[**خيارات JsonSave**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonSaveOptions) فئة لتحديد إعدادات إضافية لتصدير ورقة العمل إلى JSON.
