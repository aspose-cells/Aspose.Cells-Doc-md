## **تحويل JSON إلى مصنف Excel**

 لا داعي للتساؤل عن كيفية تحويل JSON إلى ملف Excel ، لأن مكتبة Aspose.Cells Java لديها أفضل قرار. يوفر Aspose.Cells Java API دعمًا لتحويل تنسيق JSON إلى جداول بيانات. يمكنك استخدام[**JsonLoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLoadOptions) فئة لتحديد إعدادات إضافية لاستيراد JSON إلى المصنف.
