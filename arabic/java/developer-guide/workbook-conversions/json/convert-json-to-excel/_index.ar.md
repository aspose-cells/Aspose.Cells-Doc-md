## **تحويل-JSON-إلى-كتاب-عمل-Excel**

لا داعي للتساءل حول كيفية تحويل JSON إلى ملف Excel، لأن مكتبة Aspose.Cells Java تحتوي على أفضل قرار. توفر واجهة برمجة التطبيقات Aspose.Cells Java دعمًا لتحويل التنسيق JSON إلى جداول البيانات. يمكنك استخدام الفئة [**JsonLoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLoadOptions) لتحديد الإعدادات الإضافية لاستيراد JSON إلى دفتر عمل.

يوضح المثال الكودي التالي استيراد JSON إلى دفتر عمل Excel. يرجى الرجوع إلى الكود الخاص بتحويل الملف المصدر إلى ملف xlsx الذي تم إنشاؤه بالكود للإحالة.
