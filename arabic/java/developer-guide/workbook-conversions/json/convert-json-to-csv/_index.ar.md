تدعم Aspose.Cells تحويل JSON البسيط والمتداخل إلى CSV. لهذا الغرض، توفر الواجهة البرمجية النطاقية [**JsonLayoutOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLayoutOptions) و [**JsonUtility**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonUtility). يوفر الفئة [**JsonLayoutOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLayoutOptions) الخيارات لتخطيط JSON مثل [**IgnoreArrayTitle**](https://reference.aspose.com/cells/java/com.aspose.cells/jsonlayoutoptions#IgnoreArrayTitle) (يتجاهل العنوان إذا كان المصفوفة خاصية لكائن) أو [**ArrayAsTable**](https://reference.aspose.com/cells/java/com.aspose.cells/jsonlayoutoptions#ArrayAsTable) (يعالج المصفوفة كجدول). تقوم الفئة [**JsonUtility**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonUtility) بمعالجة JSON باستخدام خيارات التخطيط المضبوطة باستخدام الفئة [**JsonLayoutOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLayoutOptions).

يوضح الكود النموذجي التالي استخدام فئتي [**JsonLayoutOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLayoutOptions) و [**JsonUtility**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonUtility) لتحميل ملف JSON المصدر وإنشاء ملف CSV الناتج.
