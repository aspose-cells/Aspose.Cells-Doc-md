{{% alert color="primary" %}} 

يوفر Aspose.Cells خاصية [Shape.getGeometry().getShapeAdjustValues()](https://reference.aspose.com/cells/java/com.aspose.cells/geometry#ShapeAdjustValues) لإجراء تغييرات على نقاط التعديل مع الأشكال. في واجهة مستخدم Microsoft Excel، تظهر التعديلات كعقد أصفر. على سبيل المثال:

- المستطيل المستدير له تعديل لتغيير القوس
- المثلث له تعديل لتغيير موقع النقطة
- لدى الهرم تعديل لتغيير عرض الجزء العلوي
- السهام لها تعديلين لتغيير شكل الرأس والذيل

سيشرح هذا المقال استخدام [Shape.getGeometry().getShapeAdjustValues()](https://reference.aspose.com/cells/java/com.aspose.cells/geometry#ShapeAdjustValues) خاصية لتغيير قيم التعديل لأشكال مختلفة.

{{% /alert %}}
