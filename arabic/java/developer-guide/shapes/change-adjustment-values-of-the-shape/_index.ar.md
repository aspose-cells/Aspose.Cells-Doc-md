{{% alert color="primary" %}} 

 يوفر Aspose.Cells[Shape.getGeometry (). getShapeAdjustValues ()](https://reference.aspose.com/cells/java/com.aspose.cells/geometry#ShapeAdjustValues) خاصية لإجراء تغييرات على نقاط الضبط مع الأشكال. في Microsoft Excel UI ، يتم عرض التعديلات كعقد ماسية صفراء. فمثلا:

- مستطيل مدور لديه تعديل لتغيير القوس
- المثلث لديه تعديل لتغيير موقع النقطة
- شبه منحرف لديه تعديل لتغيير عرض الجزء العلوي
- تحتوي الأسهم على تعديلين لتغيير شكل الرأس والذيل

 تشرح هذه المقالة استخدام[Shape.getGeometry (). getShapeAdjustValues ()](https://reference.aspose.com/cells/java/com.aspose.cells/geometry#ShapeAdjustValues) خاصية لتغيير قيمة الضبط للأشكال المختلفة.

{{% /alert %}} 
## **تغيير قيم الضبط للشكل**
يصل نموذج التعليمات البرمجية التالي إلى الأشكال الثلاثة الأولى لورقة العمل الأولى في ملف Excel المصدر ثم يغير قيم ضبط الأشكال. تظهر لقطات الشاشة أدناه كيف تبدو الأشكال قبل تغيير قيم الضبط ثم بعد تغيير قيم الضبط.
