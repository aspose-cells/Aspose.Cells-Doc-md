{{% alert color="primary" %}} 

توفر Aspose.Cells خاصية [OleObject.AutoLoad](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad) لتحديث كائن Ole عند فتح ملف Excel في Microsoft Excel. نتيجة لهذه الخاصية، سيعرض كائن Ole الصورة Ole الصحيحة التي تم إنشاؤها بواسطة Microsoft Excel.

{{% /alert %}} 
## **تحديث كائن Ole تلقائيًا عبر Microsoft Excel باستخدام Aspose.Cells**
يقوم الرمز العيني التالي بتحميل [ملف إكسل عيني](5473423.xlsx) الذي يحتوي على صورة Ole غير حقيقية. الكائن Ole في الواقع هو مستند Microsoft Word ولكن ملف Excel العيني يعرض صورة الحيوان بدلاً من صورة Microsoft Word. ولكن إذا قمت بفتح [ملف الإكسل الناتج](5473429.xlsx)، سترى أن Microsoft Excel يعرض الصورة Ole الصحيحة.

توضح اللقطة الشاشة التالية كيف تبدو [ملف إكسل العيني](5473423.xlsx) عند فتحه في Microsoft Excel.
