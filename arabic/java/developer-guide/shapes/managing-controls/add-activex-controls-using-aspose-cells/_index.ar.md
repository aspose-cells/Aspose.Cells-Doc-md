 يمكنك إضافة عناصر تحكم ActiveX باستخدام Aspose.Cells باستخدام ملف[ShapeCollection.addActiveXControl ()](https://reference.aspose.com/cells/java/com.aspose.cells/shapecollection#addActiveXControl\(int,%20int,%20int,%20int,%20int,%20int,%20int\) ) طريقة. هذه الطريقة تأخذ معلمة[نوع التحكم](https://reference.aspose.com/cells/java/com.aspose.cells/ControlType)الذي يوضح نوع عنصر تحكم ActiveX الذي يجب إضافته داخل ورقة العمل. لديها القيم التالية.

- [CHECK_BOX](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#CHECK_BOX)
- [صندوق التحرير](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#COMBO_BOX)
- [زر الأوامر](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#COMMAND_BUTTON)
- [صورة](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#IMAGE)
- [ضع الكلمة المناسبة](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#LABEL)
- [LIST_BOX](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#LIST_BOX)
- [زر الراديو](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#RADIO_BUTTON)
- [SCROLL_BAR](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#SCROLL_BAR)
- [SPIN_BUTTON](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#SPIN_BUTTON)
- [مربع الكتابة](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#TEXT_BOX)
- [TOGGLE_BUTTON](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#TOGGLE_BUTTON)
- [مجهول](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#UNKNOWN)

 بمجرد إضافة عنصر تحكم ActiveX داخل مجموعة الأشكال ، يمكنك بعد ذلك الوصول إلى كائن عنصر تحكم ActiveX عبر[الشكل](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ActiveXControl) ثم قم بتعيين خصائصها المختلفة.
