## **سيناريوهات الاستخدام المحتملة**
تقدم Aspose.Cells خاصية اللمعان لشكل أو مخطط. تحتوي على الخصائص الفرعية التالية التي يمكنك ضبطها لتحقيق نتائج مختلفة وفقًا لمتطلباتك.

- GlowEffect.Size
- GlowEffect.Transparency

اللقطة الشاشة التالية توضح واجهة Microsoft Excel لتعيين تأثير اللمعان للشكل.
