## **الحصول على مصدر بيانات الاتصال الخارجي للجدول المحوري**

يوفر Aspose.Cells القدرة على الحصول على مصدر بيانات الاتصال الخارجي للجدول المحوري. لهذا ، يوفر API الامتداد[**ExternalConnection DataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)ممتلكات[**جدول محوري**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable)صف دراسي. ال[**ExternalConnection DataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)خاصية إرجاع ال[**اتصال خارجي**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection)موضوع. يوضح مقتطف الشفرة التالي استخدام ملف[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) الخاصية للحصول على مصدر بيانات الاتصال الخارجي للجدول المحوري.
