## **الحصول على مصدر بيانات الاتصال الخارجي لجدول الدوران**

Aspose.Cells توفر القدرة على الحصول على مصدر بيانات الاتصال الخارجي لجدول الدوران. لهذا الغرض، يوفر الواجهة البرمجية تفاصيل الاستخدام لخاصية [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) لفئة [**PivotTable**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable). تعيد الخاصية [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) كائن [**ExternalConnection**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection). توضح مقتطفات الشفرة التالية استخدام الخاصية [**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) للحصول على مصدر بيانات الاتصال الخارجي لجدول الدوران.
