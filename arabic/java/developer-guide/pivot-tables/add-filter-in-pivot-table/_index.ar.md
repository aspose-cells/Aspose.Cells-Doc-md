## **سيناريوهات الاستخدام الممكنة**
 عندما تقوم بإنشاء جدول محوري ببيانات معروفة وتريد تصفية الجدول المحوري ، فإنك تحتاج إلى تعلم واستخدام عامل التصفية. يمكن أن يساعدك على تصفية البيانات التي تريدها بشكل فعال. باستخدام Aspose.Cells Java API ، يمكنك إضافة عامل تصفية على قيم الحقول في الجداول المحورية.

## **أضف عامل تصفية في Pivot Table**
يرجى الاطلاع على نموذج التعليمات البرمجية التالي. يقوم بتعيين البيانات وإنشاء PivotTable بناءً عليها. ثم أضف عامل تصفية في حقل الصف في الجدول المحوري. أخيرًا ، يحفظ المصنف بتنسيق[إخراج XLSX](out.xlsx) صيغة. بعد تنفيذ رمز المثال ، تتم إضافة جدول محوري بفلتر top10 إلى ورقة العمل.
