## **فرز مخصص في Pivot Table**
باستخدام Aspose.Cells API ، يمكنك فرز الجداول المحورية على قيم الحقول. يقوم مقتطف الشفرة التالي بتحميل نموذج ملف Excel ويضيف ثلاثة جداول محورية. الجدول المحوري الأول بدون فرز مخصص ، ويتم فرز الجدول المحوري الثاني على قيم حقل صف "SeaFood" ويتم فرز الجدول المحوري الثالث على قيم حقل العمود "28/07/2000".
