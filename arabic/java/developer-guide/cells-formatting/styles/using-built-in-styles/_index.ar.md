يوفر Aspose.Cells مجموعة كبيرة من الأنماط التي يمكن إعادة استخدامها لتنسيق خلية في وثيقة جدول حسابي. يمكننا استخدام الأنماط المضمنة في مصنفنا وأيضًا إنشاء أنماط مخصصة.

{{% /alert %}}
