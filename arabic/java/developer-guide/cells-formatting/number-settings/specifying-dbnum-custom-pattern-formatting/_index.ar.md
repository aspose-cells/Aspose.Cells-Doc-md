## **سيناريوهات الاستخدام المحتملة**

يدعم Aspose.Cells تنسيق نمط *DBNum* المخصص. على سبيل المثال، إذا كانت قيمة الخلية الخاصة بك 123 وحددت تنسيقها المخصص ك [DBNum2][$-804]General فسيتم عرضها كما يلي:  壹佰贰拾叁. يمكنك تحديد تنسيق الخلية المخصص باستخدام طرق [**Cell.getStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getStyle--) و [**Style.setCustom()**](https://reference.aspose.com/cells/java/com.aspose.cells/style/#setCustom-java.lang.String-).
