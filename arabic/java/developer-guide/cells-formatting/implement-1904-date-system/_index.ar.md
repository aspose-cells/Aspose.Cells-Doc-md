{{% alert color="primary" %}} 

Microsoft يدعم Excel نظامين للتاريخ: نظام التاريخ 1900 (نظام التاريخ الافتراضي المطبق في Excel لـ Windows) ونظام التاريخ 1904. يتم استخدام نظام التاريخ 1904 عادةً لتوفير التوافق مع ملفات Macintosh Excel وهو النظام الافتراضي إذا كنت تستخدم Excel لنظام التشغيل Macintosh. يمكنك ضبط نظام التاريخ 1904 لملفات Excel باستخدام Aspose.Cells.

{{% /alert %}} 

لتنفيذ نظام التاريخ 1904 في Microsoft Excel (على سبيل المثال Microsoft Excel 2003):

1.  من**أدوات** القائمة ، حدد**خيارات** ، وحدد ملف**عملية حسابية** التبويب.
1.  حدد ملف**1904 نظام التاريخ** اختيار.
1.  انقر**نعم**.
