تتيح Aspose.Cells للمستخدمين استخراج البيانات ذات الصلة بالنسق من ملف Excel. على سبيل المثال، يمكنك استخراج اسم النسق المطبق على دفتر العمل ولون النسق المطبق على الخلية أو حدود الخلية، إلخ.

يمكنك تطبيق نسق على دفتر العمل الخاص بك باستخدام Microsoft Excel عبر الأمر تخطيط الصفحة > النسق.
