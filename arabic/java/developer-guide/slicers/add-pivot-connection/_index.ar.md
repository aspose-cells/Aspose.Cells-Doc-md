## **سيناريوهات الاستخدام الممكنة**

 إذا كنت تريد ربط أداة تقطيع الشرائح والجدول المحوري في Excel ، فستحتاج إلى النقر بزر الماوس الأيمن فوق أداة تقطيع الشرائح وتحديد عنصر "تقرير اتصالات ...". في قائمة الخيارات ، يمكنك العمل في خانة الاختيار. وبالمثل ، إذا كنت ترغب في إقران أداة تقطيع الشرائح والجدول المحوري باستخدام Aspose.Cells Java API برمجيًا ، فيرجى استخدام[**Slicer.addPivotConnection (PivotTable pivot)**](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#addPivotConnection(com.aspose.cells.PivotTable)/) طريقة. سيتم ربط القطاعة والجدول المحوري.

## **مقسم مقسم وجدول PivotTable**

يقوم نموذج التعليمات البرمجية التالي بتحميل ملف[نموذج لملف Excel](add-pivot-connection.xlsx)الذي يحتوي على مقسم شرائح موجود. يصل إلى Slicer ثم يقرن Slicer و PivotTable. أخيرًا ، يحفظ المصنف باسم[إخراج ملف Excel](add-pivot-connection-out.xlsx).
