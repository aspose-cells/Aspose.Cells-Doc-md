Microsoft يمكن لمستخدمي Excel إضافة تعليقات إلى الخلايا في جداول البيانات لإضافة معلومات إضافية إلى الخلايا وتمييز البيانات. قد يحتاج المطورون إلى تخصيص التعليق لتحديد إعدادات المحاذاة واتجاه النص. يوفر Aspose.Cells واجهات برمجة التطبيقات (API) لإنجاز المهمة.

{{% /alert %}}

 يوفر Aspose.Cells أ[**Shape.setTextDirection ()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#TextDirection)طريقة لتعيين اتجاه النص المطلوب للتعليق.

توضح الصورة التالية التعليق بإعدادات التنسيق الخاصة به في Microsoft Excel. يتم الحصول على ملف الإخراج بعد تشغيل مقطع التعليمات البرمجية أدناه.
