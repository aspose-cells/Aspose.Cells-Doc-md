{{% alert color="primary" %}}

يمكن لمستخدمي Microsoft Excel إضافة تعليقات إلى الخلايا في جداول البيانات لإضافة معلومات إضافية إلى الخلايا وتسليط الضوء على البيانات. قد يحتاج المطورون إلى تخصيص التعليق لتحديد إعدادات المحاذاة واتجاه النص. توفر Aspose.Cells واجهات برمجة التطبيقات لإنجاز المهمة.

{{% /alert %}}

تقدم Aspose.Cells [**Shape.setTextDirection()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#TextDirection) طريقة لتحديد اتجاه النص المطلوب لتعليق.

الصورة التالية تظهر التعليق مع إعدادات الصيغة الخاصة به في Microsoft Excel. يتم الحصول على ملف الإخراج بعد تشغيل الشريط الكودي أدناه.
