## **سيناريوهات الاستخدام المحتملة**

عمومًا، يتم تحديد دليل الخطوط أو قائمة الخطوط لجميع الدفاتر ولكن في بعض الأحيان، يجب عليك تحديد مجموعة منفردة أو خاصة من الخطوط لدفاتر العمل الخاصة بك. توفر Aspose.Cells [**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs) تصنيف يمكن استخدامه لتحديد مجموعة الخطوط الفردية أو الخاصة لدفتر العمل الخاص بك.

## **تحديد مجموعة خطوط فردية أو خاصة لتقديم الدفتر**

يقوم الكود العيني التالي بتحميل [ملف إكسل عيني](67338304.xlsx) مع مجموعته الفردية أو الخاصة من الخطوط التي تم تحديدها باستخدام تصنيف [**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs). يرجى الاطلاع على [الخط العيني](67338302.zip) المستخدم داخل الكود وكذلك [الملف PDF الناتج](67338303.pdf) المولد به. تُظهر اللقطة الشاشية التالية كيف تبدو الملف PDF الناتج إذا تم العثور على الخط بنجاح.
