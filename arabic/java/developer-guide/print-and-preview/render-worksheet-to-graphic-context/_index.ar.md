{{% alert color="primary" %}} 

Aspose.Cells يمكنه الآن عرض ورقة العمل لسياق بياني. يمكن أن يكون سياق الرسوم أي شيء مثل ملف الصورة أو الشاشة أو الطابعة وما إلى ذلك. يرجى اتباع الطريقة التالية لتقديم ورقة العمل إلى سياق رسومي.

- SheetRender.toImage (int pageIndex ، Graphics2D Graphic)

{{% /alert %}} 
## **تقديم ورقة العمل إلى سياق رسومي**
 يوضح الكود التالي كيفية استخدام Aspose.Cells لتقديم ورقة العمل إلى سياق رسومي. بمجرد تنفيذ التعليمات البرمجية ، ستقوم بطباعة ورقة العمل بأكملها وملء المساحة الفارغة المتبقية باللون الأزرق في سياق الرسومات وحفظ الصورة باسم**test.png** ملف. يمكنك استخدام أي ملف اكسل المصدر لتجربة هذا الكود. يرجى أيضًا قراءة التعليقات الموجودة داخل الكود لفهم أفضل.
