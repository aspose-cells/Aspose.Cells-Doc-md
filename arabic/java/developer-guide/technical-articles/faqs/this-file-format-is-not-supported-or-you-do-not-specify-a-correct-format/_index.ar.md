## **هذا التنسيق غير مدعوم أو أنك لم تحدد تنسيقًا صحيحًا**
إذا قام المستخدم بتحديد صيغة الملف عند إنشاء دفتر عمل من قالب ملف، وغالبًا ما تكون هذه الخطأ بسبب عدم تطابق صيغة الملف المحددة مع صيغة ملف القالب الفعلية. إذا لم يقم المستخدم بتحديد صيغة الملف، وغالبًا ما يكون السبب هو أن امتداد اسم الملف لا يمثل صيغة الملف الفعلية لهذا الملف، ولا يمكن اكتشاف صيغة الملف تلقائيًا، مثل ملف CSV/TSV الذي لا يحتوي على معرفات خاصة. بالطبع، ستبلغ أيضًا عن هذا الخطأ صيغة الملفات غير المدعومة من قبل Cells.
