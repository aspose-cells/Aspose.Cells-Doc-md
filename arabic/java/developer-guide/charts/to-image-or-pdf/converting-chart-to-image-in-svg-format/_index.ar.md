{{% alert color="primary" %}} 

Scalable Vector Graphics (SVG) هو تنسيق صورة متجه مستند إلى XML للرسومات ثنائية الأبعاد التي تدعم أيضًا التفاعل والرسوم المتحركة. مواصفات SVG هي معيار مفتوح تم تطويره بواسطة World Wide Web Consortium (W3C) منذ 1999.

يتم تعريف صور SVG وسلوكياتها في ملفات نصية بتنسيق XML. هذا يعني أنه يمكن البحث عنها وفهرستها وبرمجتها وضغطها. كملفات XML ، يمكن إنشاء صور SVG وتحريرها باستخدام أي محرر نصوص ، ولكن غالبًا ما يتم إنشاؤها باستخدام برامج الرسم.

يمكن لـ Aspose.Cells حفظ الرسوم البيانية كصور بتنسيقات مختلفة مثل BMP و JPEG و PNG و GIF و SVG وما إلى ذلك. تشرح هذه المقالة كيفية حفظ المخططات كصور SVG.

{{% /alert %}} 

يوضح نموذج التعليمات البرمجية التالي كيفية استخدام Aspose.Cells لتحويل مخطط إلى صورة بتنسيق SVG. يقوم الكود بتحميل ملف Excel المصدر ثم يحفظ المخطط الأول الموجود في ورقة العمل الأولى إلى SVG.

تُظهر لقطة الشاشة التالية صورة المخطط المحولة بتنسيق SVG الذي تم إنشاؤه باستخدام نموذج التعليمات البرمجية.
