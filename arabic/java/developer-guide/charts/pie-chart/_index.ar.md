## **إنشاء مخطط دائري مع خطوط قيادة**

لتوضيح استخدام واجهة برمجة التطبيقات Aspose.Cells for Java لإنشاء رسم بياني دائري مع خطوط القائد، سنقوم أولاً بإنشاء تصنيف جديد وإدخال بيانات تعمل كمصدر بيانات السلسلة. بمجرد وضع البيانات في المكان المناسب، سنضيف تخطيطًا من نوع [**Pie**](https://reference.aspose.com/cells/java/com.aspose.cells/charttype#PIE) إلى تجميع الرسوم البيانية ونقوم بضبط جوانبه المختلفة للحصول على رؤية الرسم البياني المرغوبة.
