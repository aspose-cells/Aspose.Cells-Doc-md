## **سيناريوهات الاستخدام الممكنة**
الإصدارات المبكرة من Aspose.Cells لم تكن قادرة على التعامل مع الوحدات الآلية لمحور المخطط بشكل صحيح عندما تم تقديم المخطط للصورة أو PDF. الآن ، يدعم Aspose.Cells معالجة الوحدات الآلية لمحور المخطط. لا يوجد تغيير في الكود. فقط قم بتحويل المخطط الخاص بك إلى صورة أو PDF وسوف يعرض محور المخطط تمامًا مثل Microsoft يقوم Excel بعرضه.
## **التعامل مع الوحدات التلقائية لمحور المخطط مثل Microsoft Excel**
يقوم نموذج التعليمات البرمجية التالي بتحميل ملف[نموذج لملف Excel](61767769.xlsx)ويولد ال[إخراج PDF الرسم البياني](61767768.pdf)تُظهر لقطة الشاشة الوحدات التلقائية لمحور المخطط في مستطيلات حمراء وتقارن أيضًا نموذج مخطط ملف Excel مع مخطط الإخراج PDF. كلاهما متشابهان تمامًا.
