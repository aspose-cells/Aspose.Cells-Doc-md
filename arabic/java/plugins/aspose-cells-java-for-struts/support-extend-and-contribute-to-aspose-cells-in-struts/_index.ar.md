## **الدعم**
- إذا كنت ترغب في رؤية المشكلات المعروفة / المبلغ عنها (من قبل المستخدمين أو فريق ضمان الجودة) في التطبيق.
- أو تريد الإبلاغ عن أي مشكلة وجدت في التطبيق
- لديك أي اقتراح تحسين أو ترغب في تقديم أي طلب ميزة

**الرجاء استخدام أي من أدوات تعقب مشكلات المشروع التالية:**

- [جيثب](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **تمديد والمساهمة**
Aspose.cells Java لـ Struts 1.3 هو مصدر مفتوح وكود المصدر الخاص به متاح على مواقع الترميز الاجتماعي الرئيسية المدرجة أدناه. يتم تشجيع المطورين على تنزيل الكود المصدري والمساهمة من خلال اقتراح أو إضافة ميزة جديدة أو تحسين الميزات الموجودة ، بحيث يمكن للآخرين الاستفادة منها أيضًا.
