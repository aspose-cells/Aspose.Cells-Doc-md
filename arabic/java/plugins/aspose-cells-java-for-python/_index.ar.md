## **JPype**

JPype هو جهد للسماح لبرامج Python بالوصول الكامل إلى مكتبات فئة Java. يتم تحقيق ذلك ليس من خلال إعادة تنفيذ Python، كما قام Jython/JPython بعمله، ولكن من خلال التفاعل على المستوى الأساسي في كل من أجهزة الكمبيوتر الظاهرية.

اقرأ المزيد على [readthedocs.org](https://jpype.readthedocs.io/en/latest/userguide.html)

## **Aspose.Cells for Java**

Aspose.Cells for Java هو مكتبة فئة متقدمة لـ Java تتيح لك أداء نطاق كبير من المهام الخاصة بمعالجة المستندات مباشرة ضمن تطبيقات Java الخاصة بك.

Aspose.Cells for Java يدعم معالجة ملفات Excel (XLS، XLSX، XLSB، XLSM، XLT، XLTX، XLTM)، CSV (قيم مفصولة بفاصلة)، TSV (قيم مفصولة بعلامة التبويب)، TabDelimited (ملف نصي مفصول بشريط الألواح)، Html، MHtml، ODS (جدول بيانات OpenDocument)، SpreadsheetML، وأرقام (المستند الذي تم إنشاؤه بواسطة تطبيق "Numbers" الخاص بـ آبل). باستخدام Aspose.Cells يمكن لك إنشاء، تعديل، وتحويل المستندات دون الاعتماد على Microsoft Excel.

## **Aspose.Cells Java for Jython**

Aspose.Cells Java for Python هو مشروع يوضح / يوفر أمثلة على استخدام واجهة برمجة التطبيقات Aspose.Cells for Java في Python.

## **متطلبات النظام**

**فيما يلي متطلبات النظام لاستخدام Aspose.Cells Java for Python:**

- Jpype
- تنزيل مكون Aspose.Cells.

**إليك المنصات المدعومة:**
