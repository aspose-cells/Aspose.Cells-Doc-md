## **كيفية تكوين شفرة المصدر**
يرجى اتباع هذه الخطوات البسيطة لفتح كود المصدر وتوسيعه أثناء استخدام:

 \ 1. قم بتثبيت Python 2.7.xx فقط من:<https://www.python.org/downloads/>
 \ 2. قم بتثبيت JPype من:<https://sourceforge.net/projects/jpype/files/JPype/>
 \ 3. قم بتثبيت Java JDK من:<https://www.oracle.com/java/technologies/downloads/>
 \ 4. قم بتنزيل Aspose.Cells for Java APIs التي تريد استخدامها في مشاريع Python الخاصة بك من:<https://downloads.aspose.com/cells/java> وانسخ ملفات .jars الخاصة بواجهة برمجة التطبيقات التي تم تنزيلها إلى مجلد٪ Python٪ / lib.
 \ 5. استنساخ[Aspose.Cells Java للحصول على Python أمثلة](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) من جيثب.
