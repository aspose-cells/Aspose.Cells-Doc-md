## **مقدمة**

### **IntelliJ IDEA**

**IntelliJ IDEA** هو بيئة تطوير متكاملة (IDE) لتطوير البرمجيات بلغة الجافا. تم تطويره بواسطة JetBrains. تم إنشاؤه أصلاً للجافا وحالياً يدعم مجموعة متنوعة واسعة من اللغات، بما في ذلك جافا سكريبت، جروفي، HTML، CSS، RSS، R، هاسكيل، PHP، روبي، بايثون، سكالا، كلوجور، وكوتلن.

في تقرير **Infoworld** لعام 2010، حصلت IntelliJ على أعلى درجة في اختبار المركز من بين أربعة أدوات رئيسية لبرمجة الجافا: Eclipse، IntelliJ IDEA، NetBeans و Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) للجافا هو مكون تقرير جداول بيانات Excel® الذي يتيح لتطبيقات الجافا إنشاء وإدارة أوراق بيانات Excel® دون استخدام Microsoft Excel®. Aspose.Cells for Java هو مكون يوفر ميزات غنية جداً تقدم أكثر من مجرد وظائف تصدير البيانات الأساسية.

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java لـ IntelliJ IDEA (Maven) هو **مكون IntelliJ IDEA** يتيح لك إنشاء مشاريع maven (أي المشروع الذي تم إنشاؤه يحتوي على Aspose.Cells for Java API تبعية maven) وتنزيل أمثلة لشفرة المصدر للواجهة البرمجية لاستخدام Aspose.Cells for Java.

- يساعدك هذا المكون المساعد في العمل مع واجهة برمجة التطبيقات Aspose.Cells for Java داخل IntelliJ IDEA المفضلة لديك.

## **متطلبات النظام والمنصات المدعومة**

### **متطلبات النظام**

- مايكروسوفت ويندوز 8/7/Vista/2003/XP (بما في ذلك 64 بت) / سطح المكتب لـ Linux GNOME أو KDE / نظام التشغيل Mac OS X 10.5 أو أعلى
- الحد الأدنى لذاكرة الوصول العشوائي 1 جيجابايت، الحد الأدنى لذاكرة الوصول العشوائي الموصى به 2 جيجابايت
- 300 ميجابايت من مساحة القرص الصلب + ما لا يقل عن 1 جيجابايت للخزانات
- الحد الأدنى لدقة الشاشة 1024x768
- JDK 1.7 أو أعلى الموصى به
- IntelliJ IDEA CE/UE (13.0.1، 13.1.2 أو 14.0.2)
- اتصال بالإنترنت (يُوصى بسرعة 2 ميجابت أو أكبر)

### **المنصات المدعومة**

يدعم المكون الإضافي الإصدارات التالية من IntelliJ IDEA:

- 13.0.1 (النسخة المجتمعية / النسخة المتطورة)
- 13.1.2 (CE / UE)
- 14.0.2 (CE/ UE)

## **التحميل**

### **تحميل IntelliJ IDEA**

ستحتاج أولاً إلى تثبيت IntelliJ IDEA CE / UE قبل تنزيل مكون **Aspose.Cells Java for IntelliJ IDEA (Maven)**.

تحميل IntellliJ IDEA

1. Go to <https://www.jetbrains.com/idea/download/>
1. حدد نظام التشغيل الخاص بك
1. انقر على تحميل UE أو CE.

### **تحميل Aspose.Cells Java لبرنامج IntelliJ IDEA (Maven)**

- يمكن تنزيل المكوِّن المساعد مباشرة من [عنوان URL للإضافة في JetBrains](https://goo.gl/R4pysl) - على مستودع إضافات IntelliJ IDEA
  ثم يمكن تثبيته باستخدام الخيارات في البيئة التطويرية: 
  - اذهب إلى **ملف** > **الإعدادات**...>**الإضافات**
  - حدد "**تثبيت إضافة من القرص**"
- أو يمكن تثبيته مباشرة (دون تنزيل يدوي من عنوان URL للإضافة في JetBrains) باستخدام الخيارات في البيئة التطويرية: 
  - اذهب إلى **ملف** > **الإعدادات**...>**الإضافات**
  - حدد **"تصفح المستودعات"**
  - في مربع البحث اكتب: "Aspose"
  - حدد "**Aspose Cells Java for IntelliJ IDEA Maven**" لتثبيت المكوِّن المساعد.
