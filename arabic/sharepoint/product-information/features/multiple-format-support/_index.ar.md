{{% alert color="primary" %}}

يمكنك تحويل المستندات بين العديد من تنسيقات المستندات المكتبية الشهيرة باستخدام Aspose.Cells for SharePoint من خلال مكتبة مستندات SharePoint. يتم إجراء معظم التحويلات بدقة ودقة عالية.

{{% /alert %}}

## **تنسيقات الإدخال المدعومة**

يدعم Aspose.Cells for SharePoint التنسيقات الإدخال التالية:

- XLS – Microsoft Excel 97~2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- CSV – مستند ورقة بيانات القيم المفصولة بواسطة الفواصل
- ODS – OpenDocument
- XML – Microsoft Excel SpreadsheetML

لإنشاء المستندات، يعتمد Aspose.Cells for SharePoint على نسخة مدمجة من [Aspose.Cells for .NET](https://products.aspose.com/cells/net/)، وهي عنصر معالجة جداول البيانات الرائد في السوق من Aspose. لا يتم استخدام Microsoft Excel لإنشاء المستندات.

## **تنسيقات الإخراج المدعومة**

القائمة الكاملة لتنسيقات الإخراج المدعومة بواسطة Aspose.Cells for SharePoint هي:

- PDF – مستند محمول من أدوبي
- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM – Office Open XML for macros enabled
- TXT – Plain Text
- XML – Microsoft Excel SpreadsheetML
- ODS – OpenDocument
- CSV – Comma Separated Values
- XLSB – Microsoft Excel Binary 2007
- MHTML – Multipurpose Internet Mail Extension HTML
- XPS – XML Paper Specification
- TIFF – Tagged Image File Format
- SVG – Scalable Vector Graphics
- DIF – Drawing Interchange Format
