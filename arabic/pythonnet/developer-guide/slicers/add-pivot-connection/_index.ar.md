##  **سيناريوهات الاستخدام المحتملة**

إذا كنت تريد ربط مقسم البيانات والجدول المحوري في Excel، فستحتاج إلى النقر بزر الماوس الأيمن فوق مقسم البيانات وتحديد العنصر "الإبلاغ عن الاتصالات...". في قائمة الخيارات، يمكنك العمل على خانة الاختيار. وبالمثل، إذا كنت تريد ربط أداة تقطيع البيانات والجدول المحوري باستخدام Aspose.Cells for Python via .NET API برمجيًا، فيرجى استخدام[**Slicer.add_pivot_connection(محور)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/add_pivot_connection/#aspose.cells.pivot.PivotTable)طريقة. سيتم ربط أداة التقطيع والجدول المحوري.

##  **أداة تقطيع اللحم وPivotTable المرتبطة**

يقوم نموذج التعليمات البرمجية التالي بتحميل ملف[عينة من ملف إكسل](add-pivot-connection.xlsx)الذي يحتوي على أداة تقطيع موجودة. فهو يصل إلى Slicer ثم يقوم بربط Slicer وPivotTable. وأخيرا، فإنه يحفظ المصنف باسم[إخراج ملف إكسل](add-pivot-connection-out.xlsx).
