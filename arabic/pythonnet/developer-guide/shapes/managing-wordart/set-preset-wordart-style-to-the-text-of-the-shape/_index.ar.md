## **سيناريوهات الاستخدام المحتملة**
يمكنك تعيين نمط WordArt معد مسبقًا لنص الشكل باستخدام Aspose.Cells for Python via .NET. يرجى استخدام طرق [FontSetting.set_word_art_style()](https://reference.aspose.com/cells/python-net/aspose.cells/fontsetting/set_word_art_style/#aspose.cells.drawing.PresetWordArtStyle) أو [FontSettingCollection.set_word_art_style()](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/fontsettingcollection/set_word_art_style/#aspose.cells.drawing.PresetWordArtStyle) لهذا الغرض.

## **تعيين نمط WordArt المحدد مسبقًا لنص الشكل**
الشيفرة النموذجية التالية تنشئ مربع نص يحتوي على بعض النصوص ثم تعيين نمط WordArt معد مسبقًا لنصه باستخدام دالة [FontSetting.set_word_art_style()](https://reference.aspose.com/cells/python-net/aspose.cells/fontsetting/set_word_art_style/#aspose.cells.drawing.PresetWordArtStyle). هكذا يبدو ملف إكسل الناتج في Microsoft Excel.
