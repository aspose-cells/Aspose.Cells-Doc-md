## **سيناريوهات الاستخدام المحتملة**

أحيانًا تريد عرض النص بخط لغة الشرق الأقصى مثل اليابانية والصينية والتايلاندية، الخ. يوفر Aspose.Cells for Python via .NET خاصية [**TextOptions.far_east_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/far_east_name) التي يمكن استخدامها لتحديد اسم الخط للغات الشرق الأقصى. بالإضافة إلى ذلك، يمكنك أيضًا تحديد اسم خط اللاتينية باستخدام خاصية [**TextOptions.latin_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/latin_name).

## **تحديد اسم الشرق الأقصى واللاتيني للخط في خيارات النص للشكل**

يقوم الكود عينة التالي بإنشاء مربع نص ويضيف بعض النص الياباني داخله. ثم يُحدد أسماء الخط اللاتيني والشرق الأقصى للنص ويحفظ الدفتر كـ [ملف Excel الناتج](67338274.xlsx). تُظهر اللقطة الشاشية التالية أسماء الخط اللاتيني والشرق الأقصى لمربع النص الناتج في Microsoft Excel.
