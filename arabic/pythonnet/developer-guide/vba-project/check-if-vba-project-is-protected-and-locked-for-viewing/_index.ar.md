## التحقق مما إذا كان مشروع VBA محميًا ومقفلًا للمشاهدة في بايثون

يسمح Aspose.Cells لبايثون via .NET لك بالتحقق مما إذا كان مشروع VBA (Visual Basic for Applications) في ملف إكسل محميًا ومقفلًا للمشاهدة. لهذا، توفر واجهة برمجة التطبيقات الخاصية [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/). إذا كان مقفلًا للمشاهدة، فإن الخاصية [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/) تعيد **true**.
