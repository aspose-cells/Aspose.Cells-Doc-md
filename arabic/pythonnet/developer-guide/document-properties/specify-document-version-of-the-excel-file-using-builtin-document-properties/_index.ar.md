## **سيناريوهات الاستخدام المحتملة**

يمكنك تغيير **رقم الإصدار** لملف إكسل بالنقر بزر الماوس الأيمن على الملف واختيار خصائص > تفاصيل ثم تحرير حقل **رقم الإصدار**. يرجى استخدام خاصية [**BuiltInDocumentPropertyCollection.document_version**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/document_version) لتغييره برمجياً باستخدام APIs لـ Aspose.Cells لـ Python via .NET.
