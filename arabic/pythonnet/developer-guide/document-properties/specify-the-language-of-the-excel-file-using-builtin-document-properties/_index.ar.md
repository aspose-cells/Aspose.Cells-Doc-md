## **سيناريوهات الاستخدام المحتملة**

يمكنك تغيير **اللغة** لملف إكسل بالنقر بزر الماوس الأيمن على الملف واختيار خصائص > تفاصيل ثم تحرير حقل اللغة. يرجى استخدام الخاصية [**BuiltInDocumentPropertyCollection.language**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/language) لتغييره برمجياً باستخدام APIs لـ Aspose.Cells لـ Python via .NET.

## **تحديد لغة ملف إكسل باستخدام الخصائص المدمجة للمستند**

الشيفرة المثالية التالية تنشئ دفتر عمل وتغير الخاصية المدمجة للمستند بإسم اللغة. يرجى مشاهدة [ملف Excel الناتج](64716891.xlsx) من خلال الشيفرة ولقطة شاشة تُظهر الحقل اللغوي المعدل بواسطة [**BuiltInDocumentPropertyCollection.language**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/language) خاصية.
