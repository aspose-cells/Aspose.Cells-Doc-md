Scalable Vector Graphics (SVG) هو تنسيق صور متجه يستند إلى XML للرسومات ثنائية الأبعاد ويدعم أيضًا التفاعل والرسوم المتحركة. مواصفات SVG هي معيار مفتوح تم تطويره بواسطة اتحاد شبكة الويب العالمية (W3C) منذ عام 1999.

يتم تعريف SVG صورة وسلوكياتها في ملفات نصية بتنسيق XML. وهذا يعني أنه يمكن البحث عنها وفهرستها وكتابتها وضغطها. كملفات XML، يمكن إنشاء SVG صورة وتحريرها باستخدام أي محرر نصوص، ولكن غالبًا ما يتم إنشاؤها باستخدام برامج الرسم.

Aspose.Cells for Python via .NET يمكنه حفظ الرسم البياني في الصور بتنسيقات مختلفة مثل BMP، JPEG، PNG، GIF، SVG وما إلى ذلك. تشرح هذه المقالة كيفية حفظ الرسم البياني بتنسيق SVG.
