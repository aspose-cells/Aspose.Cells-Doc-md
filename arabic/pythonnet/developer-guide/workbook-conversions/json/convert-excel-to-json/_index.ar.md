## **تحويل دفتر العمل Excel إلى JSON**

لا داعي للتساؤل حول كيفية تحويل ورقة العمل في اكسل إلى ملف JSON، لأن مكتبة Aspose.Cells لـ Python via .NET لديها أفضل قرار. توفر واجهة برمجة التطبيقات Aspose.Cells دعم لتحويل جداول البيانات إلى تنسيق JSON. لتصدير ورقة العمل إلى JSON، قم بتمرير [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) كالمعلمة الثانية لطريقة [**Workbook.save**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions). يمكنك أيضًا استخدام فئة [**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) لتحديد الإعدادات الإضافية لتصدير ورقة العمل إلى JSON.
