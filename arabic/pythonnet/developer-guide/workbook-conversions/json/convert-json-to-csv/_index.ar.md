## **تحويل JSON إلى CSV**

Aspose.Cells for Python via .NET يدعم تحويل JSON بسيط وأيضًا JSON متداخل إلى CSV. لهذا، توفر الواجهة البرمجية [**JsonLayoutOptions**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions) و [**JsonUtility**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonutility) فئات. الفئة [**JsonLayoutOptions**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions) توفر الخيارات لتخطيط JSON مثل [**ignore_array_title**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions/ignore_array_title/)(يتجاهل العنوان إذا كان المصفوفة خاصية لكائن) أو [**array_as_table**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions/array_as_table/)(يعالج المصفوفة كجدول). الفئة [**JsonUtility**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonutility) تعالج JSON باستخدام خيارات التخطيط المُعينة بالفئة [**JsonLayoutOptions**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions).

يُظهر الكود العيني التالي استخدام واجهة البرمجة [**JsonLayoutOptions**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions) وواجهة البرمجة [**JsonUtility**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonutility) لتحميل [ملف JSON المصدر](104398869.json) وتوليد [ملف CSV الناتج](104398870.csv).
