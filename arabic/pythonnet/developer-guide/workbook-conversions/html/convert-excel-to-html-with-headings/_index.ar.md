## **تحويل Excel إلى HTML مع عناوين**

توفر Aspose.Cells for Python via NET الخيار لتصدير الرؤوس أثناء تحويل Excel إلى HTML. لهذا الغرض ، يمكنك ضبط خاصية الـ [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) على **true**. القيمة الافتراضية لخاصية [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) هي **false**. تُظهر الصورة التالية الملف الناتج الذي تم إنشاؤه بواسطة الكود التالي.
