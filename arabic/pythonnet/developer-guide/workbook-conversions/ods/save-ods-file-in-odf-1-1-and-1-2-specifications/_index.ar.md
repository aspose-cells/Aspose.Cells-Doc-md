تدعم Aspose.Cells for Python via .NET حفظ ملف ODS (جدول بيانات OpenDocument) في مواصفات ODF (شكل مستند مفتوح) 1.1 و 1.2. Aspose.Cells for Python via .NET لديها خاصية 0 تحدد استخدام مواصفات ODF 1.1 لحفظ ملفات ODS. القيمة الافتراضية لهذه الخاصية هي خاطئة، لذا يتم حفظ ملف ODS دون هذا الإعداد يستخدم مواصفات 1.2.

{{% /alert %}}
