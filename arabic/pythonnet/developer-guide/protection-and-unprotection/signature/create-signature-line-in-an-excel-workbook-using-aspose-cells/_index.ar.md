## **مقدمة**

توفر Microsoft Excel ميزة إضافة **سطر توقيع** في سجل Excel. يمكنك إضافة سطر توقيع عن طريق النقر فوق علامة **إدراج** ثم تحديد **سطر توقيع** من مجموعة **نص**.

## **كيفية إنشاء خط توقيع لإكسل**

يوفر Aspose.Cells for Python via .NET أيضًا هذه الميزة وقد كشف عن الخاصية [**Picture.signature_line**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/picture/signature_line) لهذا الغرض. ستشرح هذه المقالة كيفية استخدام هذه الخاصية لإضافة خط توقيع باستخدام Aspose.Cells for Python via .NET.
