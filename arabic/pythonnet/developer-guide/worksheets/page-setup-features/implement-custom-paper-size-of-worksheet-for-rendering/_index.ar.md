## **سيناريوهات الاستخدام المحتملة**

لا توجد خيارات مباشرة متاحة لإنشاء أحجام ورق مخصصة في MS Excel، ومع ذلك، يمكنك تعيين حجم ورق مخصص للجداول العمل المرغوبة عند تقديم ملف Excel إلى تنسيق ملف PDF. يشرح هذا المستند كيفية تعيين حجم ورق مخصص لجدول عمل باستخدام APIs Aspose.Cells لبيثون via .NET.

## **تنفيذ حجم ورق مخصص لورقة العمل للتقديم**

يتيح لك Aspose.Cells لبيثون via .NET تنفيذ حجم ورق مخصص للورقة. يمكنك استخدام الطريقة [**custom_paper_size**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup/custom_paper_size/#float-float) من الفئة [**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup) لتحديد حجم صفحة مخصص. يوضح رمز العينة التالي كيفية تحديد حجم صفحة مخصص للورقة الأولى في الدفتر. يرجى أيضًا رؤية [ملف PDF الناتج](45056028.pdf) الذي تم إنشاؤه باستخدام الرمز التالي للإحالة.
