## **سيناريوهات الاستخدام المحتملة**

لا تتوفر خيار مباشر لإنشاء أحجام ورق مخصصة في MS Excel، ومع ذلك، يمكنك تعيين حجم ورق مخصص لأوراق العمل المرغوبة عند تصيير ملف إكسل إلى صيغة PDF. يشرح هذا المستند كيفية تعيين حجم ورق مخصص لورقة عمل باستخدام API الخاصة بـ Aspose.Cells للبايثون via .NET.

## **تنفيذ حجم ورق مخصص لورقة العمل للتقديم**

يسمح لك Aspose.Cells للبايثون via .NET بتنفيذ حجم ورق مخصص لورقة العمل التي ترغب بها. يمكنك استخدام طريقة [**custom_paper_size**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup/custom_paper_size/#float-float) من فئة [**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup) لتحديد حجم صفحة مخصص. يوضح رمز العينة التالي كيفية تعيين حجم ورق مخصص لأول ورقة عمل في مصنف العمل. يرجى أيضًا مراجعة [ملف PDF الناتج](45056028.pdf) المولَّد باستخدام الكود التالي للاشارة.
