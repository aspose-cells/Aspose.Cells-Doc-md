تم تكوين خطوط الشبكة في **Aspose.Cells.ReportingServices.xml** بحيث يمكنك التحكم في ما إذا كان ملف Excel المصدر يظهر خطوط الشبكة أم لا. القيمة الافتراضية هي **إيقاف**, مما يعني أن خطوط الشبكة لا تُعرض.

{{< highlight java >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
