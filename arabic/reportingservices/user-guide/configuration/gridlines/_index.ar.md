 تم تكوين خطوط الشبكة بتنسيق**Aspose.Cells.ReportingServices.xml** حتى تتمكن من التحكم في ما إذا كان ملف Excel Microsoft المُصدَّر يعرض خطوط الشبكة. النظام الأساسي**إيقاف**، مما يعني عدم عرض خطوط الشبكة.

{{< highlight "java" >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
