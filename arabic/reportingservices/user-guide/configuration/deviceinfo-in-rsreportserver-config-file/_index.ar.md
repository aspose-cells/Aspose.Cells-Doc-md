يأخذ قسم DeviceInfo في **rsreportserver.config** المعلمات التالية التي تؤثر على سلوك Aspose.Cells:

- **FileExtension**: عندما تكون القيمة **فارغة**, يكون امتداد ملف التقرير المصدر مع The default value  وعندما تكون القيمة ليست فارغة، يتم تعيين امتداد التقرير إلى القيمة.
- **SimplePageHeaders**: عندما تكون القيمة **صحيحة**, يتم انتقال عنصر رأس التقرير كرأس صفحة Microsoft Excel. القيمة الافتراضية هي **خاطئة**.
- **SimplePageFooters**: عندما تكون **صحيحة**, يتم انتقال عنصر تذييل التقرير كتذييل صفحة Microsoft Excel. القيمة الافتراضية هي **صحيحة**.
- **PutoutHeader**: عندما تكون **صحيحة** (القيمة الافتراضية)، يتم تصدير عنصر رأس التقرير. عند **خاطئة**, لا يتم تصدير عنصر رأس التقرير. يدعم فقط امتداد Excel 2007 XLSX ( البيانات فقط).
- **PutoutFooter**: عندما تكون **صحيحة** (القيمة الافتراضية)، يتم تصدير عنصر تذييل التقرير. عند **خاطئة**, لا يتم تصديره. يدعم فقط امتداد Excel 2007 XLSX ( البيانات فقط).
- **FillTableGroupHeaderForSimpleOutPut**: **خاطئة** بشكل افتراضي. القيمة تدعم فقط امتداد Excel 2007 XLSX ( البيانات فقط).
- **NoOutPutTotalForSimpleOutPut**: **خاطئة** بشكل افتراضي. القيمة تدعم فقط امتداد Excel 2007 XLSX ( البيانات فقط).
- **لا توجد مجموعة إخراج للناتج البسيط**: **false** افتراضيًا. يدعم القيمة فقط امتداد Excel 2007 XLSX (البيانات فقط).
- **لا يوجد صفحة للناتج البسيط**: **true** افتراضيًا. القيمة تدعم فقط امتداد Excel 2007 XLSX (البيانات فقط).
- **لا توجد صفحة للناتج البسيط**: **true** افتراضيًا. القيمة تدعم فقط امتداد Excel 2007 XLSX (البيانات فقط).
- **فاصل الحقل**: يحدد فواصل الحقول. القيمة تدعم امتدادات CSV و TXT.
