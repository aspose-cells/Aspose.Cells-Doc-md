 مقدمة

يتيح لك Aspose.Cells لخدمات التقارير تقديم تقارير Excel أصلية في Microsoft SQL Server 2005 Reporting Services. بخلاف العارض المضمن للتقارير المستندة إلى Excel ، يوفر Aspose.Cells لخدمات التقارير مجموعة ميزات موسعة ويسمح بتحويل تقارير RDL إلى: تنسيق Excel 97-2003 الثنائي XLS أو SpreadsheetML أو تنسيق Excel2007 XLSX.
