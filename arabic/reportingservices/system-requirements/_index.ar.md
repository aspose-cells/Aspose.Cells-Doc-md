### **نظام الأوفيس**
يتم دعم إصدارات Microsoft Excel التالية. علاوة على ذلك، يجب تثبيت التصحيحات الإضافية للتغلب على بعض المشكلات المحتملة في Microsoft Office 2000، 2002، XP، أو 2003.

- Microsoft Excel 2000
  قم بتثبيت **lockbackRegKey.msi** لإصلاح مشكلة CLR على أجهزة Microsoft Office القديمة قبل Office 2003.
- Microsoft Excel 2002
  قم بتثبيت **lockbackRegKey.msi** لإصلاح مشكلة CLR على أجهزة Microsoft Office القديمة قبل Office 2003.
- Microsoft Excel 2003
  قم بتحديث Office 2003 دومًا (KB907417). يُصحح هذا التحديث مشكلة محتملة حيث قد لا يتم تحميل الإضافة أو العلامة الذكية أو الوثيقة الذكية إذا استخدمت .NET Framework 2.0 من Microsoft.
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
- Microsoft Excel 2016
- Microsoft Excel 2019
- Microsoft Excel 365
