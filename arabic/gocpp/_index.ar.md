**مرحبًا بك في Aspose.Cells لـ Go**

Aspose.Cells for Go via C++ هو مكتبة بلغة Go تتيح لمطوري Go إنشاء وتعديل وتحويل جداول البيانات برمجياً دون الحاجة إلى أتمتة Office أو تطبيق Microsoft Excel. يدعم Aspose.Cells for Go تنسيقات جداول البيانات مثل MS Excel 97-2003 (XLS)، و MS Excel 2007-2013/2016 (XLSX، XLSM، XLSB، وغيرها)، و Open Office XML وغيرها. يمكنك أيضًا عرض الصور من الأوراق وتحويل جداول إكسل إلى تنسيق PDF. يسمح الـ API أيضًا بإنشاء وتعديل وعرض الرسوم البيانية والأشكال. علاوة على ذلك، الحسابات الصيغية ميزة جيدة أخرى يوفرها المكون ويمكنك الاستفادة منها.
