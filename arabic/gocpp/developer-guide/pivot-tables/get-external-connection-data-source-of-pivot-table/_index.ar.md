## **الحصول على مصدر بيانات الاتصال الخارجي لجدول الدوران**

 تقدم Aspose.Cells القدرة على الحصول على مصدر بيانات الاتصال الخارجي للجدول المحوري. لهذا، يوفر API الخاص [**GetExternalConnectionDataSource()**](https://reference.aspose.com/cells/go-cpp/pivottable/getexternalconnectiondatasource/) الخاص بـ [**PivotTable**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/). الخاصية [**GetExternalConnectionDataSource()**](https://reference.aspose.com/cells/go-cpp/pivottable/getexternalconnectiondatasource/) تُرجع الكائن [**ExternalConnection**](https://reference.aspose.com/cells/cpp/aspose.cells.externalconnections/externalconnection/). يوضح المقتطف التالي استخدام الخاصية [**PivotTable.GetExternalConnectionDataSource()**](https://reference.aspose.com/cells/go-cpp/pivottable/getexternalconnectiondatasource/) للحصول على مصدر الاتصال الخارجي للجدول المحوري.
