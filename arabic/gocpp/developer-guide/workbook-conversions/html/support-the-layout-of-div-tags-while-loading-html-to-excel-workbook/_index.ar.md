{{% alert color="primary" %}} 

عادةً، يتم تجاهل تخطيط علامات DIV أثناء تحميل HTML إلى كائن مصنف Excel. ومع ذلك، إذا كنت تريد الحفاظ على تخطيط علامات DIV، قم بتعيين خاصية [GetSupportDivTag()](https://reference.aspose.com/cells/go-cpp/htmlloadoptions/getsupportdivtag/) إلى **true**. القيمة الافتراضية لهذه الخاصية هي **false**.

{{% /alert %}}
