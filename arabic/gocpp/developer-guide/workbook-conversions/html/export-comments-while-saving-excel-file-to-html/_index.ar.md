## **سيناريوهات الاستخدام المحتملة**

عند حفظ ملف Excel بصيغة HTML، لا يتم تصدير التعليقات. ومع ذلك، توفر Aspose.Cells هذه الميزة باستخدام خاصية [**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/isexportcomments/). إذا قمت بضبطها على **true**، فسيتم عرض التعليقات الموجودة في ملف Excel الخاص بك في HTML أيضًا.

## **تصدير التعليقات أثناء حفظ ملف Excel إلى HTML**

يشرح الكود النموذجي التالي استخدام خاصية [**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/isexportcomments/). تُظهر الصورة تأثير الكود على HTML عند ضبطها على **true**. يرجى تنزيل [ملف Excel العيني](50528260.xlsx) و [HTML المولد](5052826.txt) كمراجع.
