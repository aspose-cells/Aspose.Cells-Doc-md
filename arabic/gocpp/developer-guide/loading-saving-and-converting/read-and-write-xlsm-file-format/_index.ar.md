## **سيناريوهات الاستخدام المحتملة**

يدعم Microsoft Excel العديد من التنسيقات مثل XLS و XLSX و XLSM و XLSB و CSV وغيرها. يدعم Aspose.Cells أيضًا العديد من هذه التنسيقات. يوضح هذا المقال كيفية قراءة وكتابة ملف Excel بصيغة XLSM باستخدام Aspose.Cells.

## **قراءة وكتابة تنسيق ملف XLSM**
