{{% alert color="primary" %}}

Aspose.Cells يدعم تحويل HTML موجه من Microsoft Excel إلى تنسيق XLS/XLSX. يعني أن HTML الذي تم إنشاؤه بواسطة Microsoft Excel يمكن تحويله مرة أخرى إلى تنسيق XLS/XLSX باستخدام Aspose.Cells.

 بالمثل، إذا كان هناك HTML بسيط، يمكن لـ Aspose.Cells تحويله إلى نص غني بصيغة HTML. توفر Aspose.Cells طريقة [**Cell::GetHtmlString**](https://reference.aspose.com/cells/go-cpp/cell/gethtmlstring/) التي يمكنها أخذ هذا HTML البسيط وتحويله إلى نص خلية منسق.
