## **سيناريوهات الاستخدام المحتملة**

Aspose.Cells for Node.js via C++ يدعم تنسيق نمط مخصص *DBNum*. على سبيل المثال، إذا كانت قيمة الخلية 123 وأشرت تنسيقها المخصص كالـ [DBNum2][$-804]General فسيتم عرضها كـ 壹佰贰拾叁. يمكنك تحديد تنسيق الخلية المخصص باستخدام طريقتي [**Cell.getStyle()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getStyle--) و [**Style.setCustom(string)**](https://reference.aspose.com/cells/nodejs-cpp/style/#setCustom-string-).
