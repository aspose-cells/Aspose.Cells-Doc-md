يسعدنا أن نعلن عن أول إصدار عام لـ Aspose.Cells لـ PHP عبر Java. تتضمن واجهات برمجة التطبيقات وظائف معالجة جداول بيانات MS Excel في PHP (لغة برمجة مفتوحة المصدر مستخدمة على نطاق واسع). Aspose.Cells لـ PHP عبر Java هو مكون مجموعة فرعية يتضمن جميع الميزات المهمة والمفيدة الموجودة في الأصل[Aspose.Cells for Java](https://products.aspose.com/cells/java/). API الجديد قوي بنفس القدر ويتميز بمكونات غنية. وهو يدعم تحويلات تنسيق الملفات عالية الدقة من وإلى XLS و XLSX و XLSM و SpreadsheetML و CSV و Tab Delimited و HTML و MHTML و OpenDocument Spreadsheet في PHP. يرجى زيارة[صفحة مقارنة ميزات المنتج](/cells/ar/java/aspose-cells-for-php-via-java-features/)للرجوع اليها. علاوة على ذلك ، يرجى الاطلاع على الوثيقة الخاصة بكيفية القيام بذلك[بيئة الإعداد وإرشادات التثبيت الأخرى](/cells/ar/java/setup-and-installation-guidelines/)للرجوع اليها.
