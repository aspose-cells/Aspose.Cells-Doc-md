**مرحبًا بك في Aspose.Cells for PHP via Java**

Aspose.Cells for PHP via Java توفر جميع الميزات القيمة لـ Aspose.Cells for Java باستخدام JavaBridge. Aspose.Cells for PHP via Java هي مكتبة فئات تمكنك من أداء مجموعة كبيرة من مهام معالجة الجداول. Aspose.Cells for PHP via Java هي واجهة برمجة التطبيقات مقيّدة وتحتوي على ميزات غنية وتقدم تحويل صيغ الجداول، تنسيق الأوراق عمل على أدق المستويات، إدارة وتقديم الرسوم البيانية، ومحرك حساب الصيغ الموثوق به. Aspose.Cells for PHP via Java تدعم XLS، XLSX، ODS، SpreadsheetML، CSV، القوائم المحددة، HTML/MHTML، PDF، XPS وصيغ أخرى. مع Aspose.Cells for PHP via Java يمكنك إنشاء، تعديل، تحويل وعرض مستندات Excel.
