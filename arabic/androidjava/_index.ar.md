**مرحبًا بك في Aspose.Cells for Android via Java**

Aspose.Cells for Android via Java هو مكون مرن يتيح لتطبيقات Android إنشاء وإدارة أوراق العمل في Excel دون الحاجة إلى Microsoft Excel. الهدف النهائي لـ Aspose.Cells for Android via Java هو جلب كل الميزات الرائعة لـ Aspose.Cells for Java إلى منصة Android، مما يجعل التطوير الأصلي لنظام Android ممكنًا في Java، سواء بواسطة Android Studio أو أي بيئة تطوير تدعم تطوير Android. Aspose.Cells for Android via Java هو مكتبة فئة تمكّن تطبيقات Android الأصلية الخاصة بك من أداء مجموعة كبيرة من مهام معالجة المستندات. Aspose.Cells for Android via Java يدعم XLS، XLSX، CSV، SpreadsheetML، ODS، Tab Delimited، HTML، PDF، XPS وتنسيقات أخرى. باستخدام Aspose.Cells for Android via Java يمكنك إنشاء، تعديل، تحويل وتقديم أوراق العمل.
