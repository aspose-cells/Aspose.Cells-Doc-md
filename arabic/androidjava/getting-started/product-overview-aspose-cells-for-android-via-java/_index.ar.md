مرحبًا بك في توثيق Aspose.Cells for Android via Java!

Aspose.Cells هو مكون مرن يتيح لتطبيقات Android إنشاء وإدارة أوراق العمل في Excel دون الحاجة إلى تثبيت Microsoft Excel على النظام. على عكس المنتجات المماثلة من البائعين الآخرين، Aspose.Cells لا يدعم فقط إنشاء جداول بيانات وميزات تنسيق الملفات الأساسية الأخرى، ولكن أيضًا عددًا من الميزات المتقدمة. تجعل هذه الميزات المتقدمة من السهل على المطورين تلاعب محتويات جداول البيانات وتنسيق الخلايا وحماية الملف. كما يسمح Aspose.Cells أيضًا لك بإدخال بيانات إلى جداول البيانات من مصادر بيانات مختلفة، وإضافة الصيغ والوظائف الرياضية والتاريخية والمالية والنصية المعقدة الشائعة، وتدعم أيضًا التلاعب بالرسوم البيانية والصور والتعليقات وكائنات الرسم والضوابط.

{{% /alert %}} 
## **ميزات الوثيقة**
- فتح ملفات Excel العادية أو المشفرة (Excel97 - Excel2007 والإصدارات الأحدث) من مصادر مختلفة
- حفظ ملفات Excel (Excel97 - Excel2007 والإصدارات الأحدث) بتنسيقات مختلفة (XLS، XLSX/XLSM، XLTX، XLTM، XLSB، CSV، SpreadsheetML، مفصولة بالفواصل، HTML وما إلى ذلك)
- تحويل ملف Excel إلى مستند PDF
- تحويل ورقة عمل Excel إلى مستند XPS.
- قراءة وكتابة تنسيق جدول البيانات المفتوحة (ODS)
- تعديل خصائص الوثيقة لملفات Excel

- تطبيق **إعدادات تنسيق الأرقام** للخلايا

- إنشاء وإضافة **تعليقات** إلى الخلايا
- تحويل المخطط إلى ملف صورة
