{{% alert color="primary" %}} 

Aspose.Cells para Reporting Services le permite entregar Microsoft informes nativos de Excel a través de Microsoft SQL Server 2005 y 2008 Reporting Services. A diferencia del procesador integrado para informes basados en Excel Microsoft, Aspose.Cells para Reporting Services proporciona un conjunto de características ampliadas que convierten informes RDL a: formato XLS binario de Excel 97-2003, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 Formato XLSB, HTML y ODS.

{{% /alert %}}
