{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services le permite ofrecer informes nativos de Microsoft Excel a través de Microsoft SQL Server 2005 y 2008 Reporting Services. A diferencia del renderizador incorporado para informes basados en Microsoft Excel, Aspose.Cells for Reporting Services proporciona un conjunto de funciones extendidas que convierten los informes RDL a: formato binario XLS de Excel 97-2003, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 XLSB, HTML y formato ODS. 

{{% /alert %}} 

Aspose.Cells for Reporting Services admite más formatos de archivo que los exportadores nativos. A continuación, se muestran capturas de pantalla de ejemplos de informes XLS exportados con Aspose.Cells.
