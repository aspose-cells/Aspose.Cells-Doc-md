{{% alert color="primary" %}} 

Los usuarios pueden diseñar y entregar informes en Microsoft Excel directamente usando Aspose.Cells.Report.Designer, una herramienta simple y fácil de usar. No necesitan usar Visual Studio para crear informes, pero pueden ahorrar el costo de comprar licencias adicionales de Visual Studio y no tienen que perder tiempo aprendiendo a usar un entorno de desarrollo complejo.

Dado que Aspose.Cells.Report.Designer utiliza Microsoft Excel como plantilla de informe, los usuarios pueden aprovechar todas las potentes funciones y características de Microsoft Excel, incluidas tablas dinámicas, gráficos interactivos, VBA, macros y fórmulas. No todas estas funciones son compatibles con el formato de Excel estándar Microsoft que ofrece Reporting Services.

{{% /alert %}}
