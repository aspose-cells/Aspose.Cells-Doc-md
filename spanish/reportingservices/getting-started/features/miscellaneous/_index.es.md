{{% alert color="primary" %}} 

Algunas características efectivas de Aspose.Cells for Reporting Services se detallan a continuación

- Aspose.Cells for Reporting Services admite tablas con filas y pies de página agrupados. La agrupación de filas y pies de página se puede configurar en el momento del diseño al igual que otros elementos de informe.
- Aspose.Cells for Reporting Services admite parámetros y variables globales como fuentes de datos del informe.
- Los usuarios pueden usar Microsoft Query para crear fuentes de datos y consultas. Microsoft Query es una herramienta ampliamente utilizada con la que la mayoría de usuarios debería estar familiarizada.
- Aspose.Cells for Reporting Services admite tanto fuentes de datos específicas del informe como fuentes de datos compartidas creadas en el servidor de informes con antelación. Además, es posible utilizar fuentes de datos específicas del informe en el momento del diseño y especificar fuentes de datos compartidas como fuentes de datos del informe al publicar un informe.
- Aspose.Cells para Reporting Service está integrado con Microsoft Reporting Services de manera transparente y, por lo tanto, los usuarios pueden aprovechar al máximo las opciones de gestión, seguridad y entrega de los informes de Reporting Services.
