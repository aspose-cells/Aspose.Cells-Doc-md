{{% alert color="primary" %}} 

Algunas características efectivas de Aspose.Cells para Reporting Services se enumeran a continuación

- Aspose.Cells para Reporting Services admite tablas con filas y pies de página agrupados. La agrupación de filas y pies de página se puede configurar en el momento del diseño como otros elementos del informe.
- Aspose.Cells para Reporting Services admite parámetros y variables globales como fuentes de datos del informe.
- Los usuarios pueden usar Microsoft Query para crear fuentes de datos y consultas. Microsoft Query es una herramienta ampliamente utilizada con la que la mayoría de los usuarios deberían estar familiarizados.
- Aspose.Cells para Reporting Services admite orígenes de datos específicos de informes y orígenes de datos compartidos creados en Report Server por adelantado. Además, es posible utilizar fuentes de datos específicas del informe en el momento del diseño y especificar fuentes de datos compartidas como fuentes de datos del informe al publicar un informe.
- Aspose.Cells para Reporting Service se integra perfectamente con Microsoft Reporting Services, por lo que los usuarios pueden aprovechar al máximo las opciones de administración, seguridad y entrega de informes de Reporting Services.
