{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services está disponible como una evaluación gratuita ilimitada en el tiempo desde la [Página de descargas](https://downloads.aspose.com/cells/reportingservices). La versión de evaluación y las versiones con licencia del producto son iguales.

Cuando esté satisfecho con la versión de evaluación, compre una licencia. Asegúrese de entender y aceptar los términos y condiciones de la suscripción.

{{% /alert %}} 
### **Obtención y uso de la licencia**
La licencia estará disponible para descargar desde la página de pedido cuando la orden esté pagada.

El archivo de licencia es un archivo XML de texto claro, firmado digitalmente. El archivo de licencia contiene información como el nombre del cliente, el producto comprado y el tipo de licencia. No modifique el contenido del archivo de licencia, ya que hacerlo lo invalida.

Descargue la licencia a su computadora y cópiela en la carpeta que debe ser una de las siguientes carpetas.

- $[Carpeta de instalación]\Licencia\
- $[MSSQL Server Reporting Services]\ReportServer\Bin\
- $[MSSQL Server Reporting Services]\ReportServer\

El nombre del archivo de licencia puede ser uno de los siguientes según su tipo de suscripción:

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic", "Aspose.Total.lic
- Aspose.Cells.Reporting.Services - Developer Site Subscription.lic
- Aspose.Cells.Reporting.Services - Developer Subscription.lic
- Aspose.Cells.Reporting.Services - Multiple Deployment Subscription.lic
- Aspose.Cells.Reporting.Services - Single Server Deployment Subscription.lic

Para probar si el archivo de licencia se instaló correctamente, exporte cualquier informe como un libro de trabajo de Microsoft Excel. Si el libro de trabajo no contiene una hoja de marca de agua al final, el archivo de licencia se activó con éxito.

Aspose.Cells for Reporting Services inserta una hoja de marca de agua de evaluación al trabajar en el modo de evaluación. Cuando un archivo de licencia válido, por ejemplo cuando **Aspose.Cells.ReportingServices.lic** está en la carpeta $[Carpeta de instalación]\Licencia\, no hay hoja de marca de agua de evaluación.
