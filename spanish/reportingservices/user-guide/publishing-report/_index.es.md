{{% alert color="primary" %}} 

Cuando haya diseñado un informe con todos los elementos que desea que tenga, el siguiente paso es publicarlo en Reporting Server para que usted y sus colegas puedan ejecutarlo.

{{% /alert %}} 

Para publicar un informe en Report Server, siga los pasos a continuación:

1.  Hacer clic**Publicar** 

![todo:imagen_alternativa_texto](publishing-report_1.png)

 en la barra de herramientas Aspose.Cells.Report.Designer y haga clic en el**Sí** botón para guardar el informe.

![todo:imagen_alternativa_texto](publishing-report_2.png)




1.  En el cuadro de diálogo, ingrese la URL del servidor de informes y haga clic en el**Actualizar** botón.

![todo:imagen_alternativa_texto](publishing-report_3.png)

1.  Seleccione una carpeta de destino e ingrese el nombre del informe. Haga clic en el**Configuración de fuente de datos** botón para configurar las fuentes de datos.

![todo:imagen_alternativa_texto](publishing-report_4.png)



Se muestra el cuadro de diálogo Configuración de fuente de datos.

1.  En el cuadro de diálogo, ingrese la información de la fuente de datos. Hacer clic**Seleccione** para seleccionar las fuentes de datos compartidas. (Si los necesita, las fuentes de datos compartidas deben ser la principal fuente de datos del informe). Haga clic en el**DE ACUERDO** botón para completar el trabajo.

![todo:imagen_alternativa_texto](publishing-report_5.png)

1.  Hacer clic**Publicar** para publicar el informe en Report Server.

![todo:imagen_alternativa_texto](publishing-report_6.png)



botón para completar el trabajo.
