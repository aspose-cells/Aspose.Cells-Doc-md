{{% alert color="primary" %}} 

 La plantilla de informe Aspose.Cells admite variables globales de Reporting Services como fuente de datos para celdas que contienen un marcador de variables globales de Reporting Services.

[Aspose.Cells Plantilla y marcadores inteligentes](/cells/es/reportingservices/aspose-cells-template-and-smart-markers/) explica cómo funcionan los marcadores de variables globales de Reporting Services. Las variables globales normalmente se colocan en el área de texto del encabezado o pie de página de una tabla.

{{% /alert %}} 

Para agregar variables globales a los informes:

1.  Seleccione una celda.

   **Una celda seleccionada** 

![todo:imagen_alternativa_texto](adding-reporting-services-global-variables-to-report_1.png)




1.  Hacer clic**Insertar fórmula**en la barra de herramientas Aspose.Cells.Report.Designer (

![todo:imagen_alternativa_texto](adding-reporting-services-global-variables-to-report_2.png)

).

1.  Seleccione**Globales** en el panel izquierdo.
 Todas las variables globales se enumeran a la derecha.
1.  Seleccione**Reportar nombre** y haga doble clic.
 La expresión aparece en el**Editar** caja en la parte superior.

   **Selección de una variable global** 

![todo:imagen_alternativa_texto](adding-reporting-services-global-variables-to-report_3.png)




1.  Hacer clic**DE ACUERDO** para terminar.
 La celda ahora contiene un marcador de variable global.

   **Un marcador de variables globales en una celda** 

![todo:imagen_alternativa_texto](adding-reporting-services-global-variables-to-report_4.png)
