Para agregar un grupo de tabla, siga los siguientes pasos:



1) Seleccione la celda de etiqueta de grupo que está a la izquierda de la primera columna de la columna de datos de la tabla, debajo de la fila de datos de la tabla, como se muestra a continuación.

![todo:image_alt_text](adding-table-group_1.png)







2) Seleccione ”SalesOrderNumber” en el á¡rea de combinación **Set Group** para establecer el campo de grupo.



![todo:image_alt_text](adding-table-group_2.png)



El resultado se muestra a continuación.



![todo:image_alt_text](adding-table-group_3.png)



3) Agregue la fila de datos del grupo de acuerdo con las instrucciones descritas anteriormente. La fila de datos del grupo puede contener texto estático, marcadores de datos, fórmulas de Reporting Service, etc. En el ejemplo, agregamos una fila de grupo que contiene una celda de texto estático combinado y celda de fórmula de Reporting Services, como se muestra a continuación.



![todo:image_alt_text](adding-table-group_4.png)



4) Se permite más de una fila de datos de grupo en una tabla. Podemos agregar nuevamente una fila de datos de grupo, como se muestra a continuación.



![todo:image_alt_text](adding-table-group_5.png)
