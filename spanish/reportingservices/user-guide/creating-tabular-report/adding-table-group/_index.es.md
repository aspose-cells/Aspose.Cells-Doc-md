Para agregar un grupo de mesas, siga los pasos a continuación:



1) Seleccione la celda de etiqueta de grupo que se encuentra a la izquierda de la primera columna de la columna de datos de la tabla, debajo de la fila de datos de la tabla, como se muestra a continuación.

![todo:imagen_alternativa_texto](adding-table-group_1.png)







 2) Seleccione "Número de pedido de venta" en el**Establecer grupo** cuadro combinado para establecer el campo de grupo.



![todo:imagen_alternativa_texto](adding-table-group_2.png)



El resultado se muestra a continuación.



![todo:imagen_alternativa_texto](adding-table-group_3.png)



3) Agregue la fila de datos del grupo de acuerdo con las instrucciones descritas anteriormente. La fila de datos del grupo puede contener texto estático, marcadores de datos, fórmulas de servicios de informes, etc. En el ejemplo, agregamos una fila de grupo que contiene una celda de texto estático combinada y una celda de fórmula de Reporting Services, como se muestra a continuación.



![todo:imagen_alternativa_texto](adding-table-group_4.png)



4) Se permite más de una fila de datos de grupo en una tabla. Podemos agregar una fila de datos de grupo nuevamente, como se muestra a continuación.



![todo:imagen_alternativa_texto](adding-table-group_5.png)
