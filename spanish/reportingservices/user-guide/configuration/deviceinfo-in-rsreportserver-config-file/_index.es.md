La sección DeviceInfo en **rsreportserver.config** toma los siguientes parámetros que afectan el comportamiento de Aspose.Cells:

- **FileExtension**: cuando el valor es **nulo**, la extensión del archivo de informe exportado es el valor predeterminado. Cuando el valor no es nulo, se establece la extensión del informe en el valor.
- **SimplePageHeaders**: cuando el valor es **true**, el elemento de encabezado del informe se representa como un encabezado de página de Microsoft Excel. El valor predeterminado es **falso**.
- **SimplePageFooters**: cuando es **true**, el elemento de pie de página del informe se representa como un pie de página de página de Microsoft Excel. El valor predeterminado es **true**.
- **PutoutHeader**: cuando es **true** (predeterminado), se exporta el elemento de encabezado del informe. Cuando es **falso**, no se exporta el elemento de encabezado del informe. Solo admite la extensión Excel 2007 XLSX (solo datos).
- **PutoutFooter**: cuando es **true** (predeterminado), se exporta el elemento de pie de página del informe. Cuando es **falso**, no se exporta. Solo admite la extensión Excel 2007 XLSX (solo datos).
- **FillTableGroupHeaderForSimpleOutPut**: **falso** de forma predeterminada. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **NoOutPutTotalForSimpleOutPut**: **falso** de forma predeterminada. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **NoOutPutGroupForSimpleOutPut**: **falso** de forma predeterminada. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **NoDoGroupPageForSimpleOutPut**: **verdadero** de forma predeterminada. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **NoDoPageForSimpleOutPut**: **verdadero** de forma predeterminada. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **FieldDelimiter**: establece los delimitadores de campo. El valor admite las extensiones CSV y TXT.
