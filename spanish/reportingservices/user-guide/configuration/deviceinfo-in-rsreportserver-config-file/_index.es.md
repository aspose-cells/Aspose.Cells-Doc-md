 La sección DeviceInfo en**rsreportserver.config** toma los siguientes parámetros que afectan el comportamiento de Aspose.Cells:

- **Extensión de archivo** : cuando el valor es**nulo**, la extensión del archivo de informe exportado es el valor predeterminado. Cuando el valor no es nulo, la extensión del informe se establece en el valor.
- **Encabezados de página simple** : cuando el valor es**verdadero** , el elemento del encabezado del informe se representa como un encabezado de página de Excel Microsoft. El valor predeterminado es**falso**.
- **Pies de página simples** : cuando**verdadero** , el elemento de pie de página del informe se representa como un pie de página de Excel Microsoft. El valor predeterminado es**verdadero**.
- **Encabezado de salida** : cuando**verdadero** (predeterminado), se exporta el elemento del encabezado del informe. Cuando**falso**, el elemento de encabezado del informe no se exporta. Solo admite la extensión Excel 2007 XLSX (solo datos).
- **OutoutPie de página** : cuando**verdadero** (predeterminado), se exporta el elemento de pie de página del informe. Cuando**falso**No lo es. Solo admite la extensión Excel 2007 XLSX (solo datos).
- **FillTableGroupHeaderForSimpleOutPut**: **falso** por defecto. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **NoOutPutTotalForSimpleOutPut**: **falso** por defecto. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **NoOutPutGroupForSimpleOutPut**: **falso** por defecto. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **NoDoGroupPageForSimpleOutPut**: **verdadero** por defecto. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **NoDoPageForSimpleOutPut**: **verdadero** por defecto. El valor solo admite la extensión Excel 2007 XLSX (solo datos).
- **Delimitador de campo**: establece delimitadores de campo. El valor admite extensiones CSV y TXT.
