 Las líneas de cuadrícula se configuran en**Aspose.Cells.ReportingServices.xml** para que pueda controlar si un archivo de Excel Microsoft exportado muestra líneas de cuadrícula. El valor predeterminado es**apagado**, lo que significa que no se muestran las líneas de cuadrícula.

{{< highlight "java" >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
