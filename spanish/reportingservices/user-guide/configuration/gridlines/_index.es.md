Las líneas de la cuadrícula se configuran en **Aspose.Cells.ReportingServices.xml** para controlar si un archivo de Microsoft Excel exportado muestra las líneas de la cuadrícula. El valor predeterminado es **apagado**, lo que significa que las líneas de la cuadrícula no se muestran.

{{< highlight java >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
