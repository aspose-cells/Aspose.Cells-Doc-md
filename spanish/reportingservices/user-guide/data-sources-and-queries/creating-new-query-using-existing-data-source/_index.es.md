Para crear una nueva consulta utilizando una fuente de datos existente:

1. Abra Microsoft Excel.
1.  Hacer clic**Construir conjunto de datos** en el Aspose.Cells.Report.Designer (

![todo:imagen_alternativa_texto](creating-new-query-using-existing-data-source_1.png)

).

1.  En el cuadro de diálogo, seleccione una de las fuentes de datos enumeradas en el panel izquierdo. Por ejemplo,**Servidor SQL**.
1.  Hacer clic**Agregar**. 

   **Selección de una fuente de datos** 

![todo:imagen_alternativa_texto](creating-new-query-using-existing-data-source_2.png)




1. En el cuadro de diálogo, escriba un nuevo nombre de conjunto de datos, por ejemplo, SalesEmps.
1.  Hacer clic**Próximo**. 

   **Nombrar un nuevo conjunto de datos** 

![todo:imagen_alternativa_texto](creating-new-query-using-existing-data-source_3.png)



 Aspose.Cells.Report.Designer inicia Microsoft Query y se conecta a la fuente de datos seleccionada. Ahora se puede crear una nueva consulta con Microsoft Consulta.

**Construyendo una nueva consulta** 

![todo:imagen_alternativa_texto](creating-new-query-using-existing-data-source_4.png)




1.  Desde Microsoft Consultas**Archivo** menú, seleccione**Volver a Aspose.Cells.Report.Designer**. 

   **Volver al Diseñador de informes** 

![todo:imagen_alternativa_texto](creating-new-query-using-existing-data-source_5.png)



 El origen de datos SqlServer contiene dos conjuntos de datos, uno de los cuales es el nuevo conjunto de datos SalesEmps.
