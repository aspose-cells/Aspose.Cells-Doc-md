Para crear una nueva consulta usando un origen de datos existente:

1. Abre Microsoft Excel.
1. Haga clic en **Generar DataSet** en el Diseñador de Informes de Aspose.Cells (

![todo:image_alt_text](creating-new-query-using-existing-data-source_1.png)

).

1. En el cuadro de diálogo, seleccione uno de los orígenes de datos enumerados en el panel izquierdo. Por ejemplo, **SqlServer**.
1. Haga clic en **Agregar**. 

   **Seleccionar una fuente de datos** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_2.png)




1. En la ventana de diálogo, escriba un nombre nuevo para el conjunto de datos, por ejemplo, SalesEmps.
1. Haga clic en **Siguiente**. 

   **Nombrar un nuevo conjunto de datos** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_3.png)



Aspose.Cells.Report.Designer inicia Microsoft Query y se conecta a la fuente de datos seleccionada. Ahora se puede crear una nueva consulta con Microsoft Query. 

**Creando una nueva consulta** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_4.png)




1. Desde el menú **Archivo** de Microsoft Query, seleccione **Volver a Aspose.Cells.Report.Designer**. 

   **Volver al Diseñador de Informes** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_5.png)



La fuente de datos SqlServer contiene dos conjuntos de datos, uno de los cuales es el nuevo conjunto de datos SalesEmps.
