{{% alert color="primary" %}} 

El primer paso en el procedimiento para diseñar un informe en Aspose.Cells.Report.Designer es crear fuentes de datos y consultas. Hay dos formas de hacerlo. En el primer enfoque, podemos utilizar un archivo de definición de informe RDL existente creado por otras herramientas, por ejemplo utilizando Visual Studio y el archivo contiene fuentes de datos y consultas necesarias para el diseño del informe y la recuperación de datos. La segunda forma es crear nuevas fuentes y consultas en Aspose.Cells.Report.Designer.
