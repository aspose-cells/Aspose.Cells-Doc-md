{{% alert color="primary" %}} 

Puede crear orígenes de datos y consultas en Visual Studio y guardarlos en un archivo RDL. Cuando se abre el archivo RDL en Aspose.Cells.Report.Designer, sus orígenes de datos y consultas están disponibles para su uso. Por ejemplo, podemos crear una consulta llamada Detalle de Ventas de Empleados en Visual Studio y guardarla como un archivo RDL.

**El archivo RDL Detalle de Ventas de Empleados** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_1.png)

Este artículo muestra cómo trabajar con un archivo RDL en Aspose.Cells.Report.Designer.

{{% /alert %}} 

Puede abrir archivos RDL con Aspose.Cells.Report.Designer y trabajar con ellos:

1. Abre Microsoft Excel.
1. Desde la barra de herramientas de Aspose.Cells.Report.Designer, selecciona **Abrir Informe**. 

   **El botón Abrir Informe** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Selecciona el archivo RDL y ábrelo.
   Una vez que el archivo RDL se abre en Aspose.Cells.Report.Designer, las consultas aparecen en la lista de **Conjuntos de datos**. 

   **Conjuntos de datos disponibles** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_3.png)
