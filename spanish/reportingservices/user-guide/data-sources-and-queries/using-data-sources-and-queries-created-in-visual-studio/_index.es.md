{{% alert color="primary" %}} 

Puede crear fuentes de datos y consultas en Visual Studio y guardarlas en un archivo RDL. Cuando el archivo RDL se abre en Aspose.Cells.Report.Designer, sus fuentes de datos y consultas están disponibles para su uso. Por ejemplo, podemos crear una consulta denominada EmpSalesDetail en Visual Studio y guardarla como un archivo RDL.

**El archivo EmpSalesDetail RDL** 

![todo:imagen_alternativa_texto](using-data-sources-and-queries-created-in-visual-studio_1.png)

Este artículo muestra cómo trabajar con un archivo RDL en Aspose.Cells.Report.Designer.

{{% /alert %}} 

Puede abrir archivos RDL con Aspose.Cells.Report.Designer y trabajar con ellos:

1. Abra Microsoft Excel.
1.  En la barra de herramientas Aspose.Cells.Report.Designer, seleccione**Abrir informe**. 

   **El botón Abrir informe** 

![todo:imagen_alternativa_texto](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Seleccione el archivo RDL y ábralo.
 Una vez que se abre el archivo RDL en Aspose.Cells.Report.Designer, las consultas aparecen en el**conjunto de datos** lista.

   **Conjuntos de datos disponibles** 

![todo:imagen_alternativa_texto](using-data-sources-and-queries-created-in-visual-studio_3.png)
