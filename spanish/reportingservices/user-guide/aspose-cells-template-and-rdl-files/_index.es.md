{{% alert color="primary" %}} 

 Una definición de informe contiene información de diseño y recuperación de datos para un informe. El lenguaje de definición de informes (RDL) es una representación XML para la definición de informes. RDL es un esquema abierto: los desarrolladores pueden extender RDL con atributos y elementos adicionales. La extensión de archivo estándar para los archivos de definición de informes es .rdl. Su tipo MIME es texto/xml.

Una plantilla Aspose.Cells es un libro de Excel Microsoft que contiene información de diseño de informes y se guarda en archivos RDL (formato) cuando se completa el trabajo de diseño de la plantilla. Cuando se abre un archivo RDL que no contiene una plantilla Aspose.Cells, solo sus fuentes de datos y consultas están disponibles para su uso en Aspose.Cells.Report.Designer. Cuando se abre un archivo RDL que contiene una plantilla Aspose.Cells, la plantilla se recupera y se abre como un libro de trabajo en Microsoft Excel.
