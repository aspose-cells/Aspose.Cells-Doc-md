{{% alert color="primary" %}} 

El cuadro de diálogo "Dyn-Hide" le permite ocultar dinámicamente información de un informe. Le permite definir un rango de celdas en una hoja de trabajo que desea ocultar.

{{% /alert %}} 

Para ocultar información:

1.  En Microsoft Excel, haga clic en**Dyn-Ocultar** en la barra de comandos Aspose.Cells.
 Se muestra el cuadro de diálogo Dyn-Hide.

![todo:imagen_alternativa_texto](working-with-dyn-hide-form_1.png)




1. Edite la información oculta.
1.  Hacer clic**Agregar** para agregar ocultar información.

![todo:imagen_alternativa_texto](working-with-dyn-hide-form_2.png)



El área definida se oculta del informe.

Para eliminar ocultar información:

1. Seleccione una hoja de la lista de áreas ocultas en el cuadro de diálogo Dyn-Hide.
1.  Hacer clic**Borrar**.
 El área seleccionada se elimina y se muestra en el informe.
