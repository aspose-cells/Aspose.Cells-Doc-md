 Introducción

Aspose.Cells para Reporting Services le permite entregar informes nativos de Excel en Microsoft SQL Server 2005 Reporting Services. A diferencia del procesador integrado para informes basados en Excel, Aspose.Cells para Reporting Services proporciona un conjunto de funciones ampliadas y permite que los informes RDL se conviertan a: formato XLS binario Excel 97-2003, SpreadsheetML o formato XLSX Excel2007.
