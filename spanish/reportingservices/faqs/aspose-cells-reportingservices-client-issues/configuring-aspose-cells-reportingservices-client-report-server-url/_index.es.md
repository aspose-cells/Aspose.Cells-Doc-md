Para configurar una URL del servidor de informes del cliente:

1.  Haga clic en el**Abrir informe remoto** botón y obtener el**Recuperar informe** caja de diálogo.

![todo:imagen_alternativa_texto](configuring-aspose-cells-reportingservices-client-report-server-url_1.png)

1.  Haga clic en el**Configuración** botón y obtener**Configuración de URL del servidor de informes** diálogo.

![todo:imagen_alternativa_texto](configuring-aspose-cells-reportingservices-client-report-server-url_2.png)

1. Introduzca la información de configuración del servidor de informes (URL, nombre de usuario y contraseña).
 1. La URL del servidor de informes debe tener el formato http://host/ReportServer. Por ejemplo, http://172.16.204.11/ReportServer.
1. El nombre de usuario es el nombre de usuario de Windows Server. Por ejemplo, Administrador.
 1. La contraseña es la contraseña de usuario de Windows Server. Por ejemplo, 123456.
 1. El valor predeterminado del Dominio es nulo.
1.  Haga clic en el**Prueba** botón para comprobar la información de configuración.
1.  Hacer clic**Ahorrar** y guarde la información de configuración del servidor de informes.

![todo:imagen_alternativa_texto](configuring-aspose-cells-reportingservices-client-report-server-url_3.png)




1.  Hacer clic**Modificar** botón para modificar la información de configuración.
