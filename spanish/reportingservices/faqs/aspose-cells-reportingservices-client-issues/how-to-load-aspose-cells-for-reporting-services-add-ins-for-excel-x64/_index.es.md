Cuando está instalado, Aspose.Cells for Reporting Services se registra como un componente de 32 bits de forma predeterminada. No puede cargar complementos de Excel correctamente en Microsoft Excel x64. Por lo tanto, debe volver a registrar Aspose.Cells.ReportingServices.dll como componente(s) de 64 bits.
