Para establecer el formato condicional para los elementos del informe:

1. Cree y abra un archivo de definición de informe en Microsoft Excel.
 (Para los propósitos de este artículo, usaremos el caso de prueba (Company Sales 2008.rdl).
1.  Seleccione la hoja de cálculo Lista de ventas.

![todo:imagen_alternativa_texto](setting-conditional-formatting-for-report-item_1.png)




1.  Seleccione la celda I6 y agregue formato condicional a esta celda para el elemento del informe de la tabla.
 1. Formatee las celdas que tienen menos de 1000 con relleno rojo claro con texto rojo oscuro.
1. Formatee las celdas mayores de 1000 con relleno amarillo claro con texto amarillo oscuro.
1.  Establezca más formato condicional para una celda.

![todo:imagen_alternativa_texto](setting-conditional-formatting-for-report-item_2.png)




![todo:imagen_alternativa_texto](setting-conditional-formatting-for-report-item_3.png)




1.  Guarde, publique y revise el informe.

![todo:imagen_alternativa_texto](setting-conditional-formatting-for-report-item_4.png)




1.  Agregue formato condicional para esta celda para el elemento de informe de matriz o el elemento de informe de cuadro de texto.

![todo:imagen_alternativa_texto](setting-conditional-formatting-for-report-item_5.png)




1.  Guarde, publique y revise el informe.

![todo:imagen_alternativa_texto](setting-conditional-formatting-for-report-item_6.png)
