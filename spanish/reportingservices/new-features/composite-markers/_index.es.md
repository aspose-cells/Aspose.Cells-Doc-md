{{% alert color="primary" %}} 

Hemos introducido la característica de Marcadores compuestos. Ahora, el usuario puede unir marcadores como un marcador compuesto utilizando la concatenación de caracteres, es decir, “&”. Por ejemplo, *&= Parameters!ReportYear.Value & "-" &Parameters!ReportMonth.Value  &" Sales Report"*. El marcador consta de dos marcadores de parámetros y dos marcadores de texto estático.

{{% /alert %}}
