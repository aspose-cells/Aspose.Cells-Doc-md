{{% alert color="primary" %}} 

Algunos tipos de gráficos en el archivo de definición de informe de SQL Server Reporting Services 2008 no están disponibles como tipos de gráficos de Excel:

- Línea-Escalón
- Rango-Plano
- Rango-Suave
- Rango-Columna
- Rango-Barras
- Rango-Barra de error
- Rango-BoxPlot
- Forma-Túnel

Al representar elementos de gráfico que no tienen opciones equivalentes en Microsoft Excel, Aspose.Cells los produce como imágenes. 

{{% /alert %}}
