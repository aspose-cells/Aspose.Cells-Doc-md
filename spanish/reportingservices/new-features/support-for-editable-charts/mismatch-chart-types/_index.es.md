{{% alert color="primary" %}} 

Algunos tipos de gráficos en el archivo de definición de informes de SQL Server Reporting Services 2008 no están disponibles como tipos de gráficos de Excel:

- Línea escalonada
- Rango-Llano
- Rango-Suave
- Rango-Columna
- Barra de rango
- Range-ErrorBar
- Range-BoxPlot
- Embudo de forma

Al representar elementos de gráfico que no tienen opciones equivalentes en Microsoft Excel, Aspose.Cells los genera en imágenes.

{{% /alert %}}
