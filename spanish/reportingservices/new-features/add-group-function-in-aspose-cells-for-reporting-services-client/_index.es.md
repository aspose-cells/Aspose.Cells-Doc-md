Establece los atributos de grupo del elemento de informe de tabla a través de Modificar atributos. Los parámetros incluyen Tipo, Nombre, Nivel y Ocultar.

- El tipo representa el tipo de grupo. Incluye Grupo de Filas y Grupo de Columnas.
- El nombre representa el nombre del grupo.
- El nivel representa el nivel del grupo.
- Ocultar representa Ocultar grupo.

La modificación del informe implica los siguientes pasos:
Abra el cuadro de diálogo "Modificar informe" y elija la pestaña Esquema.

![todo:imagen_alternativa_texto](add-group-function-in-aspose-cells-for-reporting-services-client_1.jpg)


 Hacer clic**Modificar** y actualice Ocultar configuración.

![todo:imagen_alternativa_texto](add-group-function-in-aspose-cells-for-reporting-services-client_2.jpg)


 Hacer clic**Comprometerse** y guarde las configuraciones ocultas.

![todo:imagen_alternativa_texto](add-group-function-in-aspose-cells-for-reporting-services-client_3.jpg)
