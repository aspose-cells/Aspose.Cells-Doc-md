Aspose.Cells para Reporting Services admite conjuntos de datos compartidos de SSRS 2008R2. Cuando el Diseñador de informes Aspose.Cells abre un archivo de definición de informe que incluye conjuntos de datos compartidos, puede obtener automáticamente información del conjunto de datos que incluye campos del servidor de informes.
