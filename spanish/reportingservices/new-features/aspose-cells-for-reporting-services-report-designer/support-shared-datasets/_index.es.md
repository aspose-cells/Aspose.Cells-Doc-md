Aspose.Cells for Reporting Services admite conjuntos de datos compartidos SSRS 2008R2. Cuando Aspose.Cells Report Designer abre un archivo de definición de informe que incluye conjuntos de datos compartidos, puede obtener automáticamente información del conjunto de datos que incluye campos del servidor de informes.
