El Aspose.Cells para el Diseñador de informes de Reporting Services guarda la URL del servidor de informes al que se accede actualmente. Cuando abre de nuevo el cuadro de diálogo Administrador del servidor de informes en el Diseñador de informes Aspose.Cells, el formulario de diálogo carga automáticamente la información guardada del servidor de informes.
