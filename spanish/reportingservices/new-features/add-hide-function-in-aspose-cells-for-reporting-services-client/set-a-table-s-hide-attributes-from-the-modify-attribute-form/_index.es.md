Establecer los atributos de ocultar una tabla de informe desde **Modificar atributo**. Los parámetros incluyen:

- **Tipo** representa el tipo de objeto a ocultar. Incluye tabla, fila y columna.
- **Índice** representa la posición del objeto a ocultar. Cuando el tipo es hoja de cálculo, el valor es nulo. El valor del índice empieza desde 0.
- **Expresión** representa el valor del estado de ocultación. El valor es verdadero, falso o una fórmula de Excel. La expresión admite fórmulas de Reporting Services.
