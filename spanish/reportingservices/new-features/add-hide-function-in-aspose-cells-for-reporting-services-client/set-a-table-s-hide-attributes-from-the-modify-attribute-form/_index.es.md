Establecer los atributos ocultos de un informe de tabla de**Modificar atributo**. Los parámetros incluyen:

- **Escribe** representa el tipo de objeto oculto. Incluye tabla, fila y columna.
- **Índice** representa la posición del objeto oculto. Cuando el tipo es hoja de trabajo, el valor es nulo. El valor del índice comienza desde 0.
- **Expresión** representa el valor del estado oculto. El valor es verdadero, falso o una fórmula de Excel. La expresión admite la fórmula de Reporting Services.
