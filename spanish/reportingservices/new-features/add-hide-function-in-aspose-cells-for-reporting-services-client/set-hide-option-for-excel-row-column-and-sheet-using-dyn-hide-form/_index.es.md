Aspose.Cells for Reporting Services oculta cualquier fila, columna y hoja especificada basada en los parámetros de ocultación. Los parámetros incluyen:

- **Nombre de la hoja** representa una hoja de cálculo que especifica un objeto oculto.
- **Tipo** representa el tipo de objeto a ocultar. Incluye hoja de cálculo, fila y columna.
- **Índice de Fila/Columna** representa el objeto oculto especificado. Cuando el tipo es hoja de cálculo, el valor es nulo. El índice de columna admite caracteres. El valor del índice empieza desde 1.
- **Estado de ocultación** representa el valor del estado de ocultación. El valor es verdadero, falso o una fórmula de Excel. El parámetro solo admite parámetros de informe de Reporting Services.
