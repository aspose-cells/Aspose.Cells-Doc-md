Aspose.Cells para Reporting Services oculta cualquier conjunto específico basado en filas, columnas y hojas en los parámetros Ocultar. Los parámetros incluyen:

- **NombreHoja** representa una hoja de cálculo que especifica un objeto oculto.
- **Escribe** representa el tipo de objeto oculto. Incluye hoja de cálculo, fila y columna.
- **Índice de fila/columna**representa el objeto oculto especificado. Cuando el tipo es hoja de trabajo, el valor es nulo. El índice de columna admite caracteres. El valor del índice comienza desde 1.
- **Ocultar Estado** representa el valor de estado oculto. El valor es verdadero, falso o una fórmula de Excel. El parámetro solo admite parámetros de informe de Reporting Services.
