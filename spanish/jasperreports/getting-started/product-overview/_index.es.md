### **¡Bienvenido al Aspose.Cells para la documentación de JasperReports!**

 Aspose.Cells para JasperReports es una biblioteca que funciona como reemplazo de[JasperInformes](https://community.jaspersoft.com/project/jasperreports-library)El exportador de Excel original Microsoft, incluido con algunas mejoras excelentes. Elimina la dependencia de la biblioteca de puntos de interés y utiliza Aspose.Cells for Java en el back-end para exportar informes en Microsoft formatos de Excel (XLS, XLSX).

{{% /alert %}}

### **Descripción del producto**

Aspose.Cells para JasperReports es una solución flexible que facilita la exportación de informes de JasperReports y JasperServer a Microsoft formatos de archivo de Excel. Todas las características del informe se convierten a Microsoft hojas de cálculo de Excel con el más alto grado de precisión.

Además de la funcionalidad proporcionada por el exportador de Excel de Micrsofot original basado en puntos de interés, este componente proporciona algunas características valiosas. Las principales innovaciones relacionadas con la exportación a XLS incluyen:

- Proporcione gráficos editables en formato Excel Microsoft en lugar de imágenes estáticas.
