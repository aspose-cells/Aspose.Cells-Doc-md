{{% alert color="primary" %}}

 Aspose.Cells for JasperReports es un exportador externo para[JasperInformes](https://community.jaspersoft.com/project/jasperreports-library).

 Aspose.Cells for JasperReports se proporciona como un solo archivo JAR y se puede instalar fácilmente en computadoras que ejecutan JasperReports.[Descárguelo para iniciar una prueba](https://downloads.aspose.com/cells/jasperreports).

[Para instalar el exportador](/cells/es/jasperreports/installation/), **aspose.cells.jasperreports.jar** debe ser copiado en el**JasperReports\lib** directorio.

{{% /alert %}}
