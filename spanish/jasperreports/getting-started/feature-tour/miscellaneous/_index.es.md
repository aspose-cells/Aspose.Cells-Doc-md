{{% alert color="primary" %}} 

A continuación se enumeran algunas características efectivas de Aspose.Cells for JasperReports:

1. Aspose.Cells for JasperReports admite gráficos de Microsoft Excel para que los gráficos se exporten como gráficos editables en lugar de imágenes estáticas. De esta manera, los usuarios pueden editar manualmente los gráficos en Microsoft Excel si es necesario.
2. Aspose.Cells for JasperReports se [integra fácilmente con JasperServer](/cells/es/jasperreports/integration-with-jasperserver/).

{{% /alert %}}
