{{% alert color="primary" %}} 

Algunas características efectivas de Aspose.Cells for JasperReports se enumeran a continuación:

1. Aspose.Cells for JasperReports admite Microsoft gráficos de Excel para que los gráficos se exporten como gráficos editables en lugar de imágenes estáticas. De esta manera, los usuarios pueden editar los gráficos manualmente en Microsoft Excel si lo necesitan.
 2. Aspose.Cells for JasperReports es[fácilmente integrado con JasperServer](/cells/es/jasperreports/integration-with-jasperserver/).

{{% /alert %}}
