{{% alert color="primary" %}}

Aspose.Cells for JasperReports es un producto para exportar informes desde JasperReports y JasperServer al formato de Microsoft Excel. Aspose.Cells for JasperReports hereda todas las características del exportador integrado y también compensa algunas de las debilidades del generador de XLS original de JasperReports. Esta sección proporciona una descripción detallada de las funciones de Aspose.Cells for JasperReports. Haga clic en los enlaces correspondientes en la tabla para obtener más información sobre las funciones.

{{% /alert %}}

| :- | :- |
|[Exportar a XLS o XLSX](/cells/es/jasperreports/xls-or-xlsx-export/)|Exporte informes en formato Microsoft Excel XLS desde su aplicación sin utilizar Microsoft Excel. |
|[Fidelidad alta al diseño del informe](/cells/es/jasperreports/high-fidelity-to-the-report-design/)|El diseño y la apariencia de un informe XLS/XLSX coinciden con el original, similar a PDF u otros formatos. |
