{{% alert color="primary" %}}

Aspose.Cells para JasperReports es un producto para exportar informes de JasperReports y JasperServer al formato Microsoft Excel. Aspose.Cells para JasperReports hereda todas las características del exportador integrado y compensa algunas de las debilidades del renderizador XLS original de JasperReports. Esta sección proporciona una descripción detallada de Aspose.Cells para las características de JasperReports. Haga clic en los enlaces respectivos de la tabla para obtener más información sobre las características.

{{% /alert %}}

|**Rasgo** |**Descripción** |
|:- |:- |
|[Exportación XLS o XLSX](/cells/es/jasperreports/xls-or-xlsx-export/)| Exporte informes en formato Microsoft Excel XLS desde su aplicación sin utilizar Microsoft Excel.|
|[Alta fidelidad al diseño del informe](/cells/es/jasperreports/high-fidelity-to-the-report-design/)| El diseño y la apariencia de un informe XLS/XLSX coincide con el original, similar a PDF u otros formatos.|
|[Implementación fácil y ligera](/cells/es/jasperreports/easy-and-lightweight-deployment/)| Aspose.Cells para[JasperInformes](https://community.jaspersoft.com/project/jasperreports-library) es un[JasperInformes](https://community.jaspersoft.com/project/jasperreports-library) y el exportador JasperServer al formato Excel Microsoft.|
|[Licencias simples y asequibles](/cells/es/jasperreports/simple-and-affordable-licensing/)| Proporcionamos licencias asequibles que se utilizan fácilmente en el código fuente|
