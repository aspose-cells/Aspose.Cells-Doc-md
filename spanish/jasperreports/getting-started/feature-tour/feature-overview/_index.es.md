{{% alert color="primary" %}}

Aspose.Cells for JasperReports es un producto para exportar informes de JasperReports y JasperServer al formato Microsoft Excel. Aspose.Cells for JasperReports hereda todas las funciones del exportador integrado y compensa algunas de las debilidades del renderizador original XLS de JasperReports. Esta sección proporciona una descripción detallada de las funciones Aspose.Cells for JasperReports. Haga clic en los enlaces respectivos de la tabla para obtener más información sobre las funciones.

{{% /alert %}}

|:- |:- |
|[XLS o XLSX Exportación](/cells/es/jasperreports/xls-or-xlsx-export/)|Exporte informes en formato Microsoft Excel XLS desde su aplicación sin utilizar Microsoft Excel.|
