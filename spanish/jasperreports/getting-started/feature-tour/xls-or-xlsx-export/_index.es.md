{{% alert color="primary" %}}

 Aspose.Cells para JasperReports le permite entregar informes en formato XLS y XLSX sin pérdida de diseño o formato. Para crear hojas de cálculo en los formatos de Excel Microsoft, Aspose.Cells para JasperReports se basa en una versión integrada de[Aspose.Cells for Java](https://products.aspose.com/cells/java/), una biblioteca líder en procesamiento de hojas de cálculo de Aspose.

Microsoft No se requiere Excel para generar documentos con Aspose.Cells.
