**¡Bienvenido a Aspose.Cells for JasperReports!**

Aspose.Cells for JasperReports es una biblioteca que funciona como reemplazo del exportador original de Microsoft Excel de [JasperReports](https://community.jaspersoft.com/project/jasperreports-library), con algunas mejoras significativas. Elimina la dependencia de la biblioteca POI y utiliza Aspose.Cells for Java en el backend para exportar informes en formatos de Microsoft Excel (XLS, XLSX).
