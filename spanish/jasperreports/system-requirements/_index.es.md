{{% alert color="primary" %}} 

Para instalar Aspose.Cells para JasperReports en su sistema, su sistema debe cumplir con los siguientes requisitos:

- Java 2 Platform Standard Edition 1.6 o superior.
- JasperReports 6.3.0 o superior.

{{% /alert %}}
