## **Descargar bibliotecas requeridas**
Descargue las bibliotecas necesarias que se mencionan a continuación. Estos son los necesarios para ejecutar Aspose.Cells Java for Python ejemplos.

- [JPype](https://pypi.org/project/JPype1/)

## **Cómo configurar el código fuente**
Siga estos sencillos pasos para abrir y ampliar el código fuente mientras usa:

 \1. Instale solo Python 2.7.xx desde:<https://www.python.org/downloads/>
 \2. Instale JPype desde:<https://sourceforge.net/projects/jpype/files/JPype/>
 \3. Instale Java JDK desde:<https://www.oracle.com/java/technologies/downloads/>
 \4. Descargue las API Aspose.Cells for Java que desea usar en sus proyectos de Python desde:<https://downloads.aspose.com/cells/java> y copie los .jars de las API descargadas en la carpeta %Python%/lib.
 \5. Clon[Aspose.Cells Java for Python Ejemplos](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) de github.
