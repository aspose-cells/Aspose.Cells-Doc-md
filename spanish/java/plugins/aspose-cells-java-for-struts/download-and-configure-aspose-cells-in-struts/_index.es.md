## **Edificio Aspose.Cells Java para Struts 1.3 de Source Codes**
Después de consultar los códigos fuente de cualquiera de los repositorios anteriores, aplique los siguientes comandos mvn:

{{< highlight "java" >}}

 $ mvn -U clean package 

{{< /highlight >}}

Esto construirá "Strutsbookapp.war" en el tolder de destino.

Para implementar el archivo .war, simplemente cópielo en el directorio de la aplicación web del servidor Apache Tomcat en ejecución.
