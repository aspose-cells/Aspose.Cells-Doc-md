## **Construir Aspose.Cells Java para Struts 1.3 desde los Códigos Fuente**
Después de verificar los códigos fuente de cualquiera de los repositorios anteriores, aplique los siguientes comandos mvn:

{{< highlight java >}}

 $ mvn -U clean package 

{{< /highlight >}}

Esto creará "Strutsbookapp.war" en la carpeta de destino.

Para implementar el archivo .war simplemente cópialo al directorio webapp del servidor Apache Tomcat en ejecución.
