## **Soporte**
- Si desea ver problemas conocidos/informados (por los usuarios o el equipo de control de calidad) en la aplicación.
- O desea informar cualquier problema que haya encontrado en la aplicación
- Tiene alguna sugerencia de mejora o desea realizar alguna solicitud de función

**Utilice cualquiera de los siguientes rastreadores de problemas del proyecto:**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **Extender y contribuir**
Aspose.cells Java para Struts 1.3 es de código abierto y su código fuente está disponible en los principales sitios web de codificación social que se enumeran a continuación. Se alienta a los desarrolladores a descargar el código fuente y contribuir sugiriendo o agregando nuevas funciones o mejorando las existentes, para que otros también puedan beneficiarse de ellas.
