## **Soporte**
- Si deseas ver problemas conocidos / reportados (por los usuarios o el equipo de Q.A) en la aplicación.
- O quieres informar sobre algún problema que hayas encontrado en la aplicación.
- Tienes alguna sugerencia de mejora o te gustaría hacer una solicitud de función

**Por favor utiliza alguno de los siguientes rastreadores de problemas del proyecto:**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **Ampliar y Contribuir**
Aspose.cells Java para Struts 1.3 es de código abierto y su código fuente está disponible en los principales sitios web de codificación social listados a continuación. Se alienta a los desarrolladores a descargar el código fuente y contribuir sugiriendo o añadiendo nuevas funciones o mejorando las existentes, para que otros también puedan beneficiarse de él.
