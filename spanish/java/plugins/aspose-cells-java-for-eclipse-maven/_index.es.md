## **Introducción**
### **Eclipse IDE**
Eclipse IDE es un famoso Entorno de Desarrollo Integrado (IDE) para Java. El IDE es definitivamente el producto más conocido del proyecto de código abierto Eclipse. Hoy en día, es el entorno de desarrollo líder para Java con una cuota de mercado de aproximadamente el 60%.

El IDE Eclipse puede ser ampliado con componentes de software adicionales. Eclipse llama a estos componentes de software complementos (plug-ins). Varios proyectos de código abierto y empresas han ampliado el IDE Eclipse o han creado aplicaciones independientes (Eclipse RCP) sobre el marco de Eclipse.
### **Aspose.Cells for Java**
[Aspose.Cells for Java](http://goo.gl/gtvhlQ) es un galardonado componente de hoja de cálculo de Excel que permite a los desarrolladores de Java incrustar la capacidad de leer, escribir y manipular hojas de cálculo de Excel® (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF y formatos de archivo de imagen en sus propias aplicaciones Java sin necesidad de depender de Microsoft Excel®.

[Aspose.Cells for Java](http://goo.gl/gtvhlQ) es un componente maduro, escalable y rico en características que ofrece muchas funciones que van más allá de las simples capacidades de exportación de datos de otros proveedores. Con Aspose.Cells for Java, los desarrolladores pueden exportar datos, dar formato a hojas de cálculo al nivel más granular, importar imágenes, crear gráficos, aplicar y calcular fórmulas complejas, transmitir datos de Excel®, guardar en varios formatos y mucho más, todo esto sin necesidad de Microsoft Excel® o Automatización de Microsoft Office.
### **Aspose.Cells Java for Eclipse (Maven)**
- Aspose.Cells Java para Eclipse (Maven) es un plugin para **Eclipse IDE** por **Aspose**. Este complemento está destinado a desarrolladores que utilizan la plataforma Maven para desarrollos en Java y desean utilizar [Aspose.Cells for Java](http://goo.gl/gtvhlQ) en sus proyectos. El complemento le permite crear proyectos de Maven para utilizar el [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) y descargar también [Ejemplos de código](https://goo.gl/uxznrb) de la API. 
- El plugin proporciona las siguientes características para trabajar con [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) cómodamente dentro de **Eclipse IDE**:

**WIZARDS**:
El plugin contiene dos asistentes--

**       ****Proyecto Maven de Aspose.Cells (asistente)**

- Este asistente de Nuevo Proyecto permite a los desarrolladores crear proyectos **Maven** para utilizar [Aspose.Cells for Java](http://goo.gl/gtvhlQ) desde **Nuevo -> Proyecto -> Maven-> Proyecto Maven de Aspose.Cells.**
- La referencia de la dependencia maven de [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) se obtiene automáticamente del [Repositorio Maven de Aspose Cloud](https://repository.aspose.com/webapp/) y se agrega en el pom.xml.
- El proyecto creado siempre contendrá la versión de dependencia **Maven** más reciente disponible para [Aspose.Cells for Java API](http://goo.gl/gtvhlQ).
- Los pasos del asistente también presentan la opción de descargar [Ejemplos de código](https://goo.gl/uxznrb) para utilizar la [Aspose.Cells for Java API](http://goo.gl/gtvhlQ).

**       ****Ejemplo de código de Aspose.Cells (asistente)**

- Este asistente de Nuevo Archivo le permite copiar los [Ejemplos de Código](https://goo.gl/uxznrb) descargados en su proyecto para utilizar [Aspose.Cells for Java](http://goo.gl/gtvhlQ) desde **Nuevo -> Otro -> Java -> Ejemplo de Código de Aspose.Cells.** 
- Los ejemplos disponibles se muestran en formato de árbol desde donde el usuario puede seleccionarlos por categorías.
- Todos los ejemplos dentro de la categoría seleccionada se copiarán en la carpeta del paquete "**com.aspose.cells.examples**" del proyecto junto con los recursos necesarios dentro de la carpeta "**src/main/resources**" para ejecutar los ejemplos.
- Los [Ejemplos de Código](https://goo.gl/uxznrb) del [API Aspose.Cells for Java](http://goo.gl/gtvhlQ) son para demostrar las diversas funcionalidades del API.
- El asistente también buscará y actualizará los nuevos [Ejemplos de Código](https://goo.gl/uxznrb) disponibles desde el repositorio de ejemplos de [Aspose.Cells for Java.](https://goo.gl/uxznrb)

## **Descargando**
### **Descargar Eclipse IDE**
Primero necesitarás instalar Eclipse IDE antes de descargar el complemento Aspose.Cells Java for Eclipse (Maven).

Para descargar Eclipse IDE

1. Ir a [https://eclipse.org](https://eclipse.org/).
1. Descarga e instala el Eclipse IDE recomendado para desarrolladores de Java SE / EE.
### **Descargar Aspose.Cells Java para Eclipse (Maven)**
A continuación se presentan tres métodos recomendados para la descarga e instalación exitosa del complemento Aspose.Cells Java para Eclipse (Maven):

- Instalación de arrastrar y soltar desde [Eclipse Marketplace](https://marketplace.eclipse.org/content/asposecells-java-maven-eclipse) a su espacio de trabajo en Eclipse.
- O ir a **Ayuda** > **Instalar nuevo software...** > Ingrese la siguiente URL del sitio de actualización en **Trabajar con**: 
 cdn.aspose.com/marketplace/eclipse/java/maven/asposecells
  Luego seleccione "Aspose.Cells Java para Eclipse (Maven)" y haga clic en **Finalizar**. Acepte el acuerdo de licencia e instale el complemento.

### **¿Cómo aplicar la licencia de Aspose?**
Este complemento utiliza una versión de evaluación de Aspose.Cells. Una vez que esté satisfecho con su evaluación, puede comprar una licencia en el [sitio web de Aspose](https://purchase.aspose.com/buy).
Para eliminar el mensaje de evaluación y las limitaciones de funciones, se debe aplicar la licencia del producto. Recibirá un archivo de licencia después de haber comprado el producto. Siga los siguientes pasos para aplicar la licencia:

- Asegúrese de que el archivo de licencia tenga el nombre **Aspose.Cells.Java.lic**
- Coloque el archivo **Aspose.Cells.Java.lic** en la carpeta que contiene Aspose.Cells.jar
- Utilice el siguiente código para activar la licencia:

{{< highlight java >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **Cómo configurar el código fuente de Aspose.Cells Java para Eclipse (Maven)**
Los siguientes pasos sencillos conducirán de manera fluida a la exitosa configuración del plugin de código fuente de **"Aspose.Cells Java para Eclipse (Maven)"** en el IDE de Eclipse

1. Descargar / Clonar el código fuente.
1. Elegir **Archivo** > **Importar > General > Proyectos Existentes en Espacio de trabajo**
1. Buscar el último proyecto fuente que haya descargado
1. Seleccionar el Proyecto de Eclipse que desea importar
1. Hacer clic en **Finalizar**
1. El código del complemento de Aspose.Cells Java para Eclipse está listo para ser mejorado.
