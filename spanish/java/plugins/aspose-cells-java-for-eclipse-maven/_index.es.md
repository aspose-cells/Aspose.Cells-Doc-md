## **Introducción**
### **Eclipse IDE**
Eclipse IDE es un famoso entorno de desarrollo integrado (IDE) Java. El IDE es definitivamente el producto más conocido del proyecto Eclipse Open Source. Hoy es el entorno de desarrollo líder for Java con una cuota de mercado de aproximadamente el 60%.

El IDE de Eclipse se puede ampliar con componentes de software adicionales. Eclipse llama a estos componentes de software complementos. Varios proyectos y empresas de código abierto han ampliado el IDE de Eclipse o han creado aplicaciones independientes (Eclipse RCP) sobre el marco de trabajo de Eclipse.
### **Aspose.Cells for Java**
[Aspose.Cells for Java ](http://goo.gl/gtvhlQ)is an award-winning Excel Spreadsheet component that allows Java developers to embed the ability to read, write and manipulate Excel® spreadsheets (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF and image formatos de archivo en sus propias Java aplicaciones sin necesidad de depender de Microsoft Excel®.

[Aspose.Cells for Java](http://goo.gl/gtvhlQ)es un componente maduro, escalable y rico en características que ofrece muchas funciones que van mucho más allá de las capacidades simples de exportación de datos de otros proveedores. Con Aspose.Cells for Java, los desarrolladores pueden exportar datos, formatear hojas de cálculo al nivel más granular, importar imágenes, crear gráficos, aplicar y calcular fórmulas complejas, transmitir datos de Excel®, guardar en varios formatos y mucho más, todo esto sin necesidad de Microsoft Excel® o Microsoft Ofimática.
### **Aspose.Cells Java para eclipse (Maven)**
-  Aspose.Cells Java para Eclipse (Maven) es un complemento para**Eclipse IDE** por**Aspose.** Este complemento está destinado a desarrolladores que utilizan la plataforma Maven for Java desarrollos y desean utilizar[Aspose.Cells for Java](http://goo.gl/gtvhlQ) en sus proyectos. El complemento le permite crear proyectos maven para usar[Aspose.Cells for Java API](http://goo.gl/gtvhlQ) y también descargar[Ejemplos de código](https://goo.gl/uxznrb)del API.
-  El complemento proporciona las siguientes características para trabajar con[Aspose.Cells for Java API](http://goo.gl/gtvhlQ) dentro de**Eclipse IDE** cómodamente:

## **Requisitos del sistema y plataformas compatibles**
### **Requisitos del sistema**
- **Memoria del sistema:**2 GB o más (recomendado)
- **sistema operativo:**Cualquier sistema operativo compatible con Java VM (máquina virtual)
- **Conexión a Internet:**2 MB o más rápido (recomendado)
### **Plataformas compatibles**
- Eclipse Mars.1 (4.5.1) - Recomendado
- Eclipse Juno o posterior.
## **Descargando**
### **Descargar IDE de Eclipse**
Primero deberá instalar Eclipse IDE antes de descargar el complemento Aspose.Cells Java para Eclipse (Maven).

Para descargar el IDE de Eclipse

1.  Ir a[https://eclipse.org](https://eclipse.org/).
1. Descargue e instale los desarrolladores Eclipse IDE for Java SE / EE recomendados.
### **Descargar Aspose.Cells Java para Eclipse (Maven)**
Los siguientes son tres métodos recomendados para descargar e instalar correctamente el complemento Aspose.Cells Java para Eclipse (Maven):

-  Arrastrar y soltar la instalación desde[Mercado de Eclipse](https://marketplace.eclipse.org/content/asposecells-java-maven-eclipse) a su espacio de trabajo de Eclipse.
-  O ir a**Ayudar** > **Instalar nuevo software...**> Ingrese la siguiente URL del sitio de actualización en**Trabajar con**:
 cdn.aspose.com/marketplace/eclipse/java/maven/asposecells
 Luego seleccione "Aspose.Cells Java para Eclipse (Maven)" y**Finalizar**. Acepte el Acuerdo de licencia e instale el complemento.

### **¿Cómo solicitar la licencia Aspose?**
 Este complemento utiliza una versión de evaluación de Aspose.Cells. Una vez que esté satisfecho con su evaluación, puede comprar una licencia en el[Aspose sitio web](https://purchase.aspose.com/buy).
Para eliminar el mensaje de evaluación y las limitaciones de funciones, se debe aplicar la licencia del producto. Recibirá un archivo de licencia después de haber comprado el producto. Siga los pasos a continuación para aplicar la licencia

-  Asegúrese de que el archivo de licencia se llame como**Aspose.Cells.Java.lic**
-  Sitio**Aspose.Cells.Java.lic** archivo en la carpeta que contiene el Aspose.Cells.jar

#### **Cómo configurar el código fuente de Aspose.Cells Java para Eclipse (Maven)**
A continuación, los sencillos pasos conducirán sin problemas a una configuración exitosa de**"Aspose.Cells Java para Eclipse (Maven)"** código fuente del complemento en Eclipse IDE

1. Descargar/Clonar el código fuente.
1. Escoger**Archivo** > **Importar > General > Proyectos existentes en Workspace**
1. Busque la última fuente del proyecto que haya descargado
1. Seleccione el proyecto de Eclipse que desea importar
1. Hacer clic**Finalizar**
1. Aspose.Cells Java para el código del complemento de Eclipse ahora está listo para mejorar.
