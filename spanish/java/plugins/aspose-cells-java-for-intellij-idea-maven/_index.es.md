## **Introducción**

### **IntelliJ IDEA**

**IntelliJ IDEA** es un entorno de desarrollo integrado (IDE) de Java para desarrolladores de software. Es desarrollado por JetBrains. Originalmente creado para Java, actualmente soporta una amplia variedad de lenguajes, incluyendo JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure y Kotlin.

En un informe de **Infoworld** de 2010, IntelliJ recibió la puntuación más alta en el centro de pruebas de las cuatro principales herramientas de programación de Java: Eclipse, IntelliJ IDEA, NetBeans y Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) para Java es un componente de generación de informes de hojas de cálculo de Excel® que permite a las aplicaciones Java crear y gestionar hojas de cálculo de Excel® sin utilizar Microsoft Excel®. Aspose.Cells for Java es un componente increíblemente rico en funciones que ofrece mucho más que simples funciones básicas de exportación de datos.

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java para IntelliJ IDEA (Maven) es un **complemento de IntelliJ IDEA** que te permite crear proyectos maven (es decir, el proyecto creado contiene Aspose.Cells for Java dependencia de API maven) y descargar ejemplos de código fuente de la API para usar **Aspose.Cells for Java API**.

- Este complemento te ayuda a trabajar con **Aspose.Cells for Java API** dentro de tu IntelliJ IDEA favorito.

![todo:image_alt_text](kwkgljg.png)

**WIZARDS**:
El complemento contiene dos asistentes -----

**Asistente 1:** Proyecto Maven Aspose.Cells

- El **Asistente de Proyecto Maven Aspose** del complemento permite a los desarrolladores crear proyectos Maven Aspose.Cells for Java.
- La referencia de la dependencia maven **Aspose.Cells for Java API** se obtiene automáticamente del [Repositorio Maven de Aspose Cloud](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) y se agrega al pom.xml.
- El proyecto creado siempre contendrá la versión más reciente disponible de la Dependencia Maven para Aspose.Cells for Java.
- Los pasos del asistente también presentan la opción de descargar [Ejemplos de Código Fuente para usar Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

**Asistente 2:** Ejemplos de Aspose.Cells

- El **Asistente de Ejemplos de Aspose.Cells** te permite crear / copiar [Ejemplos de Código Fuente descargados de Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) en tu proyecto. Todos los ejemplos de la categoría seleccionada se copiarán en la carpeta del paquete "com.aspose.cells.examples" del proyecto junto con los recursos necesarios en la carpeta "src/main/resources" necesarios para ejecutar los ejemplos.

- [Ejemplos de Código Fuente de Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) están destinados a demostrar los diferentes usos de la API.

## **Requisitos del sistema y plataformas compatibles**

### **Requisitos del sistema**

- Microsoft Windows 8/7/Vista/2003/XP (incl.64-bit)  / Escritorio Linux GNOME o KDE / Mac OS X 10.5 o superior
- Mínimo 1 GB de RAM, se recomiendan 2 GB de RAM
- 300 MB de espacio en disco duro + al menos 1 G para caché
- Resolución de pantalla mínima de 1024x768
- Se recomienda JDK 1.7 o superior
- IntelliJ IDEA CE/UE (13.0.1, 13.1.2 o 14.0.2)
- Conexión a Internet (se recomienda 2 MB o más rápido)

### **Plataformas compatibles**

El complemento es compatible con las siguientes versiones de IntelliJ IDEA:

- 13.0.1 (Comunidad / Edición Ultimate)
- 13.1.2  (CE / UE)
- 14.0.2 (CE/ UE)

## **Descargando**

### **Descargando IntelliJ IDEA**

Primero necesitará instalar IntelliJ IDEA CE / UE antes de descargar el complemento **Aspose.Cells Java para IntelliJ IDEA (Maven)**.

Para descargar IntellliJ IDEA

1. Go to <https://www.jetbrains.com/idea/download/>
1. Seleccione su sistema operativo en ejecución
1. Haga clic en UE o CE para descargar.

### **Descargando el complemento Aspose.Cells Java para IntelliJ IDEA (Maven)**

- El complemento se puede descargar directamente desde la [URL de alojamiento del complemento de JetBrains](https://goo.gl/R4pysl) - en el Repositorio de complementos de IntelliJ IDEA
  y más tarde se puede instalar usando la opción del IDE:
  - Ir a **Archivo** > **Configuración**...>**Complementos**
  - Seleccionar "**Instalar complemento desde disco"**
- O se puede instalar directamente (sin descargar manualmente desde la URL de alojamiento del complemento de JetBrains) usando las opciones del IDE:
  - Ir a **Archivo** > **Configuración**...>**Complementos**
  - Selecciona **"Examinar Repositorios"**
  - En el cuadro de búsqueda, escribe: "Aspose"
  - Selecciona el plugin "**Aspose Cells Java for IntelliJ IDEA Maven**" para instalar.
