## **Introducción**

### **IDEA IntelliJ**

**IDEA IntelliJ** es un entorno de desarrollo integrado (IDE) Java para desarrolladores de software. Está desarrollado por JetBrains. Creado originalmente como for Java, actualmente es compatible con una amplia variedad de lenguajes, incluidos JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure y Kotlin.

 en un 2010**infomundo** informe, IntelliJ recibió la puntuación más alta del centro de pruebas de las cuatro principales herramientas de programación Java: Eclipse, IntelliJ IDEA, NetBeans y Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/)for Java es un componente de informes de hojas de cálculo de Excel® que permite que las aplicaciones Java creen y administren hojas de cálculo de Excel® sin usar Microsoft Excel®. Aspose.Cells for Java es un componente increíblemente rico en funciones que ofrece mucho más que funciones básicas de exportación de datos.

### **Aspose.Cells Java para IntelliJ IDEA-Maven**

-  Aspose.Cells Java para IntelliJ IDEA (Maven) es un**Complemento IntelliJ IDEA** que le permite crear proyectos maven (es decir, el proyecto creado contiene la dependencia Aspose.Cells for Java API maven) y descargar ejemplos de código fuente del API para usar**Aspose.Cells for Java API**.

-  Este plugin te ayuda a trabajar con**Aspose.Cells for Java API** dentro de su IDEA IntelliJ favorita.

**magos**:
El complemento contiene dos asistentes -----

**Mago 1:** Aspose.Cells Maven Proyecto

-  Complementos**Aspose Maven Asistente de proyectos** permite a los desarrolladores crear Aspose.Cells for Java Maven proyecto.
-  la referencia de**Aspose.Cells for Java API** maven la dependencia se obtiene automáticamente de[Aspose Nube Maven Repositorio](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) y se agrega en pom.xml.
- El proyecto creado siempre contendrá la versión disponible más reciente Maven Dependencia para Aspose.Cells for Java.
-  Los pasos del asistente también presentan la opción de descargar[Ejemplos de código fuente para usar Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

**Asistente 2:** Aspose.Cells Ejemplos

- **Aspose.Cells Asistente de ejemplos** le permite crear / copiar descargado[Ejemplos de código fuente de Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) en su proyecto. Todos los ejemplos de la categoría seleccionada se copiarán en la carpeta del paquete "com.aspose.cells.examples" del proyecto junto con los recursos necesarios dentro de la carpeta "src/main/resources" necesarios para ejecutar los ejemplos.

- [Ejemplos de código fuente de Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) están destinados a demostrar los diferentes usos del API.

## **Requisitos del sistema y plataformas compatibles**

### **Requisitos del sistema**

- Microsoft Windows 8/7/Vista/2003/XP (incl. 64 bits) / escritorio Linux GNOME o KDE / Mac OS X 10.5 o superior
- 1 GB de RAM mínimo, se recomiendan 2 GB de RAM
- 300 MB de espacio en disco duro + al menos 1 G para cachés
- Resolución de pantalla mínima de 1024x768
- Se recomienda JDK 1.7 o superior
- IntelliJ IDEA CE/UE (13.0.1, 13.1.2 o 14.0.2)
- Conexión a Internet (se recomiendan 2 MB o más rápido)

### **Plataformas compatibles**

El complemento es compatible con las siguientes versiones de IntelliJ IDEA:

- 13.0.1 (Comunidad / Edición definitiva)
- 13.1.2 (CE/UE)
- 14.0.2 (CE/UE)

## **Descargando**

### **Descargando IntelliJ IDEA**

 Primero deberá instalar IntelliJ IDEA CE / UE antes de descargar el**Aspose.Cells Java para IntelliJ IDEA (Maven)** enchufar.

Para descargar IntellliJ IDEA

1.  Ir a<https://www.jetbrains.com/idea/download/>
1. Seleccione su sistema operativo en ejecución
1. Haga clic en descarga UE o CE.

### **Descargando Aspose.Cells Java para el complemento IntelliJ IDEA (Maven)**

-  El complemento se puede descargar directamente desde el[URL de alojamiento del complemento de JetBrains](https://goo.gl/R4pysl) - en el repositorio de complementos IntelliJ IDEA
 y posteriores se pueden instalar usando la opción IDE:
 - Ir a**Archivo** > **Ajustes**...>**Complementos**
 - Seleccionar "**Instalar el complemento desde el disco"**
- O se puede instalar directamente (sin descarga manual desde la URL de alojamiento del complemento de JetBrains) usando las opciones de IDE:
 - Ir a**Archivo** > **Ajustes**...>**Complementos**
 - Seleccione**"Explorar repositorios"**
 - En el cuadro de búsqueda escriba: "Aspose"
 - Seleccionar "**Aspose Cells Java para IntelliJ IDEA Maven**"Complemento para instalar.
