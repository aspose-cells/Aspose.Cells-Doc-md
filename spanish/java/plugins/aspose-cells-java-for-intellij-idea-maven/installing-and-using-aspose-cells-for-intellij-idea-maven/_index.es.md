## **Instalación**
### **Cómo instalar Aspose.Cells Java para IntelliJ IDEA (Maven)**
Los siguientes pasos demuestran cómo instalar el complemento **Aspose.Cells Java para IntelliJ IDEA (Maven)**.

El complemento está listado en el sitio web oficial (https://goo.gl/R4pysl) de IntelliJ IDEA CE como **Aspose.Cells Java para IntelliJ IDEA Maven**, por lo que no es necesario descargarlo manualmente e instalarlo desde el disco, pero esta opción extra todavía existe en IntelliJ IDEA.

El complemento **Aspose.Cells Java para IntelliJ IDEA (Maven)** se puede instalar fácilmente (sin descarga manual) desde dentro de IntelliJ IDEA. Para hacerlo:

1. Seleccione **Configuración** desde el menú **Archivo**.
1. Haga clic en **Complementos**, luego busque en los repositorios en IntelliJ IDEA.

## **Usando**
### **Asistente de Proyecto Maven de Aspose.Cells**
Para crear un proyecto Maven de Aspose.Cells para usar la API [Aspose.Cells for Java](http://goo.gl/c1eSD2) dentro de IntelliJ IDEA, necesitas tener conexión a internet.

Luego sigue los pasos a continuación:

\1. Selecciona **Nuevo Proyecto**.
\2. Selecciona **Proyecto Maven de Aspose.Cells** 
\3. Haz clic en **Siguiente**. 

![todo:image_alt_text](m1du9a1.jpg)


\4. Proporciona un **GroupId, ArtifactId** y **Versión** para tu proyecto Maven y haz clic en **Siguiente**.

![todo:image_alt_text](khijrce.jpg)


` `Puedes seleccionar la casilla "También descargar ejemplos de código fuente" para descargar ejemplos de uso de la API (como se muestra en la captura de pantalla anterior)

\6. Proporciona un **Nombre de Proyecto** y **Ubicación** como cuando se crea una aplicación Java cualquiera

![todo:image_alt_text](0gszuiu.jpg)


\7. Haz clic en **Finalizar**.
` `Esto recuperará la referencia de la dependencia maven más reciente de la API [Aspose.Cells for Java](http://goo.gl/c1eSD2) desde el Repositorio Maven de Aspose Cloud y la configurará en el pom.xml
` `Si has seleccionado la casilla "También descargar ejemplos de código fuente". La descarga de ejemplos también comenzará desde el Repositorio de GitHub de la API [Aspose.Cells for Java](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

![todo:image_alt_text](eezoq3s.jpg)

![todo:image_alt_text](bujsm8v.jpg)

`     `Congratulations ! {{< emoticons/cross >}} You have successfully created Aspose.Cells Maven project for using [Aspose.Cells for Java API](http://goo.gl/c1eSD2).

![todo:image_alt_text](2oon4vh.jpg)

` `El proyecto Maven de la API Aspose.Cells for Java creado está listo para ser modificado según los requisitos de tu proyecto.

` `Si has optado por descargar [Códigos Fuente de Ejemplos](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples), puedes usar el Asistente de Ejemplos de Aspose.Cells para copiar/crear los ejemplos descargados en tu proyecto.
