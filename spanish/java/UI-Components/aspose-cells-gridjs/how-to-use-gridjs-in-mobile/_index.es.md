## Usar GridJs en dispositivos móviles

Puede usar GridJs de manera fluida en dispositivos móviles. 

El cliente js de GridJs puede adaptarse a la pantalla móvil.

Ejecuta la [demo de GridJs](https://github.com/aspose-cells/Aspose.Cells.Grid-for-Java/tree/master/Examples_GridJs)

El puerto predeterminado del proyecto demo es 8080. 

Abre el navegador en tu dispositivo móvil e ingresa la url http://tuip:8080/gridjsdemo/index. 

![todo:la navegación de pantalla en móvil](gridjs_edit_on_mobile.png)

Luego podemos realizar operaciones de edición como en el dispositivo PC.
