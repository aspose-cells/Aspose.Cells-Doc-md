 Esta página contiene notas de la versión para[Aspose.Cells Java 1.7.1.0 Reparación en caliente](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-java-1.7.1.0-hot-fix/)

{{% /alert %}} 

 Queridos clientes,

 ¡Hemos lanzado Aspose.Cells for Java V1.7.1! Se incluyen las siguientes mejoras y correcciones:

- Admite importar contenido y estilos de las celdas del archivo SpreadsheetML
 Admite exportar contenido y estilos de las celdas al archivo SpreadsheetML
 Admite el motor de cálculo de fórmulas
 Admite establecer la fórmula R1C1
 Admite otros dos tipos de fuentes de datos de tabla dinámica
 Corrige algunos errores al agregar tablas dinámicas.

Nota: eliminamos el método const CellValueType.FORMULA y agregamos Cell.isFormula() para decidir si la celda contiene fórmula.
