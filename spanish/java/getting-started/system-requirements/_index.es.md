{{% alert color="primary" %}} 

Aspose.Cells for Java no requiere tener Microsoft Excel instalado, ya que Aspose.Cells en sí mismo es un motor de creación, conversión y renderización de hojas de cálculo de Excel. Para ver documentos de Excel producidos por Aspose.Cells, es necesario tener al menos instalado el Visor de Microsoft Excel.

{{% /alert %}} 
## **Sistemas Operativos Compatibles**
Aspose.Cells for Java es compatible con cualquier sistema operativo que ejecute el tiempo de ejecución de Java, incluyendo, pero no limitado a:
### **Windows**
- Microsoft Windows 2000 (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows Server 2003 (x64, x86)
- Microsoft Windows Server 2008 (x64, x86)
- Microsoft Windows Server 2012 (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8 (x64, x86)
- Microsoft Windows 10 (x64, x86)
### **Linux**
- Linux (Ubuntu, openSUSE, CentOS y otros)
### **Mac**
- Mac OS X, etc.
## **Versiones de Java Compatibles**
Aspose.Cells for Java es compatible con las siguientes versiones de Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- o superior
