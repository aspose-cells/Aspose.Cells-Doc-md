## **Requisitos del sistema**

Aspose.Cells for Java admite las siguientes versiones Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- o superior

[Más detalles](/cells/es/java/system-requirements/)

## **Instalación**

### **Instale Aspose Cells for Java desde el repositorio Maven**

Maven es la forma más fácil de descargar e instalar Aspose.Cells for Java.

1.  Especifique Aspose Maven Configuración/ubicación del repositorio en su Maven pom.xml.
 2. Defina la dependencia Aspose.Cells for Java API en su pom.xml.

[Más pasos para detalles](/cells/es/java/installation/)

### **Instale Aspose Cells for Java manualmente**

1. [Descargar aspose-cells-{versión}-java.zip](https://downloads.aspose.com/cells/java/)
1. Descomprima el paquete, ingrese al directorio \JDK 1.6\, descomprima aspose-cells-{version}-java.zip.
1. Ingrese al directorio \JDK 1.6\aspose-cells-{version}-java\lib\, instale los paquetes JAR manualmente con Eclipse o Intellij.
