## **Requisitos del sistema**

Aspose.Cells for Java es compatible con las siguientes versiones de Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- o superior

[Más detalles](/cells/es/java/system-requirements/)

## **Instalación**

### **Instala Aspose Cells para Java desde el Repositorio Maven**

Maven es la forma más fácil de descargar e instalar Aspose.Cells for Java. 

1. Especifica la configuración/ubicación del Repositorio Maven de Aspose en tu pom.xml de Maven. 
2. Define la dependencia de la API Aspose.Cells for Java en tu pom.xml. 

[Más pasos para detalles](/cells/es/java/installation/)

### **Instala Aspose Cells para Java manualmente**

1. [Descarga los archivos jar de aspose-cells](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) en una carpeta.
1. Entra en la carpeta, instala los paquetes JAR manualmente con Eclipse o Intellij.
