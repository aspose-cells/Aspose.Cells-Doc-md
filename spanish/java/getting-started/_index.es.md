##  **Requisitos del sistema**

Aspose.Cells for Java admite las siguientes versiones de Java:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- o superior

[Más detalles](/cells/es/java/system-requirements/)

##  **Instalación**

###  **Instale Aspose Cells for Java desde el repositorio Maven**

 Maven es la forma más sencilla de descargar e instalar Aspose.Cells for Java.

1.  Especifique Aspose Maven Configuración/ubicación del repositorio en su Maven pom.xml.
 2. Defina la dependencia Aspose.Cells for Java API en su pom.xml.

[Más pasos para más detalles](/cells/es/java/installation/)

###  **Instalar Aspose Cells for Java manualmente**

1. [Descargar frascos de células aspose](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) en una carpeta.
1. Ingrese a la carpeta, instale los paquetes JAR manualmente con Eclipse o Intellij.
