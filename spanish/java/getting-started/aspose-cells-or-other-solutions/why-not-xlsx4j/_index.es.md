## **xlsx4j**
xlsx4j es una biblioteca Java de código abierto (ASLv2) para crear y manipular archivos xlsx de Microsoft Open XML Excel similares a docx4j para manipular archivos docx de Microsoft Open XML Word.
Es similar al kit de desarrollo OpenXML de Microsoft, pero para Java. docx4j utiliza JAXB para crear la representación del objeto en memoria.
Tendrás que tomarte el tiempo para entender JAXB y la estructura del archivo Open XML.

Lee más en [docx4java.org](https://www.docx4java.org/trac/docx4j)
## **Aspose.Cells for Java**
Aspose.Cells for Java es un componente galardonado de hoja de cálculo de Excel que permite a los desarrolladores de Java incrustar la capacidad de leer, escribir y manipular hojas de cálculo de Excel® (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF y formatos de archivos de imagen en sus propias aplicaciones Java sin depender de Microsoft Excel®.

Aspose.Cells for Java es un componente maduro, escalable y rico en funciones que ofrece muchas funciones que van más allá de las simples capacidades de exportación de datos de otros proveedores. Con Aspose.Cells for Java, los desarrolladores pueden exportar datos, formatear hojas de cálculo hasta el nivel más granular, importar imágenes, crear gráficos, aplicar y calcular fórmulas complejas, transmitir datos de Excel®, guardar en varios formatos y mucho más, todo esto sin necesidad de Microsoft Excel® ni de Automatización de Office de Microsoft.

Lee más en [aspose.com](https://products.aspose.com/cells/java/)

## **¿Por qué no xlsx4j**
xlsx4j solo puede analizar y guardar archivos xlsx de Microsoft Open XML Excel, Aspose.Cells for Java puede procesar todos los formatos de archivo de Excel (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS) y admite varias conversiones de formato de archivo.

## **Aspose.Cells Java for xlsx4j**
El proyecto Aspose.Cells for xlsx4j muestra cómo se pueden realizar diferentes tareas utilizando las API de Aspose.Cells Java en comparación con xlsx4j. El proyecto también cubre las características que solo están disponibles en las API de Aspose.Cells y no en xlsx4j para el manejo de hojas de cálculo.

Este proyecto es útil para los desarrolladores que desean comparar xlsx4j con Aspose.Cells O migrar de xlsx4j a Aspose.Cells.
## **Requisitos del sistema y plataformas compatibles**
### **Requisitos del sistema**
A continuación se presentan los requisitos del sistema para ejecutar Aspose.Cells Java para xlsx4j:

- Java 1.4 o superior instalado.
- Componente Aspose.Cells descargado.
- Bibliotecas docx4j descargadas.
### **Plataformas compatibles**
A continuación se presentan las plataformas admitidas:

- Aspose.Cells 8.3.0 o superior.
- Bibliotecas docx4j 3.1.0 o superior.
- Java IDE (Eclipse, NetBeans, IntelliJ, ...).

Para configurar **Aspose.Cells for Java API**, incluya la URL del repositorio de Aspose y la dependencia de maven de **Aspose.Cells for Java API** en su **pom.xml** de la siguiente manera:

{{< highlight java >}}

 <repositories>

    <repository>

        <id>aspose-maven-repository</id>

        <name>Aspose Maven Repository</name>

        <url>http://repository.aspose.com/repo/</url>

    </repository>

</repositories>

{{< /highlight >}}
### **Dependencia de Maven de **Aspose.Cells for Java****
{{< highlight java >}}

 <dependencies>

    <dependency>

        <groupId>com.aspose</groupId>

        <artifactId>aspose-cells</artifactId>

        <version>8.6.0</version>

    </dependency>

</dependencies>

{{< /highlight >}}
