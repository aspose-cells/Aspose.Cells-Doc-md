Aspose.Cells proporciona el método [**Workbook.ContentTypeProperties.Add**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypepropertycollection#add-java.lang.Object-) para agregar ContentTypeProperties personalizadas a un archivo de Excel. También puedes hacer que la propiedad sea opcional configurando la propiedad [**ContentTypeProperty.IsNillable**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypeproperty#IsNillable) en **true**. El siguiente fragmento de código demuestra cómo agregar ContentTypeProperties personalizadas opcionales a un archivo de Excel.
