## **Escenarios de uso posibles**
Anteriormente, las APIs de Aspose.Cells no eran capaces de renderizar el relleno degradado de las formas de WordArt cuando las hojas de cálculo se convertían al formato HTML. Ahora, las APIs de Aspose.Cells admiten renderizar el relleno degradado de las formas de WordArt. La siguiente captura de pantalla muestra el efecto del relleno degradado en HTML para versiones anteriores y recientes.
