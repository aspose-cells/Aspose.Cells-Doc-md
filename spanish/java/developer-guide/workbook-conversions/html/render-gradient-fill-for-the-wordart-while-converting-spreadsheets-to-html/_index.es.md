## **Posibles escenarios de uso**
Anteriormente, las API Aspose.Cells no podían representar el relleno degradado de las formas de WordArt cuando las hojas de cálculo se convertían al formato HTML. Ahora, las API Aspose.Cells admiten la representación del relleno degradado de las formas de WordArt. La siguiente captura de pantalla muestra el efecto del relleno degradado en HTML para versiones anteriores y recientes.
