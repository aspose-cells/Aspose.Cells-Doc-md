{{% alert color="primary" %}} 

Si desea exportar su archivo de Excel a HTML con líneas de cuadrícula, entonces por favor use la propiedad [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) y establezca su valor en **true**.

{{% /alert %}} 
## **Exportar Excel a HTML con Líneas de Cuadrícula**
El siguiente código de muestra crea un libro y llena su hoja de trabajo con algunos valores y luego lo guarda en formato HTML después de establecer la propiedad [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) en **true**.

La siguiente captura de pantalla muestra el HTML generado con este código de muestra. Como puede ver, también muestra las líneas de cuadrícula en el HTML de salida.
