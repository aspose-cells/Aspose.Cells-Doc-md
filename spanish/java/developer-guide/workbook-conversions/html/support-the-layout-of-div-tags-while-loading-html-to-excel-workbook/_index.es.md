{{% alert color="primary" %}} 

Normalmente, el diseño de las etiquetas div se ignora al cargar HTML en un objeto de libro de Excel. Sin embargo, si desea que el diseño de las etiquetas div no se ignore, entonces configure la propiedad [HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag) a **true**. El valor predeterminado de esta propiedad es **false**.

{{% /alert %}}
