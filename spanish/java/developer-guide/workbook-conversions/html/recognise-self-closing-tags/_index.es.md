HTML puede tener una variedad de formatos de etiquetas para etiquetas vacías como \<td>\</td> o \<td/>. Aspose.Cells admite ambos formatos ahora, mientras que antes solo admitía \<td>\</td>como etiquetas. Esta función se puede probar convirtiendo el archivo HTML de muestra adjunto a un archivo de Excel. El archivo HTML de muestra y los archivos de salida se pueden descargar desde los siguientes enlaces para realizar pruebas.
