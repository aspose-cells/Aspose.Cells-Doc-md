{{% alert color="primary" %}}

Aspose.Cells admite la conversión de archivos XLS que contienen imágenes y gráficos en documentos PDF. Aspose.Cells for Java puede funcionar de forma independiente para convertir una hoja de cálculo a PDF: Aspose.PDF Las API no son necesarias para la conversión.

{{% /alert %}}

El proceso se puede realizar en memoria ya que el proceso no depende de archivos XML temporales o intermediarios. Esto significa que los archivos grandes de Excel, por ejemplo, los que contienen imágenes, gráficos y otros objetos de dibujo, se pueden convertir de forma rápida y eficiente.

{{% alert color="primary" %}}

 Si la hoja de cálculo contiene fórmulas, es mejor llamar al[**Workbook.calculateFormula**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#calculateFormula()justo antes de representar a PDF. Al hacerlo, se asegura que los valores dependientes de la fórmula se vuelvan a calcular y los valores correctos se representen en PDF.

{{% /alert %}}
