Las facilidades de la estructura lógica del PDF proporcionan un mecanismo para incorporar información sobre la estructura del contenido del documento en un archivo PDF. Aspose.Cells conserva la información sobre la estructura de un documento de Microsoft Excel, como celda, fila, tabla, hoja de trabajo, imagen, forma, encabezado/pie de página, etc.

Con la opción [PdfSaveOptions.ExportDocumentStructure](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setExportDocumentStructure-boolean-), puede guardar un PDF etiquetado con la estructura del documento exportada.
