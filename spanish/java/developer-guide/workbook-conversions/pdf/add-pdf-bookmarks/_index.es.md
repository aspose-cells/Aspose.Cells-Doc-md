Este artículo proporciona información sobre cómo insertar marcadores PDF al convertir una hoja de cálculo a PDF.

Aspose.Cells le permite agregar marcadores en tiempo de ejecución. Los marcadores PDF pueden mejorar drásticamente la navegabilidad de documentos largos. Al agregar enlaces de marcadores a un documento PDF, puede tener un control preciso sobre la vista exacta que desea, no está limitado a vincular a una página. Puede configurar la vista precisa colocando la página de destino y luego crear el marcador.
