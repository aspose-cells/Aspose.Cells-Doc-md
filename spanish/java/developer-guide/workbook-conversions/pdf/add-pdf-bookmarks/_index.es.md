{{% alert color="primary" %}} 

Este artículo proporciona información sobre cómo insertar marcadores de posición en PDF al convertir una hoja de cálculo a PDF.

Aspose.Cells te permite agregar marcadores en tiempo de ejecución. Los marcadores de PDF pueden mejorar drásticamente la navegabilidad de documentos largos. Al agregar enlaces de marcadores a un documento PDF, puedes tener un control preciso sobre la vista exacta que deseas, no estás limitado a vincular a una página. Puedes configurar la vista precisa posicionando la página de destino y luego crear el marcador.

{{% /alert %}}
