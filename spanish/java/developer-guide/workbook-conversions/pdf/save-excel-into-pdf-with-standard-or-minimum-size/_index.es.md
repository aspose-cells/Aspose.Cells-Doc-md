{{% alert color="primary" %}} 

 De forma predeterminada, Aspose.Cells guarda Excel en PDF con tamaño estándar. Sin embargo, también puede guardarlo con Tamaño mínimo usando el[PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions#OptimizationType) propiedad. Acepta los siguientes valores

- [PdfOptimizationType.STANDARD](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#STANDARD)
- [PdfOptimizationType.MINIMUM_SIZE](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#MINIMUM_SIZE)

{{% /alert %}}
