{{% alert color="primary" %}}

Aspose.Cells admite la conversión de Microsoft orientado a Excel HTML al formato XLS/XLSX. Significa que el HTML generado por Microsoft Excel se puede volver a convertir al formato XLS/XLSX usando Aspose.Cells.

 Del mismo modo, si hay algún HTML simple, Aspose.Cells puede convertirlo en HTML Rich Text. Aspose.Cells proporciona[**Cell.setHtmlString()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#HtmlString) propiedad que puede tomar un HTML tan simple y convertirlo en texto de celda con formato.

{{% /alert %}}
