{{% alert color="primary" %}} 

Los hipervínculos son útiles para acceder a la información en otras hojas de trabajo o en sitios web. Microsoft Excel permite a los usuarios agregar hipervínculos al texto en las celdas y en las imágenes. Los hipervínculos de imágenes pueden facilitar la navegación por una hoja de trabajo, por ejemplo, como botones siguiente y anterior, o logotipos que se vinculan a sitios específicos. Este documento explica cómo insertar hipervínculos de imágenes en una hoja de trabajo utilizando Aspose.Cells.

{{% /alert %}} 

Aspose.Cells le permite agregar hipervínculos a imágenes en hojas de cálculo en tiempo de ejecución. Es posible configurar y modificar la sugerencia de pantalla y la dirección del enlace.
