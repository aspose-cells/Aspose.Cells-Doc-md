## **Obtenga la validación Cell en archivos ODS**

Con Aspose.Cells, puede aplicar la validación a una celda en archivos ODS. Para esto, el API proporciona el[**ObtenerValidación**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getValidation()) método de la[**Cell**](https://reference.aspose.com/cells/java/com.aspose.cells/Cell)clase.

El siguiente ejemplo de código demuestra el uso de la[**ObtenerValidación**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getValidation()) método cargando el[SAO fuente](SampleBook1.ods)archivo y leyendo la validación de la celda A9.
