{{% alert color="primary" %}}

Los usuarios de Microsoft Excel pueden agregar comentarios a las celdas de las hojas de cálculo para añadir información adicional a las celdas y resaltar los datos. Los desarrolladores pueden necesitar personalizar el comentario para especificar la configuración de alineación y la dirección del texto. Aspose.Cells proporciona APIs para lograr esta tarea.

{{% /alert %}}

Aspose.Cells proporciona un método [**Shape.setTextDirection()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#TextDirection) para establecer la dirección del texto deseada para un comentario.
