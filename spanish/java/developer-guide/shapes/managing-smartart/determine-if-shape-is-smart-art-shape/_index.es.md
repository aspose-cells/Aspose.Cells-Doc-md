## **Posibles escenarios de uso**

Smart Art Shapes son formas especiales en Microsoft Excel que le permiten crear diagramas complejos automáticamente. Puede encontrar si la forma es una forma de arte inteligente o una forma normal usando[**Forma.IsSmartArt**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#IsSmartArt)propiedad.
