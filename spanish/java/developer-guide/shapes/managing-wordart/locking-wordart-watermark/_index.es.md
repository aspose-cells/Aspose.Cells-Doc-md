{{% alert color="primary" %}}

Las APIs de Aspose.Cells permiten agregar marcas de agua WordArt en la hoja de cálculo de modo que el WordArt se convierte en un objeto que se puede mover y posicionar en la hoja de cálculo. También es posible bloquear el objeto WordArt para cualquier interacción como edición, movimiento y selección. Este artículo explica el uso del método [**Shape.setLockedProperty**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#setLockedProperty-int-boolean-) para bloquear algunos aspectos de la marca de agua.

{{% /alert %}}

## **Bloquear marca de agua WordArt**

Las APIs de Aspose.Cells permiten bloquear ciertos aspectos de la marca de agua para que la interacción del usuario pueda ser limitada o completamente bloqueada. El siguiente fragmento de código demuestra el uso de la API Aspose.Cells for Java para crear una marca de agua para cada hoja de cálculo en la hoja de cálculo cargada y bloquear la selección, el movimiento, la edición y el redimensionamiento de la marca de agua.
