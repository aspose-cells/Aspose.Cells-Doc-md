## **Escenarios de uso posibles**
Puede establecer el estilo predefinido de WordArt para el texto de la forma utilizando Aspose.Cells. Utilice los métodos [FontSetting.SetWordArtStyle()](https://reference.aspose.com/cells/java/com.aspose.cells/fontsetting#setWordArtStyle\(int\)) o [FontSettingCollection.SetWordArtStyle()](https://reference.aspose.com/cells/java/com.aspose.cells/fontsettingcollection#setWordArtStyle\(int\)) para este propósito.
