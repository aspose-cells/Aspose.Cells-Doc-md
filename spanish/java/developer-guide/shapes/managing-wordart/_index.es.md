{{% alert color="primary" %}}

Utilice WordArt para agregar efectos de texto especiales a las hojas de cálculo. Por ejemplo, estire un título en la parte superior del archivo, decore el texto y haga que se ajuste a una forma preestablecida, o aplique el texto a una hoja de Excel como una marca de agua de fondo. El WordArt se convierte en un objeto que puedes mover o colocar en hojas de cálculo para agregar decoración.

{{% /alert %}}
