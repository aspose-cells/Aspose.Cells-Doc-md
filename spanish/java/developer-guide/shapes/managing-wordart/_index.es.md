{{% alert color="primary" %}}

Use WordArt para agregar efectos de texto especiales a las hojas de cálculo. Por ejemplo, estirar un título en la parte superior del archivo, decorar texto y hacer que el texto se ajuste a una forma preestablecida, o aplicar el texto a una hoja de Excel como marca de agua de fondo. El WordArt se convierte en un objeto que se puede mover o posicionar en las hojas de cálculo para agregar decoración.

{{% /alert %}}
