Aspose.Cells proporciona funciones completas para gestionar formas en la hoja de cálculo. A veces es necesario obtener los puntos de conexión de una forma para alinear o colocar las formas en el lugar apropiado. Para este propósito, se requieren todos los puntos de conexión. El siguiente código se puede utilizar para obtener la lista de puntos de conexión de una forma donde se utiliza la función [**Shape.GetConnectionPoints()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ConnectionPoints).
