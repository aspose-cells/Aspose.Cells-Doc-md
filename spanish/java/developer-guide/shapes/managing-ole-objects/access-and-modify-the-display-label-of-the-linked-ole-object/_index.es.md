## **Posibles escenarios de uso**

Microsoft Excel le permite cambiar la etiqueta de visualización del objeto antiguo como se muestra en la siguiente captura de pantalla. También puede acceder o modificar la etiqueta de visualización del objeto Ole utilizando las API Aspose.Cells con el[**OleObject.Etiqueta**](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#Label)propiedad.
