## **Escenarios de uso posibles**

Microsoft Excel le permite cambiar la etiqueta de visualización del objeto Ole como se muestra en la siguiente captura de pantalla. También puede acceder o modificar la etiqueta de visualización del objeto Ole utilizando las API de Aspose.Cells con la propiedad [**OleObject.Label**](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#Label).
