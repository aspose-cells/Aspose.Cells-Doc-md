{{% alert color="primary" %}} 

Aspose.Cells proporciona la propiedad [OleObject.AutoLoad](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad) para actualizar el objeto OLE cuando el archivo de Excel se abre en Microsoft Excel. Debido a esta propiedad, el objeto OLE mostrará la imagen OLE correcta generada por Microsoft Excel.

{{% /alert %}} 
## **Actualizar automáticamente el objeto OLE a través de Microsoft Excel usando Aspose.Cells**
El siguiente código de muestra carga el [archivo de Excel de muestra](5473423.xlsx) que tiene una imagen OLE no real. El objeto OLE es en realidad un documento de Microsoft Word, pero el archivo de Excel de muestra muestra la imagen de un animal en lugar de la imagen de Microsoft Word. Sin embargo, si abre el [archivo de Excel de salida](5473429.xlsx), verá que Microsoft Excel muestra la imagen OLE correcta.
