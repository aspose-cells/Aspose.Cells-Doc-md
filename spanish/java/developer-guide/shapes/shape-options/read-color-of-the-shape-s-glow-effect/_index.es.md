## Posibles Escenarios de Uso

Si desea leer el color del efecto de resplandor de cualquier forma, por favor, use la propiedad [**Shape.Glow.Color**](https://reference.aspose.com/cells/java/com.aspose.cells/gloweffect#Color). Le ayudará a encontrar las diversas propiedades relacionadas con el color del efecto de resplandor aplicado en una forma.
