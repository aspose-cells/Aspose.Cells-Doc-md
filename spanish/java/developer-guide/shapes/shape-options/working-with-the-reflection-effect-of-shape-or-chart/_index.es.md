## **Posibles escenarios de uso**
 Aspose.Cells proporciona el[Forma.ReflejoEfecto](https://reference.aspose.com/cells/java/com.aspose.cells/shape#Reflection)propiedad para trabajar con el efecto de reflexión de forma o gráfico. Contiene las siguientes subpropiedades que puede configurar para lograr diferentes resultados según sus requisitos.

- [ReflectionEffect.Blur](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#Blur)
- [ReflectionEffect.Dirección](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#Direction)
- [ReflectionEffect.Distancia](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#Distance)
- [ReflectionEffect.FadeDirection](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#FadeDirection)
- [ReflectionEffect.RotWithShape](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#RotWithShape)
- [ReflectionEffect.Size](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#Size)
- [ReflectionEffect.Transparencia](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#Transparency)
- [ReflectionEffect.Tipo](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#Type)
