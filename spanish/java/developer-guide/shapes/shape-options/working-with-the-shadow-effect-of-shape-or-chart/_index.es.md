## **Posibles escenarios de uso**
 Aspose.Cells proporciona el[Forma.ShadowEffect](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ShadowEffect)propiedad para trabajar con el efecto de sombra de forma o gráfico. Contiene las siguientes subpropiedades que puede configurar para lograr diferentes resultados según sus requisitos.

- [ShadowEffect.Ángulo](https://reference.aspose.com/cells/java/com.aspose.cells/shadoweffect#Angle)
- [ShadowEffect.Blur](https://reference.aspose.com/cells/java/com.aspose.cells/shadoweffect#Blur)
- [ShadowEffect.Color](https://reference.aspose.com/cells/java/com.aspose.cells/shadoweffect#Color)
- [ShadowEffect.Distancia](https://reference.aspose.com/cells/java/com.aspose.cells/shadoweffect#Distance)
- [ShadowEffect.PresetType](https://reference.aspose.com/cells/java/com.aspose.cells/shadoweffect#PresetType)
- [ShadowEffect.Tamaño](https://reference.aspose.com/cells/java/com.aspose.cells/shadoweffect#Size)
- [ShadowEffect.Transparencia](https://reference.aspose.com/cells/java/com.aspose.cells/shadoweffect#Transparency)
