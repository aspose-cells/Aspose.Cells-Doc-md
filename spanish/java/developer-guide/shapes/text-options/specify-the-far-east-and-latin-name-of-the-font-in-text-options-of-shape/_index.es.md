## **Posibles escenarios de uso**

A veces, desea mostrar el texto en la fuente del idioma del Lejano Oriente, por ejemplo, japonés, chino, tailandés, etc. Aspose.Cells proporciona[**TextOptions.FarEastName**](https://reference.aspose.com/cells/java/com.aspose.cells/textoptions#FarEastName)propiedad que se puede utilizar para especificar el nombre de la fuente del idioma del Lejano Oriente. Además, también puede especificar el nombre de la fuente latina usando[**TextOptions.LatinName**](https://reference.aspose.com/cells/java/com.aspose.cells/textoptions#LatinName)propiedad.
