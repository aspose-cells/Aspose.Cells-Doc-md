## **Escenarios de uso posibles**

A veces quieres mostrar texto en fuentes de idioma del Lejano Oriente como japonés, chino, tailandés, etc. Aspose.Cells proporciona la propiedad [**TextOptions.FarEastName**](https://reference.aspose.com/cells/java/com.aspose.cells/textoptions#FarEastName) que se puede usar para especificar el nombre de la fuente del idioma del Lejano Oriente. Además, también puedes especificar el nombre de la fuente latina usando la propiedad [**TextOptions.LatinName**](https://reference.aspose.com/cells/java/com.aspose.cells/textoptions#LatinName).
