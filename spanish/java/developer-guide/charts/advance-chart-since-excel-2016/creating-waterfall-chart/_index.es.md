{{% alert color="primary" %}}

Un gráfico de cascada es un tipo especial de gráfico que normalmente se utiliza para mostrar cómo la posición inicial aumenta o disminuye. Microsoft Excel tiene muchos tipos de gráficos predefinidos, incluyendo columnas, líneas, sectores, barras, radar, etc. pero el gráfico de cascada está más allá de los gráficos básicos y se puede crear utilizando los tipos de gráficos existentes con poca o mucha personalización.

{{% /alert %}}

## Creación de un gráfico de cascada

Las API de Aspose.Cells permiten crear un gráfico de cascada con la ayuda de un gráfico de línea. La API también permite personalizar la apariencia del gráfico para darle la forma de una cascada configurando las propiedades más importantes para dar forma al gráfico, es decir; las propiedades [**Series.UpBars**](https://reference.aspose.com/cells/java/com.aspose.cells/series#UpBars) y [**Series.DownBars**](https://reference.aspose.com/cells/java/com.aspose.cells/series#DownBars).
