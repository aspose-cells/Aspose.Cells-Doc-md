{{% alert color="primary" %}}

 Puede usar Aspose.Cells para crear una etiqueta de datos personalizada de texto enriquecido del punto del gráfico. Aspose.Cells proporciona el[**Etiquetas de datos.caracteres()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#characters(int,%20int) ) método para devolver el[**Configuración de fuente**](https://reference.aspose.com/cells/java/com.aspose.cells/FontSetting)objeto que se puede usar para establecer las propiedades de la fuente del texto, como su color, negrita, etc.

{{% /alert %}}
