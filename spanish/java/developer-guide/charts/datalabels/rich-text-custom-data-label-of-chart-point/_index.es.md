{{% alert color="primary" %}}

Puede usar Aspose.Cells para crear una etiqueta de datos personalizada de texto enriquecido del punto del gráfico. Aspose.Cells proporciona el método [**DataLabels.characters()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#characters-int-int-) para devolver el objeto [**FontSetting**](https://reference.aspose.com/cells/java/com.aspose.cells/FontSetting) que se puede usar para establecer las propiedades de fuente del texto como su color, negrita, etc.
