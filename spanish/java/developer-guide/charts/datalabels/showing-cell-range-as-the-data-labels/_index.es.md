## Mostrar rango de celdas como etiquetas de datos en MS Excel

En Microsoft Excel 2013, puede mostrar Cell Rango para etiquetas de datos. Puedes seleccionar esta opción siguiendo estos pasos

- Seleccione Etiquetas de datos de la serie y haga clic con el botón derecho para abrir el menú emergente.
-  Haga clic en el**Formatear etiquetas de datos...** y se mostrará**Opciones de etiqueta**.
-  Marque o desmarque la casilla de verificación**La etiqueta contiene: valor desde Cells**.

## Mostrar rango de celdas como etiquetas de datos con Aspose.Cells

 Aspose.Cells proporciona el[**Etiquetas de datos.setShowCellRange()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#ShowCellRange) método para marcar o desmarcar la casilla de verificación**La etiqueta contiene: valor desde Cells** como se muestra en la captura de pantalla anterior.
