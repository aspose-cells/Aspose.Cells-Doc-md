## **Posibles escenarios de uso**

Puede cambiar el tipo de forma de las etiquetas de datos del gráfico usando el[**Etiquetas de datos.ShapeType**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#ShapeType)propiedad. Toma el valor de[**DataLabelShapeType**](https://reference.aspose.com/cells/java/com.aspose.cells/DataLabelShapeType)enumeración y cambia el tipo de forma de las etiquetas de datos en consecuencia. Algunos de sus valores son

{{< highlight "java" >}}

DataLabelShapeType.BENT_LINE_CALLOUT

DataLabelShapeType.DOWN_ARROW_CALLOUT

DataLabelShapeType.ELLIPSE

DataLabelShapeType.LINE_CALLOUT

DataLabelShapeType.RECT

etc.

{{< /highlight >}}
