##  **Posibles escenarios de uso**
Después de crear un gráfico en una hoja de trabajo, puede configurar el eje de categorías para él. En este artículo, le mostraremos cómo configurar el eje de categorías para un gráfico de Excel, usando Aspose.Cells con código de muestra.

##  **Los pasos en el código de muestra.**

1. Cree un nuevo libro de trabajo.

2. Cree un nuevo gráfico en la primera hoja de trabajo.

3. Agregue algunos valores a las celdas de la primera hoja de trabajo.

4. Ahora puede configurar el eje de categorías, hay dos formas: usar datos de celda o usar cadenas directamente, las cuales se muestran en el código de muestra.

5. Establezca el eje de valores, guarde el libro de trabajo para ver el resultado.
