## **Escenarios de uso posibles**
Después de crear un gráfico en una hoja de cálculo, puede establecer el eje de categoría para él. En este artículo, le mostraremos cómo establecer el eje de categoría para el gráfico de Excel, utilizando Aspose.Cells con un código de ejemplo.

## **Los pasos en el código de muestra**

1. Cree un nuevo libro de trabajo.

2. Cree un nuevo gráfico en la primera hoja de cálculo.

3. Agregue algunos valores a las celdas en la primera hoja de cálculo.

4. Ahora puede establecer el eje de categoría, hay dos formas: utilizando datos de celda o utilizando cadenas directamente, ambas se muestran en el código de muestra.

5. Establezca el eje de valor, guarde el libro de trabajo para ver el resultado.
