## **Posibles escenarios de uso**
Las primeras versiones de Aspose.Cells no podían manejar correctamente las unidades automáticas del eje del gráfico cuando el gráfico se representaba en imagen o PDF. Ahora, Aspose.Cells admite el manejo de unidades automáticas del eje del gráfico. No hay cambio de código. Simplemente convierta su gráfico en imagen o PDF y representará el eje del gráfico tal como lo representa Microsoft Excel.
## **Manejar unidades automáticas del eje del gráfico como Microsoft Excel**
