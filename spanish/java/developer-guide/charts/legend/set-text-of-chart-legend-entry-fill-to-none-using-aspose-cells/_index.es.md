{{% alert color="primary" %}} 

Si desea establecer el texto del relleno de la entrada de leyenda del gráfico a ninguno para que no se muestre dentro de la leyenda del gráfico, entonces establezca [LegendEntry.IsTextNoFill](https://reference.aspose.com/cells/java/com.aspose.cells/legendentry#IsTextNoFill) a **true**.

{{% /alert %}}
