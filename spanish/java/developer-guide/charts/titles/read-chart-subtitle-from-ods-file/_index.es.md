## **Leer el subtítulo del gráfico del archivo ODS**

Aspose.Cells le brinda la capacidad de leer subtítulos de gráficos en archivos ODS usando el[**Gráfico.Subtítulo**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#SubTitle)propiedad. El siguiente código de ejemplo carga el[muestra de archivo ODS](SampleChart.ods)y lee el subtítulo del gráfico usando[**Gráfico.Subtítulo**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#SubTitle)property y lo imprime en la ventana de consola. Consulte la salida de la consola del código que se proporciona a continuación como referencia.
