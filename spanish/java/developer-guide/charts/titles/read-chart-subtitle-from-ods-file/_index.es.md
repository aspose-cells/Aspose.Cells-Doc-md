## **Leer subtítulo del gráfico desde un archivo ODS**

Aspose.Cells te proporciona la capacidad de leer subtítulos de gráficos en archivos ODS utilizando la propiedad [**Chart.SubTitle**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#SubTitle). El siguiente código de muestra carga el [archivo ODS de muestra](SampleChart.ods) y lee el subtítulo del gráfico utilizando la propiedad [**Chart.SubTitle**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#SubTitle) e imprime en la ventana de la consola. Observa la salida de la consola del código a continuación para referencia.
