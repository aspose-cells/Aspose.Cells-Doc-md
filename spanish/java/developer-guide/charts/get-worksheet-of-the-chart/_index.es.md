{{% alert color="primary" %}}

 veces, desea acceder a una hoja de trabajo desde la referencia de un gráfico. Aspose.Cells proporciona el[**Gráfico.getWorksheet()**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#Worksheet) propiedad que devuelve la referencia de la hoja de cálculo que contiene el gráfico.

{{% /alert %}}

## Ejemplo

 El siguiente ejemplo muestra cómo utilizar el[**Gráfico.getWorksheet()**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#Worksheet) propiedad. El código primero imprime el nombre de la hoja de cálculo y luego accede al primer gráfico de la hoja de cálculo. Luego imprime el nombre de la hoja de trabajo nuevamente, usando el[**Gráfico.getWorksheet()**](https://reference.aspose.com/cells/java/com.aspose.cells/chart#Worksheet)propiedad.
