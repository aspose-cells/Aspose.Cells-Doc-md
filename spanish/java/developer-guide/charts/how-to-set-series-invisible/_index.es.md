## Cómo hacer que una serie sea invisible en un gráfico de Excel

En un gráfico de Excel, puede hacer clic derecho en un gráfico, hacer clic en "Seleccionar datos" y en la ventana emergente, puede configurar si una serie es visible marcándola o desmarcándola.
Puede descargar el [archivo de ejemplo](SeriesFiltered.xlsx) y operarlo en Excel como se muestra en la imagen para lograr esta función. A continuación, le mostraremos cómo lograr esto usando la biblioteca Aspose.Cells.
