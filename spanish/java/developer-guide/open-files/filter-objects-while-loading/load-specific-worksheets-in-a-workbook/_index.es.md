{{% alert color="primary" %}}

De forma predeterminada, Aspose.Cells carga la hoja de cálculo completa en la memoria. Es posible cargar solo hojas de cálculo específicas. Esto puede mejorar el rendimiento y consumir menos memoria. Este enfoque es útil al trabajar con un libro de trabajo grande que contiene varias hojas de cálculo.
