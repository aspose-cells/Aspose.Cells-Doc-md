{{% alert color="primary" %}} 

Puede acceder a la conexión de datos externos de cualquier tipo utilizando la colección [Workbook.DataConnections](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections). Uno de esos tipos de conexión de datos es WebQuery. Este artículo te mostrará cómo trabajar con la conexión de datos WebQuery. Puede crear una conexión de datos WebQuery en Microsoft Excel utilizando el menú **Datos** > **Desde la web**.

{{% /alert %}}
