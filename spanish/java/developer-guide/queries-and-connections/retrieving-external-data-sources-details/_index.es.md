{{% alert color="primary" %}} 

A veces, es posible que desee recuperar información sobre fuentes de datos externas. Por ejemplo, es posible que desee ver los datos de conexión SQL. Esta información puede incluir cualquier tipo de datos requeridos para conectarse a un servidor SQL, como la URL del servidor, el nombre de usuario, el nombre de la tabla subyacente, la consulta SQL, el tipo de consulta, la ubicación de la tabla, etc.

Aspose.Cells proporciona algunas llamadas útiles para recuperar tales detalles de bases de datos corporativas vinculadas a archivos de Excel de plantilla.

{{% /alert %}} 
## **Recuperación de detalles de fuentes de datos externas**
El siguiente ejemplo muestra cómo obtener la conexión de la base de datos y otros detalles. El ejemplo utiliza un archivo de Excel sencillo que contiene enlaces a una fuente de datos externa (SQL Server).

Cuando se ejecuta el código, los detalles de la conexión se imprimen en la consola.
