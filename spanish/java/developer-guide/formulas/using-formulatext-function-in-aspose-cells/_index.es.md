{{% alert color="primary" %}} 

*FormulaText* es una función de Excel 2013 y versiones posteriores. No es compatible con versiones anteriores como Excel 2010 o 2007, etc. Como su nombre lo indica, imprime el texto de la fórmula que se encuentra en una celda dada. Este artículo te mostrará cómo utilizar esta función usando Aspose.Cells.

{{% /alert %}}
