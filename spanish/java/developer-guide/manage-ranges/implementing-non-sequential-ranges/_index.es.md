Normalmente, los [rangos nombrados](/cells/es/java/named-ranges/) son rectangulares con celdas continuas y adyacentes entre sí. Pero a veces, puede necesitar usar un rango de celdas no secuencial en el que las celdas no son adyacentes. Aspose.Cells admite la creación de un rango nombrado con celdas no adyacentes. Para esto, la API proporciona la propiedad [**Name.RefersTo**](https://reference.aspose.com/cells/java/com.aspose.cells/name#RefersTo).
