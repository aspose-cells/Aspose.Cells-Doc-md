## **Posibles escenarios de uso**

 Aspose.Cells proporciona el[**Rango**](https://reference.aspose.com/cells/java/com.aspose.cells/Range)objeto que tiene varios métodos de utilidad que facilitan al usuario trabajar con rangos de Excel fácilmente. Este artículo ilustra el uso de los siguientes métodos o propiedades del[**Rango**](https://reference.aspose.com/cells/java/com.aspose.cells/Range)objeto.

- **Dirección**

Obtiene la dirección del rango.

- **Cell cuenta**

Obtiene todo el recuento de celdas en el rango.

- **Compensar**

Obtiene rango por desplazamiento.

- **Toda la columna**

Obtiene un objeto Range que representa la columna (o columnas) completa que contiene el rango especificado.

- **Fila completa**

Obtiene un objeto Range que representa toda la fila (o filas) que contiene el rango especificado.
