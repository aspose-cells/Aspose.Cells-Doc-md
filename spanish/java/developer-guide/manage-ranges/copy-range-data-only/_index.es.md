{{% alert color="primary" %}} 

A veces, necesitas copiar datos de un rango de celdas a otro, copiando solo los datos y no el formato. Aspose.Cells ofrece esta función mediante el método [Range.copyData](https://reference.aspose.com/cells/java/com.aspose.cells/range#copyData-com.aspose.cells.Range-)

{{% /alert %}} 
## **Copiar solo datos de rango**
Este ejemplo muestra cómo:

1. Crear un libro de trabajo.
1. Agregar datos a las celdas en la primera hoja de cálculo.
1. Crear un rango.
1. Crear un objeto de estilo con atributos de formato específicos.
1. Aplica el formato de estilo al rango.
1. Crear otro rango de celdas.
1. Copia los datos del primer rango a este segundo utilizando el método [Range.copyData](https://reference.aspose.com/cells/java/com.aspose.cells/range#copyData-com.aspose.cells.Range-)
