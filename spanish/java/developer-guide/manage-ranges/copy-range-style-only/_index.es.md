## **Copiar solo estilo de rango**
Este ejemplo crea un libro de trabajo, lo llena con datos y copia solo el estilo de un rango.

1. Crea un rango.
1. Cree un objeto de estilo con atributos de formato especificados.
1. Aplique el formato de estilo al rango.
1. Cree un segundo rango de celdas.
1. Copie el formato del primer rango al segundo rango.
