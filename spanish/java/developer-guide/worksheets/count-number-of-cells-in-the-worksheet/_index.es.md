## **Cuente el número de celdas en la hoja de trabajo**

Puede contar el número de celdas en la hoja de trabajo llamando al[**Cells.getCount**](https://reference.aspose.com/cells/net/aspose.cells/cells/properties/count)() o[**Cells.getCountLarge**](https://reference.aspose.com/cells/net/aspose.cells/cells/properties/countlarge)() captadores de propiedad como se muestra en el ejemplo de código que se proporciona a continuación.
