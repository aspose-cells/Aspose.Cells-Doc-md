## **Contar el número de celdas en la hoja de cálculo**

Puede contar el número de celdas en la hoja de cálculo llamando a los métodos getter de propiedad [**Cells.getCount**](https://reference.aspose.com/cells/net/aspose.cells/cells/properties/count)() o [**Cells.getCountLarge**](https://reference.aspose.com/cells/net/aspose.cells/cells/properties/countlarge)() como se muestra en el ejemplo de código dado a continuación.
