{{% alert color="primary" %}}

 Microsoft Excel le permite cambiar el diseño de la tabla dinámica usando*Herramientas de tabla dinámica > Diseño > Informe* Comandos del menú de diseño. Puede cambiar el diseño en estas tres formas

- Mostrar en formato compacto
- Mostrar en forma de esquema
- Mostrar en forma tabular

 Aspose.Cells también proporciona[**Tabla dinámica.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm()), [**Tabla dinámica.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm() ) y[**Tabla dinámica.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm()) métodos para cambiar el diseño de la tabla dinámica en estas tres formas.

{{% /alert %}}
