{{% alert color="primary" %}}

Aspose.Cells ahora admite la obtención de la fecha de actualización y la información actualizada por quién de un libro de trabajo.

{{% /alert %}}

[**PivotTable.getRefreshDate**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshDate) devuelve la fecha en la que se actualizó por última vez el informe de tabla dinámica. Similarmente[**PivotTable.getRefreshByWho**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshedByWho) La propiedad devuelve el nombre del usuario que actualizó el informe por última vez. El siguiente ejemplo demuestra esta función y el archivo de muestra se puede descargar desde el siguiente enlace.
