 Aspose.Cells ofrece la posibilidad de guardar la tabla dinámica en archivos ODS. Para esto, simplemente convierta un libro de trabajo con una tabla de puntos de interés existente o cree una nueva tabla dinámica y guarde el archivo en el formato ODS. Asegúrate de llamar[**Tabla dinámica. Calcular datos ()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData()antes de guardar para asegurarse de que la tabla dinámica se represente en el archivo de salida ODS.
