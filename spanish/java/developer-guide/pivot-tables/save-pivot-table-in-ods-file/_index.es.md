 Aspose.Cells ofrece la posibilidad de guardar tablas dinámicas en archivos ODS. Para esto, simplemente convierta un libro de trabajo con una tabla povot existente o cree una nueva tabla dinámica y guarde el archivo en formato ODS. Asegúrate de llamar[**Tabla dinámica. Calcular datos ()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData()antes de guardar para asegurarse de que la tabla dinámica se represente en el archivo ODS de salida.
