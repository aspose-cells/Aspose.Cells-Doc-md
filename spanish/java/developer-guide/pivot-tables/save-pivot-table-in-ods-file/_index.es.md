Aspose.Cells proporciona la capacidad de guardar una tabla dinámica en archivos ODS. Para esto, simplemente convierta un libro de trabajo con tabla dinámica existente o cree una nueva tabla dinámica y guarde el archivo en formato ODS. Asegúrese de llamar a [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData--) antes de guardar para asegurarse de que la tabla dinámica se rendere en el archivo ODS de salida.
