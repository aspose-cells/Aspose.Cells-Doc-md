## **Obtener fuente de datos de conexión externa de tabla dinámica**

Aspose.Cells proporciona la capacidad de obtener la fuente de datos de conexión externa de la tabla dinámica. Para esto, el API proporciona el[**Fuente de datos de conexión externa**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)propiedad de la[**Tabla dinámica**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable)clase. Él[**Fuente de datos de conexión externa**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)propiedad devuelve el[**Conexiónexterna**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection)objeto. El siguiente fragmento de código demuestra el uso de la[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) propiedad para obtener la fuente de datos de conexión externa de la tabla dinámica.
