## **Introducción**

Puede configurar diferentes opciones de tabla dinámica utilizando Aspose.Cells. Una de esas opciones es "Mostrar celdas vacías". Al configurar esta opción significa que todas las celdas vacías de la tabla dinámica se muestran con una cadena especificada.

Para encontrar y configurar esta opción en Microsoft Excel:

1. Seleccione una tabla dinámica y haga clic con el botón derecho.
2. Seleccione **Opciones de tabla dinámica**.
3. Seleccione la pestaña **Diseño y formato**.
4. Seleccione la opción **Mostrar celdas vacías** y especifique una cadena.

![todo:image_alt_text](setting-pivot-table-option-for-empty-cells-show_1.png)

## **Cómo configurar la opción de tabla dinámica - Mostrar celdas vacías**

Aspose.Cells proporciona las propiedades [**PivotTable.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString) y [**PivotTable.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString) para configurar la opción de tabla dinámica "Mostrar celdas vacías".
