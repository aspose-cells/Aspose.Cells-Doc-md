## **Clasificación personalizada en la tabla dinámica**
Al usar Aspose.Cells API, puede ordenar las tablas dinámicas según los valores de los campos. El siguiente fragmento de código carga el archivo de muestra de Excel y agrega tres tablas dinámicas. La primera tabla dinámica no tiene clasificación personalizada, la segunda tabla dinámica se clasifica según los valores de campo de la fila "Mariscos" y la tercera tabla dinámica se clasifica según los valores de campo de la columna "28/07/2000".

El archivo fuente de muestra y los archivos de salida se pueden descargar desde aquí para probar el código de muestra:
