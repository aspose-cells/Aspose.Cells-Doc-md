## **Orden personalizado en la tabla dinámica**
Al utilizar la API de Aspose.Cells, puede ordenar las tablas dinámicas según los valores del campo. El siguiente fragmento de código carga el archivo de Excel de muestra y agrega tres tablas dinámicas. La primera tabla dinámica está sin orden personalizado, la segunda tabla dinámica está ordenada en los valores del campo de la fila "Frutos de Mar" y la tercera tabla dinámica está ordenada en los valores del campo de la columna "28/07/2000".

El archivo fuente de ejemplo y los archivos de salida se pueden descargar desde aquí para probar el código de ejemplo:
