## **Escenarios de uso posibles**
Puedes proteger con contraseña el Proyecto VBA (Visual Basic para Aplicaciones) del libro con Aspose.Cells usando el método [VbaProject.protect()](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#protect\(boolean,%20java.lang.String\)).
## **Código de muestra**
El siguiente código de ejemplo carga el [archivo de Excel de ejemplo](43352069.xlsm), accede a su Proyecto VBA y lo protege con una contraseña. Finalmente, lo guarda como un [archivo de Excel de salida](43352070.xlsm).
