## **Posibles escenarios de uso**
 Puede proteger con contraseña el proyecto VBA (Visual Basic para aplicaciones) del libro de trabajo con Aspose.Cells usando[VbaProyecto.proteger()](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#protect\(boolean,%20java.lang.String\)) método.
## **Código de muestra**
 El siguiente código de ejemplo carga el[ejemplo de archivo de Excel](43352069.xlsm), accede a su Proyecto VBA y lo protege con una contraseña. Finalmente, lo guarda como un[archivo de salida de Excel](43352070.xlsm).
