{{% alert color="primary" %}}

 En Microsoft Excel, puede agregar una referencia de biblioteca al proyecto VBA haciendo clic en el**Herramientas > Referencias...** a mano. Se abrirá el siguiente cuadro de diálogo que le ayudará a seleccionar entre referencias existentes o explorar su biblioteca usted mismo.

![todo:image_alt_text](add-a-library-reference-to-vba-project-in-workbook_1.png)

Pero a veces es necesario agregar o registrar la referencia de la biblioteca al proyecto VBA mediante código. Puedes hacerlo usando Aspose.Cells[**VbaProject.getReferences().addRegisteredReference()**](https://reference.aspose.com/cells/java/com.aspose.cells/vbaprojectreferencecollection#addRegisteredReference(java.lang.String,%20java.lang.String)) método.
