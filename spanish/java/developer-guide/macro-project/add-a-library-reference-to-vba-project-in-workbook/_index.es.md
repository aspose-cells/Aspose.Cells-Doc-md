{{% alert color="primary" %}}

En Microsoft Excel, puede agregar una referencia de biblioteca al proyecto VBA haciendo clic manualmente en **Herramientas > Referencias...**. Se abrirá el siguiente cuadro de diálogo que le ayudará a seleccionar entre las referencias existentes o explorar su propia biblioteca.

![todo:image_alt_text](add-a-library-reference-to-vba-project-in-workbook_1.png)

Pero a veces, necesita agregar o registrar la referencia de biblioteca al proyecto VBA a través del código. Puede hacerlo usando el método de Aspose.Cells [**VbaProject.getReferences().addRegisteredReference()**](https://reference.aspose.com/cells/java/com.aspose.cells/vbaprojectreferencecollection#addRegisteredReference-java.lang.String-java.lang.String-).
