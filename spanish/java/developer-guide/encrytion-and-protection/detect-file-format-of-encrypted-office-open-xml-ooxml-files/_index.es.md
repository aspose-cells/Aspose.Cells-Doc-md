{{% alert color="primary" %}} 

**Office Open XML** (también conocido como **OOXML** o **Microsoft Open XML** (MOX)) es un formato de archivo basado en XML desarrollado por Microsoft para representar documentos de oficina como hojas de cálculo, gráficos, presentaciones y documentos de procesamiento de palabras.

{{% /alert %}} 



Aspose.Cells proporciona una forma de detectar el formato del archivo de archivos cifrados **Microsoft Open XML**. Para identificar el tipo de archivo, use el método [FileFormatUtil.detectFileFormat](https://reference.aspose.com/cells/java/com.aspose.cells/fileformatutil#detectFileFormat-java.lang.String-java.lang.String-) como se muestra en el ejemplo de código.
