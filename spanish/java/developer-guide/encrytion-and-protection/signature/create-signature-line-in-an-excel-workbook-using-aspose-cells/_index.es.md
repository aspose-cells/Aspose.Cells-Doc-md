{{% alert color="primary" %}}

Microsoft Excel proporciona una función para agregar **Línea de Firma** en libros de Excel. Puedes agregar una Línea de Firma haciendo clic en la pestaña **Insertar** y luego seleccionando **Línea de Firma** del grupo **Texto**.

Aspose.Cells también proporciona esta función y ha expuesto la propiedad [**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) para este propósito. Este artículo explicará cómo usar esta propiedad para agregar una Línea de Firma usando Aspose.Cells.

{{% /alert %}}
