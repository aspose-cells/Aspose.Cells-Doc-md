{{% alert color="primary" %}} 

Aspose.Cells le permite cargar una versión liviana del libro de trabajo en la memoria para editar su información de metadatos. Por favor, utilice la clase [com.aspose.cells.WorkbookMetadata](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookMetadata) para cargar el libro de trabajo.

{{% /alert %}}
