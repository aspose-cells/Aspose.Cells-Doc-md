{{% alert color="primary" %}}

 Aspose.Cells le permite importar un mapa XML dentro del libro de trabajo usando el[**Libro de trabajo.importXml()**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#importXml(java.lang.String,%20java.lang.String,%20int,%20int)) método. Puede importar el mapa XML usando el Excel Microsoft con los siguientes pasos

-  Seleccione**Desarrollador** pestaña
-  Hacer clic**Importar** en la sección XML y siga los pasos requeridos.

 Deberá proporcionar sus datos XML para completar la importación. Aquí hay un[muestra de datos XML](5472501.txt) que puede utilizar para la prueba.
