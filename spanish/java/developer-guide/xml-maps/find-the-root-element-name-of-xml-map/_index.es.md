Puede encontrar el nombre del *elemento raíz de XML Map* utilizando Aspose.Cells con la propiedad [**XmlMap.RootElementName**](https://reference.aspose.com/cells/java/com.aspose.cells/xmlmap#RootElementName). La siguiente captura de pantalla muestra el nombre del elemento raíz del XML Map en Microsoft Excel.
