## **Posibles escenarios de uso**

Puede vincular sus celdas a elementos del mapa XML utilizando Aspose.Cells. Utilice el[**Cells.LinkToXmlMap()**](https://reference.aspose.com/cells/java/com.aspose.cells/cells#linkToXmlMap(java.lang.String,%20int,%20int,%20java.lang.String)) método para este propósito.

## **Enlace Cells a elementos del mapa XML**

 El siguiente código de ejemplo carga el[archivo fuente excel](5472518.xlsx) que contiene el mapa XML y luego vincula las celdas A1, B2, C3, D4, E5 y F6 a los elementos del mapa XML FIELD1, FIELD2, FIELD4, FIELD5, FIELD7 y FIELD8 respectivamente y luego guarda el libro en[archivo de salida de Excel](5472517.xlsx).

 Si abres el[archivo de salida de Excel](5472517.xlsx) y haga clic en el*Desarrollador > Fuente*, verá que las celdas están vinculadas con elementos de mapa XML y también estarán resaltadas por Microsoft Excel como se muestra en esta imagen.
