{{% alert color="primary" %}} 

A veces, desea agregar íconos condicionales junto al texto en una celda para que los datos sean más significativos para los lectores. Desea utilizar algunos de los tipos de iconos de formato condicional pero sin aplicar formato condicional a las celdas. Aspose.Cells admite la función.

{{% /alert %}} 

El siguiente es un ejemplo que crea un archivo XLSX desde cero, agregando íconos condicionales a celdas con texto sin aplicar formato condicional.

Cuando se ejecuta el código, las imágenes del conjunto de iconos condicionales se agregan en el área de la celda "B2:C4", como se muestra a continuación.
