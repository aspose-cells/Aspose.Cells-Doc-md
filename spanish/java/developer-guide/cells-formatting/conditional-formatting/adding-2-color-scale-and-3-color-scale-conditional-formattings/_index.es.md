{{% alert color="primary" %}} 

**Escala de 2 colores** y**Escala de 3 colores** Los formatos condicionales se agregan de la misma manera, excepto que se diferencian por[FormatCondition.ColorScale.Is3ColorScale](https://reference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale) propiedad. Esta propiedad es**falso** para escala de 2 colores y**verdadero** para formatos condicionales de escala de 3 colores.

{{% /alert %}}
