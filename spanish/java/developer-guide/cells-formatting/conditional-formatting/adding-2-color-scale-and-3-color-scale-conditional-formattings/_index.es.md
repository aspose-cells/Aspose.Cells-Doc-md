{{% alert color="primary" %}} 

Las escalas de colores de 2 colores y 3 colores con formato condicional se agregan de la misma manera, excepto que se diferencian por la propiedad [FormatCondition.ColorScale.Is3ColorScale](https://reference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale). Esta propiedad es **false** para escalas de colores de 2 colores y **true** para escalas de colores de 3 colores con formato condicional.

{{% /alert %}}
