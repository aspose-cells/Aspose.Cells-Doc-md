## **Insertar un gráfico de chispa**
{{% alert color="primary" %}} 

Un gráfico de chispa es un gráfico pequeño en una celda de la hoja de cálculo que proporciona una representación visual de datos. Utilice gráficos de chispa para mostrar tendencias en una serie de valores, como aumentos o disminuciones estacionales, ciclos económicos o resaltar valores máximos y mínimos. Coloque un gráfico de chispa cerca de sus datos para lograr un mayor impacto. Hay tres tipos de Gráficos de Chispa: Línea, Columna y Apilado.
