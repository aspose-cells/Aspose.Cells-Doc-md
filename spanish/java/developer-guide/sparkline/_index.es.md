## **Insertar un minigráfico**
{{% alert color="primary" %}} 

Minigráfico es un gráfico diminuto en una celda de hoja de cálculo que proporciona una representación visual de los datos. Use minigráficos para mostrar tendencias en una serie de valores, como aumentos o disminuciones estacionales, ciclos económicos, o para resaltar valores máximos y mínimos. Coloque un minigráfico cerca de sus datos para lograr el mayor impacto. Hay tres tipos de minigráficos: Línea, Columna y Apilado.
