A veces, es necesario combinar varias hojas de trabajo en una sola. Esto se puede lograr fácilmente usando Aspose.Cells API. Este artículo le mostrará un ejemplo de código que lee un libro de origen y combina los datos de todas las hojas de trabajo de origen en una sola hoja de trabajo dentro de un libro de destino.

{{% /alert %}}
