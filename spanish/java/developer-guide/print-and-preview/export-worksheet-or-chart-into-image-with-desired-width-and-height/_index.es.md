{{% alert color="primary" %}}

Puede usar Aspose.Cells para exportar su hoja de trabajo o gráfico a una imagen con el ancho y el alto deseados. Proporciona[**ImageOrPrintOptions.setDesiredSize()**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#setDesiredSize(int,%20int)) para establecer el ancho y el alto deseados de la imagen exportada. El ancho y la altura se especifican en la unidad de píxeles.

{{% /alert %}}
