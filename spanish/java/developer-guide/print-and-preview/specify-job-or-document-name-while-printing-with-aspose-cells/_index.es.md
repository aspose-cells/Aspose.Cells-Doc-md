{{% alert color="primary" %}}

 Puede especificar el Nombre del trabajo o del documento mientras imprime su libro de trabajo u hoja de trabajo usando el[**WorkbookRender**](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookRender) o[**HojaRenderizar**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender) objetos. Aspose.Cells proporciona el[**WorkbookRender.toPrinter(nombreImpresora, nombreTrabajo)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ) y[**SheetRender.toPrinter(nombreImpresora, nombreTrabajo)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) métodos que puede usar para especificar el nombre del trabajo mientras imprime su libro de trabajo u hoja de trabajo

{{% /alert %}}

## **Especifique el nombre del trabajo o del documento al imprimir con Aspose.Cells**

 El código de ejemplo carga el archivo de origen de Excel y luego lo envía a la impresora especificando el nombre del trabajo o del documento mediante el[**WorkbookRender.toPrinter(nombreImpresora, nombreTrabajo)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ) y[**SheetRender.toPrinter(nombreImpresora, nombreTrabajo)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)) métodos. La captura de pantalla muestra cómo se ve el nombre del trabajo en la cola de la impresora.
