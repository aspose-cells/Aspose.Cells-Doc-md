{{% alert color="primary" %}}

Puede especificar el nombre del trabajo o documento al imprimir su libro o hoja de cálculo usando los objetos [**WorkbookRender**](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookRender) o [**SheetRender**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender). Aspose.Cells proporciona los métodos [**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter-java.lang.String-java.lang.String-) y [**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter-java.lang.String-java.lang.String-) que puede utilizar para especificar el nombre del trabajo al imprimir su libro o hoja de cálculo.

{{% /alert %}}

## **Especificar nombre de trabajo o documento al imprimir con Aspose.Cells**

El código de muestra carga el archivo de Excel de origen y luego lo envía a la impresora especificando el nombre del trabajo o documento mediante los métodos [**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter-java.lang.String-java.lang.String-) y [**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter-java.lang.String-java.lang.String-). La captura de pantalla muestra cómo se ve el nombre del trabajo en la cola de impresión.
