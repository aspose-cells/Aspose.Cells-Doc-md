{{% alert color="primary" %}}

Microsoft Excel proporciona una función que permite ajustar automáticamente la altura de una celda según su contenido. La función se llama autofit rows. Microsoft Excel no establece operación autofit en celdas fusionadas de forma nativa. A veces la función se vuelve vital para un usuario que realmente necesita implementar autofit rows en celdas fusionadas también.

{{% /alert %}}

Aspose.Cells admite esta función a través de la API [**AutoFitterOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/AutoFitterOptions). Mediante esta API, es posible ajustar automáticamente las filas en una hoja de cálculo, incluidas las celdas fusionadas.

Después de ejecutar el código, Aspose.Cells ajusta automáticamente las filas para las celdas fusionadas.
