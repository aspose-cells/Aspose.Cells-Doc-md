## **Posibles escenarios de uso**

Puedes cambiar el*Número de versión*del archivo de Excel haciendo clic derecho en el archivo y luego seleccionando*Propiedades > Detalles*y luego editar el*Número de versión*campo. Por favor use[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#DocumentVersion)property para cambiarla mediante programación usando las API Aspose.Cells.
