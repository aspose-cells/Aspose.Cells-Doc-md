## **Este formato de archivo no es compatible o no especifica un formato correcto**
Si el usuario ha especificado el formato de archivo al crear el libro de trabajo a partir de un archivo de plantilla, normalmente este error se debe a que el formato de archivo especificado no es el formato de archivo real del archivo de plantilla. Si el usuario no ha especificado el formato de archivo, normalmente se debe a que la extensión del nombre de archivo no representa el formato de archivo real de este archivo y el formato de archivo no se puede detectar automáticamente, como el archivo csv/tsv que no tiene ningún identificador especial. .
