{{% alert color="primary" %}} 

Aspose.Cells es un componente de procesamiento de hojas de cálculo que le permite generar, leer, modificar y escribir hojas de cálculo de Excel® sin utilizar Microsoft Excel®. 

- Aspose.Cells for Java - una biblioteca de clases de Java y está diseñada para funcionar en aplicaciones Java.

{{% /alert %}} 

Aunque no hay versiones especiales de Aspose.Cells para otras plataformas, es posible usar Aspose.Cells en muchos otros lenguajes de programación, incluyendo (pero no limitado a): 

|**Lenguaje de Programación** |**Aspose.Cells a Utilizar** |**Tecnología de Interoperabilidad** |
| :- | :- | :- |
|Perl |Java |Inline::Java |
|PHP |Java |Java Bridge |
|Python |Java |JPype |
En la tabla anterior se puede ver que para algunos lenguajes de programación es posible usar Aspose.Cells for Java. La siguiente tabla puede ayudarlo a elegir: 

|**Escenario** |**Cuándo Utilizar** |
| :- | :- |
|Aspose.Cells for Java via Java Puente o en Línea::Java |<p>**Ventajas**:</p><p>- Disponible en cualquier plataforma donde Java esté disponible.</p><p>- Fácil de llamar métodos estáticos, constructores con parámetros, métodos sobrecargados y usar valores enumerados.</p>|
| |<p>**Desventajas**:</p><p>- Tiene lanzamientos menos frecuentes y menos características que Aspose.Cells for .NET.</p>|
Tenga en cuenta que Aspose.Cells for Java está diseñado para aplicaciones Java y usarlo en otros lenguajes de programación está sujeto a algunas limitaciones técnicas o soluciones alternativas.
