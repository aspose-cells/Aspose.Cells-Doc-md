{{% alert color="primary" %}} 

 Aspose.Cells es un componente de procesamiento de hojas de cálculo que le permite generar, leer, modificar y escribir hojas de cálculo de Excel® sin utilizar Microsoft Excel®.

- Aspose.Cells for Java: una biblioteca de clases Java y está diseñada para funcionar en aplicaciones Java.

{{% /alert %}} 

 Aunque no hay versiones especiales de Aspose.Cells para otras plataformas, es posible usar Aspose.Cells en muchos otros lenguajes de programación, incluidos (entre otros):

|**Lenguaje de programación** |**Aspose.Cells para usar** |**Tecnología de interoperabilidad** |
|:- |:- |:- |
| Perl|Java | En línea::Java|
| PHP|Java |Java Puente|
|Python |Java | JPype|
 En la tabla anterior, puede ver que para algunos lenguajes de programación es posible usar Aspose.Cells for Java. La siguiente tabla puede ayudarlo a elegir:

|**Guión** |**Cuándo usar** |
|:- |:- |
| Aspose.Cells for Java vía Java Puente o Inline::Java|<p>**ventajas**:</p><p>- Disponible en cualquier plataforma donde esté disponible el Java.</p><p>- Métodos estáticos fáciles de llamar, constructores con parámetros, métodos sobrecargados y usar valores enumerados.</p>|
||<p>**Contras**: </p><p>- Tiene lanzamientos menos frecuentes y menos funciones que Aspose.Cells for .NET.</p>|
 Tenga en cuenta que Aspose.Cells for Java está diseñado para aplicaciones for Java y su uso en otros lenguajes de programación está sujeto a algunas limitaciones técnicas o soluciones alternativas.
