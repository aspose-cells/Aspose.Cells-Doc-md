Esta página describe las nuevas funciones Aspose.Cells más interesantes introducidas en versiones recientes.

##  Aspose.Cells for Java 23.12

* Compatibilidad con PDF/A-2 y PDF/A-3 durante la conversión a PDF
* Admite el cálculo de agrupaciones discretas de tablas dinámicas
* Soporte para exportar datos de celdas a una lista genérica.
* Soporte para importar referencias cruzadas usando tablas vinculadas para marcadores inteligentes
* Admite la función MAKEARRAY para fórmulas

##  Aspose.Cells for Java 23.11

* Soporte para convertir hojas de cálculo al archivo PDF versión 1.7 (ISO32000-1:2008)

##  Aspose.Cells for Java 23.10

* Soporte para establecer hipervínculos directamente a un rango
* Soporte para copiar varias hojas de trabajo simultáneamente
* Soporte para verificar si un rango está vacío para los datos de las celdas

##  Aspose.Cells for Java 23.9

* Soporte para calcular nuevas funciones en fórmula: LAMBDA,LET,MAP,SCAN
* Soporte para importar/exportar archivos EPUB
* Soporte para filtrar series de gráficos.

##  Aspose.Cells for Java 23.8

* Soporte para preservar el espacio de relleno en fórmulas
* Soporte para reemplazar/insertar texto con formato para texto enriquecido de celda

##  Aspose.Cells for Java 23.7

* Detección de formato de archivo mejorada para algunos formatos nuevos, como el archivo Visio y el archivo del paquete OneNote.

##  Aspose.Cells for Java 23.6

* Soporte para exportar y resaltar registros de revisión
* Soporte para recorrer objetos Row existentes de RowCollection en orden inverso
* Soporte para actualizar múltiples rangos de consolidación de tabla dinámica

##  Aspose.Cells for Java 23.5

* Soporte para actualizar referencias externas de datos de hojas externas a la hoja local del libro de trabajo actual
* Soporte para representar el control Checkbox como control interactivo en lugar de una imagen estática en GridWeb
* Soporte para seleccionar PivotArea y formatearlo
* Soporte para agregar marca de agua al convertir a pdf

##  Aspose.Cells for Java 23.4

* Soporte para calcular la función ENCODEURL
* Soporte para completar la construcción de una forma de ecuación insertando nodos relevantes paso a paso
* Soporte para calcular la expresión de la fórmula como fórmula matricial sin configurarla en un objeto de celda
* Soporte para guardar archivos xlsb en modo LightCells
* Soporte para verificar si una fuente está instalada o disponible
* Soporte para establecer fórmula en una celda especificada en ListObject
* Soporte para guardar el libro de trabajo como xhtml
* Soporte para renderizar control activex y control de formulario para GridWeb

##  Aspose.Cells for Java 23.3

* Soporte para configurar/leer/guardar nuevas funciones: LET, ENCODEURL
* Soporte para vincular fuentes de datos a enlaces externos al llamar al cálculo de fórmulas

##  Aspose.Cells for Java 23.2

* Soporte para configurar/leer/guardar nuevas funciones: SCAN, LAMBDA
* Soporte para exportar estilo a json.

##  Aspose.Cells for Java 23.1

* Admite nuevas funciones: CHOOSEROWS, CHOOSECOLS, HSTACK, VSTACK
* Soporte para obtener el autor y la fecha y hora de las revisiones.

##  Aspose.Cells for Java 22.12

* Soporte para exportar Excel a JSON con estructura jerárquica de varias capas

##  Aspose.Cells for Java 22.11

* Soporte para distinguir entre fórmula de matriz dinámica y fórmula de matriz heredada
* Soporte para leer/guardar fórmulas de matriz dinámica para formato de archivo xlsb

##  Aspose.Cells for Java 22.10

* Soporte para agregar y actualizar gráficos con un archivo de plantilla CRTX

##  Aspose.Cells for Java 22.9

* Soporte para establecer fórmulas de tabla para celdas
* Soporte para establecer fórmulas de matriz dinámica con función personalizada y motor de cálculo
* Soporte para establecer configuraciones de degradado preestablecidas para el formato de relleno degradado
* Soporte para aplicar bordes de contorno con colores de tema para Rango

##  Aspose.Cells for Java 22.8

* Admite la exportación de comentarios y notas como se muestran en la hoja a html
* Admite especificar hojas de trabajo para generar mientras se convierte a pdf/xps

##  Aspose.Cells for Java 22.7

* Soporte para leer archivos de plantilla en formato OOXML que contienen datos zip anormales y no pueden ser procesados por ZipInputStream de Java
* Soporte para exportar HTML a un solo archivo con encabezado/pie de página
* Soporte para obtener la posición absoluta y el tamaño del rango (en puntos)

##  Aspose.Cells for Java 22.6

* Mejora para calcular y rastrear fórmulas con cadena de cálculo de fórmulas

##  Aspose.Cells for Java 22.5

* Mejora del rendimiento de la memoria al configurar la fórmula de las celdas al guardar el libro en el modo LightCells

##  Aspose.Cells for Java 22.4

* Opciones mejoradas para controlar celdas en blanco para exportar CSV.
* Admite división complicada y actualización de formato condicional para operaciones de inserción/eliminación

##  Aspose.Cells for Java 22.3

* Soporte para volver a calcular celdas que dependen del nombre definido cuando se cambia el nombre definido y la cadena de cálculo del libro está habilitada.
* Soporte para obtener hojas de celda de forma recursiva según la cadena de cálculo de fórmulas.

##  Aspose.Cells for Java 22.2

* Soporte para extraer nombres definidos de NameCollection mediante un filtro de alcance específico

##  Aspose.Cells for Java 22.1

* Soporte para generar pptx editable
* Soporte para importar csv grande en varias hojas de trabajo cuando la fila/columna de datos excede el límite
* Soporte para importar csv con LightCells
* Soporte para eliminar el enlace externo especificado
* Soporte para mantener fórmulas mientras se eliminan enlaces externos

##  Aspose.Cells for Java 21.12

* Soporte para convertir Excel a scripts SQL
* Admite la importación y exportación de datos xml sin formato
* Soporte para leer la mayor cantidad de datos posible de un archivo xls dañado
* Soporte para interrumpir el proceso de SmartMarker
* Mejorar el rendimiento al agregar rangos para validación y formato condicional
* Establezca la prioridad más alta para la condición de formato recién agregada, al igual que el comportamiento de MS Excel.

##  Aspose.Cells for Java 21.11

* Admite el cálculo de la fórmula/función XLookup.
* Integre guardar json e imágenes para guardar el libro de trabajo.
* Mejore el rendimiento al convertir Excel a HTML.
