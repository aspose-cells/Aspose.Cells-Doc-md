Esta página describe las características nuevas más interesantes de Aspose.Cells introducidas en versiones recientes.

##  Aspose.Cells for Java 23.5

* Soporte para actualizar referencias externas de datos de hojas externas a la hoja local del libro de trabajo actual
* Compatibilidad para representar el control Checkbox como un control interactivo en lugar de una imagen estática en GridWeb
* Soporte para seleccionar PivotArea y formatearlo
* Soporte para agregar marca de agua al convertir a pdf

##  Aspose.Cells for Java 23.4

* Soporte para calcular la función ENCODEURL
* Soporte para completar la construcción de una forma de ecuación insertando nodos relevantes paso a paso
* Soporte para calcular la expresión de fórmula como fórmula de matriz sin establecerla en un objeto de celda
* Soporte para guardar archivos xlsb en modo LightCells
* Soporte para verificar si una fuente está instalada o disponible
* Soporte para establecer la fórmula en la celda especificada en ListObject
* Soporte para guardar el libro de trabajo como xhtml
* Soporte para renderizar control activex y control de formulario para GridWeb

##  Aspose.Cells for Java 23.3

* Soporte para configurar/leer/guardar nuevas funciones: LET, ENCODEURL
* Soporte para vincular fuentes de datos a enlaces externos al llamar al cálculo de fórmulas

##  Aspose.Cells for Java 23.2

* Soporte para configurar/leer/guardar nuevas funciones: SCAN, LAMBDA
* Soporte para exportar estilo a json

##  Aspose.Cells for Java 23.1

* Admite nuevas funciones: CHOOSEROWS, CHOOSECOLS, HSTACK, VSTACK
* Soporte para obtener el autor y la fecha y la hora de las revisiones.

##  Aspose.Cells for Java 22.12

* Soporte para exportar Excel a JSON con estructura de jerarquía de múltiples capas

##  Aspose.Cells for Java 22.11

* Compatibilidad para distinguir entre la fórmula de matriz dinámica y la fórmula de matriz heredada
* Soporte para leer/guardar fórmulas de matrices dinámicas para formato de archivo xlsb

##  Aspose.Cells for Java 22.10

* Soporte para agregar y actualizar el gráfico con el archivo de plantilla CRTX

##  Aspose.Cells for Java 22.9

* Soporte para establecer la fórmula de la tabla para las celdas.
* Soporte para establecer una fórmula de matriz dinámica con función personalizada y motor de cálculo
* Soporte para establecer configuraciones de gradiente preestablecidas para el formato de relleno de gradiente
* Soporte para aplicar bordes de contorno con colores de tema para rango

##  Aspose.Cells for Java 22.8

* Admite la exportación de comentarios y notas como se muestra en la hoja a html
* Admite la especificación de hojas de trabajo para la salida durante la conversión a pdf/xps

##  Aspose.Cells for Java 22.7

* Soporte para leer archivos de plantilla de formato OOXML que contienen datos zip anormales y no pueden ser procesados por ZipInputStream de Java
* Soporte para exportar html a un solo archivo con encabezado/pie de página
* Soporte para obtener la posición absoluta y el tamaño del Rango (en puntos)

##  Aspose.Cells for Java 22.6

* Mejora para calcular y rastrear fórmulas con cadena de cálculo de fórmulas

##  Aspose.Cells for Java 22.5

* Mejora en el rendimiento de la memoria al configurar la fórmula de las celdas al guardar el libro de trabajo en el modo LightCells

##  Aspose.Cells for Java 22.4

* Opciones mejoradas para controlar celdas en blanco para exportar CSV.
* Admite división complicada y actualización de formato condicional para operaciones de inserción/eliminación

##  Aspose.Cells for Java 22.3

* Soporte para volver a calcular las celdas que dependen del nombre definido cuando se cambia el nombre definido y se habilita la cadena de cálculo del libro.
* Soporte para obtener hojas de celda recursivamente de acuerdo con la cadena de cálculo de fórmulas

##  Aspose.Cells for Java 22.2

* Soporte para extraer nombres definidos de NameCollection por filtro de alcance especificado

##  Aspose.Cells for Java 22.1

* Soporte para generar pptx editables
* Soporte para importar csv grande en varias hojas de trabajo cuando la fila/columna de datos excede el límite
* Soporte para importar csv con LightCells
* Soporte para eliminar el enlace externo especificado
* Soporte para mantener fórmulas mientras se eliminan enlaces externos

##  Aspose.Cells for Java 21.12

* Admite la conversión de Excel a SQL Scripts
* Admite la importación y exportación de datos xml sin formato
* Soporte para leer la mayor cantidad de datos posible del archivo xls dañado
* Soporte para interrumpir el proceso de SmartMarker
* Mejore el rendimiento para agregar rangos para formato y validación condicionales
* Establezca la prioridad más alta para la condición de formato recién agregada, al igual que el comportamiento de MS Excel

##  Aspose.Cells for Java 21.11

* Admite el cálculo de la fórmula/función XLookup.
* Integre guardar json e imágenes para guardar el libro de trabajo.
* Mejore el rendimiento al convertir Excel a HTML.
