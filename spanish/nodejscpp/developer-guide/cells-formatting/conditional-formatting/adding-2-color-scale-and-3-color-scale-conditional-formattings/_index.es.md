{{% alert color="primary" %}}  
El formateo condicional de escalas de **2 colores** y **3 colores** se agrega de la misma manera, salvo que se diferencian con el método [**ColorScale.setIs3ColorScale(boolean)**](https://reference.aspose.com/cells/nodejs-cpp/colorscale/#setIs3ColorScale-boolean-). Este método es **falso** para la escala de 2 colores y **verdadero** para la escala de 3 colores.
