Aspose.Cells for Node.js via C++ proporciona la capacidad de guardar la tabla dinámica en archivos ODS. Para ello, simplemente convierte un libro de trabajo con una tabla dinámica existente o crea una nueva y guarda el archivo en formato ODS. Asegúrate de llamar a [**PivotTable.calculateData**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#calculatedata--) antes de guardar para garantizar que la tabla dinámica se renderice en el archivo ODS de salida.
