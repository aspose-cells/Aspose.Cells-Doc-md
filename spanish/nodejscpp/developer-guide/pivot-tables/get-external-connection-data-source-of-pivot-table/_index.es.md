## **Cómo obtener la fuente de datos de conexión externa de la tabla dinámica.**

Aspose.Cells for Node.js via C++ proporciona la capacidad de obtener la fuente de datos de conexión externa de la tabla dinámica. Para esto, la API proporciona el método [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) de la clase [**PivotTable**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/). El método [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) devuelve el objeto [**ExternalConnection**](https://reference.aspose.com/cells/nodejs-cpp/externalconnection/). El siguiente fragmento de código demuestra el uso del método [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) para obtener la fuente de datos de conexión externa de la tabla dinámica.
