{{% alert color="primary" %}}

Puedes configurar diferentes opciones de tabla dinámica usando Aspose.Cells for Node.js via C++. Una de estas opciones es "Mostrar en celdas vacías". Al configurar esta opción, todas las celdas vacías en una tabla dinámica se muestran como una cadena específica.

{{% /alert %}}

## **Configuración de opción de tabla dinámica en Microsoft Excel**

Para encontrar y configurar esta opción en Microsoft Excel:

1. Seleccione una tabla dinámica y haga clic derecho.
2. Seleccione **Opciones de tabla dinámica**.
3. Seleccione la pestaña **Diseño y formato**.
4. Seleccione la opción **Mostrar celdas vacías** y especifique una cadena.

## **Configurando opciones de la tabla dinámica usando Aspose.Cells for Node.js via C++**

Aspose.Cells for Node.js via C++ proporciona las propiedades [**PivotTable.setDisplayNullString**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#setDisplayNullString-boolean-) y [**PivotTable.setNullString**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#setNullString-string-) para configurar la opción "Mostrar en celdas vacías" en la tabla dinámica.
