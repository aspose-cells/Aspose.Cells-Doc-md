## **Mostrar opción de páginas de filtro de reporte**
Excel soporta crear tablas dinámicas, agregar filtros de informes y habilitar la opción "Mostrar páginas de filtros de informes". Aspose.Cells for Node.js via C++ también soporta esta función para habilitar la opción "Mostrar páginas de filtros de informes" en la tabla dinámica creada. A continuación se muestra la pantalla con la opción "Mostrar páginas de filtros de informes" en Excel.
