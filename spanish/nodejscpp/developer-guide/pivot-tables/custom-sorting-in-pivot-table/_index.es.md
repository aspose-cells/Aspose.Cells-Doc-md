## **Cómo establecer un orden personalizado en la tabla dinámica usando la biblioteca Aspose.Cells for Node.js via C++**
Usando la API Aspose.Cells for Node.js via C++, puedes ordenar tablas dinámicas por valores de campo. El siguiente fragmento de código carga un archivo de Excel de ejemplo y agrega tres tablas dinámicas. La primera sin orden personalizado, la segunda ordenada en base a los valores del campo de fila "SeaFood" y la tercera ordenada en base a los valores del campo de columna "28/07/2000".
