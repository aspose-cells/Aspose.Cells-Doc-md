## **Obtener validación de celda en archivos ODS**  

Con Aspose.Cells for Node.js via C++, puedes obtener la validación aplicada a una celda en archivos ODS. Para esto, la API proporciona el método [**Cell.getValidation()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getValidation--) de la clase [**Cell**](https://reference.aspose.com/cells/nodejs-cpp/cell/).  

El siguiente ejemplo de código demuestra cómo usar el método [**Cell.getValidation()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getValidation--) cargando el archivo [fuente ODS](101089354.ods) y leyendo la validación de la celda A9.
