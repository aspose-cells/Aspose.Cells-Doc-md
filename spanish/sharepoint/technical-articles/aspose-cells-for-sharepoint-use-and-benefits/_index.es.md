Se ha introducido Aspose.Cells for SharePoint para realizar ciertas operaciones en documentos de Microsoft Excel disponibles en las bibliotecas de documentos de SharePoint. ### **Una breve nota sobre las bibliotecas de documentos de SharePoint**
Los contenidos de los sitios de SharePoint están en forma de documentos compartidos en un lugar centralizado llamado biblioteca de documentos de SharePoint (a veces conocida como "biblioteca de documentos compartidos"). Los usuarios autorizados pueden subir documentos desde su máquina local al sitio de SharePoint. Estos documentos se almacenan en formato binario en una base de datos de SQL Server utilizada por el sitio de SharePoint. Los documentos están disponibles para otros usuarios de SharePoint en forma de una lista de hipervínculos como se muestra a continuación:

**Una biblioteca de documentos** 


### **Beneficios**
La siguiente lista destaca algunos beneficios de usar Aspose.Cells for SharePoint:

- El servidor del sitio de SharePoint no tiene la capacidad incorporada para convertir archivos de Microsoft Excel directamente en el servidor. Aspose.Cells for SharePoint agrega esta característica al servidor del sitio de SharePoint.
- Aspose.Cells for SharePoint no requiere Microsoft Excel para funcionar. Está instalado en el servidor, por lo que los usuarios de SharePoint no necesitan Microsoft Excel en sus sistemas locales para convertir documentos de Microsoft Excel de un formato a otro.
- Aspose.Cells for SharePoint no requiere ningún prerrequisito, excepto los requeridos para crear un sitio de SharePoint.
- Aspose.Cells for SharePoint es [fácil de implementar en un servidor de sitio de SharePoint](/cells/es/sharepoint/installing-aspose-cells-for-sharepoint/).
- Aspose.Cells for SharePoint admite [varios formatos de entrada y salida](/cells/es/sharepoint/multiple-format-support/).
- Aspose.Cells for SharePoint expone una API pública para los desarrolladores que necesitan desarrollar sus propias soluciones de SharePoint.
### **Convertir documentos de Microsoft Excel con Aspose.Cells for SharePoint**
Una vez que Aspose.Cells for SharePoint esté instalado en el servidor de SharePoint, la opción **Convertir con Aspose.Cells.SharePoint** está disponible en el menú contextual: 

**Encontrar la opción Convertir con...** 

Para convertir un documento de Microsoft Excel de su biblioteca de documentos de SharePoint a otros formatos, siga los siguientes pasos:

1. Haga clic en el documento de Microsoft Excel deseado y luego haga clic en Convertir con Aspose.Cells for SharePoint
   Se muestra la pantalla de Configuración de conversión. 

   **Decidir convertir** 


1. Seleccione el formato de salida.
   Aquí, también puede cambiar el nombre del archivo de salida y la carpeta de destino.
1. Haga clic en **Convertir**. 

   **Seleccionar el formato de salida** 



1. Cuando la conversión esté completa, se muestra un mensaje de éxito. 

   **Conversión exitosa** 



1. Haga clic en **Volver** para regresar a la biblioteca de documentos.
1. El documento convertido se lista en la biblioteca de documentos de SharePoint. 

   **El archivo convertido se lista en la biblioteca de documentos** 


### **Resumen**
{{% alert color="primary" %}} 

Instalar y utilizar Aspose.Cells for SharePoint es fácil. No solo agrega una función para convertir archivos de Microsoft Excel directamente en el servidor de SharePoint, sino que también elimina la necesidad de que Microsoft Excel esté instalado en las máquinas de los clientes. Además, los desarrolladores pueden utilizar la API pública de Aspose.Cells for SharePoint para desarrollar sus propias soluciones de SharePoint.

En otras palabras, es beneficioso para los administradores de sitios de SharePoint, los desarrolladores y los usuarios finales.
