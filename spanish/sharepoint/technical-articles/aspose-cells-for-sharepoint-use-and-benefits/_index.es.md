Aspose.Cells para SharePoint se introdujo para realizar ciertas operaciones en Microsoft documentos de Excel disponibles en las bibliotecas de documentos de SharePoint. ### **Una breve nota sobre las bibliotecas de documentos de SharePoint**
El contenido de los sitios de SharePoint tiene la forma de documentos compartidos en una ubicación centralizada denominada biblioteca de documentos de SharePoint (a veces conocida como "biblioteca de documentos compartida"). Los usuarios autorizados pueden cargar documentos desde su máquina local al sitio de SharePoint. Estos documentos se almacenan en formato binario en una base de datos de SQL Server utilizada por el sitio de SharePoint. Los documentos están disponibles para otros usuarios de SharePoint en forma de una lista de hipervínculos, como se muestra a continuación:

**Una biblioteca de documentos** 


### **Beneficios**
La siguiente lista destaca algunos beneficios de usar Aspose.Cells para SharePoint:

- El servidor del sitio de SharePoint no tiene capacidad integrada para convertir Microsoft archivos de Excel directamente en el servidor. Aspose.Cells para SharePoint agrega esta función al servidor del sitio de SharePoint.
- Aspose.Cells para SharePoint no requiere Microsoft Excel para funcionar. Se instala en el servidor para que los usuarios de SharePoint no necesiten Microsoft Excel en sus sistemas locales para convertir Microsoft documentos de Excel de un formato a otro.
- Aspose.Cells para SharePoint no requiere ningún requisito previo excepto los necesarios para crear un sitio de SharePoint.
-  Aspose.Cells para SharePoint es[fácil de implementar en un servidor de sitio de SharePoint](/cells/es/sharepoint/installing-aspose-cells-for-sharepoint/).
-  Aspose.Cells para compatibilidad con SharePoint[varios formatos de entrada y salida](/cells/es/sharepoint/multiple-format-support/).
- Aspose.Cells para SharePoint expone un API público para desarrolladores que necesitan desarrollar sus propias soluciones de SharePoint.
### **Conversión de documentos de Excel Microsoft con Aspose.Cells para SharePoint**
 Una vez que Aspose.Cells para SharePoint está instalado en el servidor de SharePoint, la opción**Convertir con Aspose.Cells.SharePoint** está disponible en el menú contextual:

**Encontrar la opción Convertir con...** 

Para convertir un documento de Excel Microsoft de su biblioteca de documentos de SharePoint a otros formatos, siga los pasos a continuación:

1. Haga clic en el documento de Excel Microsoft deseado y luego haga clic en Convertir con Aspose.Cells para SharePoint
 Se muestra la pantalla Configuración de conversión.

   **Decidir convertir** 


1. Seleccione el formato de salida.
 Aquí, también puede cambiar el nombre del archivo de salida y la carpeta de destino.
1.  Hacer clic**Convertir**. 

   **Selección del formato de salida** 



1.  Cuando se completa la conversión, se muestra un mensaje de éxito.

   **Conversión exitosa** 



1.  Hacer clic**Devolver** para volver a la biblioteca de documentos.
1.  El documento convertido aparece en la biblioteca de documentos de SharePoint.

   **El archivo convertido aparece en la biblioteca de documentos.** 


### **Resumen**
{{% alert color="primary" %}} 

Instalar y usar Aspose.Cells para SharePoint es fácil. No solo agrega una función para convertir archivos Microsoft Excel directamente en el servidor de SharePoint, sino que también elimina la necesidad de instalar Microsoft Excel en las máquinas cliente. Además, los desarrolladores pueden usar Aspose.Cells para el API público de SharePoint para desarrollar sus propias soluciones de SharePoint.

 En otras palabras, es beneficioso para los administradores, desarrolladores y usuarios finales del sitio de SharePoint.
