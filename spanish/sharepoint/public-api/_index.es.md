{{% alert color="primary" %}} 

Los clientes que desarrollan sus propias soluciones de SharePoint pueden obtener acceso a nuestro motor de conversión utilizando el API público. Esto incluye todos los formatos de archivo y opciones de conversión admitidos. Las especificaciones detalladas de la interfaz en nuestro[API Referencia](https://reference.aspose.com/) se acompaña con diferentes muestras C# y VB.

{{% /alert %}}
