{{% alert color="primary" %}} 

Los clientes que desarrollan sus propias soluciones de SharePoint pueden acceder a nuestra motor de conversión utilizando la API pública. Esto incluye todos los formatos de archivo admitidos y opciones de conversión. Las especificaciones detalladas de la interfaz en nuestra [Referencia de API](https://reference.aspose.com/) van acompañadas de diferentes ejemplos en C# y VB.

{{% /alert %}}
