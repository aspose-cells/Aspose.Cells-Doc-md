{{% alert color="primary" %}} 

Una de las tareas más comunes que los usuarios realizan con Aspose.Cells es la conversión de archivos individuales. 

{{% /alert %}} 
### **Convertir archivos individuales**
Para convertir un documento específico en una biblioteca de documentos: 

1. Seleccionar **Convertir con Aspose.Cells.SharePoint** en el Bloque de Control de Edición (ECB) del documento. 

   **Seleccionar el elemento 'Convertir a través de Aspose.Cells' en el ECB del documento** 

![todo:image_alt_text](convert-one-file_1.png)



Se muestra la pantalla de configuración de conversión. Aquí puedes ajustar la conversión:

1. Seleccione el formato de archivo de destino de la lista **Convertir a**.
1. Ingrese el nombre del archivo de salida en el campo **Archivo de destino**.
   Por defecto, Aspose.Cells for SharePoint completa este campo con el nombre del archivo fuente. 
   La extensión se agrega automáticamente según el formato seleccionado.
1. Inicie la conversión haciendo clic en **Convertir**.
   (O regrese a la biblioteca, descartando su configuración, haciendo clic en **Cancelar**.)
