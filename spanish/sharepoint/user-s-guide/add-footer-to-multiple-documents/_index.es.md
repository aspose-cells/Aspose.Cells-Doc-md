Si desea agregar un pie de página en varios archivos de Excel, elija la opción "Agregar pie de página con Aspose.Cells" en la barra de opciones.

![todo:image_alt_text](add-footer-to-multiple-documents_1.png)



Obtenga todos los archivos de Excel de la carpeta de origen de datos y cree una tabla de lista de archivos.

Seleccione el archivo al que se le debe agregar el pie de página, haga clic en el botón **Agregar pie de página** para agregar el pie de página a los archivos seleccionados. 

![todo:image_alt_text](add-footer-to-multiple-documents_2.png)



Las siguientes opciones están disponibles durante la configuración de agregación del pie de página:

**Sección**

Agregar posición del pie de página: Sección izquierda, Sección central y Sección derecha.

**Script del pie de página**

It represents Footer formatting script. Script commands: Command | Description| &P Current page number| &N Page count|&D Current date| &T Current time &A Sheet name &F File name without path &"<FontName>" Font name, for example: &"Arial" &"<FontName>, <FontStyle>" Font name and font style, for example: &"Arial,Bold" &<FontSize> Font size. If this command is followed by a plain number to be printed in the header, it will be separated from the font height with a space character. &G Image script For example: "&Arial,Bold&8Footer Note".
