{{% alert color="primary" %}}

Con Aspose.Cells for SharePoint puedes convertir documentos entre muchos formatos populares de documentos de oficina de una biblioteca de documentos de SharePoint. La mayoría de las conversiones se realizan con alta fidelidad y precisión.

{{% /alert %}}

## **Formatos de entrada admitidos**

Aspose.Cells for SharePoint admite los siguientes formatos de entrada:

- XLS – Microsoft Excel 97~2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- CSV - Documento de hoja de cálculo de valores separados por comas
- ODS – OpenDocument
- XML – Microsoft Excel SpreadsheetML

Para generar documentos, Aspose.Cells for SharePoint depende de una versión integrada de [Aspose.Cells for .NET](https://products.aspose.com/cells/net/), el componente líder en el procesamiento de hojas de cálculo de Aspose. Microsoft Excel no se utiliza para generar documentos.

## **Formatos de salida admitidos**

La lista completa de formatos de salida admitidos por Aspose.Cells for SharePoint es:

- PDF - Documento portátil de Adobe
- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM – Office Open XML for macros enabled
- TXT – Plain Text
- XML – Microsoft Excel SpreadsheetML
- ODS – OpenDocument
- CSV – Comma Separated Values
- XLSB – Microsoft Excel Binary 2007
- MHTML – Multipurpose Internet Mail Extension HTML
- XPS – XML Paper Specification
- TIFF – Tagged Image File Format
- SVG – Scalable Vector Graphics
- DIF – Drawing Interchange Format
