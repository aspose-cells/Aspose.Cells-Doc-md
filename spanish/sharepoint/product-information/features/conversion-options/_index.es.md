{{% alert color="primary" %}} 

Para que el proceso de conversión sea lo más flexible posible, Aspose.Cells for SharePoints le permite establecer opciones que definen cómo se tratará el archivo convertido.

{{% /alert %}} 
### **Opciones**
Las siguientes opciones están disponibles durante la conversión:
#### **Concatenar archivos**
Seleccione **Combinar archivos** para combinar archivos que originalmente estaban en diferentes formatos en un solo documento. 

Esta opción está disponible cuando selecciona una carpeta o varios archivos individuales para la conversión. Mueva los documentos en la lista hacia arriba o hacia abajo para establecer el orden en el que desea que se combinen los documentos.

#### **Convertir archivos en subcarpetas**
Seleccione **Convertir archivos en subcarpetas** para convertir de forma recursiva todos los documentos y subcarpetas en una carpeta seleccionada. 

La estructura de subcarpetas existente se crea en la carpeta de destino.

{{% alert color="primary" %}} 

Esta opción solo está disponible en SharePoint 2007. En SharePoint 2010, las subcarpetas se procesan de forma predeterminada. Para procesar solo archivos en una carpeta raíz en SharePoint 2010, seleccione los archivos individuales en lugar de la carpeta y no seleccione ninguna subcarpeta.

{{% /alert %}} 
#### **Sobrescribir archivos existentes en la carpeta de destino**
Si la carpeta de destino ya contiene archivos con el mismo nombre que los que deseas convertir, tienes dos opciones. Selecciona sobrescribir archivos existentes para reemplazar los archivos ya existentes con los recién convertidos. Desactiva la opción para omitir el procesamiento de archivos con el mismo nombre.
