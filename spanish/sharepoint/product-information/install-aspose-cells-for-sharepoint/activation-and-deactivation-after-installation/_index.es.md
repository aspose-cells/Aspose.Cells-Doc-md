{{% alert color="primary" %}} 

[Durante la instalación](/cells/es/sharepoint/installing-aspose-cells-for-sharepoint/) , Aspose.Cells para SharePoint es[activado](/cells/es/sharepoint/deployment-and-activation/) para todas las colecciones de sitios seleccionadas. Después de la instalación, utilice el**Acciones del sitio** menú en el sitio web raíz de una colección de sitios para activar y desactivar Aspose.Cells para SharePoint.
