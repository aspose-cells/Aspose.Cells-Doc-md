{{% alert color="primary" %}} 

Para desinstalar Aspose.Cells for SharePoint, ejecute el programa de instalación. Si Aspose.Cells for SharePoint ya está instalado, el programa de instalación sugiere desinstalarlo o repararlo.

{{% /alert %}} 
### **Desinstalación**
Durante la eliminación, el programa de instalación desactiva Aspose.Cells for SharePoint para todas las colecciones de sitios y retrae la solución del conjunto de servidores. 

**Desinstalación de Aspose.Cells** 

![todo:image_alt_text](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

Tenga en cuenta que al ejecutar la opción **Reparar** no reactiva Aspose.Cells for SharePoint en las colecciones de sitios. En su lugar, la característica debe activarse manualmente después de la reimplantación con la opción **Reparar**.

Desinstalar utilizando la opción **Eliminar** y luego volver a instalar puede ser más conveniente, ya que le permite seleccionar aplicaciones web y colecciones de sitios de destino durante la instalación. 

{{% /alert %}}
