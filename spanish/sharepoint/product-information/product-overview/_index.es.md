### **¡Bienvenido al Aspose.Cells para la documentación de SharePoint!**
Aspose.Cells para SharePoint es una solución flexible compatible con la conversión de hojas de cálculo que permite convertir documentos de MS Excel dentro de las aplicaciones Microsoft de SharePoint.

{{% /alert %}} 

Puede convertir múltiples formatos de documentos, como:

- PDF - Documento portátil de Adobe
- XLS-MS Excel 97~2003
- XLSX - Oficina XML abierta
- XLSM: Office Open XML para macros habilitado
- TXT - Texto sin formato
- XML - Microsoft Excel Hoja de cálculo ML
- ODS - Hoja de cálculo OpenDocument
- CSV - Valores separados por comas
- XLSB - Excel Binario 2007
- MHtml - HTML de extensión multipropósito de correo de Internet
- XPS - Especificación de papel XML
- TIFF: formato de archivo de imagen etiquetado
- SVG - Gráficos vectoriales escalables
- DIF - Formato de intercambio de dibujo

Aspose.Cells para SharePoint está diseñado para usarse con los siguientes productos:

- Windows Servicios de SharePoint 3.0 (WSS)
- Microsoft Office SharePoint Server 2007 (MOSS) Estándar
- Microsoft Office SharePoint Server 2007 (MOSS) Empresa
- Microsoft Office SharePoint Server 2010 (MOSS) Estándar
- Microsoft Office SharePoint Server 2010 (MOSS) Empresa
- Microsoft Office SharePoint Server 2013 (MOSS) Estándar
- Microsoft Office SharePoint Server 2013 (MOSS) Empresa
- Microsoft Office SharePoint Server 2016 (MOSS) estándar
- Microsoft Office SharePoint Server 2016 (MOSS) Empresa

No hay requisitos de sistema adicionales más allá de los productos anteriores.
