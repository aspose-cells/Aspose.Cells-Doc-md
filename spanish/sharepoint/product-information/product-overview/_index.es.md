### **¡Bienvenido a la Documentación de Aspose.Cells for SharePoint!**
Aspose.Cells for SharePoint es una solución flexible que admite la conversión de hojas de cálculo, lo que permite convertir documentos de MS Excel dentro de las aplicaciones de Microsoft SharePoint.

{{% /alert %}} 

Puede convertir múltiples formatos de documentos, tales como:

- PDF - Documento portátil de Adobe
- XLS - MS Excel 97~2003
- XLSX - Office Open XML
- XLSM - Office Open XML con macros habilitados
- TXT - Texto sin formato
- XML - Microsoft Excel SpreadsheetML
- ODS - Hoja de cálculo de documento OpenDocument
- CSV - Valores separados por comas
- XLSB - Binario de Excel 2007
- MHtml - HTML de extensión de correo electrónico multipropósito
- XPS - XML Paper Specification
- TIFF - Tagged Image File Format
- SVG - Scalable Vector Graphics
- DIF - Drawing Interchange Format

Aspose.Cells for SharePoint está diseñado para ser utilizado con los siguientes productos:

- Windows SharePoint Services 3.0 (WSS)
- Microsoft Office SharePoint Server 2007 (MOSS) Standard
- Microsoft Office SharePoint Server 2007 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2010 (MOSS) Standard
- Microsoft Office SharePoint Server 2010 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2013 (MOSS) Standard
- Microsoft Office SharePoint Server 2013 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2016 (MOSS) Standard
- Microsoft Office SharePoint Server 2016 (MOSS) Enterprise

No hay requisitos adicionales del sistema más allá de los productos anteriores.

Esta documentación describe las [características](/cells/es/sharepoint/features/), [instalación](/cells/es/sharepoint/install-aspose-cells-for-sharepoint/), [evaluación](/cells/es/sharepoint/evaluate-aspose-cells/), [licencia](/cells/es/sharepoint/license-aspose-cells-for-sharepoint/), casos de uso comunes y configuraciones de Aspose.Cells for SharePoint.
