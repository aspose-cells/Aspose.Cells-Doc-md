## **Introducción**
 Por favor use[IPivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table_collection#a903a0eb3d7ef995c370f4a4385b25111) método para crear una tabla dinámica dentro de la hoja de cálculo. Una vez que se crea la tabla dinámica, puede trabajar con ella usando el[IPivotTable](https://reference.aspose.com/cells/cpp/class/aspose.cells.pivot.i_pivot_table)clase.
