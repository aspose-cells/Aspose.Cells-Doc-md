{{% alert color="primary" %}} 

Utilice WordArt para agregar texto especial o efectos de imagen a hojas de cálculo. Por ejemplo, estire un título en la parte superior del archivo, decore el texto y haga que el texto se ajuste a una forma preestablecida, o aplique texto a una hoja de Excel como marca de agua de fondo. El WordArt se convierte en un objeto que puedes mover o colocar en hojas de cálculo para agregar decoración.

{{% /alert %}}
