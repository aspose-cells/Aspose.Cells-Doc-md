## **Posibles escenarios de uso**
Microsoft Excel admite muchos formatos como XLS, XLSX, XLSM, XLSB, CSV, etc. Aspose.Cells también admite muchos de estos formatos. Este artículo explica cómo leer y escribir el archivo de Excel con formato XLSB usando Aspose.Cells.
## **Leer y escribir formato de archivo XLSB**
 El siguiente código de ejemplo carga el[archivo XLSB fuente](23166992.xlsb) y lee su celda A1 y luego copia su contenido a la celda C4 y lo guarda como[archivo XLSB de salida](23166993.xlsb).
