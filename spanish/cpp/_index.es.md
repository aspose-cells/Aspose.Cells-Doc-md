**Bienvenido a Aspose.Cells for C++**

Aspose.Cells for C++ es una biblioteca nativa de C++ que permite a los desarrolladores de C++ crear, manipular y convertir hojas de cálculo programáticamente sin necesidad de Automatización de Office o la aplicación Microsoft Excel. Aspose.Cells for C++ soporta formatos de hojas de cálculo como MS Excel 97-2003 (XLS), MS Excel 2007-2013/2016 (XLSX, XLSM, XLSB, etc.), Open Office XML y otros formatos. También puedes renderizar imágenes desde hojas de cálculo y convertir hojas de cálculo de MS Excel a formato PDF. La API también te permite crear, manipular y renderizar gráficos y formas. Además, los cálculos de fórmulas son otro buen aspecto proporcionado por el componente y puedes utilizarlo.
