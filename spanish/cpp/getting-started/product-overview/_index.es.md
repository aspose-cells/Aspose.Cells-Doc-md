**Bienvenido a la sección de documentación Aspose.Cells para C++**

Aspose.Cells para C++ es una biblioteca nativa de C++ que permite a los desarrolladores de C++ crear, manipular y convertir hojas de cálculo mediante programación sin necesidad de Office Automation o la aplicación Microsoft de Excel.

El Aspose.Cells para C++ admite formatos de hoja de cálculo como el formato de libro de Excel 97-2003 y los formatos XML de Open Office.

{{% /alert %}}

## **Beneficios**

Usar Aspose.Cells para C++ en sus aplicaciones le brinda los siguientes beneficios:

- Características ricas
- Independencia de la plataforma
- Rendimiento y escalabilidad

### **Características enriquecidas**

Aspose.Cells para C++ ofrece una amplia variedad de funciones para crear, convertir y manipular hojas de cálculo:

- Propiedades integradas
- Propiedades personalizadas
- Temas
- Estilos y formato
- Validación de datos
- Formato condicional
- Hipervínculo
- Autofiltro
- Configuración de página
- Leer, escribir y calcular fórmulas
- Agrupar Filas y Columnas
- Tabla dinámica
- Mesa

### **Independencia de la plataforma**

Utilice Aspose.Cells para C++ para crear cualquier tipo de aplicación C++ de 32 y 64 bits.

### **Rendimiento y escalabilidad**

Aspose.Cells para C++ está diseñado para funcionar igualmente bien en el servidor o en el lado del cliente. Aspose.Cells para C++ es un ensamblaje nativo que se puede implementar simplemente copiándolo. No tienes que preocuparte por otros servicios o módulos.

## **Apoyo técnico**

Aspose brinda soporte técnico ilimitado y gratuito para todos sus productos. El soporte está disponible para todos los usuarios, incluida la evaluación.

 Si necesita ayuda con Aspose.Cells para C++[visita el Aspose.Forums](https://forum.aspose.com/c/cells/9). Publique su pregunta en el foro Aspose.Cells para C++ y será respondida dentro de unas horas. Las preguntas son respondidas directamente por el equipo de soporte Aspose.Cells para C++.

Cuando espere una respuesta en los foros, tenga en cuenta las diferencias de zona horaria.

Si tiene un problema con Aspose.Cells para C++, siga estos sencillos pasos para asegurarse de que se resuelva de la manera más eficiente:

- Asegúrese de usar la última versión Aspose.Cells para C++ antes de informar el problema.
- Cuando informe un problema, incluya el documento original y posiblemente un fragmento del código que causa el problema. Si necesita adjuntar varios archivos, comprímalos en uno. Es seguro adjuntar sus documentos en Aspose.Forums ya que solo usted y los desarrolladores de Aspose tendrán acceso a los archivos adjuntos.
- Intente informar un problema por hilo. Si tienes otro problema, repórtalo en un hilo separado.
