## **Requisitos de software**
Asegúrese de cumplir con los siguientes requisitos antes de descargar y ejecutar los ejemplos.

- Visual Studio 2015 o superior
## **Descargar desde GitHub**
Todos los ejemplos de Aspose.Cells for C++ están alojados en [GitHub](https://github.com/aspose-cells/Aspose.Cells-for-C/).

- Puede clonar el repositorio usando su cliente favorito de GitHub o descargar el archivo ZIP desde [aquí](https://github.com/aspose-cells/Aspose.Cells-for-C/archive/master.zip).
- Extraiga el contenido del archivo ZIP en cualquier carpeta de su computadora. Todos los ejemplos están ubicados en la carpeta **Ejemplos**.
- Abra el archivo de solución en Visual Studio y compile el proyecto.
- La carpeta **Datos** en la carpeta raíz de **Ejemplos** contiene archivos de entrada. Es obligatorio descargar la carpeta **Datos** junto con el proyecto de ejemplos.
- Abrir main.cpp, todos los ejemplos se llaman desde aquí.
- Descomente los ejemplos que desee ejecutar desde el interior del proyecto.

No dude en comunicarse a través de nuestros foros si tiene algún problema para configurar o ejecutar los ejemplos.
## **Contribuir**
Si desea agregar o mejorar un ejemplo, lo alentamos a contribuir al proyecto. Todos los ejemplos y proyectos de exhibición en este repositorio son de código abierto y se pueden utilizar libremente en sus propias aplicaciones.

Para contribuir, puedes hacer un fork del repositorio, editar el código fuente y crear una solicitud de extracción. Revisaremos los cambios e incluiremos en el repositorio si son útiles.
