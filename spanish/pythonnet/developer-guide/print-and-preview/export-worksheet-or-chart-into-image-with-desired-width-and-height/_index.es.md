{{% alert color="primary" %}}

Puedes usar Aspose.Cells para Python via .NET para exportar tu hoja o gráfico en una imagen con el tamaño deseado. La API proporciona el método [**ImageOrPrintOptions.set_desired_size()**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/set_desired_size) para establecer el tamaño deseado de la imagen exportada. El ancho y la altura se especifican en píxeles.
