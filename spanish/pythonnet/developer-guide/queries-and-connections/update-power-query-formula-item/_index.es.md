## **Escenario de uso**

Puede ocurrir que los archivos de la fuente de datos se muevan y el archivo de Excel no pueda localizar el archivo. En tales casos, la API Aspose.Cells para Python via .NET ofrece la opción de actualizar el elemento de fórmula Power Query mediante el uso de la clase [*PowerQueryFormulaItem*](https://reference.aspose.com/cells/python-net/aspose.cells.querytables/powerqueryformulaitem) para actualizar la ubicación del archivo fuente.
