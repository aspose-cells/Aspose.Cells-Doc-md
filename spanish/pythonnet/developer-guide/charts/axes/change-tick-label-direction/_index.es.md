## **Cambiar la dirección de la etiqueta del eje**

Aspose.Cells para Python via .NET le permite cambiar la dirección de las etiquetas de marcas de posición del gráfico usando la propiedad [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type). La propiedad [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type) acepta el valor de la enumeración [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/). La enumeración [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/) proporciona los siguientes miembros

- HORIZONTAL
- VERTICAL
- ROTATE90
- ROTATE270
- APILADO
