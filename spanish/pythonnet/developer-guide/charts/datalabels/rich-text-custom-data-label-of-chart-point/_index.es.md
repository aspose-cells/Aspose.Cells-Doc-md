Puede usar Aspose.Cells para Python via .NET para crear etiquetas de datos personalizadas con texto enriquecido en los puntos del gráfico. Aspose.Cells para Python via .NET proporciona el método [**DataLabels.characters()**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextframe/characters) para devolver el objeto [**FontSetting**](https://reference.aspose.com/cells/python-net/aspose.cells/fontsetting) que se puede usar para establecer las propiedades de fuente del texto, como su color, negrita, etc.
