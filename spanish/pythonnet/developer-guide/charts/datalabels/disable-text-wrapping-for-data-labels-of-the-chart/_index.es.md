{{% alert color="primary" %}}

Microsoft Excel 2013 permite a los usuarios ajustar o desajustar el texto dentro de las etiquetas de datos del gráfico. De forma predeterminada, el texto dentro de las etiquetas de datos del gráfico está en estado desajustado.

Aspose.Cells para Python via .NET proporciona una propiedad [**DataLabels.is_text_wrapped**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/datalabels/is_text_wrapped) que puede establecer en Verdadero o Falso para habilitar o deshabilitar el ajuste de texto de las etiquetas de datos respectivamente.

{{% /alert %}}
