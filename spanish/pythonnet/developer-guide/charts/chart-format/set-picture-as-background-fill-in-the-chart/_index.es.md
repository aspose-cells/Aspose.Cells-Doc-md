{{% alert color="primary" %}}

Aspose.Cells para Python via .NET permite establecer un degradado, textura, patrón o imagen como efectos de relleno para diferentes objetos, como el área de la gráfica, el área del gráfico o la caja de leyenda de un gráfico. Este documento muestra cómo agregar una imagen al fondo de un gráfico.

{{% /alert %}}

Para lograr esto, Aspose.Cells para Python via .NET proporciona la propiedad [**Chart.plot_area.area.fill_format.image_data**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/fillformat/image_data). El siguiente ejemplo de código demuestra el uso de la propiedad [**Chart.plot_area.area.fill_format.image_data**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/fillformat/image_data) para establecer una imagen como relleno de fondo en el gráfico.
