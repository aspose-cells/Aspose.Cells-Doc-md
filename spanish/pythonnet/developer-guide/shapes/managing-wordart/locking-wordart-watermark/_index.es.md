{{% alert color="primary" %}}

Las API Aspose.Cells para Python via .NET permiten agregar marcas de agua WordArt en la hoja de cálculo de tal manera que el WordArt se convierte en un objeto que puedes mover y posicionar en la hoja. También es posible bloquear el objeto WordArt para cualquier interacción, como edición, movimiento y selección. Este artículo explica el uso del método [**Shape.set_locked_property**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/set_locked_property) para bloquear algunos aspectos de la marca de agua.

{{% /alert %}}
