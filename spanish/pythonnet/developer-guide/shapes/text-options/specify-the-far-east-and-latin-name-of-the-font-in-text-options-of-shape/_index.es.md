## **Escenarios de uso posibles**

A veces quieres mostrar texto en una fuente de idioma del Lejano Oriente, por ejemplo, japonés, chino, tailandés, etc. Aspose.Cells para Python via .NET ofrece la propiedad [**TextOptions.far_east_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/far_east_name) que se puede usar para especificar el nombre de la fuente del idioma del Lejano Oriente. Además, también puedes especificar el nombre de la fuente latina usando la propiedad [**TextOptions.latin_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/latin_name).
