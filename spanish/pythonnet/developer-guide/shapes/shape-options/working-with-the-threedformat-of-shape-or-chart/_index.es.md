## **Escenarios de uso posibles**
Aspose.Cells para Python via .NET proporciona la propiedad [Shape.three_d_format](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/three_d_format/) junto con la clase [ThreeDFormat](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/threedformat) para trabajar con el formato 3D de forma o gráfico. La clase [ThreeDFormat](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/threedformat) contiene diferentes propiedades que se pueden configurar para lograr distintos resultados según los requisitos de la aplicación.
