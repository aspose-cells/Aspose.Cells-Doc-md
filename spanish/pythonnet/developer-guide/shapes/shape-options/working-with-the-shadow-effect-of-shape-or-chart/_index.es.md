## **Escenarios de uso posibles**
Aspose.Cells para Python via .NET proporciona la propiedad [Shape.shadow_effect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/shadow_effect) junto con la clase [ShadowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect) para trabajar con el efecto sombra de forma o gráfico. La clase [ShadowEffect](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect) contiene las siguientes propiedades que se pueden configurar para obtener diferentes resultados según los requisitos de la aplicación.

- [ángulo](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/angle)
- [desenfoque](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/blur)
- [color](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/color)
- [distancia](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/distance)
- [tipo_preestablecido](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/preset_type)
- [tamaño](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/size)
- [transparencia](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shadoweffect/transparency)
