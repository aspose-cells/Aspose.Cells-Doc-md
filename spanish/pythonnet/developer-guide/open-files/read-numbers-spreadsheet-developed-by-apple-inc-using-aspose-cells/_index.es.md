## **Escenarios de uso posibles**

Numbers es una aplicación de hoja de cálculo desarrollada por Apple Inc. Aspose.Cells para Python via .NET puede leer hojas de cálculo de Numbers pero no soporta escribir en ellas. Puede leer Datos, Estilo y Fórmulas de hojas de cálculo de Numbers.

El siguiente código de muestra carga la [Hoja de Cálculo de Numbers de Muestra] (sampleNumbersByAppleInc.numbers) y la convierte a [Formato de PDF de Salida] (outputNumbersByAppleInc.pdf). Descargue ambos desde los enlaces proporcionados. Puede probar el código con cualquier hoja de cálculo de Numbers. También lea los comentarios del código para obtener más ayuda.
