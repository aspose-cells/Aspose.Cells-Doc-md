{{% alert color="primary" %}}

Microsoft Excel permite a los usuarios agregar comentarios a las celdas para añadir información adicional y resaltar datos. Los desarrolladores pueden necesitar personalizar el comentario para especificar configuraciones de alineación y dirección del texto. Aspose.Cells para Python via .NET proporciona APIs para realizar esta tarea.

{{% /alert %}}

Aspose.Cells para Python via .NET proporciona una propiedad [**Shape.text_direction**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_direction) para establecer la dirección del texto en un comentario.
