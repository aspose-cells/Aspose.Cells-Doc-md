{{% alert color="primary" %}}

Microsoft Excel permite a los usuarios agregar comentarios a celdas para agregar información adicional y resaltar datos. Los desarrolladores pueden necesitar personalizar el comentario para especificar configuraciones de alineación, dirección del texto, color de fuente, etc. Aspose.Cells para Python via .NET ofrece APIs para realizar la tarea.

{{% /alert %}}

Aspose.Cells para Python via .NET proporciona una propiedad [**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body) para el color de fuente del comentario. El siguiente código de ejemplo demuestra el uso de la propiedad [**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body) para establecer la dirección del texto en un comentario.
