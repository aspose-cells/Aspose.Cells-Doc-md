## **Cómo dibujar una línea de tiempo al renderizar Excel a PDF usando la Biblioteca de Excel Aspose.Cells para Python.**
Si tienes un archivo de Excel al que se le ha aplicado una línea de tiempo y quieres exportarlo a PDF con la configuración de la línea de tiempo, Aspose.Cells for Python via .NET ahora lo admite de forma predeterminada. Simplemente exporta el archivo de Excel con la línea de tiempo a PDF, el PDF generado mostrará la línea de tiempo aplicada.
