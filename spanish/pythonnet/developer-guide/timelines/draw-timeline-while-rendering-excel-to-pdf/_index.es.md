##  **Dibuje una línea de tiempo mientras renderiza Excel en PDF**
Si tiene un archivo de Excel al que se le ha aplicado una línea de tiempo y desea exportar el Excel a PDF con la configuración de la línea de tiempo, Aspose.Cells for Python via .NET ahora lo admite de forma predeterminada. Simplemente exporte el archivo de Excel con la línea de tiempo a PDF, el PDF generado mostrará la línea de tiempo aplicada.
