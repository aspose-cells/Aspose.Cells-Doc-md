Puedes hacer que una hoja sea más atractiva agregando una imagen como fondo de la hoja. Esta característica puede ser bastante efectiva si tienes un gráfico corporativo especial que agrega un toque de fondo sin ocultar los datos en la hoja. Puedes establecer la imagen de fondo en una hoja usando las APIs de Aspose.Cells para Python via .NET.

{{% /alert %}} 

## **Cómo establecer el fondo de la hoja en Microsoft Excel**

Para establecer una imagen de fondo de hoja en Microsoft Excel (por ejemplo, Microsoft Excel 2019):

1. Desde el menú **Diseño de página**, encontrar la opción **Configurar página**, y luego hacer clic en la opción **Fondo**.
1. Seleccionar una imagen para establecer la imagen de fondo de la hoja.
