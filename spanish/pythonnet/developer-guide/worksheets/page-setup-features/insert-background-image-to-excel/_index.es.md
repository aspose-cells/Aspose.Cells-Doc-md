Puedes hacer que una hoja de trabajo sea más atractiva añadiendo una imagen como fondo de hoja. Esta característica puede ser bastante efectiva si tienes un gráfico corporativo especial que añade un toque de fondo sin ocultar los datos en la hoja. Puedes establecer una imagen de fondo para una hoja utilizando la API de Aspose.Cells for Python via .NET.

{{% /alert %}} 

## **Cómo Establecer Fondo de Hoja en Microsoft Excel**

Para establecer una imagen de fondo de hoja en Microsoft Excel (por ejemplo, Microsoft Excel 2019):

1. Desde el menú **Diseño de página**, encontrar la opción **Configurar página**, y luego hacer clic en la opción **Fondo**.
1. Seleccionar una imagen para establecer la imagen de fondo de la hoja.
