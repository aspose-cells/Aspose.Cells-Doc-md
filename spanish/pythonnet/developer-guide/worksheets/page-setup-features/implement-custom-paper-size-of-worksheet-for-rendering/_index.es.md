## **Escenarios de uso posibles**

No hay una opción directa disponible para crear tamaños de papel personalizados en MS Excel, sin embargo, puedes establecer un tamaño de papel personalizado para tus hojas de trabajo deseadas al renderizar un archivo de Excel en formato de archivo PDF. Este documento explica cómo establecer un tamaño de papel personalizado de una hoja de trabajo utilizando las APIs de Aspose.Cells for Python via .NET.

## **Implementar Tamaño de Papel Personalizado de la Hoja de Cálculo para el Renderizado**

Aspose.Cells for Python via .NET te permite implementar el tamaño de papel deseado de la hoja de trabajo. Puedes usar el método [**custom_paper_size**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup/custom_paper_size/#float-float) de la clase [**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup) para especificar un tamaño de página personalizado. El siguiente código de muestra ilustra cómo especificar un tamaño de papel personalizado para la primera hoja de trabajo en el libro.
