## **Escenarios de uso posibles**

No hay opción directa para crear tamaños de papel personalizados en MS Excel, sin embargo, puedes establecer un tamaño de papel personalizado en tus hojas deseadas al renderizar un archivo de Excel a PDF. Este documento explica cómo configurar un tamaño de papel personalizado usando las APIs de Aspose.Cells para Python via .NET.

## **Implementar Tamaño de Papel Personalizado de la Hoja de Cálculo para el Renderizado**

Aspose.Cells para Python via .NET permite implementar el tamaño de papel deseado de la hoja de cálculo. Puedes usar el método [**custom_paper_size**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup/custom_paper_size/#float-float) de la clase [**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup) para especificar un tamaño de página personalizado. El siguiente código de ejemplo ilustra cómo especificar un tamaño de papel personalizado para la primera hoja del libro. También puedes ver el [PDF de salida](45056028.pdf) generado con este código como referencia.
