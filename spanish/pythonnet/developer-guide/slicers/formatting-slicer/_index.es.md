##  **Posibles escenarios de uso**

Puede formatear la segmentación de datos en Microsoft Excel configurando su número de columnas o configurando su estilo, etc. Aspose.Cells for Python via .NET también le permite hacer esto usando el[**Segmentación.número_de_columnas**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/number_of_columns/)y[**Cortadora.style_type**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/style_type/)propiedades.
