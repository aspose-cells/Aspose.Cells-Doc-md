## **Escenarios de uso posibles**

Puede dar formato al filtro en Microsoft Excel ajustando su número de columnas o su estilo, etc. Aspose.Cells para Python via .NET también le permite hacer esto usando las propiedades [**Slicer.number_of_columns**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/number_of_columns/) y [**Slicer.style_type**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/style_type/).
