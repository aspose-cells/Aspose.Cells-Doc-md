## **Cómo dibujar un rebanador mientras se renderiza Excel a PDF usando la biblioteca de Excel de Aspose.Cells para Python**
Si tienes un archivo de Excel al que se le ha aplicado un rebanador y deseas exportar el Excel a PDF con la configuración del rebanador, Aspose.Cells para Python via .NET ahora lo admite de forma predeterminada. Simplemente exporta el archivo de Excel con el rebanador a PDF, el PDF generado mostrará el rebanador aplicado.
