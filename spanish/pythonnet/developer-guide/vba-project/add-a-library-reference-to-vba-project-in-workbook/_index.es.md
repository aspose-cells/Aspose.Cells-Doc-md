A veces, necesitas agregar o registrar la referencia de la biblioteca en el proyecto VBA a través de código. Puedes hacerlo usando el método [**VbaProject.references.add_registered_reference()**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaprojectreferencecollection/add_registered_reference/#str-str) de Aspose.Cells para Python via .NET.

{{% /alert %}}

## **Agrega una referencia de la biblioteca al proyecto VBA en Microsoft Excel**

En Microsoft Excel, puedes agregar una referencia de la biblioteca al proyecto VBA haciendo clic en **Herramientas > Referencias...** manualmente.
