## Verificar si el proyecto VBA está protegido y bloqueado para visualización en Python

Aspose.Cells para Python via .NET te permite verificar si el proyecto VBA (Visual Basic for Applications) del archivo de Excel está protegido y bloqueado para visualización. Para esto, la API proporciona la propiedad [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/). Si está bloqueado para visualización, entonces la propiedad [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/) devuelve **true**.
