**Office Open XML** (también conocido como **OOXML** o **Microsoft Open XML** (MOX)) es un formato de archivo basado en XML desarrollado por Microsoft para representar documentos de oficina como hojas de cálculo, gráficos, presentaciones y documentos de procesamiento de palabras.

{{% /alert %}} 



Aspose.Cells para Python via .NET proporciona una forma de detectar el formato del archivo de archivos cifrados **Microsoft Open XML**. Para identificar el tipo de archivo, utiliza el método [FileFormatUtil.detect_file_format](https://reference.aspose.com/cells/python-net/aspose.cells/fileformatutil/detect_file_format/#str) como se muestra en el ejemplo de código a continuación.
