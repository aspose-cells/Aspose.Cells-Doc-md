## **Introducción**

Aspose.Cells para Python via .NET admite firmar libros de trabajo con firma XAdES. Para ello, la API proporciona las clases [**DigitalSignature**](https://reference.aspose.com/cells/python-net/aspose.cells.digitalsignatures/digitalsignature) y la enumeración [**XAdESType**](https://reference.aspose.com/cells/python-net/aspose.cells.digitalsignatures/xadestype).
