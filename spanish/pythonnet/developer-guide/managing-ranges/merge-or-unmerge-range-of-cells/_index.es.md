## **Escenarios de uso posibles**

Puede usar Aspose.Cells para Python via .NET para fusionar o dividir un rango de celdas. Aspose.Cells para Python via .NET proporciona los métodos [**Range.merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/merge/#) y [**Range.un_merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/un_merge/#) con ese propósito. Este artículo explica cómo fusionar un rango de celdas en una sola celda.


## **Cómo fusionar o separar rango de celdas**

El código de ejemplo primero crea un rango - A1:D4 - luego combina las celdas en el rango en una sola celda usando el método [**Range.merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/merge/#). Del mismo modo, puede dividir celdas creando un rango y llamando al método [**Range.un_merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/un_merge/#).
