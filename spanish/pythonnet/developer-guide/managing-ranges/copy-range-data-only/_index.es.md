{{% alert color="primary" %}}

A veces, es necesario copiar datos de un rango de celdas a otro, copiando solo los datos y no el formato. Aspose.Cells for Python via .NET ofrece esta función.

Este artículo proporciona un código de ejemplo que utiliza Aspose.Cells for Python via .NET para copiar un rango de datos.

{{% /alert %}}

Este ejemplo muestra cómo:

1. Crear un libro de trabajo.
1. Agregar datos a las celdas en la primera hoja de cálculo.
1. Crear un [**Range**](https://reference.aspose.com/cells/python-net/aspose.cells/range).
1. Crea un objeto [**Style**](https://reference.aspose.com/cells/python-net/aspose.cells/style) con atributos de formato especificados.
1. Aplica el formato de estilo al rango.
1. Crear otro rango de celdas.
1. Copiar datos del primer rango a este segundo rango.
