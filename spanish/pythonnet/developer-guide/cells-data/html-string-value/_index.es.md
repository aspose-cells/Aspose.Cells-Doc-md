{{% alert color="primary" %}}

Aspose.Cells para Python via .NET admite la conversión de HTML orientado a Microsoft Excel a formato XLS/XLSX. Esto significa que el HTML generado por Microsoft Excel se puede convertir nuevamente a formato XLS/XLSX utilizando Aspose.Cells para Python via .NET.

Del mismo modo, si hay un HTML simple, Aspose.Cells para Python via .NET puede convertirlo en texto HTML enriquecido. Aspose.Cells para Python via .NET proporciona la propiedad [**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/), que puede tomar este HTML simple y convertirlo a texto de celda formateado.
