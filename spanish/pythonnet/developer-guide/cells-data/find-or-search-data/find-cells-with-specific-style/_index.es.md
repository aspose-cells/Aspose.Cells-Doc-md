{{% alert color="primary" %}}

A veces, es necesario encontrar celdas con un estilo particular aplicado. Puede utilizar Aspose.Cells para Python via .NET para encontrar todas las celdas con un estilo común. Aspose.Cells para Python via .NET proporciona la propiedad [**FindOptions.style**](https://reference.aspose.com/cells/python-net/aspose.cells/findoptions/style/) que puede utilizar para especificar el estilo para buscar celdas.

{{% /alert %}}
