## **Obtener validación de celda en archivos ODS**

Con Aspose.Cells, puede obtener la validación aplicada a una celda en archivos ODS. Para esto, la API proporciona el método [**get_validation**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) de la clase [**Cell**](https://reference.aspose.com/cells/python-net/aspose.cells/cell).

El siguiente ejemplo de código demuestra el uso del método [**get_validation**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) cargando el [archivo ODS de origen](101089354.ods) y leyendo la validación de la celda A9.
