{{% alert color="primary" %}}

Puedes usar Aspose.Cells para Python via .NET para obtener la validación aplicada a una celda. Aspose.Cells para Python via .NET proporciona el método [**Cell.get_validation()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_validation/) para este propósito. Si no hay validación aplicada a la celda, devuelve nulo.

De manera similar, puede usar el método [**Worksheet.validations.get_validation_in_cell**](https://reference.aspose.com/cells/python-net/aspose.cells/validationcollection/get_validation_in_cell/#int-int) para adquirir la validación aplicada a una celda proporcionando sus índices de fila y columna.
