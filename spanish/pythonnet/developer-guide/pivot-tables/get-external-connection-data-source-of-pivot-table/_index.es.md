## **Cómo obtener la fuente de datos de conexión externa de la tabla dinámica.**

Aspose.Cells para Python via .NET proporciona la capacidad para obtener la fuente de datos de conexión externa de la tabla dinámica. Para esto, la API proporciona la propiedad [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) de la clase [**PivotTable**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/). La propiedad [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) devuelve el objeto [**ExternalConnection**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/). El siguiente fragmento de código muestra el uso de la propiedad [**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) para obtener la fuente de datos de conexión externa de la tabla dinámica.
