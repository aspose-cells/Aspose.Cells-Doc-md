##  **Obtener fuente de datos de conexión externa de la tabla dinámica**

Aspose.Cells for Python via .NET proporciona la capacidad de obtener la fuente de datos de conexión externa de la tabla dinámica. Para ello, el API proporciona la[**fuente_de_datos_de_conexión_externa**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) propiedad de la[**Tabla dinámica**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/) clase. El[**fuente_de_datos_de_conexión_externa**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/)propiedad devuelve el[**Conexión externa**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/) objeto. El siguiente fragmento de código demuestra el uso de[**Tabla dinámica.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) propiedad para obtener la fuente de datos de conexión externa de la tabla dinámica.
