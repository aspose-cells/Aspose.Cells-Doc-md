## **Cómo mostrar la opción de páginas del filtro de informe**

Aspose.Cells para Python via .NET también admite esta función para habilitar la opción de "Mostrar páginas de filtro de informe" en la tabla dinámica creada. A continuación se muestra la pantalla que muestra la opción "Mostrar páginas de filtro de informe" en Excel.
