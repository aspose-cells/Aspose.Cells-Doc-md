## **Cómo guardar una tabla dinámica en un archivo ODS**

Aspose.Cells for Python via .NET proporciona la capacidad de guardar una tabla dinámica en archivos ODS. Para esto, simplemente convierta un libro de trabajo con una tabla dinámica existente o cree una nueva tabla dinámica y guarde el archivo en formato ODS. Asegúrese de llamar a [**PivotTable.calculate_data**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/calculate_data/#) antes de guardar para garantizar que la tabla dinámica se renderice en el archivo ODS de salida.
