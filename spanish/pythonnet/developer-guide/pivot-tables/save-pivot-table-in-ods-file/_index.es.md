 Aspose.Cells for Python via .NET proporciona la posibilidad de guardar la tabla dinámica en archivos ODS. Para esto, simplemente convierta un libro con una tabla dinámica existente o cree una nueva tabla dinámica y guarde el archivo en el formato ODS. Asegúrate de llamar[**Tabla dinámica.calcular_datos**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/calculate_data/#)antes de guardar para asegurarse de que la tabla dinámica se represente en el archivo de salida ODS.
