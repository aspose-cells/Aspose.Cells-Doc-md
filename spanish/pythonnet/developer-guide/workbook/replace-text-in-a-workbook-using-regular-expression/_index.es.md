Aspose.Cells para Python via .NET ofrece la función de reemplazar texto en un libro usando una expresión regular. Para esto, la API proporciona la propiedad [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) de la clase [**ReplaceOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions). Configurar [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) a **true** indica que la clave buscada será una expresión regular.
