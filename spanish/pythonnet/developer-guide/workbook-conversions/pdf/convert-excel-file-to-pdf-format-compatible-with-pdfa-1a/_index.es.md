##  **Posibles escenarios de uso**

PDF/A es una versión única de PDF diseñada para la conservación de documentos a largo plazo. PDF/A es una versión estandarizada ISO del formato de documento portátil (PDF), que es un formato de archivo de PDF que incorpora todas las fuentes utilizadas en el documento dentro del archivo PDF. PDF/A se diferencia de PDF en que prohíbe funciones como la vinculación de fuentes (a diferencia de la incrustación de fuentes) y el cifrado. Aspose.Cells for Python via .NET le permite guardar los archivos de Excel en archivos PDF/A compatibles con PDF (se admiten PdfA1a y PdfA1b). Este tema describe cómo guardar el libro de Excel en un archivo PDF compatible con PDF/A (PdfA1a).

##  **Convierta un archivo de Excel al formato PDF Compatible con PDFA-1a**

Los desarrolladores pueden utilizar el**[PdfSaveOptions](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/)**clase para establecer diferentes atributos para la conversión. Establecer diferentes propiedades del**[PdfSaveOptions](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/)**La clase le brinda control sobre las configuraciones de impresión, fuente, seguridad y compresión para la salida PDF. La propiedad más importante es**[PdfSaveOptions.compliance](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/compliance/)**que le permite guardar los archivos de Excel en archivos PDF compatibles con PDF/A.
