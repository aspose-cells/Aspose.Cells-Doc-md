Las facilidades de estructura lógica de PDF proporcionan un mecanismo para incorporar información, relacionada con la estructura de contenido del documento, en un archivo PDF. Aspose.Cells para Python via .NET preserva la información sobre la estructura de un documento de Microsoft Excel, como celda, fila, tabla, hoja de cálculo, imagen, forma, encabezado/pie de página, etc.

Con la opción [PdfSaveOptions.export_document_structure](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/export_document_structure/), puede guardar en un PDF etiquetado con la estructura del documento exportada.
