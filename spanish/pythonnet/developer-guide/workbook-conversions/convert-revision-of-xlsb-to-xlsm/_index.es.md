{{% alert color="primary" %}} 

Aspose.Cells for Python via .NET ahora es compatible para convertir completamente las revisiones de un archivo XLSB en un archivo XLSM. Las revisiones se encuentran dentro de la ruta \xl\revisions. Puedes verlas cambiando la extensión de tu archivo XLSB a ZIP. La ruta \xl\revisions contiene archivos con extensión .bin.

Cuando conviertes tu archivo XLSB en un archivo XLSM mediante Aspose.Cells for Python via .NET, estos archivos .bin se convierten con éxito en archivos .xml, como se muestra en estas dos capturas de pantalla.
