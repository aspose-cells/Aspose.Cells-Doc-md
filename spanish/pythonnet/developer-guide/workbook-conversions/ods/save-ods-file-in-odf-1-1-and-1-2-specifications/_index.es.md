{{% alert color="primary" %}}

 Aspose.Cells for Python via .NET admite guardar un archivo ODS (**Hoja de cálculo OpenDocument**) en las especificaciones ODF (**Formato OpenDocument**) 1.1 y 1.2. Aspose.Cells for Python via .NET tiene[**OdsSaveOptions.is_strict_schema11**](https://reference.aspose.com/cells/python-net/aspose.cells/odssaveoptions/is_strict_schema11/)Propiedad que especifica el uso de la especificación ODF 1.1 para guardar archivos ODS. El valor predeterminado de esta propiedad es *falso**, por lo que el archivo ODS guardado sin esta configuración utiliza las especificaciones 1.2.

{{% /alert %}}

El siguiente código de muestra crea un objeto de libro de trabajo, agrega algún valor a la celda A1 en la primera hoja de trabajo y luego guarda el archivo ODS en las especificaciones ODF 1.1 y 1.2. De forma predeterminada, el archivo ODS se guarda en la especificación ODF 1.2.
