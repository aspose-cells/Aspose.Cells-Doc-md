## **Convertir JSON a hoja de cálculo de Excel**
No es necesario preguntarse cómo convertir un archivo JSON a Excel, porque la biblioteca Aspose.Cells for Python via .NET tiene la mejor solución. La API de Aspose.Cells for Python via .NET brinda soporte para la conversión del formato JSON a hojas de cálculo. Puede utilizar la clase [**JsonLoadOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonLoadOptions) para especificar ajustes adicionales al importar JSON a un libro de trabajo.
