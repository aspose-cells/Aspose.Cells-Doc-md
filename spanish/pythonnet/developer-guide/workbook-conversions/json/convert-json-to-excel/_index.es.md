##  **Convertir JSON a libro de Excel**
No es necesario preguntarse cómo convertir JSON a un archivo Excel, porque la biblioteca Aspose.Cells for Python via .NET tiene la mejor decisión. Aspose.Cells for Python via .NET API proporciona soporte para convertir el formato JSON a hojas de cálculo. Puedes usar**[JsonLoadOptions](https://reference.aspose.com/cells/python-net/aspose.cells/JsonLoadOptions)** clase para especificar configuraciones adicionales para importar JSON al libro de trabajo.
