{{% /alert %}}

##  **Convertir libro de Excel a JSON**

No es necesario preguntarse cómo convertir un libro de Excel a JSON, porque la biblioteca Aspose.Cells for Python via .NET tiene la mejor decisión. Aspose.Cells API brinda soporte para convertir hojas de cálculo al formato JSON. Para exportar el libro a JSON, pase[**Guardar formato.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) como segundo parámetro de[**Libro de trabajo.guardar**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) método. También puedes usar[**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) clase para especificar configuraciones adicionales para exportar la hoja de trabajo a JSON.
