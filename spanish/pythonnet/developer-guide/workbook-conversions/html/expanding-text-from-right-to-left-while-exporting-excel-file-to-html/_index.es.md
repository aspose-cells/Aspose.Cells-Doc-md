{{% alert color="primary" %}} 

Aspose.Cells para Python via .NET ahora soporta expandir texto de derecha a izquierda al exportar archivos Excel en HTML. Esta función ha sido implementada desde la versión v8.9.0.0. Ahora, si tu archivo Excel fuente contiene texto que se expande de derecha a izquierda, Aspose.Cells lo exportará correctamente a HTML.

{{% /alert %}}
