##  **Convierta Excel a HTML con encabezados**

Aspose.Cells for Python a través de NET proporciona la opción de exportar encabezados mientras convierte Excel a HTML. Para esto, puede configurar el[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) propiedad a *verdadero**. El valor predeterminado de[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)la propiedad es *falsa**.
