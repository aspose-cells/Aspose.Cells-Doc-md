Scalable Vector Graphics (SVG) es un formato de imagen vectorial basado en XML para gráficos bidimensionales que también admite interactividad y animación. La especificación SVG es un estándar abierto desarrollado por el World Wide Web Consortium (W3C) desde 1999.

SVG las imágenes y sus comportamientos se definen en archivos de texto XML. Esto significa que se pueden buscar, indexar, programar y comprimir. Como archivos XML, las imágenes SVG se pueden crear y editar con cualquier editor de texto, pero más a menudo se crean con software de dibujo.

Aspose.Cells for Python via .NET puede guardar gráficos en imágenes en varios formatos como BMP, JPEG, PNG, GIF, SVG, etc. Este artículo explica cómo guardar un gráfico en formato SVG.
