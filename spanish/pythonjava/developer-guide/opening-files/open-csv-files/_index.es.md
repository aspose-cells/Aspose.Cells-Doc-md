## **Abrir archivos CSV**
Los archivos de valores separados por comas (CSV) contienen registros cuyos valores están delimitados o separados por comas. Puedes utilizar Microsoft Excel para guardar tu hoja de cálculo en formato CSV.

Para abrir archivos CSV, utiliza la clase [LoadOptions](https://reference.aspose.com/cells/python/asposecells.api/LoadOptions), y selecciona el valor [CSV](https://reference.aspose.com/cells/python/asposecells.api/fileformattype#CSV), predefinido en la enumeración [FileFormatType](https://reference.aspose.com/cells/python/asposecells.api/FileFormatType).
