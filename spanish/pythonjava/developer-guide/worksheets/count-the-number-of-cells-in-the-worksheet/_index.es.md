## **Contar el número de celdas en la hoja de cálculo**

Para esto, la API proporciona las propiedades [Cells.getCount](https://reference.aspose.com/cells/python/asposecells.api/cells#Count)() o [Cells.getCountLarge](https://reference.aspose.com/cells/python/asposecells.api/cells#CountLarge)().
