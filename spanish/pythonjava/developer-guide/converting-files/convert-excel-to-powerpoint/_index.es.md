Aspose.Cells for Python via Java admite la conversión de archivos de Excel (.xls, xlsx, .xlsb,xlsm), CSV y OpenOffice (.ods) a archivos PowerPoint.

{{% /alert %}}

## **Convertir libro de trabajo de Excel a PPT**

 No es necesario preguntarse cómo convertir Excel Workbook a PowerPoint, porque la biblioteca Aspose.Cells for Python via Java tiene la mejor decisión. El Aspose.Cells for Python via Java API brinda soporte para convertir hojas de cálculo al formato PowerPoint. Para exportar el libro de trabajo a PowerPoint, pase[**GuardarFormato.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) como segundo parámetro de[**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) método. También puede usar[**PptxGuardarOpciones**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) class para especificar configuraciones adicionales para exportar la hoja de trabajo a un archivo .pptx.
