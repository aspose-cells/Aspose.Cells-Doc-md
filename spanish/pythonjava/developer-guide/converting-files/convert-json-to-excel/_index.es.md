## **Convertir JSON a hoja de cálculo de Excel**
No hace falta preguntarse cómo convertir JSON a un archivo de Excel, porque la biblioteca Aspose.Cells para Python via Java tiene la mejor solución. La API Aspose.Cells para Python via Java proporciona soporte para convertir formato JSON a hojas de cálculo. Puede usar la clase [**JsonLoadOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonLoadOptions) para especificar configuraciones adicionales para importar JSON a un libro de trabajo.
