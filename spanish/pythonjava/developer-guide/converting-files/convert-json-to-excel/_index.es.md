## **Convertir JSON a libro de Excel**
 No es necesario preguntarse cómo convertir JSON a un archivo de Excel, porque la biblioteca Aspose.Cells for Python via Java tiene la mejor decisión. El Aspose.Cells for Python via Java API brinda soporte para convertir el formato JSON a hojas de cálculo. Puedes usar[**JsonLoadOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonLoadOptions) class para especificar configuraciones adicionales para importar JSON a Workbook.
