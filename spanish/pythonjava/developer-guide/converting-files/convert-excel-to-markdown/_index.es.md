## **Convertir Excel a Markdown**
Aspose.Cells para Python a través de Java admite la conversión de archivos de Excel al formato Markdown. Para exportar la hoja de trabajo activa a Markdown, pase[SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN)como segundo parámetro de la[Libro de trabajo.Guardar](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)) método. También puede usar el[MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions)class para especificar configuraciones adicionales para exportar hojas de trabajo a Markdown.
