## **Convertir texto en columnas**
Puede convertir su Texto a Columnas usando Microsoft Excel. Esta función está disponible desde*Herramientas de datos*bajo la*Datos*pestaña. Para dividir el contenido de una columna en varias columnas, los datos deben contener un delimitador específico como una coma (o cualquier otro carácter) basado en el cual Microsoft Excel divide el contenido de una celda en varias celdas. Aspose.Cells también ofrece esta función a través del[TextoAColumnas](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) método. El siguiente fragmento de código demuestra el uso de la[TextoAColumnas](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) mediante la conversión de texto en columnas con espacio como delimitador.
