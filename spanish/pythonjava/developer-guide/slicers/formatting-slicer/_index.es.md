## ** Formatear rebanador**
Usando Microsoft Excel, puedes formatear el cortador ajustando el número de columnas, estilos, etc. Aspose.Cells for Python via Java proporciona las propiedades Slicer.NumberOfColumns y Slicer.StyleType para lograr esto.

El siguiente fragmento de código carga el [archivo de Excel de ejemplo](106364964.xlsx) que contiene un cortador. Accede al cortador y configura su número de columnas y tipo de estilo, luego lo guarda como [archivo de Excel de salida](106364965.xlsx). La captura de pantalla muestra cómo se ve el cortador después de la ejecución del código de ejemplo.
