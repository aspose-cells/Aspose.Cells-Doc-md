## **Renderización de Cortador**
Aspose.Cells for Python via Java soporta la renderización de la forma del cortador. El siguiente fragmento de código carga el [archivo de Excel de ejemplo](106364974.xlsx) que contiene un cortador. Convierte la hoja de cálculo en una imagen configurando el área de impresión que cubre solo el cortador. La imagen resultante es la [imagen de salida](106364975.png) que muestra el cortador renderizado. Como puedes ver, el cortador ha sido renderizado correctamente y se ve igual que en el archivo de Excel de ejemplo.
