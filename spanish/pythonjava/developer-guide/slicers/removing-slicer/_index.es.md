## **Eliminación de la cortadora**
 Para eliminar la segmentación en Microsoft Excel, simplemente seleccione la segmentación y presione el botón*Borrar* botón. Para lograr el guardado usando Aspose.Cells for Python via Java, use el método Worksheet.getSlicers().remove(). Eliminará la cortadora de la hoja de trabajo.

El siguiente fragmento de código carga el[ejemplo de archivo de Excel](106364970.xlsx)que contiene una segmentación existente. Accede a la segmentación, la elimina y guarda el[archivo de salida de Excel](106364971.xlsx). La siguiente captura de pantalla muestra la segmentación que se eliminará.
