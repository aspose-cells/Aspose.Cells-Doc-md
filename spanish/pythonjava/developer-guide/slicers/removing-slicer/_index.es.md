## **Eliminar rebanador**
Para eliminar un cortador en Microsoft Excel, simplemente selecciona el cortador y presiona el botón *Eliminar*. Para lograrlo usando Aspose.Cells for Python via Java, utiliza el método Worksheet.getSlicers().remove(). Esto eliminará el cortador de la hoja de cálculo. 

El siguiente fragmento de código carga el [archivo de Excel de ejemplo](106364970.xlsx) que contiene un cortador existente. Accede al cortador, lo elimina y guarda el [archivo de Excel de salida](106364971.xlsx). La siguiente captura de pantalla muestra el cortador que será eliminado.
