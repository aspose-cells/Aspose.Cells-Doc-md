## **Dibuje Slicer mientras renderiza Excel a PDF**
Aspose.Cells para Python a través de Java admite la exportación de Excel a PDF con la configuración de segmentación. El API admite esto de forma predeterminada, simplemente exporte el archivo de Excel con la segmentación a PDF, el PDF generado mostrará la segmentación aplicada. El siguiente fragmento de código carga el[ejemplo de archivo de Excel](106365058.xlsx)que contiene una segmentación y guarda el libro de trabajo como el[archivo PDF de salida](106365052.xlsx).
