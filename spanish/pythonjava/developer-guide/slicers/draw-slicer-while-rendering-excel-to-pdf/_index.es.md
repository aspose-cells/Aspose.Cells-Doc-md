## **Dibujar un rebanador al renderizar Excel a PDF**
Aspose.Cells para Python via Java admite la exportación de Excel a PDF con la configuración de filtro de segmentación. La API admite esto de forma predeterminada, simplemente exporta el archivo de Excel con filtro de segmentación a PDF, el PDF generado mostrará el filtro de segmentación aplicado. El siguiente fragmento de código carga el [archivo de Excel de ejemplo](106365058.xlsx) que contiene un filtro de segmentación y guarda el libro de trabajo como el archivo PDF de [salida](106365052.xlsx).
