## **Establezca el comentario de la tabla o el objeto de lista dentro de la hoja de trabajo**
Aspose.Cells for Python via Java admite agregar el comentario de List Object. Para esto, el API proporciona el[ListObject.Comentario](https://reference.aspose.com/cells/python/asposecells.api/listobject#Comment)propiedad. El comentario añadido por el[ListObject.Comentario](https://reference.aspose.com/cells/python/asposecells.api/listobject#Comment)la propiedad será visible dentro de la*xl/tables/tableName.xml* expediente.
