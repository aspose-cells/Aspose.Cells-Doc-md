## **Objeto de lista de formato**
Una tabla es una serie de filas y columnas que contienen datos relacionados administrados independientemente de los datos de otras filas y columnas. De manera predeterminada, cada columna de la tabla tiene habilitado el filtrado en la fila del encabezado para que pueda filtrar u ordenar rápidamente los datos de su objeto de lista. Puede agregar una fila total (una fila especial en una lista que proporciona una selección de funciones agregadas útiles para trabajar con datos numéricos) al objeto de lista que proporciona una lista desplegable de funciones agregadas para cada celda de fila total.

Aspose.Cells admite el formato de objetos de lista. Para esto, te API proporciona el[ListObject](https://reference.aspose.com/cells/python/asposecells.api/ListObject) y[TableStyleType](https://reference.aspose.com/cells/python/asposecells.api/TableStyleType) clases los[TableStyleType](https://reference.aspose.com/cells/python/asposecells.api/TableStyleType)La clase contiene constantes que representan los estilos de tabla integrados.
