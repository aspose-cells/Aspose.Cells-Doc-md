## **Exportar rango de área de impresión a HTML**
Aspose.Cells for Python via Java admite exportar solo el área de impresión, es decir, el rango de celdas seleccionado en lugar de todo el área a HTML. Para esto, el API proporciona el[HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) propiedad. Estableciendo esta propiedad en**Verdadero** solo exportará el área de impresión.
