## **Exportar rango del área de impresión a HTML**
Aspose.Cells para Python via Java admite exportar solo el área de impresión, es decir, el rango de celdas seleccionado en lugar de toda la hoja a HTML. Para esto, la API proporciona la propiedad [HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly). Establecer esta propiedad en **True** solo exportará el área de impresión.
