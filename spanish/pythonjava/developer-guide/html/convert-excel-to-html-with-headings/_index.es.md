## **Convertir Excel a HTML con encabezados**
Aspose.Cells proporciona la opción de exportar los encabezados de fila y columna al convertir Excel a HTML. Esto se puede lograr utilizando la propiedad [HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) proporcionada por la API. El valor predeterminado de [HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) es **Falso**. Pase **Verdadero** como parámetro para mostrar los encabezados en el archivo HTML de salida. La siguiente imagen muestra el archivo de salida generado por el siguiente código.
