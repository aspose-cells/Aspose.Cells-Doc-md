## **Convertir Excel a HTML con encabezados**
Aspose.Cells brinda la opción de exportar encabezados de filas y columnas al convertir Excel a HTML. Esto se puede lograr usando[HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings)propiedad proporcionada por el API. El valor predeterminado de[HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) es**Falso** . Pasar**Verdadero** como parámetro para representar encabezados en el archivo HTML de salida. La siguiente imagen muestra el archivo de salida generado por el siguiente código.
