## **Cells/Cell**

Cada objeto Worksheet contiene un objeto Cells que representa la colección de todos los objetos Cell en una hoja de cálculo, como se muestra a continuación:

|**Celdas y objetos de celda**|
| :- |
|![todo:image_alt_text](aspose-cells-object-model_3.png)|
Puede usar el objeto Cell para obtener y establecer el valor, el estilo, la fórmula y otras propiedades de una sola celda.

## **CommentCollection/Comment**

Cada objeto de Hoja de cálculo también contiene un objeto de Comentarios que representa la colección de todos los objetos Comentario en una hoja de cálculo como se muestra a continuación:

|**Comentarios y objetos Comentario**|
| :- |
|![todo:image_alt_text](aspose-cells-object-model_5.png)|

## **HorizontalPageBreakCollection/HorizontalPageBreak**

Cada objeto de Hoja de cálculo contiene una colección de Saltos de página horizontales que representa una colección de todos los objetos Salto de página horizontal en una hoja de cálculo como se muestra a continuación:

|**Saltos de página horizontal y objetos Salto de página horizontal**|
| :- |
|![todo:image_alt_text](aspose-cells-object-model_6.png)|

## **HyperlinkCollection/Hyperlink**

Un objeto de Hoja de cálculo también contiene una colección de Hipervínculos que representa una colección de todos los objetos Hipervínculo en la hoja de cálculo como se muestra a continuación:

|**Hipervínculos y objetos Hipervínculo**|
| :- |
|![todo:image_alt_text](aspose-cells-object-model_7.png)|
Un objeto Hipervínculo representa un hipervínculo en la hoja de cálculo. Los desarrolladores pueden establecer la dirección del hipervínculo y otras propiedades relacionadas utilizando el objeto Hipervínculo.

## **PictureCollection/Picture**

Cada objeto de Hoja de cálculo contiene un objeto de Colección de Imágenes que representa una colección de todas las imágenes en una hoja de cálculo como se muestra a continuación:

|**Imágenes y objetos Imagen**|
| :- |
|![todo:image_alt_text](aspose-cells-object-model_8.png)|
Un objeto Imagen representa una imagen en la hoja de cálculo. Mediante el objeto Imagen, los desarrolladores no solo pueden agregar imágenes a sus hojas de cálculo, sino también colocar estas imágenes en cualquier ubicación. También es posible configurar bordes u otras propiedades de las imágenes.
