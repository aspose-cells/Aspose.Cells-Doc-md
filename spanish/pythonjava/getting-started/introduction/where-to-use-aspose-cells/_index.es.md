{{% alert color="primary" %}} 

Aspose.Cells se crea para ayudar a los desarrolladores a crear y administrar Microsoft hojas de cálculo de Excel mediante programación. Aspose.Cells no solo comprende hojas de cálculo en formato de hoja de Excel Microsoft (XLS/XLSX), sino también muchos otros formatos de archivo como valores separados por comas (CSV), delimitado por tabuladores/TSV (texto), SpreadsheetML, etc.

Aspose.Cells proporciona la mayoría de las funciones disponibles en Microsoft Excel, pero lo interesante es que Aspose.Cells puede procesar archivos de Excel de forma dinámica, lo que permite a los desarrolladores procesar miles o millones de hojas de cálculo muy rápidamente.

Este artículo analiza escenarios comunes de uso de Aspose.Cells.

{{% /alert %}} 
## **Posibles escenarios de uso**
Depende de los desarrolladores cómo usan Aspose.Cells, pero hay algunos escenarios comunes en los que Aspose.Cells puede ayudar. El uso más común de Aspose.Cells es implementarlo en el servidor web o en los servicios web para el procesamiento del lado del servidor. Algunos de los posibles escenarios de uso de Aspose.Cells se describen a continuación.
## **Gestión automatizada de hojas de cálculo**
1. Los usuarios cargan hojas de cálculo creadas manualmente en un servidor.
1. Aspose.Cells se utiliza para abrir archivos y recuperar datos.
 Después de recuperar los datos:
 1. Los datos se guardan en una base de datos.
 1. Ciertos datos se presentan en las páginas web.
 1. Algunos aspectos de las hojas de cálculo se modifican y se guardan en archivos de Excel Microsoft.
## **Aplicación web**
1. Los usuarios ingresan datos en las páginas web.
1. Los datos son recopilados por el servidor web.
1. Aspose.Cells se utiliza para completar datos en archivos nuevos o archivos de plantilla predefinidos.
1. Una vez que se crea el archivo de Excel, se puede enviar al usuario como un informe, etc.

{{% alert color="primary" %}} 

Hemos discutido algunos de los escenarios más comunes del uso de Aspose.Cells en un entorno basado en web. Pero Aspose.Cells es igualmente útil en aplicaciones de escritorio.
