## **Requisitos del sistema**
Aspose.Cells for Python via Java es una API independiente de la plataforma y puede usarse en cualquier plataforma (Windows, Linux y MacOS) donde esté instalado [Python](https://www.python.org/downloads/). La máquina debe tener Java 8 o versiones superiores antes de configurar la instalación.

## **Versión de Python**
- Python 3.5 o superior
## **Versión de Java**
- Java 1.8 o superior

## **Instalación**
### **Windows:**
- **Instalar Java y establecer las variables de entorno JAVA_HOME y PATH**
Por ejemplo:
{{< highlight java >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}

- **Instalar Aspose.Cells para Python via Java desde pypi**
Puedes usar fácilmente Aspose.Cells para Python via Java desde [pypi](https://pypi.org/project/aspose-cells/) con el siguiente comando.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}
