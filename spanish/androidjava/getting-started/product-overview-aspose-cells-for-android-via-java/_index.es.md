¡Bienvenido a la documentación de Aspose.Cells para Android via Java!

Aspose.Cells es un componente flexible que permite a las aplicaciones Android crear y administrar hojas de cálculo de Excel sin requerir que Microsoft Excel esté instalado en el sistema. A diferencia de productos similares de otros proveedores, Aspose.Cells no solo admite la generación de hojas de cálculo y otras características básicas de formato de archivos, sino también una serie de características avanzadas. Estas características avanzadas facilitan a los desarrolladores manipular el contenido de las hojas de cálculo, el formato de celdas y la protección de archivos. Aspose.Cells también le permite importar datos a las hojas de cálculo desde diferentes fuentes de datos, agregar fórmulas y funciones matemáticas, de fecha y hora, financieras o de texto comunes y complejas, y admite la manipulación de gráficos, imágenes, comentarios, objetos de dibujo y controles.

{{% /alert %}} 
## **Características del documento**
- Abrir archivos de Excel (Excel97 - Excel2007 y versiones posteriores) simples o cifrados desde diferentes fuentes
- Guardar archivos de Excel (Excel97 - Excel2007 y versiones posteriores) en diferentes formatos (XLS, XLSX/XLSM, XLTX, XLTM, XLSB, CSV, SpreadsheetML, Delimitado por tabulaciones, HTML, etc.)
- Convertir archivos de Excel a documento PDF
- Convertir hojas de cálculo de Excel a documento XPS
- Leer y escribir el formato de hoja de cálculo de OpenDocument (ODS)
- Modificar las propiedades del documento de archivos de Excel
