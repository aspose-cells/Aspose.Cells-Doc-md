## **Limitaciones de la API de Aspose.Cells para Android via Java en comparación con la API Aspose.Cells for Java**
1. La representación de WorkbookRender no es compatible.
1. No es compatible el guardado en formatos TIFF y GIF.
1. No es compatible la impresión.
