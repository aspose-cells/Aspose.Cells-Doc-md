**Bienvenido a la sección de documentación de Aspose.Cells for Go via C++**

Aspose.Cells for Go via C++ es una biblioteca nativa de Go que permite a los desarrolladores de Go crear, manipular y convertir hojas de cálculo de forma programática sin requerir Office Automation o la aplicación Microsoft Excel.

El soporte de Aspose.Cells for Go via C++ incluye formatos de hojas de cálculo como el formato de libro de Excel 97-2003 y formatos Open Office XML.

{{% /alert %}}

## **Beneficios**

Usar Aspose.Cells for Go via C++ en sus aplicaciones le ofrece los siguientes beneficios:

- Características completas
- Independencia de plataforma
- Rendimiento y escalabilidad

### **Funciones Ricas**

Aspose.Cells for Go via C++ ofrece una amplia variedad de funciones para crear, convertir y manipular hojas de cálculo:

- Propiedades Incorporadas
- Propiedades Personalizadas
- Temas
- Estilos y Formato
- Validación de Datos
- Formato condicional
- Hipervínculo
- Autofiltro
- Configuración de Página
- Lectura, Escritura y Cálculo de Fórmulas
- Agrupar Filas y Columnas
- Tabla Dinámica
- Tabla

### **Independencia de Plataforma**

Use Aspose.Cells for Go via C++ para construir cualquier tipo de aplicación Go de 64 bits.

### **Rendimiento y Escalabilidad**

Aspose.Cells for Go via C++ está diseñado para funcionar igual de bien en servidor o lado cliente. Aspose.Cells for Go via C++ es un ensamblado nativo que se puede desplegar simplemente copiándolo. No necesita preocuparse por otros servicios o módulos.

## **Soporte Técnico**

Aspose ofrece soporte técnico ilimitado y gratuito para todos sus productos. El soporte está disponible para todos los usuarios, incluidos los de evaluación.

Si necesita ayuda con Aspose.Cells for Go via C++ [visite los Foros de Aspose](https://forum.aspose.com/c/cells/9). Publique su pregunta en el foro de Aspose.Cells for Go via C++ y será respondida en unas pocas horas. Las preguntas son respondidas directamente por el equipo de soporte de Aspose.Cells para Go.

Cuando esperes una respuesta en los foros, por favor ten en cuenta las diferencias de huso horario.

Si tiene un problema con Aspose.Cells para Go en C++, siga estos pasos simples para asegurarse de que se resuelva de la manera más eficiente:

- Asegúrese de usar la última versión de Aspose.Cells para Go antes de reportar el problema.
- Al informar un problema, por favor incluye el documento original y posiblemente un fragmento del código que causa el problema. Si necesitas adjuntar varios archivos, comprímelos en uno solo. Es seguro adjuntar tus documentos en Aspose.Forums, ya que solo tú y los desarrolladores de Aspose tendrán acceso a los archivos adjuntos.
- Intente informar solo un problema por hilo. Si tiene otro problema, informe en un hilo separado.
