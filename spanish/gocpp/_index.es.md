**Bienvenido a Aspose.Cells para Go**

Aspose.Cells for Go via C++ es una biblioteca nativa en Go que permite a los desarrolladores crear, manipular y convertir hojas de cálculo de forma programática sin requerir Automatización de Office o Microsoft Excel. Aspose.Cells para Go soporta formatos de hojas de cálculo como MS Excel 97-2003 (XLS), MS Excel 2007-2013/2016 (XLSX, XLSM, XLSB, etc.), Open Office XML y otros formatos. También puedes renderizar imágenes desde las hojas y convertir hojas de cálculo de MS Excel a formato PDF. La API también permite crear, manipular y renderizar gráficos y formas. Además, el cálculo de fórmulas es otra buena función que ofrece el componente y puedes aprovecharla.
