{{% alert color="primary" %}}

Aspose.Cells ahora admite obtener la fecha de actualización y la información de quién la actualizó desde un libro de trabajo.

{{% /alert %}}

[**PivotTable.GetRefreshDate()**](https://reference.aspose.com/cells/go-cpp/pivottable/getrefreshdate/) devuelve la fecha en la que se actualizó por última vez el informe de PivotTable. De manera similar, la propiedad [**PivotTable.GetRefreshedByWho()**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/getrefreshedbywho/) devuelve el nombre del usuario que actualizó el informe por última vez. El siguiente ejemplo demuestra esta función, y el archivo de muestra se puede descargar desde el siguiente enlace.
