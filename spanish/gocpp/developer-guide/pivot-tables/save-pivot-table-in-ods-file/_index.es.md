Aspose.Cells proporciona la capacidad de guardar tablas dinámicas en archivos ODS. Para ello, puede convertir un libro de trabajo con una tabla dinámica existente o crear una nueva y guardar el archivo en formato ODS. Asegúrese de llamar a [**PivotTable::CalculateData()**](https://reference.aspose.com/cells/go-cpp/pivottable/calculatedata/) antes de guardar para garantizar que la tabla dinámica se renderice en el archivo ODS de salida.
