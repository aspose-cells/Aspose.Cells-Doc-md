## **Escenarios de uso posibles**

Puedes cambiar el idioma de un archivo Excel haciendo clic derecho en el archivo, seleccionando Propiedades > Detalles y editando el campo Idioma. Por favor, usa la propiedad [**BuiltInDocumentPropertyCollection.GetLanguage()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getlanguage/) para cambiarlo programáticamente usando las APIs de Aspose.Cells.
