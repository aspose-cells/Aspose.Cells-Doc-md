## **Crear rango de unión**
Aspose.Cells ofrece la capacidad de crear Rangos de Unión usando el método [CreateUnionRange](https://reference.aspose.com/cells/go-cpp/worksheetcollection/createunionrange/). El método [CreateUnionRange](https://reference.aspose.com/cells/go-cpp/worksheetcollection/createunionrange/) acepta dos parámetros, la dirección para crear el rango de unión y el índice de la hoja de cálculo. El método [CreateUnionRange](https://reference.aspose.com/cells/go-cpp/worksheetcollection/createunionrange/) devuelve un objeto [UnionRange](https://reference.aspose.com/cells/cpp/aspose.cells/unionrange/).
