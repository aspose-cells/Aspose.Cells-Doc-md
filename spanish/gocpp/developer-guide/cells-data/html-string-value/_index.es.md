{{% alert color="primary" %}}

Aspose.Cells admite la conversión de HTML orientado a Microsoft Excel a formato XLS/XLSX. Esto significa que el HTML generado por Microsoft Excel se puede convertir de nuevo al formato XLS/XLSX utilizando Aspose.Cells.

De manera similar, si hay HTML simple, Aspose.Cells puede convertirlo en HTML Rich Text. Aspose.Cells proporciona el método [**Cell::GetHtmlString**](https://reference.aspose.com/cells/go-cpp/cell/gethtmlstring/) que puede tomar tal HTML simple y convertirlo en texto de celda formateado.
