A veces, es necesario agregar o registrar la referencia de la biblioteca en el proyecto VBA a través del código. Puedes hacerlo usando el método [**VbaProject.References.AddRegisteredReference()**](https://reference.aspose.com/cells/go-cpp/vbaprojectreferencecollection/addregisteredreference/) de Aspose.Cells.

{{% /alert %}}

## **Agrega una referencia de la biblioteca al proyecto VBA en Microsoft Excel**

En Microsoft Excel, puedes agregar una referencia de la biblioteca al proyecto VBA haciendo clic en **Herramientas > Referencias...** manualmente.
