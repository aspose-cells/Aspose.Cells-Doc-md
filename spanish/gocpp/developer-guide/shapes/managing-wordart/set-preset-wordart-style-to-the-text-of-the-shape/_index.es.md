## **Escenarios de uso posibles**
Puede establecer un estilo de WordArt predefinido en el texto de la forma usando Aspose.Cells. Por favor, utilice [FontSetting.SetWordArtStyle()](https://reference.aspose.com/cells/go-cpp/fontsetting/setwordartstyle/) o [FontSettingCollection.SetWordArtStyle()](https://reference.aspose.com/cells/cpp/aspose.cells.drawing.texts/fontsettingcollection/setwordartstyle/) para este propósito.
