## **Escenarios de uso posibles**

Microsoft Excel le permite cambiar la etiqueta de visualización del objeto Ole como se muestra en la siguiente captura de pantalla. También puede acceder o modificar la etiqueta de visualización del objeto Ole usando las API de Aspose.Cells con los métodos [**GetLabel()**](https://reference.aspose.com/cells/go-cpp/oleobject/getlabel/) y [**SetLabel(const U16String\& value)**](https://reference.aspose.com/cells/cpp/aspose.cells.drawing/oleobject/setlabel/).
