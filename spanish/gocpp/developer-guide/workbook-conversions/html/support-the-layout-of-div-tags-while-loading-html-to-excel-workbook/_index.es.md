{{% alert color="primary" %}} 

Normalmente, se ignora el diseño de las etiquetas DIV al cargar HTML en un objeto de libro de Excel. Sin embargo, si deseas que se preserve el diseño de las etiquetas DIV, establece la propiedad [GetSupportDivTag()](https://reference.aspose.com/cells/go-cpp/htmlloadoptions/getsupportdivtag/) en **true**. El valor predeterminado de esta propiedad es **false**.

{{% /alert %}}
