## **Escenarios de uso posibles**

Cuando guardas tu archivo de Excel en HTML, los comentarios no se exportan. Sin embargo, Aspose.Cells ofrece esta función usando la propiedad [**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/isexportcomments/). Si la configuras en **true**, el HTML también mostrará los comentarios presentes en tu archivo de Excel.
