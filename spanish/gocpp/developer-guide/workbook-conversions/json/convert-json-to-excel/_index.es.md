## **Convertir JSON a hoja de cálculo de Excel** 
No necesitas preguntarte cómo convertir JSON a un archivo de Excel, porque la biblioteca Aspose.Cells for C++ tiene la mejor solución. La API Aspose.Cells admite la conversión de formato JSON a hojas de cálculo. Puedes usar la clase [JsonLoadOptions](https://reference.aspose.com/cells/go-cpp/jsonloadoptions/) para especificar configuraciones adicionales para importar JSON a Workbook.
