{{% alert color="primary" %}}

Puedes usar Aspose.Cells para crear etiquetas de datos personalizadas con texto enriquecido en los puntos del gráfico. Aspose.Cells proporciona el método [DataLabels.Characters()](https://reference.aspose.com/cells/go-cpp/charttextframe/characters/) para devolver el objeto [FontSetting](https://reference.aspose.com/cells/cpp/aspose.cells/fontsetting/) que puede usarse para configurar las propiedades de la fuente del texto, como su color, negrita, etc.
