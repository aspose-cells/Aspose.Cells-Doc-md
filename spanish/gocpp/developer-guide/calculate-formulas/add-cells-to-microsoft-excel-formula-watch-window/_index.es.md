## **Escenarios de uso posibles**

La ventana de observación de Excel es una herramienta útil para monitorear fácilmente los valores de las celdas y sus fórmulas en una ventana. Puedes abrir la *Ventana de Observación* en Microsoft Excel haciendo clic en *Fórmulas > Ventana de Observación*. Tiene un botón *Agregar Observación* que se puede usar para agregar celdas para inspección. De manera similar, puedes usar el método [**Worksheet.CellWatches.Add()**](https://reference.aspose.com/cells/go-cpp/cellwatchcollection/add_int_int/) para agregar celdas a la *Ventana de Observación* usando la API de Aspose.Cells.
