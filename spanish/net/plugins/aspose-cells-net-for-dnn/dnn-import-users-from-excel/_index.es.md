## **Introducción**
El módulo de importación de usuarios desde Excel de Aspose .NET permite a los desarrolladores importar usuarios masivos desde un archivo de Excel. Este módulo demuestra la potente característica de hoja de cálculo de Aspose.Cells.
### **Características del módulo**
Esta versión inicial del módulo está enriquecida con las siguientes características para hacer que el proceso de importación sea simple y fácil de usar:

- Le permite cargar un archivo de Excel y filtra automáticamente a los usuarios existentes.
- Omite automáticamente a los usuarios existentes para evitar cualquier confusión.
- Generar contraseñas dinámicas para los usuarios recién creados.
- Asignar los roles de seguridad "Usuarios registrados" y "Suscriptores" a los usuarios recién creados.

## **Instalación**
Una vez descargado, siga estos pasos para instalar el módulo en su sitio web de DNN:

1. Inicie sesión en su sitio como Host o con una cuenta de superusuario.
1. Vaya al menú **Host** y seleccione **Extensiones**.
1. Haga clic en **Asistente para instalar extensión**.
1. Siga las instrucciones para buscar la ubicación del archivo zip descargado, selecciónelo y luego haga clic en **Abrir**.
1. Haz clic en **Siguiente**, acepta la licencia y continúa con la instalación. Cuando hayas terminado, haz clic en el botón **Volver**.

Por favor, revisa [este video de instalación de módulos](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN para más detalles.
## **Usando**
Después de haber instalado el módulo Aspose .NET Cells Import Users from Excel for DNN, es realmente simple comenzar a usarlo en tu sitio web. Sigue estos simples pasos para empezar:

1. Asegúrate de haber iniciado sesión en DNN con una cuenta de nivel Host o Administrador.
1. Navega a la página en la que deseas agregar el módulo de exportación.
1. Selecciona **Módulos** seguido por **Agregar Nuevo Módulo** desde la cinta superior. 

1. Desde la lista, selecciona 'Aspose Import Users from Excel module for DNN' y arrástralo a la posición deseada.

Has añadido exitosamente el módulo Aspose .NET Cells Import Users from Excel para DNN a tu página. Los controles de subida de archivos serán visibles junto con el enlace de 'Archivo de Plantilla'. Puedes descargar el archivo de plantilla y formatear tu contenido de acuerdo con el archivo de plantilla. Una vez que subas tu archivo de Excel, procesará automáticamente a todos los usuarios y filtrará a los usuarios existentes. Puedes seleccionar los usuarios que deseas importar y presionar el botón 'Importar Usuarios'.
