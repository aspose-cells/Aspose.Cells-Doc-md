## **Introducción**

El módulo Aspose .NET DNN Data Exporter to Excel permite a los usuarios exportar datos directamente desde tablas de bases de datos locales o remotas, vistas y consultas personalizadas en Excel de Microsoft u hojas de cálculo de OpenOffice. Este módulo demuestra la potente funcionalidad de construcción de hojas de cálculo proporcionada por Aspose.Cells.

### **Características del módulo**

Esta versión inicial del módulo viene enriquecida con las siguientes características para hacer que el proceso de exportación sea simple y fácil de usar

- Permite conectar a la Base de Datos Local de MS SQL Server
- Permite conectar a la Base de Datos Remota de MS SQL Server
- Poblar todas las Tablas de la base de datos conectada
- Rellenar todas las vistas desde la base de datos conectada
- Permitir escribir una consulta personalizada
- Ajustar automáticamente las columnas al contenido
- Permitir omitir cadenas de más de 32k en celdas de Excel (LoadOptions)
- Aplicar formato de texto en negrita como encabezado de columna
- Permitir utilizar como fuente de datos (tabla, vistas, consulta personalizada)
- Exportar datos a documentos de Microsoft Excel (.xls, .xlsx y .xlsb)
- Exportar datos a documento de texto delimitado por tabulaciones (*.txt)
- Exportar datos a CSV (delimitado por comas) (*.csv)
- Exportar datos a hoja de cálculo de OpenDocument (*.ods)
- Opción para seleccionar el formato de salida deseado antes de exportar
- El documento exportado se envía automáticamente al navegador para su descarga

## **Instalación**

Una vez descargado, siga estos pasos para instalar el módulo en su sitio web de DNN:

1. Inicie sesión en su sitio como Host o con una cuenta de superusuario.
1. Vaya al menú **Host** y seleccione **Extensiones**.
1. Haga clic en **Asistente para instalar extensión**.
1. Siga las instrucciones para buscar la ubicación del archivo zip descargado, selecciónelo y luego haga clic en **Abrir**.
1. Haz clic en **Siguiente**, acepta la licencia y continúa con la instalación. Cuando hayas terminado, haz clic en el botón **Volver**.

Por favor, revisa [este video de instalación de módulos](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN para más detalles.

## **Usando**

Después de haber instalado el módulo Aspose .NET Database Data Exporter for DNN, es realmente fácil comenzar a usarlo en tu sitio web. Sigue estos simples pasos para empezar:

1. Asegúrate de haber iniciado sesión en DNN con una cuenta de nivel Host o Administrador.
1. Navega a la página en la que deseas agregar el módulo de exportación.
1. Selecciona **Módulos** seguido por **Agregar Nuevo Módulo** desde la cinta superior.
1. Desde la lista, selecciona 'Aspose.NET DNN Data Exporter to Excel Using Aspose.Cells' y arrástralo a la posición.
