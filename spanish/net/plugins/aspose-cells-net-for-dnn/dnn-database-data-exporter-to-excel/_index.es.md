## **Introducción**

Aspose .NET DNN Data Exporter to Excel Module permite a los usuarios exportar datos directamente desde tablas de bases de datos locales o remotas, vistas y consultas personalizadas a Microsoft Excel u OpenOffice Spreadsheet. Este módulo demuestra la potente función de creación de hojas de cálculo proporcionada por Aspose.Cells.

### **Características del módulo**

Esta versión inicial del módulo está enriquecida con las siguientes características geniales para que el proceso de exportación sea simple y fácil de usar.

- Permitir conectar la base de datos local de MS SQL Server
- Permitir conectar la base de datos remota de MS SQL Server
- Rellene todas las tablas de la base de datos conectada
- Rellene todas las vistas de la base de datos conectada
- Permitir escribir consulta personalizada
- Columnas de ajuste automático a la longitud del contenido.
- Permitir omitir cadenas de más de 32k en celdas de Excel (LoadOptions)
- Aplicar formato de columna de encabezado como texto en negrita
- Permitir usar como fuente de datos (tabla, vistas, consulta personalizada)
- Exportar datos a Microsoft Documentos de Excel (.xls, .xlsx y .xlsb)
- Exportar datos a un documento de texto delimitado por tabulaciones (*.txt)
- Exportar datos a CSV (delimitado por comas) (*.csv)
- Exportar datos a la hoja de cálculo de OpenDocument (*.ods)
- Opción para seleccionar el formato de salida deseado antes de exportar.
- El documento exportado se envía automáticamente al navegador para su descarga.

## **Instalando**

Una vez descargado, siga estos pasos para instalar el módulo en su sitio web de DNN:

1. Inicie sesión en su sitio como Anfitrión u otra cuenta de nivel de superusuario.
1.  Navegar a la**Anfitrión** menú y seleccione**Extensiones**.
1.  Hacer clic**Instalar asistente de extensión**.
1. Como se indica, busque la ubicación del archivo zip descargado, selecciónelo y luego haga clic en**Abierto**.
1.  Hacer clic**Próximo** , acepte la licencia, continúe con la instalación. Cuando haya terminado, haga clic en el**Devolver** botón.

 por favor, compruebe[video de instalacion de este modulo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) de DNN para más detalles.

## **Usando**

Después de haber instalado el módulo Aspose .NET Database Data Expoeter para DNN, es muy sencillo comenzar a usarlo en su sitio web. Siga estos sencillos pasos para comenzar:

1. Asegúrese de haber iniciado sesión en DNN con una cuenta de nivel de anfitrión o administrador.
1. Navegue a la página donde desea agregar el módulo Exportar.
1.  Seleccione**Módulos** seguido por**Agregar nuevo módulo** de la cinta superior.
1. De la lista, seleccione 'Aspose.NET DNN Data Exporter to Excel usando Aspose.Cells' y arrástrelo a su posición.

![todo:imagen_alternativa_texto](picture2.png)

Ha agregado con éxito el módulo Aspose .NET Exportador de datos de base de datos a Excel para DNN a su página.

![todo:imagen_alternativa_texto](picture3.png)

1. Ingrese o use la cadena de conexión de MS SQL Server rellenada previamente
1. Tipo de fuente de datos de Seletec (tabla, vista, consulta personalizada)
1. Seleccione o ingrese la fuente de datos (tabla, vista, consulta personalizada)
1. Seleccionar tipo de exportación
1. Haga clic en Exportar datos
1. El archivo deseado se descargará automáticamente.
