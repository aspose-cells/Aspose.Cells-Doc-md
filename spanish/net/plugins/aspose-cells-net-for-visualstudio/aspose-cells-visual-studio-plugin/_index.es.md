## **Introducción**

Microsoft Visual Studio es un entorno de desarrollo integrado (IDE) de Microsoft. Se utiliza para desarrollar programas de computadora para Microsoft Windows, así como sitios web, aplicaciones web y servicios web. Visual Studio utiliza plataformas de desarrollo de software de Microsoft como API de Windows, Windows Forms, Windows Presentation Foundation, Windows Store y Microsoft Silverlight. Puede producir tanto código nativo como código administrado.

### **Aspose.Cells for .NET**

Aspose.Cells for .NET proporciona el grupo más flexible de componentes que permiten a las aplicaciones .NET crear, modificar, convertir, generar e imprimir hojas de cálculo de Excel® sin necesidad de tener instalado Microsoft Excel® en el servidor.

Aspose.Cells for .NET soporta todos los principales formatos de archivo de Microsoft Excel y otros incluyendo XLS, XLSX, XLSM, XLTX/XLTM, HTML, CSV, ODS, Tabulado, SpreadsheetML y PDF, etc.
Para más detalles sobre la [API Aspose.Cells for .NET](https://products.aspose.com/cells/net/)

### **Explora Ejemplos de Aspose.Cells for .NET usando Visual Studio**

El Plugin de Aspose para Visual Studio - **Explora Ejemplos de Aspose.Cells for .NET** es una gran herramienta para descargar y explorar rápidamente ejemplos de la API Aspose.Cells for .NET. Ahorra mucho tiempo y esfuerzo al proporcionar una opción muy simple para seleccionar, descargar y abrir los últimos proyectos de ejemplo sin tener que salir de tu Visual Studio.

Esta versión proporciona las siguientes características:

- Compatible con Visual Studio 2010, Visual Studio 2012 y Visual Studio 2013.
- Fácil de iniciar desde el menú **Archivo > Aspose** en Visual Studio
- Ahorra tiempo y reduce la curva de aprendizaje.
- Le permite seleccionar y abrir los últimos ejemplos de Aspose.Cells for .NET API alojados en Github.com.
- Le permite abrir proyectos de ejemplos en **C#** o **VB.NET**
- Un proyecto seleccionado se abre automáticamente en Visual Studio con todas las referencias requeridas configuradas para brindarle un entorno listo para ejecutar y explorar.
- Descarga automáticamente las últimas bibliotecas de API desde **NuGet** y ejemplos desde **GitHub**.

## **Instalación**

Es muy simple y fácil de instalar el complemento Aspose Visual Studio, por favor siga estos sencillos pasos

- Haga doble clic en el archivo msi descargado
- Siga los pasos, simplemente presione Siguiente en cada pantalla para instalarlo usando la configuración estándar.

Recibirá un diálogo de éxito una vez la instalación haya sido completada.

![todo:image_alt_text](aspose-cells-visual-studio-plugin_2)

**Nota:** Por favor asegúrese de reiniciar Visual Studio para que los cambios tengan efecto.

## **Usando**

Una vez instalado, puedes lanzar fácilmente y usar el complemento interactivo como se describe a continuación:
**1.** Abre el complemento seleccionando **Archivo > Aspose**, seguido de **Nuevo Proyecto de Ejemplo Aspose.Cells** en Visual Studio.

![todo:image_alt_text](aspose-cells-visual-studio-plugin_3)

**2.** La primera vez que abres el complemento, verás el cuadro de diálogo de selección de API como se muestra a continuación. Tomará unos minutos descargar la última API y sus bibliotecas desde **NuGet** y el código fuente de los ejemplos más recientes desde **GitHub**.

![todo:image_alt_text](aspose-cells-visual-studio-plugin_4)

**3.** En el cuadro de diálogo Seleccionar una API para Ver Ejemplos, selecciona o cambia la ubicación donde se crearán los proyectos de ejemplo y selecciona o cambia el tipo de proyecto de ejemplos como **C#** o **VB.NET** y haz clic en **Continuar** para abrir el proyecto de ejemplo en Visual Studio.

![todo:image_alt_text](aspose-cells-visual-studio-plugin_1)

**4.** Nuevos proyectos de ejemplos según la selección del tipo de proyecto de **C#** o **VB.NET** en Visual Studio.

#### **Cómo configurar el código fuente**

Debe tener instalado lo siguiente para abrir y extender el código fuente

- Visual Studio 2010

Siga estos simples pasos para comenzar

1. Descargar/Clonar el código fuente.
1. Abra Visual Studio 2010 y elija **Archivo** > **Abrir Proyecto**
1. Navega hasta el último código fuente que has descargado y abre **AsposeVisualStudioPlugin.sln**

#### **Resumen del código fuente**

Hay dos proyectos en la solución

- **AsposeVisualStudioPlugin** - Contiene la integración con Visual Studio, selección y descarga automática de Aspose APIs y Ejemplos.
- **Instalador** - Proyecto de instalador Wix para generar el archivo instalador msi para el plugin.
