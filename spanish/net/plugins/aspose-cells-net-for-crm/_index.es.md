# **¿Qué es Microsoft Dynamics CRM?**
Microsoft Dynamics CRM es un paquete de software de gestión de relaciones con los clientes desarrollado por Microsoft. De manera predeterminada, el producto se enfoca principalmente en los sectores de Ventas, Marketing y Servicio (centro de ayuda), pero Microsoft ha estado comercializando Dynamics CRM como una plataforma XRM y ha estado alentando a los socios a utilizar su marco propietario (basado en .NET) para personalizarlo. Forma parte de la familia de aplicaciones comerciales de Microsoft Dynamics.

Dynamics CRM es una aplicación cliente-servidor, que, al igual que Microsoft SharePoint, es principalmente una aplicación web basada en IIS que también admite amplias interfaces de servicios web. Los clientes acceden a Dynamics CRM ya sea utilizando un navegador o mediante un complemento de cliente grueso para Microsoft Outlook.
# **Aspose.Cells for .Net**
Aspose.Cells for .NET proporciona el grupo más flexible de componentes que permiten a la aplicación .NET crear y gestionar hojas de cálculo de Excel sin requerir que Microsoft Excel esté instalado en el servidor. Aspose.Cells es uno de los principales componentes para la manipulacion de todos los tipos de formatos de Excel.

Aspose.Cells no solo admite la generación de hojas de cálculo y otras funciones básicas de formateo de archivos, sino que también soporta una serie de funciones avanzadas. Estas características avanzadas facilitan mucho la manipulación de los contenidos de las hojas de cálculo, el formateo de celdas y la protección de archivos. Aspose.Cells también permite importar datos a las hojas de cálculo desde diferentes fuentes de datos, agregar fórmulas/funciones matemáticas, de fecha, financieras, de texto comunes y complejas, admite la manipulación de gráficos, imágenes, comentarios, objetos de dibujo y controles, etc.
# **Aspose.Cells for CRM**
Aspose .NET se puede usar con Dynamics CRM de múltiples maneras. Cuando trabajas con hojas de cálculo, ya sea generación o procesamiento de hojas de cálculo, necesitas utilizar las APIs de Aspose que pueden brindar resultados rápidos con un mínimo de líneas de código.
