## **Introducción**
La herramienta de importación de actualización de Aspose es un complemento de código abierto que se utilizará en Microsoft Dynamic CRM. Este complemento es capaz de actualizar automáticamente y/o exportar datos de CRM a hojas de cálculo de Aspose.Cells. No necesitamos hacer nada manualmente e instalar este complemento y hacer que nuestro trabajo sea más fácil y rápido.

Este complemento proporciona funcionalidad para actualizar automáticamente y/o exportar desde CRM. Con un clic, importará/exportará el registro hacia/desde CRM.

Las principales características de este complemento son:

- Establecer la conexión al CRM deseado.
- Permitir al usuario seleccionar la entidad y los campos deseados.
- Obtener el registro de CRM para mostrarlo en la cuadrícula.
- Exportar el registro a hojas de cálculo de Aspose.Cells.
- Obtener el registro de las hojas de cálculo de Aspose.Cells para mostrarlo en la cuadrícula.
- Exportar el registro de la cuadrícula al CRM.
## **Requisitos del sistema y plataformas compatibles**
### **Requisitos del sistema**
Para instalar y usar la herramienta de importación de actualización de Aspose .Net para Microsoft Dynamics CRM, debe tener instalada una de las siguientes versiones de CRM

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.

No dude en contactarnos si encuentra algún problema al instalar o usar este complemento.
### **Plataformas compatibles**
Este complemento funcionará con todas las versiones de Microsoft Dynamics CRM:

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.
