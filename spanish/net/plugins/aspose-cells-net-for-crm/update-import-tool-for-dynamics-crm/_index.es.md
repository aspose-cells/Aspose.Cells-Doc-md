## **Introducción**
La herramienta de importación de actualizaciones Aspose es un complemento de código abierto que se utiliza en Microsoft Dynamic CRM. Este complemento es capaz de actualizar y/o exportar automáticamente datos de CRM a hojas de cálculo Aspose.Cells. No necesitamos hacer nada manualmente e instalar este complemento y hacer que nuestro trabajo sea más fácil y rápido.

Este complemento proporciona funcionalidad para actualizar y/o exportar automáticamente desde CRM. Un clic importará/exportará el registro a/desde CRM.

Las principales características de este complemento son:

- Establezca la conexión con el CRM deseado.
- Permita que el usuario seleccione la entidad y los campos deseados.
- Obtener el registro de la pantalla de CRM en la cuadrícula.
- Exporte el registro a la hoja de cálculo Aspose.Cells.
- Obtenga el registro de Aspose.Cells Pantalla de hoja de cálculo en cuadrícula.
- Exporte el registro de la grilla a CRM.
## **Requisitos del sistema y plataformas compatibles**
### **Requisitos del sistema**
Para instalar y utilizar Aspose .Net Update Import Tool para Microsoft Dynamics CRM, debe tener instalada una de las siguientes versiones de CRM

- Microsoft Dinámica CRM 2013.
- Microsoft Dinámica CRM 2015.
- Microsoft Dinámica CRM en línea.

No dude en contactarnos si encuentra algún problema al instalar o usar este complemento.
### **Plataformas compatibles**
Este complemento funcionará con todas las versiones de Microsoft Dynamics CRM:

- Microsoft Dinámica CRM 2013.
- Microsoft Dinámica CRM 2015.
- Microsoft Dinámica CRM en línea.
