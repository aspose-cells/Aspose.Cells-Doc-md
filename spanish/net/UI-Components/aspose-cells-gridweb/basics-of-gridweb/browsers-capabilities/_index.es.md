{{% alert color="primary" %}} 

Aspose.Cells.GridWeb es un control web basado en GUI que se puede incrustar en formularios web de ASP.NET como otros controles web proporcionados por el Framework .NET de Microsoft. Lo más importante de un control web es brindar soporte multi-navegador. Aspose.Cells.GridWeb proporciona soporte multi-navegador.

{{% /alert %}} 
## **Comparación**
Aspose.Cells.GridWeb está construido utilizando el Framework .NET de Microsoft, por lo que tiene un soporte completo en Internet Explorer (IE) de Microsoft. Sin embargo, en otros navegadores tiene algunas limitaciones menores. Este tema proporciona una comparación detallada de qué funciones son compatibles en diferentes navegadores.
