{{% alert color="primary" %}} 

Aspose.Cells.GridWeb es un control web basado en GUI que se puede incrustar en formularios web ASP.NET como otros controles web proporcionados por Microsoft .NET Framework. Lo más importante de un control web es proporcionar compatibilidad con varios navegadores. Aspose.Cells.GridWeb proporciona soporte para varios navegadores.

{{% /alert %}} 
## **Comparación**
Aspose.Cells.GridWeb está construido usando Microsoft .NET Framework por lo que es totalmente compatible con Internet Explorer (IE) de Microsoft.
