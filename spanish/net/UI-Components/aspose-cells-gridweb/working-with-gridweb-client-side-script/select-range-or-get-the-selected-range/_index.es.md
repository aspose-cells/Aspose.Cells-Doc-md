{{% alert color="primary" %}} 

Puede usar las siguientes funciones del lado del cliente para seleccionar un rango o para obtener el rango seleccionado en un GridWeb usando JavaScript.

- getSelectRange()
- setSelectRange()
- clearSelections()

getSelectRange() devuelve el último rango seleccionado. setSelectRange() selecciona el rango dado. clearSelections() borra toda la selección excluyendo la celda activa actual.
