{{% alert color="primary" %}} 

Para que sea más fácil para los desarrolladores actualizar, mantenemos un documento que describe información crítica para los usuarios existentes, especialmente aquellos que han estado usando el Aspose.Grid.Desktop anterior y ahora necesitan actualizar al Aspose.Cells.GridDesktop nuevo o fusionado.

Este tema proporciona una guía básica paso a paso para principiantes para ayudarlos a actualizar Aspose.Grid.Desktop a Aspose.Cells.GridDesktop Control y crear y usar el control Aspose.Cells.GridDesktop en aplicaciones de Windows.

{{% /alert %}} 
## **Actualice de Aspose.Grid.Desktop a Aspose.Cells.GridDesktop Control**
Los desarrolladores que trabajaron con Aspose.Grid.Desktop pueden tener problemas al usar el nuevo Aspose.Cells.GridDesktop cuando lo actualicen. A continuación se muestra el espacio de nombres que se renombra:

1. **Aspose.Grid. El espacio de nombres de escritorio ha cambiado de nombre a Aspose.Cells. Espacio de nombres de GridDesktop.**
## **Creación y uso de Aspose.Cells.GridDesktop Control**
### **Creación de un proyecto de aplicación Windows**
En primer lugar, cree un proyecto de aplicación de Windows en el que usar el control Aspose.Cells.GridDesktop. Siga los siguientes pasos;

1. **Abra el IDE de Visual Studio.NET.**
1. **En el menú Archivo, seleccione Nuevo, el Proyecto.**
1. **Seleccione la aplicación Windows en el cuadro de diálogo Nuevo proyecto.**

Después de realizar los pasos anteriores, se creará una aplicación de ventana con un formulario vacío.
### **Agregar control GridDesktop al formulario**
 Arrastre y suelte el control Aspose.Cells.GridDesktop desde el cuadro de herramientas hasta el formulario. Después de esto, aparecerá un control en el formulario donde puede realizar varias acciones en él, como**cambiar el tamaño**, **cambiar sus propiedades**, **etc.**.
### **Ejecute la aplicación**
 Finalmente, ejecute la aplicación presionando**Ctrl+F5** o haciendo clic en el**comienzo** botón. Cuando se ejecuta la aplicación, vemos un formulario que contiene un control Aspose.Cells.GridDesktop vacío. Ahora puede agregar valores a las celdas haciendo clic en ellas. La siguiente captura de pantalla muestra cómo se verá finalmente su aplicación.

![todo:imagen_alternativa_texto](how-to-use-aspose-cells-griddesktop_1.png)

No solo es posible agregar valores al control, sino también realizar otras tareas como cambiar la altura de una fila o el ancho de una columna, copiar (Ctrl+C) o cortar (Ctrl+X) datos de celda al portapapeles y pegar (Ctrl+V) datos en la celda, etc. Para realizar más operaciones, haga clic con el botón derecho en el control para ver su menú contextual.

{{% alert color="primary" %}} 

Se ha cambiado el nombre de Aspose.Grid.Desktop y ahora forma parte de Aspose.Cells, por lo que ya no es posible utilizar la versión anterior de Aspose.Grid.Desktop. Solo para enfatizar, no hay mucho trabajo involucrado en la actualización a la última versión del componente Aspose.Cells.GridDesktop. No hay cambios en el API ya que las clases con los miembros, estructuras, enumeraciones, etc. siguen siendo las mismas. El único cambio que se ha realizado es con respecto al espacio de nombres del control.

{{% /alert %}}
