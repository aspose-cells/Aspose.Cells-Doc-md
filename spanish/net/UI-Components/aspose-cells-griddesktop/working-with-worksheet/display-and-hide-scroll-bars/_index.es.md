{{% alert color="primary" %}}

Las barras de desplazamiento son útiles para navegar por el contenido de hojas de cálculo que son anchas y profundas, es decir, que tienen muchas filas y columnas. La mayoría de las aplicaciones admiten dos tipos de barra de desplazamiento:

- Barra de desplazamiento vertical
- Barra de desplazamiento horizontal

Ambos se encuentran en Microsoft Excel.

GridDesktop Aspose.Cell de Aspose.Cell proporciona barras de desplazamiento horizontales y verticales para desplazarse por el contenido de una hoja de trabajo. Con las API Aspose.Cells.GridDesktop, los desarrolladores pueden controlar la visibilidad de ambas barras de desplazamiento.

{{% /alert %}}

## **Control de la visibilidad de la barra de desplazamiento**

Para controlar la visibilidad de la barra de desplazamiento en GridDesktop, use las propiedades IsVerticalScrollBarVisible e IsHorizontalScrollBarVisible. Los ejemplos a continuación muestran cómo ocultar y mostrar las barras de desplazamiento.
