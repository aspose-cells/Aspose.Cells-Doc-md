{{% alert color="primary" %}}

Las barras de desplazamiento son útiles para navegar por el contenido de las hojas de cálculo que son anchas y profundas, es decir, que tienen muchas filas y columnas. La mayoría de las aplicaciones admiten dos tipos de barras de desplazamiento:

- Barra de desplazamiento vertical
- Barra de desplazamiento horizontal

Ambas se encuentran en Microsoft Excel.

La API GridDesktop de Aspose.Cells proporciona barras de desplazamiento horizontales y verticales para desplazarse por el contenido de una hoja de cálculo. Con las API de Aspose.Cells.GridDesktop, los desarrolladores pueden controlar la visibilidad de ambas barras de desplazamiento.

{{% /alert %}}

## **Controlar la visibilidad de la barra de desplazamiento**

Para controlar la visibilidad de la barra de desplazamiento en GridDesktop, utilice las propiedades IsVerticalScrollBarVisible e IsHorizontalScrollBarVisible. Los ejemplos a continuación muestran cómo ocultar y mostrar las barras de desplazamiento.
