{{% alert color="primary" %}} 

En este tema, discutiremos cómo cambiar la fuente y el color de fuente de una celda. Esta característica proporciona un control más profundo de las hojas de trabajo a los desarrolladores. Con esta función, los desarrolladores pueden personalizar la fuente y el color de cualquier celda deseada.

{{% /alert %}} 
## **Cambiar la fuente y el color de un Cell**
Para cambiar la fuente y el color de una celda usando Aspose.Cells.GridDesktop, siga los pasos a continuación:

-  Accede a cualquier deseado**Hoja de cálculo**
-  Accede a un**Cell** cuya fuente y color se van a cambiar
-  Crea una personalizada**Fuente**
-  Selecciona el**Fuente** del**Cell** al personalizado
-  Finalmente, establezca**Color de fuente** del**Cell** a cualquier deseado**Color**
