{{% alert color="primary" %}} 

Aspose.Cells.GridDesktop es un control basado en GUI que permite a los usuarios trabajar con sus datos en modo WYSIWYG. Hemos explorado mucho la API de Aspose.Cells.GridDesktop para realizar diferentes tipos de tareas. También hemos discutido sobre cómo controlar la configuración de formato de la cuadrícula, las hojas de cálculo, filas y columnas y celdas programáticamente utilizando la API robusta y más fácil de Aspose.Cells.GridDesktop. Pero todo eso fue desde el punto de vista de la programación. En este tema, discutiremos cómo un usuario final puede controlar la configuración de formato de Aspose.Cells.GridDesktop utilizando su **Cuadro de diálogo de estilo**.

{{% /alert %}} 
## **Usar el Cuadro de diálogo de estilo**
Para usar el **Cuadro de diálogo de estilo** en Aspose.Cells.GridDesktop, sigue los pasos a continuación:

- **Haz clic derecho** en la cuadrícula. Aparecerá un menú contextual que te ofrecerá algunas opciones.
- Selecciona la opción **Formato de celdas...** del menú contextual.

Después de seleccionar la opción **Formato de celdas...**, aparecerá el **Cuadro de diálogo de estilo**. El **Cuadro de diálogo de estilo** consta de varias pestañas, incluida: * **Número**

- **Alineación**
- **Borde**
- **Fuente**
- **Protección**

Vamos a discutir y ver cada pestaña de la **Ventana de Estilos** una por una.
### **Pestaña de Números**
La pestaña de **Números** se utiliza para establecer la configuración de **Formato de Número** de una celda. Puede seleccionar una categoría de formato numérico, establecer decimales para el valor, seleccionar un conjunto de símbolos y también seleccionar cómo mostrar un valor negativo en la celda.
### **Pestaña de Alineación**
Usando la pestaña de **Alineación**, los usuarios pueden establecer la alineación horizontal y vertical del texto. Incluso pueden usar otras opciones de control de texto como ajustar el texto, reducir la celda para ajustar el texto y fusionar celdas.
### **Pestaña de Bordes**
La pestaña de **Bordes** facilita a los usuarios dibujar bordes de cualquier tipo y color con solo unos pocos clics. Todo lo que tienen que hacer es seleccionar algunas opciones para dibujar un borde según sus deseos.
### **Pestaña de Fuente**
Usando la pestaña de **Fuente**, los usuarios pueden cambiar el nombre de la fuente, el tamaño, el color y los estilos, etc. Los usuarios también pueden obtener una vista previa de la configuración de la fuente antes de aplicarla a las celdas. Por lo tanto, facilita el trabajo de los usuarios para formatear las celdas **Just In Time**.
### **Pestaña de Protección**
La pestaña de **Protección** simplemente permite a los usuarios proteger sus celdas bloqueándolas o ocultándolas. 

{{% alert color="primary" %}} 

Los desarrolladores también pueden abrir la **Ventana de Estilos** programáticamente llamando al método **MostrarVentanaEstilos** de **GridDesktop**.
