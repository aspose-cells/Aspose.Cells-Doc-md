## **Aspose.Cells.GridDesktop – Control de Cuadrícula de Escritorio**
### **Funciones de importación/exportación**
- Importar contenido desde hojas de cálculo de Excel.
- Importar datos directamente desde un objeto DataTable.
- Exportar contenido a hojas de cálculo de Excel.
### **Características de la hoja de cálculo**
- Agregar, insertar o eliminar hojas de cálculo.
- Renombrar hojas de cálculo.
- Ordenar datos de la hoja de cálculo.
### **Funciones de fila y columna**
- Agregar, insertar o eliminar filas y columnas.
- Ocultar o mostrar filas y columnas.
- Congelar o descongelar filas y columnas.
### **Funciones de celda**
- Admite configuraciones de formato de números.
- Controlar la alineación del texto.
- Gestionar la configuración de bordes.
- Controlar los ajustes de fuente.
- Fusionar o dividir celdas.
### **Funciones de formato**
- Controlar el comportamiento, aspecto y sensación de la cuadrícula.
- Aplicar el estilo y fuente de una celda centrada a otras celdas utilizando el Pincel de formato.
- Manipular datos y estilos de celda fácilmente.
### **Características de utilidad**
- Agregar o eliminar comentarios de celda.
- Insertar o eliminar imágenes.
- Agregar o eliminar hipervínculos.
- Agregar o eliminar reglas de validación.
- Agregar o eliminar controles (como CheckBox, ComboBox, etc.) en celdas.
- Navegación por teclado de celdas.
### **Características de Seguridad**
- Protege tus celdas bloqueándolas o ocultándolas.
### **Funciones Avanzadas**
- Admite la creación y cálculo de fórmulas.
- Realiza tus acciones deseadas en diferentes eventos de la cuadrícula.
### **Versiones de Microsoft Excel compatibles**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **Plataformas Compatibles**
- Formularios de Windows
