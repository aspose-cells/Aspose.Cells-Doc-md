## **Aspose.Cells.GridDesktop: control de cuadrícula de escritorio**
### **Funciones de importación/exportación**
- Importa contenido desde hojas de cálculo de Excel.
- Importe datos directamente desde un objeto DataTable.
- Exportar contenido a hojas de cálculo de Excel.
### **Características de la hoja de trabajo**
- Agregar, insertar o eliminar hojas de trabajo.
- Cambiar el nombre de las hojas de trabajo.
- Ordenar datos de la hoja de trabajo.
### **Funciones de fila y columna**
- Agregar, insertar o eliminar filas y columnas.
- Ocultar o mostrar filas y columnas.
- Congele o descongele filas y columnas.
### **Cell Características**
- Admite configuraciones de formato de número.
- Controla la alineación del texto.
- Administre la configuración de los bordes.
- Controla la configuración de la fuente.
- Combinar o dividir celdas.
### **Funciones de formato**
- Controle el comportamiento, el aspecto y la sensación de la cuadrícula.
- Aplique el estilo y la fuente de una celda enfocada a otras celdas usando Format Painter.
- Manipule datos y estilos de celda fácilmente.
### **Funciones de utilidad**
- Agregue o elimine comentarios de celda.
- Insertar o eliminar imágenes.
- Agregar o eliminar hipervínculos.
- Agregar o eliminar reglas de validación.
- Agregue o elimine controles (como CheckBox, ComboBox, etc.) en las celdas.
- Teclado de navegación de celdas.
### **Características de seguridad**
- Proteja sus celdas bloqueándolas u ocultándolas.
### **Funciones avanzadas**
- Admite la creación y el cálculo de fórmulas.
- Realice sus acciones deseadas bajo diferentes eventos de cuadrícula.
### **Versiones de Excel compatibles con Microsoft**
- Microsoft excel 97
- Microsoft excel 2000
- Microsoft Excel XP
- Microsoft excel 2003
- Microsoft excel 2007
- Microsoft excel 2010
- Microsoft excel 2013
### **Plataformas compatibles**
- Windows Formularios
