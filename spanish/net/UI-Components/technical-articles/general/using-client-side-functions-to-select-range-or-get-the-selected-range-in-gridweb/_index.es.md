{{% alert color="primary" %}} 

Puede usar las siguientes funciones del lado del cliente para seleccionar un rango o para obtener el rango seleccionado en GridWeb usando JavaScript.

- getSelectRange()
- establecerSeleccionarRango()
- borrar selecciones ()

getSelectRange() devuelve el último rango seleccionado. setSelectRange() selecciona el rango dado. clearSelections() borra toda la selección excepto la celda activa actual.
