Aspose.Cells proporciona funciones ricas para administrar formas en la hoja de cálculo. A veces es necesario obtener los puntos de conexión de una forma para alinear o colocar las formas en el lugar adecuado. Para ello, se requieren todos los puntos de conexión. El siguiente código se puede usar para obtener la lista de puntos de conexión de una forma usando el[Forma.ConnectionPoints](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/connectionpoints) propiedad.
