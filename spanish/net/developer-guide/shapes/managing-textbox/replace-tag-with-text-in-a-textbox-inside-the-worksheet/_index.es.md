## **Posibles escenarios de uso**
Los cuadros de texto pueden tener etiquetas que se pueden reemplazar con algún texto en tiempo de ejecución para configurarlos de acuerdo con el requisito. Las etiquetas pueden ser algunas etiquetas encerradas entre paréntesis angulares '<' and '>. Puede haber múltiples etiquetas dentro de un solo cuadro de texto.
