veces, necesita copiar elementos en una hoja de cálculo, por ejemplo, imágenes, gráficos y otros objetos de dibujo, entre hojas de cálculo. Aspose.Cells admite esta función. Los gráficos, imágenes y otros objetos se pueden copiar con el mayor grado de precisión.
