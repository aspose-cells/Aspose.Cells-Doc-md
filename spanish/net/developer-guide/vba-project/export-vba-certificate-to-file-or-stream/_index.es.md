Aspose.Cells le permite exportar Certificado Digital VBA a una secuencia como un archivo o memoria. Puede acceder a los datos en bruto del certificado digital VBA utilizando la propiedad [**Workbook.VbaProject.CertRawData**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/certrawdata).
