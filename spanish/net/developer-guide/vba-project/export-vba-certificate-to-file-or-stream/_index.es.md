{{% alert color="primary" %}}

 Aspose.Cells le permite exportar el certificado digital VBA a una transmisión, como un archivo o una transmisión de memoria. Puede acceder a los datos sin procesar del certificado digital VBA utilizando el[**Libro de trabajo.VbaProject.CertRawData**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/certrawdata) propiedad.

{{% /alert %}}
