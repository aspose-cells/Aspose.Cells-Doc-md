## Verifica si el Proyecto VBA está Protegido y Bloqueado para Visualización en C#

Aspose.Cells te permite verificar si el Proyecto VBA (Visual Basic for Applications) de un archivo Excel está protegido y bloqueado para visualización. Para ello, la API proporciona la propiedad [**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing). Si está bloqueado para visualización, entonces la propiedad [**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) devuelve **true**.
