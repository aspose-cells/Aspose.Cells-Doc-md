## Compruebe si el proyecto VBA está protegido y bloqueado para su visualización en C#

Aspose.Cells le permite verificar si el proyecto VBA (Visual Basic para aplicaciones) del archivo de Excel está protegido y bloqueado para su visualización. Para esto, el API proporciona el[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) propiedad. Si está bloqueado para su visualización, entonces el[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) devoluciones de propiedad**verdadero**.
