{{% alert color="primary" %}}

A menudo necesitas saber el número de versión de la aplicación que creó un documento de Microsoft Excel. Aspose.Cells proporciona la propiedad [**Workbook.BuiltInDocumentProperties.Version**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/version) para este propósito.

{{% /alert %}}

El siguiente código de ejemplo demuestra el uso de la propiedad [**Workbook.BuiltInDocumentProperties.Version**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/version). Carga archivos de Excel creados con Microsoft Excel 2003, 2007, 2010 y 2013 e imprime el número de versión de la aplicación que creó estos documentos de Excel..
