 A menudo, necesita saber el número de versión de la aplicación que creó un documento de Excel Microsoft. Aspose.Cells proporciona el[**Workbook.BuiltInDocumentProperties.Version**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/version) propiedad para este fin.

{{% /alert %}}

 El siguiente código de ejemplo demuestra el uso de la[**Workbook.BuiltInDocumentProperties.Version**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/version)propiedad. Carga archivos de Excel creados con Microsoft Excel 2003, 2007, 2010 y 2013 e imprime el número de versión de la aplicación que creó estos documentos de Excel.
