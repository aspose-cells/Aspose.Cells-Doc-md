{{% alert color="primary" %}}

Un gráfico de cascada es un tipo especial de gráfico que normalmente se utiliza para demostrar cómo aumenta o disminuye la posición inicial. Microsoft Excel tiene muchos tipos de gráficos predefinidos, incluidos los de columnas, líneas, circulares, de barras, radiales, etc., pero el gráfico de cascada va más allá de los gráficos básicos y se puede crear utilizando los tipos de gráficos existentes con poca o más personalización.

{{% /alert %}} 

 Aspose.Cells Las API permiten crear un gráfico de cascada con la ayuda de un gráfico de líneas. El API también permite personalizar la apariencia del gráfico para darle la forma de la cascada configurando el[**Serie.UpBars**](https://reference.aspose.com/cells/net/aspose.cells.charts/series/properties/upbars) & [**Series.DownBars**](https://reference.aspose.com/cells/net/aspose.cells.charts/series/properties/downbars)propiedades.
