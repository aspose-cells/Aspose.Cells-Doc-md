{{% alert color="primary" %}}

Aspose.Cells le permite establecer un degradado, textura, patrón o imagen como efectos de relleno para diferentes objetos, como el área de trazado, el área del gráfico o el cuadro de leyenda de un gráfico. Este documento muestra cómo agregar una imagen al fondo de un gráfico.

{{% /alert %}}

Para lograr esto, Aspose.Cells proporciona la propiedad [**Chart.PlotArea.Area.FillFormat.ImageData**](https://reference.aspose.com/cells/net/aspose.cells.drawing/fillformat/properties/imagedata). El siguiente código de ejemplo demuestra el uso de la propiedad [**Chart.PlotArea.Area.FillFormat.ImageData**](https://reference.aspose.com/cells/net/aspose.cells.drawing/fillformat/properties/imagedata) para establecer una imagen como relleno de fondo en el gráfico.
