{{% alert color="primary" %}}

Aspose.Cells le permite establecer un degradado, textura, patrón o imagen como efectos de relleno para diferentes objetos, como el área de trazado, el área del gráfico o el cuadro de leyenda de un gráfico. Este documento muestra cómo agregar una imagen al fondo de un gráfico.

{{% /alert %}}

Para lograrlo, Aspose.Cells proporciona la[**Chart.PlotArea.Area.FillFormat.ImageData**](https://reference.aspose.com/cells/net/aspose.cells.drawing/fillformat/properties/imagedata) propiedad. El siguiente ejemplo de código demuestra el uso de[**Chart.PlotArea.Area.FillFormat.ImageData**](https://reference.aspose.com/cells/net/aspose.cells.drawing/fillformat/properties/imagedata) Propiedad para establecer una imagen como fondo para completar el gráfico.
