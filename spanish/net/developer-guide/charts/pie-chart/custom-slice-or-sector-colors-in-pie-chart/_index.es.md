{{% alert color="primary" %}}

Este artículo explica cómo agregar colores personalizados a las porciones/sectores de gráficos circulares. De forma predeterminada, los gráficos circulares utilizan la plantilla predeterminada de Excel Microsoft. Para usar otros colores, redefina los colores en el gráfico.

{{% /alert %}}

Para establecer un color personalizado para las porciones o sectores individuales de un gráfico circular:

1.  Acceder al[**Serie**](https://reference.aspose.com/cells/net/aspose.cells.charts/series) objetos[**ChartPoint**](https://reference.aspose.com/cells/net/aspose.cells.charts/chartpoint).
1.  Asigne el color de su elección usando el[**ChartPoint.Area.ForegroundColor**](https://reference.aspose.com/cells/net/aspose.cells.drawing/area/properties/foregroundcolor)propiedad.

Este artículo también explica cómo:

- Los datos de categoría de un gráfico.
- Un título de gráfico vinculado a una celda.
- La configuración de la fuente del título del gráfico.
- La posición de la leyenda.

{{% alert color="primary" %}}

[**ChartPoint.Area.ForegroundColor**](https://reference.aspose.com/cells/net/aspose.cells.drawing/area/properties/foregroundcolor) no es específico de los gráficos circulares, pero se puede utilizar para todo tipo de gráficos.
