{{% alert color="primary" %}}

Este artículo explica cómo agregar colores personalizados a sectores/porciones de gráficos circulares. De forma predeterminada, los gráficos circulares utilizan la plantilla predeterminada de Excel Microsoft. Para utilizar otros colores, redefina los colores en el gráfico.

{{% /alert %}}

Para establecer un color personalizado para los sectores o sectores individuales de un gráfico circular:

1.  Acceder al[**Serie**](https://reference.aspose.com/cells/net/aspose.cells.charts/series) objetos[**Punto de gráfico**](https://reference.aspose.com/cells/net/aspose.cells.charts/chartpoint).
1. Asigne el color de su elección usando el[**ChartPoint.Área.ForegroundColor**](https://reference.aspose.com/cells/net/aspose.cells.drawing/area/properties/foregroundcolor)propiedad.

Este artículo también explica cómo:

- Datos de categoría de un gráfico.
- Un título de gráfico vinculado a una celda.
- La configuración de fuente del título del gráfico.
- La posición de la leyenda.

{{% alert color="primary" %}}

[**ChartPoint.Área.ForegroundColor**](https://reference.aspose.com/cells/net/aspose.cells.drawing/area/properties/foregroundcolor) No es específico de gráficos circulares, pero se puede utilizar para todo tipo de gráficos.
