## **Posibles escenarios de uso**
Puede leer las etiquetas de los ejes de su gráfico después de calcular sus valores usando el[Gráfico.Calcular()](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/methods/calculate)método. Por favor use el[Eje.AxisLabels](https://reference.aspose.com/cells/net/aspose.cells.charts/axis/properties/axislabels)propiedad para este propósito que devolverá la lista de etiquetas de eje.
## **Lea las etiquetas de los ejes después de calcular el gráfico**
Consulte el siguiente código de ejemplo que carga el[ejemplo de archivo de Excel](64716803.xlsx)lee las etiquetas del eje de categorías del gráfico en la primera hoja de trabajo. Luego imprime los valores de las etiquetas de los ejes en la consola. Consulte la salida de la consola del código de muestra que se proporciona a continuación para obtener una referencia.
