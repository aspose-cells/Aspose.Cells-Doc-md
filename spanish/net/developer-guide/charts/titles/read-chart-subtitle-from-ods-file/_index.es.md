## **Leer subtítulo del gráfico desde un archivo ODS**

Aspose.Cells le brinda la capacidad de leer subtítulos de gráficos en archivos ODS mediante el uso de la propiedad [**Chart.SubTitle**](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/properties/subtitle). El siguiente código de ejemplo carga el [archivo ODS de muestra](89620481.ods) y lee el subtítulo del gráfico usando la propiedad [**Chart.SubTitle**](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/properties/subtitle) e imprime en la ventana de la consola. Consulte la salida de la consola del código a continuación para obtener referencia.
