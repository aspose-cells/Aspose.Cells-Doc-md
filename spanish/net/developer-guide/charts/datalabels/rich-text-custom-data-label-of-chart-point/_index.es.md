{{% alert color="primary" %}}

 Puede usar Aspose.Cells para crear una etiqueta de datos personalizada de texto enriquecido del punto del gráfico. Aspose.Cells proporciona el[**Etiquetas de datos.Personajes()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) método para devolver el[**Configuración de fuente**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) objeto que se puede usar para establecer las propiedades de la fuente del texto, como su color, negrita, etc.

{{% /alert %}}
