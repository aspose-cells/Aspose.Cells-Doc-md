{{% alert color="primary" %}}

 Puede utilizar Aspose.Cells para crear una etiqueta de datos personalizada de texto enriquecido del punto del gráfico. Aspose.Cells proporciona la[**Etiquetas de datos.Caracteres()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) método para devolver el[**Configuración de fuente**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) Objeto que se puede utilizar para establecer las propiedades de fuente del texto como su color, negrita, etc.
