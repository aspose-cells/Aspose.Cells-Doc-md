{{% alert color="primary" %}}

Puedes usar Aspose.Cells para crear una etiqueta de datos personalizada con texto enriquecido del punto del gráfico. Aspose.Cells proporciona el método [**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) para devolver el objeto [**FontSetting**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) que puede usarse para establecer las propiedades de la fuente del texto como su color, negrita, etc.

{{% /alert %}}
