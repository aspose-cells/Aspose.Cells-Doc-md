{{% alert color="primary" %}}

Microsoft Excel 2013 permite a los usuarios envolver o desenvolver texto dentro de las etiquetas de datos del gráfico. De forma predeterminada, el texto dentro de las etiquetas de datos del gráfico está en el estado ajustado.

 Aspose.Cells proporciona un[**Etiquetas de datos.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) propiedad que puede establecer en Verdadero o Falso para habilitar o deshabilitar el ajuste de texto de las etiquetas de datos, respectivamente.
