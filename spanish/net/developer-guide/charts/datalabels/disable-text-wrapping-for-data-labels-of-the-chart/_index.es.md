{{% alert color="primary" %}}

Microsoft Excel 2013 permite a los usuarios ajustar o desajustar el texto dentro de las etiquetas de datos del gráfico. De forma predeterminada, el texto dentro de las etiquetas de datos del gráfico está en estado desajustado.

Aspose.Cells proporciona una propiedad [**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) que se puede establecer en Verdadero o Falso para habilitar o deshabilitar el ajuste de texto de las etiquetas de datos, respectivamente.
