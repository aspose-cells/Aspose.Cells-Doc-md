## **Escenario de uso**

 Puede haber casos en los que los archivos de origen de datos se muevan y el archivo de Excel no pueda localizar el archivo. En tales casos, Aspose.Cells API ofrece la opción de actualizar el elemento Fórmula de Power Query mediante el[*PowerQueryFormulaItem*](https://reference.aspose.com/cells/net/aspose.cells.querytables/powerqueryformulaitem) class para actualizar la ubicación del archivo fuente.
