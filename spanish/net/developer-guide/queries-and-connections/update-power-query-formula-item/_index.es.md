## **Escenario de uso**

Puede haber casos en los que los archivos de origen de datos se muevan y el archivo de Excel no pueda ubicar el archivo. En tales casos, la API de Aspose.Cells proporciona la opción para actualizar el elemento de fórmula de Power Query mediante el uso de la clase [*PowerQueryFormulaItem*](https://reference.aspose.com/cells/net/aspose.cells.querytables/powerqueryformulaitem) para actualizar la ubicación del archivo de origen.
