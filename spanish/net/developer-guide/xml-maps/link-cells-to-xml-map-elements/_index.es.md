## **Posibles escenarios de uso**

Puede vincular sus celdas a elementos del mapa XML utilizando Aspose.Cells. Utilice el[**Cells.LinkToXmlMap()**](https://reference.aspose.com/cells/net/aspose.cells/cells/methods/linktoxmlmap)método para este propósito.

## **Enlace Cells a elementos del mapa Xml**

 El siguiente código de ejemplo carga el[archivo fuente excel](5115471.xlsx) que contiene el mapa XML y luego vincula las celdas A1, B2, C3, D4, E5 y F6 a los elementos del mapa XML FIELD1, FIELD2, FIELD4, FIELD5, FIELD7 y FIELD8 respectivamente y luego guarda el libro en[archivo de salida de Excel](5115467.xlsx).

 Si abres el[archivo de salida de Excel](5115467.xlsx) y haga clic en el botón Desarrollador > Fuente, verá que las celdas están vinculadas con elementos del mapa XML y también estarán resaltadas por Microsoft Excel como se muestra en esta imagen.
