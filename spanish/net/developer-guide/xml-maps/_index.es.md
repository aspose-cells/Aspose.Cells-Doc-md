{{% alert color="primary" %}}

 Aspose.Cells le permite importar el mapa XML dentro del libro de trabajo usando el[**Libro de trabajo.ImportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/importxml/index) método. Puede importar el mapa XML usando el Excel Microsoft con los siguientes pasos

-  Seleccione**Desarrollador** pestaña
-  Hacer clic**Importar** en la sección XML y siga los pasos requeridos.

 Deberá proporcionar sus datos XML para completar la importación. Aquí hay un[muestra de datos XML](5115037.txt) que puede utilizar para la prueba.
