{{% alert color="primary" %}}

Aspose.Cells te permite importar el mapa XML dentro del libro usando el método [**Workbook.ImportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/importxml/index). Puedes importar el Mapa XML usando Microsoft Excel con los siguientes pasos

- Selecciona la pestaña **Desarrollador**
- Haz clic en **Importar** en la sección de XML y sigue los pasos requeridos.

Deberás proporcionar tus datos XML para completar la importación. Aquí tienes un [ejemplo de datos XML](5115037.txt) que puedes usar para pruebas.
