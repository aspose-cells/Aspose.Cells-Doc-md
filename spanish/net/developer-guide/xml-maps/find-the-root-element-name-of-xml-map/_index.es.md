## **Escenarios de uso posibles**

Puede encontrar el *Nombre del elemento raíz del mapa XML* utilizando Aspose.Cells con la propiedad [**XmlMap.RootElementName**](https://reference.aspose.com/cells/net/aspose.cells/xmlmap/properties/rootelementname). La siguiente captura de pantalla muestra el nombre del elemento raíz del mapa XML en Microsoft Excel.
