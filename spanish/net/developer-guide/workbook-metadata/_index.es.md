Aspose.Cells le permite cargar una versión liviana de un libro en la memoria para editar su información de metadatos. Utilice la clase [**WorkbookMetadata**](https://reference.aspose.com/cells/net/aspose.cells.metadata/workbookmetadata) para cargar el libro.

{{% /alert %}}
