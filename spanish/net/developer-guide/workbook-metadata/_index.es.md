{{% alert color="primary" %}}

 Aspose.Cells le permite cargar una versión ligera del libro de trabajo en la memoria para editar su información de metadatos. Por favor use el[**Libro de trabajoMetadatos**](https://reference.aspose.com/cells/net/aspose.cells.metadata/workbookmetadata)clase para cargar el libro de trabajo.

{{% /alert %}}

Una vez que abra el libro de trabajo usando[**Libro de trabajo**](https://reference.aspose.com/cells/net/aspose.cells/workbook) clase, podrá leer las propiedades del documento.
