## **Actualizando a Aspose.Cells**
veces, los usuarios anteriores (p. ej., los usuarios de Aspose.Excel) pueden contraer un problema o detectar que faltan algunas funciones en la versión; alternativamente, deben actualizar a la versión Aspose.Cells más nueva para satisfacer sus necesidades. Cabe señalar que Aspose.Excel ha cambiado de nombre para convertirse en Aspose.Cells ahora, por lo que ya no podemos continuar ni hacer modificaciones con las versiones anteriores del componente Aspose.Excel. El nuevo componente Aspose.Cells es más confiable, estable y próspero con una funcionalidad mejorada y avanzada. Nos enfocamos especialmente en optimizar su desempeño para los más diversos escenarios. Solo para enfatizar, no hay mucho trabajo involucrado en la actualización al componente Aspose.Cells. Sin embargo, cuando se actualiza a él.

{{% alert color="primary" %}} 

Algunas API (clases, estructura, enumeraciones, etc.) se agregan en función de algunas características avanzadas que ahora ofrecemos, las clases anteriores (con sus miembros) se conservan y optimizan, pero se cambia el nombre de algunas clases, algunos miembros adicionales (propiedades, métodos, etc. .) también se incluyen para las clases con muy pocos excluidos.

A continuación se muestra la lista de clases a las que se les ha cambiado el nombre:

1. **Sobresalir** clase es reemplazada por**Libro de trabajo** clase.

2. **Diseñador Excel** clase es reemplazada por**WorkbookDesigner** clase.

3. **Ayudante de Excel** clase es reemplazada por**CellsHelper** clase.

 Para obtener más información, consulte nuestras demostraciones destacadas:<https://github.com/aspose-cells/Aspose.Cells-for-.NET>
