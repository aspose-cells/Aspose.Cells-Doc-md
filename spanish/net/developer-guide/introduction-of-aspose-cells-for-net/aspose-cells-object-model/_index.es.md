 El modelo de objetos Aspose.Cells proporciona información sobre las relaciones estructurales entre los objetos de la biblioteca de clases Aspose.Cells.

{{% /alert %}} 

La estructura de nivel superior del modelo de objeto Aspose.Cells se muestra a continuación de forma jerárquica.

|**Estructura de nivel superior del modelo de objeto Aspose.Cells**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_1.png)|
Como puede ver en la figura anterior, la raíz del modelo de objetos es el objeto Workbook. A continuación se proporciona una breve descripción de algunos de los objetos con fines introductorios.
## **WorksheetCollection/Worksheet**
El objeto Workbook contiene WorksheetCollection, que representa la colección de todos los objetos Worksheet en una hoja de cálculo, como se muestra a continuación:

|**Hojas de trabajo y objetos de hoja de trabajo**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_2.png)|
## **Cells/Cell**
Cada objeto Hoja de trabajo contiene un objeto Cells que representa la colección de todos los objetos Cell en una hoja de trabajo como se muestra a continuación:

|**Cells y Cell objetos**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_3.png)|
Puede usar el objeto Cell para obtener y establecer el valor, el estilo, la fórmula y otras propiedades de una sola celda.
## **ChartCollection/Gráfico**
El objeto Charts representa una colección de todos los objetos Chart en una hoja de trabajo. Cada objeto Gráfico se compone de varios otros objetos que funcionan juntos para crear y administrar gráficos. La estructura del gráfico en Aspose.Cells se muestra en el siguiente diagrama:

|**Modelo de objetos del gráfico**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_4.png)|
## **Colección de comentarios/Comentario**
Cada objeto de hoja de trabajo también contiene un objeto de comentarios que representa la colección de todos los objetos de comentario en una hoja de trabajo, como se muestra a continuación:

|**Comentarios y objetos de comentario**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_5.png)|
Un objeto Comentario se usa para agregar un comentario a cualquier celda especificada en la hoja de trabajo.
## **HorizontalPageBreakCollection/HorizontalPageBreak**
Cada objeto Worksheet contiene una HorizontalPageBreakCollection que representa una colección de todos los objetos HorizontalPageBreak en una hoja de trabajo, como se muestra a continuación:

|**Objetos HPageBreaks y HPageBreak**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_6.png)|
Se utiliza un objeto HorizontalPageBreak para crear un salto de página horizontal en la hoja de cálculo.
## **Colección de hipervínculos/Hipervínculo**
Un objeto de hoja de trabajo también contiene una HyperlinkCollection que representa una colección de todos los objetos de hipervínculo en la hoja de trabajo, como se muestra a continuación:

|**Hipervínculos y objetos de hipervínculo**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_7.png)|
Un objeto Hipervínculo representa un hipervínculo en la hoja de cálculo. Los desarrolladores pueden establecer la dirección del hipervínculo y otras propiedades relacionadas mediante el objeto Hipervínculo.
## **Colección de imágenes/Imagen**
Cada objeto Worksheet contiene un objeto PictureCollection que representa una colección de todos los objetos Picture en una hoja de trabajo, como se muestra a continuación:

|**Imágenes y objetos de imagen**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_8.png)|
Un objeto Imagen representa una imagen en la hoja de cálculo. Al usar el objeto Imagen, los desarrolladores no solo pueden agregar imágenes a sus hojas de trabajo, sino que también pueden colocar estas imágenes en cualquier lugar. También es posible establecer bordes u otras propiedades de las imágenes.
## **VerticalPageBreakCollection/VerticalPageBreak**
Cada objeto Worksheet contiene un objeto VerticalPageBreakCollection que representa una colección de todos los objetos VerticalPageBreak en una hoja de trabajo, como se muestra a continuación:

|**Objetos VPageBreaks y VPageBreak**|
|:- |
|![todo:imagen_alternativa_texto](aspose-cells-object-model_9.png)|
Un objeto VerticalPageBreak se usa para crear un salto de página vertical en la hoja de trabajo.
