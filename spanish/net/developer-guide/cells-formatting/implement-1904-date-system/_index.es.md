{{% alert color="primary" %}} 

Microsoft Excel admite dos sistemas de fechas: el sistema de fechas 1900 (el sistema de fechas predeterminado implementado en Excel para Windows) y el sistema de fechas 1904. El sistema de fechas 1904 se utiliza normalmente para proporcionar compatibilidad con archivos Excel de Macintosh y es el sistema predeterminado si utiliza Excel para Macintosh. Puede configurar el sistema de fecha 1904 para archivos de Excel usando Aspose.Cells.

{{% /alert %}} 

Para implementar el sistema de fechas 1904 en Microsoft Excel (por ejemplo Microsoft Excel 2003):

1.  Desde el**Herramientas** menú, seleccione**Opciones** y seleccione **Cálculo** pestaña.
1.  Selecciona el**sistema de fecha de 1904** opción.
1. Haga clic en Aceptar**.
