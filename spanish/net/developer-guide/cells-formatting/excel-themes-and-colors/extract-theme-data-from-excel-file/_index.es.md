{{% alert color="primary" %}}

Aspose.Cells permite a los usuarios extraer datos relacionados con el tema desde un archivo Excel. Por ejemplo, puede extraer el nombre del tema aplicado al libro de trabajo y el color del tema aplicado a la celda o los bordes de la celda, etc.

Puede aplicar el tema a su libro usando Microsoft Excel mediante el comando Diseño de página > Temas.

{{% /alert %}}
