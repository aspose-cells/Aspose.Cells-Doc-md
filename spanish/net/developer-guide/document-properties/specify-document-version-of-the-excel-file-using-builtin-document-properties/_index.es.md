## **Posibles escenarios de uso**

 Puedes cambiar el**Número de versión** del archivo de Excel haciendo clic derecho en el archivo y luego seleccionando Propiedades > Detalles y luego editando el**Número de versión** campo. Por favor use[**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion)property para cambiarla mediante programación usando las API Aspose.Cells.
