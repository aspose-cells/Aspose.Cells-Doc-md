 Aspose.Cells proporciona la función para reemplazar texto en un libro de trabajo usando una expresión regular. Para esto, el API proporciona[**Clave Regex**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey)propiedad de la[**Opciones de reemplazo**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions) clase. Configuración de la[**Clave Regex**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) a**verdadero** indica que la clave buscada será una expresión regular.
