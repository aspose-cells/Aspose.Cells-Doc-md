Aspose.Cells proporciona la función de reemplazar texto en un libro de trabajo usando una expresión regular. Para esto, la API proporciona la propiedad [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) de la clase [**ReplaceOptions**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions). Establecer el [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) en **verdadero** indica que la clave buscada será una expresión regular.

El siguiente fragmento de código demuestra el uso de la propiedad [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) mediante el uso del [archivo de Excel de muestra](101089318.xlsx).
