[Copiar solo datos de rango](/cells/es/net/copy-range-data-only/) y[Copiar datos de rango con estilo](/cells/es/net/copy-range-data-with-style/) explicó cómo copiar datos de un rango a otro solo o completo con formato. También es posible copiar solo el formato. Este artículo muestra cómo.

{{% /alert %}} 

Este ejemplo crea un libro de trabajo, lo llena con datos y copia solo el estilo de un rango.

1. Crea un rango.
1.  Crear un[**Estilo**](https://reference.aspose.com/cells/net/aspose.cells/style) objeto con atributos de formato especificados.
1. Aplique el formato de estilo al rango.
1. Cree un segundo rango de celdas.
1. Copie el formato del primer rango al segundo rango.
