##  **Posibles escenarios de uso**

Puede especificar campos de fórmula cuando importa datos a su hoja de trabajo usando el[**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas) . Esta propiedad toma la matriz booleana donde está el valor.**verdadero**significa que el campo es un campo de fórmula. Por ejemplo, si el tercer campo es un campo de fórmula, entonces el tercer valor de la matriz será *verdadero**.
