##  **Posibles escenarios de uso**

Aspose.Cells le permite desplazar la primera fila hacia abajo al insertar una tabla en la hoja de trabajo. Este documento explica cómo puede realizar la tarea utilizando las API Aspose.Cells.

##  **Desplace la primera fila hacia abajo al insertar Cells filas de la tabla de datos**

El siguiente código de muestra ilustra cómo desplazar la primera fila hacia abajo al insertar una tabla en la hoja de trabajo. Usamos una plantilla simple de archivo Excel en el código para demostrar la función. Puede ejercitar la función configurando el valor booleano[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown) atribuir a**Verdadero Falso** para entenderlo mejor.
