## **Posibles escenarios de uso**

Aspose.Cells le permite desplazar la primera fila hacia abajo al insertar una tabla en la hoja de trabajo. Este documento explica cómo puede realizar la tarea utilizando las API Aspose.Cells.

## **Desplace la primera fila hacia abajo al insertar Cells Filas de la tabla de datos**

El código de ejemplo siguiente ilustra cómo desplazar la primera fila hacia abajo al insertar una tabla en la hoja de cálculo. Usamos un archivo de Excel de plantilla simple en el código para demostrar la función. Puede ejercer la función configurando el valor booleano[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown) atribuir a**Verdadero**/**Falso** para entenderlo mejor. Por favor vea el[ejemplo de archivo de Excel](45056031.xlsx), [Salida de archivo falso de Excel](45056032.xlsx) , y[archivo de salida Excel True](45056033.xlsx)para tu referencia.
