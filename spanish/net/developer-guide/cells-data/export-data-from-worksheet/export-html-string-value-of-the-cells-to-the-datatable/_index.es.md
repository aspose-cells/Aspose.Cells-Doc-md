## **Posibles escenarios de uso**
Usando las API Aspose.Cells, los desarrolladores pueden llenar una tabla de datos con las cadenas HTML correspondientes (basadas en los valores de las celdas) mientras exportan los datos de la hoja de trabajo. Este documento explica cómo exportar cadenas HTML para los datos de celdas formateadas para llenar una tabla de datos usando las API Aspose.Cells.
