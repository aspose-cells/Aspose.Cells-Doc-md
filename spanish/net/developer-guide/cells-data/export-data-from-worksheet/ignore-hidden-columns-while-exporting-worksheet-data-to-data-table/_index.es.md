{{% alert color="primary" %}}

 A veces, desea ignorar las columnas ocultas al exportar datos de la hoja de trabajo a una tabla de datos. Puede lograrlo usando Aspose.Cells configurando el[**ExportTableOptions.PlotVisibleColumns**](https://reference.aspose.com/cells/net/aspose.cells/exporttableoptions/properties/plotvisiblecolumns) a**verdadero**. De forma predeterminada, su valor es **falso**, por lo que debes establecerlo como **verdadero** para ignorar las columnas ocultas.

{{% /alert %}}
