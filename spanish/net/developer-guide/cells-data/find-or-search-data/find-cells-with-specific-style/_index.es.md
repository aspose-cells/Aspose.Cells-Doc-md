{{% alert color="primary" %}}

 A veces, es necesario encontrar celdas con un estilo particular aplicado. Puede utilizar Aspose.Cells para buscar todas las celdas con un estilo común. Aspose.Cells proporciona la[**BuscarOpciones.Estilo**](https://reference.aspose.com/cells/net/aspose.cells/findoptions/properties/style) propiedad que puede utilizar para especificar el estilo para buscar celdas.

{{% /alert %}}

El código de este ejemplo busca todas las celdas que tienen el mismo estilo que el de la celda A1. Una vez ejecutado el código, todas las celdas que tienen el mismo estilo que A1 contienen el texto "Encontrado".
