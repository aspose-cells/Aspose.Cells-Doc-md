{{% alert color="primary" %}}

A veces, necesitas encontrar celdas con un estilo particular aplicado. Puedes usar Aspose.Cells para encontrar todas las celdas con un estilo común. Aspose.Cells proporciona la propiedad [**FindOptions.Style**](https://reference.aspose.com/cells/net/aspose.cells/findoptions/properties/style) que puedes usar para especificar el estilo por el que buscar celdas.

{{% /alert %}}

El código de este ejemplo encuentra todas las celdas que tienen el mismo estilo que la celda A1. Después de que se haya ejecutado el código, todas las celdas que tienen el mismo estilo que A1 contienen el texto "Encontrado".
