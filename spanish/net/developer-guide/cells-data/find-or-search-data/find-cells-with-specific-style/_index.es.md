{{% alert color="primary" %}}

 A veces, necesita encontrar celdas con un estilo particular aplicado. Puede usar Aspose.Cells para encontrar todas las celdas con un estilo común. Aspose.Cells proporciona el[**FindOptions.Style**](https://reference.aspose.com/cells/net/aspose.cells/findoptions/properties/style) propiedad que puede usar para especificar el estilo para buscar celdas.

{{% /alert %}}

El código de este ejemplo encuentra todas las celdas que tienen el mismo estilo que la celda A1. Después de ejecutar el código, todas las celdas que tienen el mismo estilo que A1 contienen el texto "Encontrado".
