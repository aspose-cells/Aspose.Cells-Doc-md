## **Obtenga la validación Cell en archivos ODS**

Con Aspose.Cells, puede aplicar la validación a una celda en archivos ODS. Para esto, el API proporciona el[**ObtenerValidación**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) metodo de la[**Cell**](https://reference.aspose.com/cells/net/aspose.cells/cell) clase.

El siguiente ejemplo de código demuestra el uso de la[**ObtenerValidación**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation) método cargando el[SAO fuente](101089354.ods) archivo y leyendo la validación de la celda A9.
