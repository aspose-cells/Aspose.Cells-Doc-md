##  **Obtener ancho de texto del valor Cell**

Aspose.Cells proporciona la[**CellsHelper.GetTextWidth**](https://reference.aspose.com/cells/net/aspose.cells/cellshelper/methods/gettextwidth) método que permite a los desarrolladores obtener el ancho del texto del valor de la celda.
