{{% alert color="primary" %}}

Aspose.Cells admite la conversión de Microsoft orientado a Excel HTML al formato XLS/XLSX. Es decir, el HTML generado por Microsoft Excel se puede volver a convertir al formato XLS/XLSX usando Aspose.Cells.

 De manera similar, si hay algún HTML simple, Aspose.Cells puede convertirlo en HTML texto enriquecido. Aspose.Cells proporciona[**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/properties/htmlstring) propiedad que puede tomar un HTML tan simple y convertirlo en texto de celda formateado.

{{% /alert %}}
