{{% alert color="primary" %}}

Aspose.Cells admite la conversión de HTML orientado a Microsoft Excel a formato XLS/XLSX. Esto significa que el HTML generado por Microsoft Excel se puede convertir de nuevo al formato XLS/XLSX utilizando Aspose.Cells.

Del mismo modo, si hay algo de HTML simple, Aspose.Cells puede convertirlo en texto enriquecido HTML. Aspose.Cells proporciona la propiedad [**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/properties/htmlstring) que puede tomar dicho HTML simple y convertirlo en texto de celda formateado.
