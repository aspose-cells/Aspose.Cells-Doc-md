##  **Posibles escenarios de uso**
Cuando necesite configurar datos con estilo para Cell específico, puede asignar una cadena html a Cell. Por supuesto, también puede obtener la cadena HTML de la celda. Aspose.Cells ofrece esta característica. Aspose.Cells proporciona las siguientes propiedades y métodos para ayudarlo a lograr sus objetivos.
- [**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/htmlstring)

##  **Obtenga y configure la cadena html usando Aspose.Cells**
Este ejemplo muestra cómo:

1. Cree un libro de trabajo y agregue algunos datos.
1. Obtenga el Cell específico en la primera hoja de trabajo.
1. Establezca la cadena html en Cell.
1. Obtenga la cadena html del Cell.
