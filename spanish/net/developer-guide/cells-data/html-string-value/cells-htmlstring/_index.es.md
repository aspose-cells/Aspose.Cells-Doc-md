##  **Posibles escenarios de uso**
Cuando necesite configurar datos con estilo para Cell específico, puede asignar una cadena html al Cell. Por supuesto, también puede obtener la cadena HTML de la celda. Aspose.Cells ofrece esta característica. Aspose.Cells proporciona las siguientes propiedades y métodos para ayudarle a alcanzar sus objetivos.
- [**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/htmlstring)
