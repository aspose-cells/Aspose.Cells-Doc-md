## **Escenarios de uso posibles**

Puedes especificar el nombre del autor al proteger tu libro de trabajo usando la API de Aspose.Cells. Por favor usa la propiedad [**Workbook.Settings.WriteProtection.Author**](https://reference.aspose.com/cells/net/aspose.cells/writeprotection/properties/author) para este propósito.

## **Especificar Autor al Proteger la Escritura del Libro de Trabajo**

El siguiente código de muestra explica el uso de la propiedad [**Workbook.Settings.WriteProtection.Author**](https://reference.aspose.com/cells/net/aspose.cells/writeprotection/properties/author). El código crea un libro de trabajo vacío, lo protege con una contraseña, especifica el nombre del autor y lo guarda como [archivo de Excel de salida](67338582.xlsx).
