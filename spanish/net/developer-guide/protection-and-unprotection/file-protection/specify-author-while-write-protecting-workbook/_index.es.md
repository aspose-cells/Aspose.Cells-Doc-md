## **Posibles escenarios de uso**

Puede especificar el nombre del autor mientras escribe protegiendo su libro de trabajo usando Aspose.Cells API. Utilice[**Libro de trabajo.Configuración.Protección contra escritura.Autor**](https://reference.aspose.com/cells/net/aspose.cells/writeprotection/properties/author)propiedad para este fin.
