 Aspose.Cells admite la firma de libros de trabajo con la firma XAdES. Para esto, el API proporciona[**Firma digital**](https://reference.aspose.com/cells/net/aspose.cells.digitalsignatures/digitalsignature) clase y[**Tipo XAdEST**](https://reference.aspose.com/cells/net/aspose.cells.digitalsignatures/xadestype) enumeración.

El siguiente fragmento de código demuestra el uso de la[**Firma digital**](https://reference.aspose.com/cells/net/aspose.cells.digitalsignatures/digitalsignature) clase para firmar el[fuente](101089323.xlsx) libro de trabajo.
