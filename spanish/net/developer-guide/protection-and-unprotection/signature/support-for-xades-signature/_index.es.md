## **Introducción**

Aspose.Cells proporciona soporte para firmar libros con Firma XAdES. Para esto, la API proporciona la clase [**DigitalSignature**](https://reference.aspose.com/cells/net/aspose.cells.digitalsignatures/digitalsignature) y la enumeración [**XAdESType**](https://reference.aspose.com/cells/net/aspose.cells.digitalsignatures/xadestype).
