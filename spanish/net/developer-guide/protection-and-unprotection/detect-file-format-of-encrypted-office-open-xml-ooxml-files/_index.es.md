**XML abierto de Office**(también conocido como**OOXML**o**Microsoft XML abierto**(MOX)) es un formato de archivo basado en XML desarrollado por Microsoft para representar documentos de oficina como hojas de cálculo, gráficos, presentaciones y documentos de procesamiento de texto.

{{% /alert %}} 



Aspose.Cells proporciona una forma de detectar el formato de archivo de cifrado**Microsoft XML abierto** archivos Para identificar el tipo de archivo, utilice el[FileFormatUtil.DetectFileFormat](https://reference.aspose.com/cells/net/aspose.cells/fileformatutil/methods/detectfileformat/index) como se muestra a continuación en el ejemplo de código.
