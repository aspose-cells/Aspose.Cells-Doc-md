{{% alert color="primary" %}} 

**Office Open XML** (también conocido como **OOXML** o **Microsoft Open XML** (MOX)) es un formato de archivo basado en XML desarrollado por Microsoft para representar documentos de oficina como hojas de cálculo, gráficos, presentaciones y documentos de procesamiento de palabras.

{{% /alert %}} 



Aspose.Cells proporciona una forma de detectar el formato de archivo de los archivos *Microsoft Open XML* encriptados. Para identificar el tipo de archivo, use el método [FileFormatUtil.DetectFileFormat](https://reference.aspose.com/cells/net/aspose.cells/fileformatutil/methods/detectfileformat/index) como se muestra a continuación en el ejemplo de código.
