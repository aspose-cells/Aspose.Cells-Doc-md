## **Posibles escenarios de uso**

Aspose.Cells le permite prefijar estilos de elementos de tabla con[**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/tablecssid)propiedad. Supongamos que establece esta propiedad con algún valor como**MiTest_TableCssId**, luego encontrará estilos de elementos de tabla como se muestra a continuación

{{< highlight "java" >}}

 table#MyTest_TableCssId

# MyTest_TableCssId tr

# MyTest_TableCssId col

# MyTest_TableCssId br

etc.

{{< /highlight >}}
