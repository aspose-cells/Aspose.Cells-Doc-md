{{% alert color="primary" %}} 

Aspose.Cells proporciona una propiedad HtmlSaveOptions.PresentationPreference útil para los desarrolladores que necesitan representar un mejor diseño al guardar un archivo de Excel Microsoft en formato HTML o MHT. El valor predeterminado de la propiedad es falso. Recomendamos establecer esta propiedad en verdadero para obtener una presentación más atractiva de los informes de Excel.

{{% /alert %}}
