{{% alert color="primary" %}} 

Normalmente, el diseño de las etiquetas div se ignora al cargar HTML en un objeto de libro de Excel. Sin embargo, si desea que el diseño de las etiquetas div no se ignore, configure el[HTMLLoadOptions.SupportDivTag](https://reference.aspose.com/cells/net/aspose.cells/htmlloadoptions/properties/supportdivtag) propiedad a**verdadero** . El valor predeterminado de esta propiedad es**falso**.

{{% /alert %}}
