{{% alert color="primary" %}} 

Aspose.Cells ahora admite la expansión de texto de derecha a izquierda al exportar un archivo de Excel a HTML. Esta función se ha implementado desde la v8.9.0.0. Ahora, si su archivo Excel de origen contiene texto que se expande de derecha a izquierda, entonces Aspose.Cells lo exportará a HTML correctamente.

{{% /alert %}}
