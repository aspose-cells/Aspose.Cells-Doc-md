Aspose.Cells admite la conversión de un archivo JSON (JavaScript Object Notation) a una hoja de cálculo de Excel.

{{% /alert %}}

## **Convertir JSON a hoja de cálculo de Excel**
No es necesario preguntarse cómo convertir JSON a un archivo de Excel, porque la biblioteca Aspose.Cells para .NET tiene la mejor solución. La API Aspose.Cells proporciona soporte para convertir el formato JSON a hojas de cálculo. Puede usar la clase [**JsonLoadOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonLoadOptions) para especificar configuraciones adicionales para importar JSON a la hoja de cálculo.
