{{% alert color="primary" %}}

Aspose.Cells admite guardar un archivo ODS (**Hoja de cálculo de OpenDocument**) en el ODF (**Formato de documento abierto** Especificaciones 1.1 y 1.2. Aspose.Cells tiene[**OdsSaveOptions.IsStrictSchema11**](https://reference.aspose.com/cells/net/aspose.cells/odssaveoptions/properties/isstrictschema11) propiedad que especifica el uso de la especificación ODF 1.1 para guardar archivos ODS. El valor predeterminado de esta propiedad es**falso**, por lo que el archivo ODS guardado sin esta configuración utiliza las especificaciones 1.2.

{{% /alert %}}

El código de muestra a continuación crea un objeto de libro de trabajo, agrega algún valor a la celda A1 en la primera hoja de trabajo y luego guarda el archivo ODS en las especificaciones ODF 1.1 y 1.2. De forma predeterminada, el archivo ODS se guarda en la especificación ODF 1.2.
