## **Posibles escenarios de uso**

Por lo general, usted especifica el directorio de fuentes o la lista de fuentes para todos los libros de trabajo, pero a veces tiene que especificar un conjunto de fuentes individual o privado para sus libros de trabajo. Aspose.Cells proporciona la clase ****[IndividualFontConfigs](https://reference.aspose.com/cells/net/aspose.cells/individualfontconfigs)** que se puede usar para especificar el conjunto de fuentes individual o privado para su libro de trabajo.
