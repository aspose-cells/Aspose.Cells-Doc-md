{{% alert color="primary" %}}

 Puede usar Aspose.Cells para exportar su hoja de trabajo o gráfico a una imagen con el ancho y el alto deseados. Proporciona[**ImageOrPrintOptions.SetDesiredSize()**](https://reference.aspose.com/cells/net/aspose.cells.rendering/imageorprintoptions/methods/setdesiredsize) para establecer el ancho y el alto deseados de la imagen exportada. El ancho y la altura se especifican en la unidad de píxeles.
