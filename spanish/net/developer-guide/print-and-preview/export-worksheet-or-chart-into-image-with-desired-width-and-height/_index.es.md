{{% alert color="primary" %}}

Puede utilizar Aspose.Cells para exportar su hoja de cálculo o gráfico a una imagen con el ancho y alto deseados. Proporciona el método [**ImageOrPrintOptions.SetDesiredSize()**](https://reference.aspose.com/cells/net/aspose.cells.rendering/imageorprintoptions/methods/setdesiredsize) para establecer el ancho y alto deseados de la imagen exportada. El ancho y alto se especifican en unidades de píxeles.
