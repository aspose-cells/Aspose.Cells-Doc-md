{{% alert color="primary" %}} 

A veces desea especificar el formato de píxel al representar una hoja de trabajo en formato de imagen. De forma predeterminada, Aspose.Cells utiliza 32 bits por píxel. Aspose.Cells le permite personalizar el formato de píxel (profundidad de bits) usando opciones para la imagen renderizada.

{{% /alert %}}
