{{% alert color="primary" %}}

Los comentarios se agregan a las celdas para registrar comentarios, desde detalles sobre cómo funciona una fórmula, de dónde proviene un valor o preguntas de revisores. Los comentarios juegan un papel extremadamente importante cuando varias personas discuten o revisan el mismo documento en diferentes momentos. ¿Cómo distinguir los comentarios de diferentes personas? Sí, podemos establecer un color de fondo diferente para cada comentario. Pero cuando necesitamos procesar muchos documentos y muchos comentarios, hacerlo manualmente es un desastre. Afortunadamente [**Aspose.Cells**](https://products.aspose.com/cells/net/) proporciona una API que le permite hacer esto en código.

{{% /alert %}}
