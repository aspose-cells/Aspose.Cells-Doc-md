## **Dibuje Slicer mientras renderiza Excel a PDF**
Si tiene un archivo de Excel al que se le ha aplicado una segmentación y desea exportar el Excel a PDF con la configuración de segmentación, Aspose.Cells ahora lo admite de forma predeterminada. Simplemente exporte el archivo de Excel con la segmentación a PDF, el PDF generado mostrará la segmentación aplicada.

 El siguiente código de ejemplo carga el[ejemplo de archivo de Excel](94044165.xlsx) que contiene una segmentación existente. A continuación, guarda el libro de trabajo como[archivo de salida PDF](94044166.pdf). La siguiente captura de pantalla compara el archivo Excel de origen y el archivo PDF generado.
