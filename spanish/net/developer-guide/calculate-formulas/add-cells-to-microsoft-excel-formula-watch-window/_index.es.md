## **Escenarios de uso posibles**

La Ventana de seguimiento de fórmulas de Microsoft Excel es una herramienta útil para observar los valores de las celdas y sus fórmulas de manera conveniente en una ventana. Puedes abrir la *Ventana de seguimiento* utilizando Microsoft Excel haciendo clic en *Fórmulas > Ventana de seguimiento*. Tiene el botón *Agregar Seguimiento* que se utiliza para agregar las celdas para inspección. Del mismo modo, puedes usar el método [**Worksheet.CellWatches.Add()**](https://reference.aspose.com/cells/net/aspose.cells/cellwatchcollection/methods/add/index) para agregar celdas a *Ventana de seguimiento* utilizando la API de Aspose.Cells.
