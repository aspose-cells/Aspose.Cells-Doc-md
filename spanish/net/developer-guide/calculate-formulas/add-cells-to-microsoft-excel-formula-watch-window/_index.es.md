##  **Posibles escenarios de uso**

Microsoft Excel Watch Window es una herramienta útil para observar los valores de celda y sus fórmulas cómodamente en una ventana. Puedes abrir el*Ventana de observación* utilizando Microsoft Excel haciendo clic en el*Fórmulas > Ver* *Ventana*. Tiene el *Agregar Reloj*Botón que se puede utilizar para agregar las celdas para inspección. Del mismo modo, puedes utilizar[**Hoja de trabajo.CellWatches.Add()**](https://reference.aspose.com/cells/net/aspose.cells/cellwatchcollection/methods/add/index) método para agregar celdas en*Ventana de observación*usando Aspose.Cells API.
