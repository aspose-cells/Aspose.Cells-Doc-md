## **Posibles escenarios de uso**

Microsoft Excel Watch Window es una herramienta útil para ver los valores de las celdas y sus fórmulas convenientemente en una ventana. Puedes abrir el*Ventana de vigilancia* usando Microsoft Excel haciendo clic en el*Fórmulas > Ver* *Ventana*. tiene el*Agregar reloj*botón que se puede usar para agregar las celdas para su inspección. Del mismo modo, puede utilizar[**Hoja de trabajo.CellWatches.Add()**](https://reference.aspose.com/cells/net/aspose.cells/cellwatchcollection/methods/add/index) método para agregar celdas en*Ventana de vigilancia*usando Aspose.Cells API.
