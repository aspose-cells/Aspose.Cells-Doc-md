{{% alert color="primary" %}} 

FormulaText es una función de Excel 2013 y versiones posteriores. No es compatible con versiones anteriores como Excel 2010 o 2007, etc. Como sugiere su nombre, imprime el texto de la fórmula que está presente en una celda determinada. Este artículo le mostrará cómo hacer uso de esta función usando Aspose.Cells.

{{% /alert %}}
