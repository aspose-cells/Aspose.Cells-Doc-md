{{% alert color="primary" %}}

 Puede crear una tabla de datos en Microsoft Excel usando Datos > Análisis hipotético > Tabla de datos.... Aspose.Cells ahora le permite calcular la fórmula de matriz de una tabla de datos. Por favor use[**Libro de trabajo. Calcular fórmula ()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/calculateformula) normal para el cálculo de cualquier tipo de fórmulas.

{{% /alert %}}

Si cambia el valor de la celda B1 a 100, los valores de la tabla de datos que se rellenan con el color amarillo se convertirán en 120, como se muestra en las siguientes imágenes.
