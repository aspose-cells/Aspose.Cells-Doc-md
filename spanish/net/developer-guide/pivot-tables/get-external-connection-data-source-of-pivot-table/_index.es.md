## **Obtener fuente de datos de conexión externa de tabla dinámica**

Aspose.Cells proporciona la capacidad de obtener la fuente de datos de conexión externa de la tabla dinámica. Para esto, el API proporciona el[**Fuente de datos de conexión externa**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)propiedad de la[**Tabla dinámica**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) clase. Él[**Fuente de datos de conexión externa**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)propiedad devuelve el[**Conexiónexterna**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) objeto. El siguiente fragmento de código demuestra el uso de la[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) propiedad para obtener la fuente de datos de conexión externa de la tabla dinámica.
