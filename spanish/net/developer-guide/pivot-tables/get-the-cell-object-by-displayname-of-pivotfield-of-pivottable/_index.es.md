{{% alert color="primary" %}}

Aspose.Cells proporciona el método [**PivotTable.GetCellByDisplayName()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/getcellbydisplayname) que puedes utilizar para acceder al objeto de celda por el nombre de visualización del campo de tabla dinámica. Este método es útil cuando deseas resaltar o dar formato al encabezado de tu campo de tabla dinámica. Este artículo explica cómo recuperar el objeto de celda por el nombre de visualización del campo de datos y luego aplicar formato a este.

{{% /alert %}}

Luego cambia el color de relleno y el color de fuente de la celda a azul claro y negro respectivamente.
