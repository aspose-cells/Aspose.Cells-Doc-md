 Esta página contiene notas de la versión para[Aspose.Cells .Net 3.9.0.0 Nueva versión](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-.net-3.9.0.0-new-release/)

 Queridos clientes,

 ¡Hemos lanzado Aspose.Cells for .NET v3.9!

- Por algunas razones externas, ahora la clase Excel se renombra como Workbook y la clase ExcelDesigner se renombra como WorkbookDesigner. Todas las propiedades y el método de estas dos clases no se modifican.
- Admite agregar EMF imágenes en su formato sin formato
- Se agrega un nuevo marcador inteligente
- Admite cargar archivos de texto con separador personalizado
- Se corrigió un error en la configuración de hipervínculos a archivos locales
- Errores corregidos en el método InsertRows
- Se corrigió un error en la configuración de validación de datos
- Se corrigió un error al abrir algunos archivos de Excel
