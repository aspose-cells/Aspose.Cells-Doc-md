**Uso de Aspose.Cells en el componente Report Canvas**

Robert Chilvers, 17 de marzo de 2008

{{% /alert %}}

## **Antecedentes del producto**

El componente Report Canvas permite al usuario crear informes visuales basados en un conjunto de datos precargado. El usuario puede agregar diferentes componentes a su lienzo, incluidas imágenes, cuadros de texto, gráficos y tablas, luego especifica los datos y cómo se deben agregar. Luego, el usuario puede reorganizar y cambiar el tamaño de los objetos para que se ajusten a su página. El usuario puede especificar paletas de colores y guardar plantillas para uso futuro. Aspose.Cells se utiliza para exportar todos los objetos del lienzo a Excel con los datos correctos. El componente está escrito con VB.Net en Visual Studio 2008.

## **Escenario de requisitos**

Seleccionamos Aspose.Cells debido a sus casi completas capacidades de exportación de Excel Microsoft. Lo más importante para nosotros es la capacidad de exportar gráficos y tablas, especialmente en Microsoft Excel 2007, que faltaban en otros componentes de terceros.

## **Implementación de la solución**

Cada objeto en el lienzo del informe tiene una función que pasa una instancia de la hoja de trabajo Aspose.Cells y se agrega a la hoja de trabajo. Cuando el usuario solicita una exportación, el libro de trabajo y las hojas de trabajo se inicializan y cada objeto en el lienzo del informe tiene esta función llamada.

## **Beneficios**

Aspose.Cells nos permitió crear el libro de Excel de forma totalmente independiente de Excel y luego guardar el libro de trabajo en el formato seleccionado por el usuario. Esto ahorró horas de depuración de la interacción al usar la interoperabilidad de Excel e implementar diferentes rutinas para guardar en diferentes versiones de Excel.

## **Implementación futura**

Es probable que usemos Aspose.Cells para cargar y guardar archivos de Excel. Esto incluirá la carga de plantillas de datos y la exportación de resultados.

## **Conclusión**

Hasta el momento, no hemos tenido problemas al usar los componentes Aspose.Cells y el componente debería ahorrarnos tiempo de desarrollo tanto a corto como a largo plazo. Las consultas de soporte y ventas han sido respondidas de manera rápida y útil.
