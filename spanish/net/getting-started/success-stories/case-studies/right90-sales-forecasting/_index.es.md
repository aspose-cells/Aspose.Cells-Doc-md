## **Escenario de requisitos**
Muchos equipos de ventas en empresas de fabricación se sienten muy cómodos usando Excel para capturar pronósticos y han definido procesos usando Excel como su herramienta de recopilación de datos de pronóstico. Si bien Right90 brinda una interfaz web simple pero poderosa para actualizar, implementar y administrar cambios en los pronósticos, también queríamos proporcionar una forma para que nuestros clientes de fabricación extraigan fácilmente su información de pronóstico en Excel. Además, también necesitábamos poder realizar un seguimiento de sus cambios en ese libro de trabajo y permitirles enviar fácilmente esos cambios nuevamente a nuestro sistema para su procesamiento, resumen y análisis agregados. Como solución de previsión empresarial bajo demanda, las principales prioridades de Right90 para la integración de Excel eran el rendimiento y la fiabilidad.
## **Implementación de la solución**
Comparamos varios productos de generación de archivos de Excel y elegimos implementar nuestra solución usando Aspose.Cells. La decisión de usar Aspose.Cells fue fácil gracias a su colección concisa de ejemplos de código, documentación y su atención al cliente receptiva. Ejecutamos Aspose.Cells detrás de un conjunto de servicios web en una aplicación web de alta disponibilidad y alta transacción.
## **Beneficios**
