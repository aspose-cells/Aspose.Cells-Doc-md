{{% alert color="primary" %}} 

 Cualquier documento cargado en Aspose.Cells se puede convertir a PDF que se ajuste a la[PDF 1.5](https://docs.fileformat.com/pdf/) o[PDF/A-1b](https://docs.fileformat.com/pdf/a/)especificación. Aspose.Cells hace todo lo posible para convertir fielmente los documentos de Excel Microsoft a PDF. La mayoría de las veces, el aspecto del documento final PDF será indistinguible del documento de Excel Microsoft original.

{{% /alert %}} 
## **PDF en Aspose.Cells**
La solución Aspose es completamente compatible con las siguientes características del documento cuando se convierte a PDF:

- Cell Formateo de datos y texto
- Gráficos y tablas dinámicas
- Encabezados/pies de página
- Formato condicional
- Bordes, colores de borde
- Imágenes, cuadros de texto y formas de dibujo
- Marcadores
- hipervínculos
- Combinado Cells, saltos de página, títulos impresos

 También puedes ver el[especificación detallada](https://docs.aspose.com/cells/net/convert-excel-workbook-to-pdf/) sobre cómo Aspose.Cells convierte un documento a PDF.
