{{% alert color="primary" %}} 

Aspose.Cells se puede utilizar para guardar sus libros de Excel en formato HTML. Los libros de trabajo HTML exportados están orientados a Microsoft Excel, es decir, son similares a los exportados por Microsoft Excel. También puede crear libros de trabajo desde HTML, pero este HTML debe estar orientado a Excel.

{{% /alert %}} 
## **HTML en Aspose.Cells**
La solución Aspose es totalmente compatible con las siguientes funciones de documentos al convertir a HTML:

- Cell Formateo de datos y texto
- Gráficos y tablas dinámicas
- Encabezados/pies de página
- Bordes, colores de borde
- Imágenes, cuadros de texto y formas de dibujo
- hipervínculos
