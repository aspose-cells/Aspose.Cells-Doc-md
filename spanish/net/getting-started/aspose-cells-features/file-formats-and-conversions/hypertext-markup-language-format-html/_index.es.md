{{% alert color="primary" %}} 

Aspose.Cells se puede utilizar para guardar sus libros de Excel en formato HTML. Los libros de trabajo HTML exportados están orientados a Microsoft Excel, es decir, son similares a los exportados por Microsoft Excel. También puede crear libros de trabajo a partir de HTML, pero este HTML debe estar orientado a Excel.

{{% /alert %}} 
## **HTML en Aspose.Cells**
La solución de Aspose admite completamente las siguientes características del documento al convertir a HTML:

- Datos de celda y formato de texto
- Gráficos y tablas dinámicas
- Encabezados/Pies de página
- Bordes, colores de borde
- Imágenes, cuadros de texto y formas de dibujo
- Hipervínculos
