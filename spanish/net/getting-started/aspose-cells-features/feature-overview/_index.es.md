Aspose.Cells for .NET cubre la mayoría de los entornos de desarrollo populares y plataformas de implementación.

| :- | :- |
|[.NET Framework](/cells/es/net/system-requirements/#net-frameworks)|Use Aspose.Cells for .NET para construir ASP.NET, Servicios web, WinForms u cualquier otro tipo de aplicación para cualquier versión de .NET Framework 2.0 o posterior, incluidas las versiones del Perfil del cliente. Soporte total para 32 bits y 64 bits.|
| [.NET Standard 2.0](/cells/es/net/system-requirements/#net-standard)|La carpeta 'netStandard' contiene un ensamblado para usar con frameworks que implementen .NET Standard 2.0, como .NET Core 2.1/3.1. |[COM](/cells/es/net/system-requirements/)|Puede ser un salvavidas si tiene que desarrollar utilizando tecnologías previas a .NET.|

|[Windows Azure](/cells/es/net/system-requirements/)|Aspose.Cells for .NET también se ejecuta en Windows Azure.|
