{{% alert color="primary" %}} 

Los componentes Aspose.Cells for .NET no requieren que Microsoft Excel esté instalado ni en el cliente ni en el lado del servidor.

{{% /alert %}}

## **Entornos de desarrollo**
Puede utilizar Aspose.Cells for .NET para desarrollar aplicaciones en cualquier entorno de desarrollo que apunte a la plataforma .NET, pero los siguientes entornos son compatibles explícitamente:

- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- Microsoft Visual Studio 2022

{{% alert color="primary" %}} 

Para usar Aspose.Cells, la máquina no necesita tener instalado Microsoft Excel ya que la biblioteca en sí misma es un motor de gestión de hojas de cálculo. Sin embargo, para ver el archivo excel generado por el componente, es necesario tener al menos Microsoft Excel Viewer instalado en su máquina.

{{% /alert %}} 
## **Requisitos del sistema de Aspose.Cells Grid Suite**
La suite de cuadrícula de Aspose.Cells está compuesta por tres controles:

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb
- Aspose.Cells.GridJs

Los requisitos del sistema de los controles se describen por separado.

{{% alert color="primary" %}} 

Para usar estos controles, la máquina para ejecutar Aspose.Cells.GridDesktop y Aspose.Cells.GridWeb no necesita tener instalado Microsoft Excel ya que Aspose.Cells.GridDesktop y Aspose.Cells.GridWeb pueden abrir y guardar hojas de cálculo de Excel sin necesidad de Microsoft Excel.
