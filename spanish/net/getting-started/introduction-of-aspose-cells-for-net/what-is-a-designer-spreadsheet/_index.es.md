¿Quieres acelerar el desarrollo? Puede utilizar Aspose.Cells para crear archivos de Excel desde cero con la ayuda de su API flexible. Uno de los usos más comunes de Aspose.Cells es crear y poblar archivos de Excel con datos, que pueden provenir de una base de datos existente, un usuario u otras fuentes de forma dinámica. Además de agregar datos a archivos de Excel, también necesitarás aplicar formato (apariencia y estilo).

El formateo lleva tiempo. ¿Qué podría acelerar el desarrollo?

{{% /alert %}}

## **La Solución - Hojas de cálculo de diseño**

Para acelerar su desarrollo, Aspose.Cells ofrece la emocionante función de hojas de cálculo de diseño.

### **¿Qué es una hoja de cálculo de diseño?**

Una hoja de cálculo de diseño es un archivo de Excel creado usando Microsoft Excel en el cual todo el formato ya está aplicado y el contenido puede ser decidido en el momento del diseño. Es más como un archivo de plantilla que no contiene ningún dato, solo el formato, como se muestra a continuación:

### **¿Cómo usarla?**

Después de crear una hoja de cálculo de diseño, use Aspose.Cells para leer la hoja de cálculo de diseño y agregar datos a sus posiciones especificadas (según la configuración de formato predefinida) en tiempo de ejecución. Por ejemplo, en la captura de pantalla anterior, puede ver que ya decidimos en el momento del diseño agregar las columnas **Nombre del producto**, **Nombre de la categoría**, **Cantidad por unidad** y **Unidades en stock**. Todavía no tienen ningún dato: esto se agregará en tiempo de ejecución. La figura anterior muestra una hoja de cálculo muy simple pero las hojas de cálculo de diseño le ahorran tiempo incluso en hojas de cálculo complejas.

### **Marcadores Inteligentes**

Además, Aspose.Cells te permite agregar [marcadores inteligentes](/cells/es/net/smart-markers/) a la hoja de cálculo diseñada. Los marcadores inteligentes se procesan en tiempo de ejecución utilizando el método Process de la clase [**WorkbookDesigner**](https://reference.aspose.com/cells/net/aspose.cells/workbookdesigner). Esto puede ayudarte a escribir menos código y obtener grandes resultados.
