{{% alert color="primary" %}} 

Aspose.Cells está creado para ayudar a los desarrolladores a crear y gestionar hojas de cálculo de Microsoft Excel de forma programática. Aspose.Cells no solo comprende las hojas de cálculo en el formato de Microsoft Excel (XLS/XLSX) sino también otros formatos de archivo como valores separados por comas (CSV), delimitados por tabulaciones/TSV (texto), SpreadsheetML, etc.

Aspose.Cells proporciona la mayoría de las características disponibles en Microsoft Excel, pero lo emocionante es que Aspose.Cells puede procesar archivos de Excel dinámicamente, lo que permite a los desarrolladores procesar miles o millones de hojas de cálculo muy rápidamente.

Este artículo discute escenarios comunes de uso de Aspose.Cells.

{{% /alert %}} 
## **Escenarios de uso posibles**
Depende de los desarrolladores cómo usen Aspose.Cells, pero hay pocos escenarios comunes en los que Aspose.Cells puede ayudar. El uso más común de Aspose.Cells es su implementación en el servidor web o en los servicios web para el procesamiento en el lado del servidor. Algunos de los escenarios de uso posibles de Aspose.Cells se describen a continuación.
## **Gestión automatizada de hojas de cálculo**
1. Los usuarios cargan hojas de cálculo creadas manualmente en un servidor.
1. Aspose.Cells se utiliza para abrir archivos y recuperar datos.
   Después de que se recuperan los datos:
   1. Los datos se guardan en una base de datos.
   1. Algunos datos se presentan en páginas web.
   1. Algunos aspectos de las hojas de cálculo se modifican y se guardan de nuevo en archivos de Microsoft Excel.
## **Aplicación web**
1. Los usuarios introducen datos en páginas web.
1. Los datos son recopilados por el servidor web.
1. Aspose.Cells se utiliza para poblar datos en archivos completamente nuevos o en archivos de plantillas predefinidas.
1. Después de que se crea el archivo de Excel, puede ser enviado al usuario como un informe, etc.

{{% alert color="primary" %}} 

Hemos discutido algunos de los escenarios más comunes de uso de Aspose.Cells en entornos web. Pero Aspose.Cells también es igualmente útil en aplicaciones de escritorio.
