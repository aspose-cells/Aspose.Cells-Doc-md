Esta página describe las nuevas funciones Aspose.Cells más interesantes introducidas en versiones recientes.

##  Aspose.Cells for .NET 23.12

* Compatibilidad con PDF/A-2 y PDF/A-3 durante la conversión a PDF
* Admite el cálculo de agrupaciones discretas de tablas dinámicas
* Soporte para exportar datos de celdas a una lista genérica.
* Soporte para importar referencias cruzadas usando tablas vinculadas para marcadores inteligentes
* Admite la función MAKEARRAY para fórmulas
* Soporte para renderizar texto de celda con ángulo de rotación para GridJS
* Admite más opciones para importar/exportar tablas de datos para GridDesktop
* Admite renderizar imagen con rotación para GridDesktop
* Soporte para ajustar el orden de las hojas de trabajo para GridDesktop
*  Opción de soporte para mostrar/ocultar mensajes de advertencia de fórmula para GridDesktop

##  Aspose.Cells for .NET 23.11

* Soporte para convertir hojas de cálculo al archivo PDF versión 1.7 (ISO32000-1:2008)
* Soporte para insertar controles de formulario para GridJS
* Soporte para mostrar el relleno degradado del fondo de la celda para GridJS
* Soporte para copiar/eliminar/insertar/cambiar el nombre de la hoja de trabajo para GridDesktop
* Admite la extensión automática de filas/columnas máximas al realizar la operación de pegado para GridDesktop
* Vista previa de impresión de hojas de soporte para GridDesktop
*  Proporcionar .NET compilación principal de GridDesktop

##  Aspose.Cells for .NET 23.10

* Admite la operación de pegado adaptativo cuando el tamaño del rango de destino es diferente del rango de origen
* Soporte para establecer hipervínculos directamente a un rango
* Soporte para copiar varias hojas de trabajo simultáneamente
* Soporte para verificar si un rango está vacío para los datos de las celdas

##  Aspose.Cells for .NET 23.9

* Soporte para calcular nuevas funciones en fórmula: LAMBDA,LET,MAP,SCAN
* Soporte para importar/exportar archivos EPUB
* Soporte para filtrar series de gráficos.
* Localización de soporte para GridDesktop
* Proporcionar GridWeb para la plataforma .net7-windows

##  Aspose.Cells for .NET 23.8

* Soporte para preservar el espacio de relleno en fórmulas
* Soporte para reemplazar/insertar texto con formato para texto enriquecido de celda
* Soporte para renderizar control activex y control de formulario para botón giratorio en GridJs
* Admite más tipos de validación para GridJs: longitud de texto/decimal, lista de selección, expresión de fórmula

##  Aspose.Cells for .NET 23.7

* Mejora de muchas funciones de GridJS, como controles de renderizado, guardado del lado del servidor para cuadros combinados, etc.
* Detección de formato de archivo mejorada para algunos formatos nuevos, como el archivo Visio y el archivo del paquete OneNote.
* Admite firmas XAdES para CSharp X509Certificate2
* Soporte para trabajar en Blazor Web Assembly

* Soporte para filtrar por fecha y hora en GridJs
