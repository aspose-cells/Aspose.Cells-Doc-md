Esta página describe las características nuevas más interesantes de Aspose.Cells introducidas en versiones recientes.

## Aspose.Cells for .NET 22.12

* Soporte para exportar Excel a JSON con estructura de jerarquía de múltiples capas

## Aspose.Cells for .NET 22.11

Soporte para distinguir entre fórmula de matriz dinámica y fórmula de matriz heredada
* Soporte para leer/guardar fórmulas de matrices dinámicas para formato de archivo xlsb

## Aspose.Cells for .NET 22.10

* Soporte para usar SkiaSharp como biblioteca de gráficos para NET6
* Soporte para agregar y actualizar el gráfico con el archivo de plantilla CRTX

## Aspose.Cells for .NET 22.9

* Soporte para establecer la fórmula de la tabla para las celdas
* Soporte para establecer una fórmula de matriz dinámica con función personalizada y motor de cálculo
* Soporte para establecer configuraciones de gradiente preestablecidas para el formato de relleno de gradiente
* Soporte para aplicar bordes de contorno con colores de tema para Range

## Aspose.Cells for .NET 22.8

* Admite la exportación de comentarios y notas como se muestra en la hoja a html
* Admite la especificación de hojas de trabajo para la salida durante la conversión a pdf/xps

## Aspose.Cells for .NET 22.7

* Soporte para exportar html a un solo archivo con encabezado/pie de página
* Soporte para obtener la posición absoluta y el tamaño del Rango (en puntos)

## Aspose.Cells for .NET 22.6

* Mejora para calcular y rastrear fórmulas con cadena de cálculo de fórmulas

## Aspose.Cells for .NET 22.5

Mejora en el rendimiento de la memoria al configurar la fórmula de las celdas mientras se guarda el libro de trabajo en el modo LightCells

## Aspose.Cells for .NET 22.4

* Opciones mejoradas para controlar celdas en blanco para exportar CSV.
* Admite división complicada y actualización de formato condicional para operaciones de inserción/eliminación

## Aspose.Cells for .NET 22.3

* Compatibilidad para volver a calcular las celdas que dependen del nombre definido cuando se cambia el nombre definido y se habilita la cadena de cálculo del libro.
* Soporte para obtener hojas de celda recursivamente de acuerdo con la cadena de cálculo de fórmulas

## Aspose.Cells for .NET 22.2

* Soporte para extraer nombres definidos de NameCollection por filtro de alcance especificado.

## Aspose.Cells for .NET 22.1

* Soporte para generar pptx editables
* Soporte para importar csv grande en varias hojas de trabajo cuando la fila/columna de datos excede el límite
* Soporte para importar csv con LightCells
* Soporte para eliminar el enlace externo especificado
* Soporte para mantener fórmulas mientras se eliminan enlaces externos

## Aspose.Cells for .NET 21.12

* Admite la conversión de Excel a SQL Scripts
* Admite la importación y exportación de datos xml sin formato
Soporte para leer la mayor cantidad de datos posible del archivo xls dañado
* Soporte para interrumpir el proceso de SmartMarker
* Mejore el rendimiento para agregar rangos para formato y validación condicionales
* Establezca la prioridad más alta para la condición de formato recién agregada, al igual que el comportamiento de MS Excel

## Aspose.Cells for .NET 21.11

* Admite el cálculo de la función/fórmula XLookup.
* Integre guardar json e imágenes para guardar el libro de trabajo.
* Mejore el rendimiento al convertir Excel a HTML.
