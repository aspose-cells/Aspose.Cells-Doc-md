**Bienvenido a la sección de documentación de Aspose.Cells for JavaScript mediante C++**

Aspose.Cells for JavaScript mediante C++ es una biblioteca nativa de WebAssembly que permite a desarrolladores de JavaScript/Node.js crear, manipular y convertir hojas de cálculo programáticamente sin necesidad de Office Automation o Microsoft Excel.

La biblioteca admite los principales formatos de hojas de cálculo, incluyendo Excel (XLS, XLSX, XLSB, XLSM), ODS, CSV y HTML, proporcionando una solución completa para el procesamiento de hojas de cálculo directamente en el navegador o entornos de servidor sin dependencias externas.

{{% /alert %}}

Usar Aspose.Cells for JavaScript mediante C++ en sus aplicaciones le ofrece los siguientes beneficios:

- Alto rendimiento
- Funciones completas
- Compatibilidad universal
- Independencia de plataforma
- Preparado para servidores sin necesidad de servidor

### **Funciones Ricas**

Aspose.Cells for JavaScript vía C++ ofrece una amplia variedad de funciones para crear, convertir y manipular hojas de cálculo:

- Propiedades Incorporadas
- Propiedades Personalizadas
- Temas

- Hipervínculo
- Autofiltro

### **Soporte para Navegador y Node.js**

Totalmente compatible con navegadores modernos (Chrome, Firefox, Edge, Safari) y entornos de ejecución Node.js.
Garantiza un comportamiento coherente tanto en aplicaciones del lado cliente como en del lado servidor para una integración y ejecución sin problemas.
