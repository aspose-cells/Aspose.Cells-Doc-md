{{% alert color="primary" %}} 

By default, Aspose.Cells saves Excel into PDF with Standard size. However, you can also save it with Minimum size using the [PdfSaveOptions.OptimizationType](https://apireference.aspose.com/cells/net/aspose.cells/pdfsaveoptions/properties/optimizationtype) property. It accepts the following values

- PdfOptimizationType.Standard
- PdfOptimizationType.MinimumSize
