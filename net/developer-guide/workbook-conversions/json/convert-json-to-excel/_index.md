Aspose.Cells supports converting a Json(JavaScript Object Notation) file to Excel Workbook.

{{% /alert %}}

## **Convert JSON to Excel Workbook**
No need to wonder how to convert JSON to Excel file, because Apose.Cells for .NET library has best decision. The Aspose.Cells API provides support for converting JSON format to spreadsheets. You can use [**JsonLoadOptions**](https://apireference.aspose.com/cells/net/aspose.cells/JsonLoadOptions) class to specify additional settings for importing JSON to Workbook.

The following code example demonstrates importing JSON to Excel Workbook. Please see the code to convert [source file](sample.json) to the [output1 xlsx file](sample_out.xlsx) and [output2 xlsx file](sample_out2.xlsx) generated by the code for reference.
