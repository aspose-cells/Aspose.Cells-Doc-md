Aspose.Cells provides the ability to save pivot table in ODS files. For this, you just convert a workbook with existing pivot table or create a new pivot table and save the file in the ODS format. Be sure to call [**PivotTable.CalculateData()**](https://apireference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata) before saving to ensure that is the pivot table is rendered in the output ODS file.
