##Change Adjustment Values of the Shape
Aspose.Cells for python via java provides [**Shape.getGeometry().getShapeAdjustValues()**](https://reference.aspose.com/cells/python-java/asposecells.api/geometry#ShapeAdjustValues) method to make changes to the adjustment points with shapes. In the Microsoft Excel UI, adjustments display as yellow diamond nodes. For example:
- Rounded Rectangle has an adjustment to change the arc
- Triangle has an adjustment to change the location of the point
- Trapezoid has an adjustment to change the width of the top
- Arrows have two adjustments to change the shape of the head and tail
This article will explain the use of [**Shape.getGeometry().getShapeAdjustValues()**](https://reference.aspose.com/cells/python-java/asposecells.api/geometry#ShapeAdjustValues) method to change the adjustment value of the different shapes.
