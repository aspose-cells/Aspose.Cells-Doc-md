##Data Source Error
##### **Description**
When publishing a report, Aspose.Cells.ReportingServices Client generates an error:
![todo:image_alt_text](data-source-error_1.png)
##### **Solution**
1. Click **Data Source Setup**.
1. Select **Share data source**.
1. Re-publish the report.
