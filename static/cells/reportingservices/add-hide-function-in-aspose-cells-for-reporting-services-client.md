##Add Hide Function in Aspose.Cells for Reporting Services Client
The Hide function hides specified rows and columns. There are two methods in the Aspose.Cells for Reporting Services:
- [Set hide attributes for worksheet, rows and columns](https://docs.aspose.com/cells/reportingservices/set-a-table-s-hide-attributes-from-the-modify-attribute-form/).
- [Set a table's hide attributes using **Modify Attribute**](https://docs.aspose.com/cells/reportingservices/set-hide-option-for-excel-row-column-and-sheet-using-dyn-hide-form/).
