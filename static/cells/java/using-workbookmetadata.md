##Using WorkbookMetadata
Aspose.Cells allows you to load a light-weight version of the workbook into memory to edit its metadata information. Please use the [com.aspose.cells.WorkbookMetadata](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookMetadata) class to load the workbook.
