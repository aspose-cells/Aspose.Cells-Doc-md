##Export Excel to HTML with GridLines
If you want to export your Excel file into HTML with GridLines, then please use the [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) property and set it **true**.
