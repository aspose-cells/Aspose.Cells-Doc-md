##Get Pivot Table refresh date and refresh by who information
Aspose.Cells now supports fetching the refresh date and refresh by who information from a workbook.
[**PivotTable.getRefreshDate**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshDate) returns the date on which the PivotTable report was last refreshed. Similarly [**PivotTable.getRefreshByWho**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshedByWho) property returns the name of the user who refreshed the report last time.
