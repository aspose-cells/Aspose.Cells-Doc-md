##Manage Axes of Excel Charts
In Excel charts, there are 3 kinds of Axes:
1. Horizonal(Category) Axis : X Axis
1. Vertical(Value) Axis : Y Axis
1. Depth(Series) Axis : Z Axis
## **Axis Options**
Aspose.Cells also allows to manage chart's axes at runtime，with [Axis](https://reference.aspose.com/cells/java/com.aspose.cells/Axis) object, you can change all options of Axis as done in Excel.
