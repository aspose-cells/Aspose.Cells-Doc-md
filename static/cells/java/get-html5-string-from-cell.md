##Get HTML5 string from Cell
## **Possible Usage Scenarios**
Aspose.Cells returns the HTML string of the cell using the [**getHtmlString(boolean html5)**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#HtmlString) method. If you pass **false** as a parameter, it will return you Normal HTML but if you pass **true** as a parameter, it will return HTML5 string.
