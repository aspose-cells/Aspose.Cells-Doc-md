##Formatting Slicer
## **Possible Usage Scenarios**
You can format the slicer in Microsoft Excel by setting its number of columns or by setting its style etc. Aspose.Cells also allows you to do this using the [Slicer.NumberOfColumns](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#NumberOfColumns) and [Slicer.StyleType](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#StyleType) properties.
