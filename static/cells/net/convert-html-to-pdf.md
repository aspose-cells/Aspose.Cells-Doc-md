## **C# Convert HTML to PDF**
The following C# code sample shows how to convert an HTML document to a PDF using [Aspose.Cells for .NET](https://releases.aspose.com/cells/net/).
1. Create an instance of the [HtmlLoadOptions](https://reference.aspose.com/cells/net/aspose.cells/htmlloadoptions/) class.
1. Initialize [Workbook](https://reference.aspose.com/cells/net/aspose.cells/workbook/) object.
1. Save output PDF document by calling Workbook.Save() method.
