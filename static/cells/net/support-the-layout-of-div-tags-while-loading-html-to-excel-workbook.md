##Support the layout of DIV tags while loading HTML to excel workbook
Normally, the layout of div tags are ignored while loading HTML into an excel workbook object. However, if you want the layout of div tags are not ignored, then please set the [HTMLLoadOptions.SupportDivTag](https://reference.aspose.com/cells/net/aspose.cells/htmlloadoptions/properties/supportdivtag) property to **true**. The default value of this property is **false**.
