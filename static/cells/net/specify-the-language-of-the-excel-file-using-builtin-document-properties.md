##Specify the Language of the Excel File using BuiltIn Document Properties
## **Possible Usage Scenarios**
You can change the Language of Excel file by right-clicking the file and then selecting Properties > Details and then editing the Language field. Please use [**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/language) property to change it programmatically using Aspose.Cells APIs.
