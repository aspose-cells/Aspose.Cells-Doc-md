##Copy Range Data Only
This article describes how to Copy Range Data Only with Aspose.Cells for Python via .NET library.
Sometimes, you need to copy data from one range of cells to another, copying the data only, not the formatting. Aspose.Cells for Python via .NET offers this feature.
This article provides a sample code that uses Aspose.Cells for Python via .NET to copy a range of data.
This example shows how to:
1. Create a workbook.
1. Add data to cells in the first worksheet.
1. Create a [**Range**](https://reference.aspose.com/cells/python-net/aspose.cells/range).
1. Create a [**Style**](https://reference.aspose.com/cells/python-net/aspose.cells/style) object with specified formatting attributes.
1. Apply the style formatting to the range.
1. Create another range of cells.
1. Copy data of the first range to this second range.
