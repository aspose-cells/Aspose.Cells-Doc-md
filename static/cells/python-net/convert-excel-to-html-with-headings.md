##Convert Excel to HTML with headings
This topic show you how to Convert Excel to HTML with headings using Aspose.Cells for Python via NET.
## **Convert Excel to HTML with headings**
Aspose.Cells for Python via NET provides the option to export headings while converting Excel to HTML. For this, you may set the [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) property to **true**. The default value of [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)property is **false**.
