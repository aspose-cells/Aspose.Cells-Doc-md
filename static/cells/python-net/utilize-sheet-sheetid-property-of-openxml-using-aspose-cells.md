##Utilize Sheet.SheetId property of OpenXml using Aspose.Cells
This article shows how to utilize Sheet.SheetId property of OpenXml using Aspose.Cells for Python via .NET APIs.
## **Possible Usage Scenarios**
*Sheet.SheetId* property is found inside the *DocumentFormat.OpenXml.Spreadsheet* namespace and is part of OpenXml. You can see this property and its value inside *workbook.xml* as shown in the following screenshot. Aspose.Cells for Python via .NET provides the equivalent property as [**Worksheet.tab_id**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/tab_id).
