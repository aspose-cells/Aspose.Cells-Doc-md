## Possible Usage Scenarios

Sometimes you want to display text in Far East language font e.g. Japanese, Chinese, Thai, etc. Aspose.Cells provides [TextOptions.FarEastName](https://apireference.aspose.com/java/cells/com.aspose.cells/textoptions#FarEastName) property that can be used to specify the font name of Far East language. Besides, you can also specify the Latin font name using T[extOptions.LatinName](https://apireference.aspose.com/java/cells/com.aspose.cells/textoptions#LatinName) property.
