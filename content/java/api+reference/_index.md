# Aspose.Cells for Java : API Reference


Aspose.Cells for Java is a class library that enables your applications to perform a great range of spreadsheet processing tasks. Aspose.Cells for Java supports the popular spreadsheet (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) file formats and allows exporting Excel files to PDF, XPS, HTML, MHTML, Plain Text, TIFF, JPG, PNG, BMP, and SVG.

The latest API Reference can be found at [Aspose.Cells for Java API Reference](https://apireference.aspose.com/java/cells).
