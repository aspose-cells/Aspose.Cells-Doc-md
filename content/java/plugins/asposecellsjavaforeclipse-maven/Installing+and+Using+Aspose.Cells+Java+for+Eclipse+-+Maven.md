## Installing

**Aspose.Cells Java for Eclipse (Maven)** plugin can be easily installed from the Available **Plugin** tab in the Plugin dialog.

*   To open it, select **Plugins** from the **Tools** menu in Eclipse.   
    ![](https://docs2.aspose.com/cells/java/attachments/thumbnails/5275907/5473383)

*   This adds the **Aspose.Cells Maven Project** in New Project wizard and **Aspose.Cells Code Example** in New Other wizard of Eclipse IDE.   
    ![](download/thumbnails/5275907/1404090928)

## Using

### Aspose.Cells Maven Project (wizard)

To create **Maven Project** by wizard for using [Aspose.Cells for Java API](http://goo.gl/gtvhlQ):

1.  Select **New -> Project**.
2.  Select **Aspose.Cells Maven Project** in the **Maven** category.
3.  Click **Next**.  
    ![](download/thumbnails/5275907/1000111024)
4.  Provide **Project Name, Location, GroupId, ArtifactId** and **Version** for your Maven Project and click **Finish.**  
    ![](download/thumbnails/5275907/1810020411)
5.  This will retrieve the [Aspose.Cells for Java](http://goo.gl/gtvhlQ) latest [Maven Dependency](http://maven.aspose.com/repository/ext-release-local/com/aspose/aspose-cells) reference from [Aspose Cloud Maven Repository](http://maven.aspose.com/artifactory/webapp/home.html?0) and configure it in **pom.xml**. If you have opted for **Also Download Code Examples,** downloading of the **Code Examples** will also begin from the [Aspose.Cells for Java API Examples Repository. ](https://goo.gl/uxznrb)
6.  Following **Maven** Project will be created on your **Eclipse IDE** on completion of the wizard:   
    ![](download/thumbnails/5275907/2071850032)
7.  The created **Maven Project** is configured to use **Aspose.Cells for Java API** and ready to be enhanced as per your Project requirements.  
    If you have opted to download [Code Examples](https://goo.gl/uxznrb), you can use **Aspose.Cells Code Example (wizard)** for importing needed **Code Examples** of [Aspose.Cells for Java](http://goo.gl/gtvhlQ) API into your project.

### Aspose.Cells Code Example (wizard)

**Aspose.Cells Code Example wizard** allows you to try out many samples provided for [Aspose.Cells for Java](http://goo.gl/gtvhlQ) API.

To be able to use **Aspose.Cells Code Example wizard** comfortably: it is recommended to always select **Also Download Code Examples** while creating **Maven Project** on **Aspose.Cells Maven Project** **wizard**, 

To use examples, just:

1.  Click **New -> Other** in **Eclipse**.
2.  Select **Aspose.Cells Code Example** in the **Java** category.
3.  Click **Next**.   
    ![](download/thumbnails/5275907/1218360395)
4.  Choose your project and then expand the tree to select **Code Example** category and click **Finish**.  
    ![](download/thumbnails/5275907/1399514877)
5.  This will copy the selected category **Code Examples** Java files into the project under **com.aspose.cells.examples** package. Also any required resources needed by the Code Examples will be copied into **src/main/resources** folder, as shown below:  
    ![](download/thumbnails/5275907/1267999683)
6.  Review the example code, compile and run.
7.  You can now test other examples and start building your own application using [Aspose.Cells for Java API](http://goo.gl/gtvhlQ).
