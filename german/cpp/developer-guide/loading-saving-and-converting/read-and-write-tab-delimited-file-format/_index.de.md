## **Mögliche Verwendungsszenarien**
Microsoft Excel unterstützt viele Formate wie XLS, XLSX, XLSM, XLSB, CSV, tabstoppgetrennt usw. Aspose.Cells unterstützt ebenfalls viele dieser Formate. Dieser Artikel erläutert, wie die Excel-Datei im tabstoppgetrennten Format mit Aspose.Cells gelesen und geschrieben wird.
## **Tabstoppgetrenntes Dateiformat lesen und schreiben**
