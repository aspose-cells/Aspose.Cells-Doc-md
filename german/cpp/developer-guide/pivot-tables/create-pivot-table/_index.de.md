##  **Einführung**
 Benutzen Sie bitte[PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) Methode zum Erstellen einer Pivot-Tabelle innerhalb des Arbeitsblatts. Sobald die Pivot-Tabelle erstellt ist, können Sie mit ihr arbeiten[PivotTable](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/)Klasse.
