{{% alert color="primary" %}} 

Verwenden Sie WordArt, um spezielle Text- oder Bildeffekte auf Tabellenkalkulationen hinzuzufügen. Zum Beispiel, einen Titel über die Oberseite der Datei strecken, Text dekorieren und Text an eine vordefinierte Form anpassen oder Text als Hintergrundwasserzeichen auf ein Excel-Blatt anwenden. Das WordArt wird zu einem Objekt, das Sie in Tabellenkalkulationen bewegen oder positionieren können, um Dekorationen hinzuzufügen.

{{% /alert %}}
