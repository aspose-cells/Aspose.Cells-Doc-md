## **Kopieren Sie das Design von einer Arbeitsmappe in eine andere**
 Der folgende Beispielcode zeigt, wie es geht[Thema aus einer Arbeitsmappe kopieren](45056005.xlsx) hinein[ein anderes Arbeitsheft](45056004.xlsx).
