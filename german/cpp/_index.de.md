**Willkommen bei der Aspose.Cells für C++**

Aspose.Cells für C++ ist eine native C++-Bibliothek, die es C++-Entwicklern ermöglicht, Tabellenkalkulationen programmgesteuert zu erstellen, zu manipulieren und zu konvertieren, ohne dass eine Office-Automatisierung oder Microsoft Excel-Anwendung erforderlich ist. Aspose.Cells für C++ unterstützt Tabellenkalkulationsformate wie MS Excel 97-2003 (XLS), MS Excel 2007-2013/2016 (XLSX, XLSM, XLSB usw.), Open Office XML und andere Formate. Sie können auch Bilder aus Arbeitsblättern rendern und MS Excel-Tabellen in das PDF-Format konvertieren. Mit API können Sie auch Diagramme und Formen erstellen, manipulieren und rendern. Darüber hinaus sind Formelberechnungen ein weiterer guter Aspekt, den die Komponente bietet, und Sie können sie nutzen.
