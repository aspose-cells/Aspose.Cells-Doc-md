##  **Installation von Aspose.Cells for C++ in Windows**
###  **Verwenden des Paketmanagers Nuget**
1. Suchen Sie im Paketmanager Nuget nach Aspose.Cells.
 1. Hier ist die[Link zu NuGet Paket Aspose.Cells.Cpp](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. Weitere Informationen finden Sie unter [Aspose Cells bis NuGet installieren] (https://docs.aspose.com/cells/cpp/getting-started/).

Sie können das ZIP-Paket auch unter Aspose.Cells for C++ herunterladen[Download-Seite](https://downloads.aspose.com/cells/cpp/). Das Archiv enthält auch die Offline-Dokumentation API.
###  **Verwenden von Include- und lib-Ordnern**
1.  Laden Sie das herunter/installieren Sie es[neueste Dateien](https://downloads.aspose.com/cells/cpp/).
1. Entpacken Sie den Ordner in die Produktionsumgebung.
1. Um Aspose.Cells for C++ zu verwenden, verweisen Sie auf die Verzeichnisse „Include“ und „lib/dll“ in Ihrem Projekt.
