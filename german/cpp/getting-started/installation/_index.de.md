## **Installation von Aspose.Cells for C++ unter Windows**
### **Verwenden des Nuget-Paket-Verwalters**
1. Suchen Sie in NuGet Package Manager nach Aspose.Cells. 
   1. Hier ist der [Link zum NuGet-Paket Aspose.Cells.Cpp](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. Beachten Sie die [Installieren von Aspose Cells über NuGet] für Details.(https://docs.aspose.com/cells/cpp/getting-started/).  

Sie können das ZIP-Paket auch von der Aspose.Cells for C++ [Download-Seite](https://downloads.aspose.com/cells/cpp/) herunterladen. Das Archiv enthält auch die Offline-API-Dokumentation.
### **Verwenden von Include- und lib-Ordnern**
1. Laden/Installieren Sie die [neuesten Dateien](https://downloads.aspose.com/cells/cpp/).
1. Entpacken Sie den Ordner in die Produktionsumgebung.
1. Verweisen Sie zur Verwendung von Aspose.Cells for C++ auf das Verzeichnis "Include" und "lib/dll" in Ihrem Projekt.

## **Installation von Aspose.Cells for C++ unter Linux**
1. Laden/Installieren Sie die [neuesten Dateien](https://downloads.aspose.com/cells/cpp/).
1. Entpacken Sie den Ordner in die Produktionsumgebung.
1. Um Aspose.Cells for C++ zu verwenden, verweisen Sie im Projekt auf das Verzeichnis "Include" und "libAspose.Cells.so".
