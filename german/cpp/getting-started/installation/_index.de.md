## **Installation von Aspose.Cells für C++ in Windows**
### **Mit Nuget Package Manager**
1. Suchen Sie im Nuget Package Manager nach Aspose.Cells.
 1. Hier ist die[Link zu NuGet Paket Aspose.Cells.Cpp](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. Einzelheiten finden Sie unter [Install Aspose Cells bis NuGet].(https://docs.aspose.com/cells/cpp/getting-started/).

Sie können das ZIP-Paket auch unter Aspose.Cells für C++ herunterladen[Download-Seite](https://downloads.aspose.com/cells/cpp/). Das Archiv enthält auch die Offline-Dokumentation API.
### **Verwenden von Include- und lib-Ordnern**
1.  Herunterladen/Installieren der[neueste Dateien](https://downloads.aspose.com/cells/cpp/).
1. Entpacken Sie den Ordner in die Produktionsumgebung.
1. Um Aspose.Cells für C++ zu verwenden, verweisen Sie in Ihrem Projekt auf die Verzeichnisse „Include“ und „lib“.

## **Installation von Aspose.Cells für C++ unter Linux**
1.  Laden Sie Aspose.Cells für C++ von der folgenden Seite herunter:[Download Aspose.Cells für C++ (Linux)](https://downloads.aspose.com/cells/cpp)
2. Entpacken Sie das Paket und Sie finden eine Demo, die zeigt, wie Sie Aspose.Cells für C++ für Linux verwenden.
3. Führen Sie „cd Demo“ in Ihrer Linux-Befehlszeile aus
4. Führen Sie "rm -rf build;mkdir build;cd build" aus
5. Führen Sie „cmake ..“ aus, um ein Makefile von CMakeLists.txt im Demo-Ordner zu erstellen
6. Führen Sie "make" zum Kompilieren aus
 7. Führen Sie „./demo“ aus, Sie werden das Ergebnis sehen
