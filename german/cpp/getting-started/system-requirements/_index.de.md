Aspose.Cells for C++ ist eine native C++-Bibliothek, die es C++-Entwicklern ermöglicht, Tabellenkalkulationen programmgesteuert zu erstellen, zu manipulieren und zu konvertieren, ohne dass eine Office-Automatisierung oder Microsoft Excel-Anwendung erforderlich ist.

## Entwicklungsumgebung

Sie können Aspose.Cells for C++ verwenden, wenn Sie Anwendungen für Windows oder Linux entwickeln.

### Windows

 Aspose.Cells for C++ kann verwendet werden, um Anwendungen in jeder Entwicklungsumgebung zu entwickeln, die unterstützt[Microsoft Visual Studio v142-Plattform-Toolset](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017?view=msvc-160), aber die in der folgenden Tabelle aufgeführten Umgebungen werden ausdrücklich unterstützt:

<table>  
	<tr>
			<td style="font-weight: bold; width:800px">Entwicklungsumgebungen</td>
		</tr>
  <tr>
			<td><ul><li>MicrosoftVisual Studio 2017</li><li>MicrosoftVisual Studio 2019</li></ul></td>
			</tr>
</table>

### Linux

Aspose.Cells for C++ kann verwendet werden, um Anwendungen in der Entwicklungsumgebung zu entwickeln, die C++11 oder höher unterstützen, aber die folgenden Compiler und Tools werden ausdrücklich unterstützt:

<table>  
	<tr>
			<td style="font-weight: bold; width:800px">Compiler</td>
		</tr>
  <tr>
			<td><ul><li>GCC 7.5.0 oder höher</li><li>CMake 3.0.0 oder höher</li></ul></td>
			</tr>
</table>

### Zusätzliche Abhängigkeit von Linux

 Aspose.Cells for C++ unter Linux abhängig von[libgdiplus](https://github.com/mono/libgdiplus//) Binärdateien sowohl dynamische Bibliothek als auch Tool. Bitte installieren Sie es vor der Verwendung:

- Installation von libgdiplus auf Ubuntu oder Debian</br>
`sudo apt-get install libgif-dev autoconf libtool automake build-essential gettext libglib2.0-dev libcairo2-dev libtiff-dev libexif-dev`
