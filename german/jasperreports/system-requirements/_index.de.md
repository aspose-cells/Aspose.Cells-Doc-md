{{% alert color="primary" %}} 

Um Aspose.Cells for JasperReports auf Ihrem System zu installieren, muss Ihr System die folgenden Anforderungen erfüllen:

- Java 2 Plattform Standard Edition 1.6 oder höher.
- JasperReports 6.3.0 oder höher.

{{% /alert %}}
