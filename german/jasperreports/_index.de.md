**Willkommen bei Aspose.Cells für JasperReports**

 Aspose.Cells für JasperReports ist eine Bibliothek, die als Ersatz für fungiert[JasperReports](https://community.jaspersoft.com/project/jasperreports-library)s Original Microsoft Excel Exporter, gebündelt mit einigen großartigen Verbesserungen. Es eliminiert die Abhängigkeit von der POI-Bibliothek und verwendet Aspose.Cells for Java im Backend, um Berichte in Microsoft Excel-Formaten (XLS, XLSX) zu exportieren.
