 Beseitigen Sie die Abhängigkeit der POI-Bibliothek.

 l Unterstützt verschiedene Arten von Diagrammen, einschließlich Balkendiagramm, Flächendiagramm, Kreisdiagramm, Liniendiagramm usw.
