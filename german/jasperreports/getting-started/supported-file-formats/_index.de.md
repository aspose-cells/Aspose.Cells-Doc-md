|**Format**|**Beschreibung**|**Belastung**|**Speichern**|
|:- |:- |:- |:- |
|[XLS](https://docs.fileformat.com/spreadsheet/xls/)|Speichert das Dokument in der Microsoft Excel 97-2003-Arbeitsmappe.||{{< emoticons/tick >}}|
|[XLSX](https://docs.fileformat.com/spreadsheet/xlsx/)|Speichert das Dokument in Office Open XML SpreadsheetML Arbeitsmappe oder Vorlagendatei mit oder ohne Makros.||{{< emoticons/tick >}}|
|[ODS](https://docs.fileformat.com/spreadsheet/ods/)|Speichert das Dokument in ODS (OpenDocument Spreadsheet).||{{< emoticons/tick >}}|
|[XML](https://docs.fileformat.com/web/xml/)|Erweiterbare Auszeichnungssprache||{{< emoticons/tick >}}|
|[PDF](https://docs.fileformat.com/pdf/)|Adobe Portable Document-Format.||{{< emoticons/tick >}}|
|[CSV](https://docs.fileformat.com/spreadsheet/csv/)|Speichert das Dokument in der Datei CSV (Comma Separated Value).||{{< emoticons/tick >}}|
|[TSV](https://docs.fileformat.com/spreadsheet/tsv/)|Speichert das Dokument in der Datei TSV (durch Tabulatoren getrennte Werte).||{{< emoticons/tick >}}|
|TabDelimited|Speichert das Dokument in einer tabulatorgetrennten Textdatei, ebenso wie die Datei TSV.||{{< emoticons/tick >}}|
