{{% alert color="primary" %}} 

Einige effektive Funktionen von Aspose.Cells for JasperReports sind unten aufgeführt:

1. Aspose.Cells for JasperReports unterstützt Microsoft Excel-Diagramme, sodass Diagramme als bearbeitbare Diagramme und nicht als statische Bilder exportiert werden. Auf diese Weise können Benutzer die Diagramme bei Bedarf manuell in Microsoft Excel bearbeiten.
 2. Aspose.Cells for JasperReports ist[einfach in JasperServer integriert](/cells/de/jasperreports/integration-with-jasperserver/).

{{% /alert %}}
