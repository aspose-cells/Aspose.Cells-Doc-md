{{% alert color="primary" %}} 

Einige effektive Funktionen von Aspose.Cells for JasperReports sind unten aufgeführt:

1. Aspose.Cells for JasperReports unterstützt Microsoft Excel-Diagramme, sodass Diagramme als bearbeitbare Diagramme anstelle von statischen Bildern exportiert werden. Auf diese Weise können Benutzer die Diagramme manuell in Microsoft Excel bearbeiten, falls erforderlich.
2. Aspose.Cells for JasperReports ist [einfach mit JasperServer zu integrieren](/cells/de/jasperreports/integration-with-jasperserver/).

{{% /alert %}}
