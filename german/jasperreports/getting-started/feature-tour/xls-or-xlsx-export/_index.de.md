{{% alert color="primary" %}}

 Aspose.Cells for JasperReports ermöglicht es Ihnen, Berichte im Format XLS und XLSX ohne Verlust von Layout oder Formatierung bereitzustellen. Um Tabellenkalkulationen in den Excel-Formaten Microsoft zu erstellen, stützt sich Aspose.Cells for JasperReports auf eine integrierte Version von[Aspose.Cells for Java](https://products.aspose.com/cells/java/), eine führende Tabellenkalkulationsbibliothek von Aspose.

Microsoft Excel ist nicht erforderlich, um Dokumente mit Aspose.Cells zu generieren.
