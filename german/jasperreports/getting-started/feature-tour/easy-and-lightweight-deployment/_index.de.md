{{% alert color="primary" %}}

 Aspose.Cells für JasperReports ist ein externer Exporteur für[JasperReports](https://community.jaspersoft.com/project/jasperreports-library).

 Aspose.Cells für JasperReports wird als einzelne JAR-Datei bereitgestellt und kann problemlos auf Computern installiert werden, auf denen JasperReports ausgeführt wird.[Laden Sie es herunter, um eine Testversion zu starten](https://downloads.aspose.com/cells/jasperreports).

[So installieren Sie den Exporter](/cells/de/jasperreports/installation/), **aspose.cells.jasperreports.jar** kopiert werden soll**JasperReports\lib** Verzeichnis.

{{% /alert %}}
