{{% alert color="primary" %}}

Aspose.Cells for JasperReports ist ein externer Exporteur für [JasperReports](https://community.jaspersoft.com/project/jasperreports-library).

Aspose.Cells for JasperReports wird als einzelne JAR-Datei bereitgestellt und kann mühelos auf Computern, auf denen JasperReports ausgeführt wird, installiert werden. [Laden Sie sie herunter, um eine Testversion zu starten](https://downloads.aspose.com/cells/jasperreports).

Um den Exporteur zu installieren, sollte die Datei **aspose.cells.jasperreports.jar** in das **JasperReports\lib** Verzeichnis kopiert werden.

{{% /alert %}}
