{{% alert color="primary" %}}

Aspose.Cells for JasperReports ist ein Produkt zum Exportieren von Berichten aus JasperReports und JasperServer im Microsoft Excel-Format. Aspose.Cells for JasperReports erbt alle Funktionen des integrierten Exporteurs und gleicht einige Schwächen des originalen XLS-Renderers von JasperReports aus. Dieser Abschnitt bietet eine detaillierte Beschreibung der Funktionen von Aspose.Cells for JasperReports. Klicken Sie auf die entsprechenden Links in der Tabelle, um mehr über die Funktionen zu erfahren.

{{% /alert %}}

|**Funktion** |**Beschreibung** |
| :- | :- |
|[XLS or XLSX Export](/cells/de/jasperreports/xls-or-xlsx-export/)|Exportieren Sie Berichte im Microsoft Excel XLS-Format aus Ihrer Anwendung heraus, ohne Microsoft Excel zu verwenden. |
|[Hohe Treue zum Berichtsdesign](/cells/de/jasperreports/high-fidelity-to-the-report-design/)|Das Layout und das Erscheinungsbild eines XLS/XLSX-Berichts entsprechen dem Original, ähnlich wie bei PDF oder anderen Formaten. |
|[Einfache und schlanke Bereitstellung](/cells/de/jasperreports/easy-and-lightweight-deployment/)|Aspose.Cells für [JasperReports](https://community.jaspersoft.com/project/jasperreports-library) ist ein [JasperReports](https://community.jaspersoft.com/project/jasperreports-library)- und JasperServer-Exporteur im Microsoft Excel-Format. |
