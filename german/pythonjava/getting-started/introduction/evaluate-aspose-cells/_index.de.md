{{% alert color="primary" %}}

Sie können Aspose.Cells zur Evaluierung problemlos von [pypi](https://pypi.org/project/aspose-cells/) herunterladen. Der Evaluierungsdownload ist derselbe wie der Download für die Vollversion: Die Evaluierung wird lizenziert, wenn Sie eine Lizenz anwenden.

{{% /alert %}}

Die Evaluierungsversion von Aspose.Cells (d.h. Aspose.Cells ohne angewendete Lizenz) bietet die meisten Produktfunktionen. Eine Einschränkung ist, dass immer ein zusätzliches Arbeitsblatt mit einem Evaluierungswasserzeichen in eine Arbeitsmappe eingefügt wird, wenn eine Tabelle geöffnet oder gespeichert wird. Das eingefügte zusätzliche Evaluierungswasserzeichenarbeitsblatt wird unten angezeigt:
