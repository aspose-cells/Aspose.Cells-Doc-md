## **Möchten Sie die Entwicklung beschleunigen?**

{{% alert color="primary" %}}

Sie können Aspose.Cells verwenden, um mit Hilfe des flexiblen API Excel-Dateien von Grund auf neu zu erstellen. Eine der häufigsten Anwendungen für Aspose.Cells ist das Erstellen und Füllen von Excel-Dateien mit Daten, die aus einer vorhandenen Datenbank, einem Benutzer oder anderen Quellen stammen können. dynamisch. Neben dem Hinzufügen von Daten zu Excel-Dateien müssen Sie auch schreiben, um die Formatierung (Look and Feel) anzuwenden.

Das Formatieren braucht Zeit. Was könnte die Entwicklung beschleunigen?

{{% /alert %}}

## **Die Lösung - Designer-Tabellenkalkulationen**

Um Ihre Entwicklung zu beschleunigen, bietet Aspose.Cells die aufregende Feature-Designer-Tabelle.

### **Was ist eine Designer-Tabelle?**

Ein Designer-Arbeitsblatt ist eine Excel-Datei, die mit Microsoft Excel erstellt wurde, in der alle Formatierungen bereits angewendet wurden und der Inhalt zur Entwurfszeit festgelegt werden kann. Es ist eher eine Vorlagendatei, die keine Daten enthält, sondern nur die Formatierung, wie unten gezeigt:

### **Wie benutzt man es?**

 Verwenden Sie nach dem Erstellen eines Designer-Arbeitsblatts Aspose.Cells, um das Designer-Arbeitsblatt zu lesen und Daten an den angegebenen Positionen (gemäß den vordefinierten Formateinstellungen) zur Laufzeit hinzuzufügen. Im obigen Screenshot können Sie beispielsweise sehen, dass wir uns bereits zur Entwurfszeit für das Hinzufügen entschieden haben**Produktname**, **Kategoriename**, **Menge pro Einheit** und**Einheiten auf Lager**Säulen. Sie haben noch keine Daten: diese werden zur Laufzeit hinzugefügt. Die obige Abbildung zeigt eine sehr einfache Tabellenkalkulation, aber Designer-Tabellenkalkulationen sparen Ihnen selbst bei komplexen Tabellenkalkulationen Zeit.

### **Intelligente Markierungen**

 Zusätzlich können Sie mit Aspose.Cells hinzufügen[intelligente Markierungen](/cells/de/python-java/smart-markers/) zur Designer-Tabelle. Smart-Marker werden zur Laufzeit mit verarbeitet[**WorkbookDesigner**](https://reference.aspose.com/cells/python-java/asposecells.api/WorkbookDesigner) Process-Methode der Klasse. Dies kann Ihnen helfen, weniger Code zu schreiben und trotzdem ein großartiges Ergebnis zu erzielen.

[Erfahren Sie mehr über intelligente Marker](/cells/de/python-java/smart-markers/).
