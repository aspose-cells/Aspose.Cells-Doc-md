## **Systemanforderungen**
Aspose.Cells für Python via Java ist eine plattformunabhängige API und kann auf jeder Plattform (Windows, Linux und MacOS) verwendet werden, auf der [Python](https://www.python.org/downloads/) installiert ist. Auf dem Rechner müssen Java 8 oder neuere Versionen installiert sein, bevor die Installation erfolgt.

## **Python-Version**
- Python 3.5 oder höher
## **Java-Version**
- Java 1.8 oder höher

## **Installation**
### **Windows:**
- **Java installieren und die Umgebungsvariablen JAVA_HOME und PATH setzen**
Zum Beispiel:
{{< highlight java >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}

- **Aspose.Cells für Python via Java aus pypi installieren
Sie können Aspose.Cells for Python via Java ganz einfach von [pypi](https://pypi.org/project/aspose-cells/) mit folgendem Befehl verwenden.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}

### **Linux:**
- **Java installieren**

- **Aspose.Cells für Python via Java aus pypi installieren
Sie können Aspose.Cells for Python via Java ganz einfach von [pypi](https://pypi.org/project/aspose-cells/) mit folgendem Befehl verwenden.
