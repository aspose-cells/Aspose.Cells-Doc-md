## **Konvertieren Sie Excel in SVG**
SVG(**Skalierbare Vektorgrafiken)**ist eine auf XML-Standards basierende Spezifikation für zweidimensionale Vektorgrafiken. Es ist ein offener Standard, der seit 1999 vom World Wide Web Consortium (W3C) entwickelt wird.

Aspose.Cells für Python über Java unterstützt die Konvertierung von Excel-Dateien in SVG-Bilder. Dafür sorgt die API[SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender), [ImageOrPrintOptions](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions), und[WorkbookRender](https://reference.aspose.com/cells/python/asposecells.api/WorkbookRender)Klassen.
