Aspose.Cells für Python via Java unterstützt die Konvertierung von Excel (.xls, xlsx, .xlsb, xlsm), CSV und OpenOffice (.ods) Dateien in eine PowerPoint-Datei.

{{% /alert %}}

## **Excel-Arbeitsmappe in PPT konvertieren**

Sie brauchen sich nicht zu wundern, wie man Excel-Arbeitsmappe in PowerPoint konvertiert, denn die Aspose.Cells für die Python via Java-Bibliothek trifft die beste Entscheidung. Die Aspose.Cells für Python via Java-API bietet Unterstützung für die Konvertierung von Tabellenkalkulationen in PowerPoint-Format. Um die Arbeitsmappe in PowerPoint zu exportieren, übergeben Sie [**SaveFormat.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) als zweiten Parameter der [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\))-Methode. Sie können auch die Klasse [**PptxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) verwenden, um zusätzliche Einstellungen für den Export des Arbeitsblatts in eine .pptx-Datei festzulegen.
