## **Konvertieren Sie Excel-Arbeitsmappe in PPT**

 Sie müssen sich nicht fragen, wie Sie eine Excel-Arbeitsmappe in PowerPoint konvertieren, denn die Bibliothek Aspose.Cells for Python via Java hat die beste Entscheidung. Aspose.Cells for Python via Java API bietet Unterstützung für die Konvertierung von Tabellenkalkulationen in das PowerPoint-Format. Um die Arbeitsmappe nach PowerPoint zu exportieren, pass[**SaveFormat.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) als zweiter Parameter von[**Arbeitsmappe.speichern**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) Methode. Sie können auch verwenden[**PptxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) -Klasse, um zusätzliche Einstellungen für den Export von Arbeitsblättern in eine PPTX-Datei anzugeben.
