Aspose.Cells for Python via Java unterstützt die Konvertierung von Excel- (.xls, xlsx, .xlsb, xlsm ), CSV- und OpenOffice- ( .ods) Dateien in Word-Dateien.

{{% /alert %}}

## **Excel-Arbeitsmappe in Word konvertieren**

Sie müssen sich nicht fragen, wie Sie eine Excel-Arbeitsmappe in Word konvertieren, denn die Bibliothek Aspose.Cells for Python via Java hat die beste Entscheidung. Die Aspose.Cells for Python via Java API bietet Unterstützung für die Konvertierung von Tabellenkalkulationen in das Word-Format. Um die Arbeitsmappe nach Word zu exportieren, übergeben Sie[**SaveFormat.DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) als zweiter Parameter von[**Arbeitsmappe.speichern**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) Methode. Sie können auch verwenden[**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions) -Klasse, um zusätzliche Einstellungen für den Export von Arbeitsblättern in eine .docx-Datei anzugeben.
