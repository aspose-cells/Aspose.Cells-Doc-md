Aspose.Cells für Python via Java unterstützt die Konvertierung von Excel (.xls, xlsx, .xlsb, xlsm), CSV und OpenOffice (.ods)-Dateien in eine Word-Datei.

{{% /alert %}}

## **Excel-Arbeitsmappe in Word konvertieren**

Kein Bedarf, sich zu fragen, wie man Excel-Arbeitsmappe in Word konvertiert, denn die Aspose.Cells for Python via Java-Bibliothek hat die beste Lösung. Die Aspose.Cells for Python via Java-API bietet Unterstützung für die Konvertierung von Tabellenkalkulationen in das Word-Format. Um die Arbeitsmappe in Word zu exportieren, übergeben Sie [**SaveFormat.DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) als zweiten Parameter der [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\))-Methode. Sie können auch die [**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions)-Klasse verwenden, um zusätzliche Einstellungen für den Export des Arbeitsblatts in eine .docx-Datei anzugeben.
