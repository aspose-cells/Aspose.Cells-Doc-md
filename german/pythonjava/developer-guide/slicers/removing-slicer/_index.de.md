## **Slicer entfernen**
 Um den Slicer in Microsoft Excel zu entfernen, wählen Sie einfach den Slicer aus und drücken Sie die*Löschen*Taste. Um das Speichern mit Aspose.Cells für Python über Java zu erreichen, verwenden Sie die Methode Worksheet.getSlicers().remove(). Der Slicer wird aus dem Arbeitsblatt entfernt.

Das folgende Code-Snippet lädt die[Beispiel-Excel-Datei](106364970.xlsx)die einen vorhandenen Slicer enthält. Es greift auf den Slicer zu, entfernt ihn und speichert die[Excel-Datei ausgeben](106364971.xlsx). Der folgende Screenshot zeigt den Slicer, der entfernt wird.
