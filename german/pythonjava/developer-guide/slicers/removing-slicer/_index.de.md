## **Slicer entfernen**
Um einen Slicer in Microsoft Excel zu entfernen, wählen Sie einfach den Slicer aus und drücken die *Löschen*-Taste. Um dies mit Aspose.Cells für Python via Java zu erreichen, verwenden Sie die Methode Worksheet.getSlicers().remove(). Sie entfernt den Slicer aus dem Arbeitsblatt. 

Der folgende Code-Schnipsel lädt die [Beispiel-Excel-Datei](106364970.xlsx) mit einem vorhandenen Slicer. Er greift auf den Slicer zu, entfernt ihn und speichert die [Ausgabe-Excel-Datei](106364971.xlsx). Der folgende Screenshot zeigt den Slicer, der entfernt wird.
