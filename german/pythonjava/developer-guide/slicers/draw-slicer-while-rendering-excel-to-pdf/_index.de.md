## **Slicer zeichnen beim Rendern von Excel zu PDF**
Aspose.Cells für Python via Java unterstützt das Exportieren von Excel zu PDF mit Slicer-Einstellungen. Die API unterstützt dies standardmäßig, Sie exportieren einfach die Excel-Datei mit Slicer in PDF, das generierte PDF zeigt den angewendeten Slicer. Der folgende Code-Schnipsel lädt die [Beispiel-Excel-Datei](106365058.xlsx) mit einem Slicer und speichert die Arbeitsmappe als [Ausgabe-PDF-Datei](106365052.xlsx). Der folgende Screenshot vergleicht die Quell-Excel-Datei und die generierte PDF-Datei.
