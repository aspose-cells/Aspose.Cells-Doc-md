## **Slicer zeichnen, während Excel in PDF gerendert wird**
Aspose.Cells for Python via Java unterstützt den Export von Excel in PDF mit Slicer-Einstellungen. Der API unterstützt dies standardmäßig, Sie exportieren einfach die Excel-Datei mit Slicer in PDF, das generierte PDF zeigt den angewendeten Slicer.
