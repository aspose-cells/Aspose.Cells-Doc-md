## **Exportieren Sie Kommentare, während Sie eine Excel-Datei in HTML speichern**
Wenn Excel in HTML konvertiert wird, werden Kommentare nicht exportiert. Aspose.Cells for Python via Java bietet die Funktion zum Exportieren von Kommentaren während der Konvertierung von Excel in HTML. Um dies zu erreichen, bietet die API die[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments)Eigentum. Festlegen des Werts von[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments) Eigentum zu**WAHR** exportiert Kommentare in das Ausgabe-HTML.
