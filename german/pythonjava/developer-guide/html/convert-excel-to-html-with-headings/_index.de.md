## **Konvertieren Sie Excel in HTML mit Überschriften**
Aspose.Cells bietet die Option, Zeilen- und Spaltenüberschriften zu exportieren, während Excel in HTML konvertiert wird. Dies kann durch die Verwendung erreicht werden[HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings)Eigenschaft, die von API bereitgestellt wird. Der Standardwert von[HtmlSaveOptions.ExportHeadings](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportHeadings) ist**FALSCH** . Passieren**WAHR** als Parameter zum Rendern von Überschriften in der ausgegebenen HTML-Datei. Das folgende Bild zeigt die vom folgenden Code generierte Ausgabedatei.
