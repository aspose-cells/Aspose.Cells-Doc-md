## **Exportieren Sie den Druckbereichsbereich nach HTML**
Aspose.Cells for Python via Java unterstützt das Exportieren nur des Druckbereichs also ausgewählter Zellenbereiche statt des gesamten siest auf HTML. Dafür bietet die API die[HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) Eigentum. Setzen Sie diese Eigenschaft auf**Wahr** exportiert nur den Druckbereich.
