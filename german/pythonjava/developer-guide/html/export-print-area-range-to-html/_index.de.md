## **Druckbereichsbereich nach HTML exportieren**
Aspose.Cells for Python via Java unterstützt das Exportieren nur des Druckbereichs, dh ausgewählter Zellbereiche, anstelle des gesamten Zellbereichs nach HTML. Dafür sorgt die API[HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) Eigentum. Setzen Sie diese Eigenschaft auf**WAHR** exportiert nur den Druckbereich.
