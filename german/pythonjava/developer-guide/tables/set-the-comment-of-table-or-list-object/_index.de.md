## **Den Kommentar der Tabelle oder des Listenobjekts innerhalb des Arbeitsblatts festlegen**
Aspose.Cells für Python via Java unterstützt das Hinzufügen des Kommentars des Listenobjekts. Hierfür bietet die API die [ListObject.Comment](https://reference.aspose.com/cells/python/asposecells.api/listobject#Comment) Eigenschaft. Der vom [ListObject.Comment](https://reference.aspose.com/cells/python/asposecells.api/listobject#Comment) Eigenschaft hinzugefügte Kommentar ist in der Datei *xl/tables/tableName.xml* sichtbar.
