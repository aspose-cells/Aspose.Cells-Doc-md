## **Text in Spalten konvertieren**
Sie können Ihren Text in Spalten mithilfe von Microsoft Excel konvertieren. Dieses Feature ist unter *Daten Tools* im *Daten*-Tab verfügbar. Um den Inhalt einer Spalte in mehrere Spalten aufzuteilen, sollte die Daten einen bestimmten Begrenzer enthalten, wie beispielsweise ein Komma (oder ein anderes Zeichen), anhand dessen Microsoft Excel den Inhalt einer Zelle in mehrere Zellen aufteilt. Aspose.Cells bietet dieses Feature auch über die Methode [TextToColumns](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)). Der folgende Codeausschnitt zeigt die Verwendung der Methode [TextToColumns](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) durch Konvertieren von Text in Spalten mit Leerzeichen als Trennzeichen.
