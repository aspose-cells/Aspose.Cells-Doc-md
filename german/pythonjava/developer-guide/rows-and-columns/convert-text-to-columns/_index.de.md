## **Text in Spalten umwandeln**
Sie können Ihren Text mit Microsoft Excel in Spalten umwandeln. Diese Funktion ist ab verfügbar*Datenwerkzeuge*unter dem*Daten*Tab. Um den Inhalt einer Spalte auf mehrere Spalten aufzuteilen, sollten die Daten ein bestimmtes Trennzeichen wie ein Komma (oder ein beliebiges anderes Zeichen) enthalten, auf dessen Grundlage Microsoft Excel den Inhalt einer Zelle auf mehrere Zellen aufteilt. Aspose.Cells bietet diese Funktion auch über die[TextZuSpalten](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) Methode. Das folgende Code-Snippet demonstriert die Verwendung von[TextZuSpalten](https://reference.aspose.com/cells/python/asposecells.api/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\))-Methode durch Konvertieren von Text in Spalten mit Leerzeichen als Trennzeichen.
