{{% alert color="primary" %}}

Aspose.Cells for Python via Java bietet die [**Shape.getGeometry().getShapeAdjustValues()**](https://reference.aspose.com/cells/python-java/asposecells.api/geometry#ShapeAdjustValues)-Methode, um Änderungen an den Anpassungspunkten mit Shapes vorzunehmen. Im Microsoft Excel UI werden Anpassungen als gelbe Diamantenknoten angezeigt. Zum Beispiel:

- Das abgerundete Rechteck hat eine Anpassung, um den Bogen zu ändern
- Das Dreieck hat eine Anpassung, um die Position des Punktes zu ändern
- Das Trapezoid hat eine Anpassung, um die Breite des oberen Teils zu ändern
- Pfeile haben zwei Anpassungen, um die Form des Kopfes und des Endes zu ändern

Dieser Artikel erklärt die Verwendung der [**Shape.getGeometry().getShapeAdjustValues()**](https://reference.aspose.com/cells/python-java/asposecells.api/geometry#ShapeAdjustValues)-Methode zur Änderung des Anpassungswerts verschiedener Formen.
