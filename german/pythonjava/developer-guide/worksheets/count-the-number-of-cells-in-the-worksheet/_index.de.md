## **Zählen Sie die Anzahl der Zellen im Arbeitsblatt**
Aspose.Cells for Python via Java bietet die Möglichkeit, die Anzahl der Zellen im Arbeitsblatt zu zählen. Dafür sorgt die API[Cells.getCount](https://reference.aspose.com/cells/python/asposecells.api/cells#Count)() oder[Cells.getCountLarge](https://reference.aspose.com/cells/python/asposecells.api/cells#CountLarge)() Eigenschaften.
