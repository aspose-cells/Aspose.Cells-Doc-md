## **Zählen Sie die Anzahl der Zellen im Arbeitsblatt**
Aspose.Cells für Python via Java bietet die Möglichkeit, die Anzahl der Zellen im Arbeitsblatt zu zählen. Hierfür stellt die API die [Cells.getCount](https://reference.aspose.com/cells/python/asposecells.api/cells#Count)() oder [Cells.getCountLarge](https://reference.aspose.com/cells/python/asposecells.api/cells#CountLarge)() Eigenschaften bereit.

Der folgende Codeausschnitt zeigt, wie die [Cells.getCount](https://reference.aspose.com/cells/python/asposecells.api/cells#Count)() oder [Cells.getCountLarge](https://reference.aspose.com/cells/python/asposecells.api/cells#CountLarge)() Eigenschaften verwendet werden, um die Anzahl der Zellen in einem Arbeitsblatt zu erhalten.
