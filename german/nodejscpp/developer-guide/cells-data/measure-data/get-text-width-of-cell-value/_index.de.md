## **Breite des Zellenwerts abrufen**

Manchmal müssen Entwickler die Breite des Zellwerts für die Layoutplanung berechnen. Aspose.Cells for Node.js via C++ bietet die [**CellsHelper.getTextWidth(string, Font, number)**](https://reference.aspose.com/cells/nodejs-cpp/cellshelper/#getTextWidth-string-font-number-) Methode, mit der Entwickler die Textbreite des Zellwerts ermitteln können. Das folgende Beispiel zeigt, wie [**CellsHelper.getTextWidth(string, Font, number)**](https://reference.aspose.com/cells/nodejs-cpp/cellshelper/#getTextWidth-string-font-number-) genutzt wird, um auf die Textbreite des Zellwerts zuzugreifen.
