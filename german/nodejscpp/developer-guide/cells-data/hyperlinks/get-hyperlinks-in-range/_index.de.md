## **Hyperlinks im Bereich abrufen**

 Die [**Range**](https://reference.aspose.com/cells/nodejs-cpp/range) Klasse bietet eine [**getHyperlinks()**](https://reference.aspose.com/cells/nodejs-cpp/range/#getHyperlinks--) Methode, die alle Hyperlinks im ausgewählten Bereich zurückgibt. Sie können den Hyperlink auch löschen, indem Sie die [**Hyperlink.delete()**](https://reference.aspose.com/cells/nodejs-cpp/hyperlink/#delete--) Methode aufrufen.
