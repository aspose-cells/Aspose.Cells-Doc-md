{{% alert color="primary" %}}

Manchmal müssen Sie Zellen mit einem bestimmten Stil finden. Sie können dazu Aspose.Cells for Node.js via C++ verwenden, um alle Zellen mit einem gemeinsamen Stil zu finden. Aspose.Cells bietet die [**FindOptions.setStyle(Style)**](https://reference.aspose.com/cells/nodejs-cpp/findoptions/#setStyle-style-) Methode, mit der Sie den zu suchenden Stil angeben können.

{{% /alert %}}

Der Code in diesem Beispiel sucht alle Zellen, die denselben Stil wie die Zelle A1 haben. Nachdem der Code ausgeführt wurde, enthalten alle Zellen, die denselben Stil wie A1 haben, den Text "Gefunden".
