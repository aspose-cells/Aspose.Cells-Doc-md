## **Mögliche Verwendungsszenarien**

Aspose.Cells for Node.js via C++ unterstützt das benutzerdefinierte Musterformat *DBNum*. Wenn zum Beispiel Ihr Zellenwert 123 ist und Sie das benutzerdefinierte Format [DBNum2][$-804]General angeben, wird es als 壹佰贰拾叁 angezeigt. Sie können die benutzerdefinierte Formatierung der Zelle mit den Methoden [**Cell.getStyle()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getStyle--) und [**Style.setCustom(string)**](https://reference.aspose.com/cells/nodejs-cpp/style/#setCustom-string-) festlegen.
