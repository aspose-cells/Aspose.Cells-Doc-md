## **Option für Berichtsfilterseiten anzeigen**
Excel unterstützt das Erstellen von Pivot-Tabellen, das Hinzufügen von Berichtfiltern und das Aktivieren der "Berichtfilterseiten anzeigen"-Option. Aspose.Cells for Node.js via C++ unterstützt diese Funktion ebenfalls, um die "Berichtfilterseiten anzeigen"-Option bei der erstellten Pivot-Tabelle zu aktivieren. Nachfolgend ist der Bildschirm gezeigt, der die "Berichtfilterseiten anzeigen"-Option in Excel zeigt.
