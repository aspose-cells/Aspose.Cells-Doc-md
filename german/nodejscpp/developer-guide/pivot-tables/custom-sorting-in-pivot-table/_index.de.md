## **Wie man eine benutzerdefinierte Sortierung in Pivot-Tabellen mit Aspose.Cells for Node.js via C++ festlegt**
Mit der Aspose.Cells for Node.js via C++ API können Sie Pivot-Tabellen nach Feldwerten sortieren. Der folgende Code lädt die Beispiel-Excel-Datei und fügt drei Pivot-Tabellen hinzu. Die erste Pivot-Tabelle hat keine benutzerdefinierte Sortierung, die zweite ist nach Zeilenwerten „SeaFood“ sortiert, und die dritte ist nach Spaltenwerten „28/07/2000“ sortiert.
