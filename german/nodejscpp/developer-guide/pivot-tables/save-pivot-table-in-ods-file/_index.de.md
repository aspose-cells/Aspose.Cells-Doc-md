Aspose.Cells for Node.js via C++ bietet die Möglichkeit, Pivot-Tabellen in ODS-Dateien zu speichern. Hierzu konvertieren Sie einfach eine Arbeitsmappe mit vorhandener Pivot-Tabelle oder erstellen eine neue Pivot-Tabelle und speichern die Datei im ODS-Format. Stellen Sie sicher, dass Sie [**PivotTable.calculateData**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#calculatedata--) vor dem Speichern aufrufen, um sicherzustellen, dass die Pivot-Tabelle im Ausgabedokument gerendert wird. Der folgende Codeausschnitt zeigt, wie man eine Pivot-Tabelle in einer ODS-Datei speichert.
