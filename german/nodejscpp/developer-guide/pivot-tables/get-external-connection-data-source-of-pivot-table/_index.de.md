## **Wie man die externe Verbindungsdatenquelle der Pivot-Tabelle abruft**

Aspose.Cells for Node.js via C++ ermöglicht es, die externe Verbindungsdatenquelle der Pivot-Tabelle zu erhalten. Hierfür bietet die API die [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--)-Methode der [**PivotTable**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/)-Klasse. Die [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--)-Methode gibt das [**ExternalConnection**](https://reference.aspose.com/cells/nodejs-cpp/externalconnection/)-Objekt zurück. Der folgende Codeausschnitt zeigt, wie die [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--)-Methode verwendet wird, um die externe Verbindungsdatenquelle der Pivot-Tabelle zu erhalten.
