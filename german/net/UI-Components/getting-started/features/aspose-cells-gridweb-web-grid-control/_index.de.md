## **Aspose.Cells.GridWeb – Web Grid Control**
### **Import-/Exportfunktionen**
- Import aus und Export in Excel-Tabellen.
- Importieren und exportieren Sie Daten direkt von und zu DataView- oder DataTable-Objekten.
### **Arbeitsblattfunktionen**
- Arbeitsblätter hinzufügen/entfernen.
- Unterstützt sowohl die horizontale als auch die vertikale Sortierung von Arbeitsblattdaten.
- Unterstützt die Paginierung von Arbeitsblättern.
- Tabellen von Grund auf neu erstellen.
### **Zeilen- und Spaltenfunktionen**
- Hinzufügen, Einfügen und Entfernen von Zeilen und Spalten in einem Arbeitsblatt.
- Zeilen und Spalten ein- und ausblenden.
- Einfrieren und Auftauen von Zeilen und Spalten.
- Passen Sie die Kopfzeilen von Zeilen und Spalten an.
### **Cell Eigenschaften**
- Bearbeiten Sie mehrere Zellen auf der Clientseite mit einem einzigen Beitrag zurück zur Serverseite.
- Unterstützt Zahlen-, Datums- und Zeitformateinstellungen.
- Unterstützt benutzerdefinierte Formateinstellungen
- Steuern Sie die Textausrichtung.
- Grenzen verwalten.
- Steuern Sie die Schriftarteinstellungen.
- Zellen verbinden oder teilen
### **Formatierungsfunktionen**
- Erstellen Sie webbasierte interaktive Grid-Anwendungen, die wie Desktop-Tabellenkalkulationsanwendungen aussehen und sich anfühlen.
- Unterstützt die clientseitige Zellformatierung.
- Bietet sowohl integrierte als auch anpassbare visuelle Stile.
### **Utility-Funktionen**
- Unterstützt die Validierung von Dropdown-Listen und regulären Ausdrücken.
- Kommentare hinzufügen und entfernen.
- Bilder einfügen und entfernen.
- Hyperlinks hinzufügen und entfernen.
- Tastaturnavigation.
### **Erweiterte Funktionen**
- Unterstützt das Erstellen und Berechnen von Formeln auf Webseiten.
- Bietet über 80 vordefinierte Funktionen in Bezug auf Mathematik, Text und Daten, Datum und Uhrzeit sowie Logik.
- Unterstützt das Erstellen dynamischer Pivot-Tabellen.
- Führen Sie Aktionen unter verschiedenen Grid-Ereignissen durch.
- Verwenden Sie das Aspose.Cells.GridWeb-Steuerelement im sitzungslosen Modus.
- Senden Sie Rasterdaten mit einem einzigen Klick.
