{{% alert color="primary" %}} 

In diesem Thema wird die Festlegung der Spaltenbreite und der Zeilenhöhe mithilfe der Aspose.Cells.GridDesktop API diskutiert. Manchmal enthalten Arbeitsblattzellen Werte, die breiter sind als die ursprüngliche Zelle. Dies erschwert es Benutzern, den Zellinhalt zu lesen. Einige Benutzer möchten die rechte Pfeiltaste nicht verwenden, um den vollständigen Wert in einer Zelle anzuzeigen. Um solche Probleme zu lösen, können Entwickler die Spaltenbreite ändern. Durch Ändern der Zeilenhöhe werden ähnliche Probleme gelöst.

{{% /alert %}} 
## **Wie man die Spaltenbreite festlegt**
Um die Breite einer Spalte mithilfe von Aspose.Cells.GridDesktop festzulegen:

1. Greifen Sie auf ein Arbeitsblatt zu.
1. Greifen Sie auf die Spalte zu, für die Sie die Breite ändern möchten.
1. Legen Sie die Spaltenbreite fest.

## **Wie man die Zeilenhöhe festlegt**
Um die Höhe einer Zeile mit Aspose.Cells.GridDesktop festzulegen, befolgen Sie bitte die folgenden Schritte:

1. Greifen Sie auf ein beliebiges Arbeitsblatt zu.
1. Greifen Sie auf die Zeile zu, deren Höhe Sie ändern möchten.
1. Setzen Sie die Zeilenhöhe.
