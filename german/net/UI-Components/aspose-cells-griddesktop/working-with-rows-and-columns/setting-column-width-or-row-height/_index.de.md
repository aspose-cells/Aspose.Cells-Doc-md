{{% alert color="primary" %}} 

In diesem Thema wird das Festlegen der Spaltenbreite und Zeilenhöhe mithilfe von Aspose.Cells.GridDesktop API erläutert. Manchmal enthalten Arbeitsblattzellen Werte, die breiter als die ursprüngliche Zelle sind. Dadurch wird es für Benutzer schwierig, den Zellinhalt zu lesen. Einige Benutzer möchten nicht die rechte Pfeiltaste verwenden, um den vollständigen Wert in einer Zelle anzuzeigen. Um solche Probleme zu lösen, können Entwickler die Spaltenbreite ändern. Durch Ändern der Zeilenhöhe werden ähnliche Probleme gelöst.

{{% /alert %}} 
##  **So legen Sie die Spaltenbreite fest**
So legen Sie die Breite einer Spalte mit Aspose.Cells.GridDesktop fest:

1. Greifen Sie auf ein Arbeitsblatt zu.
1. Greifen Sie auf die Spalte zu, deren Breite Sie ändern möchten.
1. Legen Sie die Spaltenbreite fest.

##  **So legen Sie die Zeilenhöhe fest**
Um die Höhe einer Zeile mit Aspose.Cells.GridDesktop festzulegen, befolgen Sie bitte die folgenden Schritte:

1. Greifen Sie auf jedes gewünschte Arbeitsblatt zu.
1. Greifen Sie auf die Zeile zu, deren Höhe Sie ändern möchten.
1. Legen Sie die Zeilenhöhe fest.
