{{% alert color="primary" %}} 

Dieses Thema behandelt die Informationen zum Einfrieren und Auftauen von Zeilen und Spalten eines Arbeitsblatts. Das Einfrieren von Spalten oder Zeilen ermöglicht es Benutzern, die Spaltenüberschriften oder Zeilentitel sichtbar zu halten, während sie zu anderen Teilen des Arbeitsblatts scrollen. Diese Funktion ist sehr hilfreich beim Scrollen durch Arbeitsblätter, die große Datenmengen enthalten. Denn in solchen Fällen, wenn Benutzer Daten scrollen, werden nur Daten nach unten gescrollt und Überschriften bleiben dort, um die Lesbarkeit der Daten zu erhöhen.

{{% /alert %}} 
## **Einfrieren von Spalten**
Um Spalten eines Arbeitsblatts mit Aspose.Cells.GridDesktop einzufrieren, befolgen Sie bitte die folgenden Schritte:

- Greifen Sie auf jedes gewünschte **Arbeitsblatt** zu
- Setzen Sie die Anzahl von **eingefrorenen Spalten** im **Arbeitsblatt**

## **Freigeben von Spalten**
Um Spalten eines Arbeitsblatts mit Aspose.Cells.GridDesktop freizugeben, befolgen Sie bitte die folgenden Schritte:

- Greifen Sie auf jedes gewünschte **Arbeitsblatt** zu
- Setzen Sie die Anzahl von **eingefrorenen Spalten** im **Arbeitsblatt** auf Null (0). Damit wird der Effekt von nicht eingefrorenen Spalten erzeugt

## **Einfrieren von Zeilen**
Um Zeilen eines Arbeitsblatts mit Aspose.Cells.GridDesktop einzufrieren, befolgen Sie bitte die folgenden Schritte:

- Greifen Sie auf jedes gewünschte **Arbeitsblatt** zu
- Legen Sie die Anzahl der **eingefrorenen Zeilen** im **Arbeitsblatt** fest

## **Zeilen einfrieren aufheben**
Um Zeilen eines Arbeitsblatts mit Aspose.Cells.GridDesktop zu entfrieren, befolgen Sie bitte die folgenden Schritte:

- Greifen Sie auf jedes gewünschte **Arbeitsblatt** zu
- Legen Sie die Anzahl der **eingefrorenen Zeilen** im **Arbeitsblatt** auf Null (0) fest. Dadurch werden die Zeilen entfroren
