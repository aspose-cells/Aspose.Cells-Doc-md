In diesem Thema wird erläutert, wie Sie den Namen eines Arbeitsblatts mithilfe von Aspose.Cells.GridDesktop ändern.

{{% /alert %}} 
## **Beispiel**
Das Umbenennen eines Arbeitsblatts ist eine einfache Aufgabe, die sehr nützlich sein kann. Zum Beispiel, wenn Sie Arbeitsblätter so beschriften möchten, dass deutlich wird, was sie enthalten. Beispielsweise kann ein Modell 12 Arbeitsblätter haben, die jeweils nach Monat benannt sind, um Ausgabeninformationen zu enthalten.

So benennen Sie ein Arbeitsblatt mit dem Aspose.Cells.GridDesktop-Steuerelement um:

1. Fügen Sie einem Formular ein Aspose.Cells.GridDesktop-Steuerelement hinzu.
1. Rufen Sie die Referenz eines gewünschten Arbeitsblatts ab.
1. Legen Sie den Namen des Arbeitsblatts mit seiner Referenz fest.

{{% alert color="primary" %}} 

 Bevor Sie den Namen des Arbeitsblatts ändern, müssen Sie auf die Referenz dieses Arbeitsblatts zugreifen. Es gibt viele Möglichkeiten, auf eine Arbeitsblattreferenz zuzugreifen. Um mehr über diese Möglichkeiten zu erfahren, beziehen Sie sich bitte auf[Zugriff auf Arbeitsblätter](/cells/de/net/accessing-worksheet/).
