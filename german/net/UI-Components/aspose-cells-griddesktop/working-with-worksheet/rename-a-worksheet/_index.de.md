In diesem Artikel wird erläutert, wie Sie den Namen eines Arbeitsblatts mithilfe von Aspose.Cells.GridDesktop ändern.

{{% /alert %}} 
## **Beispiel**
Das Umbenennen eines Arbeitsblatts ist eine einfache Aufgabe, die sehr nützlich sein kann. Wenn Sie beispielsweise Arbeitsblätter beschriften möchten, damit auf einen Blick erkennbar ist, was jedes einzelne enthält. Ein Modell könnte beispielsweise 12 Arbeitsblätter haben, die jeweils nach Monaten benannt sind, um Ausgabeninformationen zu speichern.

Um ein Arbeitsblatt unter Verwendung der Aspose.Cells.GridDesktop-Steuerung umzubenennen:

1. Fügen Sie eine Aspose.Cells.GridDesktop-Steuerung zu einem Formular hinzu.
1. Holen Sie sich die Referenz eines gewünschten Arbeitsblatts.
1. Setzen Sie den Namen des Arbeitsblatts unter Verwendung seines Verweises.

{{% alert color="primary" %}} 

Bevor Sie den Namen des Arbeitsblatts ändern, ist es erforderlich, den Verweis auf dieses Arbeitsblatt abzurufen. Es gibt viele Möglichkeiten, auf einen Arbeitsblattverweis zuzugreifen. Um mehr über diese Möglichkeiten zu erfahren, sehen Sie bitte unter [Arbeitsblätter abrufen](/cells/de/net/aspose-cells-griddesktop/accessing-worksheet/) nach.
