{{% alert color="primary" %}} 

Um Entwicklern das Upgrade zu erleichtern, pflegen wir ein Dokument, in dem wichtige Informationen für bestehende Benutzer beschrieben werden, insbesondere für diejenigen, die den älteren Aspose.Grid.Desktop verwendet haben und nun auf den neuen oder zusammengeführten Aspose.Cells.GridDesktop aktualisieren müssen.

Dieses Thema enthält eine grundlegende Schritt-für-Schritt-Anleitung für Anfänger, die ihnen beim Upgrade von Aspose.Grid.Desktop auf Aspose.Cells.GridDesktop-Steuerelement und beim Erstellen und Verwenden des Aspose.Cells.GridDesktop-Steuerelements in Windows-Anwendungen hilft.

{{% /alert %}} 
## **Aktualisieren Sie von Aspose.Grid.Desktop auf Aspose.Cells.GridDesktop Control**
Entwickler, die mit Aspose.Grid.Desktop gearbeitet haben, können bei der Verwendung des neuen Aspose.Cells.GridDesktop auf Probleme stoßen, wenn sie darauf upgraden. Im Folgenden ist der Namespace aufgeführt, der umbenannt wird:

1. **Aspose.Grid.Desktop-Namespace wurde in Aspose.Cells.GridDesktop-Namespace umbenannt.**
## **Erstellen und Verwenden von Aspose.Cells.GridDesktop Control**
### **Erstellen eines Windows-Anwendungsprojekts**
Erstellen Sie zunächst ein Windows-Anwendungsprojekt, in dem das Aspose.Cells.GridDesktop-Steuerelement verwendet werden soll. Befolgen Sie die folgenden Schritte;

1. **Öffnen Sie die Visual Studio.NET-IDE.**
1. **Wählen Sie im Menü Datei die Option Neu, das Projekt.**
1. **Wählen Sie Windows-Anwendung im Dialogfeld „Neues Projekt“ aus.**

Nachdem Sie die obigen Schritte ausgeführt haben, wird eine Fensteranwendung mit leerem Formular erstellt.
### **Hinzufügen des GridDesktop-Steuerelements zum Formular**
 Ziehen Sie das Steuerelement Aspose.Cells.GridDesktop per Drag & Drop aus der Toolbox in das Formular. Danach erscheint auf dem Formular ein Steuerelement, mit dem Sie verschiedene Aktionen ausführen können, z**Größe ändern**, **seine Eigenschaften ändern**, **etc**.
### **Führen Sie die Anwendung aus**
 Führen Sie schließlich die Anwendung aus, indem Sie auf drücken**Strg+F5** oder klicken Sie auf die**Start** Knopf. Wenn die Anwendung ausgeführt wird, sehen wir ein Formular, das ein leeres Aspose.Cells.GridDesktop-Steuerelement enthält. Jetzt können Sie Zellen Werte hinzufügen, indem Sie darauf klicken. Der folgende Screenshot zeigt, wie Ihre Anwendung letztendlich aussehen wird.

![todo: Bild_alt_Text](how-to-use-aspose-cells-griddesktop_1.png)

Es ist nicht nur möglich, dem Steuerelement Werte hinzuzufügen, sondern auch andere Aufgaben auszuführen, wie z. B. das Ändern der Höhe einer Zeile oder der Breite einer Spalte, das Kopieren (Strg+C) oder Ausschneiden (Strg+X) von Zelldaten in die Zwischenablage und Einfügen (Strg+V) von Daten in eine Zelle usw. Um weitere Vorgänge auszuführen, klicken Sie mit der rechten Maustaste auf das Steuerelement, um sein Kontextmenü anzuzeigen.

{{% alert color="primary" %}} 

Aspose.Grid.Desktop wurde umbenannt und ist Teil von Aspose.Cells geworden, sodass es nicht mehr möglich ist, ältere Versionen von Aspose.Grid.Desktop zu verwenden. Nur um es zu betonen, es ist nicht viel Arbeit mit dem Upgrade auf die neueste Version der Aspose.Cells.GridDesktop-Komponente verbunden. An der API ändert sich nichts, da die Klassen mit den Membern, Structs, Enumerations etc. gleich bleiben. Die einzige Änderung, die vorgenommen wurde, betrifft den Namensraum des Steuerelements.
