## **Mögliche Nutzungsszenarien**
 Manchmal ist es wichtig, die GridDesktop-Version zur Laufzeit zu finden. Beispielsweise möchten Sie sicherstellen, dass Sie die richtige Version verwenden, oder Sie debuggen Ihre Anwendung und möchten wissen, welche Version von GridDesktop derzeit verwendet wird. GridDesktop stellt die Methode Aspose.Cells.GridDesktop.GridDesktop.GetVersion() bereit, die die GridDesktop-Version zur Laufzeit zurückgibt.
