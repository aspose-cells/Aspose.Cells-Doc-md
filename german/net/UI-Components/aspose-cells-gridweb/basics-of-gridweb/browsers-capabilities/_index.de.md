{{% alert color="primary" %}} 

Aspose.Cells.GridWeb ist ein GUI-basiertes Websteuerelement, das in ASP.NET-Webformulare wie andere von Microsoft .NET Framework bereitgestellte Websteuerelemente eingebettet werden kann. Das Wichtigste bei einem Websteuerelement ist die Unterstützung mehrerer Browser. Aspose.Cells.GridWeb bietet eine plattformübergreifende Browserunterstützung.

{{% /alert %}} 
## **Vergleich**
Aspose.Cells.GridWeb wurde unter Verwendung des Microsoft .NET Frameworks erstellt, daher wird es vollständig vom Microsoft Internet Explorer (IE) unterstützt. Auf anderen Browsern gibt es jedoch geringfügige Einschränkungen. Dieses Thema bietet einen detaillierten Vergleich, welche Funktionen auf verschiedenen Browsern unterstützt werden.
