{{% alert color="primary" %}} 

Aspose.Cells.GridWeb ist ein GUI-basiertes Websteuerelement, das in ASP.NET-Webformulare eingebettet werden kann, wie andere Websteuerelemente, die vom Framework Microsoft .NET bereitgestellt werden. Das Wichtigste an einem Websteuerelement ist die Bereitstellung von Cross-Browser-Unterstützung. Aspose.Cells. GridWeb bietet browserübergreifende Unterstützung.

{{% /alert %}} 
## **Vergleich**
Aspose.Cells.GridWeb wird mit dem Microsoft .NET Framework erstellt, sodass es vom Internet Explorer (IE) von Microsoft vollständig unterstützt wird. Bei anderen Browsern gibt es jedoch geringfügige Einschränkungen. Dieses Thema enthält einen detaillierten Vergleich, welche Funktionen von verschiedenen Browsern unterstützt werden.
