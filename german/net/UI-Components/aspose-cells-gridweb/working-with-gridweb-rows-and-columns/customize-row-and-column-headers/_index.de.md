{{% alert color="primary" %}} 

Ähnlich wie Microsoft Excel verwendet auch Aspose.Cells.GridWeb normale Überschriften oder Beschriftungen für Zeilen (Zahlen wie 1, 2, 3 und so weiter) und Spalten (alphabetisch wie A, B, C und so weiter). Mit Aspose.Cells.GridWeb ist es auch möglich, Beschriftungen anzupassen. Dieser Artikel behandelt die Anpassung von Zeilen- und Spaltenüberschriften zur Laufzeit mithilfe der Aspose.Cells.GridWeb-API.

{{% /alert %}} 
## **Anpassen der Zeilenüberschrift**
Um die Überschrift oder Beschriftung einer Zeile anzupassen:

1. Fügen Sie der Webformularsteuerung Aspose.Cells.GridWeb hinzu.
1. Greifen Sie auf das Arbeitsblatt in der GridWorksheetCollection zu.
1. Legen Sie die Beschriftung einer bestimmten Zeile fest.
