{{% alert color="primary" %}} 

Das Umbenennen eines Arbeitsblatts kann sehr nützlich sein, wenn man mit vielen Arbeitsblättern in Aspose.Cells.GridWeb arbeitet und sich dazu entscheidet, ihre Namen zu ändern, um sie aussagekräftiger zu machen. Zum Beispiel kann ein Arbeitsblatt, das eine Rechnung enthält, in Rechnung umbenannt werden, anstelle von Blatt1. Dieser Abschnitt beschreibt dieses einfache, aber nützliche Feature.

{{% /alert %}} 
## **Arbeitsblatt umbenennen**
Alle Arbeitsblätter enthalten eine Name-Eigenschaft, die es Entwicklern ermöglicht, auf Arbeitsblattnamen zuzugreifen oder diese zu ändern. Um ein Arbeitsblatt umzubenennen:

1. Greifen Sie auf ein Arbeitsblatt aus der GridWorksheetCollection zu.
1. Benennen Sie das ausgewählte Arbeitsblatt um.



{{% alert color="primary" %}} 

Für weitere Details zum Zugriff auf Arbeitsblätter in Aspose.Cells.GridWeb lesen Sie bitte [Arbeitsblätter abrufen](/cells/de/net/aspose-cells-gridweb/access-worksheets/).

{{% /alert %}}
