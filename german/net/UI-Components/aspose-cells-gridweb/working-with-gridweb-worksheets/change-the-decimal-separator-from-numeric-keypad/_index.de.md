## **Mögliche Verwendungsszenarien**
Standardmäßig zeigt Aspose.Cells.GridWeb numerische Daten entsprechend den lokal/regionalen Einstellungen auf dem Rechner an. Sie können das Dezimaltrennzeichen von der numerischen Tastatur aus programmgesteuert über die Aspose.Cells.GridWeb-API ändern. Wenn also eine Datei in die GridWeb-Matrix importiert wird oder Sie numerische Daten (von der numerischen Tastatur) in eine neue Arbeitsblattzelle eingeben, sollte das gewünschte Dezimaltrennzeichen visuell festgelegt sein. 
## **Ändern Sie das Dezimaltrennzeichen von der numerischen Tastatur aus**
Durch die Eigenschaft **GridWorksheetCollection.NumberDecimalSeparator** können Sie das Dezimaltrennzeichen von der numerischen Tastatur aus programmgesteuert ändern. Bitte beachten Sie die Screenshots, die zeigen, wie es funktioniert.

{{% alert color="primary" %}} 

Bitte beachten Sie, dass die Änderung des Dezimaltrennzeichens nur für die visuelle Erfahrung der Benutzer in GridWeb gilt. Wenn Sie Ihre Arbeitsmappe bearbeiten und speichern, werden die numerischen Werte (in der Tabellenkalkulation) weiterhin gemäß Ihrem lokalen/regionalen Dezimaltrennzeichen gespeichert.
