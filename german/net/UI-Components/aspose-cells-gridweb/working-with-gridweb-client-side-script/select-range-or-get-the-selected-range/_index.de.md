{{% alert color="primary" %}} 

Sie können die folgenden clientseitigen Funktionen verwenden, um einen Bereich auszuwählen oder den ausgewählten Bereich in einem GridWeb mithilfe von JavaScript zu erhalten.

- getSelectRange()
- setSelectRange()
- clearSelections()

getSelectRange() gibt den zuletzt ausgewählten Bereich zurück. setSelectRange() wählt den angegebenen Bereich aus. clearSelections() löscht alle Auswahlen außer der aktuellen aktiven Zelle.
