{{% alert color="primary" %}} 

Wenn Sie das Kopieren und Einfügen von Zeilen in GridDesktop innerhalb der Steuerung oder zwischen der Steuerung und Excel aktivieren möchten, setzen Sie bitte die Eigenschaft GridDesktop.ClipboardCopyPaste auf true. Sie können diese Eigenschaft zur Entwurfszeit oder im Code festlegen. Der Standardwert dieser Eigenschaft ist false. Derzeit können nur Zellwerte kopiert und eingefügt werden, und es werden keine anderen Einstellungen der Zelle wie Format, Rahmenstil usw. kopiert.
