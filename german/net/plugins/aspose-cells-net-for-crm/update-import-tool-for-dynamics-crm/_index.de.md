## **Einführung**
Das Aspose Update Import Tool ist ein Open-Source-Add-On, das in Microsoft Dynamic CRM verwendet werden soll. Dieses Add-On ist in der Lage, Daten automatisch zu aktualisieren und/oder zu exportieren von CRM zu Aspose.Cells-Tabellenkalkulationen. Wir müssen nichts manuell tun, sondern dieses Add-On installieren und unsere Arbeit einfacher und schneller machen.

Dieses Add-On bietet die Funktionalität, automatisch von CRM zu aktualisieren und/oder zu exportieren. Ein Klick importiert/exportiert den Datensatz von/nach CRM.

Die Hauptfunktionen dieses Add-Ons sind:

- Stellen Sie die Verbindung zum gewünschten CRM her.
- Erlauben dem Benutzer, die gewünschte Entität und Felder auszuwählen.
- Den Datensatz aus CRM im Raster anzeigen.
- Exportieren des Datensatzes in Aspose.Cells-Tabellenkalkulation.
- Den Datensatz aus Aspose.Cells-Tabellenkalkulation im Raster anzeigen.
- Den Datensatz aus dem Raster in CRM exportieren.
## **Systemanforderungen und unterstützte Plattformen**
### **Systemanforderungen**
Um das Aspose .Net Update Import Tool für Microsoft Dynamics CRM zu installieren und zu verwenden, müssen Sie eine der folgenden CRM-Versionen installiert haben.

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.

Bitte zögern Sie nicht, uns zu kontaktieren, wenn Sie Probleme bei der Installation oder Verwendung dieses Add-Ons finden.
### **Unterstützte Plattformen**
Dieses Add-On funktioniert mit allen Versionen von Microsoft Dynamics CRM:

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.

Es handelt sich um ein benutzerdefiniertes Tool und erfordert keine Installation oder Deinstallation.
Sie müssen lediglich das Tool herunterladen und auf Ihrem Laufwerk extrahieren, und es ist einsatzbereit.
