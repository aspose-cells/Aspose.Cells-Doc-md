## **Einführung**
Das Aspose Update Import Tool ist ein Open-Source-Add-On zur Verwendung in Microsoft Dynamic CRM. Dieses Add-On ist in der Lage, Daten aus CRM automatisch in Aspose.Cells-Tabellen zu aktualisieren und/oder zu exportieren. Wir müssen nichts manuell tun und dieses Add-On installieren und unsere Arbeit einfacher und schneller machen.

Dieses Add-on bietet Funktionen zum automatischen Aktualisieren und/oder Exportieren aus CRM. Ein Klick importiert/exportiert den Datensatz in/aus CRM.

Hauptmerkmale dieses Addons sind:

- Stellen Sie die Verbindung zum gewünschten CRM her.
- Erlauben Sie dem Benutzer, die gewünschte Entität und Felder auszuwählen.
- Abrufen des Datensatzes aus der CRM-Anzeige im Raster.
- Exportieren Sie den Datensatz in die Tabelle Aspose.Cells.
- Holen Sie sich den Datensatz von Aspose.Cells Tabellenanzeige im Raster.
- Exportieren Sie den Datensatz aus dem Raster nach CRM.
## **Systemanforderungen und unterstützte Plattformen**
### **System Anforderungen**
Um das Aspose .Net Update Import Tool für Microsoft Dynamics CRM zu installieren und zu verwenden, muss eine der folgenden CRM-Versionen installiert sein

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.

Bitte zögern Sie nicht, uns zu kontaktieren, wenn Sie Probleme bei der Installation oder Verwendung dieses Add-ons feststellen.
### **Unterstützte Plattformen**
Dieses Addon funktioniert mit allen Versionen von Microsoft Dynamics CRM:

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.
