## **Einführung**
Das Export GridView To Excel Control ist ein ASP.NET-Serversteuerelement, das es ermöglicht, den Inhalt des GridView in Microsoft Excel oder OpenOffice-Tabellenkalkulationen mithilfe von [Aspose.Cells](https://products.aspose.com/cells/net/) zu exportieren. Es fügt die Schaltfläche **Export to Excel** oben auf dem GridView-Steuerelement hinzu. Wenn auf die Schaltfläche geklickt wird, wird der Inhalt des GridView-Steuerelements dynamisch in eine Microsoft Excel- oder OpenOffice-Tabelle exportiert und dann automatisch in weniger als ein paar Sekunden an den vom Benutzer ausgewählten Festplattenort heruntergeladen.
### **Modulfunktionen**
Diese erste Version des Steuerelements bietet die folgenden Funktionen:

- Holen Sie sich eine Offline-Kopie Ihres Online-GridView-Inhalts für Bearbeitung, Weitergabe und Druck.
- Geerbt vom Standard-ASP.NET-GridView-Steuerelement und hat daher alle seine Funktionen und Eigenschaften.
- Exportieren Sie GridView nach Xlsx, Xlsb, Xls, Txt, Csv, Ods.
- Funktioniert mit allen .NET-Versionen ab .NET 2.0.
- Möglichkeit zur Anpassung/Übersetzung des Texts der Export-Schaltfläche.
- Wenden Sie das Erscheinungsbild Ihres eigenen Themas auf die Export-Schaltfläche mit CSS an.
- Möglichkeit zum Hinzufügen einer benutzerdefinierten Überschrift oben auf dem exportierten Dokument
- Möglichkeit zum Speichern jedes exportierten Dokuments auf dem Server an einem konfigurierbaren Festplattenpfad
- Möglichkeit zum Exportieren der aktuellen Seite oder aller Seiten bei aktivierter Paginierung

## **Installation**
Es ist sehr einfach und unkompliziert, das Exportieren von GridView zu Excel Control zu installieren. Befolgen Sie bitte diese einfachen Schritte.
### **Für Visual Studio 2010, 2012 und 2013**
1. Entpacken Sie die heruntergeladene Zip-Datei
1. Doppelklicken Sie auf die VSIX-Datei Aspose.Excel.GridViewExport.vsix
1. Es wird ein Dialog angezeigt, der Ihnen die verfügbaren und unterstützten Visual Studio-Versionen auf Ihrem Computer anzeigt
1. Wählen Sie die Versionen aus, zu denen Sie die Export GridView To Excel Control hinzufügen möchten
1. Klicken Sie auf Installieren

Sie erhalten einen Erfolgsdialog, sobald die Installation abgeschlossen ist

**Hinweis:** Stellen Sie sicher, dass Sie Visual Studio neu starten, damit die Änderungen wirksam werden
### **Für Visual Studio 2005, 2008 und Express-Editionen**
Bitte befolgen Sie diese Schritte, um Export GridView To Excel Control in Visual Studio zu integrieren, um ein einfaches Drag & Drop wie bei anderen ASP.NET-Steuerungen zu ermöglichen

1. Entpacken Sie die heruntergeladene Zip-Datei
1. Stellen Sie sicher, dass Visual Studio als Administrator ausgeführt wird

Im Menü 'Extras' wählen Sie 'Werkzeugkastenelemente auswählen'

1. Klicken Sie auf Durchsuchen 
   Das Dialogfeld 'Öffnen' wird geöffnet
1. Navigieren Sie zum extrahierten Ordner und wählen Sie Aspose.Excel.GridViewExport.dll aus
1. Klicken Sie auf OK

Wenn Sie ein aspx- oder ascx-Steuerelement öffnen, sehen Sie ExportGridViewToWord unter dem allgemeinen Tab im Werkzeugkasten auf der linken Seite

### **ExportGridViewToExcel-Steuerung manuell hinzufügen**
Wenn Sie Probleme bei der Verwendung der obigen Methoden haben, die Visual Studio Toolbox verwenden, können Sie diese Steuerung manuell zu Ihrer ASP.NET-Anwendung hinzufügen, die auf einem beliebigen .NET-Framework größer als 2.0 läuft.

1. Falls Sie Visual Studio verwenden, stellen Sie sicher, dass Sie es als Administrator ausführen.
1. Fügen Sie eine Referenz zu **Aspose.Excel.GridViewExport.dll** hinzu, die im extrahierten Download-Paket in Ihrem ASP.NET-Projekt oder in Ihrer Webanwendung verfügbar ist. Stellen Sie sicher, dass Ihre Webanwendung/Visual Studio Vollzugriff auf dieses Verzeichnis hat, da andernfalls eine Zugriffsverweigerung auftreten kann.
1. Fügen Sie diese Zeile oben auf der Seite, Steuerung oder MasterPage hinzu. 

{{< highlight java >}}

 <%@ Register assembly="Aspose.Excel.GridViewExport" namespace="Aspose.Excel.GridViewExport" tagprefix="aspose" %>

{{< /highlight >}}

1. Fügen Sie das Folgende an der Stelle auf Ihrer ASP.NET-Seite, Steuerung oder MasterPage hinzu, an der Sie die Steuerung hinzufügen möchten. 

{{< highlight java >}}

 <aspose:ExportGridViewToExcel ID="ExportGridViewToExcel1" runat="server"></aspose:ExportGridViewToExcel>

{{< /highlight >}}

### **Eigenschaften der Aspose .NET Export GridView To Excel-Steuerung**
Die folgenden Eigenschaften werden bereitgestellt, um Features dieser Steuerung zu konfigurieren und zu nutzen

|**Eigenschaftsname** |**Typ** |**Beispiel/Mögliche Werte** |**Beschreibung** |
| :- | :- | :- | :- |
|ExportButtonText |string |Export to Excel |Sie können diese Eigenschaft verwenden, um den bestehenden Standardtext zu überschreiben |
|ExportButtonCssClass |string |btn btn-primary |Css-Klasse, die auf die äußere Div des Exportbuttons angewendet wird. Um css auf den Button anzuwenden, können Sie .yourClass input verwenden |
|ExportFileHeading |string |<h4>GridView Export Beispielbericht</h4> |Sie können HTML-Tags verwenden, um Ihrem Titel Stil hinzuzufügen |
|ExportOutputFormat |enum |Xlsx, Xlsb, Xls, Txt, Csv, Ods |Ausgabeformat des exportierten Dokuments. Unterstützte Formate sind Xlsx, Xlsb, Xls, Txt, Csv, Ods |
|ExportOutputPathOnServer |string |c: <br>temp |Lokaler Ausgabediskettenpfad auf dem Server, wo eine Kopie des Exports automatisch gespeichert wird. Die Anwendung muss Schreibzugriff auf diesen Pfad haben. |
|ExportDataSource |object |allRowsDataTable |Legt das Objekt fest, aus dem diese Datenbindungssteuerung ihre Liste der Datenobjekte abruft. Das Objekt muss alle Daten enthalten, die exportiert werden müssen. Diese Eigenschaft wird zusätzlich zur normalen DataSource-Eigenschaft verwendet und ist nützlich, wenn eine benutzerdefinierte Paginierung aktiviert ist und die aktuelle Seite nur die auf dem Bildschirm anzuzeigenden Zeilen abruft. |
|LicenseFilePath |string | |Lokaler Pfad auf dem Server zur Lizenzdatei. Zum Beispiel c: <br>inetpub <br>Aspose.Cells.lic |
Ein Beispiel für die Verwendung aller Eigenschaften des Export GridView to Excel-Steuerelements wird unten gezeigt

{{< highlight java >}}

 <aspose:ExportGridViewToExcel Width="800px" ID="ExportGridViewToExcel1" ExportButtonText="Export to Excel"

    CssClass="table table-hover table-bordered" ExportButtonCssClass="myClass" ExcelOutputFormat="xlsx"

    ExportOutputPathOnServer="c:\\temp" ExportFileHeading="<h1>Example Report</h1>"

    OnPageIndexChanging="ExportGridViewToExcel1_PageIndexChanging" PageSize="5" AllowPaging="True"

    LicenseFilePath="c:\inetpub\Aspose.Cells.lic" runat="server" CellPadding="4"

    ForeColor="#333333" GridLines="Both">

</aspose:ExportGridViewToExcel>
