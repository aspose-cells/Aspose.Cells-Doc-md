## **Einführung**
 Export GridView To Excel Control ist ein ASP.NET-Serversteuerelement, das den Export von GridView-Inhalten in Microsoft Excel oder OpenOffice Spreadsheets ermöglicht[Aspose.Cells](https://products.aspose.com/cells/net/) . Es fügt hinzu**Nach Excel exportieren** Schaltfläche über dem GridView-Steuerelement. Durch Klicken auf die Schaltfläche wird der Inhalt des GridView-Steuerelements dynamisch in eine Microsoft-Excel- oder OpenOffice-Tabelle exportiert und die exportierte Datei dann automatisch in nur wenigen Sekunden an den vom Benutzer ausgewählten Speicherort heruntergeladen.
### **Modulfunktionen**
Diese erste Version des Steuerelements bietet die folgenden Funktionen:

- Holen Sie sich eine Offline-Kopie Ihrer bevorzugten Online-GridView-Inhalte zum Bearbeiten, Freigeben und Drucken.
- Vom standardmäßigen GridView-Steuerelement ASP.NET geerbt und daher mit all seinen Features und Eigenschaften.
- Exportieren Sie GridView nach Xlsx, Xlsb, Xls, Txt, Csv, Ods.
- Funktioniert mit allen .NET-Versionen ab .NET 2.0.
- Möglichkeit, den Text der Export-Schaltfläche anzupassen/zu lokalisieren.
- Wenden Sie das Aussehen und Verhalten Ihres eigenen Themas auf der Schaltfläche „Exportieren“ mithilfe von CSS an.
- Option zum Hinzufügen einer benutzerdefinierten Überschrift über dem exportierten Dokument
- Option zum Speichern jedes exportierten Dokuments auf dem Server unter einem konfigurierbaren Festplattenpfad
- Option zum Exportieren der aktuellen Seite oder aller Seiten, wenn Paging aktiviert ist

## **Installieren**
Es ist sehr einfach und leicht, GridView To Excel Control zu exportieren, bitte befolgen Sie diese einfachen Schritte
### **Für Visual Studio 2010, 2012 und 2013**
1. Extrahieren Sie die heruntergeladene ZIP-Datei
1. Doppelklicken Sie auf die VSIX-Datei Aspose.Excel.GridViewExport.vsix
1. Ein Dialogfeld wird angezeigt, das Ihnen die verfügbaren und unterstützten Visual Studio-Versionen anzeigt, die auf Ihrem Computer installiert sind
1. Wählen Sie diejenigen aus, denen Sie das Export GridView To Excel-Steuerelement hinzufügen möchten.
1. Klicken Sie auf Installieren

Sie erhalten einen Erfolgsdialog, sobald die Installation abgeschlossen ist.

**Notiz:** Stellen Sie sicher, dass Sie Visual Studio neu starten, damit die Änderungen wirksam werden.
### **Für Visual Studio 2005, 2008 und Express-Editionen**
Bitte befolgen Sie diese Schritte, um das Export GridView To Excel-Steuerelement in Visual Studio für einfaches Drag & Drop zu integrieren, genau wie andere ASP.NET-Steuerelemente

1. Extrahieren Sie die heruntergeladene ZIP-Datei
1. Stellen Sie sicher, dass Sie Visual Studio als Administrator ausführen

Klicken Sie im Menü Extras auf Toolboxelemente auswählen.

1.  Klicken Sie auf Durchsuchen.
 Das Dialogfeld „Öffnen“ wird angezeigt.
1. Navigieren Sie zum extrahierten Ordner und wählen Sie Aspose.Excel.GridViewExport.dll aus
1. OK klicken.

Wenn Sie ein Aspx- oder Ascx-Steuerelement in der Toolbox auf der linken Seite öffnen, sehen Sie ExportGridViewToWord auf der Registerkarte Allgemein

### **Manuelles Hinzufügen des ExportGridViewToExcel-Steuerelements**
Wenn Sie Probleme mit den oben genannten Methoden haben, die Visual Studio Toolbox verwenden, können Sie dieses Steuerelement manuell zu Ihrer ASP.NET-Anwendung hinzufügen, die auf einem beliebigen .NET-Framework größer als 2.0 ausgeführt wird

1. Wenn Sie Visual Studio verwenden, stellen Sie sicher, dass Sie es als Administrator ausführen
1.  Verweis hinzufügen auf**Aspose.Excel.GridViewExport.dll**verfügbar als extrahiertes Downloadpaket in Ihrem ASP.NET-Projekt oder Ihrer Webanwendung. Stellen Sie sicher, dass Ihre Webanwendung/Visual Studio vollen Zugriff auf diesen Ordner hat, andernfalls erhalten Sie möglicherweise die Ausnahme „Zugriff verweigert“.
1.  Fügen Sie diese Zeile oben auf der Seite, dem Steuerelement oder der MasterPage hinzu

{{< highlight "java" >}}

 <%@ Register assembly="Aspose.Excel.GridViewExport" namespace="Aspose.Excel.GridViewExport" tagprefix="aspose" %>

{{< /highlight >}}

1.  Fügen Sie Folgendes an einer Stelle auf Ihrer ASP.NET-Seite, Ihrem Steuerelement oder Ihrer Masterseite hinzu, an der das Steuerelement hinzugefügt werden soll

{{< highlight "java" >}}

 <aspose:ExportGridViewToExcel ID="ExportGridViewToExcel1" runat="server"></aspose:ExportGridViewToExcel>

{{< /highlight >}}

### **Aspose .NET Eigenschaften von GridView in Excel-Steuerelement exportieren**
Die folgenden Eigenschaften werden zum Konfigurieren und Verwenden von coolen Funktionen bereitgestellt, die von diesem Steuerelement bereitgestellt werden

|**Name des Anwesens** |**Typ** |**Beispiel/Mögliche Werte** |**Beschreibung** |
|:- |:- |:- |:- |
| ExportButtonText| Schnur| Nach Excel exportieren| Sie können diese Eigenschaft verwenden, um vorhandenen Standardtext zu überschreiben|
|ExportButtonCssClass| Schnur| btn btn-primär| CSS-Klasse, die auf das äußere Div der Exportschaltfläche angewendet wird. Um CSS auf die Schaltfläche anzuwenden, können Sie die Eingabe .yourClass verwenden|
| ExportFileHeading| Schnur|<h4>GridView-Exportbeispielbericht</h4> | Sie können HTML-Tags verwenden, um Ihrer Überschrift Stil zu verleihen|
| ExportAusgabeformat| Aufzählung| Xlsx, Xlsb, Xls, Txt, Csv, Ods| Ausgabeformat des exportierten Dokuments. Unterstützte Formate sind Xlsx, Xlsb, Xls, Txt, Csv, Ods|
| ExportOutputPathOnServer| Schnur| c:<br> Temp| Lokaler Ausgabedatenträgerpfad auf dem Server, auf dem automatisch eine Kopie des Exports gespeichert wird. Die Anwendung muss Schreibzugriff auf diesen Pfad haben.|
| ExportDataSource| Objekt| allRowsDataTable|Legt das Objekt fest, von dem dieses Datenbindungssteuerelement seine Liste von Datenelementen abruft. Das Objekt muss alle Daten enthalten, die exportiert werden müssen. Diese Eigenschaft wird zusätzlich zur normalen DataSource-Eigenschaft verwendet und ist nützlich, wenn benutzerdefiniertes Paging aktiviert ist und die aktuelle Seite nur Zeilen abruft, die auf dem Bildschirm angezeigt werden sollen.|
| LicenseFilePath| Schnur|| Lokaler Pfad auf dem Server zur Lizenzdatei. Zum Beispiel c:<br> inetpub<br> Aspose.Cells.lic|
