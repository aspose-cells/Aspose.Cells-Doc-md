## **Einführung**
Das Aspose .NET Export Users to Excel for SiteFinity-Modul ermöglicht es Entwicklern, SiteFinity-Benutzer in Microsoft Excel oder OpenOffice Spreadsheet zu exportieren. Dieses Modul zeigt die leistungsstarke Tabellenkalkulationsfunktion von Aspose.Cells.

## **Systemanforderungen und unterstützte Plattformen**
### **Systemanforderungen**
Um Aspose.Cells .NET für Sitefinity-Add-Ons einzurichten, müssen die folgenden Anforderungen erfüllt sein:

- Sitefinity CMS läuft auf ASP.NET 4.0

Bitte zögern Sie nicht, uns zu kontaktieren, wenn Sie Probleme bei der Einrichtung dieses Sitefinity-Add-ons haben.
### **Unterstützte Plattformen**
Das Add-On wird auf allen Versionen von unterstützt

- Sitefinity CMS läuft auf ASP.NET 4.0
## **Quellcode**
Sie können den neuesten Quellcode von einem der folgenden Standorte erhalten

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-.NET/tree/master/Plugins/SiteFinity)

Sie müssen Folgendes installiert haben, um den Quellcode öffnen und erweitern zu können

1. Navigieren Sie zum neuesten heruntergeladenen Quellcode und öffnen Sie die **.sln**-Datei.
## **Installation und Verwendung**
### **Herunterladen**
Sie können den Aspose .NET Content Exporter für das Sitefinity-Modul von einem der folgenden Orte herunterladen

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-.NET/releases)
### **Installation**
Nachdem Sie es heruntergeladen haben, befolgen Sie bitte diese Schritte, um das Add-on in Ihre Sitefinity-Website zu installieren:

**Schritt 1: Dateien in Ihre Sitefinity-Installation kopieren**

Bitte entpacken Sie die heruntergeladene ZIP-Datei. Sie benötigen FTP oder direkten Zugriff auf den Sitefinity-Installationsordner auf dem Server, um Folgendes durchzuführen:

1. Kopieren Sie Aspose.Cells.dll und Aspose.SiteFinity.ExportUsersToExcel.dll in den **bin**-Ordner der Sitefinity-Installation.
1. Kopieren Sie den **Addons**-Ordner auf der obersten Ebene der Sitefinity-Installation, wo sich der **bin**-Ordner befindet.

**Schritt 2: Das Aspose Sitefinity Content Export-Add-On in Sitefinity registrieren**

1. Log into your Sitefinity CMS with an ‘**Administrator**’ account. The login page can be reached by <http://www.mywebsite.com/sitefinity>
1. Klicken Sie auf **Verwaltung** und dann auf **Einstellungen**.
   Die Seite für die Grundlegenden Einstellungen wird angezeigt.
1. Klicken Sie auf den Link **Erweitert**.
   Die Einstellungsseite erscheint.
1. Klicken Sie im linken Bereich auf **Toolboxen**, gefolgt von **Toolboxen**, dann **Seitensteuerelemente**, **Abschnitte** und **Inhalt Toolbox Abschnitt**, dann **Werkzeuge**.
1. Klicken Sie auf **Neu erstellen**.
   Das Widget-Registrierungsformular erscheint.
1. Füllen Sie die Formularfelder wie folgt aus: 
   1. Stellen Sie sicher, dass **Aktiviert** ausgewählt ist.
   1. Fügen Sie ~/Addons/Aspose.SiteFinity.ExportUsersToExcel/AsposeExportUsersToExcel.ascx hinzu.

   1. ` `in das Feld **Steuerelement CLR-Typ oder virtueller Pfad**.
   1. Fügen Sie **Name**, **Titel** und **Beschreibung** wie folgt hinzu:
      Aspose.SiteFinity.ExportUsersToExcel
      Aspose Export SiteFinity Users to Excel
      Benutzer von SiteFinity nach Excel exportieren
   1. Sie können alle anderen Felder unverändert lassen.
1. Klicken Sie auf **Änderungen speichern**, wenn Sie fertig sind.
   Das Widget ist in der Toolbox registriert und kann in Sitefinity verwendet werden.
## **Verwendung und Video-Demo**
### **Verwendung**
Nachdem Sie das Aspose Sitefinity Export Users to Excel-Add-On installiert und konfiguriert haben, ist es wirklich einfach, es auf Ihrer Website zu verwenden. Befolgen Sie bitte diese einfachen Schritte, um loszulegen:

1. Stellen Sie sicher, dass Sie in Sitefinity mit einem Administratorzugang angemeldet sind.
1. Navigieren Sie zur Seite, auf der Sie das Export-Add-On hinzufügen möchten. Stellen Sie sicher, dass die Seite im Bearbeitungsmodus geöffnet ist.
1. Wählen Sie im rechten **Widgets ziehen** Menü 'Aspose Export Benutzer nach Excel' aus und ziehen Sie es an die gewünschte Position.


Sie haben erfolgreich Aspose Sitefinity Export Benutzer nach Excel hinzugefügt.
