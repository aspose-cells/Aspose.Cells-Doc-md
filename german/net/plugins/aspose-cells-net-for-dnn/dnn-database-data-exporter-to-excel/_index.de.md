## **Einführung**

Aspose .NET DNN Data Exporter to Excel-Modul ermöglicht es Benutzern, Daten direkt aus lokalen oder entfernten Datenbanktabellen, Ansichten und benutzerdefinierten Abfragen in Microsoft Excel oder OpenOffice Spreadsheet zu exportieren. Dieses Modul zeigt die leistungsstarke Tabellenkalkulationsfunktion von Aspose.Cells.

### **Modulfunktionen**

Diese erste Version des Moduls ist mit den folgenden coolen Funktionen angereichert, um den Exportprozess einfach und benutzerfreundlich zu gestalten

- Erlauben Sie den Anschluss an die lokale MS SQL Server-Datenbank
- Erlauben Sie den Anschluss an die entfernte MS SQL Server-Datenbank
- Füllen Sie alle Tabellen aus der verbundenen Datenbank auf
- Füllen Sie alle Ansichten aus der verbundenen Datenbank auf
- Erlauben Sie die Erstellung einer benutzerdefinierten Abfrage
- Automatische Anpassung von Spalten an die Länge des Inhalts.
- Zulassen, dass Zeichenfolgen mit mehr als 32k in Excel-Zellen übersprungen werden (LoadOptions)
- Anwendung der Header-Spaltenformatierung als fettgedruckter Text
- Verwendung als Datenquelle zulassen (Tabelle, Ansichten, benutzerdefinierte Abfrage)
- Export von Daten in Microsoft Excel-Dokumente (.xls, .xlsx und .xlsb)
- Export von Daten in ein Tabstopp-Textdokument (*.txt)
- Export von Daten in CSV (durch Komma getrennt) (*.csv)
- Export von Daten in OpenDocument-Tabellenkalkulation (*.ods)
- Option zur Auswahl des gewünschten Ausgabeformats vor dem Export
- Das exportierte Dokument wird automatisch zum Herunterladen an den Browser gesendet

## **Installation**

Nachdem Sie es heruntergeladen haben, befolgen Sie bitte diese Schritte, um das Modul in Ihre DNN-Website zu installieren:

1. Melden Sie sich auf Ihrer Website als Host oder einem anderen Benutzerkonto mit Superuser-Berechtigungen an.
1. Navigieren Sie zum **Host**-Menü und wählen Sie **Erweiterungen**.
1. Klicken Sie auf **Installations-Assistent für Erweiterungen**.
1. Gehen Sie wie angegeben zum Speicherort der heruntergeladenen Zip-Datei, wählen Sie sie aus und klicken Sie dann auf **Öffnen**.
1. Klicken Sie auf **Weiter**, akzeptieren Sie die Lizenz, setzen Sie die Installation fort. Wenn die Installation abgeschlossen ist, klicken Sie auf die Schaltfläche **Zurück**.

Bitte überprüfen Sie [dieses Modul-Installationsvideo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) von DNN für weitere Details.

## **Verwendung**

Nachdem Sie das Aspose .NET Database Data Exporter für das DNN-Modul installiert haben, ist es wirklich einfach, es auf Ihrer Website zu verwenden. Befolgen Sie bitte diese einfachen Schritte, um loszulegen:

1. Stellen Sie sicher, dass Sie bei DNN mit einem Host- oder Admin-Konto angemeldet sind.
1. Navigieren Sie zu der Seite, auf der Sie das Exportmodul hinzufügen möchten.
1. Wählen Sie **Module**, gefolgt von **Neues Modul hinzufügen** im oberen Menüband.
1. Wählen Sie aus der Liste 'Aspose.NET DNN Data Exporter to Excel Using Aspose.Cells' und ziehen Sie sie an die gewünschte Position.

![todo:image_alt_text](picture2.png)

Sie haben das Aspose .NET Database Data Exporter-Modul erfolgreich zu Ihrer Seite hinzugefügt.

![todo:image_alt_text](picture3.png)

1. Geben Sie ein oder verwenden Sie den vorab ausgefüllten MS SQL Server-Verbindungszeichenfolge
1. Wählen Sie den Datentyp aus (Tabelle, Ansicht, benutzerdefinierte Abfrage)
1. Wählen Sie den Datenquelle aus (Tabelle, Ansicht, benutzerdefinierte Abfrage) oder geben Sie diese ein
1. Wählen Sie den Exporttyp aus
1. Klicken Sie auf Daten exportieren
1. Die gewünschte Datei wird automatisch heruntergeladen.
