## **Einführung**
Das Aspose .NET Import Users from Excel-Modul ermöglicht es Entwicklern, Benutzer in Bulk aus einer Excel-Datei zu importieren. Dieses Modul zeigt die leistungsstarke Tabellenkalkulation von Aspose.Cells.
### **Modulfunktionen**
Diese erste Version des Moduls ist mit den folgenden coolen Funktionen ausgestattet, um den Importprozess einfach und benutzerfreundlich zu gestalten:

- Ermöglicht das Hochladen einer Excel-Datei und filtert automatisch vorhandene Benutzer heraus.
- Überspringt automatisch die vorhandenen Benutzer, um jegliche Verwirrung zu vermeiden.
- Generiert dynamische Passwörter für die neu erstellten Benutzer.
- Weist den neu erstellten Benutzern die Sicherheitsrollen "Registrierte Benutzer" und "Abonnenten" zu.

## **Installation**
Nachdem Sie es heruntergeladen haben, befolgen Sie bitte diese Schritte, um das Modul in Ihre DNN-Website zu installieren:

1. Melden Sie sich auf Ihrer Website als Host oder einem anderen Benutzerkonto mit Superuser-Berechtigungen an.
1. Navigieren Sie zum **Host**-Menü und wählen Sie **Erweiterungen**.
1. Klicken Sie auf **Installations-Assistent für Erweiterungen**.
1. Gehen Sie wie angegeben zum Speicherort der heruntergeladenen Zip-Datei, wählen Sie sie aus und klicken Sie dann auf **Öffnen**.
1. Klicken Sie auf **Weiter**, akzeptieren Sie die Lizenz, setzen Sie die Installation fort. Wenn die Installation abgeschlossen ist, klicken Sie auf die Schaltfläche **Zurück**.

Bitte überprüfen Sie [dieses Modul-Installationsvideo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) von DNN für weitere Details.
## **Verwendung**
Nachdem Sie das Aspose .NET Cells Import Users from Excel für DNN-Modul installiert haben, ist es wirklich einfach, es auf Ihrer Website zu verwenden. Befolgen Sie bitte diese einfachen Schritte, um zu beginnen:

1. Stellen Sie sicher, dass Sie bei DNN mit einem Host- oder Admin-Konto angemeldet sind.
1. Navigieren Sie zu der Seite, auf der Sie das Exportmodul hinzufügen möchten.
1. Wählen Sie **Module**, gefolgt von **Neues Modul hinzufügen** im oberen Menüband. 

1. Wählen Sie aus der Liste das 'Aspose Import Users from Excel module for DNN' und ziehen Sie es an die gewünschte Position.

Sie haben das Aspose .NET Cells Import Users from Excel-Modul für das DNN-Modul erfolgreich zu Ihrer Seite hinzugefügt. Die Datei-Upload-Steuerungen werden zusammen mit dem Link 'Vorlagendatei' angezeigt. Sie können die Vorlagendatei herunterladen und Ihre Inhalte entsprechend formatieren. Sobald Sie Ihre Excel-Datei hochgeladen haben, werden alle Benutzer automatisch verarbeitet und bestehende Benutzer herausgefiltert. Sie können die Benutzer auswählen, die Sie importieren möchten, und auf die Schaltfläche 'Benutzer importieren' klicken.
