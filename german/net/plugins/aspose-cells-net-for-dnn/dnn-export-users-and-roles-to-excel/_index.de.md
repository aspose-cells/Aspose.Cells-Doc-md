## **Einführung**
Aspose .NET Export Benutzer und Rollen nach Excel Modul ermöglicht Entwicklern, DNN Benutzer zusammen mit ihren Rollen in Microsoft Excel oder OpenOffice Spreadsheet zu exportieren. Dieses Modul demonstriert das leistungsstarke Tabellenkalkulationsfeature von Aspose.Cells.
### **Modulfunktionen**
Diese erste Version des Moduls ist mit den folgenden coolen Funktionen angereichert, um den Exportprozess einfacher und benutzerfreundlicher zu gestalten.

- Benutzer in Microsoft Excel-Dokumente exportieren (.xls, .xlsx und .xlsb)
- Benutzer in Tabstopp-getrennte Textdokumente (* .txt) exportieren
- Benutzer in CSV (Komma getrennt) (* .csv) exportieren
- Benutzer in OpenDocument-Spreadsheet (* .ods) exportieren
- Option zur Auswahl des gewünschten Ausgabeformats vor dem Export
- Option zum Exportieren aller oder ausgewählter Benutzer in das ausgewählte Ausgabeformat für das Dokument
- Das exportierte Dokument wird automatisch zum Herunterladen an den Browser gesendet

## **Installation**
Nachdem Sie es heruntergeladen haben, befolgen Sie bitte diese Schritte, um das Modul in Ihre DNN-Website zu installieren:

1. Melden Sie sich auf Ihrer Website als Host oder einem anderen Benutzerkonto mit Superuser-Berechtigungen an.
1. Navigieren Sie zum **Host**-Menü und wählen Sie **Erweiterungen**.
1. Klicken Sie auf **Installations-Assistent für Erweiterungen**.
1. Gehen Sie wie angegeben zum Speicherort der heruntergeladenen Zip-Datei, wählen Sie sie aus und klicken Sie dann auf **Öffnen**.
1. Klicken Sie auf **Weiter**, akzeptieren Sie die Lizenz, setzen Sie die Installation fort. Wenn die Installation abgeschlossen ist, klicken Sie auf die Schaltfläche **Zurück**.

Bitte überprüfen Sie [dieses Modul-Installationsvideo](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7) von DNN für weitere Details.
## **Verwendung**
Nachdem Sie das Aspose .NET Cells Export-Benutzer und -Rollenmodul für DNN installiert haben, ist es wirklich einfach, es auf Ihrer Website zu verwenden. Befolgen Sie diese einfachen Schritte, um loszulegen:

1. Stellen Sie sicher, dass Sie bei DNN mit einem Host- oder Admin-Konto angemeldet sind.
1. Navigieren Sie zu der Seite, auf der Sie das Exportmodul hinzufügen möchten.
1. Wählen Sie **Module**, gefolgt von **Neues Modul hinzufügen** im oberen Menüband. 

1. Wählen Sie aus der Liste 'Aspose Export Users and Roles to Excel' und ziehen Sie sie an die gewünschte Position.

Sie haben das Aspose .NET Export Users and Roles to Excel for DNN-Modul erfolgreich zu Ihrer Seite hinzugefügt. Alle DNN-Benutzer werden in einer Rasteransicht angezeigt und Sie können Benutzer zum Exportieren auswählen und dann auf die Schaltfläche "Exportieren" klicken, nachdem Sie das gewünschte Ausgabeformat aus dem Dropdown-Menü ausgewählt haben. Die gewünschte Datei wird automatisch heruntergeladen.
