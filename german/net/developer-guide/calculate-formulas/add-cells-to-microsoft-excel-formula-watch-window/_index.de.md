## **Mögliche Verwendungsszenarien**

Das Microsoft Excel Formelüberwachungsfenster ist ein nützliches Tool, um die Zellwerte und -formeln bequem in einem Fenster anzeigen zu können. Sie können das *Überwachungsfenster* in Microsoft Excel öffnen, indem Sie auf *Formeln > Überwachen* *Fenster* klicken. Es hat die Schaltfläche *Überwachung hinzufügen*, die verwendet werden kann, um die Zellen zur Inspektion hinzuzufügen. Ebenso können Sie die Methode [**Worksheet.CellWatches.Add()**](https://reference.aspose.com/cells/net/aspose.cells/cellwatchcollection/methods/add/index) verwenden, um Zellen in das *Überwachungsfenster* unter Verwendung der Aspose.Cells API hinzuzufügen.
