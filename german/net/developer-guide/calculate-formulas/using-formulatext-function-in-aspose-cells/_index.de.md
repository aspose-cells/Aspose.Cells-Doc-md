{{% alert color="primary" %}} 

FormulaText ist eine Funktion von Excel 2013 und höher. Es wird von früheren Versionen wie Excel 2010 oder 2007 usw. nicht unterstützt. Wie der Name schon sagt, druckt es den Text der Formel, die in einer bestimmten Zelle vorhanden ist. Dieser Artikel zeigt Ihnen, wie Sie diese Funktion mit Aspose.Cells nutzen können.

{{% /alert %}}
