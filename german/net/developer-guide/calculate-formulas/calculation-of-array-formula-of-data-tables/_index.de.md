{{% alert color="primary" %}}

 Sie können eine Datentabelle in Microsoft Excel erstellen, indem Sie Daten > Was-wäre-wenn-Analyse > Datentabelle ... verwenden. Mit Aspose.Cells können Sie jetzt die Matrixformel einer Datentabelle berechnen. Bitte verwende[**Workbook.CalculateFormula()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/calculateformula) wie für die Berechnung beliebiger Formeln üblich.

{{% /alert %}}
