{{% alert color="primary" %}}

Sie können eine Datentabelle in Microsoft Excel erstellen, indem Sie Daten > Was-wäre-wenn-Analyse > Datentabelle... verwenden. Mit Aspose.Cells können Sie jetzt die Arrayformel einer Datentabelle berechnen. Benutzen Sie bitte[**Workbook.CalculateFormula()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/calculateformula) wie üblich für die Berechnung jeglicher Art von Formeln.

{{% /alert %}}
