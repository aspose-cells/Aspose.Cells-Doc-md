Dieser Artikel enthält Informationen zum Abrufen verbundener Zellbereiche in einem Arbeitsblatt.

Aspose.Cells ermöglicht es Ihnen, zusammengeführte Zellbereiche in einem Arbeitsblatt zu erhalten. Sie können sie auch trennen (aufteilen). Dieser Artikel zeigt den einfachsten Code mit**Aspose.Cells API** um die Aufgabe auszuführen.

{{% /alert %}}

Die Komponente stellt die[**Cells.MergedCells**](https://reference.aspose.com/cells/net/aspose.cells/cells/properties/mergedcells)-Attribut, das eine ArrayList von zusammengeführten Zellbereichen erhalten kann.
