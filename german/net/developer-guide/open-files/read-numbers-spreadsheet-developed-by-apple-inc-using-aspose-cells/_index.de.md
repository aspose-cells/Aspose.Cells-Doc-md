## **Mögliche Verwendungsszenarien**

Numbers ist eine Tabellenkalkulationsanwendung, die von Apple Inc. entwickelt wurde. Aspose.Cells kann eine Numbers-Tabelle lesen, unterstützt jedoch nicht das Schreiben darauf. Es kann Daten, Formatierungen und Formeln der Numbers-Tabelle lesen.

## **Zahlen-Tabellenkalkulation von Apple Inc. mit Aspose.Cells lesen**

Der folgende Beispielcode lädt die [Beispiel-Numbers-Arbeitsmappe](sampleNumbersByAppleInc.numbers) und konvertiert sie in das [Ausgabe-PDF-Format](outputNumbersByAppleInc.pdf). Sie müssen die [**LoadOptions**](https://reference.aspose.com/cells/net/aspose.cells/loadoptions) Klasse verwenden und [**LoadFormat.Numbers**](https://reference.aspose.com/cells/net/aspose.cells/loadformat) als Parameter in ihrem Konstruktor angeben, um sie erfolgreich zu laden. Bitte laden Sie beide von den angegebenen Links herunter. Sie können den Code mit jeder Numbers-Arbeitsmappe ausprobieren. Bitte lesen Sie auch die Kommentare des Codes für weitere Hilfe.
