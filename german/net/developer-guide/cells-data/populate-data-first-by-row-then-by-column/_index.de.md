{{% alert color="primary" %}} 

Wenn Sie eine Tabelle zuerst zeilenweise und dann spaltenweise mit Daten füllen, verbessert sich die Gesamtleistung.

{{% /alert %}} 

Das Einfügen von Daten in die Reihenfolge A1, B1, A2, B2 ist schneller als A1, A2, B1, B2. Wenn ein Arbeitsblatt viele Zellen enthält und Sie der zweiten Reihenfolge folgen, also die Daten Zeile für Zeile füllen, kann dieser Tipp das Programm viel schneller machen.
