{{% alert color="primary" %}}

 Manchmal müssen Sie Zellen finden, auf die ein bestimmter Stil angewendet wurde. Mit Aspose.Cells können Sie alle Zellen mit einem gemeinsamen Stil finden. Aspose.Cells bietet die[**FindOptions.Style**](https://reference.aspose.com/cells/net/aspose.cells/findoptions/properties/style) -Eigenschaft, mit der Sie den Stil angeben können, nach dem Zellen durchsucht werden sollen.

{{% /alert %}}

Der Code in diesem Beispiel findet alle Zellen, die denselben Stil wie Zelle A1 haben. Nachdem der Code ausgeführt wurde, enthalten alle Zellen, die den gleichen Stil wie A1 haben, den Text „Gefunden“.
