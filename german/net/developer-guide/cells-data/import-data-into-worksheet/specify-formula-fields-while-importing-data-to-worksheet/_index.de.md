## **Mögliche Verwendungsszenarien**

Beim Importieren von Daten in Ihr Arbeitsblatt können Sie mit dem [**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas) Formelfelder angeben. Diese Eigenschaft nimmt das Boolsche Array an, wobei der Wert **true** bedeutet, dass das Feld ein Formelfeld ist. Wenn beispielsweise das dritte Feld ein Formelfeld ist, wird der dritte Wert im Array **true** sein.
