##  **Mögliche Nutzungsszenarien**

Sie können Formelfelder angeben, wenn Sie Daten mithilfe von in Ihr Arbeitsblatt importieren[**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas) . Diese Eigenschaft übernimmt das boolesche Array, in dem sich der Wert befindet**WAHR**bedeutet, dass das Feld ein Formelfeld ist. Wenn das dritte Feld beispielsweise ein Formelfeld ist, ist der dritte Wert im Array *true**.
