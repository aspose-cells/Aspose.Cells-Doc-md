{{% alert color="primary" %}}

Aspose.Cells unterstützt die Konvertierung des Microsoft Excel-orientierten HTML in das XLS/XLSX-Format. Das bedeutet, dass die von Microsoft Excel generierte HTML mithilfe von Aspose.Cells wieder in das Format XLS/XLSX konvertiert werden kann.

 Wenn es ein einfaches HTML gibt, kann Aspose.Cells es in HTML Rich Text konvertieren. Aspose.Cells bietet[**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/properties/htmlstring) Eigenschaft, die ein so einfaches HTML annehmen und in formatierten Zellentext konvertieren kann.

{{% /alert %}}
