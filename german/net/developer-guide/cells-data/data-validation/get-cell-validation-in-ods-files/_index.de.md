## **Zellvalidierung in ODS-Dateien erhalten**

Hierfür bietet die API die [**GetValidation**](https://reference.aspose.com/cells/net/aspose.cells/cell/methods/getvalidation)-Methode der [**Cell**](https://reference.aspose.com/cells/net/aspose.cells/cell)-Klasse.
