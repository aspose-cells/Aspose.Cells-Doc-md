##  **Holen Sie sich Hyperlinks in Reichweite**

 Der[**Reichweite**](https://reference.aspose.com/cells/net/aspose.cells/range) Klasse bietet a[**Hyperlinks**](https://reference.aspose.com/cells/net/aspose.cells/range/properties/hyperlinks) Eigenschaft, die alle Hyperlinks im ausgewählten Bereich zurückgibt. Sie können den Hyperlink auch löschen, indem Sie die aufrufen[**Hyperlink.Löschen**](https://reference.aspose.com/cells/net/aspose.cells/hyperlink/methods/delete) Methode.
