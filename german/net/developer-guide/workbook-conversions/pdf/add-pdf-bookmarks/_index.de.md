Dieser Artikel enthält Informationen zum Einfügen von PDF-Lesezeichen beim Konvertieren einer Tabellenkalkulation in PDF.

Aspose.Cells ermöglicht es Ihnen, Lesezeichen im Handumdrehen hinzuzufügen. PDF-Lesezeichen können die Navigierbarkeit langer Dokumente drastisch verbessern. Wenn Sie Lesezeichen-Links zu einem PDF-Dokument hinzufügen, haben Sie eine genaue Kontrolle über die genaue Ansicht, die Sie möchten, Sie sind nicht darauf beschränkt, auf eine Seite zu verlinken. Sie können die genaue Ansicht einstellen, indem Sie die Zielseite positionieren und dann das Lesezeichen erstellen.

{{% /alert %}}

Bitte sehen Sie sich den folgenden Beispielcode an, um herauszufinden, wie Sie PDF-Lesezeichen hinzufügen. Der Code generiert eine einfache Arbeitsmappe, gibt PDF-Lesezeichen mit Zielorten an und generiert die PDF-Datei.
