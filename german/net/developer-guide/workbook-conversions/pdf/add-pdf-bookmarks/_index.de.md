Dieser Artikel enthält Informationen zum Einfügen von PDF-Lesezeichen beim Konvertieren einer Tabelle in PDF.

Aspose.Cells ermöglicht es Ihnen, Lesezeichen im Handumdrehen hinzuzufügen. PDF Lesezeichen können die Navigierbarkeit langer Dokumente drastisch verbessern. Wenn Sie Lesezeichen-Links zu PDF-Dokumenten hinzufügen, haben Sie eine genaue Kontrolle über die genaue Ansicht, die Sie möchten, Sie sind nicht darauf beschränkt, auf eine Seite zu verlinken. Sie können die genaue Ansicht einstellen, indem Sie die Zielseite positionieren und dann das Lesezeichen erstellen.

{{% /alert %}}

Bitte sehen Sie sich den folgenden Beispielcode an, um herauszufinden, wie Sie PDF-Lesezeichen hinzufügen. Der Code generiert eine einfache Arbeitsmappe, gibt PDF-Lesezeichen mit Zielspeicherorten an und generiert die PDF-Datei.

{{% alert color="primary" %}}

Wenn Ihre Tabelle Formeln enthält, rufen Sie am besten an[**Workbook.CalculateFormula**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/calculateformula/index) kurz vor dem Rendern der Tabelle in das Format PDF. Dadurch wird sichergestellt, dass die formelabhängigen Werte in PDF aktualisiert und korrekt gerendert werden.
