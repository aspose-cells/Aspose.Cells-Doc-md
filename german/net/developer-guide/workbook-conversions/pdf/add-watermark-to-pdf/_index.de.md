Beim Konvertieren einer Excel-Datei in eine PDF-Datei müssen Sie möglicherweise ein Wasserzeichen zur PDF-Datei hinzufügen. Die folgenden Beispiele zeigen, wie man beim Rendern als PDF Text- und Bildwasserzeichen zu PDF hinzufügt.

##   **Fügen Sie Textwasserzeichen zu PDF hinzu**

 Sie können ganz einfach Textwasserzeichen zu PDFs hinzufügen, indem Sie Text und die entsprechende Schriftart angeben. Außerdem können Sie Ausrichtung, Versatz, Drehung, Deckkraft, Vordergrund/Hintergrund und Skalierung zum Einblättern festlegen[RenderingWatermark](https://reference.aspose.com/cells/net/aspose.cells.rendering/renderingwatermark/).

##  **Bildwasserzeichen zu PDF hinzufügen**

 Sie können einem PDF ein Bildwasserzeichen hinzufügen, indem Sie einfach die Bildbytes eines Bildes angeben. Außerdem können Sie Ausrichtung, Versatz, Drehung, Deckkraft, Vordergrund/Hintergrund und Skalierung zum Einblättern festlegen[RenderingWatermark](https://reference.aspose.com/cells/net/aspose.cells.rendering/renderingwatermark/).
