## **Mögliche Nutzungsszenarien**

Benannte Ziele sind spezielle Arten von Lesezeichen oder Links in PDF, die nicht von PDF-Seiten abhängen. Das bedeutet, wenn Seiten von PDF hinzugefügt oder gelöscht werden, können Lesezeichen ungültig werden, aber benannte Ziele bleiben intakt. Um ein benanntes Ziel zu erstellen, stellen Sie bitte das ein[**PdfLesezeichenEintrag.Zielname**](https://reference.aspose.com/cells/net/aspose.cells.rendering/pdfbookmarkentry/properties/destinationname)Eigentum.
