{{% alert color="primary" %}} 

 Standardmäßig speichert Aspose.Cells Excel in PDF mit Standardgröße. Sie können es jedoch auch mit der Mindestgröße speichern[PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/net/aspose.cells/pdfsaveoptions/properties/optimizationtype) Eigentum. Es akzeptiert die folgenden Werte

- PdfOptimizationType.Standard
- PdfOptimizationType.MinimumSize
