{{% alert color="primary" %}} 

Standardmäßig speichert Aspose.Cells Excel in PDF mit Standardgröße. Sie können es jedoch auch mit Minimalgröße speichern, indem Sie die [PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/net/aspose.cells/pdfsaveoptions/properties/optimizationtype)-Eigenschaft verwenden. Es akzeptiert die folgenden Werte

- PdfOptimizationType.Standard
- PdfOptimizationType.MinimumSize
