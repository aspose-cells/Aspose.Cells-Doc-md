{{% alert color="primary" %}} 

Normalerweise werden die Layouts der div-Tags beim Laden von HTML in ein Excel-Arbeitsmappenobjekt ignoriert. Wenn Sie jedoch möchten, dass die Layouts der div-Tags nicht ignoriert werden, setzen Sie bitte die [HTMLLoadOptions.SupportDivTag](https://reference.aspose.com/cells/net/aspose.cells/htmlloadoptions/properties/supportdivtag)-Eigenschaft auf **true**. Der Standardwert dieser Eigenschaft ist **false**.
