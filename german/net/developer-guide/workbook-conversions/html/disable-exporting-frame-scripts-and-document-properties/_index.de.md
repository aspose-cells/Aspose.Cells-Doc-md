{{% alert color="primary" %}}

Beim Konvertieren einer Arbeitsmappe in HTML exportiert Aspose.Cells Frameskripte und Dokumenteigenschaften. Die Version 8.6.0 von Aspose.Cells for .NET führt eine Option ein, mit der das Exportieren von Frameskripten und Dokumenteigenschaften optional deaktiviert werden kann. Bitte verwenden Sie die HtmlSaveOptions.ExportFrameScriptsAndProperties-Eigenschaft, um den Export zu deaktivieren.

{{% /alert %}}
