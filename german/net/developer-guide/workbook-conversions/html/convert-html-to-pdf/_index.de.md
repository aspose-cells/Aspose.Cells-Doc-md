## **HTML to PDF Konvertierung in .NET**
Wie konvertiert man HTML in PDF? Mit der [Aspose.Cells for .NET](https://releases.aspose.com/cells/net/) Bibliothek können Sie HTML programmgesteuert mit wenigen Codezeilen in PDF konvertieren. Aspose.Cells for .NET ist in der Lage, plattformübergreifende Anwendungen zu erstellen, die Excel-Dateien generieren, ändern, konvertieren, rendern und drucken können.

## **C# HTML in PDF konvertieren**
Das folgende C#-Codebeispiel zeigt, wie man ein HTML-Dokument mithilfe von [Aspose.Cells for .NET](https://releases.aspose.com/cells/net/) in ein PDF konvertiert.

1. Erstellen Sie eine Instanz der Klasse [HtmlLoadOptions](https://reference.aspose.com/cells/net/aspose.cells/htmlloadoptions/).
1. Initialisieren Sie das Objekt [Workbook](https://reference.aspose.com/cells/net/aspose.cells/workbook/).
1. Speichern Sie das Ausgabe-PDF-Dokument, indem Sie die Methode Workbook.Save() aufrufen.
