Aspose.Cells unterstützt die Umwandlung einer Json (JavaScript Object Notation)-Datei in eine Excel-Arbeitsmappe.

{{% /alert %}}

## **Konvertieren Sie JSON in Excel-Arbeitsmappe**
Sie müssen sich keine Gedanken darüber machen, wie Sie JSON in eine Excel-Datei konvertieren, denn die Apose.Cells für die .NET-Bibliothek hat die beste Lösung. Die Aspose.Cells-API bietet Unterstützung für die Konvertierung des JSON-Formats in Tabellenkalkulationen. Sie können die Klasse [**JsonLoadOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonLoadOptions) verwenden, um zusätzliche Einstellungen für den Import von JSON in die Arbeitsmappe festzulegen.
