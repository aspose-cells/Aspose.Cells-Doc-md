## **Excel-Arbeitsmappe in JSON konvertieren**

Die Aspose.Cells-API unterstützt die Konvertierung von Arbeitsblättern in das JSON-Format. Um die Arbeitsmappe in JSON zu exportieren, geben Sie [**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat) als zweiten Parameter der Methode [**Workbook.Save**](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/3) an. Sie können auch die Klasse [**JsonSaveOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonSaveoptions) verwenden, um zusätzliche Einstellungen für die Exportierung des Arbeitsblatts in JSON festzulegen.

Das folgende Codebeispiel zeigt, wie das aktive Arbeitsblatt mithilfe des Enumerationselement [**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat) in Json exportiert wird. Bitte sehen Sie sich den Code zur Konvertierung der [Quelldatei](book1.xlsx) in die [Ausgabe-Json-Datei](book1.Json) an, die vom Code generiert wurde, zum Nachschlagen.
