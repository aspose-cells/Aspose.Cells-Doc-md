##  **Mögliche Nutzungsszenarien**
Wenn Sie den Rahmen für „Bereich“ festlegen möchten, müssen Sie nicht jede Zelle einzeln festlegen. Sie können den Rand des Bereichs festlegen. Aspose.Cells bietet diese Funktion.
Dieser Artikel enthält einen Beispielcode, der Aspose.Cells zum Festlegen der Bereichsgrenze verwendet.

##  **Legen Sie den Bereichsrand in Excel fest**
Um den Rand eines Bereichs in Excel festzulegen, können Sie die folgenden Schritte ausführen:
1. Wählen Sie den Zellbereich aus, auf den Sie den Rahmen anwenden möchten.
2. Suchen Sie auf der Registerkarte „Startseite“ des Menübands die Gruppe „Schriftart“.
3. Klicken Sie in der Gruppe „Schriftart“ auf die Dropdown-Schaltfläche „Rahmen“.
<br>
<img src="border.png" />
4. Wählen Sie aus den Optionen im Dropdown-Menü den Rahmentyp aus, den Sie anwenden möchten. Sie können aus voreingestellten Rahmenstilen wählen oder Ihren eigenen Rahmen anpassen.
5. Sobald Sie den gewünschten Rahmenstil ausgewählt haben, wird der Rahmen auf den ausgewählten Zellbereich angewendet.

##  **Legen Sie die Bereichsgrenze mit Aspose.Cells fest**
Dieses Beispiel zeigt, wie Sie:

1. Erstellen Sie eine Arbeitsmappe.
1. Fügen Sie Daten zu den Zellen im ersten Arbeitsblatt hinzu.
1.  Ein ... kreieren[**Bereich**](https://reference.aspose.com/cells/java/com.aspose.cells/range/).
1. Legen Sie den inneren Rand des Bereichs fest.
1. Legen Sie den äußeren Rand des Bereichs fest.
