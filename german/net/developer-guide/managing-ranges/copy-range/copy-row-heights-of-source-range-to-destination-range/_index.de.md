{{% alert color="primary" %}}

 Manchmal muss der Benutzer Zeilenhöhen des Quellbereichs in den Zielbereich kopieren. Aspose.Cells bietet[**PasteType.RowHeights**](https://reference.aspose.com/cells/net/aspose.cells/pastetype) enum für diesen Zweck. Wann werden Sie einstellen[**PasteOptions.PasteType**](https://reference.aspose.com/cells/net/aspose.cells/pasteoptions/properties/pastetype) Eigentum mit[**PasteType.RowHeights**](https://reference.aspose.com/cells/net/aspose.cells/pastetype) enum, dann werden die Höhen aller Zeilen innerhalb des Quellbereichs in den Zielbereich kopiert.

{{% /alert %}}

Sobald Sie die von diesem Code generierte Excel-Ausgabedatei in Microsoft Excel öffnen, werden Sie sehen, dass die Zeilenhöhen des Zielbereichs genau mit den Zeilenhöhen des Quellbereichs übereinstimmen.
