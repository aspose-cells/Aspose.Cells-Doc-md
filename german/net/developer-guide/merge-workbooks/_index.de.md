{{% alert color="primary" %}}

Manchmal müssen Sie Arbeitsmappen mit verschiedenen Inhalten wie Bildern, Diagrammen und Daten in einer einzigen Arbeitsmappe kombinieren. Aspose.Cells unterstützt diese Funktion. Dieser Artikel zeigt, wie Sie eine Konsolenanwendung in Visual Studio erstellen und Arbeitsmappen mit ein paar einfachen Codezeilen unter Verwendung von Aspose.Cells kombinieren.

{{% /alert %}}

## **Kombinieren von Arbeitsmappen mit Bildern und Diagrammen**

Der Beispielcode kombiniert zwei Arbeitsmappen mithilfe von Aspose.Cells zu einer einzigen Arbeitsmappe. Der Code lädt die Quellarbeitsmappen, verwendet die[**Arbeitsmappe.combine()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/combine)-Methode, um sie zu kombinieren, und speichert die Ausgabearbeitsmappe.
