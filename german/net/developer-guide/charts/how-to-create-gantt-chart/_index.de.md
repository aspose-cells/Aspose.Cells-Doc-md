##  Was ist ein Gantt-Diagramm?

Ein Gantt-Diagramm hilft Ihnen bei der Planung Ihrer Projektaufgaben und hilft Ihnen anschließend, Ihren Fortschritt zu verfolgen.

##  Gantt-Diagramm in Excel hinzufügen

Müssen Sie den Status eines einfachen Projektzeitplans mit einem Gantt-Diagramm anzeigen? Obwohl Excel keinen vordefinierten Gantt-Diagrammtyp hat, können Sie einen simulieren, indem Sie ein gestapeltes Balkendiagramm anpassen, um die Start- und Enddaten von Aufgaben anzuzeigen, wie folgt:

##  So erstellen Sie

-  Wählen Sie die Daten aus, die Sie grafisch darstellen möchten. In unserem Beispiel ist das B1:B7 und dann Einfügen**Gestapelte Bar** Diagramm.

![todo:image_alt_text](1.png)

- Wählen Sie das Diagramm aus,**Daten auswählen**->**Fügen Sie** hinzu und legen Sie den **Seriennamen fest** Und**Serienwerte** wie folgt

![todo:image_alt_text](2.png)

-  Wählen Sie das Diagramm aus und bearbeiten Sie es**Horizontale Achsenbeschriftungen (Kategorie).**

![todo:image_alt_text](3.png)

- **Achse formatieren** die Y-Achse, auswählen**Kategorien in umgekehrter Reihenfolge**
-  Wählen Sie die aus**Blaue Serie** und stellen Sie die ein**Füllen->KEIN Füllen**
- **Achse formatieren** Legen Sie für die

![todo:image_alt_text](4.png)

- **Datendateien hinzufügen** für das Diagramm
Jetzt erhalten Sie ein Gantt-Diagramm.
