##  **Mögliche Nutzungsszenarien**
Wenn die Zahlen in einem Diagramm von Datenreihe zu Datenreihe stark variieren oder Sie über gemischte Datentypen (Preis und Volumen) verfügen, zeichnen Sie eine oder mehrere Datenreihen auf einer sekundären vertikalen Achse (Wertachse) auf. Die Skala der sekundären vertikalen Achse zeigt die Werte für die zugehörigen Datenreihen. Eine sekundäre Achse funktioniert gut in einem Diagramm, das eine Kombination aus Säulen- und Liniendiagrammen zeigt.
