## **Mögliche Nutzungsszenarien**
Wenn die Zahlen in einem Diagramm von Datenreihe zu Datenreihe stark variieren oder wenn Sie gemischte Datentypen (Preis und Volumen) haben, zeichnen Sie eine oder mehrere Datenreihen auf einer sekundären vertikalen (Wert-)Achse. Die Skala der sekundären vertikalen Achse zeigt die Werte für die zugehörige Datenreihe. Eine Sekundärachse funktioniert gut in einem Diagramm, das eine Kombination aus Säulen- und Liniendiagrammen darstellt.
