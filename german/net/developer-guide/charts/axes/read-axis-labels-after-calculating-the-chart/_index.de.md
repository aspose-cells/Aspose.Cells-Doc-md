##  **Mögliche Nutzungsszenarien**
Sie können Achsenbeschriftungen Ihres Diagramms lesen, nachdem Sie dessen Werte mithilfe von berechnet haben[Chart.Calculate()](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/methods/calculate)Methode. Bitte nutzen Sie die[Axis.AxisLabels](https://reference.aspose.com/cells/net/aspose.cells.charts/axis/properties/axislabels)Eigenschaft für diesen Zweck, die die Liste der Achsenbeschriftungen zurückgibt.
