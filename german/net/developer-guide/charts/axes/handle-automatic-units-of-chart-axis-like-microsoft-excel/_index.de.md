##  **Mögliche Nutzungsszenarien**
Frühere Versionen von Aspose.Cells konnten automatische Einheiten der Diagrammachse nicht ordnungsgemäß verarbeiten, wenn das Diagramm als Bild oder PDF gerendert wurde. Jetzt unterstützt Aspose.Cells die Verarbeitung automatischer Einheiten der Diagrammachse. Es gibt keine Codeänderung. Konvertieren Sie Ihr Diagramm einfach in ein Bild oder PDF und die Diagrammachse wird genau so gerendert, wie Microsoft Excel sie rendert.
