{{% alert color="primary" %}}

 Sie können Aspose.Cells verwenden, um eine benutzerdefinierte Rich-Text-Datenbeschriftung des Diagrammpunkts zu erstellen. Aspose.Cells bietet die[**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) Methode zur Rückgabe der[**FontSetting**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) Objekt, mit dem die Schriftarteigenschaften des Textes wie Farbe, Fettigkeit usw. festgelegt werden können.

{{% /alert %}}
