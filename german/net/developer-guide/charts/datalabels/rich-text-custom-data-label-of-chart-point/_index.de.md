{{% alert color="primary" %}}

 Sie können Aspose.Cells verwenden, um eine benutzerdefinierte Rich-Text-Datenbeschriftung des Diagrammpunkts zu erstellen. Aspose.Cells bietet die[**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters) Methode, um die zurückzugeben[**Schrifteinstellung**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting) Objekt, mit dem die Schrifteigenschaften des Textes wie Farbe, Fettdruck usw. eingestellt werden können.

{{% /alert %}}

## Benutzerdefinierte Rich-Text-Datenbeschriftung des Diagrammpunkts

 Der folgende Code greift auf den ersten Diagrammpunkt der ersten Reihe zu, setzt seinen Text und legt dann die Schriftart der ersten 10 Zeichen fest, indem er seine Farbe auf Rot und Fettdruck auf setzt**wahr**.
