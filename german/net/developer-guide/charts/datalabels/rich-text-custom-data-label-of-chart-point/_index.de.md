{{% alert color="primary" %}}

Sie können Aspose.Cells verwenden, um benutzerdefinierte Rich-Text-Datenbeschriftung des Diagrammpunkts zu erstellen. Aspose.Cells bietet die Methode [**DataLabels.Characters()**](https://reference.aspose.com/cells/net/aspose.cells.charts/charttextframe/methods/characters), um das [**FontSetting**](https://reference.aspose.com/cells/net/aspose.cells/fontsetting)-Objekt zurückzugeben, das verwendet werden kann, um die Schrifteigenschaften des Textes wie seine Farbe, Fettgedrucktheit usw. festzulegen.

{{% /alert %}}
