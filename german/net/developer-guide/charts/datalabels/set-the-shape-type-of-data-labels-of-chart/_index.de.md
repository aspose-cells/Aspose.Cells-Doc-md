##  **Mögliche Nutzungsszenarien**
Sie können den Formtyp der Datenbeschriftungen des Diagramms mithilfe der Eigenschaft DataLabels.ShapeType ändern. Es übernimmt den Wert der DataLabelShapeType-Enumeration und ändert den Formtyp der Datenbeschriftungen entsprechend. Einige seiner Werte sind

{{< highlight "java" >}}

 DataLabelShapeType.BentLineCallout

DataLabelShapeType.DownArrowCallout

DataLabelShapeType.Ellipse

DataLabelShapeType.LineCallout

DataLabelShapeType.Rect

etc.

{{< /highlight >}}
