{{% alert color="primary" %}}

Microsoft Excel 2013 ermöglicht es Benutzern, Text innerhalb der Datenbeschriftungen des Diagramms umzubrechen oder aufzulösen. Standardmäßig befindet sich der Text in den Datenbeschriftungen des Diagramms im umbrochenen Zustand.

 Aspose.Cells bietet eine[**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) -Eigenschaft, die Sie auf True oder False setzen können, um den Textumbruch von Datenbeschriftungen zu aktivieren bzw. zu deaktivieren.

{{% /alert %}}
