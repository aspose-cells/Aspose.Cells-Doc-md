## **Mögliche Nutzungsszenarien**
Sie können den Wertformatcode von Diagrammserien mit festlegen[Series.ValuesFormatCode](https://reference.aspose.com/cells/net/aspose.cells.charts/series/properties/valuesformatcode)Eigentum. Diese Eigenschaft ist nicht nur für Reihen nützlich, die auf dem Bereich innerhalb des Arbeitsblatts basieren, sondern funktioniert auch gut für Reihen, die mit einem Array von Werten erstellt wurden.
## **Legen Sie den Werteformatcode der Diagrammreihe fest**
Der folgende Beispielcode fügt dem leeren Diagramm eine Reihe hinzu, die vorher keine Reihe hat. Es fügt die Reihe unter Verwendung des Arrays von Werten hinzu. Sobald es die Serie hinzugefügt hat, formatiert es sie mit dem Code $#,##0 unter Verwendung von[Series.ValuesFormatCode](https://reference.aspose.com/cells/net/aspose.cells.charts/series/properties/valuesformatcode)Eigentum und die Zahl 10000 wird zu 10.000 $.
