## **Mögliche Verwendungsszenarien**
Sie können den Werteformatcode der Diagrammserie mithilfe der [Series.ValuesFormatCode](https://reference.aspose.com/cells/net/aspose.cells.charts/series/properties/valuesformatcode) Eigenschaft einstellen. Diese Eigenschaft ist nicht nur für die Serie nützlich, die auf dem Bereich im Arbeitsblatt basiert, sondern funktioniert auch gut für die mit einem Wertearray erstellte Serie.
## **Setzen des Werteformatcodes der Diagrammserie**
Der folgende Beispielcode fügt dem leeren Diagramm eine Serie hinzu, die zuvor keine Serie hatte. Es fügt die Serie mit dem Wertearray hinzu. Sobald die Serie hinzugefügt ist, formatiert sie diese mit dem Code $#,##0 mithilfe der [Series.ValuesFormatCode](https://reference.aspose.com/cells/net/aspose.cells.charts/series/properties/valuesformatcode) Eigenschaft und die Zahl 10000 wird zu $10.000.
