{{% alert color="primary" %}}

Aspose.Cells ermöglicht es Ihnen, einen Farbverlauf, eine Textur, ein Muster oder ein Bild als Fülleffekte für verschiedene Objekte, wie z.B. den Diagrammbereich, den Chartbereich oder das Legendenfeld eines Diagramms, festzulegen. Dieses Dokument zeigt, wie Sie ein Bild als Hintergrund für ein Diagramm hinzufügen können.

{{% /alert %}}

Um dies zu erreichen, bietet Aspose.Cells die [**Chart.PlotArea.Area.FillFormat.ImageData**](https://reference.aspose.com/cells/net/aspose.cells.drawing/fillformat/properties/imagedata) Eigenschaft. Das folgende Codebeispiel demonstriert die Verwendung der [**Chart.PlotArea.Area.FillFormat.ImageData**](https://reference.aspose.com/cells/net/aspose.cells.drawing/fillformat/properties/imagedata)-Eigenschaft, um ein Bild als Hintergrundfüllung im Diagramm einzustellen.
