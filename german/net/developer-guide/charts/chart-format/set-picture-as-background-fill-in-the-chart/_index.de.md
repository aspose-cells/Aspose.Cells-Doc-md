Aspose.Cells ermöglicht es Ihnen, einen Verlauf, eine Textur, ein Muster oder ein Bild als Fülleffekte für verschiedene Objekte festzulegen, z. B. den Plotbereich, den Diagrammbereich oder das Legendenfeld eines Diagramms. Dieses Dokument zeigt, wie Sie dem Hintergrund eines Diagramms ein Bild hinzufügen.

{{% /alert %}}
