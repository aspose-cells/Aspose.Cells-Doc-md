{{% alert color="primary" %}}

Manchmal müssen Sie das Format einer Datei erkennen, bevor Sie sie öffnen, da die Dateierweiterung nicht garantieren kann, dass der Dateiinhalt geeignet ist. Die Datei könnte verschlüsselt sein (eine passwortgeschützte Datei), sodass sie nicht direkt gelesen werden kann oder nicht gelesen werden sollte. Aspose.Cells bietet die [**FileFormatUtil.DetectFileFormat()**](https://reference.aspose.com/cells/net/aspose.cells/fileformatutil/methods/detectfileformat/index)-Methode und einige relevante APIs, die Sie zum Verarbeiten von Dokumenten nutzen können.

{{% /alert %}}
