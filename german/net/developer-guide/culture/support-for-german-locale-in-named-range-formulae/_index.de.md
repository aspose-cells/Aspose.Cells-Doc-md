Englische Formeln werden in einen benannten Bereich geschrieben. Diese Excel-Datei kann in einer Umgebung geöffnet werden, in der das System auf das deutsche Gebietsschema konfiguriert ist, jedoch werden die englischen Formeln ins Deutsche übersetzt. Das folgende Beispiel zeigt diese Funktion, erfordert jedoch, dass Excel auf Deutsch installiert ist und das Systemgebietsschema ebenfalls auf Deutsch eingestellt ist.
