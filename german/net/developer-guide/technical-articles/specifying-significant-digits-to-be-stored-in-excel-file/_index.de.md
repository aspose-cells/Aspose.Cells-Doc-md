## **Mögliche Nutzungsszenarien**

Standardmäßig speichert Aspose.Cells 17 signifikante Ziffern von Double-Werten in der Excel-Datei, im Gegensatz zu MS-Excel, das nur 15 signifikante Ziffern speichert. Sie können das Standardverhalten von Aspose.Cells mithilfe von von 17 signifikanten Ziffern auf 15 signifikante Ziffern ändern[**CellsHelper.SignificantDigits**](https://reference.aspose.com/cells/net/aspose.cells/cellshelper/properties/significantdigits)Eigentum.
