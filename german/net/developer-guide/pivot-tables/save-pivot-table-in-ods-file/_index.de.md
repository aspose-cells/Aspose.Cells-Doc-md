Aspose.Cells ermöglicht das Speichern einer Pivot-Tabelle in ODS-Dateien. Dazu konvertieren Sie einfach eine Arbeitsmappe mit einer vorhandenen Pivot-Tabelle oder erstellen eine neue Pivot-Tabelle und speichern die Datei im ODS-Format. Stellen Sie sicher, dass [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata) aufgerufen wird, bevor Sie speichern, um sicherzustellen, dass die Pivot-Tabelle in der Ausgabedatei gerendert wird. Der folgende Code-Ausschnitt zeigt das Speichern der Pivot-Tabelle in einer ODS-Datei.
