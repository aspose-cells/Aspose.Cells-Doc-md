 Aspose.Cells bietet die Möglichkeit, Pivot-Tabellen in ODS-Dateien zu speichern. Dazu konvertieren Sie einfach eine Arbeitsmappe mit vorhandener Pivot-Tabelle oder erstellen eine neue Pivot-Tabelle und speichern die Datei im ODS-Format. Unbedingt anrufen[**PivotTable.CalculateData()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata) Stellen Sie vor dem Speichern sicher, dass die Pivot-Tabelle in der ODS-Ausgabedatei gerendert wird. Das folgende Code-Snippet demonstriert das Speichern einer Pivot-Tabelle in einer ODS-Datei.
