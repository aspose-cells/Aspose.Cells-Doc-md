{{% alert color="primary" %}}

Manchmal möchten Sie Pivottabellenzellen formatieren. Zum Beispiel möchten Sie Hintergrundfarbe auf Pivottabellenzellen anwenden. Aspose.Cells bietet zwei Methoden [**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) und [**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format), die Sie zu diesem Zweck verwenden können.

[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) wendet den Stil auf die gesamte Pivottabelle an, während [**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format) den Stil auf eine einzelne Zelle der Pivottabelle anwendet.

{{% /alert %}}
