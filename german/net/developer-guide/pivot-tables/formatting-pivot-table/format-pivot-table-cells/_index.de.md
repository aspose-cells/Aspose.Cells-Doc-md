 Manchmal möchten Sie Pivot-Tabellenzellen formatieren. Sie möchten beispielsweise eine Hintergrundfarbe auf Pivot-Tabellenzellen anwenden. Aspose.Cells bietet zwei Methoden[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) und[**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format), die Sie zu diesem Zweck verwenden können.

[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) wendet den Stil während auf die gesamte Pivot-Tabelle an[**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format) wendet den Stil auf eine einzelne Zelle der Pivot-Tabelle an.
