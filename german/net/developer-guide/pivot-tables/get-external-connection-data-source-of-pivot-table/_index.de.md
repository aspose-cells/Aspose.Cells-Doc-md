## **Externe Verbindungsdatenquelle der Pivot-Tabelle abrufen**

Aspose.Cells bietet die Möglichkeit, die externe Verbindungsdatenquelle der Pivot-Tabelle abzurufen. Dafür sorgt die API[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)Eigentum der[**PivotTable**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) Klasse. Das[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)Eigenschaft gibt die zurück[**Externe Verbindung**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) Objekt. Das folgende Code-Snippet demonstriert die Verwendung von[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) -Eigenschaft, um die externe Verbindungsdatenquelle der Pivot-Tabelle abzurufen.
