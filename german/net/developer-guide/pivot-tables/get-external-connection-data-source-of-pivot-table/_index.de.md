## **Abrufen der externen Verbindungsdatenquelle der Pivot-Tabelle**

Aspose.Cells bietet die Möglichkeit, die externe Verbindungsdatenquelle der Pivot-Tabelle abzurufen. Hierfür stellt die API die Eigenschaft [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) der Klasse [**PivotTable**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) bereit. Die Eigenschaft [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) gibt das Objekt [**ExternalConnection**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) zurück. Der folgende Codeausschnitt zeigt die Verwendung der Eigenschaft [**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) zur Abfrage der externen Verbindungsdatenquelle der Pivot-Tabelle.
