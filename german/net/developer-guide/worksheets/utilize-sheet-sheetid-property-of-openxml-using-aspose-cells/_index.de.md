##  **Mögliche Nutzungsszenarien**

*Sheet.SheetId*Die Immobilie befindet sich im Inneren*DocumentFormat.OpenXml.Spreadsheet* Namespace und ist Teil von OpenXml. Sie können diese Eigenschaft und ihren Wert im Inneren sehen*arbeitsmappe.xml* wie im folgenden Screenshot gezeigt. Aspose.Cells bietet die entsprechende Eigenschaft als[**Worksheet.TabId**](https://reference.aspose.com/cells/net/aspose.cells/worksheet/properties/tabid).
