## **Mögliche Nutzungsszenarien**

*Sheet.SheetId*Eigentum befindet sich innerhalb der*DocumentFormat.OpenXml.Spreadsheet*Namespace und ist Teil von OpenXml. Sie können diese Eigenschaft und ihren Wert im Inneren sehen*Arbeitsmappe.xml* wie im folgenden Screenshot gezeigt. Aspose.Cells bietet die entsprechende Eigenschaft als[**Arbeitsblatt.TabId**](https://reference.aspose.com/cells/net/aspose.cells/worksheet/properties/tabid).
