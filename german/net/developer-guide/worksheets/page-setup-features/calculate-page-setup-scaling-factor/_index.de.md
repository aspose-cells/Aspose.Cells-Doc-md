{{% alert color="primary" %}}

Wenn Sie die Seiteneinrichtungsskalierung mit festlegen**Anpassen an n Seite(n) breit und m hoch** Option, Microsoft Excel berechnet den Seiteneinrichtungs-Skalierungsfaktor. Sie können dasselbe mit berechnen[**SheetRender.PageScale**](https://reference.aspose.com/cells/net/aspose.cells.rendering/sheetrender/properties/pagescale) Eigentum. Diese Eigenschaft gibt einen Double-Wert zurück, der in einen Prozentwert konvertiert werden kann. Wenn beispielsweise 0,5 zurückgegeben wird, bedeutet dies, dass der Skalierungsfaktor 50 % beträgt.
