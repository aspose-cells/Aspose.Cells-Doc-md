## **Mögliche Nutzungsszenarien**

Sie können den Slicer in Microsoft Excel formatieren, indem Sie seine Spaltenanzahl oder seinen Stil usw. festlegen. Mit Aspose.Cells können Sie dies auch tun[**Slicer.NumberOfColumns**](https://reference.aspose.com/cells/net/aspose.cells.slicers/slicer/properties/numberofcolumns)und[**Slicer.StyleType**](https://reference.aspose.com/cells/net/aspose.cells.slicers/slicer/properties/styletype)Eigenschaften.

## **Slicer formatieren**

Bitte sehen Sie sich den folgenden Code an, er lädt die[Beispiel-Excel-Datei](67338473.xlsx)die einen Slicer enthält. Es greift auf den Slicer zu, legt die Anzahl der Spalten und den Stiltyp fest und speichert es als[Excel-Datei ausgeben](67338474.xlsx).
