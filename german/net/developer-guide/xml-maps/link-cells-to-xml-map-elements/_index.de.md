## **Mögliche Nutzungsszenarien**

Sie können Ihre Zellen mit Aspose.Cells mit XML-Map-Elementen verknüpfen. Bitte verwenden Sie die[**Cells.LinkToXmlMap()**](https://reference.aspose.com/cells/net/aspose.cells/cells/methods/linktoxmlmap)Methode zu diesem Zweck.

## **Link Cells zu XML-Kartenelementen**

 Der folgende Beispielcode lädt die[Excel-Quelldatei](5115471.xlsx) die XML Map enthält und dann die Zellen A1, B2, C3, D4, E5 und F6 mit den XML Map-Elementen FIELD1, FIELD2, FIELD4, FIELD5, FIELD7 und FIELD8 verknüpft und dann die Arbeitsmappe in speichert[Excel-Datei ausgeben](5115467.xlsx).

 Wenn Sie die öffnen[Excel-Datei ausgeben](5115467.xlsx) und klicken Sie auf die Schaltfläche Entwickler > Quelle, Sie werden sehen, dass die Zellen mit XML-Kartenelementen verknüpft sind, und sie werden auch von Microsoft Excel hervorgehoben, wie in diesem Bild gezeigt.
