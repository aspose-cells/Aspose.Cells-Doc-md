Der folgende Beispielcode exportiert die XML-Daten aller XML-Maps aus der Arbeitsmappe nacheinander. Überprüfen Sie bitte die [Beispieldatei](5115497.xlsx), die in diesem Code verwendet wurde, und die [exportierten XML-Daten der ersten XML-Map](5472487.xml).
