Aspose.Cells bietet die Option zum Festlegen einer benutzerdefinierten Datenquelle für WorkbookDesigner. Die API stellt eine überladene Methode bereit[WorkbookDesigner.SetDataSource](https://reference.aspose.com/cells/net/aspose.cells.workbookdesigner/setdatasource/methods/5)die den Namen der Quelle als ersten Parameter und die Instanz der Klasse, die implementiert, verwendet[ICellsDataTable](https://reference.aspose.com/cells/net/aspose.cells/icellsdatatable)als zweiter Parameter.

Das folgende Code-Snippet demonstriert die Verwendung von[WorkbookDesigner.SetDataSource](https://reference.aspose.com/cells/net/aspose.cells.workbookdesigner/setdatasource/methods/5)-Methode zum Festlegen der benutzerdefinierten DataSource.
