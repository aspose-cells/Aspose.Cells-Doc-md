Aspose.Cells bietet die Möglichkeit, eine benutzerdefinierte Datenquelle für WorkbookDesigner festzulegen. Die API bietet eine overloading Methode [WorkbookDesigner.SetDataSource](https://reference.aspose.com/cells/net/aspose.cells.workbookdesigner/setdatasource/methods/5), die den Namen der Quelle als ersten Parameter und die Instanz der Klasse, die [ICellsDataTable](https://reference.aspose.com/cells/net/aspose.cells/icellsdatatable) implementiert, als zweiten Parameter annimmt. 

Der folgende Codeschnipsel zeigt die Verwendung der Methode [WorkbookDesigner.SetDataSource](https://reference.aspose.com/cells/net/aspose.cells.workbookdesigner/setdatasource/methods/5), um die benutzerdefinierte Datenquelle festzulegen.
