## Überprüfen Sie, ob das VBA-Projekt geschützt und für die Anzeige in C# gesperrt ist

Mit Aspose.Cells können Sie überprüfen, ob das VBA-Projekt (Visual Basic for Applications) der Excel-Datei geschützt und für die Anzeige gesperrt ist. Dafür sorgt die API[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) Eigentum. Wenn es für die Anzeige gesperrt ist, dann die[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing) Eigenschaft Renditen**Stimmt**.
