Aspose.Cells ermöglicht es dem Benutzer zu prüfen, ob das VBA-Codeprojekt signiert ist oder nicht. Bitte verwenden Sie die [**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned)-Eigenschaft, um zu überprüfen, ob das VBA-Codeprojekt signiert ist oder nicht.

Sie können beliebige Excel-Dateien für diesen Code verwenden. Zu Testzwecken können Sie [diese im Code verwendete Excel-Datei](5115032.xlsm) verwenden.
