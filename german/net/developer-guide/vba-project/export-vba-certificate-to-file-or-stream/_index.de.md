 Aspose.Cells ermöglicht Ihnen das Exportieren des digitalen VBA-Zertifikats in einen Stream, wie z. B. einen Datei- oder Speicherstream. Auf die Rohdaten des digitalen VBA-Zertifikats können Sie über zugreifen[**Workbook.VbaProject.CertRawData**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/certrawdata) Eigentum.
