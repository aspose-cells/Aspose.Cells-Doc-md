 Sie können überprüfen, ob Ihr VBA-Projekt signiert ist oder nicht, indem Sie Microsoft Excel über verwenden**Extras > Digitale Signaturen...** Menübefehl. Ebenso können Sie es programmgesteuert mit Aspose.Cells überprüfen[**Workbook.VbaProject.IstSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned) Eigentum.
