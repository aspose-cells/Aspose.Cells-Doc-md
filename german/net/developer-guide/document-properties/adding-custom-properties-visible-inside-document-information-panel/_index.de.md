## **Hinzufügen von benutzerdefinierten Eigenschaften, die im Dokumentinformationsbereich sichtbar sind**

Aspose.Cells kann verwendet werden, um benutzerdefinierte Eigenschaften innerhalb des Arbeitsmappenobjekts hinzuzufügen, die im Dokumentinformationsbereich sichtbar sind. Sie können den Dokumentinformationsbereich in Microsoft Excel mit den Menübefehlen „Datei“ > „Info“ > „Eigenschaften“ > „Dokumentbereich anzeigen“ öffnen.

 Bitte verwende[**Workbook.ContentTypeProperties.Add()**](https://reference.aspose.com/cells/net/aspose.cells.properties/contenttypepropertycollection/methods/add/index)-Methode, um eine benutzerdefinierte Eigenschaft hinzuzufügen, die im Dokumentinformationsbereich sichtbar ist

Der folgende Beispielcode fügt zwei benutzerdefinierte Eigenschaften hinzu. Die erste Eigenschaft hat keinen Typ und die zweite Eigenschaft hat einen Typ wie DateTime. Sobald Sie die von diesem Code generierte Excel-Ausgabedatei öffnen, sehen Sie diese beiden Eigenschaften im Dokumentinformationsbereich.
