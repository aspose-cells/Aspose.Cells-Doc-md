## **Mögliche Verwendungsszenarien**

Sie können die **Versionsnummer** der Excel-Datei ändern, indem Sie mit der rechten Maustaste auf die Datei klicken und dann Eigenschaften > Details auswählen und anschließend das Feld **Versionsnummer** bearbeiten. Verwenden Sie bitte die [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion)-Eigenschaft, um sie programmgesteuert mithilfe von Aspose.Cells-APIs zu ändern.
