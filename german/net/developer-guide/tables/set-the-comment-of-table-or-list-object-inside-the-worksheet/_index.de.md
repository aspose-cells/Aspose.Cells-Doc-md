{{% alert color="primary" %}}

Sie können den Kommentar für die Tabelle oder das Listenobjekt im Arbeitsblatt mithilfe der [**ListObject.Comment**](https://reference.aspose.com/cells/net/aspose.cells.tables/listobject/properties/comment) setzen-Eigenschaft. Der Kommentar wird in der Datei xl/tables/tableName.xml sichtbar sein.

{{% /alert %}}
