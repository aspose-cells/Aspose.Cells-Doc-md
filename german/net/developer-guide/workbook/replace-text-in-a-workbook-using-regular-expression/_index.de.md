 Aspose.Cells bietet die Funktion zum Ersetzen von Text in einer Arbeitsmappe mithilfe eines regulären Ausdrucks. Dafür sorgt die API[**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey)Eigentum der[**Optionen ersetzen**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions) Klasse. Einstellung der[**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) zu**wahr** gibt an, dass der gesuchte Schlüssel ein regulärer Ausdruck sein wird.
