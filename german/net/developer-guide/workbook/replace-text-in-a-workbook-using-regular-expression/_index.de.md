Aspose.Cells bietet die Möglichkeit, Text in einer Arbeitsmappe mithilfe eines regulären Ausdrucks zu ersetzen. Hierfür bietet die API die [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey)-Eigenschaft der Klasse [**ReplaceOptions**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions). Das Setzen des [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey) auf **true** gibt an, dass der gesuchte Schlüssel ein regulärer Ausdruck ist.

Der folgende Codeausschnitt demonstriert die Verwendung der [**RegexKey**](https://reference.aspose.com/cells/net/aspose.cells/replaceoptions/properties/regexkey)-Eigenschaft, indem die [Beispiel-Excel-Datei](101089318.xlsx) verwendet wird. Die durch den folgenden Codeausschnitt generierte [Ausgabedatei](101089319.xlsx) finden Sie im Anhang.
