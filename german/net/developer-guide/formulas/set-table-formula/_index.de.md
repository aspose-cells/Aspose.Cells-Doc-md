## **Mögliche Nutzungsszenarien**
 Manchmal möchten Sie, dass eine Formel in Ihrem Tabellen- oder Listenobjekt automatisch auf neue Zeilen übertragen wird, während Sie neue Daten eingeben. Dies ist das Standardverhalten von Microsoft Excel. Um dasselbe mit Aspose.Cells zu erreichen, verwenden Sie bitte[Listenspalte.Formel](https://reference.aspose.com/cells/net/aspose.cells.tables/listcolumn/properties/formula)Eigentum.
