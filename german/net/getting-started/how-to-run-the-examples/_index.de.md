## **Software-Anforderungen**
Stellen Sie bitte sicher, dass Sie die folgenden Anforderungen erfüllen, bevor Sie die Beispiele herunterladen und ausführen.

1. Visual Studio 2015 oder höher
1. NuGet Package Manager in Visual Studio installiert. Es ist meistens bereits in Visual Studio 2015 installiert. Für Details zur Installation des NuGet Package Managers siehe: [Installation der NuGet-Clienttools](https://docs.microsoft.com/de-de/nuget/install-nuget-client-tools)
1. Gehen Sie zu Tools->Optionen->NuGet Package Manager->Paketquellen und stellen Sie sicher, dass die Option **nuget.org** ausgewählt ist
1. Das Beispielprojekt verwendet die NuGet Automatic Package Restore-Funktion, daher sollten Sie eine aktive Internetverbindung haben. Wenn Sie keine aktive Internetverbindung auf dem Rechner, auf dem die Beispiele ausgeführt werden sollen, haben, überprüfen Sie bitte [Installation](/cells/de/net/installation-and-deployment/) und fügen Sie manuell eine Referenz zu Aspose.Cells.dll im Beispielprojekt hinzu.
## **Von GitHub herunterladen**
Alle Beispiele von Aspose.Cells for .NET werden auf [GitHub](https://github.com/aspose-cells/Aspose.Cells-for-.NET) gehostet.
## **Aspose.Cells Beispiele**
- Sie können entweder das Repository mit Ihrem bevorzugten GitHub-Client klonen oder die ZIP-Datei von [hier](https://github.com/aspose-cells/Aspose.Cells-for-.NET/archive/master.zip) herunterladen.
- Entpacken Sie den Inhalt der ZIP-Datei in einen beliebigen Ordner auf Ihrem Computer. Alle Beispiele befinden sich im **Beispiele**-Ordner.
- Es gibt eine Visual Studio-Lösungsdatei für C#-Beispiele, d.h. **Aspose.Cells.Examples.CSharp.sln**.
- Das Projekt wird in Visual Studio 2015 erstellt und gewartet.
- Öffnen Sie die Lösungsdatei in Visual Studio und erstellen Sie das Projekt.
- Beim ersten Start werden die Abhängigkeiten automatisch über NuGet heruntergeladen. Sie können die DLLs auch separat von [hier](https://downloads.aspose.com/cells/net) herunterladen.
- Der **Data**-Ordner im Stammverzeichnis der **Beispiele** enthält Eingabedateien, die von CSharp-Beispielen verwendet werden. Es ist zwingend erforderlich, dass Sie den **Data**-Ordner zusammen mit dem Beispiele-Projekt herunterladen.
- Öffnen Sie **RunExamples.cs**, alle Beispiele werden von hier aus aufgerufen.
- Kommentieren Sie die Beispiele, die Sie im Projekt ausführen möchten, aus.
## **Aspose.Cells.GridDesktop Beispiele**
- Aspose.Cells.GridDesktop-Beispiele sind ebenfalls im [Aspose.Cells GitHub](https://github.com/aspose-cells/Aspose.Cells-for-.NET)-Repository enthalten und werden als Teil der ZIP-Datei, die von [hier](https://github.com/aspose-cells/Aspose.Cells-for-.NET/archive/master.zip) heruntergeladen werden kann, verfügbar sein.
- Alle Beispiele befinden sich im **Examples_GridDesktop**-Ordner.
- Ähnlich wie bei den Aspose.Cells-Beispielen ist der Name der Lösungsdatei für die GridWeb-Beispiele **Aspose.Cells.GridDesktop.Examples.CSharp.sln**.
- Öffnen Sie die Lösungsdatei in Visual Studio und erstellen Sie das Projekt.
- Alle Abhängigkeiten sind Bestandteil des Beispiele-Projekts. Sie können die DLLs auch separat von [hier](https://downloads.aspose.com/cells/net) herunterladen.
- Der **Data**-Ordner im Stammverzeichnis von **Examples_GridDesktop** enthält von Beispielen verwendete Eingabedateien. Es ist zwingend erforderlich, dass Sie den **Data**-Ordner zusammen mit dem Beispiele-Projekt herunterladen.
- Öffnen und führen Sie das Projekt aus.
- Klicken Sie im Menü auf das Beispiel, das Sie von innerhalb des Formulars ausführen möchten.
## **Beispiele für Aspose.Cells.GridWeb**
- Beispiele für Aspose.Cells.GridWeb sind ebenfalls im [Aspose.Cells GitHub](https://github.com/aspose-cells/Aspose.Cells-for-.NET) Repository enthalten und werden als Teil der herunterladbaren ZIP-Datei von [hier](https://github.com/aspose-cells/Aspose.Cells-for-.NET/archive/master.zip) verfügbar sein.
- Alle Beispiele befinden sich im Ordner **Examples_GridWeb**.
- Ähnlich wie bei den Aspose.Cells-Beispielen ist der Lösungsdateiname für GridWeb-Beispiele **Aspose.Cells.GridWeb.Examples.CSharp.sln**.
- Öffnen Sie die Lösungsdatei in Visual Studio und erstellen Sie das Projekt.
- Alle Abhängigkeiten sind als Teil der Beispielsprojekte enthalten. Sie können die DLLs auch separat von [hier](https://downloads.aspose.com/cells/net) herunterladen.
- Der **Data**-Ordner im Stammverzeichnis von **Examples_GridWeb** enthält Eingabedateien, die von den Beispielen verwendet werden. Es ist zwingend erforderlich, dass Sie den **Data**-Ordner zusammen mit dem Beispielsprojekt herunterladen.
- Öffnen und führen Sie **Examples.aspx** im Beispielsprojekt aus.
- Klicken Sie im Browser auf das Beispiel, das Sie im Projekt ausführen möchten.
