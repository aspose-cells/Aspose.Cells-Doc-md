## **Installieren Sie Aspose.Cells for .NET bis NuGet**
NuGet ist die einfachste Möglichkeit, Aspose-APIs for .NET herunterzuladen und zu installieren. Öffnen Sie Microsoft Visual Studio und den NuGet-Paket-Manager. Suchen Sie "aspose", um die gewünschte Aspose API zu finden. Klicken Sie auf "Installieren", die ausgewählte API wird heruntergeladen und in Ihrem Projekt referenziert.

Hinweis: Weitere Informationen finden Sie auf der Webseite nuget für aspose.cells:
[Aspose.Cells for .NET NuGet Paket](https://www.nuget.org/packages/Aspose.Cells/)

### **Installieren Sie Aspose.Cells mithilfe der Package Manager-GUI**
Befolgen Sie diese Schritte, um die Komponente Aspose.Cells mithilfe der Paketmanager-GUI zu referenzieren:

- Öffnen Sie Ihre Lösung/Ihr Projekt in Visual Studio.
- Klicken**Werkzeug** -> **Bibliothekspaket-Manager** -> **NuGet-Pakete verwalten**von Lösung. Sie können auf dieselbe Option auch ganz einfach über den Projektmappen-Explorer zugreifen. Klicken Sie mit der rechten Maustaste auf die Lösung oder das Projekt, und wählen Sie sie aus**NuGet-Pakete verwalten**aus dem Kontextmenü.
- Wählen**Online**aus dem linken Menü und geben Sie „Aspose.Cells“ in das Suchtextfeld ein, um das Paket Aspose.Cells .NET zu finden.
- Drücke den**Installieren**Schaltfläche neben dem Eintrag Aspose.Cells for .NET, um die neueste Version in Ihrem Projekt zu installieren.
### **Installieren Sie Aspose.Cells mithilfe der Paket-Manager-Konsole**
Sie können die folgenden Schritte ausführen, um die Komponente Aspose.Cells mithilfe der Paket-Manager-Konsole zu referenzieren:

- Öffnen Sie Ihre Lösung/Ihr Projekt in Visual Studio.
- Wählen**Werkzeug** -> **Bibliothekspaket-Manager** -> **Paket-Manager-Konsole**aus dem Menü, um die Paketmanager-Konsole zu öffnen.
 Geben Sie den Befehl „Install-Package Aspose.Cells“ ein und drücken Sie die Eingabetaste, um die neueste Vollversion in Ihrer Anwendung zu installieren. Alternativ können Sie das Suffix „-prerelease“ an den Befehl anhängen, um anzugeben, dass auch die neuste Version inklusive Hotfixes installiert werden soll.
- Sie werden sehen, dass der Tipp "Aspose.Cells wird heruntergeladen..." unten links im Fenster angezeigt wird, was darauf hinweist, dass der Download läuft.
- Nach dem Herunterladen sehen Sie die folgenden Bestätigungsmeldungen. Wenn Sie mit der EULA Aspose nicht vertraut sind, ist es eine gute Idee, die Lizenz zu lesen, auf die in der URL verwiesen wird.
- Sie sollten nun feststellen, dass Aspose.Cells erfolgreich hinzugefügt und in Ihrer Bewerbung für Sie referenziert wurde.
## **Verweis auf Aspose.Cells aus einem .NET-Projekt**
Um Aspose.Cells in einer Anwendung zu verwenden, fügen Sie einen Verweis darauf hinzu. So fügen Sie eine Referenz mit Visual Studio hinzu:

1.  Im**Lösungsforscher**, erweitern Sie den Projektknoten, dem Sie einen Verweis hinzufügen möchten.
1.  Klicken Sie mit der rechten Maustaste auf die**Verweise** Knoten für das Projekt und wählen Sie aus**Referenz hinzufügen** aus dem Menü.
1.  Im**Referenz hinzufügen** Wählen Sie im Dialogfeld die Registerkarte .NET (standardmäßig ausgewählt). Wenn Sie mit dem MSI-Installationsprogramm installiert haben, wird Aspose.Cells im oberen Bereich angezeigt.
1.  Wählen Sie es aus und klicken Sie darauf**Wählen**.

Wenn Sie nur die DLL heruntergeladen oder entpackt haben:

1.  Suchen Sie die Datei Aspose.Cells.dll mithilfe von**Durchsuche** Knopf. Aspose.Cells sollte in der angezeigt werden**Ausgewählte Komponenten** Bereich des Dialogfelds.
1.  Klicken**OK** . Die Referenz Aspose.Cells erscheint unter der**Verweise** Knoten des Projekts.
### **Verweisen auf die Komponente aus einem VS.NET 2010 Client Profile-Projekt**
Wenn das Zielframework Ihres Projekts .NET Framework 3.5/4 Client Profile ist, verwenden Sie die Komponentendatei Aspose.Cells.dll, die sich im Ordner net_ClientProfile Ihres Installationsverzeichnisses befindet. Zum Beispiel:

-  In**Lösungsforscher** von VS.NET 2010 für Ihr Projekt mit der rechten Maustaste**Verweise** und auswählen**Referenz hinzufügen**.
-  Wähle aus**Durchsuche** Registerkarte im Dialogfeld und klicken Sie auf Dropdown-Liste suchen.
- Suchen Sie die Komponentendatei Aspose.Cells.dll in Ihrem Installationsverzeichnis und wählen Sie sie aus, zum Beispiel: .../Program Files/Aspose/Aspose.Cells for .NET/Bin/net_ClientProfile/ **(Stellen Sie sicher, dass Sie das Produkt mit dem MSI-Installationsprogramm auf Ihrem Computer installiert haben .)**
-  Klicken**OK** um den Dialog zu schließen

{{% alert color="primary" %}} 

Wenn das Zielframework Ihres VS.NET 2010-Projekts „.NET Framework 4“ oder ein anderes ist, verwenden Sie einfach die Komponentendatei Aspose.Cells.dll, die sich im Ordner net4.0/net2.0 befindet.

## **Verweis auf Aspose.Cells-Grid-Steuerelemente aus einem .NET-Projekt**
Um ein Rastersteuerelement in Ihrer Anwendung zu verwenden, fügen Sie einen Verweis darauf hinzu. Gehen Sie wie folgt vor, um auf ein Rastersteuerelement in Visual Studio zu verweisen:

-  In**Lösungsforscher**, erweitern Sie den Projektknoten, dem Sie einen Verweis hinzufügen möchten.
-  Klicken Sie mit der rechten Maustaste auf die**Verweise** Knoten für das Projekt und wählen Sie aus**Referenz hinzufügen** aus dem Menü.
-  Im**Referenz hinzufügen** Dialogfeld wählen Sie die**.NET Registerkarte** (standardmäßig ausgewählt). Wenn Sie das MSI-Installationsprogramm verwendet haben, um Aspose.Cells for .NET zu installieren, werden Aspose.Cells.GridDesktop und Aspose.Cells.GridWeb im oberen Bereich angezeigt.
-  Wählen Sie sie aus und klicken Sie**Wählen**.
- Die Verweise Aspose.Cells.GridDesktop und Aspose.Cells.GridWeb werden unter dem Knoten Verweise des Projekts angezeigt.

Wenn Sie nur die DLL heruntergeladen und entpackt haben:

-  Suchen Sie die Dateien Aspose.Cells.GridDesktop.dll und Aspose.Cells.GridWeb.dll mithilfe der Schaltfläche Durchsuchen. Aspose.Cells Grid Suite wurde referenziert und sollte in der erscheinen**Ausgewählte Komponenten** Bereich des Dialogfelds.
-  Klicken**OK.**
## **Aspose.Cells for .NET deinstallieren**
Wenn Sie das MSI-Installationsprogramm zum Bereitstellen von Aspose.Cells for .NET verwendet haben, gehen Sie folgendermaßen vor, um die Komponente und die Steuerelemente, die zugehörigen Demos und Dokumentationen vollständig zu entfernen:

-  Von dem**Start** Menü, auswählen**Einstellungen** , gefolgt von**Schalttafel**.
-  Klicken**Hinzufügen entfernen Programme**.
- Wählen Sie Aspose.Cells for .NET (Version).
-  Klicken**Ändern/Entfernen** Aspose.Cells zu entfernen.
