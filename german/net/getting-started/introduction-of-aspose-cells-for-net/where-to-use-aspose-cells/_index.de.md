{{% alert color="primary" %}} 

Aspose.Cells wurde entwickelt, um Entwicklern dabei zu helfen, Microsoft Excel-Tabellen programmgesteuert zu erstellen und zu verwalten. Aspose.Cells versteht nicht nur Tabellenkalkulationen im Microsoft Excel Sheet Format (XLS/XLSX), sondern auch viele andere Dateiformate wie Comma Separated Values (CSV), Tab Delimited/TSV (Text), SpreadsheetML etc.

Aspose.Cells bietet die meisten Funktionen, die in Microsoft Excel verfügbar sind, aber das Spannende ist, dass Aspose.Cells Excel-Dateien dynamisch verarbeiten kann, was es Entwicklern ermöglicht, Tausende oder Millionen von Tabellenkalkulationen sehr schnell zu verarbeiten.

In diesem Artikel werden allgemeine Szenarien für die Verwendung von Aspose.Cells erläutert.

{{% /alert %}} 
## **Mögliche Nutzungsszenarien**
Es liegt an den Entwicklern, wie sie Aspose.Cells verwenden, aber es gibt einige gängige Szenarien, in denen Aspose.Cells hilfreich sein kann. Die häufigste Verwendung von Aspose.Cells ist die Bereitstellung auf dem Webserver oder in Webdiensten für die serverseitige Verarbeitung. Einige der möglichen Nutzungsszenarien von Aspose.Cells werden im Folgenden beschrieben.
## **Automatisierte Tabellenverwaltung**
1. Benutzer laden manuell erstellte Tabellenkalkulationen auf einen Server hoch.
1. Aspose.Cells wird verwendet, um Dateien zu öffnen und Daten abzurufen.
 Nachdem die Daten abgerufen wurden:
 1. Die Daten werden in einer Datenbank gespeichert.
 1. Bestimmte Daten werden auf Webseiten präsentiert.
 1. Einige Aspekte der Tabellenkalkulationen werden geändert und wieder in Microsoft Excel-Dateien gespeichert.
## **Internetanwendung**
1. Benutzer geben Daten auf Webseiten ein.
1. Die Datenerfassung erfolgt durch den Webserver.
1. Aspose.Cells wird verwendet, um Daten in brandneue Dateien oder vordefinierte Vorlagendateien einzufügen.
1. Nachdem die Excel-Datei erstellt wurde, kann sie als Bericht usw. an den Benutzer gesendet werden.

{{% alert color="primary" %}} 

Wir haben einige der häufigsten Szenarien für die Verwendung von Aspose.Cells in einer webbasierten Umgebung besprochen. Aber Aspose.Cells ist auch in Desktop-basierten Anwendungen gleichermaßen nützlich.
