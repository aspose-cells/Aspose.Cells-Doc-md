{{% alert color="primary" %}} 

Aspose.Cells wurde entwickelt, um Entwicklern zu helfen, Microsoft Excel-Arbeitsmappen programmgesteuert zu erstellen und zu verwalten. Aspose.Cells versteht nicht nur Arbeitsmappen im Microsoft Excel-Format (XLS/XLSX), sondern auch viele andere Dateiformate wie Kommagetrennte Werte (CSV), Tabulator-getrennte Werte/TSV (Text), SpreadsheetML etc.

Aspose.Cells bietet die meisten in Microsoft Excel verfügbaren Funktionen, doch das Spannende ist, dass Aspose.Cells Excel-Dateien dynamisch verarbeiten kann, was es Entwicklern ermöglicht, schnell Tausende oder Millionen von Arbeitsmappen zu verarbeiten.

Dieser Artikel behandelt häufige Szenarien der Verwendung von Aspose.Cells.

{{% /alert %}} 
## **Mögliche Verwendungsszenarien**
Es ist Sache der Entwickler, wie sie Aspose.Cells verwenden, doch es gibt einige gängige Szenarien, in denen Aspose.Cells helfen kann. Die häufigste Verwendung von Aspose.Cells besteht darin, es auf dem Webserver oder in Web-Services für die serverseitige Verarbeitung bereitzustellen. Einige mögliche Anwendungsszenarien von Aspose.Cells werden nachstehend beschrieben.
## **Automatisiertes Arbeitsmappen-Management**
1. Benutzer laden manuell erstellte Arbeitsmappen auf einen Server hoch.
1. Aspose.Cells wird verwendet, um Dateien zu öffnen und Daten abzurufen.
   Nachdem die Daten abgerufen wurden:
   1. Die Daten werden in einer Datenbank gespeichert.
   1. Bestimmte Daten werden auf Webseiten präsentiert.
   1. Einige Aspekte der Arbeitsmappen werden geändert und in Microsoft Excel-Dateien zurückgespeichert.
## **Webanwendung**
1. Benutzer geben Daten auf Webseiten ein.
1. Daten werden vom Webserver gesammelt.
1. Aspose.Cells wird verwendet, um Daten in brandneue Dateien oder vorgefertigte Vorlagendateien einzufügen.
1. Nachdem die Excel-Datei erstellt wurde, kann sie dem Benutzer als Bericht usw. zugesandt werden.

{{% alert color="primary" %}} 

Wir haben einige der häufigsten Szenarien zur Verwendung von Aspose.Cells in webbasierten Umgebungen diskutiert. Doch Aspose.Cells ist auch in Desktop-Anwendungen gleichermaßen nützlich.
