Aspose.Cells für Android wurde auf Version 8.7.0 aktualisiert.

Notiz
Da die Codebasis von Aspose.Cells für Android mit dem Code der relevanten Version .NET und Java übereinstimmt, sind die meisten Änderungen, Erweiterungen und Korrekturen in der
Aspose.Cells for .NET v8.6.1
, 
Aspose.Cells for .NET v8.6.2
, 
Aspose.Cells for .NET v8.6.3
, 
Aspose.Cells for .NET v8.7.0
, 
Aspose.Cells for Java v8.6.1
, 
Aspose.Cells for Java v8.6.2
, 
Aspose.Cells for Java v8.6.3
und
Aspose.Cells for Java v8.7.0
sind auch in diesem Aspose.Cells für Android v8.7.0 enthalten.
