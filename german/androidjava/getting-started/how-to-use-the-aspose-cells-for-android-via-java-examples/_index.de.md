Stellen Sie zunächst sicher, dass das ADT-Bundle installiert ist und dass Sie die neueste Version von Aspose.Cells für Android via Java heruntergeladen haben.

- Extrahieren Sie die **aspose-cells-X.X.X.jar** aus dem letzten Release-Archiv und fügen Sie sie in einem neuen Ordner unter dem Beispiel, das Sie ausführen möchten, hinzu. Fügen Sie die Datei unter einem Ordner namens **libs** hinzu.
- Öffnen Sie das Projekt in Ihrem Workspace mit Eclipse und ADT.
- Klicken Sie auf **Ausführen**, um zu starten, und wählen Sie den entsprechenden Emulator, auf dem das Paket ausgeführt werden soll.
