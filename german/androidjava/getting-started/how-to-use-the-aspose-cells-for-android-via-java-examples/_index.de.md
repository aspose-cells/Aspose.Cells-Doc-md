Stellen Sie zunächst sicher, dass Sie das ADT-Bundle installiert und die neueste Version von Aspose.Cells für Android über Java heruntergeladen haben.

- Extrahieren Sie die**aspose-cells-XXXjar**aus dem neuesten Versionsarchiv und fügen Sie es einem neuen Ordner unter dem Beispiel hinzu, das Sie ausführen möchten. Fügen Sie die Datei unter einem Ordner mit dem Namen hinzu**Bibliotheken**.
- Öffnen Sie das Projekt in Ihrem Arbeitsbereich mit Eclipse mit ADT.
- Klicken**Laufen**zu starten und den entsprechenden Emulator auszuwählen, auf dem das Paket ausgeführt werden soll.
