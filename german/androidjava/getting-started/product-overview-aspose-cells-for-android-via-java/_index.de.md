Willkommen zur Dokumentation von Aspose.Cells für Android via Java!

Aspose.Cells ist eine flexible Komponente, die es Android-Anwendungen ermöglicht, Excel-Tabellen zu erstellen und zu verwalten, ohne dass Microsoft Excel auf dem System installiert werden muss. Im Gegensatz zu ähnlichen Produkten anderer Anbieter unterstützt Aspose.Cells nicht nur die Generierung von Tabellenkalkulationen und andere grundlegende Dateiformatierungsfunktionen, sondern auch eine Reihe von erweiterten Funktionen. Diese erweiterten Funktionen machen es Entwicklern einfach, den Inhalt von Tabellenkalkulationen, Zellformatierungen und den Dateischutz zu manipulieren. Aspose.Cells ermöglicht es auch, Daten aus verschiedenen Quellen in Tabellenkalkulationen zu importieren, häufige und komplexe mathematische, Datum- und Uhrzeit-, Finanz- oder Textformeln und Funktionen hinzuzufügen. Es unterstützt außerdem das Manipulieren von Diagrammen, Bildern, Kommentaren, Zeichenobjekten und Steuerelementen.

{{% /alert %}} 
## **Dokumentfunktionen**
- Öffnen von einfachen oder verschlüsselten Excel-Dateien (Excel97 - Excel2007 und spätere Versionen) aus verschiedenen Quellen
- Excel-Dateien (Excel97 - Excel2007 und spätere Versionen) in verschiedenen Formaten speichern (XLS, XLSX/XLSM, XLTX, XLTM, XLSB, CSV, SpreadsheetML, Tabulatorgetrennt, HTML usw.)
- Konvertieren von Excel-Datei in PDF-Dokument
- Konvertieren von Excel-Tabelle in XPS-Dokument
- OpenDocument Spreadsheet (ODS)-Format lesen und schreiben
- Ändern Sie die Dokumenteigenschaften von Excel-Dateien
## **Arbeitsblattfunktionen**
## **Anzeigeoptionen**
- Ein Arbeitsblatt ausblenden oder anzeigen
- Registerkarten, Bildlaufleisten, Gitterlinien sowie Zeilen- und Spaltenüberschriften ein- oder ausblenden.
- Seitenwechselvorschau.
- Legen Sie den Zoomfaktor eines Arbeitsblatts fest.
- Ausgewählte Daten beim Scrollen sichtbar lassen, indem Sie Bereiche fixieren.
## **Arbeitsblattschutz**
- Schützen Sie den Inhalt, Objekte und Szenarien des Arbeitsblatts.
## **Seiteneinrichtungsoptionen**
- Legen Sie die Seitenausrichtung, Skalierung, Papierformat und Druckqualität fest.
- Legen Sie die Seitenumrandung für oben, links, rechts, unten fest.
- Kopf- und Fußzeilen festlegen.
- Legen Sie den Druckbereich und den Drucktitel fest.
## **Seitenumbrüche**
- Fügen Sie horizontale und vertikale Seitenumbrüche an einer bestimmten Zelle ein.
## **Zeilen & Spalten Funktionen**
- Passen Sie die Spaltenbreite an.
- Passen Sie die Breite und Höhe der Zellen automatisch an.
- Fügen Sie Zeilen und Spalten hinzu oder löschen Sie sie.
- Blenden Sie Zeilen und Spalten ein oder aus.
- Gruppieren oder trennen Sie Zeilen und Spalten.
## **Datenmanagement Funktionen**
- Fügen Sie Daten zur Laufzeit in gewünschte Zellen ein.
- Importieren Sie Daten in Arbeitsblätter aus verschiedenen Datenquellen.
- Daten aus Zellen entsprechend ihrer Datentypen abrufen
- Exportieren Sie Zelldaten als Array.
- Suchen oder Suchen Sie nach bestimmten Daten in Zellen
- Setzen Sie Formeln mit integrierten oder Add-in-Funktionen.
- Fügen Sie verschiedene Arten von Hyperlinks hinzu.
- Erstellen oder greifen Sie auf benannte Bereiche zu.
- Wenden Sie Datenvalidierung und Auto-Filterung an.
- Wenden Sie Zwischensummen auf Tabellen des Arbeitsblatts an.
## **Formatierungsfunktionen**
- Wenden Sie Nummernformatierungseinstellungen auf die Zellen an.
- Formatieren Sie die Zelle als Datum.
- Setzen Sie die **Textausrichtung**
- Mehrere Zellen zu einer einzigen zusammenführen
- Verschiedene Arten von **Schriftarteinstellungen** für die Zellen anwenden
- Verschiedene Farben auf Zellen, Schriftarten, Rasterlinien, Grafikobjekten usw. anwenden
- **Rich Text** in einer einzelnen Zelle setzen
- Verschiedene **Rahmeneinstellungen** auf Zellen anwenden
- Verschiedene **Hintergrundmuster** auf Zellen anwenden
- **Formatierungseinstellungen** auf ein Arbeitsblatt, eine Zeile oder eine Spalte usw. anwenden
- **Bedingte Formatierung** auf Zellen anwenden
## **Diagramm- und Grafikfunktionen**
- Verschiedene Arten von Diagrammen einfach erstellen
- **Diagramme** durch Festlegen ihrer verschiedenen Eigenschaften anpassen
- **Daten für die Diagramme** dynamisch festlegen
- Mithilfe von API **benutzerdefinierte Diagramme** zu Arbeitsblättern hinzufügen
- **Bilder** zur Laufzeit zu Arbeitsblättern hinzufügen
- **Kommentare** zu Zellen erstellen und hinzufügen
- Diagramm in Bilddatei konvertieren
- Arbeitsblatt in Bilddatei konvertieren
## **Erweiterte Funktionen**
- Unterstützt die **Formelberechnungen** mit einer robusten Formelberechnungs-Engine
- **Quelldaten eines Pivot-Tables** zur Laufzeit ändern
- PivotTables erstellen und manipulieren
