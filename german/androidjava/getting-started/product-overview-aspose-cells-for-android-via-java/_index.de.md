Willkommen bei der Aspose.Cells for Android via Java Dokumentation!

Aspose.Cells ist eine flexible Komponente, mit der Android-Anwendungen Excel-Tabellen erstellen und verwalten können, ohne dass Microsoft Excel auf dem System installiert sein muss. Im Gegensatz zu ähnlichen Produkten anderer Anbieter unterstützt Aspose.Cells nicht nur die Erstellung von Tabellenkalkulationen und andere grundlegende Dateiformatierungsfunktionen, sondern auch eine Reihe erweiterter Funktionen. Diese erweiterten Funktionen machen es Entwicklern leicht, den Inhalt von Tabellenkalkulationen, die Zellenformatierung und den Dateischutz zu manipulieren. Mit Aspose.Cells können Sie auch Daten aus verschiedenen Datenquellen in Tabellenkalkulationen importieren und allgemeine und komplexe mathematische, Datums- und Zeit-, Finanz- oder Textformeln und -funktionen hinzufügen. Es unterstützt außerdem das Manipulieren von Diagrammen, Bildern, Kommentaren, Zeichnen von Objekten und Steuerelementen.

## **Dokumentfunktionen**
- Öffnen Sie einfache oder verschlüsselte Excel-Dateien (Excel97 - Excel2007 und spätere Versionen) aus verschiedenen Quellen
- Speichern Sie Excel-Dateien (Excel97 - Excel2007 und spätere Versionen) in verschiedenen Formaten (XLS, XLSX/XLSM, XLTX, XLTM, XLSB, CSV, SpreadsheetML, Tabulatorgetrennt, HTML usw.)
- Konvertieren Sie die Excel-Datei in das Dokument PDF
- Konvertieren Sie die Excel-Tabelle in das Dokument XPS
- Lesen und schreiben Sie das Format OpenDocument Spreadsheet (ODS).
- Ändern Sie die Dokumenteigenschaften von Excel-Dateien
## **Arbeitsblattfunktionen**
## **Anzeigeoptionen**
- Blenden Sie ein Arbeitsblatt ein oder aus
- Registerkarten, Bildlaufleisten, Gitternetzlinien sowie Zeilen- und Spaltenüberschriften ein- oder ausblenden.
- Seitenumbrüche in der Vorschau anzeigen.
- Legen Sie die Zoomstufe eines Arbeitsblatts fest.
- Lassen Sie ausgewählte Daten sichtbar bleiben, während Sie mit Freeze-Fenstern scrollen.
## **Arbeitsblattschutz**
- Schützen Sie Arbeitsblattinhalte, Objekte und Szenarien.
## **Seiteneinrichtungsoptionen**
- Legen Sie Seitenausrichtung, Skalierung, Papiergröße und Druckqualität fest.
- Legen Sie Seitenränder für oben, links, rechts und unten fest.
- Legen Sie Kopf- und Fußzeilen fest.
- Druckbereich und Drucktitel festlegen.
## **Seitenumbrüche**
- Fügt horizontale und vertikale Seitenumbrüche an einer bestimmten Zelle ein.
## **Funktionen für Zeilen und Spalten**
- Passen Sie die Spaltenbreite an.
- Breite und Höhe der Zellen automatisch anpassen.
- Zeilen und Spalten hinzufügen oder löschen.
- Zeilen und Spalten ein- oder ausblenden.
- Zeilen und Spalten gruppieren oder Gruppierung aufheben.
## **Datenverwaltungsfunktionen**
- Fügen Sie zur Laufzeit Daten zum gewünschten Cells hinzu
- **Importieren Sie Daten in Arbeitsblätter** aus unterschiedlichen Datenquellen
- Rufen Sie Daten aus Zellen gemäß ihren Datentypen ab
- **Zellendaten in Array exportieren**
- Suchen oder suchen Sie bestimmte Daten in Zellen
-  Satz**Formeln mit integrierten oder Add-in-Funktionen**
- Fügen Sie verschiedene Arten von Hyperlinks hinzu
- Benannte Bereiche erstellen oder darauf zugreifen
-  Anwenden**Datenvalidierung** und**Automatische Filterung**
-  Anwenden**Zwischensummen** zu Arbeitsblatttabellen
## **Formatierungsfunktionen**
-  Anwenden**Zahlenformateinstellungen** für die Cells
- Formatieren Sie Cell als Datum
-  Satz**Textausrichtung**
- Führen Sie mehrere Cells zu einer einzigen zusammen
- Wenden Sie verschiedene Arten an**Schrifteinstellungen** für die Cells
- Wenden Sie verschiedene Farben auf Cells, Schriftarten, Gitternetzlinien, Grafikobjekte usw. an.
-  Satz**Rich-Text** in einer einzigen Cell
-  Bewerben Sie sich anders**Rahmeneinstellungen** unter Cells
-  Bewerben Sie sich anders**Hintergrundmuster** unter Cells
-  Anwenden**Formateinstellungen** auf einem Arbeitsblatt, Zeile oder Spalte etc.
-  Anwenden**Bedingte Formatierung** unter Cells
## **Diagramm- und Grafikfunktionen**
- Erstellen Sie ganz einfach verschiedene Arten von Diagrammen
-  Anpassen**Diagramme** indem Sie ihre verschiedenen Eigenschaften festlegen
-  Satz**Daten für die Charts** dynamisch
-  Addieren**Benutzerdefinierte Diagramme** zu Arbeitsblättern mit API
-  Addieren**Bilder** in Arbeitsblätter zur Laufzeit
-  Erstellen und hinzufügen**Bemerkungen** zu Zellen
- Diagramm in Bilddatei umwandeln
- Arbeitsblatt in Bilddatei konvertieren
## **Erweiterte Funktionen**
-  Unterstützt**Formelberechnungen** mit robuster Formelberechnungs-Engine
-  Veränderung**Quelldaten einer Pivot-Tabelle** zur Laufzeit
- PivotTables erstellen und bearbeiten
