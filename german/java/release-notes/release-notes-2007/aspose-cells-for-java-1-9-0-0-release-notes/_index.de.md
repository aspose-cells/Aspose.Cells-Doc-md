 Sehr geehrte Kunden,

 Wir haben Aspose.Cells for Java V1.9.0 freigegeben!

 Was gibt's Neues?

- Unterstützt den Export in die XML-Datei Aspose.Pdf
 Unterstützt das Kopieren von Kommentaren und benannten Bereichen beim Kopieren von Arbeitsblättern
 Unterstützt die ICustomFunction-Schnittstelle zur Berechnung von Formeln
Unterstützt die Verfolgung von Vorfahren und Angehörigen einer Zelle

 Was wurde verbessert?

- Unterstützt, nur alle Formeln in einem Arbeitsblatt zu berechnen
 Unterstützt POJO und POJOs mit intelligenten Markern

 Was ist behoben?

- Ein Fehler beim Abrufen benannter Bereiche des Arbeitsblatts wurde behoben
 Fehler beim Erstellen der PivotTable behoben
 Fehler beim Schützen von Dateien behoben
 Fehler beim Berechnen von Formeln behoben
 Fehler beim Speichern der Datei behoben.
