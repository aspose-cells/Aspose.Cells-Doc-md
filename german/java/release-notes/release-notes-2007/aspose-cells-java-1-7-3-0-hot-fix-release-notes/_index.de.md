 Sehr geehrte Kunden,

 Wir haben Aspose.Cells for Java V1.7.3 veröffentlicht! Folgende Verbesserungen und Fehlerbehebungen sind enthalten:

- Unterstützt das Einfügen/Löschen eines Bereichs.
 Unterstützt das Öffnen und Speichern verschlüsselter Excel-Dateien.
 Verbessert die Methode Cell.getStringValue(), um mehr Datums-/Zahlenformate zu unterstützen.
 Aktualisierte die Web-Demos.
 Unterstützt die Berechnung von Formeln mit JDK1.4.
 Behebt einen Fehler der Methode License.setLicense().
 Behebt einen Fehler beim Festlegen des benutzerdefinierten Zahlenformats.
Behebt einen Fehler beim Entfernen eines Arbeitsblatts aus der Arbeitsmappe.
