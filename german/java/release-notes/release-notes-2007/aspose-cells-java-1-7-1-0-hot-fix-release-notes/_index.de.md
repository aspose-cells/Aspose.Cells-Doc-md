 Sehr geehrte Kunden,

 Wir haben Aspose.Cells for Java V1.7.1 veröffentlicht! Folgende Verbesserungen und Fehlerbehebungen sind enthalten:

- Unterstützt den Import von Inhalten und Stilen der Zellen aus der SpreadsheetML-Datei
 Unterstützt den Export von Inhalten und Stilen der Zellen in eine SpreadsheetML-Datei
 Unterstützt die Berechnungsmaschine der Formel
 Unterstützt die Einstellung der R1C1-Formel
 Unterstützt zwei weitere Datenquellentypen von PivotTable
 Behebt einige Fehler beim Hinzufügen von PivotTable .

Hinweis: Wir haben die const CellValueType.FORMULA entfernt und die Methode Cell.isFormula() hinzugefügt, um zu entscheiden, ob die Zelle eine Formel enthält.
