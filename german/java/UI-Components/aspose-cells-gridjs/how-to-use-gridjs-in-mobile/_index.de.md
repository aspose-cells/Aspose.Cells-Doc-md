## Verwenden von GridJs auf mobilen Geräten

Sie können GridJs problemlos auf dem Mobilgerät verwenden. 

GridJs-Client-JavaScript kann sich an den mobilen Bildschirm anpassen.

Führen Sie das [Demo von GridJs](https://github.com/aspose-cells/Aspose.Cells-for-java/tree/master/Examples_GridJs "Demo von GridJs") aus

Der Standardport des Demo-Projekts ist 8080. 

Öffnen Sie den Browser auf Ihrem mobilen Gerät und geben Sie die URL http://yourip:8080/gridjsdemo/index ein. 

![todo:die Bildschirmnavigation auf dem Mobilgerät](gridjs_edit_on_mobile.png)

Dann können wir Bearbeitungsvorgänge wie auf dem PC-Gerät durchführen.
