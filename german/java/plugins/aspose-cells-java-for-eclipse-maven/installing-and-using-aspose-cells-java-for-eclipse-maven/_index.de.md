## **Installation**

**Aspose.Cells Java für Eclipse (Maven)**-Plugin kann bequem vom verfügbaren **Plugin**-Tab im Plugin-Dialog installiert werden.

- Um es zu öffnen, wählen Sie **Plugins** im **Tools**-Menü in Eclipse.  

![todo:image_alt_text](installing-and-using-aspose-cells-java-for-eclipse-maven_1)

- Dies fügt das **Aspose.Cells Maven-Projekt** in den Assistenten Neues Projekt und **Aspose.Cells-Codebeispiel** im Assistenten für Neues anderes hinzu.  

![todo:image_alt_text](project_1.png)

## **Verwendung**

### **Aspose.Cells Maven-Projekt (Assistent)**

Um ein **Maven-Projekt** mithilfe des Assistenten für die Verwendung von [Aspose.Cells for Java API](https://products.aspose.com/cells/java/) zu erstellen:

1. Wählen Sie **Neu -> Projekt**.
1. Wählen Sie **Aspose.Cells Maven-Projekt** in der Kategorie **Maven**.
1. Klicken Sie auf **Weiter**.

![todo:image_alt_text](project_2.png)

1. Geben Sie **Projektname, Standort, GroupId, ArtifactId** und **Version** für Ihr Maven-Projekt an und klicken Sie auf **Fertigstellen**.

![todo:image_alt_text](project_3.png)

1. Dies wird die [Aspose.Cells for Java](https://products.aspose.com/cells/java/) neuesten [Maven-Abhängigkeiten](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) Referenz aus dem [Aspose Cloud Maven-Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo) abrufen und es in der **pom.xml** konfigurieren. Wenn Sie sich für **Auch Codebeispiele herunterladen** entschieden haben, wird auch der Download der **Codebeispiele** aus dem [Aspose.Cells for Java API-Beispiele Repository](https://github.com/aspose-cells/Aspose.Cells-for-Java) beginnen.
1. Nach Abschluss des Assistenten wird das folgende **Maven**-Projekt auf Ihrem **Eclipse IDE** erstellt:  

![todo:image_alt_text](project_4.png)

1. Das erstellte **Maven-Projekt** ist konfiguriert, um die **Aspose.Cells for Java API** zu verwenden und kann gemäß den Anforderungen Ihres Projekts erweitert werden.
   Wenn Sie sich dafür entschieden haben, [Codebeispiele](https://github.com/aspose-cells/Aspose.Cells-for-Java) herunterzuladen, können Sie den **Aspose.Cells Codebeispiel-Assistent** verwenden, um die benötigten **Codebeispiele** der Aspose.Cells for Java API in Ihr Projekt zu importieren.

### **Aspose.Cells Codebeispiel-Assistent**

Der **Aspose.Cells Codebeispiel-Assistent** ermöglicht es Ihnen, viele Beispiele, die für die Aspose.Cells for Java API bereitgestellt werden, auszuprobieren.

{{% alert color="primary" %}}

Um den **Aspose.Cells Codebeispiel-Assistent** komfortabel nutzen zu können: Es wird empfohlen, **Auch Codebeispiele herunterladen** immer auszuwählen, während Sie ein **Maven-Projekt** auf dem **Aspose.Cells Maven-Projekt-Assistent** erstellen.

{{% /alert %}}

Um Beispiele zu verwenden, einfach:

1. Klicken Sie auf **Neu -> Andere** in **Eclipse**.
1. Wählen Sie **Aspose.Cells Codebeispiel** in der Kategorie **Java** aus.
1. Klicken Sie auf **Weiter**.  

![todo:image_alt_text](example_1.png)

1. Wählen Sie Ihr Projekt aus und erweitern Sie dann den Baum, um die **Codebeispiel**-Kategorie auszuwählen und klicken Sie auf **Fertigstellen**.

![todo:image_alt_text](example_2.png)

1. Dies kopiert die ausgewählten Kategorie-**Codebeispiele**-Java-Dateien in das Projekt unter dem Paket **com.aspose.cells.examples**. Auch alle benötigten Ressourcen, die von den Codebeispielen benötigt werden, werden in den Ordner **src/main/resources** kopiert, wie unten gezeigt:

![todo:image_alt_text](example_3.png)

1. Überprüfen Sie den Beispielcode, kompilieren und ausführen Sie ihn.
1. Sie können jetzt weitere Beispiele testen und damit beginnen, Ihre eigene Anwendung mit der Aspose.Cells for Java API zu entwickeln.
