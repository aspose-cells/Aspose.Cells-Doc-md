## **Installieren**

**Aspose.Cells Java für Sonnenfinsternis (Maven)** Plugin kann einfach aus dem Verfügbaren installiert werden**Plugin** Registerkarte im Plugin-Dialog.

- Um es zu öffnen, wählen Sie aus**Plugins**von dem**Werkzeug** Menü in Eclipse.

![todo: Bild_alt_Text](installing-and-using-aspose-cells-java-for-eclipse-maven_1)

-  Dies fügt die hinzu**Aspose.Cells Maven Projekt** im Assistenten für neue Projekte und**Aspose.Cells Codebeispiel** im New Other Wizard der Eclipse IDE.

![todo: Bild_alt_Text](project_1.png)

## **Verwenden**

### **Aspose.Cells Maven Projekt (Assistent)**

 Erschaffen**Maven Projekt** durch Assistenten für die Verwendung[Aspose.Cells for Java API](https://products.aspose.com/cells/java/):

1. Auswählen**Neu -> Projekt**.
1. Auswählen**Aspose.Cells Maven Projekt**in dem**Maven**Kategorie.
1. Klicken**Nächste**.

![todo: Bild_alt_Text](project_2.png)

1. Zur Verfügung stellen**Projektname, Ort, GroupId, ArtifactId**und**Ausführung**für Ihr Maven-Projekt und klicken Sie**Fertig.**

![todo: Bild_alt_Text](project_3.png)

1. Dadurch wird die abgerufen[Aspose.Cells for Java](https://products.aspose.com/cells/java/) neueste[Maven Abhängigkeit](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) Referenz von[Aspose Wolke Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo) und konfigurieren Sie es in**pom.xml**. Wenn Sie sich dafür entschieden haben**Laden Sie auch Codebeispiele herunter,** Herunterladen der**Codebeispiele** wird auch von der beginnen[Aspose.Cells for Java API Beispiele Repository.](https://github.com/aspose-cells/Aspose.Cells-for-Java)
1. Folgend**Maven** Das Projekt wird auf Ihrem erstellt**Eclipse-IDE** Nach Abschluss des Assistenten:

![todo: Bild_alt_Text](project_4.png)

1.  Das Geschaffene**Maven Projekt** ist für die Verwendung konfiguriert**Aspose.Cells for Java API** und bereit, gemäß Ihren Projektanforderungen erweitert zu werden.
Wenn Sie sich für den Download entschieden haben[Codebeispiele](https://github.com/aspose-cells/Aspose.Cells-for-Java) , können Sie verwenden**Aspose.Cells Codebeispiel (Assistent)**für den Import benötigt**Codebeispiele** von Aspose.Cells for Java API in Ihr Projekt.

### **Aspose.Cells Codebeispiel (Assistent)**

**Aspose.Cells Codebeispiel-Assistent** können Sie viele Beispiele ausprobieren, die für Aspose.Cells for Java API bereitgestellt werden.

{{% alert color="primary" %}}

 Verwenden zu können**Aspose.Cells Codebeispiel-Assistent**komfortabel: Es wird empfohlen, immer auszuwählen**Laden Sie auch Codebeispiele herunter** beim Erstellen**Maven Projekt**an**Aspose.Cells Maven Projekt** **Magier**,

{{% /alert %}}

Um Beispiele zu verwenden, einfach:

1. Klicken**Neu -> Andere** in**Finsternis**.
1. Auswählen**Aspose.Cells Codebeispiel**in dem**Java**Kategorie.
1. Klicken**Nächste**.  

![todo: Bild_alt_Text](example_1.png)

1.  Wählen Sie Ihr Projekt aus und erweitern Sie dann den Baum, um es auszuwählen**Codebeispiel** Kategorie und klicken Sie darauf**Fertig**.

![todo: Bild_alt_Text](example_2.png)

1.  Dadurch wird die ausgewählte Kategorie kopiert**Codebeispiele**Java Dateien in das Projekt unter**com.aspose.cells.examples**Paket. Außerdem werden alle erforderlichen Ressourcen, die von den Codebeispielen benötigt werden, hineinkopiert**src/main/resources**Ordner, wie unten gezeigt:

![todo: Bild_alt_Text](example_3.png)

1. Überprüfen Sie den Beispielcode, kompilieren Sie ihn und führen Sie ihn aus.
1. Sie können jetzt andere Beispiele testen und mit dem Erstellen Ihrer eigenen Anwendung beginnen, indem Sie Aspose.Cells for Java API verwenden.
