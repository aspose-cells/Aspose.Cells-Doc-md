## **Einführung**
### **Eclipse IDE**
Die Eclipse-IDE ist eine bekannte integrierte Entwicklungsumgebung (IDE) für Java. Die IDE ist definitiv das bekannteste Produkt des Eclipse Open Source-Projekts. Heutzutage ist sie die führende Entwicklungsumgebung für Java mit einem Marktanteil von etwa 60%.

Die Eclipse-IDE kann mit zusätzlichen Softwarekomponenten erweitert werden. Eclipse nennt diese Softwarekomponenten Plug-ins. Mehrere Open-Source-Projekte und Unternehmen haben die Eclipse-IDE erweitert oder eigenständige Anwendungen (Eclipse RCP) auf Grundlage des Eclipse-Frameworks erstellt.
### **Aspose.Cells for Java**
[Aspose.Cells for Java](http://goo.gl/gtvhlQ) ist eine preisgekrönte Excel-Tabellenkomponente, die es Java-Entwicklern ermöglicht, die Fähigkeit zum Lesen, Schreiben und Manipulieren von Excel®-Tabellen (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF und Bilddateiformaten in ihre eigenen Java-Anwendungen einzubetten, ohne auf Microsoft Excel® angewiesen zu sein.

[Aspose.Cells for Java](http://goo.gl/gtvhlQ) ist eine ausgereifte, skalierbare und funktionsreiche Komponente, die viele Funktionen bietet, die weit über die einfachen Datenexportfähigkeiten anderer Anbieter hinausgehen. Mit Aspose.Cells for Java können Entwickler Daten exportieren, Tabellenkalkulationen bis ins kleinste Detail formatieren, Bilder importieren, Diagramme erstellen, komplexe Formeln anwenden und berechnen, Excel®-Daten streamen, in verschiedenen Formaten speichern und vieles mehr - ganz ohne Microsoft Excel® oder Microsoft Office Automation.
### **Aspose.Cells Java for Eclipse (Maven)**
- Aspose.Cells Java für Eclipse (Maven) ist ein Plugin für die **Eclipse-IDE** von **Aspose**. Dieses Plugin richtet sich an Entwickler, die die Maven-Plattform für Java-Entwicklungen verwenden möchten und [Aspose.Cells for Java](http://goo.gl/gtvhlQ) in ihren Projekten einsetzen möchten. Das Plugin ermöglicht es Ihnen, Maven-Projekte für die Verwendung der [Aspose.Cells for Java-API](http://goo.gl/gtvhlQ) zu erstellen und auch [Codebeispiele herunterzuladen](https://goo.gl/uxznrb) für die API.

**WIZARDS**:
Das Plugin enthält zwei Assistenten-----

**       ****Aspose.Cells Maven-Projekt (Assistent)**

- Dieser Assistent für ein neues Projekt ermöglicht Entwicklern die Erstellung eines **Maven**-Projekts zur Verwendung von [Aspose.Cells for Java](http://goo.gl/gtvhlQ) von **Neu -> Projekt -> Maven -> Aspose.Cells Maven-Projekt.**
- Die Referenz der **Aspose.Cells for Java-API**-Maven-Abhängigkeit wird automatisch aus dem [Aspose Cloud Maven-Repository](https://repository.aspose.com/webapp/) abgerufen und in die pom.xml aufgenommen.
- Das erstellte Projekt enthält immer die aktuell verfügbare Version der **Maven**-Abhängigkeit für die **Aspose.Cells for Java-API**.
- Die Assistentenschritte bieten auch die Möglichkeit, [Codebeispiele herunterzuladen](https://goo.gl/uxznrb) für die Verwendung der **Aspose.Cells for Java-API**.

**       ****Aspose.Cells Codebeispiel (Assistent)**

- Dieser Assistent für eine neue Datei ermöglicht es Ihnen, die heruntergeladenen [Codebeispiele](https://goo.gl/uxznrb) in Ihr Projekt für die Verwendung von [Aspose.Cells for Java](http://goo.gl/gtvhlQ) von **Neu -> Andere -> Java -> Aspose.Cells Codebeispiel** zu kopieren. 
- Die verfügbaren Beispiele werden im Baumformat angezeigt, aus dem der Benutzer sie kategorisch auswählen kann.
- Alle Beispiele innerhalb der ausgewählten Kategorie werden in den Ordner "**com.aspose.cells.examples**" des Projekts kopiert und die erforderlichen Ressourcen im Ordner "**src/main/resources**" abgelegt, die zum Ausführen der Beispiele benötigt werden.
- [Codebeispiele](https://goo.gl/uxznrb) der [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) sind zur Demonstration verschiedener Funktionen der API gedacht.
- Der Assistent sucht auch nach und aktualisiert neu verfügbare [Codebeispiele](https://goo.gl/uxznrb) aus dem [Aspose.Cells for Java Beispiele-Repository.](https://goo.gl/uxznrb)

## **Herunterladen**
### **Eclipse-IDE herunterladen**
Sie müssen zuerst die Eclipse-IDE installieren, bevor Sie das Aspose.Cells Java für Eclipse (Maven) Plugin herunterladen.

Um die Eclipse-IDE herunterzuladen

1. Gehen Sie zu [https://eclipse.org](https://eclipse.org/).
1. Laden und installieren Sie die empfohlene Eclipse-IDE für Java SE / EE-Entwickler herunter.
### **Aspose.Cells Java für Eclipse (Maven) herunterladen**
Im Folgenden finden Sie drei empfohlene Methoden für das erfolgreiche Herunterladen und Installieren des Aspose.Cells Java für Eclipse (Maven) Plugins:

- Ziehen und ablegen Sie die Installation aus dem [Eclipse Marketplace](https://marketplace.eclipse.org/content/asposecells-java-maven-eclipse) in Ihren Eclipse-Arbeitsbereich.
- Oder gehen Sie zu **Hilfe** > **Neue Software installieren...** > Geben Sie die folgende Update-Site-URL in **Arbeitsbereich** ein:
 cdn.aspose.com/marketplace/eclipse/java/maven/asposecells
  Wählen Sie dann "Aspose.Cells Java für Eclipse (Maven)" und klicken Sie auf **Beenden**. Akzeptieren Sie die Lizenzvereinbarung und installieren Sie das Plugin.

### **Wie wendet man Aspose Lizenz an?**
Dieses Plugin verwendet eine Evaluierungsversion von Aspose.Cells. Sobald Sie mit Ihrer Evaluierung zufrieden sind, können Sie eine Lizenz auf der [Aspose-Website](https://purchase.aspose.com/buy) erwerben.
Um die Evaluierungsnachricht und die Funktionsbeschränkungen zu entfernen, muss eine Produktlizenz angewendet werden. Sie erhalten eine Lizenzdatei, nachdem Sie das Produkt erworben haben. Befolgen Sie bitte die unten stehenden Schritte, um die Lizenz anzuwenden

- Stellen Sie sicher, dass die Lizenzdatei den Namen **Aspose.Cells.Java.lic** hat
- Legen Sie die Datei **Aspose.Cells.Java.lic** in den Ordner, der die Aspose.Cells.jar-Datei enthält
- Verwenden Sie den folgenden Code zum Aktivieren der Lizenz:

{{< highlight java >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **Wie konfiguriere ich den Quellcode von Aspose.Cells Java für Eclipse (Maven)**
Die folgenden einfachen Schritte führen reibungslos zur erfolgreichen Konfiguration des **"Aspose.Cells Java für Eclipse (Maven)"** Plugin-Quellcodes in der Eclipse-IDE

1. Lade den Quellcode herunter / klonen.
1. Wählen Sie **Datei** > **Importieren > Allgemein > Bestehende Projekte in Workspace importieren**
1. Durchsuchen Sie das neueste Projektquellverzeichnis, das Sie heruntergeladen haben
1. Wählen Sie das Eclipse-Projekt aus, das Sie importieren möchten
1. Klicken Sie auf **Fertig stellen**
1. Aspose.Cells Java für Eclipse-Plugin-Code ist jetzt bereit zur Verbesserung.
