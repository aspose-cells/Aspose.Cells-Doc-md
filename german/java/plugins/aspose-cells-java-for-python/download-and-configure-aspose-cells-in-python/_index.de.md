## **Erforderliche Bibliotheken herunterladen**
Laden Sie die unten aufgeführten erforderlichen Bibliotheken herunter. Dies sind die für die Ausführung von Aspose.Cells Java for Python Beispielen erforderlichen.

- **Aspose:** [Aspose.Cells for Java Komponente](https://downloads.aspose.com/cells/java/)
- [JPype](https://pypi.org/project/JPype1/)

## **So konfigurieren Sie den Quellcode**
Bitte befolgen Sie diese einfachen Schritte, um den Quellcode zu öffnen und zu erweitern, während Sie ihn verwenden:

 \1. Installieren Sie nur Python 2.7.xx von:<https://www.python.org/downloads/>
 \2. Installieren Sie JPype von:<https://sourceforge.net/projects/jpype/files/JPype/>
 \3. Installieren Sie Java JDK von:<https://www.oracle.com/java/technologies/downloads/>
 \4. Laden Sie Aspose.Cells for Java APIs herunter, die Sie in Ihren Python-Projekten verwenden möchten:<https://downloads.aspose.com/cells/java> und kopieren Sie die .jars der heruntergeladenen APIs in den Ordner %Python%/lib.
 \5. Klon[Aspose.Cells Java for Python Beispiele](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) von github.
