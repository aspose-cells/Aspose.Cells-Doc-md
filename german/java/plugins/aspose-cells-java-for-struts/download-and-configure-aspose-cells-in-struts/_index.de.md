## **Aufbau von Aspose.Cells Java für Struts 1.3 aus den Quellcodes**
Nachdem Sie die Quellcodes aus einem der oben genannten Repositories überprüft haben, wenden Sie die folgenden mvn-Befehle an:

{{< highlight java >}}

 $ mvn -U clean package 

{{< /highlight >}}

Dies erstellt die Datei "Strutsbookapp.war" im Zielordner.

Um die .war-Datei bereitzustellen, kopieren Sie sie einfach in das Webapp-Verzeichnis des laufenden Apache Tomcat-Servers.
