## **Erstellen von Aspose.Cells Java für Struts 1.3 aus Quellcodes**
Wenden Sie nach dem Auschecken von Quellcodes aus einem der oben genannten Repositorys die folgenden mvn-Befehle an:

{{< highlight "java" >}}

 $ mvn -U clean package 

{{< /highlight >}}

Dadurch wird "Strutsbookapp.war" im Target Teller erstellt.

Um die .war-Datei bereitzustellen, kopieren Sie sie einfach in das Webapp-Verzeichnis des ausgeführten Apache Tomcat-Servers.
