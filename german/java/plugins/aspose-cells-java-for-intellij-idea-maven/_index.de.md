## **Einführung**

### **IntelliJ IDEA**

**IntelliJ IDEA** ist eine integrierte Entwicklungsumgebung (IDE) für Softwareentwickler. Es wird von JetBrains entwickelt. Ursprünglich für Java erstellt, unterstützt es derzeit eine Vielzahl von Sprachen, einschließlich JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure und Kotlin.

In einem Bericht von **Infoworld** aus dem Jahr 2010 erhielt IntelliJ die höchste Testcenter-Bewertung unter den vier besten Java-Programmierwerkzeugen: Eclipse, IntelliJ IDEA, NetBeans und Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) für Java ist eine Excel®-Tabellenkalkulationsberichtskomponente, die es Java-Anwendungen ermöglicht, Excel®-Tabellenkalkulationen zu erstellen und zu verwalten, ohne Microsoft Excel® zu verwenden. Aspose.Cells for Java ist eine unglaublich leistungsstarke Komponente, die viel mehr als nur grundlegende Datexportfunktionen bietet.

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java für IntelliJ IDEA (Maven) ist ein **IntelliJ IDEA Plugin**, das es Ihnen ermöglicht, Maven-Projekte zu erstellen (d.h. das erstellte Projekt enthält Aspose.Cells for Java API Maven-Abhängigkeit) und Beispiele für den Quellcode der API für die Verwendung von **Aspose.Cells for Java API** herunterzuladen.

- Dieses Plugin hilft Ihnen, mit **Aspose.Cells for Java API** in Ihrer bevorzugten IntelliJ IDEA zu arbeiten.

**WIZARDS**:
Das Plugin enthält zwei Assistenten -----

**Wizard 1:** Aspose.Cells Maven-Projekt

- Das **Aspose Maven Project Wizard** Plugin ermöglicht es Entwicklern, ein Aspose.Cells for Java Maven-Projekt zu erstellen.
- Die Referenz der **Aspose.Cells for Java-API**-Maven-Abhängigkeit wird automatisch aus dem [Aspose Cloud Maven-Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) abgerufen und in die pom.xml eingefügt.
- Das erstellte Projekt enthält immer die aktuell verfügbare Version der Maven-Abhängigkeit für Aspose.Cells for Java.
- Die Assistentenschritte bieten auch die Möglichkeit zum Herunterladen von [Quellcodebeispielen zur Verwendung der Aspose.Cells for Java-API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples).

**Wizard 2:** Aspose.Cells Beispiele

- Der **Aspose.Cells Beispiele-Assistent** ermöglicht das Erstellen/Kopieren heruntergeladener [Quellcodebeispiele der Aspose.Cells for Java-API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) in Ihr Projekt. Alle Beispiele der ausgewählten Kategorie werden in den Paketordner "com.aspose.cells.examples" des Projekts kopiert, zusammen mit den erforderlichen Ressourcen im Ordner "src/main/resources", die zum Ausführen der Beispiele benötigt werden.

- [Quellcodebeispiele der Aspose.Cells for Java-API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) dienen dazu, die verschiedenen Verwendungen der API zu demonstrieren.

## **Systemanforderungen und unterstützte Plattformen**

### **Systemanforderungen**

- Microsoft Windows 8/7/Vista/2003/XP (einschl. 64-Bit) / Linux GNOME oder KDE-Desktop / Mac OS X 10.5 oder höher
- Mindestens 1 GB RAM, empfohlen werden 2 GB RAM
- 300 MB Festplattenspeicherplatz + mindestens 1 GB für Caches
- Mindestens 1024x768 Bildschirmauflösung
- JDK 1.7 oder höher empfohlen
- IntelliJ IDEA CE/UE (13.0.1, 13.1.2 oder 14.0.2)
- Internetverbindung (2 MB oder schneller empfohlen)

### **Unterstützte Plattformen**

Das Plugin unterstützt die folgenden IntelliJ IDEA-Versionen:

- 13.0.1 (Community / Ultimate Edition)
- 13.1.2 (CE / UE)
- 14.0.2 (CE/ UE)

## **Herunterladen**

### **Herunterladen von IntelliJ IDEA**

Sie müssen zuerst IntelliJ IDEA CE / UE installieren, bevor Sie das **Aspose.Cells Java für IntelliJ IDEA (Maven)**-Plugin herunterladen können.

IntelliJ IDEA herunterladen

1. Go to <https://www.jetbrains.com/idea/download/>
1. Wählen Sie Ihr Betriebssystem aus
1. Klicken Sie auf den UE- oder CE-Download.

### **Herunterladen des Aspose.Cells Java für IntelliJ IDEA-Plugins (Maven)**

- Das Plugin kann direkt von der [JetBrains-Plugin-Hosting-URL](https://goo.gl/R4pysl) heruntergeladen werden - auf dem IntelliJ IDEA-Plugin-Repository
  und später kann es mit der IDE-Option installiert werden:
  - Gehen Sie zu **Datei** > **Einstellungen**...>**Plugins**
  - Wählen Sie "**Plugin von Datenträger installieren**"
- Oder es kann direkt (ohne manuelles Herunterladen von der JetBrains-Plugin-Hosting-URL) mit den IDE-Optionen installiert werden:
  - Gehen Sie zu **Datei** > **Einstellungen**...>**Plugins**
  - Wählen Sie **"In Repositorys suchen"**
  - Geben Sie im Suchfeld "Aspose" ein
  - Wählen Sie das "**Aspose Cells Java für IntelliJ IDEA Maven**"-Plugin zum Installieren aus.
