## **Einführung**

### **IntelliJ-IDEE**

**IntelliJ-IDEE** ist eine Java integrierte Entwicklungsumgebung (IDE) für Softwareentwickler. Es wird von JetBrains entwickelt. Ursprünglich erstellt for Java, unterstützt es derzeit eine Vielzahl von Sprachen, darunter JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure und Kotlin.

 Bei einem 2010**Infowelt** Bericht erhielt IntelliJ die höchste Testcenter-Punktzahl von den vier besten Java-Programmiertools: Eclipse, IntelliJ IDEA, NetBeans und Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/)for Java ist eine Berichtskomponente für Excel®-Tabellenkalkulationen, mit der Java-Anwendungen Excel®-Tabellen erstellen und verwalten können, ohne Microsoft Excel® zu verwenden. Aspose.Cells for Java ist eine unglaublich funktionsreiche Komponente, die viel mehr als nur grundlegende Datenexportfunktionen bietet.

### **Aspose.Cells Java für IntelliJ IDEA - Maven**

-  Aspose.Cells Java für IntelliJ IDEA (Maven) ist ein**IntelliJ IDEA-Plugin** Damit können Sie maven-Projekte erstellen (dh das erstellte Projekt enthält Aspose.Cells for Java API maven-Abhängigkeiten) und Quellcodebeispiele von API zur Verwendung herunterladen**Aspose.Cells for Java API**.

-  Dieses Plugin hilft Ihnen, damit zu arbeiten**Aspose.Cells for Java API** in Ihrer bevorzugten IntelliJ-IDEE.

**ZAUBERER**:
Das Plugin enthält zwei Assistenten -----

**Zauberer 1:** Aspose.Cells Maven Projekt

-  Plugins**Aspose Maven Projektassistent** lässt Entwickler Aspose.Cells for Java Maven Projekt erstellen.
-  Die Referenz von**Aspose.Cells for Java API** maven Abhängigkeit wird automatisch abgerufen[Aspose Wolke Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) und wird in der pom.xml hinzugefügt.
- Das erstellte Projekt enthält immer die neueste verfügbare Version Maven Abhängigkeit für Aspose.Cells for Java.
-  Die Schritte des Assistenten sind auch mit der Option zum Herunterladen vorhanden[Quellcodebeispiele für die Verwendung von Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

**Zauberer 2:** Aspose.Cells Beispiele

- **Aspose.Cells Beispielassistent** lässt Sie herunterladen / kopieren[Quellcode-Beispiele von Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) in Ihr Projekt. Alle Beispiele für die ausgewählte Kategorie werden zusammen mit den erforderlichen Ressourcen im Ordner „src/main/resources“, die zum Ausführen der Beispiele benötigt werden, in den Paketordner „com.aspose.cells.examples“ des Projekts kopiert.

- [Quellcode-Beispiele von Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) sind dazu gedacht, die verschiedenen Verwendungen der API zu demonstrieren.

## **Systemanforderungen und unterstützte Plattformen**

### **System Anforderungen**

- Microsoft Windows 8/7/Vista/2003/XP (inkl. 64-bit) / Linux GNOME oder KDE Desktop / Mac OS X 10.5 oder höher
- Mindestens 1 GB RAM, 2 GB RAM empfohlen
- 300 MB Festplattenspeicher + mindestens 1 GB für Caches
- 1024 x 768 minimale Bildschirmauflösung
- JDK 1.7 oder höher empfohlen
- IntelliJ IDEA CE/UE (13.0.1, 13.1.2 oder 14.0.2)
- Internetverbindung (2 MB oder schneller empfohlen)

### **Unterstützte Plattformen**

Das Plugin unterstützt die folgenden IntelliJ IDEA-Versionen:

- 13.0.1 (Community / Ultimate-Edition)
- 13.1.2 (CE / UE)
- 14.0.2 (CE/UE)

## **wird heruntergeladen**

### **Herunterladen von IntelliJ IDEA**

 Sie müssen zuerst IntelliJ IDEA CE / UE installieren, bevor Sie die herunterladen können**Aspose.Cells Java für IntelliJ IDEA (Maven)** Plugin.

Zum Herunterladen von IntelliJ IDEA

1.  Gehe zu<https://www.jetbrains.com/idea/download/>
1. Wählen Sie Ihr laufendes Betriebssystem aus
1. Klicken Sie auf UE- oder CE-Download.

### **Herunterladen von Aspose.Cells Java für das IntelliJ IDEA-Plugin (Maven)**

-  Das Plugin kann direkt von heruntergeladen werden[Hosting-URL des JetBrains-Plugins](https://goo.gl/R4pysl) - im IntelliJ IDEA Plugin-Repository
 und höher können mit der IDE-Option installiert werden:
 - Gehe zu**Datei** > **Einstellungen**...>**Plugins**
 - Auswählen "**Plugin von Datenträger installieren"**
- Oder es kann mithilfe der IDE-Optionen direkt installiert werden (ohne manuellen Download von der JetBrains-Plug-in-Hosting-URL):
 - Gehe zu**Datei** > **Einstellungen**...>**Plugins**
 - Auswählen**"Repositories durchsuchen"**
 - Geben Sie in das Suchfeld ein: "Aspose"
 - Auswählen "**Aspose Cells Java für IntelliJ IDEA Maven**" Plugin zu installieren.
