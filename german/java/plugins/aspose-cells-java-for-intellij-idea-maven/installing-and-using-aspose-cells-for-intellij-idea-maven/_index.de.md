## **Installieren**
### **So installieren Sie Aspose.Cells Java für IntelliJ IDEA (Maven)**
 Die folgenden Schritte zeigen die Installation**Aspose.Cells Java für IntelliJ IDEA (Maven)** Plugin.

 Das Plugin ist auf der CE von IntelliJ IDEA aufgeführt[offizielle Plugin-Website](https://goo.gl/R4pysl) - Plugin-Repository als**Aspose.Cells Java für IntelliJ IDEA Maven**, sodass Sie es nicht manuell herunterladen und von der Festplatte installieren müssen, aber diese zusätzliche Option ist weiterhin in IntelliJ IDEA vorhanden.

**Aspose.Cells Java für IntelliJ IDEA (Maven)** Plugin kann einfach (ohne manuellen Download) aus IntelliJ IDEA heraus installiert werden. Es zu tun:

1.  Auswählen**Einstellungen** von dem**Datei** Speisekarte.
1.  Klicken**Plugins** , durchsuchen Sie dann die Repositories in IntelliJ IDEA.

## **Verwenden**
### **Aspose.Cells Maven Projektassistent**
 Aspose.Cells Maven Projekt zur Verwendung erstellen[Aspose.Cells for Java API](http://goo.gl/c1eSD2) Innerhalb von IntelliJ IDEA benötigen Sie eine Internetverbindung.

Führen Sie dann die folgenden Schritte aus:

 \1. Auswählen**Neues Projekt**.
 \2. Auswählen**Aspose.Cells Maven Projekt** 
 \3. Klicken**Nächste**. 

![todo: Bild_alt_Text](m1du9a1.jpg)


 \4. Zur Verfügung stellen**Gruppen-ID, Artefakt-ID** und**Ausführung** für Ihr maven-Projekt und klicken Sie**Nächste.**

![todo: Bild_alt_Text](khijrce.jpg)


`    ` Sie können das Kontrollkästchen „Also Download Examples Source Code“ aktivieren, um die Beispiele für die Verwendung von API herunterzuladen (wie im obigen Schnappschuss gezeigt).

 \6. Zur Verfügung stellen**Projektname** und**Ort**genau wie beim Erstellen einer Java-Anwendung

![todo: Bild_alt_Text](0gszuiu.jpg)


 \7. Klicken**Fertig**.
 `    `Dadurch wird die abgerufen[Aspose.Cells for Java API](http://goo.gl/c1eSD2) neueste maven Abhängigkeitsreferenz von Aspose Cloud Maven Repository und konfigurieren Sie es in pom.xml
 `     `Wenn Sie das Kontrollkästchen "Also Download Example Source Code" aktiviert haben. Das Herunterladen der Beispiele beginnt ebenfalls mit der[Aspose.Cells for Java API GitHub-Repository](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

![todo: Bild_alt_Text](eezoq3s.jpg)

![todo: Bild_alt_Text](bujsm8v.jpg)

 `     `Herzlichen Glückwunsch!{{< emoticons/cross >}} Sie haben erfolgreich das Projekt Aspose.Cells Maven zur Verwendung erstellt[Aspose.Cells for Java API](http://goo.gl/c1eSD2).

![todo: Bild_alt_Text](2oon4vh.jpg)

`      `Das erstellte Projekt Aspose.Cells for Java API Maven kann gemäß den Anforderungen Ihres Projekts geändert werden.

 `      `Wenn Sie sich für den Download entschieden haben[Beispiele Quellcodes](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples), können Sie den Aspose.Cells Beispielassistenten verwenden, um die heruntergeladenen Beispiele in Ihr Projekt zu kopieren / zu erstellen.
