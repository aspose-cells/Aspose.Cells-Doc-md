## **Installation**
### **Wie man Aspose.Cells Java für IntelliJ IDEA (Maven) installiert**
Die nachstehenden Schritte zeigen, wie das Plugin **Aspose.Cells Java für IntelliJ IDEA (Maven)** installiert wird.

Das Plugin ist auf der IntelliJ IDEA-CE [offiziellen Plugin-Website](https://goo.gl/R4pysl) - Plugin-Repository als **Aspose.Cells Java für IntelliJ IDEA Maven** aufgeführt, sodass Sie es nicht manuell herunterladen und von der Festplatte installieren müssen. Diese zusätzliche Option besteht jedoch immer noch in IntelliJ IDEA.

Das Plugin **Aspose.Cells Java für IntelliJ IDEA (Maven)** kann problemlos (ohne manuellen Download) von innerhalb von IntelliJ IDEA installiert werden. So geht's:

1. Wählen Sie **Einstellungen** im **Datei**-Menü aus.
1. Klicken Sie auf **Plugins**, dann durchsuchen Sie die Repositories in IntelliJ IDEA.

## **Verwendung**
### **Aspose.Cells Maven-Projekt-Assistent**
Um ein Aspose.Cells Maven-Projekt für die Verwendung der [Aspose.Cells for Java-API](http://goo.gl/c1eSD2) innerhalb von IntelliJ IDEA zu erstellen, benötigen Sie eine Internetverbindung.

Befolgen Sie dann die folgenden Schritte:

\1. Wählen Sie **Neues Projekt**.
\2. Wählen Sie **Aspose.Cells Maven-Projekt** aus 
\3. Klicken Sie auf **Weiter**. 

![todo:image_alt_text](m1du9a1.jpg)


\4. Geben Sie eine **GroupId, ArtifactId** und **Version** für Ihr Maven-Projekt ein und klicken Sie auf **Weiter**.

![todo:image_alt_text](khijrce.jpg)


`    `Sie können das Kontrollkästchen "Beispiele für die Verwendung des APIs auch herunterladen" auswählen, um die Beispiel-Nutzungsbeispiele des APIs herunterzuladen (wie im obigen Screenshot gezeigt)

\6. Geben Sie einen **Projektnamen** und den **Standort** an, ähnlich wie bei der Erstellung jeder Java-Anwendung

![todo:image_alt_text](0gszuiu.jpg)


\7. Klicken Sie auf **Fertigstellen**.
`    `Dadurch wird die [Aspose.Cells for Java API](http://goo.gl/c1eSD2) aktuellste Maven-Abhängigkeitsreferenz aus dem Aspose Cloud Maven Repository abgerufen und in der pom.xml konfiguriert
`     `Wenn Sie die Checkbox "Auch Beispiele-Quellcode herunterladen" ausgewählt haben, beginnt auch der Download der Beispiele aus dem [Aspose.Cells for Java API GitHub-Repository](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)

![todo:image_alt_text](eezoq3s.jpg)

![todo:image_alt_text](bujsm8v.jpg)

`     `Congratulations ! {{< emoticons/cross >}} You have successfully created Aspose.Cells Maven project for using [Aspose.Cells for Java API](http://goo.gl/c1eSD2).

![todo:image_alt_text](2oon4vh.jpg)

`      `Das erstellte Aspose.Cells for Java API Maven-Projekt ist bereit, entsprechend den Anforderungen Ihres Projekts angepasst zu werden.

`      `Wenn Sie sich entschieden haben, [Beispiele-Quellcodes herunterzuladen](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples), können Sie den Aspose.Cells Examples Wizard verwenden, um die heruntergeladenen Beispiele in Ihr Projekt zu kopieren/erstellen.
