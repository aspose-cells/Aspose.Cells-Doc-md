Diese Seite beschreibt die interessantesten neuen Aspose.Cells-Funktionen, die in den letzten Versionen eingeführt wurden.

## Aspose.Cells for Java 22.12
* Unterstützung für den Export von Excel nach JSON mit mehrschichtiger Hierarchiestruktur

## Aspose.Cells for Java 22.11

Unterstützung zur Unterscheidung zwischen dynamischer Array-Formel und Legacy-Array-Formel
* Unterstützung zum Lesen/Speichern dynamischer Array-Formeln für das xlsb-Dateiformat

## Aspose.Cells for Java 22.10

* Unterstützung zum Hinzufügen und Aktualisieren von Diagrammen mit CRTX-Vorlagendatei

## Aspose.Cells for Java 22.9

* Unterstützung zum Festlegen von Tabellenformeln für Zellen
* Unterstützung zum Festlegen dynamischer Array-Formeln mit benutzerdefinierten Funktionen und Berechnungsmodulen
* Unterstützung zum Festlegen voreingestellter Verlaufseinstellungen für das Verlaufsfüllformat
* Unterstützung zum Anwenden von Umrisslinien mit Themenfarben für den Bereich

## Aspose.Cells for Java 22.8

* Unterstützt den Export von Kommentaren und Notizen, wie sie auf dem Blatt angezeigt werden, in HTML
* Unterstützung der Angabe von Arbeitsblättern zur Ausgabe während der Konvertierung in pdf/xps

## Aspose.Cells for Java 22.7

* Unterstützung zum Lesen von Vorlagendateien im OOXML-Format, die anormale ZIP-Daten enthalten und nicht von ZipInputStream von Java verarbeitet werden können
* Unterstützung für den Export von HTML in eine einzelne Datei mit Kopf-/Fußzeile der Seite
* Unterstützung, um die absolute Position und Größe des Bereichs (in Punkten) zu erhalten

## Aspose.Cells for Java 22.6

* Verbesserung beim Berechnen und Verfolgen von Formeln mit Formelberechnungskette

## Aspose.Cells for Java 22.5

Verbesserung der Speicherleistung beim Festlegen der Zellenformel beim Speichern der Arbeitsmappe im LightCells-Modus

## Aspose.Cells for Java 22.4

* Verbesserte Optionen zum Steuern leerer Zellen für den Export von CSV.
* Unterstützung komplizierter Aufteilung und Aktualisierung der bedingten Formatierung für Einfüge-/Löschvorgänge

## Aspose.Cells for Java 22.3

* Unterstützung für die Neuberechnung von Zellen, die vom definierten Namen abhängen, wenn der definierte Name geändert und die Berechnungskette der Arbeitsmappe aktiviert wird.
* Unterstützung, um die Blätter der Zelle rekursiv gemäß der Berechnungskette der Formeln zu erhalten

## Aspose.Cells for Java 22.2

* Unterstützung zum Extrahieren definierter Namen aus NameCollection durch angegebenen Bereichsfilter

## Aspose.Cells for Java 22.1

* Unterstützung zum Generieren von bearbeitbaren PPTX
* Unterstützung für den Import großer CSV-Dateien in mehrere Arbeitsblätter, wenn die Datenzeile/-spalte das Limit überschreitet
* Unterstützung für den CSV-Import mit LightCells
* Unterstützung zum Entfernen des angegebenen externen Links
* Unterstützung zum Beibehalten von Formeln beim Entfernen externer Links

## Aspose.Cells for Java 21.12

* Unterstützt die Konvertierung von Excel in SQL-Skripts
* Unterstützt den Import und Export von einfachen XML-Daten
Unterstützung, um so viele Daten wie möglich aus beschädigten XLS-Dateien zu lesen
* Unterstützung, um den Prozess von SmartMarker zu unterbrechen
* Verbessern Sie die Leistung beim Hinzufügen von Bereichen für die bedingte Formatierung und Validierung
* Legen Sie die höchste Priorität für die neu hinzugefügte Formatbedingung fest, genau wie das Verhalten von MS Excel

## Aspose.Cells for Java 21.11

* Unterstützung bei der Berechnung von XLookup-Formeln/-Funktionen.
* Integrieren Sie das Speichern von JSON und Bildern in das Speichern von Arbeitsmappen.
* Verbessern Sie die Leistung beim Konvertieren von Excel in HTML.
