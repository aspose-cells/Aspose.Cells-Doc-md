## **Willkommen in der Aspose.Cells-Dokumentation**
**Aspose.Cells** ist das flexibelste Komponente, die es Java-Anwendungen ermöglicht, Excel-Arbeitsmappen zu erstellen und zu verwalten, ohne dass **Microsoft Excel** auf dem System installiert sein muss. Im Gegensatz zu ähnlichen Produkten anderer Anbieter unterstützt Aspose.Cells nicht nur die Generierung von Tabellenkalkulationen und andere grundlegende Dateiformatierungsfunktionen, sondern auch eine Reihe von erweiterten Funktionen. Diese erweiterten Funktionen machen es für Entwickler wesentlich bequemer, den Inhalt der Tabellenkalkulation, die Formatierung von Zellen und den Dateischutz zu manipulieren. Aspose.Cells ermöglicht auch das Importieren von Daten in Tabellenkalkulationen aus verschiedenen Datenquellen, das Hinzufügen von gängigen und komplexen mathematischen, DateTime-, Finanz- und Textformeln/Funktionen, die Manipulation von Diagrammen, Bildern, Kommentaren, Zeichenobjekten und Steuerungen usw.

Aspose.Cells for Java bietet auch eine komplette Rasterlösung für Webclients; es verfügt über eine GUI-basierte Steuerung wie folgt:

- Aspose.Cells.GridWeb, das Webanwendungen unterstützt

Mit Aspose.Cells.GridWeb für die JAVA-Rastersteuerung können Entwickler ganz einfach aus Excel®-Tabellenkalkulationen importieren und in sie exportieren. Jeder Benutzer, der mit Microsoft Excel vertraut ist, kann das Erscheinungsbild des Rasters steuern. Aspose.Cells bietet eine umfangreiche Auswahl an Funktionen, wie das Steuern des Designs und Layouts des Rasters, das Verwalten von mehreren Arbeitsblättern, das Verwalten von Zellen, Zeilen und Spalten eines Arbeitsblatts, das Erstellen und Berechnen von Formeln usw.

## **Produktbeschreibung**
Aspose.Cells kann mit jeder Art von Anwendung, ob Java-Webanwendung oder Java-Desktopanwendung, verwendet werden. Darüber hinaus kann die Bibliothek auch mit anderen Lösungen wie PHP usw. verwendet werden. Aspose.Cells bietet voll ausgestattete Demos und Arbeitsbeispiele in Java, damit Entwickler ein besseres Verständnis der vom Komponenten bereitgestellten Funktionen haben.

Neben der Unterstützung von gängigen Funktionen wie dem Erstellen oder Lesen von nativen Excel-Dateien bietet Aspose.Cells auch viele erweiterte Funktionen, wie das Speichern und Öffnen von Excel-Dateien in und aus Streams, das Senden von Ausgaben an einen Clientbrowser, das Importieren und Exportieren von Daten aus einem **ResultSet** oder einem **Array**, das Importieren von Bildern und Diagrammen sowie das Erstellen von Bildern und Diagrammen über die API, das Importieren von Formeln aus einer Designer-Tabelle und das Konfigurieren des Seitenlayouts über die API.

Darüber hinaus gibt es auch eine Reihe von einzigartigen Funktionen, die die Entwickler nur in Aspose.Cells finden können, wie die Unterstützung von CSV- und SpreadsheetML-Formaten, das Hinzufügen einer Kopie eines vorhandenen Arbeitsblatts (mit vollständigen Inhalten, Bildern und Diagrammen usw.) in eine Datei, das Festlegen eines Gradientenhintergrunds für Diagramme über die API, das Erstellen von Kommentaren, Auto-Filtern und Seitenwechseln über die API, das Festlegen von komplexen Formeln, die Unterstützung von bedingten Formatierungen, die Unterstützung von Schutzoptionen, die in Microsoft Excel XP oder höher eingeführt wurden, die Manipulation von benannten Bereichen, die Suche nach APIs, die Unterstützung von benutzerdefinierten Diagramm-APIs und der Formelberechnungs-Engine usw.
## **Dokumentfunktionen**
- Öffnen von einfachen oder verschlüsselten Excel-Dateien (Excel 95, Excel97, Excel2007/2010/2013/2016/2019, Office365) aus verschiedenen Quellen
- Öffnen von einfachen Zahlen-Dateien (erstellt von Apples Anwendung "Numbers") aus verschiedenen Quellen
- Speichern von Excel-Dateien (Excel97- Excel2007/2010/2013/2016/2019) in verschiedenen Formaten (XLS, XLSX/XLSM, XLTX, XLTM, XLSB, CSV, SpreadsheetML, tabulatorbegrenzt/TSV, HTML usw.)
- Konvertieren von Excel-Datei in PDF-Dokument
- Konvertieren von Excel-Tabelle in XPS-Dokument
- Konvertieren in das verschlagwortete Bildformat **(TIFF)**
- Lesen und Schreiben im **OpenDocument Spreadsheet (ODS)**-Format
- Ändern Sie die Dokumenteigenschaften von Excel-Dateien
## **Arbeitsblattfunktionen**
### **Anzeigeoptionen**
- **Arbeitsblatt ausblenden oder anzeigen**
- **Registerkarten, Bildlaufleisten, Gitterlinien sowie Zeilen- und Spaltenüberschriften** einblenden oder ausblenden
- **Seitenumbrüche in der Vorschau anzeigen**
- Für das Arbeitsblatt den **Zoomfaktor** festlegen
- Ausgewählte Daten während des Bildlaufs mithilfe von **Spalten fixieren** sichtbar lassen
### **Arbeitsblattschutz**
- Inhalt, Objekte und Szenarien eines Arbeitsblatts **schützen**
### **Seiteneinrichtungsoptionen**
- **Seitenlayout, Skalierung, Papierformat, Druckqualität** usw. festlegen
- **Seitenränder** oben, links, rechts, unten usw. festlegen
- **Kopf- und Fußzeilen** festlegen
- **Druckbereich**, **Drucktitel** usw. festlegen
### **Seitenumbrüche**
- **Horizontale** und **vertikale Seitenumbrüche** an einer bestimmten Zelle einfügen
### **Zeilen & Spalten Funktionen**
- **Spaltenbreite** bei Bedarf anpassen
- Breite und Höhe der Zellen automatisch anhand ihres Inhalts anpassen
- Zeilen und Spalten **ein- oder ausfügen**
- **Zeilen und Spalten** ausblenden oder anzeigen
- **Zeilen und Spalten** gruppieren oder gruppieren aufheben
### **Datenmanagement Funktionen**
- Daten zur Laufzeit in gewünschte Zellen hinzufügen
- Daten aus verschiedenen Datenquellen in Arbeitsblätter **importieren**
- Daten aus Zellen entsprechend ihrer Datentypen abrufen
- **Zellendaten in Array exportieren**
- Suchen oder Suchen Sie nach bestimmten Daten in Zellen
- Setzen von **Formeln mit integrierten oder Add-In-Funktionen**
- Setzen von **Array- und R1C1-Referenzstil-Formel**
- Hinzufügen verschiedener Arten von Hyperlinks
- Erstellen oder Zugreifen auf benannte Bereiche
- Anwendung von **Datenvalidierung** und **Automatischer Filterung**
- Anwendung von **Zwischensummen** auf Arbeitsblatttabellen
### **Formatierungsfunktionen**
- Anwendung von **Zahlenformatierungseinstellungen** für Zellen
- Formatieren einer Zelle als Datum
- Setzen des **Textausrichtung**
- Mehrere Zellen zusammenführen
- Anwendung von verschiedenen Arten von **Schriftart-Einstellungen** für die Zellen
- Verschiedene Farben auf Zellen, Schriften, Gitterlinien, Grafikobjekte usw. anwenden.
- Setzen von **Rich-Text** in einer einzelnen Zelle
- Anwendung verschiedener **Rahmeneinstellungen** auf Zellen
- Anwendung verschiedener **Hintergrundmuster** auf Zellen
- Anwendung von **Formatierungseinstellungen** auf ein Arbeitsblatt, eine Zeile oder eine Spalte usw.
- Anwendung von **Bedingter Formatierung** auf Zellen
