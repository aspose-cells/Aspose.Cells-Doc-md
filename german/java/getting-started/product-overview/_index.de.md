## **Willkommen bei der Aspose.Cells-Dokumentation**
**Aspose.Cells** ist die flexibelste Komponente, mit der Java-Anwendungen Excel-Tabellen erstellen und verwalten können, ohne dass dies erforderlich ist**Microsoft Excel**auf dem System installiert werden. Im Gegensatz zu ähnlichen Produkten anderer Anbieter unterstützt Aspose.Cells nicht nur die Erstellung von Tabellenkalkulationen und andere grundlegende Dateiformatierungsfunktionen, sondern auch eine Reihe von erweiterten Funktionen. Diese erweiterten Funktionen machen es den Entwicklern viel bequemer, den Inhalt von Tabellenkalkulationen, die Zellenformatierung und den Dateischutz zu manipulieren. Aspose.Cells ermöglicht auch den Import von Daten in Tabellenkalkulationen aus verschiedenen Datenquellen, das Hinzufügen allgemeiner und komplexer mathematischer, DateTime-, Finanz-, Textformeln / -funktionen, unterstützt die Manipulation von Diagrammen, Bildern, Kommentaren, Zeichenobjekten und Steuerelementen usw.

Aspose.Cells for Java bietet auch eine vollständige Grid-Lösung für Web-Clients; Es hat eine GUI-basierte Steuerung wie folgt:

- Aspose.Cells.GridWeb, das Webanwendungen unterstützt

Mit Aspose.Cells.GridWeb for JAVA Grid Control können Entwickler ganz einfach Excel®-Tabellen importieren und exportieren. Jeder Benutzer, der mit Microsoft Excel vertraut ist, kann das Erscheinungsbild des Rasters steuern. Aspose.Cells bietet umfangreiche Funktionen wie das Steuern des Designs und Layouts des Rasters, das Verwalten mehrerer Arbeitsblätter, das Verwalten von Zellen, Zeilen und Spalten eines Arbeitsblatts, das Erstellen und Berechnen von Formeln usw.

## **Produktbeschreibung**
Aspose.Cells kann mit jeder Art von Anwendung verwendet werden, ob Java Webanwendung oder Java Desktopanwendung. Darüber hinaus kann die Bibliothek auch mit einigen anderen Lösungen wie PHP usw. verwendet werden. Aspose.Cells bietet voll funktionsfähige Demos und Arbeitsbeispiele, die in Java geschrieben wurden, damit Entwickler die von der Komponente bereitgestellten Funktionen besser verstehen können.

Neben der Unterstützung allgemeiner Funktionen wie dem Erstellen oder Lesen nativer Excel-Dateien bietet Aspose.Cells auch viele erweiterte Funktionen wie das Speichern und Öffnen von Excel-Dateien in und aus Streams, das Senden der Ausgabe an einen Client-Browser, das Importieren und Exportieren von Daten aus einem**Ergebnissatz** oder ein**Array**, Importieren von Bildern und Diagrammen, Erstellen von Bildern und Diagrammen über API, Importieren von Formeln aus einer Designer-Tabelle und Konfigurieren der Seiteneinrichtung über API.

Darüber hinaus gibt es auch eine Reihe einzigartiger Funktionen, die die Entwickler nur in Aspose.Cells finden können, wie die Unterstützung von CSV- und SpreadsheetML-Formaten, das Hinzufügen der Kopie eines vorhandenen Arbeitsblatts (mit vollständigem Inhalt, Bildern und Diagrammen usw.) zu einer Datei, Festlegen eines Verlaufshintergrunds für Diagramme bis API, Erstellen von Kommentaren, automatischen Filtern und Seitenumbrüchen bis API, Festlegen komplexer Formeln, Unterstützung bedingter Formatierung, Unterstützung von Schutzoptionen, die in Microsoft Excel XP oder höher eingeführt wurden, Bearbeiten von benannten Bereichen, Auffinden von APIs, Unterstützung von benutzerdefiniertem Diagramm API und Formelberechnungsmaschine usw.
## **Dokumentfunktionen**
- Öffnen Sie einfache oder verschlüsselte Excel-Dateien (Excel 95, Excel97, Excel2007/2010/2013/2016/2019, Office365) aus verschiedenen Quellen
- Öffnen Sie Plain Numbers-Dateien (erstellt von Apples „Numbers“-Anwendung) aus verschiedenen Quellen
- Speichern Sie Excel-Dateien (Excel97- Excel2007/2010/2013/2016/2019) in verschiedenen Formaten (XLS, XLSX/XLSM, XLTX, XLTM, XLSB, CSV, SpreadsheetML, Tab Delimited/TSV, HTML etc.)
- Konvertieren Sie eine Excel-Datei in ein PDF-Dokument
- Konvertieren Sie eine Excel-Tabelle in ein XPS-Dokument
-  In Tagged Image File Format konvertieren**(TIFF)**
-  Lesen und Schreiben**OpenDocument-Tabelle (ODS)** Format
- Ändern Sie die Dokumenteigenschaften von Excel-Dateien
## **Arbeitsblattfunktionen**
### **Anzeigeoptionen**
- **Ausblenden oder anzeigen** ein Arbeitsblatt
-  Ein- oder ausblenden**Registerkarten, Bildlaufleisten, Gitternetzlinien sowie Zeilen- und Spaltenüberschriften**
- **Seitenumbrüche in der Vorschau anzeigen**
-  Satz**Zoomstufe** für das Arbeitsblatt
-  Ausgewählte Daten beim Scrollen sichtbar bleiben lassen mit**Scheiben einfrieren**
### **Arbeitsblattschutz**
- **Schützen Sie Inhalte, Objekte und Szenarien** eines Arbeitsblattes
### **Seiteneinrichtungsoptionen**
-  Satz**Seitenausrichtung, Skalierung, Papiergröße, Druckqualität** usw.
-  Satz**Seitenränder** wie oben, links, rechts, unten usw.
-  Satz**Kopf-und Fußzeilen**
-  Satz**Druckbereich**, **Titel drucken** usw.
### **Seitenumbrüche**
-  Einfügung**horizontal** und**vertikale Seitenumbrüche** an einer bestimmten Zelle
### **Funktionen für Zeilen und Spalten**
- **Spaltenbreite anpassen** nach Bedarf
- Passen Sie die Breite und Höhe von Zellen automatisch an ihren Inhalt an
- **Einfügen oder löschen** Reihen und Spalten
- **Ausblenden oder anzeigen** Reihen und Spalten
- **Gruppieren oder Gruppierung aufheben** Reihen und Spalten
### **Datenverwaltungsfunktionen**
- Fügen Sie zur Laufzeit Daten zu gewünschten Zellen hinzu
- **Importieren Sie Daten in Arbeitsblätter** aus unterschiedlichen Datenquellen
- Rufen Sie Daten aus Zellen gemäß ihren Datentypen ab
- **Exportieren Sie Zellendaten in ein Array**
- Suchen oder suchen Sie bestimmte Daten in Zellen
-  Satz**Formeln mit integrierten oder Add-in-Funktionen**
-  Satz**Reihe** und**R1C1** Referenzstilformel
- Fügen Sie verschiedene Arten von Hyperlinks hinzu
- Benannte Bereiche erstellen oder darauf zugreifen
-  Sich bewerben**Datenvalidierung** und**automatische Filterung**
-  Sich bewerben**Zwischensummen** zu Arbeitsblatttabellen
### **Formatierungsfunktionen**
-  Sich bewerben**Zahlenformateinstellungen** für Zellen
- Zelle als Datum formatieren
-  Satz**Textausrichtung**
- Mehrere Zellen zusammenführen
- Wenden Sie verschiedene Arten an**Schriftarteinstellungen** für die Zellen
- Wenden Sie verschiedene Farben auf Zellen, Schriftarten, Gitterlinien, Grafikobjekte usw. an.
-  Satz**reichhaltiger Text** in einer einzigen Zelle
-  Bewerben Sie sich anders**Randeinstellungen** auf Zellen
-  Bewerben Sie sich anders**Hintergrundmuster** auf Zellen
-  Sich bewerben**Formateinstellungen** auf einem Arbeitsblatt, Zeile oder Spalte etc.
-  Sich bewerben**bedingte Formatierung** auf Zellen

-  Hinzufügen**Bilder** in Arbeitsblätter zur Laufzeit
- Erstellen und fügen Sie Kommentare zu Zellen hinzu
- Diagramm in Bilddatei umwandeln
- Arbeitsblatt in Bilddatei konvertieren
- Arbeitsblätter drucken
