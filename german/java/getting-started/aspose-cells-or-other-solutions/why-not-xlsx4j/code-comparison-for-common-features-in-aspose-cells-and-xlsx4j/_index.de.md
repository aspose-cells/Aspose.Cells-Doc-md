Die Entwickler, die nach Tabellenkalkulationsbibliotheken suchen, müssen möglicherweise die Funktionen durchgehen, die von den auf dem Markt verfügbaren Bibliotheken bereitgestellt werden. Bei der Auswahl einer Option werden hauptsächlich einige Faktoren berücksichtigt:

- Merkmale
- Anstrengungen erforderlich, um die Bibliothek zu benutzen
- Leistung
