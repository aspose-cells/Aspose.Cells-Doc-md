{{% alert color="primary" %}}

 Aspose.Cells können Sie ganz einfach zur Auswertung aus dem herunterladen[Download-Seite](https://releases.aspose.com/cells/java/). Der Evaluierungs-Download ist derselbe wie der Download für die Vollversion: Die Evaluierung wird lizenziert, wenn Sie eine Lizenz beantragen.

{{% /alert %}}

Die Evaluierungsversion von Aspose.Cells (d. h. Aspose.Cells ohne angewendete Lizenz) bietet die meisten Produktfunktionen. Eine Einschränkung besteht darin, dass beim Öffnen / Speichern eines beliebigen Tabellenkalkulationsdokuments immer ein zusätzliches Arbeitsblatt mit Bewertungswasserzeichen eingefügt wird.
