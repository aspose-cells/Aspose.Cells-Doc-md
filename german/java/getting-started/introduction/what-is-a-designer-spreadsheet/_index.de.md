## **Möchten Sie die Entwicklung beschleunigen?**

{{% alert color="primary" %}}

Sie können Aspose.Cells verwenden, um Excel-Dateien von Grund auf mit Hilfe seiner flexiblen API zu erstellen. Eine der häufigsten Anwendungen von Aspose.Cells besteht darin, Excel-Dateien mit Daten zu erstellen und zu füllen, die aus einer vorhandenen Datenbank, einem Benutzer oder anderen Quellen dynamisch stammen können. Neben der Hinzufügung von Daten zu Excel-Dateien müssen Sie auch die Formatierung anwenden (Aussehen und Gefühl).

Formatierung dauert Zeit. Was könnte die Entwicklung beschleunigen?

{{% /alert %}}

## **Die Lösung - Designer-Arbeitsblätter**

Um Ihre Entwicklung zu beschleunigen, bietet Aspose.Cells das aufregende Feature Designer-Arbeitsblatt.

### **Was ist ein Designer-Arbeitsblatt?**

Ein Designer-Arbeitsblatt ist eine Excel-Datei, die mit Microsoft Excel erstellt wurde und in der bereits alle Formatierungen angewendet wurden, und der Inhalt zur Entwurfszeit festgelegt werden kann. Es ähnelt eher einer Vorlagendatei, die keine Daten, sondern nur die Formatierung enthält, wie unten gezeigt:

### **Wie wird es verwendet?**

Nach Erstellung eines Designer-Arbeitsblatts verwenden Sie Aspose.Cells, um das Designer-Arbeitsblatt zu lesen und Daten zu den festgelegten Positionen hinzuzufügen (gemäß den vordefinierten Formateinstellungen) zur Laufzeit. Zum Beispiel sehen Sie im obigen Screenshot, dass wir bereits zur Entwurfszeit entschieden haben, die Spalten **Produktname**, **Kategorienname**, **Menge pro Einheit** und **Einheiten auf Lager** hinzuzufügen. Sie haben noch keine Daten: diese werden zur Laufzeit hinzugefügt. Die obige Abbildung zeigt eine sehr einfache Tabelle, aber Designer-Arbeitsblätter sparen Ihnen auch Zeit, selbst bei komplexen Tabellen.

### **Smart Marker**

Zusätzlich ermöglicht Ihnen Aspose.Cells das Hinzufügen von [intelligenten Markern](/cells/de/java/smart-markers/) zum Designer-Arbeitsblatt. Die intelligenten Marker werden zur Laufzeit mithilfe der Methode Process der Klasse [**WorkbookDesigner**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookdesigner) verarbeitet. Dies kann Ihnen dabei helfen, weniger Code zu schreiben und dennoch ein großartiges Ergebnis zu erzielen.

[Weitere Informationen zu intelligenten Markern finden Sie hier](/cells/de/java/smart-markers/).
