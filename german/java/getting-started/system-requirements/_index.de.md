{{% alert color="primary" %}} 

Aspose.Cells for Java erfordert keine Installation von Microsoft Excel, da Aspose.Cells selbst eine Engine zum Erstellen, Konvertieren und Rendering von Excel-Tabellen ist. Um von Aspose.Cells erstellte Excel-Dokumente anzuzeigen, müssen Sie mindestens Microsoft Excel Viewer installiert haben.

{{% /alert %}} 
## **Unterstützte Betriebssysteme**
Aspose.Cells for Java unterstützt jedes Betriebssystem, das die Java-Laufzeit ausführt, einschließlich, aber nicht beschränkt auf:
### **Windows**
- Microsoft Windows 2000 ( x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows Server 2003 (x64, x86)
- Microsoft Windows Server 2008 (x64, x86)
- Microsoft Windows Server 2012 (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows 7 (x64, x86)
- Microsoft Windows 8 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
### **Linux**
- Linux (Ubuntu, openSUSE, CentOS und andere)
### **Mac**
- Mac OS X usw.
## **Unterstützte Java-Versionen**
Aspose.Cells for Java unterstützt die folgenden Java-Versionen:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- oder höher
