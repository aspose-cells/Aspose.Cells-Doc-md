**Willkommen bei Aspose.Cells for Java**

Aspose.Cells for Java ist eine Programmierklassenbibliothek, die es Softwareentwicklern ermöglicht, Tabellendateien in ihren eigenen Anwendungen zu manipulieren und zu verarbeiten.

In Verbindung mit APIs und GUI-Steuerungen beschleunigt Aspose.Cells for Java die Microsoft Excel®-Programmierung und -Konvertierung.

Aspose.Cells for Java unterstützt die beliebten Tabellendateiformate (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), die Ihr Unternehmen tagtäglich verwendet. Es ermöglicht auch das Exportieren von Excel-Dateien in PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, Klartext sowie beliebte Bildformate wie TIFF, JPG, PNG, BMP und SVG.
