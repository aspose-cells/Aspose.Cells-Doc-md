**Willkommen unter Aspose.Cells for Java**

 Aspose.Cells for Java ist eine Programmierklassenbibliothek, die es Softwareentwicklern ermöglicht, Tabellenkalkulationsdateien in ihren eigenen Anwendungen zu manipulieren und zu verarbeiten.

In Kombination mit APIs und GUI-Steuerelementen beschleunigt Aspose.Cells for Java die Microsoft Excel®-Programmierung und -Konvertierung.

Aspose.Cells for Java unterstützt die gängigen Dateiformate für Tabellenkalkulationen (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), die Sie täglich im Unternehmen verwenden. Es ermöglicht auch den Export von Excel-Dateien in PDF, DOCX, PPTX, JSON, XPS, HTML, MHTML, Nur-Text und gängige Bildformate, einschließlich TIFF, JPG, PNG, BMP and SVG.
