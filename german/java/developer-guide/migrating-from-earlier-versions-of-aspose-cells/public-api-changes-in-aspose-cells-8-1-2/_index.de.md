{{% alert color="primary" %}} 

Dieses Dokument beschreibt Änderungen an Aspose.Cells API von Version 8.1.1 zu 8.1.2, die für Modul-/Anwendungsentwickler von Interesse sein könnten. Es enthält nicht nur neue und aktualisierte öffentliche Methoden, sondern auch eine Beschreibung aller Änderungen im Verhalten hinter den Kulissen in Aspose.Cells.

{{% /alert %}} 
## **Unterstützung für Warnung hinzugefügt, wenn Schriftersetzung auftritt**
Mit Aspose.Cells for Java 8.1.2 wurden die WarningInfo- und WarningType-Klassen, die IWarningCallback-Schnittstelle und die SaveOptions.WarningCallback- und ImageOrPrintOptions.WarningCallback-Eigenschaften hinzugefügt, damit die Entwickler Warnungen erhalten können, wenn beim Konvertieren von Tabellenkalkulationen in Bilder, XPS- und PDF-Formate Schriftarten ersetzt werden.

{{% alert color="primary" %}} 

 Bitte lesen Sie den ausführlichen Artikel auf[Erhalten von Warnungen für die Ersetzung von Schriftarten beim Rendern von Tabellenkalkulationen](http://aspose.com/docs/display/cellsjava/Get+Warnings+for+Font+Substitution+while+Rendering+Excel+File) für mehr Informationen.

{{% /alert %}}
## **Veraltete PdfSaveOptions.ChartImageType-Eigenschaft gelöscht**
Aspose.Cells for Java 8.1.2 hat die veraltete Eigenschaft PdfSaveOptions.ChartImageType aus dem öffentlichen API entfernt.
