 Der folgende Beispielcode bettet die Formel in die Smart-Marker-Variable mit dem Namen Test ein, und ihr Datenquellenname ist ebenfalls Test, sodass das vollständige Feld mit dem Formelparameter so aussieht**&=$Test(formel)** und nach der Ausführung des Codes, die[Endgültige Excel-Ausgabedatei](47153156.xlsx) wird Formeln in Zellen von A1 bis A5 haben.
