## Textfeld zum Arbeitsblatt in Excel hinzufügen

Im Excel-Programm (Version 07 und höher) gibt es zwei Stellen, an denen Sie Textfelder einfügen können. Eine in "Formen einfügen", die andere befindet sich auf der rechten Seite des oberen Menüs der Option "Einfügen".

## Wie erstellt man

Sie können Textfelder mit horizontalem oder vertikalem Text erstellen.

- Wählen Sie die entsprechende Option (horizontal oder vertikal)
- Klicken Sie mit der linken Maustaste auf die Seite
- Halten Sie die linke Taste gedrückt und ziehen Sie eine Strecke auf der Seite
- Lassen Sie die linke Taste los

Jetzt erhalten Sie ein Textfeld.
