## **Mögliche Nutzungsszenarien**
Textfelder können Tags haben, die zur Laufzeit durch Text ersetzt werden können, um sie den Anforderungen entsprechend zu konfigurieren. Tags können ein in spitze Klammern eingeschlossenes Label sein '<' and '>. Es können mehrere Tags in einem einzelnen Textfeld vorhanden sein.
