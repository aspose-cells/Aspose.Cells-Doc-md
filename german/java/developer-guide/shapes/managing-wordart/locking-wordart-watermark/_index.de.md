{{% alert color="primary" %}}

Aspose.Cells APIs ermöglichen das Hinzufügen von WordArt-Wasserzeichen auf das Arbeitsblatt in einer Weise, dass das WordArt zu einem Objekt wird, das auf dem Arbeitsblatt bewegt und positioniert werden kann. Es ist auch möglich, das WordArt-Objekt für jede Interaktion wie Bearbeitung, Bewegung und Auswahl zu sperren. Dieser Artikel erläutert die Verwendung der Methode [**Shape.setLockedProperty**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#setLockedProperty-int-boolean-), um einige Aspekte des Wasserzeichens zu sperren.

{{% /alert %}}

## **Sperren des WordArt-Wasserzeichens**

Aspose.Cells APIs ermöglichen das Sperren bestimmter Aspekte des Wasserzeichens, damit die Benutzerinteraktion eingeschränkt oder vollständig blockiert werden kann. Der folgende Codeausschnitt zeigt die Verwendung der API Aspose.Cells for Java, um für jedes Arbeitsblatt in der geladenen Tabellenkalkulation ein Wasserzeichen zu erstellen und die Auswahl, Bewegung, Bearbeitung und Größenänderung des Wasserzeichens zu sperren.
