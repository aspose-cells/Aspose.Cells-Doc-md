{{% alert color="primary" %}}

Verwenden Sie WordArt, um Tabellenkalkulationen spezielle Texteffekte hinzuzufügen. Strecken Sie beispielsweise einen Titel über den oberen Rand der Datei, verzieren Sie Text und passen Sie den Text an eine voreingestellte Form an oder wenden Sie den Text als Hintergrundwasserzeichen auf eine Excel-Tabelle an. Das WordArt-Objekt wird zu einem Objekt, das Sie in Tabellenkalkulationen verschieben oder positionieren können, um Dekorationen hinzuzufügen.

{{% /alert %}}
