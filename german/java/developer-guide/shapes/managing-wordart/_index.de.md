{{% alert color="primary" %}}

Verwenden Sie WordArt, um spezielle Texteffekte zu Tabellen zu hinzuzufügen. Zum Beispiel einen Titel über die Oberseite der Datei strecken, Text dekorieren und den Text in eine voreingestellte Form passen oder den Text als Hintergrundwasserzeichen auf ein Excel-Blatt anwenden. Das WordArt wird zu einem Objekt, das Sie in Tabellen bewegen oder positionieren können, um Dekorationen hinzuzufügen.

{{% /alert %}}

Das folgende Beispiel zeigt, wie ein WordArt-Objekt hinzugefügt wird, um ein Hintergrundwasserzeichen für ein Arbeitsblatt festzulegen.
