## Mögliche Nutzungsszenarien

 Wenn Sie die Farbe des Glow-Effekts einer beliebigen Form ablesen möchten, verwenden Sie bitte die[**Form.Leuchten.Farbe**](https://reference.aspose.com/cells/java/com.aspose.cells/gloweffect#Color)Eigentum. Es hilft Ihnen, die verschiedenen Eigenschaften in Bezug auf die Farbe des in einer Form angewendeten Glüheffekts zu finden.
