{{% alert color="primary" %}}

Standardmäßig lädt Aspose.Cells die gesamte Arbeitsmappe in den Speicher. Es ist möglich, nur bestimmte Blätter zu laden. Dies kann die Leistung verbessern und weniger Speicher verbrauchen. Dieser Ansatz ist nützlich, wenn mit einer großen Arbeitsmappe mit mehreren Arbeitsblättern gearbeitet wird.
