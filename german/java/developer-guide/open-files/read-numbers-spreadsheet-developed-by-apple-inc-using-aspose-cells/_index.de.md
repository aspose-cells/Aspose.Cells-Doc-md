## **Mögliche Nutzungsszenarien**

Numbers ist eine von Apple Inc. Aspose.Cells entwickelte Tabellenkalkulationsanwendung, die Numbers-Tabellenkalkulationen lesen kann, aber das Schreiben darin nicht unterstützt. Es kann die Daten, den Stil und die Formeln der Zahlentabelle lesen.

## **Read Numbers Spreadsheet Entwickelt von Apple Inc. mit Aspose.Cells**

 Der folgende Beispielcode lädt die[Tabelle mit Beispielnummern](sampleNumbersByAppleInc.numbers) und wandelt es um[Ausgabe im PDF-Format](outputNumbersByAppleInc.pdf) . Sie müssen verwenden**[LoadOptions](https://reference.aspose.com/cells/java/com.aspose.cells/LoadOptions)** Klasse und angeben**[LoadFormat.NUMBERS](https://reference.aspose.com/cells/java/com.aspose.cells/loadformat#NUMBERS)**als Parameter in seinem Konstruktor, um es erfolgreich zu laden. Bitte laden Sie beide von den angegebenen Links herunter. Du kannst den Code mit jeder Numbers-Tabelle ausprobieren. Bitte lesen Sie auch die Kommentare des Codes für weitere Hilfe.
