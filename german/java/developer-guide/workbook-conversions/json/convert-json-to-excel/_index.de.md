## **Konvertieren Sie JSON in eine Excel-Arbeitsmappe**

 Sie müssen sich nicht fragen, wie Sie JSON in eine Excel-Datei konvertieren, denn die Bibliothek Aspose.Cells Java hat die beste Entscheidung. Das Aspose.Cells Java API bietet Unterstützung für die Konvertierung des JSON-Formats in Tabellenkalkulationen. Sie können verwenden[**JsonLoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLoadOptions) -Klasse, um zusätzliche Einstellungen für den Import von JSON in Workbook anzugeben.
