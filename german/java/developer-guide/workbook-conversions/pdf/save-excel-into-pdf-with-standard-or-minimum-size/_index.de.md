{{% alert color="primary" %}} 

Standardmäßig speichert Aspose.Cells Excel in PDF mit Standardgröße. Sie können es jedoch auch mit minimaler Größe speichern, indem Sie die [PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions#OptimizationType)-Eigenschaft verwenden. Sie akzeptiert die folgenden Werte

- [PdfOptimizationType.STANDARD](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#STANDARD)
- [PdfOptimizationType.MINIMUM_SIZE](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#MINIMUM_SIZE)

{{% /alert %}}
