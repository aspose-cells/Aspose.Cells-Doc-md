Zur Kompatibilität mit älteren Versionen rendert Aspose.Cells standardmäßig beim Konvertieren von Excel in PDF Gitterlinien als gestrichelte Linie, während moderne Excel Gitterlinien als durchgezogene Linie rendert.

Mit der Option [PdfSaveOptions.GridlineTypes](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setGridlineType-int-) kann Aspose.Cells Gitterlinien auch als durchgezogene Linie rendern.
