## **Mögliche Verwendungsszenarien**
Früher konnte Aspose.Cells beim Speichern einer Excel-Datei im PDF-Format keine Office-Add-Ins rendern. Jetzt rendern Aspose.Cells es problemlos. Sie müssen keine spezielle Methode oder Eigenschaft verwenden, um Office-Add-Ins im Ausgabe-PDF zu rendern. Speichern Sie einfach Ihre Excel-Datei im PDF-Format und es werden Office-Add-Ins gerendert.
## **Office-Add-Ins beim Konvertieren von Excel in PDF anzeigen**
Der folgende Beispielcode speichert die [Beispiel-Excel-Datei](60489783.xlsx), die die Office-Add-Ins enthält. Bitte beachten Sie die [mit der vorherigen Version d.h. 17.11 generierte Ausgabe-PDF-Datei](60489781.pdf) und die [mit der neueren Version d.h. 17.12 und höher generierte Ausgabe-PDF-Datei](60489782.pdf). Die Ausgabe-PDF-Datei der früheren Version ist leer, aber die Ausgabe-PDF-Datei der neueren Version zeigt das Office-Add-In.
