{{% alert color="primary" %}}

Aspose.Cells unterstützt die Konvertierung von XLS-Dateien, die Bilder und Diagramme enthalten, in PDF-Dokumente. Die Aspose.Cells for Java kann unabhängig arbeiten, um eine Tabelle in PDF zu konvertieren: Die Aspose.PDF-APIs sind für die Konvertierung nicht erforderlich.

{{% /alert %}}

Der Prozess kann im Arbeitsspeicher durchgeführt werden, da er nicht von temporären oder Zwischen-XML-Dateien abhängt. Das bedeutet, dass große Excel-Dateien, die z. B. Bilder, Diagramme und andere Zeichenobjekte enthalten, schnell und effizient konvertiert werden können.

{{% alert color="primary" %}}

Wenn die Tabelle Formeln enthält, ist es am besten, die Methode [**Workbook.calculateFormula**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#calculateFormula--) kurz vor dem Rendern in PDF aufzurufen. Dadurch wird sichergestellt, dass formelabhängige Werte neu berechnet werden und die korrekten Werte im PDF gerendert werden.

{{% /alert %}}
