Dieser Artikel bietet Informationen dazu, wie PDF-Lesezeichen beim Konvertieren einer Tabellenkalkulation in PDF eingefügt werden können.

Aspose.Cells ermöglicht es Ihnen, Lesezeichen zur Laufzeit hinzuzufügen. PDF-Lesezeichen können die Navigierbarkeit langer Dokumente drastisch verbessern. Beim Hinzufügen von Lesezeichenlinks zu einem PDF-Dokument haben Sie eine genaue Kontrolle über die genaue Ansicht, die Sie wünschen. Sie sind nicht auf das Verlinken auf eine Seite beschränkt. Sie können die genaue Ansicht einrichten, indem Sie die Zielseite positionieren und dann das Lesezeichen erstellen.

{{% /alert %}}
