Dieser Artikel enthält Informationen zum Einfügen von PDF-Lesezeichen beim Konvertieren einer Tabelle in PDF.

Aspose.Cells ermöglicht das Hinzufügen von Lesezeichen zur Laufzeit. PDF Lesezeichen können die Navigierbarkeit langer Dokumente drastisch verbessern. Wenn Sie Lesezeichen-Links zu einem PDF-Dokument hinzufügen, haben Sie eine genaue Kontrolle über die genaue Ansicht, die Sie möchten, Sie sind nicht darauf beschränkt, auf eine Seite zu verlinken. Sie können die genaue Ansicht einstellen, indem Sie die Zielseite positionieren und dann das Lesezeichen erstellen.

{{% /alert %}}
