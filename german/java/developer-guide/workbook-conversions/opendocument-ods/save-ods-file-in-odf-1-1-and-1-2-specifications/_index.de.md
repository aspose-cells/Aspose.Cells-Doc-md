{{% alert color="primary" %}}

Aspose.Cells unterstützt das Speichern (**OpenDocument-Tabelle**) ODS-Datei in (**OpenDocument-Format** ) ODF 1.1- und ODF 1.2-Spezifikationen. Aspose.Cells hinzugefügt[**OdsSaveOptions.setStrictSchema11()**](https://reference.aspose.com/cells/java/com.aspose.cells/odssaveoptions#IsStrictSchema11) -Eigenschaft, um die ODF 1.1-Spezifikation zum Speichern von ODS-Dateien zu verwenden. Der Standardwert dieser Eigenschaft ist**FALSCH**, so dass ODS-Dateien, die ohne diese speziellen Einstellungen gespeichert werden, die 1.2-Spezifikation verwenden.

{{% /alert %}}
