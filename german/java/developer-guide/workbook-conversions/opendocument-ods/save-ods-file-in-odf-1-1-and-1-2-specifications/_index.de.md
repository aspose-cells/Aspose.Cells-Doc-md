{{% alert color="primary" %}}

Aspose.Cells unterstützt das Speichern (**OpenDocument-Tabelle**) ODS Datei in (**OpenDocument-Format** ) ODF 1.1- und ODF 1.2-Spezifikationen. Aspose.Cells hinzugefügt[**OdsSaveOptions.setStrictSchema11()**](https://reference.aspose.com/cells/java/com.aspose.cells/odssaveoptions#IsStrictSchema11) -Eigenschaft zur Verwendung der ODF 1.1-Spezifikation zum Speichern von ODS-Dateien. Der Standardwert dieser Eigenschaft ist**FALSCH**also verwendet die ODS-Datei, die ohne diese speziellen Einstellungen gespeichert wird, die 1.2-Spezifikation.

{{% /alert %}}

Der folgende Beispielcode erstellt das Arbeitsmappenobjekt, fügt einen Wert in Zelle A1 des ersten Arbeitsblatts hinzu und speichert dann die Datei ODS in ODF 1.1- und 1.2-Spezifikationen. Standardmäßig wird die Datei ODS in der ODF 1.2-Spezifikation gespeichert.
