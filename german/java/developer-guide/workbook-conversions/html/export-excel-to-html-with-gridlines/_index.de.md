{{% alert color="primary" %}} 

Wenn Sie Ihre Excel-Datei mit Rasterlinien in HTML exportieren möchten, verwenden Sie bitte die Eigenschaft [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) und setzen Sie sie auf **true**.

{{% /alert %}} 
## **Excel in HTML mit Rasterlinien exportieren**
Die folgende Beispielscode erstellt eine Arbeitsmappe und füllt ihr Arbeitsblatt mit Werten und speichert sie dann im HTML-Format, nachdem die Eigenschaft [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) auf **true** gesetzt wurde.

Der folgende Screenshot zeigt das generierte HTML mit diesem Beispielcode. Wie Sie sehen können, zeigt es auch Rasterlinien im generierten HTML an.
