{{% alert color="primary" %}} 

 Wenn Sie Ihre Excel-Datei mit GridLines in HTML exportieren möchten, dann verwenden Sie bitte die[HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) Eigenschaft und legen Sie sie fest**wahr**.

{{% /alert %}}
