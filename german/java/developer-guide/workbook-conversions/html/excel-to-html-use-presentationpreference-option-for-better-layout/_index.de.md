{{% alert color="primary" %}}

Aspose.Cells bietet eine nützliche Eigenschaft, [**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference), die Entwickler verwenden können, um ein besseres Layout zu erhalten, wenn sie eine Microsoft Excel-Datei im HTML- oder MHT-Format speichern. Der Standardwert der Eigenschaft ist false. Wir empfehlen, diese Eigenschaft auf true zu setzen, um bessere Präsentationen aus einem Excel-Bericht zu erhalten.
