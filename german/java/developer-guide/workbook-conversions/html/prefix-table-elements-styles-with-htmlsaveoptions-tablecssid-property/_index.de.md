## **Mögliche Nutzungsszenarien**

Mit Aspose.Cells können Sie Tabellenelementen Stile voranstellen[**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId)Eigentum. Angenommen, Sie setzen diese Eigenschaft mit einem Wert wie*MyTest_TableCssId*, dann finden Sie Tabellenelemente wie unten gezeigt

{{< highlight "java" >}}

table#MyTest_TableCssId

# MyTest_TableCssId tr

# MyTest_TableCssId col

# MyTest_TableCssId br

etc.

{{< /highlight >}}
