## **Mögliche Verwendungsszenarien**

Wenn Sie Ihre Excel-Datei in HTML speichern, kann die Leistung des resultierenden HTML verbessert werden, indem mit benutzerdefinierten Eigenschaften die Bilddaten nur einmal gespeichert werden müssen, wenn es mehrere Vorkommnisse desselben base64-Bildes gibt. Bitte verwenden Sie die [**HtmlSaveOptions.EnableCssCustomProperties**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions/properties/#EnableCssCustomProperties) Eigenschaft und setzen Sie sie auf **true**, während Sie in HTML speichern.
