{{% alert color="primary" %}} 

Normalerweise wird das Layout von Div-Tags beim Laden von HTML in ein Excel-Arbeitsmappenobjekt ignoriert. Wenn Sie jedoch möchten, dass das Layout von Div-Tags nicht ignoriert wird, setzen Sie bitte die Eigenschaft [HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag) auf **true**. Der Standardwert dieser Eigenschaft ist **false**.

{{% /alert %}}
