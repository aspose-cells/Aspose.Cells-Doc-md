{{% alert color="primary" %}} 

 Normalerweise wird das Layout von div-Tags beim Laden von HTML in ein Excel-Arbeitsmappenobjekt ignoriert. Wenn Sie jedoch möchten, dass das Layout von div-Tags nicht ignoriert wird, dann stellen Sie dies bitte ein[HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag) Eigentum zu**wahr** . Der Standardwert dieser Eigenschaft ist**FALSCH**.

{{% /alert %}}
