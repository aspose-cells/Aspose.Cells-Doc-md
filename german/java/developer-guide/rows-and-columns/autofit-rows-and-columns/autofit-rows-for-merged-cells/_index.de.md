{{% alert color="primary" %}}

Microsoft Excel bietet eine Funktion, mit der Sie die Höhe einer Zelle automatisch an ihren Inhalt anpassen können. Die Funktion heißt Zeilen automatisch anpassen. Microsoft Excel legt den automatischen Anpassungsvorgang für verbundene Zellen nicht nativ fest. Manchmal ist die Funktion für einen Benutzer von entscheidender Bedeutung, der wirklich Zeilen zum automatischen Anpassen auch für verbundene Zellen implementieren muss.

{{% /alert %}}

 Aspose.Cells unterstützt diese Funktion durch die[**AutoFitter-Optionen**](https://reference.aspose.com/cells/java/com.aspose.cells/AutoFitterOptions)API. Mit diesem API ist es möglich, Zeilen in einem Arbeitsblatt automatisch anzupassen, einschließlich verbundener Zellen.

Nach dem Ausführen des Codes passt Aspose.Cells Zeilen für verbundene Zellen automatisch an.
