## **Mögliche Verwendungsszenarien**
Sie können das VBA-Projekt eines Arbeitsbuchs mit Aspose.Cells durch die Methode [VbaProject.protect()](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#protect-boolean-java.lang.String-) passwortgeschützt schützen.
## **Beispielcode**
Der folgende Beispielcode lädt die [Beispiel-Excel-Datei](43352069.xlsm), greift auf ihr VBA-Projekt zu und schützt es mit einem Passwort. Schließlich wird es als [Ausgabedatei](43352070.xlsm) gespeichert.
