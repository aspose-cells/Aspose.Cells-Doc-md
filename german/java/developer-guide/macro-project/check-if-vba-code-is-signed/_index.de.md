{{% alert color="primary" %}}

 Aspose.Cells ermöglicht dem Benutzer zu überprüfen, ob das VBA-Codeprojekt signiert ist oder nicht. Bitte verwenden Sie die[**Workbook.getVbaProject().isSigned()**](https://reference.aspose.com/cells/java/com.aspose.cells/vbaproject#IsSigned) Methode, um zu überprüfen, ob das VBA-Codeprojekt signiert ist oder nicht.

{{% /alert %}}

## **Überprüfen Sie, ob der VBA-Code signiert ist**

Sie können jede Ihrer Excel-Dateien verwenden, um diesen Code zu testen. Zu Testzwecken können Sie verwenden[diese Excel-Datei, die im Code verwendet wird](5472500.xlsm).
