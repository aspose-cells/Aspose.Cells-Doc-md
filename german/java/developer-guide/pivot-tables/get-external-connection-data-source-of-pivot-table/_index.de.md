## **Externe Verbindungsdatenquelle der Pivot-Tabelle abrufen**

Aspose.Cells bietet die Möglichkeit, die externe Verbindungsdatenquelle der Pivot-Tabelle abzurufen. Dafür sorgt die API[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)Eigentum der[**PivotTable**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable)Klasse. Das[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)Eigenschaft gibt die zurück[**Externe Verbindung**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection)Objekt. Das folgende Code-Snippet demonstriert die Verwendung von[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) -Eigenschaft, um die externe Verbindungsdatenquelle der Pivot-Tabelle abzurufen.
