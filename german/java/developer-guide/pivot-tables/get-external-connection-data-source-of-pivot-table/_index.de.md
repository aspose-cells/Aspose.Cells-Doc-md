## **Abrufen der externen Verbindungsdatenquelle der Pivot-Tabelle**

Aspose.Cells bietet die Möglichkeit, die externe Verbindung Datenquelle des Pivot-Tables abzurufen. Hierfür bietet die API die Eigenschaft [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) der Klasse [**PivotTable**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable). Die Eigenschaft [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) gibt das Objekt [**ExternalConnection**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection) zurück. Der folgende Codeausschnitt demonstriert die Verwendung der Eigenschaft [**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource), um die externe Verbindung Datenquelle des Pivot-Tables abzurufen.
