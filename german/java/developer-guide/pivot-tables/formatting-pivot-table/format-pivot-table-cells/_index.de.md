{{% alert color="primary" %}}

 Manchmal möchten Sie Pivot-Tabellenzellen formatieren. Beispielsweise möchten Sie Pivot-Tabellenzellen eine Hintergrundfarbe zuweisen. Aspose.Cells bietet zwei Methoden[**PivotTable.formatAll()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#formatAll(com.aspose.cells.Style) ) und[**PivotTable.format()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#format(int,%20int,%20com.aspose.cells.Style)), die Sie zu diesem Zweck verwenden können.

[**PivotTable.formatAll()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#formatAll(com.aspose.cells.Style) ) wendet den Stil während auf die gesamte Pivot-Tabelle an[**PivotTable.format()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#format(int,%20int,%20com.aspose.cells.Style)) wendet den Stil auf eine einzelne Zelle der Pivot-Tabelle an.

{{% /alert %}}
