 Aspose.Cells bietet die Möglichkeit, Pivot-Tabellen in ODS-Dateien zu speichern. Dazu konvertieren Sie einfach eine Arbeitsmappe mit vorhandener Povot-Tabelle oder erstellen eine neue Pivot-Tabelle und speichern die Datei im ODS-Format. Unbedingt anrufen[**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData()vor dem Speichern, um sicherzustellen, dass die Pivot-Tabelle in der ODS-Ausgabedatei gerendert wird.
