Aspose.Cells bietet die Möglichkeit, Pivot-Tabellen in ODS-Dateien zu speichern. Hierfür konvertieren Sie einfach eine Arbeitsmappe mit einer vorhandenen Pivot-Tabelle oder erstellen eine neue Pivot-Tabelle und speichern die Datei im ODS-Format. Stellen Sie sicher, dass Sie [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData--) vor dem Speichern aufrufen, um sicherzustellen, dass die Pivot-Tabelle in der Ausgabedatei im ODS-Format gerendert wird. Der folgende Codeausschnitt zeigt das Speichern der Pivot-Tabelle in einer ODS-Datei.
