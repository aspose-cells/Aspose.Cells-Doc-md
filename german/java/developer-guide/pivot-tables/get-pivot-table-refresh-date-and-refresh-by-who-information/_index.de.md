{{% alert color="primary" %}}

Aspose.Cells unterstützt jetzt das Abrufen des Aktualisierungsdatums und der Aktualisierung von Wer-Informationen aus einer Arbeitsmappe.

{{% /alert %}}

[**PivotTable.getRefreshDate**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshDate) gibt das Datum zurück, an dem der PivotTable-Bericht zuletzt aktualisiert wurde. Ähnlich[**PivotTable.getRefreshByWho**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshedByWho) Die Eigenschaft gibt den Namen des Benutzers zurück, der den Bericht zuletzt aktualisiert hat.
