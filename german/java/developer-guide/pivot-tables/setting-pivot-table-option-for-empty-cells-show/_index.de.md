## **Einführung**

Sie können mithilfe von Aspose.Cells verschiedene Pivot-Tabellen-Optionen festlegen. Eine solche Option ist "Leerzellen anzeigen". Das Festlegen dieser Option bedeutet, dass alle leeren Zellen in der Pivot-Tabelle mit einem bestimmten Text angezeigt werden.

Um diese Option in Microsoft Excel zu finden und einzustellen:

1. Wählen Sie eine Pivot-Tabelle aus und klicken Sie mit der rechten Maustaste darauf.
1. Wählen Sie **PivotTable-Optionen** aus.
1. Wählen Sie das Registerkarte **Layout & Format** aus.
1. Wählen Sie die Option **Leere Zellen anzeigen** aus und geben Sie einen Text an.

## **Pivot-Tabellenoption einstellen - Leere Zellen anzeigen**

Aspose.Cells bietet die Eigenschaften [**PivotTable.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString) und [**PivotTable.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString), um die Pivot-Tabellen-Option "Leerzellen anzeigen" festzulegen.
