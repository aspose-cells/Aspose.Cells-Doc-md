{{% alert color="primary" %}}

Mit Aspose.Cells können Sie verschiedene Optionen für Pivot-Tabellen festlegen. Eine dieser Optionen ist "Für leere Zellen anzeigen". Das Setzen dieser Option bedeutet, dass alle leeren Zellen in der Pivot-Tabelle mit einer bestimmten Zeichenfolge angezeigt werden.

So finden und setzen Sie diese Option in Microsoft Excel:

1. Wählen Sie eine Pivot-Tabelle aus und klicken Sie mit der rechten Maustaste darauf.
1.  Wählen**PivotTable-Optionen**.
1.  Wähle aus**Layout & Format** Tab.
1.  Wähle aus**Für leere Zellen anzeigen** Option und geben Sie eine Zeichenfolge an.

![todo: Bild_alt_Text](setting-pivot-table-option-for-empty-cells-show_1.png)

{{% /alert %}}

Aspose.Cells bietet die[**PivotTable.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString) und[**PivotTable.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString)Eigenschaften zum Festlegen der Pivot-Tabellenoption "Für leere Zellen anzeigen".
