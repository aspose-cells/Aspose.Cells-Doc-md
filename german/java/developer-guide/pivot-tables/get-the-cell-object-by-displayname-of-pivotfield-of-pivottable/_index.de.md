{{% alert color="primary" %}} 

 Aspose.Cells bietet[PivotTable.getCellByDisplayName()](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#getCellByDisplayName\(java.lang.String\))-Methode, mit der Sie über den Anzeigenamen des Pivot-Felds auf das Zellenobjekt zugreifen können. Diese Methode ist nützlich, wenn Sie die Überschrift Ihres Pivot-Felds hervorheben oder formatieren möchten. In diesem Artikel wird erläutert, wie Sie das Zellobjekt anhand des Anzeigenamens des Datenfelds abrufen und dann eine Formatierung darauf anwenden.

{{% /alert %}} 
## **Rufen Sie das Objekt Cell nach DisplayName von PivotField von PivotTable ab**
Der folgende Code greift auf die erste Pivot-Tabelle des Arbeitsblatts zu und ruft dann die Zelle anhand des Anzeigenamens des zweiten Datenfelds der Pivot-Tabelle ab.
