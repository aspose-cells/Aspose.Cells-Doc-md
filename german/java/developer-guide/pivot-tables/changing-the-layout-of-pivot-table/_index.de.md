{{% alert color="primary" %}}

Microsoft Excel ermöglicht es Ihnen, das Layout des PivotTable mit den Befehlen im Menü *PivotTable Tools > Design > Report* Layout zu ändern. Sie können das Layout in diesen drei Formen ändern

- Im kompakten Formular anzeigen
- Im Gliederungsformular anzeigen
- Im tabellarischen Formular anzeigen

Aspose.Cells bietet auch Methoden [**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm--), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm--) und [**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm--) zur Änderung des Layouts der Pivot-Tabelle in diesen drei Formen.

{{% /alert %}}
