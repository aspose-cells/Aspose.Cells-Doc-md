{{% alert color="primary" %}}

 Excel-Anwendung bietet die**Passen Sie die Größe der Form an den Text an** Option für die DataLabels von Chart, um die Form zu vergrößern, damit der Text hineinpasst. Auf diese Option kann über die Excel-Oberfläche zugegriffen werden, indem Sie eine der Datenbeschriftungen im Diagramm auswählen. Rechtsklick und auswählen**DataLabels formatieren** Speisekarte. An**Größe & Eigenschaften** Registerkarte, erweitern**Ausrichtung** um die zugehörigen Eigenschaften einschließlich der zu offenbaren**Ändern Sie die Größe der Form, um den Text zu korrigieren** Möglichkeit.
