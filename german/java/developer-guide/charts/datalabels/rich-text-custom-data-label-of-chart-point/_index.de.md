{{% alert color="primary" %}}

 Sie können Aspose.Cells verwenden, um eine benutzerdefinierte Rich-Text-Datenbeschriftung des Diagrammpunkts zu erstellen. Aspose.Cells bietet die[**DataLabels.Zeichen ()**](https://reference.aspose.com/cells/java/com.aspose.cells/datalabels#characters(int,%20int) )-Methode, um die zurückzugeben[**Schrifteinstellung**](https://reference.aspose.com/cells/java/com.aspose.cells/FontSetting)Objekt, mit dem die Schrifteigenschaften des Textes wie Farbe, Fettdruck usw. eingestellt werden können.

{{% /alert %}}
