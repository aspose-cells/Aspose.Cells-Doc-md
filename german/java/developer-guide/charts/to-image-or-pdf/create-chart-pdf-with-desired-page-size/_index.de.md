## **Mögliche Verwendungsszenarien**
Sie können mithilfe von Aspose.Cells ein Diagramm-PDF mit der gewünschten Seitengröße erstellen und angeben, wie Sie das Diagramm innerhalb der Seite ausrichten möchten. Sie können das Diagramm oben, unten, zentriert, links, rechts usw. ausrichten. Darüber hinaus kann das Ausgabediagramm in einem Stream oder auf der Festplatte erstellt werden.
## **Erstellen Sie ein Diagramm-PDF mit gewünschter Seitengröße**
Bitte sehen Sie sich den folgenden Beispielcode an, der die [Beispieldatei Excel](64716912.xlsx) lädt, auf das erste Diagramm im Arbeitsblatt zugreift und es dann in [PDF-Ausgabe](64716911.pdf) mit der gewünschten Seitengröße konvertiert. Auf dem folgenden Screenshot ist zu sehen, dass die Seitengröße im Ausgabe-PDF gemäß dem im Code angegebenen Wert 7x7 ist und das Diagramm sowohl horizontal als auch vertikal zentriert ist.
