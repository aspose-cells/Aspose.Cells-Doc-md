## **Mögliche Nutzungsszenarien**
Sie können mit Aspose.Cells ein Diagramm PDF mit Ihrer gewünschten Seitengröße erstellen und angeben, wie Sie das Diagramm innerhalb der Seite ausrichten möchten. Sie können das Diagramm oben, unten, in der Mitte, links, rechts usw. ausrichten. Außerdem kann das Ausgabediagramm in einem Stream oder auf einer Festplatte erstellt werden.
## **Diagramm PDF mit gewünschter Seitengröße erstellen**
Bitte sehen Sie sich den folgenden Beispielcode an, der die lädt[Beispiel-Excel-Datei](64716912.xlsx), greift auf das erste Diagramm im Arbeitsblatt zu und konvertiert es dann in[Ausgang PDF](64716911.pdf)mit gewünschter Seitengröße. Der folgende Screenshot zeigt, dass die Seitengröße in der Ausgabe PDF 7 x 7 ist, wie im Code angegeben, und das Diagramm sowohl horizontal als auch vertikal zentriert ist.
