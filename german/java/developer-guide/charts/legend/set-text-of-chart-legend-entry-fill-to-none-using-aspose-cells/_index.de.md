{{% alert color="primary" %}} 

Wenn Sie den Text des Legenden-Eintrags im Diagramm auf keine Füllung setzen möchten, damit er nicht in der Diagrammlegende angezeigt wird, setzen Sie bitte [LegendEntry.IsTextNoFill](https://reference.aspose.com/cells/java/com.aspose.cells/legendentry#IsTextNoFill) auf **true**.

{{% /alert %}}
