{{% alert color="primary" %}}

Ein Wasserfalldiagramm ist ein spezieller Diagrammtyp, der normalerweise verwendet wird, um zu zeigen, wie die Ausgangsposition entweder zunimmt oder abnimmt. Microsoft Excel verfügt über viele vordefinierte Diagrammtypen, darunter Säulen-, Linien-, Torten-, Balken-, Radar-Diagramme usw., aber das Wasserfalldiagramm geht über die grundlegenden Grafiken hinaus und kann unter Verwendung der vorhandenen Diagrammtypen mit wenig oder mehr Anpassung erstellt werden.

{{% /alert %}}

## Erstellung eines Wasserfalldiagramms

Aspose.Cells APIs ermöglichen die Erstellung eines Wasserfalldiagramms mithilfe eines Liniendiagramms. Die API ermöglicht es auch, das Diagrammaussehen anzupassen, um ihm die Form des Wasserfalls zu geben, indem die wichtigsten Eigenschaften zur Formung des Diagramms eingestellt werden, eine; [**Series.UpBars**](https://reference.aspose.com/cells/java/com.aspose.cells/series#UpBars) & [**Series.DownBars**](https://reference.aspose.com/cells/java/com.aspose.cells/series#DownBars) Eigenschaften.
