{{% alert color="primary" %}}

 In diesem Artikel wird erläutert, wie Sie ein Kreisdiagramm mit Führungslinien von Grund auf neu erstellen, während Sie Aspose.Cells for Java API verwenden. In Excel ist die Option „Führungslinien anzeigen“ standardmäßig aktiviert, sodass beim Erstellen eines Kreisdiagramms in Excel die Führungslinien angezeigt werden. Beim Erstellen eines ähnlichen Diagramms mit Aspose.Cells-APIs müssen Sie jedoch explizit festlegen[**Serie.HasLeaderLines**](https://reference.aspose.com/cells/java/com.aspose.cells/series#HasLeaderLines) Eigentum.

{{% /alert %}}

## **Kreisdiagramm mit Führungslinien erstellen**

 Um die Verwendung von Aspose.Cells for Java API zum Erstellen eines Tortendiagramms mit Führungslinien zu demonstrieren, erstellen wir zunächst ein neues[**Arbeitsmappe**](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook) und geben Sie einige Daten ein, die als Seriendatenquelle dienen. Sobald die Daten vorhanden sind, werden wir a hinzufügen[**Diagramm**](https://reference.aspose.com/cells/java/com.aspose.cells/Chart) des Typs[**Kuchen**](https://reference.aspose.com/cells/java/com.aspose.cells/charttype#PIE)zur Sammlung von Diagrammen und stellen Sie die verschiedenen Aspekte ein, um die gewünschte Diagrammansicht zu erhalten.
