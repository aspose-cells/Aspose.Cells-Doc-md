{{% alert color="primary" %}}

In Excel-Diagrammen gibt es 3 Arten von Achsen:
1. Horizontale(Kategorie) Achse: X-Achse
1. Vertikale(Wert) Achse: Y-Achse
1. Tiefen(Serien) Achse: Z-Achse



{{% /alert %}}
