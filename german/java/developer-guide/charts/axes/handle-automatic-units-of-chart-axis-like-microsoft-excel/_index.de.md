## **Mögliche Nutzungsszenarien**
Frühe Versionen von Aspose.Cells konnten automatische Einheiten der Diagrammachse nicht richtig handhaben, wenn das Diagramm als Bild oder PDF gerendert wurde. Aspose.Cells unterstützt jetzt die Handhabung automatischer Einheiten der Diagrammachse. Es gibt keine Codeänderung. Konvertieren Sie einfach Ihr Diagramm in ein Bild oder PDF und es rendert die Diagrammachse genau so, wie Microsoft Excel es rendert.
