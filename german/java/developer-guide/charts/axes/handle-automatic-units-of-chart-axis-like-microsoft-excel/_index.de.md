## **Mögliche Nutzungsszenarien**
Frühere Versionen von Aspose.Cells konnten automatische Einheiten der Diagrammachse nicht richtig handhaben, wenn das Diagramm als Bild oder PDF gerendert wurde. Jetzt unterstützt Aspose.Cells die Handhabung automatischer Einheiten der Diagrammachse. Es gibt keine Codeänderung. Konvertieren Sie einfach Ihr Diagramm in ein Bild oder PDF und es wird die Diagrammachse genau so rendern, wie Microsoft Excel es rendert.
