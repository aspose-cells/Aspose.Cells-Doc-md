##  **Mögliche Nutzungsszenarien**
Nachdem Sie ein Diagramm in einem Arbeitsblatt erstellt haben, können Sie die Kategorieachse dafür festlegen. In diesem Artikel zeigen wir Ihnen, wie Sie mithilfe von Aspose.Cells und Beispielcode die Kategorieachse für ein Excel-Diagramm festlegen.

##  **Die Schritte im Beispielcode**

1. Erstellen Sie eine neue Arbeitsmappe.

2. Erstellen Sie im ersten Arbeitsblatt ein neues Diagramm.

3. Fügen Sie den Zellen im ersten Arbeitsblatt einige Werte hinzu.

4. Jetzt können Sie die Kategorieachse festlegen. Es gibt zwei Möglichkeiten: die Verwendung von Zelldaten oder die direkte Verwendung von Zeichenfolgen, die beide im Beispielcode gezeigt werden.

5. Stellen Sie die Werteachse ein und speichern Sie die Arbeitsmappe, um das Ergebnis anzuzeigen.
