## **Holen Sie sich Hyperlinks in Reichweite**

Das[**Bereich**](https://reference.aspose.com/cells/java/com.aspose.cells/range) Klasse bietet a[**getHyperlinks()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#Hyperlinks) Eigenschafts-Getter, der alle Hyperlinks im ausgewählten Bereich abruft. Sie können den Hyperlink auch löschen, indem Sie die aufrufen[**Hyperlink.löschen()**](https://reference.aspose.com/cells/java/com.aspose.cells/hyperlink#delete()) Methode.
