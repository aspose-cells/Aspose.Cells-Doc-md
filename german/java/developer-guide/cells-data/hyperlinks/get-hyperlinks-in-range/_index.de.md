## **Hyperlinks im Bereich abrufen**

Die [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/range) -Klasse stellt einen [**getHyperlinks()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#Hyperlinks)-Eigenschafts-Getter bereit, der alle Hyperlinks im ausgewählten Bereich abruft. Sie können auch den Hyperlink löschen, indem Sie die [**Hyperlink.delete()**](https://reference.aspose.com/cells/java/com.aspose.cells/hyperlink#delete--)-Methode aufrufen.
