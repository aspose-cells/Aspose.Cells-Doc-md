{{% alert color="primary" %}} 

Hyperlinks sind nützlich, um auf die Informationen auf anderen Arbeitsblättern oder auf Websites zuzugreifen. Microsoft Mit Excel können Benutzer Hyperlinks zum Text in Zellen und Bildern hinzufügen. Bild-Hyperlinks können das Navigieren in einem Arbeitsblatt erleichtern, z. B. als Weiter- und Zurück-Schaltflächen oder Logos, die auf bestimmte Websites verweisen. In diesem Dokument wird erläutert, wie Sie mit Aspose.Cells Bild-Hyperlinks in ein Arbeitsblatt einfügen.

{{% /alert %}} 

Aspose.Cells ermöglicht es Ihnen, zur Laufzeit Hyperlinks zu Bildern in Tabellenkalkulationen hinzuzufügen. Es ist möglich, den Bildschirmtipp und die Adresse des Links festzulegen und zu ändern.
