Mit der zunehmenden Verbreitung von hochauflösenden Bildschirmen erscheinen Bilder, die mit der Standard-DPI von 96 generiert wurden, oft unscharf und undeutlich. Um auf hochauflösenden Bildschirmen Klarheit zu gewährleisten, ist es wichtig, Bilder mit einer höheren DPI zu generieren. Aspose.Cells bietet die Funktionen zur Einstellung von [**ImageOrPrintOptions.HorizontalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setHorizontalResolution-int-) und [**ImageOrPrintOptions.VerticalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setVerticalResolution-int-), damit Sie hochwertige Bilder aus Excel-Dateien erstellen können, die auf hochauflösenden Displays scharf aussehen.
