{{% alert color="primary" %}}

Sie können mit Aspose.Cells Ihr Arbeitsblatt oder Diagramm in ein Bild mit der gewünschten Breite und Höhe exportieren. Es bietet [**ImageOrPrintOptions.setDesiredSize()**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#setDesiredSize-int-int-) Methode, um die gewünschte Breite und Höhe des exportierten Bildes festzulegen. Die Breite und Höhe werden in Pixeln angegeben.
