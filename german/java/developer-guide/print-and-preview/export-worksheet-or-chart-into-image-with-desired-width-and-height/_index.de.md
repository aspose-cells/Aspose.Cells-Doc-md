 Sie können Aspose.Cells verwenden, um Ihr Arbeitsblatt oder Diagramm in ein Bild mit der gewünschten Breite und Höhe zu exportieren. Es bietet[**ImageOrPrintOptions.setDesiredSize()**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#setDesiredSize(int,%20int))-Methode, um die gewünschte Breite und Höhe des exportierten Bildes festzulegen. Breite und Höhe werden in der Einheit Pixel angegeben.
