{{% alert color="primary" %}} 

Manchmal möchten Sie Informationen über externe Datenquellen abrufen. Sie möchten beispielsweise die SQL-Verbindungsdaten sehen. Diese Informationen können alle Arten von Daten enthalten, die erforderlich sind, um eine Verbindung mit einem SQL-Server herzustellen, z. B. die Server-URL, den Benutzernamen, den zugrunde liegenden Tabellennamen, die SQL-Abfrage, den Abfragetyp, den Speicherort der Tabelle usw.

Aspose.Cells bietet einige nützliche Aufrufe zum Abrufen solcher Details von Unternehmensdatenbanken, die mit Vorlagendateien in Excel verknüpft sind.

{{% /alert %}}
