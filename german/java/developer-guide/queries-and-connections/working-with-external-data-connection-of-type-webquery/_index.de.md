 Sie können auf die externe Datenverbindung jeder Art zugreifen, indem Sie die verwenden[Arbeitsmappe.Datenverbindungen](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections) Sammlung. Ein Typ einer solchen Datenverbindung ist WebQuery. Dieser Artikel zeigt Ihnen, wie Sie mit der WebQuery-Datenverbindung arbeiten. Sie können eine WebQuery-Datenverbindung in Microsoft Excel mithilfe von erstellen**Daten** > **Aus dem Web** Speisekarte.
