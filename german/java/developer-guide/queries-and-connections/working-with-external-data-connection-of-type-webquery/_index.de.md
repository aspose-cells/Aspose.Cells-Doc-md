{{% alert color="primary" %}} 

Sie können die externe Datenverbindung jeden Typs über die [Workbook.DataConnections](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#DataConnections)-Sammlung abrufen. Ein solcher Datentyp ist WebQuery. Dieser Artikel zeigt, wie Sie mit der WebQuery-Datenverbindung arbeiten. Sie können eine WebQuery-Datenverbindung in Microsoft Excel über das **Daten** > **Von Web**-Menü erstellen.

{{% /alert %}}
