{{% alert color="primary" %}} 

 Aspose.Cells ermöglicht es Ihnen, eine schlanke Version der Arbeitsmappe in den Arbeitsspeicher zu laden, um ihre Metadateninformationen zu bearbeiten. Bitte verwenden Sie die[com.aspose.cells.WorkbookMetadata](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookMetadata) Klasse, um die Arbeitsmappe zu laden.

{{% /alert %}}

Sobald Sie die Arbeitsmappe mit öffnen[Arbeitsmappe](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook) Klasse können Sie die Dokumenteigenschaften lesen.
