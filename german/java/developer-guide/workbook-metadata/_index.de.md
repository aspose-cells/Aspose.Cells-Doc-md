{{% alert color="primary" %}} 

Mit Aspose.Cells können Sie eine leichtgewichtige Version der Arbeitsmappe in den Speicher laden, um ihre Metadateninformationen zu bearbeiten. Verwenden Sie die Klasse [com.aspose.cells.WorkbookMetadata](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookMetadata), um die Arbeitsmappe zu laden.

{{% /alert %}}
