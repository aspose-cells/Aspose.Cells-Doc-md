{{% alert color="primary" %}}

Aspose.Cells ermöglicht es Ihnen, die XML-Map innerhalb der Arbeitsmappe mithilfe der [**Workbook.importXml()**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#importXml-java.lang.String-java.lang.String-int-int-)-Methode zu importieren. Sie können die XML-Map mithilfe von Microsoft Excel mit den folgenden Schritten importieren

- Wählen Sie den **Entwickler**-Tab
- Klicken Sie auf **Importieren** im XML-Bereich und befolgen Sie die erforderlichen Schritte.

Sie müssen Ihre XML-Daten bereitstellen, um den Import abzuschließen. Hier ist eine [Beispiel-XML-Daten](5472501.txt), die Sie für Tests verwenden können.
