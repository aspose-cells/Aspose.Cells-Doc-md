{{% alert color="primary" %}} 

**2-Farben-Skala** und**3-Farben-Skala** Bedingte Formatierungen werden auf die gleiche Weise hinzugefügt, außer dass sie sich durch unterscheiden[FormatCondition.ColorScale.Is3ColorScale](https://reference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale) Eigentum. Diese Eigenschaft ist**FALSCH** für 2-Farbskala u**wahr** für bedingte Formatierungen der 3-Farbskala.
