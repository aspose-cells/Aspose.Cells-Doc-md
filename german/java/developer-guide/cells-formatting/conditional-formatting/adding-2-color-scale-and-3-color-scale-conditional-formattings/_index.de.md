{{% alert color="primary" %}} 

Bedingte Formatierungen für **2-Farbskala** und **3-Farbskala** werden auf die gleiche Weise hinzugefügt, außer dass sie sich durch die Eigenschaft [FormatCondition.ColorScale.Is3ColorScale](https://reference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale) unterscheiden. Diese Eigenschaft ist **false** für 2-Farbskala und **true** für 3-Farbskala bedingte Formatierungen.

{{% /alert %}}
