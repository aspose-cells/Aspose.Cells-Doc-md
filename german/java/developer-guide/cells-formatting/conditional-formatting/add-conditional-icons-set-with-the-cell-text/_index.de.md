{{% alert color="primary" %}} 

Manchmal möchten Sie bedingte Symbole neben dem Text in einer Zelle hinzufügen, um die Daten für die Leser aussagekräftiger zu machen. Sie möchten einige der Symboltypen für die bedingte Formatierung verwenden, ohne jedoch die bedingte Formatierung auf Zellen anzuwenden. Aspose.Cells unterstützt die Funktion.

{{% /alert %}} 

Im Folgenden finden Sie ein Beispiel, das eine XLSX-Datei von Grund auf neu erstellt und bedingte Symbole zu Zellen mit Text hinzufügt, ohne bedingte Formatierung anzuwenden.

Wenn der Code ausgeführt wird, werden Bilder aus dem bedingten Symbolsatz im Zellbereich „B2:C4“ hinzugefügt, wie unten gezeigt.
