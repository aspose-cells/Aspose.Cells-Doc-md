{{% alert color="primary" %}} 

 Manchmal muss der Benutzer Zeilenhöhen des Quellbereichs in den Zielbereich kopieren. Aspose.Cells bietet[PasteType.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS) enum für diesen Zweck. Wann werden Sie einstellen[PasteOptions.setPasteType()](https://reference.aspose.com/cells/java/com.aspose.cells/pasteoptions#PasteType) Eigentum mit[PasteType.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS) enum, dann werden die Höhen aller Zeilen innerhalb des Quellbereichs in den Zielbereich kopiert.

{{% /alert %}}
