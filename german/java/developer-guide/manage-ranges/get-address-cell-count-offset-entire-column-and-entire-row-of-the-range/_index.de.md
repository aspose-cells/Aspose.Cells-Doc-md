## **Mögliche Nutzungsszenarien**

 Aspose.Cells bietet die[**Bereich**](https://reference.aspose.com/cells/java/com.aspose.cells/Range)Objekt, das über verschiedene Dienstprogrammmethoden verfügt, die dem Benutzer das einfache Arbeiten mit Excel-Bereichen erleichtern. Dieser Artikel veranschaulicht die Verwendung der folgenden Methoden oder Eigenschaften der[**Bereich**](https://reference.aspose.com/cells/java/com.aspose.cells/Range)Objekt.

- **Adresse**

Ruft die Adresse des Bereichs ab.

- **Cell Graf**

Ruft alle Zellenzahlen im Bereich ab.

- **Versatz**

Ruft den Bereich nach Offset ab.

- **Ganze Spalte**

Ruft ein Range-Objekt ab, das die gesamte Spalte (oder Spalten) darstellt, die den angegebenen Bereich enthält.

- **Ganze Reihe**

Ruft ein Range-Objekt ab, das die gesamte Zeile (oder Zeilen) darstellt, die den angegebenen Bereich enthält.
