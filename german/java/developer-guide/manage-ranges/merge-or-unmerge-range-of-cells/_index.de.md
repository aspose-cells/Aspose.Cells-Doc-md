{{% alert color="primary" %}}

Sie können Aspose.Cells verwenden, um einen Bereich von Zellen zusammenzuführen oder aufzuteilen. Aspose.Cells bietet die Methoden [**Range.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge--) und [**Range.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge--) für diesen Zweck. Dieser Artikel erklärt, wie Sie einen Bereich von Zellen in eine einzige Zelle zusammenführen können.

{{% /alert %}}

Der folgende Beispielscode erstellt zuerst einen Bereich - A1:D4 - und fusioniert dann die Zellen im Bereich in eine einzelne Zelle mit der Methode [**Range.merge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#merge--).
Ebenso ist es möglich, Zellen aufzuteilen, indem ein Bereich erstellt und die Methode [**Range.unMerge()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#unMerge--) aufgerufen wird.

Das folgende Bild zeigt die generierte Excel-Datei mit dem Beispielcode. Wie Sie sehen können, wurde der Bereich A1:D4 in eine einzige Zelle zusammengeführt.
