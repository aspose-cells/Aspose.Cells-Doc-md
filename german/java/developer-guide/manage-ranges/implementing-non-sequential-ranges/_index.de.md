Normalerweise sind [benannte Bereiche](/cells/de/java/named-ranges/) rechteckig, wobei die Zellen kontinuierlich und benachbart zueinander sind. Manchmal kann es jedoch erforderlich sein, einen nicht-sequentiellen Zellenbereich zu verwenden, bei dem Zellen nicht benachbart sind. Aspose.Cells unterstützt die Erstellung eines benannten Bereichs mit nicht benachbarten Zellen. Hierfür bietet die API die Eigenschaft [**Name.RefersTo**](https://reference.aspose.com/cells/java/com.aspose.cells/name#RefersTo) an.
