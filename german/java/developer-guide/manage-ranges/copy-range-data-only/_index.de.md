{{% alert color="primary" %}} 

Manchmal müssen Sie Daten von einem Zellbereich in einen anderen kopieren, wobei Sie nur die Daten kopieren, nicht die Formatierung. Aspose.Cells bietet diese Funktion durch Bereitstellung der[Range.copyData](https://reference.aspose.com/cells/java/com.aspose.cells/range#copyData\(com.aspose.cells.Range\)) Methode.

{{% /alert %}} 
## **Nur Bereichsdaten kopieren**
Dieses Beispiel zeigt, wie Sie:

1. Erstellen Sie eine Arbeitsmappe.
1. Fügen Sie Daten zu Zellen im ersten Arbeitsblatt hinzu.
1. Erstellen Sie einen Bereich.
1. Erstellen Sie ein Stilobjekt mit angegebenen Formatierungsattributen.
1. Wenden Sie die Stilformatierung auf den Bereich an.
1. Erstellen Sie einen weiteren Zellbereich.
1.  Kopieren Sie Daten des ersten Bereichs in diesen zweiten Bereich, indem Sie verwenden[Range.copyData](https://reference.aspose.com/cells/java/com.aspose.cells/range#copyData\(com.aspose.cells.Range\)) Methode.
