## **Union Range erstellen**
Aspose.Cells bietet die Möglichkeit, einen Union-Bereich mittels der [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/java/com.aspose.cells/worksheetcollection#createUnionRange-java.lang.String-int-) Methode zu erstellen. Die [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/java/com.aspose.cells/worksheetcollection#createUnionRange-java.lang.String-int-) Methode akzeptiert zwei Parameter, die Adresse zur Erstellung des Vereinigungsbereichs und den Index des Arbeitsblatts. Die Methode gibt ein [UnionRange](https://reference.aspose.com/cells/java/com.aspose.cells/UnionRange) Objekt zurück.

Der folgende Codeausschnitt zeigt die Erstellung eines Union-Bereichs mit der [WorksheetCollection.CreateUnionRange](https://reference.aspose.com/cells/java/com.aspose.cells/worksheetcollection#createUnionRange-java.lang.String-int-) Methode. Die vom Code generierte Ausgabedatei ist als Referenz angehängt.
