Aspose.Cells bietet die Möglichkeit, Text in einer Arbeitsmappe mittels regulärer Ausdrücke zu ersetzen. Hierfür bietet die API die Eigenschaft [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) der Klasse [**ReplaceOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/ReplaceOptions). Wenn die [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) auf **wahr** gesetzt wird, wird angezeigt, dass der gesuchte Schlüssel ein regulärer Ausdruck ist.

Der folgende Code zeigt die Verwendung der Eigenschaft [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) anhand der Verwendung der [Beispieldatei Excel](SampleRegexReplace.xlsx). Die durch den folgenden Codeausschnitt generierte [Ausgabedatei](RegexReplace_out.xlsx) ist als Referenz angehängt.
