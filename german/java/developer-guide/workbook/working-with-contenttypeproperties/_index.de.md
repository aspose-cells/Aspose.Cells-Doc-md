 Aspose.Cells bietet[**Workbook.ContentTypeProperties.Add**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypepropertycollection#add(java.lang.Object))-Methode zum Hinzufügen benutzerdefinierter ContentTypeProperties zu einer Excel-Datei. Sie können die Eigenschaft auch optional machen, indem Sie die festlegen[**ContentTypeProperty.IsNillable**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypeproperty#IsNillable) Eigentum zu**wahr**. Der folgende Codeausschnitt veranschaulicht das Hinzufügen optionaler benutzerdefinierter ContentTypeProperties zu einer Excel-Datei. Das folgende Bild zeigt beide Eigenschaften, die durch den Beispielcode hinzugefügt wurden.
