Aspose.Cells bietet eine Methode zum Hinzufügen benutzerdefinierter ContentTypeProperties zu einer Excel-Datei. Sie können die Eigenschaft auch optional machen, indem Sie die Eigenschaft [**ContentTypeProperty.IsNillable**](https://reference.aspose.com/cells/java/com.aspose.cells/contenttypeproperty#IsNillable) auf **true** setzen. Das folgende Codeschnipsel zeigt das Hinzufügen optionaler benutzerdefinierter ContentTypeProperties zu einer Excel-Datei. Das folgende Bild zeigt beide Eigenschaften, die durch den Beispielcode hinzugefügt wurden.
