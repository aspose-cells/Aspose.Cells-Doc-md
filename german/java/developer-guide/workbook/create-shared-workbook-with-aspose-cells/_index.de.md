## **Mögliche Nutzungsszenarien**

Microsoft Mit Excel können Sie die Arbeitsmappe freigeben, wie im folgenden Screenshot gezeigt. Wenn Sie die Arbeitsmappe freigeben, kann mehr als ein Benutzer die Arbeitsmappe im Netzwerk bearbeiten. Aspose.Cells ermöglicht Ihnen das Erstellen einer freigegebenen Arbeitsmappe mit[**Arbeitsmappe.Einstellungen.Freigegeben**](https://reference.aspose.com/cells/java/com.aspose.cells/workbooksettings#Shared)Eigentum.
