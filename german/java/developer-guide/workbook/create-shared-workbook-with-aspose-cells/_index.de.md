## **Mögliche Verwendungsszenarien**

Microsoft Excel ermöglicht es Ihnen, die Arbeitsmappe freizugeben, wie im folgenden Screenshot gezeigt. Wenn Sie die Arbeitsmappe freigeben, können mehrere Benutzer die Arbeitsmappe im Netzwerk bearbeiten. Mit Aspose.Cells können Sie eine freigegebene Arbeitsmappe mit der Eigenschaft [**Workbook.Settings.Shared**](https://reference.aspose.com/cells/java/com.aspose.cells/workbooksettings#Shared) erstellen.

![todo:image_alt_text](create-shared-workbook-with-aspose-cells_1.png)

## **Gemeinsame Arbeitsmappe mit Aspose.Cells erstellen**

Der folgende Beispielscode erstellt eine freigegebene Arbeitsmappe, indem die Eigenschaft [**Workbook.Settings.Shared**](https://reference.aspose.com/cells/java/com.aspose.cells/workbooksettings#Shared) als **wahr** festgelegt wird.
