## **Slicer zeichnen, während Excel in PDF gerendert wird**
Wenn Sie eine Excel-Datei haben, auf die Slicer angewendet wurde, und Sie Excel mit den Slicer-Einstellungen in PDF exportieren möchten, unterstützt Aspose.Cells dies jetzt standardmäßig. Sie exportieren einfach die Excel-Datei mit Slicer in PDF, das generierte PDF zeigt den angewendeten Slicer.

Der folgende Beispielcode lädt die[Beispiel-Excel-Datei](SampleSlicerChart.xlsx)die einen vorhandenen Slicer enthält. Es speichert dann die Arbeitsmappe als[PDF-Datei ausgeben](SampleSlicerChart.pdf). Der folgende Screenshot vergleicht die Excel-Quelldatei und die generierte PDF-Datei.
