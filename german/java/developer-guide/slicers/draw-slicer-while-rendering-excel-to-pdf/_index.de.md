## **Slicer zeichnen beim Rendern von Excel zu PDF**
Wenn Sie eine Excel-Datei haben, auf die ein Slicer angewendet wurde, und die Excel-Datei mit den Slicereinstellungen in PDF exportieren möchten, unterstützt Aspose.Cells dies jetzt standardmäßig. Sie exportieren einfach die Excel-Datei mit Slicer in PDF, das generierte PDF zeigt den angewendeten Slicer.

Der folgende Beispielcode lädt die [Beispiel-Excel-Datei](SampleSlicerChart.xlsx), die einen vorhandenen Slicer enthält. Dann speichert es die Arbeitsmappe als [Ausgabe-PDF-Datei](SampleSlicerChart.pdf). Der folgende Screenshot vergleicht die Quell-Excel-Datei und die generierte PDF-Datei.
