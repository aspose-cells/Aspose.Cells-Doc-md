## **Mögliche Verwendungsszenarien**
Aspose.Cells unterstützt das Rendern von Slicer-Formen. Wenn Sie Ihr Arbeitsblatt in ein Bild konvertieren oder Ihre Arbeitsmappe in den Formaten PDF oder HTML speichern, werden Sie sehen, dass die Slicer ordnungsgemäß gerendert werden.
## **Rendernde Slicer**
Der folgende Beispielcode lädt die [Beispiel-Excel-Datei](67338485.xlsx), die einen vorhandenen Slicer enthält. Es konvertiert das Arbeitsblatt in ein Bild, indem es den Druckbereich festlegt, der nur den Slicer abdeckt. Das erstellte Bild ist das [Ausgabebild](67338486.png), das den gerenderten Slicer zeigt. Wie Sie im Bild sehen können, wurde der Slicer ordnungsgemäß gerendert und sieht genauso aus wie in der Beispiel-Excel-Datei.
