## **Mögliche Nutzungsszenarien**
Wenn Sie den Slicer in Microsoft Excel entfernen möchten, wählen Sie ihn einfach aus und drücken Sie die*Löschen*Knopf. Wenn Sie es mit Aspose.Cells API programmgesteuert entfernen möchten, verwenden Sie bitte die[Arbeitsblatt.getSlicers().remove()](https://reference.aspose.com/cells/java/com.aspose.cells/slicercollection#remove\(com.aspose.cells.Slicer\)) Methode. Der Slicer wird aus dem Arbeitsblatt entfernt.
## **Slicer entfernen**
Der folgende Beispielcode lädt die[Beispiel-Excel-Datei](67338504.xlsx)die einen vorhandenen Slicer enthält. Es greift auf die Slicer zu und entfernt es dann. Schließlich speichert es die Arbeitsmappe als[Excel-Datei ausgeben](67338502.xlsx). Der folgende Screenshot zeigt den Slicer, der nach der Ausführung des Beispielcodes entfernt wird.
