## **Mögliche Verwendungsszenarien**
Wenn Sie den Slicer in Microsoft Excel entfernen möchten, wählen Sie ihn einfach aus und drücken Sie die *Taste Löschen*. Ebenso, wenn Sie ihn mithilfe der Aspose.Cells-API programmgesteuert entfernen möchten, verwenden Sie bitte die Methode [Worksheet.getSlicers().remove()](https://reference.aspose.com/cells/java/com.aspose.cells/slicercollection#remove\(com.aspose.cells.Slicer\)). Es wird den Slicer aus dem Arbeitsblatt entfernen.
