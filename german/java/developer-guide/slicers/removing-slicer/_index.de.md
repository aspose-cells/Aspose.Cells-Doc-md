## **Mögliche Verwendungsszenarien**
Wenn Sie den Slicer in Microsoft Excel entfernen möchten, wählen Sie ihn einfach aus und drücken Sie die *Entf*-Taste. Ebenso, wenn Sie ihn programmgesteuert mit der Aspose.Cells API entfernen möchten, verwenden Sie bitte die [Worksheet.getSlicers().remove()](https://reference.aspose.com/cells/java/com.aspose.cells/slicercollection#remove-com.aspose.cells.Slicer-) Methode. Dadurch wird der Slicer aus dem Arbeitsblatt entfernt. 
## **Slicer entfernen**
Der folgende Beispielcode lädt die [Beispiel-Excel-Datei](67338504.xlsx), die einen vorhandenen Slicer enthält. Es greift auf die Slicer zu und entfernt ihn dann.
