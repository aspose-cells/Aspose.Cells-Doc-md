## **Mögliche Verwendungsszenarien**
Sie können den Slicer in Microsoft Excel formatieren, indem Sie beispielsweise seine Anzahl von Spalten einstellen oder seinen Stil festlegen usw. Aspose.Cells ermöglicht es Ihnen auch, dies mithilfe der Eigenschaften [Slicer.NumberOfColumns](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#NumberOfColumns) und [Slicer.StyleType](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#StyleType) zu tun.
## **Formatierung Schneidwerkzeug**
Bitte sehen Sie sich den folgenden Code an, der die [Beispiel-Excel-Datei](67338501.xlsx) lädt, die einen Slicer enthält. Es greift auf den Slicer zu, setzt seine Anzahl von Spalten und den Stiltyp und speichert es als [Ausgabe-Excel-Datei](67338500.xlsx).
