## **Mögliche Nutzungsszenarien**
Sie können den Slicer in Microsoft Excel formatieren, indem Sie seine Spaltenanzahl oder seinen Stil usw. festlegen. Mit Aspose.Cells können Sie dies auch tun[Slicer.NumberOfColumns](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#NumberOfColumns)und[Slicer.StyleType](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#StyleType)Eigenschaften.
## **Slicer formatieren**
Bitte sehen Sie sich den folgenden Code an, er lädt die[Beispiel-Excel-Datei](67338501.xlsx)die einen Slicer enthält. Es greift auf den Slicer zu, legt die Anzahl der Spalten und den Stiltyp fest und speichert es als[Excel-Datei ausgeben](67338500.xlsx).
