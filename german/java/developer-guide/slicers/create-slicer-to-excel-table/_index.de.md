## **Mögliche Nutzungsszenarien**

 Ein Slicer wird verwendet, um Daten schnell zu filtern. Es kann verwendet werden, um Daten sowohl in einer Tabelle als auch in einer Pivot-Tabelle zu filtern. Microsoft Mit Excel können Sie Slicer erstellen, indem Sie eine Tabelle oder Pivot-Tabelle auswählen und dann auf klicken*Einfügen > Slicer*. Aspose.Cells ermöglicht es Ihnen auch, Slicer mit dem zu erstellen[**Worksheet.Slicers.Add()**](https://reference.aspose.com/cells/java/com.aspose.cells/slicercollection#add(com.aspose.cells.ListObject,%20com.aspose.cells.ListColumn,%20int,%20int)) Methode.
