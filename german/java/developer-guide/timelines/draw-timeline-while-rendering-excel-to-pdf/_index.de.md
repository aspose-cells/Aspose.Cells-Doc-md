## **Zeitleiste beim Rendern von Excel in PDF zeichnen**
Wenn Sie eine Excel-Datei haben, der eine Zeitachse hinzugefügt wurde, und Sie die Excel-Datei mit den Zeitleisteneinstellungen in PDF exportieren möchten, Aspose.Cells for Java unterstützt dies standardmäßig. Sie exportieren einfach die Excel-Datei mit Zeitleiste nach PDF, das generierte PDF wird die angewendete Zeitachse zeigen.

Der folgende Beispielcode lädt die [Beispiel-Excel-Datei](input.xlsx), die eine vorhandene Zeitachse enthält. Anschließend speichert er die Arbeitsmappe als [ausgegebene PDF-Datei](out.pdf). Der folgende Screenshot vergleicht die Quell-Excel-Datei und die generierte PDF-Datei.
