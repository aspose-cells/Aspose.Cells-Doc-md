## **Mögliche Verwendungsszenarien**

Sie können die *Versionsnummer* der Excel-Datei ändern, indem Sie mit der rechten Maustaste auf die Datei klicken und dann *Eigenschaften > Details* auswählen und dann das Feld *Versionsnummer* bearbeiten. Verwenden Sie bitte das [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#DocumentVersion) Eigenschaft, um es programmgesteuert mithilfe von Aspose.Cells APIs zu ändern.
