## **Legen Sie den Kommentar des Tabellen- oder Listenobjekts im Arbeitsblatt fest**

 Bitte legen Sie den Kommentar des Tabellen- oder Listenobjekts innerhalb des Arbeitsblatts mit fest[**ListObject.Kommentar**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment)Eigentum. Der Kommentar ist in der Datei xl/tables/tableName.xml sichtbar.

 Der folgende Beispielcode lädt die[Excel-Quelldatei](5472561.xlsx), setzt den Kommentar des ersten Tabellen- oder Listenobjekts innerhalb des Arbeitsblatts und der Screenshot zeigt den durch diesen Code erstellten Kommentar im roten Rechteck.
