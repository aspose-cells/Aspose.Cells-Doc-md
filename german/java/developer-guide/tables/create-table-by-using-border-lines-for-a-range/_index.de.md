{{% alert color="primary" %}}

 Manchmal möchten Sie eine Tabelle erstellen, indem Sie Rahmenlinien für a hinzufügen**Bereich**/**Zellbereich** basierend auf der Adresse der Zellen, die Sie haben. Sie können verwenden[**Cells.createRange**](https://reference.aspose.com/cells/java/com.aspose.cells/cells#createRange(int,%20int,%20boolean) )-Methode zum Erstellen einer Reihe von Zellen. Das[**Cells.createRange**](https://reference.aspose.com/cells/java/com.aspose.cells/cells#createRange(int,%20int,%20boolean) )-Methode gibt a zurück[**Bereich**](https://reference.aspose.com/cells/java/com.aspose.cells/Range) Objekt. Sie können eine erstellen[**Stil**](https://reference.aspose.com/cells/java/com.aspose.cells/Style) Objekt und geben Sie die Rahmenoptionen (oben, links, unten, rechts) entsprechend an. Später erhalten Sie möglicherweise die Zellen der[**Bereich**](https://reference.aspose.com/cells/java/com.aspose.cells/Range)und wenden Sie die gewünschte Formatierung auf die Zellen an.
