{{% alert color="primary" %}}

 Microsoft Excel bietet eine Funktion zum Hinzufügen**Signaturzeile** in Excel-Arbeitsmappen. Sie können eine Signaturzeile hinzufügen, indem Sie auf klicken**Einfügung** Tab und dann auswählen**Signaturzeile** von dem**Text** Gruppe.

 Aspose.Cells bietet auch diese Funktion und hat die freigelegt[**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) Eigentum für diesen Zweck. In diesem Artikel wird erläutert, wie Sie diese Eigenschaft verwenden, um eine Signaturzeile mit Aspose.Cells hinzuzufügen.

{{% /alert %}}
