Englische Formeln werden in den benannten Bereich geschrieben. Diese Excel-Datei kann in einer Umgebung geöffnet werden, in der das System auf deutsches Gebietsschema konfiguriert ist, die englische Formel soll jedoch in die deutsche Sprache übersetzt werden. Das folgende Beispiel demonstriert diese Funktion, erfordert jedoch, dass Excel in deutscher Sprache installiert ist und das Systemgebietsschema ebenfalls auf Deutsch eingestellt ist.
