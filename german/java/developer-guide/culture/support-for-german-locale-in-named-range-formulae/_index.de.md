Englische Formeln werden in benannten Bereich geschrieben. Diese Excel-Datei kann in einer Umgebung geöffnet werden, in der das System auf deutsche Lokalisierung konfiguriert ist. Allerdings sollen die englischen Formeln in die deutsche Sprache übersetzt werden. Das folgende Beispiel demonstriert diese Funktion, erfordert jedoch, dass Excel in deutscher Sprache installiert ist und die Systemlokalisierung ebenfalls auf Deutsch eingestellt sein muss.
