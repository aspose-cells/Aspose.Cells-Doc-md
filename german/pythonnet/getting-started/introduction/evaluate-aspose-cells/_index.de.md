{{% alert color="primary" %}}

 Aspose.Cells können Sie ganz einfach zur Auswertung herunterladen[pypi](https://pypi.org/project/aspose-cells-python/). Der Evaluierungs-Download ist derselbe wie der Download für die Vollversion: Die Evaluierung wird lizenziert, wenn Sie eine Lizenz beantragen.

{{% /alert %}}

Die Evaluierungsversion von Aspose.Cells (d. h. Aspose.Cells ohne angewendete Lizenz) bietet die meisten Produktfunktionen. Eine Einschränkung besteht darin, dass beim Öffnen oder Speichern einer Tabelle immer ein zusätzliches Arbeitsblatt mit einem Bewertungswasserzeichen eingefügt wird.
