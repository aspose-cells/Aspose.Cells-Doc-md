##  **Anzeigeformatoption „Rangfolge vom Kleinsten zum Größten“ und „Rangfolge vom Größten zum Kleinsten“.**

Aspose.Cells for Python via .NET bietet die Möglichkeit, die Anzeigeformatoption für Pivotfelder festzulegen. Hierfür sorgt die API[**PivotField.data_display_format**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfield/data_display_format/) Eigentum. Um vom Größten zum Kleinsten zu ordnen, können Sie Folgendes festlegen[**PivotField.data_display_format**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfield/data_display_format/)Eigentum zu[**PivotFieldDataDisplayFormat.RANK_LARGEST_TO_SMALLEST**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfielddatadisplayformat/). Der folgende Codeausschnitt veranschaulicht das Festlegen der Anzeigeformatoptionen.
