##  **Benutzerdefinierte Sortierung in der Pivot-Tabelle**
Mit Aspose.Cells for Python via .NET API können Sie Pivot-Tabellen nach Feldwerten sortieren. Der folgende Codeausschnitt lädt die Beispiel-Excel-Datei und fügt drei Pivot-Tabellen hinzu. Die erste Pivot-Tabelle weist keine benutzerdefinierte Sortierung auf, die zweite Pivot-Tabelle ist nach den Zeilenfeldwerten „SeaFood“ und die dritte Pivot-Tabelle nach den Feldwerten der Spalte „28.07.2000“ sortiert.
