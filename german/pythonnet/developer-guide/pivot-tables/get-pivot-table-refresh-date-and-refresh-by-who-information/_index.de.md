Aspose.Cells for Python via .NET unterstützt jetzt das Abrufen des Aktualisierungsdatums und der Aktualisierung durch wen-Informationen aus einer Arbeitsmappe.

{{% /alert %}}

[**PivotTable.refresh_date**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/refresh_date/)Gibt das Datum zurück, an dem der PivotTable-Bericht zuletzt aktualisiert wurde. Ähnlich[**PivotTable.refreshed_by_who**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/refreshed_by_who/) Die Eigenschaft gibt den Namen des Benutzers zurück, der den Bericht das letzte Mal aktualisiert hat.
