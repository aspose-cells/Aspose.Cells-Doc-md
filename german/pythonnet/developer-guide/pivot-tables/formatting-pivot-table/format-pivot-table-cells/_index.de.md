{{% alert color="primary" %}}

Manchmal möchten Sie Pivot-Tabellenzellen formatieren. Zum Beispiel möchten Sie den Pivot-Tabellenzellen eine Hintergrundfarbe zuweisen. Aspose.Cells für Python via .NET bietet zwei Methoden [**PivotTable.format_all(style)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format_all/) und [**PivotTable.format(row, column, style)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format/), die Sie für diesen Zweck verwenden können.

[**PivotTable.format_all(style)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format_all/) wendet den Stil auf die gesamte Pivottabelle an, während [**PivotTable.format(row, column, style)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format/) den Stil auf eine einzelne Zelle der Pivottabelle anwendet.
