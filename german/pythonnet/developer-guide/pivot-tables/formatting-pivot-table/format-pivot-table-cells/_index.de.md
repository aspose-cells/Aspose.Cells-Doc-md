{{% alert color="primary" %}}

 Manchmal möchten Sie Pivot-Tabellenzellen formatieren. Sie möchten beispielsweise eine Hintergrundfarbe auf Pivot-Tabellenzellen anwenden. Aspose.Cells for Python via .NET bietet zwei Methoden[**PivotTable.format_all(style)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format_all/) Und[**PivotTable.format(Zeile, Spalte, Stil)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format/), die Sie zu diesem Zweck nutzen können.

[**PivotTable.format_all(style)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format_all/) wendet den Stil auf die gesamte Pivot-Tabelle an, während[**PivotTable.format(Zeile, Spalte, Stil)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format/) Wendet den Stil auf eine einzelne Zelle der Pivot-Tabelle an.
