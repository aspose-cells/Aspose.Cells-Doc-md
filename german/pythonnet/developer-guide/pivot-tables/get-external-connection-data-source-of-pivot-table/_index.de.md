## **Wie man die externe Verbindungsdatenquelle der Pivot-Tabelle abruft**

Aspose.Cells für Python via .NET bietet die Möglichkeit, die externe Verbindungsdatenquelle der Pivot-Tabelle abzurufen. Hierfür bietet die API die Eigenschaft [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) der Klasse [**PivotTable**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/). Die Eigenschaft [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) gibt das Objekt [**ExternalConnection**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/) zurück. Der folgende Codeausschnitt zeigt die Verwendung der Eigenschaft [**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) zum Abrufen der externen Verbindungsdatenquelle der Pivot-Tabelle.
