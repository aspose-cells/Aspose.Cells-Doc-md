##  **Rufen Sie die externe Verbindungsdatenquelle der Pivot-Tabelle ab**

Aspose.Cells for Python via .NET bietet die Möglichkeit, die externe Verbindungsdatenquelle der Pivot-Tabelle abzurufen. Hierfür sorgt die API[**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) Eigentum der[**PivotTable**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/) Klasse. Der[**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/)Eigenschaft gibt die zurück[**Externe Verbindung**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/) Objekt. Der folgende Codeausschnitt demonstriert die Verwendung von[**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) -Eigenschaft, um die externe Verbindungsdatenquelle der Pivot-Tabelle abzurufen.
