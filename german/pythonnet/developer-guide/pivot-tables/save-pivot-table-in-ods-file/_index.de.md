## **So speichern Sie eine Pivot-Tabelle in einer ODS-Datei**

Aspose.Cells für Python via .NET bietet die Möglichkeit, Pivot-Tabellen in ODS-Dateien zu speichern. Dazu konvertieren Sie einfach eine Arbeitsmappe mit einer vorhandenen Pivot-Tabelle oder erstellen eine neue Pivot-Tabelle und speichern die Datei im ODS-Format. Stellen Sie sicher, dass Sie [**PivotTable.calculate_data**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/calculate_data/#) aufrufen, bevor Sie speichern, um sicherzustellen, dass die Pivot-Tabelle in der Ausgabedatei ODS gerendert wird.
