 Aspose.Cells for Python via .NET bietet die Möglichkeit, Pivot-Tabellen in ODS-Dateien zu speichern. Dazu konvertieren Sie einfach eine Arbeitsmappe mit vorhandener Pivot-Tabelle oder erstellen eine neue Pivot-Tabelle und speichern die Datei im Format ODS. Rufen Sie unbedingt an[**PivotTable.calculate_data**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/calculate_data/#)Stellen Sie vor dem Speichern sicher, dass die Pivot-Tabelle in der Ausgabedatei ODS gerendert wird.
