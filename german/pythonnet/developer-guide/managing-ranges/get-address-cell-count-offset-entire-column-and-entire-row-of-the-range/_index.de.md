## **Mögliche Verwendungsszenarien**
Aspose.Cells für Python via .NET bietet das Range-Objekt, das verschiedene Hilfsmethoden enthält, die dem Benutzer die Arbeit mit Excel-Bereichen erleichtern. Dieser Artikel veranschaulicht die Verwendung der folgenden Methoden oder Eigenschaften des Range-Objekts.

- **adresse**

Holt die Adresse des Bereichs ab.

- **zellenanzahl**

Holt alle Zellenanzahlen im Bereich ab.

- **offset_erhalten**

Holt den Bereich durch Versatz ab.

- **gesamte_spalte**

Ruft ein Range-Objekt ab, das die gesamte Spalte (oder Spalten) enthält, die den angegebenen Bereich enthält.

- **gesamte_zeile**

Ruft ein Range-Objekt ab, das die gesamte Zeile (oder Zeilen) enthält, die den angegebenen Bereich enthält.
