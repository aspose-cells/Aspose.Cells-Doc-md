## **Einführung**

In Excel können Sie einen Bereich auswählen, dann einen Bereich einfügen und andere Daten nach rechts oder nach unten verschieben.

Aspose.Cells für Python via .NET bietet die Methode [Cells.insert_range](https://reference.aspose.com/cells/python-net/aspose.cells/cells/insert_range/) zum Einfügen eines Bereichs an.
