## **Mögliche Verwendungsszenarien**

Mit Aspose.Cells for Python via .NET können Sie einen Bereich von Zellen zusammenführen oder aufteilen. Aspose.Cells for Python via .NET bietet die [**Range.merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/merge/#) und [**Range.un_merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/un_merge/#) Methoden für diesen Zweck. Dieser Artikel erläutert, wie man einen Bereich von Zellen in eine einzige Zelle zusammenführt.


## **Wie man einen Bereich von Zellen zusammenführt oder trennt**

Der folgende Beispielcode erstellt zunächst einen Bereich - A1:D4 - und fusioniert dann die Zellen im Bereich in eine einzelne Zelle mit der Methode [**Range.merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/merge/#). Ebenso können Sie Zellen aufteilen, indem Sie einen Bereich erstellen und die Methode [**Range.un_merge()**](https://reference.aspose.com/cells/python-net/aspose.cells/range/un_merge/#) aufrufen.
