Manchmal benötigen Sie jedoch einen nicht aufeinanderfolgenden Zellenbereich, in dem die Zellen nicht benachbart sind. Aspose.Cells für Python via .NET unterstützt die Erstellung eines benannten Bereichs mit nicht benachbarten Zellen.
