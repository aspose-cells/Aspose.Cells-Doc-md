## **Anwendungsszenario**

Es kann vorkommen, dass die Datenquellendateien verschoben wurden und die Excel-Datei die Datei nicht mehr finden kann. In solchen Fällen bietet die Aspose.Cells für Python via .NET API die Option, den Power-Query-Formelpunkt mit der Klasse [*PowerQueryFormulaItem*](https://reference.aspose.com/cells/python-net/aspose.cells.querytables/powerqueryformulaitem) zu aktualisieren, um den Speicherort der Quelldatei zu aktualisieren.
