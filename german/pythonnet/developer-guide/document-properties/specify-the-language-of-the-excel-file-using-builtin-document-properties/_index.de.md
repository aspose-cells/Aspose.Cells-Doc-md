## **Mögliche Verwendungsszenarien**

Sie können die Sprache der Excel-Datei ändern, indem Sie mit der rechten Maustaste auf die Datei klicken und dann Eigenschaften > Details auswählen und das Feld Sprache bearbeiten. Bitte verwenden Sie die [**BuiltInDocumentPropertyCollection.language**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/language)-Eigenschaft, um dies programmgesteuert mit Aspose.Cells für Python via .NET APIs zu ändern.
