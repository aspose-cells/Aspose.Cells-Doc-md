## **Mögliche Verwendungsszenarien**

Sie können die **Versionsnummer** Ihrer Excel-Datei ändern, indem Sie mit der rechten Maustaste auf die Datei klicken, Eigenschaften auswählen > Details öffnen und dann das Feld **Versionsnummer** bearbeiten. Verwenden Sie die [**BuiltInDocumentPropertyCollection.document_version**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/document_version)-Eigenschaft, um sie programmgesteuert mit Aspose.Cells für Python via .NET APIs zu ändern.
