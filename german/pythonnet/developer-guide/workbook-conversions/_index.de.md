##  **Konvertieren einer Excel-Arbeitsmappe in XPS**

Das Dokumentformat XPS besteht aus strukturiertem XML-Markup, das das Layout eines Dokuments und das visuelle Erscheinungsbild jeder Seite definiert, sowie Rendering-Regeln zum Verteilen, Archivieren, Rendern, Verarbeiten und Drucken von Dokumenten.

Die Markup-Sprache für XPS ist eine Teilmenge von Die verwendeten Elemente werden anhand von Pfaden und anderen geometrischen Grundelementen beschrieben.

Bei einer XPS-Datei handelt es sich tatsächlich um ein Unicode-ZIP-Archiv, das die Open Packaging Conventions verwendet und die Dateien enthält, aus denen das Dokument besteht. Dazu gehören eine XML-Markup-Datei für jede Seite, Text, eingebettete Schriftarten, Rasterbilder, 2D-Vektorgrafiken sowie Informationen zur digitalen Rechteverwaltung. Der Inhalt einer XPS-Datei kann untersucht werden, indem man sie einfach in einer Anwendung öffnet, die ZIP-Dateien unterstützt.

##  **Festlegen der Bildeinstellungen für HTML**

 Aspose.Cells for Python via .NET hat ausgesetzt**[image_options](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/image_options/)** für die**[HtmlSaveOptions](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions)**Klasse, die es Entwicklern ermöglicht, Bildeinstellungen beim Speichern von Tabellenkalkulationen im Format HTML festzulegen.

Nachfolgend finden Sie Einzelheiten zu einigen Bildeinstellungen, die angewendet werden können:

- *[Bildtyp](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/imagetype/)**: Gibt den Bildtyp an. Bitte beachten Sie, dass alle Formen, einschließlich Diagramme, in der Ausgabe HTML als Bilder gerendert werden.
- *[Smoothing_mode](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/smoothing_mode/)**: Gibt das Anti-Aliasing für Linien, Kurven und Kanten gefüllter Bereiche an.
- *[text_rendering_hint](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/text_rendering_hint/)**: Gibt die Qualität der Textwiedergabe an.
- **[Qualität](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/quality/)**: Gibt die Qualität des Bildes zwischen 0 und 100 an, wenn **[ImageType ](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/imagetype/)**wird als Jpeg angegeben.
- *[vertikale_auflösung](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/vertical_resolution/)**: Ruft die vertikale Auflösung des Bildes in Punkten pro Zoll ab oder legt diese fest.
- *[horizontale_auflösung](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/horizontal_resolution/)**: Ruft die horizontale Auflösung des Bildes in Punkten pro Zoll ab oder legt diese fest.
- **[tiff_compression](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/tiff_compression/)**: Ruft den Komprimierungstyp für die Bilder ab oder legt diesen fest, wenn **[ImageType]( https://reference.aspose.com/cells/python-net/aspose.cells.drawing/imagetype/)**wird als Tiff angegeben.
- *[transparent](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/transparent/)**: Gibt an, ob der Hintergrund eines Bildes transparent sein soll, wenn ImageFormat als PNG angegeben ist.

 Der folgende Code zeigt die Verwendung**[HtmlSaveOptions.image_options](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/image_options/)**um unterschiedliche Präferenzen anzugeben.
