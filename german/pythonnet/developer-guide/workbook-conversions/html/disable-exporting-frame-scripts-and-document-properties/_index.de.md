{{% alert color="primary" %}}

Aspose.Cells for Python via .NET exportiert Frame-Skripte und Dokumenteigenschaften, wenn eine Arbeitsmappe in HTML konvertiert wird. Version 8.6.0 von Aspose.Cells für Python via .NET führt eine Option ein, mit der das Exportieren von Frame-Skripten und Dokumenteigenschaften optional deaktiviert werden kann. Verwenden Sie die Eigenschaft HtmlSaveOptions.ExportFrameScriptsAndProperties, um das Exportieren zu deaktivieren.

{{% /alert %}}
