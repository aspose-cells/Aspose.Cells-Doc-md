## **Mögliche Verwendungsszenarien**

Beim Speichern Ihrer Excel-Datei in HTML werden Kommentare nicht exportiert. Aspose.Cells für Python via .NET bietet diese Funktion jedoch mit der [**HtmlSaveOptions.is_export_comments**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/is_export_comments)-Eigenschaft. Wenn Sie diese auf **true** setzen, werden Kommentare in Ihrer Excel-Datei ebenfalls angezeigt.
