{{% alert color="primary" %}} 

Aspose.Cells für Python via .NET unterstützt jetzt das Expandieren von Text von rechts nach links beim Exportieren von Excel-Dateien in HTML. Dieses Feature wurde seit Version v8.9.0.0 implementiert. Wenn Ihre Quelldatei Text enthält, der von rechts nach links expandiert, exportiert Aspose.Cells dies korrekt in HTML.

{{% /alert %}}
