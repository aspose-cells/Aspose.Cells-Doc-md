## **HTML-zu-PDF-Konvertierung in Python**
Wie konvertiert man HTML in PDF? Mit [Aspose.Cells for Python via NET](https://pypi.org/project/aspose-cells-python/) Bibliothek können Sie mit wenigen Zeilen Code HTML programmgesteuert in PDF konvertieren. Aspose.Cells for Python via NET ist in der Lage, plattformübergreifende Anwendungen zu erstellen, die die Fähigkeit haben, alle Excel-Dateien zu generieren, zu ändern, zu konvertieren, zu rendern und zu drucken.

## **Python HTML in PDF konvertieren**
Der folgende C#-Codeausschnitt zeigt, wie ein HTML-Dokument mit [Aspose.Cells for Python via NET](https://pypi.org/project/aspose-cells-python/) in ein PDF konvertiert wird.

1. Erstellen Sie eine Instanz der [HtmlLoadOptions](https://reference.aspose.com/cells/python-net/aspose.cells/htmlloadoptions/) Klasse.
1. Initialisieren Sie das [Arbeitsmappe](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/) Objekt.
1. Speichern Sie das Ausgabedokument als PDF, indem Sie die Methode Workbook.save() aufrufen.
