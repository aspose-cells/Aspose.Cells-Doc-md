{{% alert color="primary" %}} 

Aspose.Cells für Python via .NET bietet eine nützliche Eigenschaft HtmlSaveOptions.PresentationPreference für Entwickler, die eine bessere Layoutdarstellung beim Speichern einer Microsoft Excel-Datei in HTML- oder MHT-Format benötigen. Der Standardwert der Eigenschaft ist false. Wir empfehlen, diese Eigenschaft auf true zu setzen, um eine ansprechendere Präsentation von Excel-Berichten zu erhalten.

{{% /alert %}}
