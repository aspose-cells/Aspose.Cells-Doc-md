{{% alert color="primary" %}} 

Wenn Sie Ihre Excel-Datei mit GridLines in HTML exportieren möchten, verwenden Sie bitte die [HtmlSaveOptions.export_grid_lines](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_grid_lines/) Eigenschaft und setzen sie auf **true**.
