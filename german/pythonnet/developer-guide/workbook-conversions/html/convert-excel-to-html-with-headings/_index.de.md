## **Excel in HTML mit Überschriften konvertieren**

Aspose.Cells for Python via NET bietet die Möglichkeit, Überschriften beim Konvertieren von Excel in HTML zu exportieren. Hierzu können Sie die [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)-Eigenschaft auf **true** setzen. Der Standardwert der [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)-Eigenschaft ist **false**.
