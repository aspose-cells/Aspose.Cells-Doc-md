##  **Konvertieren Sie Excel mit Überschriften in HTML**

Aspose.Cells for Python über NET bietet die Möglichkeit, Überschriften beim Konvertieren von Excel in HTML zu exportieren. Hierzu können Sie die festlegen[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) Eigenschaft auf *true**. Der Standardwert von[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)Eigenschaft ist *falsch**.
