{{% alert color="primary" %}} 

Aspose.Cells für Python via .NET unterstützt jetzt die vollständige Konvertierung von Überarbeitungen von XLSB-Dateien in XLSM-Dateien. Die Überarbeitungen befinden sich im Pfad \xl\revisions. Sie können sie anzeigen, indem Sie die Dateierweiterung Ihrer XLSB-Datei in ZIP ändern. Der Pfad \xl\revisions enthält Dateien mit der Erweiterung .bin.

Wenn Sie Ihre XLSB-Datei mit Aspose.Cells for Python via .NET in eine XLSM-Datei konvertieren, werden diese .bin-Dateien erfolgreich in .xml-Dateien umgewandelt, wie in diesen beiden Screenshots gezeigt.
