Aspose.Cells for Python via .NET unterstützt jetzt die vollständige Konvertierung von Revisionen der Datei XLSB in die Datei XLSM. Revisionen befinden sich im Pfad \xl\revisions. Sie können sie anzeigen, indem Sie die Dateierweiterung XLSB in ZIP ändern. Der Pfad \xl\revisions enthält Dateien mit der Erweiterung .bin.

Wenn Sie Ihre XLSB-Datei mit Aspose.Cells for Python via .NET in eine XLSM-Datei konvertieren, werden diese .bin-Dateien erfolgreich in .xml-Dateien konvertiert, wie in diesen beiden Screenshots gezeigt.
