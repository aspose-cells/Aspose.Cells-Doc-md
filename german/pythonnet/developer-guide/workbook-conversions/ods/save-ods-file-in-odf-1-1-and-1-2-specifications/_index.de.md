{{% alert color="primary" %}}

 Aspose.Cells for Python via .NET unterstützt das Speichern einer ODS-Datei (**OpenDocument Spreadsheet**) in den ODF-Spezifikationen (**OpenDocument Format**) 1.1 und 1.2. Aspose.Cells for Python via .NET hat[**OdsSaveOptions.is_strict_schema11**](https://reference.aspose.com/cells/python-net/aspose.cells/odssaveoptions/is_strict_schema11/)Eigenschaft, die die Verwendung der ODF 1.1-Spezifikation zum Speichern von ODS-Dateien angibt. Der Standardwert dieser Eigenschaft ist *false**, sodass die ohne diese Einstellung gespeicherte Datei ODS die 1.2-Spezifikationen verwendet.

{{% /alert %}}

Der folgende Beispielcode erstellt ein Arbeitsmappenobjekt, fügt der Zelle A1 im ersten Arbeitsblatt einen Wert hinzu und speichert dann die Datei ODS in den ODF-Spezifikationen 1.1 und 1.2. Standardmäßig wird die Datei ODS in der ODF 1.2-Spezifikation gespeichert.
