PDF-Funktionen für die logische Struktur bieten einen Mechanismus zum Einbinden von Informationen bezüglich der Dokumentinhaltsstruktur in eine PDF-Datei. Aspose.Cells for Python via .NET bewahrt Informationen über die Struktur eines Microsoft Excel-Dokuments, wie Zelle, Zeile, Tabelle, Arbeitsblatt, Bild, Form, Kopf-/Fußzeile usw.

Mit der Option [PdfSaveOptions.export_document_structure]https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/export_document_structure/) können Sie in einem getaggten Dokument mit der Nummer PDF speichern und die Dokumentstruktur exportieren.
