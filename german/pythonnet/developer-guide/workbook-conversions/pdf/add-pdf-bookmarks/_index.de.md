Dieser Artikel bietet Informationen dazu, wie PDF-Lesezeichen beim Konvertieren einer Tabellenkalkulation in PDF eingefügt werden können.

Mit Aspose.Cells für Python via .NET können Sie Lesezeichen im Handumdrehen hinzufügen. PDF-Lesezeichen können die Navigierbarkeit langer Dokumente erheblich verbessern. Wenn Sie Lesezeichenlinks zu einem PDF-Dokument hinzufügen, haben Sie eine präzise Kontrolle über die genaue Ansicht, die Sie wünschen. Sie sind nicht darauf beschränkt, nur auf eine Seite zu verlinken. Sie können die genaue Ansicht einrichten, indem Sie die Zielseite positionieren und dann das Lesezeichen erstellen.

{{% /alert %}}

Bitte sehen Sie sich den folgenden Beispielcode an, um zu erfahren, wie PDF-Lesezeichen hinzugefügt werden können. Der Code generiert eine einfache Arbeitsmappe, gibt PDF-Lesezeichen mit Zielorten an und generiert die PDF-Datei.

{{< gist "aspose-cells-gists" "7bb30376b4d40cdfd596286870fb9752" "PDF-AddPDFBookmarks-1.py" >}}

{{% alert color="primary" %}}

Wenn Ihre Arbeitsmappe Formeln enthält, ist es am besten, [**Workbook.calculate_formula**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/calculate_formula/#) kurz vor dem Rendern der Arbeitsmappe im PDF-Format aufzurufen. Dadurch wird sichergestellt, dass die formelabhängigen Werte aktualisiert und im PDF-Format korrekt dargestellt werden.
