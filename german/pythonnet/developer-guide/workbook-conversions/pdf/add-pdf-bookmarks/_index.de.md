Dieser Artikel enthält Informationen zum Einfügen von PDF-Lesezeichen beim Konvertieren einer Tabelle in PDF.

Aspose.Cells for Python via .NET ermöglicht Ihnen das spontane Hinzufügen von Lesezeichen. PDF-Lesezeichen können die Navigation in langen Dokumenten drastisch verbessern. Wenn Sie Lesezeichen-Links zum Dokument PDF hinzufügen, können Sie genau die gewünschte Ansicht steuern und sind nicht auf die Verknüpfung mit einer Seite beschränkt. Sie können die genaue Ansicht einrichten, indem Sie die Zielseite positionieren und dann das Lesezeichen erstellen.

{{% /alert %}}

Bitte sehen Sie sich den folgenden Beispielcode an, um herauszufinden, wie Sie PDF-Lesezeichen hinzufügen. Der Code generiert eine einfache Arbeitsmappe, gibt PDF-Lesezeichen mit Zielspeicherorten an und generiert die Datei PDF.
