##  **Mögliche Nutzungsszenarien**

Zuvor konnten Office-Add-Ins mit Aspose.Cells for Python via .NET nicht gerendert werden, wenn die Excel-Datei im Format PDF gespeichert wurde. Jetzt macht Aspose.Cells for Python via .NET alles in Ordnung. Sie müssen keine spezielle Methode oder Eigenschaft verwenden, um Office-Add-Ins in der Ausgabe PDF darzustellen. Speichern Sie einfach Ihre Excel-Datei im Format PDF und schon werden Office-Add-Ins gerendert.
