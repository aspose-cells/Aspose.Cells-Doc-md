##  **Mögliche Nutzungsszenarien**

Früher wurde Aspose.Cells for Python via .NET erstellt[**PDFBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) für ein normales Blatt. Aber jetzt können auch Aspose.Cells for Python via .NET erstellen[**PDFBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) für Diagrammblatt. Da das Diagrammblatt außer Zelle A1 keine andere Zelle enthält, wird es erstellt[**PDFBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) Nur für Zelle A1.

##  **Erstellen Sie einen PDFBookmarkEntry für das Diagrammblatt**

 Der folgende Beispielcode lädt die[Beispiel-Excel-Datei](61767756.xlsx) welches vier Blätter hat. Zwei davon sind normale Blätter und die anderen beiden sind Kartenblätter. Es erstellt wie folgt vier Lesezeicheneinträge

- Lesezeichen-I
- Lesezeichen-II-Chart1
- Lesezeichen-III
- Lesezeichen-IV-Chart2
