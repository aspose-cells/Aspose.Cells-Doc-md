## **Mögliche Verwendungsszenarien**

Früher erstellte Aspose.Cells für Python via .NET [**PdfBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) für ein normales Blatt. Aber jetzt kann Aspose.Cells für Python via .NET [**PdfBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) auch für ein Diagrammblatt erstellen. Da ein Diagrammblatt außer Zelle A1 keine anderen Zellen hat, wird es nur [**PdfBookmarkEntry**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/) für die Zelle A1 erstellen.
