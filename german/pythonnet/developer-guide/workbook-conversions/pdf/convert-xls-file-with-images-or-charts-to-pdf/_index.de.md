Aspose.Cells for Python via .NET unterstützt die Konvertierung von XLS-Dateien, die Bilder und Diagramme enthalten, in PDF-Dokumente. Aspose.Cells for Python via .NET API können unabhängig arbeiten, um eine Tabelle in PDF zu konvertieren: Aspose.PDF for .NET ist für die Konvertierung nicht erforderlich. Der Prozess kann im Speicher durchgeführt werden, da der Prozess nicht auf temporäre oder zwischengeschaltete XML-Dateien angewiesen ist. Dies bedeutet, dass große Excel-Dateien, die beispielsweise Bilder, Diagramme und andere Zeichenobjekte enthalten, schnell und effizient konvertiert werden können.
