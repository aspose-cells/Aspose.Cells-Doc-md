## **Excel-Arbeitsmappe in JSON konvertieren**

Die Aspose.Cells for Python via .NET API bietet Unterstützung beim Konvertieren von Tabellenkalkulationen in das JSON-Format. Um die Arbeitsmappe in JSON zu exportieren, geben Sie als zweiten Parameter der Methode [**Workbook.save**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) an. Sie können auch die [**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions)-Klasse verwenden, um zusätzliche Einstellungen für den Export des Arbeitsblatts in JSON festzulegen.

Das folgende Codebeispiel zeigt, wie das aktive Arbeitsblatt mithilfe des Enumerationselement [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) in Json exportiert wird. Bitte sehen Sie sich den Code zur Konvertierung der [Quelldatei](book1.xlsx) in die [Ausgabe-Json-Datei](book1.Json) an, die vom Code generiert wurde, zum Nachschlagen.
