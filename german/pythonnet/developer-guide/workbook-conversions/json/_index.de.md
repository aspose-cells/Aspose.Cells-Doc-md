##  **Konvertieren Sie die Excel-Arbeitsmappe in JSON**

Aspose.Cells for Python via .NET API bietet Unterstützung für die Konvertierung von Tabellenkalkulationen in das Format JSON. Um die Arbeitsmappe nach JSON zu exportieren, übergeben Sie[**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) als zweiter Parameter von[**Arbeitsmappe.speichern**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) Methode. Sie können auch verwenden[**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) Klasse, um zusätzliche Einstellungen für den Export des Arbeitsblatts nach JSON anzugeben.

 Das folgende Codebeispiel veranschaulicht den Export eines aktiven Arbeitsblatts nach Json mithilfe von[**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) Aufzählungsmitglied.
