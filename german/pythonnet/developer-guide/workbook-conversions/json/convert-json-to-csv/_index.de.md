## **JSON in CSV konvertieren**

Aspose.Cells f端r Python via .NET unterst端tzt die Konvertierung von einfachem sowie verschachteltem JSON in CSV. Dazu bietet die API die Klassen [**JsonLayoutOptions**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions) und [**JsonUtility**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonutility). Die Klasse [**JsonLayoutOptions**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions) bietet die Optionen f端r das JSON-Layout wie [**ignore_array_title**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions/ignore_array_title/) (ignoriert den Titel, wenn das Array eine Eigenschaft eines Objekts ist) oder [**array_as_table**](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions/array_as_table/) (verarbeitet das Array als Tabelle).
