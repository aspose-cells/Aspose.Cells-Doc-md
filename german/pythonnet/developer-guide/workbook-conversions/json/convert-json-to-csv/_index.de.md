##  **Konvertieren Sie JSON in CSV**

Aspose.Cells for Python via .NET unterstützt die Konvertierung von einfachen sowie verschachtelten JSON in CSV. Dafür sorgt die API**[JsonLayoutOptions](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions)** Und**[JsonUtility](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonutility)** Klassen. Der**[JsonLayoutOptions](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions)**Die Klasse bietet die Optionen für das Layout JSON**[ignore_array_title](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions/ignore_array_title/)**(ignoriert den Titel, wenn das Array eine Eigenschaft eines Objekts ist) oder ** [array_as_table](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions/array_as_table/)**(verarbeitet das Array als Tabelle). Das **[JsonUtility](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonutility)**Die Klasse verarbeitet JSON unter Verwendung der mit dem festgelegten Layoutoptionen**[JsonLayoutOptions](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions)**Klasse.

Das folgende Codebeispiel demonstriert die Verwendung von**[JsonLayoutOptions](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonlayoutoptions)**Und**[JsonUtility](https://reference.aspose.com/cells/python-net/aspose.cells.utility/jsonutility)** Klassen zum Laden der[Quelldatei JSON](104398869.json) und erzeugt die[Ausgabedatei CSV](104398870.csv).
