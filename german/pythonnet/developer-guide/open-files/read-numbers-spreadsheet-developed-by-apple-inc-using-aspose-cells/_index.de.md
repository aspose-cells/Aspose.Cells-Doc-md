## **Mögliche Verwendungsszenarien**

Numbers ist eine Tabellenkalkulationsanwendung von Apple Inc. Aspose.Cells for Python via .NET kann Numbers-Tabellen lesen, unterstützt jedoch kein Schreiben in diese. Es kann Daten, Stile und Formeln von Numbers-Tabellen lesen.

## **Numbers-Tabellen lesen, entwickelt von Apple Inc., mit Aspose.Cells Python via .NET**

Der folgende Beispielcode lädt die [Beispiel-Numbers-Arbeitsmappe](sampleNumbersByAppleInc.numbers) und konvertiert sie in das [Ausgabe-PDF-Format](outputNumbersByAppleInc.pdf). Sie müssen die [**LoadOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/loadoptions) Klasse verwenden und [**LoadFormat.NUMBERS**](https://reference.aspose.com/cells/python-net/aspose.cells/loadformat) als Parameter in ihrem Konstruktor angeben, um sie erfolgreich zu laden. Bitte laden Sie beide von den angegebenen Links herunter. Sie können den Code mit jeder Numbers-Arbeitsmappe ausprobieren. Bitte lesen Sie auch die Kommentare des Codes für weitere Hilfe.
