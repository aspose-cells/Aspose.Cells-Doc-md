## **Mögliche Verwendungsszenarien**

Sie können das gemeinsame Arbeitsblatt in Excel schützen oder entsperren, wie im folgenden Screenshot gezeigt. Aspose.Cells für Python via .NET unterstützt diese Funktion ebenfalls mit den Methoden [**Workbook.protect_shared_workbook()**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/protect_shared_workbook) und [**Workbook.unprotect_shared_workbook()**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/unprotect_shared_workbook).
