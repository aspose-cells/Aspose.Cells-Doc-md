## **Mögliche Verwendungsszenarien**

Wenn Sie ein neues Blatt zu einer Arbeitsmappe hinzufügen, enthält es die standardmäßigen *Seiteneinrichtungseinstellungen*. Es kann vorkommen, dass Sie die Einstellungen ([**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup)) von einem Arbeitsblatt auf ein anderes übertragen müssen. In diesem Dokument wird erläutert, wie Sie die Seiteneinrichtungseinstellungen von einem Arbeitsblatt auf ein anderes mithilfe der Aspose.Cells for Python via .NET APIs kopieren.
