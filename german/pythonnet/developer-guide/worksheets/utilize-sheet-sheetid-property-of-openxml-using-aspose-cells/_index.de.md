## **Mögliche Verwendungsszenarien**

*Sheet.SheetId*-Eigenschaft ist innerhalb des *DocumentFormat.OpenXml.Spreadsheet*-Namespace zu finden und ist Teil von OpenXml. Sie können diese Eigenschaft und ihren Wert innerhalb von *workbook.xml* wie in der folgenden Bildschirmaufnahme sehen. Aspose.Cells für Python via .NET bietet die entsprechende Eigenschaft als [**Worksheet.tab_id**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/tab_id).
