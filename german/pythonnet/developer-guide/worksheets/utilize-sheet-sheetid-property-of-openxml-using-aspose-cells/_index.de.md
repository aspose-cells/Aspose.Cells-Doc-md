## **Mögliche Verwendungsszenarien**

*Sheet.SheetId* Eigenschaft befindet sich im Namespace *DocumentFormat.OpenXml.Spreadsheet* und ist Teil von OpenXml. Sie können diese Eigenschaft und ihren Wert in *workbook.xml* sehen, wie im folgenden Screenshot dargestellt. Aspose.Cells für Python via .NET bietet die entsprechende Eigenschaft als [**Worksheet.tab_id**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/tab_id).
