Aspose.Cells für Python via .NET ermöglicht es, die Farbe einzelner Tabellenblatt-Registerkarten anzupassen, um sie hervorzuhaben. Zum Beispiel können Sie Ausgaben rot, Verkäufe grün, Vermögenswerte blau usw. darstellen.

{{% /alert %}} 

## **Wie man die Tabellenblatt-Registerkartenfarbe mit Microsoft Excel festlegt**
1. Klicken Sie mit der rechten Maustaste auf eine Registerkarte im Registerblatt am unteren Rand des aktuellen Arbeitsblatts.
1. Wählen Sie **Registerkartenfarbe**.
1. Wählen Sie eine Farbe aus der Palette.
1. Klicken Sie auf **OK**.
