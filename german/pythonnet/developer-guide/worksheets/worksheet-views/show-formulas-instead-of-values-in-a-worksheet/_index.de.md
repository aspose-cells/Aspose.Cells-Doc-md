{{% alert color="primary" %}}

Es ist möglich, Formeln anstelle von berechneten Werten in Microsoft Excel mithilfe der **Formeln anzeigen** Option im **Formeln** Register anzuzeigen. Wenn Formeln angezeigt werden, zeigt Microsoft Excel die Formeln im Arbeitsblatt an. Das gleiche Ergebnis kann mit Aspose.Cells für Python via .NET erreicht werden.

{{% /alert %}}

Aspose.Cells für Python via .NET stellt eine [**Worksheet.show_formulas**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/show_formulas) Eigenschaft bereit. Setzen Sie diese auf **true**, um Microsoft Excel anzuzeigen.
