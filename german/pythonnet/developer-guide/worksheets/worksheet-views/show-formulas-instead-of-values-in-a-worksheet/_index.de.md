{{% alert color="primary" %}}

Es ist möglich, Formeln anstelle berechneter Werte in Microsoft Excel anzuzeigen, indem die **Formeln anzeigen**-Option im **Formeln**-Ribbon verwendet wird. Wenn Formeln angezeigt werden, zeigt Microsoft Excel Formeln im Arbeitsblatt an. Sie können dasselbe auch mit Aspose.Cells für Python via .NET erreichen.

{{% /alert %}}

Aspose.Cells für Python via .NET bietet eine [**Worksheet.show_formulas**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/show_formulas) Eigenschaft. Stellen Sie diese auf **true**, um Microsoft Excel anzuweisen, Formeln anzuzeigen.
