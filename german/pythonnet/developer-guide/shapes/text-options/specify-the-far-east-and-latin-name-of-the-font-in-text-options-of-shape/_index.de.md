## **Mögliche Verwendungsszenarien**

Manchmal möchten Sie Text in ostasiatischer Schriftart anzeigen, z.B. Japanisch, Chinesisch, Thailändisch usw. Aspose.Cells für Python via .NET bietet die [**TextOptions.far_east_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/far_east_name)-Eigenschaft, mit der Sie den Schriftartnamen der ostasiatischen Sprache angeben können. Außerdem können Sie den lateinischen Fontnamen mit der [**TextOptions.latin_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/latin_name)-Eigenschaft festlegen.
