## **Mögliche Verwendungsszenarien**

Aspose.Cells for Python via .NET ermöglicht es, die Ränder jeder Form oder Kommentare mit der [**Shape.text_body.text_alignment**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/fontsettingcollection/text_alignment)-Eigenschaft festzulegen. Diese Eigenschaft gibt das Objekt der Klasse [**ShapeTextAlignment**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/shapetextalignment) zurück, das verschiedene Eigenschaften hat, z.B. [**top_margin_pt**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/shapetextalignment/top_margin_pt), [**left_margin_pt**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/shapetextalignment/left_margin_pt), [**bottom_margin_pt**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/shapetextalignment/bottom_margin_pt), [**right_margin_pt**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/shapetextalignment/right_margin_pt) usw., die verwendet werden können, um die oberen, linken, unteren und rechten Ränder festzulegen.
