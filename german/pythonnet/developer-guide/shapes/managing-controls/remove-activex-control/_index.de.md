## **AktiveX-Steuerung entfernen**

Aspose.Cells für Python via .NET bietet die Möglichkeit, ActiveX-Steuerelemente aus Arbeitsmappen zu entfernen. Dafür stellt die API die Methode [**Shape.remove_active_x_control**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/remove_active_x_control) bereit. Das folgende Code-Snippet zeigt die Verwendung der [**Shape.remove_active_x_control**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/remove_active_x_control)-Methode, um ein ActiveX-Steuerelement zu entfernen.
