## **Mögliche Verwendungsszenarien**
Sie können den voreingestellten WordArt-Stil auf den Text der Form setzen, indem Sie Aspose.Cells für Python via .NET verwenden. Bitte verwenden Sie die Methoden [FontSetting.set_word_art_style()](https://reference.aspose.com/cells/python-net/aspose.cells.fontsetting/set_word_art_style/#aspose.cells.drawing.PresetWordArtStyle) oder [FontSettingCollection.set_word_art_style()](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/fontsettingcollection/set_word_art_style/#aspose.cells.drawing.PresetWordArtStyle) für diesen Zweck.
