{{% alert color="primary" %}}

Manchmal müssen Sie die Breite und Höhe des Zellenwerts berechnen, um den Zellenwert innerhalb der Zelle anzupassen. Aspose.Cells for Python via .NET bietet dafür [**Cell.get_width_of_value()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_width_of_value/#) und [**Cell.get_height_of_value()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_height_of_value/#) Methoden. Mit diesen Methoden können Sie die Breite und Höhe des Zellenwerts berechnen und dann die Breite der Spalte und Höhe der Zeile dieser Zelle entsprechend einstellen. Dadurch wird der Zellenwert innerhalb der Zelle angepasst oder eingepasst.

Alternativ können Sie auch Zeilen und Spalten Ihrer Zelle oder Zellenbereichs mit den Aspose.Cells for Python via .NET APIs automatisch anpassen.
