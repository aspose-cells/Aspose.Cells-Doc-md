{{% alert color="primary" %}} 

Hyperlinks sind nützlich, um auf Informationen in anderen Arbeitsblättern oder auf Websites zuzugreifen. Microsoft Excel ermöglicht es Benutzern, Hyperlinks auf Text in Zellen und auf Bilder hinzuzufügen. Bild-Hyperlinks können die Navigation in einem Arbeitsblatt erleichtern, zum Beispiel als Navigationsbuttons oder Logos, die auf bestimmte Websites verlinken. Dieses Dokument erläutert, wie Bild-Hyperlinks in einem Arbeitsblatt mithilfe von Aspose.Cells for Python via .NET eingefügt werden.

{{% /alert %}} 

Aspose.Cells for Python via .NET ermöglicht es Ihnen, zur Laufzeit Hyperlinks zu Bildern in Tabellenkalkulationen hinzuzufügen.
