{{% alert color="primary" %}}

Aspose.Cells für Python via .NET unterstützt die Konvertierung von Microsoft Excel-orientiertem HTML in das XLS/XLSX-Format. Das bedeutet, dass das von Microsoft Excel generierte HTML mithilfe von Aspose.Cells für Python via .NET wieder in das XLS/XLSX-Format konvertiert werden kann.

Ebenso kann Aspose.Cells für Python via .NET einfaches HTML in HTML-Rich-Text konvertieren. Aspose.Cells für Python via .NET bietet die [**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/)-Eigenschaft, die ein solches einfaches HTML aufnehmen und in formatierten Zelltext umwandeln kann.

{{% /alert %}}
