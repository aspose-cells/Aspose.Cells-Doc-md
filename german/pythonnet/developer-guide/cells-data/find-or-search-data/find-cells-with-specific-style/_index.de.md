{{% alert color="primary" %}}

Manchmal müssen Sie Zellen mit einem bestimmten Stil finden. Sie können Aspose.Cells for Python via .NET verwenden, um alle Zellen mit einem gemeinsamen Stil zu finden. Aspose.Cells for Python via .NET bietet die [**FindOptions.style**](https://reference.aspose.com/cells/python-net/aspose.cells/findoptions/style/)-Eigenschaft, die Sie verwenden können, um den Stil für die Zellsuche festzulegen.

{{% /alert %}}

Der Code in diesem Beispiel sucht alle Zellen, die denselben Stil wie die Zelle A1 haben. Nachdem der Code ausgeführt wurde, enthalten alle Zellen, die denselben Stil wie A1 haben, den Text "Gefunden".
