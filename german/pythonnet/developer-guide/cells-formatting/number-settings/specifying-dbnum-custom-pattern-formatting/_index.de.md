## **Mögliche Verwendungsszenarien**

Aspose.Cells für Python via .NET unterstützt das benutzerdefinierte Muster *DBNum*. Zum Beispiel, wenn Ihr Zellwert 123 ist und Sie es mit [DBNum2][$-804]General formatieren, wird es als 壹佰贰拾叁 angezeigt. Sie können Ihre eigene Zellformatierung mit der Methode [**Cell.get_style()**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/get_style/#) und der Eigenschaft [**Style.custom**](https://reference.aspose.com/cells/python-net/aspose.cells/style/custom) festlegen.
