{{% alert color="primary" %}} 

Microsoft Excel unterstützt zwei Datensysteme: das 1900-Datumsystem (das Standard-Datumsystem in Excel für Windows) und das 1904-Datumsystem. Das 1904-Datumsystem wird normalerweise verwendet, um die Kompatibilität mit Macintosh-Excel-Dateien sicherzustellen und ist das Standard-System, wenn Sie Excel für Macintosh verwenden. Sie können das 1904-Datumsystem für Excel-Dateien mit Aspose.Cells für Python via .NET einstellen.

{{% /alert %}} 

Um das 1904-Datensystem in Microsoft Excel zu implementieren (zum Beispiel Microsoft Excel 2003):

1. Wählen Sie im **Extras**-Menü die Option **Optionen** und wählen Sie den Tab **Berechnung**.
1. Wählen Sie die Option **1904-Datensystem** aus.
1. Klicken Sie auf **OK**.
