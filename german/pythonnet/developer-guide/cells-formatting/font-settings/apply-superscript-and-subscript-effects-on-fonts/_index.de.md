Aspose.Cells für Python via .NET bietet die Funktionalität, Superscript (Text über der Grundlinie) und Subscript (Text unterhalb der Grundlinie) Effekte auf Text anzuwenden.

{{% /alert %}}

## **Arbeiten mit Überlagerungs- und Indizierungseffekten**

Wenden Sie den Überlagerungseffekt an, indem Sie die [**Style.font**](https://reference.aspose.com/cells/python-net/aspose.cells/font) des Objekts [**is_superscript**](https://reference.aspose.com/cells/python-net/aspose.cells/font/is_superscript) Eigenschaft auf **true** setzen. Für die Anwendung der Indizierung setzen Sie die [**Style.font**](https://reference.aspose.com/cells/python-net/aspose.cells/font) des Objekts [**is_subscript**](https://reference.aspose.com/cells/python-net/aspose.cells/font/is_subscript) Eigenschaft auf **true**.
