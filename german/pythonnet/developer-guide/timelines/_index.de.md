## **Mögliche Verwendungsszenarien**

Anstatt Filter anzupassen, um Daten anzuzeigen, können Sie eine PivotTable-Zeitachse verwenden - eine dynamische Filteroption, mit der Sie ganz einfach nach Datum/Uhrzeit filtern und den Zeitraum, den Sie möchten, mit einem Schieberegler steuern können. Microsoft Excel ermöglicht es Ihnen, eine Zeitachse zu erstellen, indem Sie eine Pivot-Tabelle auswählen und dann auf *Einfügen > Zeitachse* klicken. Auch Aspose.Cells für Python via .NET ermöglicht es Ihnen, eine Zeitachse mithilfe der [**Worksheet.timelines.add()**](https://reference.aspose.com/cells/python-net/aspose.cells.timelines/timelinecollection/#methods)-Methode zu erstellen.
