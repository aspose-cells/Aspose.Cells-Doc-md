## **Erstellen Sie ein Diagramm-PDF mit gewünschter Seitengröße**
Sie können ein Diagramm-PDF mit Ihrer gewünschten Seitengröße erstellen, indem Sie Aspose.Cells für Python via .NET verwenden, und festlegen, wie das Diagramm innerhalb der Seite ausgerichtet werden soll, z. B. oben, unten, zentriert, links, rechts usw. Außerdem kann das Ausgabe-Diagramm als Stream oder auf Festplatte erstellt werden. Sehen Sie sich den folgenden Beispielcode an, der die [Beispieldatei](64716906.xlsx) lädt, das erste Diagramm im Arbeitsblatt abruft und dann in ein [Ausdruck-PDF](64716907.pdf) mit gewünschter Seitengröße konvertiert. Der folgende Screenshot zeigt, dass die Seitengröße im Output-PDF 7x7 beträgt, wie im Code angegeben, und das Diagramm sowohl horizontal als auch vertikal zentriert ausgerichtet ist.
