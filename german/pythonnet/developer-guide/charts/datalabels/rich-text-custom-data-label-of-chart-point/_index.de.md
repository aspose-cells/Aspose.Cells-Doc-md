{{% alert color="primary" %}}

Sie können Aspose.Cells für Python via .NET verwenden, um reichhaltige Text-Benutzerdefinierte Datenbeschriftungen für den Diagrammpunkt zu erstellen. Aspose.Cells für Python via .NET bietet die [**DataLabels.characters()**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextframe/characters)-Methode, um das [**FontSetting**](https://reference.aspose.com/cells/python-net/aspose.cells/fontsetting)-Objekt zurückzugeben, das verwendet werden kann, um die Schriftarteigenschaften des Textes wie Farbe, Fettdruck usw. festzulegen.
