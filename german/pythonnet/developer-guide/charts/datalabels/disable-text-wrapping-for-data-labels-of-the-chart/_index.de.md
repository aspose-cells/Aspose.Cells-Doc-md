{{% alert color="primary" %}}

Microsoft Excel 2013 ermöglicht es Benutzern, Texte in den Datenbeschriftungen des Diagramms umzubrechen oder nicht umzubrechen. Standardmäßig ist der Text in den Datenbeschriftungen des Diagramms umgebrochen.

Aspose.Cells für Python via .NET bietet eine [**DataLabels.is_text_wrapped**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/datalabels/is_text_wrapped)-Eigenschaft, die Sie auf True oder False setzen können, um den Textumbruch der Datenbeschriftungen jeweils zu aktivieren oder zu deaktivieren.

{{% /alert %}}
