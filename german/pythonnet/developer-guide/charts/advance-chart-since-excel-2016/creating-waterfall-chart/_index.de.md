{{% alert color="primary" %}}

Ein Wasserfalldiagramm ist ein spezieller Diagrammtyp, der normalerweise verwendet wird, um zu zeigen, wie die Ausgangsposition entweder zunimmt oder abnimmt. Microsoft Excel verfügt über viele vordefinierte Diagrammtypen, darunter Säulen-, Linien-, Torten-, Balken-, Radar-Diagramme usw., aber das Wasserfalldiagramm geht über die grundlegenden Grafiken hinaus und kann unter Verwendung der vorhandenen Diagrammtypen mit wenig oder mehr Anpassung erstellt werden.

{{% /alert %}} 

Aspose.Cells für Python via .NET APIs ermöglichen es, mit Hilfe eines Liniendiagramms ein Wasserfalldiagramm zu erstellen. Die API erlaubt auch die Anpassung des Diagramm-Aussehens, um die Form eines Wasserfalls durch Setzen der [**Series.up_bars**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/series/up_bars)- und [**Series.down_bars**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/series/down_bars)-Eigenschaften zu erhalten.
