{{% alert color="primary" %}}

Wenn Sie den Text der Diagrammlegendeneintragsfüllung auf keine setzen möchten, sodass er nicht im Diagrammlegendenbereich angezeigt wird, setzen Sie bitte [**LegendEntry.is_text_no_fill**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/legendentry/is_text_no_fill) auf **true**.

{{% /alert %}}

Der folgende Beispielcode setzt den Text der zweiten Diagrammlegendeneintragsfüllung auf keine. Laden Sie bitte die [Beispieldatei Excel](5115219.xlsx) herunter, die in diesem Code verwendet wird, und die [Ausgabedatei Excel](5115217.xlsx), die von ihr generiert wird, zur Referenz.
