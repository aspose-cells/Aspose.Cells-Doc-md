## **Mögliche Verwendungsszenarien**

Microsoft Excel ermöglicht es, die Arbeitsmappe wie im folgenden Screenshot freizugeben. Wenn Sie die Arbeitsmappe freigeben, können mehrere Benutzer die Arbeitsmappe im Netzwerk bearbeiten. Aspose.Cells for Python via .NET ermöglicht es, eine gemeinsame Arbeitsmappe mit der [**Workbook.settings.shared**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/shared)-Eigenschaft zu erstellen.
