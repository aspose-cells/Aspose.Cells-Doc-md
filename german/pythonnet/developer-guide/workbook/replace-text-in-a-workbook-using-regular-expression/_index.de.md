Aspose.Cells for Python via .NET bietet die Funktion, Text in einem Arbeitsbuch mithilfe eines regulären Ausdrucks zu ersetzen. Dafür stellt die API die [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key)-Eigenschaft der [**ReplaceOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions)-Klasse bereit. Wenn das [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) auf **true** gesetzt ist, bedeutet dies, dass der gesuchte Schlüssel ein regulärer Ausdruck ist.

Der folgende Codeausschnitt demonstriert die Verwendung der [**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key)-Eigenschaft, indem die [Beispiel-Excel-Datei](101089318.xlsx) verwendet wird.
