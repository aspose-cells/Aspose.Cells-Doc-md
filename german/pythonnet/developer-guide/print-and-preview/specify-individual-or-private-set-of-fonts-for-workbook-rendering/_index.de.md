## **Mögliche Verwendungsszenarien**

In der Regel geben Sie den Schriftartenordner oder eine Liste von Schriftarten für alle Workbooks an, aber manchmal müssen Sie individuelle oder private Schriftart-Sets für Ihre Workbooks festlegen. Die Klasse [**IndividualFontConfigs**](https://reference.aspose.com/cells/python-net/aspose.cells/individualfontconfigs) in Aspose.Cells für Python via .NET kann verwendet werden, um individuelle oder private Schriftartsets zu spezifizieren.
