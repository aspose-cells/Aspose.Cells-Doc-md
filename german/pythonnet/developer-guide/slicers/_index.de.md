## **Mögliche Verwendungsszenarien**

Ein Slicer wird verwendet, um Daten schnell zu filtern. Es kann sowohl verwendet werden, um Daten in einer Tabelle als auch in einem Pivot-Table zu filtern. Microsoft Excel ermöglicht es Ihnen, einen Slicer zu erstellen, indem Sie eine Tabelle oder ein Pivot-Table auswählen und dann auf *Einfügen > Slicer* klicken. Aspose.Cells für Python via .NET ermöglicht es Ihnen ebenfalls, einen Slicer mithilfe der Methode [**Worksheet.slicers.add()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/add/#aspose.cells.pivot.PivotTable-int-int-aspose.cells.pivot.PivotField) zu erstellen.
