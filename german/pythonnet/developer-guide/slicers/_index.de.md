##  **Mögliche Nutzungsszenarien**

Um Daten schnell zu filtern, wird ein Slicer verwendet. Es kann zum Filtern von Daten sowohl in einer Tabelle als auch in einer Pivot-Tabelle verwendet werden. Microsoft In Excel können Sie einen Slicer erstellen, indem Sie eine Tabelle oder Pivot-Tabelle auswählen und dann auf *Einfügen > Slicer* klicken. Aspose.Cells for Python via .NET ermöglicht Ihnen auch die Erstellung eines Slicers mit dem[**Worksheet.slicers.add()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/add/#aspose.cells.pivot.PivotTable-int-int-aspose.cells.pivot.PivotField)Methode.
