## **Mögliche Verwendungsszenarien**

Wenn Sie den Slicer in Microsoft Excel entfernen möchten, wählen Sie ihn einfach aus und drücken Sie die *Löschen*-Taste. Ebenso, wenn Sie ihn mithilfe des Aspose.Cells für Python via .NET-APIs programmgesteuert entfernen möchten, verwenden Sie bitte die Methode [**Worksheet.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/). Dies wird den Slicer aus dem Arbeitsblatt entfernen.
