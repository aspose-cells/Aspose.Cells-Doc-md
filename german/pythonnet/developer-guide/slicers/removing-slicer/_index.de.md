##  **Mögliche Nutzungsszenarien**

 Wenn Sie den Slicer in Microsoft Excel entfernen möchten, wählen Sie ihn einfach aus und drücken Sie die Taste*Löschen* Taste. Wenn Sie es ebenfalls programmgesteuert mit Aspose.Cells for Python via .NET API entfernen möchten, verwenden Sie bitte die[**Worksheet.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/)Methode. Dadurch wird der Slicer aus dem Arbeitsblatt entfernt.
