## **Mögliche Verwendungsszenarien**

Wenn Sie den Slicer und die Pivot-Tabelle in Excel entkoppeln möchten, müssen Sie mit der rechten Maustaste auf den Slicer klicken und das Element "Berichtsverbindungen..." auswählen. In der Optionsliste können Sie das Kontrollkästchen bedienen. Ebenso, wenn Sie den Slicer und die Pivot-Tabelle mithilfe des Aspose.Cells für Python via .NET-APIs programmgesteuert entkoppeln möchten, verwenden Sie bitte die Methode [**Slicer.remove_pivot_connection(pivot)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/remove_pivot_connection/#aspose.cells.pivot.PivotTable). Diese wird den Slicer und die Pivot-Tabelle entkoppeln.
