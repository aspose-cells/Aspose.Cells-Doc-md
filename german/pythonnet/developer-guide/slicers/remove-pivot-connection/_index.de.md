##  **Mögliche Nutzungsszenarien**

 Wenn Sie Slicer und Pivot-Tabelle in Excel trennen möchten, müssen Sie mit der rechten Maustaste auf Slicer klicken und den Eintrag „Verbindungen melden…“ auswählen. In der Optionsliste können Sie das Kontrollkästchen bedienen. Wenn Sie Slicer und Pivot-Tabelle mithilfe von Aspose.Cells for Python via .NET API programmgesteuert trennen möchten, verwenden Sie bitte die[**Slicer.remove_pivot_connection(pivot)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/remove_pivot_connection/#aspose.cells.pivot.PivotTable)Methode. Dadurch werden Slicer und Pivot-Tabelle getrennt.
