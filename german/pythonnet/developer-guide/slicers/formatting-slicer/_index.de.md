##  **Mögliche Nutzungsszenarien**

Sie können den Slicer in Microsoft Excel formatieren, indem Sie die Anzahl der Spalten oder den Stil usw. festlegen. Aspose.Cells for Python via .NET ermöglicht Ihnen dies auch mit dem[**Slicer.number_of_columns**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/number_of_columns/)Und[**Slicer.style_type**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/style_type/)Eigenschaften.

##  **Formatierungs-Slicer**

Bitte sehen Sie sich den folgenden Code an, er lädt die[Beispiel-Excel-Datei](67338473.xlsx)das einen Slicer enthält. Es greift auf den Slicer zu, legt die Anzahl der Spalten und den Stiltyp fest und speichert ihn unter[Excel-Datei ausgeben](67338474.xlsx).
