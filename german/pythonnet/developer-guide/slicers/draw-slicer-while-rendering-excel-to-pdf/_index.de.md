##  **Zeichnen Sie Slicer beim Rendern von Excel auf PDF**
Wenn Sie eine Excel-Datei haben, auf die ein Slicer angewendet wurde, und Sie Excel mit den Slicer-Einstellungen nach PDF exportieren möchten, unterstützt Aspose.Cells for Python via .NET dies jetzt standardmäßig. Sie exportieren einfach die Excel-Datei mit Slicer nach PDF, die generierte PDF zeigt den angewendeten Slicer an.

 Der folgende Beispielcode lädt die[Beispiel-Excel-Datei](94044165.xlsx) das einen vorhandenen Slicer enthält. Anschließend wird die Arbeitsmappe als gespeichert[Ausgabedatei PDF](94044166.pdf). Der folgende Screenshot vergleicht die Excel-Quelldatei und die generierte Datei PDF.
