## **Wie man beim Rendern von Excel zu PDF mit Hilfe der Aspose.Cells für Python Excel-Bibliothek den Slicer zeichnet**
Wenn Sie eine Excel-Datei haben, in der ein Slicer angewendet wurde und Sie die Excel-Datei mit den Slicer-Einstellungen exportieren möchten, unterstützt Aspose.Cells für Python via .NET dies jetzt standardmäßig. Sie exportieren einfach die Excel-Datei mit Slicer als PDF, das generierte PDF zeigt den angewendeten Slicer.
