**Willkommen bei Aspose.Cells für Go**

Aspose.Cells for Go via C++ ist eine native Go-Bibliothek, die Go-Entwicklern ermöglicht, Tabellenprogrammgesteuert zu erstellen, zu manipulieren und zu konvertieren, ohne Office Automation oder Microsoft Excel-Anwendung zu benötigen. Aspose.Cells für Go unterstützt Formate wie MS Excel 97-2003 (XLS), MS Excel 2007-2013/2016 (XLSX, XLSM, XLSB usw.), Open Office XML und andere Formate. Sie können auch Bilder aus Tabellenblättern rendern und MS Excel-Tabellen in PDF umwandeln. Die API ermöglicht außerdem die Erstellung, Manipulation und Darstellung von Diagrammen und Formen. Außerdem ist die Formelbehandlung eine weitere gute Funktion, die das Komponenten bietet und genutzt werden kann.
