## **Hyperlinks im Bereich abrufen**

Die Klasse [**Range**](https://reference.aspose.com/cells/go-cpp/range/) bietet eine [**GetHyperlinks()**](https://reference.aspose.com/cells/go-cpp/range/gethyperlinks/)-Eigenschaft, die alle Hyperlinks im ausgewählten Bereich zurückgibt. Sie können auch den Hyperlink durch Aufruf der Methode [**Hyperlink.Delete**](https://reference.aspose.com/cells/cpp/aspose.cells/hyperlink/delete/) löschen.
