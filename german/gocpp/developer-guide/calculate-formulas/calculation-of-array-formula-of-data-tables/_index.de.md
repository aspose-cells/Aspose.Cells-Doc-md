{{% alert color="primary" %}}

Sie können in Microsoft Excel über Daten > Was-wäre-wenn-Analyse > Datentabelle... eine Datentabelle erstellen. Aspose.Cells ermöglicht es Ihnen jetzt, die Array-Formel einer Datentabelle zu berechnen. Bitte verwenden Sie [**Workbook.CalculateFormula()**](https://reference.aspose.com/cells/go-cpp/workbook/calculateformula/) wie normal, um jegliche Art von Formeln zu berechnen.

{{% /alert %}}
