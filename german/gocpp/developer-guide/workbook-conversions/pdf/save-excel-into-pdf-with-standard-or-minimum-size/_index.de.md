{{% alert color="primary" %}} 

Standardmäßig speichert Aspose.Cells Excel in PDF mit Standardgröße. Sie können es jedoch auch mit minimaler Größe speichern, indem Sie die Eigenschaft [PdfSaveOptions.GetOptimizationType()](https://reference.aspose.com/cells/go-cpp/pdfsaveoptions/getoptimizationtype/) verwenden. Sie akzeptiert die folgenden Werte:

- PdfOptimizationType::Standard
- PdfOptimizationType::MinimumSize

{{% /alert %}}
