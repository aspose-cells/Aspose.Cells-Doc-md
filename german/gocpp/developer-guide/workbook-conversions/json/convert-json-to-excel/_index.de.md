## **Konvertieren Sie JSON in Excel-Arbeitsmappe** 
Es ist nicht nötig zu überlegen, wie JSON in eine Excel-Datei umgewandelt wird, denn die Bibliothek Aspose.Cells for C++ bietet die beste Lösung. Das Aspose.Cells API unterstützt die Konvertierung des JSON-Formats in Tabellenkalkulationen. Sie können die Klasse [JsonLoadOptions](https://reference.aspose.com/cells/go-cpp/jsonloadoptions/) verwenden, um zusätzliche Einstellungen für den Import von JSON in die Arbeitsmappe festzulegen.
