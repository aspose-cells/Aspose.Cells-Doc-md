## **Mögliche Verwendungsszenarien**

Beim Speichern Ihrer Excel-Datei als HTML können Sie verschiedene Kreuztypen für Zelltexte angeben. Standardmäßig generiert Aspose.Cells HTML entsprechend Microsoft Excel. Wenn Sie den Kreuztyp auf [**CrossHideRight**](https://reference.aspose.com/cells/go-cpp/htmlcrosstype) ändern, werden alle Textüberlagerungen oder Überlappungen rechts im Zelltext verborgen.
