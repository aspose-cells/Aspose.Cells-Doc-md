{{% alert color="primary" %}} 

Normalerweise wird das Layout der DIV-Tags beim Laden von HTML in ein Excel-Arbeitsmappenobjekt ignoriert. Möchtest du jedoch das Layout der DIV-Tags beibehalten, stelle die Eigenschaft [GetSupportDivTag()](https://reference.aspose.com/cells/go-cpp/htmlloadoptions/getsupportdivtag/) auf **true**. Der Standardwert dieser Eigenschaft ist **false**.

{{% /alert %}}
