{{% alert color="primary" %}} 

Wenn Sie Ihre Excel-Datei mit GridLines nach HTML exportieren möchten, verwenden Sie bitte die [HtmlSaveOptions.GetExportGridLines()](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/getexportgridlines/) Eigenschaft und setzen Sie sie auf **true**.

{{% /alert %}}
