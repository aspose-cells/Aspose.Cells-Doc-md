## **Mögliche Verwendungsszenarien**

Wenn Sie Ihre Excel-Datei in HTML speichern, können Sie bei mehreren Vorkommen eines Base64-Bildes mit Kundeneigenschaften die Bilddaten nur einmal speichern, um die Leistung des resultierenden HTML zu verbessern. Bitte verwenden Sie [**HtmlSaveOptions.GetEnableCssCustomProperties()**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/getenablecsscustomproperties/) Property und setzen Sie es auf **true** beim Speichern als HTML.
