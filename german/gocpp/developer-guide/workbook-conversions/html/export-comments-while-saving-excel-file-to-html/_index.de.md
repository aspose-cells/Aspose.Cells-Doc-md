## **Mögliche Verwendungsszenarien**

Wenn Sie Ihre Excel-Datei in HTML speichern, werden Kommentare nicht exportiert. Aspose.Cells bietet diese Funktion jedoch mit der [**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/isexportcomments/)-Eigenschaft. Wenn Sie sie auf **true** setzen, werden die Kommentare in Ihrem Excel in HTML angezeigt.
