## **HTML in PDF-Konvertierung in C++**
Wie konvertiert man HTML in PDF? Mit der [Aspose.Cells for C++](https://releases.aspose.com/cells/go-cpp/) Bibliothek können Sie HTML ganz einfach programmgesteuert in PDF umwandeln, mit nur wenigen Zeilen Code. Aspose.Cells for C++ ist in der Lage, plattformübergreifende Anwendungen zu erstellen und alle Excel-Dateien zu generieren, zu modifizieren, zu konvertieren, anzuzeigen und zu drucken.

## **C++ HTML in PDF konvertieren**
Das folgende C++-Codebeispiel zeigt, wie man ein HTML-Dokument in ein PDF mit [Aspose.Cells for C++](https://releases.aspose.com/cells/go-cpp/) konvertiert.

1. Erstellen Sie eine Instanz der [HtmlLoadOptions](https://reference.aspose.com/cells/go-cpp/htmlloadoptions/) Klasse.
1. Initialisieren Sie das [Workbook](https://reference.aspose.com/cells/go-cpp/workbook/) Objekt.
1. Speichern Sie das Ausgabe-PDF-Dokument, indem Sie die Methode Workbook.Save() aufrufen.
