{{% alert color="primary" %}}

Microsoft Excel ermöglicht es, das Layout der Pivot-Tabelle mit den Befehlen *PivotTable Tools > Design > Report Layout* zu ändern. Sie können das Layout in diesen drei Formen ändern:

- Im kompakten Formular anzeigen
- Im Gliederungsformular anzeigen
- Im tabellarischen Formular anzeigen

Aspose.Cells bietet auch die Methoden [**PivotTable.ShowInCompactForm()**](https://reference.aspose.com/cells/go-cpp/pivottable/showincompactform/), [**PivotTable.ShowInOutlineForm()**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/showinoutlineform/) und [**PivotTable.ShowInTabularForm()**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/showintabularform/), um das Layout der Pivot-Tabelle in diesen drei Formen zu ändern.
