Aspose.Cells bietet die Möglichkeit, Pivot-Tabellen im ODS-Format zu speichern. Dafür können Sie entweder eine Arbeitsmappe mit einer bestehenden Pivot-Tabelle konvertieren oder eine neue Pivot-Tabelle erstellen und die Datei im ODS-Format speichern. Stellen Sie sicher, dass Sie [**PivotTable::CalculateData()**](https://reference.aspose.com/cells/go-cpp/pivottable/calculatedata/) vor dem Speichern aufrufen, um sicherzustellen, dass die Pivot-Tabelle im Ausgabe-ODS-Datei gerendert wird.
