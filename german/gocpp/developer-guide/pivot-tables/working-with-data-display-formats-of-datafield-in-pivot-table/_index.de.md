## **"Rank Kleinste bis Größte" und "Rank Größte bis Kleinste" Anzeigeformatoption**

Aspose.Cells bietet die Möglichkeit, das Anzeigeformat für Pivot-Felder festzulegen. Hierfür stellt die API die [**PivotField.GetCalculationType()**](https://reference.aspose.com/cells/go-cpp/pivotshowvaluessetting/getcalculationtype/)-Eigenschaft bereit. Um nach Größe zu ranken, können Sie die [**PivotField.GetCalculationType()**](https://reference.aspose.com/cells/go-cpp/pivotshowvaluessetting/getcalculationtype/)-Eigenschaft auf [**PivotFieldDataDisplayFormat.RankLargestToSmallest**](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivotfielddatadisplayformat/) setzen. Das folgende Code-Beispiel zeigt, wie die Anzeigeformatoptionen eingestellt werden.
