## **Mögliche Verwendungsszenarien**

Numbers ist eine Tabellenkalkulationsanwendung, die von Apple Inc. entwickelt wurde. Aspose.Cells kann eine Numbers-Datei lesen, aber nicht darin schreiben. Es kann die Daten, Formatierungen und Formeln der Numbers-Tabelle lesen.

## **Zahlen-Tabellenkalkulation von Apple Inc. mit Aspose.Cells lesen**

Der folgende Beispielcode lädt die [Beispieltabelle Numbers von Apple](sampleNumbersByAppleInc.numbers) und konvertiert sie in [Ausgabe PDF-Format](outputNumbersByAppleInc.pdf). Sie müssen die **[LoadOptions](https://reference.aspose.com/cells/go-cpp/loadoptions/)** Klasse verwenden und **[LoadFormat::Numbers](https://reference.aspose.com/cells/go-cpp/loadformat/)** als Parameter im Konstruktor angeben, um sie erfolgreich zu laden. Laden Sie beide Dateien von den angegebenen Links herunter. Sie können den Code mit jeder Numbers-Tabelle versuchen. Lesen Sie auch die Kommentare im Code für weitere Hilfe.
