{{% alert color="primary" %}}

Wenn Sie den Text der Diagrammlegendeneintragsfüllung auf keine setzen möchten, sodass er nicht im Diagrammlegendenbereich angezeigt wird, setzen Sie bitte [**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/go-cpp/legendentry/istextnofill/) auf **true**.
