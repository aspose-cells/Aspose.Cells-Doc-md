{{% alert color="primary" %}}

Sie können überprüfen, ob Ihr VBA-Projekt mit Microsoft Excel signiert ist, über das Menü **Tools > Digitale Signaturen...**. Ebenso können Sie es programmatisch mit Aspose.Cells [**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/go-cpp/vbaproject/issigned/) überprüfen.

{{% /alert %}}
