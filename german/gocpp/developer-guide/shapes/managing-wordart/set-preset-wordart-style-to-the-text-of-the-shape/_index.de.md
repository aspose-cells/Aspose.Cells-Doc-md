## **Mögliche Verwendungsszenarien**
Sie können den voreingestellten WordArt-Stil auf den Text der Form mit Aspose.Cells setzen. Bitte verwenden Sie [FontSetting.SetWordArtStyle()](https://reference.aspose.com/cells/go-cpp/fontsetting/setwordartstyle/) oder [FontSettingCollection.SetWordArtStyle()](https://reference.aspose.com/cells/cpp/aspose.cells.drawing.texts/fontsettingcollection/setwordartstyle/) für diesen Zweck
