## **Mögliche Verwendungsszenarien**

Sie können die Sprache einer Excel-Datei ändern, indem Sie mit der rechten Maustaste auf die Datei klicken, Eigenschaften > Details auswählen und dann das Sprachfeld bearbeiten. Verwenden Sie [**BuiltInDocumentPropertyCollection.GetLanguage()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getlanguage/) Eigenschaft, um sie programmgesteuert mit den APIs von Aspose.Cells zu ändern.
