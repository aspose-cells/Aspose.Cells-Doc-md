## **Mögliche Verwendungsszenarien**

Sie können die **Versionsnummer** einer Excel-Datei ändern, indem Sie mit der rechten Maustaste auf die Datei klicken, Eigenschaften > Details auswählen und dann das Feld **Versionsnummer** bearbeiten. Verwenden Sie [**BuiltInDocumentPropertyCollection.GetDocumentVersion()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getdocumentversion/) Eigenschaft, um sie programmgesteuert mit den APIs von Aspose.Cells zu ändern.
