## **Mögliche Verwendungsszenarien**  

Microsoft Excel ermöglicht das Teilen des Arbeitsbuchs, wie im folgenden Screenshot gezeigt. Wenn Sie das Arbeitsbuch teilen, können mehrere Benutzer auf das Netzwerk zugreifen und es bearbeiten. Aspose.Cells ermöglicht es Ihnen, mit der [**Workbook.GetShared()**](https://reference.aspose.com/cells/go-cpp/workbooksettings/getshared/)-Eigenschaft ein geteiltes Arbeitsbuch zu erstellen.
