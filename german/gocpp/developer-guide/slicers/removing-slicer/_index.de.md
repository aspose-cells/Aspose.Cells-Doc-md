## **Mögliche Verwendungsszenarien**

Wenn Sie einen Slicer in Microsoft Excel entfernen möchten, wählen Sie ihn einfach aus und drücken Sie die *Entfernen*-Taste. Ebenso, wenn Sie ihn programmatisch mit der API von Aspose.Cells entfernen möchten, verwenden Sie die [**Worksheet.Slicers.Remove()**](https://reference.aspose.com/cells/go-cpp/slicercollection/remove/)-Methode. Dadurch wird der Slicer aus dem Arbeitsblatt entfernt.
