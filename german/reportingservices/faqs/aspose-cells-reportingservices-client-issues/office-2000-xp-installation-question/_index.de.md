##### **Beschreibung**
Wenn Microsoft Excel 2002 oder Microsoft Excel 2000 Aspose.Cells.ReportingServices.Client-Add-Ins lädt, erzeugt Microsoft Excel einen Automatisierungsfehler.
##### **Lösung**
{{% alert color="primary" %}} 

 Installieren**lockbackRegKey.msi**. Dadurch wird das CLR-Problem auf Office-Computern behoben, die älter als Office 2003 sind.

{{% /alert %}}
