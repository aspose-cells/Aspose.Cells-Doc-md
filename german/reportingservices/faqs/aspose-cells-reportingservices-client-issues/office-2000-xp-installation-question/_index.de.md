##### **Beschreibung**
Wenn Microsoft Excel 2002 oder Microsoft Excel 2000 Aspose.Cells.ReportingServices.Client-Add-Ins lädt, tritt ein Automatisierungsfehler auf.
##### **Lösung**
{{% alert color="primary" %}} 

Installieren Sie **lockbackRegKey.msi**. Dies behebt das CLR-Problem auf Bürorechnern älter als Office 2003.

{{% /alert %}}
