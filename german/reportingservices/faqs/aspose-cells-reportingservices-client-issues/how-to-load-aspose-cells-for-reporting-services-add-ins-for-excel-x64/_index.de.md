Nach der Installation wird Aspose.Cells for Reporting Services standardmäßig als 32-Bit-Komponente registriert. Es kann Excel-Add-Ins in Microsoft Excel x64 nicht korrekt laden. Daher müssen Sie die Aspose.Cells.ReportingServices.dll als 64-Bit-Komponente neu registrieren.
