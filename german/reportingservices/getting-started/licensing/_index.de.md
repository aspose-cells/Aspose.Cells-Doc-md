{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services ist als kostenlose, zeitlich unbegrenzte Testversion von der [Download-Seite](https://downloads.aspose.com/cells/reportingservices) verfügbar. Die Test- und lizenzierten Versionen des Produkts sind identisch.

Wenn Sie mit der Testversion zufrieden sind, kaufen Sie eine Lizenz. Stellen Sie sicher, dass Sie die Abonnementbedingungen verstehen und akzeptieren.

{{% /alert %}} 
### **Erhalt und Verwendung der Lizenz**
Die Lizenz steht zum Download auf der Bestellseite zur Verfügung, sobald die Bestellung bezahlt ist.

Die Lizenzdatei ist eine klartextliche, digital signierte XML-Datei. Die Lizenzdatei enthält Informationen wie den Clientnamen, das gekaufte Produkt und den Lizenztyp. Ändern Sie den Inhalt der Lizenzdatei nicht, da dies sie ungültig macht.

Laden Sie die Lizenz auf Ihren Computer herunter und kopieren Sie sie in den Order, der einer der folgenden Orte sein muss.

- $[Installationsordner]\Lizenz\
- $[MSSQL Server Reporting Services]\ReportServer\Bin\
- $[MSSQL Server Reporting Services]\ReportServer\

Der Name der Lizenzdatei kann je nach Ihrem Abonnementtyp einer der folgenden sein.

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic", "Aspose.Total.lic
- Aspose.Cells.Reporting.Services - Developer Site Subscription.lic
- Aspose.Cells.Reporting.Services - Developer Subscription.lic
- Aspose.Cells.Reporting.Services - Mehrfach-Bereitstellungsabonnement.lic
- Aspose.Cells.Reporting.Services - Einzelserver-Bereitstellungsabonnement.lic

Um zu testen, ob die Lizenzdatei korrekt installiert wurde, exportieren Sie einen Bericht als Microsoft Excel-Arbeitsmappe. Wenn die Arbeitsmappe am Ende kein Wasserzeichen-Arbeitsblatt enthält, wurde die Lizenzdatei erfolgreich aktiviert.

Aspose.Cells for Reporting Services fügt ein Evaluierungswasserzeichen-Arbeitsblatt ein, wenn im Evaluierungsmodus gearbeitet wird. Wenn eine gültige Lizenzdatei vorhanden ist, beispielsweise wenn **Aspose.Cells.ReportingServices.lic** im $[Installationsordner]\Lizenz\-Ordner vorhanden ist, gibt es kein Evaluierungswasserzeichen-Arbeitsblatt.
