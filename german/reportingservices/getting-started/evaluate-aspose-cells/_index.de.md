{{% alert color="primary" %}}

Sie können Aspose.Cells for Reporting Services leicht zum Testen von der [Download-Seite](https://downloads.aspose.com/cells/reportingservices) herunterladen. Der Evaluierungsdownload entspricht dem Kaufdownload. Die Evaluierungsversion wird einfach lizenziert, nachdem eine [Lizenz angewendet](/cells/de/reportingservices/licensing/) wurde.

{{% /alert %}}

Die Evaluierungsversion von Aspose.Cells for Reporting Services (ohne spezifizierte Lizenz) bietet die Produktfunktionalität in gewisser Hinsicht, jedoch nicht vollständig. Eine Einschränkung besteht darin, dass sie jedes Arbeitsblattdokument beim Öffnen/Speichern immer mit einem zusätzlichen Arbeitsblatt mit Evaluierungswasserzeichen einfügt.

![todo:image_alt_text](evaluate-aspose-cells_2.png)

Wenn Sie Aspose.Cells for Reporting Services ohne Evaluierungsversionseinschränkungen testen möchten, können Sie eine [30-tägige temporäre Lizenz](https://purchase.aspose.com/temporary-license) anfordern.
