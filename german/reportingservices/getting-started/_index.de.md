Aspose.Cells for Reporting Services besteht aus **zwei Teilen** wie folgt:

- ***Aspose.Cells.Report.Designer***

&nbsp;&nbsp;&nbsp;&nbsp;Dies ist ein Berichtsdesigner-Tool, das in MS Excel im Add-Ins... Formular integriert ist. Mit Aspose.Cells.Report.Designer können Benutzer eine Berichtsvorlage in MS Excel anstelle von Visual Studio entwerfen. Die Berichtsvorlage wird dann in die Reporting Services RDL-Datei eingebettet und kann auf dem Berichtsserver veröffentlicht werden. Benutzer können den Bericht auch in Aspose.Cells.Report.Designer anzeigen und exportieren.

- ***Aspose.Cells.Renderer for Reporting Services***

&nbsp;&nbsp;&nbsp;&nbsp;Aspose.Cells.Renderer für Reporting Services ist tatsächlich eine Implementierung der Microsoft Reporting Services Rendering Extension. Es kann RDL-Berichte rendern, die sowohl von Visual Studio als auch von Aspose.Cells.Report.Designer entworfen wurden.
