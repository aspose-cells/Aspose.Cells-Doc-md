{{% alert color="primary" %}} 

Es gibt zwei Arten von Formeln: 

1. Reporting Services-Formeln und
1. die von Aspose.Cells für Reporting Services bereitgestellte dynamische Microsoft Excel-Formel.

Die Verwendung von Reporting Services-Formeln in Aspose.Cells.Report.Designer entspricht dem Anwenden von Formeln in Visual Studio. Sie können Reporting Services-Formeln basierend auf den Werten von Zellen in einer Berichtsvorlage festlegen. Die Formeln werden berechnet, und die Berechnungsergebnisse werden zur Renderzeit zurück in die Zellen eingefüllt.

Dynamische Microsoft Excel-Formeln werden nicht berechnet, sondern gemäß bestimmter Regeln und Vorschriften in echte Microsoft Excel-Formeln umgewandelt, bevor sie zur Renderzeit zurück in die Zellen eingefügt werden.
