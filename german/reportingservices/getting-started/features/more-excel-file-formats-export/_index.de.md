{{% alert color="primary" %}} 

Aspose.Cells für Reporting Services ermöglicht Ihnen, native Microsoft Excel-Berichte über Microsoft SQL Server 2005 und 2008 Reporting Services bereitzustellen. Anders als der integrierte Renderer für Microsoft Excel-basierte Berichte bietet Aspose.Cells für Reporting Services einen erweiterten Funktionssatz, der RDL-Berichte konvertiert in: Excel 97-2003 binäres XLS-Format, CSV, SpreadsheetML, Excel 2007 XLSX , Excel 2007 XLSM, Excel 2007 XLSB-, HTML- und ODS-Format.

{{% /alert %}} 

Aspose.Cells für Reporting Services unterstützt mehr Dateiformate als die nativen Exportprogramme. Unten sehen Sie Screenshots von XLS-Beispielberichten, die mit Aspose.Cells exportiert wurden.
