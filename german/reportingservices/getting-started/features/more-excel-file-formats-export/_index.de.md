{{% alert color="primary" %}} 

Mit Aspose.Cells for Reporting Services können Sie native Microsoft Excel-Berichte über Microsoft SQL Server 2005 und 2008 Reporting Services bereitstellen. Anders als der integrierte Renderer für Microsoft Excel-basierte Berichte bietet Aspose.Cells for Reporting Services einen erweiterten Funktionssatz, der RDL-Berichte konvertiert in: Excel 97-2003 binäres XLS-Format, CSV, SpreadsheetML, Excel 2007 XLSX , Excel 2007 XLSM, Excel 2007 XLSB, HTML- und ODS-Format.

{{% /alert %}} 

Aspose.Cells for Reporting Services unterstützt mehr Dateiformate als die nativen Exporter. Unten sehen Sie Screenshots von XLS-Beispielberichten, die mit Aspose.Cells exportiert wurden.
