{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services ermöglicht es Ihnen, native Microsoft Excel-Berichte durch Microsoft SQL Server 2005 und 2008 Reporting Services bereitzustellen. Im Gegensatz zum integrierten Renderer für auf Microsoft Excel basierende Berichte bietet Aspose.Cells for Reporting Services einen erweiterten Funktionsumfang, der RDL-Berichte in folgende Formate konvertiert: Excel 97-2003 binäres XLS-Format, CSV, SpreadsheetML, Excel 2007 XLSX, Excel 2007 XLSM, Excel 2007 XLSB, HTML und ODS-Format. 

{{% /alert %}} 

Aspose.Cells for Reporting Services unterstützt mehr Dateiformate als die nativen Exporteure. Unten befinden sich Screenshots von Beispielen von XLS-Berichten, die mit Aspose.Cells exportiert wurden.
