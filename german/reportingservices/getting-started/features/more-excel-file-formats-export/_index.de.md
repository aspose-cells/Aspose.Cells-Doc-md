{{% alert color="primary" %}} 

Mit Aspose.Cells for Reporting Services können Sie native Microsoft Excel-Berichte über Microsoft SQL Server 2005 und 2008 Reporting Services bereitstellen. Im Gegensatz zum integrierten Renderer für Microsoft EXCEL-basierte Berichte liefert Aspose.Cells for Reporting Services einen erweiterten Merkmal, der RDL-Berichte in: Excel 97-2003 Binary XLS, CSV, XLS, EXCEL 2007, CSV, Excel 20SpreadsheetML, EXCEL 2007 076111111111193481, CONUDRET. HTML und ODS Format.

{{% /alert %}} 

Aspose.Cells for Reporting Services unterstützt mehr Dateiformate als die nativen Exporter. Unten sehen Sie Screenshots von XLS-Beispielberichten, die mit Aspose.Cells exportiert wurden.
