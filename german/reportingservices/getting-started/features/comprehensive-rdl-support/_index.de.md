{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services unterstützt die RDL-Spezifikation vollständig. Dies bedeutet, dass Sie vorhandene RDL-Berichte problemlos zu einer Aspose-Lösung portieren können.

{{% /alert %}} 
**Unterstützte RDL-Elemente**
Aspose.Cells unterstützt die folgenden RDL-Elemente: 

- Seite, Kopfzeilen, Fußzeilen
- Textfelder
- Bilder
- Diagramme
- Listen
- Tabellen
- Matrizen
- Stile
- Rechtecke, Linien
- Unterrichtsberichte
- Tablix
- Messgerät
