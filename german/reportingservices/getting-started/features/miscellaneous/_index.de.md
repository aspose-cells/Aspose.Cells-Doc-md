{{% alert color="primary" %}} 

Einige effektive Eigenschaften von Aspose.Cells for Reporting Services sind unten aufgeführt

- Aspose.Cells for Reporting Services unterstützt Tabellen mit gruppierten Zeilen und Fußzeilen. Die Gruppierung von Zeilen und Fußzeilen kann zur Entwurfszeit wie andere Berichtselemente eingerichtet werden.
- Aspose.Cells for Reporting Services unterstützt Parameter und globale Variablen als Datenquellen des Berichts.
- Benutzer können Microsoft Query verwenden, um Datenquellen und Abfragen zu erstellen. Microsoft Query ist ein weit verbreitetes Tool, das die meisten Benutzer kennen und mit dem sie vertraut sind.
- Aspose.Cells for Reporting Services unterstützt sowohl berichtsspezifische Datenquellen als auch freigegebene Datenquellen, die auf dem Berichtsserver im Voraus erstellt wurden. Darüber hinaus ist es möglich, berichtsspezifische Datenquellen zur Entwurfszeit zu verwenden und beim Veröffentlichen eines Berichts freigegebene Datenquellen als Berichtsdatenquellen anzugeben.
- Aspose.Cells for Reporting Service ist nahtlos in Microsoft Reporting Services integriert, sodass Benutzer voll von Reporting Services' Berichtsverwaltung, Sicherheit und Zustellungsoptionen profitieren können.

{{% /alert %}}
