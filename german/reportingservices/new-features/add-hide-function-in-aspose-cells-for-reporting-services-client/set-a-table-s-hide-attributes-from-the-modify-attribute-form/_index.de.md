Setzen Sie die Hide-Attribute eines Tabellenberichts aus dem **Attribut ändern**. Die Parameter beinhalten:

- **Typ** repräsentiert den zu verbergenden Objekttyp. Dies beinhaltet Tabelle, Zeile und Spalte.
- **Index** repräsentiert die Position des zu verbergenden Objekts. Wenn der Typ Arbeitsblatt ist, ist der Wert null. Der Indexwert beginnt bei 0.
- **Ausdruck** repräsentiert den Ausblendedkeit-Wert. Der Wert ist true, false oder eine Excel-Formel. Der Ausdruck unterstützt Reporting Services-Formeln.
