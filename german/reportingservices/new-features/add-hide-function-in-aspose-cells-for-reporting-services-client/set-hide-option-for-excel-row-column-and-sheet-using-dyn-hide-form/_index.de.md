Aspose.Cells for Reporting Services blendet alle angegebenen Zeilen, Spalten und Blätter basierend auf den versteckten Parametern aus. Zu den Parametern gehört:

- **Blattname** repräsentiert ein Arbeitsblatt, das ein zu verbergendes Objekt spezifiziert.
- **Typ** repräsentiert den zu verbergenden Objekttyp. Dies beinhaltet Arbeitsblatt, Zeile und Spalte.
- **Zeilen-/Spaltenindex** repräsentiert das spezifizierte zu verbergende Objekt. Wenn der Typ Arbeitsblatt ist, ist der Wert null. Der Spaltenindex unterstützt Zeichen. Der Indexwert beginnt bei 1.
- **Verbergungsstatus** repräsentiert den Ausblendungsstatus-Wert. Der Wert ist true, false oder eine Excel-Formel. Der Parameter unterstützt nur Berichtsparameter von Reporting Services.
