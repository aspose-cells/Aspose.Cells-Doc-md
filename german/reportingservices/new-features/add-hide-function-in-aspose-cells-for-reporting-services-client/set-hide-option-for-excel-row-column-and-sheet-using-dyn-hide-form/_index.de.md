Aspose.Cells for Reporting Services blendet alle angegebenen zeilen-, spalten- und blattbasierten Sätze in den Hide-Parametern aus. Zu den Parametern gehören:

- **Blattname** stellt ein Arbeitsblatt dar, das ein verborgenes Objekt angibt.
- **Typ** stellt den Objekttyp „Ausblenden“ dar. Es enthält Arbeitsblatt, Zeile und Spalte.
- **Zeilen-/Spaltenindex** stellt das angegebene Hide-Objekt dar. Wenn der Typ Arbeitsblatt ist, ist der Wert null. Spaltenindex unterstützt Zeichen. Der Indexwert beginnt bei 1.
- **Staat ausblenden** stellt den Zustandswert für das Ausblenden dar. Der Wert ist wahr, falsch oder eine Excel-Formel. Der Parameter unterstützt nur Reporting Services-Berichtsparameter.
