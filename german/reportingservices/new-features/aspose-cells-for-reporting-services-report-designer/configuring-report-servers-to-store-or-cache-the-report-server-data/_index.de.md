Der Aspose.Cells for Reporting Services Report Designer speichert die aktuelle aufgerufene Berichtsserver-URL. Wenn Sie den Dialogfeld für den Berichtsserver-Manager im Aspose.Cells Report Designer erneut öffnen, lädt das Dialogformular automatisch die gespeicherten Berichtsserverinformationen.
