Der Berichts-Designer Aspose.Cells für Reporting Services speichert die aktuell aufgerufene Berichtsserver-URL. Wenn Sie den Report Server Manager-Dialog im Aspose.Cells Report Designer erneut öffnen, lädt das Dialogformular automatisch die gespeicherten Report Server-Informationen.
