Aspose.Cells for Reporting Services unterstützt freigegebene Datensätze von SSRS 2008R2. Wenn der Aspose.Cells Report Designer eine Berichtsdefinitionsdatei öffnet, die freigegebene Datensätze enthält, kann er automatisch Datensatzinformationen abrufen, die Felder vom Berichtsserver enthalten.
