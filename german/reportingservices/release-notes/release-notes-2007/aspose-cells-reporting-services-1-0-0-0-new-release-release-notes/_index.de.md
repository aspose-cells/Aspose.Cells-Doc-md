 Einführung

Aspose.Cells für Reporting Services ermöglicht Ihnen, native Excel-Berichte in Microsoft SQL Server 2005 Reporting Services bereitzustellen. Anders als der integrierte Renderer für Excel-basierte Berichte bietet Aspose.Cells für Reporting Services einen erweiterten Funktionsumfang und ermöglicht die Konvertierung von RDL-Berichten in: Excel 97-2003 binäres XLS-Format, SpreadsheetML oder Excel2007 XLSX-Format.
