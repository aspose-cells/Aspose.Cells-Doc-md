 Diese Seite enthält Versionshinweise für[Aspose.Cells für Reporting Services 1.5.2](https://downloads.aspose.com/cells/reportingservices/new-releases/aspose.cells-for-reporting-services-1.5.2/)

 Wir freuen uns, Aspose.Cells für Reporting Services 1.5.2 ankündigen zu können!

 Was hat sich geändert?



-  Optimiert das Rendering für große Datenberichte.
-  Behebt ein Installationsproblem für Windows 7.
-  Behebt das Paging für das Tablix-Berichtselement.
-  Behandelt das Rendern von mehr Gruppen für Tablix.
-  Reduziert die Speichernutzungsrate.
