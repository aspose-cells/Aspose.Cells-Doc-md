{{% alert color="primary" %}} 

Wenn Sie einen Bericht mit allen gewünschten Elementen entworfen haben, ist der nächste Schritt, ihn auf den Berichtsserver zu veröffentlichen, sodass Sie und Ihre Kollegen ihn ausführen können.

{{% /alert %}} 

So veröffentlichen Sie einen Bericht auf dem Berichtsserver. Befolgen Sie die folgenden Schritte:

1. Klicken Sie auf **Veröffentlichen** 

![todo:image_alt_text](publishing-report_1.png)

in der Aspose.Cells.Report.Designer-Symbolleiste und klicken Sie auf die **Ja**-Schaltfläche, um den Bericht zu speichern. 

![todo:image_alt_text](publishing-report_2.png)




1. Geben Sie im Dialogfeld die URL des Berichtsservers ein und klicken Sie auf die **Aktualisieren**-Schaltfläche.

![todo:image_alt_text](publishing-report_3.png)

1. Wählen Sie einen Zielordner aus und geben Sie den Berichtsnamen ein. Klicken Sie auf die Schaltfläche **DataSource Setup**, um Datenquellen einzurichten. 

![todo:image_alt_text](publishing-report_4.png)



Der Dialog zur Datenquelleneinrichtung wird angezeigt.

1. Geben Sie im Dialogfeld die Informationen zur Datenquelle ein. Klicken Sie auf **Auswählen**, um die freigegebenen Datenquellen auszuwählen. (Wenn Sie diese benötigen, sollten die freigegebenen Datenquellen die Hauptdatenquelle des Berichts sein). Klicken Sie auf die Schaltfläche **OK**, um die Arbeit abzuschließen.

![todo:image_alt_text](publishing-report_5.png)

1. Klicken Sie auf **Veröffentlichen**, um den Bericht auf den Report Server zu veröffentlichen. 

![todo:image_alt_text](publishing-report_6.png)



Schaltfläche, um die Arbeit abzuschließen.
