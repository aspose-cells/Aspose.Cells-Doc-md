1. Öffnen Sie Microsoft Excel.
1.  Klicken**Datensatz erstellen**auf der Symbolleiste Aspose.Cells.Report.Designer (

![todo: Bild_alt_Text](mapping-query-parameters-to-report-parameters_1.png)

).

1.  Wählen Sie beispielsweise einen Datensatz im linken Bereich aus**EmpSalesDetail**. 

   **Auswählen eines Datensatzes** 

![todo: Bild_alt_Text](mapping-query-parameters-to-report-parameters_2.png)




1. Klicken Sie im rechten Bereich auf die Registerkarte Parameter, um die Parameter des Datensatzes aufzulisten.
1.  Drücke den**Gebundener Berichtsparameter** Säule.
 Alle vorab erstellten Berichtsparameter werden aufgelistet.
1.  Wählen Sie einen Parameter aus.

   **Zuordnungsparameter** 

![todo: Bild_alt_Text](mapping-query-parameters-to-report-parameters_3.png)




1.  Wiederholen Sie diesen Vorgang, um alle Abfrageparameter den Berichtsparametern zuzuordnen.

   **Kartierung abgeschlossen** 

![todo: Bild_alt_Text](mapping-query-parameters-to-report-parameters_4.png)

1.  Klicken**Nahe**.
