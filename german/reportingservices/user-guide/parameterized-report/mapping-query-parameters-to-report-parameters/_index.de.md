1. Öffnen Sie Microsoft Excel.
1. Klicken Sie auf **Datensatz erstellen** auf der Aspose.Cells.Report.Designer-Symbolleiste (

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_1.png)

).

1. Wählen Sie einen Datensatz im linken Bereich aus, z. B. **EmpSalesDetail**. 

   **Auswählen eines Datensatzes** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_2.png)




1. Klicken Sie auf die Registerkarte Parameter im rechten Bereich, um die Parameter des Datensatzes aufzulisten.
1. Klicken Sie auf die Spalte **Gebundener Berichtsparameter**.
   Alle im Voraus erstellten Berichtsparameter werden aufgelistet. 
1. Wählen Sie einen Parameter aus. 

   **Parameter zuordnen** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_3.png)




1. Wiederholen Sie die Zuordnung aller Abfrageparameter zu Berichtsparametern. 

   **Zuordnung abgeschlossen** 

![todo:image_alt_text](mapping-query-parameters-to-report-parameters_4.png)

1. Klicken Sie auf **Schließen**.
