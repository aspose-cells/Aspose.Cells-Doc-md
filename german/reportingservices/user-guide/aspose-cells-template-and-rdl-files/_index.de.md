{{% alert color="primary" %}} 

 Eine Berichtsdefinition enthält Datenabruf- und Layoutinformationen für einen Bericht. Report Definition Language (RDL) ist eine XML-Darstellung für die Berichtsdefinition. RDL ist ein offenes Schema: Entwickler können RDL mit zusätzlichen Attributen und Elementen erweitern. Die Standarddateierweiterung für Berichtsdefinitionsdateien ist .rdl. Sein MIME-Typ ist text/xml.

Eine Aspose.Cells-Vorlage ist eine Microsoft-Excel-Arbeitsmappe, die Informationen zum Berichtslayout enthält und in RDL-Dateien (Format) gespeichert wird, wenn die Vorlagenentwurfsarbeit abgeschlossen ist. Wenn eine RDL-Datei geöffnet wird, die keine Aspose.Cells-Vorlage enthält, stehen nur ihre Datenquellen und Abfragen zur Verwendung in Aspose.Cells.Report.Designer zur Verfügung. Wenn eine RDL-Datei geöffnet wird, die eine Aspose.Cells-Vorlage enthält, wird die Vorlage abgerufen und als Arbeitsmappe in Microsoft Excel geöffnet.
