{{% alert color="primary" %}} 

Eine Berichtsdefinition enthält Datenabruf und Layoutinformationen für einen Bericht. Die Report Definition Language (RDL) ist eine XML-Repräsentation für die Berichtsdefinition. RDL ist ein offenes Schema: Entwickler können RDL um zusätzliche Attribute und Elemente erweitern. Die Standarddateierweiterung für Berichtsdefinitionsdateien ist .rdl. Ihr MIME-Typ ist text/xml. 

Eine Aspose.Cells-Vorlage ist eine Microsoft Excel-Arbeitsmappe, die Layoutinformationen für Berichte enthält und in RDL-Dateien (Format) gespeichert wird, wenn die Vorlagengestaltungsarbeit abgeschlossen ist. Wenn eine RDL-Datei geöffnet wird, die keine Aspose.Cells-Vorlage enthält, stehen nur ihre Datenquellen und Abfragen für die Verwendung in Aspose.Cells.Report.Designer zur Verfügung. Wenn eine RDL-Datei geöffnet wird, die eine Aspose.Cells-Vorlage enthält, wird die Vorlage abgerufen und als Arbeitsmappe in Microsoft Excel geöffnet.

{{% /alert %}}
