So fügen Sie Tabellenfußzeilen hinzu:

1.  Wählen Sie die Fußzeilen-Tag-Zelle direkt links neben der ersten Spalte der Tabelle, aber unter der Datenzeile der Tabellengruppe (oder Tabellendatenzeile, wenn die Tabelle keine Tabellengruppe enthält):

   **Auswählen der Fußzeilen-Tag-Zelle** 

![todo: Bild_alt_Text](adding-table-footers_1.png)




1.  Klicken**Fußzeile festlegen**auf der Symbolleiste Aspose.Cells.Report.Designer (

![todo: Bild_alt_Text](adding-table-footers_2.png)

). 

**Das Footer-Tag-Set** 

![todo: Bild_alt_Text](adding-table-footers_3.png)




1. Fügen Sie die Datenzeile der Fußzeile hinzu.
 Fußzeilendatenzeilen können statischen Text, Datenmarkierungen, Reporting Services-Formeln usw. enthalten. Hier haben wir eine Fußzeile hinzugefügt, die eine zusammengeführte statische Textzelle und eine Reporting Services-Formelzelle enthält.
