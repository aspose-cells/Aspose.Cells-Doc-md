So erstellen Sie eine neue Tabelle:

1. Öffnen Sie Microsoft Excel.
1.  Klicken**Bericht öffnen** 

![todo: Bild_alt_Text](adding-base-information-for-table_1)

 `  ` auf der Aspose.Cells.Report.Designer-Symbolleiste, um eine RDL-Berichtsdatei zu öffnen, die die im Voraus erstellten Datenquellen, Datensätze und Berichtsparameter enthält. Sobald die Datei erfolgreich geöffnet wurde, stehen alle ihre Informationen zur Verfügung, beispielsweise werden ihre Datensätze in der entsprechenden DataSet-Combobox aufgelistet.

![todo: Bild_alt_Text](adding-base-information-for-table_2.png)

1. Öffnen Sie eine Microsoft Excel-Tabelle und gestalten Sie den Tabellenkopf wie folgt:

   **Gestaltung des Tabellenkopfes** 

![todo: Bild_alt_Text](adding-base-information-for-table_3.png)

1.  Wählen Sie einen Datensatz „EmpSalesDetail“ aus dem entsprechenden Datensatz-Kombinationsfeld-Steuerelement aus.

   **Auswählen eines Datensatzes** 

![todo: Bild_alt_Text](adding-base-information-for-table_4.png)

1.  Alle Felder des Datensatzes „EmpSalesDetail“ sind in der Felder-Kombinationsbox aufgelistet. Wählen Sie die Zelle unter der Spalte „OrderNumber“ Cell und dann das Feld „SalesOrderNumber“ aus dem Kombinationsfeld „Felder“, um eine Datenmarkierung in die Zelle einzufügen.

   **Einfügen eines Feldes** 

![todo: Bild_alt_Text](adding-base-information-for-table_5.png)



**Der Feldcode zeigt an, wo ein Feld eingefügt wurde.** 

![todo: Bild_alt_Text](adding-base-information-for-table_6.png)

1.  Wiederholen Sie Schritt 5, um die Felder ProdCat und SubCat einzufügen.

   **Alle Felder eingefügt**
