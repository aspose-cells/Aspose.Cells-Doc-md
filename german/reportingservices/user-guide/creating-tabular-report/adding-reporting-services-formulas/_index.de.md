Um Reporting Services-Formeln zur Tabellendatenzelle hinzuzufügen, befolgen Sie die nachstehenden Schritte:

- Wählen Sie die Tabellendatenzelle aus und klicken Sie auf **Formel einfügen** in der Aspose.Cells.Report.Designer-Symbolleiste (

![todo:image_alt_text](adding-reporting-services-formulas_1.png)

).

![todo:image_alt_text](adding-reporting-services-formulas_2.png)




- Wählen Sie „DataSet“ im linken Panel, wählen Sie „EmpSalesDetail“ im mittleren Panel und doppelklicken Sie auf „Sum(EmpSalesDetail.OrderQty)“ im rechten Panel. Ein Formelausdruck erscheint im oberen Eingabefeld. Klicken Sie auf die Schaltfläche **OK**, um den Ausdruck in die Zelle einzufügen.

![todo:image_alt_text](adding-reporting-services-formulas_3.png)

- Das Ergebnis wird folgendermaßen angezeigt.


**Der eingefügte Ausdruck** 

![todo:image_alt_text](adding-reporting-services-formulas_4.png)

- Wiederholen Sie die obigen Schritte, um der Verkaufsspalte eine Formel hinzuzufügen, wie im Folgenden dargestellt.
