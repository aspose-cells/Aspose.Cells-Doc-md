{{% alert color="primary" %}} 

Der erste Schritt bei der Erstellung eines Berichts im Aspose.Cells.Report.Designer besteht darin, Datenquellen und Abfragen zu erstellen. Dies kann auf zwei Arten erfolgen. Im ersten Ansatz können wir eine vorhandene RDL-Berichtsdefinitionsdatei verwenden, die von anderen Tools erstellt wurde, z. B. unter Verwendung von Visual Studio und die Datei enthält die für das Berichtsdesign und die Datenabfrage erforderlichen Datenquellen und Abfragen. Der zweite Weg besteht darin, neue Quellen und Abfragen im Aspose.Cells.Report.Designer zu erstellen.
