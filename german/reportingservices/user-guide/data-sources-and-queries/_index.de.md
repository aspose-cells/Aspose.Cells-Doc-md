{{% alert color="primary" %}} 

Der erste Schritt bei der Vorgehensweise zum Entwerfen eines Berichts in Aspose.Cells.Report.Designer ist das Erstellen von Datenquellen und Abfragen. Dazu gibt es zwei Möglichkeiten. Beim ersten Ansatz können wir eine vorhandene RDL-Berichtsdefinitionsdatei verwenden, die von anderen Tools erstellt wird, beispielsweise mithilfe von Visual Studio, und die Datei enthält Datenquellen und Abfragen, die für das Berichtsdesign und den Datenabruf erforderlich sind. Die zweite Möglichkeit besteht darin, neue Quellen und Abfragen in Aspose.Cells.Report.Designer zu erstellen.

{{% /alert %}}
