{{% alert color="primary" %}} 

Sie können Datenquellen und Abfragen in Visual Studio erstellen und in einer RDL-Datei speichern. Wenn die RDL-Datei in Aspose.Cells.Report.Designer geöffnet wird, stehen ihre Datenquellen und Abfragen zur Verfügung. Beispielsweise können wir in Visual Studio eine Abfrage mit dem Namen EmpSalesDetail erstellen und als RDL-Datei speichern.

**Die EmpSalesDetail-RDL-Datei** 

![todo: Bild_alt_Text](using-data-sources-and-queries-created-in-visual-studio_1.png)

Dieser Artikel zeigt, wie Sie mit einer RDL-Datei in Aspose.Cells.Report.Designer arbeiten.

{{% /alert %}} 

Sie können RDL-Dateien mit Aspose.Cells.Report.Designer öffnen und damit arbeiten:

1. Öffnen Sie Microsoft Excel.
1.  Wählen Sie in der Symbolleiste Aspose.Cells.Report.Designer aus**Bericht öffnen**. 

   **Die Schaltfläche Bericht öffnen** 

![todo: Bild_alt_Text](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Wählen Sie die RDL-Datei aus und öffnen Sie sie.
 Sobald die RDL-Datei in Aspose.Cells.Report.Designer geöffnet wird, erscheinen die Abfragen in der**Datensatz** aufführen.

   **Verfügbare Datensätze**
