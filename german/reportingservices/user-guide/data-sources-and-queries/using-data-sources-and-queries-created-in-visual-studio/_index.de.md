{{% alert color="primary" %}} 

Sie können Datenquellen und Abfragen in Visual Studio erstellen und in einer RDL-Datei speichern. Wenn die RDL-Datei im Aspose.Cells.Report.Designer geöffnet wird, sind ihre Datenquellen und Abfragen verfügbar. Zum Beispiel können wir eine Abfrage namens EmpSalesDetail in Visual Studio erstellen und als RDL-Datei speichern.

**Die EmpSalesDetail RDL-Datei** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_1.png)

In diesem Artikel wird gezeigt, wie man mit einer RDL-Datei im Aspose.Cells.Report.Designer arbeitet.

{{% /alert %}} 

Sie können RDL-Dateien mit dem Aspose.Cells.Report.Designer öffnen und mit ihnen arbeiten:

1. Öffnen Sie Microsoft Excel.
1. Wählen Sie in der Aspose.Cells.Report.Designer-Symbolleiste **Open Report** aus. 

   **Die Schaltfläche Open Report** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_2.png)




1. Wählen Sie die RDL-Datei aus und öffnen Sie sie.
   Sobald die RDL-Datei in Aspose.Cells.Report.Designer geöffnet ist, erscheinen die Abfragen in der **Datensatz**-Liste. 

   **Verfügbare Datensätze** 

![todo:image_alt_text](using-data-sources-and-queries-created-in-visual-studio_3.png)
