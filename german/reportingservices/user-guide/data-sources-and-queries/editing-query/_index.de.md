Um eine Abfrage im Aspose.Cells.Report.Designer zu bearbeiten:

1. Öffnen Sie Microsoft Excel.
1. Klicken Sie auf **Build DataSet** auf der Symbolleiste des Aspose.Cells.Report.Designer (

![todo:image_alt_text](editing-query_1.png)

).

1. Wählen Sie im Dialogfeld einen Datensatz aus, z. B. **EmpSalesDetail**.
1. Klicken Sie auf **Bearbeiten**. 

   **Auswählen eines Datensatzes** 

![todo:image_alt_text](editing-query_2.png)




1. Aspose.Cell.Report.Designer startet Microsoft Query und öffnet die Abfrage zum Bearbeiten. 

   **Bearbeiten einer Abfrage in Microsoft Query** 

![todo:image_alt_text](editing-query_3.png)




1. Wählen Sie im **Datei**-Menü von Microsoft Query **Zurück zum Aspose.Cells.Report.Designer** aus 

   **Zurück zum Berichtsdesigner** 

![todo:image_alt_text](editing-query_4.png)
