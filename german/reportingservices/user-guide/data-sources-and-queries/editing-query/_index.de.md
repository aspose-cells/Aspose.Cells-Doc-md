So bearbeiten Sie eine Abfrage in Aspose.Cells.Report.Designer:

1. Öffnen Sie Microsoft Excel.
1.  Klicken**Datensatz erstellen**auf der Symbolleiste Aspose.Cells.Report.Designer (

![todo: Bild_alt_Text](editing-query_1.png)

).

1.  Wählen Sie im Dialogfenster einen Datensatz aus, z. B.**EmpSalesDetail**.
1.  Klicken**Bearbeiten**. 

   **Auswählen eines Datensatzes** 

![todo: Bild_alt_Text](editing-query_2.png)




1.  Aspose.Cell.Report.Designer startet Microsoft Query und öffnet die Query zur Bearbeitung.

   **Bearbeiten einer Abfrage in Microsoft Abfrage** 

![todo: Bild_alt_Text](editing-query_3.png)




1.  Von Microsoft Abfrage's**Datei** Menü, auswählen**Zurück zu Aspose.Cells.Report.Designer** 

   **Kehren Sie zum Berichtsdesigner zurück** 

![todo: Bild_alt_Text](editing-query_4.png)
