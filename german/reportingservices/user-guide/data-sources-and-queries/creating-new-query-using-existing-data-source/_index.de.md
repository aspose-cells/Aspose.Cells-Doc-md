So erstellen Sie eine neue Abfrage mit einer vorhandenen Datenquelle:

1. Öffnen Sie Microsoft Excel.
1.  Klicken**Datensatz erstellen** im Aspose.Cells.Report.Designer (

![todo: Bild_alt_Text](creating-new-query-using-existing-data-source_1.png)

).

1.  Wählen Sie im Dialogfeld eine der im linken Bereich aufgeführten Datenquellen aus. Zum Beispiel,**SQL Server**.
1.  Klicken**Addieren**. 

   **Auswählen einer Datenquelle** 

![todo: Bild_alt_Text](creating-new-query-using-existing-data-source_2.png)




1. Geben Sie im Dialogfeld einen neuen Datensatznamen ein, z. B. SalesEmps.
1.  Klicken**Nächste**. 

   **Benennen eines neuen Datensatzes** 

![todo: Bild_alt_Text](creating-new-query-using-existing-data-source_3.png)



 Aspose.Cells.Report.Designer startet die Microsoft-Abfrage und stellt eine Verbindung zur ausgewählten Datenquelle her. Mit Microsoft Abfrage kann nun eine neue Abfrage erstellt werden.

**Erstellen einer neuen Abfrage** 

![todo: Bild_alt_Text](creating-new-query-using-existing-data-source_4.png)




1.  Von Microsoft Abfrage's**Datei** Menü, auswählen**Zurück zu Aspose.Cells.Report.Designer**. 

   **Kehren Sie zum Berichtsdesigner zurück** 

![todo: Bild_alt_Text](creating-new-query-using-existing-data-source_5.png)
