Um eine neue Abfrage unter Verwendung einer vorhandenen Datenquelle zu erstellen:

1. Öffnen Sie Microsoft Excel.
1. Klicken Sie auf **Datensatz erstellen** im Aspose.Cells.Report.Designer (

![todo:image_alt_text](creating-new-query-using-existing-data-source_1.png)

).

1. Wählen Sie im Dialogfeld eine der im linken Bereich aufgeführten Datenquellen aus. Zum Beispiel **SqlServer**.
1. Klicken Sie auf **Hinzufügen**. 

   **Auswahl einer Datenquelle** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_2.png)




1. Geben Sie im Dialogfeld einen neuen Datensatznamen ein, z. B. SalesEmps.
1. Klicken Sie auf **Weiter**. 

   **Benennen eines neuen Datensatzes** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_3.png)



Der Aspose.Cells.Report.Designer startet Microsoft Query und verbindet sich mit der ausgewählten Datenquelle. Es kann nun eine neue Abfrage mit Microsoft Query erstellt werden. 

**Erstellen einer neuen Abfrage** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_4.png)




1. Wählen Sie im **Datei**-Menü von Microsoft Query **Zurück zum Aspose.Cells.Report.Designer** aus. 

   **Zurück zum Berichtsdesigner** 

![todo:image_alt_text](creating-new-query-using-existing-data-source_5.png)
