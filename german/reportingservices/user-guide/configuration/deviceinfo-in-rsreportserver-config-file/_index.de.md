 Der DeviceInfo-Abschnitt in**rsreportserver.config** nimmt die folgenden Parameter, die das Verhalten von Aspose.Cells beeinflussen:

- **Dateierweiterung** : wenn der Wert ist**Null**, ist die Dateierweiterung des exportierten Berichts der Standardwert. Wenn der Wert nicht null ist, wird die Erweiterung des Berichts auf den Wert gesetzt.
- **SimplePageHeader** : wenn der Wert ist**wahr** wird das Kopfzeilenelement des Berichts als Microsoft Excel-Seitenkopf gerendert. Der Standardwert ist**FALSCH**.
- **EinfacheSeitenFußzeilen** : wann**wahr** , wird das Fußzeilenelement des Berichts als Microsoft Excel-Seitenfuß gerendert. Der Standardwert ist**wahr**.
- **PutoutHeader** : wann**wahr** (Standard) wird das Kopfzeilenelement des Berichts exportiert. Wann**FALSCH**, wird das Kopfzeilenelement des Berichts nicht exportiert. Unterstützt nur die Excel 2007-Erweiterung XLSX (nur Daten).
- **PutoutFooter** : wann**wahr** (Standard) wird das Fußzeilenelement des Berichts exportiert. Wann**FALSCH**, es ist nicht. Unterstützt nur die Excel 2007-Erweiterung XLSX (nur Daten).
- **FillTableGroupHeaderForSimpleOutPut**: **FALSCH** standardmäßig. Der Wert unterstützt nur die Excel 2007-Erweiterung XLSX (nur Daten).
- **NoOutPutTotalForSimpleOutPut**: **FALSCH** standardmäßig. Der Wert unterstützt nur die Excel 2007-Erweiterung XLSX (nur Daten).
- **NoOutPutGroupForSimpleOutPut**: **FALSCH** standardmäßig. Der Wert unterstützt nur die Excel 2007-Erweiterung XLSX (nur Daten).
- **NoDoGroupPageForSimpleOutPut**: **wahr** standardmäßig. Der Wert unterstützt nur die Excel 2007-Erweiterung XLSX (nur Daten).
- **NoDoPageForSimpleOutPut**: **wahr** standardmäßig. Der Wert unterstützt nur die Excel 2007-Erweiterung XLSX (nur Daten).
- **Feldbegrenzer**: legt Feldtrennzeichen fest. Der Wert unterstützt die Erweiterungen CSV und TXT.
