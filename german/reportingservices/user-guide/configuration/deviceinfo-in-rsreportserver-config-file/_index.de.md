 Der DeviceInfo-Abschnitt in**rsreportserver.config** nimmt die folgenden Parameter, die das Verhalten von Aspose.Cells beeinflussen:

- **Dateierweiterung** : wenn der Wert ist**Null**, ist die Dateierweiterung des exportierten Berichts der Standardwert. Wenn der Wert nicht null ist, wird die Erweiterung des Berichts auf den Wert gesetzt.
- **SimplePageHeader** : wenn der Wert ist**Stimmt** , wird das Kopfzeilenelement des Berichts als Microsoft Excel-Seitenkopf gerendert. Der Standardwert ist**FALSCH**.
- **EinfacheSeitenFußzeilen** : Wenn**Stimmt** , wird das Fußzeilenelement des Berichts als Microsoft Excel-Seitenfuß gerendert. Der Standardwert ist**Stimmt**.
- **PutoutHeader** : Wenn**Stimmt** (Standard) wird das Kopfzeilenelement des Berichts exportiert. Wann**FALSCH**, wird das Kopfzeilenelement des Berichts nicht exportiert. Unterstützt nur die Erweiterung Excel 2007 XLSX (nur Daten).
- **PutoutFooter** : Wenn**Stimmt** (Standard) wird das Fußzeilenelement des Berichts exportiert. Wann**FALSCH**es ist nicht. Unterstützt nur die Erweiterung Excel 2007 XLSX (nur Daten).
- **FillTableGroupHeaderForSimpleOutPut**: **FALSCH** standardmäßig. Der Wert unterstützt nur die Erweiterung Excel 2007 XLSX (nur Daten).
- **NoOutPutTotalForSimpleOutPut**: **FALSCH** standardmäßig. Der Wert unterstützt nur die Erweiterung Excel 2007 XLSX (nur Daten).
- **NoOutPutGroupForSimpleOutPut**: **FALSCH** standardmäßig. Der Wert unterstützt nur die Erweiterung Excel 2007 XLSX (nur Daten).
- **NoDoGroupPageForSimpleOutPut**: **Stimmt** standardmäßig. Der Wert unterstützt nur die Erweiterung Excel 2007 XLSX (nur Daten).
- **NoDoPageForSimpleOutPut**: **Stimmt** standardmäßig. Der Wert unterstützt nur die Erweiterung Excel 2007 XLSX (nur Daten).
- **Feldbegrenzer**: legt Feldtrennzeichen fest. Der Wert unterstützt CSV- und TXT-Erweiterungen.
