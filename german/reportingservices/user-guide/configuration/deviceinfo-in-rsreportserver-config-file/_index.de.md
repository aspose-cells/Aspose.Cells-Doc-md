Der DeviceInfo-Abschnitt in **rsreportserver.config** enthält die folgenden Parameter, die das Verhalten von Aspose.Cells beeinflussen:

- **FileExtension**: Wenn der Wert **null** ist, ist die exportierte Berichtsdateierweiterung der Standardwert. Ist der Wert nicht null, wird die Erweiterung des Berichts auf den Wert festgelegt.
- **SimplePageHeaders**: Wenn der Wert **true** ist, wird das Rahmenkopf-Element des Berichts als Microsoft Excel-Rahmenkopf gerendert. Der Standardwert ist **false**.
- **SimplePageFooters**: Wenn **true**, wird das Rahmenfußzeilen-Element des Berichts als Microsoft Excel-Rahmenfußzeile gerendert. Der Standardwert ist **true**.
- **PutoutHeader**: Wenn **true** (Standard), wird das Rahmenkopf-Element des Berichts exportiert. Wenn **false**, wird das Rahmenkopf-Element nicht exportiert. Unterstützt nur die XLSX (nur Daten) -Erweiterung von Excel 2007.
- **PutoutFooter**: Wenn **true** (Standard), wird das Rahmenfußzeilen-Element des Berichts exportiert. Wenn **false**, wird es nicht exportiert. Unterstützt nur die XLSX (nur Daten) -Erweiterung von Excel 2007.
- **FillTableGroupHeaderForSimpleOutPut**: Standardmäßig **false**. Der Wert unterstützt nur die XLSX (nur Daten) -Erweiterung von Excel 2007.
- **NoOutPutTotalForSimpleOutPut**: Standardmäßig **false**. Der Wert unterstützt nur die XLSX (nur Daten) -Erweiterung von Excel 2007.
- **NoOutPutGroupForSimpleOutPut**: Standardmäßig **false**. Der Wert unterstützt nur die XLSX (nur Daten) -Erweiterung von Excel 2007.
- **NoDoGroupPageForSimpleOutPut**: Standardmäßig **true**. Der Wert unterstützt nur die XLSX (nur Daten) -Erweiterung von Excel 2007.
- **NoDoPageForSimpleOutPut**: Standardmäßig **true**. Der Wert unterstützt nur die XLSX (nur Daten) -Erweiterung von Excel 2007.
- **FieldDelimiter**: Legt Feldtrennzeichen fest. Der Wert unterstützt CSV- und TXT-Erweiterungen.
