 Gitternetzlinien sind in konfiguriert**Aspose.Cells.ReportingServices.xml** So können Sie steuern, ob eine exportierte Microsoft Excel-Datei Gitterlinien anzeigt. Der Standardwert ist**aus**, was bedeutet, dass Gitternetzlinien nicht angezeigt werden.

{{< highlight "java" >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
