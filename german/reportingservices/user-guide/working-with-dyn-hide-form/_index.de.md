{{% alert color="primary" %}} 

Mit dem Dialog "Dyn-Hide" können Sie Informationen dynamisch aus einem Bericht ausblenden. Sie können damit einen Zellbereich auf einem Arbeitsblatt definieren, den Sie ausblenden möchten.

{{% /alert %}} 

Informationen ausblenden:

1.  Klicken Sie in Microsoft Excel auf**Dyn-Hide** in der Aspose.Cells-Befehlsleiste.
 Der Dyn-Hide-Dialog wird angezeigt.

![todo: Bild_alt_Text](working-with-dyn-hide-form_1.png)




1. Bearbeiten Sie die Ausblendungsinformationen.
1.  Klicken**Addieren** Informationen zum Ausblenden hinzuzufügen.

![todo: Bild_alt_Text](working-with-dyn-hide-form_2.png)



Der definierte Bereich wird aus dem Bericht ausgeblendet.

So entfernen Sie Ausblendinformationen:

1. Wählen Sie im Dyn-Hide-Dialog ein Blatt aus der Liste der ausgeblendeten Bereiche aus.
1.  Klicken**Löschen**.
 Der ausgewählte Bereich wird gelöscht und im Bericht angezeigt.
