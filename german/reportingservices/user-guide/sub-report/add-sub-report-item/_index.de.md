So fügen Sie ein untergeordnetes Berichtselement hinzu:

1. Wählen Sie die Zelle aus, die auf das Element des Unterberichts verweist.
1.  Klicken**Unterbericht** (

![todo: Bild_alt_Text](add-sub-report-item_1.png)

).
 Das Unterberichtsformular wird angezeigt.

**Das SubReport-Formular** 

![todo: Bild_alt_Text](add-sub-report-item_2.png)




1.  Klicken**Unterbericht hinzufügen**.
 Das Formular Bericht abrufen wird angezeigt.
1.  Wählen Sie einen Bericht aus.

   **Auswählen eines Berichts** 

![todo: Bild_alt_Text](add-sub-report-item_3.png)




1.  Klicken**Nahe**. 

   **Ein Unterbericht wurde ausgewählt** 

![todo: Bild_alt_Text](add-sub-report-item_4.png)




1.  Doppelklicken Sie auf die Parameterliste, um das Dialogfeld "Formel bearbeiten" anzuzeigen.

   **Der Dialog Formel bearbeiten** 

![todo: Bild_alt_Text](add-sub-report-item_5.png)




1.  Bearbeiten und/oder aktualisieren Sie einen Parameterwert.

   **Benennen des Unterberichts** 

![todo: Bild_alt_Text](add-sub-report-item_6.png)




1.  Klicken**Unterbericht einfügen**. 

   **Der in eine Berichtstabelle eingefügte Unterbericht** 

![todo: Bild_alt_Text](add-sub-report-item_7.png)
