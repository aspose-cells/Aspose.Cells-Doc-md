Zum Hinzufügen eines untergeordneten Berichtselements:

1. Wählen Sie die Zelle aus, die das untergeordnete Berichtselement referenzieren wird.
1. Klicken Sie auf **Unterbericht** (

![todo:image_alt_text](add-sub-report-item_1.png)

).
Das Unterberichtsformular wird angezeigt. 

**Das Unterberichtsformular** 

![todo:image_alt_text](add-sub-report-item_2.png)




1. Klicken Sie auf **Unterbericht hinzufügen**.
   Das Formular zum Abrufen des Berichts wird angezeigt.
1. Wählen Sie einen Bericht aus. 

   **Auswahl eines Berichts** 

![todo:image_alt_text](add-sub-report-item_3.png)




1. Klicken Sie auf **Schließen**. 

   **Ein Unterbericht wurde ausgewählt** 

![todo:image_alt_text](add-sub-report-item_4.png)




1. Doppelklicken Sie auf die Parameterliste, um den Dialog zum Bearbeiten der Formel anzuzeigen. 

   **Der Dialog zum Bearbeiten der Formel** 

![todo:image_alt_text](add-sub-report-item_5.png)




1. Bearbeiten und/oder Aktualisieren eines Parameterwerts. 

   **Benennung des Unterberichts** 

![todo:image_alt_text](add-sub-report-item_6.png)




1. Klicken Sie auf **Unterbericht einfügen**. 

   **Der Unterbericht in eine Berichtstabelle eingefügt** 

![todo:image_alt_text](add-sub-report-item_7.png)
