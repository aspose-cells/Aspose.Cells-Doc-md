{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services (einschließlich des Aspose.Cells.ReportingServices-Clients, Add-Ins für Microsoft Excel und Aspose.Cells.ReportingServices Server) muss ordnungsgemäß installiert sein. Darüber hinaus sollte Microsoft SQL Server vor der Installation von Aspose.Cells for Reporting Services installiert werden.

{{% /alert %}} 
### **Office-System**
Die folgenden Versionen von Microsoft Excel werden unterstützt. Darüber hinaus müssen zusätzliche Patches installiert werden, um mögliche Probleme für Microsoft Office 2000, 2002, XP oder 2003 zu beheben.

- Microsoft Excel 2000
  Installieren Sie **lockbackRegKey.msi**, um ein CLR-Problem auf Microsoft Office-Maschinen, die älter als Office 2003 sind, zu beheben.
- Microsoft Excel 2002
  Installieren Sie **lockbackRegKey.msi**, um ein CLR-Problem auf Microsoft Office-Maschinen, die älter als Office 2003 sind, zu beheben.
- Microsoft Excel 2003
  Immer aktualisieren auf Office 2003 (KB907417). Dieses Update behebt ein potenzielles Problem, bei dem ein Add-In, ein Smart Tag oder ein Smartdokument nicht geladen wird, wenn es das Microsoft .NET Framework 2.0 verwendet.
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
- Microsoft Excel 2016
- Microsoft Excel 2019
- Microsoft Excel 365
