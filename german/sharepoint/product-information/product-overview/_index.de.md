### **Willkommen bei der Aspose.Cells für SharePoint-Dokumentation!**
Aspose.Cells für SharePoint ist eine flexible Lösung zur Unterstützung der Tabellenkonvertierung, die es ermöglicht, MS Excel-Dokumente innerhalb der Microsoft SharePoint-Anwendungen zu konvertieren.

{{% /alert %}} 

Sie können mehrere Dokumentformate konvertieren, wie z. B.:

- PDF - Portables Dokument von Adobe
- XLS-MS Excel 97~2003
- XLSX – Office Open XML
- XLSM – Office Open XML für Makros aktiviert
- TXT - Einfacher Text
- XML - Microsoft Excel-TabellenkalkulationML
- ODS - OpenDocument-Tabellenkalkulation
- CSV - Kommagetrennte Werte
- XLSB - Excel-Binärdatei 2007
- MHtml - Mehrzweck-Internet-Mail-Erweiterung HTML
- XPS - XML-Papierspezifikation
- TIFF - Tagged Image File Format
- SVG - Skalierbare Vektorgrafiken
- DIF - Zeichnungsaustauschformat

Aspose.Cells für SharePoint ist für die Verwendung mit den folgenden Produkten konzipiert:

- Windows SharePoint-Dienste 3.0 (WSS)
- Microsoft Office SharePoint Server 2007 (MOSS)-Standard
- Microsoft Office SharePoint Server 2007 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2010 (MOSS) Standard
- Microsoft Office SharePoint Server 2010 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2013 (MOSS) Standard
- Microsoft Office SharePoint Server 2013 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2016 (MOSS) Standard
- Microsoft Office SharePoint Server 2016 (MOSS) Enterprise

Außer den oben genannten Produkten gibt es keine zusätzlichen Systemanforderungen.

 Diese Dokumentation beschreibt die[Merkmale](/cells/de/sharepoint/features/), [Installation](/cells/de/sharepoint/install-aspose-cells-for-sharepoint/), [Auswertung](/cells/de/sharepoint/evaluate-aspose-cells/), [Lizenzierung](/cells/de/sharepoint/license-aspose-cells-for-sharepoint/), allgemeine Anwendungsfälle und Einstellungen von Aspose.Cells für SharePoint.
