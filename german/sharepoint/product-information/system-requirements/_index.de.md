### **Serveranforderungen**
Um Aspose.Cells for SharePoint zu installieren, benötigen Sie einen Computer, der eines der folgenden Betriebssysteme ausführt:

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft SharePoint Server 2010
- Microsoft SharePoint Server 2013

{{% alert color="primary" %}} 

Wir empfehlen, Service Pack 2 oder höher für diese Produkte zu installieren.

{{% /alert %}}
### **Client-Anforderungen**
Sie benötigen einen Browser, um auf SharePoint-Websites zuzugreifen. Aspose.Cells for SharePoint ist getestet mit:

- Microsoft Internet Explorer
- Mozilla FireFox
- Google Chrome
- Safari
