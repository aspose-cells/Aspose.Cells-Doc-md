Aspose.Cells for SharePoint bietet eine Symbolleiste, die einige wertvolle Optionen und Befehle umfasst:

- Konvertieren mit Aspose.Cells
- Dokument mit Aspose.Cells anzeigen
- Im Dokument mit Aspose.Cells suchen
- Wasserzeichen mit Aspose.Cells hinzufügen
- Fußzeile mit Aspose.Cells hinzufügen
