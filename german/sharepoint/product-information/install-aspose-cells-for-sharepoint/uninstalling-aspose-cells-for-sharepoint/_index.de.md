{{% alert color="primary" %}} 

Um Aspose.Cells für SharePoint zu deinstallieren, führen Sie das Setup-Programm aus. Wenn Aspose.Cells für SharePoint bereits installiert ist, schlägt das Setup-Programm vor, es zu entfernen oder zu reparieren.

{{% /alert %}} 
### **Deinstallation**
 Beim Entfernen deaktiviert das Setup-Programm Aspose.Cells für SharePoint für alle Websitesammlungen und zieht die Lösung von der Serverfarm zurück.

**Aspose.Cells deinstallieren** 

![todo: Bild_alt_Text](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

 Bitte beachten Sie, dass das Ausführen der**Reparatur** Option reaktiviert Aspose.Cells für SharePoint auf Websitesammlungen nicht. Stattdessen muss die Funktion nach der erneuten Bereitstellung mit manuell aktiviert werden**Reparatur** Möglichkeit.

 Deinstallation über die**Entfernen** Option und anschließende Neuinstallation möglicherweise bequemer, da Sie während der Installation Zielwebanwendungen und Websitesammlungen auswählen können.

{{% /alert %}}
