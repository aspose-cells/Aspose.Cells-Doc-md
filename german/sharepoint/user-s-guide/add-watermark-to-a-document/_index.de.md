Wenn Sie ein Wasserzeichen in einer Excel-Datei hinzufügen möchten, wählen Sie bitte die Option "Wasserzeichen mit Aspose.Cells hinzufügen" im Bearbeitungssteuerelement (ECB) des Dokuments aus.

![todo:image_alt_text](add-watermark-to-a-document_1.png)

Die folgenden Optionen stehen während des Setups für das Hinzufügen eines Wasserzeichens zur Verfügung:

![todo:image_alt_text](add-watermark-to-a-document_2.png)

**Wasserzeichen-Datei**

Die Datei muss eine Bilddatei sein.
