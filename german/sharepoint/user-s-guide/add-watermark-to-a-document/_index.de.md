Wenn Sie Wasserzeichen in einer Excel-Datei hinzufügen möchten, wählen Sie bitte die Option „Wasserzeichen mit Aspose.Cells hinzufügen“ im Bearbeitungskontrollblock (ECB) des Dokuments.

![todo: Bild_alt_Text](add-watermark-to-a-document_1.png)

Die folgenden Optionen sind während der Einrichtung zum Hinzufügen von Wasserzeichen verfügbar:

![todo: Bild_alt_Text](add-watermark-to-a-document_2.png)

**Wasserzeichendatei**

Die Datei muss eine Bilddatei sein.
