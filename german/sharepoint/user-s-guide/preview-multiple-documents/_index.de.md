 Sie können mehrere Excel-Dokumente in der Dokumentbibliothek in der Vorschau anzeigen.

 So zeigen Sie alle Dateien in einem Ordner in der Vorschau an:

1.  Auswählen**Vorschaudokument mit Aspose.Cells** in der ECB für Ihren gewünschten Ordner, während Sie im übergeordneten Ordner bleiben.

So zeigen Sie mehrere Dateien in einem Ordner in der Vorschau an:

1. Wählen Sie mehrere Dateien aus.
1.  Wählen Sie bei noch ausgewählten Dateien aus**Vorschaudokument mit Aspose.Cells**.

**Dokumentvorschau innerhalb von SharePoint**

Wenn Sie mehrere Dokumente in der Vorschau anzeigen, zeigt der Datenbereich das erste Dokument in der Liste. Die Dokumentenliste zeigt alle in der Vorschau angezeigten Dokumente. So zeigen Sie ein anderes Dokument an:

1. Wählen Sie ein Dokument in der Liste aus.
1.  Klicken**Vorschau**.
