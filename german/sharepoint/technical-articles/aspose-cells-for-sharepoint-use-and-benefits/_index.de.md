### **Eine kurze Notiz zu SharePoint-Dokumentbibliotheken**
Die Inhalte auf SharePoint-Websites liegen in Form von Dokumenten vor, die an einem zentralen Ort namens SharePoint-Dokumentbibliothek (manchmal auch als "gemeinsame Dokumentbibliothek" bezeichnet) freigegeben sind. Autorisierte Benutzer können Dokumente von ihrem lokalen Rechner auf die SharePoint-Site hochladen. Diese Dokumente werden im binären Format in einer von der SharePoint-Site verwendeten SQL Server-Datenbank gespeichert. Dokumente werden anderen SharePoint-Benutzern in Form einer Liste von Hyperlinks wie unten gezeigt zur Verfügung gestellt:

### **Vorteile**
Die folgende Liste hebt einige Vorteile der Nutzung von Aspose.Cells for SharePoint hervor:

- Der SharePoint-Server verfügt nicht über die eingebaute Fähigkeit, Microsoft Excel-Dateien direkt auf dem Server zu konvertieren. Aspose.Cells for SharePoint fügt diese Funktion dem SharePoint-Server hinzu.
- Aspose.Cells for SharePoint benötigt nicht Microsoft Excel, um zu funktionieren. Es ist auf dem Server installiert, sodass SharePoint-Benutzer Microsoft Excel auf ihren lokalen Systemen nicht benötigen, um Microsoft Excel-Dokumente von einem Format in ein anderes zu konvertieren.
- Aspose.Cells for SharePoint erfordert keine anderen Voraussetzungen außer denen, die für die Erstellung einer SharePoint-Site erforderlich sind.
- Aspose.Cells for SharePoint ist [einfach auf einem SharePoint-Site-Server zu installieren](/cells/de/sharepoint/installing-aspose-cells-for-sharepoint/).
- Aspose.Cells for SharePoint unterstützt [eine Reihe von Ein- und Ausgabeformaten](/cells/de/sharepoint/multiple-format-support/).
- Aspose.Cells for SharePoint stellt eine öffentliche API für Entwickler bereit, die ihre eigenen SharePoint-Lösungen entwickeln müssen.
### **Konvertieren von Microsoft Excel-Dokumenten mit Aspose.Cells for SharePoint**
Sobald Aspose.Cells for SharePoint auf dem SharePoint-Server installiert ist, steht die Option **Mit Aspose.Cells.SharePoint konvertieren** im Kontextmenü zur Verfügung: 

**Die Option ‚Mit... konvertieren‘ finden** 

Um ein Microsoft Excel-Dokument aus Ihrer SharePoint-Dokumentenbibliothek in andere Formate zu konvertieren, befolgen Sie die untenstehenden Schritte:

1. Klicken Sie auf das gewünschte Microsoft Excel-Dokument und dann auf Mit Aspose.Cells for SharePoint konvertieren
   Der Bildschirm für die Konvertierungseinstellungen wird angezeigt. 

   **Entscheidung zur Konvertierung** 


1. Wählen Sie das Ausgabeformat aus.
   Hier können Sie auch den Ausgabedateinamen und den Zielpfad ändern.
1. Klicken Sie auf **Konvertieren**. 

   **Auswahl des Ausgabeformats** 



1. Wenn die Konvertierung abgeschlossen ist, wird eine Erfolgsmeldung angezeigt. 

   **Erfolgreiche Konvertierung** 



1. Klicken Sie auf **Zurück**, um zur Dokumentenbibliothek zurückzukehren.
1. Das konvertierte Dokument wird in der SharePoint-Dokumentenbibliothek aufgelistet. 

   **Die konvertierte Datei ist in der Dokumentenbibliothek aufgeführt** 


### **Zusammenfassung**
{{% alert color="primary" %}} 

Die Installation und Verwendung von Aspose.Cells for SharePoint ist einfach. Es fügt nicht nur eine Funktion zur direkten Konvertierung von Microsoft Excel-Dateien auf dem SharePoint-Server hinzu, sondern beseitigt auch die Notwendigkeit, dass Microsoft Excel auf den Client-Maschinen installiert ist. Darüber hinaus können Entwickler die öffentliche API von Aspose.Cells for SharePoint nutzen, um ihre eigenen SharePoint-Lösungen zu entwickeln.

Mit anderen Worten: Es ist vorteilhaft für SharePoint-Websiteadministratoren, Entwickler und Endbenutzer.
