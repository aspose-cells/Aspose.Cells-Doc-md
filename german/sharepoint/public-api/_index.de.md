{{% alert color="primary" %}} 

Kunden, die ihre eigenen SharePoint-Lösungen entwickeln, können über die öffentliche Rufnummer API auf unsere Konvertierungs-Engine zugreifen. Dies umfasst alle unterstützten Dateiformate und Konvertierungsoptionen. Die detaillierten Schnittstellenspezifikationen in unserer[API Referenz](https://reference.aspose.com/) wird mit verschiedenen C#- und VB-Samples begleitet.

{{% /alert %}}
