{{% alert color="primary" %}} 

Kunden, die ihre eigenen SharePoint-Lösungen entwickeln, können über die öffentliche API auf unseren Konvertierungsmotor zugreifen. Dies beinhaltet alle unterstützten Dateiformate und Konvertierungsoptionen. Die detaillierten Schnittstellenspezifikationen in unserer [API Referenz](https://reference.aspose.com/) werden von verschiedenen C#- und VB-Beispielen begleitet.

{{% /alert %}}
