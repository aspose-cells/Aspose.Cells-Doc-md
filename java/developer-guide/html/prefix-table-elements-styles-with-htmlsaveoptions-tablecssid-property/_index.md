## **Possible Usage Scenarios**

Aspose.Cells allows you to prefix table elements styles with [**HtmlSaveOptions.TableCssId**](https://apireference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId) property. Suppose, you set this property with some value like *MyTest_TableCssId*, then you will find table elements styles like shown below

{{< highlight java >}}

table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}
