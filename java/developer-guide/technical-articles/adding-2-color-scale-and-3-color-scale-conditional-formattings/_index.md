{{% alert color="primary" %}} 

**2-Color Scale** and **3-Color Scale** Conditional Formattings are added in the same way except they are differed by [FormatCondition.ColorScale.Is3ColorScale](https://apireference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale) property. This property is **false** for 2-Color Scale and **true** for 3-Color Scale Conditional Formattings.
