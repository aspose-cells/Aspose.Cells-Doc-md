{{% alert color="primary" %}}

You can use Aspose.Cells to export your worksheet or chart into an image with the desired width and height. It provides [**ImageOrPrintOptions.setDesiredSize()**](https://apireference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#setDesiredSize(int,%20int)) method to set the desired width and height of the exported image. The width and height are specified in the unit of pixels.
