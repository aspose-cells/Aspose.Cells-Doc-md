## **Possible Usage Scenarios**
Aspose.Cells allows you to save the workbook in *Strict Open XML Spreadsheet* format. For that purpose, it provides the [Workbook.Settings.Compliance](https://apireference.aspose.com/java/cells/com.aspose.cells/workbooksettings#Compliance) property. If you set its value as [OoxmlCompliance.ISO_29500_2008_STRICT](https://apireference.aspose.com/java/cells/com.aspose.cells/ooxmlcompliance#ISO_29500_2008_STRICT), then the output Excel file will be saved in *Strict Open XML Spreadsheet* format.
