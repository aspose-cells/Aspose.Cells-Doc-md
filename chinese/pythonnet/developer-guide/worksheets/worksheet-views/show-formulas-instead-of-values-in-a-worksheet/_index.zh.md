{{% alert color="primary" %}}

在 Microsoft Excel 中，可以使用“公式”功能选项页中的“显示公式”选项来显示公式而不是计算值。当显示公式时，Microsoft Excel 会在工作表中显示公式。您可以通过 Aspose.Cells for Python via .NET 来实现相同的功能。

{{% /alert %}}

Aspose.Cells for Python via .NET 提供了一个 [**Worksheet.show_formulas**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/show_formulas) 属性。将其设置为 **true** 可以让 Microsoft Excel 显示公式。
