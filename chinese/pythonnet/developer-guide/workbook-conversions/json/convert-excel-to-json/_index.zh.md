##  **将 Excel 工作簿转换为 JSON**

无需想知道如何将 Excel 工作簿转换为 JSON，因为 Aspose.Cells for Python via .NET 库具有最佳决策。 Aspose.Cells API 支持将电子表格转换为 JSON 格式。要将工作簿导出到 JSON，请通过[**保存格式.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat)作为第二个参数[**工作簿.保存**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions)方法。您还可以使用[**Json保存选项**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions)类指定将工作表导出到 JSON 的其他设置。
