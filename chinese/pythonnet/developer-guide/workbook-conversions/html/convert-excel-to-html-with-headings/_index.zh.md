##  **将 Excel 转换为带标题的 HTML**

Aspose.Cells for Python via NET 提供了在将 Excel 转换为 HTML 时导出标题的选项。为此，您可以设置[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)属性为 *true**。默认值为[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)属性是*假**。下图显示了以下代码生成的输出文件。
