{{% alert color="primary" %}}

Aspose.Cells for Python 通过.NET支持在ODF（OpenDocument Format）1.1和1.2规范中保存ODS（OpenDocument Spreadsheet）文件。Aspose.Cells for Python 通过.NET有一个属性，用于指定将ODS文件保存为ODF 1.1规范。此属性的默认值为false，因此未使用此设置保存的ODS文件将使用1.2规范。
