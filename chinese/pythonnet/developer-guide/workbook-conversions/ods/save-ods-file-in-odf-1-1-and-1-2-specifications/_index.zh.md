{{% alert color="primary" %}}

 Aspose.Cells for Python via .NET 支持以 ODF (**OpenDocument Format**) 1.1 和 1.2 规范保存 ODS 文件 (**OpenDocument Spreadsheet**)。 Aspose.Cells for Python via .NET 有[**OdsSaveOptions.is_strict_schema11**](https://reference.aspose.com/cells/python-net/aspose.cells/odssaveoptions/is_strict_schema11/)属性，指定使用 ODF 1.1 规范来保存 ODS 文件。该属性的默认值为*false**，因此没有此设置保存的ODS文件使用1.2规范。

{{% /alert %}}
