## **可能的使用场景**

以前，Aspose.Cells for Python via .NET不能在将Excel文件保存为PDF格式时呈现Office加载项。现在Aspose.Cells for Python via .NET可以很好地呈现它。您无需使用任何特殊方法或属性来在输出PDF中呈现Office加载项。只需将Excel文件保存为PDF格式，它将呈现Office加载项。
