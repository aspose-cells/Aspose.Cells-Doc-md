Microsoft Excel 允许您在打印或保存为 PDF 格式时打印注释，具有以下选项

- 没有任何
- 在表的末尾
- 如表中所示

 Aspose.Cells for Python via .NET 提供[**打印注释类型**](https://reference.aspose.com/cells/python-net/aspose.cells/printcommentstype/)枚举来支持相同的功能。这[**打印注释类型**](https://reference.aspose.com/cells/python-net/aspose.cells/printcommentstype/)枚举有以下成员

- 打印无评论
- 就地打印
- 打印表结束
