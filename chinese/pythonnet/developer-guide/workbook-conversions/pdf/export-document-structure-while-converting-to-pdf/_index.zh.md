PDF 逻辑结构工具提供了将文档内容结构信息合并到 PDF 文件中的机制。Aspose.Cells for Python via .NET 保留了有关 Microsoft Excel 文档的结构信息，例如单元格、行、表格、工作表、图像、形状、页眉/页脚等。

通过选项 [PdfSaveOptions.export_document_structure](https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/export_document_structure/) ，可以将文档结构导出为标记的 PDF。
