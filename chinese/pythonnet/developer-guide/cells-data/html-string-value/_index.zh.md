{{% alert color="primary" %}}

Aspose.Cells for Python via .NET支持将基于Microsoft Excel的HTML转换为XLS/XLSX格式。这意味着，由Microsoft Excel生成的HTML可以使用Aspose.Cells for Python via .NET转换回XLS/XLSX格式。

同样，如果有一些简单的HTML，Aspose.Cells for Python via .NET可以将其转换为HTML富文本。Aspose.Cells for Python via .NET提供一个[**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/)属性，可以接受这样的简单HTML并将其转换为格式化的单元格文本。

{{% /alert %}}
