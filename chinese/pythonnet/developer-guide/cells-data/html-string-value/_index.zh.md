{{% alert color="primary" %}}

通过.NET的Aspose.Cells for Python支持将面向Microsoft Excel的HTML转换为XLS/XLSX格式。这意味着由Microsoft Excel生成的HTML可以使用Aspose.Cells for Python通过.NET转换回XLS/XLSX格式。

同样，如果有一些简单的HTML，Aspose.Cells for Python通过.NET可以将其转换为HTML富文本。Aspose.Cells for Python通过.NET提供[**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/)属性，可以接受这样一个简单的HTML并将其转换为格式化的单元格文本。

{{% /alert %}}
