Aspose.Cells for Python via .NET允许您将宏代码指定给表单控件（如按钮）。请使用[**Shape.macro_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/macro_name)属性为工作簿中的表单控件分配新的宏代码。

{{% /alert %}}

    MsgBox "Welcome to Aspose!"

End Sub
