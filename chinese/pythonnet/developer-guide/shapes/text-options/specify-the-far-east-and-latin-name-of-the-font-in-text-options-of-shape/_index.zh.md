## **可能的使用场景**

有时您想在远东语言字体（如日语、中文、泰语等）中显示文本。Aspose.Cells for Python via .NET 提供 [**TextOptions.far_east_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/far_east_name) 属性，可用来指定远东语言的字体名称。此外，您还可以使用 [**TextOptions.latin_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/latin_name) 属性指定拉丁字体名称。
