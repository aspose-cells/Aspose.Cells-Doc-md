## **如何在将 Excel 渲染为 PDF 时使用 Aspose.Cells for Python Excel 库绘制时间轴**
如果您有一个应用了时间轴的 Excel 文件，想要将 Excel 导出为 PDF 并保留时间轴设置，Aspose.Cells for Python 通过 .NET 现在默认支持这一点。只需将应用了时间轴的 Excel 文件导出为 PDF，生成的 PDF 将保留已应用的时间轴。
