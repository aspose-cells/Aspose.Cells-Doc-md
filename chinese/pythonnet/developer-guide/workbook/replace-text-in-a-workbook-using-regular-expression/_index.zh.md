Aspose.Cells for Python via .NET提供了用正则表达式在工作簿中替换文本的功能。为此，API提供了[**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key)属性的[**ReplaceOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions)类。将[**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key)设置为**true**表示搜索的关键词是正则表达式。
