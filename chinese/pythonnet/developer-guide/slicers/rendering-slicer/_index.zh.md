##  **可能的使用场景**
Aspose.Cells for Python via .NET 支持切片形状的渲染。如果将工作表转换为图像或将工作簿保存为 PDF 或 HTML 格式，您将看到切片器已正确渲染。

##  **渲染切片器**
以下示例代码加载[Excel 文件示例](67338479.xlsx)包含现有切片器。它通过设置仅覆盖切片器的打印区域将工作表转换为图像。流动的图像是[输出图像](67338480.png)显示渲染的切片器。如您所见，切片器已正确呈现，并且看起来与示例 Excel 文件中的相同。
