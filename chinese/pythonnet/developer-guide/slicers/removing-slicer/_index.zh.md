##  **可能的使用场景**

如果您想删除 Microsoft Excel 中的切片器，只需选择它并按*删除*按钮。同样，如果您想以编程方式使用 Aspose.Cells for Python via .NET API 删除它，请使用[**Worksheet.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/)方法。它将从工作表中删除切片器。
