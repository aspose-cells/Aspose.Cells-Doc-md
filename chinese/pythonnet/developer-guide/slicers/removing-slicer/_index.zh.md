## **可能的使用场景**

如果您要在 Microsoft Excel 中移除数据透视表筛选器，只需选择它，然后按*删除*按钮。同样，如果您要使用 Aspose.Cells for Python via .NET API 在程序中移除它，请使用 [**Worksheet.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/) 方法。它将从工作表中删除数据透视表筛选器。
