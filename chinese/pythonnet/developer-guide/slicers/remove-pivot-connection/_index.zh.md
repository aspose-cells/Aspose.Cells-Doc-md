## **可能的使用场景**

如果您想要在 Excel 中取消关联切片器和数据透视表，您需要右键单击切片器并选择“报表连接...”选项。在选项列表中，您可以操作勾选框。类似地，如果您想要使用 Aspose.Cells for Python 通过 .NET API 编程方式取消关联切片器和数据透视表，请使用 [**Slicer.remove_pivot_connection(pivot)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/remove_pivot_connection/#aspose.cells.pivot.PivotTable) 方法。它将取消关联切片器和数据透视表。
