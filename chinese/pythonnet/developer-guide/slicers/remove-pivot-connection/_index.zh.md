## **可能的使用场景**

如果您要在 Excel 中取消关联数据透视表筛选器和数据透视表，您需要右键单击筛选器，然后选择"报表连接..."项。在选项列表中，您可以操作复选框。同样，如果您要使用 Aspose.Cells for Python via .NET API 在程序中取消关联数据透视表筛选器和数据透视表，请使用 [**Slicer.remove_pivot_connection(pivot)**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/remove_pivot_connection/#aspose.cells.pivot.PivotTable) 方法。它会取消关联数据透视表筛选器和数据透视表。
