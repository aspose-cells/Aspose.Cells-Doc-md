Aspose.Cells for Python via .NET允许您使用 [**Workbook.import_xml()**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/import_xml/#str-str-int-int) 方法导入工作簿内的XML映射。您可以按照以下步骤在Microsoft Excel中导入XML映射。

- 选择** 开发人员** 选项卡
- 单击 **在 XML 部分导入** 并按照所需步骤操作。

您需要提供您的 XML 数据以完成导入。这里是一个[样本 XML 数据](5115037.txt)，您可以用于测试。
