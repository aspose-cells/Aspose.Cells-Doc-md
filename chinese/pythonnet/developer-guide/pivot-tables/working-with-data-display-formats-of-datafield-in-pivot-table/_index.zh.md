##  **“从小到大排名”和“从大到小排名”显示格式选项**

Aspose.Cells for Python via .NET 提供设置数据透视字段显示格式选项的功能。为此，API 提供了[**PivotField.data_display_format**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfield/data_display_format/)财产。要从大到小排列，您可以设置[**PivotField.data_display_format**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfield/data_display_format/)财产给[**PivotFieldDataDisplayFormat.RANK_LARGEST_TO_SMALLEST**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivotfielddatadisplayformat/)。以下代码片段演示了设置显示格式选项。
