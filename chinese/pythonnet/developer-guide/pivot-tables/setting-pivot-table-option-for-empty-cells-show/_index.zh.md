{{% alert color="primary" %}}

您可以使用 Aspose.Cells for Python 通过 .NET 设置不同的数据透视表选项。其中一个选项是“显示空单元格”。通过设置此选项，数据透视表中的所有空单元格将显示为指定的字符串。

{{% /alert %}}

## **如何在Microsoft Excel中设置数据透视表选项**

在Microsoft Excel中查找并设置此选项：

1. 选择一个数据透视表，右键单击。
1. 选择**数据透视表选项**。
1. 选择**布局和格式**选项卡。
1. 选择**显示空单元格**选项并指定一个字符串。

## **如何使用 Aspose.Cells for Python Excel 库设置数据透视表选项**

Aspose.Cells for Python 通过 .NET 为设置“显示报表筛选页”数据透视表选项提供了[**PivotTable.display_null_string**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/display_null_string/)和[**PivotTable.null_string**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/null_string/)属性。
