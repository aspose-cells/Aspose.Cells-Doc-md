## **可能的使用场景**

Microsoft Excel允许对数据透视表的Pivot字段进行分组。

## **如何在数据透视表中对Pivot字段进行分组**

以下示例代码加载了[示例Excel文件](64716818.xlsx)，并使用[**PivotTable.set_manual_group_field()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/set_manual_group_field/#int-float-float-list-float)方法对第一个数据透视字段进行分组。然后刷新和计算数据透视表的数据，并将工作簿保存为[输出Excel文件](64716817.xlsx)。屏幕截图显示了示例代码对示例Excel文件的影响。从屏幕截图可以看出，第一个数据透视字段现在按月份和季度分组。
