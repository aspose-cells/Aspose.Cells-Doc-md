##  **合并功能**

Aspose.Cells for Python via .NET 可用于将 ConsolidationFunction 应用于数据透视表的数据字段（或值字段）。在 Microsoft Excel 中，您可以右键单击值字段，然后选择**值字段设置...**选项，然后选择选项卡*汇总值依据**。从那里，您可以选择任何您选择的合并函数，如求和、计数、平均值、最大值、最小值、乘积、非重复计数等。

 Aspose.Cells for Python via .NET 提供[**合并功能**](https://reference.aspose.com/cells/python-net/aspose.cells/consolidationfunction/)枚举支持以下合并功能。

- 合并函数.AVERAGE
- 合并函数.COUNT
- 合并函数.COUNT_NUMS
- ConsolidationFunction.DISTINCT_COUNT
- 合并函数.MAX
- 合并函数.MIN
- 合并功能.产品
- 合并函数.STD_DEV
- ConsolidationFunction.STD_DEVP
