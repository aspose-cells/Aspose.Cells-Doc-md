Aspose.Cells for Python via .NET支持修改现有的SQL数据连接。本文将解释如何使用Aspose.Cells for Python via .NET修改SQL数据连接的不同属性。

同样，Aspose.Cells for Python via .NET提供了访问和修改数据连接的手段，使用Workbook.DataConnections集合。
