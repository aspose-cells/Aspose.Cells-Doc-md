{{% alert color="primary" %}}

Aspose.Cells for Python via .NET提供Worksheet.QueryTables集合，该集合通过索引返回QueryTable对象。它具有以下两个属性

- QueryTable.AdjustColumnWidth
- QueryTable.PreserveFormatting

这两个都是布尔值。你可以在Microsoft Excel中通过数据 > 连接 > 属性查看它们。

{{% /alert %}}
