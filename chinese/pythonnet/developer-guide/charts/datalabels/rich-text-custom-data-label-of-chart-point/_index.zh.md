您可以使用Aspose.Cells for Python via .NET为图表点创建丰富文本的自定义数据标签。该API提供[**DataLabels.characters()**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextframe/characters)方法，返回[**FontSetting**](https://reference.aspose.com/cells/python-net/aspose.cells/fontsetting)对象，可用来设置文本的字体属性，如颜色、粗细等。
