{{% alert color="primary" %}}

Microsoft Excel 2013允许用户在图表的数据标签中换行或取消换行。默认情况下，图表的数据标签中的文字是处于换行状态。

Aspose.Cells for Python via .NET提供了一个[**DataLabels.is_text_wrapped**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/datalabels/is_text_wrapped)属性，您可以将其设为True或False，以启用或禁用数据标签的文本换行。
