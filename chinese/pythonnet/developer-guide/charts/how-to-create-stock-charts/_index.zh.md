## **什么是股票图表?**

股票图表是一种用来追踪包括商品、股票和加密货币等交易资产价格变化的图表。它们可以显示随时间的高低值，以及开盘和收盘值。Aspose.Cells for Python via .NET 提供四种股票图表，要使用这些图表，必须拥有正确的数据集并按正确顺序选择列。

以下数据集显示了某股票的日常交易信息。我们将使用这些数据创建四种类型的股票图表：高-低-收（HLC）股票图表，开-高-低-收（OHLC）图表，量-高-低-收（VHLC）股票图表，以及量-开-高-低-收（VOHLC）股票图表。
