**欢迎来到 Aspose.Cells 获取 C++ 文档部分**

C++ 的 Aspose.Cells 是本机 C++ 库，使 C++ 开发人员能够以编程方式创建、操作和转换电子表格，而无需 Office 自动化或 Microsoft Excel 应用程序。

C++ 的 Aspose.Cells 支持电子表格格式，例如 Excel 97-2003 工作簿格式和 Open Office XML 格式。

{{% /alert %}}

## **好处**

在您的应用程序中将 Aspose.Cells 用于 C++ 可为您带来以下好处：

- 丰富的功能
- 平台独立性
- 性能和可扩展性

### **丰富的功能**

Aspose.Cells 为 C++ 提供了用于创建、转换和操作电子表格的广泛功能：

- 内置属性
- 自定义属性
- 主题
- 样式和格式
- 数据验证
- 条件格式
- 超级链接
- 自动过滤
- 页面设置
- 读、写和计算公式
- 分组行和列
- 数据透视表
- 桌子

### **平台独立性**

使用 Aspose.Cells for C++ 构建任何类型的 32 位和 64 位 C++ 应用程序。

### **性能和可扩展性**

Aspose.Cells for C++ 旨在在服务器端或客户端执行同样出色的性能。 Aspose.Cells for C++ 是一个本地程序集，可以通过简单地复制它来部署。您不必担心其他服务或模块。
