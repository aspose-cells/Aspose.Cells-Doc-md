{{% alert color="primary" %}} 

Aspose.Cells for JasperReports的一些有效功能如下：

1. Aspose.Cells for JasperReports支持Microsoft Excel图表，因此图表会以可编辑图表的形式导出，而不是静态图片。这样，用户可以在Microsoft Excel中手动编辑图表。
2. Aspose.Cells for JasperReports 可轻松与JasperServer集成。

{{% /alert %}}
