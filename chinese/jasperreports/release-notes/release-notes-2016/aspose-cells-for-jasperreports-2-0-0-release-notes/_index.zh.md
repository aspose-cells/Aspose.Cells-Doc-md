### **公共 API 和向后不兼容的更改**
以下是对公共 API 所做的任何更改的列表，例如添加、重命名、删除或弃用的成员，以及对 JasperReports 对 Aspose.Cells 所做的任何非向后兼容更改。如果您对列出的任何更改有疑虑，请在 Aspose.Cells 支持论坛上提出。
#### **支持JasperServer V6.3.0**
我们删除了那些支持旧版本 JasperReports 的版本，只在发布存档中保留支持 JasperServer v6.3.0 的版本。如果用户需要使用旧版本的JasperReports，请下载旧版本的JasperReports Aspose.Cells。
