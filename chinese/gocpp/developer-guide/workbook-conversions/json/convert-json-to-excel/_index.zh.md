## **将JSON转换为Excel工作簿** 
不用担心如何将 JSON 转换为 Excel 文件，因为 Aspose.Cells for C++ 库提供了最佳方案。Aspose.Cells API 支持将 JSON 格式转换为电子表格。你可以使用 [JsonLoadOptions](https://reference.aspose.com/cells/go-cpp/jsonloadoptions/) 类来为导入 JSON 到工作簿设置额外的参数。
