## **可能的使用场景**

你可以通过右键点击文件，选择 属性 > 详细信息，并编辑“语言”字段，来更改Excel文件的语言。请使用 [**BuiltInDocumentPropertyCollection.GetLanguage()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getlanguage/) 属性通过 Aspose.Cells API 以编程方式更改。
