## 用C++检测VBA项目是否受保护且锁定查看

Aspose.Cells允许你检测Excel文件的VBA（Visual Basic for Applications）项目是否已受保护且锁定查看。API提供[**VbaProject.GetIslockedForViewing()**](https://reference.aspose.com/cells/go-cpp/vbaproject/getislockedforviewing/)属性，如果被锁定查看，则返回[**VbaProject.GetIslockedForViewing()**](https://reference.aspose.com/cells/go-cpp/vbaproject/getislockedforviewing/)为**true**。
