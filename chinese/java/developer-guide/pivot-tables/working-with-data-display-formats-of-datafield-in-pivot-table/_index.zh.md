## **“从最小到最大排列”和“从最大到最小排列”显示格式选项**

Aspose.Cells 提供了为数据透视字段设置显示格式选项的能力。为此，API 提供了[**PivotField.DataDisplayFormat**](https://reference.aspose.com/cells/java/com.aspose.cells/pivotfield#DataDisplayFormat)财产。要从大到小排列，您可以设置[**PivotField.DataDisplayFormat**](https://reference.aspose.com/cells/java/com.aspose.cells/pivotfield#DataDisplayFormat)财产给[**PivotFieldDataDisplayFormat.RANK_LARGEST_TO_SMALLEST**](https://reference.aspose.com/cells/java/com.aspose.cells/pivotfielddatadisplayformat#RANK_LARGEST_TO_SMALLEST).以下代码片段演示了设置显示格式选项。
