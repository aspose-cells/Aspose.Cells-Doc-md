{{% alert color="primary" %}}

您可以使用Aspose.Cells设置不同的数据透视表选项。其中一个选项是"对于空单元格显示"。设置此选项意味着数据透视表中的所有空单元格都显示为指定的字符串。

在Microsoft Excel中查找并设置此选项：

1. 选择一个数据透视表并右键单击。
1. 选择**数据透视表选项**。
1. 选择**布局和格式**选项卡。
1. 选择**显示空单元格**选项并指定一个字符串。

![todo:image_alt_text](setting-pivot-table-option-for-empty-cells-show_1.png)

{{% /alert %}}

Aspose.Cells提供了用于设置"对于空单元格显示"数据透视表选项的[**PivotTable.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString)和[**PivotTable.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString)属性。
