## **介绍**

您可以使用Aspose.Cells设置不同的数据透视表选项。其中之一是“显示空单元格”。设置此选项意味着数据透视表中的所有空单元格都将显示指定的字符串。

要在Microsoft Excel中查找并设置此选项：

1. 选择一个数据透视表并右键单击。
2. 选择**数据透视表选项**。
3. 选择**布局和格式**选项卡。
4. 选择**对空单元格显示**选项并指定字符串。
