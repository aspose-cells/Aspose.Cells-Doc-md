{{% alert color="primary" %}}

您可以使用 Aspose.Cells 设置不同的数据透视表选项。其中一个选项是“显示空单元格”。设置此选项意味着数据透视表中的所有空单元格都以指定的字符串显示。

要在 Microsoft Excel 中查找并设置此选项：

1. 选择一个数据透视表并右键单击它。
1. 选择**数据透视表选项**.
1. 选择**布局和格式**标签。
1. 选择**对于空单元格显示**选项并指定一个字符串。

![待办事项：图像_替代_文本](setting-pivot-table-option-for-empty-cells-show_1.png)

{{% /alert %}}

Aspose.Cells 提供了[**数据透视表.setDisplayNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#DisplayNullString)和[**数据透视表.setNullString()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#NullString)用于设置“对于空单元格显示”数据透视表选项的属性。
