## **获取数据透视表的外部连接数据源**

Aspose.Cells提供了获取数据透视表的外部连接数据源的能力。为此，API提供了[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)类的[**PivotTable**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable)属性。[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)属性返回[**ExternalConnection**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection)对象。以下代码片段演示了如何使用[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)属性获取数据透视表的外部连接数据源。
