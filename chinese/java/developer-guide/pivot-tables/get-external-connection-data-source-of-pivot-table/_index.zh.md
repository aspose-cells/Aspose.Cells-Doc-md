## **获取数据透视表的外部连接数据源**

Aspose.Cells 提供获取数据透视表外接数据源的功能。为此，API 提供了[**外部连接数据源**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)的财产[**数据透视表**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable)班级。这[**外部连接数据源**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)属性返回[**外部连接**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection)目的。下面的代码片段演示了使用[**数据透视表.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)属性获取数据透视表的外部连接数据源。
