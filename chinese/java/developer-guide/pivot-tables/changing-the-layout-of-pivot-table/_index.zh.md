{{% alert color="primary" %}}

Microsoft Excel允许您使用*数据透视表工具* > *设计* > *报表*布局菜单命令更改数据透视表的布局。您可以将布局更改为这三种形式

- 显示为紧凑形式
- 显示为大纲形式
- 显示为表格形式

Aspose.Cells还提供了[**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm())、[**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm())和[**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm())方法来更改数据透视表在这三种形式下的布局。

{{% /alert %}}
