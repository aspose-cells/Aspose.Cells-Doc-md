{{% alert color="primary" %}}

Microsoft Excel 允许您使用更改数据透视表的布局*数据透视表工具 > 设计 > 报表*布局菜单命令。你可以改变这三种形式的Layout

- 以紧凑形式显示
- 以大纲形式显示
- 以表格形式显示

Aspose.Cells还提供[**数据透视表.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm()), [**数据透视表.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm() ） 和[**数据透视表.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm()) 方法来改变这三种形式的数据透视表的布局。
