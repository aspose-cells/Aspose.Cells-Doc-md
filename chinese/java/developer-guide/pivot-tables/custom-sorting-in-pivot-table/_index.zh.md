## **数据透视表中的自定义排序**
使用Aspose.Cells API，可以对字段值进行数据透视表排序。 以下代码片段加载了示例excel文件并添加了三个数据透视表。 第一个数据透视表没有自定义排序，第二个数据透视表按"SeaFood"行字段值排序，第三个数据透视表按"28/07/2000"列字段值排序。
