## **数据透视表中的自定义排序**
通过使用 Aspose.Cells API，您可以根据字段值对数据透视表进行排序。以下代码片段加载示例 excel 文件并添加三个数据透视表。第一个数据透视表没有自定义排序，第二个数据透视表按“SeaFood”行字段值排序，第三个数据透视表按“28/07/2000”列字段值排序。
