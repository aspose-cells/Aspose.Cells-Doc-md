## **可能的使用场景**

Microsoft Excel 允许将工作簿共享，如以下屏幕截图所示。共享工作簿后，多个用户可以在网络上编辑该工作簿。Aspose.Cells 可以通过 [**Workbook.Settings.Shared**](https://reference.aspose.com/cells/java/com.aspose.cells/workbooksettings#Shared) 属性创建共享工作簿。
