Aspose.Cells提供[工作簿.dispose()](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#dispose\(\) ) 方法来释放非托管资源[工作簿](https://reference.aspose.com/cells/java/com.aspose.cells/Workbook)目的。 dispose 模式仅用于访问非托管资源的对象，例如文件和管道句柄、注册表句柄、等待句柄或指向非托管内存块的指针。这是因为垃圾收集器在回收未使用的托管对象方面非常有效，但无法回收非托管对象。
