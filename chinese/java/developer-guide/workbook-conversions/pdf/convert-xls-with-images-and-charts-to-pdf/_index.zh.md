{{% alert color="primary" %}}

Aspose.Cells支持将包含图像和图表的XLS文件转换为PDF文档。 Aspose.Cells for Java 可独立工作以将电子表格转换为PDF：不需要使用Aspose.PDF API来进行转换。

{{% /alert %}}

该过程可在内存中进行，不依赖临时或中间 XML 文件。这意味着大型 Excel 文件（例如含有图片、图表和其他绘图对象的文件）可快速高效地转换。

{{% alert color="primary" %}}

如果电子表格中包含公式，最好在转换为 PDF 前调用 [**Workbook.calculateFormula**](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#calculateFormula--) 方法。这样做可以确保重新计算公式依赖的值，并在 PDF 中呈现正确的值。

{{% /alert %}}
