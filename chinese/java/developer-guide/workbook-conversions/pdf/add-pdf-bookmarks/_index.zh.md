{{% alert color="primary" %}} 

本文提供有关如何在将电子表格转换为 PDF 时插入 PDF 书签的信息。

Aspose.Cells 允许您在运行时添加书签。 PDF 书签可以大大提高长文档的导航性。将书签链接添加到 PDF 文档时，您可以精确控制所需的确切视图，而不仅限于链接到页面。您可以通过定位目标页面来设置精确视图，然后创建书签。

{{% /alert %}}
