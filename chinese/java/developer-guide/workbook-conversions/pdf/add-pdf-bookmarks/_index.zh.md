本文提供了将电子表格转换为PDF时如何插入PDF书签的信息。

Aspose.Cells允许您在运行时添加书签。PDF书签可以极大地提高对长文档的导航。在向PDF文档添加书签链接时，您可以精确控制所需的确切视图，不仅限于链接到特定页。您可以通过定位目标页面来设置精确视图，然后创建书签。
