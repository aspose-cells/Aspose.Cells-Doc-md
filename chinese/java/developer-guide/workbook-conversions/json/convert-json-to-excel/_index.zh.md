## **将JSON转换为Excel工作簿**

不必纠结如何将JSON转换为Excel文件，因为Aspose.Cells Java库有最佳决策。Aspose.Cells Java API支持将JSON格式转换为电子表格。您可以使用[**JsonLoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLoadOptions)类指定导入JSON到工作簿的附加设置。
