{{% alert color="primary" %}}

Aspose.Cells 支持保存（**OpenDocument电子表格**ODS 文件在 (**开放文档格式** ODF 1.1 和 ODF 1.2 规范。 Aspose.Cells 已添加[**OdsSaveOptions.setStrictSchema11()**](https://reference.aspose.com/cells/java/com.aspose.cells/odssaveoptions#IsStrictSchema11)使用 ODF 1.1 规范保存 ODS 文件的属性。该属性的默认值为**错误的**所以没有这个特殊设置保存的 ODS 文件将使用 1.2 规范。

{{% /alert %}}
