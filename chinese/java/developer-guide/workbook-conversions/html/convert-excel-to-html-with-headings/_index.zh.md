## **将 Excel 转换为带有标题的 HTML**

Aspose.Cells 提供了在将 Excel 转换为 HTML 时导出标题的选项。为此，您可以将[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings)属性设置为 **true**。[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings) 属性的默认值为 **false**。以下图片显示了以下代码生成的输出文件。
