{{% alert color="primary" %}}

Aspose.Cells提供了一个[**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference)属性，开发人员可以使用它在将Microsoft Excel文件保存为HTML或MHT格式时获得更好的布局。该属性的默认值为**false**。我们建议将此属性设置为**true**，以从Excel报表中获得更美观的呈现。
