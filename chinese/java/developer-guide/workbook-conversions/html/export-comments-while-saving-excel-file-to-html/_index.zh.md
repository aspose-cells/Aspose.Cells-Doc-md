## **可能的使用场景**

当您将Excel文件保存为HTML时，注释不会被导出。但是，Aspose.Cells提供了**[**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#IsExportComments)**属性来支持此功能。如果将其设置为**true**，则HTML也将显示Excel文件中的注释。
