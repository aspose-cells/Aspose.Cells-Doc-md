{{% alert color="primary" %}} 

如果您想要将Excel文件导出为带有网格线的HTML，请使用 [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) 属性并将其设置为 **true**。

{{% /alert %}} 
## **将Excel导出为带有网格线的HTML**
以下示例代码创建一个工作簿，并在其工作表中填充一些值，然后在将 [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) 设置为 **true** 后将其保存为HTML格式。

下面的截图显示了使用此示例代码生成的输出HTML。正如您所见，它还在输出HTML中显示了网格线。
