{{% alert color="primary" %}} 

如果您希望将Excel文件导出为具有网格线的HTML，请使用[HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines)属性，并将其设置为**true**。

{{% /alert %}}
