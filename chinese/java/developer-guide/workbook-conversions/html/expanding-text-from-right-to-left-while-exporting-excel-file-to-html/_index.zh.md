{{% alert color="primary" %}} 

Aspose.Cells现在支持将Excel文件中从右向左扩展的文本导出为HTML。该功能自v8.9.0.0版本已经实现。现在，如果您的源Excel文件包含从右向左扩展的文本，那么Aspose.Cells将正确地将其导出为HTML。

{{% /alert %}}
