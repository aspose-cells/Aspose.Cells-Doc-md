{{% alert color="primary" %}} 

Aspose.Cells 现在支持在将 Excel 文件导出为 HTML 时从右向左扩展文本。该功能从 v8.9.0.0 开始实现。现在，如果您的源 Excel 文件包含任何从右向左扩展的文本，则 Aspose.Cells 会将其正确导出为 HTML。

{{% /alert %}}
