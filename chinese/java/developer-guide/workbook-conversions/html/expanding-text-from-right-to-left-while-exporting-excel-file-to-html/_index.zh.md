{{% alert color="primary" %}} 

Aspose.Cells现在支持将文本从右向左展开，同时将Excel文件导出为HTML。自版本v8.9.0.0起已实现此功能。现在，如果您的源Excel文件包含从右向左展开的文本，则Aspose.Cells将正确地将其导出到HTML中。

{{% /alert %}}
