{{% alert color="primary" %}} 

通常，将 HTML 加载到 Excel 工作簿对象时，会忽略 div 标签的布局。但是，如果你想让布局中的div标签不被忽略，那么请设置[HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag)财产给**真的** .该属性的默认值为**错误的**.
