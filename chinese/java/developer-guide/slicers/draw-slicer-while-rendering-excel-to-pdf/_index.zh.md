## **在将 Excel 渲染为 PDF 时绘制切片器**
如果您有一个应用了切片器的 Excel 文件，并且您想要使用切片器设置将 Excel 导出为 PDF，Aspose.Cells 现在默认支持此功能。您只需将带有切片器的 Excel 文件导出为 PDF，生成的 PDF 将显示应用的切片器。

下面的示例代码加载[示例 Excel 文件](SampleSlicerChart.xlsx)包含一个现有的切片器。然后将工作簿另存为[输出PDF文件](SampleSlicerChart.pdf).以下屏幕截图比较了源 Excel 文件和生成的 PDF 文件。
