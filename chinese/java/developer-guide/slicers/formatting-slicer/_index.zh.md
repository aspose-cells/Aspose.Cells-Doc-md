您可以通过设置切片器的列数或样式等来格式化Microsoft Excel中的切片器。Aspose.Cells也允许您使用[Slicer.NumberOfColumns](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#NumberOfColumns)和[Slicer.StyleType](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#StyleType)属性来完成此操作。
