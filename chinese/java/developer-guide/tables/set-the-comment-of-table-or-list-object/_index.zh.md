## **设置工作表内表格或列表对象的批注**

请使用[**ListObject.Comment**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment)属性在工作表中设置表格或列表对象的注释。 注释将在xl/tables/tableName.xml文件中可见。
