## **设置工作表内表格或列表对象的注释**

请使用 [**ListObject.Comment**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment) 属性设置工作表内表格或列表对象的批注。该批注将在xl/tables/tableName.xml文件内可见。

以下示例代码加载了[源excel文件](5472561.xlsx)，设置了工作表内第一个表或列表对象的批注，并且屏幕截图显示了此代码创建的批注在红色矩形中。
