## **可能的使用场景**

Aspose.Cells 允许您在将表格插入工作表时将第一行向下移动。本文档解释了如何使用 Aspose.Cells API 完成任务。

## **插入 Cells 数据表行时将第一行向下移动**

以下示例代码说明了在将表格插入工作表时如何将第一行向下移动。我们在代码中使用一个简单的模板 Excel 文件来演示该功能。您可以通过设置布尔值来使用该功能[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/java/com.aspose.cells/importtableoptions#ShiftFirstRowDown)归于**真假**更好地理解它。请参阅[示例 Excel 文件](45056043.xlsx), [输出Excel False文件](45056044.xlsx)， 和[输出Excel真文件](45056045.xlsx)供你参考。
