有时，您可能想要检索有关外部数据源的信息。例如，您可能希望查看 SQL 连接数据。此信息可以包括连接到 SQL 服务器所需的任何类型的数据，例如服务器 URL、用户名、基础表名、SQL 查询、查询类型、表的位置等。

Aspose.Cells 提供了一些有用的调用，用于从链接到模板 Excel 文件的公司数据库中检索此类详细信息。
