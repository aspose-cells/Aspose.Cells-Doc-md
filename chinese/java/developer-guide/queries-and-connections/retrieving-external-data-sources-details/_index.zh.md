有时，您可能希望检索有关外部数据源的信息。例如，您可能希望查看SQL连接数据。这些信息可以包括连接到SQL服务器所需的任何类型的数据，例如服务器URL、用户名、底层表名、SQL查询、查询类型、表的位置等。

Aspose.Cells提供了一些用于从与模板Excel文件链接的公司数据库中检索此类详细信息的有用调用。
