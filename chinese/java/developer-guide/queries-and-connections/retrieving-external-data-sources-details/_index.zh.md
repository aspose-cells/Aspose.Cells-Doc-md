有时，您可能需要检索有关外部数据源的信息。例如，您可能希望查看SQL连接数据。这些信息可以包括与连接到SQL服务器所需的任何类型数据，例如服务器URL、用户名、基础表名、SQL查询、查询类型、表的位置等。

Aspose.Cells为从链接到模板Excel文件的公司数据库中检索此类详细信息提供了一些有用的调用。
