{{% alert color="primary" %}} 

Aspose.Cells支持修改现有的SQL数据连接。 本文将解释如何使用Aspose.Cells修改SQL数据连接的不同属性。

您可以通过**数据 > 连接**菜单命令添加或查看Microsoft Excel中的数据连接。

类似地，Aspose.Cells提供了一种访问和修改工作簿中数据连接的方法，使用 Workbook.getDataConnections() 集合。
