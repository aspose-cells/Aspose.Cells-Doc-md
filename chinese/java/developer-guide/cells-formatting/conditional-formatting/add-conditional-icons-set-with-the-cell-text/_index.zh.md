{{% alert color="primary" %}} 

有时，您希望在单元格中的文本旁边添加条件图标，以使数据对读者更有意义。您想要使用某些条件格式图标类型，但不对单元格应用条件格式。 Aspose.Cells 支持该功能。

{{% /alert %}} 

下面是一个示例，它从头开始创建一个 XLSX 文件，将条件图标添加到包含文本的单元格而不应用条件格式。
