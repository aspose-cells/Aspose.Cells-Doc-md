{{% alert color="primary" %}} 

**2-Color Scale**和**3-Color Scale**条件格式被相同的方式添加，除了它们通过[FormatCondition.ColorScale.Is3ColorScale](https://reference.aspose.com/cells/java/com.aspose.cells/colorscale#Is3ColorScale)属性进行区分。这个属性对于2-Color Scale来说是**false**，对于3-Color Scale来说是**true**。

{{% /alert %}}
