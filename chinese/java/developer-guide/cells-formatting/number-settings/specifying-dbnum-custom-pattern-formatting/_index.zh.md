## **可能的使用场景**

Aspose.Cells支持*DBNum*自定义模式格式化。例如，如果您的单元格值为123，并通过[DBNum2][$-804]General指定其自定义格式，则它将显示为壹佰贰拾叁。您可以使用[**Cell.getStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getStyle()）和[**Style.setCustom()**](https://reference.aspose.com/cells/java/com.aspose.cells/style#Custom)方法指定单元格的自定义格式。
