## **可能的使用场景**

Aspose.Cells 支持*数据库编号*自定义模式格式。例如，如果您的单元格值为 123，并且您将其自定义格式指定为 [DBNum2][$-804]General，那么它将显示为壹佰辰拾取。您可以使用指定单元格的自定义格式[**Cell.getStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getStyle()） 和[**样式.setCustom()**](https://reference.aspose.com/cells/java/com.aspose.cells/style#Custom)方法。
