## **数据格式设置**

格式化用于区分不同类型的信息，并清晰显示数据。

格式表示一种样式，定义为一组特征，如字体和字号、数字格式、单元格边框、单元格底纹、缩进、对齐和文字方向。边框提供了进一步突出信息的方式。边框是围绕单元格或一组单元格画的线。

数字格式也使数据更有意义。通过应用不同的数字格式，您可以改变数字的外观，而不改变其本质。

Aspose.Cells让您快速轻松地绘制单元格和范围边框。它还让您应用字体并填充单元格。该组件足够高效，您可以格式化整个行或列，设置对齐方式，在单元格中换行和旋转文本。Aspose.Cells进一步支持Microsoft Excel支持的所有数字格式。

## **摘要**

{{% alert color="primary" %}}

工作表数据格式化很重要，因为数据的外观可能会改变其含义。如果您计划打印工作表、将其通过电子邮件发送给客户或展示给管理层，请考虑格式是否适合查看。
谨慎使用颜色、阴影、边框、字体、数字格式、对齐、缩进和方向可以改变杂乱的数据和易于操作和理解的工作表之间的差异。
