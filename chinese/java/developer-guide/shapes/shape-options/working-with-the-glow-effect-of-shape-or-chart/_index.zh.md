## **可能的使用场景**
Aspose.Cells 提供了 Shape.Glow 属性来处理形状或图表的发光效果。它包含以下子属性，您可以根据需要设置这些子属性以获得不同的结果。

- 发光效果.大小
- 发光效果.透明度
