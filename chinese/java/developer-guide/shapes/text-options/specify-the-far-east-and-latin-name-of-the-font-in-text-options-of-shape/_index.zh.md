## **可能的使用场景**

有时，您希望使用远东语言字体（例如日语、中文、泰语等）显示文本。Aspose.Cells提供了[**TextOptions.FarEastName**](https://reference.aspose.com/cells/java/com.aspose.cells/textoptions#FarEastName)属性，可用于指定远东语言的字体名称。此外，还可以使用[**TextOptions.LatinName**](https://reference.aspose.com/cells/java/com.aspose.cells/textoptions#LatinName)属性指定拉丁字体名称。
