## **可能的使用场景**

Aspose.Cells 允许您使用[**形状.TextBody.TextAlignment**](https://reference.aspose.com/cells/java/com.aspose.cells/fontsettingcollection#TextAlignment)财产。此属性返回的对象[**形状文本对齐**](https://reference.aspose.com/cells/java/com.aspose.cells/ShapeTextAlignment)具有不同属性的类，例如[**顶边距点**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#TopMarginPt), [**左边距点**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#LeftMarginPt), [**底边距点**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#BottomMarginPt), [**右边距点**](https://reference.aspose.com/cells/java/com.aspose.cells/shapetextalignment#RightMarginPt)等，可用于设置上、左、下和右边距。
