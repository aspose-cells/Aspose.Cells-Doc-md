Aspose.Cells APIs允许以一种方式在工作表上添加WordArt水印，使WordArt成为一个可以在工作表上移动和定位的对象。还可以锁定WordArt对象，以阻止任何交互，如编辑、移动和选择。本文解释了使用[**Shape.setLockedProperty**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#setLockedProperty(int,%20boolean)）方法锁定标记的一些方面。
