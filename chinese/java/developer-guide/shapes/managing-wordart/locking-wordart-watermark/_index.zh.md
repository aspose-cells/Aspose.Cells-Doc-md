Aspose.Cells API 允许在工作表上添加艺术字水印，使艺术字成为可以在工作表上移动和定位的对象。也可以锁定艺术字对象以进行任何交互，例如编辑、移动和选择。这篇文章解释了使用[**形状.setLockedProperty**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#setLockedProperty(int,%20boolean)) 方法来锁定水印的几个方面。
