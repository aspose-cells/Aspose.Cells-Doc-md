Aspose.Cells 提供了[OleObject.自动加载](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad)在 Microsoft Excel 中打开 excel 文件时刷新 OLE 对象的属性。由于此属性，OLE 对象将显示由 Microsoft Excel 生成的正确 OLE 图像。

## **使用 Aspose.Cells 通过 Microsoft Excel 自动刷新 OLE 对象**
