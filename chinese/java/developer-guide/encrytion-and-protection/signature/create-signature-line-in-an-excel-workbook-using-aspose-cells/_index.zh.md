Microsoft Excel提供了一个添加功能**签名行**在 Excel 工作簿中。您可以通过单击**插入**选项卡然后选择**签名行**来自**文本**团体。

Aspose.Cells也提供了这个功能，并暴露了[**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine)为此目的的财产。本文将解释如何使用此属性添加使用 Aspose.Cells 的签名行。

{{% /alert %}}
