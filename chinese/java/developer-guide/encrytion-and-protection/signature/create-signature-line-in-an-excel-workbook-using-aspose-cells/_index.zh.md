{{% alert color="primary" %}}

Microsoft Excel提供了在Excel工作簿中添加 **签名行** 的功能。您可以通过单击 **插入** 选项卡，然后从 **文本** 组中选择 **签名行** 来添加签名行。

Aspose.Cells也提供了这个功能，并为此暴露了 [**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine) 属性。本文将解释如何使用此属性来使用Aspose.Cells添加签名行。

{{% /alert %}}
