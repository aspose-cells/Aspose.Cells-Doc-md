## **可能的使用场景**

您可以通过右键单击文件，然后选择*属性 > 详细信息*，编辑*版本号*字段来更改 Excel 文件的*版本号*。请使用 [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#DocumentVersion) 属性通过 Aspose.Cells API 进行编程更改。
