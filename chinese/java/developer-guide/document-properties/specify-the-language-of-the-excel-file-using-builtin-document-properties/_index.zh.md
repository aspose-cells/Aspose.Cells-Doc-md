## **可能的使用场景**

您可以通过右键单击文件，然后选择属性 > 详细信息，并编辑语言字段来更改Excel文件的语言。请使用[**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#Language)属性使用Aspose.Cells API以编程方式更改语言。
