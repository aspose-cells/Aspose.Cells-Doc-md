随着高分辨率屏幕的普及，默认 96 DPI 生成的图像常常显得模糊不清。为了确保在高分辨率屏幕上清晰显示，必须以更高的 DPI 生成图像。Aspose.Cells 提供了设置 [**ImageOrPrintOptions.HorizontalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setHorizontalResolution-int-) 和 [**ImageOrPrintOptions.VerticalResolution**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions/#setVerticalResolution-int-) 的功能，允许您从 Excel 文件创建高质量图像，在高分辨率显示屏上看起来清晰。
