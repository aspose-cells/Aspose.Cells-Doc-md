## **可能的使用场景**

通常情况下，您可以为所有工作簿指定字体目录或字体列表，但有时需要为工作簿指定个别或私有字体集。Aspose.Cells 提供了[**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs)类，可用于为工作簿指定个别或私有字体集。
