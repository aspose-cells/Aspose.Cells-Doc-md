## **可能的使用场景**

通常，您为所有工作簿指定字体目录或字体列表，但有时，您必须为您的工作簿指定单独或私有字体集。Aspose.Cells提供了[**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs) 类，可用于为您的工作簿指定单独或私有字体集。
