## **可能的使用场景**

通常，您为所有工作簿指定字体目录或字体列表，但有时，您必须为工作簿指定单独或私有的字体集。 Aspose.Cells提供[**个人字体配置**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs)可用于为您的工作簿指定单独或私有字体集的类。
