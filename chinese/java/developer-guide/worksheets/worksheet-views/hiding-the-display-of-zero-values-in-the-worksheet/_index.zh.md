{{% alert color="primary" %}} 

有时，您需要在电子表格中隐藏零值。这可能是个人偏好，也可能是格式化标准。

{{% /alert %}} 

要在Microsoft Excel中隐藏工作表中的零值（例如Microsoft Excel 2003）：

1. 从**工具**菜单中选择**选项**，然后选择**视图**选项卡。
1. 取消选中**零值**选项。
1. 点击**确定**。
