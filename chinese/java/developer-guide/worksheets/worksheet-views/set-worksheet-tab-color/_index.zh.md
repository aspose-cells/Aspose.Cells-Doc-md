{{% alert color="primary" %}}

Aspose.Cells允许您更改个别工作表标签的颜色，使其在其他工作表中突出显示。例如，您可以将Expenses设为红色，Sales设为绿色，Assets设为蓝色等。

{{% /alert %}}

## **使用Microsoft Excel设置工作表标签颜色**

1. 在当前工作表底部的选项卡表中右键单击一个选项卡。
1. 选择**选项卡颜色**。
1. 从调色板中选择一种颜色。
1. 单击**确定**。

**标签已经被着色为红色、绿色和蓝色**
