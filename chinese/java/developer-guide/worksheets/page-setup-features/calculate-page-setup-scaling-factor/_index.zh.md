当您使用设置页面设置缩放时**适合 n 页宽乘以 m 高**选项，Microsoft Excel 计算页面设置比例因子。您可以使用计算相同的东西[SheetRender.getPageScale()](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#PageScale)财产。此属性返回一个可以转换为百分比值的双精度值。例如，如果它返回 0.5079621076，则表示比例因子为 51%。
