## **可能的使用场景**

在 MS Excel 中没有直接选项可用于创建自定义纸张尺寸，但是，您可以在将 Excel 文件呈现为 PDF 文件格式时设置所需工作表的自定义纸张尺寸。本文档说明如何使用 Aspose.Cells API 设置工作表的自定义纸张尺寸。
