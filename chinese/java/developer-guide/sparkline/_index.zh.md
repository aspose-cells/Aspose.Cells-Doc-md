## **插入迷你图**
{{% alert color="primary" %}} 

迷你图是工作表单元格中的一个小图表，可提供数据的可视化表示。使用迷你图显示一系列值的趋势，例如季节性增加或减少、经济周期，或突出显示最大值和最小值。将迷你图放置在其数据附近以获得最大影响。迷你图分为三种类型：线型、柱型和堆叠型。
