## **插入一个Sparkline**
{{% alert color="primary" %}} 

Sparkline是工作表单元格中的微型图表，提供数据的可视表示。使用Sparkline显示一系列值的趋势，例如季节性增加或减少，经济周期，或突出最大和最小值。将Sparkline放置在其数据附近以获得最大效果。有三种类型的Sparkline：折线，柱状和堆叠。
