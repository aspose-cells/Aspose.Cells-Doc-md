## **可能的使用场景**
您可以使用 Aspose.Cells 读取或写入 ActiveX ComboBox 控件的值。请通过以下方式访问 ActiveX 控件[形状.ActiveXControl](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ActiveXControl)属性并通过检查其类型[ActiveXControl.类型](https://reference.aspose.com/cells/java/com.aspose.cells/activexcontrol#Type)财产，它应该返回[控件类型.ComboBox](https://reference.aspose.com/cells/java/com.aspose.cells/controltype#COMBO_BOX)值，然后将其类型转换为[组合框ActiveXControl](https://reference.aspose.com/cells/java/com.aspose.cells/ComboBoxActiveXControl)对象并读取或修改其各种属性。

请下载[示例 excel 文件](5473374.xlsx)用于以下示例代码和[输出excel文件](5473375.xlsx)由它产生。
