## **获取范围内的超链接**

 [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/range) 类提供了一个 [**getHyperlinks()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#Hyperlinks) 属性 getter，用于获取选定范围内的所有超链接。您还可以通过调用 [**Hyperlink.delete()**](https://reference.aspose.com/cells/java/com.aspose.cells/hyperlink#delete--) 方法删除超链接。
