超链接对于访问其他工作表或网站上的信息非常有用。Microsoft Excel允许用户在单元格文本和图像上添加超链接。图像超链接可以使工作表的导航更加便捷，例如，作为下一个和上一个按钮，或者链接到特定网站的标识。本文档解释了如何使用Aspose.Cells在工作表中插入图像超链接。

{{% /alert %}} 

Aspose.Cells允许您在运行时向电子表格中的图像添加超链接。可以设置和修改链接的屏幕提示和地址。
