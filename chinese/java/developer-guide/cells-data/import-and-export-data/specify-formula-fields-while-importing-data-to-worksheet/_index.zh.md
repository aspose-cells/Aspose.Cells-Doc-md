## **可能的使用场景**

在使用[**ImportTableOptions.setFormulas()**](https://reference.aspose.com/cells/java/com.aspose.cells/importtableoptions#IsFormulas)方法将数据导入工作表时，您可以指定公式字段。此方法使用布尔数组，其中值true表示该字段是公式字段。例如，如果第三个字段是公式字段，则数组中的第三个值将为true。
