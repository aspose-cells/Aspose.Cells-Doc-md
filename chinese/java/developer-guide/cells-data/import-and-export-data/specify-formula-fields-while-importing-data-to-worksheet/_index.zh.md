## **可能的使用场景**

您可以使用[**ImportTableOptions.setFormulas()**](https://reference.aspose.com/cells/java/com.aspose.cells/importtableoptions#IsFormulas)的方法在导入数据时指定公式字段。此方法接受布尔数组，其中值**true**表示字段是公式字段。例如，如果第三个字段是公式字段，则数组中的第三个值将为**true**。
