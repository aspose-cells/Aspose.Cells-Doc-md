{{% alert color="primary" %}} 

有时，您需要计算单元格值的宽度和高度，以适应单元格内的单元格值。Aspose.Cells提供[Cell.getWidthOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getWidthOfValue\(\))和[Cell.getHeightOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getHeightOfValue\(\))方法来实现此目的。通过使用这些方法，您可以计算单元格值的宽度和高度，然后分别设置该单元格的列宽和行高，这将调整或适应单元格内的单元格值。

另外，您还可以使用Aspose.Cells API来[自动调整单元格或单元格范围的行和列](/cells/zh/java/autofit-rows-and-columns/)。

{{% /alert %}}
