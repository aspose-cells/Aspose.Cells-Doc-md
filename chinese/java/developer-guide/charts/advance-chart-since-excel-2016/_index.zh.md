## **可能的使用场景**
Aspose.Cells支持读取和操作Microsoft Excel 2016图表，这些图表在Microsoft Excel 2013或更早版本中不存在。
## **读取和操作Excel 2016图表**
以下示例代码加载了包含 Microsoft Excel 2016 图表的 [源Excel文件](23166980.xlsx) ，并读取第一个工作表中的所有图表，逐个更改其标题以符合其图表类型。以下截图显示了执行代码之前的源Excel文件。您可以看到，所有图表的图表标题都相同。

![todo:image_alt_text](read-and-manipulate-excel-2016-charts_1.png)

以下截图显示了执行代码后的 [输出Excel文件](23166978.xlsx) 。您可以看到，图表标题已根据其图表类型更改。
