这段文字将告诉您如何创建股票图表，其中包括四种类型:
- **HLC** – 高-低-收。
- **OHLC** – 开-高-低-收。
- **VHLC** – 成交量-高-低-收。
- **VOHLC** – 成交量-开-高-低-收。

## **什么是股票图表?**

股票图表是用来跟踪交易资产价格变化的特定图表。这些资产包括商品、股票和加密货币。它们可以让您在同一个图表中看到随着时间变化的高低价和开盘收盘价。Aspose.Cells提供了4种股票图表，要使用这些图表，您必须有正确的数据集，并且必须以正确的顺序选择列。
