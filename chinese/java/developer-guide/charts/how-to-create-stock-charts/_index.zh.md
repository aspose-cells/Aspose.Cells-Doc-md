这一段将告诉您如何创建股票图表，其中包括四种类型：
- **HLC** -高低收。
- **OHLC** -开-高-低-收。
- **VHLC** -成交量-高-低-收。
- **VOHLC** -成交量-开-高-低-收。

## **什么是股票图表？**

股票图表是一种特定的图表，用于跟踪交易资产价格的变化。这些资产如商品、股票和加密货币。它们允许您在同一个图表中查看随时间变化的高低值，以及开盘和收盘值。Aspose.Cells提供了4种股票图表，要使用这些图表，您必须有正确的数据集并且必须按正确的顺序选择列。
