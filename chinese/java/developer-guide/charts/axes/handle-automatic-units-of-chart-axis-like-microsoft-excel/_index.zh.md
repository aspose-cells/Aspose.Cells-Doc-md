## **可能的使用场景**
早期版本的Aspose.Cells在将图表渲染为图像或PDF时无法正确处理图表轴的自动单位。现在，Aspose.Cells支持处理图表轴的自动单位。没有代码更改。只需将您的图表转换为图像或PDF，它将像Microsoft Excel一样渲染图表轴。
