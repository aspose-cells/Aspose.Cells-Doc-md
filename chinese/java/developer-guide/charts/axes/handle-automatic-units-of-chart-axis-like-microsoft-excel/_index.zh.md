## **可能的使用场景**
早期版本的Aspose.Cells不能正确处理图表轴的自动单位，当图表呈现为图像或PDF时。现在，Aspose.Cells支持处理图表轴的自动单位。没有代码更改。只需将您的图表转换为图像或PDF，它将呈现图表轴，就像Microsoft Excel呈现它一样。
