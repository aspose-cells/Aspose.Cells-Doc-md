{{% alert color="primary" %}}

Excel应用程序为图表的数据标签提供了**调整形状以适应文本**选项，以增加形状的大小以使文本适合其中。可以通过在Excel接口上选择图表上的任何数据标签来访问此选项。右键单击并选择**格式数据标签**菜单。在**大小和属性**选项卡上展开**对齐**以显示相关属性，包括**调整形状以适应文本**选项。
