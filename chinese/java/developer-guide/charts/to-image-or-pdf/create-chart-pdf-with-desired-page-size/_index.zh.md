## **可能的使用场景**
您可以使用 Aspose.Cells 创建具有所需页面大小的图表 PDF 并指定您希望如何在页面内对齐图表。您可以将图表对齐到顶部、底部、中心、左侧、右侧等。此外，输出图表可以在流中或磁盘上创建。
