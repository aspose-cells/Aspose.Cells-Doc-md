{{% alert color="primary" %}} 

如果您希望将图表的图例条目填充文本设置为无，以便它不显示在图例内部，请将[LegendEntry.IsTextNoFill](https://reference.aspose.com/cells/java/com.aspose.cells/legendentry#IsTextNoFill)设置为**true**。

{{% /alert %}}
