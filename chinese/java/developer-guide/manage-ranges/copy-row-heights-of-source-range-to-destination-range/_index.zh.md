{{% alert color="primary" %}} 

有时用户需要将源范围的行高复制到目标范围。 Aspose.Cells提供[粘贴类型.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS)枚举用于此目的。什么时候设置[粘贴选项.setPasteType()](https://reference.aspose.com/cells/java/com.aspose.cells/pasteoptions#PasteType)财产与[粘贴类型.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS)枚举然后源范围内所有行的高度将被复制到目标范围。
