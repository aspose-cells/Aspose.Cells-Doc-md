## **剪切和粘贴单元格**

Aspose.Cells 提供了在工作表内部移动和粘贴单元格的能力，使用 [**Cells**](https://reference.aspose.com/cells/java/com.aspose.cells/Cells) 集合的 [**insertCutCells**](https://reference.aspose.com/cells/java/com.aspose.cells/cells#insertCutCells-com.aspose.cells.Range-int-int-int-) 方法。 [**insertCutCells**](https://reference.aspose.com/cells/java/com.aspose.cells/cells#insertCutCells-com.aspose.cells.Range-int-int-int-) 接受以下参数。

- [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/Range)：要剪切的单元格范围。
- 行索引：要插入单元格的行索引。
- 列索引：要插入单元格的列索引。
- [**ShiftType**](https://reference.aspose.com/cells/java/com.aspose.cells/ShiftType)：列的移动方向。
