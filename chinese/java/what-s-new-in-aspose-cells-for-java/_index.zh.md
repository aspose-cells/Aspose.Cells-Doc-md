本页面描述了最近发布的Aspose.Cells新功能中最有趣的内容。

## Aspose.Cells for Java 24.7
* 在公式中支持DROP功能
* 改善对于复杂参数的LET和MAKEARRAY函数计算性能
* 在图表中支持星型字符的轮廓
* 支持将对象制作为装饰品
* 支持交换系列
* 改进TreeMap图表的显示
* 支持将"覆盖单元格"的图片转换为"单元格内"
* 默认支持加载本地Office云字体
* 在导入JSON数据时支持保留JSON模式
* 支持导出为AZW3格式文件
* 支持在为GridJS的单元格设置自动换行后自动设置高度
* 支持使用流写入方式生成JSON数据

## Aspose.Cells for Java 24.6
* 渲染增强功能：颜色饱和度和色温调整，双色调，Webp 图片
* 透视表增强功能：合并区域，透视字段的“显示数值为”
* 支持 Range.ToHtml()/ToImage() 以方便用户操作区域数据

## Aspose.Cells for Java 24.5
* 支持新功能: XMATCH, XOR
* 支持转置范围内的数据
* 支持获取高级筛选设置的属性
* 支持将单元格中放置的图片呈现为pdf/图片

## Aspose.Cells for Java 24.4
* 为用户的便利提供中断监视器的简单实现
* 在将OLE对象转换为pdf时支持嵌入附件
* 支持将ChartX的自动形状呈现为图片
* 支持将图片嵌入单元格（ms excel中的"放置在单元格内"操作）
* 改进导入Numbers文件：合并单元格，隐藏行/列，……等

## Aspose.Cells for Java 24.3
* 支持导入AZW3文件
* 支持以odf 1.3标准保存ods
* 支持计算EVALUATE函数

## Aspose.Cells for Java 24.2
* 支持新的函数: TEXTAFTER, TEXTBEFORE, TEXTSPLIT, TOROW, TOCOL
* 支持检测oxps格式
* 改进对大型数据集排序的性能

## Aspose.Cells for Java 24.1

* 支持 jakarta.servlet 用于 GridWeb
* 支持导入包含用户指定标头行/列和每个工作表总行/列的csv数据
* 支持导出带有自定义样式的html
* 支持合并区域的自动填充
* 支持从右向左、从下往上自动填充数据

## Aspose.Cells for Java 23.12

* 支持转换为pdf时符合PDF/A-2和PDF/A-3标准
* 支持对数据透视表进行离散分组计算
* 支持将单元格数据导出为通用列表
* 支持使用链接表导入交叉引用的智能标记
* 支持公式的MAKEARRAY函数

## Aspose.Cells for Java 23.11

* 支持将电子表格转换为PDF文件版本1.7(ISO32000-1:2008)

## Aspose.Cells for Java 23.10

* 支持将超链接直接设置为一个范围
* 支持同时复制多个工作表
* 支持检查一个范围的单元数据是否为空

## Aspose.Cells for Java 23.9

* 支持在公式中计算新函数：LAMBDA,LET,MAP,SCAN
* 支持导入/导出EPUB文件
* 支持过滤图表系列

## Aspose.Cells for Java 23.8

* 支持保留公式中的填充空格
* 支持以富文本格式替换/插入文本的Cell

## Aspose.Cells for Java 23.7

* 对一些新格式进行了增强的文件格式检测，如visio文件，OneNote包文件。

## Aspose.Cells for Java 23.6

* 支持导出和突出显示修订日志
* 支持按相反顺序从RowCollection中遍历现有的Row对象
* 支持刷新数据透视表的多个合并范围

## Aspose.Cells for Java 23.5

* 支持更新外部工作簿中外部工作表数据的外部引用到当前工作簿的本地工作表

* 支持在转换为pdf时添加水印

## Aspose.Cells for Java 23.4

* 支持计算函数 ENCODEURL

* 支持在列表对象中将公式设置到指定的单元格

## Aspose.Cells for Java 23.3

## Aspose.Cells for Java 23.2

## Aspose.Cells for Java 23.1
