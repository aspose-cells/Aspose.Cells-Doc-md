## **安装**
### **如何在IntelliJ IDEA上安装Aspose.Cells Java - Maven**
以下步骤演示了如何安装**Aspose.Cells Java for IntelliJ IDEA (Maven)**插件。

插件在IntelliJ IDEA的CE [official plugin website](https://goo.gl/R4pysl) -插件存储库中列为**Aspose.Cells Java for IntelliJ IDEA Maven**，因此您无需手动下载并从磁盘上安装它，但在IntelliJ IDEA中仍有此额外选项。

**Aspose.Cells Java for IntelliJ IDEA (Maven)**插件可以在IntelliJ IDEA中轻松（无需手动下载）安装。要执行此操作:

1. 从**文件**菜单中选择**设置**。
1.单击**插件**，然后在IntelliJ IDEA中浏览存储库。 

![todo:image_alt_text](lbtci11.jpg)
### **Aspose.Cells Maven 项目向导**
安装此插件将引入一个新的项目类型 - 

![todo:image_alt_text](aspose_small.png)

**Aspose.Cells Maven 项目** - 到 IntelliJ IDEA中，这将使您能够创建Aspose.Cells基于Maven的项目来使用[Aspose.Cells for Java API。](http://goo.gl/c1eSD2)。 

![todo:image_alt_text](m1du9a1.jpg)
### **Aspose.Cells示例向导**
安装后，在“新建”菜单下（右键单击->新建**或**文件菜单->新建）将出现一个用于创建[使用Aspose.Cells for Java API的源代码示例](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)的选项 -

## **使用**
### **Aspose.Cells Maven 项目向导**
要在IntelliJ IDEA中为使用[Aspose.Cells for Java API](http://goo.gl/c1eSD2)创建Aspose.Cells Maven项目，您需要连接到互联网。

然后按照以下步骤：

\1.选择**新建项目**。
\2.选择**Aspose.Cells Maven 项目** 
\3.单击**下一步**。 

![todo:image_alt_text](m1du9a1.jpg)


\4.为您的Maven项目提供**GroupId，ArtifactId**和 **Version**，然后单击**下一步**。

![todo:image_alt_text](khijrce.jpg)


` `您可以勾选“还下载示例源代码”复选框来下载API的示例用法示例（如上快照所示）

\6.提供**项目名称**和**位置**，就像创建任何Java应用程序时一样

![todo:image_alt_text](0gszuiu.jpg)


\7.单击**完成**。
` `这将从Aspose Cloud Maven存储库检索[Aspose.Cells for Java API](http://goo.gl/c1eSD2)最新的Maven依赖项引用并在pom.xml中配置它
` `如果您选择了“还下载示例源代码”复选框。还将从[Aspose.Cells for Java API GitHub存储库](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)开始下载示例

![todo:image_alt_text](eezoq3s.jpg)

![todo:image_alt_text](bujsm8v.jpg)

`     `Congratulations ! {{< emoticons/cross >}} You have successfully created Aspose.Cells Maven project for using [Aspose.Cells for Java API](http://goo.gl/c1eSD2).

![todo:image_alt_text](2oon4vh.jpg)

` `创建的Aspose.Cells for Java API Maven项目已准备好根据项目的需求进行修改。

` `如果您选择下载[示例源代码](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)，您可以使用Aspose.Cells 示例向导将下载的示例复制/创建到项目中。
### **Aspose.Cells示例向导**

然后在IntelliJ IDEA上按照以下步骤操作：

\1. 选择 **File** => **New** - 如下所示：
