## **介绍**

### **IntelliJ IDEA**

**IntelliJ IDEA**是由JetBrains开发的用于软件开发人员的Java集成开发环境（IDE）。最初是为Java创建的，目前支持广泛的语言，包括JavaScript、Groovy、HTML、CSS、RSS、R、Haskell、PHP、Ruby、Python、Scala、Clojure和Kotlin。

根据2010年**Infoworld**的报告，IntelliJ在四款主要Java编程工具（Eclipse、IntelliJ IDEA、NetBeans和Oracle JDeveloper）中获得了最高的测试中心得分

### **Java的Aspose.Cells**

[Aspose.Cells](https://products.aspose.com/cells/java/) for Java是一款Excel®电子表格报告组件，可让Java应用程序在不使用Microsoft Excel®的情况下创建和管理Excel®电子表格。Aspose.Cells for Java是一个功能非常丰富的组件，提供的功能远远超出了基本的数据导出功能。

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java for IntelliJ IDEA（Maven）是一个**IntelliJ IDEA插件**，可让您创建maven项目（即创建的项目包含Aspose.Cells for Java API maven依赖项）并下载该API的源代码示例以使用**Aspose.Cells for Java API**。

- 该插件可帮助您在您喜爱的IntelliJ IDEA中使用**Aspose.Cells for Java API**。

**向导**：
该插件包含两个向导-----

**向导1：** Aspose.Cells Maven项目

- 插件的**Aspose Maven项目向导**允许开发人员创建Aspose.Cells for Java Maven项目。
- **Aspose.Cells for Java API** maven依赖项的引用会自动从[Aspose Cloud Maven存储库](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells)获取并添加到pom.xml中。
- 创建的项目将始终包含最新版本的Aspose.Cells for Java的Maven依赖项。
- 向导步骤还提供了下载[用于使用Aspose.Cells for Java API的源代码示例](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)的选项

**向导2：** Aspose.Cells示例

- **Aspose.Cells示例向导**允许您将下载的[Aspose.Cells for Java API的源代码示例](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)创建/复制到您的项目中。所选类别的所有示例将被复制到项目的"com.aspose.cells.examples"包文件夹中，并与运行示例所需的"src/main/resources"文件夹中的资源一起复制。

- [Aspose.Cells for Java API的源代码示例](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)用于演示API的不同用法。

## **系统要求和支持的平台**

### **系统要求**

- Microsoft Windows 8/7/Vista/2003/XP（包括64位） / Linux GNOME或KDE桌面 / Mac OS X 10.5或更高版本
- 最低1 GB RAM，推荐2 GB RAM
- 300 MB硬盘空间 + 至少1 G的缓存
- 最小屏幕分辨率为1024x768
- JDK 1.7或更高版本推荐
- IntelliJ IDEA CE/UE (13.0.1, 13.1.2或14.0.2)
- 网络连接（建议2 MB或更快）

### **支持的平台**

插件支持以下IntelliJ IDEA版本:

- 13.0.1（社区版/旗舰版）
- 13.1.2（CE/UE）
- 14.0.2（CE/UE）

## **下载中**

### **正在下载IntelliJ IDEA**

您需要先安装IntelliJ IDEA CE/UE才能下载**Aspose.Cells Java for IntelliJ IDEA (Maven)**插件。

下载IntellliJ IDEA

1. Go to <https://www.jetbrains.com/idea/download/>
1. 选择您正在运行的操作系统
1. 点击UE或CE下载。

### **正在下载Aspose.Cells Java for IntelliJ IDEA插件(Maven)**

- 插件可以直接从[JetBrains插件托管URL](https://goo.gl/R4pysl)下载 - 在IntelliJ IDEA插件仓库中
  后来可以使用IDE选项安装:
  - 转到**文件** > **设置**...>**插件**
  - 选择"**从磁盘安装插件"**
- 或者可以使用IDE选项直接安装（无需从JetBrains插件托管URL手动下载）:
  - 转到**文件** > **设置**...>**插件**
  - 选择**"浏览存储库"**
  - 在搜索框中输入:"Aspose"
  - 选择"**Aspose Cells Java for IntelliJ IDEA Maven**"插件进行安装。
