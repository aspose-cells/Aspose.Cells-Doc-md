## **介绍**

### **IntelliJ IDEA**

**IntelliJ IDEA**是由JetBrains开发的用于软件开发人员的Java集成开发环境（IDE）。最初为Java创建，目前支持包括JavaScript、Groovy、HTML、CSS、RSS、R、Haskell、PHP、Ruby、Python、Scala、Clojure和Kotlin在内的多种语言。

根据**Infoworld** 2010年的一份报告，IntelliJ在四大Java编程工具（Eclipse、IntelliJ IDEA、NetBeans和Oracle JDeveloper）中获得了最高的测试中心分数。

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) for Java是一款能让Java应用程序无需使用Microsoft Excel®即可创建和管理Excel®电子表格的报表组件。Aspose.Cells for Java是一个功能丰富的组件，提供了远不止基本的数据导出功能。

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java for IntelliJ IDEA (Maven)是一个**IntelliJ IDEA插件**，可以创建包含Aspose.Cells for Java API maven依赖的maven项目，并下载API的源代码示例以使用**Aspose.Cells for Java API**。

- 该插件可帮助您在您喜欢的IntelliJ IDEA中使用**Aspose.Cells for Java API**。

**WIZARDS**:
该插件包含两个向导 -----

**向导1：** Aspose.Cells Maven项目

- 插件的**Aspose Maven Project Wizard**让开发人员创建Aspose.Cells for Java Maven项目。
- [Aspose.Cells for Java API](http://goo.gl/gtvhlQ)的maven依赖引用会自动从[Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells)中获取并添加到pom.xml中。
- 创建的项目将始终包含最新可用版本的Aspose.Cells for Java的Maven依赖。
- 向导步骤中还提供了下载[使用Aspose.Cells for Java API的源代码示例](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)的选项

**向导2：** Aspose.Cells 示例

- **Aspose.Cells Examples Wizard**允许您创建/复制已下载的[Aspose.Cells for Java API源代码示例](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)到您的项目中。所选类别的所有示例将复制到项目的"com.aspose.cells.examples"包文件夹中，以及"src/main/resources"文件夹中运行示例所需的必要资源。

- [Aspose.Cells for Java API源代码示例](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples)旨在演示API的不同用法。

## **系统要求和支持的平台**

### **系统要求**

- Microsoft Windows 8/7/Vista/2003/XP（包括64位）/Linux GNOME或KDE桌面/Mac OS X 10.5或更高版本
- 最低1GB RAM，推荐2GB RAM
- 300 MB硬盘空间 + 至少1G用于缓存
- 最低分辨率1024x768
- 建议使用JDK 1.7或更高版本
- 推荐使用IntelliJ IDEA CE/UE（13.0.1、13.1.2或14.0.2）
- 网络连接（建议2MB或更快）

### **支持的平台**

插件支持以下IntelliJ IDEA版本：

- 13.0.1（社区版/旗舰版）
- 13.1.2 (CE/UE)
- 14.0.2 (CE/ UE)

## **下载**

### **下载IntelliJ IDEA**

你需要先安装IntelliJ IDEA CE/UE，然后再下载**Aspose.Cells Java for IntelliJ IDEA（Maven）**插件。

下载IntelliJ IDEA

1. Go to <https://www.jetbrains.com/idea/download/>
1. 选择你正在使用的操作系统
1. 点击下载UE或CE。

### **下载Aspose.Cells Java for IntelliJ IDEA插件（Maven）**

- 插件可以直接从[JetBrains插件托管URL](https://goo.gl/R4pysl)下载 - 在IntelliJ IDEA插件仓库上
  然后可以使用IDE选项进行安装：
  - 转到**文件** > **设置**...>**插件**
  - 选择"**从磁盘安装插件**"
- 或者可以直接安装（无需从JetBrains插件托管URL手动下载）使用IDE选项：
  - 转到**文件** > **设置**...>**插件**
  - 选择**"浏览仓库"**
  - 在搜索框中输入：“Aspose”
  - 选择 “**Aspose Cells Java for IntelliJ IDEA Maven**” 插件进行安装。
