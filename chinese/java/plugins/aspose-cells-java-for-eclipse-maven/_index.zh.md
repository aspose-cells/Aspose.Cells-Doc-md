## **介绍**
### **Eclipse IDE**
Eclipse IDE是著名的Java集成开发环境（IDE）。该IDE绝对是Eclipse开源项目中最知名的产品。如今，它是市场份额约为60%的Java领先开发环境。

Eclipse IDE可以通过额外的软件组件进行扩展。Eclipse称这些软件组件为插件。几个开源项目和公司已经扩展了Eclipse IDE，或者在Eclipse框架之上创建了独立应用程序（Eclipse RCP）。
### **Java的Aspose.Cells**
[Aspose.Cells for Java](http://goo.gl/gtvhlQ)是屡获殊荣的Excel电子表格组件，允许Java开发人员将读取、编写和操作Excel®电子表格（XLS、XLSX、XLSM、XLSB、XLTX、SpreadsheetML、CSV、ODS）、HTML、MHTML、PDF和图像文件格式的能力嵌入到他们自己的Java应用程序中，而无需依赖于Microsoft Excel®。

[Aspose.Cells for Java](http://goo.gl/gtvhlQ)是一个成熟、可扩展且功能丰富的组件，提供许多远远超出其他供应商简单数据导出能力的功能。使用Aspose.Cells for Java，开发人员可以导出数据，将电子表格格式化到最细粒度级别，导入图像，创建图表，应用和计算复杂的公式，流式传输Excel®数据，以各种格式保存等等 - 所有这些都不需要依赖于Microsoft Excel®或Microsoft Office Automation。
### **Aspose.Cells Java for Eclipse (Maven)**
- Aspose.Cells Java for Eclipse（Maven）是由Aspose提供的**Eclipse IDE**的插件。该插件适用于在Java开发中使用Maven平台的开发人员，并希望在他们的项目中使用[Aspose.Cells for Java](http://goo.gl/gtvhlQ)。该插件允许您为使用[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)创建maven项目，并下载[代码示例](https://goo.gl/uxznrb)。 
- 该插件为在**Eclipse IDE**中舒适地使用[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)提供以下功能：

**向导**：
该插件包含两个向导-----

**       ****Aspose.Cells Maven项目（向导）**

- 此新项目向导允许开发人员为使用[Aspose.Cells for Java](http://goo.gl/gtvhlQ)创建Maven项目，路径为**新建 -> 项目 -> Maven -> Aspose.Cells Maven项目**。
- 从[Aspose Cloud Maven仓库](https://repository.aspose.com/webapp/)自动提取[Aspose.Cells for Java API](http://goo.gl/gtvhlQ) maven依赖项的引用，并将其添加到pom.xml中。
- 创建的项目将始终包含最新可用版本的[Aspose.Cells for Java API](http://goo.gl/gtvhlQ) Maven依赖项。
- 向导步骤还提供了下载[代码示例](https://goo.gl/uxznrb)以使用[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)的选项。

**       ****Aspose.Cells代码示例（向导）**

- 此新文件向导允许您将下载的[代码示例](https://goo.gl/uxznrb)复制到项目中，以便从**新建 -> 其他 -> Java -> Aspose.Cells代码示例**使用[Aspose.Cells for Java](http://goo.gl/gtvhlQ)。 
- 可用的示例以树状格式显示，用户可以按类别选择它们。
- 所选类别中的所有示例将被复制到项目的"**com.aspose.cells.examples**"包文件夹中，并包含在"**src/main/resources**"文件夹中运行所需的资源。
- [Aspose.Cells for Java API](http://goo.gl/gtvhlQ)的[代码示例](https://goo.gl/uxznrb)旨在演示API的各种功能。
- 向导还将查找并更新来自[Aspose.Cells for Java示例存储库](https://goo.gl/uxznrb)的新可用[代码示例](https://goo.gl/uxznrb)。

### **如何应用 Aspose 许可证？**
此插件使用 Aspose.Cells 的评估版本。一旦您对评估满意，您可以在 [Aspose 网站](https://purchase.aspose.com/buy) 购买许可证。
要移除评估信息和功能限制，应用产品许可证。购买产品后，您将收到一个许可证文件。请按照以下步骤应用许可证

- 确保许可证文件命名为 **Aspose.Cells.Java.lic**
- 将 **Aspose.Cells.Java.lic** 文件放置在包含 Aspose.Cells.jar 的文件夹中
- 使用以下代码激活许可证：

{{< highlight java >}}

 License license = new License();

license.setLicense("Aspose.Cells.Java.lic");

{{< /highlight >}}

#### **如何配置Aspose.Cells Java插件的Eclipse（Maven）源代码**
以下简单步骤将顺利地引导您成功配置在Eclipse IDE中**"Aspose.Cells Java插件的Eclipse（Maven）"**插件源代码

1. 下载/克隆源代码
1. 选择**文件** > **导入 > 一般 > 导入现有的项目到工作区**
1. 浏览到您下载的最新项目源代码
1. 选择要导入的Eclipse项目
1. 点击**完成**
1. Aspose.Cells Java插件的Eclipse源代码现在已准备好进行增强
