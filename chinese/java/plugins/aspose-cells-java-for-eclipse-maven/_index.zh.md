## **介绍**
### **Eclipse IDE**
Eclipse IDE是一款著名的Java集成开发环境（IDE）。该IDE无疑是Eclipse开源项目中最知名的产品。如今，它是市场份额约为60%的Java领先开发环境。

Eclipse IDE可以通过附加软件组件进行扩展。Eclipse称这些软件组件为插件。一些开源项目和公司扩展了Eclipse IDE，或者在Eclipse框架之上创建独立的应用程序（Eclipse RCP）。
### **Aspose.Cells for Java**
[Aspose.Cells for Java](http://goo.gl/gtvhlQ)是一款屡获殊荣的Excel电子表格组件，允许Java开发人员将读取、写入和操作Excel®电子表格(XLS、XLSX、XLSM、XLSB、XLTX、SpreadsheetML、CSV、ODS)、HTML、MHTML、PDF和图像文件格式嵌入到他们自己的Java应用程序中，而无需依赖Microsoft Excel®。

[Aspose.Cells for Java](http://goo.gl/gtvhlQ) 是一款成熟、可扩展且功能丰富的组件，提供了许多远远超出其他供应商简单数据导出功能的功能。使用Aspose.Cells for Java，开发人员可以导出数据，对电子表格进行最精细级别的格式化，导入图像，创建图表，应用和计算复杂的公式，流式传输Excel®数据，以各种格式保存等等 - 所有这些都不需要Microsoft Excel®或Microsoft Office自动化。
### **Aspose.Cells Java for Eclipse (Maven)**
- Aspose.Cells Java for Eclipse (Maven)是**Aspose**开发的用于**Eclipse IDE**的插件。该插件适用于使用Maven平台进行Java开发并希望在项目中使用[Aspose.Cells for Java](http://goo.gl/gtvhlQ)的开发人员。该插件允许你创建使用[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)的maven项目，并下载API的[代码示例](https://goo.gl/uxznrb)。

**WIZARDS**:
该插件包含两个向导-----

**       ****Aspose.Cells Maven Project（向导）**

- 通过此新建项目向导，开发人员可以创建**Maven**项目以使用[Aspose.Cells for Java](http://goo.gl/gtvhlQ)，路径为**New -> Project -> Maven-> Aspose.Cells Maven Project.**
- [Aspose.Cells for Java API](http://goo.gl/gtvhlQ)的maven依赖引用会自动从[Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/)中获取并添加到pom.xml中。
- 创建的项目将始终包含[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)的最新可用版本**Maven**依赖。
- 向导步骤还提供了下载[Code Examples](https://goo.gl/uxznrb)的选项，以使用[Aspose.Cells for Java API](http://goo.gl/gtvhlQ)。

**       ****Aspose.Cells代码示例（向导）**

- 通过此新建文件向导，您可以将下载的[Code Examples](https://goo.gl/uxznrb)复制到项目中，以使用[Aspose.Cells for Java](http://goo.gl/gtvhlQ)，路径为**New -> Other -> Java -> Aspose.Cells Code Example.** 
- 可用的示例以树形格式显示，用户可以选择分类地进行选择。
- 所选类别中的所有示例将与项目的"**com.aspose.cells.examples**"包文件夹一起复制，还将与运行示例所需的资源一起复制到"**src/main/resources**"文件夹中。
- [Aspose.Cells for Java API](http://goo.gl/gtvhlQ)的[Code Examples](https://goo.gl/uxznrb)旨在演示API的各种用法。
- 向导还将查找并更新新版本[Code Examples](https://goo.gl/uxznrb)从[Aspose.Cells for Java示例存储库](https://goo.gl/uxznrb)中获取。
## **系统要求和支持的平台**

- 推荐 Eclipse Mars.1 (4.5.1)
- Eclipse Juno 或更新版本。

### **如何应用Aspose许可证?**
此插件使用 Aspose.Cells 的评估版本。一旦您满意评估结果，您可以在 [Aspose 网站](https://purchase.aspose.com/buy) 购买许可证。
要移除评估消息和功能限制，需要应用产品许可证。购买该产品后，您将收到一个许可证文件。请按照以下步骤申请许可证。

- 确保许可证文件命名为 **Aspose.Cells.Java.lic**
- 将 **Aspose.Cells.Java.lic** 文件放置在包含 Aspose.Cells.jar 的文件夹中
- 使用以下代码激活许可证：

#### **如何配置 Aspose.Cells Java for Eclipse（Maven）的源代码**
以下简单步骤将顺利导致成功配置 **"Aspose.Cells Java for Eclipse（Maven）"** 插件源代码在 Eclipse IDE 中

1. 下载/克隆源代码。
1. 选择 **文件** > **导入 > 一般 > 从现有项目导入到工作区**
1. 浏览到您已下载的最新项目源代码
1. 选择要导入的 Eclipse 项目
1. 单击 **完成**
1. Aspose.Cells Java for Eclipse 插件代码现在已准备好加强。
