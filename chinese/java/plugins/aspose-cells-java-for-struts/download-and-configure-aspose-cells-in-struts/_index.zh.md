## **从源代码构建 Aspose.Cells Java for Struts 1.3**
从上述任何存储库中检出源代码后，应用以下 mvn 命令：

{{< highlight "java" >}}

 $ mvn -U clean package 

{{< /highlight >}}

这将在目标出纳器中构建“Strutsbookapp.war”。

要部署 .war 文件，只需将其复制到正在运行的 Apache tomcat 服务器 webapp 目录。
