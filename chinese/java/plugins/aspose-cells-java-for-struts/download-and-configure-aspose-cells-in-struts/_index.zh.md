## **从源码构建Struts 1.3中的Aspose.Cells Java**
从上述任一存储库检出源代码后，应用以下mvn命令：

{{< highlight java >}}

 $ mvn -U clean package 

{{< /highlight >}}

这将在目标目录中构建"Strutsbookapp.war"文件。

要部署.war文件，只需将其复制到运行中的Apache Tomcat服务器的webapp目录中。
