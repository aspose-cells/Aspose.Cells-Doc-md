## **从源代码构建Aspose.Cells Java for Struts 1.3**
从上述任何存储库检出源代码后，应用以下mvn命令：

{{< highlight java >}}

 $ mvn -U clean package 

{{< /highlight >}}

这将在目标文件夹中构建"Strutsbookapp.war"。

要部署.war文件，只需将其复制到正在运行的Apache tomcat服务器的webapp目录。
