## **支持**
- 如果您想在应用程序中查看已知/报告的问题（由用户或 QA 团队）。
- 或者您想报告您在应用程序中发现的任何问题
- 有任何改进建议或喜欢提出任何功能要求

**请使用以下任何项目的问题跟踪器：**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **扩展和贡献**
Aspose.cells Java for Struts 1.3 是开源的，其源代码可以在下面列出的主要社交编码网站上获得。鼓励开发人员下载源代码并通过建议或添加新功能或改进现有功能来做出贡献，以便其他人也可以从中受益。
