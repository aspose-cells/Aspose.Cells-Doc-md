## **系统要求**

Aspose.Cells for Java支持以下Java版本：

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- 或以上

[更多细节](/cells/zh/java/system-requirements/)

## **安装**

### **从 Maven 存储库安装 Aspose Cells for Java**

 Maven 是下载和安装 Aspose.Cells for Java 最简单的方法。

1. 在 Maven pom.xml 中指定 Aspose Maven 存储库配置/位置。
 2. 在 pom.xml 中定义 Aspose.Cells for Java API 依赖项。

[详细步骤更多](/cells/zh/java/installation/)

### **手动安装Aspose Cells for Java**

1. [下载 aspose-cells-{version}-java.zip](https://downloads.aspose.com/cells/java/)
1. 解压压缩包，进入\JDK 1.6\目录，解压aspose-cells-{version}-java.zip。
1. 进入\JDK 1.6\aspose-cells-{version}-java\lib\目录，使用Eclipse或Intellij手动安装JAR包。
