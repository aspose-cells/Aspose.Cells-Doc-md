##  **系统要求**

Aspose.Cells for Java 支持以下 Java 版本：

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- 或以上

[更多细节](/cells/zh/java/system-requirements/)

##  **安装**

###  **从 Maven 存储库安装 Aspose Cells for Java**

 Maven 是下载和安装 Aspose.Cells for Java 的最简单方法。

1. 在 Maven pom.xml 中指定 Aspose Maven 存储库配置/位置。
 2. 在 pom.xml 中定义 Aspose.Cells for Java API 依赖项。

[更多步骤详情](/cells/zh/java/installation/)

###  **手动安装 Aspose Cells for Java**

1. [下载 aspose-cells jars](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/)到一个文件夹中。
1. 进入该文件夹，使用Eclipse或Intellij手动安装JAR包。
