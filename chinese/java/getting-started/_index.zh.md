## **系统要求**

Aspose.Cells for Java支持以下Java版本：

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- 或更高版本

[更多详情](/cells/zh/java/system-requirements/)

## **安装**

### **从Maven仓库安装Aspose Cells for Java**

Maven是下载和安装Aspose.Cells for Java的最简单方式。 

1. 在您的Maven pom.xml中指定Aspose Maven仓库的配置/位置。 
2. 在您的pom.xml中定义Aspose.Cells for Java API依赖项。 

[更多详细步骤](/cells/zh/java/installation/)

### **手动安装Aspose Cells for Java**

1. [下载aspose-cells jars](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/)到一个文件夹中。
1. 进入文件夹，使用Eclipse或Intellij手动安装JAR包。
