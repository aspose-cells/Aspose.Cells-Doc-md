## **系统要求**

Aspose.Cells for Java 支持以下 Java 版本：

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- 或更高

[更多详情](/cells/zh/java/system-requirements/)

## **安装**

### **从Maven仓库安装Aspose Cells for Java**

Maven 是下载和安装 Aspose.Cells for Java 的最简单方式。 

1. 在Maven的pom.xml中指定Aspose Maven仓库的配置/位置。 
2. 在您的 pom.xml 中定义 Aspose.Cells for Java API 依赖。 

[详情请参阅更多步骤](/cells/zh/java/installation/)

### **手动安装Aspose Cells for Java**

1. 将aspose-cells jars（https://releases.aspose.com/java/repo/com/aspose/aspose-cells/）下载到一个文件夹中。
1. 进入文件夹，使用Eclipse或Intellij手动安装JAR包。
