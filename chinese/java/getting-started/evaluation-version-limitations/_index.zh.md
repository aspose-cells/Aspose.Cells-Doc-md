##  **评估版限制**

评估版**Aspose.Cells**产品（未指定许可证）提供完整的产品功能，但仅限于在一个程序中打开 100 个文件以及带有评估水印的额外工作表。

限制如下所示：

###  **第一个限制：打开文件的数量**

运行程序时，您只能打开 100 个 Excel 文件。如果您的应用程序超过此数量，则会抛出异常。

###  **第二个限制：带有评估水印的工作表**
该工作表将始终显示为活动工作表。仅在许可版本中，您可以将活动工作表设置为其他工作表。
