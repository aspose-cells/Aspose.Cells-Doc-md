有时我们会遇到以下问题：为什么我们应该使用Aspose产品而不是xlsx4j？这个问题很容易回答：功能和功能。
{{% /alert %}}


## **xlsx4j**
xlsx4j 是一个用于创建和操作 Microsoft Open XML Excel xlsx 文件的开源（ASLv2）Java库，类似于 docx4j 用于操作 Microsoft Open XML Word docx 文件。
它类似于 Microsoft 的 OpenXML SDK，但适用于 Java。docx4j 使用 JAXB 创建内存对象表示。
您需要花时间了解 JAXB 和 Open XML 文件结构。

在 [docx4java.org](https://www.docx4java.org/trac/docx4j) 上阅读更多信息。
## **Aspose.Cells for Java**
Aspose.Cells for Java是一款屡获殊荣的Excel电子表格组件,可让Java开发人员在其自己的Java应用程序中嵌入读取、写入和操作Excel®电子表格(XLS、XLSX、XLSM、XLSB、XLTX、SpreadsheetML、CSV、ODS)、HTML、MHTML、PDF和图像文件格式的能力,而无需依赖Microsoft Excel®。

Aspose.Cells for Java是一个成熟、可扩展且功能丰富的组件，提供了许多远超其他供应商简单数据导出功能的功能。使用Aspose.Cells for Java，开发人员可以导出数据、格式化电子表格到最细粒度的水平，导入图像，创建图表，应用和计算复杂公式，流式传输Excel®数据，以各种格式保存等等 - 所有这些都无需Microsoft Excel®或Microsoft Office Automation的需求。

在 [aspose.com](https://products.aspose.com/cells/java/) 上阅读更多信息。

## **为什么不使用 xlsx4j**
xlsx4j只能解析和保存Microsoft Open XML Excel xlsx文件，Aspose.Cells for Java可以处理所有Excel文件格式(XLS、XLSX、XLSM、XLSB、XLTX、SpreadsheetML、CSV、ODS)并支持各种文件格式转换。

## **Aspose.Cells Java 用于 xlsx4j**
Aspose.Cells 用于 xlsx4j 项目展示了使用 Aspose.Cells Java API 与 xlsx4j 相比可以执行不同任务的方式。该项目还涵盖了仅在 Aspose.Cells API 中而不在 xlsx4j 中处理电子表格的功能。

该项目对于想要比较 xlsx4j 与 Aspose.Cells 或从 xlsx4j 迁移到 Aspose.Cells 的开发人员非常有帮助。
## **系统要求和支持的平台**
### **系统要求**
以下是执行 Aspose.Cells Java 用于 xlsx4j 的系统要求：

- 已安装 Java 1.4 或更高版本。
- 已下载 Aspose.Cells 组件。
- 已下载 docx4j 库。
### **支持的平台**
以下是支持的平台：

- Aspose.Cells 8.3.0 或以上版本。
- docx4j 3.1.0 或以上库。
- Java 集成开发环境（Eclipse、NetBeans、IntelliJ 等）。
## **下载**
可在以下所有社交编码网站上下载运行示例的发布版本:

### **GitHub**
- [Aspose.Cells Java for Xlsx4j - v 1.0](https://github.com/aspose-cells/Aspose.Cells-for-Java/releases/tag/Aspose.Cells_Java_for_Xlsx4j-v1.0.0)

要配置**Aspose.Cells for Java API**，请在您的**pom.xml**中包含Aspose存储库URL和Aspose.Cells for Java API Maven依赖，如下所示:
### **Aspose Maven 仓库**
{{< highlight java >}}

 <repositories>

    <repository>

        <id>aspose-maven-repository</id>

        <name>Aspose Maven Repository</name>

        <url>http://repository.aspose.com/repo/</url>

    </repository>

</repositories>

{{< /highlight >}}
### **Aspose.Cells for Java Maven依赖**
{{< highlight java >}}

 <dependencies>

    <dependency>

        <groupId>com.aspose</groupId>

        <artifactId>aspose-cells</artifactId>

        <version>8.6.0</version>

    </dependency>

</dependencies>

{{< /highlight >}}

**注意：** 请使用可用的 Aspose API 最新版本。
