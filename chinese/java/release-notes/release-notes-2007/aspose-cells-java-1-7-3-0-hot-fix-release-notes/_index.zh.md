亲爱的客户，

我们已经发布了 Aspose.Cells for Java V1.7.3！包括以下增强和修复：

- 支持插入/删除范围。
支持打开和保存加密的Excel文件。
增强 Cell.getStringValue() 方法以支持更多日期/数字格式。
更新了网络演示。
支持用JDK1.4计算公式。
修复了 License.setLicense() 方法的错误。
修复了设置自定义数字格式的错误。
修复了从工作簿中删除工作表的错误。
