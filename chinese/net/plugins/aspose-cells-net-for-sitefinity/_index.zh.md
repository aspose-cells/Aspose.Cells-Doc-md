## **入门**
### **介绍**
#### **什么是 Sitefinity？**
Sitefinity 是一个基于 ASP.NET 的现代 Web 内容管理系统 (CMS)，为业务专业人员提供直观的 Web 内容管理和强大的开发环境。以下是我们为这个流行的 CMS 创建的项目。
#### **Aspose.Cells for .NET**
Aspose.Cells for .NET 是一个高级类库 for .NET，它使您能够直接在 .NET 应用程序中执行各种电子表格处理任务。

使用 Aspose.Cells，您可以生成、修改、转换、呈现和打印电子表格，而无需使用 Microsoft Excel。

Aspose.Cells for .NET 支持XSL、XSLX、OpenOffice等多种格式。
