## **介绍**
Aspose .NET数据库数据导出至Excel对Umbraco模块允许用户将数据直接从本地或远程数据库表、视图和自定义查询导出到Microsoft Excel或OpenOffice电子表格。该模块演示了Aspose.Cells提供的强大的电子表格构建功能。该模块的初始版本丰富了以下酷功能，以使导出过程简单易用。
### **模块特性**
此附加组件的初始版本具有以下功能：

- 允许连接本地 MS SQL Server 数据库
- 允许连接远程 MS SQL Server 数据库
- 从连接的数据库中填充所有表
- 从连接的数据库中填充所有视图
- 允许编写自定义查询
- 自动调整列宽以适合内容长度
- 允许在Excel单元格中跳过超过32k的字符串（LoadOptions）
- 将表头列格式设置为粗体文本
- 允许用作数据源（表、视图、自定义查询）
- 将数据导出到 Microsoft Excel 文档（.xls、.xlsx 和 .xlsb）
- 将数据导出到制表符分隔文本文档（*.txt）
- 将数据导出到CSV（逗号分隔）文件（*.csv）
- 将数据导出到OpenDocument电子表格（*.ods）
- 在导出之前选择所需的输出格式。
- 导出的文档会自动发送到浏览器进行下载。

## **系统要求和支持的平台**
### **系统要求**
为设置Aspose .NET数据库数据导出至Excel对Umbraco模块，您需要满足以下要求：

- Umbraco 6.2.5和Umbraco 6版本
- 使用MS SQL Server的Umbraco
- Microsoft .Net Framework 4.0

**注意：** 不支持Umbraco 7及以上版本。我们期待听取您的反馈，并在下个版本中添加对Umbraco 7的支持。
### **支持的平台**
此模块支持的所有版本

- 运行在ASP.NET 4.0上的Umbraco 6.0

## **安装**
下载后，请按照以下步骤将此包安装到您的 Umbraco 网站中：

1. 登录Umbraco的**Developer**部分，例如`http://www.myblog.com/umbraco`
1. 从树中展开 **Packages** 文件夹
1. 从这里可以通过选择**安装本地包**或浏览**Umbraco Package Repository**来安装软件包。
1. 如果安装 **本地包**，不要解压包，而是将 zip 加载到 Umbraco 中
1. 按照屏幕上的指示操作。

**注意:** 安装时可能出现‘超出最大请求长度’的错误。您可以通过更新Umbraco Web.config文件中的‘maxRequestLength’值轻松解决此问题。
<httpRuntime requestValidationMode="2.0" enableVersionHeader="false" maxRequestLength="25000" />
## **使用**
安装完Aspose .NET Database Data Exporter to Excel for Umbraco模块后，开始在您的网站上使用它真的很简单。请按照以下简单步骤开始使用。

1. 确保您已登录到Umbraco**开发者**部分，例如 `http://www.myblog.com/umbraco/`
1. 在屏幕左下角的部分列表中，单击**设置**。
1. 展开**模板**节点并选择要添加到的模板，例如Textpage。
1. 选择要将导出按钮添加到的选定模板中的位置。通常您可能希望将其添加到页面的右上角或页面底部。
1. 在顶部功能区中单击**插入宏**。
1. 从**选择宏**（Aspose .NET Database Data Exporter to Excel for Umbraco）中，选择最近安装的Aspose .NET Database Data Exporter to Excel for Umbraco宏，然后单击**确定**。

请查看下面的屏幕截图以获取详细信息。 

![todo:image_alt_text](umbraco-database-data-exporter-to-excel_2)

您已成功将Aspose .NET Database Data Exporter to Excel模块添加到您的页面。

![todo:image_alt_text](umbraco-database-data-exporter-to-excel_1)

1. 输入或使用预填的 MS SQL Server 连接字符串。
1. 选择数据源类型 (表、视图、自定义查询)。
1. 选择或输入数据源 (表、视图、自定义查询)。
1. 选择导出类型。
1. 单击导出数据。
1. 期望的文件将会自动下载。
