## **介绍**
Aspose 更新导入工具是用于 Microsoft Dynamic CRM 的开源插件。此插件能够自动将数据从 CRM 更新和/或导出到 Aspose.Cells 电子表格。我们不需要手动执行任何操作并安装此插件，使我们的工作更轻松、更快捷。

此插件提供自动更新和/或从 CRM 导出的功能。单击一次即可将记录导入/导出 CRM。

这个插件的主要特点是：

- 建立与所需 CRM 的连接。
- 允许用户选择所需的实体和字段。
- 从 CRM 中获取记录显示在网格中。
- 将记录导出到 Aspose.Cells 电子表格。
- 从 Aspose.Cells 电子表格显示在网格中获取记录。
- 将记录从网格导出到 CRM。
## **系统要求和支持的平台**
### **系统要求**
为了安装和使用 Aspose .Net Update Import Tool for Microsoft Dynamics CRM，您需要安装以下 CRM 版本之一

- Microsoft 动态客户关系管理 2013。
- Microsoft 动态 CRM 2015。
- Microsoft 在线动态 CRM。

如果您在安装或使用此插件时发现任何问题，请随时与我们联系。
### **支持的平台**
此插件适用于所有版本的 Microsoft Dynamics CRM：

- Microsoft 动态客户关系管理 2013。
- Microsoft 动态 CRM 2015。
- Microsoft 在线动态 CRM。
