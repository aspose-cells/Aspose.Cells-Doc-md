## **介绍**
Aspose更新导入工具是一个开源附加组件，可在Microsoft Dynamic CRM中使用。该附加组件能够自动更新和/或导出CRM数据到Aspose.Cells电子表格中。我们无需手动操作，只需安装此附加组件即可使工作更轻松和更快。

该附加组件提供自动从CRM更新和/或导出的功能。单击即可将记录从CRM导入/导出到CRM。

此附加组件的主要功能包括：

- 建立与所需CRM的连接。
- 允许用户选择所需的实体和字段。
- 在网格中显示来自CRM的记录。
- 将记录导出到Aspose.Cells电子表格。
- 从Aspose.Cells电子表格中获取记录并在网格中显示。
- 将记录从网格导出到CRM。
## **系统要求和支持的平台**
### **系统要求**
为了安装和使用Aspose .Net更新导入工具用于Microsoft Dynamics CRM，您需要安装以下任一CRM版本

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.

如果您在安装或使用此附加功能时遇到任何问题，请随时与我们联系。
### **支持的平台**
此插件将适用于所有版本的Microsoft Dynamics CRM:

- Microsoft Dynamics CRM 2013.
- Microsoft Dynamics CRM 2015.
- Microsoft Dynamics CRM Online.
