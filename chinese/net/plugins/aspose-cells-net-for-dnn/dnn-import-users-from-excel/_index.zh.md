## **介绍**
Aspose.NET从Excel导入用户模块允许开发人员从Excel文件中导入批量用户。该模块展示了Aspose.Cells强大的电子表格功能。
### **模块特性**
该模块的初始版本具有以下酷功能，使导入过程简单易用:

- 允许您上传Excel文件并自动过滤现有用户。
- 自动跳过现有用户，以避免混乱。
- 为新创建的用户生成动态密码。
- 为新创建的用户分配"已注册用户"和"订阅者"安全角色。

## **安装**
下载后，请按照以下步骤将模块安装到您的DNN网站中:

1. 以Host或其他超级用户级别的帐户登录网站。
1. 导航到**Host**菜单并选择**Extensions**。
1. 点击**Install Extension Wizard**。
1. 按照指示浏览到已下载zip文件的位置，选择文件，然后点击**打开**。
1. 点击**下一页**，接受许可证，继续安装。完成后，单击**返回**按钮。

请查看[DNN的此模块安装视频](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)以获取更多详情。
## **使用**
安装完Aspose .NET Cells Import Users from Excel for DNN模块后，您可以在网站上轻松使用它。请按照以下简单的步骤开始使用：

1. 确保您已使用主机或管理员级别账户登录到DNN。
1. 导航至您要添加导出模块的页面。
1. 从顶部功能区选择**模块**，然后点击**添加新模块**。 

1. 从列表中选择 'Aspose Import Users from Excel module for DNN' 并将其拖放到位置。

您已成功添加Aspose .NET Cells Import Users from Excel module for DNN模块到您的页面。文件上传控件将与“模板文件”链接一起显示。您可以下载模板文件，并按照模板文件的格式进行内容格式化。上传Excel文件后，所有用户将自动处理并筛选出现有用户。您可以选择要导入的用户并按“导入用户”按钮。
