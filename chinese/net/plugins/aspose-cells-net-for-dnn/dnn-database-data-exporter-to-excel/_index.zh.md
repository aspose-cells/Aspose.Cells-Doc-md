## **介绍**

Aspose .NET DNN 数据导出到 Excel 模块允许用户直接从本地或远程数据库表、视图和自定义查询中将数据导出到 Microsoft Excel 或 OpenOffice 电子表格。该模块展示了由 Aspose.Cells 提供的功能强大的电子表格构建功能。

### **模块特性**

该模块的初始版本具有以下酷功能，以使导出过程简单易用

- 允许连接本地 MS SQL Server 数据库
- 允许连接远程 MS SQL Server 数据库
- 填充来自已连接数据库的所有表
- 填充来自已连接数据库的所有视图
- 允许编写自定义查询
- 自动适应列的内容长度。
- 允许跳过超过32k的字符串填充到 Excel 单元格（LoadOptions）
- 将标题列格式设置为粗体文本
- 允许使用作为数据源（表，视图，自定义查询）
- 将数据导出到Microsoft Excel文档（.xls，.xlsx和.xlsb）
- 将数据导出到制表符分隔的文本文档（*.txt）
- 将数据导出为CSV（逗号分隔） (*.csv)
- 将数据导出为OpenDocument电子表格（*.ods）
- 在导出之前选择所需的输出格式的选项
- 导出的文档会自动发送到浏览器进行下载

## **安装**

下载后，请按照以下步骤将模块安装到您的DNN网站中：

1. 以主机或其他超级用户级别帐户登录到站点。
1. 导航至 **主机** 菜单，选择 **扩展**。
1. 单击 **安装扩展向导**。
1. 按照指示浏览到下载的zip文件位置，选择文件，然后点击 **打开**。
1. 点击 **下一步**，接受许可，继续安装。安装完成后，点击 **返回** 按钮。

请查看DNN提供的[此模块安装视频](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)以获取更多详细信息。

## **使用**

安装 Aspose .NET Database Data Expoeter for DNN 模块后，在您的网站上开始使用它非常简单。请按照以下简单步骤开始使用

1. 确保您已使用主机或管理员级别帐户登录到DNN。
1. 导航到要添加导入模块的页面。
1. 从列表中选择 ‘Aspose从Excel导入用户的DNN模块’，将其拖放到位置中。
1. 从列表中选择‘Aspose.NET DNN数据导出器到Excel使用Aspose.Cells’，并将其拖动到位置。

您已成功将Aspose .NET数据库数据导出器添加到DNN模块到您的页面。

1. 输入或使用预填的MS SQL Server连接字符串
1. 选择数据源类型（表、视图、自定义查询）
1. 选择或输入数据源（表、视图、自定义查询）
1. 选择导出类型
1. 单击导出数据
1. 所需文件将自动下载。
