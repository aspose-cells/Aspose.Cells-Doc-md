## **介绍**

Aspose .NET DNN 数据导出器到 Excel 模块允许用户直接从本地或远程数据库表、视图导出数据，并通过自定义查询将数据导出到 Microsoft Excel 或 OpenOffice 电子表格。该模块演示了Aspose.Cells提供的强大的电子表格构建功能。

### **模块特点**

该模块的初始版本增加了以下很酷的功能，使导出过程简单易用

- 允许连接本地 MS SQL Server 数据库
- 允许连接远程 MS SQL Server 数据库
- 从连接的数据库填充所有表
- 从连接的数据库填充所有视图
- 允许编写自定义查询
- 自动调整列以适应内容长度。
- 允许跳过 excel 单元格中超过 32k 的字符串 (LoadOptions)
- 将标题列格式应用为粗体文本
- 允许用作数据源（表、视图、自定义查询）
- 将数据导出到 Microsoft Excel 文档（.xls、.xlsx 和 .xlsb）
- 将数据导出到制表符分隔的文本文档 (*.txt)
- 将数据导出到 CSV（逗号分隔）(*.csv)
- 将数据导出到 OpenDocument 电子表格 (*.ods)
- 导出前选择所需输出格式的选项。
- 导出的文档会自动发送到浏览器进行下载。

## **安装中**

下载后，请按照以下步骤将模块安装到您的 DNN 网站：

1. 以主持人或其他超级用户级别帐户登录您的站点。
1. 导航到**主持人**菜单并选择**扩展**.
1. 点击**安装扩展向导**.
1. 按照指示浏览到下载的 zip 文件的位置，选择它然后单击**打开**.
1. 点击**下一个** 接受许可证，继续安装。完成后单击**返回**按钮。

请检查[本模块安装视频](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)来自 DNN 的更多详细信息。

## **使用**

安装 Aspose .NET Database Data Expoeter for DNN 模块后，在您的网站上开始使用它真的很简单。请按照以下简单步骤开始：

1. 确保您使用主持人或管理员级别的帐户登录到 DNN。
1. 导航到要添加导出模块的页面。
1. 选择**模块**其次是**添加新模块**从顶部功能区。
1. 从列表中选择“Aspose.NET DNN Data Exporter to Excel Using Aspose.Cells”并将其拖动到位。
