## **介绍**

Aspose .NET DNN 数据导出至 Excel 模块允许用户直接从本地或远程数据库表、视图和自定义查询导出数据到 Microsoft Excel 或 OpenOffice 电子表格。该模块展示了由 Aspose.Cells 提供的强大的电子表格构建功能。

### **模块特性**

该模块的初始版本增加了以下酷炫功能，使导出过程简单易用：

- 允许连接本地 MS SQL Server 数据库
- 允许连接远程 MS SQL Server 数据库
- 从连接的数据库中填充所有表
- 从连接的数据库中填充所有视图
- 允许编写自定义查询
- 自动调整列宽以适合内容长度
- 允许在Excel单元格中跳过超过32k的字符串（LoadOptions）
- 将表头列格式设置为粗体文本
- 允许用作数据源（表、视图、自定义查询）
- 将数据导出到 Microsoft Excel 文档（.xls、.xlsx 和 .xlsb）
- 将数据导出到制表符分隔文本文档（*.txt）
- 将数据导出到CSV（逗号分隔）文件（*.csv）
- 将数据导出到OpenDocument电子表格（*.ods）
- 在导出之前选择所需的输出格式。
- 导出的文档会自动发送到浏览器进行下载。

## **安装**

下载后，请按照以下步骤将模块安装到您的DNN网站中:

1. 以Host或其他超级用户级别的帐户登录网站。
1. 导航到**Host**菜单并选择**Extensions**。
1. 点击**Install Extension Wizard**。
1. 按照指示浏览到已下载zip文件的位置，选择文件，然后点击**打开**。
1. 点击**下一页**，接受许可证，继续安装。完成后，单击**返回**按钮。

请查看[DNN的此模块安装视频](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)以获取更多详情。

## **使用**

安装Aspose .NET Database Data Expoeter for DNN模块后，开始在您的网站上使用非常简单。请按照以下简单步骤开始使用：

1. 确保您已使用主机或管理员级别账户登录到DNN。
1. 导航至您要添加导出模块的页面。
1. 从顶部功能区选择**模块**，然后点击**添加新模块**。
1. 从列表中选择‘Aspose.NET DNN Data Exporter to Excel Using Aspose.Cells’，并将其拖放到位置。

您已成功将Aspose .NET Database Data Exporter to Excel for DNN模块添加到页面。

1. 选择或输入数据源 (表、视图、自定义查询)。
1. 选择导出类型。
1. 单击导出数据。
