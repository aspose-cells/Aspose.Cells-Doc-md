## **支持的平台**
Aspose.Cells for .NET产品涵盖了大多数流行的开发环境和部署平台。 

|**功能**|**描述**|
| :- | :- |
|[.NET Framework](/cells/zh/net/system-requirements/#net-frameworks)|使用Aspose.Cells for .NET构建ASP.NET、Web服务、WinForms或任何其他类型的应用程序，适用于.NET Framework 2.0或更高版本，包括客户端配置版本。完全支持32位和64位。
|[.NET Standard 2.0](/cells/zh/net/system-requirements/#net-standard)|'netStandard'文件夹包含用于实现.NET Standard 2.0的框架的程序集，如.NET Core 2.1/3.1。最新版本的Aspose.Cells API也支持.NET 5.0、.NET 6.0、.NET 7.0和.NET8.0。|
|[COM](/cells/zh/net/system-requirements/)|如果您必须使用较早的.NET技术进行开发，COM可能是一个救命稻草。|
|[Mono](/cells/zh/net/system-requirements/)|您可以使用Aspose.Cells for .NET构建与Mono兼容的应用程序。
|[Windows Azure](/cells/zh/net/system-requirements/)|Aspose.Cells for .NET也支持在Windows Azure上运行。
