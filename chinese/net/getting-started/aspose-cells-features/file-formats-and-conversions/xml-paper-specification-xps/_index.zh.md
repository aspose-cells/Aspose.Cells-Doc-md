## **Aspose.Cells 中的 XPS**
加载到 Aspose.Cells 中的任何文档都可以转换为 XPS。 Aspose.Cells 使用相同的高保真分页布局和呈现引擎在其所有支持的固定布局格式（例如 PDF、XPS 等）中生成输出。

值得注意的是，Aspose.Cells 直接生成 XPS（包括 TrueType 字体的子集化），而不依赖于 .NET Framework 3.5 中的 Windows Presentation Foundation 类，这使得 Aspose.Cells 可以在运行较早版本的 .NET 或完全不同的平台上生成 XPS。
