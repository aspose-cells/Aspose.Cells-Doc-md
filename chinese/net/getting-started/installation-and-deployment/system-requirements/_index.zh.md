{{% alert color="primary" %}} 

Aspose.Cells for .NET 组件无需在客户端或服务器端上安装 Microsoft Excel。

{{% /alert %}}

## **开发环境**
您可以使用Aspose.Cells for .NET在任何针对.NET平台的开发环境中开发应用程序，但以下环境受到明确支持:

- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- 微软Visual Studio 2012
- 微软Visual Studio 2013
- 微软Visual Studio 2015
- 微软Visual Studio 2017
- 微软Visual Studio 2019
- 微软Visual Studio 2022

{{% alert color="primary" %}} 

要使用Aspose.Cells，机器本身不需要安装Microsoft Excel，因为该库本身就是电子表格管理引擎。但是，若要查看组件生成的Excel文件，则需要在机器上至少安装Microsoft Excel Viewer。

{{% /alert %}} 
## **Aspose.Cells网格套件系统要求**
Aspose.Cells网格套件由三个控件组成：

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb
- Aspose.Cells.GridJs

这些控件的系统要求分别描述。

{{% alert color="primary" %}} 

要使用这些控件，运行Aspose.Cells.GridDesktop和Aspose.Cells.GridWeb的机器不需要安装Microsoft Excel，因为Aspose.Cells.GridDesktop和Aspose.Cells.GridWeb可以打开和保存Excel电子表格，无需Microsoft Excel。
