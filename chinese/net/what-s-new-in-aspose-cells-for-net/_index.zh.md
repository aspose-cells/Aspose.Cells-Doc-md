本页面介绍了最新版本中引入的最有趣的 Aspose.Cells 新功能。

## Aspose.Cells for .NET 24.5
* 支持新功能：XMATCH, XOR
* 支持转置范围的数据
* 支持获取高级筛选设置的属性
* 支持将单元格中放置的图片呈现为pdf/图像
* 支持将样式复制到GridJS的自动填充
* 支持GridJS的图表颜色设置

## Aspose.Cells for .NET 24.4
* 为用户提供简单的中断监视器实现以方便使用
* 支持将OLE对象转换为pdf时嵌入附件
* 支持将ChartX的自动形状呈现为图像
* 支持将图像嵌入到单元格中（ms excel中图片的“放置在单元格”操作）
* 对导入Numbers文件进行改进：合并单元格，隐藏行/列等

## Aspose.Cells for .NET 24.3
* 支持导入AZW3文件
* 支持用odf 1.3标准保存ods
* 支持计算EVALUATE函数
* 支持GridDesktop的筛选：Top10，日期，文本
* 在GridJS的插入菜单和帮助提示中添加更多公式
* 为GridJS的公式菜单添加分类功能
* 当计算结果为日期时间时，为GridJS自动设置单元格日期格式

## Aspose.Cells for .NET 24.2
* 支持新功能：TEXTAFTER,TEXTBEFORE,TEXTSPLIT,TOROW,TOCOL
* 支持检测oxps格式
* 提高对大型数据集排序的性能
* 改进GridDesktop的有关筛选的操作：数值筛选，关键字搜索，清除筛选等

## Aspose.Cells for .NET 24.1

* 支持NET8
* 支持导入具有用户指定标题行/列和每个工作表总行/列的csv数据
* 支持使用自定义样式导出html
* 支持合并区域的自动填充
* 支持从右到左、从下到上自动填充数据

## Aspose.Cells for .NET 23.12

* 在转换为pdf时支持PDF/A-2和PDF/A-3合规性
* 支持对数据透视表的离散分组计算
* 支持将单元格数据导出到通用列表
* 支持使用链接表导入智能标记的交叉引用
* 支持公式的MAKEARRAY函数
* 支持使用GridJS旋转角度呈现单元格文本
* 支持对GridDesktop的数据表进行更多导入/导出选项
* 支持在GridDesktop中以旋转方式渲染图片
* 支持调整GridDesktop的工作表顺序
* 支持在GridDesktop中显示/隐藏公式警告消息选项 

## Aspose.Cells for .NET 23.11

* 支持将电子表格转换为PDF文件版本1.7(ISO32000-1:2008)
* 支持在GridJS中插入表单控件
* 支持在GridJS中显示单元格背景的渐变填充
* 支持在GridDesktop中复制/删除/插入/重命名工作表
* 在GridDesktop中进行粘贴操作时自动扩展最大行/列
* 支持GridDesktop的SheetPrintingPreview
* 提供GridDesktop的.NET Core版本 

## Aspose.Cells for .NET 23.10

* 当目标区域的大小与源区域不同时，支持自适应粘贴操作
* 可直接为范围设置超链接
* 支持同时复制多个工作表
* 支持检查某个范围的单元格数据是否为空

## Aspose.Cells for .NET 23.9

* 支持计算公式中的新函数：LAMBDA、LET、MAP、SCAN
* 支持导入/导出EPUB文件
* 支持筛选图表系列
* 支持GridDesktop的本地化
* 为.net7-windows平台提供GridWeb

## Aspose.Cells for .NET 23.8

* 支持保留公式中的填充空间
* 支持使用格式设置替换/插入富文本单元格的文本
* 支持在GridJs中呈现activex控件和表单控件以支持旋转按钮
* 为GridJs支持更多类型的验证：文本长度/小数，选择列表，公式表达式

## Aspose.Cells for .NET 23.7

* 增强了GridJS的许多功能，如渲染控件，combobox的服务器端保存回传等等
* 增强了对一些新格式的文件格式检测，如visio文件，OneNote包文件
* 为CSharp X509Certificate2支持XAdES签名
* 支持在Blazor Web Assembly中运行

## Aspose.Cells for .NET 23.6

* 支持导出和突出显示修订日志
* 支持从RowCollection中反向遍历现有的Row对象
* 支持刷新数据透视表的多个合并范围
* 在GridJs中显示图表工作表
* 在GridJs中显示单元格溢出内容
* 在GridJs中显示背景图片
* 在GridJs中按日期时间筛选

## Aspose.Cells for .NET 23.5

## Aspose.Cells for .NET 23.4

* 支持在ListObject中设置指定单元格的公式
* 支持将工作簿另存为xhtml
* 支持在GridWeb中呈现activex控件和表单控件

## Aspose.Cells for .NET 23.3

* 支持设置/读取/保存新函数：LET，ENCODEURL

## Aspose.Cells for .NET 23.2

## Aspose.Cells for .NET 23.1
