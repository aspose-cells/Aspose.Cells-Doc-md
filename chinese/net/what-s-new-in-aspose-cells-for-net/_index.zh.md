本页介绍了最近版本中引入的最有趣的新 Aspose.Cells 功能。

##  Aspose.Cells for .NET 24.3
* 支持导入AZW3文件
* 支持以odf 1.3标准保存ods
* 支持计算EVALUATE函数
* 支持 GridDesktop 过滤器：Top10、日期、文本
* 在插入菜单中添加更多公式以及 GridJS 的帮助提示
* 为 GridJS 的公式菜单添加分类功能
* 当计算结果为日期时间时，自动将GridJS的单元格设置为日期格式

##  Aspose.Cells for .NET 24.2
* 支持新功能：TEXTAFTER、TEXTBEFORE、TEXTSPLIT、TOROW、TOCOL
* 支持检测oxps格式
* 提高大型数据集排序的性能
* GridDesktop过滤操作增强：数字过滤、关键词搜索、清除过滤。

##  Aspose.Cells for .NET 24.1

* 支持NET8
* 支持导入带有用户指定的标题行/列以及每张表的总行/列的 csv 数据
* 支持导出带有自定义样式的html
* 支持合并区域自动填充
* 支持从右到左、从下到上自动填充数据

##  Aspose.Cells for .NET 23.12

* 转换为 pdf 时支持 PDF/A-2 和 PDF/A-3 合规性
* 支持计算数据透视表的离散分组
* 支持将单元格数据导出到通用列表
* 支持使用智能标记的链接表导入交叉引用
* 支持公式的MAKEARRAY函数
* 支持 GridJS 以旋转角度渲染单元格文本
* 支持 GridDesktop 导入/导出数据表的更多选项
* 支持GridDesktop旋转渲染图片
* 支持调整GridDesktop的工作表顺序
* 支持GridDesktop显示/隐藏公式警告消息的选项

##  Aspose.Cells for .NET 23.11

* 支持将电子表格转换为PDF文件版本1.7(ISO32000-1:2008)
* 支持为GridJS插入表单控件
* 支持GridJS显示单元格背景的渐变填充
* 支持GridDesktop复制/删除/插入/重命名工作表
* 支持GridDesktop粘贴操作时自动扩展最大行/列
* 支持 GridDesktop 的 SheetPrintingPreview
* 提供 .NET GridDesktop 核心构建

##  Aspose.Cells for .NET 23.10

* 支持目标范围大小与源范围大小不同时的自适应粘贴操作
* 支持直接设置范围的超链接
* 支持同时复制多个工作表
* 支持检查某一区域的单元格数据是否为空

##  Aspose.Cells for .NET 23.9

* 支持计算公式中的新函数：LAMBDA、LET、MAP、SCAN
* 支持导入/导出 EPUB 文件
* 支持过滤图表系列
* 支持 GridDesktop 本地化
* 为.net7-windows平台提供GridWeb

##  Aspose.Cells for .NET 23.8

* 支持在公式中保留填充空间
* 支持用单元格的富文本格式替换/插入文本
* 支持在GridJs中渲染旋转按钮的activex控件和表单控件
* 支持 GridJs 更多类型的验证：文本长度/小数、选择列表、公式表达式

##  Aspose.Cells for .NET 23.7
