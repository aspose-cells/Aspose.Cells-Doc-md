本页面描述了最近发布的Aspose.Cells新功能中最有趣的内容。

## Aspose.Cells for .NET 24.6
* 渲染增强功能：颜色饱和度和色温调整，双色调，Webp 图片
* 透视表增强功能：合并区域，透视字段的“显示数值为”
* 支持 Range.ToHtml()/ToImage() 以方便用户操作区域数据

## Aspose.Cells for .NET 24.5
* 支持新功能: XMATCH, XOR
* 支持转置范围内的数据
* 支持获取高级筛选设置的属性
* 支持将单元格中放置的图片呈现为pdf/图片
* 支持将样式复制到GridJS的自动填充
* 支持GridJS的图表颜色设置

## Aspose.Cells for .NET 24.4
* 为用户的便利提供中断监视器的简单实现
* 在将OLE对象转换为pdf时支持嵌入附件
* 支持将ChartX的自动形状呈现为图片
* 支持将图片嵌入单元格（ms excel中的"放置在单元格内"操作）
* 改进导入Numbers文件：合并单元格，隐藏行/列，……等

## Aspose.Cells for .NET 24.3
* 支持导入AZW3文件
* 支持以odf 1.3标准保存ods
* 支持计算EVALUATE函数
* 支持GridDesktop的筛选功能：前10，日期，文本
* 在GridJS的插入菜单和帮助提示中添加更多公式
* 为GridJS的公式菜单添加分类特性
当计算结果为日期时间时, 自动将单元格设置为GridJS的日期格式

## Aspose.Cells for .NET 24.2
* 支持新的函数: TEXTAFTER, TEXTBEFORE, TEXTSPLIT, TOROW, TOCOL
* 支持检测oxps格式
* 改进对大型数据集排序的性能
* 增强查询操作功能: GridDesktop的数字过滤, 关键词搜索, 清除筛选

## Aspose.Cells for .NET 24.1

* 支持 NET8
* 支持导入包含用户指定标头行/列和每个工作表总行/列的csv数据
* 支持导出带有自定义样式的html
* 支持合并区域的自动填充
* 支持从右向左、从下往上自动填充数据

## Aspose.Cells for .NET 23.12

* 支持转换为pdf时符合PDF/A-2和PDF/A-3标准
* 支持对数据透视表进行离散分组计算
* 支持将单元格数据导出为通用列表
* 支持使用链接表导入交叉引用的智能标记
* 支持公式的MAKEARRAY函数
* 支持GridJS渲染具有旋转角度的单元格文本
* 对于GridDesktop导入/导出数据表格，支持更多选项
* 支持在GridDesktop中旋转图片渲染
* 支持调整GridDesktop中工作表的顺序
* 支持在GridDesktop中显示/隐藏公式警告消息的选项 

## Aspose.Cells for .NET 23.11

* 支持将电子表格转换为PDF文件版本1.7(ISO32000-1:2008)
* 支持在GridJS中插入表单控件
* 支持在GridJS中显示单元格背景的渐变填充
* 支持在GridDesktop中复制/删除/插入/重命名工作表
* 在GridDesktop进行粘贴操作时支持自动扩展最大行/列
* 支持GridDesktop的SheetPrintingPreview
* 提供GridDesktop的.NET Core构建 

## Aspose.Cells for .NET 23.10

* 支持在目标范围的大小与源范围不同时进行自适应粘贴操作
* 支持将超链接直接设置为一个范围
* 支持同时复制多个工作表
* 支持检查一个范围的单元数据是否为空

## Aspose.Cells for .NET 23.9

## Aspose.Cells for .NET 23.8

* 支持保留公式中的填充空格

## Aspose.Cells for .NET 23.7
