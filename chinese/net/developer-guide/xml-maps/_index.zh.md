{{% alert color="primary" %}}

 Aspose.Cells 允许您使用[**工作簿.ImportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/importxml/index)方法。您可以通过以下步骤使用 Microsoft Excel 导入 XML 地图

- 选择**开发商**标签
- 点击**进口**在 XML 部分并按照所需的步骤进行操作。

您需要提供 XML 数据才能完成导入。这里有一个[示例 XML 数据](5115037.txt)您可以用于测试。
