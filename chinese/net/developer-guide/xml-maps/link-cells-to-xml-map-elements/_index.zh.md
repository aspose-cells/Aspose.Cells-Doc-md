## **可能的使用场景**

您可以使用 Aspose.Cells 将单元格链接到 XML Map 元素。请使用 [**Cells.LinkToXmlMap()**](https://reference.aspose.com/cells/net/aspose.cells/cells/methods/linktoxmlmap) 方法来实现此目的。

## **将单元格链接到 Xml Map 元素**

以下示例代码加载了包含 XML Map 的 [源 Excel 文件](5115471.xlsx)，然后将单元格 A1、B2、C3、D4、E5 和 F6 链接到XML Map 元素 FIELD1、FIELD2、FIELD4、FIELD5、FIELD7 和 FIELD8，并将工作簿保存为 [输出 Excel 文件](5115467.xlsx)。

如果您打开 [输出 Excel 文件](5115467.xlsx) 并单击“开发人员”>“源”按钮，您将看到单元格与 XML Map 元素相链接，并且如图所示将被 Microsoft Excel 高亮显示。
