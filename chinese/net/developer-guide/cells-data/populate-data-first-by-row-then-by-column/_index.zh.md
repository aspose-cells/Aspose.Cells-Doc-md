{{% alert color="primary" %}} 

首先按行然后按列填充电子表格中的数据可以提高整体性能。

{{% /alert %}} 

将数据放入序列 A1、B1、A2、B2 比 A1、A2、B1、B2 更快。如果工作表中有很多单元格，并且您遵循第二个顺序，即逐行填充数据，则此技巧可以使程序更快。
