##  **可能的使用场景**

当您将数据导入工作表时，您可以使用以下命令指定公式字段：[**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas) 。该属性采用布尔数组，其中值**真的**表示该字段是公式字段。例如，如果第三个字段是公式字段，则数组中的第三个值将为 *true**。
