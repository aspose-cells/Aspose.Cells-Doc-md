## **可能的使用场景**

您可以使用[**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas)在导入数据到工作表时指定公式字段。此属性接受布尔数组，其中值true表示该字段是公式字段。例如，如果第三个字段是公式字段，则数组中的第三个值将为true。
