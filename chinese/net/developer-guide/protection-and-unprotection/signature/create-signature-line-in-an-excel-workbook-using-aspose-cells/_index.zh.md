Microsoft Excel提供了在Excel工作簿中添加**签名行**的功能。您可以通过单击**插入**选项卡，然后从**文本**组中选择**签名行**来添加签名行。

Aspose.Cells也提供了此功能，并已为此目的公开了[**Picture.SignatureLine**](https://reference.aspose.com/cells/net/aspose.cells.drawing/picture/properties/signatureline)属性。本文将解释如何使用此属性使用Aspose.Cells添加签名行。
