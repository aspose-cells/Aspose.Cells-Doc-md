Aspose.Cells也提供了这个功能，并暴露了[**图片.签名行**](https://reference.aspose.com/cells/net/aspose.cells.drawing/picture/properties/signatureline)为此目的的财产。本文将解释如何使用此属性添加使用 Aspose.Cells 的签名行。
