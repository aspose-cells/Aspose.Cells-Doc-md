Aspose.Cells支持修改现有的SQL数据连接。本文将解释如何使用Aspose.Cells修改SQL数据连接的不同属性。

你可以通过**数据 > 连接**菜单命令在Microsoft Excel中添加或查看数据连接。

同样，Aspose.Cells提供了使用Workbook.DataConnections集合访问和修改数据连接的方法。
