## 在C#中检查VBA项目是否已受保护和锁定以供查看

Aspose.Cells允许您检查Excel文件的VBA（Visual Basic for Applications）项目是否受保护并锁定以供查看。为此，API提供[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing)属性。如果已锁定以供查看，则[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing)属性将返回**true**。
