## 在 C# 中检查 VBA 项目是否被保护和锁定以供查看

Aspose.Cells 允许您检查 Excel 文件的 VBA (Visual Basic for Applications) 项目是否受到保护并锁定以供查看。为此，API 提供了[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing)财产。如果它被锁定以供查看，则[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing)财产回报**真的**.
