## **获取数据透视表的外部连接数据源**

Aspose.Cells 提供获取数据透视表的外部连接数据源的功能。为此，API 提供了 [**PivotTable**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) 类的 [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) 属性。 [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) 属性返回 [**ExternalConnection**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) 对象。以下代码片段演示了使用 [**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) 属性获取数据透视表的外部连接数据源的用法。
