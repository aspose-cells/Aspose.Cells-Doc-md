有时，您希望格式化数据透视表单元格。例如，您希望对数据透视表单元格应用背景颜色。Aspose.Cells提供了两个方法[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall)和[**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format)，您可以用于此目的。

[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) 将样式应用于整个数据透视表，而[**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format)仅将样式应用于数据透视表的单个单元格。
