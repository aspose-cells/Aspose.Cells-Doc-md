Aspose.Cells API 可以在工作表上添加 WordArt 水印，使 WordArt 成为可以在工作表上移动和定位的对象。还可以锁定 WordArt 对象，防止任何编辑、移动和选择。本文介绍了该方法的使用[**Shape.SetLockedProperty**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/methods/setlockedproperty)进行水印某些方面的锁定。

{{% /alert %}}
