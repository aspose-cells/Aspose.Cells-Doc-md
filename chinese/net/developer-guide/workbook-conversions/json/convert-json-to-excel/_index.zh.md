Aspose.Cells 支持将 Json(JavaScript Object Notation) 文件转换为 Excel 工作簿。

{{% /alert %}}

## **将 JSON 转换为 Excel 工作簿**
无需考虑如何将 JSON 转换为 Excel 文件，因为 Apose.Cells for .NET 库有最佳决策。 Aspose.Cells API 提供了将JSON格式转换为电子表格的支持。您可以使用[**JsonLoad选项**](https://reference.aspose.com/cells/net/aspose.cells/JsonLoadOptions)类以指定将 JSON 导入工作簿的其他设置。
