## **可能的使用场景**

Aspose.Cells允许您使用 [**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/tablecssid) 属性前缀表元素样式。假设您将此属性设置为**MyTest_TableCssId**，那么您将找到以下表元素样式

{{< highlight java >}}

 table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}
