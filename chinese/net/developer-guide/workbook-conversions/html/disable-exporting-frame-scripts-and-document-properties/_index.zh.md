Aspose.Cells 在将工作簿转换为 HTML 时导出框架脚本和文档属性。 Aspose.Cells for .NET 的 8.6.0 版本引入了一个选项，允许您有选择地禁用导出框架脚本和文档属性。请使用 HtmlSaveOptions.ExportFrameScriptsAndProperties 属性禁用导出。

{{% /alert %}}
