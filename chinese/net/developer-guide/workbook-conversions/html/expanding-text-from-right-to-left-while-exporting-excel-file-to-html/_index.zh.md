{{% alert color="primary" %}} 

Aspose.Cells现在支持导出Excel文件到HTML时从右向左展开文本，从v8.9.0.0开始实现该功能。现在，如果您的源 Excel 文件包含任何从右向左扩展的文本，则 Aspose.Cells 会将其正确导出为 HTML。

{{% /alert %}}
