Aspose.Cells 为需要在将 Microsoft Excel 文件保存为 HTML 或 MHT 格式时需要呈现更好布局的开发人员提供了一个有用的 HtmlSaveOptions.PresentationPreference 属性。该属性的默认值为 false。我们建议将此属性设置为 true 以获得更具吸引力的 Excel 报告显示。
