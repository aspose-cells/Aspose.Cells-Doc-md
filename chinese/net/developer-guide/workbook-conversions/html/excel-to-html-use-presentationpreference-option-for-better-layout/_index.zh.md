{{% alert color="primary" %}} 

Aspose.Cells为需要将Microsoft Excel文件保存为HTML或MHT格式时获得更好布局的开发人员提供了HtmlSaveOptions.PresentationPreference属性。 该属性的默认值为false。 我们建议将此属性设置为true，以获得Excel报告的更吸引人呈现。
