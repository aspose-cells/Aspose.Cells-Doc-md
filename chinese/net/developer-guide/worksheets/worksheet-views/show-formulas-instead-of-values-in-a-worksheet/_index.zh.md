{{% alert color="primary" %}}

可以使用 Excel 在 Microsoft 中显示公式而不是计算值**显示公式**选项从**公式**丝带。显示公式时，Microsoft Excel 在工作表中显示公式。您可以使用 Aspose.Cells 实现相同的目的。

{{% /alert %}}

Aspose.Cells提供了[**工作表.ShowFormulas**](https://reference.aspose.com/cells/net/aspose.cells/worksheet/properties/showformulas)财产。将此设置为**真的**设置Microsoft Excel显示公式。
