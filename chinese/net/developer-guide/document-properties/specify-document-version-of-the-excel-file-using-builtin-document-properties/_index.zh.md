## **可能的使用场景**

您可以右键单击文件并选择属性 > 详细信息来更改Excel文件的**版本号**，然后编辑**版本号**字段。请使用 [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion) 属性使用Aspose.Cells API进行编程更改。
