Microsoft 自 Office 2007 以来，Office 在应用程序窗口顶部用功能区替换了菜单和工具栏。功能区是可自定义的。
Aspose.Cells 让您可以

- 保留 Ribbon XML 而不解析它，
- 读取和写入功能区 XML 而不解析它，
- 获取和设置功能区 XML 数据。

如果要更改功能区 XML，则必须使用 XML 解析器或其他功能区 XML 工具对其进行解析。
