{{% alert color="primary" %}}

Aspose.Cells 允许用户从 Excel 文件中提取主题相关数据。例如，您可以提取应用于工作簿的主题名称以及应用于单元格或单元格边框的主题颜色等。

您可以通过 Microsoft Excel 的“页面布局”>“主题”命令向工作簿应用主题。
