Aspose.Cells 允许用户从 Excel 文件中提取主题相关数据。例如，您可以提取应用于工作簿的主题名称和应用于单元格或单元格边框的主题颜色等。

您可以使用 Microsoft Excel 通过页面布局 > 主题命令将主题应用到您的工作簿。
