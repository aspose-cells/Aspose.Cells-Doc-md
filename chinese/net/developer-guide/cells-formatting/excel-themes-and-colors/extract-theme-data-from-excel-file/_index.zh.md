Aspose.Cells允许用户从Excel文件中提取主题相关数据。例如，您可以提取应用于工作簿的主题名称和应用于单元格或单元格边框的主题颜色等。

您可以通过“页面布局”>“主题”命令使用 Microsoft Excel 将主题应用到工作簿。
