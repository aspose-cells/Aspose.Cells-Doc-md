{{% alert color="primary" %}}

Aspose.Cells允许用户从Excel文件中提取与主题相关的数据。例如，您可以提取应用于工作簿的主题名称以及应用于单元格或单元格边框的主题颜色等。

您可以在Microsoft Excel中通过“页布局” > “主题”命令向工作簿应用主题。
