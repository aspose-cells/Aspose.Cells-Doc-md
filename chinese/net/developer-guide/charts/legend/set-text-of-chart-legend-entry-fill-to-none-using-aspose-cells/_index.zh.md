如果您想将图表图例条目填充的文本设置为无，以便它不会显示在图表图例中，请设置[**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/net/aspose.cells.charts/legendentry/properties/istextnofill)为*真**。

{{% /alert %}}
