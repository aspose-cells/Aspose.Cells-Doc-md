{{% alert color="primary" %}}

如果您想将图表图例条目的文本填充设置为无，这样它就不会显示在图表图例中，那么请设置[**图例条目.IsTextNoFill**](https://reference.aspose.com/cells/net/aspose.cells.charts/legendentry/properties/istextnofill)到**真的**.

{{% /alert %}}
