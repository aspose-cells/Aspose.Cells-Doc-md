{{% alert color="primary" %}}

Microsoft Excel 2013 允许用户在图表的数据标签内环绕或展开文本。默认情况下，图表数据标签内的文本处于环绕状态。

Aspose.Cells提供了[**数据标签.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped)您可以设置 True 或 False 的属性分别启用或禁用数据标签的文本换行。
