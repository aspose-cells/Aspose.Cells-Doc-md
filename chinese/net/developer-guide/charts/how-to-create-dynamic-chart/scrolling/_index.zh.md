## **可能的使用场景**
动态滚动图是一种用于显示随时间变化的数据的图形表示。它旨在提供数据的实时视图，允许用户跟踪连续更新和趋势。随着添加新数据，图表会不断更新自身，并自动滚动以显示最新信息。

动态滚动图通常在各行业中广泛应用，如金融、股票市场分析、天气追踪和社交媒体分析。它们使用户能够可视化和分析数据模式，并根据实时信息做出明智决策。

这些图表通常是互动的，允许用户放大或缩小，浏览历史数据，并调整时间间隔。它们通常支持多个数据系列，提供不同指标和它们之间相关性的全面视图。

总的来说，动态滚动图是监控和分析时序数据的宝贵工具，促进实时决策，并增强数据可视化能力。

## **笔记**
在生成的文件中，您可以操作滚动条，而图表会动态计算最新的10组数据。这是通过样例代码中的“OFFSET”公式来实现的:

```
"=OFFSET(Sheet1!$B$2,Sheet1!$G$20,0,Sheet1!$H$20,1)"
```

您可以尝试将单元格“Sheet1!$H$20”中的数字“10”更改为“15”，这样动态图表将计算最新的15组数据。现在我们已成功使用Aspose.Cells创建了一个动态滚动图。
