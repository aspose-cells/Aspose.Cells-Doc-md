{{% alert color="primary" %}}

Aspose.Cells允许您为不同对象（例如图表的绘图区域、图表区域或图例框）设置渐变、纹理、图案或图片作为填充效果。本文展示了如何将图片添加到图表的背景中。

{{% /alert %}}
