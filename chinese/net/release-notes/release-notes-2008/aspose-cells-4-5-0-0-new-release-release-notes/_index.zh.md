什么是新的？



- ` `将工作表转换为图像。
- ` `支持将 Excel 文件转换为 HTML 文件格式。
- ` `支持数据从左到右排序。



有什么改进？



- ` `公式计算引擎支持更多功能
- ` ` 在自动过滤器中支持更多自定义过滤器。
- ` `增强的 Excel2007 Xlsx 支持
- ` ` 增强的 SpreadsheetML 支持
- ` `增强的xls2Pdf转换
- ` `增强图表2图像转换。
- ` `增强智能标记
- ` `支持从模板文件中获取单选按钮的链接单元格。



什么是固定的？



- ` ` 读写图表中的错误。
- ` `计算公式中的错误
- ` ` 打开和保存一些特殊的 xls 文件时出现错误。
- ` ` Cells.ImportDataTable 方法中的错误
- ` ` 插入行/列时更新图表引用时出现错误。
