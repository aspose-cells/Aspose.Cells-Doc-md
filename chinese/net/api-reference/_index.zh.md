{{% alert color="primary" %}} 

Aspose.Cells for .NET是一款电子表格编程库，允许软件开发人员在其应用程序内部操作和转换电子表格文件。Aspose.Cells for .NET支持流行的电子表格（XLS、XLSX、XLSM、XLSB、XLTX、XLTM、CSV、SpreadsheetML、ODS）文件格式，允许将Excel文件导出为PDF、XPS、HTML、MHTML、纯文本、TIFF、JPG、PNG、BMP和SVG。

最新的API参考可以在[Aspose.Cells for .NET API参考](https://reference.aspose.com/cells/net)找到。

{{% /alert %}}
