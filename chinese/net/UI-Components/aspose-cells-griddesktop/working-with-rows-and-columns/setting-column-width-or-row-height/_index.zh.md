{{% alert color="primary" %}} 

本主题讨论使用 Aspose.Cells.GridDesktop API 设置列宽和行高。有时，工作表单元格包含的值比原始单元格宽。这使用户难以阅读单元格内容。有些用户不喜欢使用右箭头键来查看单元格中的完整值。为了解决此类问题，开发人员可以更改列宽。更改行高可以解决类似的问题。

{{% /alert %}}

1. 访问要更改宽度的列。
1. 设置列宽。
