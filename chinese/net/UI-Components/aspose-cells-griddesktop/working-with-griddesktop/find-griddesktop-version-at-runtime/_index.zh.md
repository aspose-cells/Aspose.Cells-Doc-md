## **可能的使用场景**
有时，在运行时查找 GridDesktop 版本很重要。例如，您想确保您使用的是正确的版本，或者您正在调试您的应用程序并且您想知道当前使用的 GridDesktop 的版本。 GridDesktop 提供 Aspose.Cells.GridDesktop.GridDesktop.GetVersion() 方法，该方法在运行时返回 GridDesktop 版本。
