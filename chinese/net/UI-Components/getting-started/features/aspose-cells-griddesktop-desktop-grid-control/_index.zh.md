## **Aspose.Cells.GridDesktop – 桌面网格控件**
### **导入/导出功能**
- 从Excel电子表格中导入内容。
- 直接从DataTable对象中导入数据。
- 导出内容到Excel电子表格。
### **工作表功能**
- 添加、插入或删除工作表。
- 重命名工作表。
- 对工作表数据进行排序。
### **行和列功能**
- 添加、插入或删除行和列。
- 隐藏或显示行和列。
- 冻结或取消冻结行和列。
### **单元格功能**
- 支持数字格式设置。
- 控制文本对齐。
- 管理边框设置。
- 控制字体设置。
- 合并或拆分单元格。
### **格式特性**
- 控制网格的行为、外观和感觉。
- 使用格式刷将焦点单元格的样式和字体应用到其他单元格。
- 轻松操作数据和单元格样式。
### **实用功能**
 - 添加或删除单元格注释。
 - 插入或删除图片。
 - 添加或删除超链接。
 - 添加或删除验证规则。
 - 在单元格中添加或删除控件（如复选框、组合框等）。
 - 单元格的键盘导航。
### **安全功能**
 - 通过锁定或隐藏保护您的单元格。
### **高级功能**
 - 支持创建和计算公式。
 - 在不同的网格事件下执行您所需的操作。
### **支持的 Microsoft Excel 版本**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **支持的平台**
- Windows 窗体
