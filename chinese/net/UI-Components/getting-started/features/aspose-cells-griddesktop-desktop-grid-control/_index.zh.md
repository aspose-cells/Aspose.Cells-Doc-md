## **Aspose.Cells.GridDesktop – 桌面网格控件**
### **导入/导出功能**
- 从 Excel 电子表格导入内容。
- 直接从 DataTable 对象导入数据。
- 将内容导出到 Excel 电子表格。
### **工作表功能**
- 添加、插入或删除工作表。
- 重命名工作表。
- 对工作表数据进行排序。
### **行列特征**
- 添加、插入或删除行和列。
- 隐藏或显示行和列。
- 冻结或解冻行和列。
### **Cell 特点**
- 支持数字格式设置。
- 控制文本对齐方式。
- 管理边框设置。
- 控制字体设置。
- 合并或拆分单元格。
### **格式化功能**
- 控制网格的行为、外观和感觉。
- 使用格式刷将焦点单元格的样式和字体应用到其他单元格。
- 轻松处理数据和单元格样式。
### **实用功能**
- 添加或删除单元格注释。
- 插入或删除图片。
- 添加或删除超链接。
- 添加或删除验证规则。
- 在单元格中添加或删除控件（如 CheckBox、ComboBox 等）。
- 单元格的键盘导航。
### **安全功能**
- 通过锁定或隐藏它们来保护您的细胞。
### **高级功能**
- 支持创建和计算公式。
- 在不同的网格事件下执行您想要的操作。
### **支持的 Microsoft Excel 版本**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **支持的平台**
- Windows 表格
