## **Aspose.Cells.GridWeb – 网页网格控件**
### **导入/导出功能**
- 从 Excel 电子表格导入和导出到 Excel 电子表格。
- 直接从 DataView 或 DataTable 对象导入和导出数据。
### **工作表功能**
- 添加/删除工作表。
- 支持工作表数据的水平和垂直排序。
- 支持工作表分页。
- 从头开始创建电子表格。
### **行列特征**
- 在工作表中添加、插入和删除行和列。
- 隐藏和显示行和列。
- 冻结和解冻行和列。
- 自定义行和列的标题。
### **Cell 特点**
- 使用单个回发到服务器端在客户端编辑多个单元格。
- 支持数字、日期和时间格式设置。
- 支持自定义格式设置
- 控制文本对齐。
- 管理边界。
- 控制字体设置。
- 合并或拆分单元格
### **格式化功能**
- 创建外观和感觉都像桌面电子表格应用程序的基于 Web 的交互式网格应用程序。
- 支持客户端单元格格式化。
- 提供内置和可自定义的视觉样式。
### **实用功能**
- 支持下拉列表和正则表达式验证。
- 添加和删除评论。
- 插入和删除图片。
- 添加和删除超链接。
- 键盘导航。
### **高级功能**
- 支持在网页上创建和计算公式。
- 提供 80 多个与数学、文本和数据、日期和时间以及逻辑相关的预定义函数。
- 支持创建动态数据透视表。
- 在不同的网格事件下执行操作。
- 在无会话模式下使用 Aspose.Cells.GridWeb 控件。
- 单击一次即可提交网格数据。
### **支持的浏览器**
- Microsoft Internet Explorer
- 火狐浏览器
- 歌剧
- Google 铬
- Microsoft 边
### **Microsoft 支持 Excel**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **支持的平台**
- ASP.NET 网页表格
