## **Aspose.Cells.GridWeb – Web Grid 控件**
### **导入/导出功能**
- 从 Excel 电子表格中导入和导出。
- 直接从 DataView 或 DataTable 对象导入和导出数据。
### **工作表功能**
- 添加/移除工作表。
- 支持工作表数据的水平和垂直排序。
- 支持工作表分页。
- 从头开始创建电子表格。
### **行和列功能**
- 在工作表中添加、插入和删除行和列。
- 隐藏和显示行和列。
- 冻结和解冻行和列。
- 自定义行和列的标题。
### **单元格功能**
- 在客户端通过单次回发到服务器端编辑多个单元格。
- 支持数字、日期和时间格式设置。
- 支持自定义格式设置。
- 控制文本对齐。
- 管理边框。
- 控制字体设置。
- 合并或拆分单元格。
### **格式特性**
- 创建外观和感觉像桌面电子表格应用程序的基于Web的交互性网格应用程序。
- 支持客户端单元格格式设置。
- 提供内置和可自定义的视觉样式。
### **实用功能**
- 支持下拉列表和正则表达式验证。
- 添加和删除注释。
- 插入和删除图片。
- 添加和删除超链接。
- 键盘导航。
### **高级功能**
- 支持在网页上创建和计算公式。
- 提供了80多个与数学、文本和数据、日期和时间以及逻辑相关的预定义函数。
- 支持创建动态数据透视表。
- 在不同的网格事件下执行操作。
- 在无会话模式下使用Aspose.Cells.GridWeb控件。
- 单击提交网格数据。
### **支持的浏览器**
- 微软互联网浏览器
- 火狐浏览器
- Opera
- 谷歌浏览器
- 微软Edge浏览器
### **支持的Microsoft Excel**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **支持的平台**
- ASP.NET Web Forms
