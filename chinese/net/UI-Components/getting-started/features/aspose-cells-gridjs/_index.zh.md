## **Aspose.Cells.GridJs**
### **导入/导出功能**
- 从 Excel 电子表格导入和导出到 Excel 电子表格。

### **工作表功能**
- 添加/删除/重命名工作表。


### **行列特征**
- 在工作表中添加、插入和删除行和列。
- 隐藏和显示行和列。
- 冻结和解冻行和列。
 
### **Cell 特点**
- 编辑多个单元格。
- 支持数字、日期、货币和时间格式设置。
- 支持自定义格式设置。
- 支持背景颜色设置。
- 支持自动填充。
- 控制文本对齐。
- 控制文本换行。
- 管理边界。
- 控制字体设置。
- 合并或拆分单元格
### **格式化功能**
- 创建外观和感觉都像桌面电子表格应用程序的基于 Web 的交互式网格应用程序。
- 支持单元格格式。

### **实用功能**
- 支持下拉列表验证。
- 添加/删除/搜索评论。
- 旋转/调整大小/移动图片/形状。

- 键盘导航。
### **高级功能**
- 支持在网页上创建和计算公式。
- 提供 80 多个与数学、文本和数据、日期和时间以及逻辑相关的预定义函数。

### **支持的浏览器**
- Microsoft Internet Explorer
- 火狐浏览器
- 歌剧
- Google 铬
- Microsoft 边
### **Microsoft 支持 Excel**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **支持的平台**
- ASP.NET核心
