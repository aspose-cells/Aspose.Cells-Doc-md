{{% alert color="primary" %}} 

有时，您需要找到一个简单的方法来处理包含公式的单元格。您需要一种方式来快速浏览所有公式，以检查错误。这可以帮助您跟踪公式中使用的数据。

Aspose.Cells.GridWeb提供了一个名为Microsoft Excel中的"显示公式"的功能。

{{% /alert %}} 
## **使用显示公式功能**
您只需要使用快捷键"CTRL + ~"即可在工作表中显示所有公式。
