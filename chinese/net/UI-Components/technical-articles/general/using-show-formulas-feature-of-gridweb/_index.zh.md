{{% alert color="primary" %}} 

有时，您需要为包含公式的单元格找到一种简单的方法。您需要有一种方法来快速通读所有公式以检查错误。这可以帮助您跟踪公式中使用的数据。

Aspose.Cells.GridWeb 提供了在 Microsoft Excel 中称为显示公式的功能。

{{% /alert %}} 
## **使用显示公式功能**
您只需使用快捷键“CTRL + ~”即可显示工作表中的所有公式。
