{{% alert color="primary" %}} 

您可以使用以下客户端功能来使用JavaScript选择GridWeb中的范围或获取选定的范围。

- getSelectRange()
- setSelectRange()
- clearSelections()

getSelectRange()返回最后选定的范围。setSelectRange()选择给定的范围。clearSelections()清除除当前活动单元格之外的所有选定内容。
