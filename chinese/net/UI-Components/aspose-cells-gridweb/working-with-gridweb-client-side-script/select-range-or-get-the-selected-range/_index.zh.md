{{% alert color="primary" %}} 

您可以使用以下客户端函数来使用JavaScript选择Range或获取GridWeb中的选定范围。

- getSelectRange()
- setSelectRange()
- clearSelections()

getSelectRange()返回最后选择的范围。setSelectRange()选择给定的范围。clearSelections()清除所有选择，不包括当前活动单元格。
