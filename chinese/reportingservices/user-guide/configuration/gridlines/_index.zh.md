网格线配置在**Aspose.Cells.ReportingServices.xml**这样您就可以控制导出的 Microsoft Excel 文件是否显示网格线。默认值为**离开**，表示不显示网格线。

{{< highlight "java" >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
