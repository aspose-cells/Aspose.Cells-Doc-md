可以在 **Aspose.Cells.ReportingServices.xml** 中配置网格线，以控制导出的 Microsoft Excel 文件是否显示网格线。默认值为 **off**，表示不显示网格线。

{{< highlight java >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
