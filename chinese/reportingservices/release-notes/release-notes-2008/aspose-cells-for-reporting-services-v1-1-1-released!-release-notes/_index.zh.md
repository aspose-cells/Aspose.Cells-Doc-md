有什么改进？

- 重写客户端工具，提高稳定性
- 支持子报表设计
- 支持手动输入报表范围
- 支持的复合标记

什么是固定的？

- 修复了在同一台机器上安装多个 SQL Server Reporting Serverices 实例时的安装错误
- 修复了不选择数据集时插入公式的错误
