介绍

Aspose.Cells for Reporting Services 允许您在 Microsoft SQL Server 2005 Reporting Services 中提供本机 Excel 报告。与基于 Excel 的报表的内置渲染器不同，Aspose.Cells for Reporting Services 提供扩展功能集并允许将 RDL 报表转换为：Excel 97-2003 二进制 XLS 格式、SpreadsheetML 或 Excel2007 XLSX 格式。
