{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services（包括Aspose.Cells.ReportingServices客户端、Microsoft Excel加载项和Aspose.Cells.ReportingServices服务器）需要正确安装。此外，在安装Aspose.Cells for Reporting Services之前应先安装Microsoft SQL Server。

{{% /alert %}} 
### **Office 系统**
支持以下Microsoft Excel版本。此外，您需要安装附加补丁以解决Microsoft Office 2000、2002、XP或2003可能出现的一些问题。

- Microsoft Excel 2000
  在早于Office 2003的Microsoft Office计算机上安装**lockbackRegKey.msi**以解决CLR问题。
- Microsoft Excel 2002
  在早于Office 2003的Microsoft Office计算机上安装**lockbackRegKey.msi**以解决CLR问题。
- Microsoft Excel 2003
  始终升级至Office 2003（KB907417）。此更新可纠正使用Microsoft .NET Framework 2.0的加载项、智能标记或智能文档可能无法加载的潜在问题。
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
- Microsoft Excel 2016
- Microsoft Excel 2019
- Microsoft Excel 365
