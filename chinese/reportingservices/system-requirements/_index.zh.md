{{% alert color="primary" %}} 

需要正确安装Aspose.Cells for Reporting Services（包括Aspose.Cells.ReportingServices客户端、Microsoft Excel的插件和Aspose.Cells.ReportingServices服务器）。此外，安装Aspose.Cells for Reporting Services之前应先安装Microsoft SQL Server。

{{% /alert %}} 
### **Office系统**
支持以下Microsoft Excel版本。此外，您需要安装附加的补丁来解决Microsoft Office 2000、2002、XP或2003可能出现的一些问题。

- Microsoft Excel 2000
  安装**lockbackRegKey.msi**以解决早于Office 2003的Microsoft Office机器上的CLR问题。
- Microsoft Excel 2002
  安装**lockbackRegKey.msi**以解决早于Office 2003的Microsoft Office机器上的CLR问题。
- Microsoft Excel 2003
  始终更新到Office 2003（KB907417）。此更新纠正了在使用Microsoft .NET Framework 2.0的插件、智能标记或智能文档时可能出现的潜在问题。
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
- Microsoft Excel 2016
- Microsoft Excel 2019
- Microsoft Excel 365
