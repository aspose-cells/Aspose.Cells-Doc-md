Aspose.Cells for Reporting Services 隐藏隐藏参数中指定的任何基于行、列和工作表的集合。参数包括：

- **工作表名称**表示指定隐藏对象的工作表。
- **类型**表示隐藏对象类型。它包括工作表、行和列。
- **行/列索引**表示指定的隐藏对象。当类型为工作表时，值为空。列索引支持字符。索引值从1开始。
- **隐藏状态**表示隐藏状态值。值为 true、false 或 Excel 公式。该参数仅支持 Reporting Services 报表参数。
