{{% alert color="primary" %}} 

将图表项目呈现到 Microsoft Excel 图表中:

1. 打开 **Aspose.Cells.ReportingServices.xml** 文件。
1. 修改 **Aspose.Cells.ReportingServices.xml** 文件中的配置参数。
1. 添加所需报告的配置信息。

例如:

{{< highlight java >}}

 <Chart >

<Report name= "Employee Sales Summary 2008">

</Report >

</Chart> 

{{< /highlight >}}
