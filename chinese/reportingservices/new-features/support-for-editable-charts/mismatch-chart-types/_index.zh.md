{{% alert color="primary" %}} 

SQL Server Reporting Services 2008 报表定义文件中的某些图表类型不可用作 Excel 图表类型：

- 线步进
- 山脉平原
- 范围平滑
- 范围列
- 范围栏
- 范围误差条
- 范围箱线图
- 形状漏斗

在 Microsoft Excel 中呈现没有等效选项的图表项时，Aspose.Cells 将它们输出到图像。

{{% /alert %}}
