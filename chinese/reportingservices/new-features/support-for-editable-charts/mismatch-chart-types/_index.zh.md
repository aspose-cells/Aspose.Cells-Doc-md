{{% alert color="primary" %}} 

在SQL Server报表服务2008报告定义文件中，一些图表类型不可用作Excel图表类型:

- Line-Stepped
- Range-Plain
- Range-Smooth
- Range-Column
- Range-Bar
- Range-ErrorBar
- Range-BoxPlot
- Shape-Funnel

在渲染在Microsoft Excel中没有等价选项的图表项时，Aspose.Cells将其输出为图像。 

{{% /alert %}}
