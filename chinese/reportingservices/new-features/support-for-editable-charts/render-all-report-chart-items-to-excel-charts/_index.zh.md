将报表中的所有图表项呈现为可编辑的 Excel 图表：

1. 打开**Aspose.Cells.ReportingServices.xml**文件。
1. 修改配置参数如下：

{{< highlight "java" >}}

 <Chart value="all">

</Chart>

{{< /highlight >}}

当值设置为“all”时，所有图表项都呈现为可编辑图表。
