{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services 允许您通过 Microsoft SQL Server 2005 和 2008 Reporting Services 提供本机 Microsoft Excel 报告。 Unlike the built-in renderer for Microsoft Excel-based reports, Aspose.Cells for Reporting Services provides an extended feature set that convert RDL reports to: Excel 97-2003 binary XLS format, CSV, SpreadsheetML, Excel 2007 XLSX , Excel 2007 XLSM, Excel 2007 XLSB, HTML 和 ODS 格式。

{{% /alert %}}
