我们很高兴地宣布 Aspose.Cells 用于 SharePoint v1.0.0！

什么是新的：

- 支持 MS Excel 输入格式：XLS、XLSX、XLST、XLSM、CSV、ODS、SpreadsheetML、TXT。
- 支持输出格式：XLS、XLSX、CSV、HTML、ODS、PDF、SpreadsheetML、TXT。
