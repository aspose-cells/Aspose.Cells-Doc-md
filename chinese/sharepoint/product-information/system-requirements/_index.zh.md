### **服务器要求**
要安装Aspose.Cells for SharePoint，您需要使用以下之一的计算机：

- Windows SharePoint Services 3.0
- Microsoft Office SharePoint Server 2007
- Microsoft SharePoint Server 2010
- Microsoft SharePoint Server 2013

{{% alert color="primary" %}} 

我们建议您安装这些产品的Service Pack 2或更高版本。

{{% /alert %}}
### **客户端要求**
您需要浏览器访问SharePoint站点。Aspose.Cells for SharePoint经过以下测试：

- 微软互联网浏览器
- 火狐浏览器
- 谷歌浏览器
- Safari
