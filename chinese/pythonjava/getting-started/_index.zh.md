## **系统要求**
 Aspose.Cells for Python via Java 与平台无关 API 可以在任何平台（Windows，Linux 和 MacOS）上使用，其中[Python](https://www.python.org/downloads/)已安装。在设置安装之前，机器必须具有 Java 8 或更高版本。

## **Python版本**
- Python 3.5 或更高
## **Java版本**
- Java 1.8 或更高版本

## **安装**
### **Windows:**
- **安装Java并设置JAVA_HOME和PATH环境变量**
例如：
{{< highlight "java" >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}
  
- **通过pypi的Java为Python安装Aspose.Cells**
您可以通过 Java 轻松地将 Aspose.Cells 用于 Python 来自[皮皮](https://pypi.org/project/aspose-cells/)使用以下命令。
{{< highlight "java" >}}

 $ pip install aspose-cells

{{< /highlight >}}

### **Linux：**
- **安装 Java**
  
- **通过pypi的Java为Python安装Aspose.Cells**
您可以通过 Java 轻松地将 Aspose.Cells 用于 Python 来自[皮皮](https://pypi.org/project/aspose-cells/)使用以下命令。

{{< /highlight >}}

### **苹果系统：**
