## **将 Excel 文件保存为 HTML 时导出注释**
当 Excel 转换为 HTML 时，不会导出注释。 Aspose.Cells for Python via Java 提供Excel转HTML时导出注释的功能。为此，API 提供了[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments)财产。设定值[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments)财产给**真的**将在输出 HTML 中导出注释。
