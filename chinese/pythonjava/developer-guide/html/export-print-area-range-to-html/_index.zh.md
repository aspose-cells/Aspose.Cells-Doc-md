## **将打印区域范围导出为HTML**
Aspose.Cells for Python via Java支持仅将打印区域（即选择的单元格范围）而不是整个工作表导出为HTML。为此，API提供了[HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly)属性。将此属性设置为**True**将只导出打印区域。
