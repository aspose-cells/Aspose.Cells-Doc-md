## **导出打印区域范围为HTML**
Aspose.Cells for Python via Java 支持仅导出打印区域，即选定的单元格范围，而不是整个她到 HTML。为此，API 提供了[HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly)财产。将此属性设置为**真的**只会导出打印区域。
