## 在 Aspose.Cells for Python via Java 中为工作表添加图标

如果您需要使用 [Aspose.Cells for Python via Java](https://products.aspose.com/cells/python-java/) 在 Excel 文件中添加“图标”，本文件可以为您提供一些帮助。

### 插入到Excel中
对应于插入图标操作的Excel界面如下：

![](1.png)

- 选择要插入到工作表中的图标的位置
- 左键单击*插入*->*图标*
- 在打开的窗口中，选择上图中红色矩形所示的图标
- 单击 *插入*，将其插入到 Excel 文件中。

效果如下:
