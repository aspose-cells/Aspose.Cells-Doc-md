## **将 Excel 转换为图像**
Aspose.Cells for Python via Java 支持将Excel文件转换为不同的图像格式图像。为此，API 提供了[图纸渲染](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)和[图像或打印选项](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions)类。此类表示将呈现为图像的工作表。这[图纸渲染](https://reference.aspose.com/cells/python/asposecells.api/SheetRender)类提供了[印象（）](https://reference.aspose.com/cells/python/asposecells.api/sheetrender#toImage\(int,%20java.lang.String\)) 将工作表转换为图像文件的方法。支持 BMP、PNG、JPEG、TIFF 和 EMF 格式。
