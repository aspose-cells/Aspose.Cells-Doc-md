## **将 JSON 转换为 Excel 工作簿**
无需考虑如何将 JSON 转换为 Excel 文件，因为通过 Java 库将 Aspose.Cells 转换为 Python 是最佳决策。 Aspose.Cells for Python via Java API 提供了JSON格式转电子表格的支持。您可以使用[**JsonLoad选项**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonLoadOptions)类以指定用于将 JSON 导入工作簿的其他设置。
