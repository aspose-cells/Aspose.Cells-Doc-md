Aspose.Cells for Python via Java 支持将工作簿转换为 Json（JavaScript 对象表示法）文件。

{{% /alert %}}

## **将 Excel 工作簿转换为 JSON**

无需考虑如何将 Excel 工作簿转换为 JSON，因为通过 Java 库将 Aspose.Cells 转换为 Python 是最佳决策。 Aspose.Cells for Python via Java API 提供了将电子表格转换为JSON格式的支持。要将工作簿导出到 JSON，请传递[**保存格式.JSON**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat)作为第二个参数[**工作簿.保存**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)） 方法。您也可以使用[**JsonSave选项**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonSaveOptions)类以指定将工作表导出到 JSON 的其他设置。
