Aspose.Cells for Python via Java 支持将 Excel（.xls、xlsx、.xlsb、xlsm）、CSV 和 OpenOffice(.ods) 文件转换为 PowerPoint 文件。

{{% /alert %}}

## **将 Excel 工作簿转换为 PPT**

无需考虑如何将 Excel 工作簿转换为 PowerPoint，因为通过 Java 库将 Aspose.Cells 转换为 Python 是最佳决策。 Aspose.Cells for Python via Java API 提供了将电子表格转换为PowerPoint格式的支持。要将工作簿导出到 PowerPoint，请通过[**保存格式.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat)作为第二个参数[**工作簿.保存**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)） 方法。您也可以使用[**PptxSave选项**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions)类以指定将工作表导出到 .pptx 文件的其他设置。
