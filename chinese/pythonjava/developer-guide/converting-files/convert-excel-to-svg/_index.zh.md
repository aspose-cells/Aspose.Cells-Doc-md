## **将 Excel 转换为 SVG**
SVG(**可缩放矢量图形）**是基于 XML 标准的二维矢量图形规范。它是一个开放标准，自 1999 年以来一直由万维网联盟 (W3C) 开发。

Aspose.Cells for Python via Java 支持将Excel文件转换为SVG图像。为此，API 提供了[图纸渲染](https://reference.aspose.com/cells/python/asposecells.api/SheetRender), [图像或打印选项](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions)， 和[工作簿渲染](https://reference.aspose.com/cells/python/asposecells.api/WorkbookRender)类。
