## **创建数据透视表**
Aspose.Cells for Python via Java 提供创建数据透视表的功能。要使用 Aspose.Cells 创建数据透视表，请按照以下步骤操作：

1. 通过使用将一些数据添加到工作表单元格[Cell](https://reference.aspose.com/cells/python/asposecells.api/Cell)对象的[设定值](https://reference.aspose.com/cells/python/asposecells.api/cell#Value)财产。此数据将用作数据透视表的数据源。
1. 通过调用将数据透视表添加到工作表[数据透视表集合](https://reference.aspose.com/cells/python/asposecells.api/PivotTableCollection)[添加](https://reference.aspose.com/cells/python/asposecells.api/pivottablecollection#add\(java.lang.Object\)方法，封装在[工作表](https://reference.aspose.com/cells/python/asposecells.api/Worksheet)目的。
1. 访问[数据透视表](https://reference.aspose.com/cells/python/asposecells.api/PivotTable)对象来自[数据透视表集合](https://reference.aspose.com/cells/python/asposecells.api/PivotTableCollection)通过传递[数据透视表](https://reference.aspose.com/cells/python/asposecells.api/PivotTable)指数。
1. 使用封装在[数据透视表集合](https://reference.aspose.com/cells/python/asposecells.api/PivotTableCollection)对象来管理数据透视表。
