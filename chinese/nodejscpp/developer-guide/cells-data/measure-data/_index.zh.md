有时候您需要计算单元格值的宽度和高度，以使单元格值适合单元格内。Aspose.Cells 为此目的提供了 [**Cell.getWidthOfValue()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getWidthOfValue--) 和 [**Cell.getHeightOfValue()**](https://reference.aspose.com/cells/nodejs-cpp/cell/#getHeightOfValue--) 方法。通过使用这些方法，您可以计算单元格值的宽度和高度，然后分别设置该单元格的列宽和行高，从而调整或使单元格值适合单元格内。  

或者，您也可以使用Aspose.Cells API [自动调整您的单元格或单元格区域的行和列](/cells/zh/nodejs-cpp/autofit-rows-and-columns/)。
