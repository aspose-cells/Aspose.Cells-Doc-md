{{% alert color="primary" %}}

有时你想格式化数据透视表的单元格。例如，为数据透视表单元格应用背景色。Aspose.Cells for Node.js via C++ 提供了两个方法 [**PivotTable.formatAll(style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#formatAll-style-) 和 [**PivotTable.format(row, column, style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#format-number-number-style-)，你可以利用它们来实现。

[**PivotTable.formatAll(style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#formatAll-style-) 将样式应用到整个数据透视表，而 [**PivotTable.format(row, column, style)**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#format-number-number-style-) 将样式应用到数据透视表的单个单元格。

{{% /alert %}}
