## **Possible Usage Scenarios**  

You can change the **Version number** of an Excel file by right‑clicking the file, selecting **Properties > Details**, and then editing the **Version number** field. Please use [**BuiltInDocumentPropertyCollection.GetDocumentVersion()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getdocumentversion/) property to change it programmatically using Aspose.Cells APIs.
