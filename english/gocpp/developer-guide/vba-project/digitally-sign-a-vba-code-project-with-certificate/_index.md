{{% alert color="primary" %}} 

You can digitally sign your VBA code project using Aspose.Cells with its [**Workbook.VbaProject.Sign()**](https://reference.aspose.com/cells/go-cpp/vbaproject/sign/) method. Please follow these steps to verify whether your Excel file is digitally signed with a certificate.

- Click **Visual Basic** from the **Developer** tab to open **Visual Basic for Applications IDE**.  
- Click **Tools** > **Digital Signatures...** in the **Visual Basic for Applications IDE**.

It will show the **Digital Signature Form** indicating whether the document is digitally signed with a certificate or not.

{{% /alert %}}
