You can make a worksheet more appealing by adding a picture as a worksheet background. This feature can be quite effective if you have a special corporate graphic that adds a touch of background without obscuring the data on the sheet. You can set a background picture for a sheet using the Aspose.Cells API.

{{% /alert %}} 

## **Setting Sheet Background in Microsoft Excel**

To set a sheet's background image in Microsoft Excel (for example, Microsoft Excel 2019):

1. From the **Page Layout** menu, find the **Page Setup** option, and then click the **Background** option.  
2. Select a picture to set as the sheet's background.
