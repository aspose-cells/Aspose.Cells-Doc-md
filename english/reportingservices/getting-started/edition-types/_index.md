## Overview

Aspose.Cells for Reporting Services is built to integrate seamlessly with Microsoft SQL Server Reporting Services (SSRS).  
Whether you need to:

- **Design** Excel‑style reports using the familiar Aspose.Cells.Report.Designer,
- **Render** them efficiently on the server,
- **Publish** and preview directly from SSRS,

all functionality is provided by this single edition—no extra licensing tiers or hidden modules.

### Key Benefits

- **Full‑featured Excel support** – Pivot tables, dynamic formulas, VBA/macros, and interactive charts work exactly as they do in the desktop version of Excel.  
- **Native SSRS integration** – Use SSRS parameters, global variables, and formulas as data sources without additional configuration.  
- **One‑time purchase** – No need to manage multiple editions; the feature list above covers everything you’ll ever need for reporting services.

## Frequently Asked Questions

**Q: Are there any hidden costs or add‑ons?**  
A: No. All the capabilities listed in the table are included in the single edition.

**Q: Can I extend the functionality with custom code?**  
A: Yes. Aspose.Cells exposes a rich API that can be called from custom SSRS extensions or external services.

**Q: Is VBA execution sandboxed?**  
A: VBA macros run in a secure, isolated environment, ensuring server stability and security.
