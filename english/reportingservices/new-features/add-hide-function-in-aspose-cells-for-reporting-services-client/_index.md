## Overview  

The **Hide** function lets you conceal specified rows, columns, or entire worksheets in an Excel report generated by Aspose.Cells for Reporting Services.

There are two ways to set hide attributes:

1. **Set hide attributes for a worksheet, rows, or columns** – Use the **Modify Attribute** form to define hide options for individual cells, rows, columns, or the whole sheet.  
   <br/>[Set hide attributes for worksheet, rows and columns](/cells/reportingservices/set-a-table-hide-attributes-from-the-modify-attribute-form/)

2. **Set a table's hide attributes using the **Modify Attribute** dialog** – Apply hide settings to an entire table at once.  
   <br/>[Set hide option for Excel row, column and sheet using dyn‑hide form](/cells/reportingservices/set-hide-option-for-excel-row-column-and-sheet-using-dyn-hide-form/)
