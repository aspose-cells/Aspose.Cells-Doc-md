## Overview

Aspose.Cells for Reporting Services continuously evolves to meet the growing demands of developers and report designers.  
The **New Features** section below highlights the most recent additions, each designed to improve functionality, user experience, and integration flexibility.

## New Features

| Feature | Description |
|---------|-------------|
| **[Composite Markers](/cells/reportingservices/composite-markers/)** | Combine multiple marker types (image, text, shape) into a single, reusable marker for richer visualizations. |
| **[Sub‑reports](/cells/reportingservices/sub-reports/)** | Embed complete reports within a parent report, enabling modular design and easier maintenance. |
| **[Integrate with Microsoft Report Viewer in Local Mode](/cells/reportingservices/integrate-with-microsoft-report-viewer-in-local-mode/)** | Seamlessly run Aspose.Cells reports using the Microsoft Report Viewer control without a server‑side processing engine. |
| **[Extended Rendering Extensions Support](/cells/reportingservices/extended-rendering-extensions-support/)** | Access additional output formats and rendering options beyond the default set (e.g., PDF/A, SVG, TIFF). |
| **[Support for Microsoft Excel 2007/2010 Custom UI](/cells/reportingservices/support-for-microsoft-excel-2007-or-2010-custom-ui/)** | Preserve custom UI XML (ribbons, tabs) when exporting to Excel 2007/2010, maintaining a familiar user interface. |
| **[Support for Editable Charts](/cells/reportingservices/support-for-editable-charts/)** | Generate charts that remain fully editable in Excel, allowing end‑users to modify data series, styles, and layouts. |
| **[Support for Digital Signatures](/cells/reportingservices/support-for-digital-signatures/)** | Secure your reports with X.509 digital signatures, ensuring authenticity and integrity of exported documents. |
| **[Add Matrix in Aspose.Cells for Reporting Services Client](/cells/reportingservices/add-matrix-in-aspose-cells-for-reporting-services-client/)** | Introduce matrix (pivot‑style) data regions directly within the client designer for advanced data summarization. |
| **[Add Hide Function in Aspose.Cells for Reporting Services Client](/cells/reportingservices/add-hide-function-in-aspose-cells-for-reporting-services-client/)** | Dynamically hide rows, columns, or entire report items based on expression‑driven conditions. |
| **[Aspose.Cells for Reporting Services Report Designer](/cells/reportingservices/aspose-cells-for-reporting-services-report-designer/)** | A dedicated visual designer that simplifies report layout creation, preview, and publishing. |
| **[Add Settings for Page Setup in Aspose.Cells for Reporting Services Renderer](/cells/reportingservices/add-settings-for-page-setup-in-aspose-cells-for-reporting-services-renderer/)** | Fine‑tune page margins, orientation, paper size, and scaling directly from the renderer configuration. |
| **[Add Group Function in Aspose.Cells for Reporting Services Client](/cells/reportingservices/add-group-function-in-aspose-cells-for-reporting-services-client/)** | Group data rows or columns on the client side, enabling hierarchical reporting without additional server logic. |
| **[Add Sorting Feature for Rendering Table Item](/cells/reportingservices/add-sorting-feature-for-rendering-table-item/)** | Allow end‑users to sort table rows interactively when the report is viewed in supported formats (e.g., HTML, Excel). |

## How to Use These Features

1. **Update the library** – Ensure you are running the latest version of Aspose.Cells for Reporting Services (download from the official site or NuGet).  
2. **Enable the feature** – Most features are toggled via the report designer or renderer settings; refer to each feature’s page for exact steps.  
3. **Test your report** – Render the report in the target format (PDF, XLSX, HTML, etc.) and verify that the new behavior appears as expected.  
4. **Deploy** – Once validated, publish the updated report to your Reporting Services server or embed it in your application.
