Aspose.Cells for Python via .NET allows you to import the XML map inside the workbook using the [**Workbook.import_xml()**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/import_xml/#str-str-int-int) method. You can import an XML map using Microsoft Excel with the following steps:

- Select the **Developer** tab  
- Click **Import** in the XML section and follow the required steps.

You will need to provide your XML data to complete the import. Here is a [sample XML data file](5115037.txt) that you can use for testing.
