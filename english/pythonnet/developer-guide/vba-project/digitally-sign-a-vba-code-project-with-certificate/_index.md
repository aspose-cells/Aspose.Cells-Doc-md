{{% alert color="primary" %}}

You can digitally sign your VBA code project using Aspose.Cells for Python via .NET with its [**Workbook.vba_project.sign()**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/sign/#aspose.cells.digitalsignatures.DigitalSignature) method. Please follow these steps to check if your excel file is digitally signed with a certificate.

- Click **Visual Basic** from the **Developer** tab to open **Visual Basic for Applications IDE**
- Click **Tools** > **Digital Signatures...**  of **Visual Basic for Applications IDE**

and it will show the **Digital Signature Form** showing if the document is digitally signed with a certificate or not.

{{% /alert %}}
