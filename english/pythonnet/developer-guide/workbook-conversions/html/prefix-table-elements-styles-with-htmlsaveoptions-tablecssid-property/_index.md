## **Possible Usage Scenarios**

Aspose.Cells allows you to prefix table element styles with the [**HtmlSaveOptions.table_css_id**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/table_css_id) property. Suppose you set this property to a value such as **MyTest_TableCssId**; then you will find table element styles as shown below:

{{< highlight java >}}

table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}
