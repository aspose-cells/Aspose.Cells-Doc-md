Hyperlinks are useful for accessing information on other worksheets or on websites. Microsoft Excel lets users add hyperlinks to text in cells and to images. Image hyperlinks can make navigating a worksheet easier, for example, as next and previous buttons, or logos that link to particular sites. This document explains how to insert image hyperlinks in a worksheet using Aspose.Cells for Python via .NET.

{{% /alert %}}
