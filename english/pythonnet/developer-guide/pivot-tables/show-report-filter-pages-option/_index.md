## **How to Show report filter pages option**  

Excel supports creating pivot tables, adding report filters, and enabling the "Show Report Filter Pages" option. Aspose.Cells for Python via .NET also supports this feature, enabling the "Show Report Filter Pages" option on the created pivot table. The following screen shows the "Show Report Filter Pages" option in Excel.
