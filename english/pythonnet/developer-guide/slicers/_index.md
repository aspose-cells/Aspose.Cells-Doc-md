## **Possible Usage Scenarios**

A slicer is used to filter data quickly. It can be used to filter data both in a table and a pivot table. Microsoft Excel allows you to create a slicer by selecting a table or pivot table and then clicking *Insert > Slicer*. Aspose.Cells for Python via .NET also allows you to create a slicer using the [**Worksheet.slicers.add()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/add/#aspose.cells.pivot.PivotTable-int-int-aspose.cells.pivot.PivotField) method.
