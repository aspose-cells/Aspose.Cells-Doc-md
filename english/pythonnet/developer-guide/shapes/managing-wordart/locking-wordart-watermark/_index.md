{{% alert color="primary" %}}

Aspose.Cells for Python via .NET APIs allow adding WordArt watermarks on the worksheet in a way that the WordArt becomes an object that you can be moved and positioned on the worksheet. It is also possible to lock the WordArt object for any interaction such as editing, movement & selection. This article explains the usage of [**Shape.set_locked_property**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/set_locked_property) method to lock a few aspects of the watermark.
