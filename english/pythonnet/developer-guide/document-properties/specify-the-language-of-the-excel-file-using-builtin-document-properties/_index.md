## **Possible Usage Scenarios**

You can change the language of an Excel file by right‑clicking the file, selecting **Properties > Details**, and then editing the **Language** field. Please use the **[BuiltInDocumentPropertyCollection.language](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/language)** property to change it programmatically using Aspose.Cells for Python via .NET APIs.
