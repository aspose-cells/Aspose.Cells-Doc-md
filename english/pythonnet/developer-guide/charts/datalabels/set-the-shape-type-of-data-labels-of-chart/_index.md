## **Possible Usage Scenarios**
You can change the shape type of data labels of the chart using the DataLabels.ShapeType property. It takes a value from the DataLabelShapeType enumeration and changes the shape type of data labels accordingly. Some of its values are

{{< highlight java >}}

DataLabelShapeType.BENT_LINE_CALLOUT

DataLabelShapeType.DOWN_ARROW_CALLOUT

DataLabelShapeType.ELLIPSE

DataLabelShapeType.LINE_CALLOUT

DataLabelShapeType.RECT

etc.

{{< /highlight >}}
