{{% alert color="primary" %}}

Aspose.Cells provides the functionality to apply superscript (text above the baseline) and subscript (text below the baseline) effects to text.

{{% /alert %}}

## **Working with Superscript and Subscript**

Apply the superscript effect by setting the [**Style.Font**](https://reference.aspose.com/cells/net/aspose.cells/font) object's [**IsSuperscript**](https://reference.aspose.com/cells/net/aspose.cells/font/properties/issuperscript) property to **true**. To apply subscript, set the [**Style.Font**](https://reference.aspose.com/cells/net/aspose.cells/font) object's [**IsSubscript**](https://reference.aspose.com/cells/net/aspose.cells/font/properties/issubscript) property to **true**.
