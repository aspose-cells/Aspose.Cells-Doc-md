{{% alert color="primary" %}}

You can retrieve the equation text of a chart trendline using Aspose.Cells. Aspose.Cells provides the [**Trendline.DataLabels.Text**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/text) property, which returns the equation text of a chart trendline. To use this property, you must first call the [**Chart.Calculate()**](https://reference.aspose.com/cells/net/aspose.cells.charts/chart/methods/calculate) method.
