## Check if VBA Project is Protected and Locked for Viewing in C#

Aspose.Cells allows you to check if the VBA (Visual Basic for Applications) project of an Excel file is protected and locked for viewing. For this, the API provides the **VbaProject.IsLockedForViewing** property. If it is locked for viewing, then the **VbaProject.IsLockedForViewing** property returns **true**.
