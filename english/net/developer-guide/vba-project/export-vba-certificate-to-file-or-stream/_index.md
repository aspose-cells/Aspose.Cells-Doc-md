{{% alert color="primary" %}}

Aspose.Cells allows you to export a VBA digital certificate to a stream, such as a file or a memory stream. You can access the raw data of the VBA digital certificate using the [**Workbook.VbaProject.CertRawData**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/certrawdata) property.
