{{% alert color="primary" %}}

You can digitally sign your VBA code project using Aspose.Cells with its [**Workbook.VbaProject.Sign()**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/methods/sign) method. Please follow these steps to check if your Excel file is digitally signed with a certificate.

- Click **Visual Basic** from the **Developer** tab to open **Visual Basic for Applications IDE**
- Click **Tools** > **Digital Signatures...** in the **Visual Basic for Applications IDE**

It will display the **Digital Signature Form**, indicating whether the document is digitally signed with a certificate.

{{% /alert %}}
