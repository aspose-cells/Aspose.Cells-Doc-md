{{% alert color="primary" %}}

Microsoft Excel allows you to change the layout of a PivotTable using the *PivotTable Tools > Design > Report Layout* menu commands. You can change the layout in these three forms:

- Show in Compact Form  
- Show in Outline Form  
- Show in Tabular Form  

Aspose.Cells also provides [**PivotTable.ShowInCompactForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showincompactform), [**PivotTable.ShowInOutlineForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showinoutlineform) and [**PivotTable.ShowInTabularForm()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/showintabularform) methods to change the layout of a PivotTable in these three forms.
