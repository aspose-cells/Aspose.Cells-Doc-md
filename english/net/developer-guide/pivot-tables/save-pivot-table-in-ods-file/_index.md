Aspose.Cells provides the ability to save a pivot table in ODS files. For this, you simply convert a workbook with an existing pivot table, or create a new pivot table, and save the file in the ODS format. Be sure to call [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata) before saving to ensure that the pivot table is rendered in the output ODS file.
