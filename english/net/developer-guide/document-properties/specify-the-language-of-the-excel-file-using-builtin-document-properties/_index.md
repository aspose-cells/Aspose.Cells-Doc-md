## **Possible Usage Scenarios**

You can change the language of an Excel file by right‑clicking the file, selecting **Properties > Details**, and then editing the **Language** field. Please use the **[BuiltInDocumentPropertyCollection.Language](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/language)** property to change it programmatically using Aspose.Cells APIs.
