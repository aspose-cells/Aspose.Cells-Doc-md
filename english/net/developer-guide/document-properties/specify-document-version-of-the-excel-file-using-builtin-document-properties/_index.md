## **Possible Usage Scenarios**

You can change the **Version number** of an Excel file by right‑clicking the file, then selecting **Properties > Details**, and then editing the **Version number** field. Please use [**BuiltInDocumentPropertyCollection.DocumentVersion**](https://reference.aspose.com/cells/net/aspose.cells.properties/builtindocumentpropertycollection/properties/documentversion) property to change it programmatically using Aspose.Cells APIs.
