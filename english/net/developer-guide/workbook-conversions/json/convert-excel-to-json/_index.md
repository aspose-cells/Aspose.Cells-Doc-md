## **Convert Excel Workbook to JSON**

No need to wonder how to convert an Excel workbook to JSON, because Aspose.Cells for .NET provides the best solution. The Aspose.Cells API provides support for converting spreadsheets to JSON format. To export the workbook to JSON, pass [**SaveFormat.Json**](https://reference.aspose.com/cells/net/aspose.cells/saveformat) as the second parameter of [**Workbook.Save**](https://reference.aspose.com/cells/net/aspose.cells.workbook/save/methods/3) method. You may also use [**JsonSaveOptions**](https://reference.aspose.com/cells/net/aspose.cells/JsonSaveoptions) class to specify additional settings for exporting a worksheet to JSON.
