## **Possible Usage Scenarios**

Aspose.Cells allows you to prefix the styles of table elements with the [**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/net/aspose.cells/htmlsaveoptions/properties/tablecssid) property. Suppose you set this property to a value such as **MyTest_TableCssId**; you will then find table element styles as shown below:

{{< highlight java >}}
table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.
{{< /highlight >}}
