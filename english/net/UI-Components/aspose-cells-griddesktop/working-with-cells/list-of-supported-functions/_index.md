## **Introduction**
There are a lot of builtâ€‘in functions supported by the Aspose.Cells.GridDesktop control. These functions belong to the following categories:

- **Math Functions**, provides a huge set of functions that can be used in numerous mathematical calculations
- **Text and Data Functions**, provides various functions to perform different kinds of operations on text and data
- **Date and Time Functions**, provides some useful functions related to date and time
- **Logical Functions**, provides a small set of logical functions to handle logical operations on data
