## Basics of GridWeb
GridWeb is a simple-to-use .NET/Java web control for displaying and editing spreadsheet files.

It is easy and simple to use.

It helps you build an online web editor for spreadsheet files quickly.

It also supports importing and exporting all kinds of spreadsheet format files, which are 100% compatible with MS Excel files.
