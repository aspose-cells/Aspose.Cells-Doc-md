**Welcome to the Aspose.Cells for JavaScript via C++ documentation section**

Aspose.Cells for JavaScript via C++ is a native WebAssembly library that enables JavaScript/Node.js developers to create, manipulate, and convert spreadsheets programmatically without requiring Office Automation or Microsoft Excel.

The library supports major spreadsheet formats including Excel (XLS, XLSX, XLSB, XLSM), ODS, CSV, and HTML, providing a complete solution for spreadsheet processing directly in the browser or server‑side environments without external dependencies.

{{% /alert %}}

Aspose.Cells for JavaScript via C++ offers a wide range of features for creating, converting, and manipulating spreadsheets:

- Built‑In Properties

- Data Validation
- Conditional Formatting

Ensures consistent behavior across both client‑side and server‑side applications for seamless integration and execution.
