{{% alert color="primary" %}}

Microsoft Excel allows you to change the layout of a PivotTable using *PivotTable Tools > Design > Report* Layout menu commands. You can change the layout in these three forms:

- Show in Compact Form
- Show in Outline Form
- Show in Tabular Form

Aspose.Cells also provides [**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm--), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm--) and [**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm--) methods to change the layout of a PivotTable in these three forms.

{{% /alert %}}
