## **Possible Usage Scenarios**

You can change the *Language* of Excel file by right-clicking the file and then selecting *Properties > Details* and then editing the *Language* field. Please use [**BuiltInDocumentPropertyCollection.Language**](https://reference.aspose.com/cells/java/com.aspose.cells/builtindocumentpropertycollection#Language) property to change it programmatically using Aspose.Cells APIs.
