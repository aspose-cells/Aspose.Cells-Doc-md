{{% alert color="primary" %}}

In Microsoft Excel, you can add a library reference to the VBA project by clicking **Tools > References...**. It will open the following dialog box, which helps you select from existing references or browse for your library.

![todo:image_alt_text](add-a-library-reference-to-vba-project-in-workbook_1.png)

But sometimes you need to add or register a library reference to the VBA project through code. You can do it using Aspose.Cells [**VbaProject.getReferences().addRegisteredReference()**](https://reference.aspose.com/cells/java/com.aspose.cells/vbaprojectreferencecollection#addRegisteredReference-java.lang.String-java.lang.String-) method.
