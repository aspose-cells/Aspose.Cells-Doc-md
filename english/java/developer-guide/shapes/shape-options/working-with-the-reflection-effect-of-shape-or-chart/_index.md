## **Possible Usage Scenarios**
Aspose.Cells provides the [Shape.ReflectionEffect](https://reference.aspose.com/cells/java/com.aspose.cells/shape#getReflection--) property to work with the reflection effect of shape or chart. It contains the following sub-properties which you can set to achieve different results as per your requirements.

- [ReflectionEffect.Blur](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#getBlur--)
- [ReflectionEffect.Direction](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#getDirection--)
- [ReflectionEffect.Distance](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#getDistance--)
- [ReflectionEffect.FadeDirection](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#getFadeDirection--)
- [ReflectionEffect.RotWithShape](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#getRotWithShape--)
- [ReflectionEffect.Size](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#getSize--)
- [ReflectionEffect.Transparency](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#getTransparency--)
- [ReflectionEffect.Type](https://reference.aspose.com/cells/java/com.aspose.cells/reflectioneffect#getType--)
