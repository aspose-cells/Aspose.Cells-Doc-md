## **Possible Usage Scenarios**
Aspose.Cells provides the Shape.Glow property to work with the glow effect of a shape or chart. It contains the following sub‑properties which you can set to achieve different results as per your requirements.

- GlowEffect.Size
- GlowEffect.Transparency
