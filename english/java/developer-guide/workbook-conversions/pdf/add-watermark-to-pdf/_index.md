While converting an Excel file to PDF, you may have a requirement to add a watermark to the PDF file. The following examples show how to add text and image watermarks to PDF while rendering to PDF.

## **Add text watermark to PDF**

You can easily add a text watermark to PDF by specifying the text and the corresponding font. Also, you can set alignment, offset, rotation, opacity, foreground/background, and scale to the page in [RenderingWatermark](https://reference.aspose.com/cells/java/com.aspose.cells/renderingwatermark/).

## **Add image watermark to PDF**

You can add an image watermark to PDF just by specifying the image bytes of an image. Also, you can set alignment, offset, rotation, opacity, foreground/background, and scale to the page in [RenderingWatermark](https://reference.aspose.com/cells/java/com.aspose.cells/renderingwatermark/).
