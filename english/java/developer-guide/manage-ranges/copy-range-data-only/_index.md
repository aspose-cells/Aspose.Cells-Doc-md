{{% alert color="primary" %}} 

Sometimes, you need to copy data from one range of cells to another, copying only the data, not the formatting. Aspose.Cells offers this feature by providing the [Range.copyData](https://reference.aspose.com/cells/java/com.aspose.cells/range#copyData-com.aspose.cells.Range-) method.

{{% /alert %}} 
## **Copy Range Data Only**
This example shows how to:

1. Create a workbook.  
2. Add data to cells in the first worksheet.  
3. Create a range.  
4. Create a style object with specified formatting attributes.  
5. Apply the style formatting to the range.  
6. Create another range of cells.  
7. Copy the data from the first range to the second range using the [Range.copyData](https://reference.aspose.com/cells/java/com.aspose.cells/range#copyData-com.aspose.cells.Range-) method.
