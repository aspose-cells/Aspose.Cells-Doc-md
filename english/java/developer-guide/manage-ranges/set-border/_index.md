## **Possible Usage Scenarios**
When you want to set the border for a range, you don't need to set each cell individually. You can set the border on the range. Aspose.Cells offers this feature.  
This article provides a sample code that uses Aspose.Cells to set a range border.

## **Set Range border in Excel**
To set the border of a range in Excel, you can follow these steps:
1. Select the range of cells to which you want to apply the border.
2. In the **Home** tab of the ribbon, locate the **Font** group.
3. Within the **Font** group, click the **Borders** drop‑down button.  
   <img src="border.png" />
4. Choose the type of border you want to apply from the options in the drop‑down menu. You can select from preset border styles or customize your own border.
5. Once you have selected the desired border style, the border will be applied to the selected range of cells.

## **Set Range border using Aspose.Cells**
This example shows how to:

1. Create a workbook.  
2. Add data to cells in the first worksheet.  
3. Create a [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/range/).  
4. Set the inner border of the range.  
5. Set the outer border of the range.
