{{% alert color="primary" %}}

You can specify Job or Document Name while printing your workbook or worksheet using the [**WorkbookRender**](https://reference.aspose.com/cells/java/com.aspose.cells/WorkbookRender) or [**SheetRender**](https://reference.aspose.com/cells/java/com.aspose.cells/SheetRender) objects. Aspose.Cells provides the [**WorkbookRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter-java.lang.String-java.lang.String-) and [**SheetRender.toPrinter(printerName, jobName)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter-java.lang.String-java.lang.String-) methods which you can use to specify Job Name while printing your workbook or worksheet

{{% /alert %}}
