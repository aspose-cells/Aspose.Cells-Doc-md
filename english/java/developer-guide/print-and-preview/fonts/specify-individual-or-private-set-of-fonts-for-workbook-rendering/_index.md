## **Possible Usage Scenarios**

Generally, you specify the fonts directory or list of fonts for all workbooks but sometimes, you have to specify individual or private set of fonts for your workbooks. Aspose.Cells provides [**IndividualFontConfigs**](https://reference.aspose.com/cells/java/com.aspose.cells/individualfontconfigs) class that can be used to specify the individual or private set of fonts for your workbook.
