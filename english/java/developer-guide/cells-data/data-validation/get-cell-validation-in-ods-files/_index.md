## **Get Cell Validation in ODS Files**

With Aspose.Cells,  you can get the validation applied to a cell in ODS files. For this, the API provides the [**GetValidation**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getValidation--) method of the [**Cell**](https://reference.aspose.com/cells/java/com.aspose.cells/Cell) class.

The following code sample demonstrates the use of the [**GetValidation**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getValidation--) method by loading the [source ODS](SampleBook1.ods) file and reading the validation of the cell A9.
