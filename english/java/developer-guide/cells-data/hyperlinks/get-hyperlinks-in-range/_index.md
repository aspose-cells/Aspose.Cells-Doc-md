## **Get Hyperlinks in Range**

The [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/range) class provides a [**getHyperlinks()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#getHyperlinks--) property getter which gets all the hyperlinks in the selected range. You may also delete the Hyperlink by calling the [**Hyperlink.delete()**](https://reference.aspose.com/cells/java/com.aspose.cells/hyperlink#delete--) method.
