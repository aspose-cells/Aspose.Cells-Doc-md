{{% alert color="primary" %}}

Aspose.Cells allows you to import an XML map into a workbook using the **Workbook.importXml()** method. You can import an XML map using Microsoft Excel with the following steps:

- Select the **Developer** tab  
- Click **Import** in the XML section and follow the required steps.

You will need to provide your XML data to complete the import. Here is a [sample XML data](5472501.txt) that you can use for testing.
