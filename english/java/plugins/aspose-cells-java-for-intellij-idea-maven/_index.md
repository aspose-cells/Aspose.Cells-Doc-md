## **Introduction**

### **IntelliJ IDEA**

**IntelliJ IDEA** is a Java integrated development environment (IDE) for software developers. It is developed by JetBrains. Originally created for Java, it currently supports a wide variety of languages, including JavaScript, Groovy, HTML, CSS, RSS, R, Haskell, PHP, Ruby, Python, Scala, Clojure, and Kotlin.

In a 2010 **InfoWorld** report, IntelliJ received the highest test center score out of the four top Java programming tools: Eclipse, IntelliJ IDEA, NetBeans and Oracle JDeveloper.

### **Aspose.Cells for Java**

[Aspose.Cells](https://products.aspose.com/cells/java/) for Java is an Excel® spreadsheet reporting component that enables Java applications to create and manage Excel® spreadsheets without using Microsoft Excel®. Aspose.Cells for Java is an incredibly feature‑rich component that offers much more than just basic data export functions.

### **Aspose.Cells Java for IntelliJ IDEA - Maven**

- Aspose.Cells Java for IntelliJ IDEA (Maven) is an **IntelliJ IDEA Plugin** which lets you create Maven projects (i.e., the created project contains Aspose.Cells for Java API Maven dependency) and download source code examples of the API for using **Aspose.Cells for Java API**.

- This plugin helps you work with **Aspose.Cells for Java API** inside your favorite IntelliJ IDEA.

**WIZARDS**:  
The plugin contains two wizards:

**Wizard 1:** Aspose.Cells Maven Project

- Plugin's **Aspose Maven Project Wizard** lets developers create Aspose.Cells for Java Maven projects.  
- The reference of **Aspose.Cells for Java API** Maven dependency is automatically fetched from the [Aspose Cloud Maven Repository](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) and is added to the `pom.xml`.  
- The created project will always contain the most recent available Maven dependency version for Aspose.Cells for Java.  
- The wizard steps also present an option for downloading [Source Code Examples for using Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples).

**Wizard 2:** Aspose.Cells Examples

- **Aspose.Cells Examples Wizard** lets you create or copy downloaded source code examples of Aspose.Cells for Java API into your project. All the examples for the selected category will be copied to the project's `com.aspose.cells.examples` package folder along with the required resources within the `src/main/resources` folder needed to run the examples.  

- [Source Code Examples of Aspose.Cells for Java API](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Examples) are meant for demonstrating the different usages of the API.

## **Downloading**

### **Downloading IntelliJ IDEA**

You will need to first install IntelliJ IDEA CE / UE before downloading the **Aspose.Cells Java for IntelliJ IDEA (Maven)** plugin.

To download IntelliJ IDEA:

1. Go to <https://www.jetbrains.com/idea/download/>  
2. Select your operating system  
3. Click on **UE** or **CE** download.

### **Downloading Aspose.Cells Java for IntelliJ IDEA plugin (Maven)**

- The plugin can be directly downloaded from the [JetBrains plugin hosting URL](https://goo.gl/R4pysl) on the IntelliJ IDEA Plugin Repository and later installed via the IDE:
  - Go to **File** > **Settings** > **Plugins**  
  - Select **Install Plugin from Disk...**  

- Or it can be installed directly (without manual downloading) using the IDE:
  - Go to **File** > **Settings** > **Plugins**  
  - Select **Browse Repositories**  
  - In the search box type: **Aspose**  
  - Select **Aspose Cells Java for IntelliJ IDEA Maven** plugin to install.
