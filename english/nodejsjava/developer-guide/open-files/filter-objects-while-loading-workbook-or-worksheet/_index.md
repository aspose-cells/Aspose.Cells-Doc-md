## **Possible Usage Scenarios**
Please use [LoadOptions.setLoadFilter](https://reference.aspose.com/cells/nodejs/LoadOptions#setLoadFilter) method while filtering data from the workbook. But if you want to filter data from individual worksheets, then you will have to override the [LoadFilter.startSheet](https://reference.aspose.com/cells/nodejs/LoadFilter#startSheet) method. Please provide an appropriate value from the **LoadDataFilterOptions** enumeration while creating or working with [LoadFilter](https://reference.aspose.com/cells/nodejs/loadfilter).

The **LoadDataFilterOptions** enumeration has the following possible values.

- ALL
- BOOK_SETTINGS
- CELL_BLANK
- CELL_BOOL
- CELL_DATA
- CELL_ERROR
- CELL_NUMERIC
- CELL_STRING
- CELL_VALUE
- CHART
- CONDITIONAL_FORMATTING
- DATA_VALIDATION
- DEFINED_NAMES
- DOCUMENT_PROPERTIES
- DRAWING
- FORMULA
- HYPERLINKS
- MERGED_AREA
- NONE
- OLE_OBJECT
- PICTURE
- PIVOT_TABLE
- REVISION
- SETTINGS
- SHAPE
- SHEET_DATA
- SHEET_SETTINGS
- STRUCTURE
- STYLE
- TABLE
- VBA
- XML_MAP
