## **Possible Usage Scenarios**

Sometimes users face a problem of duplicate columns while exporting data from a worksheet into the data table. DataTable cannot have duplicate columns, so duplicate columns must be renamed before you could export worksheet data to the data table. Aspose.Cells for Node.js via C++ can rename the duplicate columns automatically according to the strategy specified by you with [**RenameStrategy**](https://reference.aspose.com/cells/nodejs-cpp/renamestrategy/) property. If you specify [**RenameStrategy**](https://reference.aspose.com/cells/nodejs-cpp/renamestrategy/), columns will be renamed like column1, column2, column3, etc., and if you specify [**RenameStrategy**](https://reference.aspose.com/cells/nodejs-cpp/renamestrategy/), then columns will be renamed like columnA, columnB, columnC, etc.

## **Rename duplicate columns automatically while exporting worksheet data**

The following sample code adds some data in the first three columns of the worksheet, but all columns have the same name i.e. *People*. Then it exports the data from the worksheet into a data table by specifying [**RenameStrategy**](https://reference.aspose.com/cells/nodejs-cpp/renamestrategy/) strategy. It then prints the column names of the data table generated by Aspose.Cells for Node.js via C++. The following screenshot shows the data table with exported data in the visualizer. As you can see, duplicate columns have been renamed to PeopleA, PeopleB, etc.

## **Sample Code**

```javascript
const AsposeCells = require("aspose.cells.node");
const path = require("path");

// The path to the documents directory.
const dataDir = path.join(__dirname, "data");
const filePath = path.join(dataDir, "sample.xlsx");

// Create a workbook.
const wb = new AsposeCells.Workbook();

// Access first worksheet.
const ws = wb.getWorksheets().get(0);

// Write the same column name in columns A, B and C.
const columnName = "People";
ws.getCells().get("A1").putValue(columnName);
ws.getCells().get("B1").putValue(columnName);
ws.getCells().get("C1").putValue(columnName);

// Insert data in column A, B and C.
ws.getCells().get("A2").putValue("Data");
ws.getCells().get("B2").putValue("Data");
ws.getCells().get("C2").putValue("Data");

// Create ExportTableOptions and specify that you want to rename 
// duplicate column names automatically via RenameStrategy property.
const opts = new AsposeCells.ExportTableOptions();
opts.setExportColumnName(true);
opts.setRenameStrategy(AsposeCells.RenameStrategy.Letter);

// Export data to data table, duplicate column names will be renamed automatically.
const dataTable = ws.getCells().exportDataTable(0, 0, 4, 3, opts);

// Now print the column names of the data table generated by Aspose.Cells while exporting worksheet data.
for (let i = 0; i < dataTable.getColumns().getCount(); i++) {
    console.log(dataTable.getColumns().get(i).getColumnName());
}
```
