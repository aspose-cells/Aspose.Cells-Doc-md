## **Possible Usage Scenarios**

You can sort data in the column using a custom list. This can be done using [**DataSorter.addKey**](https://reference.aspose.com/cells/nodejs-cpp/datasorter/#addKey-number-sortorder-string-) method. However, this method works only if the items in the custom list do not have commas inside them. If they have commas like "USA,US", "China,CN" etc., then you must use [**DataSorter.addKey(number, SortOrder, string[])**)](https://reference.aspose.com/cells/nodejs-cpp/datasorter/#addKey-number-sortorder-stringarray-) method. Here, the last parameter is not String but an Array of Strings.
