## **How to Get Pivot Table Refresh Date and Refresh‑by‑Who Information**
[**PivotTable.getRefreshDate()**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getRefreshDate--) returns the date on which the PivotTable report was last refreshed. Similarly, the [**PivotTable.getRefreshedByWho()**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getRefreshedByWho--) property returns the name of the user who refreshed the report most recently.
