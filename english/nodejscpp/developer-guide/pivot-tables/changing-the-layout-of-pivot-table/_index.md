## **How to Change the Layout of Pivot Table in MS Excel**
Microsoft Excel allows you to change the layout of a pivot table using *PivotTable Tools > Design > Report Layout* menu commands. You can change the layout in these three forms:

- Show in Compact Form
- Show in Outline Form
- Show in Tabular Form

## **How to Change the Layout of Pivot Table Using Aspose.Cells for Node.js via C++**
Aspose.Cells for Node.js via C++ library also provides [**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#showInCompactForm--), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#showInOutlineForm--) and [**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#showInTabularForm--) methods to change the layout of a pivot table in these three forms.
