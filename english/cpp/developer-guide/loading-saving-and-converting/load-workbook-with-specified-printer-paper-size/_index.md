{{% alert color="primary" %}}

You can specify the printer paper size of your choice while loading your workbook using the [**LoadOptions.SetPaperSize()**](https://reference.aspose.com/cells/cpp/aspose.cells/loadoptions/setpapersize/) method. Please note that if you create a new file in MS Excel, you will find the paper size is the same as the setting of the default printer on your machine.

{{% /alert %}}

The following sample code illustrates the usage of the [**LoadOptions.SetPaperSize()**](https://reference.aspose.com/cells/cpp/aspose.cells/loadoptions/setpapersize/)Â method. It first creates a workbook, then saves it to a memory stream in XLSX format. Next, it loads the workbook with A5 paper size and saves it in PDF format. It then loads the workbook again with A3 paper size and saves it again in PDF format. If you open the output PDFs and check their paper sizes, you will see that they are different: one is A5 and the other is A3. Please download the [A5 output PDF](PrinterSize-a5_out.pdf) and [A3 output PDF](PrinterSize-a3_out.pdf) generated by the code for your reference.
