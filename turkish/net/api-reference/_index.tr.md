{{% alert color="primary" %}} 

Aspose.Cells for .NET, yazılım geliştiricilerin kendi uygulamaları içinden elektronik tablo dosyalarını manipüle etmelerine olanak tanıyan bir elektronik tablo programlama kütüphanesidir. Aspose.Cells for .NET popüler elektronik tablo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) dosya formatlarını destekler ve Excel dosyalarının PDF, XPS, HTML, MHTML, Düz Metin, TIFF, JPG, PNG, BMP ve SVG'ye dönüştürülmesine olanak tanır.

En son API Başvurusuna [Aspose.Cells for .NET API Başvurusu](https://reference.aspose.com/cells/net) adresinden ulaşılabilir.

{{% /alert %}}
