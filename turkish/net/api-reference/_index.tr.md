{{% alert color="primary" %}} 

Aspose.Cells for .NET, yazılım geliştiricilerin elektronik tablo dosyalarını kendi uygulamaları içinden değiştirmelerine ve dönüştürmelerine olanak tanıyan bir elektronik tablo programlama kitaplığıdır. Aspose.Cells for .NET, popüler elektronik tablo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) dosya formatlarını destekler ve Excel dosyalarının PDF, XPS, HTML, MHTML, Düz Metin, TIFF, JPG, PNG, 076 formatlarına aktarılmasına olanak tanır 1113481 ve SVG.

 En son API Referansını şu adreste bulabilirsiniz:[Aspose.Cells for .NET API Referans](https://reference.aspose.com/cells/net).
