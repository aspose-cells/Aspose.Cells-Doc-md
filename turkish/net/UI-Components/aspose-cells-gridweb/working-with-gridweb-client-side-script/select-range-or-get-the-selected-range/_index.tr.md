JavaScript kullanarak GridWeb'de bir aralık seçmek veya seçili aralığı almak için aşağıdaki istemci tarafı işlevlerini kullanabilirsiniz.

- getSelectRange()
- setSelectRange()
- clearSelections()

getSelectRange(), son seçilen aralığı döndürür. setSelectRange(), verilen aralığı seçer. clearSelections(), geçerli etkin hücre dışında tüm seçimi temizler.
