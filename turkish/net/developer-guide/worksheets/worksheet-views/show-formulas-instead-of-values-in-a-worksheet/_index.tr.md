{{% alert color="primary" %}}

 Microsoft Excel'de hesaplanan değerler yerine formülleri göstermek mümkündür.**Formülleri Göster** seçeneği**formüller**kurdele. Formüller gösterildiğinde, Microsoft Excel formülleri çalışma sayfasında görüntüler. Aynı şeyi Aspose.Cells'i kullanarak da elde edebilirsiniz.

{{% /alert %}}

 Aspose.Cells bir sağlar[**Çalışma Sayfası.Formülleri Göster**](https://reference.aspose.com/cells/net/aspose.cells/worksheet/properties/showformulas) Emlak. Bunu şuna ayarla:**doğru**Microsoft Excel'i formülleri gösterecek şekilde ayarlamak için.
