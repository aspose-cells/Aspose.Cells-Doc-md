Aspose.Cells, kullanıcıların Temayla ilgili verileri Excel dosyasından çıkarmasına olanak tanır. Örneğin, çalışma kitabına uygulanan Tema Adını ve hücreye veya hücrenin kenarlıklarına uygulanan Tema Rengini vb. çıkarabilirsiniz.

Microsoft Excel'de Sayfa Düzeni > Temalar komutunu kullanarak Temayı çalışma kitabınıza uygulayabilirsiniz.
