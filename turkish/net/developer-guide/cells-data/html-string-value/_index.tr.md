{{% alert color="primary" %}}

Aspose.Cells, Microsoft Excel odaklı HTML'in XLS/XLSX biçimine dönüştürülmesini destekler. Bu, Microsoft Excel tarafından oluşturulan HTML'in, Aspose.Cells kullanılarak XLS/XLSX biçimine dönüştürülebileceği anlamına gelir.

 Benzer şekilde, eğer basit bir HTML varsa, Aspose.Cells onu HTML Zengin Metin'e dönüştürebilir. Aspose.Cells sağlar[**Cell.HtmlString**](https://reference.aspose.com/cells/net/aspose.cells/cell/properties/htmlstring) Bu kadar basit bir HTML'i alıp onu biçimlendirilmiş hücre metnine dönüştürebilen özellik.

{{% /alert %}}
