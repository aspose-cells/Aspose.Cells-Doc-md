## **Olası Kullanım Senaryoları**

Aspose.Cells, bir tabloyu çalışma sayfasına eklerken ilk satırı aşağı kaydırmanıza olanak tanır. Bu belge, Aspose.Cells API'larını kullanarak görevi nasıl gerçekleştirebileceğinizi açıklar.

## **Hücreler Veri Tablosu Satırlarını Eklerken İlk Satırı Aşağıya Kaydırma**

Aşağıdaki örnek kod, bir tabloyu çalışma sayfasına eklerken ilk satırı aşağı kaydırmayı göstermektedir. Özelliği boolean [**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown)'yi **True**/**False** olarak ayarlayarak özelliği kullanabilirsiniz. Daha iyi anlamak için lütfen örnek Excel dosyasını inceleyin.
