## **Olası Kullanım Senaryoları**

Aspose.Cells, çalışma kitabını şuraya kaydetmenizi sağlar:*Kesin Açık XML Elektronik Tablosu*biçim. Bu amaçla, sağladığı**[Workbook.Settings.Compliance](https://reference.aspose.com/cells/net/aspose.cells/workbooksettings/properties/compliance)**Emlak. Değerini olarak ayarlarsanız**[OoxmlCompliance.Iso29500_2008_Strict](https://reference.aspose.com/cells/net/aspose.cells/ooxmlcompliance)**, çıktı Excel dosyası Strict Open XML Elektronik Tablosu biçiminde kaydedilecektir.
