 Aspose.Cells şunları sağlar:[**OleObject.AutoLoad**](https://reference.aspose.com/cells/net/aspose.cells.drawing/oleobject/properties/autoload) Excel dosyası Microsoft Excel'de açıldığında OLE nesnesini yenileme özelliği. Bu özellik nedeniyle OLE nesnesi, Microsoft Excel tarafından oluşturulan doğru OLE görüntüsünü görüntüler.

{{% /alert %}}

 Aşağıdaki örnek kod,[örnek excel dosyası](5115231.xlsx) gerçek olmayan bir OLE görüntüsüne sahip olan. OLE nesnesi aslında bir Microsoft Word belgesidir ancak örnek excel dosyası Microsoft Word görüntüsü yerine hayvan görüntüsünü gösterir. Ama eğer açarsan[çıktı excel dosyası](5115225.xlsx), Microsoft Excel'in doğru OLE görüntüsünü gösterdiğini göreceksiniz.
