Bazen pivot tablo hücrelerini biçimlendirmek istersiniz. Örneğin, pivot tablo hücrelerine arka plan rengi uygulamak istiyorsunuz. Aspose.Cells iki yöntem sunar[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) Ve[**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format), bu amaçla kullanabilirsiniz.

[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) stili pivot tablonun tamamına uygularken[**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format) stili pivot tablonun tek bir hücresine uygular.
