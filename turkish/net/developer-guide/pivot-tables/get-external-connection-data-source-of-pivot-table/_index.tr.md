## **Pivot Tablonun Dış Bağlantı Veri Kaynağını Alın**

Aspose.Cells, pivot tablonun harici bağlantı veri kaynağını alma olanağı sağlar. Bunun için API,[**Harici Bağlantı Veri Kaynağı**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)mülkiyeti[**Pivot tablo**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) sınıf. bu[**Harici Bağlantı Veri Kaynağı**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource)özellik döndürür[**Harici Bağlantı**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) nesne. Aşağıdaki kod parçacığı,[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) pivot tablonun harici bağlantı veri kaynağını alma özelliği.
