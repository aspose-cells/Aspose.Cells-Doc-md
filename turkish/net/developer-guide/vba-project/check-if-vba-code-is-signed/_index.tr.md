Aspose.Cells, kullanıcının VBA kod projesinin imzalı olup olmadığını kontrol etmesine izin verir. Lütfen VBA kod projesinin imzalı olup olmadığını kontrol etmek için [**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/issigned) özelliğini kullanın.

{{% /alert %}}

Bu kodu test etmek için herhangi bir excel dosyasını kullanabilirsiniz. Test amaçlı [bu kodda kullanılan excel dosyasını](5115032.xlsm) kullanabilirsiniz.
