##  **Olası Kullanım Senaryoları**
Aspose.Cells'in ilk sürümleri, grafik görüntüye veya PDF'e dönüştürüldüğünde grafik ekseninin otomatik birimlerini düzgün şekilde işleyemiyordu. Artık Aspose.Cells, grafik ekseninin otomatik birimlerinin işlenmesini destekliyor. Kod değişikliği yoktur. Grafiğinizi resme veya PDF'e dönüştürün; grafik eksenini tıpkı Microsoft Excel'in oluşturduğu gibi oluşturacaktır.
