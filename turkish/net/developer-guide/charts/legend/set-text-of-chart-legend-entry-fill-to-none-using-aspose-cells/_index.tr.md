 Grafiğin açıklama girişi dolgu metnini, grafik açıklaması içinde görüntülenmemesi için hiçbiri olarak ayarlamak istiyorsanız, lütfen[**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/net/aspose.cells.charts/legendentry/properties/istextnofill) ile**doğru**.

{{% /alert %}}
