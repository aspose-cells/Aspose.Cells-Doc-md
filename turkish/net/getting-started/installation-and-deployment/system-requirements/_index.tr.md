## **Geliştirme Ortamları**
Aspose.Cells for .NET platformunu hedefleyen herhangi bir geliştirme ortamında uygulama geliştirmek için kullanabilirsiniz, ancak aşağıdaki ortamlar açıkça desteklenir:

- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- Microsoft Visual Studio 2022

{{% alert color="primary" %}} 

Aspose.Cells'i kullanmak için makinede Microsoft Excel'in yüklü olması gerekmez, çünkü kitaplığın kendisi bir elektronik tablo yönetim motorudur. Ancak bileşen tarafından oluşturulan excel dosyasını görüntülemek için makinenizde en az Microsoft Excel Viewer kurulu olmalıdır.

{{% /alert %}} 
## **Aspose.Cells Grid Suite Sistem Gereksinimleri**
Aspose.Cells Grid Suite iki kontrolden oluşur:

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb

Her iki kontrolün sistem gereksinimleri ayrı ayrı açıklanmıştır.

{{% alert color="primary" %}} 

Bu iki kontrolü kullanmak için Aspose.Cells.GridDesktop ve Aspose.Cells.GridWeb çalıştıracak makinede Aspose.Cells.GridDesktop ve Aspose.Cells.GridWeb olarak Microsoft Excel'in kurulu olması gerekmez. Microsoft Excel'e gerek kalmadan Excel elektronik tablolarını açıp kaydedebilir.
