## **Geliştirme Ortamları**
Aspose.Cells for .NET .NET platformunu hedefleyen herhangi bir geliştirme ortamında uygulamalar geliştirmek için kullanılabilir, ancak aşağıdaki ortamlar açıkça desteklenir:

- Microsoft Visual Studio 2005
- Microsoft Visual Studio 2008
- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- Microsoft Visual Studio 2022

{{% alert color="primary" %}} 

Aspose.Cells'ı kullanmak için kütüphane kendisi bir elektronik tablo yönetim motoru olduğu için makinenin Microsoft Excel yüklü olmasına gerek yoktur. Ancak, bileşen tarafından üretilen excel dosyasını görüntülemek için makinenizde en azından Microsoft Excel Viewer yüklü olmalıdır.

{{% /alert %}} 
## **Aspose.Cells Grid Suite Sistem Gereksinimleri**
Aspose.Cells Grid Suite üç kontrol tarafından oluşur:

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb
- Aspose.Cells.GridJs

Kontrollerin sistem gereksinimleri ayrı ayrı açıklanmıştır.

{{% alert color="primary" %}} 

Bu kontrolleri kullanmak için Aspose.Cells.GridDesktop ve Aspose.Cells.GridWeb'i çalıştırmak için makinenizin Microsoft Excel yüklü olmasına gerek yoktur çünkü Aspose.Cells.GridDesktop ve Aspose.Cells.GridWeb Microsoft Excel'e ihtiyaç duymadan Excel elektronik tablolarını açabilir ve kaydedebilir.
