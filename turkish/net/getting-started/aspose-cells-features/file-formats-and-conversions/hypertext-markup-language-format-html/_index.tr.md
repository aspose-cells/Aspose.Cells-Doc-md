{{% alert color="primary" %}} 

Aspose.Cells, Excel çalışma kitaplarınızı HTML biçiminde kaydetmek için kullanılabilir. Dışa aktarılan HTML çalışma kitapları Microsoft Excel odaklıdır, yani Microsoft Excel tarafından dışa aktarılanlara benzerler. HTML'den çalışma kitapları da oluşturabilirsiniz, ancak bu HTML Excel odaklı olmalıdır.

{{% /alert %}} 
## **Aspose.Cells'de HTML**
Aspose çözümü, HTML'ye dönüştürürken aşağıdaki belge özelliklerini tamamen destekler:

- Cell Veri ve Metin Biçimlendirme
- Grafikler ve Pivot Tablolar
- Üstbilgiler/Altbilgiler
- Kenarlıklar, Kenarlık Renkleri
- Görüntüler, Metin Kutuları ve Çizim Şekilleri
- köprüler
