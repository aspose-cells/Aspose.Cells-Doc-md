## **Desteklenen Platformlar**
Aspose.Cells for .NET ürünü, popüler geliştirme ortamlarının ve dağıtım platformlarının çoğunu kapsamaktadır. 

|**Özellik**|**Açıklama**|
| :- | :- |
|[.NET Framework](/cells/tr/net/system-requirements/#net-frameworks)|ASP.NET, Web Hizmetleri, WinForms veya .NET Framework sürümleri 2.0 veya sonrası için herhangi bir türde uygulama geliştirmek için Aspose.Cells for .NET kullanın. 32-bit ve 64-bit için tam destek.|
|[.NET Standard 2.0](/cells/tr/net/system-requirements/#net-standard)|'netStandard' klasörü, .NET Core 2.1/3.1 gibi .NET Standard 2.0 uygulayan çerçevelerle kullanmak için derlemeyi içerir. Ayrıca, Aspose.Cells API'nin en son sürümü .NET 5.0, .NET 6.0, .NET 7.0 ve .NET8.0'ı da desteklemektedir.|
|[COM](/cells/tr/net/system-requirements/)|Ön-.NET teknolojileri kullanarak geliştirme yapmak zorunda kalırsanız hayat kurtarıcı olabilir.|
|[Mono](/cells/tr/net/system-requirements/)|Mono ile uygulamalar geliştirmek için Aspose.Cells for .NET'yi kullanabilirsiniz.|
|[Windows Azure](/cells/tr/net/system-requirements/)|Aspose.Cells for .NET aynı zamanda Windows Azure'da çalışır.|
