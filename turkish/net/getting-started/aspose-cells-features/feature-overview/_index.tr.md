##  **Desteklenen Platformlar**
Aspose.Cells for .NET ürünü, popüler geliştirme ortamlarının ve dağıtım platformlarının çoğunu kapsar.

|**Özellik**|**Tanım**|
| :- | :- |
|[.NET Çerçeve](/cells/tr/net/system-requirements/#net-frameworks)|ASP.NET, Web Hizmetleri, WinForms veya İstemci Profili sürümleri de dahil olmak üzere herhangi bir .NET Framework sürüm 2.0 veya üzeri için başka türde bir uygulama oluşturmak için Aspose.Cells for .NET'i kullanın. 32 bit ve 64 bit için tam destek.|
|[.NET Standart 2.0](/cells/tr/net/system-requirements/#net-standard)|'netStandard' klasörü, .NET Core 2.1/3.1 gibi .NET Standard 2.0'ı uygulayan çerçeveyle kullanılacak derlemeyi içerir. Aspose.Cells API'in en son sürümü ayrıca .NET 5.0, .NET 6.0, .NET 7.0 ve .NET8.0'ı da destekler.|
|[COM](/cells/tr/net/system-requirements/)|.NET öncesi teknolojileri kullanarak geliştirme yapmanız gerekiyorsa cankurtaran olabilir.|
|[Mono](/cells/tr/net/system-requirements/)|Mono ile uygulama oluşturmak için Aspose.Cells for .NET'i kullanabilirsiniz.|
|[Windows Azure](/cells/tr/net/system-requirements/)|Aspose.Cells for .NET ayrıca Windows Azure'da da çalışır.|
