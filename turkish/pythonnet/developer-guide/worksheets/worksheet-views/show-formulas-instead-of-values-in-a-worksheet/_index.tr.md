{{% alert color="primary" %}}

Microsoft Excel'de hesaplanmış değerler yerine formülleri göstermek mümkündür ve bunun için **Formüller** şeridindeki **Formülleri Göster** seçeneğini kullanabilirsiniz. Formüller gösterildiğinde, Microsoft Excel, çalışma sayfasındaki formülleri gösterir. Aynı şeyi Aspose.Cells for Python via .NET kullanarak da başarabilirsiniz.

{{% /alert %}}

Aspose.Cells for Python via .NET, **true** olarak ayarlanması gerekli olan bir [**Worksheet.show_formulas**](https://reference.aspose.com/cells/python-net/aspose.cells/worksheet/show_formulas) özelliği sağlar. Bu, Microsoft Excel'in formülleri göstermesi için ayarlanır.
