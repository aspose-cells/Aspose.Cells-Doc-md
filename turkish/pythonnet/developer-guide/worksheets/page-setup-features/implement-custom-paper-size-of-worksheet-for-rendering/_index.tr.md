## **Olası Kullanım Senaryoları**

MS Excel'de özel kağıt boyutları oluşturmak için doğrudan bir seçenek bulunmamaktadır, ancak Excel dosyasını PDF'ye dönüştürürken istediğiniz sayfaların kağıt boyutunu ayarlayabilirsiniz. Bu belge, Aspose.Cells for Python via .NET API'leri kullanarak bir sayfaya özel kağıt boyutu ayarlama yollarını anlatır.

## **Otomatik Olarak Çalışma Sayfası için Özel Kağıt Boyutunun Uygulanması**

Aspose.Cells for Python via .NET, istediğiniz sayfa boyutunu uygulamanıza olanak tanır. [**custom_paper_size**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup/custom_paper_size/#float-float) metodunu [**PageSetup**](https://reference.aspose.com/cells/python-net/aspose.cells/pagesetup) sınıfında kullanarak özel bir sayfa boyutu belirtebilirsiniz. Aşağıdaki örnek kod, çalışma kitabındaki ilk sayfa için özel bir kağıt boyutu belirlemeyi gösterir. Ayrıca, aşağıdaki kodla oluşturulan [çıktı PDF'sine](45056028.pdf) de bakabilirsiniz.
