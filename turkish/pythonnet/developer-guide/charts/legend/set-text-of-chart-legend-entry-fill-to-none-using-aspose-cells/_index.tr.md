{{% alert color="primary" %}}

Eğer grafik efsanesinin giriş doldurmasını hiçbiri olarak ayarlamak istiyorsanız, lütfen [**LegendEntry.is_text_no_fill**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/legendentry/is_text_no_fill)'yi **true** olarak ayarlayın.
