## **Tick Etiketi Yönünü Değiştirme**

Aspose.Cells for Python via .NET size, grafik tekerlek çubuğu etiketi yönünü [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type) özelliğini kullanarak değiştirme imkanı sağlar. [**TickLabels.direction_type**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/ticklabels/direction_type) özelliği, [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/) öbek nesne değerini kabul eder. [**ChartTextDirectionType**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/charttextdirectiontype/) öbeği, aşağıdaki üyeleri sağlar

- YATAY
- DİKEY
- DÖNÜŞ 90
- DÖNÜŞ 270
- Yığılı
