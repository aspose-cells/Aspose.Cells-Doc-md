## Python'da VBA Projesinin Korunduğunu ve Görüntülemeye Kapanmış olduğunu Kontrol Et

Aspose.Cells for Python via .NET, Excel dosyasının VBA (Visual Basic for Applications) Projesinin korunduğunu ve görüntülemeye kapatıldığını kontrol etmenizi sağlar. Bunun için API, [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/) özelliğini sunar. Eğer görüntüleme için kilitliyse, o zaman [**VbaProject.islocked_for_viewing**](https://reference.aspose.com/cells/python-net/aspose.cells.vba/vbaproject/islocked_for_viewing/) özelliği **doğru** döner.
