{{% alert color="primary" %}}

Aspose.Cells for Python via .NET'i kullanarak farklı pivot tablo seçeneklerini ayarlayabilirsiniz. Bu seçeneklerden biri "Boş hücreleri göster" seçeneğidir. Bu seçeneği ayarladığınızda pivot tablodaki tüm boş hücreler belirtilen dize olarak görüntülenir.

{{% /alert %}}

##  **Microsoft Excel'de Pivot Tablo Seçeneğinin Ayarlanması**

Bu seçeneği Microsoft Excel'de bulup ayarlamak için:

1. Bir pivot tablo seçin ve sağ tıklayın.
1. *Özet Tablo Seçenekleri**'ni seçin.
1.  Şunu seçin:**Düzen ve Format** sekmesi.
1.  Şunu seçin:**Boş hücrelerin gösterimi için** seçeneğini seçin ve bir dize belirtin.

##  **Aspose.Cells for Python via .NET Kullanarak Pivot Tablo Seçeneğinin Ayarlanması**

Aspose.Cells for Python via .NET şunları sağlar[**PivotTable.display_null_string**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/display_null_string/) Ve[**PivotTable.null_string**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/null_string/) "Boş hücreler için göster" pivot tablo seçeneğinin ayarlanmasına yönelik özellikler.
