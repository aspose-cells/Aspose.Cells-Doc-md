## **Olası Kullanım Senaryoları**

Microsoft Excel, aşağıdaki ekran görüntüsünde gösterildiği gibi çalışma kitabınızı paylaşmanıza izin verir. Çalışma kitabını paylaştığınızda, ağ üzerinde birden fazla kullanıcı çalışma kitabını düzenleyebilir. Aspose.Cells for Python via .NET, [**Workbook.settings.shared**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/shared) özelliği ile paylaşılan çalışma kitabı oluşturmaya olanak tanır.

![todo:image_alt_text](create-shared-workbook-with-aspose-cells_1.png)

## **Aspose.Cells for Python via .NET ile Paylaşılan Çalışma Kitabı Oluştur**

Aşağıdaki örnek kod, [**Workbook.settings.shared**](https://reference.aspose.com/cells/python-net/aspose.cells/workbooksettings/shared) özelliğini **true** olarak ayarlayarak paylaşılan bir çalışma kitabı oluşturur. Microsoft Excel'de [çıktı Excel dosyasını](55541786.xlsx) açtığınızda, bu ekran görüntüsünde gösterildiği gibi çıktı çalışma kitabının adıyla birlikte **Paylaşılan** göreceksiniz.
