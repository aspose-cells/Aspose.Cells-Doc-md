## **Olası Kullanım Senaryoları**

Bazen, Japonya, Çin, Tayland gibi, Doğu Asya dili fontunda metin göstermek istersiniz. Aspose.Cells for Python via .NET, Doğu Asya dillerinin font adını belirtmek için [**TextOptions.far_east_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/far_east_name) özelliğini sağlar. Ayrıca, Latin font adını [**TextOptions.latin_name**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing.texts/textoptions/latin_name) özelliği kullanarak belirleyebilirsiniz.
