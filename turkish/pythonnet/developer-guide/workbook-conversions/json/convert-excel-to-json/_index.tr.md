##  **Excel Çalışma Kitabını JSON'e dönüştürün**

Excel Çalışma Kitabı'nı JSON'e nasıl dönüştüreceğinizi merak etmenize gerek yok, çünkü Aspose.Cells for Python via .NET kütüphanesi en iyi karara sahiptir. Aspose.Cells API, e-tabloların JSON biçimine dönüştürülmesi için destek sağlar. Çalışma kitabını JSON'e aktarmak için şunu iletin:[**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) ikinci parametre olarak[**Çalışma kitabı.kaydet**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) yöntem. Ayrıca kullanabilirsiniz[**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) Çalışma sayfasını JSON'e aktarmak için ek ayarları belirtmek üzere sınıf.
