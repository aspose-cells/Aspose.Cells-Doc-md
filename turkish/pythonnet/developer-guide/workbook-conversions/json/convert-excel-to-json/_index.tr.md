## **Excel Çalışma Kitabını JSON'a Dönüştür**

Excel Çalışma Kitabını JSON'a dönüştürmenin nasıl yapılacağını merak etmeye gerek yok, çünkü Aspose.Cells for Python via .NET kütüphanesinin en iyi kararı vardır. Aspose.Cells API, elektronik tabloları JSON formatına dönüştürmeyi destekler. Çalışma kitabını JSON'a dışa aktarmak için, [**Workbook.save**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) yönteminin ikinci parametresi olarak [**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) geçirin. Ayrıca [**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) sınıfını kullanarak çalışma sayfasını JSON'a dışa aktarmak için ek ayarlar belirleyebilirsiniz.
