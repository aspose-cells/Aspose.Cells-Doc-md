##  **Excel Çalışma Kitabını JSON'e dönüştürün**

Aspose.Cells for Python via .NET API, e-tabloların JSON biçimine dönüştürülmesi için destek sağlar. Çalışma kitabını JSON'e aktarmak için şunu iletin:[**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) ikinci parametre olarak[**Çalışma kitabı.kaydet**](https://reference.aspose.com/cells/python-net/aspose.cells/workbook/save/#str-aspose.cells.SaveOptions) yöntem. Ayrıca kullanabilirsiniz[**JsonSaveOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/JsonSaveoptions) Çalışma sayfasını JSON'e aktarmak için ek ayarları belirtmek üzere sınıf.

 Aşağıdaki kod örneği, etkin çalışma sayfasının kullanılarak Json'e aktarılmasını gösterir.[**SaveFormat.JSON**](https://reference.aspose.com/cells/python-net/aspose.cells/saveformat) numaralandırma üyesi Lütfen dönüştürülecek koda bakın[Kaynak dosyası](book1.xlsx) -e[çıktı Json dosyası](book1.Json)referans kodu tarafından oluşturulur.
