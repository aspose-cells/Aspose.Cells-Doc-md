{{% alert color="primary" %}} 

 Aspose.Cells for Python via .NET, Excel dosyasını HTML'ye dışa aktarırken sağa sola metin genişletmeyi destekler. Bu özellik, v8.9.0.0 sürümünden beri uygulanmaktadır. Kaynak Excel dosyanızda sağdan sola genişleyen herhangi bir metin varsa, Aspose.Cells bunu doğru şekilde HTML'ye aktarır.

{{% /alert %}}
