##  **Başlıklarla Excel'i HTML'e dönüştürün**

Aspose.Cells for Python ile NET Excel'i HTML'e dönüştürürken başlıkları dışa aktarma seçeneği sunar. Bunun için[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)özelliği *true** olarak değiştirin. Varsayılan değeri[**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)özellik *yanlış**.
