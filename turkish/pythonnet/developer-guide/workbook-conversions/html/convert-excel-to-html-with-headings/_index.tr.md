## **Başlıkları İle Excel'i HTML'e Dönüştürme**

Aspose.Cells için Python via NET, Excel'i HTML'e dönüştürürken başlıkları dışa aktarma seçeneği sağlar. Bunun için [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) özelliğini **true** olarak ayarlayabilirsiniz. [**HtmlSaveOptions.export_row_column_headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/) özelliğinin varsayılan değeri **false**'dir.
