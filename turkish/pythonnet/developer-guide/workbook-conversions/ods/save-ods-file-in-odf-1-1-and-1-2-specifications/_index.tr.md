{{% alert color="primary" %}}

 Aspose.Cells for Python via .NET, ODS dosyasının (**OpenDocument Elektronik Tablosu**) ODF (**OpenDocument Formatı**) 1.1 ve 1.2 spesifikasyonlarına kaydedilmesini destekler. Aspose.Cells for Python via .NET var[**OdsSaveOptions.is_strict_schema11**](https://reference.aspose.com/cells/python-net/aspose.cells/odssaveoptions/is_strict_schema11/)ODS dosyalarını kaydetmek için ODF 1.1 spesifikasyonunun kullanımını belirten özellik. Bu özelliğin varsayılan değeri *yanlış** olduğundan, bu ayar olmadan kaydedilen ODS dosyası 1.2 özelliklerini kullanır.

{{% /alert %}}

Aşağıdaki örnek kod, bir çalışma kitabı nesnesi oluşturur, ilk çalışma sayfasındaki A1 hücresine bir miktar değer ekler ve ardından ODS dosyasını ODF 1.1 ve 1.2 belirtimlerine kaydeder. Varsayılan olarak ODS dosyası ODF 1.2 spesifikasyonuna kaydedilir.
