## **Olası Kullanım Senaryoları**

Excel dosyasının dilini değiştirmek için dosyaya sağ tıklayın ve ardından Özellikler > Ayrıntılar > ve Dili alanını düzenleyin. Programlı olarak değiştirmek için [**BuiltInDocumentPropertyCollection.language**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/language) özelliğini Aspose.Cells for Python via .NET API'leriyle kullanın.

Aşağıdaki örnek kod bir çalışma kitabı oluşturur ve Dil adı verilen yerleşik belge özelliğini değiştirir.
