## **Olası Kullanım Senaryoları**

Excel dosyasının **Sürüm numarasını** sağ tıklayarak ve ardından Özellikler > Ayrıntılar > ve **Sürüm numarası** alanını düzenleyerek değiştirebilirsiniz. Programlı olarak değiştirmek için [**BuiltInDocumentPropertyCollection.document_version**](https://reference.aspose.com/cells/python-net/aspose.cells.properties/builtindocumentpropertycollection/document_version) özelliğini Aspose.Cells for Python via .NET API'leriyle kullanın.
