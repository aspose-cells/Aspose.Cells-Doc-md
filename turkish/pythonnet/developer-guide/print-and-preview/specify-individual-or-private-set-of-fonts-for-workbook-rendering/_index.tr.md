## **Olası Kullanım Senaryoları**

Genellikle, tüm çalışma kitapları için fontlar dizini veya font listesi belirlenir, ancak bazen, çalışma kitabınız için bireysel veya özel font setleri belirtmeniz gerekebilir. Aspose.Cells for Python via .NET, çalışma kitabınız için bireysel veya özel font setleri belirlemek üzere kullanılabilen [**IndividualFontConfigs**](https://reference.aspose.com/cells/python-net/aspose.cells/individualfontconfigs) sınıfını sağlar.
