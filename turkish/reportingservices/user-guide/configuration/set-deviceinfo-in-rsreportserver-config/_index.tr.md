- **FileExtension** 
  Değer null olduğunda, dışa aktarılan rapor dosyası uzantı adı varsayılan değer olacaktır. Değer null olmadığında, dışa aktarılan rapor dosyasının uzantı adı değer olacaktır.
- **SimplePageHeaders** 
  Değer true olduğunda, rapor başlık öğesini Excel sayfa başlığına dönüştürür. Varsayılan değer false'dur.
- **SimplePageFooters** 
  Değer true olduğunda, rapor altbilgi öğesini Excel sayfa altbilgisine dönüştürür. Varsayılan değer true'dur.
- **PutoutHeader** 
  Değer true olduğunda, rapor başlık öğesini dışa aktarır. Değer false olduğunda, rapor başlık öğesini dışa aktarmaz. Varsayılan değer true'dur. Değer yalnızca Excel2007Xlsx(Yalnızca Veri) uzantısını destekler.
- **PutoutFooter** 
  Değer true olduğunda, rapor altbilgi öğesini dışa aktarır. Değer false olduğunda, rapor altbilgi öğesini dışa aktarmaz. Varsayılan değer true'dur. Değer yalnızca Excel2007Xlsx(Yalnızca Veri) uzantısını destekler.
- **FillTableGroupHeaderForSimpleOutPut** 
  Varsayılan değer false'dur. Değer yalnızca Excel2007Xlsx(Yalnızca Veri) uzantısını destekler.
- **NoOutPutTotalForSimpleOutPut** 
  Varsayılan değer false'dur. Değer yalnızca Excel2007Xlsx(Yalnızca Veri) uzantısını destekler.
- **NoOutPutGroupForSimpleOutPut** 
  Varsayılan değer false'dur. Değer yalnızca Excel2007Xlsx(Yalnızca Veri) uzantısını destekler.
- **NoDoGroupPageForSimpleOutPut** 
  Varsayılan değer true'dur. Değer yalnızca Excel2007Xlsx(Yalnızca Veri) uzantısını destekler.
- **NoDoPageForSimpleOutPut** 
  Varsayılan değer true'dur. Değer yalnızca Excel2007Xlsx(Yalnızca Veri) uzantısını destekler.
- **FieldDelimiter** 
  Alan ayraçlarını ayarlar. Değer CSV ve TXT uzantılarını destekler.
