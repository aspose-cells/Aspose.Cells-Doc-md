{{% alert color="primary" %}} 

SQL Server Reporting Services 2008 rapor tanımı dosyasındaki bazı grafik türleri, Excel grafik türleri olarak mevcut değildir:

- Çizgi Adımlı
- Aralık-Düz
- Menzil-Pürüzsüz
- Aralık Sütunu
- Menzil Çubuğu
- Menzil-ErrorBar
- Range-BoxPlot
- Şekil-Huni

Microsoft Excel'de eşdeğer seçeneklere sahip olmayan grafik öğelerini işlerken, Aspose.Cells bunları görüntülere verir.

{{% /alert %}}
