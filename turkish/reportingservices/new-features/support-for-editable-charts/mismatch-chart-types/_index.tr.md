{{% alert color="primary" %}} 

SQL Server Reporting Services 2008 rapor tanım dosyasındaki bazı grafik tipleri, Excel grafik tipleri olarak mevcut değil:

- Çizgi-Adımlı
- Aralık-Düz
- Aralık-Düzgün
- Aralık-Sütun
- Aralık-Çubuk
- Aralık-HataÇubuğu
- Aralık-KutuGrafiği
- Şekil-Trambolin

Microsoft Excel'de eşdeğer seçenekleri olmayan grafik öğelerini çıkardığında Aspose.Cells onları resimlere çıkarır. 

{{% /alert %}}
