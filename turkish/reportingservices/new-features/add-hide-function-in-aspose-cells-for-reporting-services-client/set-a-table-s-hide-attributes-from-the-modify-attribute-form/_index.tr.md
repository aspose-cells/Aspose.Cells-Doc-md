 Bir tablo raporunun gizleme özniteliklerini şuradan ayarlayın:**Niteliği Değiştir**. Parametreler şunları içerir:

- **Tip** gizleme nesnesi türünü temsil eder. Tablo, satır ve sütun içerir.
- **dizin**gizleme nesnesinin konumunu temsil eder. Tür çalışma sayfası olduğunda, değer boştur. İndeks değeri 0'dan başlar.
- **İfade** gizleme durumu Değerini temsil eder. Değer true, false veya bir Excel formülüdür. İfade, Raporlama Servisleri formülünü destekler.
