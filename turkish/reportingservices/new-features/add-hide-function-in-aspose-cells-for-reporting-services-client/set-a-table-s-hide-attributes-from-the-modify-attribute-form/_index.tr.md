**Öznitelikleri Değiştir** kullanarak bir tablo raporunun gizleme özniteliklerini ayarlayın. Parametreler şunları içerir:

- **Tür**, gizleme nesne türünü temsil eder. Tablo, satır ve sütunu içerir.
- **Endeks**, gizleme nesnesinin konumunu temsil eder. Tür çalışma sayfası olduğunda değer boştur. Endeks değeri 0'dan başlar.
- **İfade**, gizleme durumu Değerini temsil eder. Değer true, false veya bir Excel formülü olabilir. İfade, Reporting Services formülünü destekler.
