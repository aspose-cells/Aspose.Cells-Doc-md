## **Sistem Gereksinimleri**
Aspose.Cells for Python via Java, platformdan bağımsız bir API'dir ve [Python](https://www.python.org/downloads/) kurulu olan herhangi bir platformda (Windows, Linux ve MacOS) kullanılabilir. Kurulumu yapmadan önce makinenin Java 8 veya daha yeni sürümlere sahip olması gerekir.

## **Python Sürümü**
- Python 3.5 veya daha yeni
## **Java Sürümü**
- Java 1.8 veya daha yeni

## **Kurulum**
### **Windows:**
- **Java'yı yükleyin ve JAVA_HOME ve PATH ortam değişkenlerini ayarlayın**
Örneğin:
{{< highlight java >}}

JAVA_HOME=C:\jdk1.8.0_131

PATH=C:\jdk1.8.0_131\bin;

{{< /highlight >}}

- **Aspose.Cells for Python via Java'ü pypi'dan kurun**
Aşağıdaki komutla Aspose.Cells for Python via Java'ü [pypi](https://pypi.org/project/aspose-cells/) üzerinden kolayca kullanabilirsiniz.
{{< highlight java >}}

 $ pip install aspose-cells

{{< /highlight >}}
