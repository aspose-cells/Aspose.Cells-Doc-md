## **Excel'i Markdown'a Dönüştür**
Aspose.Cells for Python via Java, Excel dosyalarını Markdown biçimine dönüştürmeyi destekler. Aktif çalışma sayfasını Markdown'e dışa aktarmak için, [SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN) 'i [Workbook.Kaydet](https://reference.aspose.com/cells/python/asposecells.api/workbook#save\(java.lang.String,%20int\)) metodunun ikinci parametresi olarak geçirin. Ek ayarları belirtmek için [MarkdownSaveOptions](https://reference.aspose.com/cells/python/asposecells.api/MarkdownSaveOptions) sınıfını da kullanabilirsiniz.

Aşağıdaki örnek kod, [SaveFormat.Markdown](https://reference.aspose.com/cells/python/asposecells.api/saveformat#MARKDOWN) numaralandırma üyesini kullanarak, etkin çalışma sayfasını Markdown'e dışa aktarma işlemini göstermektedir. Referans için, kod tarafından oluşturulan [çıktı Markdown dosyasını](Book1.txt) görmek için çıktı görseline bakın.
