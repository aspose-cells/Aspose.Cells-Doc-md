## **Excel Çalışma Kitabını PPT'ye Dönüştür**

Excel Çalışma Kitabını PowerPoint'e dönüştürme konusunda ne yapılacağını merak etmenize gerek yok, çünkü Aspose.Cells for Python via Java kütüphanesi en iyi seçeneği sunar. Aspose.Cells for Python via Java API, elektronik tabloları PowerPoint biçimine dönüştürmeyi destekler. Çalışma kitabını PowerPoint'e dışa aktarmak için [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)) yönteminin ikinci parametresi olarak [**SaveFormat.PPTX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat)'ı geçirin. Ayrıca, çalışma sayfasını .pptx dosyasına dışa aktarmak için [**PptxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/PptxSaveOptions) sınıfını kullanabilirsiniz.
