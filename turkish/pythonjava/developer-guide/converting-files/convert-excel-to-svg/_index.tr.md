## **Excel'i SVG'e dönüştür**
SVG(**Ölçeklendirilebilir Vektör Grafiği)**iki boyutlu vektör grafikleri için XML standartlarına dayalı bir belirtimdir. World Wide Web Consortium (W3C) tarafından 1999'dan beri geliştirilmekte olan açık bir standarttır.

Aspose.Cells for Python via Java, Excel dosyalarının SVG resimlere dönüştürülmesini destekler. Bunun için API,[SheetRender](https://reference.aspose.com/cells/python/asposecells.api/SheetRender), [ResimVeyaBaskıSeçenekleri](https://reference.aspose.com/cells/python/asposecells.api/ImageOrPrintOptions), ve[Çalışma KitabıRender](https://reference.aspose.com/cells/python/asposecells.api/WorkbookRender)sınıflar.
