## **Excel Çalışma Kitabını Word'e Dönüştür**

Excel Workbook'u Word'e nasıl çevireceğimi merak etmenize gerek yok çünkü Aspose.Cells for Python via Java kütüphanesi en iyi kararı veriyor. Aspose.Cells for Python via Java API, elektronik tabloları Word formatına dönüştürmek için destek sağlar. Çalışma kitabını Word'e aktarmak için,[**SaveFormat.DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat) ikinci parametre olarak[**çalışma kitabı.kaydet**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\) ) yöntem. Ayrıca kullanabilirsiniz[**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions) çalışma sayfasını .docx dosyasına dışa aktarmak için ek ayarlar belirtmek üzere sınıf.
