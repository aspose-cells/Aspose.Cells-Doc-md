## **Excel Çalışma Kitabını Word'e Dönüştür**

Excel Çalışma Kitabını Word biçimine dönüştürmek için nasıl bir yol izleneceğinizi merak etmeyin, çünkü Aspose.Cells for Python via Java kütüphanesi en iyi kararı sunar. Aspose.Cells for Python via Java API, elektronik tabloları Word biçimine dönüştürmeyi destekler. Çalışma kitabını Word'e dışa aktarmak için [**Workbook.save**](https://reference.aspose.com/cells/python-java/asposecells.api/workbook#save\(java.lang.String,%20int\)) yönteminin ikinci parametresi olarak [**SaveFormat.DOCX**](https://reference.aspose.com/cells/python-java/asposecells.api/saveformat)'ı geçirin. Ayrıca, çalışma sayfasını .docx dosyasına dışa aktarmak için [**DocxSaveOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/DocxSaveOptions) sınıfını kullanabilirsiniz.
