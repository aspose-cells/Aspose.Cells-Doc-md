## **JSON'u Excel Workbook'e dönüştür**
JSON'u Excel dosyasına nasıl dönüştüreceğinizi merak etmenize gerek yok, çünkü Aspose.Cells for Python via Java kütüphanesi en iyi kararı sunar. Aspose.Cells for Python via Java API, JSON formatını elektronik tablolara dönüştürmeyi destekler. JSON'u Çalışma Kitabına aktarmak için [**JsonLoadOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonLoadOptions) sınıfını kullanabilirsiniz.

Aşağıdaki kod örneği, JSON'u Excel Çalışma Kitabına aktarmayı göstermektedir. Referans için, [kaynak dosyasını](sample.json) kod tarafından oluşturulan dosyaya dönüştürmek için kodu inceleyin.
