## **Excel'i HTML'ye dönüştürürken Yer Paylaşımlı İçeriği Gizle**
Excel dosyanızı HTML'ye kaydettiğinizde, hücre dizileri için farklı çapraz türler belirleyebilirsiniz. Varsayılan olarak, Aspose.Cells, Microsoft Excel'e göre HTML oluşturur, ancak değiştirdiğinizde[HtmlSaveOptions.HtmlCrossStringType](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#HtmlCrossStringType)ile[GEÇMEK_SAKLAMAK_SAĞ](https://reference.aspose.com/cells/python/asposecells.api/htmlcrosstype#CROSS_HIDE_RIGHT)daha sonra hücrenin sağ tarafında, hücre dizisiyle örtüşen veya örtüşen tüm dizileri gizler.

Aşağıdaki örnek kod,[örnek excel dosyası](sampleHidingOverlaidContentWithCrossHideRight.xlsx)ve onu kaydeder[çıktı HTML'si](HTML-outputHidingOverlaidContentWithCrossHideRight.zip)ayarladıktan sonra[HtmlSaveOptions.HtmlCrossStringType](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#HtmlCrossStringType)olarak[GEÇMEK_SAKLAMAK_SAĞ](https://reference.aspose.com/cells/python/asposecells.api/htmlcrosstype#CROSS_HIDE_RIGHT). Ekran görüntüsü nasıl olduğunu açıklıyor[GEÇMEK_SAKLAMAK_SAĞ](https://reference.aspose.com/cells/python/asposecells.api/htmlcrosstype#CROSS_HIDE_RIGHT)çıktı HTML'sini varsayılan çıktıdan etkiler.
