## Aspose.Cells for Python via Java ile Çalışma Sayfasına İkonlar Ekle

Bir Excel dosyasına 'ikonlar' eklemek için [Aspose.Cells for Python via Java](https://products.aspose.com/cells/python-java/) kullanmanız gerekiyorsa, bu belge size yardımcı olabilir.

### Excel'e Ekleme
Ekle simgesine karşılık gelen Excel arayüzü aşağıdaki gibidir:

![](1.png)

- Çalışma sayfasına eklemek istediğiniz simgenin konumunu seçin
- *Ekle*->*Simgeler*ü tıklayın
- Açılan pencerede, yukarıdaki resimde kırmızı dikdörtgen içindeki simgeyi seçin
- Excel dosyasına eklenecek, *Ekle* butonuna sol tıklayın.

Efekt aşağıdaki gibidir:

![](2.png)

Burada, [Aspose.Cells for Python via Java](https://products.aspose.com/cells/python-java/) kullanarak ikonlar eklemenize yardımcı olacak *örnek kod* hazırladık. Ayrıca gerekli [örnek dosya](sample.xlsx) ve bir ikon [kaynak dosyası](icon.zip) da vardır. Excel arayüzünü kullanarak, [kaynak dosyası](icon.zip) ile aynı görüntüleme efektine sahip bir ikon ekledik ve [örnek dosya](sample.xlsx) içerisinde gösterdik.
