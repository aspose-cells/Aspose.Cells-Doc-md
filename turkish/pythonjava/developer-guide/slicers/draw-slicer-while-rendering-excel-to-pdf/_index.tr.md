## **Excel'i PDF'e dönüştürürken Dilimleyici çizin**
Aspose.Cells for Python via Java, dilimleyici ayarlarıyla Excel'in PDF'e aktarılmasını destekler. API bunu varsayılan olarak destekler, Excel dosyasını dilimleyici ile PDF'e dışa aktarmanız yeterlidir, oluşturulan PDF dilimleyicinin uygulandığını gösterecektir. Aşağıdaki kod parçacığı,[örnek excel dosyası](106365058.xlsx)bir dilimleyici içerir ve çalışma kitabını[çıktı PDF dosyası](106365052.xlsx). Aşağıdaki ekran görüntüsü, kaynak Excel dosyasını ve oluşturulan PDF dosyasını karşılaştırır.
