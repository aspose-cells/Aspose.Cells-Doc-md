## **HTML'ye desteklenmeyen kenarlık stiliyle Excel**
Microsoft Excel, Web Tarayıcıları tarafından desteklenmeyen bazı kesikli kenarlık türlerini destekler. Bu tür dosyalar Aspose.Cells kullanılarak HTML'ye dönüştürüldüğünde, bu kenarlıklar kaldırılır. Ancak, Aspose.Cells for Python via Java, benzer kenarlıkların görüntülenmesini destekler.[HtmlSaveOptions.ExportSimilarBorderStyle](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportSimilarBorderStyle)Emlak. değerini ayarlayabilirsiniz[HtmlSaveOptions.ExportSimilarBorderStyle](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportSimilarBorderStyle) mülkiyet**Doğru** desteklenmeyen sınırları dışa aktarmak için.

Aşağıdaki örnek kod,[örnek excel dosyası](sampleExportSimilarBorderStyle.xlsx)aşağıdaki ekran görüntüsünde gösterildiği gibi bazı desteklenmeyen kenarlıklar içerir. Ekran görüntüsü ayrıca etkisini göstermektedir[HtmlSaveOptions.ExportSimilarBorderStyle](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportSimilarBorderStyle)içindeki mülk[çıktı HTML'si](outputExportSimilarBorderStyle.zip).
