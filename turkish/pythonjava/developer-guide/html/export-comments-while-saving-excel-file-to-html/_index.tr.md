## **Excel dosyasını HTML'ye kaydederken Yorumları Dışa Aktar**
Excel HTML'ye dönüştürüldüğünde yorumlar dışa aktarılmaz. Java aracılığıyla Python için Aspose.Cells, Excel'den HTML'ye dönüştürme sırasında yorumları dışa aktarma özelliği sağlar. Bunu başarmak için API,[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments)Emlak. değerini ayarlama[HtmlSaveOptions.IsExportComments](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#IsExportComments) mülkiyet**Doğru** çıktı HTML'sindeki yorumları dışa aktarır.
