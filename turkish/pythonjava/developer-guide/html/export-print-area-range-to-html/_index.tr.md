## **Yazdırma alanı aralığını HTML olarak dışa aktar**

Bunun için API,[HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) Emlak. Bu özellik şu şekilde ayarlanıyor:**Doğru** yalnızca yazdırma alanını dışa aktarır.
