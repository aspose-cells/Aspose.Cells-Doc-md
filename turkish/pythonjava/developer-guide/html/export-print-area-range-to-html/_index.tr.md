## **HTML'e Baskı Alanı Aralığını Dışa Aktar**
Aspose.Cells for Python via Java, yalnızca belirlenmiş hücre aralığı olan baskı alanını tüm sayfaya değil sadece HTML olarak dışa aktarmayı destekler. Bunun için API, [HtmlSaveOptions.ExportPrintAreaOnly](https://reference.aspose.com/cells/python/asposecells.api/htmlsaveoptions#ExportPrintAreaOnly) özelliğini sağlar. Bu özelliği **True** olarak ayarlamak, yalnızca baskı alanını dışa aktarır.
