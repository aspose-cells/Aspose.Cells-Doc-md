 JasperReports v1.0.0.0 için Aspose.Cells'i duyurmaktan mutluluk duyuyoruz!

 Ne var ne yok:

 Statik resimler yerine Microsoft Excel formatında düzenlenebilir grafikler sağlayın.

 l JasperReports için Aspose.Cells'i JasperServer'a entegre etme desteği.

 l POI kitaplığının bağımlılığını ortadan kaldırın.

 l Çubuk grafik, alan grafiği, pasta grafiği, çizgi grafiği vb. dahil olmak üzere çeşitli grafik türlerini destekleyin.
