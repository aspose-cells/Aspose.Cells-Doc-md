{{% alert color="primary" %}} 

JasperReports için Aspose.Cells'i sisteminize yüklemek için sisteminizin aşağıdaki gereksinimleri karşılaması gerekir:

- Java 2 Platform Standard Edition 1.6 veya üstü.
- JasperReports 6.3.0 veya üstü.

{{% /alert %}}
