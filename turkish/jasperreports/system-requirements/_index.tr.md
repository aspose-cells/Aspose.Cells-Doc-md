{{% alert color="primary" %}} 

Aspose.Cells for JasperReports'ü sistemimize yüklemek için sisteminizin aşağıdaki gereksinimleri karşılaması gerekmektedir:

- Java 2 Platform Standard Edition 1.6 veya daha yükseği.
- JasperReports 6.3.0 veya daha yükseği.

{{% /alert %}}
