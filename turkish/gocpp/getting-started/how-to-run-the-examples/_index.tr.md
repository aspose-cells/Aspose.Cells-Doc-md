## **Yazılım Gereksinimleri**

Örnekleri indirip çalıştırmadan önce lütfen aşağıdaki gereksinimleri karşıladığınızdan emin olun.

- Go 1.13 veya daha yeni sürüm
- Metin düzenleyici veya IDE

## **GitHub'dan İçe Aktar**

Aspose.Cells for Go via C++ örneklerinin tamamı [GitHub](https://github.com/aspose-cells/aspose.cells-go-cpp-examples/) üzerinde barındırılmaktadır.

- En sevdiğiniz GitHub istemcinizle depo klonlayabilir veya [buradan](https://github.com/aspose-cells/aspose.cells-go-cpp-examples/archive/master.zip) ZIP dosyasını indirebilirsiniz.
- ZIP dosyasının içeriğini bilgisayarınızdaki herhangi bir klasöre çıkarın. Tüm örnekler **Örnekler** klasöründe bulunmaktadır.
- Çözüm dosyasını Metin düzenleyici veya IDE'de açın ve projeyi derleyin.
- **Örnekler** klasörünün kök klasöründe **Data** klasörü giriş dosyalarını içermektedir. Örnekler projesiyle birlikte **Data** klasörünü indirmeniz gerekmektedir.
- main.go'yu açın, tüm örnekler buradan çağrılır.
- Çalıştırmak istediğiniz örnekleri dosya içinde yorum satırını kaldırın.
- go run .

Lütfen, örneklerin kurulumu veya çalıştırılmasıyla ilgili herhangi bir sorununuz varsa, Forumlarımızı kullanarak bizimle iletişime geçmekten çekinmeyin.

## **Katkıda bulun**

Bir örnek eklemek veya geliştirmek isterseniz, projeye katkıda bulunmanızı teşvik ederiz. Bu depoda bulunan tüm örnekler ve vitrin projeleri açık kaynaklıdır ve kendi uygulamalarınızda özgürce kullanılabilir.

Katkıda bulunmak için reposu klonlayabilir, kaynak kodunu düzenleyebilir ve bir pull isteği oluşturabilirsiniz. Değişiklikleri gözden geçireceğiz ve faydalı bulunursa reposuna dahil edeceğiz.
