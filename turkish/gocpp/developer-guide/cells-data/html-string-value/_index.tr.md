{{% alert color="primary" %}}

Aspose.Cells, Microsoft Excel odaklı HTML'yi XLS/XLSX formatına dönüştürmeyi destekler. Bu, Microsoft Excel tarafından oluşturulan HTML'nin Aspose.Cells kullanılarak tekrar XLS/XLSX formatına dönüştürülebileceği anlamına gelir.

Benzer şekilde, eğer basit bir HTML varsa, Aspose.Cells onu HTML Zengin Metne dönüştürebilir. [**Cell::GetHtmlString**](https://reference.aspose.com/cells/go-cpp/cell/gethtmlstring/) yöntemi, böyle basit bir HTML alabilir ve biçimlendirilmiş hücre metnine dönüştürebilir.
