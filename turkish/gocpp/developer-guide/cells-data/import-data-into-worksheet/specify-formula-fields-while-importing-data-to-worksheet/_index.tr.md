## **Olası Kullanım Senaryoları**

Çalışma sayfanıza veri aktardığınızda formül alanlarını belirleyebilirsiniz. [**ImportTableOptions.GetIsFormulas()**](https://reference.aspose.com/cells/go-cpp/importtableoptions/getisformulas/). Bu özellik, formül alanı olduğunda **true** değeri alır. Örneğin, üçüncü alan bir formül alanı ise, dizideki üçüncü değer **true** olacaktır.
