 Microsoft Excel kullanarak, VBA projenizin imzalanıp imzalanmadığını **Araçlar > Dijital İmzalar...** menüsü aracılığıyla kontrol edebilirsiniz. Benzer şekilde, Aspose.Cells [**Workbook.VbaProject.IsSigned**](https://reference.aspose.com/cells/go-cpp/vbaproject/issigned/) kullanılarak programatik olarak da kontrol edebilirsiniz.
