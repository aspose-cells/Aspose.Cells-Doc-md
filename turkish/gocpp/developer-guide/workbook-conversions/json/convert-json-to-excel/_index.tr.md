## **JSON'u Excel Workbook'e dönüştür** 
JSON'yu bir Excel dosyasına nasıl dönüştüreceğinizi merak etmeye gerek yok, çünkü Aspose.Cells for C++ kütüphanesinin en iyi çözümü var. Aspose.Cells API, JSON formatını elektronik tablolarına dönüştürmek için destek sağlar. JSON'u Çalışma Kitabına içe aktarmak için ek ayarları belirtmek üzere [JsonLoadOptions](https://reference.aspose.com/cells/go-cpp/jsonloadoptions/) sınıfını kullanabilirsiniz. 

Aşağıdaki kod örneği, JSON'u Excel Çalışma Kitabına içe aktarmayı göstermektedir. Lütfen başvuru için kodu inceleyin [kaynak dosyası](örnek.json) kod tarafından oluşturulan xlsx dosyası.
