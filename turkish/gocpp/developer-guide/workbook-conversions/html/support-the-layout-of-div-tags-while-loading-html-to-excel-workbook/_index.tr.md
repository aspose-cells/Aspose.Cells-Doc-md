{{% alert color="primary" %}} 

Normalde, HTML'yi Excel çalışma kitabına yüklerken DIV etiketlerinin düzeni göz ardı edilir. Ancak, DIV etiketlerinin düzenini korumak istiyorsanız, [GetSupportDivTag()](https://reference.aspose.com/cells/go-cpp/htmlloadoptions/getsupportdivtag/) özelliğini **true** olarak ayarlayın. Bu özelliğin varsayılan değeri **false**.

{{% /alert %}}
