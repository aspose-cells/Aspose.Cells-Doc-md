{{% alert color="primary" %}}

Eğer grafik efsanesinin giriş doldurmasını hiçbiri olarak ayarlamak istiyorsanız, lütfen [**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/go-cpp/legendentry/istextnofill/)'yi **true** olarak ayarlayın.

{{% /alert %}}
