## **Olası Kullanım Senaryoları**

Bir Excel dosyasının **Sürüm numarasını** sağ tıklayıp özellikler > ayrıntılar menüsünden değiştirebilirsiniz ve ardından **Sürüm numarası** alanını düzenleyin. Programatik olarak değiştirmek için Lütfen [**BuiltInDocumentPropertyCollection.GetDocumentVersion()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getdocumentversion/) özelliğini kullanın.
