## **Olası Kullanım Senaryoları**

 Bir Excel dosyasının dilini değiştirmek için dosyaya sağ tıklayın ve özellikler > ayrıntılar menüsünü seçin, sonra Dil alanını düzenleyin. Programatik olarak değiştirmek için [**BuiltInDocumentPropertyCollection.GetLanguage()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getlanguage/) özelliğini kullanın.
