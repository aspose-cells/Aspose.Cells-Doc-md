### **Aspose.Cells for SharePoint Belgelerine Hoş Geldiniz!**
Aspose.Cells for SharePoint, Microsoft SharePoint uygulamaları içinde MS Excel belgelerinin dönüştürülmesini mümkün kılan bir elektronik tablo dönüştürme çözümüdür.

{{% /alert %}} 

Şu belge formatlarını dönüştürebilirsiniz:

- PDF - Adobe Taşınabilir Belge
- XLS - MS Excel 97~2003
- XLSX - Office Open XML
- XLSM - Makrolar etkin Office Açık XML
- TXT - Düz Metin
- XML - Microsoft Excel Elektronik Tablo Dili
- ODS - OpenDocument Elektronik Tablo
- CSV - Virgülle Ayrılmış Değerler
- XLSB - Excel Binary 2007
- MHtml - Çok Amaçlı İnternet Posta Eklentisi HTML
- XPS - XML Paper Specification
- TIFF - Tagged Image File Format
- SVG - Scalable Vector Graphics
- DIF - Drawing Interchange Format

Aspose.Cells for SharePoint, aşağıdaki ürünlerle kullanılacak şekilde tasarlanmıştır:

- Windows SharePoint Services 3.0 (WSS)
- Microsoft Office SharePoint Server 2007 (MOSS) Standard
- Microsoft Office SharePoint Server 2007 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2010 (MOSS) Standard
- Microsoft Office SharePoint Server 2010 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2013 (MOSS) Standard
- Microsoft Office SharePoint Server 2013 (MOSS) Enterprise
- Microsoft Office SharePoint Server 2016 (MOSS) Standard
- Microsoft Office SharePoint Server 2016 (MOSS) Enterprise

Yukarıdaki ürünler dışında ek sistem gereksinimleri bulunmamaktadır.

Bu belge, Aspose.Cells for SharePoint'un [özelliklerini](/cells/tr/sharepoint/features/), [kurulumunu](/cells/tr/sharepoint/install-aspose-cells-for-sharepoint/), [değerlendirmesini](/cells/tr/sharepoint/evaluate-aspose-cells/), [lisanslamasını](/cells/tr/sharepoint/license-aspose-cells-for-sharepoint/), yaygın kullanım durumlarını ve ayarlarını tanımlar.
