## **Rapor filtre sayfalarını göster seçeneği**
Excel, pivot tablolar oluşturma, rapor filtreleri ekleme ve "Rapor Filtre Sayfalarını Göster" seçeneğini etkinleştirme özelliklerini destekler. Aspose.Cells for Node.js via C++ bu özelliği de destekler ve oluşturulan pivot tabloda "Rapor Filtre Sayfalarını Göster" seçeneğini etkinleştirir. Aşağıda, Excel'de "Rapor Filtre Sayfalarını Göster" seçeneğinin gösterildiği ekran bulunmaktadır.
