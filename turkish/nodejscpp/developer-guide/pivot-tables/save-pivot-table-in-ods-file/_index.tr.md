Aspose.Cells for Node.js via C++, pivot tabloyu ODS dosyaları olarak kaydetme yeteneği sağlar. Bunun için, mevcut pivot tabloyu içeren veya yeni bir pivot tablo oluşturan bir çalışma kitabını ODS biçiminde kaydedersiniz. Kaydetmeden önce [**PivotTable.calculateData**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#calculatedata--) 'nın çağrıldığından emin olun, böylece çıktı ODS dosyasına pivot tablo doğru şekilde render edilir.
