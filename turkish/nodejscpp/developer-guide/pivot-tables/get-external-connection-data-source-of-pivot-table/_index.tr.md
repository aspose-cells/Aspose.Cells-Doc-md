## **Pivot Tablosunun Dış Bağlantı Veri Kaynağını Alma Yöntemi**

Aspose.Cells for Node.js via C++, pivot tablonun dış bağlantı veri kaynağını elde etme yeteneği sağlar. Bunu yapmak için, API [**PivotTable**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/) sınıfının [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) yöntemini sağlar. [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) yöntemi, [**ExternalConnection**](https://reference.aspose.com/cells/nodejs-cpp/externalconnection/) nesnesini döndürür. Aşağıdaki kod parçacığı, [**getExternalConnectionDataSource**](https://reference.aspose.com/cells/nodejs-cpp/pivottable/#getExternalConnectionDataSource--) yönteminin pivot tablonun dış bağlantı veri kaynağını almak için kullanılmasını göstermektedir.
