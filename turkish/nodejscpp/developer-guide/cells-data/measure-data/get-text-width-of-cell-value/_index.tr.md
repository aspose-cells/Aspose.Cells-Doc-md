## **Hücre Değeri Metnin Genişliğini Alma**

Aspose.Cells for Node.js via C++, geliştiricilerin hücrenin değerinin metin genişliğini almalarını sağlayan [**CellsHelper.getTextWidth(string, Font, number)**](https://reference.aspose.com/cells/nodejs-cpp/cellshelper/#getTextWidth-string-font-number-) metodunu sağlar.
