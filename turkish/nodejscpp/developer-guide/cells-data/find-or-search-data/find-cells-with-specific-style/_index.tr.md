{{% alert color="primary" %}}

Bazen belirli bir stili uygulanmış hücreleri bulmanız gerekir. Ortak stile sahip tüm hücreleri bulmak için Aspose.Cells for Node.js via C++ kullanabilirsiniz. Aspose.Cells, aramak istediğiniz stili belirtmek için kullanabileceğiniz [**FindOptions.setStyle(Style)**](https://reference.aspose.com/cells/nodejs-cpp/findoptions/#setStyle-style-) yöntemini sağlar.

{{% /alert %}}

Bu örnek kod, A1 hücresiyle aynı stile sahip tüm hücreleri bulur. Kod çalıştırıldıktan sonra, A1 ile aynı stile sahip tüm hücreler "Bulundu" metnini içerir.
