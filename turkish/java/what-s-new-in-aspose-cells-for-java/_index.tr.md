Bu sayfa, son sürümlerde tanıtılan en ilginç yeni Aspose.Cells özelliklerini açıklar.

## Aspose.Cells for Java 25.4
* HTML'e dışa aktarmak için sayfa setini belirtmeyi destekler
* Bir çalışma sayfasının ızgara çizgisi rengini alma ve ayarlamayı destekler
* Düzenlenebilir grafiklerin DOCX dosyalarına dışa aktarılmasını destekler
* Hassasiyet etiketlerinin okunmasını ve yazılmasını destekler
* Daha fazla stil özelliğini Numbers'a aktarmayı destekler
* Markdown formatında şablon dosyalarını içe aktarmayı destekler
* Formüllerde herhangi bir özel fonksiyon kullanılıp kullanılmadığını tespit etmeyi destekler (`Cell` ve `Workbook` içindeki)

## Aspose.Cells for Java 25.3
* Bir hücrenin görüntü stilini hücre nesnesini almadan elde etmeyi destekler
* Çalışma kitabındaki kullanılan fontların ikili içeriğini alma desteği
* Excel dosyasını çalışma kitabı yapısına JSON'a dönüştürmeyi destekler
* Hücre değeri, biçimlendirme, formül ve yorumu Numbers dosyasına dışa aktarmayı destekler
* HTML'yi kaydederken satır içi stil/hazırlığı HTML öğeleri içine yerleştirmeye destekler
* Oluşturulan HTML'de seçilen sayfa sekmesini vurgulama desteği
* GridJs için daha fazla kısa yol tuşunu destekler
* Excel'den kopyalanan içeriği, GridJs için çoklu hedef alanlara yapıştırmayı destekler

## Aspose.Cells for Java 25.2
* LightCells ile kaydederken hücreye zengin metin ayarlamayı destekler
* Farklı format dosyalarını okuma desteği
* dBase (dbf) format dosyası okuma ve yazma desteği
* WPS uyumlu gömülü resmi destekle
* GridJs'de salt okunur alan ayarlamayı destekle
* MS Excel'den GridJS'e sayı biçimi ayarını kopyalamayı destekle
* Kullanıcının kolaylığı için GridJS'de otomatik işlemleri geliştirme, örneğin boş satır/sütun uzatma, sayfa kaydırma, ...

## Aspose.Cells for Java 25.1
* Kullanıcının özel motorundaki dinamik dizi formülü için hesaplamayı atlamayı destekle, özel fonksiyon kullanımı
* Kullanıcının özel motorundaki dış tablo referansını ifade etmesini almayı destekler
* Otomatik şekiller ve serbest şekilleri ODS dosyasına kaydetmeyi destekler
* Tablo içinde formlar ayarlarını okuma ve yazma desteği
* Font'daki Format 2 alt tablosunu destekler
* Satır verisi listesinin hücrelere içe aktarımını basitleştirir
* GridJs için hücre ekleme/silme/güncelleme olaylarını daha fazla destekle
* GridJs için yerelleştirilmiş ondalık ayırıcı desteği
* GridJs için filtre listesi arama desteği
* Bir sayfada birden çok örneği destekle

## Aspose.Cells for Java 24.12
* Hücre aralığı ekleme/silme sırasında formül değişikliklerini bildirme desteği
* BYROW ve BYCOL lambda fonksiyonlarını destekle
* LineGradientBrush, PathGradientBrush, HatchBrush ve TextureBrush'ı pcl'ye render etmeyi destekler
* HTML kaydederken inline stil veya formatlamayı css yerine yerleştirerek destekler
* Yazıya, hücreye renk gibi veri sıralayıcı anahtar olarak ekleme desteği
* PowerQueryFormulas'da sıra düzeni güncelleme ve grup eklemeyi destekler
* PivotField filtrelerinin çoklu filtreler, gün boyunca filtreleme gibi iyileştirmeleri
* GridJs'de satır/sütun eklemede veya silmede formül referansını yenilemeyi destekler
* GridJs için güncelleme izleme desteği
* GridJs'de sütunlar arasında merkezi hizalama desteği

## Aspose.Cells for Java 24.11
* xlsx'den pcl'ye dönüştürme desteği
* PivotTable için harici kaynağın kaynağını almak için destek

* Hatalar, varlıklar gibi zengin değerleri içe aktarma/dışa aktarma desteği
* Excel'deki gibi İmza Satırı ekleme desteği
* Oluşturulan SVG'nin farklı framework'ler için aynı davranışı göstermesini sağlamak için geliştirme
* Serbest biçimli şekiller ekleme iyileştirmeleri
* GridJs için farklı veri türleri (tarih, saat, para birimi...) için sayı biçimi türünü ayarlama desteği
* GridJs için satır/kolon stiline sınır çizgileri çizme desteği
* GridJs'nin görüntüleme ve düzenleme iyileştirmeleri
