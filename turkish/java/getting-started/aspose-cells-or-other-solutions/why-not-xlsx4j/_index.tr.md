## **xlsx4j**
xlsx4j, Microsoft Open XML Excel xlsx dosyalarını oluşturmak ve manipüle etmek için açık kaynak (ASLv2) Java kütüphanesidir, Microsoft Open XML Word docx dosyalarını manipüle etmek için docx4j gibi.
Bu, Java için Microsoft’un OpenXML SDK'sına benzer. docx4j, iç bellek nesnesi temsilini oluşturmak için JAXB kullanır.
JAXB'yı ve Open XML dosya yapısını anlamak için zaman ayırmanız gerekecek.

[docx4java.org](https://www.docx4java.org/trac/docx4j) adresinde daha fazla bilgi edinin
## **Aspose.Cells for Java**
Aspose.Cells for Java, Java geliştiricilerinin, Microsoft Excel®'e veya Microsoft Office Automation'a güvenmeksizin kendi Java uygulamalarına Excel® elektronik tablolarını (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF ve görüntü dosya biçimlerini okuma, yazma ve işleme yeteneğini entegre etmelerine olanak tanır.

Aspose.Cells for Java, diğer satıcıların basit veri dışa aktarma yeteneklerinin ötesinde birçok fonksiyon sunan olgun, ölçeklenebilir ve özellik zengin bir bileşen olup, Aspose.Cells for Java ile geliştiriciler veri dışa aktarabilir, elektronik tablolara en ince düzeyde biçimlendirme yapabilir, görüntüleri içe aktarabilir, çizelge oluşturabilir, karmaşık formülleri uygulayabilir ve hesaplayabilir, Excel® verilerini akıtabilir, çeşitli biçimlerde kaydedebilir ve bunun gibi birçok işlemi Microsoft Excel® veya Microsoft Office Automation gerektirmeksizin gerçekleştirebilir.

[aspose.com](https://products.aspose.com/cells/java/) adresinde daha fazla bilgi edinin

## **Neden xlsx4j değil**
xlsx4j sadece Microsoft Open XML Excel xlsx dosyalarını ayrıştırabilir ve kaydedebilir, Aspose.Cells for Java tüm Excel dosya biçimlerini (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS) işleyebilir ve çeşitli dosya biçimi dönüşümlerini destekler.

## **Aspose.Cells Java for xlsx4j**
Proje Aspose.Cells for xlsx4j, Aspose.Cells Java API'leri kullanarak farklı görevlerin xlsx4j ile karşılaştırıldığında nasıl gerçekleştirilebileceğini gösterir. Proje ayrıca elektronik tablolarla ilgili Aspose.Cells API'lerinde sadece Aspose.Cells API'lerinde bulunan ancak xlsx4j'de bulunmayan özellikleri kapsar.

Bu proje, geliştiricilerin xlsx4j'yi Aspose.Cells ile karşılaştırmak veya xlsx4j'den Aspose.Cells'e geçiş yapmak isteyen geliştiriciler için faydalıdır.
## **Sistem Gereksinimleri ve Desteklenen Platformlar**
### **Sistem Gereksinimleri**
Aspose.Cells Java için xlsx4j'nin çalıştırılması için aşağıdaki sistem gereksinimleri bulunmaktadır:

- Java 1.4 veya üstü yüklü.
- Aspose.Cells bileşeni indirildi.
- docx4j kütüphaneleri indirildi.
### **Desteklenen Platformlar**
Aşağıdaki platformlar desteklenmektedir:

- Aspose.Cells 8.3.0 veya üstü.
- docx4j 3.1.0 veya üstü kütüphaneler.
- Java IDE (Eclipse, NetBeans, IntelliJ, ...).
## **İndirme**
Çalışan örnek sürümlerini aşağıda belirtilen tüm sosyal kodlama sitelerinden indirebilirsiniz:

### **GitHub**
- [Aspose.Cells Java for Xlsx4j - v 1.0](https://github.com/aspose-cells/Aspose.Cells-for-Java/releases/tag/Aspose.Cells_Java_for_Xlsx4j-v1.0.0)

**Aspose.Cells for Java API**'yi yapılandırmak için Aspose deposu URL'sini ve Aspose.Cells for Java API maven bağımlılığını **pom.xml** dosyanıza aşağıdaki gibi ekleyin:
### **Aspose Maven Deposu**
{{< highlight java >}}

 <repositories>

    <repository>

        <id>aspose-maven-repository</id>

        <name>Aspose Maven Repository</name>

        <url>http://repository.aspose.com/repo/</url>

    </repository>

</repositories>

{{< /highlight >}}
### **Aspose.Cells for Java Maven Bağımlılığı**
{{< highlight java >}}

 <dependencies>

    <dependency>

        <groupId>com.aspose</groupId>

        <artifactId>aspose-cells</artifactId>

        <version>8.6.0</version>

    </dependency>

</dependencies>

{{< /highlight >}}

**Not:** Lütfen Aspose API'nin mevcut en son sürümünü kullanın.
