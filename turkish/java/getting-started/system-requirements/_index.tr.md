{{% alert color="primary" %}} 

Aspose.Cells for Java, Microsoft Excel'in kurulu olmasını gerektirmez, çünkü Aspose.Cells'in kendisi bir Excel elektronik tablo oluşturma, dönüştürme ve işleme motorudur. Aspose.Cells tarafından üretilen Excel belgelerini görüntülemek için en az Microsoft Excel Viewer kurulu olmalıdır.

{{% /alert %}} 
## **Desteklenen İşletim Sistemleri**
Aspose.Cells for Java, bunlarla sınırlı olmamak üzere, Java çalışma zamanını çalıştıran tüm işletim sistemlerini destekler:
### **Windows**
- Microsoft Windows 2000 (x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows Sunucu 2003 ( x64, x86)
- Microsoft Windows Sunucu 2008 ( x64, x86)
- Microsoft Windows Sunucu 2012 ( x64, x86)
- Microsoft Windows Vista ( x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
### **linux**
- Linux (Ubuntu, openSUSE, CentOS ve diğerleri)
### **Mac**
- Mac OS X vb.
## **Desteklenen Java Versiyonlar**
Aspose.Cells for Java, aşağıdaki Java sürümlerini destekler:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- ya da üzerinde
