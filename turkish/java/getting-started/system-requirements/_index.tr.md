{{% alert color="primary" %}} 

Aspose.Cells for JavaAspose.Cells'in kendisi bir Excel elektronik tablo oluşturma, dönüştürme ve görüntüleme motoru olduğu için Microsoft Excel yüklü olmasını gerektirmez. Aspose.Cells tarafından üretilen Excel belgelerini görüntülemek için en azından Microsoft Excel Görüntüleyici programının yüklü olması gerekmektedir.

{{% /alert %}} 
## **Desteklenen İşletim Sistemleri**
Aspose.Cells for Java, Java çalışma zamanını içeren herhangi bir işletim sistemini destekler, ancak bununla sınırlı değildir:
### **Windows**
- Microsoft Windows 2000 (x64, x86)
- Microsoft Windows XP (x64, x86)
- Microsoft Windows Server 2003 (x64, x86)
- Microsoft Windows Server 2008 (x64, x86)
- Microsoft Windows Server 2012 (x64, x86)
- Microsoft Windows Vista (x64, x86)
- Microsoft Windows 7 ( x64, x86)
- Microsoft Windows 8 ( x64, x86)
- Microsoft Windows 10 ( x64, x86)
### **Linux**
- Linux (Ubuntu, openSUSE, CentOS and others)
### **Mac**
- Mac OS X etc.
## **Supported Java Versions**
Aspose.Cells for Java desteklediği Java sürümleri:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- veya üstü
