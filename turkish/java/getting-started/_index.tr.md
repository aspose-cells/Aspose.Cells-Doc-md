## **Sistem Gereksinimleri**

Aspose.Cells for Java desteklediği Java sürümleri:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- veya üstü

[Daha fazla detay](/cells/tr/java/system-requirements/)

## **Kurulum**

### **Aspose Cells for Java'yı Maven Deposundan Yükle**

Maven, Aspose.Cells for Java'yi indirmenin en kolay yoludur. 

1. Maven pom.xml dosyanızda Aspose Maven Deposu yapılandırmasını/yerini belirtin. 
2. pom.xml dosyanızda Aspose.Cells for Java API bağımlılığını tanımlayın. 

[Detaylar için daha fazla adım](/cells/tr/java/installation/)

### **Aspose Cells for Java'yı manuel olarak yükle**

1. [Aspose-cells jar dosyalarını](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/) bir klasöre indirin.
1. Klasöre girin, Eclipse veya Intellij ile JAR paketlerini el ile kurun.
