## **Destek**
- Kullanıcılar veya Q.A ekibi tarafından bilinen / rapor edilen sorunları uygulamada görmek istiyorsanız.
- Ya da uygulamada bulduğunuz herhangi bir sorunu bildirmek isterseniz
- Herhangi bir iyileştirme öneriniz varsa veya herhangi bir özellik isteği yapmak istiyorsanız

**Lütfen aşağıdaki projenin herhangi bir sorun takipçisini kullanın:**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **Genişletme ve Katkı Sağlama**
Aspose.Cells Java for Struts 1.3 açık kaynaklıdır ve kaynak kodu aşağıda listelenen büyük sosyal kodlama web sitelerinde mevcuttur. Geliştiriciler, kaynak kodunu indirerek önerilerde bulunabilir veya yeni özellikler ekleyerek mevcut olanları geliştirebilir, böylece diğerleri de bundan faydalanabilir.
