## **Destek**
- Bilinen / bildirilen sorunları (kullanıcılar veya QA ekibi tarafından) uygulamada görmek isterseniz.
- Veya uygulamada bulduğunuz herhangi bir sorunu bildirmek istiyorsunuz
- Herhangi bir iyileştirme öneriniz varsa veya herhangi bir özellik isteğinde bulunmak isterseniz

**Lütfen aşağıdaki projenin sorun izleyicisinden herhangi birini kullanın:**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **Genişletin ve Katkıda Bulunun**
Struts 1.3 için Aspose.cells Java açık kaynaktır ve kaynak kodu aşağıda listelenen başlıca sosyal kodlama web sitelerinde mevcuttur. Geliştiricilerin kaynak kodunu indirmeleri ve yeni özellikler önererek veya ekleyerek veya mevcut olanları geliştirerek katkıda bulunmaları teşvik edilir, böylece diğerleri de bundan faydalanabilir.
