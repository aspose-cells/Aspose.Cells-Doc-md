## **JPype**

JPype, python programlarının java sınıf kütüphanelerine tam erişimini sağlama çabasıdır. Bu, Jython/JPython'in yaptığı gibi Python'u yeniden uygulamayla değil, her iki Sanal Makinede de doğal düzeyde arayüzle gerçekleştirilir.

[readthedocs.org](https://jpype.readthedocs.io/en/latest/userguide.html) adresinde daha fazlasını okuyun

## **Aspose.Cells for Java**

Aspose.Cells for Java, Java için gelişmiş bir sınıf kitaplığıdır ve Java uygulamalarınızın içinde doğrudan belge işleme görevlerinin büyük bir yelpazesini gerçekleştirmenizi sağlar.

Aspose.Cells for Java, Excel (XLS, XLSX, XLSB, XLSM, XLT, XLTX, XLTM), CSV (Virgülle Ayrılmış Değer), TSV (Sekmeyle Ayrılmış Değerler), TabDelimited (Sekmeyle ayrılmış metin dosyası), Html, MHtml, ODS (Açık Belge Elektronik Tablosu), SpreadsheetML ve Numbers (belge Apple'ın "Numbers" uygulaması tarafından oluşturuldu) dosyalarının işlenmesini destekler. Aspose.Cells ile Microsoft Excel kullanmadan belge üretebilir, değiştirebilir ve dönüştürebilirsiniz.

## **Aspose.Cells Java for Jython**

Aspose.Cells Java for Python, Python'da Aspose.Cells for Java API kullanımı örneklerini gösteren/belirten bir projedir.

## **Sistem Gereksinimleri**

**Aspose.Cells Java for Python kullanmak için sistem gereksinimleri aşağıdaki gibidir:**

- Jpype
- Aspose.Cells bileşeni indirildi.

## **Desteklenen Platformlar**

**Aşağıdakiler desteklenen platformlardır:**

- Python 2.7.x.x
