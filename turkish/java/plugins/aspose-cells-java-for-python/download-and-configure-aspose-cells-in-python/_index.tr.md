## **Kaynak kodu nasıl yapılandırılır**
Kullanırken kaynak kodunu açmak ve genişletmek için lütfen şu basit adımları izleyin:

 \1. Şuradan yalnızca Python 2.7.xx'i kurun:<https://www.python.org/downloads/>
 \2. JPype'ı şuradan yükleyin:<https://sourceforge.net/projects/jpype/files/JPype/>
 \3. Java JDK'yı şuradan yükleyin:<https://www.oracle.com/java/technologies/downloads/>
 \4. Python projelerinizde kullanmak istediğiniz Aspose.Cells for Java API'leri şu adresten indirin:<https://downloads.aspose.com/cells/java> ve indirilen API'lerin .jar dosyalarını %Python%/lib klasörüne kopyalayın.
 \5. Klon[Aspose.Cells Java için Python Örnekler](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python) github'dan.
