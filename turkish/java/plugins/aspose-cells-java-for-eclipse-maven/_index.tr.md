## **Giriş**
### **Eclipse IDE**
Eclipse IDE, ünlü bir Java Entegre Geliştirme Ortamı (IDE) 'dir. Eclipse Open Source projesinin en tanınmış ürünüdür. Günümüzde Java için önde gelen geliştirme ortamı olup pazar payı yaklaşık olarak %60'tır.

Eclipse IDE, ek yazılım bileşenleri ile genişletilebilir. Eclipse, bu yazılım bileşenlerine Plug-in der. Birçok Open Source projesi ve şirket, Eclipse IDE'yi genişletmiş veya Eclipse çerçevesi üzerinde bağımsız uygulamalar (Eclipse RCP) oluşturmuştur.
### **Aspose.Cells for Java**
[Aspose.Cells for Java], Java geliştiricilerin kendi Java uygulamalarına, Excel® elektronik tablolarını (XLS, XLSX, XLSM, XLSB, XLTX, SpreadsheetML, CSV, ODS), HTML, MHTML, PDF ve görüntü dosyası formatlarını okuma, yazma ve manipüle etme yeteneğini eklemelerini sağlayan ödüllü bir Excel Elektronik Tablo bileşenidir ve Microsoft Excel® veya Microsoft Office Automation'a güvenmek zorunda kalmazlar.

[Aspose.Cells for Java], olgun, ölçeklenebilir ve özellik açısından zengin bir bileşen olup diğer sağlayıcıların basit veri dışa aktarma yeteneklerinin çok ötesinde birçok işlev sunar. Aspose.Cells for Java ile yapılandırılan projeler veri aktarabilir, elektronik tabloları en ayrıntılı düzeye kadar biçimlendirebilir, görüntüleri içe aktarabilir, grafikler oluşturabilir, karmaşık formülleri uygulayabilir ve hesaplayabilir, Excel® verisini akabilir, çeşitli formatlarda kaydedebilir ve çok daha fazlasını yapabilir - ve tüm bunları Microsoft Excel® veya Microsoft Office Automation'a ihtiyaç duymadan yapabilirler.

**       ****Aspose.Cells Maven Projesi (sihirbazı)**

- Bu Yeni Proje sihirbazı, geliştiricilere [Aspose.Cells for Java](http://goo.gl/gtvhlQ) 'yi kullanmak için **New -> Project -> Maven-> Aspose.Cells Maven Project** yolundan Maven projesi oluşturma imkanı sunar.

- Oluşturulan proje, [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) için her zaman en son kullanılabilir sürüm Maven Bağımlılığını içerecektir.
- Sihirbaz adımları ayrıca [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) 'yi kullanmak için [Kod Örnekleri](https://goo.gl/uxznrb) indirme seçeneği sunar.

**       ****Aspose.Cells Kod Örneği (sihirbazı)**

- Bu Yeni Dosya sihirbazı, [Kod Örnekleri](https://goo.gl/uxznrb) indirdiğiniz klasörü projenize kullanabilmek için [Aspose.Cells for Java](http://goo.gl/gtvhlQ) işaretli yerden **Yeni -> Diğer -> Java -> Aspose.Cells Kod Örneği**'ne kopyalamanıza olanak sağlar. 
- Mevcut örnekler, kullanıcıların kategorik olarak seçebilecekleri ağaç formatında görüntülenir.
- Seçilen kategori içindeki tüm örnekler, projenin "**com.aspose.cells.examples**" paket klasörüne ve gerekli kaynakları içeren "**src/main/resources**" klasörüne kopyalanır.
- [Aspose.Cells for Java API](http://goo.gl/gtvhlQ) 'nin [Kod Örnekleri](https://goo.gl/uxznrb) , API'nin çeşitli yönlerini göstermek içindir.
- Sihirbaz ayrıca [Aspose.Cells for Java örnek deposu](https://goo.gl/uxznrb) 'ndan indirilebilecek yeni [Kod Örnekleri](https://goo.gl/uxznrb) için güncellemeleri arar ve getirir.
## **Sistem Gereksinimleri ve Desteklenen Platformlar**

- **Sistem Belleği:** 2 GB veya daha fazla (Tavsiye Edilir)
- **OS:** Java VM'yi destekleyen herhangi bir işletim sistemi
- **İnternet Bağlantısı:** 2 MB veya daha hızlı (Tavsiye edilen)
### **Desteklenen Platformlar**
- Önerilen: Eclipse Mars.1 (4.5.1)
- Eclipse Juno ya da daha yeni.

Bu Eklenti, Aspose.Cells'in değerlendirme sürümünü kullanmaktadır. Değerlendirmenizden memnun kalırsanız, lisansı [Aspose websitesinden](https://purchase.aspose.com/buy) satın alabilirsiniz.

#### **Aspose.Cells Java for Eclipse (Maven) kaynak kodu nasıl yapılandırılır**
Aşağıdaki basit adımlar, Eclipse IDE'de **"Aspose.Cells Java for Eclipse (Maven)"** eklenti kaynak kodunun başarılı bir şekilde yapılandırılmasını sağlar

1. Kaynak kodu İndir / Klonla

1. İndirdiğiniz en son proje kaynağına göz atın
1. İçe aktarmak istediğiniz Eclipse Projesini seçin
1. **Bitir**'e tıklayın
1. Aspose.Cells Java for Eclipse eklenti kodu artık geliştirmeye hazır
