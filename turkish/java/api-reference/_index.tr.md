{{% alert color="primary" %}} 

Aspose.Cells for Java, uygulamalarınızın geniş bir elektronik tablo işleme görevlerini yerine getirmesini sağlayan bir sınıf kütüphanesidir. Aspose.Cells for Java, popüler elektronik tablo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) dosya biçimlerini destekler ve Excel dosyalarının PDF, XPS, HTML, MHTML, Düz Metin, TIFF, JPG, PNG, BMP ve SVG formatlarına aktarılmasına izin verir.

En son API Referansına [Aspose.Cells for Java API Referansı](https://reference.aspose.com/cells/java) adresinden ulaşabilirsiniz.

{{% /alert %}}
