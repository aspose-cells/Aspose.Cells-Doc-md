 Ne var ne yok?

- Yay, Oval, Dikdörtgen, Yorum, Metin Kutusu gibi şekiller için zengin metin eklemeyi destekler
 Şekil için her tür yerleştirmeyi destekler
 Şablondan tüm şekilleri okumayı destekler
 SmartMarker'da dizi ayarlamayı destekler

 Neler Geliştirildi?

- Şeklin konumunu ve boyutunu değiştirmek için yeni yöntemler eklendi ve bazı eski yöntemler kullanımdan kaldırıldı
 Yeni API için bazı demo kodları ve şablonlar ekleyin
 Kullanımdan kaldırılan API çağrısını demolarda yeni API ile değiştirin

 Neler Düzeltildi?

- Formül hesaplamadaki bazı hatalar düzeltildi
 Excel2007 XLSX biçimini dışa aktarma hatası düzeltildi
 Kopyalama stilindeki bir hata düzeltildi
 Formül kopyalama hatası düzeltildi
 Görüntü boyutunu kaydetme hatası düzeltildi

 Yorumlar
