Biz
 Aspose.Cells for Java v7.1.2'yi duyurmaktan mutluluk duyuyoruz!

 Yeni özellikler

 – POJO'yu içe aktarmak için Cells.importCustomObjects() yöntemini destekleyin

 Geliştirmeler

- autoFitColumns() yöntemleri için iyileştirmeler yapıldı
- Kılavuz çizgileri açık olan Sayfadan Görüntüye özelliği için performans

 İstisnalar

- Cell.getDisplayStyle() yöntemi çağrılırken NullPointerException oluştu

 Hatalar

- Dağılım tablosu için maksimum değer düzgün bir şekilde kuşatılamaz
- Şekillerdeki metin, oluşturulan PDF dosyasında kayboldu veya şekiller aralığının dışına taşındı
- Grafikte Tarih Etiketleri için seri adı gösterilmedi
