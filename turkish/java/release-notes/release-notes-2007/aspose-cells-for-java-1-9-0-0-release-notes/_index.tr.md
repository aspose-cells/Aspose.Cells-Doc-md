 Bu sayfa için sürüm notları içerir[Aspose.Cells for Java 1.9.0.0](https://downloads.aspose.com/cells/java/new-releases/aspose.cells-for-java-1.9.0.0/)

 Sayın müşteriler,

 Aspose.Cells for Java V1.9.0 sürümünü yayınladık!

 Ne var ne yok?

- Aspose.Pdf xml dosyasına aktarmayı destekler
 Çalışma sayfasıyla başa çıkarken yorumları ve adlandırılmış aralıkları kopyalamayı destekler
 Formülleri hesaplamak için ICustomFunction arayüzünü destekler
Bir hücrenin predentlerini ve bağımlılarını izlemeyi destekler

 Neler iyileştirildi?

- Yalnızca bir çalışma sayfasındaki tüm formülleri hesaplamayı destekler
 Akıllı işaretleyicilerle POJO ve POJO'ları destekler

 Neler düzeltildi?

- Çalışma sayfasının adlandırılmış aralıklarını alma hatası düzeltildi
 PivotTable oluşturma hatası düzeltildi
 Dosyayı koruma hatası düzeltildi
 Formüllerin hesaplanmasındaki bir hata düzeltildi
 Dosyayı kaydetme hatası düzeltildi.
