 Sayın müşteriler,

 Aspose.Cells for Java V1.7.0 sürümünü yayınladık! Aşağıdaki geliştirmeler ve düzeltmeler dahildir:

- CSV formatındaki dosyayı açmayı/kaydetmeyi destekler
 PivotTable eklemeyi destekler
 Akıllı işaretleyici işleme performansını artırır
 Çalışma sayfasıyla başa çıkma hatasını düzeltir
Grafiğin DataLabels hatasını düzeltir
 v1.6'dan sonraki diğer iyileştirmeler ve düzeltmeler
