{{% alert color="primary" %}} 

Bazen kullanıcı kaynak aralığın satır yüksekliklerini hedef aralığa kopyalamak isteyebilir. Aspose.Cells, bu amaç için [PasteType.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS) enumunu sağlar. [PasteType.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS) enumu ile [PasteOptions.setPasteType()](https://reference.aspose.com/cells/java/com.aspose.cells/pasteoptions#PasteType) özelliğini ayarladığınızda, kaynak aralık içindeki tüm satırların yükseklikleri hedef aralığa kopyalanır.

{{% /alert %}}
