{{% alert color="primary" %}} 

 Bazen kullanıcının kaynak aralığın satır yüksekliklerini hedef aralığa kopyalaması gerekir. Aspose.Cells sağlar[YapıştırTürü.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS) bu amaçla enum. ne zaman ayarlayacaksın[PasteOptions.setPasteType()](https://reference.aspose.com/cells/java/com.aspose.cells/pasteoptions#PasteType) ile mülk[YapıştırTürü.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS) enum sonra kaynak aralık içindeki tüm satırların yükseklikleri hedef aralığa kopyalanacaktır.

{{% /alert %}}
