Eski sürümlerle uyumluluk için, Aspose.Cells varsayılan olarak Excel'den PDF'ye dönüştürürken ızgara çizgisini noktalı çizgi olarak gösterir. Ancak, modern Excel günümüzde ızgara çizgisini katı çizgi olarak gösterir.

[PdfSaveOptions.GridlineTypes](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setGridlineType-int-) seçeneğiyle, Aspose.Cells ızgara çizgisini düz çizgi olarak da gösterebilir.
