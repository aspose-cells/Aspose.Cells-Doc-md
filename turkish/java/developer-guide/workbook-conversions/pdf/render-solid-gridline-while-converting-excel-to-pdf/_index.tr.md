Aspose.Cells, eski sürümlerle uyumluluk için Excel'i PDF'ye dönüştürürken ızgaralı çizgiyi varsayılan olarak noktalı çizgi olarak oluşturur. Ancak, modern Excel günümüzde ızgaralı çizgiyi düz çizgi olarak oluşturur.

[PdfSaveOptions.GridlineTypes](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions/#setGridlineType-int-) seçeneği ile Aspose.Cells ayrıca ızgara çizgilerini kesintisiz çizgi olarak oluşturabilir.
