Bu davranışı kullanarak değiştirebilirsiniz.[**ImageOrPrintOptions.OutputBlankPageWhenNothingToPrint**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#OutputBlankPageWhenNothingToPrint) Emlak. ne zaman ayarlayacaksın**doğru**, boş sayfayı yazdıracaktır.
