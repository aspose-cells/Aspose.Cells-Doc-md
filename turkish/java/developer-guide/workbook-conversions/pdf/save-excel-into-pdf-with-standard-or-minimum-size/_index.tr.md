 Varsayılan olarak, Aspose.Cells, Excel'i Standart boyutta PDF'ye kaydeder. Ancak, Minimum boyut ile de kaydedebilirsiniz.[PdfSaveOptions.OptimizationType](https://reference.aspose.com/cells/java/com.aspose.cells/pdfsaveoptions#OptimizationType) Emlak. Aşağıdaki değerleri kabul eder

- [PdfOptimizationType.STANDART](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#STANDARD)
- [PdfOptimizationType.MINIMUM_SIZE](https://reference.aspose.com/cells/java/com.aspose.cells/pdfoptimizationtype#MINIMUM_SIZE)
