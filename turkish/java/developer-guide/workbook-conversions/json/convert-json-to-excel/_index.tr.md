## **JSON'u Excel Workbook'e dönüştür**

JSON'u Excel dosyasına nasıl dönüştüreceğinizi merak etmeyin, çünkü Aspose.Cells Java kütüphanesi en iyi kararı verir. Aspose.Cells Java API, JSON formatını elektronik tablolara dönüştürmeyi destekler. JSON'u Çalışma Kitabına içe aktarmak için [**JsonLoadOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLoadOptions) sınıfını kullanabilir ve ek ayarları belirtebilirsiniz.

Aşağıdaki kod örneği, JSON'u Excel Çalışma Kitabına içe aktarmayı göstermektedir. Lütfen başvuru için kodu inceleyin [kaynak dosyası](örnek.json) kod tarafından oluşturulan xlsx dosyası.
