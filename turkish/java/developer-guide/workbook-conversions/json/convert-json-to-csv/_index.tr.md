Aspose.Cells, basit ve iç içe JSON'u CSV'ye dönüştürmeyi destekler. Bunun için API şunları sağlar:[**JsonLayoutOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLayoutOptions)ve[**Json Yardımcı Programı**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonUtility)sınıflar. bu[**JsonLayoutOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLayoutOptions)class gibi JSON düzeni için seçenekler sunar[**Dizi Başlığını Yoksay**](https://reference.aspose.com/cells/java/com.aspose.cells/jsonlayoutoptions#IgnoreArrayTitle)(dizi bir nesnenin özelliğiyse başlığı yok sayar) veya[**DiziAsTablosu**](https://reference.aspose.com/cells/java/com.aspose.cells/jsonlayoutoptions#ArrayAsTable)(diziyi tablo olarak işler). bu[**Json Yardımcı Programı**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonUtility)class ile ayarlanan düzen seçeneklerini kullanarak JSON'u işler.[**JsonLayoutOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLayoutOptions)sınıf.

Aşağıdaki kod örneği, kullanımını gösterir[**JsonLayoutOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonLayoutOptions)ve[**Json Yardımcı Programı**](https://reference.aspose.com/cells/java/com.aspose.cells/JsonUtility)yüklemek için sınıflar[kaynak JSON dosyası](SampleJson.json)ve oluşturur[çıkış CSV dosyası](SampleJson_out.csv).
