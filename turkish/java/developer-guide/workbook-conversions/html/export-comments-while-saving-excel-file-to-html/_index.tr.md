## **Olası Kullanım Senaryoları**

Excel dosyanızı HTML'e kaydettiğinizde, yorumlar dışa aktarılmaz. Ancak Aspose.Cells, [**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#IsExportComments) özelliğini kullanarak bu özelliği sağlar. **true** olarak ayarlarsanız, HTML ayrıca Excel dosyanızdaki yorumları da görüntüler.

## **Excel dosyasını HTML olarak kaydederken yorumları dışa aktar**

Aşağıdaki örnek kod, [**HtmlSaveOptions.IsExportComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#IsExportComments) özelliğinin kullanımını açıklar. Ekran görüntüsü, özelliğin **true** olarak ayarlandığında HTML üzerindeki etkisini göstermektedir. Referans için lütfen [örnek Excel dosyasını](50528270.xlsx) ve oluşturulan [HTML'yi](50528269) indirin.
