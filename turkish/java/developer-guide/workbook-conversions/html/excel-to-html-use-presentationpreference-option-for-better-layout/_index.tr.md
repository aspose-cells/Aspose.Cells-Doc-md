{{% alert color="primary" %}}

Aspose.Cells, bir Microsoft Excel dosyasını HTML veya MHT biçimine kaydederken daha iyi bir düzen elde etmek için kullanılabilecek bir özellik olan [**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference) sağlar. Özelliğin varsayılan değeri **false**'dur. Size önerimiz, bu özelliği **true** olarak ayarlayarak Excel raporundan daha iyi görünümlü sunumlar elde etmenizdir.

{{% /alert %}}
