Aspose.Cells faydalı bir özellik sağlar,[**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference) , geliştiricilerin bir Microsoft Excel dosyasını HTML veya MHT biçiminde kaydederken daha iyi bir düzen elde etmek için kullanabileceği. Özelliğin varsayılan değeri**YANLIŞ** . Bu özelliği şu şekilde ayarlamanızı öneririz:**doğru** bir Excel raporundan daha iyi görünen sunumlar elde etmek için.
