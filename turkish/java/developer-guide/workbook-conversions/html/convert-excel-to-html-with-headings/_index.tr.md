## **Excel'i başlıklarla HTML'ye dönüştürün**

Aspose.Cells, Excel'i HTML'ye dönüştürürken başlıkları dışa aktarma seçeneği sunar. Bunun için ayarlayabilirsiniz[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings)mülkiyet**doğru**. varsayılan değeri[**HtmlSaveOptions.ExportHeadings**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportHeadings)özellik**yanlış**.
