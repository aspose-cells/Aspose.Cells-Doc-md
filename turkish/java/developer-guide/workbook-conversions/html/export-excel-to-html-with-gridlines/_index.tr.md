{{% alert color="primary" %}} 

Excel dosyanızı GridLines ile HTML'e dışa aktarmak istiyorsanız, lütfen [HtmlSaveOptions.ExportGridLines](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#ExportGridLines) özelliğini kullanın ve bu **true** olarak ayarlayın.

{{% /alert %}}
