## **Olası Kullanım Senaryoları**

Excel dosyanızı HTML'e kaydettiğinizde, Aspose.Cells Downlevel Koşullu Yorumları açığa çıkarır. Bu koşullu yorumlar genellikle eski Internet Explorer sürümleri ile ilgilidir ve modern Web Tarayıcıları için gereksizdir. Detaylı bilgiyi aşağıdaki bağlantıda okuyabilirsiniz.

- [Koşullu yorum - Downlevel-açıklanan koşullu yorum](https://en.wikipedia.org/wiki/Conditional_comment#Downlevel-revealed_conditional_comment)

Aspose.Cells, bu Downlevel Açığa Çıkarılmış Yorumları [**HtmlSaveOptions.DisableDownlevelRevealedComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#DisableDownlevelRevealedComments) özelliğini **true** olarak ayarlayarak ortadan kaldırmanızı sağlar.

## **HTML'ye kaydederken Downlevel Açıklanan Yorumları Devre Dışı Bırak**

Aşağıdaki örnek kod, [**HtmlSaveOptions.DisableDownlevelRevealedComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#DisableDownlevelRevealedComments) özelliğinin kullanımını göstermektedir. Yönlendirilmemiş durumda **true** olarak ayarlanmadığında bu özelliğin etkisi olan ekran görüntüsünü aşağıdan indirebilirsiniz.
