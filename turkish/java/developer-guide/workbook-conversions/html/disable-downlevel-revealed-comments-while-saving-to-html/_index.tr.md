## **Olası Kullanım Senaryoları**

Excel dosyanızı HTML'ye kaydettiğinizde Aspose.Cells, Alt Düzey Koşullu Yorumları gösterir. Bu koşullu yorumlar çoğunlukla Internet Explorer'ın eski sürümleriyle ilgilidir ve modern Web Tarayıcılarıyla ilgisizdir. Onlarla ilgili detaylı bilgiyi aşağıdaki linkten okuyabilirsiniz.

- [Koşullu yorum - Alt seviye tarafından açıklanan koşullu yorum](https://en.wikipedia.org/wiki/Conditional_comment#Downlevel-revealed_conditional_comment)

Aspose.Cells, bu Alt Düzey Açığa Çıkan Yorumları ortadan kaldırmanıza olanak tanır.[**HtmlSaveOptions.DisableDownlevelRevealedComments**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#DisableDownlevelRevealedComments)mülkiyet**doğru**.
