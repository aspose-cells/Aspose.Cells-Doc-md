## **Olası Kullanım Senaryoları**

Excel dosyanızı HTML'e kaydederken, bir temel64 görüntüde birden fazla kez kullanılan öğe için, özel özellik ile görüntü verisi yalnızca bir kez kaydedilir ve böylece oluşan HTML'nin performansı artırılabilir. Lütfen [**HtmlSaveOptions.EnableCssCustomProperties**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions/properties/#EnableCssCustomProperties) özelliğini kullanın ve HTML'e kaydederken **true** olarak ayarlayın.
