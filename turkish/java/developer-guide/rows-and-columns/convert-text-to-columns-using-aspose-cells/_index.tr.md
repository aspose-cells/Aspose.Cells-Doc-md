## **Aspose.Cells Kullanarak Metni Sütunlara Dönüştürme**
Aşağıdaki örnek kod, [TextToColumns](https://reference.aspose.com/cells/java/com.aspose.cells/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) metodunun kullanımını açıklar. Kod önce ilk çalışma sayfasının A sütununa bazı insan isimleri ekler. İlk ve soyadı boşluk karakteriyle ayrılmıştır. Ardından A sütununa [TextToColumns](https://reference.aspose.com/cells/java/com.aspose.cells/cells#textToColumns\(int,%20int,%20int,%20com.aspose.cells.TxtLoadOptions\)) metodunu uygular ve çıktı excel dosyası olarak kaydeder. Eğer [çıktı excel dosyasını](25395230.xlsx) açarsanız, ilk isimleri A sütununda, soyadlarını ise B sütununda görebilirsiniz.
