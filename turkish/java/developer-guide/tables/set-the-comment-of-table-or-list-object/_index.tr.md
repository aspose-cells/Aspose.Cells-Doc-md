## **Çalışma Sayfası içinde Masa veya Liste Nesnesi Yorumunu Ayarlayın**

Lütfen çalışma sayfasındaki Tablo veya Liste Nesnesinin yorumunu [**ListObject.Comment**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment) özelliğini kullanarak ayarlayın. Yorum, xl/tables/tableName.xml dosyası içinde görüntülenecektir.

Aşağıdaki örnek kod, [kaynak excel dosyasını](5472561.xlsx) yükler, çalışma sayfasındaki ilk tablo veya liste nesnesinin yorumunu ayarlar ve ekran görüntüsü, bu kod tarafından oluşturulan yorumu kırmızı dikdörtgen içinde gösterir.
