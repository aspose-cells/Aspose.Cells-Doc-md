Aspose.Cells, normal bir ifade kullanarak bir çalışma kitabındaki metni değiştirme özelliği sağlar. Bunun için API şunları sağlar:[**normal ifade anahtarı**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)mülkiyeti[**Seçenekleri Değiştir**](https://reference.aspose.com/cells/java/com.aspose.cells/ReplaceOptions)sınıf. ayarlamak[**normal ifade anahtarı**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)ile**doğru**aranan anahtarın normal bir ifade olacağını belirtir.

Aşağıdaki kod parçacığı,[**normal ifade anahtarı**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)özelliğini kullanarak[örnek excel dosyası](SampleRegexReplace.xlsx). bu[çıktı dosyası](RegexReplace_out.xlsx)aşağıdaki kod parçacığı tarafından oluşturulan referans için eklenmiştir.
