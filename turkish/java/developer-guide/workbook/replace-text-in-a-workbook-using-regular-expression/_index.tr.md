Aspose.Cells, düzenli bir ifade kullanarak bir çalışma kitabındaki metni değiştirme özelliğini sağlar. Bu için API, [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) özelliğini içeren [**ReplaceOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/ReplaceOptions) sınıfını sağlar. [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) ayarını **true** olarak belirlemek aranan anahtarın düzenli bir ifade olacağını gösterir.

Aşağıdaki kod parçası, [örnek excel dosyası](SampleRegexReplace.xlsx) kullanılarak [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey) özelliğinin nasıl kullanılacağını gösterir. Aşağıdaki kod parçası tarafından oluşturulan [çıkış dosyası](RegexReplace_out.xlsx) referans için eklenmiştir.
