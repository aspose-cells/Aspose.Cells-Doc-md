## **Excel'i PDF'ye dönüştürürken Zaman Çizelgesi çizin**
Zaman çizelgesi uygulanmış bir Excel dosyanız varsa ve Excel'i zaman çizelgesi ayarlarıyla PDF'ye dışa aktarmak istiyorsanız, Aspose.Cells for Java artık bunu varsayılan olarak desteklemektedir. Excel dosyasını zaman çizelgesiyle birlikte PDF'ye dışa aktarmanız yeterlidir, oluşturulan PDF uygulanan zaman çizelgesini gösterecektir.

 Aşağıdaki örnek kod,[örnek excel dosyası](input.xlsx) mevcut bir zaman çizelgesini içerir. Daha sonra çalışma kitabını şu şekilde kaydeder:[çıktı PDF dosyası](out.pdf). Aşağıdaki ekran görüntüsü, kaynak Excel dosyasını ve oluşturulan PDF dosyasını karşılaştırır.
