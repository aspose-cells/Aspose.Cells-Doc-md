## **Olası Kullanım Senaryoları**

 Dilimleyici ve pivot tabloyu Excel'de ilişkilendirmek istiyorsanız, dilimleyiciye sağ tıklayıp "Bağlantıları Raporla..." öğesini seçmeniz gerekir. Seçenek listesinde, onay kutusu üzerinde işlem yapabilirsiniz. Benzer şekilde, programlı olarak Aspose.Cells Java API kullanarak dilimleyici ve pivot tabloyu ilişkilendirmek istiyorsanız, lütfen[**Slicer.addPivotConnection(PivotTable pivotu)**](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#addPivotConnection(com.aspose.cells.PivotTable)/) yöntem. Dilimleyici ve pivot tabloyu ilişkilendirir.

## **İlişkili Dilimleyici ve PivotTable**

Aşağıdaki örnek kod,[örnek excel dosyası](add-pivot-connection.xlsx)mevcut bir dilimleyici içerir. Dilimleyici'ye erişir ve ardından Dilimleyici ile PivotTable'ı ilişkilendirir. Son olarak, çalışma kitabını şu şekilde kaydeder:[çıktı excel dosyası](add-pivot-connection-out.xlsx).
