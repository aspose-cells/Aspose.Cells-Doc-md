## **Olası Kullanım Senaryoları**

Microsoft Excel, aşağıdaki ekran görüntüsünde gösterildiği gibi Ole Nesnesinin görüntü etiketini değiştirmenize olanak tanır. Ayrıca, Aspose.Cells API'leri kullanarak da Ole nesnesinin görüntü etiketine erişebilir veya onu değiştirebilirsiniz [**OleObject.Label**](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#Label) özelliği ile.
