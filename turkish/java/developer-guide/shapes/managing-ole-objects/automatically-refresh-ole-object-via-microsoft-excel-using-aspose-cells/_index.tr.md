 Aspose.Cells şunları sağlar:[OleObject.AutoLoad](https://reference.aspose.com/cells/java/com.aspose.cells/oleobject#AutoLoad) Excel dosyası Microsoft Excel'de açıldığında OLE nesnesini yenileme özelliği. ## **Aspose.Cells kullanarak Microsoft Excel aracılığıyla OLE nesnesini otomatik olarak yenile**
