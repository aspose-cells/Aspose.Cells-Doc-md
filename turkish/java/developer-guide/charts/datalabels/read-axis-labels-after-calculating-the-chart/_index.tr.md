## **Olası Kullanım Senaryoları**

Lütfen [**Chart.calculate()**](https://reference.aspose.com/cells/java/com.aspose.cells/chart/#calculate--) yöntemini kullanarak grafik değerlerini hesaplama işleminden sonra grafiğinizin ekseni etiketlerini okuyun. Bu amaçla lütfen [**Axis.getAxisTexts()**](https://reference.aspose.com/cells/java/com.aspose.cells/axis/#getAxisTexts--) yöntemini kullanın, bu amaçla ekseni etiketlerinin listesini döndürecektir.

## **Grafik Hesaplandıktan Sonra Eksen Etiketlerini Okuma**
