Excel grafiklerinde 3 tür Eksen vardır:
1. Yatay(Kategori) Eksen : X Ekseni
1. Dikey(Değer) Eksen : Y Eksen
1. Derinlik(Seri) Ekseni : Z Ekseni



{{% /alert %}}
