## **Olası Kullanım Senaryoları**
Aspose.Cells'in önceki sürümleri, grafik görüntüye veya PDF'e dönüştürüldüğünde otomatik grafik ekseni birimlerini düzgün bir şekilde işleyemezdi. Artık Aspose.Cells, otomatik grafik ekseni birimlerinin işlenmesini destekliyor. Kod değişikliği yoktur. Sadece grafiğinizi resme veya PDF'e dönüştürün ve grafik eksenini tıpkı Microsoft Excel'in oluşturduğu gibi işleyecektir.
