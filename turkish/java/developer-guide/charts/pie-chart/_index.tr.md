 Bu makalede, Aspose.Cells for Java API kullanılırken sıfırdan lider çizgileri olan bir pasta grafiğin nasıl oluşturulacağı açıklanmaktadır. Excel'de 'Lider çizgileri göster' seçeneği varsayılan olarak ayarlanmıştır, böylece Excel'de bir pasta grafiği oluşturduğunuzda lider çizgiler gösterilir. Ancak, Aspose.Cells API'leri ile benzer bir grafik oluştururken, açıkça ayarlamanız gerekir.[**Series.HasLeaderLines**](https://reference.aspose.com/cells/java/com.aspose.cells/series#HasLeaderLines) Emlak.
