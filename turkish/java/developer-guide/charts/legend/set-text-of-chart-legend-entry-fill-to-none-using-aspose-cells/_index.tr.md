{{% alert color="primary" %}} 

Grafiğin lejant giriş dolgu metnini hiçbir şey olarak ayarlamak istiyorsanız böylece grafiğin lejant içinde görüntülenmemesi için lütfen [LegendEntry.IsTextNoFill](https://reference.aspose.com/cells/java/com.aspose.cells/legendentry#IsTextNoFill)’i **true** olarak ayarlayın.

{{% /alert %}}
