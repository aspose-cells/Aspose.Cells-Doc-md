{{% alert color="primary" %}}

 Aspose.Cells, bir Pivot Tabloyu bir Çalışma Sayfasından silmek veya kaldırmak için bir özellik sağlar. Pivot tablo nesnesini veya pivot tablo konumunu kullanarak pivot tabloyu silebilirsiniz. lütfen[**Worksheet.getPivotTables().remove()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottablecollection#remove(com.aspose.cells.PivotTable) ) pivot tablo nesnesini kullanarak pivot tabloyu silme yöntemi ve[**Worksheet.getPivotTables().removeAt()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottablecollection#removeAt(int)pivot tablo koleksiyonu içindeki konumunu kullanarak bir pivot tablo nesnesini silme yöntemi.

{{% /alert %}}
