## **Pivot Tablosunda Özel Sıralama**
Aspose.Cells API'sını kullanarak, alan değerlerine göre Pivot Tablolarını sıralayabilirsiniz. Aşağıdaki kod parçası, örnek excel dosyasını yükler ve üç özet tablo ekler. İlk özet tablosu özel sıralamasızdır, ikinci özet tablosu 'Deniz Ürünleri' satır alanı değerlerine göre sıralanmıştır ve üçüncü özet tablosu '28/07/2000' sütun alanı değerlerine göre sıralanmıştır.
