## **Özet Tabloda özel sıralama**
Aspose.Cells API kullanarak Pivot Tabloları alan değerlerine göre sıralayabilirsiniz. Aşağıdaki kod parçacığı, örnek excel dosyasını yükler ve üç pivot tablo ekler. Birinci pivot tablo özel sıralama içermez, ikinci pivot tablo "Deniz Ürünleri" satır alanı değerlerine göre ve üçüncü pivot tablo "28/07/2000" sütun alanı değerlerine göre sıralanır.
