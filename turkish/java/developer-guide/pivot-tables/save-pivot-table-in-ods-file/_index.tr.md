Aspose.Cells, özet tabloyu ODS dosyalarında kaydetme olanağı sağlar. Bunun için mevcut bir özet tabloya sahip bir çalışma kitabını dönüştürmeniz veya yeni bir özet tablosu oluşturmanız ve dosyayı ODS formatında kaydetmeniz yeterlidir. Çıktıdaki özet tablonun, çıkış ODS dosyasında görüntülenmesini sağlamak için kaydetmeden önce [**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData--) çağırmayı unutmayın. Aşağıdaki kod parçacığı, özet tabloyu ODS dosyasına kaydetmeyi gösterir.
