{{% alert color="primary" %}}

Microsoft Excel, Pivot Tablosu Düzenini değiştirmenize izin verir. Bu, *PivotTable Araçları > Tasarım > Rapor* Düzeni'nden menü komutları kullanılarak yapılabilir. Düzeni bu üç formda değiştirebilirsiniz:

- Kompakt Formda Göster
- Açıklamalı Formda Göster
- Tablo Formunda Göster

Aspose.Cells ayrıca pivot tablosunun bu üç biçimde yerleşimini değiştirmek için [**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm--), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm--) ve [**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm--) yöntemlerini sağlar.

{{% /alert %}}
