{{% alert color="primary" %}}

 Microsoft Excel kullanarak Pivot Tablo Düzenini değiştirmenize izin verir*PivotTable Araçları > Tasarım > Rapor* Düzen menüsü komutları. Düzeni bu üç formda değiştirebilirsiniz.

- Kompakt Formda Göster
- Anahat Formunda Göster
- Tablo Şeklinde Göster

 Aspose.Cells ayrıca sağlar[**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm()), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm() ) ve[**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm()) bu üç formdaki pivot tablonun düzenini değiştirme yöntemleri.

{{% /alert %}}
