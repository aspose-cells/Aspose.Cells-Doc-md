## **Pivot Tablonun Dış Bağlantı Veri Kaynağını Alın**

Aspose.Cells, pivot tablonun harici bağlantı veri kaynağını alma olanağı sağlar. Bunun için API,[**Harici Bağlantı Veri Kaynağı**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)mülkiyeti[**Pivot tablo**](https://reference.aspose.com/cells/java/com.aspose.cells/PivotTable)sınıf. bu[**Harici Bağlantı Veri Kaynağı**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource)özellik döndürür[**Harici Bağlantı**](https://reference.aspose.com/cells/java/com.aspose.cells/ExternalConnection)nesne. Aşağıdaki kod parçacığı,[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#ExternalConnectionDataSource) pivot tablonun harici bağlantı veri kaynağını alma özelliği.
