## **Olası Kullanım Senaryoları**

Aspose.Cells, *DBNum* özel desen biçimlendirmesini destekler. Örneğin, hücre değeriniz 123 ise ve özel biçimlendirmeyi [DBNum2][$-804]Genel olarak belirtirseniz, bunun gibi 壹佰贰拾叁 olarak gösterilecektir. Hücrenin özel biçimlendirmesini [**Cell.getStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getStyle--) ve [**Style.setCustom()**](https://reference.aspose.com/cells/java/com.aspose.cells/style/#setCustom-java.lang.String-) yöntemlerini kullanarak belirtebilirsiniz.
