## **Olası Kullanım Senaryoları**

Aspose.Cells destekler*DBNum*özel desen biçimlendirme. Örneğin, hücre değeriniz 123 ise ve özel biçimlendirmesini [DBNum2][$-804]General olarak belirtirseniz, 壹佰贰拾叁 gibi görüntülenecektir. Kullanarak hücrenin özel biçimlendirmesini belirleyebilirsiniz.[**Cell.getStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getStyle()) ve[**Style.setCustom()**](https://reference.aspose.com/cells/java/com.aspose.cells/style#Custom)yöntemler.
