## **ODS Dosyalarında Hücre Doğrulamasını Al**

Aspose.Cells ile ODS dosyalarında bir hücreye uygulanan doğrulamayı alabilirsiniz. Bunun için API, [**GetValidation**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getValidation--) yöntemini ve [**Cell**](https://reference.aspose.com/cells/java/com.aspose.cells/Cell) sınıfını sağlar.

Aşağıdaki kod örneği, [**GetValidation**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getValidation--) yönteminin kullanımını gösterir; bu yöntemle [kaynak ODS](SampleBook1.ods) dosyası yüklenir ve A9 hücresinin doğrulaması okunur.
