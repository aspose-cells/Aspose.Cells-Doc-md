## **Aralıktaki Bağlantıları Al**

Seçilen aralıktaki tüm bağlantıları almak için, [**Range**](https://reference.aspose.com/cells/java/com.aspose.cells/range) sınıfı [**getHyperlinks()**](https://reference.aspose.com/cells/java/com.aspose.cells/range#Hyperlinks) özelliği getter sağlar. Ayrıca [**Hyperlink.delete()**](https://reference.aspose.com/cells/java/com.aspose.cells/hyperlink#delete--) yöntemini çağırarak Bağlantıyı silebilirsiniz.
