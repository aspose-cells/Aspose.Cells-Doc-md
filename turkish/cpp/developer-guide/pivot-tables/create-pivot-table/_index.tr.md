## **Giriş**
Lütfen çalışma sayfası içinde bir pivot tablo oluşturmak için [PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) yöntemini kullanın. Pivot tablo oluşturulduktan sonra, [PivotTable](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/) sınıfı ile onun üzerinde çalışabilirsiniz.
