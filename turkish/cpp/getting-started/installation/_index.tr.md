## **Windows'de C++ için Aspose.Cells kurulumu**
### **Nuget Paket Yöneticisini Kullanma**
1. Nuget Paket Yöneticisi'nde Aspose.Cells'i arayın.
 1. İşte[Bağlantı NuGet Paket Aspose.Cells.Cpp](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. Ayrıntılar için [Kurulum Aspose Cells ila NuGet]'e bakın.(https://docs.aspose.com/cells/cpp/getting-started/).

Ayrıca C++ için Aspose.Cells'den ZIP paketini indirebilirsiniz.[indirme sayfası](https://downloads.aspose.com/cells/cpp/). Arşiv ayrıca çevrimdışı API belgelerini de içerir.
### **Include ve lib Klasörlerini Kullanma**
1.  İndirin/Yükleyin[son dosyalar](https://downloads.aspose.com/cells/cpp/).
1. Klasörü üretim ortamına sıkıştırın.
1. Aspose.Cells'i C++ için kullanmak üzere projenizdeki "Include" ve "lib" dizinine başvurun.

## **Linux'ta C++ için Aspose.Cells kurulumu**
1.  Aşağıdaki sayfadan C++ için Aspose.Cells'i indirin:[C++(Linux) için Aspose.Cells'i indirin](https://downloads.aspose.com/cells/cpp)
2. Paketi açın ve Linux için C++ için Aspose.Cells'in nasıl kullanılacağı hakkında bir Demo bulacaksınız.
3. Linux komut satırınızda "cd Demo"yu çalıştırın
4. "rm -rf build;mkdir build;cd build" komutunu çalıştırın
5. "cmake .." komutunu çalıştırın, Demo klasöründe CMakeLists.txt tarafından bir Makefile oluşturacaktır.
6. Derlemek için "make" komutunu çalıştırın
 7. "./demo" komutunu çalıştırın, sonucu göreceksiniz
