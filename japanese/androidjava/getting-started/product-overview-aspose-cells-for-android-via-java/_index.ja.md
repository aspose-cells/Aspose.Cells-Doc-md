Aspose.Cells for Android via Java ドキュメントへようこそ！

Aspose.Cells は、Android アプリケーションがシステムに Microsoft Excel をインストールする必要なしに Excel スプレッドシートを作成および管理できる柔軟なコンポーネントです。他社製品とは異なり、Aspose.Cells は、スプレッドシートの生成および他の基本的なファイルフォーマット機能だけでなく、多数の高度な機能もサポートしています。これらの高度な機能により、開発者はスプレッドシートの内容、セルの書式設定、ファイルの保護を簡単に操作できます。Aspose.Cells はさらに、さまざまなデータソースからスプレッドシートにデータをインポートしたり、一般的な数学、日付と時刻、財務、テキストの数式や関数を追加したりすることもできます。さらに、チャート、画像、コメント、描画オブジェクト、コントロールを操作することもサポートしています。

{{% /alert %}} 
## **ドキュメントの機能**
- 異なるソースから（Excel97 - Excel2007 およびそれ以降のバージョンの）プレーンまたは暗号化された Excel ファイルを開く
- 異なる形式（XLS、XLSX/XLSM、XLTX、XLTM、XLSB、CSV、SpreadsheetML、タブ区切り、HTML など）で（Excel97 - Excel2007 およびそれ以降のバージョンの）Excel ファイルを保存
- Excel ファイルを PDF ドキュメントに変換
- Excel スプレッドシートを XPS ドキュメントに変換
- OpenDocument Spreadsheet（ODS）形式の読み込みと書き込み
- Excel ファイルのドキュメントプロパティを変更
## **ワークシート機能**
## **表示オプション**
- ワークシートを非表示または表示
- タブ、スクロールバー、グリッド線、行および列見出しの表示または非表示
- ページ ブレークをプレビューします。
- ワークシートのズームレベルを設定
- フリーズ ペインを使用してスクロールしても選択したデータを表示したままにします。
## **ワークシートの保護**
- ワークシートの内容、オブジェクト、シナリオを保護
## **ページ設定オプション**
- ページの向き、スケーリング、用紙サイズ、印刷品質を設定
- 上端、左端、右端、下端に対するページ余白を設定
- ヘッダーとフッターを設定します。
- 印刷エリアおよび印刷タイトルを設定
## **ページブレーク**
- 指定したセルに水平および垂直の改ページを挿入します。
## **行および列の機能**
- 列の幅を調整
- セルの幅と高さを自動調整
- 行および列を追加または削除
- 行と列の表示または非表示。
- 行と列をグループ化またはグループ化解除します。
## **データ管理機能**
- ランタイムで必要なセルにデータを追加
- 異なるデータソースからワークシートにデータをインポート
- セルからデータを**データ型**に従って取得
- セルのデータを配列にエクスポート
- セル内の特定のデータを**検索**
- 組み込みまたはアドイン関数を使用した数式を設定
- 異なる種類のハイパーリンクを追加
- 名前付き範囲を作成またはアクセス
- データ検証およびオートフィルタリングを適用
- ワークシートテーブルに小計を適用
## **書式設定機能**
- セルに数値形式設定を適用
- セルを日付として書式設定
- **テキストの配置**を設定します
- 複数のセルを1つにマージします
- セルに異なる**フォント設定**を適用します
- セル、フォント、グリッド線、グラフィックオブジェクトなどに異なる色を適用します
- 1つのセルに**リッチテキスト**を設定します
- セルに異なる**境界線設定**を適用します
- セルに異なる**背景パターン**を適用します
- ワークシート、行、列などに**フォーマット設定**を適用します
- セルに**条件付き書式設定**を適用します
## **チャートおよびグラフィック機能**
- 簡単にさまざまな種類のグラフを作成します
- 異なるプロパティを設定して**グラフ**をカスタマイズします
- チャートのデータを**動的に設定**します
- APIを使用してワークシートに**カスタムチャート**を追加します
- ワークシートに**画像**をランタイムで追加します
- セルに**コメント**を作成および追加します
- グラフを画像ファイルに変換する
- ワークシートを画像ファイルに変換する
## **高度な機能**
- 頑健な式計算エンジンを使用して**式の計算**をサポートします
- ピボットテーブルの**ソースデータ**をランタイムで変更します
- ピボットテーブルを作成および操作します
