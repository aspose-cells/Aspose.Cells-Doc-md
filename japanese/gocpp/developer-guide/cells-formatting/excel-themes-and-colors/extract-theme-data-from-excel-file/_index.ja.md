{{% alert color="primary" %}}

Aspose.Cellsは、Excelファイルからテーマに関するデータを抽出できます。たとえば、ワークブックに適用されたテーマ名やセルに適用されたテーマカラー、セルのボーダーなどを抽出可能です。

Microsoft Excelのページレイアウト > テーマコマンドを使用して、テーマをワークブックに適用できます。
