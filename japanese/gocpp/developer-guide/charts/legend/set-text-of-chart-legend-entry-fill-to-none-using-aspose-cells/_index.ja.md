{{% alert color="primary" %}}

チャートの凡例エントリの塗りつぶしのテキストを表示しないようにするには、[**LegendEntry.IsTextNoFill**](https://reference.aspose.com/cells/go-cpp/legendentry/istextnofill/)を**true**に設定してください。

{{% /alert %}}
