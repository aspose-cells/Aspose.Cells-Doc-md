## **可能な使用シナリオ**

Excelファイルの言語を変更するには、ファイルを右クリックして[プロパティ] > [詳細設定]を選択し、言語フィールドを編集します。プログラムで変更するには、[**BuiltInDocumentPropertyCollection.GetLanguage()**](https://reference.aspose.com/cells/go-cpp/builtindocumentpropertycollection/getlanguage/)プロパティをAspose.Cells APIを使用して使用してください。
