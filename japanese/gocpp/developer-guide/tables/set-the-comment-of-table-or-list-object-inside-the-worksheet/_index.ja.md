ワークシート内のテーブルまたはリストオブジェクトのコメントを [ListObject.SetComment](https://reference.aspose.com/cells/go-cpp/listobject/setcomment/) メソッドを使用して設定してください。コメントは xl/tables/tableName.xml ファイル内に表示されます。
