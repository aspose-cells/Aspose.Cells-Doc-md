スプレッドシートからHTMLファイルを生成することは非常に一般的です。列のサイズは多くの場合「pt」で定義されています。ただし、生成されたテーブルの幅が大きい場合、600pxのコンテナパネルの幅に適合させる必要がある場合があります。この場合、生成されたテーブルの幅が大きい場合に水平スクロールバーが表示される可能性があります。[**HtmlSaveOptions.GetWidthScalable()**](https://reference.aspose.com/cells/go-cpp/htmlsaveoptions/getwidthscalable/)を**true**に設定すると、この固定サイズをemやパーセントなどのスケーラブルな単位に変更して、より良い表示を行うことができます。
