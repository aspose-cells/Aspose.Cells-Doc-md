## **可能な使用シナリオ**

Microsoft Excelは、多くのフォーマット（XLS、XLSX、XLSM、XLSB、CSVなど）をサポートしています。Aspose.Cellsもこれらの多くのフォーマットをサポートしています。この記事は、Aspose.Cellsを使用してXLSBフォーマットのExcelファイルを読み書きする方法について説明します。
