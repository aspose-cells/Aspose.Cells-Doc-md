## **可能な使用シナリオ**

Microsoft ExcelはXLS、XLSX、XLSM、XLSB、CSV、タブ区切りなど、多くのフォーマットをサポートしています。Aspose.Cellsもこれらの多くをサポートしています。この記事では、Aspose.Cellsを使用してタブ区切り形式のExcelファイルを読み書きする方法について説明します。
