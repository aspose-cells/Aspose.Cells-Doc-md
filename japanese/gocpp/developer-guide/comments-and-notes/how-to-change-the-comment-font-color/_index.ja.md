{{% alert color="primary" %}} 

Microsoft Excelでは、ユーザーはセルにコメントを追加して追加情報を表示し、データを強調表示することができます。開発者は、コメントをカスタマイズして配置設定、テキスト方向のフォントカラーなどを指定する必要があります。Aspose.Cellsは、このタスクを実行するためのAPIを提供しています。

{{% /alert %}}
