序章

Aspose.Cells for Reporting Services を使用すると、Microsoft SQL Server 2005 Reporting Services でネイティブ Excel レポートを配信できます。 Excel ベースのレポートの組み込みレンダラーとは異なり、Aspose.Cells for Reporting Services は拡張機能セットを提供し、RDL レポートを Excel 97-2003 バイナリ XLS 形式、SpreadsheetML または Excel2007 XLSX 形式に変換できます。
