{{% alert color="primary" %}} 

Aspose.Cells for Reporting Services（Aspose.Cells.ReportingServicesクライアント、Microsoft Excelのアドイン、およびAspose.Cells.ReportingServices Serverを含む）を適切にインストールする必要があります。さらに、 Microsoft SQL Serverをインストールする前にAspose.Cells for Reporting Servicesをインストールする必要があります。

{{% /alert %}} 
### **Office System**
以下のMicrosoft Excelのバージョンがサポートされています。また、Microsoft Office 2000、2002、XP、または2003のいくつかの問題を克服するために追加のパッチをインストールする必要があります。

- Microsoft Excel 2000
  Office 2003より古いMicrosoft Officeマシン上のCLRの問題を修正するには**lockbackRegKey.msi**をインストールしてください。
- Microsoft Excel 2002
  Office 2003より古いMicrosoft Officeマシン上のCLRの問題を修正するには**lockbackRegKey.msi**をインストールしてください。
- Microsoft Excel 2003
  常にOffice 2003（KB907417）に更新してください。この更新プログラムは、Microsoft .NET Framework 2.0を使用するアドイン、スマートタグ、またはスマートドキュメントがロードされない可能性のある問題を修正します。
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
- Microsoft Excel 2016
- Microsoft Excel 2019
- Microsoft Excel 365
