グリッド線はで構成されます**Aspose.Cells.ReportingServices.xml**エクスポートされた Microsoft Excel ファイルにグリッド線を表示するかどうかを制御できます。デフォルト値は**オフ**、グリッド線が表示されないことを意味します。

{{< highlight "java" >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
