グリッドラインは**Aspose.Cells.ReportingServices.xml**で構成されているため、エクスポートされたMicrosoft Excelファイルがグリッドラインを表示するかどうかを制御できます。デフォルト値は**off**で、これはグリッドラインが表示されないことを意味します。

{{< highlight java >}}

 <Aspose>

...

  <DataWorkSheet> 

    <GridLine value="off"/>

  </DataWorkSheet>

...

</Aspose>

{{< /highlight >}}
