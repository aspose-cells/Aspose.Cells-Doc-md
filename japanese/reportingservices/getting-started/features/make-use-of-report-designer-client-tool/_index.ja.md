{{% alert color="primary" %}} 

ユーザーは、シンプルで使いやすいツールである Aspose.Cells.Report.Designer を使用して、レポートを設計し、Microsoft Excel に直接配信できます。レポートを作成するために Visual Studio を使用する必要はありませんが、追加の Visual Studio ライセンスを購入するコストを節約でき、複雑な開発環境の使用方法を学ぶのに時間を費やす必要がありません。

Aspose.Cells.Report.Designer は Microsoft Excel をレポート テンプレートとして使用するため、ユーザーは、ピボット テーブル、インタラクティブ チャート、VBA、マクロ、数式など、Microsoft Excel のすべての強力な機能を利用できます。これらの機能のすべてが、Reporting Services が提供する標準の Microsoft Excel 形式でサポートされているわけではありません。

{{% /alert %}} 
- **Aspose.Cells.Report.Designer ツールバー**
**ユーザーは、Aspose.Cells.Report.Designer ツールバーを使用して、RDL レポート ファイルを操作できます。**
