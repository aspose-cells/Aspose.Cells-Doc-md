**Aspose.Cells for Reporting Services ドキュメンテーションへようこそ!**
Aspose.Cells は受賞歴のあるスプレッドシート コンポーネントであり、開発者は Microsoft Excel に依存することなく、Excel スプレッドシートの読み取り、書き込み、および操作の機能を独自のアプリケーションに組み込むことができます。


**製品の概要**
Aspose.Cells for Reporting Services は、Reporting Services で Microsoft Excel レポートを配信する必要がある開発者向けのソリューションです。 Aspose.Cells.Report.Designer ツールを使用して、レポート サービスに柔軟でユーザー フレンドリーな機能を追加します。データ ソースの設定、クエリの作成、レポートのデザイン、Report Server へのレポートの発行など、Reporting Services でのレポートの開発に関連するすべてのタスクを実行できます。 Aspose.Cells.Report.Designer は Microsoft Excel をレポート テンプレートとして使用するため、ユーザーは Microsoft Excel の強力な機能を利用できます。

Aspose.Cells for Reporting Services コンポーネントを使用してレポートをデザインするには、2 つの方法があります。空の Microsoft Excel テンプレートから始めて、データ ソースの設定からレポート サーバーへのレポートの発行までのタスクを完了します。または、データベース クエリを使用できる既存の RDL レポートから始めます。この方法を取ると、ユーザーはレポートのデザインのみに集中する必要があります。この方法は、レポートのデザインとクエリの作成を分離したいユーザーにとって便利です。後者は比較的複雑で、データベースと SQL のスキルを持つユーザーが行う必要があります。

**Aspose.Cells for Reporting Services を使用してレンダリングされたサンプル レポート** 

![todo:画像_代替_文章](product-overview_2.png)

Aspose.Cells for Reporting Services は 2 つの部分で構成されています。
- **Aspose.Cells.Report.Designer**
これは、アドイン フォームで Microsoft Excel と統合されているレポート デザイナー ツールです。 Aspose.Cells.Report.Designer を使用すると、ユーザーは Visual Studio の代わりに Microsoft Excel でレポート テンプレートをデザインできます。次に、レポート テンプレートが Reporting Services RDL ファイルに埋め込まれ、レポート サーバーに発行できるようになります。ユーザーは、Aspose.Cells.Report.Designer でレポートをプレビューおよびエクスポートすることもできます。
- **Aspose.Cells.レンダラー for Reporting Services**
Aspose.Cells.Renderer for Reporting Services は、Microsoft の Reporting Services Rendering Extension の実装です。 Visual Studio と Aspose.Cells.Report.Designer の両方で設計された RDL レポートをレンダリングできます。
