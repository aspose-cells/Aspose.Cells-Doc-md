{{% alert color="primary" %}} 

- 以前のバージョンの Aspose.Cells for Reporting Services がインストールされている場合は、まずそれをアンインストールしてください。見る[アンインストール中 Aspose.Cells for Reporting Services](/cells/ja/reportingservices/uninstalling-aspose-cells-for-reporting-services/)コンポーネントの取り外し方法に関するトピック。
- [Aspose.Cells for Reporting Services ダウンロードページにアクセス](https://downloads.aspose.com/cells/reportingservices)コンポーネントの最新バージョンを含む Aspose.Cells for Reporting Services MSI インストーラーをダウンロードします。
 - ReportingServices の場合は Aspose.Cells Msi は、Microsoft SQL Server 2008 Reporting Services 以降の製品のみをサポートします。
- ダウンロードしたファイルを実行し、インストーラーの指示に従います。

{{% /alert %}} 
**インストールプロセス**
MSI インストーラーを使用して Aspose.Cells をインストールするには:

1. ようこそ画面は、インストールの最初の画面です。クリック**次**.

![todo:画像_代替_文章](using-msi-installer_1.png)




1. 使用許諾契約を読んで同意し、クリックします**次**.

![todo:画像_代替_文章](using-msi-installer_2.png)




1. インストールフォルダを選択してクリック**次**. 

![todo:画像_代替_文章](using-msi-installer_3.png)




1. クリックして、Aspose.Cells for Reporting Services をインストールすることを確認します。**インストール**. 

![todo:画像_代替_文章](using-msi-installer_4.png)



インストールの進行状況が画面に表示されます。

![todo:画像_代替_文章](using-msi-installer_5.png)




1. インストールが完了したら、**終了**インストーラーを閉じます。

![todo:画像_代替_文章](using-msi-installer_6.png)
