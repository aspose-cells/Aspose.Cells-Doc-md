{{% alert color="primary" %}} 

- 以前のバージョンの Aspose.Cells for Reporting Services がインストールされている場合は、まずアンインストールしてください。[Aspose.Cells for Reporting Services のアンインストール](/cells/ja/reportingservices/uninstalling-aspose-cells-for-reporting-services/)トピックを参照してコンポーネントを削除する方法について確認します。
- [Aspose.Cells for Reporting Services ダウンロードページ](https://downloads.aspose.com/cells/reportingservices) にアクセスし、最新バージョンのコンポーネントを提供する Aspose.Cells for Reporting Services MSI インストーラーをダウンロードします。
    - Aspose.Cells for ReportingServices Msi は、Microsoft SQL Server 2008 Reporting Services およびそれ以降の製品をサポートしています。
- ダウンロードしたファイルを実行し、インストーラーの指示に従います。

{{% /alert %}} 
**インストールプロセス**
MSI インストーラーを使用して Aspose.Cells をインストールするには:

1. ウェルカム画面がインストールの最初の画面です。**次へ** をクリックします。

![todo:image_alt_text](using-msi-installer_1.png)




1. ライセンス契約書を読んで承諾し、**次へ** をクリックします。

![todo:image_alt_text](using-msi-installer_2.png)




1. インストールフォルダーを選択し、**次へ** をクリックします。 

![todo:image_alt_text](using-msi-installer_3.png)




1. **インストール** をクリックして、Aspose.Cells for Reporting Services をインストールすることを確認します。 

![todo:image_alt_text](using-msi-installer_4.png)



インストールの進行状況が画面に表示されます。 

![todo:image_alt_text](using-msi-installer_5.png)




1. インストールが完了したら、**完了** をクリックしてインストーラーを閉じます。

![todo:image_alt_text](using-msi-installer_6.png)
