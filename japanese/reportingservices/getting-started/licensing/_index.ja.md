{{% alert color="primary" %}} 

 Aspose.Cells for Reporting Services は、[ダウンロードページ](https://downloads.aspose.com/cells/reportingservices).製品の評価版とライセンス版は同じです。

評価版に満足したら、ライセンスを購入してください。サブスクリプションの利用規約を理解し、同意していることを確認してください。

{{% /alert %}} 
### **ライセンスの取得と使用**
ライセンスは、注文の支払いが完了すると、注文ページからダウンロードできるようになります。

ライセンス ファイルは、デジタル署名されたクリア テキストの XML ファイルです。ライセンス ファイルには、クライアント名、購入した製品、ライセンス タイプなどの情報が含まれています。ライセンス ファイルの内容は変更しないでください。変更すると無効になります。

ライセンスをコンピュータにダウンロードし、次のいずれかのフォルダにコピーします。

- $[インストールフォルダ]\ライセンス\
- $[MSSQL Server レポート サービス]\ReportServer\Bin\
- $[MSSQL Server レポート サービス]\ReportServer\

ライセンス ファイルの名前は、サブスクリプションの種類に基づいて次のいずれかになります。

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic"、"Aspose.Total.lic
- Aspose.Cells.Reporting.Services - 開発者サイト Subscription.lic
- Aspose.Cells.Reporting.Services - 開発者 Subscription.lic
- Aspose.Cells.Reporting.Services - 複数の展開 Subscription.lic
- Aspose.Cells.Reporting.Services - 単一サーバー展開 Subscription.lic

ライセンス ファイルが正しくインストールされたことをテストするには、レポートを Microsoft Excel ワークブックとしてエクスポートします。ワークブックの最後にウォーターマーク ワークシートが含まれていない場合、ライセンス ファイルは正常にアクティブ化されています。

 Aspose.Cells for Reporting Services は、評価モードで作業しているときに評価透かしワークシートを挿入します。有効なライセンス ファイルがある場合。**Aspose.Cells.ReportingServices.lic** $[インストール フォルダー]\License\ フォルダーにある場合、評価用透かしワークシートはありません。
