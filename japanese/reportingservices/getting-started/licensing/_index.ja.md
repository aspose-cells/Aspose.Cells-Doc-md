{{% alert color="primary" %}} 

Aspose.Cells for Reporting Servicesは、[ダウンロードページ](https://downloads.aspose.com/cells/reportingservices)から無料で、期限なしの評価版として入手できます。 製品の評価版とライセンス版は同じです。

評価版に満足したら、ライセンスを購入してください。サブスクリプションの利用条件に同意し理解していることを確認してください。

{{% /alert %}} 
### **ライセンスの入手と使用**
注文が支払われたときに、注文ページからライセンスファイルをダウンロードできます。

ライセンスファイルは、クライアント名、購入製品、ライセンスタイプなどの情報を含む、クリアテキストのデジタルに署名されたXMLファイルです。 ライセンスファイルの内容を変更しないでください。変更すると無効になります。

ライセンスをコンピュータにダウンロードし、次のフォルダーの1つにコピーしてください。

- $[Installation Folder]\License\
- $[MSSQL Server Reporting Services]\ReportServer\Bin\
- $[MSSQL Server Reporting Services]\ReportServer\

ライセンスファイルの名前は、以下のサブスクリプションタイプに基づいて1つになります。

- Aspose.Cells.Reporting.Services.lic
- Aspose.Total.ReportingServices.lic
- Aspose.Custom.lic", "Aspose.Total.lic
- Aspose.Cells.Reporting.Services - Developer Site Subscription.lic
- Aspose.Cells.Reporting.Services - Developer Subscription.lic
- Aspose.Cells.Reporting.Services - Multiple Deployment Subscription.lic
- Aspose.Cells.Reporting.Services - Single Server Deployment Subscription.lic

ライセンスファイルが正しくインストールされたかどうかをテストするには、どの報告書もMicrosoft Excelワークブックとしてエクスポートします。ワークブックに最後にウォーターマークのワークシートが含まれていない場合、ライセンスファイルは正常にアクティブ化されています。

Aspose.Cells for Reporting Servicesは、評価モードで作業する際に評価透かしワークシートを挿入します。たとえば、有効なライセンスファイル（例：**Aspose.Cells.ReportingServices.lic**が$[Installation Folder]\License\フォルダにある場合）がある場合、評価透かしワークシートはありません。
