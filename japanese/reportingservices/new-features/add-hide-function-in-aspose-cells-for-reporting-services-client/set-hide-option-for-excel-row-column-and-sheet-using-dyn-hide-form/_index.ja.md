Aspose.Cells for Reporting Servicesは、非表示パラメータに基づいて指定された行、列、およびシートを非表示にします。パラメータには次が含まれます：

- **SheetName** は隠しオブジェクトを指定するワークシートを表します。
- **Type** は隠しオブジェクトのタイプを表します。ワークシート、行、列を含みます。
- **行/列インデックス** は指定された隠しオブジェクトを表します。タイプがワークシートの場合、値はnullです。列インデックスは文字をサポートします。インデックス値は1から始まります。
- **隠し状態** は隠し状態の値を表します。値はtrue、false、またはExcelの数式です。パラメータはReporting Servicesレポートパラメータのみサポートします。
