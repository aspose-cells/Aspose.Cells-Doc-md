Aspose.Cells for Reporting Services は、Hide パラメーターで指定された行、列、およびシート ベースのセットを非表示にします。パラメータには次のものがあります。

- **シート名**非表示オブジェクトを指定するワークシートを表します。
- **タイプ**非表示オブジェクト タイプを表します。ワークシート、行、列が含まれます。
- **行/列インデックス**指定された非表示オブジェクトを表します。タイプがワークシートの場合、値は null です。列インデックスは文字をサポートしています。インデックス値は 1 から始まります。
- **状態を非表示**非表示状態の値を表します。値は true、false、または Excel 式です。このパラメーターは、Reporting Services レポート パラメーターのみをサポートします。
