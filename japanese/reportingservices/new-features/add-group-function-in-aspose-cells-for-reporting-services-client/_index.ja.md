[属性の変更] を使用してテーブル レポート アイテムのグループ属性を設定します。パラメータには、タイプ、名前、レベル、および非表示が含まれます。

- タイプは、グループ タイプを表します。行グループと列グループが含まれます。
- 名前はグループ名を表します。
- レベルは、グループ レベルを表します。
- 非表示は、グループの非表示を表します。

レポートの変更には、次の手順が含まれます。
[レポートの変更] ダイアログ ボックスを開き、[アウトライン] タブを選択します。

![todo:画像_代替_文章](add-group-function-in-aspose-cells-for-reporting-services-client_1.jpg)


クリック**変更**ボタンをクリックして、Hide 構成を更新します。

![todo:画像_代替_文章](add-group-function-in-aspose-cells-for-reporting-services-client_2.jpg)


クリック**専念**ボタンをクリックして非表示設定を保存します。

![todo:画像_代替_文章](add-group-function-in-aspose-cells-for-reporting-services-client_3.jpg)
