レポート内のすべてのグラフ アイテムを編集可能な Excel グラフとして表示するには:

1. 開く**Aspose.Cells.ReportingServices.xml**ファイル。
1. 構成パラメータを次のように変更します。

{{< highlight "java" >}}

 <Chart value="all">

</Chart>

{{< /highlight >}}

値を「all」に設定すると、すべてのグラフ項目が編集可能なグラフとして表示されます。
