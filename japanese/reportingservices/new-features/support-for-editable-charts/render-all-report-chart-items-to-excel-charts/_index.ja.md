{{% alert color="primary" %}} 

レポートのすべてのチャートアイテムを編集可能なExcelチャートとしてレンダリングするには:

1. **Aspose.Cells.ReportingServices.xml** ファイルを開きます。
1. 次のように構成パラメータを変更します: 

{{< highlight java >}}

 <Chart value="all">

</Chart>

{{< /highlight >}}

値が「all」に設定されている場合、すべてのチャートアイテムが編集可能なチャートとしてレンダリングされます。
