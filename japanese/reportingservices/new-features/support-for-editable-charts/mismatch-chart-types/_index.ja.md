{{% alert color="primary" %}} 

SQL Server Reporting Services 2008 レポート定義ファイルの一部のグラフの種類は、Excel のグラフの種類として使用できません。

- ラインステップ
- レンジプレーン
- レンジスムーズ
- 範囲列
- 範囲バー
- Range-ErrorBar
- Range-BoxPlot
- 形状漏斗

Microsoft Excel に同等のオプションがないグラフ アイテムをレンダリングすると、Aspose.Cells はそれらを画像に出力します。

{{% /alert %}}
