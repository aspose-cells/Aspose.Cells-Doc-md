Aspose.Cells for Reporting Services Report Designer は、現在アクセスされているレポート サーバーの URL を保存します。 Aspose.Cells レポート デザイナーで [レポート サーバー マネージャー] ダイアログを再度開くと、保存されているレポート サーバー情報がダイアログ フォームに自動的に読み込まれます。
