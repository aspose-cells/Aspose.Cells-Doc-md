Aspose.Cells for JasperReports のいくつかの効果的な機能を以下に示します。

1. Aspose.Cells for JasperReports は Microsoft Excel チャートをサポートしているため、チャートは静止画ではなく編集可能なチャートとしてエクスポートされます。このようにして、ユーザーは必要に応じて Microsoft Excel でチャートを手動で編集できます。
 2. Aspose.Cells for JasperReports は[JasperServer と簡単に統合](/cells/ja/jasperreports/integration-with-jasperserver/).

{{% /alert %}}
