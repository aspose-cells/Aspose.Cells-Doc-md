{{% alert color="primary" %}}

Aspose.Cells for JasperReportsは、JasperReportsおよびJasperServerからレポートをMicrosoft Excel形式にエクスポートするための製品です。 Aspose.Cells for JasperReportsは、組込みエクスポータのすべての機能を継承し、JasperReportsの元のXLSレンダラのいくつかの弱点を補うようになっています。このセクションでは、Aspose.Cells for JasperReportsの機能の詳細な説明を提供します。テーブル内のリンクをクリックして、各機能について詳しく学んでください。

{{% /alert %}}

|**機能** |**説明** |
| :- | :- |
|[XLSまたはXLSXエクスポート](/cells/ja/jasperreports/xls-or-xlsx-export/)|Microsoft Excel XLS形式としてレポートをエクスポートしますが、Microsoft Excelを利用せずにアプリケーションから。|
|[レポートデザインへの高い忠実度](/cells/ja/jasperreports/high-fidelity-to-the-report-design/)|XLS/XLSXレポートのレイアウトと外観がオリジナルと同じように一致し、PDFやその他の形式と似ています。|
|[簡単で軽量な展開](/cells/ja/jasperreports/easy-and-lightweight-deployment/)|Aspose.Cells for [JasperReports](https://community.jaspersoft.com/project/jasperreports-library)は、Microsoft Excel形式への[JasperReports](https://community.jaspersoft.com/project/jasperreports-library)およびJasperServerエクスポータです。|
|[シンプルで手頃な価格のライセンス](/cells/ja/jasperreports/simple-and-affordable-licensing/)|ソースコードで簡単に使用できる手頃な価格のライセンスを提供します。|
|[ワールドクラスの無料技術サポート](/cells/ja/jasperreports/world-class-free-technical-support/)|製品の開発者によって直接提供される無制限の技術サポートが提供されます。|
|[その他](/cells/ja/jasperreports/miscellaneous/)|ユーザーにとってより有効な機能が提供されます。|
