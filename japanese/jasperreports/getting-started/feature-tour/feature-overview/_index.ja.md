{{% alert color="primary" %}}

Aspose.Cells for JasperReports は、JasperReports および JasperServer からレポートを Microsoft Excel 形式にエクスポートするための製品です。 Aspose.Cells for JasperReports は組み込みのエクスポーターのすべての機能を継承し、JasperReports の元の XLS レンダラーのいくつかの弱点を補っています。このセクションでは、Aspose.Cells for JasperReports の機能について詳しく説明します。機能の詳細については、表内のそれぞれのリンクをクリックしてください。

{{% /alert %}}

|**特徴** |**説明** |
|:- |:- |
|[XLS または XLSX エクスポート](/cells/ja/jasperreports/xls-or-xlsx-export/)|Microsoft Excel を使用せずに、アプリケーションからレポートを Microsoft Excel XLS 形式でエクスポートします。|
|[レポートデザインへの高い忠実度](/cells/ja/jasperreports/high-fidelity-to-the-report-design/)|XLS/XLSX レポートのレイアウトと外観は、PDF やその他の形式と同様に、元のレポートと同じです。|
|[簡単で軽量な導入](/cells/ja/jasperreports/easy-and-lightweight-deployment/)| Aspose.Cells[ジャスパーレポート](https://community.jaspersoft.com/project/jasperreports-library)です[ジャスパーレポート](https://community.jaspersoft.com/project/jasperreports-library)JasperServer エクスポーターを Microsoft Excel 形式に変換します。|
|[シンプルで手頃なライセンス](/cells/ja/jasperreports/simple-and-affordable-licensing/)|ソースコードで簡単に使用できる手頃な価格のライセンスを提供します|
|[ワールドクラスの無料技術サポート](/cells/ja/jasperreports/world-class-free-technical-support/)|無制限のテクニカル サポートは、製品の開発者によって直接提供されます。|
|[その他](/cells/ja/jasperreports/miscellaneous/)|ユーザーにとってより効果的な機能が提供されます。|
