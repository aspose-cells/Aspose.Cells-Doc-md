{{% alert color="primary" %}}

Aspose.Cells for JasperReports は [JasperReports](https://community.jaspersoft.com/project/jasperreports-library) の外部エクスポータです。

Aspose.Cells for JasperReports は単一の JAR ファイルとして提供され、JasperReports を実行しているコンピューターに簡単にインストールできます。[ダウンロードしてトライアルを開始](https://downloads.aspose.com/cells/jasperreports)しましょう。

[エクスポータをインストールする](/cells/ja/jasperreports/installation/)には、**aspose.cells.jasperreports.jar** を **JasperReports\lib** ディレクトリにコピーする必要があります。

{{% /alert %}}
