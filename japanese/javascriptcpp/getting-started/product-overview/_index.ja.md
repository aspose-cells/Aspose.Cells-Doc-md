**Aspose.Cells for JavaScript via C++ ドキュメントセクションへようこそ**

Aspose.Cells for JavaScript via C++は、Office AutomationやMicrosoft Excelを必要とせずに、JavaScript/Node.js開発者がスプレッドシートをプログラムで作成、操作、変換できるネイティブWebAssemblyライブラリです。

このライブラリは、Excel（XLS、XLSX、XLSB、XLSM）、ODS、CSV、およびHTMLを含む主要なスプレッドシートフォーマットをサポートし、ブラウザまたはサーバーサイド環境で外部依存なしでスプレッドシート処理の完全なソリューションを提供します。

{{% /alert %}}

## **Benefits**

アプリケーションでAspose.Cells for JavaScript経由のC++を使用することにより、以下の利点があります：

- 高性能
- 豊富な機能
- 汎用性の高い互換性
- プラットフォームの独立性
- サーバーレス対応
- 簡単な統合

### **豊富な機能**

Aspose.Cells for JavaScript via C++は、スプレッドシートの作成、変換、操作のための幅広い機能を提供します。

- 組み込みのプロパティ
- カスタムプロパティ
- テーマ
- スタイルと書式
- データの検証
- 条件付き書式
- ハイパーリンク
- オートフィルター
- ページ設定
- 読み込み、書き込み、および数式の計算
- 行と列のグループ化
- ピボットテーブル
- テーブル

### **ブラウザとNode.jsをサポート**

最新のブラウザ（Chrome、Firefox、Edge、Safari）およびNode.jsランタイムと完全に互換性があります。
クライアント側とサーバー側のアプリケーションの両方で一貫した動作を保証し、シームレスな統合と実行を実現します。
