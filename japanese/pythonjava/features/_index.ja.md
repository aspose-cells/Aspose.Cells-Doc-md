このページは、Aspose.Cells for Python via Javaの機能を要約しています。

{{% /alert %}} 
## **ファイル形式と変換**
- XLS、SpreadsheetML（XLSX、XML）、Comma Separated Value（CSV）、HTML、OpenDocument Spreadsheet（ODS）、PDF、Tab Delimited、Plain Text（TXT）
## **レンダリングと印刷**
- スプレッドシートをベクトル画像（EMF）にレンダリング
- ワークシートをラスター画像（マルチページTIFF、GIF）にレンダリング
- ワークシートをラスター画像（PNG、JPEGなど）にレンダリング
- 高い信頼性でスプレッドシートをPDFにレンダリング
## **スプレッドシートのコンテンツ機能**
- ワークブック内またはワークブック間でワークシートをコピーまたは移動
- 画像の挿入、一般的なおよびカスタムチャートの作成、APIを使用したチャートのグラデーション背景の設定
- セルにコメントを追加
- Microsoft Excel 2007以降のテーマと色
- オートフィルターの作成
- データの検証を実装
- データのソートを実装
- テキストの検索および置換
- セルの結合/分割
- 行と列のグループ化/グループ解除
- カスタムページ区切りの作成
- 複雑なExcel数式の計算
- Microsoft Excel 2007以降でサポートされている高度な条件付き書式をサポート
- ウィンドウ枠の固定/解除
- データをリンクするためのハイパーリンクの挿入
- Smart Markersの実装
- ドキュメントプロパティ設定の指定
- ワークシートの保護/保護解除
- Excel XPおよびそれ以降のバージョンで導入された高度な保護オプションの指定
- リストオブジェクト/Microsoft Excel テーブルの作成
- サブトータルの作成
- フォームコントロールやその他の図形/オブジェクトの挿入
- ピボットテーブルとピボットチャートの作成
- アドイン、VBA、マクロの保護または削除
- 名前付き範囲の操作
- スプレッドシートからOLEオブジェクトの追加、保持、または抽出
- Microsoft Excelのスパークラインの実装
- ファイルの暗号化
## **スプレッドシートの書式設定機能**
- フォント、色、効果、罫線、シェーディングを含むセルのすべての文字の書式設定を適用
- セルの数値書式設定（Microsoft Excelでサポートされている）のすべての設定を適用
- すべての種類のテキスト配置設定を設定
- セルの異なる種類のフォント設定を適用
- セル、フォント、グリッド線、グラフィックオブジェクトなどに異なる色を適用
- 単一のセルで異なるリッチテキストの書式設定を適用
- セルに異なる境界設定を適用
- セルに異なる背景パターンを適用
- ワークシート、行、列、またはセル範囲などにフォーマット設定を適用
## **ページ設定機能**
- ページの向き、拡大率、用紙サイズを調整
- 余白とページの中央揃えの指定
- ヘッダーやフッターの作成または編集
- 印刷エリアを設定したり、タイトルを印刷したり、グリッド線を表示したりするなど。
