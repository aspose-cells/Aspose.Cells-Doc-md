## **Excel を PDF にレンダリングしながらスライサーを描画する**
Aspose.Cells for Python via Java は、スライサー設定で Excel から PDF へのエクスポートをサポートします。 API はデフォルトでこれをサポートします。スライサーを含む Excel ファイルを PDF にエクスポートするだけで、生成された PDF にスライサーが適用されたことが表示されます。次のコード スニペットは、[サンプル Excel ファイル](106365058.xlsx)スライサーを含み、ワークブックを[出力PDFファイル](106365052.xlsx).次のスクリーンショットは、ソースの Excel ファイルと生成された PDF ファイルを比較しています。
