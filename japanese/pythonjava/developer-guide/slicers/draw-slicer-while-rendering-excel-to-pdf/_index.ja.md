## **ExcelをPDFにレンダリングする際にスライサーを描画する**
Aspose.Cells for Python via Java は、スライサー設定を含むExcelをPDFにエクスポートするのをサポートしています。APIはこれをデフォルトでサポートしており、スライサー付きのExcelファイルを単純にPDFにエクスポートすると、生成されたPDFに適用されたスライサーが表示されます。
