## **レンダリング スライサー**
Aspose.Cells for Python via Java は、スライサー形状のレンダリングをサポートします。次のコード スニペットは、[サンプル Excel ファイル](106364974.xlsx)スライサーが含まれています。スライサーのみをカバーする印刷領域を設定することで、ワークシートを画像に変換します。流れるイメージは[出力画像](106364975.png)レンダリングされたスライサーを示しています。ご覧のとおり、スライサーは適切にレンダリングされており、サンプルの Excel ファイルと同じように見えます。
