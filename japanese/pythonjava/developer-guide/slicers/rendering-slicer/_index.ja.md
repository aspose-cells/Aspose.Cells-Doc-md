## **スライサーのレンダリング**
Aspose.Cells for Python via Java は、スライサー形状のレンダリングをサポートしています。次のコードスニペットは、スライサーを含む[sample Excel file](106364974.xlsx)をロードし、スライサーのみをカバーする印刷エリアを設定してワークシートを画像に変換します。生成された画像は、レンダリングされたスライサーを示す[output image](106364975.png)です。スライサーが正しくレンダリングされており、サンプルExcelファイルと同じように見えます。
