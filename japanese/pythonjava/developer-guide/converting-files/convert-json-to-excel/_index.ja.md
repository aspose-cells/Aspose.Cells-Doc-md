Aspose.Cells for Python via Java は、Json（JavaScript Object Notation）ファイルを Excel Workbook に変換する機能をサポートしています。

{{% /alert %}}

## **JSONをExcelワークブックに変換する**
Aspose.Cells for Python via Java ライブラリには最適な選択肢が用意されているため、JSON を Excel ファイルに変換する方法を疑問に思う必要はありません。Aspose.Cells for Python via Java API では、JSON フォーマットをスプレッドシートに変換するサポートが提供されています。JSON をワークブックにインポートするための追加設定を指定するために [**JsonLoadOptions**](https://reference.aspose.com/cells/python-java/asposecells.api/JsonLoadOptions) クラスを使用できます。

以下のコード例は、JSON を Excel Workbook にインポートする方法を示しています。参照のためにコードを確認して、[ソースファイル](sample.json) をコードによって生成されたファイルに変換する方法を参照してください。
