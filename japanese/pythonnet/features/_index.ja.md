このページは Aspose.Cells for Python via .NET の特徴をまとめたものです。

{{% /alert %}} 
## **ファイル形式と変換**
- XLS、SpreadsheetML (XLSX、XML)、XLSB、XLSM、DOCX、PPTX、JSON、カンマ区切り値 (CSV)、HTML、OpenDocument スプレッドシート (ODS)、PDF、タブ区切り、プレーン テキスト (TXT)、および TIFF を含む一般的な画像形式、JPG、PNG、BMP、および SVG。
## **レンダリングと印刷**
- スプレッドシートをベクター画像 (EMF) にレンダリングする
- ワークシートをラスター イメージにレンダリング (マルチページ TIFF、GIF)
- ワークシートをラスター イメージ (PNG、JPEG など) にレンダリングします。
- スプレッドシートを高忠実度で PDF にレンダリング
## **スプレッドシートのコンテンツ機能**
- ワークブック内またはワークブック間でワークシートをコピーまたは移動する
- API を使用して、画像を挿入し、従来のグラフとカスタム グラフを作成し、グラフのグラデーション背景を設定します。
- セルにコメントを追加する
- Microsoft Excel 2007 以降のバージョンのテーマと色
- 自動フィルターを作成する
- データ検証を実装する
- データの並べ替えを実装する
- テキストの検索と置換
- 合流・分割 Cells
- 行と列のグループ化/グループ解除
- カスタム改ページを作成する
- 複雑な Excel 数式を計算する
- Microsoft Excel 2007 以降のバージョンでサポートされている高度な条件付き書式をサポートします
- ペインのフリーズ/フリーズ解除
- ハイパーリンクを挿入してデータをリンクする
- スマート マーカーの実装
- ドキュメントのプロパティ設定を指定する
- ワークシートの保護/保護解除
- Excel XP 以降のバージョンで導入された高度な保護オプションを指定する
- リスト オブジェクトの作成/Microsoft Excel テーブル
- 小計の作成
- フォーム コントロールおよびその他の描画図形/オブジェクトを挿入する
- ピボット テーブルとピボット チャートを作成する
- アドイン、VBA、マクロを保持または削除する
- 名前付き範囲を操作する
- スプレッドシートから OLE オブジェクトを追加、保存、または抽出します。
- Microsoft Excel スパークラインを実装する
- ファイルを暗号化する
## **スプレッドシートの書式設定機能**
- フォント、色、効果、境界線、陰影など、セル内のすべての文字書式を適用します
- セルにすべての数値形式設定 (Microsoft Excel でサポート) を適用します。
- すべての種類のテキスト配置設定を設定する
- セルにさまざまな種類のフォント設定を適用する
- セル、フォント、グリッド線、グラフィック オブジェクトなどにさまざまな色を適用します。
- つのセルにさまざまなリッチ テキスト書式を適用する
- セルに異なる境界線設定を適用する
- セルに異なる背景パターンを適用する
- ワークシート、行、列、またはセルの範囲などに書式設定を適用します。
## **PageSetup 機能**
- ページの向き、スケーリング、用紙サイズを調整する
- 余白とページの中央揃えを指定する
- ヘッダーやフッターを作成または編集する
- 印刷範囲の設定、タイトルの印刷、グリッド線のオンなど。
