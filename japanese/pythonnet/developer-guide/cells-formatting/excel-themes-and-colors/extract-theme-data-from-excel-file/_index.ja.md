{{% alert color="primary" %}}

Aspose.Cells for Python via .NETは、Excelファイルからテーマ関連のデータを抽出する機能を提供します。例えば、ブックに適用されたテーマ名やセルやセルの罫線に適用されたテーマカラーなどを抽出できます。

Microsoft Excelを使用してワークブックにテーマを適用できます。[ページレイアウト] > [テーマ]コマンドを使用します。

{{% /alert %}}
