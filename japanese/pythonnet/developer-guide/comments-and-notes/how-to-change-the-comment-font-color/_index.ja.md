{{% alert color="primary" %}}

Microsoft Excelでは、セルへのコメント追加が可能で、追加情報を記載したりデータを強調できます。開発者は、整列設定、テキスト方向、フォント色などをカスタマイズする必要があります。Aspose.Cells for Python via .NETは、そのためのAPIを提供します。

{{% /alert %}}

Aspose.Cells for Python via .NETは、コメントのフォントの色を設定する[**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body)プロパティを提供しています。以下のサンプルコードは、[**Shape.text_body**](https://reference.aspose.com/cells/python-net/aspose.cells.drawing/shape/text_body)プロパティを使用してコメントのテキスト方向を設定する例です。
