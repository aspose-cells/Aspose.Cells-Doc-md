PDF 論理構造機能は、文書コンテンツ構造に関する情報を PDF ファイルに組み込むメカニズムを提供します。 Aspose.Cells for Python via .NET は、セル、行、テーブル、ワークシート、画像、図形、ヘッダー/フッターなど、Microsoft Excel ドキュメントの構造に関する情報を保存します。

オプション [PdfSaveOptions.export_document_struction]https://reference.aspose.com/cells/python-net/aspose.cells/pdfsaveoptions/export_document_struction/) を使用すると、エクスポートされたドキュメント構造をタグ付き PDF に保存できます。
