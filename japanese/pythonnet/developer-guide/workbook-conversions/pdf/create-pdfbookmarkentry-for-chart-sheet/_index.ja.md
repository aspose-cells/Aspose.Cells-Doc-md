##  **考えられる使用シナリオ**

以前は、Aspose.Cells for Python via .NET が作成されていました。[**PDFブックマークエントリ**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/)ノーマルシートの場合。しかし、現在は Aspose.Cells for Python via .NET も作成できます[**PDFブックマークエントリ**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/)チャートシート用。グラフシートにはセルA1以外のセルがないため、[**PDFブックマークエントリ**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/pdfbookmarkentry/)セル A1 のみ。

##  **グラフ シートの PdfBookmarkEntry を作成する**

次のサンプルコードは、[サンプル Excel ファイル](61767756.xlsx) 4枚のシートがあります。そのうちの 2 枚は通常のシートで、残りの 2 枚はチャート シートです。次のように 4 つのブックマーク エントリを作成します。

- ブックマーク-I
- ブックマーク-II-チャート1
- ブックマーク-III
- ブックマーク-IV-チャート2
