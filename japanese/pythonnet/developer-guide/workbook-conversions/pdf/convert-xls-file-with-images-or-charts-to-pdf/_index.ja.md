Aspose.Cells for Python via .NET は、画像とグラフを含む XLS ファイルから PDF ドキュメントへの変換をサポートしています。 Aspose.Cells for Python via .NET API は独立して機能して、スプレッドシートを PDF に変換できます。変換には Aspose.PDF for .NET は必要ありません。このプロセスは一時ファイルや中間 XML ファイルに依存しないため、メモリ内で実行できます。これは、画像、グラフ、その他の描画オブジェクトを含む大きな Excel ファイルを迅速かつ効率的に変換できることを意味します。
