##  **考えられる使用シナリオ**

以前は、Excel ファイルが PDF 形式で保存されている場合、Aspose.Cells for Python via .NET は Office アドインを表示できませんでした。 Aspose.Cells for Python via .NET で問題なく表示されるようになりました。出力 PDF で Office アドインをレンダリングするために特別なメソッドやプロパティを使用する必要はありません。Excel ファイルを PDF 形式で保存するだけで、Office アドインがレンダリングされます。
