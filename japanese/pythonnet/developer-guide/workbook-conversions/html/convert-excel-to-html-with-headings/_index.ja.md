##  **Excelを見出し付きでHTMLに変換**

Aspose.Cells for Python via NET では、Excel を HTML に変換するときに見出しをエクスポートするオプションが提供されます。このために、[**HtmlSaveOptions.export_row_column_Headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)プロパティを *true** に設定します。のデフォルト値[**HtmlSaveOptions.export_row_column_Headings**](https://reference.aspose.com/cells/python-net/aspose.cells/htmlsaveoptions/export_row_column_headings/)プロパティは *false** です。次の画像は、次のコードによって生成された出力ファイルを示しています。
