{{% alert color="primary" %}}

ウォーターフォールチャートは、通常、出発位置が増加または減少する方法を示すために使用される特別なタイプのチャートです。Microsoft Excelには、カラム、折れ線、円、棒、レーダーなどの事前定義済みのチャートタイプが多数ありますが、ウォーターフォールチャートは基本的なグラフを超えたものであり、既存のチャートタイプを使用して少しのカスタマイズまたはより多くのカスタマイズで作成できます。

{{% /alert %}} 

Aspose.Cells for Python via .NET APIは、ラインチャートを利用してウォーターフォールチャートを作成できます。また、チャートの外観をカスタマイズし、[**Series.up_bars**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/series/up_bars)と[**Series.down_bars**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/series/down_bars)のプロパティを設定することでウォーターフォールの形状に調整可能です。
