Microsoft Excel 2013では、データラベルにセル範囲を表示できます。Aspose.Cells for Python via .NETはこの機能をサポートしています。

{{% /alert %}}

## **セル範囲をデータラベルとして表示するためのチェックボックス**

Microsoft Excelでセルの範囲をデータラベルとして表示するには：

1. シリーズのデータラベルを選択して、コンテキストメニューを開くために右クリックします。
1. **データラベルの書式設定**を選択します。ラベルのオプションが表示されます。
1. **ラベルに値を含める**オプションを選択またはクリアします。

以下のサンプルコードは、チャートシリーズのデータラベルにアクセスし、**true**に**ラベルに値を含める**オプションを選択するための[**DataLabels.show_cell_range**](https://reference.aspose.com/cells/python-net/aspose.cells.charts/datalabels/show_cell_range)プロパティを設定します。
