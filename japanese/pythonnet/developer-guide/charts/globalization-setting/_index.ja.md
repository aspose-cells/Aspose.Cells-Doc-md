このトピックでは、チャートをローカライズされた画像に変換し、チャートのローカライズされた地域を設定する方法を説明します。

{{% /alert %}}

## **シナリオ**

チャートのローカライズされた地域を設定する必要があるのはいつですか？

Excelでスペインの地域設定を使用してチャートを含むXLSXファイルを開くと、チャートのタイトルや凡例などの要素がスペイン語で表示されます。ただし、Aspose.Cellsを使用してこのチャートを画像として保存すると、これらの要素はデフォルトで英語のままになる場合があります：

**! [グローバル・イシュー](GlobalIssue.png)**

これは、出力画像内のチャート凡例がExcelのローカリゼーションと一致しないために発生します。これを解決するには、チャートのローカライズされた地域設定を構成します。

## **サポートされている要素**

以下のチャート要素はローカリゼーション設定に従って表示されます：

| **対応要素**      | **デフォルト値（英語）**       |
|-----------------------------|-----------------------------------|
| 軸タイトル名             | 軸タイトル                        |
| 軸ユニット名             | 百、千...                        |
| チャートタイトル名        | チャートタイトル                |
| 凡例増加名                | 増加                              |
| 凡例減少名                | 減少                              |
| 凡例合計名                | 合計                              |
| その他の名前              | その他                            |
| シリーズ名                | シリーズ                        |
