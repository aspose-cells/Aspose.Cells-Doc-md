##  **Excel を PDF にレンダリングするときにスライサーを描画する**
スライサーが適用された Excel ファイルがあり、スライサー設定を使用して Excel を PDF にエクスポートする場合、Aspose.Cells for Python via .NET はこれをデフォルトでサポートするようになりました。スライサーを含む Excel ファイルを PDF にエクスポートするだけで、生成された PDF にはスライサーが適用されたことが表示されます。

次のサンプルコードは、[サンプル Excel ファイル](94044165.xlsx)既存のスライサーが含まれています。次に、ワークブックを次のように保存します。[出力PDFファイル](94044166.pdf)。次のスクリーンショットは、ソース Excel ファイルと生成された PDF ファイルを比較しています。
