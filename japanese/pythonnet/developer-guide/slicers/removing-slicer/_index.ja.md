##  **考えられる使用シナリオ**

Microsoft Excel でスライサーを削除したい場合は、それを選択して、*消去*ボタン。同様に、プログラムで Aspose.Cells for Python via .NET API を使用して削除する場合は、[**Worksheet.slicers.remove()**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicercollection/)方法。ワークシートからスライサーが削除されます。
