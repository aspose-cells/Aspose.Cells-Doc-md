##  **考えられる使用シナリオ**

Microsoft Excel では、列数を設定したり、スタイルを設定したりすることで、スライサーを書式設定できます。 Aspose.Cells for Python via .NET では、[**スライサー.number_of_columns**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/number_of_columns/)そして[**スライサー.style_type**](https://reference.aspose.com/cells/python-net/aspose.cells.slicers/slicer/style_type/)プロパティ。
