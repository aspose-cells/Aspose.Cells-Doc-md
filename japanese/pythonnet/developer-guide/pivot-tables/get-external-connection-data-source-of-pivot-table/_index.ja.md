## **Pivot Table の外部接続データソースの取得方法**

Aspose.Cells for Python via .NET は、Pivot Table の外部接続データソースを取得する機能を提供します。このために、API は [**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) クラスの [**PivotTable**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/) プロパティを提供します。[**external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) プロパティは [**ExternalConnection**](https://reference.aspose.com/cells/python-net/aspose.cells.externalconnections/externalconnection/) オブジェクトを返します。次のコードスニペットは、Pivot Table の外部接続データソースを取得するために [**PivotTable.external_connection_data_source**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/external_connection_data_source/) プロパティを使用する方法を示しています。
