{{% alert color="primary" %}}

場合によっては、ピボット テーブルのセルを書式設定したい場合があります。たとえば、ピボット テーブルのセルに背景色を適用するとします。 Aspose.Cells for Python via .NET には 2 つの方法があります[**PivotTable.format_all(スタイル)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format_all/)そして[**PivotTable.format(行、列、スタイル)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format/)、この目的に使用できます。

[**PivotTable.format_all(スタイル)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format_all/)スタイルをピボット テーブル全体に適用します。[**PivotTable.format(行、列、スタイル)**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/format/)スタイルをピボット テーブルの単一のセルに適用します。

{{% /alert %}}
