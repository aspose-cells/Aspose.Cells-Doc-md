##  **考えられる使用シナリオ**

Microsoft Excel では、ピボット テーブルのピボット フィールドをグループ化できます。ピボット フィールドに関連するデータが大量にある場合、多くの場合、データをセクションにグループ化すると便利です。 Aspose.Cells for Python via .NET も、[**PivotTable.set_manual_group_field()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/set_manual_group_field/#int-float-float-list-float)方法。

##  **ピボット テーブルのグループ ピボット フィールド**

次のサンプルコードは、[サンプル Excel ファイル](64716818.xlsx)そして、最初のピボットフィールドでグループ化を実行します。[**PivotTable.set_manual_group_field()**](https://reference.aspose.com/cells/python-net/aspose.cells.pivot/pivottable/set_manual_group_field/#int-float-float-list-float)方法。次に、ピボット テーブルのデータを更新して計算し、ワークブックを次の名前で保存します。[Excelファイルを出力](64716817.xlsx)。スクリーンショットは、サンプル Excel ファイルに対するサンプル コードの効果を示しています。スクリーンショットでわかるように、最初のピボット フィールドは月と四半期ごとにグループ化されています。
