Aspose.Cells for Python via .NETは、正規表現を使用してワークブック内のテキストを置き換える機能を提供します。これには、APIの[**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key)プロパティと[**ReplaceOptions**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions)クラスがあります。[**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key)を**true**に設定すると、検索キーが正規表現であることを示します。

次のコードスニペットは、[**ReplaceOptions.regex_key**](https://reference.aspose.com/cells/python-net/aspose.cells/replaceoptions/regex_key) プロパティの使用例を示しています。[サンプルExcelファイル](101089318.xlsx)を使用して、このプロパティを使用したコードによって生成された[出力ファイル](101089319.xlsx)が添付されています。
