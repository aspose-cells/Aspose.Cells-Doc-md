{{% alert color="primary" %}} 

ハイパーリンクは、他のワークシートやウェブサイトの情報にアクセスするために便利です。Microsoft Excelでは、セル内のテキストや画像にハイパーリンクを追加することができます。画像ハイパーリンクは、ワークシートのナビゲーションを容易にするためのものであり、例えば、「次へ」「前へ」ボタンや特定のサイトへリンクするロゴなどがあります。このドキュメントでは、Aspose.Cells for Python via .NETを使用してワークシートに画像ハイパーリンクを挿入する方法について説明します。

{{% /alert %}} 

Aspose.Cells for Python via .NETを使用して、スプレッドシート内の画像にハイパーリンクを追加することができます。リンクのスクリーンチップやアドレスを設定および変更することが可能です。次のサンプルコードは、ワークシートに画像ハイパーリンクを追加する方法を示しています。
