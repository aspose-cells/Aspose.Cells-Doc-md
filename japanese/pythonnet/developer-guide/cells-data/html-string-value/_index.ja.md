{{% alert color="primary" %}}

Aspose.Cells for Python via .NET は、Microsoft Excel 向けの HTML を XLS/XLSX 形式に変換する機能をサポートしています。つまり、Microsoft Excel によって生成された HTML を Aspose.Cells for Python via .NET を使用して XLS/XLSX 形式に変換することができます。

同様に、簡単な HTML がある場合は、Aspose.Cells for Python via .NET が HTML リッチテキストに変換できます。Aspose.Cells for Python via .NET は、[**Cell.html_string**](https://reference.aspose.com/cells/python-net/aspose.cells/cell/html_string/) プロパティを提供し、このような簡単な HTML を取得して、フォーマットされたセルのテキストに変換できます。

{{% /alert %}}
