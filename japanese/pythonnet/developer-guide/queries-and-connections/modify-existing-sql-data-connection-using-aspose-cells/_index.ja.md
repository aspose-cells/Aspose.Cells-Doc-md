Aspose.Cells for Python via .NET は既存の SQL データ接続の変更をサポートします。この記事では、Aspose.Cells for Python via .NET を使用して SQL データ接続のさまざまなプロパティを変更する方法を説明します。

Microsoft Excel内のデータ接続を追加または参照するには、**データ > 接続** メニューコマンドに従ってください。

同様に、Aspose.Cells for Python via .NET は Workbook.DataConnections コレクションを使用してデータコネクションにアクセスおよび変更する手段を提供します。
