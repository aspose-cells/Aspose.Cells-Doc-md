## **可能な使用シナリオ**

Aspose.Cells for Python via .NETを使い、[**ImageOrPrintOptions.page_index**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/page_index)と[**ImageOrPrintOptions.page_count**](https://reference.aspose.com/cells/python-net/aspose.cells.rendering/imageorprintoptions/page_count)プロパティを用いてExcelファイルのページシーケンスを画像にレンダリングできます。これらのプロパティは、ページ数が多い場合に、いくつかだけをレンダリングしたいときに役立ちます。処理時間とレンダリングのメモリ消費も節約できます。
