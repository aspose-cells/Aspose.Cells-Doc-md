## **可能な使用シナリオ**

一般的に、すべてのワークブックに対してフォントのディレクトリまたはフォントの一覧を指定しますが、場合によっては個別またはプライベートなフォントセットを指定する必要があります。Aspose.Cells for Python via .NETは、ワークブック用に個別またはプライベートのフォントセットを指定できる[**IndividualFontConfigs**](https://reference.aspose.com/cells/python-net/aspose.cells/individualfontconfigs)クラスを提供します。
