{{% alert color="primary" %}}

Aspose.Cellsは、Microsoft Excel向けのHTMLをXLS/XLSXフォーマットに変換できることをサポートします。つまり、Microsoft Excelが生成したHTMLをAspose.Cellsを使ってXLS/XLSXに再変換できます。

同様に、簡単なHTMLがある場合、Aspose.CellsはそれをHTMLリッチテキストに変換できます。Aspose.Cellsは、[**Cell.setHtmlString(string)**](https://reference.aspose.com/cells/nodejs-cpp/cell/#setHtmlString-string-) メソッドを提供しており、そのようなシンプルなHTMLを取り込み、フォーマットされたセルテキストに変換します。

{{% /alert %}}
