{{% alert color="primary" %}} 

ハイパーリンクは他のワークシートやウェブサイトの情報へのアクセスに便利です。Microsoft Excelは、セル内のテキストや画像にハイパーリンクを追加でき、これによりナビゲーションが簡単になります。例えば、次へ・前へボタンや特定のサイトにリンクしたロゴなどです。この記事では、Aspose.Cells for Node.js via C++を使用してワークシートに画像ハイパーリンクを挿入する方法を解説します。

{{% /alert %}}
