## **Aspose.Cells for Node.js via C++ライブラリを使用してピボットテーブルのカスタム並べ替えを設定する方法。**
Aspose.Cells for Node.js via C++ APIを使用して、ピボットテーブルのフィールド値で並べ替えが可能です。以下のコードスニペットは、サンプルExcelファイルを読み込み、3つのピボットテーブルを追加します。最初のピボットテーブルはカスタム並べ替えなし、2つ目は「SeaFood」行フィールド値で並べ替え、3つ目は「28/07/2000」列フィールド値で並べ替えられます。
