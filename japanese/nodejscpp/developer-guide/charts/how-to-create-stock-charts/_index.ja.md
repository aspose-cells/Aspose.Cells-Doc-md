- **VOHLC** – 出来高-始値-高値-安値-終値。

## **株価チャートとは何ですか？**

株価チャートは、商品、株式、暗号通貨などの取引資産の価格変動を追跡するための特定のチャートです。これにより、時間経過における高値と安値、始値と終値を一つのチャートで見ることができます。Aspose.Cells for Node.js via C++は4つの株価チャートを提供し、それらを使用するには適切なデータセットを用意し、正しい順序で列を選択する必要があります。

以下のデータセットは、株式の日次取引情報を示します。このデータを使用して、4つのタイプの株式チャートを作成します：ハイロークローズ（HLC）、オープンハイロークローズ（OHLC）、ボリュームハイロークローズ（VHLC）、およびボリュームオープンハイロークローズ（VOHLC）。
