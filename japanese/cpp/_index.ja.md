**Aspose.Cells for C++ へようこそ**

Aspose.Cells for C++ は、Office Automation や Microsoft Excel アプリケーションを必要とせずに、C++ デベロッパーがプログラムでスプレッドシートを作成、操作、変換できるようにするネイティブの C++ ライブラリです。Aspose.Cells for C++ は、MS Excel 97-2003 (XLS)、MS Excel 2007-2013/2016 (XLSX、XLSM、XLSB など)、Open Office XML などのスプレッドシート形式をサポートしています。また、ワークシートから画像をレンダリングしたり、MS Excel スプレッドシートを PDF 形式に変換したりすることもできます。API はまた、チャートや図形の作成、操作、レンダリングを許可します。さらに、コンポーネントによって提供される数式計算も利用できます。
