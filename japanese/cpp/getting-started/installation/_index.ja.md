##  **Aspose.Cells for C++ を Windows にインストールする**
###  **Nuget パッケージマネージャーの使用**
1. Nuget パッケージ マネージャーで、Aspose.Cells を検索します。
 1. ここにあります[NuGet パッケージ Aspose.Cells.Cpp へのリンク](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. 詳細については、[インストール Aspose Cells ～ NuGet] を参照してください。(https://docs.aspose.com/cells/cpp/getting-started/)。

Aspose.Cells for C++ から ZIP パッケージをダウンロードすることもできます。[ダウンロードページ](https://downloads.aspose.com/cells/cpp/)。アーカイブには、オフライン API ドキュメントも含まれています。
###  **Include フォルダーと lib フォルダーの使用**
1. ダウンロード/インストールします。[最新のファイル](https://downloads.aspose.com/cells/cpp/).
1. フォルダーを実稼働環境に解凍します。
1. Aspose.Cells for C++ を使用するには、プロジェクト内の「Include」ディレクトリと「lib / dll」ディレクトリを参照します。
