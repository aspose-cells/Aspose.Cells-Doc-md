## **Windows に Aspose.Cells for C++ をインストールする**
### **Nuget パッケージ マネージャーの使用**
1. Nuget パッケージ マネージャーで、Aspose.Cells を検索します。
 1. ここに[NuGet パッケージ Aspose.Cells.Cpp へのリンク](https://www.nuget.org/packages/Aspose.Cells.Cpp).
1. 詳細については、[Aspose Cells から NuGet をインストールする] (https://docs.aspose.com/cells/cpp/getting-started/) を参照してください。

Aspose.Cells for C++ から ZIP パッケージをダウンロードすることもできます。[ダウンロードページ](https://downloads.aspose.com/cells/cpp/).アーカイブには、オフラインの API ドキュメントも含まれています。
### **include および lib フォルダーの使用**
1. ダウンロード/インストール[最新のファイル](https://downloads.aspose.com/cells/cpp/).
1. フォルダを本番環境に解凍します。
1. Aspose.Cells for C++ を使用するには、プロジェクト内の「Include」および「lib」ディレクトリを参照してください。

## **Linux に Aspose.Cells for C++ をインストールする**
1. 次のページから Aspose.Cells for C++ をダウンロードします。[ダウンロード Aspose.Cells for C++(Linux)](https://downloads.aspose.com/cells/cpp)
2. パッケージを解凍すると、Linux で Aspose.Cells for C++ を使用する方法に関するデモが見つかります。
3. Linux コマンド ラインで「cd Demo」を実行します。
4. 「rm -rf build;mkdir build;cd build」を実行します
5. 「cmake ..」を実行すると、Demo フォルダに CMakeLists.txt で Makefile が作成されます
6.「make」を実行してコンパイルします
7. 「./demo」を実行すると、結果が表示されます
