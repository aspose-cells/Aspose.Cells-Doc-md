## 開発環境

Windows または Linux 用のアプリケーションを開発する場合は、Aspose.Cells for C++ を使用できます。

### Windows

 Aspose.Cells for C++ は、サポートされている任意の開発環境でアプリケーションを開発するために使用できます。[Microsoft Visual Studio v142 プラットフォーム ツールセット](https://docs.microsoft.com/en-us/cpp/porting/binary-compat-2015-2017?view=msvc-160)、ただし、次の表にリストされている環境は明示的にサポートされています。

<table>  
	<tr>
			<td style="font-weight: bold; width:800px">開発環境</td>
		</tr>
  <tr>

			</tr>
</table>

### Linux への追加の依存関係

Aspose.Cells Linux の for C++ は以下に依存します[libgdiplus](https://github.com/mono/libgdiplus//)動的ライブラリとツールの両方のバイナリ。使用する前にインストールしてください:

- Ubuntu または Debian への libgdiplus のインストール</br>
`sudo apt-get install libgif-dev autoconf libtool automake build-essential gettext libglib2.0-dev libcairo2-dev libtiff-dev libexif-dev`
