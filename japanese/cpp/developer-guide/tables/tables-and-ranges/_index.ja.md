##  **導入**
Microsoft Excel でテーブルを作成し、付属のテーブル機能を使い続けたくない場合があります。代わりに、テーブルのように見えるものが必要です。書式設定を失わずにデータをテーブルに保持するには、テーブルを通常のデータ範囲に変換します。 Aspose.Cells は、テーブルおよびリスト オブジェクトに対して Microsoft Excel のこの機能をサポートしています。
##  **Microsoft Excelの使用**
使用**範囲に変換**書式設定を失わずにテーブルを範囲にすばやく変換する機能。 Microsoft Excel 2007/2010:

1. テーブル内の任意の場所をクリックして、アクティブ セルがテーブルの列にあることを確認します。
1. で**デザイン**タブの**ツール**グループで、*範囲に変換** をクリックします。

{{% alert color="primary" %}} 

テーブルが範囲に変換されると、テーブル機能は使用できなくなります。たとえば、行ヘッダーには並べ替えとフィルターの矢印が含まれなくなり、数式で使用されていた構造化参照 (テーブル名を使用する参照) は通常のセル参照に変わります。

{{% /alert %}}
