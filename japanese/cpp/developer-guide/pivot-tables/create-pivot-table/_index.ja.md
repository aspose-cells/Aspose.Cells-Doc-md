## **紹介**
ワークシート内でピボットテーブルを作成するには、[PivotTableCollection.Add()](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottablecollection/add/) メソッドを使用してください。ピボットテーブルが作成されたら、[PivotTable](https://reference.aspose.com/cells/cpp/aspose.cells.pivot/pivottable/) クラスを使用して操作できます。
## **ピボットテーブルの作成**
