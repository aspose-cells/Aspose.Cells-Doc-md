### **サーバー要件**
Aspose.Cells for SharePoint をインストールするには、次のいずれかを実行しているコンピューターが必要です。

- Windows SharePoint サービス 3.0
- Microsoft オフィス SharePoint サーバー 2007
- Microsoft SharePoint サーバー 2010
- Microsoft SharePoint サーバー 2013

{{% alert color="primary" %}} 

これらの製品には Service Pack 2 以降をインストールすることをお勧めします。

{{% /alert %}}
### **クライアント要件**
SharePoint サイトにアクセスするにはブラウザーが必要です。 Aspose.Cells for SharePoint は以下でテストされています。

- Microsoft インターネット エクスプローラー
- モジラ ファイアフォックス
- Google クロム
- サファリ
