{{% alert color="primary" %}} 

Aspose.Cells for SharePoint をアンインストールするには、セットアップ プログラムを実行します。 Aspose.Cells for SharePoint が既にインストールされている場合、セットアップ プログラムはそれを削除または修復することを提案します。

{{% /alert %}} 
### **アンインストール**
削除中、セットアップ プログラムはすべてのサイト コレクションの Aspose.Cells for SharePoint を非アクティブ化し、サーバー ファームからソリューションを撤回します。
