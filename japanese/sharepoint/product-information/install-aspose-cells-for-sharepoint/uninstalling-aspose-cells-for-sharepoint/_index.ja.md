{{% alert color="primary" %}} 

Aspose.Cells for SharePointをアンインストールするには、セットアッププログラムを実行します。Aspose.Cells for SharePointがすでにインストールされている場合、セットアッププログラムは削除または修復を提案します。

{{% /alert %}} 
### **アンインストール**
削除中、セットアッププログラムはすべてのサイトコレクションからAspose.Cells for SharePointを非アクティブ化し、ソリューションをサーバーファームから取り消します。 

**Aspose.Cellsのアンインストール** 

![todo:image_alt_text](uninstalling-aspose-cells-for-sharepoint_1.png)




{{% alert color="primary" %}} 

必ずしも**修復**オプションを実行しても、Aspose.Cells for SharePointをサイトコレクションで再アクティブ化するわけではありません。代わりに、**修復**オプションの後に機能を再展開してから、手動でアクティブ化する必要があります。

 **削除**オプションを使用してアンインストールし、その後再インストールする方が便利かもしれません。これにより、インストール中にターゲットのWebアプリケーションとサイトコレクションを選択できます。 

{{% /alert %}}
