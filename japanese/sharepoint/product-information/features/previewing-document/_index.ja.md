Aspose.Cells for Share Point は、SharePoint から直接ドキュメントをプレビューすることをサポートしています。次の形式はプレビューをサポートしています。

- XLS – Microsoft エクセル 97 - 2003
- XLSX – Office オープン XML
- XLSM、XLTX、XLTM – オフィス オープン XML
- XML – Microsoft Excel スプレッドシートML
