Aspose.Cells for SharePointには、いくつかの貴重なオプションとコマンドが含まれるツールバーが提供されています:

- Aspose.Cellsで変換
- Aspose.Cellsでドキュメントをプレビュー
- Aspose.Cellsでドキュメント内を検索
- Aspose.Cellsでウォーターマークを追加
- Aspose.Cellsでフッターを追加
