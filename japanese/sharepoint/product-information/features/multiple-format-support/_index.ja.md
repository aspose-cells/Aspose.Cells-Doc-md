{{% alert color="primary" %}}

Aspose.Cells for SharePointを使用して、SharePointドキュメントライブラリ間で多くの人気のあるオフィス文書形式を変換できます。ほとんどの変換は高い忠実度と精度で行われます。

{{% /alert %}}

## **サポートされている入力形式**

Aspose.Cells for SharePointは次の入力形式をサポートしています:

- XLS – Microsoft Excel 97~2003
- XLSX – Office Open XML
- XLSM, XLTX, XLTM – Office Open XML
- CSV – カンマ区切り値のワークシートドキュメント
- ODS – OpenDocument
- XML – Microsoft Excel SpreadsheetML

文書を生成するためには、Aspose.Cells for SharePointはAsposeのトップ市場の表計算処理コンポーネント[Aspose.Cells for .NET](https://products.aspose.com/cells/net/)の組み込みバージョンに依存しています。Microsoft Excelは文書を生成するために使用されません。

## **サポートされている出力形式**

Aspose.Cells for SharePointがサポートしている出力フォーマットの完全なリストは次のとおりです：

- PDF – Adobe Portable Document
- XLS – Microsoft Excel 97 - 2003
- XLSX – Office Open XML
- XLSM – Office Open XML for macros enabled
- TXT – Plain Text
- XML – Microsoft Excel SpreadsheetML
- ODS – OpenDocument
- CSV – Comma Separated Values
- XLSB – Microsoft Excel Binary 2007
- MHTML – Multipurpose Internet Mail Extension HTML
- XPS – XML Paper Specification
- TIFF – Tagged Image File Format
- SVG – Scalable Vector Graphics
- DIF – Drawing Interchange Format
