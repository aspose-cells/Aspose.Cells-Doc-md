### **SharePoint ドキュメント ライブラリに関する簡単なメモ**
SharePoint サイトのコンテンツは、SharePoint ドキュメント ライブラリ (「共有ドキュメント ライブラリ」とも呼ばれる) と呼ばれる一元化された場所で共有されるドキュメントの形式になっています。許可されたユーザーは、ローカル マシンから SharePoint サイトにドキュメントをアップロードできます。これらのドキュメントは、SharePoint サイトで使用される SQL Server データベースにバイナリ形式で保存されます。ドキュメントは、以下に示すように、ハイパーリンクのリストの形式で他の SharePoint ユーザーが利用できるようになります。

### **利点**
次のリストは、Aspose.Cells for SharePoint を使用する利点をいくつか示しています。

- SharePoint サイト サーバーには、Microsoft Excel ファイルをサーバー上で直接変換する機能が組み込まれていません。 Aspose.Cells for SharePoint は、この機能を SharePoint サイト サーバーに追加します。
- Aspose.Cells for SharePoint は Microsoft Excel を必要としません。これはサーバーにインストールされるため、SharePoint ユーザーは、Microsoft Excel ドキュメントをある形式から別の形式に変換するためにローカル システムに Microsoft Excel をインストールする必要はありません。
- Aspose.Cells for SharePoint には、SharePoint サイトの作成に必要なものを除いて、前提条件は必要ありません。
-  Aspose.Cells for SharePoint は[SharePoint サイト サーバーへの展開が容易](/cells/ja/sharepoint/installing-aspose-cells-for-sharepoint/).
-  Aspose.Cells for SharePoint 対応[多数の入出力形式](/cells/ja/sharepoint/multiple-format-support/).
- Aspose.Cells for SharePoint は、独自の SharePoint ソリューションを開発する必要がある開発者向けにパブリック API を公開します。
### **Microsoft の Excel ドキュメントを Aspose.Cells for SharePoint に変換する**
Aspose.Cells for SharePoint が SharePoint サーバーにインストールされると、オプション**Aspose.Cells.SharePoint で変換**コンテキスト メニューで使用できます。

**Convert with... オプションを見つける** 

Microsoft Excel ドキュメントを SharePoint ドキュメント ライブラリから他の形式に変換するには、次の手順に従います。

1. 目的の Microsoft Excel ドキュメントをクリックし、[Aspose.Cells for SharePoint で変換] をクリックします。
変換設定画面が表示されます。

   **転向を決意** 


1. 出力形式を選択します。
ここで、出力ファイル名と宛先フォルダーを変更することもできます。
1. クリック**変換**. 

   **出力形式の選択** 



1. 変換が完了すると、成功メッセージが表示されます。

   **変換成功** 



1. クリック**戻る**ドキュメント ライブラリに戻ります。
1. 変換されたドキュメントは、SharePoint ドキュメント ライブラリに一覧表示されます。

   **変換されたファイルはドキュメント ライブラリに一覧表示されます** 


### **概要**
{{% alert color="primary" %}} 

Aspose.Cells for SharePoint のインストールと使用は簡単です。 Microsoft Excel ファイルを SharePoint サーバー上で直接変換する機能が追加されるだけでなく、クライアント マシンに Microsoft Excel をインストールする必要がなくなります。さらに、開発者は Aspose.Cells for SharePoint の public API を使用して、独自の SharePoint ソリューションを開発できます。

つまり、for SharePoint サイト管理者、開発者、およびエンド ユーザーにとって有益です。
