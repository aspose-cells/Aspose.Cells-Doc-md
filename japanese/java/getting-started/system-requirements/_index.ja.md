{{% alert color="primary" %}} 

Aspose.Cells for Javaは、Microsoft Excelをインストールする必要がなく、Aspose.Cells自体がExcelスプレッドシートの作成、変換、およびレンダリングエンジンであるため、Aspose.Cellsが生成したExcel文書を表示するには、少なくともMicrosoft Excelビューアをインストールする必要があります。

{{% /alert %}} 
## **サポートされるオペレーティング システム**
Aspose.Cells for Javaは、Javaランタイムを実行する任意のオペレーティングシステムをサポートしていますが、以下に限定されません:
### **Windows**
- Microsoft Windows 2000 ( x64, x86)
- Microsoft Windows XP ( x64, x86)
- Microsoft Windows Server 2003 (x64、x86)
- Microsoft Windows Server 2008 (x64、x86)
- Microsoft Windows Server 2012 (x64、x86)
- Microsoft Windows Vista (x64、x86)
- Microsoft Windows 7 (x64、x86)
- Microsoft Windows 8 (x64、x86)
- Microsoft Windows 10 (x64、x86)
### **Linux**
- Linux（Ubuntu、openSUSE、CentOSなど）
### **Mac**
- Mac OS Xなど。
## **サポートされるJavaバージョン**
Aspose.Cells for Javaは次のJavaバージョンをサポートします:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- 以上
