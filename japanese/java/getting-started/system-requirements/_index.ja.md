{{% alert color="primary" %}} 

Aspose.Cells for Java では、Microsoft Excel をインストールする必要はありません。Aspose.Cells 自体が Excel スプレッドシートの作成、変換、およびレンダリング エンジンであるためです。 Aspose.Cells によって生成された Excel ドキュメントを表示するには、少なくとも Microsoft Excel ビューアーがインストールされている必要があります。

{{% /alert %}} 
## **サポートされているオペレーティング システム**
Aspose.Cells for Java は、Java ランタイムを実行するすべてのオペレーティング システムをサポートします。
### **Windows**
- Microsoft Windows 2000 (x64、x86)
- Microsoft Windows XP (x64、x86)
- Microsoft Windows サーバー 2003 (x64、x86)
- Microsoft Windows サーバー 2008 (x64、x86)
- Microsoft Windows サーバー 2012 (x64、x86)
- Microsoft Windows ビスタ (x64、x86)
- Microsoft Windows 7 (x64、x86)
- Microsoft Windows 8 (x64、x86)
- Microsoft Windows 10 (x64、x86)
### **Linux**
- Linux (Ubuntu、openSUSE、CentOS など)
### **マック**
- Mac OS X など
## **サポートされている Java バージョン**
Aspose.Cells for Java は、次の Java バージョンをサポートしています。

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- 以上
