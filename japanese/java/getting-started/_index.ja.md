## **システム要件**

Aspose.Cells for Javaは次のJavaバージョンをサポートします:

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- 以上

[詳細はこちら](/cells/ja/java/system-requirements/)

## **インストール**

### **MavenリポジトリからAspose Cells for Javaをインストールする**

MavenはAspose.Cells for Javaをダウンロードしてインストールする最も簡単な方法です。 

1. Mavenのpom.xmlでAspose Mavenリポジトリの設定/場所を指定します。 
2. pom.xmlでAspose.Cells for Java API依存関係を定義します。 

[詳細な手順はこちら](/cells/ja/java/installation/)

### **Aspose Cells for Javaを手動でインストールする**

1. [Aspose-cells jars](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/)をフォルダーにダウンロードします。
1. フォルダーに入り、EclipseまたはIntellijでJARパッケージを手動でインストールします。

EclipseでAspose Cells for Javaをインストールする:
