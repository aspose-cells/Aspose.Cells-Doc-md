##  **システム要求**

Aspose.Cells for Java は、次の Java バージョンをサポートします。

- J2SE 6.0 (1.6)
- J2SE 7.0 (1.7)
- 以上

[さらに詳しく](/cells/ja/java/system-requirements/)

##  **インストール**

###  **Aspose リポジトリから Aspose Cells for Java をインストールします**

Maven は、Aspose.Cells for Java をダウンロードしてインストールする最も簡単な方法です。

1. Maven pom.xml で Aspose Maven リポジトリの構成/場所を指定します。
 2. pom.xml で Aspose.Cells for Java API 依存関係を定義します。

[詳細については、さらなるステップをご覧ください](/cells/ja/java/installation/)

###  **Aspose Cells for Java を手動でインストールします**

1. [aspose-cells jar をダウンロード](https://releases.aspose.com/java/repo/com/aspose/aspose-cells/)フォルダーに入れます。
1. フォルダーに入り、Eclipse または Intellij を使用して JAR パッケージを手動でインストールします。

Eclipse で Aspose Cells for Java をインストールします。

**![Eclipse インストール Aspose Cells for Java](Eclipse_install.png)**

Intellij で Aspose Cells for Java をインストールします。

**![IntelliJ インストール Aspose Cells for Java](IntelliJ_install.png)**
