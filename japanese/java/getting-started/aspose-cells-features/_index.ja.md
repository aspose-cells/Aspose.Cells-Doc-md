## **サポートされているプラットフォーム**
Aspose.Cells for Java製品は、Javaをサポートする人気のある開発環境および展開プラットフォームのほとんどをカバーしています。

|**機能**|**説明**|
| :- | :- |
|[Java](/cells/ja/java/installation/)|Aspose.Cells for Javaは1.6以降のバージョンで利用可能で、Javaがインストールされた場所ならどこでも実行できます。
## **ファイル形式と変換**
世界中の多くの顧客が、Aspose.Cells for Javaが提供する徹底的なドキュメントのインポートおよびエクスポート機能を利用しています。

|**機能**|**説明**|**Aspose.Cells for Java**|
| :- | :- | :- |
|高品質な変換|いくつかの人気のあるドキュメントフォーマット間で素早くかつ信頼性の高い変換を行います。|はい|
|Microsoft Excel Workbook (XLS)|見つけることのできる、最速かつ最も完全なXLS形式の実装です。|はい|
|SpreadsheetML (XLSX, XML)|Aspose.CellsはOOXML、Flat OPC、またExcel 2003 XMLをサポートしています。さらに、<br> このコンポーネントは、XLSX、XLTX、XLSM、XLTM、XLSBなどの主要なExcel 2007以降のバージョンのファイル形式をすべてサポートしています。|はい|
|Comma Separated Value (CSV)|さまざまなアプリケーションと連携するためのCSV形式の包括的なサポート。|はい|
|HTML|HTMLへの保存が可能です。|はい|
|OpenDocument (ODS)|Aspose.CellsはOpenDocument Spreadsheet（.ods）形式の文書の読み込みと保存をサポートしています。|はい|
|Adobe Portable Document Format (PDF)|高い忠実度で任意のスプレッドシートをPDFに変換します。|はい|
|Tab Delimited|タブ区切りファイル形式の文書の読み込みと保存。|はい|
|Plain Text (TXT)|プレーンテキスト形式で保存することで、テキストの抽出が容易になります。|はい|
## **レンダリングと印刷**
Aspose.Cells for Javaは、スプレッドシートのページを高い忠実度でレンダリングします。

|**機能**|**Aspose.Cells for Java**|
| :- | :- |
|Microsoft Excel®のレイアウトドキュメントをページに高い忠実度で配置します（まさにMicrosoft Excel®がそれを行うかのように）。|はい|
|ワークシートページをラスターイメージ（PNG、JPEG、BMPなど）にレンダリングします。|はい|
|スプレッドシートページをベクターイメージ（EMF）にレンダリングします。|はい|
|画像の解像度、品質、圧縮などのオプションを指定します。|はい|
|Javaの印刷インフラストラクチャを使用してスプレッドシートを印刷します。 このコンポーネントには、Microsoft Excelの印刷プレビューで表示されるワークシートを印刷するための組み込みの印刷メソッドがあります。|はい|
|レンダリングまたは印刷前にページ番号やその他のフィールドを更新します。|はい|
