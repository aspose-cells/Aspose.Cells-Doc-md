## **サポートされているプラットフォーム**
Aspose.Cells for Java 製品は、Java をサポートする一般的な開発環境と展開プラットフォームのほとんどをカバーしています。

|**特徴**|**説明**|
|:- |:- |
|[Java](/cells/ja/java/installation/)|Aspose.Cells for Java は 1.6 以降のバージョンで利用可能で、Java がインストールされている任意の場所で実行されます。|
## **ファイル形式と変換**
世界中の多くのお客様が、Aspose.Cells for Java が提供する完全なドキュメントのインポートおよびエクスポート機能に依存しています。

|**特徴**|**説明**|**Aspose.Cells for Java**|
|:- |:- |:- |
|高品質のコンバージョン|迅速かつ確実に、いくつかの一般的なドキュメント形式間で変換します。|はい|
|Microsoft エクセルワークブック (XLS)|あなたが見つけることができる最速かつ最も完全な XLS フォーマットの実装。|はい|
|SpreadsheetML (XLSX、XML)| Aspose.Cells は、OOXML、Flat OPC、および Excel 2003 XML をサポートしています。さらに、<br>このコンポーネントは、すべての主要な Excel 2007 以降のバージョンのファイル形式 (XLSX、XLTX、XLSM、XLTM、XLSB など) をサポートしています。|はい|
|カンマ区切り値 (CSV)|さまざまなアプリケーションと対話するための CSV 形式の包括的なサポート。|はい|
|HTML|HTMLとして保存できます。|はい|
|オープンドキュメント (ODS)|Aspose.Cells は、ドキュメントを OpenDocument スプレッドシート (.ods) 形式で読み込んで保存できます。|はい|
|Adobe Portable Document Format (PDF)|スプレッドシートを忠実度の高い PDF に変換します。|はい|
|タブ区切り|ドキュメントをタブ区切りファイル形式で読み込んで保存します。|はい|
|平文 (TXT)|プレーンテキスト形式で保存することにより、テキストを簡単に抽出できます。|はい|
## **レンダリングと印刷**
Aspose.Cells for Java は、スプレッドシート ページの忠実度の高いレンダリングを提供するようになりました。

|**特徴**|**Aspose.Cells for Java**|
|:- |:- |
|ドキュメントを Microsoft Excel® に忠実にページにレイアウトします (Microsoft Excel® とまったく同じように)。|はい|
|ワークシート ページをラスター イメージにレンダリングします (PNG、JPEG、BMP など)。|はい|
|スプレッドシート ページをベクター イメージにレンダリングします (EMF)。|はい|
|画像の解像度、品質、圧縮、およびその他のオプションを指定します。|はい|
|Java 印刷インフラストラクチャを使用してスプレッドシートを印刷します。コンポーネントには、印刷するための組み込みの印刷メソッドがあります<br>Microsoft Excel の印刷プレビューに示されているワークシート。|はい|
|レンダリングまたは印刷する前に、ページ番号およびその他のフィールドを更新します。|はい|

## **スプレッドシートの書式設定機能**
Aspose.Cells for Java は、すべてのスプレッドシート要素の書式設定プロパティへの詳細なプログラム アクセスを提供します。

|**特徴**|**Aspose.Cells for Java**|
|:- |:- |
|フォント、色、効果、境界線、陰影など、すべての文字書式をセルに適用します。|はい|
|セルにすべての数値形式設定 (Microsoft Excel でサポート) を適用します。|はい|
|すべてのタイプのテキスト配置設定を設定します。|はい|
|セルにさまざまな種類のフォント設定を適用します。|はい|
|セル、フォント、グリッド線、グラフィック オブジェクトなどにさまざまな色を適用します。|はい|
|1 つのセルにさまざまなリッチ テキスト形式を適用します。|はい|
|セルに異なる境界設定を適用します。|はい|
|セルにさまざまな背景パターンを適用します。|はい|
|ワークシート、行、列、セル範囲などに書式設定を適用します。|はい|
