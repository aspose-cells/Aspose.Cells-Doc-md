## **xlsx4j**
xlsx4jは、Microsoft Open XML Excel xlsxファイルの作成および操作を行うためのオープンソース（ASLv2）Javaライブラリです。
これは、MicrosoftのOpenXML SDKに似ていますが、Java向けのものです。docx4jは、JAXBを使用してメモリ内のオブジェクト表現を作成します。
JAXBとOpen XMLファイルの構造を理解するための時間を取る必要があります。

[docx4java.org](https://www.docx4java.org/trac/docx4j)で詳細を読む
## **Aspose.Cells for Java**
Aspose.Cells for Javaは、Java開発者が自分自身のJavaアプリケーションにExcel®スプレッドシート(XLS、XLSX、XLSM、XLSB、XLTX、SpreadsheetML、CSV、ODS)、HTML、MHTML、PDF、画像ファイル形式の読み取り、書き込み、操作機能を埋め込むことを可能にする優れたExcelスプレッドシートコンポーネントです。

Aspose.Cells for Javaは、成熟し、スケーラブルで機能豊富なコンポーネントであり、他のベンダーの単純なデータエクスポート機能をはるかに超える多くの機能を提供しています。Aspose.Cells for Javaを使用すると、開発者はデータをエクスポートし、スプレッドシートを最も詳細なレベルでフォーマットし、画像をインポートし、チャートを作成し、複雑な数式を適用および計算し、Excel®データをストリーミングし、さまざまなフォーマットで保存し、それ以外にも多くを行います - すべてがMicrosoft Excel®またはMicrosoft Office Automationに依存する必要はありません。

[aspose.com](https://products.aspose.com/cells/java/)で詳細を読んでください

## **なぜxlsx4jではないのか**
xlsx4jはMicrosoft Open XML Excel xlsxファイルを解析して保存することができますが、Aspose.Cells for JavaはすべてのExcelファイル形式（XLS、XLSX、XLSM、XLSB、XLTX、SpreadsheetML、CSV、ODS）を処理し、さまざまなファイル形式の変換をサポートしています。

## **Aspose.Cells Java for xlsx4j**
Project Aspose.Cells for xlsx4jは、Aspose.Cells Java APIを使用してさまざまなタスクを実行する方法を示し、xlsx4jとの比較を行います。また、スプレッドシートの処理においてAspose.Cells APIでのみ使用できる機能もカバーしています。

このプロジェクトは、xlsx4jとAspose.Cellsを比較したり、xlsx4jからAspose.Cellsに移行したい開発者に役立ちます。
## **システム要件およびサポートされるプラットフォーム**
### **システム要件**
Aspose.Cells Java for xlsx4jを実行するためのシステム要件は次のとおりです。

- Java 1.4以上がインストールされていること。
- Aspose.Cellsコンポーネントをダウンロードしてください。
- docx4jライブラリがダウンロードされていること。
### **サポートされているプラットフォーム**
サポートされるプラットフォームは次のとおりです。

- Aspose.Cells 8.3.0以上。
- docx4j 3.1.0以上のライブラリ。
- Java IDE（Eclipse、NetBeans、IntelliJなど）。
## **ダウンロード**
実行例の次のリリースを以下で言及されるすべてのソーシャルコーディングサイトからダウンロードできます:

### **GitHub**
- [Aspose.Cells Java for Xlsx4j - v 1.0](https://github.com/aspose-cells/Aspose.Cells-for-Java/releases/tag/Aspose.Cells_Java_for_Xlsx4j-v1.0.0)

「Aspose.Cells for Java API」を構成するには、AsposeリポジトリURLと「Aspose.Cells for Java API」のmaven依存関係を、以下のように「pom.xml」に含めます。
### **Aspose Mavenリポジトリ**
{{< highlight java >}}

 <repositories>

    <repository>

        <id>aspose-maven-repository</id>

        <name>Aspose Maven Repository</name>

        <url>http://repository.aspose.com/repo/</url>

    </repository>

</repositories>

{{< /highlight >}}
### **Aspose.Cells for Java Maven依存関係**
{{< highlight java >}}

 <dependencies>

    <dependency>

        <groupId>com.aspose</groupId>

        <artifactId>aspose-cells</artifactId>

        <version>8.6.0</version>

    </dependency>

</dependencies>

{{< /highlight >}}

**注意：** 利用可能なAspose APIの最新バージョンを使用してください。
