## **Aspose.Cells ドキュメントへようこそ**
**Aspose.Cells**Java アプリケーションで Excel スプレッドシートを作成および管理できる最も柔軟なコンポーネントです。**Microsoft エクセル**システムにインストールします。他のベンダーの同様の製品とは異なり、Aspose.Cells は、スプレッドシートの生成やその他の基本的なファイル フォーマット機能をサポートするだけでなく、多くの高度な機能もサポートします。これらの高度な機能により、開発者はスプレッドシートの内容、セルの書式設定、およびファイル保護を操作することがより便利になります。 Aspose.Cells では、さまざまなデータ ソースからスプレッドシートにデータをインポートしたり、一般的で複雑な数学、DateTime、財務、テキストの数式/関数を追加したり、チャート、写真、コメント、描画オブジェクト、コントロールなどの操作をサポートしたりすることもできます。

Aspose.Cells for Java は、Web クライアント用の完全なグリッド ソリューションも提供します。次のような GUI ベースのコントロールがあります。

- Aspose.Cells.Web アプリケーションをサポートする GridWeb

Aspose.Cells.GridWeb for JAVA グリッド コントロールを使用すると、開発者は Excel® スプレッドシートから簡単にインポートおよびエクスポートできます。 Microsoft Excel に慣れているユーザーなら誰でも、グリッドのルック アンド フィールを制御できます。 Aspose.Cells は、グリッドのデザインとレイアウトの制御、複数のワークシートの管理、ワークシートのセル、行と列の管理、数式の作成と計算などの豊富な機能を提供します。

## **製品説明**
Aspose.Cells は、Java Web アプリケーションまたは Java デスクトップ アプリケーションのいずれの種類のアプリケーションでも使用できます。さらに、このライブラリは、PHP などの他のソリューションでも使用できます。

Aspose.Cells は、ネイティブ Excel ファイルの作成や読み取りなどの一般的な機能をサポートするだけでなく、ストリームへの Excel ファイルの保存とストリームからのオープン、クライアント ブラウザーへの出力の送信、データのインポートとエクスポートなどの多くの高度な機能も提供します。**結果セット**または**配列**、画像とチャートのインポート、API による画像とチャートの作成、デザイナー スプレッドシートからの数式のインポート、および API によるページ設定の構成。

さらに、CSV および SpreadsheetML 形式のサポート、既存のワークシートのコピー (完全なコンテンツ、画像、グラフなど) のファイルへの追加など、開発者が Aspose.Cells でのみ見つけることができる独自の機能も多数あります。 API までのチャートのグラデーション背景の設定、API までのコメント、オートフィルター、改ページの作成、複雑な数式の設定、条件付き書式のサポート、Microsoft で導入された保護オプションのサポート Excel XP 以降、名前付き範囲の操作、API の検索、カスタム チャート API のサポート数式計算エンジンなど
## **ドキュメントの機能**
- さまざまなソースからプレーンまたは暗号化された Excel ファイル (Excel 95、Excel97、Excel2007/2010/2013/2016/2019、Office365) を開く
- さまざまなソースから (Apple の「Numbers」アプリケーションで作成された) Plain Numbers ファイルを開く
- Excel ファイル (Excel97- Excel2007/2010/2013/2016/2019) をさまざまな形式 (XLS、XLSX/XLSM、XLTX、XLTM、XLSB、CSV、SpreadsheetML、タブ区切り/TSV、HTML など) で保存します。
- Excel ファイルを PDF ドキュメントに変換する
- Excel スプレッドシートを XPS ドキュメントに変換する
- タグ付き画像ファイル形式に変換**(TIFF)**
- 読み書き**OpenDocument スプレッドシート (ODS)**フォーマット
- Excel ファイルのドキュメント プロパティを変更する
