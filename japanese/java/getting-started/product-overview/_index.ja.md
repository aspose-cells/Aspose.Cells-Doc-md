## **Aspose.Cells のドキュメントへようこそ**
**Aspose.Cells** は、システムに **Microsoft Excel** をインストールする必要なく、Java アプリケーションが Excel スプレッドシートを作成および管理できる最も柔軟なコンポーネントです。他社製品とは異なり、Aspose.Cells はスプレッドシートの生成やその他の基本的なファイル書式設定機能だけでなく、多くの高度な機能もサポートしています。これらの高度な機能により、開発者はスプレッドシートのコンテンツ、セルの書式設定、ファイルの保護などをより便利に操作できます。Aspose.Cells はさまざまなデータソースからスプレッドシートにデータをインポートしたり、一般的な数学、日付時刻、金融、テキストの数式/関数を追加したり、チャート、画像、コメント、図形オブジェクト、コントロールなどを操作することもできます。

Aspose.Cells for Java は Web クライアント用の完全な Grid ソリューションも提供します。次のような GUI ベースのコントロールがあります:

- Web アプリケーションをサポートする Aspose.Cells.GridWeb。

Aspose.Cells.GridWeb for JAVA Grid コントロールを使用すると、開発者は簡単に Excel® スプレッドシートのインポートおよびエクスポートができます。Microsoft Excel に詳しいどんなユーザーでも、グリッドの外観と感触を制御できます。Aspose.Cells は、グリッドのデザインとレイアウトの制御、複数のワークシートの管理、ワークシートのセル、行、列の管理、数式の作成と計算など、豊富な機能を提供しています。

## **製品説明**
Aspose.Cells は Java Web アプリケーションや Java デスクトップアプリケーションなど、あらゆる種類のアプリケーションと使用できます。さらに、このライブラリは PHP などの他のソリューションとも使用できます。Aspose.Cells は、コンポーネントが公開する機能をよりよく理解するための Java で記述された完全な機能を備えたデモや作業例を提供しています。

Aspose.Cells は、ネイティブの Excel ファイルの作成や読み込みなどの一般的な機能をサポートするだけでなく、Excel ファイルの保存と開くこと、クライアントブラウザへの出力、**ResultSet** または **Array** からのデータのインポートとエクスポート、画像とチャートのインポート、API を使用した画像とチャートの作成、デザイナースプレッドシートからの数式のインポート、および API を使用したページの設定など、多くの高度な機能も提供しています。

さらに、Aspose.Cells でのみ開発者が見つけることができる、CSV と SpreadsheetML 形式のサポート、既存のワークシートのコピー（内容、画像、チャートなどを含む）をファイルに追加する機能、API を使用したチャートのグラデーション背景の設定、コメント、オートフィルター、API を使用した改ページ、複雑な数式の設定、Microsoft Excel XP 以降で導入された保護オプションのサポート、名前の範囲の操作、API を指向したカスタムチャート API および数式計算エンジンのサポートなど、一意の機能も多数提供しています。
## **ドキュメントの機能**
- 異なるソースからプレーンまたは暗号化された Excel ファイルを開く（Excel 95、Excel97、Excel2007/2010/2013/2016/2019、Office365）
- 異なるソースから作成されたプレーンナンバーズファイル（Apple の "Numbers" アプリケーションによる）を開く
- Excel ファイルを異なる形式（XLS、XLSX/XLSM、XLTX、XLTM、XLSB、CSV、SpreadsheetML、Tab Delimited/TSV、HTML など）で保存
- Excel ファイルを PDF ドキュメントに変換
- Excel スプレッドシートを XPS ドキュメントに変換
- タグ付き画像ファイル形式 **(TIFF)** に変換
- **OpenDocument Spreadsheet (ODS)** 形式を読み込み、書き込む
- Excel ファイルのドキュメントプロパティを変更
## **ワークシート機能**
### **表示オプション**
- ワークシートの非表示または表示
- **タブ、スクロールバー、グリッド線、行と列のヘッダー**の表示または非表示
- **ページ区切りのプレビュー**
- ワークシートの**ズームレベル**の設定
- **フリーズペイン**を使用して選択したデータをスクロールしても表示されたままにする
### **ワークシートの保護**
- ワークシートの内容、オブジェクト、シナリオの**保護**
### **ページ設定オプション**
- **ページの向き、拡大縮小、用紙サイズ、印刷品質**の設定
- 上部、左部、右部、下部などの**ページ余白**の設定
- **ヘッダーとフッター**の設定
- **印刷範囲**や**印刷タイトル**の設定
### **ページブレーク**
- 指定したセルに**水平**および**垂直なページ区切り**を挿入
### **行および列の機能**
- 必要に応じて**列幅の調整**
- セルの内容に応じて幅と高さを**自動調整**
- 行や列を**挿入または削除**
- 行や列を**非表示または表示**
- 行や列を**グループ化またはグループの解除**
### **データ管理機能**
- 実行時に希望のセルにデータを**追加**
- 異なるデータソースからワークシートにデータを**インポート**
- セルからデータを**データ型**に従って取得
- セルデータを**配列**に**エクスポート**
- セル内の特定のデータを**検索**
- 組み込み関数やアドイン関数を使用した**数式**の設定
- **配列**および**R1C1**の参照スタイルの数式の設定
- 異なる種類の**ハイパーリンク**の追加
- 名前付き範囲の作成またはアクセス
- **データの検証**および**自動フィルタリング**の適用
- ワークシートテーブルに**小計**を適用
### **書式設定機能**
- セルに対する**数値形式設定**の適用
- セルを日付として**フォーマット**
- **テキストの整列**の設定
- 複数のセルを**結合**
- セルに異なる種類の**フォント設定**を適用する
- セル、フォント、グリッド線、グラフィックオブジェクトなどに異なる色を適用する。
- 1つのセルに**リッチテキスト**を設定する
- セルに異なる**境界線設定**を適用する
- セルに異なる**背景パターン**を適用する
- ワークシート、行、列などに**形式設定**を適用する
- セルに**条件付き書式**を適用する
## **チャートおよびグラフィック機能**
- 簡単に異なる種類のグラフを作成する
- プロパティを設定して**グラフ**をカスタマイズする
- グラフのデータを動的に設定する
- ワークシートに**カスタムグラフ**を追加する
- ワークシートに**画像**を動的に追加する
- セルにコメントを作成して追加する
- グラフを画像ファイルに変換する
- ワークシートを画像ファイルに変換する
- ワークシートを印刷する
## **高度な機能**
- 頑健な式計算エンジンを使用して**数式計算**をサポートする
- ピボットテーブルの**ソースデータ**を動的に変更する
- ピボットテーブルを作成および操作する
- VBAコードまたはマクロを操作する。
## **対応プラットフォーム**

- Linux
- Unix
