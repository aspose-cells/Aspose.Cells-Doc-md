## **GitHub からダウンロード**
Aspose.Cells for Java のすべての例は、[GitHub](https://github.com/aspose-cells/Aspose.Cells-for-java) .お気に入りの GitHub クライアントを使用してリポジトリのクローンを作成するか、ZIP ファイルを次の場所からダウンロードできます。[ここ](https://github.com/aspose-cells/Aspose.Cells-for-java/archive/master.zip).

 ZIP ファイルの内容をコンピューター上の任意のフォルダーに抽出します。すべての例は、**例**フォルダ。

の**資力**フォルダーには、例で使用される入力ドキュメントまたはファイルが含まれています。

これらの例は、任意の IDE で実行/実行できますが、デモンストレーションの目的で、**エクリプス**.使用**ファイル -> インポート**選択した例をインポートしてパスを指定するオプション。

プロジェクトに必要な jar ファイル (ライブラリ) を追加します。

各例は、個別の実行可能なクラスとして提供されます。ファイルを右クリックし、ファイルの実行を選択します。

{{% alert color="primary" %}} 

例は次の形式です。**エクリプス プロジェクト**.を使用して Eclipse で例を開いてください。**ファイル > ファイル システムからプロジェクトを開く...**メニューコマンドを選択して、手順に従います。これで、例をうまく実行できるはずです。

{{% /alert %}} 
## **例を実行する**
サンプルを Eclipse IDE で開いたら、任意の .java ファイルを選択します。たとえば、これを選択します。

TilePictureAsTextureInsideShape.java

の中で見つかった

Examples>src>AsposeCellsExamples>DrawingObjects ディレクトリ

またはパッケージ内

AsposeCellsExamples.DrawingObjects

そしてそれを実行します。

## **助ける**
例を追加または改善したい場合は、プロジェクトに貢献することをお勧めします。このリポジトリ内のすべての例とショーケース プロジェクトはオープン ソースであり、独自のアプリケーションで自由に使用できます。

貢献するには、リポジトリをフォークし、ソース コードを編集してプル リクエストを作成します。変更を確認し、役立つと判断された場合はリポジトリに含めます。
