## **GitHub からダウンロード**
Aspose.Cells for Java のすべての例は [GitHub](https://github.com/aspose-cells/Aspose.Cells-for-java) でホストされています。お気に入りの GitHub クライアントを使用してリポジトリをクローンするか、[こちら](https://github.com/aspose-cells/Aspose.Cells-for-java/archive/master.zip) から ZIP ファイルをダウンロードできます。

ZIP ファイルの内容をコンピュータの任意のフォルダに展開します。すべての例は **Examples** フォルダにあります。

**Resources** フォルダには、例で使用される入力ドキュメントまたはファイルが含まれています。

お好みの IDE でこれらの例を実行/実行できますが、デモンストレーション目的として **Eclipse** を使用しています。 **ファイル** -> **インポート** オプションを使用して選択した例をインポートし、パスを指定します。

プロジェクトに必要なjarファイル（ライブラリ）を追加します。

各例は個別の実行可能クラスとして提供されます。ファイルを右クリックし、[ファイルの実行]を選択してください。

{{% alert color="primary" %}} 

例は **Eclipse Project** の形式で提供されます。**ファイル > ファイルシステムからのプロジェクトのインポート...** メニューコマンドを使用してEclipseで例を開き、手順に従って実行できるはずです。

{{% /alert %}} 
## **例を実行する**
Eclipse IDEで例を開いたら、任意の .java ファイルを選択します。たとえば、これを選択します。

TilePictureAsTextureInsideShape.java

これの中に見つかります

Examples>src>AsposeCellsExamples>DrawingObjects ディレクトリ内

またはパッケージの中に

AsposeCellsExamples.DrawingObjects

そして実行します。

## **貢献**
例を追加または向上させたい場合は、プロジェクトに寄与することをお勧めします。このリポジトリのすべての例とショーケースプロジェクトはオープンソースであり、自分のアプリケーションで自由に使用できます。

貢献するには、リポジトリをフォークし、ソースコードを編集してプルリクエストを作成します。変更をレビューし、役立つと判断されればリポジトリに含めます。
