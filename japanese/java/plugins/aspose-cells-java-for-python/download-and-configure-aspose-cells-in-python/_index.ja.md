## **必要なライブラリをダウンロード**
下記の必要なライブラリをダウンロードします。これらは、Aspose.Cells Java for Python の例を実行するために必要です。

- **Aspose:** [Aspose.Cells for Java コンポーネント](https://downloads.aspose.com/cells/java/)
- [JPype](https://pypi.org/project/JPype1/)
## **ソーシャル コーディング サイトからサンプルをダウンロードする**
実行中のサンプルの次のリリースは、以下のソーシャル コーディング サイトからダウンロードできます。

-----
### **GitHub**
- **Aspose.Cells Java for Python 例** 
  - [Aspose.Cells Java for Python](https://github.com/aspose-cells/Aspose.Cells-for-Java/releases/tag/Aspose.Cells_Java_for_Python-v1.0)
## **ソースコードの構成方法**
使用中にソース コードを開いて拡張するには、次の簡単な手順に従ってください。

 \1.以下から Python 2.7.xx のみをインストールします。<https://www.python.org/downloads/>
 \2.以下から JType をインストールします。<https://sourceforge.net/projects/jpype/files/JPype/>
 \3.以下から Java JDK をインストールします。<https://www.oracle.com/java/technologies/downloads/>
 \4. Python プロジェクトで使用する Aspose.Cells for Java API を次からダウンロードします。<https://downloads.aspose.com/cells/java>ダウンロードした API の .jar を %Python%/lib フォルダーにコピーします。
 \5.クローン[Aspose.Cells Java for Python 例](https://github.com/aspose-cells/Aspose.Cells-for-Java/tree/master/Plugins/Aspose-Cells-Java-for-Python)ギブハブから。
