## **インストール**

**Aspose.Cells Java for Eclipse (Maven)** プラグインは、プラグインダイアログの利用可能な **プラグイン** タブから簡単にインストールできます。

- Eclipse の **ツール** メニューから **プラグイン** を選択して開きます。  

![todo:image_alt_text](installing-and-using-aspose-cells-java-for-eclipse-maven_1)

- これにより、**Aspose.Cells Maven Project** が新しいプロジェクトウィザードに追加され、Eclipse IDE の新しいその他のウィザードに **Aspose.Cells Code Example** が追加されます。  

![todo:image_alt_text](project_1.png)

## **を使用する**

### **Aspose.Cells Maven プロジェクト (ウィザード)**

以下の手順で [Aspose.Cells for Java API](https://products.aspose.com/cells/java/) を使用するための **Maven プロジェクト** を作成します：

1. **新規 -> プロジェクト** を選択します。
1. **Maven** カテゴリ内の **Aspose.Cells Maven Project** を選択します。
1. **次へ** をクリックします。

![todo:image_alt_text](project_2.png)

1. Maven プロジェクトの **プロジェクト名、場所、GroupId、ArtifactId** および **バージョン** を指定し、**完了** をクリックします。

![todo:image_alt_text](project_3.png)

1. これにより、**pom.xml** に [Aspose.Cells for Java](https://products.aspose.com/cells/java/) の最新の [Maven 依存関係](https://repository.aspose.com/webapp/#/artifacts/browse/tree/General/repo/com/aspose/aspose-cells) 参照が取得され、構成されます。**Code Examples** もダウンロードする場合は、[Aspose.Cells for Java API Examples Repository](https://github.com/aspose-cells/Aspose.Cells-for-Java) から **Code Examples** もダウンロードが開始されます。
1. ウィザードの完了時に **Eclipse IDE** に次の **Maven** プロジェクトが作成されます：  

![todo:image_alt_text](project_4.png)

1. 作成された **Maven プロジェクト** は **Aspose.Cells for Java API** を使用するように構成されており、プロジェクトの要件に応じて拡張できる状態です。
   [Code Examples](https://github.com/aspose-cells/Aspose.Cells-for-Java) をダウンロードすることを選択した場合、**Aspose.Cells Code Example (ウィザード)** を使用して、必要な **Code Examples** を Aspose.Cells for Java API にインポートできます。

### **Aspose.Cells Code Example (ウィザード)**

**Aspose.Cells Code Example ウィザード** を使用すると、Aspose.Cells for Java API 用に用意された多くのサンプルを試すことができます。

{{% alert color="primary" %}}

**Aspose.Cells Maven Project** の作成時に常に **Also Download Code Examples** を選択することをお勧めします。

{{% /alert %}}

例を使用するには、単に：

1. **Eclipse** で **新規 -> その他** をクリックします。
1. **Java** カテゴリ内の **Aspose.Cells Code Example** を選択します。
1. **次へ** をクリックします。  

![todo:image_alt_text](example_1.png)

1. プロジェクトを選択し、ツリーを展開して **Code Example** カテゴリを選択し、**完了** をクリックします。

![todo:image_alt_text](example_2.png)

1. 選択したカテゴリの **Code Examples** Java ファイルが、**com.aspose.cells.examples** パッケージの下にプロジェクトにコピーされます。また、Code Examples で必要なリソースも **src/main/resources** フォルダにコピーされます（以下のように表示されます）。

![todo:image_alt_text](example_3.png)

1. 例示コードを確認し、コンパイルして実行します。
1. 他の例をテストしたり、Aspose.Cells for Java APIを使用して独自のアプリケーションを構築を開始できます。
