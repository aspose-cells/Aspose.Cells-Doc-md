## **サポート**
- アプリケーションで既知の/報告された問題（ユーザーまたはQ.Aチームによる）を表示したい場合。
- または、アプリケーションで見つけた問題を報告したい場合。
- 改善提案をしたい場合、または機能のリクエストをしたい場合

**次のプロジェクトの問題トラッカーを使用してください:**

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-Java/issues)
## **拡張と貢献**
Struts 1.3のAspose.cells Javaはオープンソースであり、そのソースコードは以下の主要なソーシャルコーディングウェブサイトにて利用可能です。開発者はソースコードをダウンロードし、新しい機能の提案や追加、既存機能の改善を行うことを推奨されており、他の人々もそれを利益とすることができます。
