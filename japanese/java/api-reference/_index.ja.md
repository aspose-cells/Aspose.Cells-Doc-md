{{% alert color="primary" %}} 

Aspose.Cells for Javaは、アプリケーションが多岐にわたるスプレッドシート処理タスクを実行できるクラスライブラリです。Aspose.Cells for Javaは人気のあるスプレッドシート（XLS、XLSX、XLSM、XLSB、XLTX、XLTM、CSV、SpreadsheetML、ODS）ファイル形式をサポートし、ExcelファイルをPDF、XPS、HTML、MHTML、プレインテキスト、TIFF、JPG、PNG、BMP、SVGにエクスポートすることができます。

最新のAPIリファレンスは[Aspose.Cells for Java APIリファレンス](https://reference.aspose.com/cells/java)で見つけることができます。

{{% /alert %}}
