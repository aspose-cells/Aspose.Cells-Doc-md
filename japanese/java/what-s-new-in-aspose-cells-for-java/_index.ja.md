このページでは、最近のリリースで導入された最も興味深い Aspose.Cells の新機能について説明します。

## Aspose.Cells for Java 22.12
* 多層階層構造で Excel を JSON にエクスポートするサポート

## Aspose.Cells for Java 22.11

動的配列数式と従来の配列数式を区別するためのサポート
xlsb ファイル形式の動的配列数式の読み取り/保存をサポート

## Aspose.Cells for Java 22.10

CRTX テンプレート ファイルを使用したチャートの追加と更新のサポート

## Aspose.Cells for Java 22.9

セルの表式の設定をサポート
カスタム関数と計算エンジンを使用した動的配列数式の設定のサポート
*グラデーション塗りつぶしフォーマットのプリセットグラデーション設定の設定をサポート
範囲のテーマ カラーでアウトライン ボーダーを適用するサポート

## Aspose.Cells for Java 22.8

シートに表示されているコメントとメモを html にエクスポートするサポート
pdf/xps への変換中に出力するワークシートの指定をサポート

## Aspose.Cells for Java 22.7

異常な zip データを含み、Java の ZipInputStream で処理できない OOXML 形式のテンプレート ファイルの読み込みをサポート
ページ ヘッダー/フッターを含む 1 つのファイルに html をエクスポートするサポート
Range の絶対位置とサイズの取得をサポート (ポイント単位)

## Aspose.Cells for Java 22.6

* 数式計算チェーンによる数式の計算とトレースの改善

## Aspose.Cells for Java 22.5

LightCells モードでワークブックを保存する際のセル数式設定のメモリ パフォーマンスの改善

## Aspose.Cells for Java 22.4

CSV をエクスポートするための空白セルを制御する拡張オプション。
* 挿入/削除操作のための条件付き書式の複雑な分割と更新をサポート

## Aspose.Cells for Java 22.3

定義名が変更され、ワークブックの計算チェーンが有効になっている場合、定義名に依存するセルの再計算をサポートします。
* 数式の計算チェーンに従ってセルのリーフを再帰的に取得するサポート

## Aspose.Cells for Java 22.2

指定されたスコープ フィルターによって NameCollection から定義された名前を抽出するサポート

## Aspose.Cells for Java 22.1

編集可能な pptx の生成をサポート
データの行/列が制限を超えた場合に、大きな csv を複数のワークシートにインポートするサポート
LightCells での csv インポートのサポート
指定した外部リンクの削除をサポート
外部リンクを削除しながら数式を保持するためのサポート

## Aspose.Cells for Java 21.12

Excel から SQL スクリプトへの変換をサポート
プレーン xml データのインポートとエクスポートをサポート
破損した xls ファイルから可能な限り多くのデータを読み取るためのサポート
SmartMarker のプロセスを中断するサポート
条件付き書式と検証の範囲を追加する際のパフォーマンスを向上させます
※MS Excelの挙動と同様に、新たに追加された書式条件の優先度を最優先に設定

## Aspose.Cells for Java 21.11

XLookup 式/関数の計算をサポートします。
*保存するjsonと画像を保存するワークブックに統合します。
* Excel を HTML に変換する際のパフォーマンスを向上させます。
