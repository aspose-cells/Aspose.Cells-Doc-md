お客様各位

Aspose.Cells for Java V1.9.0 をリリースしました！

新着情報？

- Aspose.Pdf xml ファイルへのエクスポートをサポート
ワークシートコピー時のコメントと名前付き範囲のコピーをサポート
計算式の ICustomFunction インターフェイスをサポート
細胞の前身と後身の追跡をサポート

何が改善されましたか？

- ワークシート内のすべての数式のみの計算をサポート
スマート マーカーを使用して POJO と POJO をサポート

何が修正されましたか？

- ワークシートの名前付き範囲を取得するバグを修正しました
ピボットテーブル作成の不具合を修正
ファイル保護の不具合を修正
計算式のバグを修正
ファイル保存のバグを修正しました。
