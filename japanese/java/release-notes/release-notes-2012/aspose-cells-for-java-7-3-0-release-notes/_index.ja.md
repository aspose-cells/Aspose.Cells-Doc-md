私たちです
Aspose.Cells for Java v7.3.0 を発表させていただきます!

新着情報

- 読む
および MHT ファイルの書き込み
- サポート
XML マップ
- 申し込み
MS Excel 2007/2010 のテーマをチャートに



その他の注目すべき機能、拡張機能、および修正点を以下に示します。

新機能

 – 遷移式評価の設定をサポート

機能強化

- ユーザーが指定したロケールでセル値をフォーマットする

例外

- サポートされていないフォント ファイルにより、PDF への保存が例外で失敗する可能性があります

バグ

- Cell.setR1C1Formula() メソッドは、列オフセットがないと正しく機能しませんでした
- 名前付き範囲が 2 回作成されました
- XLSM ファイルの保存時に sortNames() メソッドによって例外が発生する
- 分数の数値が正しくフォーマットされていませんでした
- PrintOrderType.OVER に対して生成されたブランク PDF_それから_下
- 生成された PDF の背景色とグリッドが正しくない
- 切片と勾配関数が正しく計算されませんでした
- Excel 2007 ファイル形式のピボット フィールド アイテムの 33k 制限を削除
- 表記法 1:1 は IF 関数ではサポートされていません
- DATEDIF 数式が正しく計算されませんでした
- 名前付き行の場合のセルの不適切なルックアップ
