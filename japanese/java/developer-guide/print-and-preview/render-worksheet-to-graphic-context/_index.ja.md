Aspose.Cells はワークシートをグラフィック コンテキストにレンダリングできるようになりました。グラフィック コンテキストは、画像ファイル、画面、プリンターなどのようなものです。ワークシートをグラフィック コンテキストにレンダリングするには、次の方法を使用してください。

- SheetRender.toImage(int pageIndex, Graphics2D グラフィック)
