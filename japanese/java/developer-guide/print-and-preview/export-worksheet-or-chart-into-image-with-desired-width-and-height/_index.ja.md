{{% alert color="primary" %}}

Aspose.Cellsを使用して、所定の幅と高さでワークシートまたはグラフをイメージにエクスポートできます。エクスポートされるイメージの幅と高さを指定するための[**ImageOrPrintOptions.setDesiredSize()**](https://reference.aspose.com/cells/java/com.aspose.cells/imageorprintoptions#setDesiredSize-int-int-) メソッドを提供します。幅と高さはピクセル単位で指定されます。
