{{% alert color="primary" %}} 

ワークシートからサムネイルを生成すると便利です。サムネイルは、Word 文書または PowerPoint プレゼンテーションに貼り付けて、ワークシートの内容をプレビューできる小さな画像です。元のドキュメントをダウンロードするためのリンクを含む Web ページに追加することができ、他にも多くの用途があります。

{{% /alert %}} 

Aspose.Cells for Java では、ワークシートを画像ファイルに出力できるので、サムネイルが簡単に作れます。
