## **Aspose.Cells で印刷中にジョブまたはドキュメント名を指定する**

サンプル コードは、ソース Excel ファイルを読み込み、ジョブまたはドキュメント名を指定してプリンターに送信します。[**WorkbookRender.toPrinter(プリンター名、ジョブ名)**](https://reference.aspose.com/cells/java/com.aspose.cells/workbookrender#toPrinter(java.lang.String,%20java.lang.String) ） と[**SheetRender.toPrinter(プリンター名、ジョブ名)**](https://reference.aspose.com/cells/java/com.aspose.cells/sheetrender#toPrinter(java.lang.String,%20java.lang.String)メソッド。スクリーンショットは、ジョブ名がプリンター キューでどのように表示されるかを示しています。
