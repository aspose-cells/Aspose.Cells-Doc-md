{{% alert color="primary" %}}

Aspose.Cellsは、Microsoft ExcelファイルをHTMLまたはMHT形式に保存する際に、開発者が使用できる便利な[**HtmlSaveOptions.PresentationPreference**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#PresentationPreference)プロパティを提供します。このプロパティのデフォルト値は**false**です。良い見た目のプレゼンテーションを得るためには、このプロパティを**true**に設定することをお勧めします。

{{% /alert %}}
