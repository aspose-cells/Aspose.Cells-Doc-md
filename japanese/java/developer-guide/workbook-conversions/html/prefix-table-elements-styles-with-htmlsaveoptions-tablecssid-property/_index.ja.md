## **可能な使用シナリオ**

Aspose.Cellsを使用すると、[**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId)プロパティでテーブル要素スタイルにプレフィックスを付けることができます。たとえば、*MyTest_TableCssId*のような値でこのプロパティを設定した場合、以下に示すようなテーブル要素のスタイルが見つかります

{{< highlight java >}}

table#MyTest_TableCssId

#MyTest_TableCssId tr

#MyTest_TableCssId col

#MyTest_TableCssId br

etc.

{{< /highlight >}}
