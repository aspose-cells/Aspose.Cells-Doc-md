## **考えられる使用シナリオ**

Aspose.Cells を使用すると、テーブル要素のスタイルに接頭辞を付けることができます[**HtmlSaveOptions.TableCssId**](https://reference.aspose.com/cells/java/com.aspose.cells/htmlsaveoptions#TableCssId)財産。このプロパティに次のような値を設定するとします。*MyTest_TableCssId*、次に示すようなテーブル要素スタイルが見つかります

{{< highlight "java" >}}

table#MyTest_TableCssId

# MyTest_TableCssId tr

# MyTest_TableCssId col

# MyTest_TableCssId br

etc.

{{< /highlight >}}
