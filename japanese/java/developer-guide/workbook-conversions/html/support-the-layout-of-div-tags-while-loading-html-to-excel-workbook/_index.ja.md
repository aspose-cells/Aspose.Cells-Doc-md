{{% alert color="primary" %}} 

通常、HTMLをエクセルブックオブジェクトにロードする際にDIVタグのレイアウトは無視されます。ただし、DIVタグのレイアウトを無視しない場合は、[HtmlLoadOptions.SupportDivTag](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#SupportDivTag)プロパティを**true**に設定してください。このプロパティのデフォルト値は**false**です。
