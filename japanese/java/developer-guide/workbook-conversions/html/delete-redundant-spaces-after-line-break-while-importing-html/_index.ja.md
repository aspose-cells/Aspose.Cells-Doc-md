{{% alert color="primary" %}} 

 [HtmlLoadOptions.DeleteRedundantSpaces](https://reference.aspose.com/cells/java/com.aspose.cells/htmlloadoptions#DeleteRedundantSpaces)プロパティを使用し、**true**に設定してラインブレイクタグの後に出てくる余分なスペースをすべて削除してください。既定では、このプロパティは**false**に設定されており、余分なスペースは出力Excelファイルに保持されます。

{{% /alert %}}
