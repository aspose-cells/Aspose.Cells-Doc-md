この記事では、スプレッドシートをPDFに変換する際にPDFブックマークを挿入する方法についての情報を提供します。

Aspose.Cellsを使用すると、実行時にPDFブックマークを追加することができます。PDFブックマークを追加すると、長いドキュメントのナビゲーションが大幅に改善されます。 PDFドキュメントへのブックマークリンクを追加する際に、望む正確なビューに対して細かい制御ができます。ページへのリンクに限定されることはありません。ターゲットページの位置を配置し、ブックマークを作成することで、正確なビューを設定できます。

{{% /alert %}}
