この記事では、スプレッドシートを PDF に変換するときに PDF ブックマークを挿入する方法について説明します。

Aspose.Cells を使用すると、実行時にブックマークを追加できます。 PDF ブックマークを使用すると、長いドキュメントの操作性が大幅に向上します。 PDF ドキュメントにブックマーク リンクを追加すると、ページへのリンクに限定されず、必要な正確なビューを正確に制御できます。ターゲット ページを配置して正確なビューを設定し、ブックマークを作成できます。

{{% /alert %}}
