## **Aspose.CellsのXPS**
Aspose.Cells に読み込まれたドキュメントはすべて XPS に変換できます。

Aspose.Cells は、同じ忠実度の高いページ レイアウトおよびレンダリング エンジンを使用して、サポートされているすべての固定レイアウト形式 (PDF、XPS など) で出力を生成します。
