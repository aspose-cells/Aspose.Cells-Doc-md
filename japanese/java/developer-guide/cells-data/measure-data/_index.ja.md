{{% alert color="primary" %}} 

セル値をセル内に収めるために、セル値の幅と高さを計算する必要がある場合があります。 Aspose.Cells提供[Cell.getWidthOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getWidthOfValue\(\) ） と[Cell.getHeightOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getHeightOfValue\(\)この目的のためのメソッド。これらのメソッドを使用すると、セル値の幅と高さを計算し、そのセルの列の幅と行の高さをそれぞれ設定できます。これにより、セル内のセル値が調整または適合されます。

または、次のこともできます[セルまたはセル範囲の行と列を自動調整する](/cells/ja/java/autofit-rows-and-columns/)Aspose.Cells API を使用します。
