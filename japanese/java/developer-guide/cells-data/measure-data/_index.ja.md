{{% alert color="primary" %}} 

場合によっては、セルの値をセル内にフィットさせるために、セル値の幅と高さを計算する必要があります。 Aspose.Cells では、[Cell.getWidthOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getWidthOfValue\(\)) および [Cell.getHeightOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getHeightOfValue\(\)) メソッドがそのために提供されています。 これらのメソッドを使用することで、セル値の幅と高さを計算し、そのセルの列の幅と行の高さを設定することができます。 そうすることで、セル値がセル内に調整されます。

また、Aspose.Cells の API を使用して、セルまたはセル範囲の[行と列を自動調整](/cells/ja/java/autofit-rows-and-columns/)することもできます。
