{{% alert color="primary" %}} 

時には、セルの値の幅と高さを計算してセル内に収める必要があります。Aspose.Cellsは、この目的のために[Cell.getWidthOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getWidthOfValue--)と[Cell.getHeightOfValue()](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getHeightOfValue--)メソッドを提供します。これらのメソッドを使用してセルの値の幅と高さを計算し、その後に列の幅と行の高さをそれぞれ設定することで、セルの値をセル内に調整またはフィットさせることができます。

また、Aspose.Cells の API を使用して、セルまたはセル範囲の[行と列を自動調整](/cells/ja/java/autofit-rows-and-columns/)することもできます。
