{{% alert color="primary" %}} 

ハイパーリンクは、他のワークシートや Web サイトの情報にアクセスするのに役立ちます。 Microsoft Excel では、ユーザーがセル内のテキストや画像にハイパーリンクを追加できます。画像のハイパーリンクを使用すると、次へボタンや前へボタン、または特定のサイトにリンクするロゴなど、ワークシートの移動が簡単になります。このドキュメントでは、Aspose.Cells を使用してワークシートに画像のハイパーリンクを挿入する方法について説明します。

{{% /alert %}} 

Aspose.Cells を使用すると、実行時にスプレッドシート内の画像にハイパーリンクを追加できます。リンクの画面のヒントとアドレスを設定および変更することができます。
