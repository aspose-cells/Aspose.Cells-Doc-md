## **このファイル形式はサポートされていないか、正しい形式を指定していません**
ユーザーがテンプレート ファイルからワークブックを作成するときにファイル形式を指定した場合、通常、このエラーは、指定されたファイル形式がテンプレート ファイルの実際のファイル形式ではないことが原因で発生します。ユーザーがファイル形式を指定していない場合、通常は、ファイル名拡張子がこのファイルの実際のファイル形式を表しておらず、特別な識別子を持たない csv/tsv ファイルなどのファイル形式を自動的に検出できないことが原因です。 .
