## **このファイル形式はサポートされていないか、正しい形式が指定されていません**
ユーザーがテンプレートファイルからワークブックを作成するときにファイル形式を指定した場合、一般的にこのエラーは指定されたファイル形式が実際のテンプレートファイルのファイル形式と異なるために発生します。ファイル形式が未指定の場合、一般的にファイル名の拡張子がこのファイルの実際のフォーマットを表していないか、自動的に検出できないためです。たとえば、csv/tsvファイルは特別な識別子を持たないためです。もちろん、Cellsがサポートしていないファイル形式でもこのエラーが表示されます。
