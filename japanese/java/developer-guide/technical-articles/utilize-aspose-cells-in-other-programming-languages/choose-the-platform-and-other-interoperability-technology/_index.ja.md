{{% alert color="primary" %}} 

Aspose.Cells は、Microsoft Excel® を使用せずに Excel® スプレッドシートを生成、読み取り、変更、書き込みできるスプレッドシート処理コンポーネントです。

- Aspose.Cells for Java - Java クラス ライブラリであり、Java アプリケーションで動作するように設計されています。

{{% /alert %}} 

他のプラットフォーム用の特別な Aspose.Cells バージョンはありませんが、以下を含む (ただしこれらに限定されません) 他の多くのプログラミング言語で Aspose.Cells を使用できます。

|**プログラミング言語** |**使用するAspose.Cells** |**相互運用技術** |
|:- |:- |:- |
|パール|Java |インライン::Java|
| PHP|Java |Java ブリッジ|
|Python |Java |JPype|
上記の表から、一部のプログラミング言語では Aspose.Cells for Java を使用できることがわかります。次の表は、選択に役立ちます。

|**シナリオ** |**いつ使用するか** |
|:- |:- |
|Aspose.Cells for Java via Java ブリッジまたはインライン::Java|<p>**長所**:</p><p>Java が利用可能なすべてのプラットフォームで利用できます。</p><p>- 静的メソッド、パラメーター付きのコンストラクター、オーバーロードされたメソッドを簡単に呼び出し、列挙値を使用できます。</p>|
||<p>**短所**: </p><p>Aspose.Cells for .NET よりリリース頻度が低く、機能も少ない。</p>|
 Aspose.Cells for Java は for Java アプリケーションとして設計されており、他のプログラミング言語で使用すると、いくつかの技術的な制限または回避策が適用されることに注意してください。
