{{% alert color="primary" %}}

Microsoft Excel のユーザーはスプレッドシートのセルにコメントを追加して、追加情報を提供し、データを強調することができます。開発者はコメントをカスタマイズして配置設定やテキスト方向を指定する必要があるかもしれません。Aspose.Cells はこれを実現するための API を提供します。

{{% /alert %}}

Aspose.Cells はコメントの望ましいテキスト方向を設定するための [**Shape.setTextDirection()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#TextDirection) メソッドを提供しています。
