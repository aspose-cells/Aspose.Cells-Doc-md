{{% alert color="primary" %}} 

ソース範囲の行の高さをコピー先範囲にコピーする必要がある場合があります。 Aspose.Cells提供[PasteType.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS)この目的のための列挙。いつ設定するか[PasteOptions.setPasteType()](https://reference.aspose.com/cells/java/com.aspose.cells/pasteoptions#PasteType)のプロパティ[PasteType.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS)enum の場合、ソース範囲内のすべての行の高さが宛先範囲にコピーされます。

{{% /alert %}} 
## **ソース範囲の行の高さをコピー先範囲にコピー**
次のサンプル コードは、使用方法を説明します。[PasteType.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW_HEIGHTS)ソース範囲の行の高さを宛先範囲にコピーする列挙型。 Microsoft Excel でこのコードによって生成された出力 Excel ファイルを開くと、宛先範囲の行の高さがソース範囲の行の高さとまったく同じであることがわかります。
