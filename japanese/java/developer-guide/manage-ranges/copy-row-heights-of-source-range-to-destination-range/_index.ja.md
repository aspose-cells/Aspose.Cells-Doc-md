{{% alert color="primary" %}} 

時には、ユーザーがソース範囲の行高さを宛先範囲にコピーする必要があります。Aspose.Cellsは[PasteType.ROW_HEIGHTS](https://reference.aspose.com/cells/java/com.aspose.cells/pastetype#ROW-HEIGHTS)列挙型を提供します。[PasteOptions.setPasteType()](https://reference.aspose.com/cells/java/com.aspose.cells/pasteoptions#PasteType)プロパティをこの列挙型に設定すると、ソース範囲内のすべての行の高さが宛先範囲にコピーされます。

{{% /alert %}}
