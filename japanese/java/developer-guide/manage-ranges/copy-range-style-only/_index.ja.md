[Copy Range Data Only](/cells/ja/java/copy-range-data-only/)と[Copy Range Data with Style](/cells/ja/java/copy-range-data-with-style/)では、範囲から別の範囲にデータを書式付きでコピーする方法について説明しています。この記事では、範囲の書式のみをコピーする方法を示します。

{{% /alert %}} 
## **ソースの範囲の行の高さのみをコピー**
この例では、ワークブックを作成し、データで埋め、範囲のスタイルのみをコピーします。

1. 範囲を作成します。
1. 指定された書式属性を持つスタイルオブジェクトを作成します。
1. 範囲にスタイルを適用します。
1. 別のセルの範囲を作成します。
最初の範囲の書式を2番目の範囲にコピーします。
