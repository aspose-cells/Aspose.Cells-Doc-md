Aspose.Cellsを使用すると、正規表現を使用してブック内のテキストを置換する機能が提供されます。これにより、APIは[**ReplaceOptions**](https://reference.aspose.com/cells/java/com.aspose.cells/ReplaceOptions)クラスの[**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)プロパティを提供します。 [**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)を**true**に設定することで、検索されたキーが正規表現であることを示します。

次のコードスニペットは、[サンプルエクセルファイル](SampleRegexReplace.xlsx)を使用して[**RegexKey**](https://reference.aspose.com/cells/java/com.aspose.cells/replaceoptions#RegexKey)プロパティの使用を示しています。次のコードスニペットによって生成される[出力ファイル](RegexReplace_out.xlsx)が参照用に添付されています。
