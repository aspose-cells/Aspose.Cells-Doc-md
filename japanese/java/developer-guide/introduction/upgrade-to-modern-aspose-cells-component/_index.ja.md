## **Aspose.Cellsへのアップグレード**
以前のユーザー (例: Aspose.Excel ユーザー) が問題を抱えている場合や、バージョンにいくつかの機能が欠けていることに気付いた場合があります。あるいは、要件を満たすために新しい Aspose.Cells にアップグレードする必要がある場合もあります。 Aspose.Excel は現在 Aspose.Cells に名前が変更されていることに注意してください。そのため、Aspose.Excel コンポーネントの古いバージョンを続行または修正することはできません。新しい Aspose.Cells コンポーネントは、強化された高度な機能により、より信頼性が高く、安定して繁栄しています。最も多様なシナリオでパフォーマンスを最適化することに特に重点を置いています。強調しておくと、Aspose.Cells コンポーネントへのアップグレードに伴う労力はそれほど多くありません。ただし、アップグレードすると、

**ご注意ください：** 

一部の API (クラス、構造体、列挙型など) は、現在提供されている高度な機能に基づいて追加されています。以前のクラス (メンバーを含む) は保持および最適化されていますが、いくつかのクラスの名前が変更され、いくつかの追加メンバー (プロパティ、メソッドなど) が追加されています。 .) は、非常に少数が除外されたクラスにも含まれています。
## **以下は、名前が変更されたクラスのリストです。**
1. **エクセル**クラスは**ワークブック**クラス。
1. **エクセルデザイナー**クラスは**WorkbookDesigner**クラス。
1. **エクセルヘルパー**クラスは**セルヘルパー**クラス。
