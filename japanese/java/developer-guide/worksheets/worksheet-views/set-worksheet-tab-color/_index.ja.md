{{% alert color="primary" %}}

Aspose.Cells を使用すると、個々のワークシートタブの色を変更して目立たせることができます。たとえば、Expenses を赤、Sales を緑、Assets を青などにすることができます。

{{% /alert %}}

## **Microsoft Excel でワークシートのタブの色を設定する**

1. 現在のワークシートの下部にあるタブシートでタブを右クリックします。
1. **タブの色**を選択します。
1. パレットから色を選択します。
1. **OK** をクリックします。

**タブは赤、緑、青に色付けされました**
