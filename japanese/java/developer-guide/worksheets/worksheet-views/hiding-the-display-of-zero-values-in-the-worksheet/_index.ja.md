{{% alert color="primary" %}} 

時には、スプレッドシートのゼロ値を非表示にする必要があります。個人の好みであるか、フォーマットの基準であるかもしれません。

{{% /alert %}} 

Microsoft Excel でワークシートでゼロ値を非表示にするには:

1. **ツール** メニューから **オプション** を選択し、次に **表示** タブを選択します。
1. **ゼロ値** オプションを選択解除します。
1. **OK** をクリックします。
