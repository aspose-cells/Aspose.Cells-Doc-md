{{% alert color="primary" %}} 

場合によっては、スプレッドシートでゼロの値を非表示にする必要があります。それは個人的な好みかもしれませんし、書式設定の標準かもしれません。

{{% /alert %}} 

Microsoft Excel (たとえば、Microsoft Excel 2003) でワークシートのゼロ値を非表示にするには:

1. から**ツール**メニュー、選択**オプション**を選択し、**意見**タブ。
1. 選択を解除**ゼロ値**オプション。
1. クリック**わかった**.
