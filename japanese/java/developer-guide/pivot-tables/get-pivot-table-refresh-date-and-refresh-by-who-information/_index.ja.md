{{% alert color="primary" %}}

Aspose.Cells は、ワークブックからの更新日と更新者による情報の取得をサポートするようになりました。

{{% /alert %}}

[**PivotTable.getRefreshDate**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshDate)ピボットテーブル レポートが最後に更新された日付を返します。同様に[**PivotTable.getRefreshByWho**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#RefreshedByWho)プロパティは、前回レポートを更新したユーザーの名前を返します。次の例は、この機能を示しており、サンプル ファイルは次のリンクからダウンロードできます。
