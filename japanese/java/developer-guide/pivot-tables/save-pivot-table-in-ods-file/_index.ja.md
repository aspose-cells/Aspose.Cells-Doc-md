Aspose.Cells は、ピボット テーブルを ODS ファイルに保存する機能を提供します。このためには、既存のピボット テーブルを含むワークブックを変換するか、新しいピボット テーブルを作成して、ファイルを ODS 形式で保存するだけです。必ず電話してください[**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData()を保存する前に、ピボット テーブルが出力 ODS ファイルにレンダリングされることを確認してください。次のコード スニペットは、ピボット テーブルを ODS ファイルに保存する方法を示しています。
