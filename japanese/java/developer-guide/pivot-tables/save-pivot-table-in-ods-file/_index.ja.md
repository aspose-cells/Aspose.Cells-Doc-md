Aspose.Cellsは、ODSファイルにピボットテーブルを保存する機能を提供しています。これには、既存のピボットテーブルを含むワークブックを変換するか、新しいピボットテーブルを作成してファイルをODS形式で保存するだけです。出力されたODSファイルでピボットテーブルが正常にレンダリングされるようにするために、保存前に[**PivotTable.CalculateData()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#calculateData--)を呼び出すことを忘れないでください。以下のコードスニペットは、ODSファイルにピボットテーブルを保存する方法を示しています。
