{{% alert color="primary" %}}

Microsoft Excel では、ピボット テーブルのレイアウトを次のように変更できます。*ピボットテーブル ツール > デザイン > レポート*レイアウト メニュー コマンド。この3つのフォームでレイアウトを変更できます

- コンパクトに表示
- アウトライン形式で表示
- 表形式で表示

Aspose.Cellsも提供しています[**PivotTable.showInCompactForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInCompactForm()), [**PivotTable.showInOutlineForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInOutlineForm() ） と[**PivotTable.showInTabularForm()**](https://reference.aspose.com/cells/java/com.aspose.cells/pivottable#showInTabularForm()) これら 3 つの形式でピボット テーブルのレイアウトを変更するメソッド。
