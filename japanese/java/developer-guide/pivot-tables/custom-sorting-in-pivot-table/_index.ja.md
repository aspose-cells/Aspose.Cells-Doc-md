## **ピボット テーブルでのカスタム並べ替え**
Aspose.Cells API を使用すると、フィールド値でピボット テーブルを並べ替えることができます。次のコード スニペットは、サンプルの Excel ファイルを読み込み、3 つのピボット テーブルを追加します。最初のピボット テーブルはカスタム ソートなしで、2 番目のピボット テーブルは「SeaFood」行フィールド値でソートされ、3 番目のピボット テーブルは「28/07/2000」列フィールド値でソートされています。

サンプル ソース ファイルと出力ファイルは、サンプル コードをテストするために、ここからダウンロードできます。
