## **レポート フィルター ページの表示オプション**

Excel では、ピボット テーブルの作成、レポート フィルターの追加、および [レポート フィルター ページの表示] オプションの有効化がサポートされています。 Aspose.Cells は、作成されたピボット テーブルで [レポート フィルター ページの表示] オプションを有効にするこの機能もサポートしています。以下は、Excel でオプションを表示した画面です。

![todo:画像_代替_文章](show-report-filter-pages-option_1.png)

このオプションの結果として、作成されたワークブックにはより多くのワークシートが含まれます。レポート フィルターのすべての可能な値を別のワークシートに分割します。この例では、「位置」に対するフィルターがあり、データには 3 つの異なる位置 (A、B、C) があります。この機能は、A、B、C という名前の 3 つのワークシートを追加します。これらは同じピボット テーブルですが、オプション A、B、C が事前に選択されています。
