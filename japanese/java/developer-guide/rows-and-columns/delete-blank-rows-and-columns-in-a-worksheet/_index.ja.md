{{% alert color="primary" %}} 

ワークシートからすべての空白の行と列を削除することができます。例えば、Microsoft ExcelファイルからPDFファイルを生成し、データを含む行と列のみを変換したい場合に便利です。

空の行と列を削除するために以下のAspose.Cellsのメソッドを使用します:

1. 空白行を削除するには、[Cells.deleteBlankRows()](https://reference.aspose.com/cells/java/com.aspose.cells/cells#deleteBlankRows--) メソッドを使用します。
1. 空白列を削除するには、[Cells.deleteBlankColumns()](https://reference.aspose.com/cells/java/com.aspose.cells/cells#deleteBlankColumns--) メソッドを使用します。
