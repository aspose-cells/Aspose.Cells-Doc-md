{{% alert color="primary" %}} 

ワークシートから空白の行と列をすべて削除することができます。これは、たとえば、Microsoft Excel ファイルから PDF ファイルを生成し、データを含む行と列のみを変換する場合に便利です。

次の Aspose.Cells メソッドを使用して、空の行と列を削除します。

1. 空白行を削除するには、[Cells.deleteBlankRows()](https://reference.aspose.com/cells/java/com.aspose.cells/cells#deleteBlankRows\(\)） 方法。
1. 空白の列を削除するには、[Cells.deleteBlankColumns()](https://reference.aspose.com/cells/java/com.aspose.cells/cells#deleteBlankColumns\(\)） 方法。
