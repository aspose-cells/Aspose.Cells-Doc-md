Aspose.Cells は、スプレッドシートで形状を管理するための豊富な機能を提供します。図形を適切な場所に整列または配置するために、図形の接続ポイントを取得する必要がある場合があります。この目的のために、すべての接続ポイントが必要です。次のコードを使用して、形状の接続ポイントのリストを取得できます。[**Shape.GetConnectionPoints()**](https://reference.aspose.com/cells/java/com.aspose.cells/shape#ConnectionPoints)関数が使用されます。
