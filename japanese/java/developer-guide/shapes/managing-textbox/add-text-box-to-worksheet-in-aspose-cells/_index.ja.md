Excel でワークシートにテキストボックスを追加

Excel プログラム（07以降のバージョン）では、テキストボックスを挿入できるのは2か所あります。「挿入」の下に「図形プレースメント」と「挿入」オプションの上に右側にもう一箇所あります。

作成方法

水平または垂直のテキストボックスを作成できます。

- 対応するオプション（水平または垂直）を選択します。
- ページ上で左クリックします。
- 左ボタンを押したまま、ページ上で距離をドラッグします。
- 左ボタンを離します。

これでテキストボックスができます。
