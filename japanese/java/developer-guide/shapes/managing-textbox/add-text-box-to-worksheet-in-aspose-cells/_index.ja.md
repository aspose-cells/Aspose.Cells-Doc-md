## 作成方法

横書きまたは縦書きのテキストを含むテキスト ボックスを作成できます。

- 対応するオプションを選択します (水平または垂直)
- ページを左クリック
- 左ボタンを押したまま、ページ上で距離をドラッグします
- 左ボタンを離す

これで、テキスト ボックスが表示されます。
