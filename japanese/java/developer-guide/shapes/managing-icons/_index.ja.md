## Aspose.Cells のワークシートにアイコンを追加

使用する必要がある場合[Aspose.Cells](https://products.aspose.com/cells/)Excel ファイルに「アイコン」を追加するには、このドキュメントが役立ちます。

アイコンの挿入操作に対応する Excel インターフェースは次のとおりです。

![](1.png)

- ワークシートに挿入するアイコンの位置を選択
- 左クリック*入れる*->*アイコン*
- 開いたウィンドウで、上図の赤い長方形のアイコンを選択します
- 左クリック*入れる*、Excelファイルに挿入されます。

効果は次のとおりです。

![](2.png)
