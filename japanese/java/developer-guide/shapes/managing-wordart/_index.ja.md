スプレッドシートに特殊なテキスト効果を追加するためにWordArtを使用します。たとえば、タイトルをファイルの上部に伸ばしたり、テキストを装飾したり、テキストをプリセットされた形に合わせたり、Excelシートにテキストを適用したりします。WordArtはオブジェクトになり、スプレッドシートに配置したり移動したりして装飾を追加できます。

{{% /alert %}}
