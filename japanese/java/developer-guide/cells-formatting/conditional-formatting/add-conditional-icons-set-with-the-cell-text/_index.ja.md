{{% alert color="primary" %}} 

場合によっては、セル内のテキストの横に条件付きアイコンを追加して、読み手にとってデータをより意味のあるものにしたいことがあります。セルに条件付き書式を適用せずに、いくつかの条件付き書式アイコンの種類を使用したい。 Aspose.Cells はこの機能をサポートしています。

{{% /alert %}} 

以下は、XLSX ファイルを最初から作成し、条件付き書式を適用せずにテキストを含むセルに条件付きアイコンを追加するサンプルです。

コードが実行されると、以下に示すように、条件付きアイコン セットの画像がセル領域 "B2:C4" に追加されます。
