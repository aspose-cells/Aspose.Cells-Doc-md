{{% alert color="primary" %}} 

時には、データを読み手にとってより意味のあるものにするために、セル内のテキストの隣に条件付きアイコンを追加したい場合があります。特定の条件付き書式設定のアイコンタイプの一部を使用したいが、セルに条件付き書式設定を適用したくはありません。Aspose.Cellsはこの機能をサポートしています。

{{% /alert %}} 

以下は、条件付きアイコンを追加してセル内のテキストにXLSXファイルをゼロから作成するサンプルですが、その際に条件付き書式設定をセルに適用せずにアイコンを追加しています。

コードを実行すると、条件付きアイコンセットの画像が、「B2:C4」のセル領域に次のように追加されます。
