{{% alert color="primary" %}} 

Excel ファイルの未使用のスタイルは、容量を消費するだけでなく、PDF、HTML などのさまざまな形式に変換するときにパフォーマンスの問題を引き起こします。Aspose.Cells は、[Workbook.removeUnusedStyles()](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#removeUnusedStyles\(\)) を使用して、ブック内の未使用のスタイルをすべて削除します。

{{% /alert %}} 
## **ワークブック内の未使用のスタイルを削除する**
次のコードは、の使用法を説明しています[Workbook.removeUnusedStyles()](https://reference.aspose.com/cells/java/com.aspose.cells/workbook#removeUnusedStyles\(\)）。コードは[テンプレートエクセルファイル](5473451.xlsx)提供されたリンクからダウンロードできます。という名前の未使用のスタイルが含まれています**Asposeスタイル**、このスタイルと他のすべての未使用のスタイルは、コードの実行後に削除されます。詳細については、次のスクリーンショットを参照してください。
