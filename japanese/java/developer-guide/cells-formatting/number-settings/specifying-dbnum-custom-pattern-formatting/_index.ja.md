## **可能な使用シナリオ**

Aspose.Cellsは、*DBNum*のカスタムパターン書式をサポートしています。たとえば、セルの値が123で、そのカスタム書式を [DBNum2][$-804]一般 と指定した場合、それは壹佰贰拾叁のように表示されます。[**Cell.getStyle()**](https://reference.aspose.com/cells/java/com.aspose.cells/cell#getStyle--)および[**Style.setCustom()**](https://reference.aspose.com/cells/java/com.aspose.cells/style/#setCustom-java.lang.String-)メソッドを使用してセルのカスタム書式を指定できます。
