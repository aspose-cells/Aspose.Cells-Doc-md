{{% alert color="primary" %}}

Aspose.Cellsを使用すると、Excelファイルからテーマに関連するデータを抽出できます。たとえば、ワークブックに適用されたテーマの名前やセルまたはセルの境界に適用されたテーマ色などを抽出できます。

Microsoft Excelを使用してワークブックにテーマを適用できます。[ページレイアウト] ＞ [テーマ] コマンドを使用します。
