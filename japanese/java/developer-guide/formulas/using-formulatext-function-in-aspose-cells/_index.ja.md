{{% alert color="primary" %}} 

*FormulaText*はExcel 2013以降の関数です。Excel 2010や2007などの以前のバージョンではサポートされていません。その名前が示すように、これは指定されたセルに含まれている数式のテキストを表示します。この記事では、Aspose.Cellsを使用したこの機能の利用方法を紹介します。

{{% /alert %}}
