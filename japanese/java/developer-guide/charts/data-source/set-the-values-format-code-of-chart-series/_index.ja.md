## **考えられる使用シナリオ**

を使用して、チャート シリーズの値の形式コードを設定できます。[**Series.ValuesFormatCode**](https://reference.aspose.com/cells/java/com.aspose.cells/series#ValuesFormatCode)財産。このプロパティは、ワークシート内の範囲に基づくシリーズに役立つだけでなく、値の配列で作成されたシリーズにも適しています。

## **チャート シリーズの値のフォーマット コードを設定する**

次のサンプル コードは、シリーズがない空のチャートにシリーズを追加します。値の配列を使用してシリーズを追加します。シリーズが追加されると、コード $#,##0 でフォーマットされます[**Series.ValuesFormatCode**](https://reference.aspose.com/cells/java/com.aspose.cells/series#ValuesFormatCode)プロパティと番号 10000 は $10,000 になります。スクリーンショットは、コードの効果を示しています[サンプル Excel ファイル](51740736.xlsx)と[出力エクセルファイル](51740735.xlsx)実行後。
