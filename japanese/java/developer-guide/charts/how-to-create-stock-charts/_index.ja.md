この段落では、次の 4 つのタイプを含む株価チャートの作成方法について説明します。
- **HLC** – 高値、安値、終値。
- **OHLC** – 始値-高値-安値-終値。
- **VHLC** – 出来高-高値-安値-終値。
- **VOHLC**– 出来高-始値-高値-安値-終値。

{{% /alert %}}

##  **株価チャートとは何ですか?**

株価チャートは、取引される資産の価格の変化を追跡するために使用される特定のチャートです。商品、株式、暗号通貨などの資産。これらを使用すると、時間の経過に伴う高値と安値、始値と終値を 1 つのチャートで確認できます。 Aspose.Cells には 4 つの株価チャートがあり、これらを使用するには、適切なデータ セットが利用可能であり、列を正しい順序で選択する必要があります。
