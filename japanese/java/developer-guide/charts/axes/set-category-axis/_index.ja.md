##  **考えられる使用シナリオ**
ワークシートにグラフを作成した後、そのグラフにカテゴリ軸を設定できます。この記事では、Aspose.Cellsを使ってExcelグラフに項目軸を設定する方法をサンプルコード付きで紹介します。

##  **サンプルコードの手順**

1. 新しいワークブックを作成します。

2. 最初のワークシートに新しいグラフを作成します。

3. 最初のワークシートのセルに値を追加します。

4. これでカテゴリ軸を設定できるようになりました。方法は 2 つあります。セル データを使用する方法と、文字列を直接使用する方法です。どちらもサンプル コードに示されています。

5. 値軸を設定し、ワークブックを保存して結果を表示します。
