{{% alert color="primary" %}} 

チャートの凡例エントリ塗りのテキストを無しに設定して、チャートの凡例内に表示されないようにするには、[LegendEntry.IsTextNoFill](https://reference.aspose.com/cells/java/com.aspose.cells/legendentry#IsTextNoFill) を **true** に設定してください。

{{% /alert %}}
