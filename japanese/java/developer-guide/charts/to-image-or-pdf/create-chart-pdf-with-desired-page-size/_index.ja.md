## **考えられる使用シナリオ**
Aspose.Cells を使用して希望のページ サイズでチャート PDF を作成し、ページ内でチャートをどのように配置するかを指定できます。チャートを上、下、中央、左、右などに揃えることができます。さらに、出力チャートはストリームまたはディスク上に作成できます。
## **目的のページ サイズでチャート PDF を作成する**
をロードする次のサンプル コードを参照してください。[サンプル Excel ファイル](64716912.xlsx)、ワークシート内の最初のチャートにアクセスし、それを次のように変換します[PDF出力](64716911.pdf)希望のページサイズで。次のスクリーンショットは、出力 PDF のページ サイズがコード内で指定されている 7x7 であり、チャートが水平方向と垂直方向の両方で中央揃えになっていることを示しています。
