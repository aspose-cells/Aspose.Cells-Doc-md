## **可能な使用シナリオ**
Microsoft Excelでスライサーをフォーマットすることができます。Aspose.Cellsでも、[Slicer.NumberOfColumns](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#NumberOfColumns) および[Slicer.StyleType](https://reference.aspose.com/cells/java/com.aspose.cells/slicer#StyleType) プロパティを使用して同じことができます。
## **スライサーの書式設定**
次のコードをご覧ください。スライサーが含まれる[sample Excel file](67338501.xlsx)をロードし、スライサーにアクセスして、その列数とスタイルタイプを設定し、[output Excel file](67338500.xlsx)として保存します。スクリーンショットは、サンプルコードの実行後にスライサーがどのように見えるかを示しています。
