## **可能な使用シナリオ**
Aspose.Cellsはスライサーのレンダリングをサポートしています。ワークシートを画像に変換したり、ワークブックをPDFまたはHTML形式で保存すると、スライサーが適切にレンダリングされます。
## **スライサーのレンダリング**
次のサンプルコードは、既存のスライサーが含まれる[sample Excel file](67338485.xlsx)をロードし、スライサーをカバーする印刷エリアを設定してワークシートをイメージに変換します。生成された画像は、[output image](67338486.png)として示されています。
