{{% alert color="primary" %}}

Microsoft ExcelはExcelブック内に**署名行**を追加する機能を提供しています。**挿入**タブをクリックし、**テキスト**グループから**署名行**を選択して、署名行を追加できます。

Aspose.Cellsもこの機能を提供し、この目的に[**Picture.setSignatureLine()**](https://reference.aspose.com/cells/java/com.aspose.cells/picture#SignatureLine)プロパティを公開しています。この記事では、このプロパティを使用して署名行を追加する方法について説明します。

{{% /alert %}}
