**Office Open XML** （または**OOXML**、**Microsoft Open XML**（MOX）としても知られる）は、Microsoftによって開発されたXMLベースのファイル形式で、スプレッドシート、チャート、プレゼンテーション、ワード処理などのオフィスドキュメントを表現するために使用されます。

{{% /alert %}}
