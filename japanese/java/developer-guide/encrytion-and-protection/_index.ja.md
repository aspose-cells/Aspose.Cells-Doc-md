Aspose.Cells for Java は、ファイルから MS Excel のようなデータ レベルまで、幅広い保護と保護解除を提供します。
{{% /alert %}}


## [**Excel ファイルの暗号化と復号化**](/cells/ja/java/encrypt-and-decrypt-excel-files/)
他人が Excel ファイルのデータにアクセスできないようにするには、Excel ファイルをパスワードで保護します。

## [**ブックの保護と保護解除**](/cells/ja/java/protect-and-unprotect-workbook-structure/)
他のユーザーが非表示のワークシートを表示したり、ワークシートを追加、移動、削除、非表示にしたり、ワークシートの名前を変更したりできないようにするために、Excel ブックの構造をパスワードで保護できます。

## [**ワークシートの保護と保護解除**](/cells/ja/java/protect-and-unprotect-worksheet/)
他のユーザーが誤ってまたは故意にワークシート内のデータを変更、移動、または削除するのを防ぐために、Excel ワークシートのセルをロックし、シートをパスワードで保護することができます。

## [**共有ブックの保護と保護解除**](/cells/ja/java/password-protect-or-unprotect-the-shared-workbook/)
共有ブックを保護および保護解除します。

## [**Office ファイルのデジタル署名**](/cells/ja/java/assign-and-validate-digital-signatures/)
ドキュメントのコンテンツの信頼性を保護するには、デジタル署名を追加できます。
