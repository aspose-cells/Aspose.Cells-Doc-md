## **可能な使用シナリオ**

Aspose.Cells APIを使用して、ブックを書き込み保護する際に著者名を指定できます。この目的で [**Workbook.Settings.WriteProtection.Author**](https://reference.aspose.com/cells/java/com.aspose.cells/writeprotection#Author) プロパティを使用してください。
