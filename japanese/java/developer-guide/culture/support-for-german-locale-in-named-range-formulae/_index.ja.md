英語の数式は、名前付きの領域に書き込まれます。この Excel ファイルは、システムがドイツ語ロケールに設定されている環境で開くことができますが、英語の式はドイツ語に翻訳されます。次の例はこの機能を示していますが、Excel をドイツ語でインストールする必要があり、システム ロケールもドイツ語に設定する必要があります。
