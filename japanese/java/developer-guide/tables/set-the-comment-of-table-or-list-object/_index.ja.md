## **ワークシート内のテーブルまたはリストオブジェクトのコメントを設定してください**

ワークシート内のテーブルまたはリストオブジェクトのコメントを[**ListObject.Comment**](https://reference.aspose.com/cells/java/com.aspose.cells/listobject#Comment)プロパティを使用して設定してください。コメントはxl/tables/tableName.xmlファイル内で表示されます。

次のサンプルコードは、[ソースエクセルファイル](5472561.xlsx)を読み込み、ワークシート内の最初のテーブルまたはリストオブジェクトのコメントを設定し、このコードによって作成されたコメントを赤い矩形で示したスクリーンショットを示しています。
