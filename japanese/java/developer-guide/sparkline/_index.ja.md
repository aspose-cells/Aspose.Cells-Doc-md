## **スパークラインを挿入する**
{{% alert color="primary" %}} 

スパークラインは、ワークシートセル内の小さなチャートであり、データの視覚的表現を提供します。スパークラインを使用して、値の系列における傾向、季節ごとの増減、経済サイクル、最大値や最小値のハイライトなどを表示できます。最大の効果を得るために、データの近くにスパークラインを配置します。スパークラインには、ライン、カラム、スタックドの3つのタイプがあります。

{{% /alert %}}
