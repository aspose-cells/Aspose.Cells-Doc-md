## **スパークラインを挿入する**
{{% alert color="primary" %}} 

スパークラインは、データを視覚的に表現するワークシート セル内の小さなグラフです。スパークラインを使用して、季節的な増減、景気循環など、一連の値の傾向を示したり、最大値と最小値を強調したりします。スパークラインをデータの近くに配置すると、最大の効果が得られます。スパークラインには、折れ線、列、積み上げの 3 種類があります。
