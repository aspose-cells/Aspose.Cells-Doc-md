{{% alert color="primary" %}} 

外部データ ソースに関する情報を取得したい場合があります。たとえば、SQL 接続データを確認したい場合があります。この情報には、サーバー URL、ユーザー名、基になるテーブル名、SQL クエリ、クエリの種類、テーブルの場所など、SQL サーバーに接続するために必要なあらゆる種類のデータを含めることができます。

Aspose.Cells は、テンプレート Excel ファイルにリンクされた企業データベースからそのような詳細を取得するための便利な呼び出しを提供します。
