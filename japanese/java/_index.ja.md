**Aspose.Cells for Javaへようこそ**

Aspose.Cells for Java は、ソフトウェア開発者が自分のアプリケーション内でスプレッドシートファイルを操作、処理できるようにするプログラミングクラスライブラリです。

APIとGUIコントロールと組み合わせることで、Aspose.Cells for Java はMicrosoft Excel®のプログラミングと変換をスピードアップします。

Aspose.Cells for Java は、ビジネスで毎日使用される人気のあるスプレッドシート(XLS、XLSX、XLSM、XLSB、XLTX、XLTM、CSV、SpreadsheetML、ODS)ファイルフォーマットをサポートしています。また、ExcelファイルをPDF、DOCX、PPTX、JSON、XPS、HTML、MHTML、プレーンテキストやTIFF、JPG、PNG、BMP、SVGなどの一般的な画像形式にエクスポートすることも可能です。
