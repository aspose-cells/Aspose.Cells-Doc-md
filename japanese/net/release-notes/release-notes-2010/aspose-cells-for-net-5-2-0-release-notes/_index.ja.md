{{% alert color="primary" %}} 

このページには、[Aspose.Cells for .NET 5.2.0](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-5.2.0/)

{{% /alert %}} 

Aspose.Cells for .NET v5.2.0 を発表できることをうれしく思います。

 Aspose.Cells の変更点:

- XLSX ファイルのデジタル署名の署名と検証をサポートします。
 XLS および XLSX ファイルの R/W ピボット テーブルをサポートします。
ピボット テーブル データをセルに計算します。
 31の強化と修正。

 Aspose.Cells.GridWeb の変更点:

- MS Excel の画像のインポート/エクスポートをサポートします。



 Aspose.Cells.GridDesktop の変更点:

- 名前付き範囲をサポートします。
 1 つの修正と 1 つの強化。
