お客様各位

Aspose.Cells for .NET v4.1 をリリースしました！

- 条件付き書式をゼロから作成するための API を提供
- ピボット テーブルをゼロから作成するための API を提供
- CheckBox クラスの LinkedCell プロパティを追加
- 新しい C# クイック スタート デモを追加
- 式の設定における2つの小さなバグを修正
- 計算式のバグを修正
- Windows Vista RC2 で Aspose.Cells を使用する際の問題を修正しました。この問題は難読化ツールによって引き起こされます。
- PDF ファイルへの変換中のページの向きのバグを修正しました
- 共有数式で行をコピーするバグを修正しました
