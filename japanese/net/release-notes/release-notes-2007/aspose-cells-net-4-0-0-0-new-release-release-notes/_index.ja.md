お客様各位

Aspose.Cells for .NET V4.0 をリリースしました！

- クラス名の変更: ExcelHelper->CellsHelper、ExcelException->CellsException
- AutoFitRow メソッドを整形してより正確に
- 大量の名前付き範囲を追加する際の問題を解決するために Worksheets.SortNames メソッドを追加しました。
- 新しいチャートのデモを追加
- 数式設定のバグを修正
- ファイルを CSV 形式でストリームに保存する際のバグを修正しました
