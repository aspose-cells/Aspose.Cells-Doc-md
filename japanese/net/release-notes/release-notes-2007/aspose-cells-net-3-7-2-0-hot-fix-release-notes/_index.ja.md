Aspose.Cells for .NET v3.7.2 をリリースしました！

- より多くの Excel 関数をサポートするための強化された数式計算エンジン
- Excel ファイルの読み込み中の最適化されたパフォーマンス
- ファイルを SpreadsheetML としてエクスポートするバグを修正しました
- チャートの TickLabel 設定のバグを修正しました
- 多くのコメントの挿入とリスト検証のバグを修正
