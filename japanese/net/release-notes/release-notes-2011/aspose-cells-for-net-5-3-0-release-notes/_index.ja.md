{{% alert color="primary" %}} 

このページには、[Aspose.Cells for .NET 5.3.0](https://downloads.aspose.com/cells/net/new-releases/aspose.cells-for-.net-5.3.0/)

{{% /alert %}} 

Aspose.Cells for .NET v5.3.0 を発表できることをうれしく思います。

 Aspose.Cells の変更点:

- Chart-to-Image 機能が強化されました。
 Excel-to-PDF 機能が強化されました。
 R/W ODS ファイル機能が強化されました。
行の自動調整機能が強化されました。
 63 の機能強化と修正。

 Aspose.Cells.GridWeb の変更点:

- FireFox のコントロールのすべての機能をサポートします。
リスト コレクションを DataTable オブジェクトに変換するためのサポートを提供します。
 SMALL 関数/式をサポートします。
 Excel ファイルにオートフィルターをインポートします。
 IE のクライアント スクリプトのパフォーマンスが強化されました。
数式を計算するときに最大行/列を自動的に拡張するかどうかをサポートします。
 1 つの修正と 1 つの強化。
