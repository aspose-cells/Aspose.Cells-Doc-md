## **紹介**
Aspose .NET Import Users from Excelモジュールは、Excelファイルから大量のユーザーをインポートすることができます。このモジュールは、Aspose.Cellsのパワフルなスプレッドシート機能を示しています。
### **モジュールの機能**
この初期バージョンのモジュールには、次の優れた機能が備わっており、インポートプロセスをシンプルかつ使いやすくしています:

- Excelファイルをアップロードし、既存のユーザーを自動的にフィルタリングします。
- 既存のユーザーを自動的にスキップして、混乱を避けます。
- 新しく作成されたユーザーに動的なパスワードを割り当てます。
- 「登録ユーザー」と「サブスクライバー」のセキュリティロールを新しく作成されたユーザーに割り当てます。

## **インストール**
ダウンロードしたら、次の手順に従ってモジュールをDNNウェブサイトにインストールしてください

1. **ホスト**メニューに移動し、**拡張機能**を選択します
1. **拡張機能ウィザードのインストール**をクリックします
1. 指示に従い、ダウンロードしたzipファイルの場所を参照し、選択して**開く**をクリックします
1. **次へ**をクリックし、ライセンスを受け入れ、インストールを続行します。完了したら**戻る**ボタンをクリックします
[このモジュールのインストールビデオ](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)を詳細についてご確認ください

詳細については、DNNの[このモジュールのインストールビデオ](https://www.dnnsoftware.com/community/learn/video-library/view-video/video/542/view/details/how-to-install-a-module-in-dotnetnuke-7)をご確認ください。
## **を使用する**
Aspose .NET Cells Import Users from Excel for DNNモジュールをインストールした後、ウェブサイトで使用を開始することは非常に簡単です。次の簡単な手順に従って開始してください

1. **ホスト**または**管理者**レベルのアカウントでDNNにログインしてください
1. エクスポートモジュールを追加したいページに移動してください
1. 上部リボンから**モジュール**の後に**新しいモジュールの追加**を選択してください 

1. リストから『Aspose Import Users from Excel module for DNN』を選択し、それを適切な位置にドラッグアンドドロップしてください

Aspose .NET Cells Import Users from Excelモジュールをページに正常に追加しました。ファイルアップロードコントロールと「テンプレートファイル」リンクが表示されます。テンプレートファイルをダウンロードし、内容をテンプレートファイルに従って書式設定できます。Excelファイルをアップロードすると、自動的にすべてのユーザーが処理され、既存のユーザーがフィルタリングされます。インポートするユーザーを選択し、「ユーザーのインポート」ボタンを押すことができます
