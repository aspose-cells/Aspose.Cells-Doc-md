以下は、ツールを使用するために従う必要があるスクリーンショットの手順です。

![todo:画像_代替_文章](using-and-configuring-crm-update-import-tool_1)

-  CRM との接続を作成するには、[CRM にログイン] をクリックし、CRM 組織情報を追加します。

![todo:画像_代替_文章](using-and-configuring-crm-update-import-tool_2)

- 接続が確立されたら、エンティティを選択し、目的の属性のリストを選択します。

![todo:画像_代替_文章](using-and-configuring-crm-update-import-tool_3)

- **CRM から取得:**ここをクリックして、CRM からのデータをグリッドに表示します
- **Excel にエクスポート:**ここをクリックして、ファイルを提供して CRM から Excel ファイルにデータをエクスポートします。
- **Excel から取得:**ここをクリックしてソース Excel ファイルを選択すると、Excel ファイルのデータがグリッドに表示されます。
- **CRM へのエクスポート:**ここをクリックしてソース Excel ファイルを選択すると、選択したエンティティの CRM に Excel からデータがインポートされます。 (Excel ファイルの列名は CRM のスキーマ名と一致する必要があることに注意してください)
