以下は、ツールを使用するために従う必要があるスクリーンショットの手順です:

![todo:image_alt_text](using-and-configuring-crm-update-import-tool_1)

- CRMとの接続を作成するには、CRMへのログインをクリックし、CRM組織情報を追加します 

![todo:image_alt_text](using-and-configuring-crm-update-import-tool_2)

- 接続が確立されたら、エンティティを選択し、必要な属性のリストを選択します。 

![todo:image_alt_text](using-and-configuring-crm-update-import-tool_3)

- **CRMから取得:** ここをクリックして、CRMからのデータをグリッドに表示します
- **Excelにエクスポート:** ここをクリックして、ファイルを提供してCRMからExcelにデータをエクスポートします。
- **Excelから取得:** ここをクリックし、ソースのExcelファイルを選択すると、Excelファイルのデータがグリッドに表示されます。
- **CRMにエクスポート:** ここをクリックし、ソースのExcelファイルを選択すると、ExcelからCRMにデータをインポートします。 (Excelファイルの列名がCRMのスキーマ名と一致している必要があることに注意してください)
