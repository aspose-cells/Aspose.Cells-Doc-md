## **序章**

Microsoft Visual Studio は、Microsoft の統合開発環境 (IDE) です。Microsoft Windows のコンピューター プログラム、および Web サイト、Web アプリケーション、Web サービスの開発に使用されます。 Visual Studio は、Windows API、Windows Forms、Windows Presentation Foundation、Windows Store、Microsoft Silverlight などの Microsoft ソフトウェア開発プラットフォームを使用します。ネイティブ コードとマネージ コードの両方を生成できます。

### **Aspose.Cells for .NET**

Aspose.Cells for .NET は、サーバーに Microsoft Excel® をインストールすることなく、.NET アプリケーションが Excel® スプレッドシートを作成、変更、変換、レンダリング、および印刷できるようにする最も柔軟なコンポーネント グループを提供します。

Aspose.Cells for .NET は、すべての主要な Microsoft Excel およびその他のファイル形式 (XLS、XLSX、XLSM、XLTX/XLTM、HTML、CSV、ODS、タブ区切り、SpreadsheetML、PDF など) をサポートしています。
詳細については、[Aspose.Cells for .NET API](https://products.aspose.com/cells/net/)

### **Aspose.Cells for .NET Visual Studio を使用した例を調べる**

Aspose Visual Studio プラグイン -**Aspose.Cells for .NET の例を調べる**Aspose.Cells for .NET API の例をすばやくダウンロードして探索するための優れたツールです。 Visual Studio を離れることなく、最新のサンプル プロジェクトをシームレスに選択、ダウンロード、開くための非常にシンプルなオプションを提供することで、多くの時間と労力を節約できます。

このバージョンは、次の機能を提供します。

- Visual Studio 2010 と Visual Studio 2012 と Visual Studio 2013 をサポートします。
- から簡単に起動**ファイル > Aspose** Visual Studio のメニュー

- 開くことを許可する**C#**また**VB.NET**サンプル プロジェクト
- 選択したプロジェクトは Visual Studio で自動的に開かれ、必要なすべての参照が設定されており、すぐに実行して探索できる環境が提供されます。

#### **ソースコードの構成方法**

ソースコードを開いて拡張するには、以下をインストールする必要があります

開始するには、次の簡単な手順に従ってください

1. ソースコードをダウンロード/クローンします。
1. Visual Studio 2010 を開き、選択します**ファイル** > **プロジェクトを開く**
1. ダウンロードして開いた最新のソースコードを参照します**AsposeVisualStudioPlugin.sln**

#### **ソースコードの概要**

ソリューションには 2 つのプロジェクトがあります

- **AsposeVisualStudioプラグイン** Visual Studio との統合、Aspose API とサンプルの選択と自動ダウンロードが含まれています。
- **インストーラ** プラグイン用の msi インストーラー ファイルを生成する Wix インストーラー プロジェクト。
