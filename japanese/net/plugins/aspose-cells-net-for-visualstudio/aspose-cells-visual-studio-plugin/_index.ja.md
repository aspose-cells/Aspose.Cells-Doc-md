## **紹介**

Microsoft Visual Studio は、マイクロソフトの統合開発環境（IDE）です。Microsoft Windows 向けのコンピュータプログラムや、Web サイト、Web アプリケーション、Web サービスを開発するために使用されます。Visual Studio は、Windows API、Windows Forms、Windows Presentation Foundation、Windows Store、Microsoft Silverlight などの Microsoft ソフトウェア開発プラットフォームを使用します。ネイティブコードと管理コードの両方を生成できます。

### **Aspose.Cells for .NET**

Aspose.Cells for .NET は、Microsoft Excel® をサーバーにインストールせずに、.NET アプリケーションがExcel®スプレッドシートを作成、変更、変換、描画、印刷するための最も柔軟なコンポーネント群を提供します。

Aspose.Cells for .NET は、すべての主要な Microsoft Excel およびその他のファイル形式（XLS、XLSX、XLSM、XLTX/XLTM、HTML、CSV、ODS、タブ区切り、SpreadsheetML、PDF など）をサポートしています。
[Aspose.Cells for .NET API](https://products.aspose.com/cells/net/)の詳細について

### **Visual Studio を使用してAspose.Cells for .NETの例を探索する**

Aspose Visual Studio プラグイン - **Aspose.Cells for .NET の例を探索** は、最新の例プロジェクトを簡単に選択してダウンロードし、Visual Studio を離れることなく開くことができるため、多くの時間と労力を節約できます。

このバージョンには、次のような機能が提供されます：

- Visual Studio 2010 および Visual Studio 2012 および Visual Studio 2013 をサポート
- Visual Studio の **Aspose** メニューから簡単に起動できます
- 時間を節約し、学習曲線を短縮します。
- Github.com でホストされている最新の Aspose.Cells for .NET API の例を選択して開くことができます。
- **C#** または **VB.NET** の例プロジェクトを開くことができます
- 選択したプロジェクトは、必要な参照がすべて設定された状態で、Visual Studio で自動的に開かれ、実行および調査が可能な環境が提供されます。
- **NuGet** から最新の API ライブラリと **GitHub** から例を自動的にダウンロードします。

## **インストール**

Aspose Visual Studio Plugin のインストールは非常に簡単です。次の簡単な手順に従ってください

- ダウンロードした msi ファイルをダブルクリックします
- ステップに従って、標準設定を使用してインストールする場合は、各画面で単に「次へ」を押すことができます。

インストールが完了すると成功のダイアログが表示されます。

![todo:image_alt_text](aspose-cells-visual-studio-plugin_2)

**注意:** 変更を反映させるためにVisual Studioを再起動してください。

## **を使用する**

インストールされると、以下に説明するように、対話型プラグインを簡単に起動して使用できます
**1.** Visual Studio で **ファイル > Aspose** を選択し、次に **新しい Aspose.Cells 例プロジェクト** を選択してプラグインを開きます。

![todo:image_alt_text](aspose-cells-visual-studio-plugin_3)

**2.** プラグインを初めて開くと、以下に示すような API 選択ダイアログが表示されます。最新の API およびそれらのライブラリを **NuGet** から、そして最新の例題のソースコードを **GitHub** からダウンロードするには、数分かかります。

![todo:image_alt_text](aspose-cells-visual-studio-plugin_4)

**3.** 「例のプロジェクトが作成される場所」を選択または変更し、「例のプロジェクトの種類」を **C#** または **VB.NET** に選択または変更し、「続行」をクリックして、Visual Studio で例プロジェクトを開きます。

![todo:image_alt_text](aspose-cells-visual-studio-plugin_1)

**4.** Visual Studio で **C#** または **VB.NET** のプロジェクトが新しく作成されます。

## **サポート、拡張、貢献**

### **サポート**

Asposeが立ち上がって最初の日から、良い製品だけを提供するだけでは不十分だと分かっていました。良いサービスも提供する必要がありました。私たち自身も開発者であり、技術的な問題やソフトウェアの不具合が必要なことを妨げるときにどれだけイライラするか理解しています。私たちは問題を解決するためにここにいて、それを作り出すためではありません。

そのため、無料サポートを提供しています。製品を購入したか、評価を使用しているかに関わらず、私たちの製品を使用するすべての人にフルの注意と尊敬を提供する価値があります。

Aspose Visual Studio Plugin に関連する問題や提案については、以下のいずれかのプラットフォームを使用してログを残すことができます

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-.NET/issues)

### **拡張と貢献**

Visual Studio 用の **Aspose.Cells for .NET の例** はオープンソースです。そのソースコードは以下にリストされた主要なソーシャルコーディングウェブサイトで利用できます。開発者はソースコードをダウンロードし、自分の要件に応じて機能を拡張することを推奨します。

#### **ソースコード**

最新のソースコードを以下の場所から取得できます

- [Github](https://github.com/aspose-cells/Aspose.Cells-for-.NET)


#### **ソースコードの構成方法**

以下のアイテムがインストールされている必要があります

- Visual Studio 2010

開始するための簡単なステップに従ってください

1. ソースコードをダウンロード/クローンします。
1. Visual Studio 2010を開き、**ファイル** > **プロジェクトを開く** を選択してください
1. ダウンロードした最新のソースコードに移動し、**AsposeVisualStudioPlugin.sln** を開きます

#### **ソースコードの概要**

このソリューションには2つのプロジェクトがあります

- **AsposeVisualStudioPlugin** - Visual Studio との統合、Aspose API および例の選択および自動ダウンロードを含みます
- **Installer** - プラグインのための msi インストーラファイルを生成するための Wix インストーラプロジェクトです
