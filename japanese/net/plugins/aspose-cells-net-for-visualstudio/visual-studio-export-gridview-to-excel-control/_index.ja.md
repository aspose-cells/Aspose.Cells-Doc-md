## **序章**
Export GridView To Excel Control は ASP.NET サーバー コントロールで、GridView のコンテンツを Microsoft Excel または OpenOffice スプレッドシートにエクスポートできます。[Aspose.Cells](https://products.aspose.com/cells/net/) .追加します**Excel にエクスポート**GridView コントロールの上にあるボタン。ボタンをクリックすると、GridView コントロールのコンテンツが Microsoft Excel または OpenOffice スプレッドシートに動的にエクスポートされ、エクスポートされたファイルがユーザーが選択したディスクの場所に数秒で自動的にダウンロードされます。
### **モジュールの機能**
コントロールのこの初期バージョンは、次の機能を提供します。

- お気に入りのオンライン GridView コンテンツのオフライン コピーを取得して、編集、共有、印刷することができます。
- デフォルトの ASP.NET GridView コントロールから継承されているため、そのすべての機能とプロパティを備えています。
- GridView を Xlsx、Xlsb、Xls、Txt、Csv、Ods にエクスポートします。
- .NET 2.0 以降のすべての .NET バージョンで動作します。
- エクスポート ボタンのテキストをカスタマイズ/ローカライズする機能。
- css を使用して、[エクスポート] ボタンに独自のテーマのルック アンド フィールを適用します。
- エクスポートされたドキュメントの上にカスタム見出しを追加するオプション
- エクスポートされた各ドキュメントをサーバー上の構成可能なディスク パスに保存するオプション
- ページングが有効な場合に現在のページまたはすべてのページをエクスポートするオプション

このコントロールを使用すると、GridView を次のさまざまなファイル形式でエクスポートできます。

1. GridView を Excel にエクスポートする
1. GridView を Xlsx にエクスポートする

## **インストール**
Export GridView To Excel Control のインストールは非常にシンプルで簡単です。次の簡単な手順に従ってください。
### **Visual Studio 2010、2012、および 2013 の場合**
1. ダウンロードしたzipファイルを解凍します
1. VSIX ファイル Aspose.Excel.GridViewExport.vsix をダブルクリックします。
1. ダイアログが表示され、マシンにインストールされている利用可能でサポートされている Visual Studio のバージョンが表示されます。
1. Export GridView To Excel Control を追加するものを選択します。
1. [インストール] をクリックします

インストールが完了すると、成功のダイアログが表示されます。

**ノート：**変更を有効にするために、必ず Visual Studio を再起動してください。
### **Visual Studio 2005、2008、および Express エディションの場合**
次の手順に従って、Export GridView To Excel Control を Visual Studio に統合し、他の ASP.NET コントロールと同様に簡単にドラッグ アンド ドロップしてください。

1. ダウンロードしたzipファイルを解凍します
1. Visual Studio を管理者として実行してください。

[ツール] メニューの [ツールボックス アイテムの選択] をクリックします。

1.  [参照] をクリックします。
 [開く] ダイアログ ボックスが表示されます。
1. 解凍したフォルダーを参照し、Aspose.Excel.GridViewExport.dll を選択します。
1. [OK] をクリックします。

左側のツールボックスで aspx または ascx コントロールを開くと、[全般] タブの下に ExportGridViewToWord が表示されます。

### **ExportGridViewToExcel コントロールを手動で追加する**
Visual Studio Toolbox を使用する上記の方法を使用して問題が発生した場合は、2.0 以上の .NET フレームワークで実行されている ASP.NET アプリケーションにこのコントロールを手動で追加できます。

1. Visual Studio を使用している場合は、必ず管理者として実行してください。
1. への参照を追加**Aspose.Excel.GridViewExport.dll**ASP.NET プロジェクトまたは Web アプリケーションの抽出されたダウンロード パッケージで利用できます。 Web アプリケーション/Visual Studio がこのフォルダーに完全にアクセスできることを確認してください。そうしないと、Access is denied 例外が発生する可能性があります。
1. この行をページ、コントロール、または MasterPage の先頭に追加します

{{< highlight "java" >}}

 <%@ Register assembly="Aspose.Excel.GridViewExport" namespace="Aspose.Excel.GridViewExport" tagprefix="aspose" %>

{{< /highlight >}}

1. コントロールを追加する ASP.NET ページ、コントロール、またはマスターページの場所に次を追加します。

{{< highlight "java" >}}

 <aspose:ExportGridViewToExcel ID="ExportGridViewToExcel1" runat="server"></aspose:ExportGridViewToExcel>

{{< /highlight >}}

### **Aspose .NET GridView を Excel コントロールのプロパティにエクスポート**
次のプロパティは、このコントロールによって提供される優れた機能を構成および使用するために公開されています

|**プロパティ名** |**タイプ** |**例/可能な値** |**説明** |
|:- |:- |:- |:- |
|ExportButtonText|ストリング|Excel にエクスポート|このプロパティを使用して、既存のデフォルト テキストを上書きできます|
|ExportButtonCssClass|ストリング| btn btn-プライマリ|エクスポート ボタンの外側の div に適用される Css クラス。ボタンに css を適用するには、.yourClass 入力を使用できます|
|ExportFileHeading|ストリング|<h4>GridView エクスポート サンプル レポート</h4> |html タグを使用して見出しにスタイルを追加できます|
|ExportOutputFormat|列挙|Xlsx、Xlsb、Xls、Txt、Csv、オッズ|エクスポートされたドキュメントの出力形式。サポートされている形式は、Xlsx、Xlsb、Xls、Txt、Csv、Ods です。|
| ExportOutputPathOnServer|ストリング| c:<br>温度|エクスポートのコピーが自動的に保存されるサーバー上のローカル出力ディスク パス。アプリケーションには、このパスへの書き込みアクセスが必要です。|
| ExportDataSource|物体|allRowsDataTable|このデータ バインド コントロールがデータ項目のリストを取得するオブジェクトを設定します。オブジェクトには、エクスポートする必要があるすべてのデータが含まれている必要があります。このプロパティは、通常の DataSource プロパティに加えて使用され、カスタム ページングが有効で、現在のページが画面に表示される行のみをフェッチする場合に役立ちます。|
| LicenseFilePath|ストリング||サーバー上のライセンス ファイルへのローカル パス。たとえば、c:<br> inetpub<br> Aspose.Cells.lic|
すべてのプロパティが使用されている GridView を Excel コントロールにエクスポートする例を以下に示します。

{{< highlight "java" >}}

 <aspose:ExportGridViewToExcel Width="800px" ID="ExportGridViewToExcel1" ExportButtonText="Export to Excel"

    CssClass="table table-hover table-bordered" ExportButtonCssClass="myClass" ExcelOutputFormat="xlsx"

    ExportOutputPathOnServer="c:\\temp" ExportFileHeading="<h1>Example Report</h1>"

    OnPageIndexChanging="ExportGridViewToExcel1_PageIndexChanging" PageSize="5" AllowPaging="True"

    LicenseFilePath="c:\inetpub\Aspose.Cells.lic" runat="server" CellPadding="4"

    ForeColor="#333333" GridLines="Both">

</aspose:ExportGridViewToExcel>


{{< /highlight >}}

- Aspose.Excel.GridViewExport.Website - Excel のエクスポート可能な GridView コントロールをテストするための Web プロジェクト
