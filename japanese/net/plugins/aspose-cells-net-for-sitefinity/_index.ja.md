### **序章**
#### **サイトフィニティとは？**
Sitefinity は最新の ASP.NET ベースの Web コンテンツ管理システム (CMS) であり、ビジネス プロフェッショナル向けの直感的な Web コンテンツ管理と堅牢な開発環境を提供します。この人気のある CMS 用に作成したプロジェクトを次に示します。
#### **Aspose.Cells for .NET**
Aspose.Cells for .NET は高度なクラス ライブラリ for .NET で、.NET アプリケーション内でさまざまなスプレッドシート処理タスクを直接実行できます。

Aspose.Cells を使用すると、Microsoft Excel を使用せずに、スプレッドシートを生成、変更、変換、レンダリング、および印刷できます。

Aspose.Cells for .NET は、XSL、XSLX、OpenOffice などの多くの形式をサポートしています。
