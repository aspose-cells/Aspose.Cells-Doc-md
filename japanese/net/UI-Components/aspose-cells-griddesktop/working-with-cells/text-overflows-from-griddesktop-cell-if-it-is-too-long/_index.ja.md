## **考えられる使用シナリオ**
Microsoft 次の隣接するセルが空で、テキストが長すぎてセル内に収まらない場合、Excel でテキストがオーバーフローします。 GridDesktop の Aspose.Cells もこの機能をサポートしています。ただし、この機能は GridDesktop の以前のバージョン (17.11、17.10 など) ではサポートされていませんでした。17.12 以降の新しいバージョンでは、この機能がサポートされています。
## **テキストが長すぎると GridDesktop セルからはみ出す**
次のスクリーンショットは、テキストが長すぎてセル幅に収まらない場合に、GridDesktop セルからはみ出すことを示しています。スクリーンショットは、テキスト オーバーフローに関連する新しいバージョンと古いバージョンの GridDesktop の動作を示しています。スクリーンショットでわかるように、この機能は 17.12 以降でサポートされています。
