## **可能な使用シナリオ**
Microsoft Excelは、隣接する空のセルがあり、テキストがセルに収まりきらない場合にテキストをオーバーフローさせます。Aspose.Cells for GridDesktopもこの機能をサポートしています。ただし、この機能は以前のバージョンのGridDesktop（例えば17.11、17.10など）ではサポートされていませんでした。新しいバージョン（17.12以降）では、この機能がサポートされています。
