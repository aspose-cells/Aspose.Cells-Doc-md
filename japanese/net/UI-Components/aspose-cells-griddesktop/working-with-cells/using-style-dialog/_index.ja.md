Aspose.Cells.GridDesktop は、ユーザーが WYSIWYG モードでデータを操作できる GUI ベースのコントロールです。さまざまな種類のタスクを実行するために、Aspose.Cells.GridDesktop の API を詳しく調べました。また、Aspose.Cells.GridDesktop の堅牢で簡単な API を使用して、グリッド、ワークシート、行と列、および Cells の書式設定をプログラムで制御する方法についても説明しました。しかし、それはすべてプログラミングの観点からのものでした。このトピックでは、単純なエンド ユーザーが Aspose.Cells.GridDesktop の書式設定を制御する方法について説明します。**スタイルダイアログ**.

{{% /alert %}} 
## **スタイル ダイアログの使用**
使用するには**スタイルダイアログ**Aspose.Cells.GridDesktop で、次の手順に従ってください。

- **右クリック**グリッド上。いくつかのオプションを提供する前に、コンテキスト メニューがポップアップします。
- 選択する**フォーマット Cells...**コンテキストメニューからオプション。

選択後**フォーマット Cells...**オプション、**スタイルダイアログ**あなたの前に現れるでしょう。**スタイルダイアログ**次のようないくつかのタブで構成されています。* **番号**

- **アライメント**
- **国境**
- **フォント**
- **保護**

の各タブについて説明し、表示してみましょう**スタイルダイアログ**一つずつ。
### **番号タブ**
**番号**タブを使用して設定します**数値形式**セルの設定。数値形式のカテゴリを選択し、値の小数点以下の桁数を設定し、記号セットを選択して、セルに負の値を表示する方法を選択することもできます。
### **配置タブ**
使用する**アライメント**タブで、ユーザーはテキストの水平方向と垂直方向の配置を設定できます。テキストの折り返し、テキストに合わせてセルを縮小、セルの結合など、他のテキスト制御オプションを使用することもできます。
### **ボーダータブ**
**国境**タブを使用すると、ユーザーは数回クリックするだけで、任意の種類と色の境界線を簡単に描くことができます。彼らがしなければならないのは、いくつかのオプションを選択して、希望に応じて境界線を引くことだけです.
### **フォントタブ**
使用する**フォント**タブで、ユーザーはフォント名、サイズ、色、スタイルなどを変更できます。ユーザーは、セルに適用する前にフォント設定をプレビューすることもできます。そのため、ユーザーはセルの書式設定を簡単に行うことができます**ジャストインタイム**.
### **保護タブ**
**保護**タブを使用すると、ユーザーはセルをロックまたは非表示にすることでセルを保護できます。
