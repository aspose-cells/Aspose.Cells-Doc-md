{{% alert color="primary" %}} 

開発者は、結果を Microsoft Excel スプレッドシート ファイルとして保存せずに、Web ページから GridWeb コンテンツを印刷する必要がある場合があります。 Aspose.Cells.GridWeb コントロールは、クライアント側関数を介してこの機能をサポートします。

{{% /alert %}}
