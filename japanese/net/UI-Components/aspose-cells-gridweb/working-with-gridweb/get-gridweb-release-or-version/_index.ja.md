Aspose.Cells.GridWeb.GridWeb.GetVersion()メソッドを使用して、Aspose.Cells.GridWebのリリースまたは実行バージョンを取得してください。 このバージョンは、コードを実行時にGridWebバージョンを見つけるのに役立ちます。

{{% /alert %}}
