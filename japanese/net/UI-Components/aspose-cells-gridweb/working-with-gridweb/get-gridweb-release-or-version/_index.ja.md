Aspose.Cells.GridWeb.GridWeb.GetVersion() メソッドを使用して、Aspose.Cells.GridWeb のリリースまたは実行中のバージョンを取得してください。このバージョンは、実行時にコードを介して GridWeb のバージョンを見つけるのに役立ちます。

{{% /alert %}}
