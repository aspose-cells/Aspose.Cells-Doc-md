## GridWeb の基礎
GridWeb は、表示およびスプレッドシートファイルの編集に使用するシンプルな .net/java webcontrol です。
{{% alert color="primary" %}} 

使いやすくシンプルです。

スプレッドシートファイルのオンライン Web エディタを素早く構築できます。

MS Excel ファイルと完全に互換性のあるあらゆるスプレッドシート形式ファイルのインポートとエクスポートもサポートします。
