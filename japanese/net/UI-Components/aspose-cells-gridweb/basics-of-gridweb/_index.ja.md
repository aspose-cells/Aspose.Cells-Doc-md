##  GridWeb の基本
GridWeb は、スプレッドシート ファイルを表示および編集するための、使いやすい .net/java Web コントロールです。
{{% alert color="primary" %}} 

使い方は簡単です。

スプレッドシート ファイル用のオンライン Web エディタを迅速に構築するのに役立ちます。

また、MS Excel ファイルと 100% 互換性のあるあらゆる種類のスプレッドシート形式ファイルのインポートとエクスポートもサポートしています。
