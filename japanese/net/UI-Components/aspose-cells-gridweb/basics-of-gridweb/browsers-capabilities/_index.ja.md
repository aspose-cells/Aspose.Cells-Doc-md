{{% alert color="primary" %}} 

Aspose.Cells.GridWeb は、Microsoft .NET フレームワークによって提供される他の Web コントロールと同様に、ASP.NET Web フォームに埋め込むことができる GUI ベースの Web コントロールです。 Web コントロールで最も重要なことは、クロスブラウザー サポートを提供することです。 Aspose.Cells.GridWeb は、クロスブラウザー サポートを提供します。

{{% /alert %}} 
## **比較**
Aspose.Cells.GridWeb は Microsoft .NET フレームワークを使用して構築されているため、Microsoft の Internet Explorer (IE) で完全にサポートされています。ただし、他のブラウザでは小さな制限があります。このトピックでは、さまざまなブラウザーでサポートされている機能を詳細に比較します。
