## **Aspose.Cells.GridDesktop – デスクトップ グリッド コントロール**
### **インポート/エクスポート機能**
- Excel スプレッドシートからコンテンツをインポートします。
- DataTable オブジェクトから直接データをインポートします。
- コンテンツを Excel スプレッドシートにエクスポートします。
### **ワークシートの機能**
- ワークシートを追加、挿入、または削除します。
- ワークシートの名前を変更します。
- ワークシート データを並べ替えます。
### **行と列の機能**
- 行と列を追加、挿入、または削除します。
- 行と列を表示または非表示にします。
- 行と列を凍結または凍結解除します。
### **Cell 特徴**
- 数値形式の設定をサポートします。
- テキストの配置を制御します。
- ボーダー設定を管理します。
- フォント設定を制御します。
- セルを結合または分割します。
### **フォーマット機能**
- グリッドの動作、ルック アンド フィールを制御します。
- Format Painter を使用して、フォーカスされたセルのスタイルとフォントを他のセルに適用します。
- データとセル スタイルを簡単に操作します。
### **ユーティリティ機能**
- セルのコメントを追加または削除します。
- 写真を挿入または削除します。
- ハイパーリンクを追加または削除します。
- 検証規則を追加または削除します。
- セルにコントロール (CheckBox、ComboBox など) を追加または削除します。
- セルのキーボード ナビゲーション。
### **セキュリティ機能**
- セルをロックまたは非表示にして、セルを保護します。
### **高度な機能**
- 数式の作成と計算をサポートします。
- さまざまなグリッド イベントで目的のアクションを実行します。
### **サポートされている Microsoft Excel バージョン**

- Microsoft エクセル2000
- Microsoft エクセルXP
- Microsoft エクセル2003
- Microsoft エクセル 2007

- Microsoft エクセル 2013
