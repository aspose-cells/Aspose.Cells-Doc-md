## **Aspose.Cells.GridDesktop – デスクトップグリッドコントロール**
### **インポート/エクスポート機能**
- Excelスプレッドシートからのコンテンツのインポート。
- DataTableオブジェクトから直接データをインポート。
- コンテンツをExcelスプレッドシートにエクスポート。
### **ワークシート機能**
- ワークシートの追加、挿入、または削除。
- ワークシートの名前の変更。
- ワークシートデータの並べ替え。
### **行と列の機能**
- 行と列の追加、挿入、または削除。
- 行と列の表示または非表示。
- 行と列の固定または解除。
### **セル機能**
- 数値形式設定がサポートされています。
- テキストの配置を制御します。
- 境界線設定を管理します。
- フォント設定を制御。
- セルの結合または分割。
### **書式設定機能**
- グリッドの動作、外観、見た目を制御します。
- フォーマットのコピーとペースト機能を使用して、フォーカスを当てたセルのスタイルとフォントを他のセルに適用します。
- データとセルのスタイルを簡単に操作します。
### **ユーティリティ機能**
- セルのコメントを追加または削除します。
- 画像を挿入または削除します。
- ハイパーリンクを追加または削除します。
- バリデーションルールを追加または削除します。
- セルにコントロール（CheckBox、ComboBoxなど）を追加または削除します。
- キーボードによるセルのナビゲーション。
### **セキュリティ機能**
- ロックまたは非表示によってセルを保護します。
### **高度な機能**
- 数式の作成と計算をサポートします。
- 異なるグリッドイベント下で任意のアクションを実行します。
### **サポートされているMicrosoft Excelのバージョン**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **対応プラットフォーム**
- Windowsフォーム
