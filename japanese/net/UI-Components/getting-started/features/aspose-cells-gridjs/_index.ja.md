## **Aspose.Cells.GridJs**
### **インポート/エクスポート機能**
- Excel スプレッドシートからインポートおよびエクスポートします。

### **ワークシートの機能**
- ワークシートの追加/削除/名前変更。


### **行と列の機能**
- ワークシートで行と列を追加、挿入、および削除します。
- 行と列の非表示と表示。
- 行と列を固定および固定解除します。
 
### **Cell 特徴**
- 複数のセルを編集します。
- 数値、日付、通貨、および時刻のフォーマット設定をサポートします。
- カスタム形式の設定をサポートします。
- 背景色の設定をサポートします。
- 自動入力をサポートします。
- テキストの配置を制御します。
- テキストの折り返しを制御します。
- 境界を管理します。
- フォント設定を制御します。
- セルを結合または分割する
### **フォーマット機能**
- デスクトップ スプレッドシート アプリケーションのようなルック アンド フィールの Web ベースの対話型グリッド アプリケーションを作成します。
- セルの書式設定をサポートします。

### **ユーティリティ機能**
- ドロップダウン リストの検証をサポートします。
- コメントを追加/削除/検索します。
- 写真/図形を回転/サイズ変更/移動します。

- キーボード ナビゲーション。
### **高度な機能**
- Web ページでの数式の作成と計算をサポートします。
- 数学、テキストとデータ、日付と時刻、およびロジックに関連する 80 以上の定義済み関数を提供します。
