## **Aspose.Cells.GridJs**
### **インポート/エクスポート機能**
- Excelスプレッドシートからのインポートおよびエクスポート。

### **ワークシート機能**
- ワークシートの追加/削除/名前変更。


### **行と列の機能**
- ワークシート内の行と列の追加、挿入、削除。
- 行と列の表示および非表示。
- 行と列の固定および固定解除。

### **セル機能**
- 複数のセルを編集します。
- 数値、日付、通貨、時間の形式設定をサポートします。
- カスタム形式設定をサポートします。
- 背景色設定をサポートします。
- オートフィルをサポートします。
- テキストの整列を制御。
- テキストの折り返しを制御します。
- 枠線を管理。
- フォント設定を制御。
- セルの結合または分割
### **書式設定機能**
- デスクトップのスプレッドシートアプリケーションのように見える、ウェブベースのインタラクティブなグリッドアプリケーションを作成。
- セルの書式設定をサポートします。

### **ユーティリティ機能**
- ドロップダウンリストの検証をサポートします。
- コメントの追加/削除/検索を行います。
- 画像/図形の回転/サイズ変更/移動を行います。

- キーボードナビゲーション。
### **高度な機能**
-Webページ上での数式の作成および計算のサポート。
- 数学、テキストおよびデータ、日付および時刻、および論理に関連する80以上の事前定義された関数のサポート。

### **対応ブラウザ**
- マイクロソフトインターネットエクスプローラ
- Mozilla FireFox
- Opera
- Google Chrome
- Microsoft Edge
### **マイクロソフトExcelサポート**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **対応プラットフォーム**
- ASP.NET core
