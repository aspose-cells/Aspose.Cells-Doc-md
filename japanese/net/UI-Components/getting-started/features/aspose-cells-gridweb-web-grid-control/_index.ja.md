## **Aspose.Cells.GridWeb – Web グリッド コントロール**
### **インポート/エクスポート機能**
- Excel スプレッドシートからインポートおよびエクスポートします。
- DataView または DataTable オブジェクトとの間でデータを直接インポートおよびエクスポートします。
### **ワークシートの機能**
- ワークシートを追加/削除します。
- ワークシート データの水平方向と垂直方向の両方の並べ替えをサポートします。
- ワークシートのページネーションをサポートします。
- スプレッドシートをゼロから作成します。
### **行と列の機能**
- ワークシートで行と列を追加、挿入、および削除します。
- 行と列の非表示と表示。
- 行と列を固定および固定解除します。
- 行と列のヘッダーをカスタマイズします。
### **Cell 特徴**
- クライアント側で複数のセルを編集し、サーバー側に 1 つのポストを返します。
- 数値、日付、および時刻のフォーマット設定をサポートします。
- カスタムフォーマット設定をサポート
- テキストの配置を制御します。
- 境界を管理します。
- フォント設定を制御します。
- セルを結合または分割する
### **フォーマット機能**
- デスクトップ スプレッドシート アプリケーションのようなルック アンド フィールの Web ベースの対話型グリッド アプリケーションを作成します。
- クライアント側のセルの書式設定をサポートします。
- 組み込みの視覚スタイルとカスタマイズ可能な視覚スタイルの両方を提供します。
### **ユーティリティ機能**
- ドロップダウン リストと正規表現の検証をサポートします。
- コメントを追加および削除します。
- 写真の挿入と削除。
- ハイパーリンクを追加および削除します。
- キーボード ナビゲーション。
### **高度な機能**
- Web ページでの数式の作成と計算をサポートします。
- 数学、テキストとデータ、日付と時刻、およびロジックに関連する 80 以上の定義済み関数を提供します。
- 動的ピボット テーブルの作成をサポートします。
- 異なるグリッド イベントでアクションを実行します。
- セッションレス モードで Aspose.Cells.GridWeb コントロールを使用します。
- ワンクリックでグリッド データを送信します。
