## **Aspose.Cells.GridWeb – Web Grid Control**
### **インポート/エクスポート機能**
- Excelスプレッドシートからのインポートおよびエクスポート。
- DataViewまたはDataTableオブジェクトからのデータの直接インポートとエクスポート。
### **ワークシート機能**
- ワークシートの追加/削除。
- ワークシートデータの水平および垂直ソートをサポート。
- ワークシートのページ分割をサポート。
- ゼロからスプレッドシートを作成。
### **行と列の機能**
- ワークシート内の行と列の追加、挿入、削除。
- 行と列の表示および非表示。
- 行と列の固定および固定解除。
- 行と列のヘッダーのカスタマイズ。
### **セル機能**
- クライアント側で複数のセルを編集し、サーバー側に1回のポストバックで送信。
- 数値、日付、および時刻の書式設定をサポート。
- カスタム書式設定をサポート。
- テキストの整列を制御。
- 枠線を管理。
- フォント設定を制御。
- セルの結合または分割
### **書式設定機能**
- デスクトップのスプレッドシートアプリケーションのように見える、ウェブベースのインタラクティブなグリッドアプリケーションを作成。
- クライアント側のセル書式設定をサポート。
- 組み込みおよびカスタマイズ可能なビジュアルスタイルを提供。
### **ユーティリティ機能**
- ドロップダウンリストおよび正規表現のバリデーションをサポート。
- コメントの追加と削除。
- 画像の挿入および削除。
- ハイパーリンクの追加と削除。
- キーボードナビゲーション。
### **高度な機能**
-Webページ上での数式の作成および計算のサポート。
- 数学、テキストおよびデータ、日付および時刻、および論理に関連する80以上の事前定義された関数のサポート。
- 動的ピボットテーブルの作成のサポート。
- 異なるグリッドイベントの下でのアクションの実行。
- セッションレスモードでAspose.Cells.GridWebコントロールを使用。
- 1回のクリックでグリッドデータを送信。
### **対応ブラウザ**
- マイクロソフトインターネットエクスプローラ
- Mozilla FireFox
- Opera
- Google Chrome
- Microsoft Edge
### **マイクロソフトExcelサポート**
- Microsoft Excel 97
- Microsoft Excel 2000
- Microsoft Excel XP
- Microsoft Excel 2003
- Microsoft Excel 2007
- Microsoft Excel 2010
- Microsoft Excel 2013
### **対応プラットフォーム**
- ASP.NET Web Forms
