{{% alert color="primary" %}} 

Aspose.Cells for .NET は、ソフトウェア開発者が独自のアプリケーション内からスプレッドシート ファイルを操作および変換できるスプレッドシート プログラミング ライブラリです。 Aspose.Cells for .NET は、一般的なスプレッドシート (XLS、XLSX、XLSM、XLSB、XLTX、XLTM、CSV、SpreadsheetML、ODS) ファイル形式をサポートし、Excel ファイルを PDF、XPS、HTML、MHTML、プレーン テキスト、TIFF、JPG、PNG、 BMP、および SVG。

最新の API リファレンスは次の場所にあります。[Aspose.Cells for .NET API 参照](https://reference.aspose.com/cells/net).
