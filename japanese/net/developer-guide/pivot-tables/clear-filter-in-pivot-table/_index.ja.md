##  **考えられる使用シナリオ**
既知のデータを使用してピボット テーブルを作成し、ピボット テーブルをフィルター処理したい場合は、フィルターを学習して使用する必要があります。これは、必要なデータを効果的にフィルタリングするのに役立ちます。 Aspose.Cells API を使用すると、ピボット テーブルのフィールド値に対してフィルターを操作できます。

##  **Excelのピボットテーブルのフィルターをクリアする**
Excel のピボット テーブルでフィルターをクリアするには、次の手順に従います。

1. フィルターをクリアするピボットテーブルを選択します。
2. ピボット テーブルでクリアするフィルターのドロップダウン矢印をクリックします。
3. ドロップダウン メニューから [フィルターをクリア] を選択します。
<img src="1.png" width=80% />
4. ピボット テーブルからすべてのフィルターをクリアする場合は、Excel のリボンのピボットテーブル分析タブにある [フィルターのクリア] ボタンをクリックすることもできます。
<img src="2.png" width=80% />

##  **C# を使用してピボット テーブルのフィルターをクリアする**
Aspose.Cells を使用してピボット テーブルのフィルターをクリアします。次のサンプル コードを参照してください。
1. データを設定し、それに基づいてピボットテーブルを作成します。
2. ピボット テーブルの行フィールドにフィルターを追加します。
 3. ワークブックを保存します。[出力XLSX](out_add.xlsx)フォーマット。サンプル コードを実行すると、top10 フィルターを備えたピボット テーブルがワークシートに追加されます。
 4. 特定のピボットフィールドのフィルターをクリアします。フィルターをクリアするコードを実行すると、特定のピボットフィールドのフィルターがクリアされます。をご確認ください。[出力XLSX](out_delete.xlsx).
