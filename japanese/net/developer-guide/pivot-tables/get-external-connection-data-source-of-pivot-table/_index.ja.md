## **ピボットテーブルの外部接続データソースの取得**

Aspose.Cells は、ピボットテーブルの外部接続データソースを取得する機能を提供しています。そのために、API は [**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) クラスの [**PivotTable**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable) プロパティを提供します。[**ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) プロパティは [**ExternalConnection**](https://reference.aspose.com/cells/net/aspose.cells.externalconnections/externalconnection) オブジェクトを返します。次のコードスニペットは、[**PivotTable.ExternalConnectionDataSource**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/properties/externalconnectiondatasource) プロパティを使用してピボットテーブルの外部接続データソースを取得する方法を示しています。
