Aspose.Cellsは、ピボットテーブルをODSファイルに保存する機能を提供します。これには、既存のピボットテーブルを含むワークブックを変換するか、新しいピボットテーブルを作成してファイルをODS形式で保存するだけです。出力されるODSファイルでピボットテーブルが表示されるようにするために、保存前に[**PivotTable.CalculateData()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/calculatedata)を呼び出すことを確認してください。次のコードスニペットは、ODSファイルにピボットテーブルを保存する方法を示しています。
