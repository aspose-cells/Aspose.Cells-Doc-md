{{% alert color="primary" %}}

時々、ピボットテーブルセルの書式を設定したいことがあります。たとえば、ピボットテーブルセルに背景色を適用したい場合があります。Aspose.Cells では、この目的で使用できる２つの方法 [**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) および [**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format) を提供しています。

[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall) は、ピボットテーブル全体にスタイルを適用し、[**PivotTable.Format()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format) はピボットテーブルの個々のセルにスタイルを適用します。

{{% /alert %}}
