場合によっては、ピボット テーブルのセルを書式設定したい場合があります。たとえば、ピボット テーブルのセルに背景色を適用するとします。 Aspose.Cells は 2 つの方法を提供します[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall)そして[**ピボットテーブル.フォーマット()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format)、この目的に使用できます。

[**PivotTable.FormatAll()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/formatall)スタイルをピボット テーブル全体に適用します。[**ピボットテーブル.フォーマット()**](https://reference.aspose.com/cells/net/aspose.cells.pivot/pivottable/methods/format)スタイルをピボット テーブルの単一のセルに適用します。
