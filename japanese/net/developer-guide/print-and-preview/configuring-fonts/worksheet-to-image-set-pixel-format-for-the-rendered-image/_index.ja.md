{{% alert color="primary" %}} 

ワークシートをイメージ形式にレンダリングするときに、ピクセル形式を指定したい場合があります。デフォルトでは、Aspose.Cells は 32 ビット/ピクセルを使用します。 Aspose.Cells では、レンダリング イメージのオプションを使用して、ピクセル形式 (ビット深度) をカスタマイズできます。

{{% /alert %}}
