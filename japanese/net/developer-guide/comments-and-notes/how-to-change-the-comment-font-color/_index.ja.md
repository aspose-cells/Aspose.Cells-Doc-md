Microsoft Excel では、ユーザーがセルにコメントを追加して、追加情報を追加し、データを強調表示することができます。開発者は、配置設定、テキストの方向、フォントの色などを指定するために、コメントをカスタマイズする必要がある場合があります。Aspose.Cells は、タスクを実行するための API を提供します。

{{% /alert %}}
