{{% alert color="primary" %}}

Microsoft Excel では、ユーザーがセルにコメントを追加して、追加情報を追加し、データを強調表示することができます。開発者は、配置設定とテキスト方向を指定するためにコメントをカスタマイズする必要がある場合があります。 Aspose.Cells は、タスクを実行するための API を提供します。

{{% /alert %}}

Aspose.Cells は[**Shape.TextDirection**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textdirection)コメントのテキスト方向を設定するプロパティ。次のサンプル コードは、[**Shape.TextDirection**](https://reference.aspose.com/cells/net/aspose.cells.drawing/shape/properties/textdirection)コメントのテキスト方向を設定するプロパティ。
