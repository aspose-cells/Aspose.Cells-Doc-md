## C# で VBA プロジェクトが保護され、表示用にロックされているかどうかを確認する

Aspose.Cells では、Excel ファイルの VBA (Visual Basic for Applications) プロジェクトが保護され、表示のためにロックされているかどうかを確認できます。このために、API は[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing)財産。表示のためにロックされている場合、[**VbaProject.IslockedForViewing**](https://reference.aspose.com/cells/net/aspose.cells.vba/vbaproject/properties/islockedforviewing)プロパティリターン**真実**.
