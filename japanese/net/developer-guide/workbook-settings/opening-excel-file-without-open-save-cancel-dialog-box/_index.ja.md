{{% alert color="primary" %}} 

このドキュメントでは、[開く-保存-キャンセル] ダイアログ ボックスを表示せずに、ブラウザーで Microsoft Excel ファイルを開く方法について説明します。

ここで、ファイルの直接ダウンロードを許可しないセキュリティ制限は、Aspose ではなく、Microsoft (または他のブラウザ ベンダー) によって実施されていることに注意してください。これは、潜在的に有害なファイルがローカル マシンにダウンロードされるのをブロックおよび制限するために課されます。 .

クライアントのローカル システムが、ダウンロードを促す [開く-保存-キャンセル] ダイアログを表示せずにダウンロードを許可するのは危険です。非常に大きなセキュリティ リスクになるため、Aspose から利用できるオプションや回避策はありません。

{{% /alert %}}

上で説明したように、ファイルが本当に必要かどうかの確認なしに、システム上でファイルを開いたり実行したりできるようにすることは、セキュリティ上のリスクです。一部のファイルにはウイルスが含まれており、サイトによっては、確認を求めずに有害なファイルをマシンにダウンロードしようとします。したがって、ユーザーがファイルを確認する必要があり、ダウンロードまたは実行する前にそのソースを確認できるように、ダウンロード プロンプトなしでファイルのダウンロードを許可することはお勧めしません。ダウンロード ダイアログ ボックスを無効にすると、システムはウイルス、トロイの木馬、ハッカーに対して無防備になり、システムに影響を与える可能性があります。
## **[開く-保存-キャンセル] ダイアログ ボックスを使用せずにファイルを開く**
これはセキュリティ上の大きな懸念事項ですが、Microsoft は依然として Internet Explorer の設定を提供し、ユーザーがファイル ダウンロードの [開く]、[保存]、[キャンセル] プロンプトを無効にできるようにします。

Windows エクスプローラー:

1. 上で**ツール**メニュー、選択**フォルダ オプション**.
1. [フォルダ オプション] ダイアログの [ファイルの種類] タブをクリックします。
1. XLS 拡張子のファイル タイプを選択します。
1. クリック**高度**. 
ダイアログボックスが表示されます。下部に 3 つのオプションがあります。
1. チェックを外す**ダウンロード後に開くことを確認する**.
1.  番目のオプションを選択します -**同じウィンドウで閲覧する** Microsoft Excel スタンドアロンを起動せずに Internet Explorer で Excel ファイルを表示するには。

|**ファイル タイプ オプションの編集**|
|:- |

この設定により、ファイルをダウンロードまたは開くときに [開く]、[保存]、[キャンセル] ダイアログが表示されることなく、ファイルを Web ブラウザーで直接実行できます。
