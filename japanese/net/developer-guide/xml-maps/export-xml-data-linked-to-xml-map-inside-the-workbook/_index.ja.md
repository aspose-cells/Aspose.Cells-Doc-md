## **ブック内のXMLマップにリンクされたXMLデータをエクスポート**

ブック内のXMLマップにリンクされたXMLデータをエクスポートするには、[**Workbook.ExportXml()**](https://reference.aspose.com/cells/net/aspose.cells/workbook/methods/exportxml/index) メソッドを使用してください。サンプルコードは、[5115497.xlsx](sample excel file)と[5472487.xml](exported XML data of the first XML Map)をご確認ください。
