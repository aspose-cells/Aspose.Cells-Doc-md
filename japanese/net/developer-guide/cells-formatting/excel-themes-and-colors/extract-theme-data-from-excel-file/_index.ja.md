Aspose.Cells を使用すると、ユーザーは Excel ファイルからテーマ関連データを抽出できます。たとえば、ブックに適用されているテーマ名や、セルやセルの境界線に適用されているテーマの色などを抽出できます。

Microsoft Excel の [ページ レイアウト] > [テーマ] コマンドを使用してワークブックにテーマを適用できます。
