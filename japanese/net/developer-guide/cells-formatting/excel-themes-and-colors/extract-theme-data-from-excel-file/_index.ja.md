{{% alert color="primary" %}}

Aspose.Cells を使用すると、ユーザーは Excel ファイルからテーマ関連のデータを抽出できます。たとえば、ワークブックに適用されているテーマ名、セルまたはセルの境界線に適用されているテーマの色などを抽出できます。

Microsoft Excel の [ページ レイアウト] > [テーマ] コマンドを使用して、ワークブックにテーマを適用できます。
