{{% alert color="primary" %}}

Aspose.Cellsを使用すると、Excelファイルからテーマ関連のデータを抽出することができます。たとえば、ワークブックに適用されているテーマ名や、セルまたはその境界に適用されているテーマカラーなどを抽出できます。

Microsoft Excelを使用してワークブックにテーマを適用できます。[ページレイアウト] > [テーマ]コマンドを使用します。

{{% /alert %}}
