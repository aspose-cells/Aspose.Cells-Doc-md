{{% alert color="primary" %}} 

ハイパーリンクは、他のワークシートまたは Web サイト上の情報にアクセスするのに便利です。 Microsoft Excel では、セル内のテキストや画像にハイパーリンクを追加できます。画像のハイパーリンクを使用すると、たとえば、「次へ」および「前へ」ボタンや、特定のサイトにリンクするロゴとして、ワークシート内を簡単に移動できます。このドキュメントでは、Aspose.Cells を使用してワークシートに画像のハイパーリンクを挿入する方法について説明します。

{{% /alert %}} 

Aspose.Cells を使用すると、実行時にスプレッドシート内の画像にハイパーリンクを追加できます。リンクの画面チップやアドレスを設定・変更することができます。次のサンプル コードは、ワークシートに画像のハイパーリンクを追加する方法を示しています。
