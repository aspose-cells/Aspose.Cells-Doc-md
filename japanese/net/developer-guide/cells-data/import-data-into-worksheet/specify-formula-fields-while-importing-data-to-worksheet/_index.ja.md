##  **考えられる使用シナリオ**

データをワークシートにインポートするときに、[**ImportTableOptions.IsFormulas**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/isformulas) 。このプロパティはブール配列を受け取ります。値は**真実**フィールドが数式フィールドであることを意味します。たとえば、3 番目のフィールドが数式フィールドの場合、配列内の 3 番目の値は *true** になります。
