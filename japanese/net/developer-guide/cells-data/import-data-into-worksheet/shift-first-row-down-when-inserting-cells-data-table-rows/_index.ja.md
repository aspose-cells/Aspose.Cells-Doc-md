## **可能な使用シナリオ**

Aspose.Cellsを使用すると、ワークシートにテーブルを挿入する際に最初の行を下にシフトできます。このドキュメントは、Aspose.Cells APIを使用してタスクを達成する方法について説明しています。

## **セルデータテーブル行を挿入するときに、最初の行を下にシフト**

以下のサンプルコードは、ワークシートにテーブルを挿入する際に最初の行を下にシフトする方法を示しています。機能を実証するためにコード内で簡単なテンプレートExcelファイルを使用します。機能をより良く理解するために、[**ImportTableOptions.ShiftFirstRowDown**](https://reference.aspose.com/cells/net/aspose.cells/importtableoptions/properties/shiftfirstrowdown)属性を**True**/**False**に設定して機能を試してください。[サンプルExcelファイル](45056031.xlsx)、[出力Excel Falseファイル](45056032.xlsx)、および[出力Excel Trueファイル](45056033.xlsx)を参照してください。
