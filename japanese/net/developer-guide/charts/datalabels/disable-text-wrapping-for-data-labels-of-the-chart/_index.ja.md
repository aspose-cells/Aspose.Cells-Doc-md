{{% alert color="primary" %}}

Microsoft Excel 2013 では、ユーザーはグラフのデータ ラベル内でテキストをラップまたはアンラップできます。デフォルトでは、グラフのデータ ラベル内のテキストは折り返された状態になっています。

Aspose.Cells は[**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped) True または False を設定して、データ ラベルのテキスト ラッピングをそれぞれ有効または無効にできます。

{{% /alert %}}
