{{% alert color="primary" %}}

Microsoft Excel 2013 では、ユーザーはグラフのデータ ラベル内のテキストを折り返すか折り返さないことができます。デフォルトでは、グラフのデータ ラベル内のテキストは折り返された状態になっています。

Aspose.Cells は、[**DataLabels.IsTextWrapped**](https://reference.aspose.com/cells/net/aspose.cells.charts/datalabels/properties/istextwrapped)このプロパティを True または False に設定して、それぞれデータ ラベルのテキストの折り返しを有効または無効にできます。

{{% /alert %}}
