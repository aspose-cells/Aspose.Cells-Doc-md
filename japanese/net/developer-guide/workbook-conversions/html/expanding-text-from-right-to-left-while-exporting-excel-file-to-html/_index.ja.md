{{% alert color="primary" %}} 

Aspose.Cells は、Excel ファイルを HTML にエクスポートする際にテキストを右から左に展開する機能を v8.9.0.0 以降でサポートしています。元の Excel ファイルに右から左に展開するテキストが含まれている場合、Aspose.Cells はそれを適切に HTML にエクスポートします。

{{% /alert %}}
