## **考えられる使用シナリオ**

Excel ファイルを HTML に保存すると、コメントはエクスポートされません。ただし、Aspose.Cells は、[**HtmlSaveOptions.IsExportComments**](https://docs.aspose.com/cells/net/export-comments-while-saving-excel-file-to/)財産。設定すれば**真実**の場合、HTML は Excel ファイルに存在するコメントも表示します。
