{{% alert color="primary" %}}

ワークシート内のテーブルまたはリストオブジェクトのコメントを設定するには、[**ListObject.Comment**](https://reference.aspose.com/cells/net/aspose.cells.tables/listobject/properties/comment) プロパティを使用します。コメントは xl/tables/tableName.xml ファイル内で表示されます。
