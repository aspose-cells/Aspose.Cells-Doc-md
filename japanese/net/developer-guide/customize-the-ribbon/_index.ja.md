Microsoft Office は、Office 2007 以降、メニューとツールバーをアプリケーション ウィンドウの上部にあるリボンに置き換えました。リボンはカスタマイズ可能です。
Aspose.Cells を使用すると、

- 解析せずにリボン XML を保持し、
- 解析せずにリボン XML を読み書きします。
- リボン XML データを取得および設定します。

リボン XML を変更する場合は、XML パーサーまたは他のリボン XML ツールで解析する必要があります。
