## **紹介**

Microsoft ExcelはExcelブック内に**署名行**を追加する機能を提供しています。**挿入**タブをクリックし、**テキスト**グループから**署名行**を選択して、署名行を追加できます。

## **Excelファイルの署名行を作成する方法**

Aspose.Cellsもこの機能を提供し、この目的に[**Picture.SignatureLine**](https://reference.aspose.com/cells/net/aspose.cells.drawing/picture/properties/signatureline)プロパティを公開しています。この記事では、このプロパティを使用して署名行を追加する方法について説明します。

次のサンプルコードは、[**Picture.SignatureLine**](https://reference.aspose.com/cells/net/aspose.cells.drawing/picture/properties/signatureline)プロパティを使用して署名行を追加し、ワークブックを保存します。
