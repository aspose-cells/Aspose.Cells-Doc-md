{{% alert color="primary" %}} 

Aspose.Cells を使用して、Excel ワークブックを HTML 形式で保存できます。エクスポートされた HTML ワークブックは Microsoft Excel 向けです。つまり、Microsoft Excel でエクスポートされたものと似ています。 HTML からワークブックを作成することもできますが、この HTML は Excel 指向である必要があります。

{{% /alert %}} 
## **Aspose.Cells の HTML**
Aspose ソリューションは、HTML への変換時に次のドキュメント機能を完全にサポートします。

- Cell データとテキストのフォーマット
- グラフとピボット テーブル
- ヘッダー/フッター
- ボーダー、ボーダーの色
- 画像、テキストボックス、図形の描画
- ハイパーリンク
