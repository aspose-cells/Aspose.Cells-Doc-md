{{% alert color="primary" %}} 

Aspose.Cellsを使用してExcelワークブックをHTML形式で保存できます。エクスポートされたHTMLワークブックは、Microsoft Excelと同様に出力されます。また、HTMLからワークブックを作成することも可能ですが、このHTMLはMicrosoft Excelに準拠している必要があります。

{{% /alert %}} 
## **Aspose.CellsのHTML**
Asposeの解決方法は、次のドキュメント機能への変換を利用する際にしっかりとサポートしています:

- セルデータとテキストの書式設定
チャートとピボットテーブル
ヘッダー/フッター
境界線、境界線の色
イメージ、テキストボックス、図形の描画
- ハイパーリンク
