## **Aspose.Cells – Microsoft Excel スプレッドシート管理ライブラリ**
### **ドキュメントの機能**
- 異なるソースからプレーンまたは暗号化された Microsoft Excel ファイル（Excel97 - Excel2007/2010/2013）を開きます。
- Excel ファイル（Excel 97 - Excel2007/2010/2013）を様々な形式（XLS、XLSX/XLSM、XLTX、XLTM、XLSB、CSV、SpreadsheetML、タブ区切り、TSV、HTML など）で保存します。
- Excel ファイルを PDF ドキュメントに変換します。
- Excel スプレッドシートを XPS ドキュメントに変換します。
- Tagged Image File Format (TIFF) に変換します。
- OpenDocument Spreadsheet (ODS) 形式を読み取りおよび書き込みます。
- Excel ファイルのドキュメント プロパティを変更します。
### **ワークシート機能**
#### **表示オプション**
- ワークシートを非表示または表示します。
- タブ、スクロールバー、グリッド線、行ヘッダーおよび列ヘッダーを表示または非表示にします。
- ページ ブレークをプレビューします。
- ワークシートのズーム レベルを設定します。
- フリーズ ペインを使用してスクロールしても選択したデータを表示したままにします。
#### **ワークシートの保護**
- ワークシートのコンテンツ、オブジェクト、シナリオを保護します。
#### **ページ設定オプション**
- ページの向き、拡大縮小、用紙サイズ、印刷品質などを設定します。
- 上部、左、右、下などのページ余白を設定します。
- ヘッダーとフッターを設定します。
- 印刷エリア、印刷タイトルなどを設定します。
#### **ページブレーク**
- 指定したセルに水平および垂直の改ページを挿入します。
### **行および列の機能**
- 行の高さを調整します。
- 必要に応じて列の幅を調整します。
- セルの幅と高さをコンテンツに応じて自動調整します。
- 行と列を挿入または削除します。
- 行と列の表示または非表示。
- 行と列をグループ化またはグループ化解除します。
### **データ管理機能**
- ワークシート内の任意のセルにアクセスします。
- 実行時に希望するセルにデータを追加します。
- 異なるデータソースからワークシートにデータをインポートします。
- データの種類に応じてセルからデータを取得します。
- セルのデータを DataTable にエクスポートします。
- セル内の特定のデータを検索または検索します。
- 組み込みまたはアドイン関数を使用して数式を設定します。
- 配列および R1C1 参照スタイルの数式を設定します。
- 異なる種類のハイパーリンクを追加します。
- 名前付き範囲を作成またはアクセスします。
- データの検証および自動フィルタリングを適用します。
### **書式設定機能**
- セルに異なるスタイルを適用します。
- セルに数値形式の設定を適用します。
- セルを日付として書式設定します。
- テキストの配置を設定します。
- 複数のセルをマージする。
- セルに異なる種類のフォント設定を適用する。
- セル、フォント、グリッド線、グラフィックオブジェクトなどに異なる色を適用する。
- 単一のセルにリッチテキストを設定する。
- セルに異なる境界線設定を適用する。
- セルに異なる背景パターンを適用する。
- ワークシート、行、列などにフォーマット設定を適用する。
- セルに条件付き書式を適用する。
### **チャートおよびグラフィック機能**
- カラム、バー、折れ線、円、散布図などのすべての標準的なチャートタイプをサポートする。
- 異なる種類のチャートを簡単に作成する。
- 異なるプロパティを設定してチャートをカスタマイズする。
- チャートのデータを動的に設定する。
- APIを使用してワークシートにカスタムチャートを追加する。
- ランタイムでワークシートに画像を追加する。
- セルにコメントを作成し追加する。
- チャートを画像ファイルに変換する。
- ワークシートを画像ファイルに変換する。
- ワークシートを印刷する。
### **高度な機能**
- デザイナースプレッドシートを作成し、スマートマーカーを使用する。
- 頑健な式計算エンジンによる式計算を迅速に行う。
- COMクライアントからAspose.Cellsにアクセスする。
- ランタイムでピボットテーブルのソースデータを変更する。
- ピボットテーブルを作成し操作する。
- VBAコードまたはマクロを操作する。
### **サポートされているプラットフォーム**
- デスクトップWindowsアプリケーション
- エンタープライズWebアプリケーション
