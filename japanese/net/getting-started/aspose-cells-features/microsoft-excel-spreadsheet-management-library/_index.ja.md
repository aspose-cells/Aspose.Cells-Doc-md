## **Aspose.Cells – Microsoft Excel スプレッドシート管理ライブラリ**
### **ドキュメントの機能**
- さまざまなソースからプレーンまたは暗号化された Microsft Excel ファイル (Excel97 - Excel2007/2010/2013) を開きます。
- Excel ファイル (Excel 97 - Excel2007/2010/2013) をさまざまな形式 (XLS、XLSX/XLSM、XLTX、XLTM、XLSB、CSV、SpreadsheetML、タブ区切り、TSV、HTML など) で保存します。
- Excel ファイルを PDF ドキュメントに変換します。
- Excel スプレッドシートを XPS ドキュメントに変換します。
- タグ付き画像ファイル形式 (TIFF) に変換します。
- OpenDocument スプレッドシート (ODS) 形式の読み取りと書き込み。
- Excel ファイルのドキュメント プロパティを変更します。
### **ワークシートの機能**
#### **表示オプション**
- ワークシートを非表示または表示します。
- タブ、スクロール バー、グリッド線、および行と列のヘッダーを表示または非表示にします。
- 改ページをプレビューします。
- ワークシートのズーム レベルを設定します。
- フリーズ ペインを使用してスクロールしている間、選択したデータを表示したままにします。
#### **ワークシートの保護**
- ワークシートのコンテンツ、オブジェクト、およびシナリオを保護します。
#### **ページ設定オプション**
- ページの向き、倍率、用紙サイズ、印刷品質などを設定します。
- 上、左、右、下などのページ余白を設定します。
- ヘッダーとフッターを設定します。
- 印刷範囲、印刷タイトルなどを設定します。
#### **改ページ**
- 指定したセルに水平および垂直の改ページを挿入します。
### **行と列の機能**
- 行の高さを調整します。
- 必要に応じて列幅を調整します。
- セルの内容に応じてセルの幅と高さを自動調整します。
- 行と列を挿入または削除します。
- 行と列を表示または非表示にします。
- 行と列をグループ化またはグループ解除します。
### **データ管理機能**
- ワークシート内の任意のセルにアクセスします。
- 実行時に必要なセルにデータを追加します。
- さまざまなデータ ソースからワークシートにデータをインポートします。
- データ型に従ってセルからデータを取得します。
- セル データを DataTable にエクスポートします。
- セル内の特定のデータを検索または検索します。
- 組み込み関数またはアドイン関数を使用して数式を設定します。
- 配列と R1C1 参照スタイル式を設定します。
- さまざまな種類のハイパーリンクを追加します。
- 名前付き範囲を作成またはアクセスします。
- データ検証と自動フィルタリングを適用します。
### **フォーマット機能**
- セルにさまざまなスタイルを適用します。
- 数値書式設定をセルに適用します。
- セルを日付としてフォーマットします。
- テキストの配置を設定します。
- 複数のセルを結合します。
- セルにさまざまな種類のフォント設定を適用します。
- セル、フォント、グリッド線、グラフィック オブジェクトなどにさまざまな色を適用します。
- 1 つのセルにリッチ テキストを設定します。
- セルに異なる境界設定を適用します。
- セルにさまざまな背景パターンを適用します。
- ワークシート、行、列などに書式設定を適用します。
- セルに条件付き書式を適用します。
### **チャートとグラフィック機能**
- 縦棒グラフ、棒グラフ、折れ線グラフ、円グラフ、散布図など、すべての標準的なグラフ タイプをサポートします。
- さまざまな種類のチャートを簡単に作成します。
- さまざまなプロパティを設定してチャートをカスタマイズします。
- グラフのデータを動的に設定します。
- API を使用してカスタム グラフをワークシートに追加します。
- 実行時にワークシートに画像を追加します。
- コメントを作成してセルに追加します。
- チャートを画像ファイルに変換します。
- ワークシートを画像ファイルに変換します。
- ワークシートを印刷します。
### **高度な機能**
- デザイナー スプレッドシートを作成し、スマート マーカーを操作します。
- 堅牢な数式計算エンジンによる高速な数式計算。
- COM クライアントから Aspose.Cells にアクセスします。
- 実行時にピボット テーブルのソース データを変更します。
- ピボット テーブルを作成して操作します。
- VBA コードまたはマクロを操作します。
### **サポートされているプラットフォーム**
- デスクトップ Windows アプリケーション
- エンタープライズ Web アプリケーション
