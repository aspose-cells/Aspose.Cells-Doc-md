## **Aspose.Cells for .NET から NuGet をインストールします**
NuGet は、Aspose API for .NET をダウンロードしてインストールする最も簡単な方法です。Microsoft Visual Studio と NuGet パッケージ マネージャーを開きます。 「aspose」を検索して、目的の Aspose API を見つけます。「インストール」をクリックすると、選択した API がダウンロードされ、プロジェクトで参照されます。

注: 詳細については、aspose.cells の nuget Web ページにアクセスしてください。
[Aspose.Cells for .NET NuGet パッケージ](https://www.nuget.org/packages/Aspose.Cells/)

### **パッケージ マネージャー GUI を使用して Aspose.Cells をインストールします。**
パッケージ マネージャー GUI を使用して Aspose.Cells コンポーネントを参照するには、次の手順に従います。

- Visual Studio でソリューション/プロジェクトを開きます。
- クリック**ツール** -> **ライブラリ パッケージ マネージャー** -> **NuGet パッケージの管理**ソリューションから。ソリューション エクスプローラーから同じオプションに簡単にアクセスすることもできます。ソリューションまたはプロジェクトを右クリックして、**NuGet パッケージの管理**コンテキストメニューから。
- 選択する**オンライン**左側のメニューから検索テキスト ボックスに「Aspose.Cells」と入力して、Aspose.Cells .NET パッケージを検索します。
- クリック**インストール**Aspose.Cells for .NET エントリの横にあるボタンをクリックして、最新バージョンをプロジェクトにインストールします。
### **パッケージ マネージャー コンソールを使用して Aspose.Cells をインストールします。**
以下の手順に従って、パッケージ マネージャー コンソールを使用して Aspose.Cells コンポーネントを参照できます。

- Visual Studio でソリューション/プロジェクトを開きます。
- 選択する**ツール** -> **ライブラリ パッケージ マネージャー** -> **パッケージ マネージャー コンソール**メニューからパッケージ マネージャー コンソールを開きます。
 コマンド「Install-Package Aspose.Cells」を入力し、Enter キーを押して最新のフル リリースをアプリケーションにインストールします。または、コマンドに「-prerelease」サフィックスを追加して、ホット フィックスを含む最新リリースもインストールするように指定できます。
- ウィンドウの左下に「Aspose.Cells をダウンロードしています...」というヒントが表示され、ダウンロードが進行中であることが示されます。
- ダウンロードすると、次の確認メッセージが表示されます。 Aspose EULA に慣れていない場合は、URL で参照されているライセンスを読むことをお勧めします。
- Aspose.Cells が正常に追加され、アプリケーションで参照されていることがわかります。
## **.NET プロジェクトからの Aspose.Cells の参照**
アプリケーションで Aspose.Cells を使用するには、それへの参照を追加します。 Visual Studio を使用して参照を追加するには:

1. の中に**ソリューション エクスプローラー**で、参照を追加するプロジェクト ノードを展開します。
1. を右クリックします。**参考文献**プロジェクトのノードを選択し、**参照を追加**メニューから。
1. の中に**参照を追加**ダイアログ ボックスで、.NET タブを選択します (デフォルトで選択されています)。 MSI インストーラーを使用してインストールした場合、Aspose.Cells が上部ペインに表示されます。
1. 選択してクリック**選択する**.

DLL のみをダウンロードまたは解凍した場合:

1. を使用して Aspose.Cells.dll ファイルを見つけます。**ブラウズ**ボタン。 Aspose.Cells が**選択されたコンポーネント**ダイアログ ボックスのペイン。
1. クリック**わかった** . Aspose.Cells 参照が下に表示されます**参考文献**プロジェクトのノード。
### **VS.NET 2010 Client Profile プロジェクトからのコンポーネントの参照**
プロジェクトのターゲット フレームワークが .NET Framework 3.5/4 Client Profile の場合、インストール ディレクトリの net_ClientProfile フォルダーにある Aspose.Cells.dll コンポーネント ファイルを使用します。例えば：

- の**ソリューション エクスプローラー**プロジェクトのVS.NET 2010の、右クリック**参考文献**そして選択**参照を追加**.

- クリック**わかった**ダイアログを閉じる

## **.NET プロジェクトからの Aspose.Cells グリッド コントロールの参照**
アプリケーションでグリッド コントロールを使用するには、グリッド コントロールへの参照を追加します。 Visual Studio でグリッド コントロールを参照するには、次の手順を実行します。

- の**ソリューション エクスプローラー**で、参照を追加するプロジェクト ノードを展開します。
- を右クリックします。**参考文献**プロジェクトのノードを選択し、**参照を追加**メニューから。
- の中に**参照を追加**ダイアログ ボックスで、**.NET タブ**(デフォルトで選択されています)。 MSI インストーラーを使用して Aspose.Cells for .NET をインストールした場合、Aspose.Cells.GridDesktop および Aspose.Cells.GridWeb が上部ペインに表示されます。
- それらを選択してクリックします**選択する**.
- Aspose.Cells.GridDesktop および Aspose.Cells.GridWeb 参照は、プロジェクトの参照ノードの下に表示されます。

DLL のみをダウンロードして解凍した場合:

-  [参照] ボタンを使用して、Aspose.Cells.GridDesktop.dll および Aspose.Cells.GridWeb.dll ファイルを見つけます。 Aspose.Cells Grid Suite が参照されており、**選択されたコンポーネント**ダイアログ ボックスのペイン。
- クリック**わかった。**
## **アンインストール中 Aspose.Cells for .NET**
MSI インストーラーを使用して Aspose.Cells for .NET を展開した場合は、次の手順に従って、コンポーネントとコントロール、関連するデモとドキュメントを完全に削除します。

- から**始める**メニュー、選択**設定**、 に続く**コントロールパネル**.
- クリック**プログラムの追加と削除**.
- Aspose.Cells for .NET (バージョン) を選択します。
- クリック**変更/削除**Aspose.Cells を削除します。
