### **コマンドラインの使用**
1. Windows ディレクトリで cmd.exe を検索し、右クリックして [Run as...] を選択します。**管理者**
 2. 次に、コマンド プロンプトで次のコマンドを実行します: msiexec /i<your path>/Aspose.Cells.msi と入力します。
### **コントロール パネルの使用**
- [開始] をクリックします。
- コントロールパネルをクリック
- [ユーザー アカウントと家族の安全] をクリックします。
- [ユーザー アカウント] をクリックします。
- [ユーザー アカウント制御をオンまたはオフにする] をクリックします。
- チェックボックスをオフにします
- [OK] をクリックします。
