## **開発環境**
Aspose.Cells for .NET を使用して、.NET プラットフォームを対象とする任意の開発環境でアプリケーションを開発できますが、次の環境が明示的にサポートされています。

- Microsoft ビジュアルスタジオ 2005
- Microsoft ビジュアルスタジオ 2008
- Microsoft ビジュアルスタジオ 2010
- Microsoft ビジュアルスタジオ 2012
- Microsoft ビジュアルスタジオ 2013
- Microsoft ビジュアルスタジオ 2015
- Microsoft ビジュアルスタジオ 2017
- Microsoft ビジュアルスタジオ 2019
- Microsoft ビジュアルスタジオ 2022

{{% alert color="primary" %}} 

Aspose.Cells を使用するために、マシンに Microsoft がインストールされている必要はありません。ライブラリ自体がスプレッドシート管理エンジンであるため、Excel がインストールされています。ただし、コンポーネントによって生成された Excel ファイルを表示するには、マシンに少なくとも Microsoft Excel Viewer がインストールされている必要があります。

{{% /alert %}} 
## **Aspose.Cells グリッド スイートのシステム要件**
Aspose.Cells Grid Suite は 2 つのコントロールで構成されています。

- Aspose.Cells.GridDesktop
- Aspose.Cells.GridWeb

両方のコントロールのシステム要件は、個別に説明されています。

{{% alert color="primary" %}} 

これら 2 つのコントロールを使用するには、Aspose.Cells.GridDesktop および Aspose.Cells.GridWeb を実行するマシンに Microsoft Excel をインストールする必要はありません。
