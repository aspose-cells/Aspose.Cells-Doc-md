## **セキュリティ例外の問題**
一部のユーザーは、Aspose.Cells を使用しようとしているときに「セキュリティ例外」エラーを受け取る場合があります。この問題は通常、Web アプリケーションで発生します。
### **説明**
 Aspose.Cells に電話する必要があります**Win32 GDI API**いくつかの重要な機能を提供します。 Web サーバーの信頼レベルが厳密である場合、このセキュリティ例外がスローされることがあります。
### **解決**
「アンマネージ アセンブリへの呼び出しを許可する」を有効にして、Aspose.Cells.dll セキュリティ アクセス許可を付与する新しいアクセス許可セットを作成してみてください。
