## **ソフトウェア要件**
サンプルをダウンロードして実行する前に、次の要件を満たしていることを確認してください。

1. Visual Studio 2015 以降
1. NuGet パッケージ マネージャーが Visual Studio にインストールされました。ほとんどの場合、Visual Studio 2015 に既にインストールされています。NuGet パッケージ マネージャーのインストール方法の詳細については、以下を確認してください。[NuGet クライアント ツールのインストール](https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools)
1. Tools->Options->NuGet Package Manager->Package Sources に移動し、オプションが**nuget.org**チェックされています
1. サンプル プロジェクトは NuGet 自動パッケージ復元機能を使用しているため、アクティブなインターネット接続が必要です。サンプルを実行するマシンにアクティブなインターネット接続がない場合は、チェックしてください[インストール](/cells/ja/net/installation-and-deployment/)サンプル プロジェクトに Aspose.Cells.dll への参照を手動で追加します。
## **GitHub からダウンロード**
Aspose.Cells for .NET のすべての例は、[GitHub](https://github.com/aspose-cells/Aspose.Cells-for-.NET).
## **Aspose.Cells 例**
- お気に入りの GitHub クライアントを使用してリポジトリのクローンを作成するか、ZIP ファイルを次の場所からダウンロードできます。[ここ](https://github.com/aspose-cells/Aspose.Cells-for-.NET/archive/master.zip).
-  ZIP ファイルの内容をコンピューター上の任意のフォルダーに抽出します。すべての例は、**例**フォルダ。
- C# の例の Visual Studio ソリューション ファイルがあります。**Aspose.Cells.Examples.CSharp.sln**.
- プロジェクトは、Visual Studio 2015 で作成および管理されます。
- Visual Studio でソリューション ファイルを開き、プロジェクトをビルドします。
- 最初の実行時に、依存関係は NuGet 経由で自動的にダウンロードされます。DLL を個別にダウンロードすることもできます。[ここ](https://downloads.aspose.com/cells/net).
- **データ**のルート フォルダにあるフォルダ**例**CSharp の例で使用された入力ファイルが含まれています。をダウンロードすることが必須です。**データ**フォルダとサンプル プロジェクト。
- 開ける**RunExamples.cs**、すべての例はここから呼び出されます。
- プロジェクト内から実行するサンプルのコメントを外します。
## **Aspose.Cells.GridDesktop の例**
- Aspose.Cells.GridDesktop の例も含まれています[Aspose.Cells GitHub](https://github.com/aspose-cells/Aspose.Cells-for-.NET)からダウンロードできる ZIP ファイルの一部として利用できます。[ここ](https://github.com/aspose-cells/Aspose.Cells-for-.NET/archive/master.zip).
- すべての例は次の場所にあります。**Examples_GridDesktop**フォルダ。
- Aspose.Cells の例と同様に、GridWeb の例のソリューション ファイル名は**Aspose.Cells.GridDesktop.Examples.CSharp.sln**.
- Visual Studio でソリューション ファイルを開き、プロジェクトをビルドします。
- すべての依存関係は、サンプル プロジェクトの一部として含まれています。から個別に DLL をダウンロードすることもできます。[ここ](https://downloads.aspose.com/cells/net).
- **データ**のルート フォルダにあるフォルダ**Examples_GridDesktop**例で使用される入力ファイルが含まれています。をダウンロードすることが必須です。**データ**フォルダとサンプル プロジェクト。
- プロジェクトを開いて実行します。
- フォーム内から実行したいメニューの例をクリックします。
