## **ソフトウェア要件**
例をダウンロードして実行する前に、以下の要件を満たしていることを確認してください。

1. Visual Studio 2015 またはそれ以降
1. Visual Studio にインストールされている NuGet パッケージマネージャ。Visual Studio 2015 にはほとんどインストールされています。NuGet パッケージマネージャのインストール方法の詳細については、[NuGet クライアント ツールのインストール](https://docs.microsoft.com/en-us/nuget/install-nuget-client-tools) を参照してください。
1. Tools->Options->NuGet Package Manager->Package Sources へ移動し、オプション **nuget.org** がチェックされていることを確認してください
1. 例プロジェクトは NuGet 自動パッケージの復元機能を使用しているため、アクティブなインターネット接続が必要です。実行するマシンにアクティブなインターネット接続がない場合は、[インストール](/cells/ja/net/installation-and-deployment/) を参照し、例プロジェクトに Aspose.Cells.dll の参照を手動で追加してください。
## **GitHub からダウンロード**
Aspose.Cells for .NETのすべての例は[GitHub](https://github.com/aspose-cells/Aspose.Cells-for-.NET)でホストされています。
## **Aspose.Cells の例**
- お気に入りの GitHub クライアントを使用してリポジトリをクローンするか、[こちら](https://github.com/aspose-cells/Aspose.Cells-for-.NET/archive/master.zip) から ZIP ファイルをダウンロードしてください。
- ZIP ファイルの内容をコンピュータの任意のフォルダに展開します。すべての例は **Examples** フォルダにあります。
- C# の例用の Visual Studio ソリューション ファイルがあります。 **Aspose.Cells.Examples.CSharp.sln**。
- プロジェクトは Visual Studio 2015 で作成および管理されています。
- Visual Studio でソリューションファイルを開いてプロジェクトを構築します。
- 初回実行時に、依存関係は自動的に NuGet を介してダウンロードされます。[こちら](https://downloads.aspose.com/cells/net) から DLL を別途ダウンロードすることもできます。
- **Examples** のルートフォルダにある **Data** フォルダには、CSharp の例で使用される入力ファイルが含まれています。例プロジェクトと一緒に **Data** フォルダをダウンロードする必要があります。
- **RunExamples.cs** を開き、全ての例はここから呼び出されます。
- プロジェクト内から実行したい例のコメントを外します。
## **Aspose.Cells.GridDesktop の例**
- Aspose.Cells.GridDesktop の例は [Aspose.Cells GitHub](https://github.com/aspose-cells/Aspose.Cells-for-.NET) リポジトリにも含まれており、[こちら](https://github.com/aspose-cells/Aspose.Cells-for-.NET/archive/master.zip) からダウンロード可能な ZIP ファイルの一部として利用できます。
- すべての例は **Examples_GridDesktop** フォルダにあります。
- Aspose.Cells の例と同様に、GridWeb の例のソリューション ファイル名は **Aspose.Cells.GridDesktop.Examples.CSharp.sln** です。
- Visual Studio でソリューションファイルを開いてプロジェクトを構築します。
- すべての依存関係は例のプロジェクトの一部として含まれています。[こちら](https://downloads.aspose.com/cells/net) から DLL を別途ダウンロードすることもできます。
- **Examples_GridDesktop** のルートフォルダにある **Data** フォルダには、例で使用される入力ファイルが含まれています。例プロジェクトと一緒に **Data** フォルダをダウンロードする必要があります。
- プロジェクトを開いて実行します。
- フォーム内から実行したいメニューの例をクリックします。
## **Aspose.Cells.GridWeb の例**
- Aspose.Cells.GridWeb の例は[Aspose.Cells GitHub](https://github.com/aspose-cells/Aspose.Cells-for-.NET)リポジトリに含まれており、ZIPファイルは[こちら](https://github.com/aspose-cells/Aspose.Cells-for-.NET/archive/master.zip)からダウンロード可能です。
- すべての例は**Examples_GridWeb**フォルダにあります。
- Aspose.Cells の例と同様に、GridWeb の例のソリューションファイルの名前は**Aspose.Cells.GridWeb.Examples.CSharp.sln**です。
- Visual Studio でソリューションファイルを開いてプロジェクトを構築します。
- すべての依存関係は例のプロジェクトの一部として含まれています。DLL は[こちら](https://downloads.aspose.com/cells/net)から個別にダウンロードすることもできます。
- **Examples_GridWeb**のルートフォルダにある**Data**フォルダには、例で使用される入力ファイルが含まれています。例のプロジェクトと一緒に**Data**フォルダをダウンロードすることが必須です。
- 例のプロジェクトで**Examples.aspx**を開いて実行します。
- プロジェクト内から実行したい例をブラウザでクリックします。
