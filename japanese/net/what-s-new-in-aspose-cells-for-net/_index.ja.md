このページでは、最近のリリースで導入された最も興味深い Aspose.Cells の新機能について説明します。

##  Aspose.Cells for .NET 23.5

* 外部シート データの外部参照を現在のワークブックのローカル シートに更新するサポート
* チェックボックス コントロールを静的な画像ではなく対話型コントロールとしてレンダリングするサポート
* ピボットエリアの選択とフォーマットのサポート
* PDFへの変換中に透かしを追加するサポート

##  Aspose.Cells for .NET 23.4

* 関数ENCODEURLの計算のサポート
* 関連するノードを段階的に挿入することで方程式図形の構築を完了するためのサポート
* 数式をセルオブジェクトに設定せずに配列数式として計算できるようになりました。
* LightCells モードでの xlsb ファイルの保存のサポート
* つのフォントがインストールされているか、または利用可能であるかを確認するサポート
* ListObject 内の指定されたセルに数式を設定するサポート
* ワークブックを xhtml として保存するサポート
* GridWeb の activex コントロールとフォーム コントロールのレンダリングのサポート

##  Aspose.Cells for .NET 23.3

* 新しい関数の設定/読み取り/保存のサポート: LET、ENCODEURL
* 数式計算を呼び出すときにデータ ソースを外部リンクにリンクするサポート

##  Aspose.Cells for .NET 23.2

* 新しい関数の設定/読み取り/保存のサポート: SCAN、LAMBDA
* スタイルを JSON にエクスポートするサポート

##  Aspose.Cells for .NET 23.1

* 新しい関数のサポート: CHOSEROWS、CHOOSECOLS、HSTACK、VSTACK
* リビジョンから作成者と日付時刻を取得するためのサポート

##  Aspose.Cells for .NET 22.12

* 多層階層構造を持つ Excel JSON へのエクスポートのサポート

##  Aspose.Cells for .NET 22.11

* 動的配列数式とレガシー配列数式の区別のサポート
* xlsb ファイル形式の動的配列数式の読み取り/保存のサポート

##  Aspose.Cells for .NET 22.10

* SkiaSharp を NET6 のグラフィックス ライブラリとして使用するためのサポート
* CRTX テンプレート ファイルを使用したチャートの追加と更新のサポート

##  Aspose.Cells for .NET 22.9

* セルの表式の設定のサポート
* カスタム関数と計算エンジンによる動的配列数式の設定のサポート
* グラデーション塗りつぶしフォーマットのプリセットグラデーション設定のサポート
* 範囲のテーマカラーを使用したアウトライン境界線の適用のサポート

##  Aspose.Cells for .NET 22.8

* シートに表示されているコメントとメモの HTML へのエクスポートをサポート
* PDF/XPSへの変換中に出力するワークシートの指定をサポート

##  Aspose.Cells for .NET 22.7

* ページヘッダー/フッターを含む単一ファイルへの HTML のエクスポートのサポート
* Range の絶対位置とサイズ (ポイント単位) を取得するためのサポート

##  Aspose.Cells for .NET 22.6

* 数式計算チェーンによる数式の計算とトレースの改善

##  Aspose.Cells for .NET 22.5

* LightCells モードでワークブックを保存する際のセル数式設定のメモリ パフォーマンスの改善

##  Aspose.Cells for .NET 22.4

* CSV をエクスポートするための空白セルを制御するオプションが強化されました。
* 挿入/削除操作の条件付き書式の複雑な分割と更新をサポート

##  Aspose.Cells for .NET 22.3

* 定義名が変更され、ワークブックの計算チェーンが有効になっている場合に、定義名に依存するセルの再計算をサポートします。
* 数式の計算チェーンに従ってセルのリーフを再帰的に取得するためのサポート

##  Aspose.Cells for .NET 22.2

* 指定されたスコープ フィルターによって NameCollection から定義された名前を抽出するためのサポート。

##  Aspose.Cells for .NET 22.1

* 編集可能なpptxの生成をサポート
* データ行/列が制限を超えた場合に、大きな CSV を複数のワークシートにインポートするサポート
* LightCells を使用した CSV インポートのサポート
* 指定された外部リンクの削除のサポート
* 外部リンクを削除しながら数式を保持するサポート

##  Aspose.Cells for .NET 21.12

* Excel から SQL スクリプトへの変換をサポート
* プレーン XML データのインポートとエクスポートのサポート
* 破損した xls ファイルから可能な限り多くのデータを読み取るためのサポート
* SmartMarker のプロセスを中断するためのサポート
* 条件付き書式設定と検証のための範囲を追加するパフォーマンスの向上
* MS Excelの動作と同様に、新しく追加したフォーマット条件の優先度を最高に設定します。

##  Aspose.Cells for .NET 21.11

* XLookup 式/関数の計算をサポートします。
* JSON と画像の保存をワークブックの保存に統合します。
* Excel を HTML に変換する際のパフォーマンスが向上しました。
